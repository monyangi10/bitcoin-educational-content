## บทนำ



Mini Miner braiins BMM 100 เป็นผลิตภัณฑ์ที่สร้างขึ้นโดย Mining pool Braiins อุปกรณ์นี้มีการออกแบบที่น่าสนใจและเงียบมาก มันผลิตพลังการคำนวณได้ 1.1 Th/s และใช้พลังงานประมาณ 40 วัตต์ แตกต่างจากอุปกรณ์อื่น ๆ มันไม่ใช่โอเพนซอร์ส แต่ติดตั้งได้ง่ายมากจริง ๆ ใช้เวลาเพียงไม่กี่คลิกเท่านั้น! Mini Miner BMM 100 เป็นเวอร์ชันแรกที่ปล่อยออกมา ตอนนี้เวอร์ชัน 2 กำลังผลิตอยู่ ชื่อว่า BMM 101 ซึ่งแตกต่างจากเวอร์ชันแรกด้วยการมีหน้าจอที่ใหญ่ขึ้นและมี Wi-Fi แต่ขั้นตอนการติดตั้งยังคงเหมือนเดิม



คุณยังสามารถค้นหาข้อมูลสำคัญเพิ่มเติมได้โดยตรวจสอบคู่มือฉบับสมบูรณ์ได้โดยตรงที่ [เว็บไซต์ของผู้ผลิต](https://braiins.com/hardware/mini-Miner-bmm-100)

## การเชื่อมต่อ Mini Miner



คุณจะต้องเชื่อมต่ออุปกรณ์กับอินเทอร์เน็ตผ่านสายอีเธอร์เน็ต โปรดทราบว่าในเวอร์ชันใหม่ (BMM 101) ไม่จำเป็นต้องทำเช่นนี้อีกต่อไป กลับมาที่ Miner ขนาดเล็กนี้ เมื่อเราหาตำแหน่งได้แล้ว เราจะต้องเชื่อมต่อมันกับสายอินเทอร์เน็ตก่อน แล้วจึงเชื่อมต่อกับไฟฟ้า: อุปกรณ์จะเปิดโดยอัตโนมัติและแสดงที่อยู่ IP บนหน้าจอ



## การกำหนดค่า



เราจำเป็นต้องเปิดเบราว์เซอร์และป้อนที่อยู่ IP ที่แสดงให้เราเห็น mini Miner ในแถบค้นหา ฉันขอเตือนคุณว่าในการค้นหาอุปกรณ์บนเครือข่ายคุณจะต้องอยู่ในพื้นที่ ดังนั้นคุณจะต้องเชื่อมต่อคอมพิวเตอร์ที่คุณใช้งานกับเครือข่ายเดียวกันกับ mini Miner เมื่อเราป้อนที่อยู่ IP แล้วให้กด enter และหน้าล็อกอินเข้าสู่ระบบปฏิบัติการของ mini Miner ซึ่งคือ Braiins OS จะปรากฏบนหน้าจอ



![image](assets/en/06.webp)



ในการเข้าสู่ระบบ คุณจะต้องป้อน `root` เป็นชื่อผู้ใช้ของคุณ ในขณะที่คุณสามารถเว้นรหัสผ่านว่างไว้ได้ คลิกที่เข้าสู่ระบบและแดชบอร์ด Miner ขนาดเล็กของคุณจะปรากฏขึ้น



![image](assets/en/07.webp)

### Braiins pool



ในการเชื่อมต่อกับพูลนี้ เราจำเป็นต้องสร้างบัญชี พูลนี้ยังทำการชำระเงินโดยใช้ Lightning Network ดังนั้นเราจะสามารถรับ Sats ได้ไม่กี่รายการต่อวัน ในการทำเช่นนี้เราจำเป็นต้องตั้งค่า Address lightning เพื่อรับรางวัล หากคุณไม่ทราบวิธีการสร้างบัญชีบน braiins pool หรือวิธีการตั้งค่า Address lightning ของคุณ คุณสามารถทำตามคำแนะนำนี้:



https://planb.academy/it/tutorials/mining/pool/braiins-pool-557be706-35a9-4375-a563-d55ab5c69f55

เมื่อเสร็จสิ้นแล้ว เราจะอยู่ในแดชบอร์ดของ Braiins pool สิ่งที่เราต้องทำคือบอกพูลว่าเราต้องการเชื่อมต่อกับหนึ่งในเครื่องขุดของเรา ดังนั้นทางด้านซ้ายของหน้าจอคุณจะพบกับรายการหลายรายการ เราต้องไปที่ "workers"



![image](assets/en/04.webp)



และเราจำเป็นต้องคลิกที่ปุ่มสีม่วงทางด้านขวาที่เขียนว่า "Connect workers."



![image](assets/en/05.webp)



นี่คือหน้าต่างที่มีข้อมูลที่เราต้องการเพื่อเชื่อมต่อ Miner ขนาดเล็กของเรากับพูล ที่นี่การเปลี่ยนแปลงเดียวที่เราสามารถทำได้คือการเลือก Stratum V2 หากต้องการทราบว่า Stratum v2 คืออะไร ดูรายการนี้ใน [glossary](https://planb.academy/en/resources/glossary/stratum-v2)



![image](assets/en/10.webp)



ตอนนี้เราจำเป็นต้องคัดลอกสตริงนี้ที่เริ่มต้นด้วย stratumv2 ดังนั้นเราคลิกที่สัญลักษณ์ "คัดลอก" เล็ก ๆ จากนั้นเราไปที่แดชบอร์ดของ Miner ขนาดเล็กของเราที่เราได้ทิ้งไว้ในส่วนการกำหนดค่าและพูล เราคลิกที่เพิ่มพูลใหม่



![image](assets/en/11.webp)



และวางสตริงที่เราคัดลอกลงในช่องด้านล่าง URL ของพูล



![image](assets/en/12.webp)



ตอนนี้เราจำเป็นต้องเพิ่มชื่อผู้ใช้และรหัสผ่าน กลับไปที่แดชบอร์ดของพูล ใต้ตรงนี้เรายังมี userID และรหัสผ่าน userID และชื่อผู้ใช้ของเรา ซึ่งเป็นชื่อที่เราให้เมื่อสร้างบัญชี รวมถึงชื่อของ Miner ที่เราต้องการใส่เข้าไป คุณสามารถตัดสินใจได้ว่าจะให้ชื่อกับอุปกรณ์ที่คุณกำลังเชื่อมต่อกับพูลหรือไม่ มันเป็นทางเลือก แต่แนะนำให้ใส่เข้าไป เพราะถ้าคุณเชื่อมต่ออุปกรณ์หลายตัวจะทำให้ง่ายต่อการระบุทันที หากคุณไม่ต้องการใส่อะไรแทน คุณสามารถปล่อย `workerName` ไว้ได้



![image](assets/en/13.webp)



จากนั้นเราไปที่ mini Miner ของเราและป้อนชื่อผู้ใช้ ที่นี่เราจะป้อนในกรณีของฉัน "finalstepbitcoin" ซึ่งเป็น userID ของฉัน, miniminer dot. นี่คือชื่อที่ฉันตัดสินใจตั้งให้กับอุปกรณ์ หากคุณไม่ต้องการตั้งชื่อก็เพียงแค่เขียน userid dot workername ในกรณีของฉันจะเป็น finalstepbitcoin.workername เมื่อคุณป้อนชื่อผู้ใช้แล้วคุณสามารถเลือกพาสเวิร์ดและเขียนลงในช่องว่าง คุณยังสามารถใส่ anithing123 ซึ่งเป็นที่แสดงในหน้าจอสระ แต่เพียงแค่ต้องการบอกว่าคุณสามารถใส่พาสเวิร์ดใดก็ได้ที่คุณต้องการ



เมื่อคุณป้อนข้อมูลทั้งหมดแล้ว คุณต้องกดปุ่มบันทึกทางด้านขวา (ปุ่มที่มีรูปร่างเหมือนแผ่นดิสก์) และด้วยวิธีนี้คุณได้ตั้งค่าข้อมูลสระใน mini Miner แล้ว



![image](assets/en/14.webp)



ตอนนี้คุณต้องกลับไปที่แดชบอร์ดของพูลและคลิกที่ "เชื่อมต่อแล้ว! กลับไป"



![image](assets/en/15.webp)



เราได้เชื่อมต่อ mini Miner ของเรากับ braiins pool แล้ว! ตอนนี้คุณสามารถเห็นมันในรายการของคนงานได้แล้ว หากมันไม่ปรากฏขึ้นมา ให้ทำการรีเฟรชและรอสักครู่ เมื่อมันปรากฏขึ้น ให้ตรวจสอบว่ามันมีสถานะ "OK" พร้อมเครื่องหมายถูกสีเขียว



![image](assets/en/17.webp)



หากคุณกลับไปที่แดชบอร์ด คุณควรเริ่มเห็นการเคลื่อนไหวบนกราฟและเห็น Hashrate ของอุปกรณ์ของเรา ซึ่งหมายความว่าสระกำลังรับงานของเราและด้วยเหตุนี้เราจึงกำลังขุดอยู่ในทุกเจตนาและวัตถุประสงค์

### Public Pool



ผ่านพูลนี้เราสามารถลองเสี่ยงโชคและขุดเดี่ยว โดยพึ่งพาพูล ในกรณีนี้เราจะไม่ได้รับรางวัล แต่เราจะได้รับรางวัลเต็มจำนวนหากเราสามารถขุดบล็อกได้สำเร็จ จากนั้นเราจะเชื่อมโยงไปยังพูลสาธารณะ พูลที่เป็นโอเพ่นซอร์สทั้งหมดที่ใช้เฉพาะ Mining เราเปิดหน้าต่างใหม่บนเบราว์เซอร์และไปที่ [web.public-pool.io](https://web.public-pool.io/#/).



![image](assets/en/18.webp)



มีหน้าที่มีข้อมูลทั้งหมดที่เราต้องการ จากนั้นเราคัดลอกที่อยู่ stratum ที่นั่น



![image](assets/en/19.webp)



จากนั้นเรากลับไปที่แดชบอร์ดของ Miner ขนาดเล็กของเรา ไปที่การกำหนดค่าและไปที่พูล คลิกที่เพิ่มพูลใหม่ (กระบวนการเดียวกับที่เห็นด้านบน) และวาง 'stratum address' ภายใต้ pool url.



![image](assets/en/20.webp)



ตอนนี้ให้เรากลับไปที่หน้าสระว่ายน้ำและดูว่าในช่องชื่อผู้ใช้เราต้องป้อนที่อยู่ Bitcoin ซึ่งจะเป็นที่อยู่ที่เราจะได้รับรางวัลในกรณีที่เราขุดบล็อกสำเร็จ จากนั้นใส่จุดและตามด้วยชื่ออุปกรณ์ของเรา เช่นเดียวกับที่เราทำก่อนหน้านี้กับ Braiins Pool ส่วนรหัสผ่านเราสามารถเลือกเองได้



![image](assets/en/21.webp)



เรากลับไปที่ mini Miner และภายใต้ชื่อผู้ใช้เราวางที่อยู่ Bitcoin ตามด้วยจุดและชื่อ ฉันจะใส่ `miniminer` ในรหัสผ่านแทนฉันจะใส่ test คุณใส่อะไรก็ได้ที่คุณต้องการ



![image](assets/en/22.webp)



ตอนนี้เราบันทึกการตั้งค่าและปิดใช้งาน Braiins pool.



![image](assets/en/23.webp)
