Misty Breez เป็น wallet แบบ Lightning self-holding ที่พัฒนาโดย Breez โดยอิงจากชุดพัฒนาซอฟต์แวร์ของพวกเขาและเครือข่าย **Liquid** ที่พัฒนาโดย BlockStream.


มันมาพร้อมกับแนวทางใหม่ทั้งหมดในการดำเนินการโดยไม่ต้องใช้โหนด Lightning: เป็นไปได้ว่าเป็น **ตัวเปลี่ยนเกม** ในการโอนระหว่างเครือข่าย Bitcoin


ในบทแนะนำนี้ เราจะอธิบายว่า wallet นี้ทำงานอย่างไรและให้ภาพรวมที่สมบูรณ์แก่คุณ



## Misty Breez ทำงานอย่างไร?



Misty Breez เป็นการใช้งานโดยไม่มีโหนด Lightning เป็นแบ็กเอนด์ ได้รับการพัฒนาบนพื้นฐานของ Breez SDK และ Liquid



Liquid เป็นเลเยอร์คู่ขนานกับเครือข่าย Bitcoin ที่มีการปรับปรุงความเร็วและต้นทุนการทำธุรกรรมอย่างมีนัยสำคัญ เลเยอร์นี้ช่วยให้ Misty Breez ไม่จำเป็นต้องใช้โหนด Lightning และสามารถใช้บริการแลกเปลี่ยนจากบุคคลที่สาม เช่น **Boltz** เพื่อให้แน่ใจว่ามีการทำงานร่วมกันระหว่างเครือข่าย Liquid และเครือข่าย Lightning อย่ารีบร้อน เราจะกลับมาที่นี่อีกครั้ง



สำหรับตอนนี้ มาเริ่มการผจญภัยของเรากับ Misty Breez wallet กันเถอะ



## เริ่มต้นใช้งาน Misty Breez



แอปพลิเคชันมือถือ Misty Breez สามารถดาวน์โหลดได้จากแพลตฟอร์มดาวน์โหลดอย่างเป็นทางการ เช่น Google Play Store (บน Android) และ Apple Store (บน iOS) คุณยังสามารถถูกนำทางไปยังแอปที่ถูกต้องจากเว็บไซต์ [Misty Breez] อย่างเป็นทางการ (https://breez.technology/misty/)



⚠️ Make sure you don't confuse Misty Breez with the Breez wallet.



⚠️ **IMPORTANT**: เพื่อความปลอดภัยของบิตคอยน์ของคุณ จำเป็นต้องดาวน์โหลดแอปพลิเคชันจากแพลตฟอร์มอย่างเป็นทางการเพื่อรับรองความถูกต้องของแอปพลิเคชัน



![download-misty-breez](assets/fr/01.webp)



ในบทแนะนำนี้ เราจะเริ่มต้นจากอุปกรณ์ Android อย่างไรก็ตาม แต่ละขั้นตอนและคุณสมบัติเฉพาะที่อธิบายไว้ในส่วนนี้สามารถใช้กับ iOS ได้เช่นกัน



เมื่อติดตั้ง Misty Breez คุณจะมีตัวเลือกในการสร้าง wallet ใหม่หรือกู้คืน Lightning wallet เก่าที่คุณมีคำกู้คืน


ในบทแนะนำนี้ เราเลือกที่จะสร้าง wallet ใหม่



⚠️Misty Breez กำลังอยู่ในขั้นตอนการพัฒนา ดังนั้นเราขอแนะนำให้คุณเริ่มต้นด้วยปริมาณที่เหมาะสม



![create-wallet](assets/fr/02.webp)


### บันทึกคำกู้คืนของคุณ :


หนึ่งในสิ่งแรกที่คุณควรทำเมื่อสร้าง wallet ใหม่คือการสำรองคำกู้คืน 12 คำของคุณ


นี่คือเคล็ดลับบางประการเกี่ยวกับวิธีการสำรองข้อมูลวลีสำรองของคุณ



https://planb.academy/tutorials/wallet/backup/backup-mnemonic-22c0ddfa-fb9f-4e3a-96f9-46e2a7954270

ในการสำรองวลีของคุณ ให้เลือกเมนู **Preferences > Security** จากนั้นเลือกตัวเลือก **Check your Backup Phrase**



![backup](assets/fr/03.webp)


เพื่อความปลอดภัยเพิ่มเติม คุณยังสามารถ**สร้างรหัส PIN** เพื่อยืนยันการเข้าถึง wallet ของคุณได้




ค้นหาสกุลเงินท้องถิ่นของคุณในหลากหลายสกุลเงินที่ Misty Breez ยอมรับ กำหนดค่าสกุลเงินของคุณจากเมนู **Preferences > Fiat Currencies** จากนั้นเลือกสกุลเงินหรือสกุลเงินที่คุณต้องการ



![devises](assets/fr/04.webp)



### การทำธุรกรรมครั้งแรกของคุณ


หากคุณคุ้นเคยกับ Breez wallet อยู่แล้ว คุณจะไม่รู้สึกแปลกใจกับ Misty Breez ที่มี Interface ที่ใช้งานง่ายเลย



ในเมนู Interface **Balance** ให้คลิกที่ตัวเลือก **Receive** เพื่อสร้างใบแจ้งหนี้เพื่อรับบิตคอยน์ของคุณบน wallet ของคุณ



⚠️ Misty Breez จะขอให้คุณเปิดการแจ้งเตือนสำหรับแอปพลิเคชันในการตั้งค่าของโทรศัพท์ของคุณเพื่อให้คุณมีสิทธิ์ได้รับที่อยู่ Lightning



ด้วย Misty Breez, คุณสามารถ :




- รับบิตคอยน์บนเครือข่าย Lightning ตั้งแต่ **100 ซาโตชิ** จนถึง **25,000,000 ซาโตชิ**.
- รับบิตคอยน์บนเครือข่ายหลัก Bitcoin จาก **25,000 ซาโตชิ**.



![transactions](assets/fr/05.webp)



นี่คือจุดเริ่มต้นของความมหัศจรรย์ของ Misty Breez


ต่างจาก Breez wallet ซึ่งให้คุณใช้ Lightning node และขอให้คุณรับผิดชอบค่าใช้จ่ายในการเปิดและปิดช่องทางการชำระเงินด้วยตัวเอง Misty Breez ไม่ขอให้คุณทำอะไรเลย ดังที่ได้กล่าวไว้ก่อนหน้านี้ Misty Breez ไม่ได้ทำงานบนพื้นฐานของ Lightning node เลยด้วยซ้ำ



มาดูเบื้องหลังกันให้ละเอียดขึ้น



ในความเป็นจริง คุณเป็นเจ้าของ Liquid wallet ซึ่งเชื่อมโยงกับ Misty Breez wallet ของคุณ ตามหลักเหตุผล คุณจะจัดการ L-BTC (Liquid Bitcoin) ที่อัตราคงที่ซึ่งเชื่อมโยงกับบริการแปลง satoshis ใต้ทะเลของบุคคลที่สามที่จะช่วยให้คุณสามารถทำงานร่วมกับเครือข่าย Lightning ได้



เมื่อคุณได้รับการชำระเงินบน Misty Breez wallet ผู้ส่งของคุณจะส่ง satoshis ซึ่งจะผ่านบริการแปลงเช่น Boltz (ที่ปัจจุบันใช้โดย Misty Breez) เพื่อแปลง satoshis ที่ส่งเป็น L-BTC ซึ่งจะได้รับบน Misty Breez wallet ของคุณ (เชื่อมโยงกับ Liquid wallet)


นี่คือแผนภาพที่เรียบง่ายของกระบวนการเบื้องหลัง.



![lnswap-in](assets/fr/06.webp)



คลิกที่ Interface ในเมนู **Balance** คลิกที่ตัวเลือก **Send** เพื่อชำระใบแจ้งหนี้ Lightning


แทรกใบแจ้งหนี้ Lightning, ที่อยู่ Lightning ของผู้รับ หรือเพียงแค่สแกนรหัส QR บนใบแจ้งหนี้เพื่อทำการชำระเงินของคุณ



![send-bitcoins](assets/fr/07.webp)



เบื้องหลัง คุณเปิดใช้งาน Liquid wallet ที่เชื่อมโยงกับ Misty Breez wallet ของคุณเพื่อแปลงค่าเทียบเท่าของ L-BTC เป็น satoshis ผ่าน Boltz จากนั้นโอน satoshis เหล่านี้ไปยัง Lightning wallet ของผู้รับของคุณ (ที่มีอยู่บนเครือข่าย Lightning)



![send-bitcoin-bts](assets/fr/08.webp)



คุณลักษณะนี้ของโครงสร้างพื้นฐาน Misty Breez ช่วยให้ผู้ใช้สามารถทำธุรกรรมได้แม้ว่า Misty Breez จะออฟไลน์



สำหรับผู้มีประสบการณ์มากขึ้น ยังมีเมนู **Preferences > Developers** ที่ให้รายละเอียดเพิ่มเติมเกี่ยวกับ:




- เวอร์ชันของชุดพัฒนาโปรแกรม Breez
- คีย์สาธารณะของคุณ Misty Breez wallet.
- ผู้กู้, ตัวระบุเฉพาะที่ได้มาจากกุญแจสาธารณะหลัก
- ยอดคงเหลือ wallet ของคุณ
- เคล็ดลับ Liquid สำหรับการส่งจำนวนเล็กน้อยของ L-BTC
- Bitcoin Tip สำหรับการส่งจำนวนเล็กน้อยของ Bitcoin



คุณยังสามารถดำเนินการบางอย่างได้ เช่น การซิงโครไนซ์กับเครือข่าย Liquid การสำรองกุญแจของคุณ การแชร์บันทึกกิจกรรมของคุณ และการเลือกสแกนเครือข่าย Liquid ใหม่
