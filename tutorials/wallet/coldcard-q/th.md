wallet เป็นอุปกรณ์อิเล็กทรอนิกส์ที่ทุ่มเทให้กับการจัดการและรักษาความปลอดภัยของคีย์ส่วนตัวของ Bitcoin wallet แตกต่างจากซอฟต์แวร์วอลเล็ต (หรือฮอตวอลเล็ต) ที่ติดตั้งบนเครื่องทั่วไปที่มักเชื่อมต่อกับอินเทอร์เน็ต ฮาร์ดแวร์วอลเล็ตช่วยให้คีย์ส่วนตัวถูกแยกออกทางกายภาพ ลดความเสี่ยงจากการละเมิดลิขสิทธิ์และการโจรกรรม


วัตถุประสงค์หลักของฮาร์ดแวร์ wallet คือการลดฟังก์ชันการทำงานของอุปกรณ์ให้มากที่สุดเท่าที่จะเป็นไปได้เพื่อที่จะลดพื้นผิวการโจมตีให้น้อยลง พื้นผิวการโจมตีที่น้อยลงยังหมายถึงเวกเตอร์การโจมตีที่เป็นไปได้น้อยลง กล่าวคือ จุดอ่อนในระบบที่ผู้โจมตีอาจใช้ประโยชน์เพื่อเข้าถึงบิตคอยน์จะน้อยลง


ขอแนะนำให้ใช้ฮาร์ดแวร์ wallet เพื่อรักษาความปลอดภัยของบิตคอยน์ของคุณ โดยเฉพาะอย่างยิ่งหากคุณถือครองในปริมาณมาก ไม่ว่าจะเป็นในมูลค่าสัมบูรณ์หรือเป็นสัดส่วนของสินทรัพย์ทั้งหมดของคุณ


กระเป๋าเงินฮาร์ดแวร์ถูกใช้ร่วมกับซอฟต์แวร์การจัดการ wallet บนคอมพิวเตอร์หรือสมาร์ทโฟน ซอฟต์แวร์นี้จัดการการสร้างธุรกรรม แต่ลายเซ็นเข้ารหัสที่จำเป็นในการทำให้ธุรกรรมเหล่านี้ถูกต้องจะดำเนินการเฉพาะภายในฮาร์ดแวร์ wallet เท่านั้น ซึ่งหมายความว่าคีย์ส่วนตัวจะไม่ถูกเปิดเผยต่อสภาพแวดล้อมที่อาจมีช่องโหว่


กระเป๋าเงินฮาร์ดแวร์มอบการป้องกันสองชั้นให้กับผู้ใช้: ในด้านหนึ่ง พวกมันรักษาความปลอดภัยให้กับบิตคอยน์ของคุณจากการโจมตีระยะไกลโดยการเก็บคีย์ส่วนตัวแบบออฟไลน์ และในอีกด้านหนึ่ง พวกมันมักจะมีความทนทานทางกายภาพมากขึ้นต่อความพยายามในการดึงคีย์ออกมา และมันคือเกณฑ์ความปลอดภัยทั้ง 2 ข้อนี้ที่เราสามารถตัดสินและจัดประเภทโมเดลต่าง ๆ ในตลาดได้


ในบทแนะนำนี้ ฉันอยากจะแนะนำให้คุณรู้จักกับหนึ่งในโซลูชันเหล่านั้น: **COLDCARD Q**.

## แนะนำ COLDCARD Q


COLDCARD Q เป็นฮาร์ดแวร์ Bitcoin-only wallet ที่พัฒนาโดยบริษัท Coinkite ของแคนาดา ซึ่งเป็นที่รู้จักจากรุ่น MK ก่อนหน้านี้ Q แทนผลิตภัณฑ์ที่ทันสมัยที่สุดของพวกเขาจนถึงปัจจุบัน และถูกวางตำแหน่งให้เป็นฮาร์ดแวร์ Bitcoin wallet ที่ดีที่สุด


ในแง่ของฮาร์ดแวร์ COLDCARD Q มาพร้อมกับคุณสมบัติทั้งหมดที่จำเป็นสำหรับประสบการณ์การใช้งานที่ดีที่สุด:




- หน้าจอ LCD ขนาดใหญ่ช่วยให้การนำทางและการใช้งานง่ายขึ้น;
- แป้นพิมพ์ QWERTY แบบเต็ม;
- กล้องในตัวสำหรับสแกนคิวอาร์โค้ด;
- ช่องใส่การ์ด microSD สองช่อง ;
- ตัวเลือกพลังงานที่แยกออกมาอย่างสมบูรณ์ผ่านแบตเตอรี่ AAA สามก้อน (ไม่รวมอยู่ด้วย) หรือผ่านสาย USB-C ;
- สอง Secure Elements จากผู้ผลิตสองรายที่แตกต่างกันเพื่อเพิ่มความปลอดภัย;
- ความสามารถในการสื่อสารผ่าน NFC.


ในความเห็นของฉัน COLDCARD Q มีข้อเสียเพียงสองข้อเท่านั้น ข้อแรก เนื่องจากมีฟีเจอร์มากมาย ทำให้มันค่อนข้างเทอะทะ โดยมีขนาดเกือบ 13 ซม. ยาว และ 8 ซม. กว้าง ซึ่งเกือบจะเท่ากับขนาดของสมาร์ทโฟนขนาดเล็ก นอกจากนี้ยังค่อนข้างหนาเนื่องจากช่องใส่แบตเตอรี่ หากคุณกำลังมองหาฮาร์ดแวร์ wallet ที่มีขนาดเล็กและพกพาได้มากกว่า MK4 ที่มีขนาดกะทัดรัดกว่าน่าจะเป็นตัวเลือกที่ดีกว่า ข้อเสียที่สองคือราคาของอุปกรณ์ ซึ่งตั้งราคาไว้ที่ **$239.99** บนเว็บไซต์ทางการ ซึ่งแพงกว่า MK4 ถึง $72 ซึ่งทำให้ Q แข่งขันโดยตรงกับกระเป๋าเงินฮาร์ดแวร์ระดับพรีเมียมอย่าง Ledger Flex หรือ Passport ของ Foundation


![CCQ](assets/fr/001.webp)


ในด้านซอฟต์แวร์ COLDCARD Q มีการติดตั้งที่ดีเทียบเท่ากับอุปกรณ์อื่นๆ ของ Coinkite พร้อมด้วยคุณสมบัติขั้นสูงมากมาย:




- ทอยลูกเต๋าไปที่ generate วลีการกู้คืนของคุณเอง ;
- รหัส PIN ;
- นับถอยหลังสู่การล็อค PIN ขั้นสุดท้าย ;
- BIP39 passphrase ;
- รหัสล็อคสุดท้าย ;
- การนับถอยหลังการเชื่อมต่อ ;
- SeedXOR;
- BIP85...


โดยสรุปแล้ว COLDCARD Q มอบประสบการณ์การใช้งานที่ดีกว่า MK4 และอาจเหมาะสำหรับผู้ใช้ระดับกลางถึงขั้นสูงที่มองหาความสะดวกในการใช้งานที่มากขึ้น


COLDCARD Q มีวางจำหน่ายสำหรับขาย [บนเว็บไซต์ทางการของ Coinkite](https://store.coinkite.com/store/coldcard) นอกจากนี้ยังสามารถซื้อได้จากผู้ค้าปลีก


## กำลังเตรียมบทแนะนำ


เมื่อคุณได้รับ COLDCARD ขั้นตอนแรกคือการตรวจสอบบรรจุภัณฑ์เพื่อให้แน่ใจว่าไม่ได้ถูกเปิดออก หากบรรจุภัณฑ์เสียหาย อาจบ่งชี้ว่าอุปกรณ์ wallet ถูกดัดแปลงและอาจไม่ใช่ของแท้


![CCQ](assets/fr/002.webp)


เมื่อคุณเปิดกล่อง คุณควรพบรายการต่อไปนี้:




- COLDCARD Q ในถุงปิดผนึก;
- การ์ดสำหรับบันทึกวลีช่วยจำของคุณ


![CCQ](assets/fr/003.webp)


ตรวจสอบให้แน่ใจว่าถุงไม่ได้ถูกเปิดหรือเสียหาย นอกจากนี้ตรวจสอบว่าหมายเลขบนถุงของคุณตรงกับหมายเลขบนกระดาษภายในถุง เก็บหมายเลขนี้ไว้สำหรับการอ้างอิงในอนาคต


![CCQ](assets/fr/004.webp)


หากคุณต้องการใช้งาน COLDCARD โดยไม่เชื่อมต่อกับคอมพิวเตอร์ (air-gap) ให้ใส่แบตเตอรี่ AAA สามก้อนที่ด้านหลังของอุปกรณ์ อีกทางเลือกหนึ่ง คุณสามารถเชื่อมต่อกับคอมพิวเตอร์ของคุณผ่านสาย USB-C ได้


![CCQ](assets/fr/005.webp)


สำหรับบทแนะนำนี้ คุณจะต้องใช้ Sparrow Wallet เพื่อจัดการ Bitcoin wallet บนคอมพิวเตอร์ของคุณ ดาวน์โหลด [Sparrow Wallet](https://sparrowwallet.com/download/) จากเว็บไซต์ทางการ ฉันขอแนะนำอย่างยิ่งให้คุณตรวจสอบทั้งความถูกต้อง (ด้วย GnuPG) และความสมบูรณ์ (ผ่านแฮช) ก่อนดำเนินการติดตั้ง หากคุณไม่ทราบวิธีการทำเช่นนี้ ให้ทำตามบทแนะนำนี้:


https://planb.academy/tutorials/computer-security/data/integrity-authenticity-21d0420a-be02-4663-94a3-8d487f23becc

## การเลือก PIN code


คุณสามารถเปิดใช้งาน COLDCARD ของคุณได้โดยกดปุ่มที่มุมซ้ายบน


![CCQ](assets/fr/006.webp)


กดปุ่ม "*ENTER*" เพื่อยอมรับเงื่อนไขการใช้งาน


![CCQ](assets/fr/007.webp)


COLDCARD Q ของคุณจะแสดงตัวเลขที่ด้านบนของหน้าจอ ตรวจสอบให้แน่ใจว่าตัวเลขนี้ตรงกับตัวเลขบนถุงที่ปิดผนึกและบนชิ้นส่วนพลาสติกภายในถุง ซึ่งจะช่วยให้มั่นใจได้ว่าพัสดุของคุณไม่ได้ถูกเปิดระหว่างเวลาที่ Coinkite บรรจุและเวลาที่คุณเปิด กด "*ENTER*" เพื่อดำเนินการต่อ


![CCQ](assets/fr/008.webp)


ไปที่เมนู "*เลือก PIN Code*" และยืนยันด้วยปุ่ม "*ENTER*"


![CCQ](assets/fr/009.webp)


รหัส PIN นี้ใช้เพื่อปลดล็อก COLDCARD ของคุณ ดังนั้นจึงเป็นการป้องกันการเข้าถึงทางกายภาพโดยไม่ได้รับอนุญาต รหัส PIN นี้ไม่เกี่ยวข้องกับการสร้างคีย์เข้ารหัสของ wallet ของคุณ ดังนั้น แม้จะไม่มีการเข้าถึงรหัส PIN นี้ การครอบครองวลีช่วยจำของคุณจะช่วยให้คุณสามารถเข้าถึงบิตคอยน์ของคุณได้อีกครั้ง


รหัส PIN ของ COLDCARD แบ่งออกเป็นสองส่วน: คำนำหน้าและคำต่อท้าย ซึ่งแต่ละส่วนสามารถมีได้ระหว่าง 2 ถึง 6 หลัก รวมทั้งหมด 4 ถึง 12 หลัก เมื่อคุณปลดล็อก COLDCARD ของคุณ คุณจะต้องป้อนคำนำหน้าก่อน จากนั้นอุปกรณ์จะแสดงคำ 2 คำให้คุณดู จากนั้นป้อนคำต่อท้าย คำสองคำนี้จะถูกให้คุณในขั้นตอนการกำหนดค่านี้ และควรบันทึกไว้อย่างระมัดระวัง เนื่องจากคุณจะต้องใช้คำเหล่านี้ทุกครั้งที่ปลดล็อก COLDCARD ของคุณ หากคำสองคำที่แสดงระหว่างการปลดล็อกตรงกับคำที่คุณบันทึกไว้ระหว่างการกำหนดค่า นี่จะยืนยันว่าอุปกรณ์ของคุณไม่ได้ถูกบุกรุกตั้งแต่การใช้งานครั้งล่าสุด


คลิกอีกครั้งที่ "*เลือก PIN*"


![CCQ](assets/fr/010.webp)


ยืนยันว่าคุณได้อ่านคำเตือนแล้ว


![CCQ](assets/fr/011.webp)


คุณจะเลือก PIN code ของคุณตอนนี้ เราแนะนำให้ใช้ PIN code ที่ยาวและสุ่ม ตรวจสอบให้แน่ใจว่าคุณเก็บรหัสนี้ไว้ในที่ที่แตกต่างจากที่เก็บ COLDCARD ของคุณ คุณสามารถใช้การ์ดที่ให้มาในพัสดุของคุณเพื่อบันทึกรหัสนี้


ป้อนคำนำหน้าที่คุณเลือก จากนั้นกดปุ่ม "*ENTER*" เพื่อยืนยันส่วนแรกของรหัส PIN ของคุณ


![CCQ](assets/fr/012.webp)


คำสองคำป้องกันฟิชชิ่งจะปรากฏบนหน้าจอของคุณ จากนั้นบันทึกไว้อย่างระมัดระวังเพื่อใช้อ้างอิงในอนาคต คุณสามารถใช้การ์ดที่รวมอยู่ในแพ็คเกจของคุณเพื่อจดบันทึกคำเหล่านั้นได้


![CCQ](assets/fr/013.webp)


จากนั้นป้อนส่วนที่สองของรหัส PIN ของคุณและกด "*ENTER*".


![CCQ](assets/fr/014.webp)


ยืนยันรหัส PIN ของคุณโดยป้อนอีกครั้ง ตรวจสอบว่าคำป้องกันการฟิชชิ่งสองคำตรงกับที่คุณบันทึกไว้ก่อนหน้านี้


![CCQ](assets/fr/015.webp)


จากนี้ไป ทุกครั้งที่คุณปลดล็อก COLDCARD ของคุณ อย่าลืมตรวจสอบความถูกต้องของคำป้องกันฟิชชิ่งสองคำที่ปรากฏบนหน้าจอหลังจากที่คุณป้อนรหัส PIN prefix ของคุณ


## อัปเดตเฟิร์มแวร์


เมื่อใช้เครื่องของคุณเป็นครั้งแรก สิ่งสำคัญคือต้องตรวจสอบและอัปเดตเฟิร์มแวร์ ในการทำเช่นนี้ ให้เข้าถึงเมนู "*Advanced/Tools*"


![CCQ](assets/fr/016.webp)


**สำคัญ:** หากคุณกำลังวางแผนที่จะอัปเกรดเฟิร์มแวร์และนี่ไม่ใช่ครั้งแรกที่คุณใช้ COLDCARD (เช่น คุณมี wallet ที่สร้างบน COLDCARD แล้ว) โปรดตรวจสอบให้แน่ใจว่าคุณมีวลีช่วยจำและมันใช้งานได้ (รวมถึง passphrase ที่เป็นทางเลือก หากมี) นี่เป็นสิ่งสำคัญเพื่อหลีกเลี่ยงการสูญเสียบิตคอยน์ของคุณในกรณีที่เกิดปัญหาระหว่างการอัปเดตอุปกรณ์


เลือก "*อัปเกรดเฟิร์มแวร์*"


![CCQ](assets/fr/017.webp)


เลือก "*แสดงเวอร์ชัน*"

## สร้าง Bitcoin wallet


ตอนนี้ถึงเวลาที่จะ generate ใหม่ Bitcoin wallet! ในการทำเช่นนี้ คุณจำเป็นต้องสร้างวลีช่วยจำ บน Coldcard คุณมีสามวิธีในการสร้างวลีนี้:




- ใช้เฉพาะเครื่องสร้างตัวเลขสุ่มภายใน (TRNG);
- ใช้การผสมผสานระหว่าง TRNG และการทอยลูกเต๋าเพื่อเพิ่มเอนโทรปี;
- ใช้การทอยลูกเต๋าเท่านั้น


**สำหรับผู้เริ่มต้นและผู้ใช้ระดับกลาง เราแนะนำให้ใช้เฉพาะเครื่องสร้างตัวเลขสุ่มภายในของ COLDCARD เท่านั้น**


ฉันไม่แนะนำตัวเลือกที่ใช้ลูกเต๋าเพียงอย่างเดียว เนื่องจากการดำเนินการที่ไม่ดีอาจนำไปสู่ประโยคที่มีเอนโทรปีไม่เพียงพอ ซึ่งจะทำให้ความปลอดภัยของ wallet ของคุณตกอยู่ในอันตราย


อย่างไรก็ตาม ตัวเลือกที่ดีที่สุดคืออย่างแน่นอนตัวเลือกที่สอง ซึ่งรวมการใช้ TRNG กับแหล่งเอนโทรปีภายนอก วิธีนี้รับประกันเอนโทรปีขั้นต่ำที่เทียบเท่ากับ TRNG เพียงอย่างเดียว และเพิ่มระดับความปลอดภัยเพิ่มเติมในกรณีที่เครื่องกำเนิดภายในล้มเหลว (โดยสมัครใจหรือไม่) โดยการเลือกตัวเลือกนี้ ซึ่งรวม TRNG และการทอยลูกเต๋า คุณจะได้รับประโยชน์จากการควบคุมที่มากขึ้นในการสร้างประโยค โดยไม่เพิ่มความเสี่ยงในกรณีที่คุณดำเนินการไม่ดี


คลิกที่ "*New Seed Words*"


![CCQ](assets/fr/029.webp)


คุณสามารถเลือกความยาวของประโยคได้ ฉันแนะนำให้คุณเลือกประโยคที่มีความยาว 12 คำ เนื่องจากจัดการได้ง่ายกว่าและมีความปลอดภัย wallet ไม่แพ้ประโยคที่มีความยาว 24 คำ


![CCQ](assets/fr/030.webp)


จากนั้น COLDCARD จะแสดงวลีการกู้คืนที่สร้างโดย TRNG หากคุณต้องการเพิ่มเอนโทรปีภายนอกเพิ่มเติม ให้กดปุ่ม "*4*"


![CCQ](assets/fr/031.webp)


สิ่งนี้จะพาคุณไปยังหน้าที่คุณสามารถเพิ่มเอนโทรปีโดยการทอยลูกเต๋า ทอยให้ได้มากที่สุดเท่าที่จะทำได้ (แนะนำให้ทอยอย่างน้อย 50 ครั้ง แต่ถ้าน้อยกว่านี้ก็ไม่เป็นไรเพราะคุณได้รับประโยชน์จากเอนโทรปีของ TRNG อยู่แล้ว) และบันทึกผลลัพธ์ด้วยปุ่ม "*1*" ถึง "*6*" เมื่อเสร็จสิ้น กด "*ENTER*" เพื่อยืนยัน


![CCQ](assets/fr/032.webp)


จะแสดงวลีช่วยจำใหม่ โดยอิงจากเอนโทรปีที่คุณเพิ่งให้และของ TRNG


**คำเตือน: วลีช่วยจำนี้ให้การเข้าถึงบิตคอยน์ของคุณอย่างเต็มที่และไม่มีข้อจำกัด** ใครก็ตามที่ครอบครองวลีนี้สามารถขโมยเงินของคุณได้ แม้จะไม่มีการเข้าถึงทางกายภาพไปยัง COLDCARD ของคุณก็ตาม วลี 12 คำนี้จะคืนการเข้าถึงบิตคอยน์ของคุณในกรณีที่สูญหาย ถูกขโมย หรือเสียหายของ COLDCARD ของคุณ ดังนั้นจึงเป็นสิ่งสำคัญมากที่จะต้องบันทึกไว้อย่างระมัดระวังและเก็บไว้ในที่ปลอดภัย


คุณสามารถเขียนลงบนกระดาษแข็งที่มาพร้อมกับ COLDCARD ของคุณ หรือเพื่อความปลอดภัยเพิ่มเติม ฉันแนะนำให้คุณสลักมันลงบนแผ่นรองสแตนเลสเพื่อป้องกันจากความเสี่ยงของไฟไหม้ น้ำท่วม หรือการพังทลาย ไม่ว่าในกรณีใด อย่าบันทึกมันลงบนสื่อดิจิทัล มิฉะนั้นคุณอาจสูญเสียบิตคอยน์ของคุณได้


เขียนคำที่ปรากฏบนหน้าจอลงบนสื่อทางกายภาพที่คุณเลือก ขึ้นอยู่กับกลยุทธ์ด้านความปลอดภัยของคุณ คุณอาจพิจารณาทำสำเนาทางกายภาพของประโยคหลายชุด (แต่ที่สำคัญ อย่าแยกมันออก) สิ่งสำคัญคือต้องเก็บคำไว้เป็นลำดับหมายเลขและตามลำดับ


เห็นได้ชัดว่า **คุณต้องไม่แชร์คำเหล่านี้** บนอินเทอร์เน็ต ยกเว้นในบทเรียนนี้ ตัวอย่าง wallet นี้จะถูกใช้เฉพาะบน Testnet และจะถูกลบเมื่อสิ้นสุดบทเรียน


หลังจากเขียนคำลงไปแล้ว ให้กด "*ENTER*".


![CCQ](assets/fr/033.webp)


ในการตรวจสอบว่าคุณได้บันทึกวลีของคุณอย่างถูกต้องหรือไม่ ระบบจะขอให้คุณยืนยันคำบางคำ เลือกหมายเลขที่ตรงกับแต่ละคำบนแป้นพิมพ์ตัวเลข


![CCQ](assets/fr/034.webp)


wallet ของคุณถูกสร้างขึ้นแล้ว! ที่มุมขวาบนของหน้าจอ คุณสามารถเห็นลายนิ้วมือคีย์ส่วนตัวหลักของคุณ กด "*ENTER*".

## การตั้งค่า wallet ใหม่บน Sparrow


มีหลายตัวเลือกสำหรับการสร้างการสื่อสารระหว่างซอฟต์แวร์ Sparrow Wallet และ COLDCARD ของคุณ วิธีที่ตรงไปตรงมาที่สุดคือการใช้สาย USB-C อย่างไรก็ตาม โดยค่าเริ่มต้น COLDCARD ของคุณจะปิดการสื่อสารผ่านสายและ NFC เพื่อเปิดใช้งานอีกครั้ง ให้ไปที่ "*Settings*" จากนั้น "*Hardware On/Off*" และเปิดใช้งานตัวเลือกการสื่อสารที่ต้องการ


![CCQ](assets/fr/037.webp)


หากคุณต้องการเก็บ COLDCARD ของคุณให้แยกออกจากคอมพิวเตอร์โดยสิ้นเชิง คุณสามารถเลือกใช้การสื่อสารแบบ "air-gap" ทางอ้อม โดยใช้ QR codes หรือ microSD card นี่คือวิธีที่เราจะใช้ในบทแนะนำนี้

ก่อนที่จะใช้ที่อยู่ที่เสนอโดย Sparrow Wallet ให้ตรวจสอบบนหน้าจอ COLDCARD ของคุณ การปฏิบัตินี้ช่วยให้คุณยืนยันได้ว่าที่อยู่ที่แสดงบน Sparrow ไม่ใช่การหลอกลวง และฮาร์ดแวร์ wallet มีคีย์ส่วนตัวที่จำเป็นในการใช้จ่ายบิตคอยน์ที่ปลอดภัยด้วยที่อยู่นี้ในภายหลัง วิธีนี้ช่วยให้คุณหลีกเลี่ยงการโจมตีหลายประเภทได้


ในการดำเนินการตรวจสอบนี้ ให้คลิกที่เมนู "*Address Explorer*" บน COLDCARD


![CCQ](assets/fr/054.webp)


เลือกประเภทของสคริปต์ที่คุณใช้บน Sparrow ในกรณีของฉันคือ Segwit P2WPKH ฉันคลิกที่มัน

จากนั้นคุณสามารถดูที่อยู่ที่ได้มาของคุณในลำดับที่แตกต่างกัน


![CCQ](assets/fr/056.webp)


ตรวจสอบกับ Sparrow ว่าที่อยู่ตรงกันหรือไม่ ในกรณีของฉัน ที่อยู่ที่มีเส้นทางอนุพันธ์ `m/84'/1'/0'/0/0` คือ `tb1qwfwwvzssep4wyjg3vsgezmwa037ehvd4fhmjvr` ทั้งบน Sparrow และ COLDCARD


![CCQ](assets/fr/057.webp)


อีกวิธีหนึ่งในการยืนยันความเป็นเจ้าของที่อยู่นี้คือการสแกน QR code ของมันโดยตรงไปยัง Sparrow จาก COLDCARD ของคุณ จากหน้าจอหลักของ COLDCARD ของคุณ เลือก "*Scan Any QR Code*" คุณยังสามารถใช้ปุ่ม "*QR*" บนแป้นพิมพ์ได้อีกด้วย

สแกนรหัส QR ของที่อยู่ที่แสดงบน Sparrow Wallet

ที่อยู่ได้รับการยืนยันเรียบร้อยแล้ว

![CCQ](assets/fr/062.webp)


สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการติดฉลาก ฉันขอแนะนำบทแนะนำอื่นนี้:

## ส่งบิตคอยน์


ตอนนี้ที่คุณได้รับ sats แรกของคุณใน wallet ที่ปลอดภัยด้วย COLDCARD แล้ว คุณก็สามารถใช้จ่ายได้เช่นกัน!


เช่นเคย เริ่มต้นด้วยการเปิดและปลดล็อก COLDCARD Q ของคุณ จากนั้นเปิด Sparrow Wallet และไปที่แท็บ "*Send*" เพื่อเตรียมการทำธุรกรรมใหม่


![CCQ](assets/fr/064.webp)


หากคุณต้องการ "ควบคุมเหรียญ" หรือเลือกเฉพาะ UTXOs ที่จะใช้ในธุรกรรม ให้ไปที่แท็บ "*UTXOs*" เลือก UTXOs ที่คุณต้องการใช้จ่าย จากนั้นคลิกที่ "*Send Selected*" คุณจะถูกนำไปยังหน้าจอเดียวกันในแท็บ "*Send*" แต่มี UTXOs ของคุณที่เลือกไว้แล้วสำหรับธุรกรรม


![CCQ](assets/fr/065.webp)


ป้อนที่อยู่ปลายทาง คุณยังสามารถป้อนที่อยู่หลายแห่งได้โดยคลิกที่ปุ่ม "*+ เพิ่ม*"


![CCQ](assets/fr/066.webp)


เขียน "*ป้ายกำกับ*" เพื่อจดจำวัตถุประสงค์ของค่าใช้จ่ายนี้


![CCQ](assets/fr/067.webp)


เลือกจำนวนเงินที่จะส่งไปยังที่อยู่นี้


![CCQ](assets/fr/068.webp)


ปรับอัตราค่าธรรมเนียมของธุรกรรมของคุณตามตลาดปัจจุบัน


![CCQ](assets/fr/069.webp)


ตรวจสอบให้แน่ใจว่าพารามิเตอร์การทำธุรกรรมทั้งหมดของคุณถูกต้อง จากนั้นคลิกที่ "*Create Transaction*"


![CCQ](assets/fr/070.webp)


หากทุกอย่างเป็นที่พอใจของคุณ ให้คลิกที่ "*Finalize Transaction for Signing*"


![CCQ](assets/fr/071.webp)


เมื่อคุณสร้างธุรกรรมใน Sparrow เสร็จแล้ว ก็ถึงเวลาที่จะเซ็นด้วย COLDCARD ของคุณ เพื่อส่ง PSBT (ธุรกรรมที่ยังไม่ได้เซ็น) ไปยังอุปกรณ์ของคุณ คุณมีหลายตัวเลือก หากเปิดใช้งานการส่งข้อมูลผ่านสาย คุณสามารถส่งธุรกรรมโดยตรงผ่านการเชื่อมต่อ USB-C ได้ เช่นเดียวกับฮาร์ดแวร์ wallet อื่น ๆ ในกรณีนี้ บน Sparrow คุณจะต้องคลิกที่ปุ่ม "*Sign*" ที่มุมล่างขวา ในตัวอย่างของฉัน ปุ่มนี้ถูกบล็อกเพราะ COLDCARD ไม่ได้เชื่อมต่อด้วยสายเคเบิล


![CCQ](assets/fr/072.webp)


หากคุณต้องการรักษาการเชื่อมต่อแบบ "air-gap" โดยไม่ต้องมีการติดต่อโดยตรงระหว่างฮาร์ดแวร์ wallet และคอมพิวเตอร์ของคุณ คุณมี 2 ตัวเลือก ตัวเลือกแรกและซับซ้อนกว่าคือการใช้การ์ด microSD ใส่การ์ด microSD ลงในคอมพิวเตอร์ของคุณ บันทึกธุรกรรมผ่านปุ่ม "*Save Transaction*" บน Sparrow จากนั้นโอนการ์ดนี้ไปยังพอร์ตบน COLDCARD ของคุณ


![CCQ](assets/fr/073.webp)


จากนั้นเข้าถึงเมนู "*Ready To Sign*"


![CCQ](assets/fr/074.webp)


ตรวจสอบรายละเอียดการทำธุรกรรมบน COLDCARD ของคุณ รวมถึงที่อยู่ผู้รับ จำนวนเงินที่ส่ง และค่าธรรมเนียมการทำธุรกรรม


![CCQ](assets/fr/075.webp)


หากทุกอย่างถูกต้อง กดปุ่ม "*ENTER*" เพื่อเซ็นชื่อธุรกรรม


![CCQ](assets/fr/076.webp)


จากนั้นใส่ microSD กลับเข้าไปในคอมพิวเตอร์ของคุณและบน Sparrow คลิกที่ "*Load Transaction*" เพื่อโหลดธุรกรรมที่ลงนามแล้วจาก microSD คุณจะสามารถทำการตรวจสอบขั้นสุดท้ายก่อนที่จะอัปโหลดไปยังเครือข่าย Bitcoin


![CCQ](assets/fr/077.webp)


วิธีที่สองในการเซ็นด้วย COLDCARD ของคุณใน Air-Gap ซึ่งง่ายกว่าวิธี microSD มาก คือการสแกน PSBT โดยตรงผ่านกล้องของอุปกรณ์ บน Sparrow ให้เลือก "*Show QR*"


![CCQ](assets/fr/078.webp)


บน COLDCARD เลือก "*Scan Any QR Code*". คุณยังสามารถใช้ปุ่ม "*QR*" บนแป้นพิมพ์ได้อีกด้วย


![CCQ](assets/fr/079.webp)


ใช้กล้องของ COLDCARD เพื่อสแกน QR code ที่แสดงบน Sparrow


![CCQ](assets/fr/080.webp)


รายละเอียดการทำธุรกรรมจะปรากฏอีกครั้งเพื่อการตรวจสอบ กด "*ENTER*" เพื่อเซ็นชื่อหากทุกอย่างเป็นที่พอใจของคุณ


![CCQ](assets/fr/081.webp)


COLDCARD ของคุณจะแสดงธุรกรรมที่ลงนามแล้วเป็น QR code จากนั้นใช้เว็บแคมของคอมพิวเตอร์ของคุณเพื่อสแกน QR code นี้โดยเลือก "*Scan QR*" บน Sparrow


![CCQ](assets/fr/082.webp)


ธุรกรรมที่คุณลงนามแล้วตอนนี้สามารถมองเห็นได้บน Sparrow ตรวจสอบอีกครั้งว่าทุกอย่างถูกต้อง จากนั้นคลิกที่ "*Broadcast Transaction*" เพื่อกระจายธุรกรรมบนเครือข่าย Bitcoin


![CCQ](assets/fr/083.webp)


คุณสามารถติดตามธุรกรรมของคุณได้ในแท็บ "*ธุรกรรม*" ของ Sparrow Wallet


![CCQ](assets/fr/084.webp)
