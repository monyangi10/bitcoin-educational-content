### Umugereka A - Gusuzuma dosiye ya Green Wallet


Gusuzuma ivyo waronse bisigura kugenzura ko dosiye waronse itahinduwe kuva yasohorwa n’uwayikoze.


Ivyo tubikora mu kugenzura ko umukono (wakozwe n’urufunguzo rw’ibanga rw’abahinguzi) hamwe n’idosiye yavanwe n’urufunguzo rwa bose rw’abahinguzi bigarukana igisubizo c’UKURI igihe unyura mu gikorwa ca gpg –verify. Nzokwereka ingene ivyo wobikora mu nyuma.


Mbere, turaronka urufunguzo rwo gusinya:


Ku bwa Linux, fungura terminal, hanyuma ukoreshe iri tegeko (ushobora gusa gukopa no gushiramwo umwandiko, hanyuma ugashiramwo ibimenyetso vy'ugusubiramwo):


```bash
gpg --keyserver keyserver.ubuntu.com --recv-keys "04BE BF2E 35A2 AF2F FDF1 FA5D E7F0 54AA 2E76 E792"
```


Ku Mac, ukora ivyo nyene, kiretse ko uzokenera gukuraho no gushiramwo GPG Suite mbere.


Ku bijanye na Windows, ukora ivyo nyene, kiretse ko uzobanza gukura no gushiramwo GPG4Win.


Uzoronka ouput ivuga ko urufunguzo rwa bose rwashizwemwo.


![image](assets/9.webp)


Iyi shusho ifise akaranga ka alt ataco kamaze; Izina rya dosiye yayo ni ishusho-3-1024x162.urubuga


Igikurikira, turakeneye kuronka dosiye irimwo Hash ya porogarama. Bibitswe kuri paji ya GitHub ya Blockstream. Ubanze genda kuri page yabo y’amakuru hano, hanyuma ukande kuri link ya “desktop”. Bizotwara kuri paji yasohotse vuba kuri GitHub maze aho uzobona uruja n’uruza rwa dosiye SHA256SUMS.asc, ariyo nyandiko y’inyandiko irimwo Hash yasohowe na Blockstream ya porogarama twakuye.


![image](assets/10.webp)


GitHub:


![image](assets/11.webp)


Si ngombwa, ariko maze kubika kuri disiki, nahinduye izina rya “SHA256SUMS.asc” ndarigira “SHA256.txt” kugira ngo nshobore gufungura dosiye kuri Mac nkoresheje umuhinduzi w’inyandiko. Ibi nivyo vyari muri dosiye:


![image](assets/12.webp)


Icanditswe turiko turakurikira kiri hejuru. Bivanye n’idosiye twakuye, hariho igisohoka ca Hash gihuye n’ico tuzogereranya naco mu nyuma.


Igice co hasi c’inyandiko kirimwo umukono wakozwe ku butumwa buri hejuru – ni dosiye zibiri muri imwe.


Urutonde ntaco rumaze, ariko imbere yo gusuzuma Hash, turaza kugenzura ko ubutumwa bwa Hash ari ubw’ukuri (ni ukuvuga ko butahinduwe).


Gufungura ikibanza. Ukeneye kuba uri mu bubiko bubereye aho dosiye SHA256SUMS.asc yavanwe. Dufate ko wayikuye mu bubiko bwa “Downloads”, kuri Linux na Mac, hindura ububiko nk’ubu (bushingiye ku nyuguti nini):


```bash
cd Downloads
```


Ego cane, ubwirizwa gutera <enter> inyuma y’ivyo bishimwe. Ku bijanye na Windows, fungura CMD (itegeko), wandike ico kintu nyene (naho bitajanye n’inyuguti nini).


Ku bijanye na Windows na Mac, wari ukeneye kuba waramaze gukuraho GPG4Win na GPG Suite, nk’uko vyategetswe imbere. Ku bijanye na Linux, gpg izana n’Igikoresho co Gukoresha. Kuva ku nzira (canke CMD kuri Windows), wandike iri tegeko:


```bash
gpg --verify SHA256SUMS.asc
```


Inyuguti nyayo y'izina rya dosiye (mu mutuku) ishobora kuba itandukanye ku musi uzana dosiye, rero urabe neza ko itegeko rihuye n'izina rya dosiye nk'uko ryakuweko. Ushobora kuronka iki gisohoka, maze ukirengagiza imburi yerekeye umukono wizigiwe – ivyo bisigura gusa ko utabwiye mudasobwa wizigiye urufunguzo rwa bose twazanye mbere.


![image](assets/13.webp)


Iyi shusho ifise akaranga ka alt ataco kamaze; Izina rya dosiye yayo ni ishusho-4-1024x165.urubuga


Ivyo bimenyetso vyemeza ko umukono ari mwiza, kandi turizigiye ko urufunguzo rw’ibanga rwa “info@greenaddress.it” rwashize umukono ku makuru (raporo ya Hash).


Ubu rero dukwiye Hash dosiye yacu ya zip twakuweko maze tukagereranya igisohoka nk’uko casohowe. Zirikana ko muri dosiye SHA256SUMS.asc, hariho akajambo gatoyi kavuga ngo “Hash: SHA512” bikantera urujijo, kuko dosiye ifise ibisohoka vya SHA256 imbere, rero ivyo nzovyirengagiza.


Ku Mac na Linux, fungura terminal, genda aho dosiye ya zip yavanywe (kumbure uzokenera kwandika “cd Downloads” kandi, kiretse utarafunga terminal kuva ico gihe). Mu nzira ushobora kwama usuzuma ububiko urimwo wanditse PWD (“ububiko bukora), kandi nimba ivyo vyose ari ivy’amahanga, birakenewe kuraba videwo yihuta ya YouTube urondera “ingene wogenda muri Linux/Mac/Windows file system”.


Kugira ngo ubone dosiye, wandike ibi:


```bash
shasum -a 256 BlockstreamGreen_MacOS_x86_64.zip
```


Ushobora kugenzura neza uko dosiye yawe yitwa, hanyuma ugahindura umwandiko uri mu bururu uri hejuru nimba bikenewe.
