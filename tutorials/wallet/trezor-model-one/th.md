*เครดิตภาพ: [Trezor.io](https://trezor.io/)*



Trezor Model One เป็น Hardware Wallet รุ่นแรกที่เคยเปิดตัว เปิดตัวในปี 2014 โดย SatoshiLabs หลังจากมีอยู่มากว่าสิบปี มันยังคงเป็นตัวเลือกที่น่าสนใจ โดยเฉพาะสำหรับผู้ใช้ที่มองหา Hardware Wallet ที่เข้าถึงได้ทั้งในด้านเทคนิคและงบประมาณ ในความเป็นจริง มันมีราคาอยู่ที่ €49 บนเว็บไซต์ทางการของ Trezor มันเป็นหนึ่งในกระเป๋าเงินฮาร์ดแวร์เพียงไม่กี่รุ่นในช่วงราคานี้ มันอยู่กึ่งกลางระหว่างอุปกรณ์ระดับเริ่มต้นที่ประมาณ €20 เช่น Tapsigner ซึ่งมักจะไม่มีหน้าจอ และอุปกรณ์ระดับกลางที่ประมาณ €80 เช่น Ledger Nano S Plus หรือ Trezor Safe 3



Model One มีหน้าจอ OLED ขาวดำขนาด 0.96 นิ้วและปุ่มกดสองปุ่ม ทำงานโดยไม่ใช้แบตเตอรี่ ใช้เพียงการเชื่อมต่อ micro-USB สำหรับพลังงานและการแลกเปลี่ยนข้อมูล



![Image](assets/fr/01.webp)



ข้อเสียหลักของ Model One คือการขาด Secure Element ซึ่งทำให้มีความเสี่ยงต่อการโจมตีทางกายภาพหลายรูปแบบ บางรูปแบบสามารถดำเนินการได้ค่อนข้างง่าย การโจมตีเหล่านี้อาจรวมถึงการวิเคราะห์ช่องทางเสริมเพื่อกำหนด PIN ของอุปกรณ์ หรือเทคนิคขั้นสูงเพื่อดึงข้อมูลที่เข้ารหัส seed เพื่อทำการ brute-force ในภายหลัง โปรดทราบว่าการโจมตีเหล่านี้ต้องการการเข้าถึงอุปกรณ์ทางกายภาพ อย่างไรก็ตาม ความเสี่ยงนี้สามารถลดลงได้อย่างมากโดยการใช้ passphrase BIP39 ที่แข็งแกร่ง หากคุณเลือกใช้ Hardware Wallet นี้ ฉันขอแนะนำอย่างยิ่งให้คุณกำหนดค่า passphrase



Model One มีข้อดีที่สำคัญสองประการ:




- มันขึ้นอยู่กับสถาปัตยกรรมโอเพ่นซอร์สอย่างสมบูรณ์ ต่างจากรุ่นที่ใหม่กว่าที่มี Secure Element ส่วนประกอบฮาร์ดแวร์และซอฟต์แวร์ทั้งหมดของ Model One สามารถตรวจสอบได้;
- มันมาพร้อมกับหน้าจอ ตามความรู้ของฉัน นี่เป็น Hardware Wallet เพียงเครื่องเดียวในตลาดในช่วงราคานี้ที่มีหน้าจอ นี่เป็นคุณสมบัติที่สำคัญมาก เนื่องจากช่วยให้สามารถตรวจสอบข้อมูลที่ลงนามและที่อยู่รับได้ จึงป้องกันการโจมตีทางดิจิทัลได้หลายรูปแบบ



Trezor Model One อาจเป็นตัวเลือกที่ชาญฉลาดสำหรับผู้เริ่มต้นและผู้ใช้ระดับกลางที่มีงบประมาณจำกัด อย่างไรก็ตาม สิ่งสำคัญคือต้องตระหนักถึงข้อจำกัดในด้านการป้องกันทางกายภาพ เนื่องจากไม่มี Secure Element หากงบประมาณของคุณจำกัด นี่เป็นตัวเลือกที่ดี แต่ถ้าคุณสามารถเลือกใช้รุ่นที่ดีกว่าได้ เช่น Trezor Safe 3 ที่ราคา €79 จะเป็นการดีกว่า เพราะมี Secure Element รวมอยู่ด้วย



## แกะกล่อง Trezor Model One



เมื่อคุณได้รับ Model One ของคุณ ตรวจสอบให้แน่ใจว่ากล่องและซีลยังคงอยู่ในสภาพสมบูรณ์เพื่อยืนยันว่าพัสดุไม่ได้ถูกเปิด นอกจากนี้ยังจะมีการตรวจสอบซอฟต์แวร์เพื่อยืนยันความถูกต้องและความสมบูรณ์ของอุปกรณ์เมื่อมีการตั้งค่าในภายหลัง



เนื้อหาในกล่องประกอบด้วย :




- Trezor Model One;
- กระดาษแข็งสำหรับบันทึกวลีช่วยจำของคุณ, สติกเกอร์และคำแนะนำ;
- สาย USB-A เป็น micro-USB



![Image](assets/fr/02.webp)



การนำทางอุปกรณ์นั้นง่ายมาก:




- คลิกขวาเพื่อยืนยันและดำเนินการในขั้นตอนถัดไป;
- ใช้ปุ่มซ้ายเพื่อย้อนกลับ

https://planb.academy/tutorials/wallet/desktop/sparrow-c674e2ac-d46f-4c82-92a7-7d1b0e262f5d

[ดาวน์โหลด Trezor Suite จากเว็บไซต์ทางการ](https://trezor.io/trezor-suite)

## การตั้งรหัส PIN



ขั้นตอนถัดไปคือการใส่รหัส PIN รหัส PIN จะปลดล็อก Trezor ของคุณ ดังนั้นจึงให้การป้องกันการเข้าถึงทางกายภาพโดยไม่ได้รับอนุญาต รหัส PIN นี้ไม่ได้มีส่วนเกี่ยวข้องในการสร้างคีย์เข้ารหัสของ wallet ของคุณ ดังนั้นแม้จะไม่มีการเข้าถึงรหัส PIN การครอบครองวลีช่วยจำ 12 คำของคุณจะช่วยให้คุณสามารถเข้าถึงบิตคอยน์ของคุณได้อีกครั้ง



บน Trezor Suite คลิกที่ "*Continue to PIN*" จากนั้นคลิกที่ปุ่ม "*Set PIN*"



![Image](assets/fr/15.webp)



ยืนยันบน Model One.



![Image](assets/fr/16.webp)



เราแนะนำให้เลือก PIN code ที่สุ่มมากที่สุดเท่าที่จะเป็นไปได้ อย่าลืมบันทึกโค้ดนี้ในที่ที่แยกต่างหากจากที่เก็บ Trezor ของคุณ (เช่น ในโปรแกรมจัดการรหัสผ่าน) คุณสามารถกำหนด PIN code ได้ระหว่าง 8 ถึง 50 หลัก ฉันแนะนำให้คุณเลือก PIN code ที่ยาวที่สุดเท่าที่จะเป็นไปได้เพื่อเพิ่มความปลอดภัย



รหัส PIN ต้องถูกป้อนใน Trezor Suite บนคอมพิวเตอร์ของคุณโดยการคลิกที่จุดที่ตรงกับตัวเลข ตามการกำหนดค่าคีย์บอร์ดที่แสดงบน Trezor Model One



วิธีการป้อน PIN นี้จำเป็นทุกครั้งที่คุณปลดล็อก Trezor Model One ของคุณ ไม่ว่าจะผ่าน Trezor Suite หรือ Sparrow Wallet

![Image](assets/fr/18.webp)



กรุณาใส่รหัส PIN ของคุณอีกครั้งเพื่อยืนยัน



![Image](assets/fr/19.webp)



บน Trezor Suite คลิกที่ปุ่ม "*Complete setup*"



![Image](assets/fr/20.webp)



การกำหนดค่าของ Model One ของคุณเสร็จสมบูรณ์แล้ว หากคุณต้องการ คุณสามารถเปลี่ยนชื่อและหน้าแรกของ Hardware Wallet ของคุณได้



![Image](assets/fr/21.webp)



เราจะไม่จำเป็นต้องใช้ซอฟต์แวร์ Trezor Suite อีกต่อไป ยกเว้นเพื่อดำเนินการอัปเดตเฟิร์มแวร์ปกติบน Hardware Wallet ของคุณ หรือหากคุณต้องการทดสอบการกู้คืน ตอนนี้เราจะใช้ Sparrow เพื่อจัดการ wallet เนื่องจากซอฟต์แวร์นี้เหมาะสมอย่างยิ่งสำหรับการใช้งานเฉพาะ Bitcoin



## การตั้งค่า wallet บน Sparrow Wallet



เริ่มต้นด้วยการดาวน์โหลดและติดตั้ง Sparrow Wallet [จากเว็บไซต์ทางการ](https://sparrowwallet.com/) บนคอมพิวเตอร์ของคุณ หากคุณยังไม่ได้ทำเช่นนั้น



เมื่อคุณเปิด Sparrow Wallet แล้ว ให้แน่ใจว่าโปรแกรมเชื่อมต่อกับโหนด Bitcoin ซึ่งจะแสดงด้วยเครื่องหมายถูกที่มุมล่างขวาของ Interface หากคุณมีปัญหาในการเชื่อมต่อ Sparrow ฉันแนะนำให้คุณดูที่จุดเริ่มต้นของบทแนะนำนี้:



https://planb.academy/tutorials/wallet/desktop/sparrow-c674e2ac-d46f-4c82-92a7-7d1b0e262f5d

## วิธีการส่งบิตคอยน์ด้วย Trezor Model One?



ตอนนี้ที่คุณได้รับ Satss แรกใน wallet ที่ปลอดภัยด้วย Model One ของคุณแล้ว คุณก็สามารถใช้จ่ายได้เช่นกัน! เชื่อมต่อ Trezor ของคุณกับคอมพิวเตอร์ เปิดตัว Sparrow Wallet จากนั้นไปที่แท็บ "*ส่ง*" เพื่อสร้างธุรกรรมใหม่



![Image](assets/fr/36.webp)



หากคุณต้องการ *Coin Control* นั่นคือ เลือกเฉพาะเจาะจงว่า UTXOs ใดที่จะใช้ในธุรกรรม ให้ไปที่แท็บ "*UTXOs*" เลือก UTXOs ที่คุณต้องการใช้จ่าย จากนั้นคลิกที่ "*Send Selected*" คุณจะถูกนำไปยังหน้าจอเดียวกันในแท็บ "*Send*" แต่มี UTXOs ของคุณที่เลือกไว้แล้วสำหรับธุรกรรม



![Image](assets/fr/37.webp)



ป้อนที่อยู่ปลายทาง คุณยังสามารถป้อนหลายที่อยู่ได้โดยคลิกที่ปุ่ม "*+ เพิ่ม*"



![Image](assets/fr/38.webp)
