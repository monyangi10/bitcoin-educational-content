_คู่มือนี้นำมาจาก [บทเรียน Bitcoin Workshops](https://officinebitcoin.it/lezioni/jadeele/index.html)_



บทแนะนำนี้สร้างขึ้นด้วย Jade Classic แต่การดำเนินการก็ใช้ได้กับผู้ที่มี Jade Plus เช่นกัน



หลังจากเริ่มต้น Jade คุณสามารถเริ่มใช้งานได้และ--เพื่อทำเช่นนั้น--เลือกจอแสดงผล wallet



Jade เป็นอุปกรณ์ที่สามารถใช้ร่วมกับ wallet หลายตัว หรือแอปพลิเคชันคู่หูตามที่ Blockstream ระบุไว้บนเว็บไซต์ของตน



ในบทแนะนำนี้ คุณจะได้เห็นขั้นตอนการใช้ Electrum Wallet ผ่านการเชื่อมต่อสาย USB



## การถ่ายโอนคีย์สาธารณะ



เปิดและเปิดใช้งาน Jade ที่คุณได้ตั้งค่าไว้แล้ว เมื่อคุณเปิดเครื่องมันจะมีลักษณะดังนี้:




![img](assets/en/32.webp)



หากคุณเลือก _Unlock Jade_ คุณจะได้รับเมนูที่คุณต้องเลือกวิธีการเชื่อมต่ออุปกรณ์ของคุณกับแอปคู่หู



ด้วย Electrum คุณสามารถเชื่อมต่อ Jade ผ่าน USB เท่านั้น ดังนั้นเลือกวิธีนี้



เปิดตัว Electrum ซึ่งจะเปิดการเสนอเป็นตัวเลือกเริ่มต้นเพื่อเปิด wallet ล่าสุดที่ใช้



หากนี่เป็นครั้งแรกที่คุณเชื่อมต่อ Jade กับ Electrum ให้เลือก _Create New Wallet_ แล้วจากนั้นเลือก _Finish_



![img](assets/en/34.webp)



ชื่อ wallet.



![img](assets/en/35.webp)



เลือกมาตรฐาน Wallet



![img](assets/en/36.webp)



เมื่อเลือก keystore จำเป็นต้องเลือก _ใช้ฮาร์ดแวร์ดีไวซ์_



![img](assets/en/37.webp)



Electrum เริ่มการสแกนอุปกรณ์ฮาร์ดแวร์



![img](assets/en/38.webp)



โดยการเชื่อมต่อ USB เข้ากับคอมพิวเตอร์ (ซึ่งได้เชื่อมต่อกับด้าน USB C ของ Jade แล้ว) ฮาร์ดแวร์ wallet จะปรากฏในโหมดล็อค Jade จะปลดล็อคโดยการใส่รหัส PIN หกหลักที่ตั้งไว้ระหว่างการตั้งค่า




![img](assets/en/39.webp)



อุปกรณ์ฮาร์ดแวร์ที่ปลดล็อกแล้ว Electrum ตรวจพบ Jade ดำเนินการต่อโดยคลิก _ถัดไป_



![img](assets/en/40.webp)



ณ จุดนี้ Electrum ขอให้คุณตั้งค่าสคริปต์นโยบาย: เลือก _Native Segwit_



![img](assets/en/41.webp)



ระยะของการถ่ายโอนกุญแจสาธารณะจาก wallet จาก Jade ไปยังการแสดงผล Electrum เริ่มต้นขึ้น



เมื่อการส่งออกกุญแจสาธารณะเสร็จสิ้น กระบวนการก็เสร็จสมบูรณ์



นาฬิกาแบบดูอย่างเดียวพร้อมแล้วและ Electrum แจ้งเตือนการเสร็จสิ้นด้วยหน้าจอต่อไปนี้



![img](assets/en/42.webp)



wallet ถูกสร้างขึ้นจริงและคุณสามารถเริ่มสำรวจได้: คุณสามารถดู _ที่อยู่_, _ข้อมูลกระเป๋าเงิน_, และที่สำคัญที่สุด คุณสามารถเห็นในมุมขวาล่างที่บ่งบอกว่าเป็นอุปกรณ์ของ Blockstream จุดสีเขียวถัดจากโลโก้ Blockstream แสดงว่าอุปกรณ์เปิดอยู่และเชื่อมต่อกับเครือข่ายท้องถิ่นอย่างถูกต้อง



![img](assets/en/43.webp)



## การรับและการใช้จ่ายธุรกรรม



จากเมนู _Receive_ ของ Electrum, generate ให้ใช้ `scriptPubKey` (address) เพื่อรับเงินเสมอ เริ่มต้นด้วยจำนวนเล็กน้อยและทำการทดสอบรับ+ใช้จ่าย



![img](assets/en/44.webp)



เมื่อได้รับ satss แล้ว คุณสามารถตรวจสอบการมาถึงของพวกเขาในเมนู _History_



![img](assets/en/45.webp)



![img](assets/en/46.webp)



เมื่อการทำธุรกรรมได้รับการยืนยันแล้ว คุณสามารถใช้ UTXO นี้และทำการทดสอบให้เสร็จสิ้น



ค่าใช้จ่ายเกี่ยวข้องกับการใช้ Jade ในการลงนาม



ไปที่เมนู _Send_ ของ Electrum วาง scriptPubKey และตรวจสอบให้ดี



![img](assets/en/47.webp)



เมื่อเสร็จแล้ว ให้กด _Pay_.



หน้าต่างการทำธุรกรรมจะเปิดขึ้น ซึ่งเป็นสิ่งสำคัญที่จะต้องตั้งค่าธรรมเนียมการทำธุรกรรมให้ถูกต้อง เมื่อคุณตั้งค่าทั้งหมดเสร็จแล้ว ให้คลิกที่ _Preview_ ที่มุมล่างขวา



![img](assets/en/48.webp)



หน้าต่างการทำธุรกรรมแสดงรายละเอียดสำคัญบางประการ โดยเฉพาะอย่างยิ่งสถานะ: `Unsigned`.



ณ ขั้นตอนนี้ คุณยังสามารถเห็นคำสั่ง _Sign_ ซึ่งคุณต้องคลิกเพื่อแนบลายเซ็นด้วย Jade



![img](assets/en/49.webp)



ขณะนี้เริ่มต้นขั้นตอนการสื่อสารระหว่างจอแสดงผล wallet และอุปกรณ์ฮาร์ดแวร์ ซึ่ง Electrum จะแจ้งเตือนให้คุณปฏิบัติตามคำแนะนำบนอุปกรณ์ฮาร์ดแวร์ที่เปิดอยู่และพร้อมสำหรับการลงนาม



![img](assets/en/50.webp)



**อย่างไรก็ตาม ก่อนอื่นคุณควรตรวจสอบสิ่งที่คุณกำลังเซ็น: พารามิเตอร์ทั้งหมดของธุรกรรมที่คุณเพิ่งตั้งค่าไว้ จะปรากฏบน Jade** และคุณสามารถตรวจสอบทั้งหมดได้



![img](assets/en/51.webp)



ในการดำเนินการต่อ ตรวจสอบให้แน่ใจว่าคุณวางเคอร์เซอร์ไว้ที่ลูกศร `→` เสมอ ซึ่งนำไปสู่ขั้นตอนถัดไป และอย่าวางที่ `X` เว้นแต่คุณต้องการยุติการดำเนินการโดยไม่ทำให้เสร็จสิ้น



ส่วนการตรวจสอบสิ้นสุดด้วยการแสดงค่าธรรมเนียม ณ จุดนี้การยืนยันเทียบเท่ากับการลงลายมือชื่อของคุณ



![img](assets/en/52.webp)



ชั่วขณะหนึ่ง Jade ประมวลผลการทำงาน เมื่อเสร็จสิ้นแล้วจะกลับไปที่เมนูหลัก



![img](assets/en/53.webp)



ในขณะที่อยู่บน Electrum คุณสามารถดูสถานะของธุรกรรม ซึ่งได้เปลี่ยนจาก `Unsigned` เป็น `Signed` และตอนนี้คุณสามารถเผยแพร่ได้โดยการคลิก _Broadcast_



![img](assets/en/54.webp)



wallet, ที่ผ่านการทดสอบแล้ว สามารถใช้รับ UTXO ที่ตั้งใจจะเก็บรักษาไว้อย่างปลอดภัย



![img](assets/en/55.webp)



คู่มือนี้เป็นตัวอย่างวิธีการใช้ Jade ของคุณ ซึ่งเชื่อมต่อผ่าน USB กับ wallet ที่ใช้ดูเท่านั้น Electrum เป็นตัวอย่างคลาสสิก แต่คุณอาจชอบซอฟต์แวร์ wallet อื่น ๆ มากกว่า Jade ส่งออกคีย์สาธารณะไปยังวอลเล็ตอื่น ๆ หลายแห่ง: ค้นหาฟังก์ชันที่คล้ายกันที่คุณอ่านในบทแนะนำนี้ เพื่อแนะนำคุณและค้นหาวิธีการใช้แอปคู่หูที่คุณชื่นชอบ
