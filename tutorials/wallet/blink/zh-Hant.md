## Blink Wallet 是什麼？


![image](assets/fr/01.webp)


忘記您認為 Bitcoin 錢包複雜的一切。Blink 是 Bitcoin 錢包的 iPhone。


Blink 的前身是 Bitcoin Beach Wallet，它是一款用戶友好型移動應用程式，可將 Bitcoin 帶到世界上任何地方的每個人手中。它最初是 [為薩爾瓦多的 Bitcoin Beach 社群而建](https://bitcoinmagazine.com/business/Bitcoin-beach-Wallet-renamed-to-blink)，現在則幫助全球各地的人們毫不費力地發送、接收和使用 Bitcoin。


無論您是 Bitcoin 的新手還是老手，本指南都會教您入門所需的一切知識。


### 主要功能：



- 雙貨幣支援：輕鬆持有 Bitcoin (BTC) 和 Stablesats (等值美元)
- Lightning Network: 快速、便宜的 Bitcoin 交易
- Stablesats：在使用 Bitcoin 的支付網路時，保持您的美元價值穩定
- 設定簡單：只需您的電話號碼
- 全球商家地圖：尋找接受 Bitcoin 的商家


最漂亮的部分是什麼？您只需輕輕一按，即可在「我想持有 Bitcoin，因為我認為它會上漲」和「我只想用穩定的美元來買咖啡錢」之間切換。

## 瞭解您的 Wallet Interface


![image](assets/fr/02.webp)


當您第一次打開 [Blink Wallet](https://blink.sv/)，您會看到令人耳目一新的簡潔畫面。沒有圖表向您尖叫，沒有閃爍的數字，只有您的餘額和一些簡單的按鈕。


### 主畫面概觀



- 總結餘：顯示 Bitcoin 和美元結餘的合計值
- 帳戶餘額：
 - Bitcoin：所示為 Sats（最小的 Bitcoin 單元）
 - 美元：  您的美元價值穩定



- 動作按鈕：您需要知道的四個按鈕。
 - 轉換
 - 接收
 - 發送
 - 掃描 QR
- 交易歷史：您之前的所有交易
- 價格圖表：Bitcoin 價格追蹤（點選圖表圖示）


### 基本設定與如何設定您的 Lightning Address


#### 設定您的 Lightning Address



- 移至設定 → 您的 Lightning Address
- 選擇您的自訂 Address（例如，"yourname@pay.blink.sv"）
- 這將成為您永久的 Bitcoin Address，任何人都可以用它來寄錢給您


#### 其他要設定的設定：



- 預設貨幣：選擇餘額的顯示方式
- 預設帳戶：選擇流入資金的去向
- 設定您的電子郵件：新增用於登入復原的電子郵件
- 裝置上的安全性：啟用 PIN 碼或生物辨識
- 交易限制：
 - 接收：無限制
 - 提款：2,000 美元/天
 - Wallet 至 Wallet 轉移：4,000美元/天


## 如何使用 Blink Wallet 接收 Bitcoin


### 方法 1：QR Code (最常見)



- 在 Blink Wallet 主畫面上點選「接收」。
- 選擇您要接收的貨幣：
 - Bitcoin (BTC)，或
 - 美元 (Stablesats)
- 請求特定金額：
 - 點選「設定金額
 - 輸入您想收到的金額
 - 與寄件者分享產生的 QR 代碼或付款連結
- 允許彈性金額（無設定值）：
 - 跳過「設定金額」步驟
 - 只需分享 QR 代碼
 - 寄件者決定寄送的數量
 - 寄件者決定寄送的數量


### 方法 2：閃電 Address



- 與任何人分享您的閃電 Address (yourname@pay.blink.sv)
- 他們只需使用此 Address 即可直接傳送 Bitcoin
- 無需 QR 代碼或發票
- 可與任何 Lightning 相容的 Wallet 搭配使用


### 方法 3：On-Chain Bitcoin (傳統)



- 在接收螢幕中，點選 "Onchain"。
- 使用此功能進行傳統 Bitcoin 交易
- 注意：費用較高，確認時間較慢
- 最適合數量較大或沒有 Lightning 時使用


## 如何使用 Blink Wallet 傳送 Bitcoin


### 使用 QR 碼傳送



- 點選主畫面上的「傳送
- 點選相機圖示掃描 QR 代碼
- 選擇要從哪個帳戶傳送 (Bitcoin 或 USD)
- 審核金額和費用
- 確認付款


### 與付款要求/發票一起發送



- 點選「傳送」。
- 貼上付款要求 (從文字/電子郵件複製)
- 遵循發送 QR 代碼的相同步驟


### 傳送至閃電地址



- 點選「傳送」。
- 輸入或貼上閃電 Address（看起來像電子郵件）
- 輸入您要傳送的金額
- 選擇您的傳送帳號並確認


### 傳送給其他 Blink 使用者



- 傳送至其他 Blink Wallet 使用者時零費用
- 收件人自動成為聯絡人，方便日後交易


## 使用 Blink Wallet 在 Bitcoin 和美元之間轉換


Blink Wallet 最強大的功能之一，就是能在 Bitcoin 和 Stablesats (USD) 之間進行無縫轉換。請遵循此簡單的步驟指南：



- 點選主畫面上的「轉換」按鈕
- 選擇轉換方向：
 - Bitcoin → USD
 - 美元 → Bitcoin
- 設定金額：
 - 輸入特定的 Bitcoin 或美元金額，或
 - 選擇結餘的百分比 (25%、50%、75%、100%)
- 檢視轉換率
- 確認轉換以完成程序。


使用個案：


- 當 Bitcoin 價格下跌時，兌換成美元。
- 當您想要長期持有或投資時，請轉換為 Bitcoin。
- 以美元消費，避免日常交易中的價格波動。
- 在 BTC 和美元之間切換，從市場變動中獲利。

## 需要考慮的重要權衡


### 保管性質


Blink 是託管 Wallet，意即他們代表您持有和管理您的資金。



- 優點：設定簡單、Interface 易於使用、提供客戶支援
- 缺點：您無法控制您的 [私人金鑰](https://www.blink.sv/blog/not-your-keys-not-your-coins)，這表示您必須仰賴 Blink 來管理您的資金


### 交易限制



- 存在每日限制
- 可根據要求提高限額
- 若金額較大，可考慮自行保管選項


### 隱私權考量



- 建立帳戶所需的電話號碼
- 交易記錄會儲存在 Blink 的伺服器上，這可能不適合所有的隱私偏好。


## 善用 Blink Wallet


### 最佳實務：



- 從少量開始 - 先從少量開始練習
- 使用 Lightning 位址 - 公開分享您的 Address，方便付款
- 瞭解差異 - 知道何時使用 Lightning vs On-Chain
- 持續學習 - 使用賺取部分建立 Bitcoin 知識
- 考慮您的需求 - 評估保管解決方案是否符合您的使用情況


### 何時使用各項功能：



- Bitcoin 平衡：適用於長期持有/堆疊
- 美元結餘：用於穩定消費資金，避免波動
- 閃電付款：進行快速、便宜的交易
- On-Chain 付款：適用於較大金額或無法使用 Lightning 時
