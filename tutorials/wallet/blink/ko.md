## Blink Wallet란 무엇인가요?


![image](assets/fr/01.webp)


Bitcoin 지갑이 복잡하다고 생각하는 모든 것을 잊어버리세요. Blink는 Bitcoin 지갑의 아이폰입니다.


이전에 Bitcoin Beach Wallet로 알려진 Blink는 사용자 친화적인 모바일 앱으로, 전 세계 어디서나 누구나 Bitcoin를 사용할 수 있도록 지원합니다. 원래는 엘살바도르의 Bitcoin Beach 커뮤니티(https://bitcoinmagazine.com/business/Bitcoin-beach-Wallet-renamed-to-blink)를 위해 만들어졌으며, 현재는 전 세계 사람들이 Bitcoin를 손쉽게 주고받고 사용할 수 있도록 돕고 있습니다.


이 가이드는 Bitcoin를 처음 사용하든 숙련된 사용자이든 상관없이 시작하기 위해 알아야 할 모든 것을 안내합니다.


### 주요 기능:



- 이중 통화 지원: Bitcoin(BTC)과 스테이블샛(USD 상당)을 모두 쉽게 보유할 수 있습니다
- Lightning Network: 빠르고 저렴한 Bitcoin 거래
- 스테이블삿: Bitcoin의 결제 네트워크를 사용하는 동안 USD로 가치를 안정적으로 유지하세요
- 간단한 설정: 휴대폰 번호만 있으면 됩니다
- 글로벌 판매자 지도: Bitcoin 허용 업체 찾기


가장 멋진 부분은? 탭 한 번으로 "오를 것 같아서 Bitcoin를 보유하고 싶어요"와 "커피값으로 안정적인 달러를 원해요"로 전환할 수 있다는 점입니다.

## Wallet Interface 이해


![image](assets/fr/02.webp)


Blink Wallet](https://blink.sv/)을 처음 열면 상쾌하고 깔끔한 화면이 펼쳐집니다. 요란한 차트나 번쩍이는 숫자 대신 잔액과 간단한 버튼만 있습니다.


### 메인 화면 개요



- 총 잔액: Bitcoin와 USD 잔액을 합산한 값을 표시합니다
- 계정 잔액:
 - Bitcoin: Sats(가장 작은 Bitcoin 장치)에 표시됨
 - 달러:  USD로 안정적인 가치



- 동작 버튼: 알아야 할 네 가지 버튼
 - 변환
 - 수신
 - 보내기
 - QR 스캔
- 거래 내역: 모든 이전 거래
- 가격 차트: Bitcoin 가격 추적(차트 아이콘 탭)


### 필수 설정 및 Lightning Address 설정 방법


#### 라이트닝 Address 설정하기



- 설정 → 내 Lightning Address로 이동합니다
- 사용자 지정 Address 선택(예: "yourname@pay.blink.sv")
- 이것은 누구나 송금에 사용할 수 있는 영구적인 Bitcoin Address이 됩니다


#### 구성할 기타 설정:



- 기본 통화: 잔액 표시 방식 선택
- 기본 계정: 입금되는 자금의 사용처 선택
- 이메일을 설정합니다: 로그인 복구용 이메일 추가
- 온디바이스 보안: PIN 또는 생체 인식 사용
- 거래 한도:
 - 수신: 무제한
 - 인출: $2,000/일
 - Wallet에서 Wallet로 전송: 4,000/일


## 블링크 Wallet를 사용하여 Bitcoin을 수신하는 방법


### 방법 1: QR코드(가장 일반적인 방법)



- Blink Wallet 홈 화면에서 '수신'을 탭합니다.
- 받고자 하는 통화를 선택합니다:
 - Bitcoin(BTC) 또는
 - USD(스테이블스탯)
- 특정 금액을 요청하려면
 - "금액 설정"을 탭합니다
 - 수령하고자 하는 금액을 입력하세요
 - 생성된 QR코드 또는 결제 링크를 발신자와 공유하세요
- 유연한 금액을 허용하려면(설정된 값 없음):
 - '금액 설정' 단계 건너뛰기
 - QR 코드를 공유하기만 하면 됩니다
 - 발신자가 전송할 금액을 결정합니다
 - 발신자가 전송할 금액을 결정합니다


### 방법 2: 라이트닝 Address



- 라이트닝 Address(yourname@pay.blink.sv)을 다른 사람과 공유하세요
- 이 Address만 사용하여 Bitcoin을 직접 전송할 수 있습니다
- QR코드나 인보이스가 필요하지 않습니다
- 모든 라이트닝 호환 Wallet과 함께 사용 가능


### 방법 3: On-Chain Bitcoin(기존)



- 수신 화면에서 "온체인"을 탭합니다
- 기존 Bitcoin 트랜잭션에 사용하세요
- 참고: 더 높은 수수료와 느린 확인 시간
- 더 많은 양을 사용하거나 라이트닝을 사용할 수 없는 경우에 적합합니다


## 블링크 Bitcoin를 사용하여 Wallet을 보내는 방법


### QR 코드로 보내기



- 메인 화면에서 '보내기'를 탭합니다
- 카메라 아이콘을 탭하여 QR 코드를 스캔합니다
- 송금할 계정 선택(Bitcoin 또는 USD)
- 금액 및 수수료 검토
- 결제 확인


### 결제 요청/송장과 함께 보내기



- "보내기"를 탭합니다
- 결제 요청 붙여넣기(문자/이메일에서 복사)
- QR코드 전송과 동일한 단계를 따르세요


### 라이트닝 주소로 보내기



- "보내기"를 탭합니다
- 라이트닝 Address(이메일처럼 보이는)을 입력하거나 붙여넣기합니다
- 송금할 금액을 입력합니다
- 보내는 계정을 선택하고 확인


### 다른 Blink 사용자에게 보내기



- 다른 Blink Wallet 사용자에게 전송 시 수수료 없음
- 수신자가 자동으로 연락처가 되어 향후 거래를 쉽게 할 수 있습니다


## Blink Wallet을 사용하여 Bitcoin과 USD 간 변환하기


Blink Wallet의 가장 강력한 기능 중 하나는 Bitcoin와 스테이블샛(USD)을 원활하게 변환할 수 있는 기능입니다. 간단한 단계별 가이드를 따라하세요:



- 메인 화면에서 '변환' 버튼을 탭합니다
- 전환 방향을 선택합니다:
 - Bitcoin → USD
 - USD → Bitcoin
- 금액을 설정합니다:
 - 특정 Bitcoin 또는 USD 금액을 입력하거나
 - 잔액의 백분율(25%, 50%, 75%, 100%)을 선택합니다
- 전환율 검토
- 전환을 확인하여 프로세스를 완료합니다.


사용 사례:


- Bitcoin 가격이 하락할 때 USD로 전환하세요.
- 장기 보유 또는 투자를 원할 경우 Bitcoin로 전환하세요.
- 일일 거래에서 가격 변동성을 피하려면 USD로 지출하세요.
- BTC와 USD를 전환하여 시장 움직임을 활용하세요.


## Blink Wallet의 판매자 기능


### Bitcoin 수용 기업 찾기



- 하단의 '지도' 아이콘을 탭합니다
- Bitcoin 결제를 허용하는 위치 찾기
- 비즈니스를 탭합니다:

### 비즈니스 소유자용: POS 기능


설정 → 대금 수령 방법에서 고급 판매자 도구에 액세스합니다:



- 라이트닝 금전 등록기:
 - 웹 기반 POS 단말기
 - 브라우저가 있는 모든 기기에서 사용
 - 특정 금액에 대한 인보이스 생성
- 인쇄 가능한 QR 코드:
 - 비즈니스용 generate QR 코드
 - 고객이 스캔할 수 있도록 인쇄 및 디스플레이
 - 금액 지정 없음 - 고객이 팁/결제 금액을 직접 선택


블링크 POS](https://www.blink.sv/blog/transform-your-payment-experience-with-the-blink-pos)에서 더 많은 인사이트를 얻고, [블링크 브랜딩 및 온보딩 리소스](https://www.blink.sv/en/brand)에서 유용한 판매자 리소스를 살펴보시기 바랍니다.

## 고려해야 할 중요한 트레이드 오프


### 관리 성격


Blink는 커스터디 Wallet로, 회원님을 대신하여 자금을 보관하고 관리합니다.



- 장점: 간단한 설정, 사용자 친화적인 Interface, 고객 지원 가능
- 단점: [개인 키](https://www.blink.sv/blog/not-your-keys-not-your-coins)를 제어할 수 없으므로, 자금 관리를 Blink에 의존해야 합니다


### 거래 한도



- 일일 한도가 존재합니다
- 요청 시 한도를 늘릴 수 있습니다
- 더 큰 금액의 경우 자체 보관 옵션을 고려하세요


### 개인정보 보호 고려 사항



- 계정 생성에 필요한 전화번호
- 거래 내역은 Blink의 서버에 저장되며, 이는 모든 개인정보 보호 기본 설정에 적합하지 않을 수 있습니다

## 최종 생각: 간단하고 강력하며 일상에서 바로 사용할 수 있는 기능


![image](assets/fr/03.webp)


[Blink Wallet](https://blink.sv/)은 Bitcoin의 세계로 들어가는 훌륭한 진입점을 제공합니다. 사용 편의성, 이중 통화 지원, [Lightning Network](https://www.blink.sv/blog/what-is-the-lightning-network)과의 통합이 결합되어 있어 이상적인 제품입니다:



- Bitcoin 초보자
- 일일 지출 및 수령
- 중소규모 거래
- Bitcoin에 대해 알아보기
- Bitcoin 결제를 수락하는 비즈니스


Blink Wallet은 쉽고 편리하게 사용할 수 있도록 설계되었지만, 개인정보 보호, 제어 또는 거래 제한 등 자신에게 가장 적합한 것이 무엇인지 항상 고려하는 것이 좋습니다. Bitcoin에 익숙해지면 필요에 맞는 다른 Wallet 옵션을 살펴볼 수 있습니다


Blink Wallet은 사용하기 쉽고 실제 결제용으로 제작되었습니다. 직접 사용해보고 기능을 살펴보고 실제 사용을 통해 Bitcoin에 대한 자신감과 이해도를 높여보세요.
