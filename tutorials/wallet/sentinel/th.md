> เก็บคีย์ส่วนตัวของคุณให้เป็นส่วนตัว

ในบทความนี้ เราจะสำรวจทุกสิ่งที่คุณจำเป็นต้องรู้เกี่ยวกับกระเป๋าเงินแบบดูอย่างเดียว เราจะพูดถึงวิธีการทำงานและตรวจสอบแอปพลิเคชันต่างๆ ที่มีอยู่ในตลาด สุดท้ายนี้ เราจะเสนอคำแนะนำโดยละเอียดเกี่ยวกับหนึ่งในแอปพลิเคชัน wallet แบบดูอย่างเดียวที่ได้รับความนิยมมากที่สุด: Sentinel


## Watch-Only Wallet คืออะไร?


wallet แบบดูอย่างเดียว หรือ wallet แบบอ่านอย่างเดียว เป็นซอฟต์แวร์ประเภทหนึ่งที่ออกแบบมาเพื่อให้ผู้ใช้สามารถสังเกตธุรกรรมที่เกี่ยวข้องกับคีย์สาธารณะ Bitcoin หนึ่งหรือมากกว่า โดยไม่ต้องเข้าถึงคีย์ส่วนตัวที่เกี่ยวข้อง


แอปพลิเคชันประเภทนี้จะเก็บรักษาเฉพาะข้อมูลที่จำเป็นสำหรับการตรวจสอบ Bitcoin wallet รวมถึงการดูยอดคงเหลือและประวัติการทำธุรกรรม แต่จะไม่มีการเข้าถึงคีย์ส่วนตัว ดังนั้นจึงเป็นไปไม่ได้ที่จะใช้จ่ายบิตคอยน์ที่ถืออยู่ใน wallet บนแอปพลิเคชันที่ใช้สำหรับดูเท่านั้น

![watch-only](assets/en/1.webp)

Watch-only มักใช้ร่วมกับฮาร์ดแวร์ wallet ซึ่งช่วยให้สามารถจัดเก็บคีย์ส่วนตัวของ wallet แบบ "cold" บนอุปกรณ์ที่ไม่เชื่อมต่อกับอินเทอร์เน็ต ซึ่งมีพื้นผิวการโจมตีที่น้อยที่สุด แยกคีย์ส่วนตัวออกจากสภาพแวดล้อมที่อาจมีความเสี่ยง ในทางกลับกัน แอปพลิเคชัน watch-only จะจัดเก็บเฉพาะคีย์สาธารณะขยาย (`xpub`, `zpub`, ฯลฯ) ของ Bitcoin wallet คีย์หลักนี้ไม่อนุญาตให้ค้นพบคีย์ส่วนตัวที่เกี่ยวข้องและดังนั้นจึงไม่อนุญาตให้ใช้จ่ายบิตคอยน์ อย่างไรก็ตาม มันอนุญาตให้สร้างคีย์สาธารณะลูกและที่อยู่รับ ด้วยความรู้เกี่ยวกับที่อยู่ของ wallet ที่ได้รับการรักษาความปลอดภัยโดยฮาร์ดแวร์ wallet แอปพลิเคชัน watch-only สามารถติดตามธุรกรรมเหล่านี้บนเครือข่าย Bitcoin ให้ผู้ใช้สามารถตรวจสอบยอดคงเหลือและ generate ที่อยู่รับใหม่ โดยไม่ต้องเชื่อมต่อฮาร์ดแวร์ wallet ทุกครั้ง


## ใช้ Watch-Only Wallet ตัวไหน?


ปัจจุบัน แอปพลิเคชัน watch-only ที่ครบถ้วนที่สุดคือ [Sentinel](https://github.com/wanderingking072/sentinel-android) ซึ่งพัฒนาโดยทีมงานของ Samourai Wallet ในตอนแรก และปัจจุบันดูแลโดยชุมชน โดยรวบรวมฟังก์ชันที่จำเป็นทั้งหมดสำหรับกระเป๋าเงิน watch-only ที่ดี:



- รองรับคีย์ขยาย คีย์สาธารณะ และที่อยู่;
- ความสามารถในการจัดระเบียบหลายบัญชีหรือกระเป๋าเงินเป็นคอลเลกชัน;
- การสร้างที่อยู่เพื่อรับบิตคอยน์บนฮาร์ดแวร์ wallet โดยไม่ต้องใช้งานโดยตรง;
- ความสามารถในการสร้างและส่งธุรกรรมแบบออฟไลน์;
- ตัวเลือกในการเชื่อมต่อกับโหนด Bitcoin ของตนเอง;
- การผสานรวม Tor เพื่อความเป็นส่วนตัวที่เพิ่มขึ้น

ข้อเสียเฉพาะของ Sentinel อยู่ที่แอปพลิเคชันนี้มีให้ใช้งานเฉพาะบน Android และไม่รองรับกระเป๋าเงินแบบหลายลายเซ็น ดังนั้น หากคุณมีอุปกรณ์ Android และ wallet ของคุณเป็นแบบลายเซ็นเดียวแบบคลาสสิก ฉันขอแนะนำ Sentinel

สำหรับผู้ที่ต้องการติดตาม wallet แบบหลายลายเซ็น Blue Wallet เป็นแอปพลิเคชันเดียวที่ฉันรู้จักที่มีโหมดดูอย่างเดียวสำหรับกระเป๋าเงินประเภทนี้ และสามารถใช้งานได้ทั้งบน Android และ iOS


สำหรับผู้ใช้ iOS ที่กำลังมองหาทางเลือกแทน Sentinel, [Green Wallet](https://blockstream.com/green/) หรือ [Blue Wallet](https://bluewallet.io/watch-only/) อาจเป็นตัวเลือก แม้ว่าฟังก์ชันการดูอย่างเดียวของพวกเขาจะไม่ครอบคลุมเท่า Sentinel ก็ตาม

![watch-only](assets/notext/2.webp)


## วิธีใช้ Sentinel Watch-Only Wallet?


### การติดตั้งและการตั้งค่า


เริ่มต้นด้วยการติดตั้งแอปพลิเคชัน Sentinel คุณสามารถทำได้โดยใช้ [APK ที่พร้อมดาวน์โหลดบน github repository ของโครงการ](https://github.com/wanderingking072/sentinel-android/releases)















จากนั้นคุณจะต้องเชื่อมต่อกับ Dojo ของคุณเองอย่างบังคับ เนื่องจากเซิร์ฟเวอร์ของ Samourai Wallet ไม่สามารถใช้งานได้อีกต่อไป หากคุณยังไม่มี Dojo ของตัวเอง คุณสามารถใช้ Dojo ที่ชุมชนจัดให้บนเว็บไซต์ [The Dojo Bay](https://dojobay.pw/) หรือทำตามบทแนะนำของเราเพื่อติดตั้งของคุณเอง :

https://planb.academy/tutorials/wallet/mobile/sentinel-9876f960-e964-4d20-8a6e-36231de1f4d9

![watch-only](assets/notext/4.webp)






จากนั้นคุณจะไปถึงหน้าหลักของ Sentinel


![watch-only](assets/notext/6.webp)


ในการเริ่มต้น คุณสามารถตั้งค่าแอปพลิเคชันได้ คลิกที่จุดเล็ก ๆ สามจุดที่มุมขวาบน จากนั้นคลิกที่ `Settings`


![watch-only](assets/notext/7.webp)

โดยการเลือก `รหัส PIN ของผู้ใช้` คุณมีตัวเลือกในการตั้งรหัสผ่านเพื่อรักษาความปลอดภัยในการเข้าถึง wallet ที่เป็นแบบดูเท่านั้นของคุณ คุณยังสามารถเปลี่ยนสกุลเงินอ้างอิงสำหรับการแปลงยอดคงเหลือของคุณเป็นสกุลเงินเฟียต หรือแม้กระทั่งซ่อนมูลค่าเฟียตโดยการเปิดใช้งานตัวเลือก `ซ่อนมูลค่าเฟียต` สำหรับความปลอดภัยที่เพิ่มขึ้น คุณสามารถเปิดใช้งาน `ปิดการใช้งานการจับภาพหน้าจอ` ซึ่งจะป้องกันการจับภาพหน้าจอใด ๆ ของแอปพลิเคชัน Sentinel ของคุณและหลีกเลี่ยงการเปิดเผยข้อมูลบนหน้าจอภายนอก

![watch-only](assets/notext/8.webp)


ในเมนูการตั้งค่านี้ คุณยังมีตัวเลือกในการสำรองข้อมูล Sentinel ของคุณ


### การใช้ Watch-Only Wallet


จากหน้าแรก กดปุ่ม `NEW` สีฟ้าเพื่อเพิ่มคีย์สาธารณะขยายใหม่เพื่อติดตาม จากนั้นคุณมีตัวเลือกในการสแกน QR โค้ดของคีย์ของคุณ หรือวางคีย์ (`xpub`, `zpub`...) โดยตรงโดยเลือก `Paste Pubkey`.


![watch-only](assets/notext/9.webp)


โดยทั่วไปแล้ว `xpub` ของ wallet ของคุณสามารถเข้าถึงได้โดยตรงผ่านซอฟต์แวร์การจัดการ wallet ที่คุณใช้ ตัวอย่างเช่น หากคุณจัดการฮาร์ดแวร์ wallet ของคุณด้วย Sparrow ข้อมูลนี้จะพบได้ในแท็บ `Settings` ภายใต้ส่วน `Keystore`


![watch-only](assets/notext/10.webp)


หลังจากป้อนคีย์สาธารณะที่ขยายใน Sentinel แอปพลิเคชันจะเสนอให้คุณสร้างคอลเลกชันใหม่ คอลเลกชันหมายถึงชุดของคีย์สาธารณะที่ขยายซึ่งจัดระเบียบไว้ด้วยกัน ตัวเลือกนี้ให้คุณสามารถไม่เพียงแค่แสดงรายการ `xpubs` ทั้งหมดของคุณ แต่ยังจัดประเภทพวกมันอย่างเป็นระเบียบ ตัวอย่างเช่น หากคุณมี Samourai Wallet ที่มีหลายบัญชี (ฝาก, premix, postmix...) คุณสามารถรวบรวมบัญชีทั้งหมดเหล่านี้ภายใต้คอลเลกชัน `Samourai` สำหรับกระเป๋าเงินที่จัดการสำหรับครอบครัวของคุณ คุณอาจสร้างคอลเลกชันชื่อ `Family`


เลือก `Create new collection` จากนั้นป้อนชื่อสำหรับคีย์ที่คุณเพิ่งรวมเข้าด้วยกัน ตัวอย่างเช่น หากฉันสแกนบัญชีฝากของ Samourai wallet ของฉัน ฉันจะตั้งชื่อคีย์นี้ว่า `Deposit` คลิกที่ `SAVE` เพื่อเสร็จสิ้น


![watch-only](assets/notext/11.webp)


ถัดไป ตั้งชื่อให้กับคอลเลกชันนี้และกดไอคอนการตรวจสอบที่มุมขวาบนของหน้าจอเพื่อบันทึกคอลเลกชัน คอลเลกชันของคุณจะปรากฏบนหน้าจอหลักของ Sentinel แล้ว


![watch-only](assets/notext/12.webp)


หากคุณต้องการเพิ่มกุญแจสาธารณะที่ขยายเพิ่มเติม ให้คลิกที่ `NEW` อีกครั้งและป้อนกุญแจของคุณ


![watch-only](assets/notext/13.webp)

จากนั้นคุณจะได้รับการแจ้งให้เลือกคอลเลกชันที่คุณต้องการรวมคีย์นี้เข้าไป หรือสร้างคอลเลกชันใหม่ ตัวอย่างเช่น ในกรณีของฉัน ฉันได้ตั้งค่าคอลเลกชันเฉพาะสำหรับ Ledger wallet ของฉัน

![watch-only](assets/notext/14.webp)


หากต้องการดูคีย์ที่ขยายของคอลเลกชันอย่างละเอียด เพียงคลิกที่มัน จากนั้นคุณสามารถนำทางผ่านแท็บต่างๆ เพื่อดูประวัติการทำธุรกรรมได้


![watch-only](assets/notext/15.webp)


จากคอลเลกชัน โดยการแตะที่จุดเล็ก ๆ สามจุดที่มุมขวาบน จากนั้นแตะที่ `ดูเอาต์พุตที่ยังไม่ได้ใช้` คุณสามารถเข้าถึงรายการ UTXOs ที่ถือโดย wallet ที่ติดตามได้


![watch-only](assets/notext/16.webp)


### การส่งและรับบิตคอยน์จาก Sentinel


เช่นเดียวกับ wallet ที่ดีสำหรับการดูอย่างเดียว Sentinel ช่วยให้คุณสามารถ generate ที่อยู่รับเพื่อรับบิตคอยน์บน wallet ที่ติดตามได้ แต่ Sentinel ยังมีฟีเจอร์ขั้นสูงอีกอย่างหนึ่ง: การสร้างและการกระจายธุรกรรม Bitcoin ที่ลงนามบางส่วน (PSBT) ดังนั้น wallet ที่ถือกุญแจส่วนตัวสามารถลงนามในธุรกรรมนี้ ซึ่งเมื่อได้รับการลงนามแล้ว สามารถกระจายบนเครือข่าย Bitcoin โดย Sentinel มาดูกันว่าทำทั้งหมดนี้ได้อย่างไร


**คำเตือน ไม่แนะนำให้รับบิตคอยน์ในที่อยู่รับที่ไม่ได้รับการยืนยันโดย wallet เอง** หาก wallet ที่ถือกุญแจส่วนตัว เช่น wallet แบบฮาร์ดแวร์ ไม่ได้ยืนยันอย่างชัดเจนว่าที่อยู่ใดเกี่ยวข้องกับมัน การส่งบิตคอยน์ไปยังที่อยู่นี้ถือเป็นการปฏิบัติที่เสี่ยง จริงๆ แล้ว หากไม่มีการยืนยันนี้ จะไม่มีการรับประกันว่าที่อยู่นั้นเป็นของ wallet ของคุณจริงๆ ดังนั้น ควรใช้ฟังก์ชันการรับของ wallet แบบดูได้อย่างเดียวด้วยความระมัดระวัง โดยคำนึงว่าทุนที่ส่งไปอาจสูญหายได้


หากต้องการรับบิตคอยน์ผ่าน Sentinel ให้เลือกคอลเลกชันที่สนใจ จากนั้นคลิกที่แท็บที่ตรงกับคีย์สาธารณะขยายที่คุณต้องการโอนเงินไปยัง


![watch-only](assets/notext/17.webp)


สุดท้าย ให้คลิกที่ไอคอนลูกศรที่มุมล่างซ้ายของหน้าจอ จากนั้น Sentinel จะสร้างที่อยู่รับที่ว่างเปล่าสำหรับคุณ คุณสามารถคัดลอกหรือสแกนโดยใช้ QR โค้ดได้


![watch-only](assets/notext/18.webp)


ไปที่ generate ถึง PSBT จาก Sentinel และเริ่มต้นการทำธุรกรรมการใช้จ่าย ไปที่คีย์ขยายของ wallet ที่คุณต้องการทำการชำระเงิน ตัวอย่างเช่น บัญชีเงินฝากของฉันบน Samourai wallet จากนั้นคลิกที่ไอคอนลูกศรที่มุมล่างขวาของหน้าจอ


![watch-only](assets/notext/19.webp)


ป้อนพารามิเตอร์ทั้งหมดที่เกี่ยวข้องกับธุรกรรมของคุณ:



- ป้อนที่อยู่ของผู้รับ (โดยการคลิกที่ไอคอน QR code คุณมีตัวเลือกในการสแกนที่อยู่นี้);
- ระบุจำนวนเงินที่จะส่งไปยังที่อยู่นี้;
- กำหนดค่าธรรมเนียมการทำธุรกรรม


เมื่อคุณกรอกข้อมูลในช่องที่จำเป็นทั้งหมดสำหรับการทำธุรกรรมของคุณแล้ว ให้กดปุ่ม `COMPOSE UNSIGNED TRANSACTION`

จากนั้นคุณจะเข้าถึง PSBT ซึ่งแสดงถึงธุรกรรม Bitcoin ที่ถูกสร้างขึ้นแต่ยังไม่ได้ลงนาม เนื่องจาก Sentinel ไม่มีสิทธิ์เข้าถึงกุญแจส่วนตัวของคุณ คุณมีตัวเลือกที่จะคัดลอกธุรกรรมนี้ ส่งออกเป็นไฟล์ `.psbt` หรือสแกนผ่าน QR code แบบเคลื่อนไหว


![watch-only](assets/notext/21.webp)


จากนั้น ไปที่ wallet ของคุณที่มีคีย์ส่วนตัวเพื่อเซ็นชื่อธุรกรรม (Samourai, ฮาร์ดแวร์ wallet...)


![watch-only](assets/notext/22.webp)


เมื่อทำการลงนามธุรกรรมแล้ว คุณสามารถกลับไปที่ Sentinel เพื่อกระจายธุรกรรมได้ ในการทำเช่นนี้ จากเมนูหลัก ให้คลิกที่จุดเล็ก ๆ สามจุดที่มุมขวาบน จากนั้นคลิกที่ `Broadcast transaction`.


![watch-only](assets/notext/23.webp)


คุณมีตัวเลือกในการส่ง PSBT ที่ลงนามแล้วของคุณได้สามวิธี:



- โดยการวางจากคลิปบอร์ดของคุณโดยตรง;
- โดยการนำเข้าจากไฟล์ `.psbt`;
- โดยการสแกนผ่านคิวอาร์โค้ด


![watch-only](assets/notext/24.webp)


เมื่อป้อนธุรกรรมที่ลงนามแล้วในกรอบสีเทา คุณสามารถคลิกที่ปุ่มสีเขียว `BROADCAST TRANSACTION` เพื่อกระจายธุรกรรมนั้นบนเครือข่าย Bitcoin Sentinel จะให้ TXID แก่คุณ
