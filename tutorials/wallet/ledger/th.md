### การตั้งค่าอุปกรณ์


Ledger มาพร้อมกับสาย USB ของตัวเอง ตรวจสอบให้แน่ใจว่าคุณใช้สายนี้และไม่ใช่สายเก่าๆ บางสาย USB ใช้สำหรับจ่ายไฟเท่านั้น สายนี้สามารถส่งข้อมูลและจ่ายไฟได้ เมื่อฉันใช้สาย USB สำหรับชาร์จโทรศัพท์ที่มีอยู่ทั่วไป อุปกรณ์ไม่สามารถเชื่อมต่อได้


เชื่อมต่อกับคอมพิวเตอร์ของคุณและอุปกรณ์จะเปิดเครื่อง

2. กู้คืนจากวลีการกู้คืน


โดยพื้นฐานแล้ว มันถามว่าคุณต้องการให้เครื่องสร้าง seed ให้คุณหรือถ้าคุณมีอยู่แล้วที่คุณต้องการใช้ การสร้าง seed ของคุณเองเป็นวิธีที่ดีที่สุด แต่การทำเช่นนั้นอย่างปลอดภัยเป็นเรื่องที่ซับซ้อนมากและอยู่นอกเหนือขอบเขตของบทความนี้ เลือก "ตั้งค่าเป็นอุปกรณ์ใหม่"

จากนั้นจะแสดงคำ 24 คำที่คุณต้องวนผ่านและเขียนลงไป

### เพิ่ม passphrase


ตอนนี้ที่เรามีแอป Bitcoin แล้ว เราสามารถเพิ่ม passphrase เข้าไปในวลี seed ของเราได้ เราไม่สามารถทำเช่นนั้นได้ก่อนหน้านี้เมื่อ seed ถูกสร้างขึ้นครั้งแรก เพราะในตอนเริ่มต้น เราไม่มีแอป Bitcoin และเราจำเป็นต้องเชื่อมต่อกับ Ledger Live เพื่อให้ได้มันมา


ไปที่เมนู “settings” ภายในอุปกรณ์ จากนั้นไปที่เมนูย่อย “security” แล้วเลือก passphrase คุณจะเห็น “Advanced feature” คลิกปุ่มขวา คุณจะเห็น “read manuel…” และหลังจากคลิกปุ่มขวาอีกครั้ง คุณจะเห็น “back” แต่นั่นยังไม่จบ โดยสัญชาตญาณ คุณอาจคิดว่าเป็นเช่นนั้น แต่คลิกปุ่มขวาอีกครั้ง คุณจะเห็น “set up passphrase”


คุณสามารถเลือกที่จะ "แนบกับ PIN" หรือ "ตั้งค่าแบบชั่วคราว" ได้ ฉันแนะนำให้ "แนบกับ PIN" ด้วยวิธีนี้ คุณสามารถเข้าถึงกระเป๋าเงินต่างๆ ได้ขึ้นอยู่กับ PIN ที่คุณป้อนเมื่อเปิดอุปกรณ์ครั้งแรก หากคุณ "ตั้งค่าแบบชั่วคราว" คุณจะต้องป้อน passphrase ทุกครั้งที่คุณต้องการเข้าถึง wallet นั้น แต่จะเป็นจาก PIN เริ่มต้นเสมอ


ป้อน passphrase และยืนยัน


มันจะถามคุณถึง "PIN ปัจจุบัน" ซึ่งไม่ใช่ PIN ที่คุณกำลังเชื่อมโยงกับ passphrase ใหม่ แต่เป็น PIN ที่คุณป้อนเมื่อคุณเปิดอุปกรณ์สำหรับเซสชันนี้


ตอนนี้คุณสามารถออกไปยังเมนูหลักได้โดยเลือกตัวเลือกย้อนกลับหลายครั้ง


### กำลังดู Wallet


ในบทความก่อนหน้านี้ ฉันได้อธิบายวิธีการดาวน์โหลดและตรวจสอบ Sparrow wallet และวิธีการเชื่อมต่อกับโหนดของคุณเอง หรือโหนดสาธารณะ คุณควรทำตามคำแนะนำเหล่านี้:



- ติดตั้ง Bitcoin Core ( https://armantheparman.com/bitcoincore/)



- ติดตั้ง Sparrow Bitcoin Wallet (https://armantheparman.com/download-sparrow/)



- เชื่อมต่อ Sparrow Bitcoin Wallet กับ Bitcoin Core (https://armantheparman.com/sparrowcore/)


ทางเลือกหนึ่งในการใช้ Sparrow Bitcoin Wallet คือ Electrum Desktop Wallet แต่ฉันจะอธิบาย Sparrow Bitcoin Wallet ต่อไปเนื่องจากฉันเห็นว่ามันดีที่สุดสำหรับคนส่วนใหญ่ ผู้ใช้ขั้นสูงอาจต้องการใช้ Electrum เป็นทางเลือก


เราจะโหลดมันขึ้นมาและเชื่อมต่อ Ledger โดยที่ wallet มี passphrase อยู่ภายใน wallet นี้ไม่เคยถูกเปิดเผยต่อ Ledger Live เพราะมันถูกสร้างขึ้นหลังจากที่เราเชื่อมต่ออุปกรณ์กับ Ledger Live ตรวจสอบให้แน่ใจว่าคุณจะไม่เชื่อมต่อมันกับ Ledger Live อีกครั้งเพื่อไม่ให้เปิดเผย wallet ส่วนตัวใหม่ของคุณ

โปรดสังเกตที่ช่องทำเครื่องหมาย “มีธุรกรรมที่มีอยู่” หากนี่คือ wallet ที่คุณเคยใช้มาก่อน ให้ทำเครื่องหมายในช่องนี้ มิฉะนั้นยอดคงเหลือของคุณจะแสดงเป็นศูนย์อย่างไม่ถูกต้อง การทำเครื่องหมายในช่องนี้จะขอให้ Sparrow ตรวจสอบฐานข้อมูลของ Bitcoin Core (บล็อกเชน) สำหรับธุรกรรมก่อนหน้า สำหรับคำแนะนำนี้ เรากำลังใช้ wallet ใหม่เอี่ยม ดังนั้นคุณสามารถปล่อยให้ช่องนี้ไม่ถูกทำเครื่องหมายได้


![image](assets/16.webp)


คลิกที่ “Connected Hardware Wallet” และตรวจสอบให้แน่ใจว่าอุปกรณ์เชื่อมต่ออยู่ เปิดเครื่องแล้ว ป้อน PIN แล้ว และคุณได้เข้าสู่แอป Bitcoin แล้ว


![image](assets/17.webp)


คลิก “สแกน” แล้วคลิก “นำเข้า Keystore” ในหน้าจอถัดไป


![image](assets/18.webp)


ไม่มีอะไรให้แก้ไขในหน้าจอถัดไป Ledger ได้กรอกข้อมูลให้คุณแล้ว คลิก "Apply"


![image](assets/19.webp)


หน้าจอถัดไปจะให้คุณเพิ่มรหัสผ่าน อย่าสับสนกับ “passphrase”; หลายคนจะสับสน การตั้งชื่อนี้ไม่ค่อยดีนัก รหัสผ่านนี้จะช่วยให้คุณล็อก wallet นี้บนคอมพิวเตอร์ของคุณ มันเฉพาะเจาะจงกับซอฟต์แวร์นี้บนคอมพิวเตอร์นี้ มันไม่ใช่ส่วนหนึ่งของกุญแจส่วนตัว Bitcoin ของคุณ


![image](assets/20.webp)


หลังจากหยุดชั่วคราวขณะที่คอมพิวเตอร์กำลังประมวลผล คุณจะเห็นปุ่มทางด้านซ้ายเปลี่ยนจากสีเทาเป็นสีน้ำเงิน ยินดีด้วย wallet ของคุณพร้อมใช้งานแล้ว ทำและส่งธุรกรรมได้ตามใจคุณ


![image](assets/21.webp)


### การรับ


หากต้องการรับบิตคอยน์ ให้ไปที่แท็บที่อยู่ทางด้านซ้ายและเลือกหนึ่งในที่อยู่เพื่อรับ เพียงคลิกขวาที่ที่อยู่ที่คุณต้องการ แล้วเลือก "คัดลอกที่อยู่" จากนั้นไปที่การแลกเปลี่ยนของคุณที่เงินถูกส่งมาและวางที่อยู่นั้นลงไป หรือคุณอาจให้ที่อยู่นี้แก่ลูกค้าที่สามารถใช้เพื่อชำระเงินให้คุณได้


เมื่อคุณใช้ wallet เป็นครั้งแรก คุณควรได้รับจำนวนเล็กน้อย ฝึกฝนการใช้จ่ายไปยังที่อยู่อื่น ไม่ว่าจะภายใน wallet หรือกลับไปยังการแลกเปลี่ยน เพื่อพิสูจน์ว่า wallet ทำงานตามที่คาดหวัง


เมื่อคุณทำเช่นนั้นแล้ว คุณต้องสำรองคำที่คุณเขียนลงไป สำเนาเดียวไม่เพียงพอ ควรมีสำเนากระดาษอย่างน้อยสองชุด (โลหะจะดีกว่า) และเก็บไว้ในสองสถานที่ที่แตกต่างกันและปลอดภัยดี สิ่งนี้จะลดความเสี่ยงจากภัยธรรมชาติที่อาจทำลาย HWW และสำเนากระดาษของคุณในเหตุการณ์เดียว ดู “การใช้กระเป๋าเงินฮาร์ดแวร์ Bitcoin” สำหรับการอภิปรายอย่างเต็มรูปแบบในเรื่องนี้


### กำลังส่ง


![image](assets/22.webp)


เมื่อทำการชำระเงิน คุณจำเป็นต้องวางที่อยู่ที่คุณต้องการชำระในช่อง “Pay to” คุณไม่สามารถปล่อยให้ช่อง Label ว่างได้ มันเป็นเพียงสำหรับบันทึกของกระเป๋าเงินของคุณเอง แต่ Sparrow ไม่อนุญาตให้ทำเช่นนั้น – เพียงแค่ใส่อะไรบางอย่าง (มีเพียงคุณเท่านั้นที่จะเห็นมัน) ใส่จำนวนเงินและคุณยังสามารถปรับค่าธรรมเนียมที่คุณต้องการได้ด้วยตนเอง


Wallet ไม่สามารถลงนามในธุรกรรมได้หากไม่ได้เชื่อมต่อ HWW นั่นคือหน้าที่ของ HWW – เพื่อรับธุรกรรม ลงนาม และส่งกลับมาให้พร้อมลายเซ็น ตรวจสอบให้แน่ใจว่าเมื่อคุณลงนามบนอุปกรณ์ คุณได้ตรวจสอบที่อยู่ที่คุณกำลังชำระเงินว่าเหมือนกันทั้งบนอุปกรณ์และบนหน้าจอคอมพิวเตอร์ และใบแจ้งหนี้ที่คุณได้รับ (เช่น คุณอาจได้รับอีเมลให้ชำระเงินไปยังที่อยู่ที่กำหนด)


โปรดทราบด้วยว่าหากคุณเลือกใช้เหรียญที่มีมูลค่ามากกว่าจำนวนเงินที่ต้องชำระ ส่วนที่เหลือจะถูกส่งกลับไปยังที่อยู่เปลี่ยนแปลงของกระเป๋าเงินของคุณ บางคนไม่ทราบเรื่องนี้และได้ตรวจสอบธุรกรรมของตนบนบล็อกเชนสาธารณะ และคิดว่า bitcoin บางส่วนถูกส่งไปยังที่อยู่ของผู้โจมตี แต่ในความเป็นจริง มันคือที่อยู่เปลี่ยนแปลงของพวกเขาเอง


### เฟิร์มแวร์


ในการอัปเดตเฟิร์มแวร์ คุณจำเป็นต้องเชื่อมต่อกับ Ledger Live หากคุณต้องการทำเช่นนี้ คุณควรล้างข้อมูลในอุปกรณ์ก่อน และตรวจสอบให้แน่ใจว่าคุณมีคำสำรองและ passphrase พร้อมสำหรับการกู้คืนอุปกรณ์ เหตุผลที่ฉันชอบล้างข้อมูลในอุปกรณ์ก่อนคือคุณต้องเชื่อมต่ออุปกรณ์ของคุณกับ Ledger Live เพื่ออัปเดตเฟิร์มแวร์ และฉันไม่ต้องการให้อุปกรณ์ wallet ใหม่ของคุณ (อันที่มี passphrase) สัมผัสกับ Ledger Live เลย ฉันไม่ไว้ใจว่า Ledger จะไม่ดึงข้อมูลคีย์สาธารณะจากอุปกรณ์เมื่อฉันเชื่อมต่อกับ Ledger Live พวกเขาอ้างว่าไม่ทำ แต่ฉันไม่สามารถตรวจสอบได้ด้วยตัวเอง เว้นแต่ฉันจะอ่านโค้ดและเข้าใจฮาร์ดแวร์ภายในด้วย


### บทสรุป


บทความนี้แสดงให้คุณเห็นวิธีการใช้ Ledger HWW ในวิธีที่ปลอดภัยและเป็นส่วนตัวมากกว่าที่โฆษณาไว้ – แต่บทความนี้เพียงอย่างเดียวไม่เพียงพอ ดังที่ฉันได้กล่าวไว้ตั้งแต่ต้น คุณควรผสมผสานกับข้อมูลที่ให้ไว้ใน “การใช้กระเป๋าเงินฮาร์ดแวร์ Bitcoin“
