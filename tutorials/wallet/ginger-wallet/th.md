Ginger Wallet เป็นโอเพ่นซอร์สที่ไม่ใช่การดูแล Bitcoin wallet ที่มุ่งเน้นความลับและความเป็นส่วนตัว มันเริ่มต้นชีวิตเป็น fork จาก Wasabi Wallet (หลังจากเวอร์ชัน 2.0.7.2 - ใบอนุญาต MIT)



Ginger Wallet รักษาสถาปัตยกรรมทางเทคนิคของ Wasabi ไว้ในขณะที่เพิ่มคุณสมบัติเฉพาะบางอย่าง ตามที่ระบุใน [เอกสาร Ginger Wallet](https://docs.gingerwallet.io/why-ginger/difference.html#gingerwallet) Wasabi เน้นที่ **ความเป็นอิสระและการควบคุม** ในขณะที่ Ginger มุ่งเน้นที่ **ความง่ายในการใช้งาน ความปลอดภัย และประสบการณ์ที่เรียบง่าย** ทำให้สามารถเข้าถึงได้สำหรับผู้ที่ไม่คุ้นเคยกับด้านเทคนิคมากนัก



Ginger Wallet เป็นซอฟต์แวร์ wallet สำหรับคอมพิวเตอร์เท่านั้น (ไม่มีแอปพลิเคชันมือถือ)



## Coinjoin คืออะไร?



**coinjoin** เป็นโครงสร้างธุรกรรมพิเศษ Bitcoin ที่รวมผู้เข้าร่วมหลายคนไว้ในธุรกรรมร่วมเดียว กลไกนี้จะผสมรายการของผู้ใช้ต่างๆ เข้าด้วยกันในธุรกรรมทั่วไป ทำให้ยากมาก - ถ้าทำอย่างถูกต้องก็แทบจะเป็นไปไม่ได้ - ที่จะติดตามเงินทุน ดังนั้นจึงแทบจะเป็นไปไม่ได้สำหรับผู้สังเกตการณ์ภายนอกที่จะระบุแหล่งที่มาและปลายทางของบิตคอยน์ที่เกี่ยวข้องได้อย่างแน่นอน ซึ่งแตกต่างจากธุรกรรม Bitcoin แบบดั้งเดิม



สำหรับคุณ ในฐานะผู้ใช้ coinjoin ช่วยรักษาความลับของคุณได้ ตัวอย่างเช่น หากคุณได้รับเงินบริจาค 10,000 sats บนที่อยู่ Bitcoin ผู้ส่งสามารถติดตามเงินเหล่านี้และในบางกรณีอาจสรุปได้ว่าคุณถือบิตคอยน์ในปริมาณมากขึ้น หรือสังเกตกิจกรรมของคุณ โดยการทำ coinjoin หลังจากการบริจาค 10,000 sats นี้ คุณจะทำลายความสามารถในการติดตาม: ผู้ส่งจะไม่สามารถสรุปข้อมูลใดๆ เกี่ยวกับคุณจากการชำระเงินนี้ได้อีกต่อไป



Chaumian coinjoin มอบความปลอดภัยในระดับสูง เนื่องจากเงินทุนยังคงอยู่ภายใต้การควบคุมเฉพาะของผู้ใช้ตลอดเวลา แม้แต่ผู้ดำเนินการของเซิร์ฟเวอร์ประสานงานก็ไม่สามารถเบี่ยงเบนบิตคอยน์ของผู้เข้าร่วมได้ไม่ว่ากรณีใด ๆ ทั้งผู้ใช้และผู้ประสานงานไม่จำเป็นต้องไว้วางใจกัน: แต่ละคนยังคงควบคุมกุญแจส่วนตัวของตนเอง และได้รับอนุญาตเพียงผู้เดียวในการตรวจสอบธุรกรรม ดังนั้นจึงไม่มีบุคคลที่สามสามารถยึดบิตคอยน์ของคุณระหว่างการทำ coinjoin หรือสร้างความเชื่อมโยงโดยตรงระหว่างอินพุตและเอาต์พุตของคุณได้

![screen](assets/fr/21.webp)



**หมายเหตุ** :




- สำหรับการยืนยันตัวตนแบบสองปัจจัย ตรวจสอบให้แน่ใจว่าแอปพลิเคชันการยืนยันตัวตนของคุณรองรับโปรโตคอล SHA256 และรหัส 8 หลัก Ginger Wallet ต้องการรหัส 2FA 8 หลักเพื่อเพิ่มความปลอดภัย รูปแบบที่ยาวขึ้นนี้ทำให้รหัสยากต่อการคาดเดาหรือถูกบุกรุกมากขึ้น มอบการป้องกันที่ดียิ่งขึ้นต่อการเข้าถึงโดยไม่ได้รับอนุญาต
- ตามค่าเริ่มต้น การรับส่งข้อมูลเครือข่ายทั้งหมดของ Ginger จะผ่าน Tor โดยไม่จำเป็นต้องกำหนดค่าด้วยตนเอง หาก Tor ทำงานอยู่แล้วในระบบของคุณ Ginger จะให้ความสำคัญกับ Tor โดยอัตโนมัติ



แต่เมื่อคุณปิดใช้งาน Tor ในการตั้งค่า ความเป็นส่วนตัวของคุณจะยังคงได้รับการรักษาโดยทั่วไป ยกเว้นในสองสถานการณ์:




- ในระหว่าง Coinjoin ผู้ประสานงานสามารถเชื่อมโยงข้อมูลนำเข้าและส่งออกของคุณกับที่อยู่ IP ของคุณได้;
- เมื่อทำการกระจายธุรกรรม โหนดที่เป็นอันตรายที่คุณเชื่อมต่ออาจเชื่อมโยงธุรกรรมของคุณกับ IP ของคุณได้



อย่าลืมกด **เสร็จสิ้น** (ที่มุมล่างขวา) ทุกครั้ง เพื่อบันทึกการตั้งค่าของคุณ การตั้งค่าบางอย่างต้องการให้ Ginger Wallet รีสตาร์ทเพื่อให้มีผล



นอกจากนี้ แถบค้นหาที่ด้านบนของกระเป๋าเงินช่วยให้คุณค้นหาและเข้าถึงพารามิเตอร์ใดๆ เป็นต้น...

### การกำหนดค่าพอร์ตโฟลิโอ



สามารถสร้างกระเป๋าเงินหลายใบในแอปพลิเคชันได้ ดังนั้นแต่ละ wallet สามารถกำหนดค่าให้เหมาะสมกับความต้องการของคุณได้ ในการทำเช่นนั้น ให้คลิกที่ **จุดสามจุด** ด้านหน้าชื่อ wallet จากนั้นคลิกที่ **การตั้งค่าพอร์ตโฟลิโอ**



![screen](assets/fr/23.webp)



ตามที่คุณเห็น นอกเหนือจากพารามิเตอร์ wallet แล้ว คุณจะสามารถดู UTXOs ของคุณ (รายการโทเค็นที่คุณเป็นเจ้าของ), สถิติ และข้อมูล wallet (เช่น กุญแจสาธารณะที่ขยายออก)



ในการกลับไปยังการตั้งค่า wallet ของเรา เมื่อคุณคลิกที่พารามิเตอร์ wallet คุณจะถูกนำไปยังแท็บต่อไปนี้:




- ทั่วไป** (ที่ซึ่งคุณสามารถเปลี่ยนชื่อ wallet ได้) ;



![screen](assets/fr/24.webp)





- Coinjoin** (ที่คุณสามารถปรับแต่งการตั้งค่า coinjoin สำหรับ wallet นี้) ;



![screen](assets/fr/25.webp)





- เครื่องมือ** (ที่คุณสามารถตรวจสอบ seedphrase ของคุณ, ซิงโครไนซ์ wallet ของคุณอีกครั้ง, หรือลบมันได้)



![screen](assets/fr/26.webp)

หากคุณได้ทำการสั่งซื้อแล้ว คุณสามารถดูประวัติการทำธุรกรรมของคุณได้โดยคลิกที่ปุ่ม **คำสั่งซื้อก่อนหน้า** รายการธุรกรรมที่ผ่านมาและสถานะของพวกเขาจะถูกแสดงขึ้น





- เลือกข้อเสนอที่เหมาะสมสำหรับคุณ



ณ จุดนี้ คุณจะเห็นรายการข้อเสนอทั้งหมดที่มีอยู่ สำหรับแต่ละข้อเสนอ คุณมี :




 - ชื่อผู้จัดหา (1) ;
 - จำนวนบิตคอยน์ที่เทียบเท่ากับจำนวนเงินที่ป้อนก่อนหน้านี้ วิธีการชำระเงิน และค่าธรรมเนียมการซื้อ (2) ;
 - ปุ่ม **Accept** (3)



![screen](assets/fr/35.webp)



ค่าธรรมเนียมที่ระบุในข้อเสนอไม่ได้เป็นค่าใช้จ่ายเพิ่มเติม พวกมันได้รวมอยู่ในจำนวนเงินรวมของข้อเสนอแล้ว



มุมขวาบนของหน้าจอที่มีป้ายกำกับว่า **ทั้งหมด** ให้คุณกรองข้อเสนอโดยวิธีการชำระเงิน วิธีการชำระเงินที่คุณเลือกจะถูกตั้งค่าเป็นค่าเริ่มต้น แต่สามารถเปลี่ยนแปลงได้ตลอดเวลา



![screen](assets/fr/36.webp)



หากคุณพบข้อเสนอที่เหมาะสม ให้คลิกที่ปุ่ม **ยอมรับ** เพื่อดำเนินการซื้อ จากนั้นคุณจะถูกเปลี่ยนเส้นทางไปยังหน้าของผู้ขาย ซึ่งคุณสามารถทำธุรกรรมให้เสร็จสมบูรณ์ได้



### ขายบิตคอยน์



ผู้ใช้ Ginger Wallet สามารถขาย Bitcoin ได้ ปุ่ม **ขาย** จะมองเห็นได้เฉพาะเมื่อมีเงินทุนใน wallet เท่านั้น





- คลิกที่ **ขาย**.



![screen](assets/fr/37.webp)





- เช่นเดียวกับตัวเลือก **ซื้อ** เมื่อคุณใช้ฟังก์ชันขายเป็นครั้งแรก คุณต้องเลือกประเทศของคุณก่อนที่จะดำเนินการขายบิตคอยน์





- ถัดไป คุณต้องป้อนจำนวนบิตคอยน์ที่คุณต้องการขาย คุณสามารถป้อนจำนวนนี้ในรูปแบบ BTC หรือในสกุลเงินทั่วไป เช่น ดอลลาร์สหรัฐ (USD)





- เมื่อคุณทำเสร็จแล้ว คุณจะเห็นรายการข้อเสนอที่มีอยู่ เลือกข้อเสนอการขายที่เหมาะกับคุณ จากนั้นคลิก **ยอมรับ** เพื่อดำเนินการต่อ





- ตอนนี้คุณต้องทำธุรกรรมให้เสร็จสิ้น:
 - เมื่อคุณยอมรับข้อเสนอแล้ว คุณจะถูกเปลี่ยนเส้นทางไปยังหน้าของผู้จัดหา;
 - ทำตามคำแนะนำในหน้าซัพพลายเออร์ ;
 - ในบางจุด คุณจะได้รับที่อยู่ผู้รับและจำนวนเงินที่ต้องส่ง;
 - จากนั้นกลับไปที่ Ginger Wallet เพื่อดำเนินการต่อ;
 - เมื่อกลับไปที่ Ginger Wallet กล่องโต้ตอบจะปรากฏขึ้น ให้คุณดำเนินการต่อโดยคลิก **Send**.



สิ่งนี้จะเปิดหน้าจอ **ส่ง** พร้อมที่อยู่ของผู้รับและจำนวนเงินที่กรอกไว้ล่วงหน้า คุณยังสามารถใช้ปุ่ม **ส่ง** บนหน้าจอหลักได้อีกด้วย แม้ว่าคุณจะสามารถส่งธุรกรรมด้วยตนเองได้ แต่เราขอแนะนำให้คุณดำเนินการผ่านกล่องโต้ตอบเพื่อกระบวนการที่เหมาะสมยิ่งขึ้น



## การทำ coinjoin บน Ginger Wallet



![Vidéo](https://youtu.be/AJe67RDfB1A)



ปกป้องความลับของบิตคอยน์ของคุณด้วย **Coinjoin** ซึ่งรวมเข้ากับ Ginger Wallet โดยตรง wallet ใช้ **WabiSabi** ซึ่งเป็นโปรโตคอล Chaumian coinjoin ที่ออกแบบมาเพื่ออำนวยความสะดวกในการทำ coinjoin ที่เข้าถึงได้ง่ายและมีประสิทธิภาพมากขึ้น



ขึ้นอยู่กับคุณที่จะเลือกกลยุทธ์การรวมเหรียญ (อัตโนมัติหรือด้วยตนเอง) ที่เหมาะสมกับคุณที่สุด



Ginger Coinjoin พร้อมใช้งานทันทีที่คุณดาวน์โหลด (ไม่ต้องมีขั้นตอนเพิ่มเติม) โดยอัตโนมัติ Ginger Coinjoin จะทำงานในพื้นหลังเพื่อปกป้องความเป็นส่วนตัวของคุณในทุกการทำธุรกรรม ในทางปฏิบัติ ผู้เล่น Coinjoin จะปรากฏขึ้นเมื่อใดก็ตามที่คุณมียอดคงเหลือที่สามารถทำให้ไม่ระบุตัวตนได้



สำหรับการเริ่มต้นใช้งาน coinjoin แบบแมนนวล มันเป็นการดำเนินการเพียงคลิกเดียว เริ่มรอบและรอให้การทำธุรกรรม coinjoin ถูกสร้างและยืนยัน คุณจะเห็นคะแนนการไม่ระบุตัวตนในอินเทอร์เฟซ



สามารถทำการผสมหลายครั้งจนกว่าจะถึงระดับความไม่เปิดเผยตัวตนที่ต้องการได้ คุณยังสามารถยกเว้นบางส่วนจากการผสมได้อีกด้วย



ตามค่าเริ่มต้น Ginger ใช้ผู้ประสานงานของตัวเองพร้อมพารามิเตอร์ที่กำหนดไว้ล่วงหน้าและค่าธรรมเนียมที่รับประกัน Coinjoins ของโทเค็นที่มีมูลค่ามากกว่า 0.03 BTC จะมีค่าธรรมเนียมผู้ประสานงาน 0.3% นอกเหนือจากค่าธรรมเนียม mining รายการที่มีมูลค่า 0.03 BTC หรือน้อยกว่า รวมถึงการรีมิกซ์ จะได้รับการยกเว้นจากค่าธรรมเนียมผู้ประสานงาน แม้หลังจากการทำธุรกรรมเพียงครั้งเดียว ดังนั้น การชำระเงินด้วยเงินทุน Coinjoin จะทำให้ทั้งผู้ส่งและผู้รับสามารถรีมิกซ์เหรียญของพวกเขาได้โดยไม่ต้องเสียค่าธรรมเนียมผู้ประสานงาน



Ginger ชอบการทำ coinjoin ที่มีผู้เข้าร่วมมากกว่าในรอบที่เล็กกว่าและเร็วกว่า การทำ coinjoin ที่ใหญ่กว่าจะให้ความเป็นนิรนามมากขึ้น ค่าใช้จ่ายต่ำกว่า และประสิทธิภาพของพื้นที่บล็อกที่มากขึ้น
