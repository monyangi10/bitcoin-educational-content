wallet เป็นอุปกรณ์อิเล็กทรอนิกส์ที่ทุ่มเทให้กับการจัดการและความปลอดภัยของคีย์ส่วนตัวของ Bitcoin wallet ไม่เหมือนกับซอฟต์แวร์วอลเล็ต (หรือฮอตวอลเล็ต) ที่ติดตั้งบนเครื่องทั่วไปที่มักเชื่อมต่อกับอินเทอร์เน็ต ฮาร์ดแวร์วอลเล็ตช่วยให้สามารถแยกคีย์ส่วนตัวออกจากกันทางกายภาพ ลดความเสี่ยงจากการแฮ็กและการโจรกรรม


เป้าหมายหลักของฮาร์ดแวร์ wallet คือการลดฟังก์ชันการทำงานของอุปกรณ์เพื่อลดพื้นผิวการโจมตี พื้นผิวการโจมตีที่เล็กลงยังหมายถึงเวกเตอร์การโจมตีที่มีศักยภาพน้อยลง กล่าวคือ จุดอ่อนในระบบที่ผู้โจมตีอาจใช้ประโยชน์เพื่อเข้าถึงบิตคอยน์ก็จะน้อยลงด้วย


ขอแนะนำให้ใช้ฮาร์ดแวร์ wallet เพื่อรักษาความปลอดภัยของบิตคอยน์ของคุณ โดยเฉพาะอย่างยิ่งหากคุณถือครองจำนวนมาก ไม่ว่าจะเป็นในมูลค่าสัมบูรณ์หรือเป็นสัดส่วนของสินทรัพย์ทั้งหมดของคุณ


กระเป๋าเงินฮาร์ดแวร์ถูกใช้ร่วมกับซอฟต์แวร์การจัดการ wallet บนคอมพิวเตอร์หรือสมาร์ทโฟน ซอฟต์แวร์นี้จัดการการสร้างธุรกรรม แต่ลายเซ็นเข้ารหัสที่จำเป็นในการยืนยันธุรกรรมเหล่านี้จะทำเฉพาะภายในฮาร์ดแวร์ wallet เท่านั้น ซึ่งหมายความว่าคีย์ส่วนตัวจะไม่ถูกเปิดเผยต่อสภาพแวดล้อมที่อาจมีความเสี่ยง


กระเป๋าเงินฮาร์ดแวร์มอบการป้องกันสองชั้นให้กับผู้ใช้: ในด้านหนึ่ง พวกมันรักษาความปลอดภัยให้กับบิตคอยน์ของคุณจากการโจมตีระยะไกลโดยการเก็บคีย์ส่วนตัวแบบออฟไลน์ และในอีกด้านหนึ่ง พวกมันมักจะมีความต้านทานทางกายภาพที่ดีกว่าต่อความพยายามในการดึงคีย์ออกมา และมันคือเกณฑ์ความปลอดภัยทั้ง 2 ข้อนี้ที่สามารถใช้ในการตัดสินและจัดอันดับรุ่นต่างๆ ที่มีอยู่ในตลาด


ในบทแนะนำนี้ ฉันขอเสนอให้ค้นพบหนึ่งในโซลูชันเหล่านี้: Tapsigner โดย Coinkite.


## บทนำสู่ Tapsigner


Tapsigner เป็นฮาร์ดแวร์ wallet ที่ออกแบบในรูปแบบของบัตร NFC โดยบริษัท Coinkite ซึ่งเป็นที่รู้จักในการผลิต Coldcards


![TAPSIGNER NUNCHUK](assets/notext/01.webp)


Tapsigner ช่วยให้สามารถจัดเก็บคู่ของคีย์ส่วนตัวหลักและ chain code ตามมาตรฐาน BIP32 เพื่อสร้างต้นไม้ของคีย์เข้ารหัสลับ คีย์เหล่านี้สามารถใช้ในการลงนามธุรกรรมโดยการวาง Tapsigner ไว้ใกล้โทรศัพท์หรือเครื่องอ่านบัตร NFC

บัตร NFC นี้ขายในราคา $19.99 ซึ่งถือว่าราคาย่อมเยากว่าอุปกรณ์เก็บรักษาเงินดิจิทัลอื่น ๆ ที่มีอยู่ในตลาด อย่างไรก็ตาม เนื่องจากรูปแบบของมัน Tapsigner จึงไม่มีตัวเลือกมากมายเท่าอุปกรณ์อื่น ๆ แน่นอนว่าไม่มีแบตเตอรี่ ไม่มีกล้อง และไม่มีเครื่องอ่านการ์ด micro SD เนื่องจากมันเป็นบัตร ในความเห็นของฉัน ข้อเสียที่ใหญ่ที่สุดคือการขาดหน้าจอบนฮาร์ดแวร์ wallet ซึ่งทำให้มันเสี่ยงต่อการโจมตีระยะไกลบางประเภทมากขึ้น จริง ๆ แล้ว สิ่งนี้บังคับให้ผู้ใช้ต้องเซ็นชื่อโดยไม่เห็นและต้องเชื่อถือสิ่งที่พวกเขาเห็นบนหน้าจอคอมพิวเตอร์ของพวกเขา


แม้จะมีข้อจำกัด แต่ Tapsigner ก็อาจน่าสนใจเนื่องจากราคาที่ลดลง wallet นี้สามารถใช้เพื่อเพิ่มความปลอดภัยของการใช้จ่าย wallet นอกเหนือจากการออม wallet ที่ได้รับการป้องกันโดยฮาร์ดแวร์ wallet ที่ติดตั้งหน้าจอ นอกจากนี้ยังเป็นทางออกที่ดีสำหรับผู้ที่ถือบิตคอยน์ในจำนวนเล็กน้อยและไม่ต้องการลงทุนหลายร้อยยูโรในอุปกรณ์ที่ซับซ้อนมากขึ้น ยิ่งไปกว่านั้น การใช้ Tapsigner ในการกำหนดค่ามัลติซิก หรืออาจจะในระบบ wallet ที่มีการล็อกเวลาในอนาคต สามารถให้ประโยชน์ที่น่าสนใจได้


## วิธีการซื้อ Tapsigner?


Tapsigner มีวางจำหน่ายสำหรับการซื้อ [บนเว็บไซต์ทางการของ Coinkite](https://store.coinkite.com/store/category/tapsigner) หากต้องการซื้อในร้านค้าจริง คุณสามารถค้นหา [รายชื่อผู้ค้าปลีกที่ได้รับการรับรอง](https://coinkite.com/resellers) บนเว็บไซต์ได้เช่นกัน


คุณจะต้องมีโทรศัพท์ที่รองรับการสื่อสาร NFC หรืออุปกรณ์ USB เพื่ออ่านบัตร NFC ที่ความถี่มาตรฐาน 13.56 MHz

![TAPSIGNER NUNCHUK](assets/notext/31.webp)


เมื่อการทำธุรกรรมของผู้ส่งถูกกระจายออกไป คุณจะเห็นมันปรากฏบน wallet ของคุณ


![TAPSIGNER NUNCHUK](assets/notext/32.webp)


คลิกที่ "*ดูเหรียญ*"


![TAPSIGNER NUNCHUK](assets/notext/33.webp)


เลือก UTXO ใหม่ของคุณ


![TAPSIGNER NUNCHUK](assets/notext/34.webp)


คลิกที่ "*+*" ถัดจาก "*Tags*" เพื่อเพิ่มป้ายกำกับให้กับ UTXO ของคุณ นี่เป็นวิธีปฏิบัติที่ดี เพราะจะช่วยให้คุณจำที่มาของเหรียญของคุณและเพิ่มความเป็นส่วนตัวสำหรับการใช้จ่ายในอนาคต

## วิธีการกู้คืน wallet ในกรณีที่สูญเสีย Tapsigner?


หากคุณทำ Tapsigner หาย คุณสามารถกู้คืน wallet ของคุณได้โดยใช้รหัสที่จดไว้ด้านหลังบัตร ดังนั้นจึงเป็นสิ่งสำคัญที่จะต้องบันทึกรหัสนี้แยกจาก Tapsigner เพราะหากบัตรสูญหาย การเข้าถึงรหัสนี้ก็จะสูญหายไปด้วย คุณจะต้องมีการสำรองข้อมูลที่เข้ารหัสของ wallet ด้วย


สำหรับการกู้คืน เราจะใช้แอป Nunchuk แต่โปรดทราบว่านี่หมายถึงการรักษาความปลอดภัยชั่วคราวของเงินทุนของคุณใน wallet แบบร้อน หาก Tapsigner ของคุณกำลังรักษาความปลอดภัยจำนวนมาก ให้พิจารณาทำตามกระบวนการกู้คืนเดียวกันด้วย Coldcard ใหม่แทน


เปิดแอป Nunchuk และคลิกที่ปุ่ม "*+*" ถัดจากแท็บ "*Keys*"


![TAPSIGNER NUNCHUK](assets/notext/49.webp)


เลือก "*เพิ่มกุญแจ NFC*"


![TAPSIGNER NUNCHUK](assets/notext/50.webp)


เลือกตัวเลือก "*กู้คืนคีย์ TAPSIGNER จากการสำรองข้อมูล*"


![TAPSIGNER NUNCHUK](assets/notext/51.webp)


จากนั้นคุณจะถูกเปลี่ยนเส้นทางไปยังตัวสำรวจไฟล์ของอุปกรณ์ของคุณ ค้นหาและเลือกไฟล์สำรองข้อมูลที่เข้ารหัสของ wallet ของคุณ โดยปกติชื่อของไฟล์นี้จะเริ่มต้นด้วย `backup...`.


![TAPSIGNER NUNCHUK](assets/notext/52.webp)


ป้อนรหัสผ่านที่ถอดรหัสไฟล์สำรอง รหัสผ่านนี้ตรงกับรหัสที่จดไว้ในตอนแรกที่ด้านหลังของ Tapsigner ของคุณ


![TAPSIGNER NUNCHUK](assets/notext/53.webp)

จากนั้นเลือกชื่อสำหรับการกู้คืนของคุณ wallet.

![TAPSIGNER NUNCHUK](assets/notext/54.webp)


คุณได้เข้าถึงบิตคอยน์ของคุณอีกครั้งแล้ว wallet ของคุณตอนนี้ถูกจัดการเป็น wallet ร้อนที่มองเห็นได้ในแท็บ "*Keys*" ของแอป Nunchuk ต่อไป คุณจำเป็นต้องสร้างชุดกุญแจเข้ารหัสใหม่ในส่วน "*Wallets*" โดยการเชื่อมโยงกุญแจนี้กับมัน ในการทำเช่นนี้ คุณสามารถทำตามขั้นตอนอีกครั้งในส่วน "*How to create a wallet on a Tapsigner?*" ของบทแนะนำนี้


![TAPSIGNER NUNCHUK](assets/notext/55.webp)


หากคุณทำ Tapsigner ของคุณหาย ฉันขอแนะนำอย่างยิ่งให้คุณโอนบิตคอยน์ของคุณไปยัง wallet อื่นที่คุณเป็นเจ้าของทันที โดยควรได้รับการป้องกันโดยฮาร์ดแวร์ wallet อันที่จริง Tapsigner ที่คุณทำหายอาจตกไปอยู่ในมือของคนผิด ดังนั้นจึงเป็นเรื่องสำคัญที่จะต้องล้างข้อมูล wallet ที่คุณเพิ่งกู้คืนและหยุดใช้งาน


ขอแสดงความยินดี ตอนนี้คุณพร้อมใช้งาน Tapsigner แล้ว! หากคุณพบว่าบทแนะนำนี้มีประโยชน์ ฉันจะขอบคุณมากหากคุณสามารถกดไลค์ด้านล่างได้ อย่าลังเลที่จะแบ่งปันบทความนี้บนเครือข่ายสังคมของคุณ ขอบคุณมาก!
