## Introducción


En este videotutorial aprenderás:


- Descarga y flasheo del firmware de Krux
- Generar una nueva clave privada y exportarla mediante código QR
- Emparejamiento con Blue Wallet en el móvil para uso con una sola contraseña
- Configuración de una Multisig Wallet con Sparrow wallet
- Explorar las funciones adicionales del dispositivo de firma Krux


![video](https://youtu.be/WiwmpO1k2zA)


Al final de este vídeo, estará totalmente equipado para utilizar Krux como una alternativa segura y de bricolaje para la autocustodia.
