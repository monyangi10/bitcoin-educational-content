## Introduction

In this video tutorial you’ll learn:
- Downloading and flashing the Krux firmware
- Generating a new private key and exporting it via QR code
- Pairing with Blue Wallet on mobile for single-sig use
- Setting up a multisig wallet with Sparrow Wallet
- Exploring additional features of the Krux signing device

![video](https://youtu.be/WiwmpO1k2zA)

By the end of this video, you’ll be fully equipped to use Krux as a secure, DIY alternative for self-custody.
