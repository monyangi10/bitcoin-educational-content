SeedSigner คือฮาร์ดแวร์ wallet ที่คุณประกอบเองจากฮาร์ดแวร์มาตรฐาน โดยปกติจะอยู่รอบๆ Raspberry Pi Zero wallet นี้เรียกว่า "*stateless*": ไม่เหมือนกับรุ่นอื่นๆ ส่วนใหญ่ในตลาด (Coldcard, Trezor, Ledger, ฯลฯ) มันไม่เก็บข้อมูลใดๆ ในหน่วยความจำถาวร และทำงานเฉพาะใน RAM เท่านั้น ผลที่ตามมาคือ seed ของพอร์ตโฟลิโอของคุณจะไม่ถูกเก็บไว้ใน SeedSigner ทุกครั้งที่คุณเริ่มต้นใหม่ คุณจะต้องกรอกข้อมูลเพื่อเปิดใช้งานอุปกรณ์ในการเซ็นชื่อธุรกรรมของคุณ วิธีที่พบมากที่สุดคือการบันทึก seed ของคุณเป็น QR code ซึ่งคุณจะสแกนทุกครั้งที่ใช้งาน (*SeedQR*)



อย่างไรก็ตาม วิธีการนี้มีความเสี่ยงอย่างมาก: seed ต้องสามารถเข้าถึงได้ในรูปแบบข้อความที่ชัดเจนเพื่อให้สามารถสแกนได้ ในกรณีที่เกิดการโจรกรรมหรือการบุกรุก ผู้โจมตีอาจเข้าถึงได้ง่ายและขโมยบิตคอยน์ของคุณ



ในการเอาชนะจุดอ่อนนี้ SeedSigner สามารถรวมกับ [**Seedkeeper**](https://satochip.io/product/seedkeeper/) ซึ่งเป็นสมาร์ทการ์ดที่พัฒนาโดย Satochip ซึ่งช่วยให้สามารถจัดเก็บวลีช่วยจำ (หรือความลับอื่น ๆ) ใน secure element ที่ได้รับการป้องกันด้วยรหัส PIN แอปเพล็ต Seedkeeper เป็นโอเพ่นซอร์ส และ secure element ของมันได้รับการรับรอง EAL6+ เมื่อใช้ร่วมกับ SeedSigner มันมีคุณสมบัติด้านความปลอดภัยที่น่าสนใจมาก: กุญแจของคุณยังคงได้รับการจัดการแบบออฟไลน์ทั้งหมด คุณลงนามธุรกรรมของคุณบนหน้าจอที่เชื่อถือได้ และ seed ได้รับการป้องกันทางกายภาพในสมาร์ทการ์ดที่ทนต่อการโจมตีทางกายภาพ



สิ่งที่คุณต้องใช้ในการติดตั้งให้เสร็จสมบูรณ์มีดังนี้:




- อุปกรณ์ที่จำเป็นสำหรับ SeedSigner แบบคลาสสิก: Raspberry Pi Zero, หน้าจอ Waveshare ขนาด 1.3 นิ้ว, กล้องที่เข้ากันได้ และการ์ด microSD (คุณจะพบรายละเอียดเพิ่มเติมในบทแนะนำ SeedSigner ด้านล่าง);
- ชุดขยาย SeedSigner มีจำหน่าย [ในร้านค้าอย่างเป็นทางการของ Satochip](https://satochip.io/product/seedsigner-extension-kit/) ซึ่งช่วยให้คุณอ่านและเขียนไปยังสมาร์ทการ์ดได้โดยตรงจาก SeedSigner ของคุณ อีกทางเลือกหนึ่งคือการใช้เครื่องอ่านสมาร์ทการ์ดภายนอก ซึ่งสามารถเชื่อมต่อด้วยสายเคเบิลไปยังพอร์ต Micro-USB บน Raspberry Pi อย่างไรก็ตาม ฉันยังไม่ได้ทดสอบวิธีการนี้ด้วยตัวเอง;
- Seedkeeper หรืออีกทางเลือกหนึ่งคือสมาร์ทการ์ดเปล่าที่จะติดตั้งแอปเพล็ต Seedkeeper (ชุดขยายที่ขายโดย Satochip รวมถึงสมาร์ทการ์ดเปล่าแล้ว)



![Image](assets/fr/01.webp)



บทแนะนำนี้ครอบคลุมสองสถานการณ์:




- หากคุณมีพอร์ตโฟลิโอ Bitcoin ที่จัดการผ่าน SeedSigner ของคุณอยู่แล้ว เพียงติดตั้งเฟิร์มแวร์ใหม่ จากนั้นคุณสามารถใช้ wallet ที่มีอยู่ของคุณต่อไปได้ โดยครั้งนี้ใช้ Seedkeeper เพื่อเพิ่มความปลอดภัย
- หากคุณยังไม่มี Bitcoin wallet ที่เชื่อมโยงกับ SeedSigner ของคุณ คุณจะต้องทำตามขั้นตอน **5** และ **6** ของบทแนะนำที่กล่าวถึงด้านล่าง ส่วนเหล่านี้อธิบายวิธีการ generate วลีช่วยจำกับ SeedSigner บันทึกผ่าน *SeedQR* และจากนั้นเชื่อมต่อ wallet นี้กับ Sparrow Wallet เพื่อจัดการ ฉันจะไม่อธิบายขั้นตอนเหล่านี้ที่นี่และ **ฉันสมมติว่าคุณมี Bitcoin wallet ที่ใช้งานได้แล้ว ซึ่งกำหนดค่าด้วย Sparrow และ SeedSigner ของคุณ**



https://planb.academy/tutorials/wallet/hardware/seedsigner-2b274bff-6fc8-407a-92d7-f6ec4d1fadfb

## 1. ติดตั้งเฟิร์มแวร์



ในการใช้ SeedSigner ของคุณร่วมกับ Seedkeeper คุณจำเป็นต้องติดตั้งเฟิร์มแวร์ทางเลือกที่แตกต่างจากเฟิร์มแวร์ดั้งเดิมของ SeedSigner เพื่อรองรับการอ่านสมาร์ทการ์ด สำหรับสิ่งนี้ [ฉันแนะนำให้ใช้ fork จาก "*3rdIteration*"](https://github.com/3rdIteration/seedsigner) ดาวน์โหลด [เวอร์ชันล่าสุดของอิมเมจ](https://github.com/3rdIteration/seedsigner/releases) (`.zip`) ที่ตรงกับรุ่น Raspberry Pi ที่คุณใช้งานอยู่



![Image](assets/fr/02.webp)



หากคุณยังไม่มี ให้ดาวน์โหลดซอฟต์แวร์ [Balena Etcher] (https://etcher.balena.io/) จากนั้นดำเนินการดังนี้:




- ใส่การ์ด microSD ลงในคอมพิวเตอร์ของคุณ;
- เปิดตัว Etcher ;
- เลือกไฟล์ `.zip` ที่คุณเพิ่งดาวน์โหลด;
- เลือก microSD card เป็นเป้าหมาย;
- คลิกที่ `Flash!`.



![Image](assets/fr/03.webp)



รอจนกว่ากระบวนการจะเสร็จสิ้น: microSD ของคุณพร้อมใช้งานแล้ว ตอนนี้คุณสามารถดำเนินการประกอบอุปกรณ์ของคุณต่อได้



สำหรับรายละเอียดเพิ่มเติมเกี่ยวกับการติดตั้งเฟิร์มแวร์และการตรวจสอบซอฟต์แวร์ (ขั้นตอนที่ฉันแนะนำให้คุณทำอย่างยิ่ง) ดูบทแนะนำต่อไปนี้:



https://planb.academy/tutorials/wallet/hardware/seedsigner-2b274bff-6fc8-407a-92d7-f6ec4d1fadfb

![video](https://youtu.be/jqE8HDMCImA)



เริ่มต้นด้วยการติดตั้งกล้องบน Raspberry Pi Zero โดยค่อยๆ ใส่กล้องเข้าไปในพินของกล้องและล็อคด้วยแถบสีดำ จากนั้นวาง Pi ที่ด้านล่างของเคส โดยตรวจสอบให้แน่ใจว่าพอร์ตต่างๆ ตรงกับช่องเปิดที่สอดคล้องกัน



![Image](assets/fr/04.webp)



จากนั้นเชื่อมต่อเครื่องอ่านสมาร์ทการ์ดเข้ากับขา GPIO ของ Raspberry Pi Zero



![Image](assets/fr/05.webp)



เลื่อนฝาครอบพลาสติกไปที่เครื่องอ่านสมาร์ทการ์ดจนกว่าจะอยู่ในตำแหน่งที่ถูกต้อง



![Image](assets/fr/06.webp)



จากนั้นเพิ่มหน้าจอไปยังขา GPIO ของส่วนขยาย



![Image](assets/fr/07.webp)

![Image](assets/fr/08.webp)



คุณสามารถเชื่อมต่อ SeedSigner ของคุณได้ทั้งทางพอร์ต Micro-USB ของ Raspberry Pi Zero หรือทางพอร์ต USB-C ของส่วนขยาย ทั้งสองตัวเลือกใช้งานได้ รอสักครู่เพื่อเริ่มต้น จากนั้นคุณควรเห็นหน้าจอต้อนรับปรากฏขึ้น

## 4. บันทึก SeedQR ที่มีอยู่บน Seedkeeper



![video](https://youtu.be/X-vmFHU9Ec8)



ตอนนี้ที่ Seedkeeper ของคุณใช้งานได้แล้ว คุณสามารถบันทึก Bitcoin wallet mnemonic ของคุณลงบนสมาร์ทการ์ดได้ เพื่อเริ่มต้น ให้เปิด SeedSigner ของคุณตามปกติ จากนั้นสแกน *SeedQR* ของ wallet ของคุณเพื่อโหลดเข้าสู่อุปกรณ์ เมื่อ seed ถูกนำเข้าแล้ว เพียงเลือก `Done`



![Image](assets/fr/15.webp)



เมื่อโหลด seed แล้ว ให้เข้าถึงเมนู `Backup Seed`



![Image](assets/fr/16.webp)



จากนั้นใส่ Seedkeeper ของคุณลงในไดรฟ์ SeedSigner และเลือกตัวเลือก `To SeedKeeper`



![Image](assets/fr/17.webp)



SeedSigner จะขอให้คุณป้อนรหัส PIN สำหรับ Seedkeeper ของคุณ เนื่องจากนี่เป็นบัตรเปล่า จึงยังไม่มีการกำหนดรหัสใด ๆ ให้ป้อนรหัสใด ๆ เพื่อข้ามขั้นตอนนี้ จากนั้นยืนยัน



![Image](assets/fr/18.webp)



SeedSigner ตรวจพบว่า Seedkeeper ยังไม่ได้รับการเริ่มต้น (เช่น ยังไม่ได้ตั้งรหัสผ่าน) คลิก `I Understand` เพื่อดำเนินการต่อ



![Image](assets/fr/19.webp)



ตอนนี้เลือก PIN โค้ดใหม่สำหรับ Seedkeeper ของคุณ ระหว่าง 4 ถึง 16 ตัวอักษร เพื่อความปลอดภัยเพิ่มเติม เลือกรหัสที่ยาวและสุ่ม: นี่คืออุปสรรคเดียวที่ปกป้องการเข้าถึงทางกายภาพไปยังวลีช่วยจำของคุณ



อย่าลืมบันทึก PIN นี้ทันทีที่สร้างขึ้น ไม่ว่าจะในโปรแกรมจัดการรหัสผ่านที่เชื่อถือได้ หรือบนสื่อทางกายภาพแยกต่างหากขึ้นอยู่กับกลยุทธ์ของคุณ ในกรณีหลังนี้ ต้องแน่ใจว่าไม่เก็บสื่อที่มี PIN ไว้ในที่เดียวกับ Seedkeeper ของคุณ มิฉะนั้นการป้องกันจะไม่มีประสิทธิภาพ สิ่งสำคัญคือต้องมีสำเนาสำรอง: **หากไม่มี PIN นี้ คุณจะไม่สามารถเข้าถึง seed ของคุณได้ และบิตคอยน์ของคุณจะสูญหาย**.



![Image](assets/fr/20.webp)



จากนั้นคุณสามารถกำหนด `Label` ที่เชื่อมโยงกับวลีช่วยจำของคุณได้ ป้ายกำกับนี้มีประโยชน์หากคุณจัดเก็บความลับหลายรายการบน Seedkeeper เพื่อให้คุณสามารถระบุได้ง่าย



![Image](assets/fr/21.webp)



วลีช่วยจำของคุณถูกบันทึกลงบนสมาร์ทการ์ดแล้ว



![Image](assets/fr/22.webp)



ในแง่ของกลยุทธ์ด้านความปลอดภัย มีหลายวิธีที่เป็นไปได้ ขึ้นอยู่กับความต้องการและระดับความเสี่ยงที่คุณเผชิญ โดยส่วนตัวแล้ว ฉันแนะนำให้คุณเก็บสำเนา seed ไว้อย่างน้อย 2 ชุด:




- นี่เป็นครั้งแรกสำหรับสมาร์ทการ์ด ซึ่งคุณสามารถเก็บไว้ให้เข้าถึงได้ง่ายสำหรับการดำเนินงานในชีวิตประจำวัน เช่น การยืนยันที่อยู่หรือการลงนามธุรกรรม วิธีนี้มีความเป็นไปได้ในทางปฏิบัติ (ดังที่เราจะเห็นในส่วนที่ 5) และยังคงปลอดภัยด้วยการป้องกันที่เสนอโดยรหัส PIN ดังนั้นคุณสามารถเก็บไว้ให้เข้าถึงได้โดยไม่มีความเสี่ยงใหญ่หลวง;
- สำเนาที่สองของวลีช่วยจำที่ไม่ได้เข้ารหัสของคุณ ซึ่งทำหน้าที่เป็นการสำรองข้อมูลขั้นสูงสุดของพอร์ตโฟลิโอของคุณ ใช้เฉพาะในกรณีที่สูญหายหรือถูกขโมย Seedkeeper เนื่องจากเวอร์ชันนี้ไม่ได้เข้ารหัส จึงต้องเก็บไว้ในสถานที่ที่แยกต่างหากและปลอดภัยยิ่งขึ้น เพื่อป้องกันการถูกโจมตีพร้อมกันของการสำรองข้อมูลทั้ง 2 ชุด



ขึ้นอยู่กับกลยุทธ์การป้องกันและโปรไฟล์ความเสี่ยงของคุณ คุณยังสามารถทำซ้ำ seed บน Seedkeepers หลายตัวที่แตกต่างกัน หรือสร้างสำเนาทางกายภาพหลายชุดของการจดจำ เพื่อเรียนรู้เพิ่มเติมเกี่ยวกับแนวทางปฏิบัติเหล่านี้ โปรดดูบทแนะนำต่อไปนี้:

## 6. การใช้ Seedkeeper กับ passphrase BIP39



คุณกำลังใช้ passphrase เพื่อปกป้องพอร์ตโฟลิโอ Bitcoin ของคุณหรือไม่? คุณยังสามารถลงทะเบียนมันใน Seedkeeper ของคุณ ควบคู่ไปกับ seed ของคุณได้อีกด้วย โซลูชันนี้จะช่วยให้คุณสามารถโหลด wallet ของคุณลงบน SeedSigner ได้อย่างรวดเร็ว โดยไม่ต้องป้อน passphrase ด้วยตนเองบนแป้นพิมพ์ขนาดเล็กทุกครั้งที่คุณใช้งาน



ฉันพบว่าวิธีนี้น่าสนใจเป็นพิเศษ เนื่องจากช่วยให้คุณได้รับประโยชน์จากข้อดีด้านความปลอดภัยของ passphrase ในขณะที่กำจัดข้อจำกัดที่เกี่ยวข้องกับการใช้งานในชีวิตประจำวัน นี่คือตัวอย่างของการกำหนดค่าที่ฉันจะแนะนำ:




- เก็บ seed และ passphrase ของคุณไว้ใน Seedkeeper โดยป้องกันด้วยรหัส PIN ที่แข็งแกร่ง (สิ่งนี้สำคัญ) การสำรองข้อมูลนี้จะช่วยให้คุณสามารถใช้ wallet ของคุณได้อย่างง่ายดายในชีวิตประจำวัน หากคุณต้องการ คุณสามารถทำซ้ำข้อมูลนี้ใน Seedkeeper ที่สองได้;
- นอกจากนี้ ให้เก็บสำเนาที่ชัดเจนของคำช่วยจำและ passphrase ของคุณไว้บนกระดาษหรือโลหะ นี่คือการสำรองข้อมูลครั้งสุดท้ายของคุณหากคุณสูญเสีย Seedkeeper หรือ PIN ของมัน อย่าลืมเก็บสำเนาเหล่านี้ไว้ในสถานที่ต่างๆ กัน เพื่อไม่ให้ถูกโจมตีพร้อมกันได้



ในรูปแบบการตั้งค่านี้ หากมีใครได้ครอบครองข้อความช่วยจำแบบไม่เข้ารหัสของคุณเพียงอย่างเดียว พวกเขาจะไม่สามารถขโมยอะไรได้หากไม่รู้จัก passphrase (แน่นอนว่าต้องมั่นใจว่ามันแข็งแกร่งพอที่จะต้านทานการโจมตีแบบ brute-force) ในทางกลับกัน หากมีใครค้นพบ passphrase ของคุณในรูปแบบข้อความธรรมดา มันจะยังคงไม่สามารถใช้งานได้หากไม่มีวลีช่วยจำที่สอดคล้องกัน



สุดท้ายนี้ หากมีใครสามารถเข้าถึง Seedkeeper ของคุณที่มี seed และ passphrase อยู่ พวกเขาจะไม่สามารถดึงข้อมูลใดๆ ได้หากไม่ทราบรหัส PIN ซึ่งต่างจาก passphrase รหัสนี้ไม่สามารถถูกโจมตีด้วยวิธี brute-force ได้ เนื่องจากสมาร์ทการ์ดจะล็อคตัวเองโดยอัตโนมัติหลังจากพยายามใส่รหัสผิด 5 ครั้ง



ความปลอดภัยของการกำหนดค่านี้จึงขึ้นอยู่กับ 2 ประเด็นสำคัญ:




- **passphrase strong**: มันต้องยาว สุ่ม และประกอบด้วยตัวอักษรหลากหลาย ความซับซ้อนไม่ใช่ปัญหาสำหรับคุณ เพราะคุณจะต้องป้อนมันเพียงครั้งเดียวบนแป้นพิมพ์ระหว่างการเริ่มต้น; หลังจากนั้น มันจะถูกส่งโดย Seedkeeper ;
- รหัส PIN **ที่แข็งแกร่ง** สำหรับ Seedkeeper: สุ่มและประกอบด้วย 16 อักขระ



ในการตั้งค่าการติดตั้งนี้ เริ่มต้นด้วยการโหลด passphrase ของคุณเข้าไปใน SeedSigner ตามวิธีปกติ คุณสามารถทำตามขั้นตอนที่อธิบายไว้ในบทแนะนำนี้:



https://planb.academy/tutorials/wallet/backup/seedsigner-passphrase-7a61f64d-aa03-4bcf-8308-00c89a74cffe

เมื่อพอร์ตโฟลิโอของคุณที่มี passphrase ถูกโหลดเข้าสู่ SeedSigner อย่างถูกต้องแล้ว ให้เปิดเมนู `Seeds` และเลือกพิมพ์เขียวที่ตรงกับพอร์ตโฟลิโอนี้ โปรดทราบว่าพิมพ์เขียวนี้แตกต่างจากพอร์ตโฟลิโอที่ไม่มี passphrase

ป้อนรหัส PIN ของคุณเพื่อปลดล็อก Seedkeeper จากนั้นกำหนดป้ายกำกับให้กับความลับนี้ คุณสามารถใช้ลายนิ้วมือเป็นป้ายกำกับเพื่อรักษารูปแบบของการปฏิเสธที่น่าเชื่อถือ หรือระบุอย่างชัดเจนว่า `Passphrase Wallet` เป็นต้น



![Image](assets/fr/30.webp)



พอร์ตโฟลิโอ passphrase ของคุณได้ลงทะเบียนใน Seedkeeper แล้ว

## 7. ตัวเลือกเพิ่มเติม



ในเมนู `Tools > Smartcard Tools` คุณจะพบตัวเลือกหลายอย่างสำหรับการจัดการ Seedkeeper ของคุณ:





- ในเมนู `Common Tools` คุณสามารถ :
 - ตรวจสอบความถูกต้องของบัตร;
 - เปลี่ยนรหัส PIN ;
 - เปลี่ยนป้ายกำกับที่เกี่ยวข้องกับความลับของคุณ ;
 - ปิดใช้งานฟังก์ชัน NFC (แนะนำหากใช้เฉพาะเครื่องอ่านชิป) ;
 - ทำการรีเซ็ตเป็นค่าเริ่มต้นจากโรงงาน





- ในเมนู `SeedKeeper Functions` คุณสามารถ :
 - ปรึกษารายการความลับที่ลงทะเบียน ;
 - บันทึกความลับใหม่ ;
 - ลบความลับที่มีอยู่ ;
 - บันทึกหรือโหลดตัวระบุของคุณ (ฟังก์ชันที่มีประโยชน์สำหรับพอร์ตโฟลิโอ multi-sig)





- ในเมนู `DIY Tools` คุณสามารถ :
 - กำลังคอมไพล์แอปเพล็ต Seedkeeper ;
 - ติดตั้งแอปเพล็ตบนการ์ดเปล่า ;
 - ลบแอปเพล็ต Seedkeeper เพื่อรีเซ็ตและทำให้ว่างเปล่าอีกครั้ง



ตอนนี้คุณรู้วิธีใช้ Seedkeeper เพื่อสำรองข้อมูลพอร์ตโฟลิโอของคุณอย่างปลอดภัยร่วมกับ SeedSigner แล้ว
