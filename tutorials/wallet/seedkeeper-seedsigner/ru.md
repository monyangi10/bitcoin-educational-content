SeedSigner - это аппаратный wallet, который вы собираете сами из стандартного оборудования, обычно на базе Raspberry Pi Zero. Этот wallet называется "*stateless*": в отличие от большинства других моделей на рынке (Coldcard, Trezor, Ledger и т.д.), он не хранит никаких данных в постоянной памяти, а работает только в режиме реального времени из оперативной памяти. В результате seed вашего портфеля никогда не сохраняется в SeedSigner. При каждом перезапуске вам нужно будет заполнить его, чтобы устройство могло подписывать ваши сделки. Самый распространенный способ - сохранить seed в виде QR-кода, который вы затем сканируете при каждом использовании (*SeedQR*).



Однако такой подход сопряжен со значительным риском: seed должен оставаться доступным в виде открытого текста, чтобы его можно было просканировать. В случае кражи или вторжения злоумышленник может легко завладеть им и украсть ваши биткоины.



Чтобы преодолеть эту слабость, SeedSigner можно объединить со смарт-картой [**Seedkeeper**](https://satochip.io/product/seedkeeper/), разработанной компанией Satochip. Это позволяет хранить мнемонические фразы (или другие секреты) в secure element, защищенной PIN-кодом. Апплет Seedkeeper имеет открытый исходный код, а его secure element сертифицирован по стандарту EAL6+. Используемый в сочетании с SeedSigner, он предлагает очень интересную функцию безопасности: ваши ключи остаются полностью автономными, вы подписываете свои транзакции на доверенном экране, а seed физически защищен в смарт-карте, устойчивой к физическим атакам.



Для завершения установки вам понадобятся следующие предметы:




- Обычное оборудование, необходимое для классического SeedSigner: Raspberry Pi Zero, 1,3-дюймовый экран Waveshare, совместимая камера и карта microSD (более подробную информацию вы найдете в руководстве по SeedSigner ниже);
- Набор расширений SeedSigner, доступный [в официальном магазине Satochip] (https://satochip.io/product/seedsigner-extension-kit/), который позволяет читать и записывать данные на смарт-карту непосредственно с SeedSigner. Другой вариант - использовать внешний считыватель смарт-карт, который можно подключить кабелем к порту Micro-USB на Raspberry Pi. Однако я сам не тестировал это решение;
- Seedkeeper или пустая смарт-карта, на которую можно установить апплет Seedkeeper (комплект расширений, продаваемый Satochip, уже включает пустую смарт-карту).



![Image](assets/fr/01.webp)



В этом уроке рассматриваются два сценария:




- Если у вас уже есть портфель Bitcoin, управляемый через SeedSigner, просто установите новую прошивку. После этого вы можете продолжать использовать имеющийся wallet, на этот раз используя Seedkeeper для дополнительной безопасности.
- Если у вас еще нет Bitcoin wallet, связанного с вашим SeedSigner, вам нужно выполнить шаги **5** и **6** из приведенного ниже руководства. В этих разделах объясняется, как набрать мнемоническую фразу generate с помощью SeedSigner, сохранить ее через *SeedQR*, а затем подключить этот wallet к Sparrow Wallet, чтобы управлять им. Я не буду углубляться в эти процедуры, и **я предполагаю, что у вас уже есть работающий Bitcoin wallet, настроенный с Sparrow и вашим SeedSigner**.



https://planb.academy/tutorials/wallet/hardware/seedsigner-2b274bff-6fc8-407a-92d7-f6ec4d1fadfb

## 1. Установите прошивку



Чтобы использовать SeedSigner с Seedkeeper, необходимо установить альтернативную прошивку, отличную от прошивки оригинального SeedSigner, для поддержки чтения смарт-карт. Для этого [я рекомендую использовать fork от "*3rdIteration*"](https://github.com/3rdIteration/seedsigner). Скачайте [последнюю версию образа](https://github.com/3rdIteration/seedsigner/releases) (`.zip`), соответствующую используемой модели Raspberry Pi.



![Image](assets/fr/02.webp)



Если у вас его еще нет, загрузите программу [Balena Etcher] (https://etcher.balena.io/), а затем выполните следующие действия:




- Вставьте карту microSD в компьютер;
- Запуск этчера ;
- Выберите файл `.zip`, который вы только что загрузили;
- Выберите карту microSD в качестве цели;
- Нажмите на кнопку `Flash!



![Image](assets/fr/03.webp)



Дождитесь завершения процесса: теперь ваша microSD готова к использованию. Теперь можно переходить к сборке устройства.



Более подробную информацию об установке прошивки и проверке программного обеспечения (что я настоятельно рекомендую вам сделать) вы найдете в следующем руководстве:



https://planb.academy/tutorials/wallet/hardware/seedsigner-2b274bff-6fc8-407a-92d7-f6ec4d1fadfb

## 2. Сборка считывателя смарт-карт



![video](https://youtu.be/jqE8HDMCImA)



Начните с установки камеры на Raspberry Pi Zero, аккуратно вставив ее в штырь камеры и зафиксировав черным язычком. Затем положите Pi на дно корпуса, следя за тем, чтобы порты совпадали с соответствующими отверстиями.



![Image](assets/fr/04.webp)



Затем подключите считыватель смарт-карт к контактам GPIO Raspberry Pi Zero.



![Image](assets/fr/05.webp)



Сдвиньте пластиковую крышку на считыватель смарт-карт, чтобы она заняла правильное положение.



![Image](assets/fr/06.webp)



Затем добавьте экран к контактам GPIO расширения.



![Image](assets/fr/07.webp)



Наконец, вставьте карту microSD с прошивкой в боковой порт Raspberry Pi Zero.



![Image](assets/fr/08.webp)



Теперь вы можете подключить SeedSigner либо через порт Micro-USB Raspberry Pi Zero, либо через порт USB-C удлинителя. Оба варианта работают. Подождите несколько секунд для запуска, затем вы увидите экран приветствия.



![Image](assets/fr/09.webp)



Для получения более подробной информации о начальной настройке SeedSigner я рекомендую следующее руководство:



https://planb.academy/tutorials/wallet/hardware/seedsigner-2b274bff-6fc8-407a-92d7-f6ec4d1fadfb

## 3. Прошивка смарт-карты с помощью апплета Seedkeeper (необязательно)



![video](https://youtu.be/NF4HemyEcOY)



Если у вас уже есть Seedkeeper, вы можете пропустить этот шаг и сразу перейти к шагу 4. В этом разделе мы рассмотрим, как установить апплет Seedkeeper на пустую смарт-карту (метод "сделай сам").



Чтобы начать работу, откройте меню `Инструменты > Инструменты для смарт-карт` на вашем SeedSigner.



![Image](assets/fr/10.webp)



Затем выберите `DIY Tools > Install Applet`.



![Image](assets/fr/11.webp)



Вставьте смарт-карту в считывающее устройство SeedSigner, чипом вниз, затем выберите апплет `SeedKeeper`.



![Image](assets/fr/12.webp)



Пожалуйста, будьте терпеливы во время установки: процесс может занять несколько десятков секунд.



![Image](assets/fr/13.webp)



После того как апплет будет успешно установлен, можно переходить к шагу 4.



![Image](assets/fr/14.webp)



## 4. Сохраните существующий SeedQR в Seedkeeper



![video](https://youtu.be/X-vmFHU9Ec8)



Теперь, когда ваш Seedkeeper работает, вы можете сохранить мнемонику Bitcoin wallet на смарт-карте. Для начала включите SeedSigner как обычно, затем отсканируйте *SeedQR* вашего wallet, чтобы загрузить его в устройство. После того как seed будет импортирован, просто выберите `Done`.



![Image](assets/fr/15.webp)



Когда seed будет загружен, откройте меню `Backup Seed`.



![Image](assets/fr/16.webp)



Затем вставьте Seedkeeper в дисковод SeedSigner и выберите опцию `To SeedKeeper`.



![Image](assets/fr/17.webp)



Затем SeedSigner попросит вас ввести PIN-код для вашего Seedkeeper. Поскольку это пустая карта, код еще не определен. Введите любой код, чтобы пропустить этот шаг, а затем подтвердите.



![Image](assets/fr/18.webp)



SeedSigner обнаруживает, что Seedkeeper еще не инициализирован (т.е. пароль не установлен). Нажмите `Я понял`, чтобы продолжить.



![Image](assets/fr/19.webp)



Теперь выберите новый PIN-код вашего Seedkeeper - от 4 до 16 символов. Для большей безопасности выберите длинный случайный код: это единственный барьер, защищающий физический доступ к вашей мнемонической фразе.



Не забудьте сохранить этот PIN-код сразу после его создания либо в надежном менеджере паролей, либо на отдельном физическом носителе, в зависимости от вашей стратегии. В последнем случае не храните носитель с PIN-кодом в том же месте, что и Seedkeeper, иначе защита станет неэффективной. Важно иметь резервную копию: **Без этого PIN-кода вы не сможете получить доступ к своему seed, и ваши биткоины будут потеряны**.



![Image](assets/fr/20.webp)



Затем вы можете задать `ярлык`, связанный с вашей мнемонической фразой. Эта метка будет полезна, если вы храните в Seedkeeper несколько секретов, чтобы их можно было легко идентифицировать.



![Image](assets/fr/21.webp)



Теперь ваша мнемоническая фраза сохранена на смарт-карте.



![Image](assets/fr/22.webp)



Что касается стратегии безопасности, то здесь возможно несколько подходов, в зависимости от ваших потребностей и уровня риска. Лично я рекомендую хранить как минимум 2 копии seed:




- Это первый случай для смарт-карт, которые можно держать под рукой для повседневных операций, таких как проверка адреса или подписание транзакций. Этот метод практичен (как мы увидим в части 5) и остается безопасным благодаря защите, обеспечиваемой PIN-кодом, так что вы можете держать ее под рукой без особого риска;
- Вторая копия вашей незашифрованной мнемонической фразы, служащая конечной резервной копией вашего портфеля, которая будет использоваться только в случае потери или кражи Seedkeeper. Поскольку эта версия не зашифрована, она должна храниться в отдельном, более надежном месте, чтобы предотвратить одновременную компрометацию двух резервных копий.



В зависимости от стратегии защиты и профиля риска вы можете продублировать seed на нескольких разных Seedkeepers или создать несколько физических копий мнемоники. Чтобы узнать больше об этих способах, ознакомьтесь со следующим учебным пособием:



https://planb.academy/tutorials/wallet/backup/backup-mnemonic-22c0ddfa-fb9f-4e3a-96f9-46e2a7954270


## 5. Загрузка seed из Seedkeeper



![video](https://youtu.be/ms0Iq_IyaoE)



Теперь вы можете использовать Seedkeeper для загрузки мнемонической фразы в SeedSigner при запуске и, таким образом, подписывать транзакции Bitcoin. Для начала включите SeedSigner, подключив его к сети, затем откройте меню `Seeds`.



![Image](assets/fr/23.webp)



Затем выберите опцию `From SeedKeeper`.



![Image](assets/fr/24.webp)



Вставьте Seedkeeper в считыватель смарт-карт, затем введите PIN-код, чтобы разблокировать его. Подтвердите ввод, нажав правую нижнюю кнопку подтверждения `KEY3`.



![Image](assets/fr/25.webp)



Seedkeeper может содержать несколько секретов, поэтому SeedSigner предложит вам выбрать тот, который вы хотите загрузить. Отображаемая метка соответствует имени, которое вы задали на шаге 4. Если, как в моем случае, вы зарегистрировали только один seed, будет доступен только один вариант.



![Image](assets/fr/26.webp)



Теперь ваш seed загружен. Проверьте, что это правильный wallet, сравнив отпечаток пальца, отображаемый на экране, с отпечатком, указанным в настройках Sparrow Wallet. Этот отпечаток также был указан при первом создании wallet.



Если вы используете passphrase, вы можете применить его на этом этапе (см. часть 6 этого руководства). В противном случае просто нажмите на `Done`.



![Image](assets/fr/27.webp)



После этого вы можете использовать wallet как обычно: проверять адреса доставки и подписывать транзакции, как и с классическим SeedSigner. Подробнее о том, как им пользоваться, читайте в специальном руководстве:



https://planb.academy/tutorials/wallet/hardware/seedsigner-2b274bff-6fc8-407a-92d7-f6ec4d1fadfb

## 6. Использование Seedkeeper с passphrase BIP39



Вы используете passphrase для защиты портфеля Bitcoin? Вы можете зарегистрировать его в Seedkeeper вместе с seed. Это решение позволит вам быстро загрузить wallet в SeedSigner, без необходимости вручную вводить passphrase на маленькой клавиатуре каждый раз, когда вы его используете.



Я нахожу этот метод особенно интересным, поскольку он позволяет воспользоваться преимуществами безопасности passphrase, устраняя при этом ограничения, связанные с его повседневным использованием. Вот пример конфигурации, которую я рекомендую:




- Храните seed и passphrase в Seedkeeper, защищенном надежным PIN-кодом (это очень важно). Эта резервная копия позволит вам легко использовать wallet на ежедневной основе. При желании вы можете продублировать эту информацию на втором Seedkeeper;
- Также храните четкую копию вашей мнемоники и passphrase на бумаге или металле. Это ваш запасной вариант на случай потери Seedkeeper или его PIN-кода. Храните эти копии в разных местах, чтобы их нельзя было скомпрометировать одновременно.



При такой конфигурации, если кто-то завладеет только вашей мнемоникой в открытом тексте, он не сможет ничего украсть, не зная passphrase (при условии, конечно, что она достаточно сильна, чтобы противостоять атаке методом перебора). И наоборот, если кто-то обнаружит ваш passphrase в виде обычного текста, он останется непригодным для использования без соответствующей мнемонической фразы.



Наконец, если кому-то удастся получить физический доступ к вашему Seedkeeper, содержащему seed и passphrase, он не сможет ничего извлечь, не зная PIN-кода. В отличие от passphrase, этот код не может быть взломан, так как смарт-карта автоматически блокируется после 5 неверных попыток.



Таким образом, безопасность этой конфигурации основана на двух основных моментах:




- Сильный **passphrase**: он должен быть длинным, случайным и содержать большое количество символов. Его сложность не является проблемой для вас, так как вам придется ввести его только один раз на клавиатуре во время инициализации; после этого он будет передан Seedkeeper;
- **сильный PIN-код** для Seedkeeper: также случайный и состоит из 16 символов.



Чтобы настроить эту установку, начните с загрузки passphrase в SeedSigner обычным способом. Вы можете следовать процедуре, подробно описанной в этом руководстве:



https://planb.academy/tutorials/wallet/backup/seedsigner-passphrase-7a61f64d-aa03-4bcf-8308-00c89a74cffe

После того как портфель с passphrase будет правильно загружен в SeedSigner, откройте меню `Seeds` и выберите отпечаток, соответствующий этому портфелю. Обратите внимание, что этот след отличается от следа портфеля без passphrase.



![Image](assets/fr/28.webp)



Затем нажмите на `Backup Seed`, вставьте Seedkeeper в дисковод и выберите `To SeedKeeper`.



![Image](assets/fr/29.webp)



Введите PIN-код, чтобы разблокировать Seedkeeper, а затем присвойте этому секрету метку. Вы можете оставить отпечаток пальца в качестве метки для сохранения правдоподобности, или, например, явно указать `Passphrase Wallet`.



![Image](assets/fr/30.webp)



Ваш портфель passphrase теперь зарегистрирован на Seedkeeper.



![Image](assets/fr/31.webp)



При следующем запуске просто вставьте Seedkeeper в дисковод, а затем перейдите в меню `Seeds > From SeedKeeper`.



![Image](assets/fr/32.webp)



Введите PIN-код, чтобы разблокировать смарт-карту, а затем выберите wallet, соответствующий вашему passphrase.



![Image](assets/fr/33.webp)



Проверьте отпечаток passphrase и wallet, затем подтвердите.



![Image](assets/fr/34.webp)



Теперь вы можете получить доступ к своему портфелю с помощью passphrase и подписывать свои сделки так же, как вы обычно делаете это на SeedSigner.



## 7. Дополнительные опции



В меню `Инструменты > Smartcard Tools` вы найдете несколько опций для управления Seedkeeper:





- В меню `Общие инструменты` вы можете :
 - Проверьте подлинность карты;
 - Изменение PIN-кода ;
 - Измените ярлыки, связанные с вашими секретами;
 - Отключите функцию NFC (рекомендуется использовать только считыватель чипов) ;
 - Выполните сброс к заводским настройкам.





- В меню `Функции SeedKeeper` вы можете :
 - Ознакомьтесь со списком зарегистрированных секретов;
 - Сохраните новый секрет ;
 - Удаление существующего секрета ;
 - Сохраните или загрузите свои дескрипторы (полезная функция для портфолио multi-sig).





- В меню `DIY Tools` вы можете :
 - Компиляция апплета Seedkeeper ;
 - Установите апплет на чистую карту памяти;
 - Удалите апплет Seedkeeper, чтобы сбросить его и снова сделать пустым.



Теперь вы знаете, как использовать Seedkeeper для безопасного резервного копирования вашего портфеля в сочетании с SeedSigner.
