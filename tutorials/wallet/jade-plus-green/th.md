## Jade Plus รุ่นความปลอดภัย


Jade Plus ใช้โมเดลความปลอดภัยที่อิงจาก "secure element เสมือน" ซึ่งถูกทำให้เป็นจริงโดย "oracle ที่มองไม่เห็น" ในแง่ที่เป็นรูปธรรม กลไกนี้รวม PIN ที่ผู้ใช้เลือกไว้ ความลับที่โฮสต์บน Jade และความลับที่ถือโดย oracle (เซิร์ฟเวอร์ที่ดูแลโดย Blockstream) เพื่อสร้างคีย์ AES-256 ที่กระจายอยู่ในสองหน่วยงาน ในระหว่างการเริ่มต้น การแลกเปลี่ยน ECDH จะรักษาความปลอดภัยในการสื่อสารกับ oracle และเข้ารหัสวลีการกู้คืนบนฮาร์ดแวร์ wallet ในแง่ปฏิบัติ เมื่อคุณต้องการเข้าถึง seed เพื่อเซ็นชื่อธุรกรรม คุณจำเป็นต้องเข้าถึง:




- ถึงตัวอุปกรณ์ Jade Plus เอง;
- ป้อน PIN เพื่อปลดล็อกอุปกรณ์ ;
- และถึงความลับของเทพพยากรณ์


ข้อได้เปรียบหลักของวิธีการนี้คือการไม่มีจุดล้มเหลวเพียงจุดเดียวในระดับฮาร์ดแวร์ เนื่องจากหากผู้โจมตีสามารถเข้าถึง Jade ของคุณได้ การดึงคีย์จะต้องมีการประนีประนอมทั้ง Jade และออราเคิลพร้อมกัน โมเดลนี้ยังหมายความว่า Jade Plus เป็นโอเพ่นซอร์สทั้งหมด หลีกเลี่ยงข้อจำกัดที่เกี่ยวข้องกับการใช้ส่วนประกอบที่ปลอดภัยทางกายภาพจริง เช่นที่ใช้ใน Ledger เป็นต้น


ข้อเสียของระบบนี้คือการใช้ Jade Plus ขึ้นอยู่กับออราเคิลที่ดูแลโดย Blockstream หากออราเคิลนี้ไม่สามารถเข้าถึงได้ จะไม่สามารถใช้ฮาร์ดแวร์ wallet โดยตรงกับ PIN ได้ อย่างไรก็ตาม นี่ไม่ได้หมายความว่า bitcoins ของคุณจะสูญหาย เพราะยังสามารถกู้คืนได้โดยใช้ recovery phrase ของคุณ ซึ่งคุณสามารถป้อนใน Jade Plus ในโหมด "*stateless*" เพื่อหลีกเลี่ยงการพึ่งพานี้ คุณยังสามารถกำหนดค่าและจัดการเซิร์ฟเวอร์ออราเคิลของคุณเองได้

## การตั้งรหัส PIN


คลิกที่ปุ่ม "*Continue*" เพื่อเลือก PIN code ของ Jade ของคุณ


![JADE-PLUS-GREEN](assets/fr/24.webp)

ตัวเลือก "*เปลี่ยนชื่อ*" ช่วยให้คุณปรับแต่งชื่อของ wallet ของคุณ ซึ่งมีประโยชน์อย่างยิ่งหากคุณจัดการกระเป๋าเงินหลายใบในแอปพลิเคชันเดียวกัน เมนู "*หน่วย*" ช่วยให้คุณเปลี่ยนหน่วยพื้นฐานของ wallet ของคุณได้ ตัวอย่างเช่น คุณสามารถเลือกแสดงในรูปแบบซาโตชิแทนที่จะเป็นบิตคอยน์ สุดท้าย เมนู "*พารามิเตอร์*" ให้คุณเข้าถึงตัวเลือกอื่น ๆ ได้ ที่นี่ ตัวอย่างเช่น คุณจะพบกุญแจสาธารณะที่ขยายออกและตัวบรรยาย ซึ่งมีประโยชน์หากคุณวางแผนที่จะตั้งค่า wallet แบบดูอย่างเดียวจาก Jade ของคุณ


![JADE-PLUS-GREEN](assets/fr/32.webp)


หากต้องการเชื่อมต่อกับ Jade ของคุณอีกครั้งหลังจากปิดเครื่อง ให้กดปุ่มเปิด/ปิดที่ด้านล่างของอุปกรณ์ บนแอปพลิเคชัน Green ให้เลือกอุปกรณ์ของคุณจากหน้าแรก:


![JADE-PLUS-GREEN](assets/fr/33.webp)


จากนั้นป้อนรหัส PIN บน Jade ของคุณ แล้วคุณจะเชื่อมต่ออีกครั้ง


![JADE-PLUS-GREEN](assets/fr/34.webp)


Jade ของคุณถูกปลดล็อกผ่าน "virtual secure element" ของ Blockstream (ดูในส่วนแรกของบทช่วยสอนนี้) ซึ่งต้องการการเชื่อมต่อ Bluetooth กับแอปพลิเคชัน Green หากคุณพบปัญหาในการเชื่อมต่อ Bluetooth ระหว่างการปลดล็อก ให้ลองยกเลิกการเชื่อมโยงและเชื่อมโยงอุปกรณ์ทั้งสองใหม่ หากปัญหายังคงอยู่ คุณยังสามารถปลดล็อก Jade ของคุณได้โดยเลือกตัวเลือก "*QR Scan*" และทำตามคำแนะนำที่มีอยู่ [บนเว็บไซต์ Blockstream](https://jadefw.blockstream.com/pinqr/index.html)


ก่อนที่คุณจะได้รับบิตคอยน์ครั้งแรกใน wallet ของคุณ **ฉันขอแนะนำอย่างยิ่งให้คุณทำการทดสอบการกู้คืนแบบว่างเปล่า** จดบันทึกข้อมูลอ้างอิงบางอย่าง เช่น xpub หรือที่อยู่รับครั้งแรกของคุณ จากนั้นลบ wallet ของคุณในแอป Green และบน Jade Plus ในขณะที่ยังว่างเปล่า (`Options -> Device -> Factory Reset`) จากนั้นลองกู้คืน wallet ของคุณโดยใช้การสำรองข้อมูลกระดาษของวลีช่วยจำ ตรวจสอบว่าข้อมูลคุกกี้ที่สร้างขึ้นหลังจากการกู้คืนตรงกับที่คุณเขียนไว้ในตอนแรกหรือไม่ หากตรงกัน คุณสามารถมั่นใจได้ว่าการสำรองข้อมูลกระดาษของคุณเชื่อถือได้ หากต้องการทราบข้อมูลเพิ่มเติมเกี่ยวกับวิธีการดำเนินการทดสอบการกู้คืน โปรดดูบทแนะนำอื่นนี้ :


https://planb.academy/tutorials/wallet/backup/recovery-test-5a75db51-a6a1-4338-a02a-164a8d91b895

## รับบิตคอยน์


ตอนนี้ที่ Bitcoin wallet ของคุณตั้งค่าเสร็จเรียบร้อยแล้ว คุณพร้อมที่จะรับ sats ตัวแรกของคุณแล้ว! เพียงคลิกที่ปุ่ม "*Receive*" บนแอปพลิเคชัน Green


![JADE-PLUS-GREEN](assets/fr/35.webp)


Green แสดงที่อยู่การรับ แต่ก่อนที่จะใช้งาน จำเป็นต้องตรวจสอบบน Jade เพื่อยืนยันว่ามันเป็นของ wallet ของเราอย่างแท้จริง ในการทำเช่นนี้ ให้คลิกที่ปุ่ม "*Verify on device*"


![JADE-PLUS-GREEN](assets/fr/36.webp)


ตรวจสอบที่ Jade ว่าที่อยู่ตรงกับ Green หรือไม่ จากนั้นคลิกที่ปุ่มเพื่อยืนยัน


![JADE-PLUS-GREEN](assets/fr/37.webp)


คุณสามารถแชร์ที่อยู่กับผู้ชำระเงินเพื่อรับบิตคอยน์ใน wallet ของคุณได้แล้ว เมื่อธุรกรรมถูกกระจายบนเครือข่าย มันจะแสดงใน wallet ของคุณ รอจนกว่าคุณจะได้รับการยืนยันเพียงพอเพื่อพิจารณาว่าธุรกรรมนั้นเป็นที่สิ้นสุด


![JADE-PLUS-GREEN](assets/fr/38.webp)


## ส่งบิตคอยน์


ด้วยบิตคอยน์ใน wallet ของคุณ ตอนนี้คุณสามารถส่งบิตคอยน์ได้แล้ว คลิกที่ "*Send*"


![JADE-PLUS-GREEN](assets/fr/39.webp)


ในหน้าถัดไป ให้ป้อนที่อยู่ของผู้รับ คุณสามารถป้อนด้วยตนเองหรือสแกน QR โค้ด


![JADE-PLUS-GREEN](assets/fr/40.webp)


เลือกจำนวนเงินที่ต้องการชำระ


![JADE-PLUS-GREEN](assets/fr/41.webp)


ที่ด้านล่างของหน้าจอ คุณสามารถเลือกอัตราค่าธรรมเนียมสำหรับการทำธุรกรรมนี้ได้ คุณมีทางเลือกในการปฏิบัติตามคำแนะนำของแอปพลิเคชันหรือปรับแต่งค่าธรรมเนียมของคุณเอง ยิ่งค่าธรรมเนียมสูงเมื่อเทียบกับธุรกรรมที่รอดำเนินการอื่น ๆ การทำธุรกรรมของคุณจะได้รับการดำเนินการเร็วขึ้น สำหรับข้อมูลตลาดค่าธรรมเนียม กรุณาเยี่ยมชม [Mempool.space](https://mempool.space/) ในส่วน "*Transaction Fees*"


![JADE-PLUS-GREEN](assets/fr/42.webp)


คลิกที่ "*ถัดไป*" เพื่อเข้าสู่หน้าจอสรุปธุรกรรม ตรวจสอบให้แน่ใจว่าที่อยู่ จำนวนเงิน และค่าธรรมเนียมถูกต้อง


![JADE-PLUS-GREEN](assets/fr/43.webp)


หากทุกอย่างเป็นไปด้วยดี ให้เลื่อนปุ่มสีเขียวที่ด้านล่างของหน้าจอไปทางขวาเพื่อเซ็นและกระจายธุรกรรมบนเครือข่าย Bitcoin


![JADE-PLUS-GREEN](assets/fr/44.webp)


คุณถูกขอให้ยืนยันการทำธุรกรรมบน Jade.


![JADE-PLUS-GREEN](assets/fr/45.webp)


ตรวจสอบให้แน่ใจว่าที่อยู่ของผู้รับถูกต้อง คลิกที่เครื่องหมายถูกเพื่อยืนยัน


![JADE-PLUS-GREEN](assets/fr/46.webp)


ตรวจสอบว่าจำนวนเงินที่เรียกเก็บถูกต้อง จากนั้นยืนยัน


![JADE-PLUS-GREEN](assets/fr/47.webp)


ธุรกรรมของคุณได้รับการลงนามและออกอากาศจาก Green แล้ว


![JADE-PLUS-GREEN](assets/fr/48.webp)


ขอแสดงความยินดี ตอนนี้คุณรู้วิธีตั้งค่าและใช้งาน Jade Plus กับแอปพลิเคชันมือถือ Blockstream Green ผ่านการเชื่อมต่อ Bluetooth แล้ว หากคุณพบว่าบทแนะนำนี้มีประโยชน์ ฉันจะขอบคุณมากหากคุณจะให้คะแนนนิ้วโป้งสีเขียวด้านล่าง อย่าลังเลที่จะแบ่งปันบทความนี้บนเครือข่ายสังคมของคุณ ขอบคุณที่แบ่งปัน!
