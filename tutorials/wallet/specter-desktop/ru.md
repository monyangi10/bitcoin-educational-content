Specter Desktop - это приложение с открытым исходным кодом (лицензия MIT), разрабатываемое Cryptoadvance с 2019 года, которое облегчает управление кошельками Bitcoin с помощью аппаратных кошельков (Ledger, Trezor, Coldcard, BitBox02, Passport и т. д.) и собственной инфраструктуры Bitcoin (узел Bitcoin Core или сервер Electrum). Приложение особенно эффективно в конфигурациях с несколькими подписями, позволяя вам обеспечивать безопасность больших сумм за счет распределения мощности подписи между несколькими независимыми аппаратными кошельками.



**В этом уроке вы узнаете, как:**




- Установите и настройте Specter Desktop на своем компьютере (Windows, macOS или Linux)
- Подключите Specter к серверу Electrum (в данном примере мы будем использовать Umbrel)
- Создание простого wallet с аппаратным обеспечением wallet (Coldcard)
- Получайте и отправляйте биткоины, сохраняя полный суверенитет
- Настройка мультисигнатурного wallet 2 на 3 с несколькими аппаратными кошельками
- Установка Specter на сервер Umbrel (расширенный бонус)



Все ваши транзакции будут проверяться локально через вашу собственную инфраструктуру, без передачи информации на внешние серверы, что гарантирует вашу конфиденциальность и финансовый суверенитет. Всегда проверяйте транзакции на экране оборудования wallet перед подписанием.



## Загрузка и установка



Посетите официальный сайт Specter Desktop, чтобы загрузить приложение.



![Page d'accueil Specter](assets/fr/01.webp)



На странице загрузки выберите версию, соответствующую вашей операционной системе: macOS, Windows или Linux.



![Téléchargement selon l'OS](assets/fr/02.webp)



После загрузки установите приложение в соответствии с обычными инструкциями вашей операционной системы. Для macOS перетащите значок в "Приложения". Для Windows запустите программу установки. Для Linux следуйте инструкциям пакета.



## Начальная конфигурация



При первом запуске Specter Desktop просит вас выбрать тип подключения. Вы можете подключиться к серверу Electrum или к собственному узлу Bitcoin Core.



![Choix du type de connexion](assets/fr/03.webp)



В этом примере мы будем использовать соединение с сервером Electrum, работающим на Umbrel.



Для получения дополнительной информации обратитесь к нашему руководству по Umbrel:



https://planb.academy/tutorials/node/bitcoin/umbrel-8b0e3b5b-d3cf-4a1e-8bb8-1ad2db4dd848

Этот вариант обеспечивает более быструю синхронизацию, чем Bitcoin Core. При желании вы можете выбрать "Bitcoin Core" и настроить подключение к локальному узлу. Следующие шаги остаются неизменными независимо от вашего выбора.



Выберите "Подключение Electrum", затем выберите "Введите мой собственный", чтобы настроить свой собственный сервер Electrum.



![Configuration Electrum](assets/fr/04.webp)



Введите адрес вашего сервера Electrum. В нашем случае с Umbrel, адрес будет `umbrel.local` с портом `50001`. Нажмите на кнопку "Connect", чтобы установить соединение.



После подключения появится экран приветствия с контрольным списком для начала работы. Теперь вам нужно добавить аппаратные кошельки.



![Écran d'accueil](assets/fr/05.webp)



## Добавление оборудования wallet



В левом меню нажмите "Добавить устройство", чтобы добавить оборудование wallet.



Specter Desktop поддерживает множество аппаратных кошельков: Trezor, Ledger, BitBox02, Coldcard, KeepKey, Keystone, Cobo Vault и многие другие.



Если вы хотите узнать больше, посмотрите наши руководства по аппаратным средствам wallet.



![Sélection du type de hardware wallet](assets/fr/06.webp)



Выберите оборудование для wallet. В данном примере мы используем Coldcard MK4.



Ниже вы найдете наше руководство по использованию этой фурнитуры wallet:



https://planb.academy/tutorials/wallet/hardware/coldcard-5d44dd94-423d-4e37-9a8c-3fc38b45ce59

Для Coldcard необходимо экспортировать открытые ключи с оборудования wallet либо через USB-соединение, либо на карту памяти microSD.



![Import des clés du Coldcard](assets/fr/07.webp)



Следуйте инструкциям для экспорта ключей с Coldcard. Дайте аппарату wallet имя (здесь "MK4 Tuto"). После того как ключи будут импортированы, вы можете создать wallet с одним ключом или добавить другие аппаратные кошельки для создания wallet с несколькими подписями.



![Dispositif ajouté](assets/fr/08.webp)



## Создание портфолио



После добавления оборудования wallet нажмите "Создать единый ключ wallet", чтобы создать wallet с одной подписью.



Дайте портфелю имя (например, "Wallet для tuto") и выберите тип адреса. Выберите "Segwit", чтобы использовать родные адреса bech32, оптимизирующие транзакционные издержки.



![Configuration du portefeuille](assets/fr/09.webp)



После создания портфолио Specter предлагает сохранить резервный PDF-файл, содержащий всю публичную информацию, необходимую для восстановления портфолио (дескрипторы, расширенные открытые ключи). Этот файл не содержит ваших закрытых ключей.



![Sauvegarde du portefeuille](assets/fr/10.webp)



## Получение биткоинов



Чтобы получить биткоины, выберите свой wallet в левом меню, а затем нажмите на вкладку "Получить".



Specter автоматически генерирует новый адрес приема с QR-кодом.



![Génération d'une adresse de réception](assets/fr/11.webp)



Вы можете скопировать адрес или отсканировать QR-код. Всегда проверяйте адрес на экране оборудования wallet, прежде чем передавать его кому-либо.



## Просмотр истории и адресов



После получения биткоинов вы можете просмотреть свои транзакции на вкладке "Транзакции".



![Historique des transactions](assets/fr/12.webp)



На вкладке "Адреса" вы можете просмотреть все адреса, созданные вашим портфелем, с указанием статуса их использования и соответствующих сумм.



![Liste des adresses](assets/fr/13.webp)



## Отправить биткойны



Чтобы отправить биткоины, перейдите на вкладку "Отправить". Введите адрес получателя, сумму для отправки и отметьте дополнительные опции, если хотите вручную выбрать UTXO (контроль монет).



![Création d'une transaction](assets/fr/14.webp)



Нажмите на кнопку "Создать неподписанную транзакцию", чтобы создать транзакцию. Затем Specter попросит вас подписать транзакцию с помощью оборудования wallet.



![Signature de la transaction](assets/fr/15.webp)



Если вы используете Coldcard, у вас будет выбор: подписать через USB или использовать карту microSD (с воздушной загрузкой). Подтвердите транзакцию на аппаратном экране wallet, тщательно проверив адрес назначения и сумму.



После того как транзакция подписана, вы можете транслировать ее в сети Bitcoin.



![Options de diffusion](assets/fr/16.webp)



Нажмите на кнопку "Отправить транзакцию", чтобы отправить транзакцию. Specter подтвердит, что транзакция была отправлена, и вы сможете отслеживать ее состояние на вкладке "Транзакции".



![Diffusion de la transaction](assets/fr/17.webp)



## Создание и использование портфеля с несколькими подписями



Одним из главных достоинств Specter Desktop является возможность упростить управление портфелями с несколькими подписями. Мультисигма wallet требует нескольких подписей для авторизации транзакции, что устраняет единую точку отказа. Например, в конфигурации 2-on-3 для подтверждения расходов требуется две подписи из трех отдельных аппаратных кошельков.



Чтобы создать мультисигму wallet, начните с добавления всех подписывающих аппаратных кошельков через "Добавить устройство". В этом примере мы будем использовать три разных аппаратных кошелька: Coldcard MK4 (уже добавленный ранее), Passport и Ledger. Такая диверсификация производителей повышает безопасность, позволяя избежать зависимости от одной цепочки поставок или прошивки.



Вот ссылки на учебники по Ledger и Passport:



https://planb.academy/tutorials/wallet/hardware/ledger-nano-s-plus-75043cb3-2e8e-43e8-862d-ca243b8215a4

https://planb.academy/tutorials/wallet/hardware/passport-74e53858-3fa2-43f9-b866-573297546236

Добавьте Passport, назвав оборудование wallet (например, "Passport multi") и импортировав его ключи с помощью карты microSD или QR-кода. Затем нажмите кнопку "Продолжить", чтобы продолжить.



![Ajout du Passport](assets/fr/23.webp)



Затем добавьте Ledger, подключив его через USB и открыв приложение Bitcoin на оборудовании wallet. Назовите его (например, "ledger multi") и нажмите "Получить через USB", а затем "Продолжить", чтобы импортировать его открытые ключи.



![Ajout du Ledger](assets/fr/24.webp)



После регистрации трех аппаратных кошельков в Specter нажмите на кнопку "Добавить wallet" и выберите опцию "Множественная подпись", чтобы создать wallet с несколькими подписями.



![Choix du type de wallet](assets/fr/25.webp)



Выберите три аппаратных кошелька, которые вы хотите включить в кворум мультиподписи: MK4 Tuto, Passport multi и ledger multi. Нажмите "Продолжить", чтобы перейти к следующему шагу.



![Sélection des hardware wallets pour le multisig](assets/fr/26.webp)



Выберите конфигурацию мультиподписи. Выберите "Segwit" в качестве типа адреса, чтобы воспользоваться оптимизированными комиссиями. Параметр "Требуемые подписи для авторизации транзакций (м из 3)" позволяет определить порог: для конфигурации "2 на 3" требуется 2 подписи. Каждое оборудование wallet отображает соответствующий мультисигмальный ключ. Нажмите кнопку "Создать wallet", чтобы завершить создание.



![Configuration 2-sur-3 Segwit](assets/fr/27.webp)



Теперь ваше портфолио с несколькими подписями "Multi tuto" создано. Specter сразу же рекомендует сохранить резервный PDF-файл, содержащий дескриптор портфолио. Нажмите на "Сохранить резервную копию PDF", чтобы загрузить этот важный файл.



![Wallet multisig créé](assets/fr/28.webp)



Specter также позволяет экспортировать информацию wallet в каждый из аппаратных кошельков через QR-код или файл. Это позволяет некоторым аппаратным кошелькам (например, Coldcard или Passport) хранить конфигурацию multisig непосредственно в своей памяти.



Для Passport разблокируйте устройство, затем перейдите в "Manage Account" > "Connect Wallet" > "Specter" > "Multisig" > "QR Code", затем отсканируйте QR-код, сгенерированный Specter. Затем Passport попросит вас отсканировать адрес приема с вашего wallet для подтверждения конфигурации multisig.



Для MK4 подключите его к компьютеру и разблокируйте. Затем нажмите "Сохранить файл MK4 Tuto" и сохраните файл на MK4. В следующий раз, когда вы будете подписывать оборудование wallet, MK4 будет использовать этот файл для завершения настройки мультисигмы.



![Export vers les hardware wallets](assets/fr/29.webp)



К вашему сведению, вы можете в любое время получить доступ к резервным копиям на вкладке "Настройки" вашего портфолио, затем "Экспорт":



![Accès au backup PDF](assets/fr/30.webp)



Повседневное использование остается похожим на простой wallet: вы получаете адреса generate, как обычно. Чтобы отправить биткоины, перейдите на вкладку "Отправить", введите адрес получателя и сумму, затем нажмите "Создать неподписанную транзакцию".



![Création d'une transaction multisig](assets/fr/31.webp)



Specter создает PSBT (Partially Signed Bitcoin Transaction) и выводит сообщение "Acquired 0 of 2 signatures". Теперь вы должны подписать как минимум два из трех аппаратных кошельков. Нажмите на первый аппаратный wallet (например, "MK4 Tuto"), чтобы подписать его Coldcard, затем на второй (например, "Passport multi"), чтобы получить вторую необходимую подпись.



![Signature de la transaction](assets/fr/32.webp)



После того как вы получили 2 необходимые подписи (в интерфейсе отображается "Получено 2 из 2 подписей" и "Транзакция готова к отправке"), нажмите на кнопку "Отправить транзакцию", чтобы транслировать транзакцию в сеть Bitcoin.



![Transaction prête à être diffusée](assets/fr/33.webp)



Такой подход с несколькими подписями особенно хорошо подходит для компаний (несколько руководителей должны утверждать расходы), семей (защита наследства нескольких поколений) или частных лиц, управляющих крупными суммами (географическое распределение аппаратных кошельков, чтобы противостоять локальным катастрофам).



### Критическая важность резервного копирования с несколькими подписями



**Обратите внимание**: резервное копирование многосигового портфеля принципиально отличается от резервного копирования односигового портфеля. Одних только фраз восстановления (фраз seed) недостаточно для восстановления многосигового портфеля. Необходимо также создать резервную копию **output descriptor** (output descriptor), содержащую информацию о конфигурации портфеля из нескольких знаков.



output descriptor содержит важные данные: расширенные открытые ключи (xpubs) каждого соподписанта, порог подписи (в нашем примере - 2 на 3), тип используемого скрипта (native, nested или legacy Segwit), а также пути обхода для каждого оборудования wallet. Без этого дескриптора, даже если у вас есть две из трех фраз восстановления, вы не сможете восстановить wallet или получить доступ к своим биткоинам. Дескриптор позволяет вашему программному обеспечению узнать, как объединить открытые ключи для generate с адресами Bitcoin, соответствующими вашим средствам.



Specter Desktop автоматически генерирует резервный PDF-файл при создании мультисиг-портфеля. Этот PDF-файл содержит полный дескриптор, отпечатки пальцев каждого оборудования wallet и всю публичную информацию, необходимую для восстановления. **Этот файл не содержит ваших приватных ключей** и поэтому сам по себе не позволяет вам тратить ваши биткоины, но он позволяет любому, кто получит к нему доступ, увидеть всю историю ваших транзакций и баланс.



Чтобы правильно создать резервную копию конфигурации мультиподписи, выполните следующую процедуру: после создания портфолио перейдите на вкладку "Настройки", затем "Экспорт" и выберите "Сохранить резервную копию PDF". Создайте несколько копий этого PDF-файла: распечатайте как минимум две копии на бумаге, а также сохраните зашифрованную цифровую копию. Храните по одной копии PDF с каждой из ваших фраз для восстановления в географически разных местах.



Выгравируйте фразы для восстановления на огнеупорных и водонепроницаемых металлических пластинах, чтобы гарантировать их долговечность. Никогда не недооценивайте важность этих резервных копий: если вы потеряете папку `~/.specter` на своем компьютере и потеряете один из своих аппаратных кошельков без резервной копии дескрипторов, все ваши средства будут безвозвратно потеряны, даже при конфигурации 2 на 3. Многозначное резервирование защищает от потери аппаратного wallet, но только если вы правильно создали резервную копию дескриптора вашего wallet.



## Преимущества и ограничения Specter Desktop



**Преимущества**: Оптимальная конфиденциальность благодаря полной локальной проверке без использования сторонних серверов. Гибкость мультиподписей для расширенных конфигураций (корпоративных, семейных, индивидуальных). Широкая поддержка аппаратного обеспечения wallet с полной совместимостью (USB и air-gapped).



**Ограничения**: Значительное обучение продвинутым концепциям Bitcoin (UTXOs, дескрипторы, пути деривации).



## Лучшие практики



Всегда проверяйте адреса и суммы на экране оборудования wallet перед валидацией, чтобы защитить себя от вредоносных программ.



Храните резервные копии PDF отдельно от семян. Эти открытые дескрипторы можно хранить в банковском хранилище или зашифрованном облаке, что позволит восстановить их без раскрытия закрытых ключей.



Протестируйте восстановление на суммах token, прежде чем использовать портфели с крупными средствами. Создавайте, тестируйте, удаляйте и восстанавливайте, чтобы проверить правильность своих процедур.



Поддерживайте Specter и прошивку в актуальном состоянии. Распределите соподписантов с несколькими подписями по географическому принципу (дом/офис/близко), чтобы противостоять локальным катастрофам. Используйте описательные ярлыки для упрощения бухгалтерского учета и налоговых деклараций.



## Бонус: установка на сервер Bitcoin (Umbrel, RaspiBlitz, Start9)



Если вы уже являетесь владельцем сервера Bitcoin, такого как Umbrel, RaspiBlitz, MyNode или Start9, вы можете установить Specter Desktop прямо из их магазина приложений. Такой подход дает несколько существенных преимуществ: приложение автоматически конфигурируется с вашим локальным узлом Bitcoin Core, оно остается доступным 24/7 через веб-интерфейс с любого устройства в вашей сети, и вы даже можете получить к нему безопасный удаленный доступ через Tor. Вся ваша инфраструктура Bitcoin сосредоточена на одном выделенном сервере, что упрощает управление и укрепляет ваш суверенитет.



### Установка из магазина приложений Umbrel



Из интерфейса Umbrel перейдите в App Store и найдите Specter Desktop. Нажмите на "Установить", чтобы запустить установку.



![App Store Umbrel - Specter Desktop](assets/fr/18.webp)



После завершения установки откройте Specter Desktop на вашем Umbrel. На экране приветствия вам будет предложено выбрать тип подключения. Если вы используете Specter на Umbrel, нажмите "Обновить настройки", чтобы настроить соединение.



![Écran de bienvenue Specter sur Umbrel](assets/fr/19.webp)



Выберите "Удаленное USB-подключение Specter", чтобы разрешить использование аппаратных USB-кошельков, подключенных к вашему локальному компьютеру, при использовании Specter на удаленном сервере Umbrel.



![Configuration Remote Specter USB](assets/fr/20.webp)



Следуйте отображаемым инструкциям для настройки моста HWI. Вам необходимо зайти в настройки моста устройства и добавить домен `http://umbrel.local:25441` в белый список. Нажмите на кнопку "Обновить", чтобы сохранить конфигурацию.



![HWI Bridge Settings](assets/fr/21.webp)



Если вы также хотите использовать аппаратные USB-кошельки с локального компьютера, загрузите приложение Specter Desktop на свой компьютер и установите значение "Да, я запускаю Specter удаленно". Нажмите на кнопку "Сохранить", чтобы завершить настройку.



![Configuration connexion remote dans l'app](assets/fr/22.webp)



## Заключение



Specter Desktop демократизирует расширенные конфигурации Bitcoin, делая мультиподпись доступной без ущерба для суверенитета и конфиденциальности. Для пользователей, управляющих значительными суммами денег, он превращает институциональные практики в решения, которые могут быть использованы частными лицами.



Хотя приложение требует первоначальных инвестиций в инфраструктуру и обучение, оно обеспечивает полный суверенитет: контроль над инфраструктурой проверки, физическое владение ключами и транзакции, свободные от наблюдения третьих сторон. Будь то частное лицо, защищающее свои сбережения, семья, создающая сейф для нескольких поколений, или компания, управляющая денежными потоками, Specter Desktop - это эталонный инструмент, позволяющий совместить максимальную безопасность и абсолютный суверенитет.
