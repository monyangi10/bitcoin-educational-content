![cover](assets/cover.webp)



Spectre Desktop ni porogaramu y’inkomoko yuguruye (uruhusha rwa MIT) yateguwe na Cryptoadvance kuva mu 2019 ishobora kworohereza uburongozi bw’amasakoshi ya Bitcoin n’amasakoshi yawe y’ibikoresho (Ledger, Trezor, Coldcard, BitBox02, Pasiporo, n’ibindi) n’imiterere yawe bwite ya Bitcoin. umukozi). Iryo koraniro riraruta ayandi cane cane mu gutunganya imikono myinshi, bikagufasha kuronka amahera menshi mu gutanga ububasha bwo gusinya hagati y’ibikoresho vyinshi vyigenga.



**Muri iyi nyigisho, uzomenya ingene:**




- Shiraho kandi utunganye Spectre Desktop kuri mudasobwa yawe (amadirisha, macOS canke Linux)
- Huza Spectre na server ya Electrum (tuzokoresha Umbrel muri aka karorero)
- Rema wallet yoroshe n'ibikoresho vya wallet (Coldcard)
- Kwakira no kohereza bitcoins n'ubusegaba bushitse
- Gushinga wallet y'imikono myinshi 2-ku-3 ifise amasakoshi menshi y'ibikoresho
- Shiraho Spectre kuri server ya Umbrel (agashimwe k'imbere)



Ivyo ukoresha vyose bizokwemezwa mu karere kawe biciye ku bikorwa remezo vyawe bwite, ata makuru uzorungika ku ba serveri bo hanze, ivyo bikaba bizokwemeza ibanga ryawe n’ubusegaba bwawe mu vy’amahera. Imisi yose banza usuzume amafaranga akoreshwa ku rubuga rwawe rwa wallet imbere yo gusinya.



## Gukuraho no gushiramwo



Sura urubuga rwemewe rwa Spectre Desktop kugira ngo ubone iyo porogarama.



![Page d'accueil Specter](assets/fr/01.webp)



Ku rupapuro rwo gukuraho, hitamwo verisiyo ihuye n’ivyo ukoresha: macOS, Windows canke Linux.



![Téléchargement selon l'OS](assets/fr/02.webp)



Iyo umaze kuyikura kuri interineti, shiramwo iyo porogarama nk’uko amabwirizwa asanzwe akoreshwa na sisitemu yawe. Ku bwa macOS, kurura ikimenyetso mu Bikoresho. Ku bijanye na Windows, ukoreshe igikoresho co gushiramwo. Ku Linux, ukurikize amabwirizwa y'ivyo bikoresho.



## Imiterere y'intango



Ku gutangura kwa mbere, Spectre Desktop iragusaba guhitamwo ubwoko bw’ihuriro ryawe. Ushobora kwifatanya na server ya Electrum canke n'umurongo wawe bwite wa Bitcoin Core.



![Choix du type de connexion](assets/fr/03.webp)



Muri aka karorero, tuzokoresha ubufatanye na server ya Electrum ikoresha Umbrel.



Ushaka kumenya vyinshi, raba inyigisho yacu y'umutaka:



https://planb.academy/tutorials/node/bitcoin/umbrel-8b0e3b5b-d3cf-4a1e-8bb8-1ad2db4dd848

Iyi nzira itanga uguhuza vyihuta kuruta Bitcoin Core. Niba ushaka, ushobora guhitamwo "Bitcoin Core" maze ugatunganya uruja n'uruza ku nzira yawe yo mu karere. Intambwe zikurikira ziguma ari zo nyene utitaye ku mahitamwo yawe.



Hitamwo "Ihuza rya Electrum" hanyuma uhitemwo "Injira ryanje" kugira ngo utunganye server yawe ya Electrum.



![Configuration Electrum](assets/fr/04.webp)



Injira aderesi ya server yawe ya Electrum. Mu gihe cacu n'umutaka, aderesi izoba `umutaka.local` n'icuma `50001`. Fyonda kuri "Connect" kugira ngo ushireho iyo nzira.



Iyo umaze gufatanya, igicapo c’akabazo kiraboneka, kirimwo urutonde rw’ibintu vyo gusuzuma kugira ngo utangure. Ubu ukeneye kwongerako ama wallets yawe y’ibikoresho.



![Écran d'accueil](assets/fr/05.webp)



## Kwongera ibikoresho vya wallet



Mu rutonde rw'ibubamfu, fyonda ku "Add device" kugira ngo wongereko ibikoresho vyawe vya wallet.



Spectre Desktop ifasha amasakoshi menshi y’ibikoresho: Trezor, Ledger, BitBox02, Coldcard, KeepKey, Keybuye, Cobo Vault, n’ibindi vyinshi.



Niba wifuza kumenya vyinshi, raba inyigisho zacu z’ibikoresho wallet.



![Sélection du type de hardware wallet](assets/fr/06.webp)



Hitamwo ibikoresho vyawe vya wallet. Muri aka karorero, turiko turakoresha Coldcard MK4.



Aha hepfo uzosanga inyigisho yacu y’iyi hardware ya wallet:



https://planb.academy/tutorials/wallet/hardware/coldcard-5d44dd94-423d-4e37-9a8c-3fc38b45ce59

Ku Coldcard, ukeneye gutanga imfunguruzo za bose zivuye mu bikoresho vya wallet biciye ku nzira ya USB canke ikarita ya microSD.



![Import des clés du Coldcard](assets/fr/07.webp)



Kurikiza amabwirizwa yerekanwa kugira ngo urungike hanze imfunguruzo zivuye kuri Coldcard yawe. Ha izina ry'ibikoresho vyawe vya wallet (aha "MK4 Tuto"). Imfunguruzo zimaze kwinjizwa, ushobora gukora wallet n'urufunguzo rumwe, canke ukongerako ibindi bikoresho vy'amasakoshi ku wallet y'imikono myinshi.



![Dispositif ajouté](assets/fr/08.webp)



## Guhingura ububiko



Amaze kwongerako ibikoresho vyawe vya wallet, kanda kuri "Rema urufunguzo rumwe wallet" kugira ngo ureme wallet y'umukono umwe.



Ha igitabu cawe izina (nk'akarorero "Wallet for tuto") hanyuma uhitemwo ubwoko bw'aderesi. Hitamwo "Segwit" kugira ngo ukoreshe aderesi za bech32 zituma amafaranga y'ibikorwa agenda neza.



![Configuration du portefeuille](assets/fr/09.webp)



Igihe igitabu cawe kimaze kuremwa, Spectre itanga kubika dosiye ya PDF y’ububiko irimwo amakuru yose ya bose akenewe kugira ngo ugarure igitabo cawe (ibisobanuro, imfunguruzo za bose zagutse). Iyi dosiye ntirimwo imfunguruzo zawe z'ibanga.



![Sauvegarde du portefeuille](assets/fr/10.webp)



## Kwakira ama bitcoins



Kugira ngo uronke ama bitcoins, hitamwo wallet yawe mu rutonde rw'ibubamfu, hanyuma ukande ku rubuga "Receive".



Spectre ica itanga aderesi nshasha yo kwakira abantu ifise kode ya QR.



![Génération d'une adresse de réception](assets/fr/11.webp)



Ushobora gukopa iyo aderesi canke ugacapura kode ya QR. Imisi yose urabe aderesi iri ku rubuga rwawe rwa wallet imbere y’uko uyiha umuntu uwo ari we wese.



## Raba amateka n'amaderesi



Uhejeje kwakira ama bitcoins, urashobora kubona amafaranga yawe mu gice ca "Transactions".



![Historique des transactions](assets/fr/12.webp)



Urupapuro rwa "Aderesi" rugufasha kubona aderesi zose zakozwe n'ibitabo vyawe, n'ingene zikoreshwa n'amahera ajanye.



![Liste des adresses](assets/fr/13.webp)



## Wohereze ama bitcoins



Kugira ngo wohereze ama bitcoins, fyonda ku rubuga rwa "Ohereza". Injira aderesi y’uwuzokwakira, amahera azorungikwa, hanyuma usuzume amahitamwo ateye imbere nimba wipfuza guhitamwo n’amaboko UTXOs (coin control).



![Création d'une transaction](assets/fr/14.webp)



Fyonda kuri "Rema Igikorwa kitagira Umukono" kugira ngo wubake igicuruzwa. Spectre izoca igusaba gusinya iyo nzira n’ibikoresho vyawe vya wallet.



![Signature de la transaction](assets/fr/15.webp)



Niba ukoresha Coldcard, uzoba ufise amahitamwo yo gusinya kuri USB canke ukoresheje ikarita ya microSD (ifise umuyaga). Wemeze ko ivyo ugurisha ku rubuga rwawe rwa wallet, usuzume witonze aderesi y’aho uja n’amahera.



Iyo iyo nzira imaze gusinywa, urashobora kuyitangaza ku rubuga rwa Bitcoin.



![Options de diffusion](assets/fr/16.webp)



Fyonda kuri "Ohereza amafaranga" kugira ngo wohereze amafaranga. Spectre izokwemeza ko amafaranga yawe yoherejwe, kandi ushobora gukurikirana uko ameze mu gice c'amafaranga.



![Diffusion de la transaction](assets/fr/17.webp)



## Guhingura no gukoresha ububiko bw'imikono myinshi



Kimwe mu bintu bikomeye vya Spectre Desktop ni ubushobozi bwayo bwo kworohereza uburongozi bw’ibitabo vy’imikono myinshi. Multisig wallet isaba imikono myinshi kugira ngo umuntu yemerere gucuruza, gutyo bigakuraho ikintu kimwe co kunanirwa. Nk’akarorero, ugutunganya 2-ku-3 bisaba imikono ibiri ivuye mu bikoresho bitatu bitandukanye kugira ngo umuntu yemeze amahera yose akoreshwa.



Kugira ngo ureme multisig wallet, tangura wongeyeko amasakoshi yose y'ibikoresho vy'ugusinya biciye ku "Kwongerako igikoresho". Muri aka karorero, tuzokoresha amasakoshi atatu atandukanye: Coldcard MK4 (yamaze kwongerwako mbere), Passport na Ledger. Ukwo guhinduranya abahinguzi birakomeza umutekano mu kwirinda kwizigira uruhererekane rumwe rw’ibikoresho canke porogarama imwe.



Aha niho hari amahuzu y’inyigisho za Ledger na Passport:



https://planb.academy/tutorials/wallet/hardware/ledger-nano-s-plus-75043cb3-2e8e-43e8-862d-ca243b8215a4

https://planb.academy/tutorials/wallet/hardware/passport-74e53858-3fa2-43f9-b866-573297546236

Yongerako Pasiporo mu kwita izina ry'ibikoresho vya wallet (nk'akarorero "Pasiporo multi") hanyuma ushiremwo imfunguruzo zayo biciye ku ikarita ya microSD canke kode ya QR. Hanyuma ukande kuri "Continue" kugira ngo ukomeze.



![Ajout du Passport](assets/fr/23.webp)



Hanyuma wongereko Ledger mu kuyifatanya biciye kuri USB hanyuma ugafungura porogarama ya Bitcoin ku bikoresho vya wallet. Ihe izina (nk'akarorero "ledger multi") hanyuma ukande kuri "Get via USB" hanyuma "Bandanya" kugira ngo ushiremwo imfunguruzo zayo za bose.



![Ajout du Ledger](assets/fr/24.webp)



Umaze kwandika ama wallet yawe atatu muri Spectre, fyonda ku "Ongera wallet" hanyuma uhitemwo "Isinya myinshi" kugira ngo ureme wallet y'imisinya myinshi.



![Choix du type de wallet](assets/fr/25.webp)



Hitamwo amasakoshi atatu y’ibikoresho wipfuza gushiramwo mu rwego rwawe rw’imikono myinshi: MK4 Tuto, Passport multi na ledger multi. Fyonda kuri "Bandanya" kugira ngo ugende ku ntambwe ikurikira.



![Sélection des hardware wallets pour le multisig](assets/fr/26.webp)



Hitamwo imiterere yawe y'imikono myinshi. Hitamwo "Segwit" nk'ubwoko bw'aderesi kugira ngo wungukire ku mafaranga meza. "Imikono isabwa kugira ngo wemerere ibikorwa (m kuri 3)" ishobora kugufasha gusobanura urugero: ku bijanye n'imiterere ya 2-ku-3, imikono 2 irakenewe. Igikoresho cose ca wallet kigaragaza urufunguzo rwaco rwa multisig rujanye n’ico. Fyonda kuri "Rema wallet" kugira ngo uheze kurema.



![Configuration 2-sur-3 Segwit](assets/fr/27.webp)



Igitabo cawe c'imikono myinshi "Multi tuto" ubu cararemwe. Spectre ica igusaba ko ubika dosiye ya PDF y’ububiko irimwo insobanuro y’ibikorwa. Fyonda kuri "Save Backup PDF" kugira ngo ubone iyi dosiye ihambaye.



![Wallet multisig créé](assets/fr/28.webp)



Spectre kandi iragufasha kwohereza amakuru ya wallet muri buri nkweto yawe y’ibikoresho biciye ku kode ya QR canke dosiye. Ivyo bituma amasakoshi amwe amwe y’ibikoresho (nka Coldcard canke Passport) ashobora kubika amakuru menshi mu bwenge bwayo.



Ku bijanye na Passport, fungura igikoresho cawe hanyuma ugende kuri "Gucungera konti" > "Huza Wallet" > "Specter" > "Multisig" > "Kode ya QR", hanyuma ushireko kode ya QR yakozwe na Specter. Passport yawe izoca igusaba gucapura aderesi y’ukwakira iva kuri wallet yawe kugira ngo wemeze ivyerekeye multisig.



Ku bijanye na MK4, uyishire muri PC yawe maze uyifungure. Hanyuma ukande kuri "Bika dosiye ya MK4 Tuto" hanyuma ubike iyo dosiye kuri MK4 yawe. Igihe kizoza ushize umukono ku bikoresho vyawe vya wallet, MK4 izokoresha iyo dosiye kugira ngo iheze gutunganya multisig.



![Export vers les hardware wallets](assets/fr/29.webp)



Ku makuru yawe, ushobora kuronka amakuru y'inyuma igihe cose ukoresheje "Imiterere" y'ibitabo vyawe, hanyuma "Usohora":



![Accès au backup PDF](assets/fr/30.webp)



Ikoreshwa ry’umusi ku musi riguma rimeze nk’iry’i wallet yoroshe: wewe generate uronka amaderesi nk’uko bisanzwe. Kugira ngo wohereze bitcoins, genda kuri "Kohereza", winjize aderesi y'uwuzokwakira n'amahera, hanyuma ukande kuri "Rema Ibikorwa Bitagira Umukono".



![Création d'une transaction multisig](assets/fr/31.webp)



Spectre yubaka PSBT (Partially Signed Bitcoin Transaction) kandi yerekana "Yaronse 0 ku mikono 2". Ubu utegerezwa gusinya n’imiburiburi bibiri muri bitatu vy’ibikoresho vyawe. Fyonda ku gikoresho ca mbere ca wallet (nk'akarorero "MK4 Tuto") kugira ngo usinye Coldcard yawe, hanyuma ku ca kabiri (nk'akarorero "Passport multi") kugira ngo ubone umukono wa kabiri usabwa.



![Signature de la transaction](assets/fr/32.webp)



Uhejeje kuronka amasinyatire 2 asabwa (interface yerekana "Waronse amasinyatire 2 kuri 2" na "Igurisha ryiteguye koherezwa"), ukande kuri "Ohereza Igurisha" kugira ngo utangaze iryo gucuruza ku rubuga rwa Bitcoin.



![Transaction prête à être diffusée](assets/fr/33.webp)



Ubwo buryo bwo gushirako amasinya menshi burabereye cane cane amashirahamwe (abarongozi benshi barakeneye kwemeza amahera akoreshwa), imiryango (ugukingira iragi ry’ibisekuru vyinshi), canke abantu ku giti cabo bacungera amahera menshi (ugukwiragiza amasakoshi y’ibikoresho mu karere kugira ngo bashobore guhangana n’ivyago vyo mu karere).



### Akamaro gakomeye k'ububiko bw'imikono myinshi



**Iyumvire**: gushigikira igitabu ca multisig bitandukanye cane n’ugushigikira igitabu kimwe. Amajambo yawe yo gusubirana (amajambo seed) yonyene ntahagije kugira ngo usubizeho igitabu c’ibintu vyinshi. Utegerezwa kandi gukora ububiko bwa **output descriptor** (output descriptor), irimwo amakuru y'imiterere y'ibitabo vyawe vy'amazina menshi.



output descriptor irimwo amakuru y’ingenzi: imfunguruzo za bose zagutse (xpubs) z’umuntu wese afata umukono, urugero rw’umukono (2-ku-3 mu karorero kacu), ubwoko bw’inyandiko ikoreshwa (Segwit y’imvukira, y’ivyatsi canke y’iragi), n’inzira z’uguca mu nzira z’ibikoresho vya wallet vyose. Ata n’iyi ndondoro, naho woba ufise amajambo abiri muri atatu yo gusubirana, ntuzoshobora gusubira kwubaka wallet yawe canke ngo ushikire ama bitcoins yawe. Ico gisobanuro kiratuma porogarama yawe imenya ingene yofatanya imfunguruzo za bose kuri generate amaderesi ya Bitcoin ahuye n’amahera yawe.



Spectre Desktop ihita itanga dosiye ya PDF iyo uremye igitabo cawe ca multisig. Iyi PDF irimwo insobanuro yuzuye, ibimenyetso vy’intoke vy’ibikoresho vyose vya wallet, n’amakuru yose ya bose asabwa kugira ngo bisubirwemwo. **Iyi dosiye nta mfunguruzo zawe z’ibanga irimwo** rero ubwayo ntikwemerera gukoresha amafaranga yawe ya bitcoins, ariko iremesha umuntu wese ayironka kubona amateka yawe yose y’ibikorwa vyawe n’amahera ufise.



Kugira ngo ukore ububiko bwiza bw'imiterere yawe y'imikono myinshi, kurikiza ubu buryo: umaze gukora igitabo cawe, fyonda ku rubuga rwa "Imiterere", hanyuma "Sohora" hanyuma uhitemwo "Bika ububiko bwa PDF". Rema kopi nyinshi z’iyi PDF: ucape n’imiburiburi kopi zibiri ku mpapuro, kandi ubike kopi ya digitale ipfutse. Bika kopi imwe ya PDF iri kumwe n’amajambo yawe yose yo gusubirana, mu bibanza bitandukanye.



Capa amajambo yawe yo gukira ku masahani y’ivyuma adashobora guterwa n’umuriro n’amazi kugira ngo ashobore kubaho igihe kirekire. Ntukigere ukengera akamaro k’ivyo bipimo: iyo utakaje dosiye `~/.specter` kuri mudasobwa yawe KANDI utakaje imwe mu nkuru zawe z’ibikoresho ata nkuru y’ibisobanuro, amahera yawe yose azotakara ataco ashobora gukora, mbere n’aho woba ufise ubuhinga bwa 2-ku-3. Multi-signature redundancy irinda gutakaza ibikoresho vya wallet, ariko gusa iyo warashigikiye neza igisobanuro ca wallet yawe.



## Inyungu n'imipaka ya Spectre Desktop



**Ivyiza**: Ibanga ryiza cane n’ukwemezwa kwose kw’aho hantu ata ba serveri b’abandi. Uguhinduranya imikono myinshi ku mitunganyirize iteye imbere (ishirahamwe, umuryango, umuntu ku giti ciwe). Ivyuma vyinshi wallet bifasha n’ugukorana neza (USB n’ivy’umuyaga).



**Imipaka**: Ivyigwa bihambaye ku vyiyumviro vya Bitcoin biteye imbere (UTXOs, ibisobanuro, inzira z’ugukura).



## Ibikorwa vyiza



Imisi yose urabe amaderesi n’amahera ku gicapo c’ibikoresho vyawe wallet imbere y’uko uvyemeza, kugira ngo wikingire porogarama mbi.



Gumana ama backups ya PDF atandukanye n’imbuto zawe. Ivyo bisobanuro vya bose birashobora kubikwa mu nzu y’ububiko bwa banki canke mu gicu gipfutse, bikaba vyoroshe gusubirana ata mfunguruzo zawe z’ibanga zigaragara.



Igerageza gusubirana ku mahera ya token imbere yo gukoresha ibitabo vyawe bifise amahera menshi. Rema, ugerageze, usibe kandi usubiremwo kugira ngo wemeze uburyo bwawe.



Gumana Spectre na firmware yawe bishasha. Gutanga abafatanyabikorwa bawe bafise amasinya menshi mu karere (i muhira/ibiro/hafi) kugira ngo ushobore kwihanganira ibiza vyo mu karere. Koresha ibimenyetso vy'insobanuro kugira ngo worohereze ivy'itunganywa ry'amakonte n'ivy'imisoro.



## Ivyiza: Gushiramwo kuri server ya Bitcoin (Umuttaka, RaspiBlitz, Gutangura9)



Niba usanzwe ufise umukozi wa Bitcoin nka Umbrel, RaspiBlitz, MyNode canke Start9, urashobora gushiramwo Spectre Desktop uhereye mu bubiko bwabo bw’ibikoresho. Ubwo buryo buratanga inyungu nyinshi zihambaye: iyo porogarama yitunganya ubwayo n’uruzitiro rwawe rwa Bitcoin Core, iguma ishobora gukoreshwa 24/7 biciye ku rubuga rwa interface ukoresheje igikoresho cose kiri ku rubuga rwawe, kandi urashobora mbere kuyironka ata nkomanzi uri kure biciye kuri Tor. Ivyubatswe vyawe vyose vya Bitcoin bishizwe hamwe kuri server imwe yihariye, bikaba vyorosha uburongozi no gukomeza ubusegaba bwawe.



### Gushiramwo bivuye mu iduka rya porogaramu ry'umutaka



Uvuye ku rubuga rwawe rwa Umbrel, genda kuri App Store urondere Spectre Desktop. Fyonda kuri "Install" kugira ngo utangure gushiramwo.



![App Store Umbrel - Specter Desktop](assets/fr/18.webp)



Igihe ushizeho, fungura Spectre Desktop ku Mutaka wawe. Igishushanyo c’akabazo kizogusaba guhitamwo ubwoko bw’ihuriro ryawe. Niba ukoresha Spectre kuri Umbrel yawe, fyonda ku "Update settings" kugira ngo utunganye ihuriro.



![Écran de bienvenue Specter sur Umbrel](assets/fr/19.webp)



Hitamwo "Ihuzwa rya USB rya Spectre kure" kugira ngo ushobore gukoresha amasakoshi y'ibikoresho vya USB ihuye na mudasobwa yawe yo mu karere mu gihe ukoresha Spectre kuri server ya Umbrel iri kure.



![Configuration Remote Specter USB](assets/fr/20.webp)



Kurikiza amabwirizwa yerekanwa kugira ngo utunganye ikiraro ca HWI. Ukeneye gushika ku mirongo y'ikiraro c'igikoresho maze ukongerako urwego `http://umbrel.local:25441` ku rutonde rwera. Fyonda kuri "Update" kugira ngo ubike ivyashizweho.



![HWI Bridge Settings](assets/fr/21.webp)



Niba ushaka gukoresha amasakoshi yawe ya USB kuri mudasobwa yawe, fungura porogaramu ya Spectre Desktop kuri mashini yawe maze uyishire kuri "Ego, nkoresha Spectre kure". Fyonda kuri "Save" kugira ngo uheze gutegura.



![Configuration connexion remote dans l'app](assets/fr/22.webp)



## Iciyumviro



Spectre Desktop ikoresha demokarasi mu mirongo ya Bitcoin iteye imbere, ituma umuntu ashobora gukoresha amasinya menshi ataco atakaje ku busegaba bwawe canke ibanga ryawe. Ku bakoresha bacungera amahera menshi, bihindura imigenzo y’inzego ikaba inyishu zishobora gukoreshwa n’abantu ku giti cabo.



Naho iyo porogarama isaba ishoramari ry’intango mu bikorwa remezo no mu kwiga, itanga ubusegaba bushitse: kugenzura ibikorwa remezo vy’ukwemeza, ubutunzi bw’umubiri bw’imfunguruzo, n’ibikorwa vy’ubudandaji bitagira uwundi muntu abigenzura. Waba uri umuntu ku giti ciwe ariko aracungera amahera yazigamye, umuryango urema isandugu y’amahera y’uruvyaro rwinshi, canke ishirahamwe ricungera uruja n’uruza rw’amahera, Spectre Desktop ni igikoresho co guhuza umutekano mwinshi n’ubusegaba bushitse.
