Sparrow wallet ni porogaramu yo kwibungabunga Bitcoin Wallet yateguwe na Craig Raw. Iyi porogaramu y’inkomoko yuguruye ishimwa cane n’aba bitcoiners kubera ibiranga vyinshi n’ubuhinga bwa Interface.


Hari uburyo bubiri bwo gukoresha Sparrow:




- Nka Hot Wallet, aho imfunguruzo zawe z’ibanga zibikwa kuri PC yawe.
- Nk'umuyobozi wa Cold Wallet, aho imfunguruzo z'ibanga zifatwa kuri Hardware Wallet. Muri ubu buryo, Sparrow ikoresha gusa amakuru ya bose ya Wallet, ikurikirana amahera, itanga amaderesi, kandi yubaka amafaranga, ariko umukono wa Hardware Wallet urakenewe kugira ngo ayo mafaranga abe afise akamaro. Ishobora rero gusubirira ibikorwa nka Ledger Live canke Trezor Suite.


Sparrow ishigikira amasakoshi y’umukono umwe n’ay’amasakoshi menshi, kandi ishoboza gucunga amazi y’amasakoshi menshi. Nk’akarorero, urashobora gucungera icarimwe Wallet imwe ifatanye na Ledger, iyindi na Trezor, kandi ukagira n’iyindi Hot Wallet.


Iyi porogaramu kandi iratanga ubuhinga buteye imbere bwo kugenzura Coin, bikagufasha guhitamwo neza UTXOs uzokoresha mu bikorwa vyawe kugira ngo ukoreshe neza ibanga ryawe.


Ku bijanye n'uguhuza, Sparrow iragufasha guhuza n'uruzitiro rwawe rwa Bitcoin, haba kure biciye kuri Server ya Electrum, canke na Bitcoin core. Birashoboka kandi gukoresha node ya bose nimba utaragira iyawe. Amahuza yo kure akorwa biciye kuri Tor.

## Guhuza n'urubuga rwa Bitcoin


Kugira ngo ukoreshe urubuga rwa Bitcoin no gutangaza amafaranga yawe, Sparrow itegerezwa kuba ifatanye n’urubuga rwa Bitcoin. Hari uburyo butatu nyamukuru bwo gushinga iyo nzira:



- 🟡 Gukoresha node ya bose, ni ukuvuga gufatanya n’iyindi node yemerera mwene izo nzira. Niba udafise node yawe bwite ya Bitcoin, iyi nzira iragufasha gutangura gukoresha Sparrow ningoga. Ariko rero, iyo node uhuzako izobona amafaranga yawe yose, ivyo bikaba vyoshobora gutuma ibanga ryawe rihungabana. Kugira ububasha ku mfunguruzo zawe ni ngombwa, ariko kugira node yawe bwite ni vyiza kuruta. Rero ukoreshe iyo nzira gusa nimba uriko uratangura, mu gihe uzi ingorane zishobora gutera ubuzima bwite bwawe.
- 🟢 Gufatanya n'umurongo wa Bitcoin core. Niba ufise node yawe bwite ya Bitcoin core, urashobora kuyifatanya na Sparrow wallet, haba mu karere nimba Bitcoin core ishizwe ku mashini imwe, canke kure.
- 🔵 Guhuza biciye kuri server ya Electrum. Niba node yawe ya Bitcoin ifise Electrs, nk'uko biri ku bisubizo vy'node-in-a-box nka Umbrel canke Start9, ushobora kuyifatanya uri kure ya Sparrow.


**Ni vyiza gukoresha connection biciye kuri Electrs canke Bitcoin core ku node yawe bwite kugira ngo ugabanye ivy'ukwizigira uwundi muntu kandi ukoreshe neza ibanga ryawe**

### Gufatanya na Bitcoin core 🟢 .


Uburyo bwa kabiri bwo gufatanya n'uruzitiro rwa Bitcoin ni uguhuza Sparrow n'uruzitiro rwa Bitcoin core. Iyo Bitcoin core ishizwe kuri iyo mashini nyene, ivyemezo bizoba biciye ku dosiye y’amakuru. Niba Bitcoin core iri ku mashini iri kure, uzokenera gukoresha ijambobanga ryasobanuwe muri dosiye `Bitcoin.conf`.


Urashobora kubona ko iyo ukoresheje urudodo rwa pruned Bitcoin core, ntuzoshobora kugarura Wallet irimwo amafaranga y'imbere y'amabuye yabitswe mu karere. Ariko rero, ku Wallet nshasha yaremwe kuri Sparrow, ivyo ntibizoba ikibazo: amafaranga yawe mashasha azoboneka, mbere n’aho woba ufise urudodo rwa pruned.


Kugira ngo utunganye urudodo rwa Bitcoin core, ushobora kuraba imwe muri izi nyigisho zikurikira, bivanye n'uburyo ukoresha:


https://planb.network/tutorials/node/bitcoin/bitcoin-core-mac-windows-9684ab02-e0af-41c9-8102-86ac7c7727f3

https://planb.network/tutorials/node/bitcoin/bitcoin-core-linux-568c13a6-8746-4d63-8e95-f4a61c5ae0ed

Ku Sparrow, genda ku rubuga "*Bitcoin core*".


![Image](assets/fr/07.webp)


**Na Bitcoin core yo mu karere:**


Niba Bitcoin core ishizwe kuri mudasobwa yawe, rondera dosiye `Bitcoin.conf` mu madosiye ya porogarama. Niba iyi dosiye itahari, urashobora kuyirema. Gufungura n'umuhinduzi w'inyandiko hanyuma winjize umurongo ukurikira:


```ini
server=1
```


Hanyuma ubike ivyo wahinduye.


Ushobora kandi kubikora biciye ku gishushanyo ca Bitcoin-QT ca Interface mu kuja kuri "*Ivyagezwe*" > "*Amahitamwo...*" hanyuma ugakoresha amahitamwo ya "*Gushoboza umukozi wa RPC*".


Ntukibagire gusubira gufungura porogarama umaze guhindura ivyo.


![Image](assets/fr/08.webp)


Hanyuma usubire kuri Sparrow wallet winjize inzira ija kuri dosiye yawe y'amakuru, akenshi iri muri dosiye imwe na `Bitcoin.conf`, bivanye n'uburyo ukoresha:


| **macOS** | ~/Library/Application Support/Bitcoin |
| ----------- | ------------------------------------- |
| **Windows** | %APPDATA%\Bitcoin |
| **Linux** | ~/.Bitcoin |

![Image](assets/fr/09.webp)


Siga ibindi bipimo nk'ivya kera, URL `127.0.0.1` n'icuma `8332`, hanyuma ukande kuri "*Igerageza ry'Ihuza*".


![Image](assets/fr/10.webp)


Iryo sano rirashingwa. Ikimenyetso ca Green kizoboneka mu mfuruka iburyo hasi kugira ngo kigaragaze ko uhuye n’urudodo rwa Bitcoin core.


![Image](assets/fr/11.webp)


**N'ivyo kure Bitcoin core:**


Niba Bitcoin core ishizwe ku yindi mashini ifatanye n’urubuga rumwe, banza urondere dosiye `Bitcoin.conf` mu madosiye ya porogarama. Niba iyi dosiye itarabaho, urashobora kuyirema. Gufungura iyi dosiye n'umuhinduzi w'inyandiko wongereko umurongo ukurikira:


```ini
server=1
```


Uhejeje guhindura dosiye, urabe ko uyibika muri dosiye ibereye sisitemu yawe:


| **macOS** | ~/Library/Application Support/Bitcoin |
| ----------- | ------------------------------------- |
| **Windows** | %APPDATA%\Bitcoin |
| **Linux** | ~/.Bitcoin |

Ivyo bishobora kandi gukorwa biciye ku gishushanyo Bitcoin-QT Interface Interface. Genda kuri "*Ivyagezwe*", hanyuma "*Amahitamwo...*", hanyuma ukoreshe amahitamwo ya "*Gushoboza server ya RPC*" mu guca akamenyetso mu gasandugu kahuye. Niba dosiye `Bitcoin.conf` itahari, ushobora kuyihingura ukoresheje iyi Interface ukanda kuri "*Gufungura dosiye y'imiterere*".


![Image](assets/fr/12.webp)


Rondera IP Address y’imashini yakira Bitcoin core ku rubuga rwawe rwo mu karere. Kugira ngo ubikore, urashobora gukoresha igikoresho nk’iki [Igikoresho co gupima IP y’ishavu](https://angryip.org/). Reka twiyumvire, kubera impaka, ko IP Address y'urudodo rwawe ari `192.168.1.18`.


Mu dosiye `Bitcoin.conf`, wongeremwo imirongo ikurikira, ushireho `rpcbind=192.168.1.18` kugira ngo ihure na IP Address y'urudodo rwawe.


```ini
[main]
rpcbind=127.0.0.1
rpcbind=192.168.1.18
rpcallowip=127.0.0.1
rpcallowip=192.168.1.0/24
```


![Image](assets/fr/13.webp)


Mu dosiye `Bitcoin.conf`, wongeremwo izina ry'ukoresha n'ijambobanga ry'amahuza yo kure. Raba neza ko usubiriza `loic` izina ryawe ry'ukoresha na `ijambobanga ryanje` n'ijambobanga rikomeye:


```ini
rpcuser=loic
rpcpassword=my_password
```


![Image](assets/fr/14.webp)


Uhejeje guhindura no kubika dosiye, nusubire gufungura porogarama ya Bitcoin-QT.


Ubu ushobora gusubira kuri Sparrow wallet. Genda ku rubuga rwa "*Ukoresha / Guca*". Injira izina ry'ukoresha n'ijambobanga watunganije muri dosiye `Bitcoin.conf`. Siga ibindi bipimo nk'ivya kera, ni ukuvuga URL `127.0.0.1` n'icuma `8332`. Hanyuma ukande kuri "*Igerageza ry'Ihuza*".


![Image](assets/fr/15.webp)


Iryo sano rirashingwa. Ikimenyetso ca Green kizoboneka mu mfuruka iburyo hasi kugira ngo kigaragaze ko uhuye n’urudodo rwa Bitcoin core.


![Image](assets/fr/16.webp)

## Rema Hot Wallet


Ubu Sparrow wallet itunganijwe kugira ngo ivugane n'urubuga rwa Bitcoin, witeguye gukora Wallet yawe ya mbere. Iki gice kikuyobora mu kurema Hot Wallet, ni ukuvuga Wallet imfunguruzo zayo z’ibanga zibitswe kuri mudasobwa yawe. Kubera ko orodinateri yawe ari imashini igoye cane ifatanye na Internet, iratanga ahantu hanini cane ho gutera. Ku bw’ivyo, Hot Wallet ikwiye gukoreshwa gusa ku ma bitcoins makeyi. Kugira ngo ubike amahera menshi, hitamwo Wallet itekanye iri kumwe na Hardware Wallet. Nimba ari vyo uriko urarondera, urashobora gusimbuka ukaja mu gice gikurikira.


Kugira ngo ureme Hot Wallet, uhereye ku mugaragaro w'intango wa Sparrow wallet, kanda kuri "*Dosiye*" hanyuma kuri "*Wallet nshasha*".


![Image](assets/fr/21.webp)


Injira izina rya Wallet yawe hanyuma ukande kuri "*Rema Wallet*".


![Image](assets/fr/22.webp)


Hejuru ya Interface, ushobora guhitamwo gukora "*Umukono umwe*" canke "*Umukono mwinshi*" Wallet. Hasi gusa, hitamwo ubwoko bw'inyandiko yo gufunga UTXOs zawe. Ndagusavye gukoresha urugero rwa nyuma: "*Taproot (P2TR)*".


![Image](assets/fr/23.webp)


Hanyuma ukande kuri "*Software Wallet nshasha canke yashizwe hanze*".


![Image](assets/fr/24.webp)


Hitamwo urugero rwa BIP39, kuko rushigikirwa na porogarama hafi zose za Bitcoin Wallet. Inyuma y’aho, uhitemwo uburebure bw’ijambo ryawe ryo gukira. Ubu, ijambo ry’amajambo 12 rirahagije, kuko ryose ritanga umutekano usa, ariko ijambo ry’amajambo 12 ryoroshe gukiza.


![Image](assets/fr/25.webp)


Fyonda kuri buto ya "*generate Nshasha*" kugira ngo generate ijambo rya Mnemonic rya Wallet yawe. Iri jambo ritanga uburenganzira bwo gushika ku bitcoins zawe zose, ataco zibujijwe. Umuntu wese afise iri jambo arashobora kwiba amahera yawe, mbere n’aho atashobora gushika kuri mudasobwa yawe.


Iryo jambo ry’amajambo 12 risubizaho uburenganzira bwo gukoresha amafaranga yawe y’ibiceri (bitcoins) iyo mudasobwa yawe itakaye, yibwe canke ivunitse. Ni ngombwa rero cane ko umuntu ayizigama neza kandi akayibika ahantu heza.


Ushobora kuyindika ku mpapuro canke, kugira ngo urushirize kugira umutekano, ukayicapura ku cuma kitagira ibara kugira ngo uyikingire umuriro, umwuzure canke ngo isenyuke. Guhitamwo uburyo bwo gukoresha Mnemonic yawe bizovana n’ingene ukoresha umutekano, ariko nimba uriko urakoresha Sparrow nk’uburyo bwo gukoresha Wallet bushushe burimwo amahera aringaniye, impapuro zikwiye kuba zihagije.


Kugira ngo umenye vyinshi ku buryo bwiza bwo kubika no gucunga ijambo ryawe rya Mnemonic, ndagusavye cane gukurikira iyi yindi nyigisho, cane cane nimba uri umutanguzi:


https://planb.network/tutorials/wallet/backup/backup-mnemonic-22c0ddfa-fb9f-4e3a-96f9-46e2a7954270

![Image](assets/fr/26.webp)


**Biragaragara ko udashobora kwigera usangira aya majambo kuri internet, nk’uko ndabigira muri iyi nyigisho. Aka karorero Wallet kazokoreshwa gusa kuri Testnet kandi kazokurwaho ku mpera y'inyigisho.**


Ushobora kandi guhitamwo kwongerako passphrase BIP39 ukanda ku gasandugu "*Koresha passphrase*". Imburi: gukoresha passphrase birashobora kuba ngirakamaro cane, ariko iyo udatahura ingene ikora, birashobora gutera ingorane nyinshi. Ni co gituma ndaguhanura cane gusoma iyi nkuru ngufi y’ivyiyumviro ku bijanye n’ico kibazo:


https://planb.network/tutorials/wallet/backup/passphrase-a26a0220-806c-44b4-af14-bafdeb1adce7

Umaze kubika Mnemonic yawe na passphrase iyo ari yo yose ku buryo bugaragara, kanda kuri "*Kwemeza ububiko*".


![Image](assets/fr/27.webp)


Subira winjize amajambo yawe 12 kugira ngo wemeze ko yabitswe neza, hanyuma ukande kuri "*Create Keystore*".


![Image](assets/fr/28.webp)


Hanyuma ukande kuri "*Imbere Keystore*" kugira ngo generate imfunguruzo zawe za Wallet zivuye mu mvugo ya Mnemonic.


![Image](assets/fr/29.webp)


Fyonda kuri "*Shiraho*" kugira ngo uheze gukora Wallet.


![Image](assets/fr/30.webp)


Gushinga ijambobanga rikomeye kugira ngo ushobore gushika kuri Sparrow wallet yawe. Ni vyiza ko ubika iri jambobanga mu mucungerezi w'ijambobanga, kugira ngo ntuvyibagire. Zirikana ko iri jambobanga ata ruhara rifise mu gutuma imfunguruzo zawe ziva. Ikoreshwa gusa kugira ngo ushikire Wallet yawe uciye kuri Sparrow wallet. Rero, naho ata iri jambobanga, ijambo ryawe rya Mnemonic rizohagije kugira ngo ushikire ama bitcoins yawe ukoresheje porogarama iyo ari yo yose ihuye na BIP39.


![Image](assets/fr/31.webp)


Hot Wallet yawe ubu yararemwe. Ushobora guca ku gice ca *Kwakira Bitcoins* c'iyi nyigisho nimba udategura gukoresha Hardware Wallet na Sparrow.


## Gucungera igikoresho ca Cold Wallet


Uburyo bwa kabiri bwo gukoresha Sparrow wallet ni ukuyishiraho nk'umuyobozi wa Wallet afise Hardware Wallet. Muri iyi ntunganyo, imfunguruzo z’ibanga za Bitcoin Wallet yawe ziguma kuri Hardware Wallet gusa, mu gihe Sparrow ishobora gushika ku makuru ya bose gusa. Ubwo buryo buratanga umutekano mwinshi kuruta amasakoshi ya Hot twavuganye haruguru, kuko imfunguruzo z’ibanga zibikwa ku gikoresho kidasanzwe, akenshi kikaba gifise igice c’umutekano, kidahuye na Internet, bigatuma gifise ahantu hato cane ho gutera kuruta mudasobwa isanzwe.


Hari uburyo bubiri nyamukuru bwo gufatanya Hardware Wallet yawe na Sparrow:




- Kubera umugozi, ukoreshwa cane n’ibikoresho vy’urugero rwo kwinjira nka Trezor Safe 3 canke Ledger Nano S Plus ;
- Mu buryo bwa Air-Gap, ni ukuvuga ata nzira y’uguhuza, biciye ku ikarita ya MicroSD canke kode ya QR Exchange.


Sparrow irashigikira ubwo buryo bwose bwo guhanahana amakuru kandi irahuye n’ibikoresho vyinshi vy’amasakoshi biri kw’isoko.


Kuri iyi nyigisho, nzoba ndiko nkoresha Ledger Nano S ifise umugozi, ariko uburyo bwo kubikora burasa n’ubwo mu buryo bwa Air-Gap. Uzosanga amakuru yihariye kuri Hardware Wallet yawe mu nyigisho yayo yihariye kuri Plan ₿ Network.


Imbere yo gutangura, urabe neza ko Wallet isanzwe itunganijwe kuri Hardware Wallet yawe. Niba ukoresha umugozi, uwuhuze na mudasobwa yawe biciye ku nsinga.


Kugira ngo winjize ivyo bita "*Keystore*" (amakuru ya bose akenewe kugira ngo ushobore gucunga Wallet) muri Sparrow wallet, fyonda ku rubuga "*Dosiye*", hanyuma kuri "*Wallet nshasha*".


![Image](assets/fr/32.webp)


Izina rya Wallet yawe hanyuma ukande kuri "*Rema Wallet*". Ndaguhanura ngo winjize izina rya Hardware Wallet yawe kugira ngo uzoyimenye bitagoranye mu nyuma.


![Image](assets/fr/33.webp)


Hejuru ya Interface, hitamwo hagati ya "*Umukono umwe*" canke "*Umukono mwinshi*" Wallet. Nk'akarorero kacu, tuzotunganya ikimenyetso kimwe Wallet.


Hasi gusa, hitamwo ubwoko bw'inyandiko yo gufunga UTXOs zawe. Niba Hardware Wallet yawe ibishigikiye, ndagusavye guhitamwo "*Taproot (P2TR)*".


![Image](assets/fr/34.webp)


Igikurikira, uburyo bwo kubikora buratandukanye bivanye n’uburyo ukoresha mu guhuza. Niba ukoresha uburyo bw'Ikirere, hitamwo "*Ikirere Hardware Wallet*". Hanyuma ukurikize amabwirizwa yihariye ku gikoresho cawe.


![Image](assets/fr/35.webp)


Niba ukoresha umugozi wo guhuza, nk'uko vyagenze kuri jewe, hitamwo "*Ihuye Hardware Wallet*".


![Image](assets/fr/36.webp)


Fyonda kuri "*Scan*" kugira Sparrow imenye igikoresho cawe. Raba neza ko ifunguwe kandi ifunguwe. Ku bigereranyo bimwe bimwe, nka Ledger, uzokenera gufungura porogaramu "*Bitcoin*" kugira ngo ushobore kumenya.


![Image](assets/fr/37.webp)


Hitamwo "*Injiza ububiko bw'urufunguzo*".


![Image](assets/fr/38.webp)


Fyonda kuri "*Shiraho*" kugira ngo uheze gukora Wallet.


![Image](assets/fr/39.webp)


Gushinga ijambobanga rikomeye kugira ngo ushobore gushika kuri Sparrow wallet yawe. Iri jambobanga rizokingira imfunguruzo zawe za bose, amaderesi n’amateka y’ibikorwa vyawe. Turagusavye ko wobika mu mucungerezi w’ijambobanga. Zirikana ko iri jambobanga ata ruhara rifise mu gutuma imfunguruzo zawe ziva. Naho ataco ufise, urashobora gusubira kuronka ama bitcoins yawe ukoresheje Mnemonic yawe biciye ku porogarama iyo ari yo yose ihuye na BIP39.


![Image](assets/fr/40.webp)


Ubuyobozi bwawe Wallet ubu bwatunganijwe kuri Sparrow.


![Image](assets/fr/41.webp)


## Kwakira ama bitcoins


None ko Wallet yawe ishizwe kuri Sparrow, urashobora kwakira ama bitcoins. Gusa n'ushobora gushika ku rutonde rwa "*Kwakira*".


![Image](assets/fr/42.webp)


Sparrow izokwerekana Address ya mbere itakoreshejwe muri Wallet yawe. Ushobora kwongerako "*Label*" kuri iyi Address kugira ngo ikwibutse inkomoko y'izo satoshi muri kazoza.


![Image](assets/fr/43.webp)


Nimba uriko urakoresha Hot Wallet, Address yerekanwa irashobora gukoreshwa ubwo nyene, mu kuyikopa canke mu gucapura kode ya QR ijana na yo.


Niba ukoresha Hardware Wallet, birahambaye cane ko usuzuma Address ku gicapo c’igikoresho imbere y’uko uyikoresha. Ku bikoresho bifise amashanyarazi, huza kandi ufungure Hardware Wallet yawe, hanyuma muri Sparrow, ukande kuri "*Erekana Address*". Raba neza ko Address yerekanwa kuri Hardware Wallet yawe ihuye n'iyo yerekanwa kuri Sparrow.


![Image](assets/fr/44.webp)


Ku bakoresha Hardware Wallet Air-Gap, ukugenzura kwa Address kurahindagurika bivanye n’igikoresho. Raba inyigisho yihariye ya Plan ₿ Network kugira ngo ubone amabwirizwa nyayo.


Igihe amafaranga amaze gutangazwa n'uwuyitanga, uzobona abonetse mu gice ca "*Ibikorwa*". Ushobora kuyikanda kugira ngo ubone ibindi, nka txid.


![Image](assets/fr/45.webp)


Mu gice ca "*Adresses*", uzosanga urutonde rw'amaderesi yose y'inbox yawe. Ushobora kubona nimba vyamaze gukoreshwa be n’uko hariho ikimenyetso congereweko. *Receive*" aderesi ni izo Sparrow yerekana iyo ukanda kuri "*Receive*" kandi zigenewe kwishura. Aderesi za "*Change*" zikoreshwa kuri Exchange mu bikorwa vyawe, ni ukuvuga gusubiza igice kitakoreshejwe c'ama UTXO yawe mu vyo winjiza.


![Image](assets/fr/46.webp)


Igipande ca "*UTXOs*" kikwereka UTXOs zawe zose, ni ukuvuga ibice vya Bitcoin ufise. Ushobora kubona igitigiri ca UTXO imwe imwe be n’ikimenyetso kijana na yo.

## Wohereze ama bitcoins


None ko ufise ama satoshi makeyi muri Wallet yawe, urafise kandi uburenganzira bwo kurungika amwe. Naho hari uburyo bwinshi bwo kubikora, ndagusavye gukoresha urutonde rwa "*UTXOs*" kugira ngo ushobore kugenzura neza amafaranga ukoresha (*Coin control*), aho kuja ku rutonde rwa "*Send*" (naho urutonde rwa nyuma rushobora guhagije nimba uri umutanguzi).


![Image](assets/fr/48.webp)


Hitamwo UTXOs wipfuza gukoresha nk'inyungu z'iyi nzira, hanyuma ukande kuri "*Ohereza Ivyatowe*". Ubwo buryo buratuma ushobora guhitamwo amasoko akwiriye cane mu ma UTXO yawe, bivanye n’amahera ukoresha be n’ibimenyetso ushirako iyo uyakiriye, kugira ngo ushobore gutuma amahera yawe aguma ari ibanga. Raba neza ko umubare w’ama UTXO yatowe ari munini kuruta uwo wipfuza kohereza.


![Image](assets/fr/49.webp)


Injira Address y'uwaronse mu kibanza ca "*Pay to*". Ushobora kandi gucapura Address ukoresheje kamera yawe yo kuri interineti ukanda ku kimenyetso c’ivyo bikoresho. Buto ya "*+Add*" ishobora kugufasha kwishura amaderesi menshi mu gihe kimwe.


![Image](assets/fr/50.webp)


Wongereko ikimenyetso ku vyo ugurisha kugira ngo bikwibutse intumbero yavyo. Iryo kete rizojanye n'ibara ryawe rya Exchange.


![Image](assets/fr/51.webp)


Injira amahera azorungikwa kuri iyi Address.


![Image](assets/fr/52.webp)


Guhindura igipimo c’amahera bivanye n’ingene isoko rimeze ubu. Ivyo ushobora kubigira mu kwinjiza agaciro k’amahera yose canke mu guhindura igipimo c’amahera n’igikoresho co guterera.


![Image](assets/fr/53.webp)


Hasi muri Interface, ushobora guhitamwo hagati ya "*Ubushobozi*" na "*Ubuzima bwite*". Ku bijanye nanje, uburyo bwa "*Ubuzima bwite*" ntibuboneka, kuko mfise UTXO imwe gusa muri iyi Wallet. "*Ubushobozi*" buhuye n'ugucuruza kwa kera, mu gihe "*Ubuzima bwite*" ari ugucuruza kw'ubwoko bwa Stonewall, uburyo bwo gucuruza bukomeza ibanga ryawe mu kwigana mini-CoinJoin, ivyo bikaba bituma isesengura ry'uruzitiro rirushiriza kugorana.


![Image](assets/fr/54.webp)


Sparrow yerekana igishushanyo c’incamake kigaragaza ivyo winjiza, ivyo usohora n’amahera y’ugucuruza (menya ko amafaranga mu vy’ukuri atari umusaruro, bitandukanye n’ico iki kigereranyo kivuga). Niba unezerewe na vyose, kanda kuri "*Rema Ibikorwa*".


![Image](assets/fr/55.webp)


Ivyo bigushikana kuri paji idondora neza Elements y’ivyo waguriye. Suzuma ko amakuru yose ari ukuri, hanyuma ukande kuri "*Sozera Ibikorwa vyo Gusinya*".


![Image](assets/fr/56.webp)


Ni vyiza ko ugumana Sighash y’imbere. Kugira ngo utahure igituma, raba iyi nyigisho, aho ndabasigurira ivyo ukeneye kumenya vyose ku vyerekeye Sighash:


https://planb.network/courses/46b0ced2-9028-4a61-8fbc-3b005ee8d70f

Ku rubuga rukurikira, amahitamwo arahindagurika bivanye n’ubwoko bwa Wallet uriko urakoresha:




- Ku bijanye na Hardware Wallet Air-Gap, kanda kuri "*Show QR*" kugira ngo ugaragaze PSBT ushobora gusinya n'igikoresho cawe, hanyuma ushire PSBT yashizweko umukono muri Sparrow ukoresheje "*Scan QR*". Ihitamwo rya "*Bika Ibikorwa*" rikora mu buryo busa n'ubwo, ariko n'uguhinduranya kuri microSD ;
- Ku Hot Wallet, gusa ukande kuri "*Sinya*" hanyuma winjize ijambobanga rya Wallet kugira ngo usinye ;
- Ku Hardware Wallet ifise umurongo, ukande kandi kuri "*Sign*" kugira ngo wohereze amafaranga atashizweko umukono ku gikoresho cawe.


![Image](assets/fr/57.webp)


Ku rupapuro rwawe rwa Hardware Wallet, nusuzume urupapuro rwa Address rw’uwaronse, amahera yoherejwe be n’ivyo usaba. Niba vyose bimeze neza, genda ushireko umukono.


Iryo soko rimaze gusinywa, rizosubira kugaragara muri Sparrow, ryiteguye gutangazwa ku rubuga rwa Bitcoin kugira ngo rizokurikirane ryinjizwe mu gice. Niba vyose bimeze neza, fyonda ku "*Itangazo ry'Itangazamakuru*".

## Gucungera no gutunganya amasakoshi kuri Sparrow


Mu gice ca "*Ivyagezwe*", uzosanga amakuru arambuye kuri Wallet yawe, nka :



- Ubwoko bw’ibitabo (ivy’ikimenyetso kimwe canke multi-sig) ;
- Ubwoko bw'inyandiko ikoreshwa ;
- Izina mwashize kuri Wallet ;
- Igicapo c’urufunguzo rw’umukuru;
- Inzira y'inyuma ;
- Urufunguzo rwa bose rwagutse rwa konti.


![Image](assets/fr/60.webp)


Buto ya "*Export*" iragufasha kwohereza hanze amakuru yawe ya Wallet kugira ngo ushobore kuyakoresha mu yindi porogarama mu gihe ugumya amakuru yashizweho muri Sparrow.


Buto ya "*Ongera konti*" iragufasha kwongerako konti y'inyongera kuri Wallet yawe. Konti ihuye n’urutonde rw’amaderesi atandukanye. Ico kintu gishobora kuba ngirakamaro, nk’akarorero, nimba wipfuza gutandukanya konti y’umuntu ku giti ciwe n’iy’ubudandaji, ukoresheje ijambo rimwe Mnemonic.


Buto ya "*Advanced*" itanga uburenganzira bwo gushika ku mirongo iteye imbere, nk'uguhindura ubushakashatsi bwa Sparrow bwa Address no guhindura ijambobanga rya Wallet.


![Image](assets/fr/61.webp)


Iyo ufunze Sparrow wallet, Wallet yawe irafunga ubwayo. Igihe uzofungura iyo porogarama, idirisha rizogusaba gufungura Wallet yawe ukoresheje ijambobanga ryayo.


![Image](assets/fr/62.webp)


Niba iri dirisha ridafunguka, canke niba ushaka gufungura irindi Wallet kuri Sparrow, fyonda ku rubuga rwa "*Dosiye*" hanyuma uhitemwo "*Gufungura Wallet*".


![Image](assets/fr/63.webp)


Ivyo bizofungura File Manager yawe muri dosiye aho Sparrow ibika ama wallet yawe. Hitamwo gusa Wallet wipfuza gufungura hanyuma winjize ijambobanga kugira ngo uyifungure.


![Image](assets/fr/64.webp)


Mu "*Dosiye*" iri munsi ya "*Ivyagezwe*", uzosanga amaparametere y'ihuriro ry'urubuga rwa Bitcoin yaramaze gusuzumwa mu bice vya kera. Ushobora kandi guhindura ibipimo bitandukanye nk'igice gikoreshwa, amafaranga y'amafranga yo guhindura, n'inkomoko z'amakuru.


![Image](assets/fr/65.webp)


"*View*" tab itanga amahitamwo yo guhindura no kuronka amabwirizwa y'ingirakamaro, nka "*Refresh Wallet*", isubiramwo ubushakashatsi bw'ibikorwa vya Wallet yawe.


![Image](assets/fr/66.webp)


"*Ibikoresho*" tab ihuriza hamwe ibikoresho vyinshi biteye imbere, harimwo :



- "*Sinya/Gusuzuma Ubutumwa*" bigufasha kwemeza ko ufise Address yakira canke gusuzuma umukono.
- "*Send To Many*" itanga Interface yoroshe yo gukora amafaranga kuri aderesi nyinshi zakira icarimwe, bikaba vyiza ku gukoresha amafaranga menshi.
- "*Sweep Private Key*" igufasha kugarura amafaranga y'ibiceri vy'amahera y'ibanga yoroshe maze ukayarungika kuri Sparrow wallet yawe. Ivyo bishobora kuba ngirakamaro cane cane ku bafise ama bitcoins yo mu ntango z’imyaka ya 2010, imbere y’igihe c’ama wallets ya HD.
- "Verify Download" igenzura ubutungane n'ukuri kw'iporogarama zakuwe imbere y'uko uyishira ku gikoresho cawe.
- "*Restart In*" iragufasha guhindukira ukaja ku ma wallet yawe ku nzira za Testnet canke Signet. Ivyo birashobora kuba ngirakamaro nimba wipfuza gushika ku nzira z’igerageza zifise ibiceri bitagira agaciro nyakuri.


![Image](assets/fr/67.webp)


Ubu urazi vyose ku bijanye na porogarama ya Sparrow wallet, igikoresho ciza cane co gucunga amasakoshi yawe ya Bitcoin ku musi ku musi.


Niwaba wabonye ko iyi nyigisho ari ngirakamaro, noshima cane iyo usiga urukumu rwa Green aha hepfo. Ntutinye kubisangiza abandi ku mbuga zanyu zo guhanahana amakuru. Murakoze cane!
