Ако използвате Ledger, вероятно сте установили, че трябва да преминете през софтуера Ledger Live, поне за първоначалната конфигурация на устройството, за да проверите неговата автентичност и да инсталирате приложението Bitcoin на него. След това конфигуриране обаче много биткойнисти предпочитат да използват специализиран софтуер за управление на Bitcoin wallet, например Sparrow или Liana, а не Ledger Live. Въпреки че Ledger произвеждат отлични хардуерни портфейли, които бързо включват най-новите функции на Bitcoin, техният софтуер не е непременно адаптиран към специфичните нужди на биткойнърите. Всъщност Ledger Live включва много функции, предназначени за алткойни, докато опциите, предназначени за управление на Bitcoin wallet, са ограничени. Но проблемът с Sparrow и Liana (засега) е, че те не позволяват да инсталирате приложението Bitcoin на Ledger.


За да заобиколите необходимостта от използване на Ledger Live по време на първоначалната конфигурация на вашия Ledger, можете да използвате инструмента Bacca (или "Ledger Installer"). Този софтуер ви позволява да инсталирате и актуализирате приложението Bitcoin, да проверите автентичността на вашия Ledger и дори по-късно да актуализирате фърмуера на устройството. Bacca е създаден от Антоан Пойнсо (Darosior), разработчик на ядрото на Bitcoin в Chaincode Labs, съосновател [на Revault и Liana](https://wizardsardine.com/) и Pythcoiner.


В този урок ще ви покажа как да използвате този инструмент, така че да можете да се справите без софтуера Ledger Live завинаги и да продължите да се наслаждавате на устройствата Ledger. Работи на всички устройства: Nano S Classic, Nano S Plus, Nano X, Flex и Stax.


---
*Моля, имайте предвид, че този инструмент е сравнително нов и неговите разработчици уточняват, че все още е **в етап на тестване**. Те препоръчват да се използва само за тестови цели, а не за устройство, предназначено за разполагане на истински Bitcoin wallet, въпреки че това е възможно. В тази връзка ви препоръчвам да следвате препоръките на разработчиците на този инструмент, които са посочени [в README на тяхното хранилище в GitHub](https://github.com/darosior/ledger_installer).*

## Предварителни условия


На компютъра ви са необходими два инструмента, за да използвате Bacca:




- Git ;
- Rust.


Ако вече сте ги инсталирали, можете да пропуснете тази стъпка.


**Линукс:**


В дистрибуция на Linux Git обикновено е предварително инсталиран. За да проверите дали Git е инсталиран на вашата система, можете да въведете следната команда в терминала :


```bash
git --version
```


Ако нямате инсталиран Git в системата си, ето командата за инсталирането му в Debian :


```bash
sudo apt install git
```


Накрая, за да инсталирате средата за разработка на Rust, използвайте командата :


```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```


**Windows:**


За да инсталирате Git, отидете на [официалния уебсайт на проекта](https://git-scm.com/). Изтеглете софтуера и следвайте инструкциите за инсталиране.


![BACCA](assets/fr/01.webp)


Продължете по същия начин, за да инсталирате Rust от [официалния уебсайт](https://www.rust-lang.org/tools/install).


![BACCA](assets/fr/02.webp)


**MacOS:**


Ако Git все още не е инсталиран на вашата система, отворете терминал и изпълнете следната команда, за да го инсталирате:


```bash
git --version
```


Ако в системата ви не е инсталиран Git, ще се отвори прозорец с предложение да инсталирате Xcode, който включва Git. Просто следвайте инструкциите на екрана, за да продължите с инсталацията.


За да инсталирате Rust, изпълнете следната команда:


```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```


## Монтаж на Bacca


Отворете терминал и отидете в папката, в която искате да запишете софтуера, след което изпълнете следната команда:


```bash
git clone https://github.com/darosior/ledger_installer.git
```


Навигирайте до директорията със софтуера:


```bash
cd ledger_installer
```


След това използвайте Cargo, за да компилирате проекта и да стартирате графичния потребителски интерфейс Bacca:


```bash
cargo run -p ledger_manager_gui
```


Вече имате достъп до интерфейса на софтуера.


![BACCA](assets/fr/03.webp)


## Конфигуриране на Ledger


Преди да започнете, ако вашият Ledger е нов, уверете се, че сте настроили ПИН кода и сте запазили фразата за възстановяване. За тези първоначални стъпки не е необходимо да използвате Ledger Live. Просто свържете вашия Ledger чрез USB кабела, за да го захраните. Ако не сте сигурни как да продължите с тези две стъпки, можете да се обърнете към началото на ръководството, специфично за вашия модел:

## Използване на Bacca


Свържете Ledger към компютъра и го отключете, като използвате зададения ПИН код. Bacca трябва автоматично да открие вашия Ledger.


![BACCA](assets/fr/04.webp)


За да потвърдите автентичността на вашия Ledger, кликнете върху бутона "*Check*". За да продължите, ще трябва да оторизирате връзката на вашето устройство Ledger.


![BACCA](assets/fr/05.webp)


След това Bacca ще ви информира дали вашият Ledger е истински. Ако не е, това означава, че устройството е било компрометирано или че е фалшиво. В този случай незабавно спрете да го използвате.


![BACCA](assets/fr/06.webp)


В менюто "*Apps*" можете да разгледате списъка с приложения, които вече са инсталирани на вашия Ledger.


![BACCA](assets/fr/07.webp)


За да инсталирате приложението Bitcoin, щракнете върху "*Install*", след което разрешете инсталирането на Ledger.


![BACCA](assets/fr/08.webp)


Приложението е добре инсталирано.


![BACCA](assets/fr/09.webp)


Ако нямате инсталирана най-новата версия на приложението Bitcoin, Bacca ще покаже бутона "*Актуализирай*" вместо индикацията "*Най-новата*". Просто щракнете върху този бутон, за да актуализирате приложението.


![BACCA](assets/fr/10.webp)


Сега, когато вашият Ledger е правилно конфигуриран с последната версия на приложението Bitcoin, вие сте готови да импортирате и използвате вашия wallet в софтуер за управление, като например Sparrow или Liana, без да се налага да преминавате през Ledger Live!
