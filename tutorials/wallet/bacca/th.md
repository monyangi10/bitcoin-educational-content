หากคุณใช้ Ledger คุณอาจพบว่าคุณต้องใช้ซอฟต์แวร์ Ledger Live อย่างน้อยสำหรับการตั้งค่าอุปกรณ์เบื้องต้น เพื่อตรวจสอบความถูกต้องและติดตั้งแอปพลิเคชัน Bitcoin บนอุปกรณ์ อย่างไรก็ตาม หลังจากการตั้งค่านี้แล้ว ผู้ใช้บิทคอยน์หลายคนมักจะเลือกใช้ซอฟต์แวร์การจัดการ Bitcoin wallet เฉพาะทาง เช่น Sparrow หรือ Liana แทนที่จะใช้ Ledger Live แม้ว่า Ledger จะผลิตกระเป๋าเงินฮาร์ดแวร์ที่ยอดเยี่ยมซึ่งรวมคุณสมบัติ Bitcoin ล่าสุดอย่างรวดเร็ว แต่ซอฟต์แวร์ของพวกเขาอาจไม่จำเป็นต้องปรับให้เข้ากับความต้องการเฉพาะของผู้ใช้บิทคอยน์ จริงๆ แล้ว Ledger Live มีคุณสมบัติมากมายที่ออกแบบมาสำหรับอัลท์คอยน์ ในขณะที่ตัวเลือกที่อุทิศให้กับการจัดการ Bitcoin wallet นั้นมีจำกัด แต่ปัญหาของ Sparrow และ Liana (ในขณะนี้) คือพวกเขาไม่อนุญาตให้คุณติดตั้งแอปพลิเคชัน Bitcoin บน Ledger


ในการข้ามความจำเป็นในการใช้ Ledger Live ในระหว่างการตั้งค่าเริ่มต้นของ Ledger ของคุณ คุณสามารถใช้เครื่องมือ Bacca (หรือ "Ledger Installer") ซอฟต์แวร์นี้ช่วยให้คุณติดตั้งและอัปเดตแอปพลิเคชัน Bitcoin ตรวจสอบความถูกต้องของ Ledger ของคุณ และแม้กระทั่งอัปเดตเฟิร์มแวร์ของอุปกรณ์ในภายหลัง Bacca ถูกสร้างขึ้นโดย Antoine Poinsot (Darosior) นักพัฒนา Bitcoin Core ที่ Chaincode Labs ผู้ร่วมก่อตั้ง [ของ Revault และ Liana](https://wizardsardine.com/) และ Pythcoiner


ในบทแนะนำนี้ ฉันจะแสดงวิธีการใช้เครื่องมือนี้ให้คุณ เพื่อที่คุณจะได้ไม่ต้องใช้ซอฟต์แวร์ Ledger Live อีกต่อไป และยังคงเพลิดเพลินกับอุปกรณ์ Ledger ได้ มันใช้งานได้กับทุกอุปกรณ์: Nano S Classic, Nano S Plus, Nano X, Flex และ Stax

## ข้อกำหนดเบื้องต้น


บนคอมพิวเตอร์ของคุณ คุณจะต้องใช้เครื่องมือสองอย่างในการใช้ Bacca:




- Git ;
- Rust.


หากคุณได้ติดตั้งแล้ว คุณสามารถข้ามขั้นตอนนี้ได้


**ลินุกซ์:**


บนการกระจาย Linux โดยทั่วไป Git จะถูกติดตั้งล่วงหน้าอยู่แล้ว เพื่อที่จะตรวจสอบว่า Git ถูกติดตั้งบนระบบของคุณหรือไม่ คุณสามารถพิมพ์คำสั่งต่อไปนี้ในเทอร์มินัล :


```bash
git --version
```


หากคุณยังไม่ได้ติดตั้ง Git บนระบบของคุณ นี่คือคำสั่งสำหรับติดตั้งบน Debian :


```bash
sudo apt install git
```


สุดท้ายนี้ ในการติดตั้งสภาพแวดล้อมการพัฒนา Rust ของคุณ ให้ใช้คำสั่ง :


```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```


**Windows:**


ในการติดตั้ง Git ให้ไปที่[เว็บไซต์ทางการของโปรเจกต์](https://git-scm.com/) ดาวน์โหลดซอฟต์แวร์และทำตามคำแนะนำในการติดตั้ง


![BACCA](assets/fr/01.webp)


ดำเนินการในลักษณะเดียวกันเพื่อติดตั้ง Rust จาก [เว็บไซต์ทางการ](https://www.rust-lang.org/tools/install)


![BACCA](assets/fr/02.webp)


**MacOS:**


หาก Git ยังไม่ได้ติดตั้งบนระบบของคุณ ให้เปิดเทอร์มินัลและรันคำสั่งต่อไปนี้เพื่อติดตั้ง:


```bash
git --version
```


หาก Git ยังไม่ได้ติดตั้งในระบบของคุณ จะมีหน้าต่างเปิดขึ้นเสนอให้คุณติดตั้ง Xcode ซึ่งรวมถึง Git ด้วย เพียงทำตามคำแนะนำบนหน้าจอเพื่อดำเนินการติดตั้งต่อไป


ในการติดตั้ง Rust ให้รันคำสั่งต่อไปนี้:


```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```


## การติดตั้ง Bacca


เปิดเทอร์มินัลและไปที่โฟลเดอร์ที่คุณต้องการบันทึกซอฟต์แวร์ จากนั้นรันคำสั่งต่อไปนี้:


```bash
git clone https://github.com/darosior/ledger_installer.git
```


ไปที่ไดเรกทอรีซอฟต์แวร์:


```bash
cd ledger_installer
```


จากนั้นใช้ Cargo เพื่อคอมไพล์โปรเจ็กต์และรัน Bacca GUI:


```bash
cargo run -p ledger_manager_gui
```


คุณสามารถเข้าถึงอินเทอร์เฟซของซอฟต์แวร์ได้แล้ว


![BACCA](assets/fr/03.webp)

ก่อนที่คุณจะเริ่ม หาก Ledger ของคุณเป็นเครื่องใหม่ ตรวจสอบให้แน่ใจว่าคุณได้ตั้งค่ารหัส PIN และบันทึกวลีการกู้คืนแล้ว คุณไม่จำเป็นต้องใช้ Ledger Live สำหรับขั้นตอนเริ่มต้นเหล่านี้ เพียงเชื่อมต่อ Ledger ของคุณผ่านสาย USB เพื่อจ่ายไฟให้กับมัน หากคุณไม่แน่ใจว่าจะดำเนินการสองขั้นตอนนี้อย่างไร คุณสามารถดูที่จุดเริ่มต้นของบทแนะนำเฉพาะสำหรับรุ่นของคุณ:

## การใช้ Bacca


เชื่อมต่อ Ledger ของคุณกับคอมพิวเตอร์และปลดล็อกโดยใช้รหัส PIN ที่คุณตั้งไว้ Bacca ควรตรวจจับ Ledger ของคุณโดยอัตโนมัติ


![BACCA](assets/fr/04.webp)


ในการยืนยันความถูกต้องของ Ledger ของคุณ ให้คลิกที่ปุ่ม "*Check*" คุณจะต้องอนุญาตการเชื่อมต่อบนอุปกรณ์ Ledger ของคุณเพื่อดำเนินการต่อ


![BACCA](assets/fr/05.webp)


Bacca จะบอกคุณว่าตัว Ledger ของคุณเป็นของแท้หรือไม่ หากไม่ใช่ นั่นหมายความว่าอุปกรณ์อาจถูกบุกรุกหรือเป็นของปลอม ในกรณีนี้ ให้หยุดใช้งานทันที


![BACCA](assets/fr/06.webp)


ในเมนู "*Apps*" คุณสามารถตรวจสอบรายชื่อแอปพลิเคชันที่ติดตั้งแล้วบน Ledger ของคุณ


![BACCA](assets/fr/07.webp)


ในการติดตั้งแอปพลิเคชัน Bitcoin ให้คลิกที่ "*ติดตั้ง*" จากนั้นอนุญาตการติดตั้งบน Ledger ของคุณ


![BACCA](assets/fr/08.webp)


แอปพลิเคชันติดตั้งเรียบร้อยแล้ว


![BACCA](assets/fr/09.webp)


หากคุณไม่ได้ติดตั้งแอปพลิเคชัน Bitcoin เวอร์ชันล่าสุด Bacca จะแสดงปุ่ม "*Update*" แทนที่การแสดงผล "*Latest*". เพียงคลิกที่ปุ่มนี้เพื่ออัปเดตแอปพลิเคชัน.


![BACCA](assets/fr/10.webp)


ตอนนี้ที่ Ledger ของคุณถูกตั้งค่าอย่างถูกต้องด้วยเวอร์ชันล่าสุดของแอปพลิเคชัน Bitcoin คุณพร้อมที่จะนำเข้าและใช้ wallet ของคุณบนซอฟต์แวร์การจัดการเช่น Sparrow หรือ Liana โดยไม่ต้องผ่าน Ledger Live!


หากคุณพบว่าบทแนะนำนี้มีประโยชน์ ฉันจะรู้สึกขอบคุณมากหากคุณจะกดนิ้วโป้งสีเขียวด้านล่างนี้ อย่าลังเลที่จะแบ่งปันบทความนี้บนเครือข่ายสังคมของคุณ ขอบคุณมาก!
