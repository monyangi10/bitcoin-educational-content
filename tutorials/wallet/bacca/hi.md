यदि आप Ledger का उपयोग करते हैं, तो आपने पाया होगा कि आपको इसकी प्रामाणिकता की जांच करने और इस पर Bitcoin एप्लिकेशन इंस्टॉल करने के लिए, कम से कम प्रारंभिक डिवाइस कॉन्फ़िगरेशन के लिए Ledger लाइव सॉफ़्टवेयर से गुजरना होगा। हालाँकि, इस कॉन्फ़िगरेशन के बाद, कई बिटकॉइनर्स Ledger लाइव के बजाय स्पैरो या Liana जैसे विशेष Bitcoin Wallet प्रबंधन सॉफ़्टवेयर का उपयोग करना पसंद करते हैं। हालाँकि Ledger उत्कृष्ट हार्डवेयर वॉलेट बनाता है जो नवीनतम Bitcoin सुविधाओं को जल्दी से शामिल करता है, लेकिन उनका सॉफ़्टवेयर बिटकॉइनर्स की विशिष्ट आवश्यकताओं के लिए आवश्यक रूप से अनुकूलित नहीं है। वास्तव में, Ledger लाइव में altcoins के लिए डिज़ाइन की गई कई सुविधाएँ शामिल हैं, जबकि Bitcoin Wallet प्रबंधन के लिए समर्पित विकल्प सीमित हैं। लेकिन स्पैरो और Liana (फिलहाल) के साथ समस्या यह है कि वे आपको Ledger पर Bitcoin एप्लिकेशन इंस्टॉल करने की अनुमति नहीं देते हैं।


अपने Ledger के आरंभिक कॉन्फ़िगरेशन के दौरान Ledger लाइव का उपयोग करने की आवश्यकता को बायपास करने के लिए, आप Bacca टूल (या "Ledger इंस्टॉलर") का उपयोग कर सकते हैं। यह सॉफ़्टवेयर आपको Bitcoin एप्लिकेशन इंस्टॉल और अपडेट करने, अपने Ledger की प्रामाणिकता सत्यापित करने और बाद में डिवाइस के फ़र्मवेयर को अपडेट करने की अनुमति देता है। Bacca को एंटोनी पॉइंसोट (दारोसियोर), चेनकोड लैब्स में Bitcoin कोर डेवलपर, रेवॉल्ट और Liana के सह-संस्थापक (https://wizardsardine.com/) और पायथकॉइनर द्वारा बनाया गया था।


इस ट्यूटोरियल में, मैं आपको दिखाऊंगा कि इस टूल का उपयोग कैसे करें, ताकि आप Ledger लाइव सॉफ़्टवेयर के बिना भी काम चला सकें, और फिर भी Ledger डिवाइस का आनंद ले सकें। यह सभी डिवाइस पर काम करता है: नैनो एस क्लासिक, नैनो एस प्लस, नैनो एक्स, फ्लेक्स और स्टैक्स।


---
*कृपया ध्यान दें कि यह उपकरण काफी नया है, और इसके डेवलपर्स ने निर्दिष्ट किया है कि यह अभी भी **परीक्षण चरण में है**। वे इसे केवल परीक्षण उद्देश्यों के लिए उपयोग करने की सलाह देते हैं, न कि किसी ऐसे डिवाइस के लिए जिसका उद्देश्य वास्तविक Bitcoin Wallet को होस्ट करना है, हालाँकि ऐसा करना संभव है। इस संबंध में, मेरा सुझाव है कि आप इस उपकरण के डेवलपर्स की सिफारिशों का पालन करें, जो [उनके GitHub रिपॉजिटरी के README पर](https://github.com/darosior/ledger_installer) निर्दिष्ट हैं।*

## आवश्यक शर्तें


अपने कंप्यूटर पर Bacca का उपयोग करने के लिए आपको दो उपकरणों की आवश्यकता होगी:




- गिट;
- जीडब्ल्यू-11.


यदि आपने इन्हें पहले ही स्थापित कर लिया है, तो आप इस चरण को छोड़ सकते हैं।


**लिनक्स:**


लिनक्स डिस्ट्रीब्यूशन पर, Git आमतौर पर पहले से इंस्टॉल होता है। यह जाँचने के लिए कि आपके सिस्टम पर Git इंस्टॉल है या नहीं, आप टर्मिनल में निम्न कमांड टाइप कर सकते हैं:


```bash
git --version
```


यदि आपके सिस्टम पर Git स्थापित नहीं है, तो इसे Debian पर स्थापित करने के लिए यह कमांड है:


```bash
sudo apt install git
```


अंत में, अपने Rust विकास वातावरण को स्थापित करने के लिए, कमांड का उपयोग करें:


```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```


**विंडोज़:**


Git को इंस्टॉल करने के लिए, [प्रोजेक्ट की आधिकारिक वेबसाइट](https://git-scm.com/) पर जाएं। सॉफ़्टवेयर डाउनलोड करें और इंस्टॉलेशन निर्देशों का पालन करें।


![BACCA](assets/fr/01.webp)


[आधिकारिक वेबसाइट](https://www.Rust-lang.org/tools/install) से Rust को स्थापित करने के लिए उसी तरीके से आगे बढ़ें।


![BACCA](assets/fr/02.webp)


**मैक ओएस:**


यदि आपके सिस्टम पर Git पहले से स्थापित नहीं है, तो टर्मिनल खोलें और इसे स्थापित करने के लिए निम्नलिखित कमांड चलाएँ:


```bash
git --version
```


अगर आपके सिस्टम पर Git इंस्टॉल नहीं है, तो एक विंडो खुलेगी जिसमें आपको Xcode इंस्टॉल करने के लिए कहा जाएगा, जिसमें Git शामिल है। इंस्टॉलेशन के साथ आगे बढ़ने के लिए बस ऑन-स्क्रीन निर्देशों का पालन करें।


Rust को स्थापित करने के लिए निम्नलिखित कमांड चलाएँ:


```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```


## बक्का स्थापना


टर्मिनल खोलें और उस फ़ोल्डर पर जाएं जहां आप सॉफ़्टवेयर को सहेजना चाहते हैं, फिर निम्नलिखित कमांड चलाएँ:


```bash
git clone https://github.com/darosior/ledger_installer.git
```


सॉफ़्टवेयर निर्देशिका पर जाएँ:


```bash
cd ledger_installer
```


फिर प्रोजेक्ट को संकलित करने और Bacca GUI चलाने के लिए Cargo का उपयोग करें:


```bash
cargo run -p ledger_manager_gui
```


अब आपके पास Interface सॉफ्टवेयर तक पहुंच है।


![BACCA](assets/fr/03.webp)


## Ledger को कॉन्फ़िगर करना


शुरू करने से पहले, अगर आपका Ledger नया है, तो सुनिश्चित करें कि आपने पिन कोड सेट कर लिया है और रिकवरी वाक्यांश सहेज लिया है। इन शुरुआती चरणों के लिए आपको Ledger लाइव की ज़रूरत नहीं है। बस अपने Ledger को USB केबल के ज़रिए कनेक्ट करें ताकि उसे पावर मिल सके। अगर आपको नहीं पता कि इन दो चरणों को कैसे आगे बढ़ाया जाए, तो आप अपने मॉडल के लिए खास ट्यूटोरियल की शुरुआत देख सकते हैं:

## बक्का का उपयोग


अपने Ledger को अपने कंप्यूटर से कनेक्ट करें और अपने द्वारा सेट किए गए पिन कोड का उपयोग करके इसे अनलॉक करें। Bacca को स्वचालित रूप से आपके Ledger का पता लगाना चाहिए।


![BACCA](assets/fr/04.webp)


अपने Ledger की प्रामाणिकता की पुष्टि करने के लिए, "*चेक*" बटन पर क्लिक करें। जारी रखने के लिए आपको अपने Ledger डिवाइस पर कनेक्शन को अधिकृत करना होगा।


![BACCA](assets/fr/05.webp)


इसके बाद Bacca आपको सूचित करेगा कि आपका Ledger असली है या नहीं। अगर यह असली नहीं है, तो इसका मतलब है कि डिवाइस के साथ छेड़छाड़ की गई है या यह नकली है। इस मामले में, इसका उपयोग तुरंत बंद कर दें।


![BACCA](assets/fr/06.webp)


"*ऐप्स*" मेनू में, आप अपने Ledger पर पहले से इंस्टॉल किए गए एप्लिकेशन की सूची देख सकते हैं।


![BACCA](assets/fr/07.webp)


Bitcoin एप्लिकेशन को इंस्टॉल करने के लिए, "*इंस्टॉल करें*" पर क्लिक करें, फिर अपने Ledger पर इंस्टॉलेशन को अधिकृत करें।


![BACCA](assets/fr/08.webp)


एप्लीकेशन अच्छी तरह से स्थापित है.


![BACCA](assets/fr/09.webp)


यदि आपके पास Bitcoin एप्लिकेशन का नवीनतम संस्करण इंस्टॉल नहीं है, तो Bacca "*नवीनतम*" संकेत के बजाय "*अपडेट*" बटन प्रदर्शित करेगा। एप्लिकेशन को अपडेट करने के लिए बस इस बटन पर क्लिक करें।


![BACCA](assets/fr/10.webp)


अब जबकि आपका Ledger, Bitcoin अनुप्रयोग के नवीनतम संस्करण के साथ सही ढंग से कॉन्फ़िगर हो गया है, तो आप अपने Wallet को Ledger लाइव के बिना, स्पैरो या Liana जैसे प्रबंधन सॉफ्टवेयर पर आयात और उपयोग करने के लिए तैयार हैं!


अगर आपको यह ट्यूटोरियल उपयोगी लगा, तो मैं आभारी रहूंगा यदि आप नीचे Green थंब छोड़ दें। इस लेख को अपने सोशल नेटवर्क पर शेयर करने में संकोच न करें। आपका बहुत-बहुत धन्यवाद!


मैं आपको GnuPG पर इस ट्यूटोरियल को देखने की भी सलाह देता हूँ, जो बताता है कि अपने सॉफ़्टवेयर को इंस्टॉल करने से पहले उसकी अखंडता और प्रामाणिकता की जाँच कैसे करें। यह एक महत्वपूर्ण अभ्यास है, खासकर जब Wallet प्रबंधन सॉफ़्टवेयर जैसे Liana या Sparrow को इंस्टॉल किया जाता है:
