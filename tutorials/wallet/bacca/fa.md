اگر از Ledger استفاده می‌کنید، احتمالاً متوجه شده‌اید که باید حداقل برای پیکربندی اولیه دستگاه، از نرم‌افزار Ledger Live استفاده کنید تا اصالت آن را بررسی کرده و برنامه Bitcoin را روی آن نصب کنید. با این حال، پس از این پیکربندی، بسیاری از بیت‌کوینرها ترجیح می‌دهند به جای Ledger Live از نرم‌افزار مدیریت تخصصی Bitcoin Wallet مانند Sparrow یا Liana استفاده کنند. اگرچه Ledger کیف پول‌های سخت‌افزاری عالی تولید می‌کند که به سرعت شامل آخرین ویژگی‌های Bitcoin می‌شوند، اما نرم‌افزار آن‌ها لزوماً با نیازهای خاص بیت‌کوینرها سازگار نیست. در واقع، Ledger Live شامل ویژگی‌های زیادی است که برای آلت‌کوین‌ها طراحی شده‌اند، در حالی که گزینه‌های اختصاص داده شده به مدیریت Bitcoin Wallet محدود هستند. اما مشکل Sparrow و Liana (در حال حاضر) این است که اجازه نصب برنامه Bitcoin روی Ledger را نمی‌دهند.


برای دور زدن نیاز به استفاده از Ledger Live در طول پیکربندی اولیه Ledger خود، می‌توانید از ابزار Bacca (یا "Ledger Installer") استفاده کنید. این نرم‌افزار به شما اجازه می‌دهد تا برنامه Bitcoin را نصب و به‌روزرسانی کنید، اصالت Ledger خود را تأیید کنید و حتی بعداً فریمور دستگاه را به‌روزرسانی کنید. Bacca توسط آنتوان پوینسو (Darosior)، توسعه‌دهنده اصلی Bitcoin در Chaincode Labs، هم‌بنیان‌گذار [Revault و Liana](https://wizardsardine.com/) و Pythcoiner ایجاد شده است.


در این آموزش، به شما نشان خواهم داد که چگونه از این ابزار استفاده کنید، تا بتوانید برای همیشه بدون نرم‌افزار Ledger Live کار کنید و همچنان از دستگاه‌های Ledger لذت ببرید. این ابزار بر روی تمام دستگاه‌ها کار می‌کند: Nano S Classic، Nano S Plus، Nano X، Flex و Stax.

## پیش‌نیازها


در رایانه خود، برای استفاده از Bacca به دو ابزار نیاز خواهید داشت:




- گیت ;
- Rust.


اگر قبلاً آن‌ها را نصب کرده‌اید، می‌توانید از این مرحله عبور کنید.


**لینوکس:**


در یک توزیع لینوکس، Git به طور کلی از پیش نصب شده است. برای بررسی اینکه آیا Git روی سیستم شما نصب شده است، می‌توانید دستور زیر را در ترمینال تایپ کنید:


```bash
git --version
```


اگر Git روی سیستم شما نصب نیست، در اینجا دستور نصب آن روی یک Debian آمده است:


```bash
sudo apt install git
```


در نهایت، برای نصب محیط توسعه Rust خود، از دستور زیر استفاده کنید:


```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```


**ویندوز:**


برای نصب Git، به [وب‌سایت رسمی پروژه](https://git-scm.com/) بروید. نرم‌افزار را دانلود کرده و دستورالعمل‌های نصب را دنبال کنید.


![BACCA](assets/fr/01.webp)


به همان روش برای نصب Rust از [وب‌سایت رسمی](https://www.Rust-lang.org/tools/install) اقدام کنید.


![BACCA](assets/fr/02.webp)


**مک‌اواس:**


اگر Git قبلاً روی سیستم شما نصب نشده است، یک ترمینال باز کنید و فرمان زیر را برای نصب آن اجرا کنید:


```bash
git --version
```


اگر Git روی سیستم شما نصب نشده باشد، یک پنجره باز می‌شود که به شما پیشنهاد می‌دهد Xcode را نصب کنید، که شامل Git است. به سادگی دستورالعمل‌های روی صفحه را دنبال کنید تا نصب ادامه یابد.


برای نصب Rust، فرمان زیر را اجرا کنید:


```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```


## نصب Bacca


یک ترمینال باز کنید و به پوشه‌ای که می‌خواهید نرم‌افزار را در آن ذخیره کنید بروید، سپس فرمان زیر را اجرا کنید:


```bash
git clone https://github.com/darosior/ledger_installer.git
```


به دایرکتوری نرم‌افزار بروید:


```bash
cd ledger_installer
```


سپس از Cargo برای کامپایل پروژه و اجرای رابط کاربری Bacca استفاده کنید:


```bash
cargo run -p ledger_manager_gui
```


شما اکنون به نرم‌افزار Interface دسترسی دارید.


![BACCA](assets/fr/03.webp)


## پیکربندی Ledger


قبل از شروع، اگر Ledger شما جدید است، مطمئن شوید که کد PIN را تنظیم کرده و عبارت بازیابی را ذخیره کرده‌اید. برای این مراحل اولیه نیازی به Ledger Live ندارید. کافی است Ledger خود را از طریق کابل USB به برق متصل کنید. اگر مطمئن نیستید که چگونه با این دو مرحله پیش بروید، می‌توانید به ابتدای آموزش مخصوص مدل خود مراجعه کنید:

## استفاده از Bacca


Ledger خود را به رایانه متصل کرده و با استفاده از کد PIN که تنظیم کرده‌اید، آن را باز کنید. Bacca باید به طور خودکار Ledger شما را شناسایی کند.


![BACCA](assets/fr/04.webp)


برای تأیید اصالت Ledger خود، روی دکمه "*بررسی*" کلیک کنید. برای ادامه، باید اتصال را در دستگاه Ledger خود تأیید کنید.


![BACCA](assets/fr/05.webp)


سپس باکا به شما اطلاع خواهد داد که آیا Ledger شما اصل است یا خیر. اگر اینطور نیست، این نشان می‌دهد که یا دستگاه به خطر افتاده است یا تقلبی است. در این صورت، بلافاصله استفاده از آن را متوقف کنید.


![BACCA](assets/fr/06.webp)


در منوی "*برنامه‌ها*" می‌توانید لیست برنامه‌های از پیش نصب‌شده روی Ledger خود را مشاهده کنید.


![BACCA](assets/fr/07.webp)


برای نصب برنامه Bitcoin، روی "*Install*" کلیک کنید، سپس نصب را روی Ledger خود تأیید کنید.


![BACCA](assets/fr/08.webp)


برنامه به خوبی نصب شده است.


![BACCA](assets/fr/09.webp)


اگر آخرین نسخه برنامه Bitcoin نصب نشده باشد، Bacca به جای نمایش "*Latest*" دکمه "*Update*" را نشان خواهد داد. به سادگی روی این دکمه کلیک کنید تا برنامه به‌روزرسانی شود.


![BACCA](assets/fr/10.webp)


اکنون که Ledger شما به‌درستی با آخرین نسخه از برنامه Bitcoin پیکربندی شده است، آماده‌اید تا Wallet خود را بر روی نرم‌افزارهای مدیریت مانند Sparrow یا Liana وارد و استفاده کنید، بدون اینکه نیاز باشد از Ledger Live عبور کنید!
