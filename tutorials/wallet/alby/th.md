## Alby ส่วนขยาย



ส่วนขยาย Alby เป็นเครื่องมือที่ช่วยให้เว็บเบราว์เซอร์ของคุณสามารถโต้ตอบกับเครือข่าย Bitcoin และชั้น Lightning Network ได้อย่างง่ายดายและปลอดภัย มีลักษณะเด่นสามประการ:




- Lightning Network wallet: เชื่อมโยงโหนดหรือบัญชี Alby ของคุณเพื่อส่งและรับบิตคอยน์อย่างรวดเร็วและราคาถูกผ่านเลเยอร์ Lightning Network
- การชำระเงินที่ราบรื่นผ่านเว็บ: มันขจัดความจำเป็นในการสแกน QR โค้ดหรือสลับระหว่างแอปพลิเคชันสำหรับการชำระเงินด้วยบิตคอยน์บนเว็บไซต์ที่รองรับ Lightning มันช่วยให้การทำธุรกรรมเป็นไปอย่างราบรื่นด้วยการคลิกเพียงครั้งเดียว หรือโดยไม่ต้องยืนยันหากคุณได้ตั้งงบประมาณไว้แล้ว
- ผู้จัดการ Nostr: ส่วนขยายนี้จัดการคีย์ Nostr ของคุณ ทำให้ง่ายต่อการเชื่อมต่อและโต้ตอบกับแอปพลิเคชัน Nostr ที่ทำหน้าที่เป็นผู้ลงนามที่ปลอดภัยโดยไม่ต้องเปิดเผยคีย์ส่วนตัวของคุณต่อทุกแพลตฟอร์ม

## เชื่อมต่อกับส่วนขยาย



ในบทแนะนำนี้ เราจะใช้ส่วนขยาย Alby ในเบราว์เซอร์ Firefox ภายใต้ระบบปฏิบัติการ Ubuntu อย่างไรก็ตาม มันยังสามารถใช้ได้บน Windows และกับเบราว์เซอร์เช่น Chrome



คุณสามารถเพิ่มส่วนขยาย Alby ไปยังเบราว์เซอร์ของคุณได้โดยไปที่ร้านส่วนขยาย [Firefox] (https://addons.mozilla.org/fr/firefox/addon/alby/) หรือร้านส่วนขยาย [Chrome] (https://chromewebstore.google.com/detail/alby-bitcoin-wallet-for-l/iokeahhehimjnekafflcihljlcjccdbe)

## การชำระเงินครั้งแรกของคุณ



เมื่อเข้าสู่ระบบแล้ว คุณสามารถคลิกที่ส่วนขยาย Alby ในแถบเครื่องมือเพื่อเข้าถึงพอร์ตโฟลิโอของคุณ



![buzzin](assets/fr/06.webp)



เมื่อคุณสร้างบัญชี Alby ของคุณแล้ว คุณจะต้องเชื่อมต่อกับ wallet เพื่อใช้จ่าย satoshis ในการเชื่อมโยง bitcoin wallet กับบัญชี Alby ของคุณ เราแนะนำให้คุณใช้โหนด Alby Hub ซึ่งคุณสามารถตั้งค่าได้บนคอมพิวเตอร์ของคุณหรือสมัครแผนที่เสนอโดย Alby



![hubplan](assets/fr/13.webp)

ในการสร้างโหนด Alby ของคุณเอง เราขอแนะนำบทแนะนำ Alby Hub ของเรา
