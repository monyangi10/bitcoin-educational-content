## Muun Wallet - คู่มือผู้ใช้ฉบับสมบูรณ์


คู่มือผู้ใช้ฉบับสมบูรณ์ (พร้อมภาพหน้าจอ) สำหรับแอปพลิเคชัน Muun; แอปพลิเคชันมือถือที่ใช้งานง่าย Bitcoin wallet ที่ช่วยให้คุณทำธุรกรรมบน Lightning Network ได้


![image](assets/0.5.webp)


### ดาวน์โหลด Muun และสร้าง wallet ของคุณ


ก่อนอื่น คุณต้องดาวน์โหลดแอปพลิเคชันมือถือ ซึ่งมีให้บริการทั้งบน iOS และ Android อย่าลืมดาวน์โหลดเวอร์ชันที่ถูกต้องเสมอ บางครั้งอาจมีแอปพลิเคชันปลอมในตลาด ฉันแนะนำให้ค้นหาเว็บไซต์ทางการของ Muun wallet ซึ่งคือ https://muun.com/ และใช้ลิงก์สำหรับระบบที่คุณเลือก (iOS/Android) จากนั้นคุณจะมั่นใจได้ว่าคุณกำลังใช้แอปพลิเคชันทางการ


![image](assets/1.webp)


เมื่อคุณเปิดแอปพลิเคชัน คุณจะมีตัวเลือกในการสร้าง wallet ใหม่หรือกู้คืนที่มีอยู่แล้ว มาเริ่มต้นด้วยการสร้าง wallet ใหม่กันก่อน ฉันจะแสดงขั้นตอนสำหรับการกู้คืน wallet ในภายหลัง กด "สร้าง wallet ใหม่"


![image](assets/2.webp)


Muun wallet จากนั้นขอให้คุณสร้างหมายเลขประจำตัวส่วนบุคคล (PIN) สี่หลัก ด้วย PIN คุณจะเพิ่มความปลอดภัยให้กับ wallet ของคุณในกรณีที่มีผู้ไม่ประสงค์ดีขโมยโทรศัพท์ของคุณและส่งผลให้บิตคอยน์ของคุณถูกขโมยไปด้วย


![image](assets/3.webp)


ที่นี่เลย, แอปพลิเคชันได้สร้าง wallet ใหม่เอี่ยมที่จะกลายเป็นหน้าแรกแล้ว ตอนนี้คุณจำเป็นต้องรักษาข้อมูลการกู้คืน wallet ที่เกี่ยวข้องให้ปลอดภัยก่อนที่จะส่งเงินใดๆ ไปยังมัน เพราะนั่นจะเป็นการปฏิบัติที่เสี่ยง


![image](assets/4.webp)


### การสำรองข้อมูลคีย์


กดที่กล่อง "สำรองข้อมูล wallet ของคุณ" แล้วคุณจะถูกเปลี่ยนเส้นทางไปยังแท็บ "ความปลอดภัย" กระบวนการสำรองข้อมูลของ Muun ถูกแบ่งออกเป็นสามขั้นตอนอย่างสะดวก ไม่จำเป็นต้องทำทั้งสามขั้นตอนให้ครบ แต่การรวมกันของขั้นตอนเหล่านี้จะให้การป้องกันสูงสุด


![image](assets/5.webp)


ตัวเลือกแรกช่วยให้คุณเชื่อมต่อ wallet ของคุณกับที่อยู่อีเมล นอกเหนือจากการป้องกันด้วยรหัสผ่าน ตัวเลือกนี้เป็นทางเลือกและสามารถข้ามได้โดยไม่มีปัญหา หากคุณต้องการใช้ ให้กด "1: สำรองข้อมูล wallet ของคุณ" จากนั้นกด "เริ่ม" บนหน้าจอถัดไปและป้อนที่อยู่อีเมล หน้าจอถัดไปจะแจ้งให้คุณทราบว่าคุณจำเป็นต้องยืนยันอีเมลโดยการเข้าถึงกล่องจดหมายของคุณและคลิกที่ลิงก์ที่ให้ไว้ในอีเมล Muun


![image](assets/6.webp)


เมื่อยืนยันอีเมลแล้ว คุณจะได้รับการแจ้งให้สร้างรหัสผ่าน จากนั้นคุณจะต้องทำเครื่องหมายในสองช่องเพื่อระบุว่าคุณเข้าใจว่าการกู้คืน wallet หากจำเป็น จะต้องใช้ที่อยู่อีเมลและรหัสผ่านที่คุณเพิ่งเลือก ซึ่งแตกต่างจากโปรแกรมแบบดั้งเดิมที่อนุญาตให้คุณรีเซ็ตรหัสผ่านในกรณีที่สูญหายหรือหลงลืม ดังนั้นโปรดตรวจสอบให้แน่ใจว่าคุณได้จดบันทึกทุกอย่างแล้ว


![image](assets/7.webp)

![image](assets/8.webp)


แท็บ "ความปลอดภัย" ตอนนี้แสดงว่าคุณมีการสำรองข้อมูลพื้นฐานแล้ว คุณสามารถกลับไปที่แท็บ "Wallet" และใช้แอปพลิเคชันเพื่อทำธุรกรรมให้เสร็จสมบูรณ์ (ฟังก์ชันเหล่านี้จะมีรายละเอียดเพิ่มเติมในคู่มือนี้) โดยรู้ว่า wallet สามารถกู้คืนได้แล้ว อย่างไรก็ตาม ฉันแนะนำให้ใช้ตัวเลือกความปลอดภัย #2 เพื่อ generate รหัสสำรองเพิ่มเติม ในกรณีที่รหัสผ่านที่ป้อนในตัวเลือก #1 ถูกละเมิดหรือหากคุณไม่ต้องการใช้ตัวเลือกการกู้คืนทางอีเมล


![image](assets/9.webp)


ตัวเลือก "การสำรองข้อมูลทางเลือก" ของ Muun นั้นคล้ายกับวลีช่วยจำที่ใช้โดยแอปพลิเคชัน wallet หลายตัวที่ผู้ใช้ Bitcoin หลายคนคุ้นเคย กด "เริ่ม" เพื่อแสดงรหัสการกู้คืนของคุณและจดลงบนกระดาษ (แอปพลิเคชันจะเซ็นเซอร์ภาพหน้าจอในหน้าที่แสดงรหัส) เมื่อจดแล้ว ให้เปรียบเทียบกับรหัสที่แสดงบนหน้าจอเพราะคุณจะต้องพิมพ์ซ้ำในแอปพลิเคชันเพื่อยืนยันความถูกต้องของมัน


อีกครั้ง Muun ขอให้คุณยืนยันความเข้าใจเกี่ยวกับปัญหานี้ ซึ่งก็คือคุณจะต้องใช้รหัส 32 ตัวอักษรนี้ในกรณีที่คุณลืมรหัสผ่านที่ตั้งไว้ก่อนหน้านี้


การสำรองข้อมูล wallet ตอนนี้มีความปลอดภัยอย่างมากตามมาตรฐานสมัยใหม่ที่เรารู้จัก อย่างไรก็ตาม แอปพลิเคชัน Muun มีตัวเลือกความปลอดภัยที่สามที่เรียกว่า "ชุดฉุกเฉิน" การสร้างชุดฉุกเฉินจะช่วยให้คุณกู้คืน wallet ของคุณได้โดยไม่จำเป็นต้องผ่าน Muun กล่าวอีกนัยหนึ่งคือโดยใช้ซอฟต์แวร์ Bitcoin wallet ที่แตกต่างจาก Muun


![image](assets/10.webp)


หลังจากกด "สร้างชุดฉุกเฉิน" คุณจะได้รับคำอธิบายว่าชุดนี้มาในรูปแบบของเอกสาร PDF ที่มีข้อมูล & คำแนะนำเกี่ยวกับการโอนเงินของคุณอย่างอิสระ ชุดนี้สามารถเก็บไว้บนคลาวด์ได้โดยไม่ต้องกังวลเนื่องจากต้องใช้ "รหัสกู้คืน" ของคุณซึ่งไม่ได้รวมอยู่ในเอกสาร ปัดหน้าจอเพื่อเข้าถึงหน้าการสร้างชุด

มีสามตัวเลือกที่คุณสามารถเลือกได้:

- สำรองข้อมูลด้วยตนเองด้วยแอปพลิเคชันในเครื่องบนอุปกรณ์ของคุณ

### กู้คืนด้วยที่อยู่อีเมล


หากคุณใช้ตัวเลือกการสำรองข้อมูล #1 ของ Muun ให้ป้อนที่อยู่อีเมลที่เลือกในขณะนั้น เนื่องจากตัวเลือกนี้เป็นทางเลือก คุณยังสามารถเลือกดำเนินการต่อด้วยรหัสการกู้คืน ซึ่งเป็นตัวเลือก #2 ที่เสนอโดย Muun ได้เช่นกัน มาดูตัวเลือกอีเมลกันก่อน


![image](assets/15.webp)


เมื่อคุณป้อนที่อยู่อีเมลของคุณแล้ว Muun จะแจ้งให้คุณทราบว่าได้ส่งอีเมลถึงคุณแล้วและคุณจำเป็นต้องเข้าถึงอีเมลนั้นเพื่ออนุญาตการกู้คืน wallet ตรวจสอบกล่องจดหมายของคุณ (รวมถึงส่วนสแปม) และใช้ลิงก์ที่ให้ไว้ในอีเมลของ Muun คุณจะถูกเปลี่ยนเส้นทางไปยังแอปพลิเคชันซึ่งหน้าจอจะขอให้คุณป้อนรหัสผ่านที่เกี่ยวข้องกับที่อยู่อีเมลที่ลงทะเบียนไว้


![image](assets/16.webp)


ขั้นตอนสุดท้ายคือการสร้างหมายเลขประจำตัวส่วนบุคคล จากนั้นคุณจะกลับไปยังหน้าหลักของ wallet ที่คุ้นเคย ซึ่งจะแสดงยอดคงเหลือที่เกี่ยวข้องกับมัน


![image](assets/17.webp)


### การใช้ "รหัสกู้คืน"


เมื่อกู้คืนการเข้าถึง wallet ที่มีอยู่ คุณสามารถเลือกใช้รหัสการกู้คืน ("Recovery Code" ตามที่กำหนดโดย Muun) ที่คุณได้จดบันทึกไว้ก่อนหน้านี้หากคุณเลือกตัวเลือกสำรอง #2


กระบวนการนี้คล้ายกับที่อธิบายไว้ในส่วนก่อนหน้า; การกู้คืนผ่านอีเมล เพียงเลือกใช้ตัวเลือก "กู้คืนด้วยรหัสการกู้คืน" และป้อนในช่องที่แสดงบนหน้าจอ หาก wallet ของคุณได้รับการสำรองข้อมูลด้วยอีเมลนอกเหนือจากรหัสการกู้คืน Muun จะขอให้คุณตรวจสอบกล่องจดหมายเพื่ออนุมัติกระบวนการกู้คืน ซึ่งคุณสามารถทำให้เสร็จสิ้นได้เมื่อคุณกลับไปที่แอปพลิเคชันหลังจากคลิกที่ลิงก์ที่ให้ไว้ อีกครั้ง คุณต้องสร้างหมายเลขประจำตัวส่วนบุคคล เพียงเท่านี้ คุณจะสามารถเข้าถึง wallet ของคุณได้อีกครั้ง


### การกู้คืนโดยใช้ชุดฉุกเฉิน


ในการกู้คืน wallet ของคุณโดยไม่ใช้แอปพลิเคชัน Muun Wallet คุณจะต้องใช้ชุดฉุกเฉินของคุณ ซึ่งเป็นตัวเลือกการกู้คืนที่สามที่เสนอโดย Muun ตัวเลือกนี้ช่วยให้คุณสามารถส่งเงินที่ถืออยู่ใน Muun wallet ของคุณไปยังที่อยู่ Bitcoin อื่น ๆ ได้ ดังนั้นโปรดตรวจสอบให้แน่ใจว่าคุณมี wallet ทางเลือกที่สามารถ generate ที่อยู่ที่คุณจะส่งเงินไปยังที่นั่น


เข้าถึงเอกสาร PDF ที่คุณบันทึกไว้เมื่อสร้างชุดอุปกรณ์ เอกสารนี้มีคำแนะนำที่จำเป็นในการกู้คืน wallet ของคุณ โปรดทราบว่าฟีเจอร์นี้ต้องใช้คอมพิวเตอร์เดสก์ท็อปหรือแล็ปท็อป เนื่องจากคุณจำเป็นต้องดาวน์โหลดสคริปต์ที่สร้างโดยทีมพัฒนา Muun ลิงก์นี้รวมอยู่ในอีเมลแล้ว แต่ฉันจะแชร์ที่นี่อีกครั้ง: https://github.com/muun/recovery


ชุดฉุกเฉินมาพร้อมกับรหัสยืนยัน ซึ่งคุณได้ใช้เพื่อยืนยันการสร้างชุดแล้ว รวมถึงกุญแจสองดอก กุญแจเหล่านี้จะจำเป็นเมื่อคุณเปิดใช้งานสคริปต์กู้คืน Muun ดังนั้นให้แน่ใจว่าคุณมีมันอยู่ในมือระหว่างการดำเนินการ

## การรับธุรกรรม


### Bitcoin แท็บ


เราจะครอบคลุมส่วน "รับ" ของ Muun wallet และฟังก์ชันต่างๆ ของมัน หน้าหลักของแอปพลิเคชันของคุณคือแท็บ "Wallet" ยอดคงเหลือของคุณจะแสดงอยู่ตรงกลาง และคุณสามารถแตะที่มันเพื่อสลับระหว่างการซ่อนจำนวนเงินและการแสดงจำนวนเงินได้ เราจะพูดถึงการตั้งค่าทั้งหมดของแอปพลิเคชันในบทความนี้ภายหลัง ตอนนี้ให้กด "รับ" เพื่อสำรวจฟังก์ชันนี้


![image](assets/22.webp)


ในหน้านี้ คุณสามารถเลือกที่จะรับธุรกรรมบนเครือข่าย Bitcoin และ Lightning ได้ ที่อยู่ใหม่ (และรหัส QR ที่เกี่ยวข้อง) ที่ตรงกับเครือข่ายที่ต้องการจะแสดงขึ้นมา โดยค่าเริ่มต้น ที่อยู่ Bitcoin จะแสดงเมื่อคุณมาถึงหน้าจอ "รับ" การแตะที่รหัส QR จะทำให้ที่อยู่ถูกคัดลอกไปยังคลิปบอร์ดของอุปกรณ์ของคุณ คุณสามารถแชร์ที่อยู่ไปยังแอปพลิเคชันอื่นได้ง่ายๆ ด้วยปุ่ม "แชร์" และคุณยังสามารถคัดลอกที่อยู่ด้วยปุ่ม "คัดลอก" การกดไอคอนรูปตาที่ท้ายที่อยู่จะแสดงที่อยู่ทั้งหมด ทำให้คุณสามารถเปรียบเทียบกับที่อยู่ที่คัดลอกไปยังคลิปบอร์ดระหว่างการแชร์ได้


![image](assets/23.webp)


ข้อมูลนี้มีทุกอย่างที่คุณต้องการเพื่อรับธุรกรรมบนเครือข่าย Bitcoin นอกจากนี้ Muun ยังมีตัวเลือกการปรับแต่งบางอย่างภายใต้เมนู "การตั้งค่า Address" อันดับแรก คุณสามารถเพิ่มจำนวนเงินในคำอธิบายที่อยู่ได้ ประการที่สอง คุณสามารถเลือกใช้ที่อยู่ Segwit (ตัวเลือกเริ่มต้น) หรือที่อยู่แบบดั้งเดิม (legacy)


![image](assets/24.webp)


โดยการกด "Add +" คุณสามารถเพิ่มจำนวนเงินเฉพาะไปยังที่อยู่ได้ ทำให้ง่ายขึ้นสำหรับฝ่ายที่ส่ง ตัวเลือกนี้เป็นทางเลือก โปรดทราบว่าเมื่อป้อนจำนวนเงินแล้ว ปุ่ม "Copy" ในหน้าก่อนหน้าจะเพิ่มข้อมูลไปยังที่อยู่ที่คัดลอก ("bitcoin:" เป็นคำนำหน้า ตามด้วยจำนวนเงินเป็นคำต่อท้าย) เพื่อหลีกเลี่ยงการปรับเปลี่ยนนี้ในทันที ให้กด QR code โดยตรงเพื่อคัดลอกที่อยู่ ข้อมูลจำนวนเงินจะยังคงยึดติดกับมัน นอกจากนี้ แอปพลิเคชันยังอนุญาตให้คุณเลือกป้อนจำนวนเงินในสกุลเงินที่คุณเลือก ทำให้กระบวนการแปลงเป็น BTC ง่ายขึ้น


![image](assets/25.webp)


เกี่ยวกับการเลือกประเภทที่อยู่, Segwit หรือ Legacy, ฉันแนะนำให้คง Segwit ไว้ ที่อยู่ประเภทนี้ (เริ่มต้นด้วย "bc1") ช่วยลดขนาดของข้อมูลการทำธุรกรรมและลดค่าธรรมเนียมการทำธุรกรรมที่เกี่ยวข้อง อย่างไรก็ตาม อาจจำเป็นต้องใช้ระบบ "Legacy" (ที่อยู่เริ่มต้นด้วย "3") ในกรณีที่ wallet หรือซอฟต์แวร์ไม่เข้ากันกับที่อยู่ Segwit ดังนั้นจึงเป็นสิ่งสำคัญที่จะต้องรู้วิธีแยกแยะระหว่างสองประเภทนี้


![image](assets/26.webp)


## แท็บสายฟ้า


ในการรับธุรกรรมผ่านเครือข่าย Lightning คุณจำเป็นต้องแตะที่แท็บที่มีชื่อเดียวกันที่ด้านบนของหน้าจอ ขณะนี้จะมีการแสดงรหัส QR ที่มีที่อยู่ Lightning ซึ่งคุณสามารถคัดลอกและแชร์ได้ในลักษณะเดียวกับที่อยู่ Bitcoin ที่กล่าวถึงก่อนหน้านี้ในคู่มือนี้ ฉันขอเตือนคุณว่าเครือข่าย Lightning ช่วยให้คุณเพลิดเพลินกับความเร็วในการทำธุรกรรมที่เกือบจะทันทีนอกเหนือจากค่าธรรมเนียมการทำธุรกรรมที่เป็นเพียงเศษเสี้ยวของค่าธรรมเนียมบนเครือข่าย Bitcoin


![image](assets/27.webp)


ตัวเลือกการปรับแต่งสามารถพบได้ภายใต้เมนู "Invoice settings" ที่นี่คุณสามารถแก้ไขจำนวนเงินที่เกี่ยวข้องกับที่อยู่โดยการแตะที่ "Add +" จากประสบการณ์ของฉันกับเครือข่าย Lightning ฉันเชื่อว่ามันดีกว่าที่จะใส่จำนวนเงินเมื่อสร้างธุรกรรมเพราะกระเป๋าเงินหลายตัวไม่ตอบสนองดีต่อใบแจ้งหนี้ที่ว่างเปล่า คุณจะสังเกตเห็นว่ามีตัวจับเวลาหมดอายุในเมนูนี้ ในแอปพลิเคชันนี้ ตัวจับเวลาถูกตั้งไว้ที่ 60 นาที หลังจากนั้นที่อยู่จะไม่ถูกต้อง โปรดทราบว่า Muun สร้างที่อยู่ Lightning ใหม่ทุกครั้งที่คุณทำการเปลี่ยนแปลงจำนวนเงินหรือเมื่อคุณออกจากหน้าและกลับมาใหม่


![image](assets/28.webp)


## การใช้ฟังก์ชัน LNURL


Muun wallet มีความสามารถในการใช้ LNURL เพื่อรับธุรกรรม ฟังก์ชันนี้ซึ่งเปิดใช้งานโดยการแตะที่สัญลักษณ์การสแกนสี่เหลี่ยมที่มุมขวาบนของหน้า มีข้อดีหลายประการ รวมถึงการหลีกเลี่ยงความจำเป็นในการแชร์ใบแจ้งหนี้เพื่อรับธุรกรรม แทนที่จะทำเช่นนั้น คุณเพียงแค่สแกน QR โค้ดเพื่อรับข้อมูลการชำระเงิน ซึ่งคุณสามารถตรวจสอบเพื่อยืนยันกระบวนการทำธุรกรรมได้


![image](assets/29.webp)


Muun จะแสดงหน้าคำอธิบายให้คุณดูในตอนแรก (ดูภาพหน้าจอด้านบน) จากนั้นจะขอให้คุณเปิดใช้งานการป้อนข้อมูลกล้องของอุปกรณ์ของคุณ ซึ่งเป็นขั้นตอนที่จำเป็นสำหรับการใช้งานแอปพลิเคชัน โปรดทราบว่า LNURL addresses ยังไม่ได้รับการสนับสนุนจากกระเป๋าเงิน Lightning ทั้งหมด ในบรรดาที่รองรับโดยทั่วไปจะมีเพียงความสามารถในการใช้ LNURL เพื่อรับธุรกรรมเท่านั้น ไม่ใช่เพื่อส่งธุรกรรม

![image](assets/32.webp)


เมื่อคุณมาถึงหน้า "ส่ง" หากคุณมีที่อยู่ที่คัดลอกไว้ในอุปกรณ์ของคุณแล้ว Muun จะจดจำรูปแบบที่อยู่ (Bitcoin หรือ Lightning) และแนะนำให้ใช้เพื่อส่งธุรกรรมผ่านข้อความที่มีกรอบ

เมื่อเตรียมการทำธุรกรรม Bitcoin คุณจำเป็นต้องป้อนจำนวนเงินที่จะส่ง ตรวจสอบให้แน่ใจว่าที่อยู่ปลายทางที่แสดงอยู่ด้านบนของหน้าจอตรงกับที่อยู่ที่คัดลอกมาก่อนหน้านี้ ด้านล่างของจำนวนเงินที่จะส่ง Muun จะแสดงยอดคงเหลือของ wallet ของคุณและให้คุณเลือกใช้เงินทั้งหมดของคุณ ซึ่งเป็นคุณสมบัติที่มีประโยชน์มากหากคุณต้องการล้าง wallet ของคุณให้หมดและหลีกเลี่ยงการทิ้ง "ฝุ่น" (ซาโตชิเล็กน้อย) ไว้เบื้องหลัง

หลังจากยืนยันจำนวนเงินที่จะส่งแล้ว Muun จะขอให้คุณเขียนบันทึกในหน้าถัดไป นี่เป็นการตรวจสอบเพิ่มเติม และคุณสามารถเขียนอะไรก็ได้ตามที่คุณต้องการ ไม่ว่าจะเกี่ยวข้องหรือไม่ก็ตาม


![image](assets/36.webp)


การตรวจสอบรายละเอียดการทำธุรกรรมครั้งสุดท้ายเป็นสิ่งจำเป็นก่อนที่จะทำการส่งออก ตรวจสอบที่อยู่และจำนวนเงินที่ป้อน จากนั้นปรับแต่งค่าธรรมเนียมการทำธุรกรรมหากจำเป็นโดยกดไอคอนดินสอสีน้ำเงินทางด้านขวาของ "ค่าธรรมเนียมเครือข่าย" การทำความเข้าใจพื้นฐานเกี่ยวกับวิธีการทำงานของกลุ่มธุรกรรม Bitcoin (mempool) เป็นประสบการณ์การเรียนรู้ที่ดีที่อาจช่วยให้คุณประหยัด satoshis ได้มากในระยะยาว!

ซอฟต์แวร์ของ Muun ใช้อัลกอริทึมโดยค่าเริ่มต้นที่คำนวณค่าธรรมเนียมการทำธุรกรรมที่จำเป็นสำหรับการยืนยันภายใน 30 นาทีหรือน้อยกว่า นี่คือสิ่งที่จะปรากฏเมื่อคุณพยายามแก้ไขค่าธรรมเนียมการทำธุรกรรม ปุ่ม "ป้อนค่าธรรมเนียมด้วยตนเอง" ช่วยให้คุณปรับแต่งรายละเอียดนี้ได้เอง ซึ่งเป็นคุณสมบัติที่มีประโยชน์มากหากคุณต้องการการยืนยันที่รวดเร็วขึ้น หรือในทางกลับกัน หากคุณมีขอบเขตการดำเนินการที่กว้างขวาง

โดยการเลือกที่จะป้อนจำนวนค่าธรรมเนียมการทำธุรกรรมด้วยตนเอง คุณจะถูกนำไปยังหน้าต่างใหม่ที่แสดงจำนวนที่จะป้อนใน sat/vbyte (ซาโตชิต่อไบต์เสมือน) Muun ยังแสดงประมาณการเวลายืนยันที่เกี่ยวข้องกับจำนวนที่เลือก รวมถึงค่าใช้จ่ายใน BTC และสกุลเงินเฟียตที่คุณเลือก

![image](assets/39.webp)


กลับไปที่หน้ารายละเอียดการทำธุรกรรมและกด "ส่ง" Voilà, การทำธุรกรรมของคุณถูกส่งออกไปบนเครือข่าย Bitcoin! คุณจะถูกเปลี่ยนเส้นทางไปยังหน้าแรกของ wallet ซึ่งคุณจะเห็นการหักจากยอดเงินของคุณ ที่ด้านล่างของหน้าจอมีลูกศรที่คุณสามารถกดเพื่อดูประวัติการทำธุรกรรมของคุณ การทำธุรกรรมที่คุณเพิ่งทำจะถูกเพิ่มไปยังช่วงเวลาที่เริ่มต้นของการออกใบเสร็จ


![image](assets/40.webp)


กดที่รายการเพื่อดูรายละเอียดของธุรกรรมเฉพาะ ธุรกรรมของคุณจะได้รับการยืนยันเมื่อมีคนขุดเพิ่มบล็อกใหม่ที่มีธุรกรรมของคุณลงในเชน Muun แชร์ transaction ID ที่ด้านล่างของหน้าจอ ทำให้คุณสามารถตรวจสอบสถานะของธุรกรรมของคุณบนบล็อกเอ็กซ์พลอเรอร์ได้

## ผ่านทาง Lightning Network


ตอนนี้ให้เราใช้ใบแจ้งหนี้ Bolt 11 (ใบแจ้งหนี้ Lightning แบบดั้งเดิม/ค่าเริ่มต้น) เพื่อทำธุรกรรม คัดลอกหรือสแกนที่อยู่ Lightning บนหน้า "ส่ง" คุณจะถูกเปลี่ยนเส้นทางไปยังหน้าที่แสดงรายละเอียดของใบแจ้งหนี้ปัจจุบัน จำนวนธุรกรรมจะแสดง (รวมถึงค่าธรรมเนียมเครือข่าย) พร้อมกับบันทึกหรือคำอธิบายที่เขียนบนใบแจ้งหนี้ และตัวจับเวลาหมดอายุที่ด้านล่าง โปรดทราบว่าค่าธรรมเนียมการทำธุรกรรมไม่สามารถแก้ไขได้สำหรับธุรกรรม Lightning ค่าธรรมเนียมจะถูกกำหนดโดยเส้นทางของช่องทางที่ต้องใช้เพื่อไปถึงผู้รับ


![image](assets/42.webp)


(นี่คือคำเตือนที่แสดงบนหน้าจอเมื่อใช้ใบแจ้งหนี้เปล่า หมายความว่าไม่มีจำนวนเงินที่กรอกไว้ล่วงหน้า กระเป๋าเงินบางประเภทสนับสนุนใบแจ้งหนี้ประเภทนี้และอนุญาตให้คุณกำหนดจำนวนเงินเองได้ ซึ่งไม่ใช่กรณีสำหรับ Muun)


![image](assets/43.webp)


การกดไอคอนรูปตาจะแสดงรายละเอียดของโหนด Lightning ที่คุณกำลังจัดการในธุรกรรมนี้ คุณยังมีตัวเลือกในการดูเว็บสำรวจเพื่อข้อมูลเพิ่มเติม นี่เป็นตัวอย่างที่ดีของการสรุปทางเทคนิคที่ Muun ทำได้


![image](assets/44.webp)


เมื่อคุณกด "ส่ง" การทำธุรกรรมของคุณจะเริ่มต้นและโดยปกติจะเสร็จสิ้นในเสี้ยววินาที จำนวนเงินที่ชำระจะถูกหักออกจากยอดคงเหลือของคุณ ซึ่งสามารถดูได้ที่หน้าแรกของแอปพลิเคชัน กลับไปที่ประวัติการทำธุรกรรมของคุณเพื่อดูการยืนยันการชำระเงินทันที


![image](assets/45.webp)


โปรดทราบว่าในประวัติการทำธุรกรรม Lightning และ Bitcoin จะถูกแยกแยะด้วยสัญลักษณ์ที่แตกต่างกัน หากต้องการดูรายละเอียดของธุรกรรม Lightning ให้แตะที่มันบนหน้าจอประวัติของคุณ


![image](assets/46.webp)


## การตั้งค่าแอปพลิเคชัน


แท็บที่สามบนหน้าแรก "การตั้งค่า" คือที่ที่คุณสามารถค้นหาการตั้งค่าของแอปพลิเคชันได้ หน้านี้มีความยาวที่น่าประหลาดใจ โดยเฉพาะเมื่อเปรียบเทียบกับกระเป๋าเงินมือถือยอดนิยมอื่น ๆ ในความเห็นของฉัน นี่ไม่ใช่ข้อเสีย ตรงกันข้าม ฉันเห็นว่ามันเป็นข้อดีในแง่ของความเรียบง่าย


![image](assets/47.webp)


ในหมวดทั่วไป คุณสามารถเลือกหน่วยบัญชีและสกุลเงินที่คุณต้องการ รวมถึงธีมการแสดงผลของแอปพลิเคชัน (แบบมืดหรือแบบสว่าง) ซึ่งจะถูกกำหนดในเบื้องต้นตามระบบของอุปกรณ์ของคุณ


สำหรับหน่วยบัญชีของแอปพลิเคชัน เลือกระหว่าง Bitcoin (BTC) หรือ Satoshi (SAT) เพื่อข้อมูลของคุณ Satoshi เป็นเศษส่วนที่เล็กที่สุดของบิตคอยน์ ซึ่งอยู่ในตำแหน่งทศนิยมที่แปด (1 SAT = 0.00000001 BTC) การกำหนด wallet ใน Satoshis มักจะได้รับความนิยมเมื่อใช้เครือข่าย Lightning เป็นหลักกับจำนวนเงินที่น้อย


Muun มีสกุลเงินให้เลือกมากมาย ทำให้ง่ายต่อการค้นหาอัตราแลกเปลี่ยน BTC ที่จำเป็นสำหรับการทำธุรกรรมและ/หรือความต้องการส่วนบุคคลของคุณ


หากคุณจำเป็นต้องเปลี่ยนรหัสผ่านการกู้คืน wallet คุณสามารถทำได้ในหน้าการตั้งค่า ตรวจสอบให้แน่ใจว่าคุณมีรหัสผ่านปัจจุบันหรือรหัสการกู้คืนและเข้าถึงอีเมลของคุณ


![image](assets/48.webp)


ป้อนรหัสผ่านปัจจุบันของคุณหรือเลือกป้อนรหัสกู้คืนของคุณเพื่อเริ่มการรีเซ็ต Muun จะส่งอีเมลไปยังที่อยู่ที่ลงทะเบียนไว้ก่อนหน้านี้


![image](assets/49.webp)

![image](assets/50.webp)


ส่วนการตั้งค่าขั้นสูงประกอบด้วยสองรายการ: Bitcoin Network และ Lightning Network ใน Bitcoin Network เรามีตัวเลือกในการเปิดใช้งานที่อยู่รับ Taproot (bc1p, ประเภทที่อยู่ใหม่ล่าสุด) โดยค่าเริ่มต้น


![image](assets/51.webp)


ภายใน Lightning Network คุณจะพบ:



- โปรโตคอลการรับ: เลือกเครือข่ายการรับเริ่มต้นของคุณที่แสดงบนหน้าจอรับ ฟังก์ชันทดลอง Unified ก็มีให้ใช้งานเช่นกัน นี่คือรหัส QR ที่รวมที่อยู่ทั้ง Bitcoin และ Lightning ไว้ด้วยกัน อย่างไรก็ตาม มีซอฟต์แวร์ Bitcoin เพียงไม่กี่ตัวที่รองรับฟีเจอร์นี้ในขณะนี้

-


- Turbo Channels: ตัวเลือกนี้ช่วยให้คุณสามารถเปิดหรือปิดใช้งานฟีเจอร์ Turbo channels ได้ โดยค่าเริ่มต้นจะเปิดใช้งานอยู่


![image](assets/52.webp)


ในการทำความเข้าใจสิ่งที่เรียกว่าช่องทาง Turbo เราต้องรู้ก่อนว่าธุรกรรม Lightning ดำเนินการผ่านช่องทางจากผู้ใช้หนึ่งไปยังอีกผู้ใช้หนึ่ง และช่องทางเหล่านี้ต้องได้รับการสนับสนุนทางการเงินในขั้นต้นโดยธุรกรรมบนบล็อกเชน Bitcoin


ช่องเทอร์โบช่วยให้คุณเริ่มทำธุรกรรมบนเครือข่าย Lightning ได้แม้กระทั่งก่อนที่ธุรกรรม on-chain ใด ๆ จะได้รับการยืนยัน การปิดใช้งานฟังก์ชันนี้จะหมายความว่าคุณจะต้องรอนานขึ้นอย่างมากในการทำธุรกรรมบนเครือข่าย Lightning เพื่อแลกกับความปลอดภัยที่เพิ่มขึ้นของเงินทุนของคุณ เนื่องจากมิฉะนั้นคุณจะต้องเชื่อใจว่า Muun จะไม่กระทำการที่เป็นอันตราย (การใช้จ่ายซ้ำซ้อนที่เปิดเผยต่อสาธารณะมาก) ในขณะที่รอให้ธุรกรรมของคุณได้รับการยืนยันบนบล็อกเชน


ที่ด้านล่างของหน้าการตั้งค่าคือ "ออกจากระบบ" คุณสามารถใช้ฟังก์ชันนี้หากคุณต้องการให้แอปพลิเคชันตัดการเชื่อมต่อ wallet ที่รู้จักในปัจจุบัน ซึ่งจะช่วยให้คุณสามารถสร้าง wallet ใหม่หรือ นำเข้า/กู้คืน ที่มีอยู่แล้วได้
