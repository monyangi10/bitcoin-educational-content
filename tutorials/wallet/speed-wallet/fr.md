## Qu'est ce que c'est Speed Wallet ?

Pour les nouveaux bitcoiners, le choix du portefeuille qui sera utilisé pour vous permettre de stocker et échanger vos bitcoins est crucial et peut affecter la sécurité de vos avoirs.

- De faire des achats de services digitaux grâce à une intégration de Bitrefill.

- D'accepter bitcoin dans votre commerce au travers d'un point de vente depuis l'application.

## Débuter avec Speed Wallet

Speed Wallet est un portefeuille custodial multidevises supportant des transactions sur le réseau principal de Bitcoin, les transactions sur le réseau Lightning mais aussi les transactions en devises stable USDT (Tether USD). Dans ce tutoriel, nous nous focaliserons exclusivement sur l'utilisation de Speed Wallet sur les réseaux de Bitcoin.

Speed Wallet est disponible en application mobile sur les plateformes Android (Google Play Store) et iOS (Apple Store) mais également en extension Chrome qui vous permet d'effectuer des transactions depuis votre ordinateur.

Vous pouvez retrouver sur le site web officiel de [Speed Wallet](https://speed.app) les différents liens vers les plateformes de téléchargement.

⚠️ **IMPORTANT**: Il est primordial de télécharger vos portefeuilles sur des plateformes de téléchargement officiels afin de garantir l'authenticité de l'application.

Dans ce tutoriel, nous nous focaliserons sur la plateforme Android de l'application mais tous les processus qui seront présentés dans ce tutoriel sont également valables sur iOS.

Speed Wallet requiert la création d'un compte utilisateur obligatoire pour son utilisation. Vous pouvez en créer un à partir de votre compte Google ou définir un email et un mot de passe fort.

### Effectuer votre première transaction avec Speed Wallet

Si vous avez déjà une expérience avec des portefeuilles Bitcoin, vous ne serez absolument pas déphasé. Speed Wallet vous propose une interface intuitive pour faciliter votre prise en main.

Sur la page d'accueil au menu **Wallet**, vous retrouverez :

- Une section pour la monnaie avec la quelle vous souhaitez opérer.

- Votre solde dans la monnaie choisie.

- Une option pour vous permettre de créer et de personnaliser votre adresse Lightning afin de recevoir des paiements Lightning.

![lightning-address](assets/fr/03.webp)

⚠️ **NOTE** : Une adresse Lightning est un alias sous forme d'email unique qui vous permet de simplifier la lecture de votre URL Lightning. Assurez-vous de fournir l'adresse Lightning correcte lorsque vous souhaitez recevoir des paiements Lightning.

**Recevoir des paiements sur Speed Wallet :**

Cliquez sur le bouton **Recevoir** puis sélectionnez la couche sur laquelle vous souhaitez recevoir puis le montant de la transaction pour partager votre facture à votre expéditeur.

Vous pouvez également utiliser simplement votre adresse Lightning lorsque vous souhaitez donner à votre expéditeur la flexibilité de définir le montant de la transaction.

![receive-bitcoins](assets/fr/04.webp)

- **Envoyer des bitcoins avec Speed Wallet** :

A partir du bouton **Envoyer**, vous pouvez envoyer des bitcoins en utilisant l'adresse Lightning, la facture Lightning ou l'adresse Bitcoin (mainnet) de votre destinataire.

Speed Wallet détectera, en fonction du format de votre adresse, le réseau sur lequel il devra opérer ce qui vous donne une expérience plus fluide.

![send-btc](assets/fr/05.webp)

### Explorez plus Speed Wallet

L'une des premières particularités de Speed Wallet est qu'il vous permet d'acheter et d'échanger des devises sans sortir de l'application. Ainsi, vous pouvez obtenir des bitcoins à partir du solde d'une autre devise de votre portefeuille. Par exemple vous pouvez acheter du bitcoin avec les USDT que vous possédez déjà dans le portefeuille, et cela, avec de moindres frais d'échanges.

Les options **Buy** et **Swap**, vous permettent d'échanger des bitcoins pour les autres devises disponibles dans Speed.

- **Acheter du Bitcoin avec sa carte de crédit** : Speed Wallet facilite l'acquisition de bitcoin à partir des monnaies fiduciaires que vous utilisez quotidiennement. Il intègre un agrégateur de paiement qui vous permet de payer des bitcoins avec votre carte de crédit.

![credit-card-buy-btc](assets/fr/06.webp)

- **Acheter Bitcoin à partir d'autres cryptomonnaies** : Vous pouvez échanger vos USDT, USDC pour avoir des bitcoins dans votre portefeuille et vice-versa. Au travers cette option, Speed Wallet simplifie le processus d'achat et de vente de Bitcoin sans pour autant se référer à des plateformes d'échange externes. Vous pouvez donc faire des échanges à partir de 20.000 satoshis, environ $20 au taux actuel, sans quitter votre portefeuille Speed.

## Obtenez des récompenses

En plus des applications intégrés, Speed Wallet met en place un processus d'évolution basée sur votre utilisation du portefeuille. En fonction de la régularité de vos transactions, vous obtiendrez des jetons Speed qui vous rendent éligibles à une récompense que vous pourrez recevoir en bitcoins.

Un système d'invitation existe également sur Speed Wallet qui vous permet de gagner des bitcoins chaque fois que quelqu'un utilise votre lien pour créer son portefeuille Speed Wallet.

Pour cela, vous devez être éligible après vérifications des critères suivants :

- Le programme de parrainage est disponible dans votre pays.

- Votre compte date d'au moins 24 Heures.

- Vous avez au moins **1000 Satoshis ou 1 USDT** sur votre portefeuille.

- L'invitation et les récompenses ne sont pas restreints pour votre compte.

## Speed Wallet pour les commerces

Speed wallet accorde une importance particulière à l'intégration et à l'utilisation de Bitcoin et des stablecoins dans les commerces partout dans le monde.

A partir de [Speed Business](https://www.tryspeed.com/), vous avez un agrégateur de paiement unifié pour accepter Bitcoin, propulsé par le Lightning Network que vous pouvez utiliser dans votre boutique, en ligne ou en physique.

Essentiellement axée sur les paiements vous retrouverez les options suivantes :

- **Les paiements en ligne**: Avec cette option, vous pourrez accepter bitcoin comme moyen de paiement de vos produits digitaux, via des liens de paiement, la facturation et les souscriptions.

- **Les paiements sur site**: Pour encaisser des paiements dans votre boutique.

- **Les paiements instantanés**: Une option qui vous permet de gérer les remboursements, les retraits, les dépenses mais aussi les fiches de paie de vos employés directement depuis l'interface de Speed Business.

- **Les paiements de plateforme**: Connectez votre compte Speed Business à des outils externes que vous utilisez afin d'effectuer des transferts et paiements vers ces plateformes.
