對於新的比特幣使用者而言，選擇用來儲存和 Exchange 您的比特幣的 Wallet 是至關重要的，它會影響您的資產安全。



有鑑於此，Speed Wallet 是一個直覺且安全的生態系統，可讓您兩者兼顧：





- 儲存和 Exchange 您的 bitcoins。





- 透過 Bitrefill 整合購買數位服務。





- 在您的企業中透過銷售點應用程式接受 Bitcoin。

## 開始使用 Speed Wallet



Speed Wallet 是多貨幣托管 Wallet，支援 Bitcoin 主網絡上的交易、Lightning Network 交易以及 USDT (Tether USD) 穩定貨幣交易。在本教程中，我們將專注於在 Bitcoin 網路上使用 Speed Wallet。



Speed Wallet 是 Android (Google Play 商店) 和 iOS (Apple 商店) 平台上的行動應用程式，同時也是 Chrome 瀏覽器的延伸功能，可讓您在電腦上進行交易。



您可以在 [Speed Wallet] 官方網站 (https://speed.app) 找到下載平台的連結。



⚠️ **重要**：必須從官方下載平台下載錢包，以保證應用程式的真實性。



在本教程中，我們將專注於應用程式的 Android 平台，但本教程中將介紹的所有流程在 iOS 上同樣有效。



Speed Wallet 需要建立使用者帳戶。您可以從 Google 帳戶建立一個，或定義一個強大的電子郵件和密碼。



https://planb.network/tutorials/computer-security/authentication/bitwarden-0532f569-fb00-4fad-acba-2fcb1bf05de9

![download](assets/fr/01.webp)



建立帳戶後，我們建議您設定雙重認證系統或 PIN/生物指紋驗證。這些設定可增加額外的安全層級，讓您在存取 Wallet 之前以及從 Speed Wallet 進行 Bitcoin 交易時，都能進行自我驗證。



https://planb.network/tutorials/computer-security/authentication/authy-a76ab26b-71b0-473c-aa7c-c49153705eb7

您也可以從「設定」中自訂餘額的顯示方式，在選項中選擇您的當地貨幣。



### 使用 Speed Wallet 進行首次交易



如果您已有使用 Bitcoin 錢包的經驗，就完全不會脫節。Speed Wallet 為您提供直觀的 Interface 協助您入門。



在 **Wallet** 首頁，您可以找到 ：





- 您希望使用的貨幣部分。





- 您以所選貨幣計算的餘額。





- 可選擇建立和個人化您的 Lightning Address，以接收 Lightning 付款。



![lightning-address](assets/fr/03.webp)



⚠️ **註**：Lightning Address是一個獨特的電子郵件別名，可以簡化讀取您的Lightning URL。當您希望收到Lightning付款時，請務必提供正確的Lightning Address。





- 在 Speed Wallet 上接收付款：



按一下**接收**按鈕，然後選擇您要接收的 Layer 和交易金額，與您的寄件者分享您的 Invoice。



當您想要讓寄件者彈性定義交易金額時，您也可以簡單地使用您的 Lightning Address。



![receive-bitcoins](assets/fr/04.webp)





- 使用 **Speed Wallet** 傳送比特幣：



從 **Send** 按鈕，您可以使用收件人的 Lightning Address、Lightning Invoice 或 Bitcoin (Mainnet) Address 傳送 bitcoins。



Speed Wallet 會根據您的 Address 格式，偵測它應該在哪個網路上運作，帶給您更順暢的體驗。



![send-btc](assets/fr/05.webp)



### 探索更多極速 Wallet



Speed Wallet 的首要功能之一是讓您在不離開應用程式的情況下購買和 Exchange 貨幣。通過這種方式，您可以從 Wallet 中另一種貨幣的餘額中獲取比特幣。例如，您可以用 Wallet 中已有的 USDT 購買 Bitcoin，Exchange 費用更低。



**Buy** 和 **Swap** 選項可讓您以 Exchange 比特幣換取 Speed 中可用的其他貨幣。





- 使用信用卡購買 **Bitcoin**：Speed Wallet 可讓您輕鬆地從您每天使用的法定貨幣中取得 Bitcoin。它結合了支付聚合器，讓您可以用信用卡支付比特幣。



![credit-card-buy-btc](assets/fr/06.webp)





- 從其他加密貨幣**購買 Bitcoin**：您可以 Exchange 您的 USDT、USDC 在您的 Wallet 中買入比特幣，反之亦然。透過此選項，Speed Wallet 簡化了買賣 Bitcoin 的過程，無需參考外部 Exchange 平台。因此，您可以在不離開您的 Speed Wallet 的情況下，以低至 20,000 薩托希（以目前匯率計算約為 20 美元）的價格進行交易。

## 賺取獎勵



除了內建的應用程式之外，Speed Wallet 還會根據您對 Wallet 的使用情況，實施進化程序。根據您交易的規律性，您將賺取 Speed 代幣，讓您有資格獲得獎勵，您可以比特幣的方式獲得獎勵。



Speed Wallet 也具有邀請系統，每次有人使用您的連結來建立他們的 Speed Wallet Wallet，您就可以賺取比特幣。



要符合資格，您必須滿足以下條件：





- 您所在的國家可提供贊助計畫。





- 您的帳戶至少已使用 24 小時。





- 您的投資組合中至少有 **1000 Satoshis 或 1 USDT**。





- 邀請和獎勵並不限於您的帳戶。

## 適用於零售業的速度 Wallet



Speed Wallet 特別強調 Bitcoin 和穩定幣在全球企業中的整合與使用。



從 [Speed Business](https://www.tryspeed.com/)，您有一個統一的付款聚合器來接受 Bitcoin，由 Lightning Network 所提供，您可以在您的商店中使用，無論是線上或實體商店。



基本上以付款為主，您可以找到下列選項：





- **線上付款**：使用此選項，您可以透過付款連結、發票和訂閱，接受 Bitcoin 作為數位產品的付款方式。





- **現場付款**：在您的店內收取付款。





- **即時付款**：可讓您直接從 Speed Business Interface 管理報銷、提款、開銷和員工薪資單的選項。





- **平台付款**：將您的 Speed Business 帳戶連接到您使用的外部工具，以便向這些平台進行轉帳和付款。
