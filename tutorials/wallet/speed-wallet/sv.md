## Vad är Speed Wallet?



För nya bitcoiners är valet av Wallet som kommer att användas för att lagra och Exchange dina bitcoins avgörande och kan påverka säkerheten för dina tillgångar.



Med detta i åtanke är Speed Wallet ett intuitivt och säkert ekosystem som gör att du kan göra både och:





- Förvara och Exchange dina bitcoins.





- Köp digitala tjänster genom Bitrefill-integration.





- Acceptera Bitcoin i ditt företag genom en applikation för försäljningsställen.

## Komma igång med Speed Wallet



Speed Wallet är en depå Wallet med flera valutor som stöder transaktioner på Bitcoin huvudnätverket, Lightning Network transaktioner och även USDT (Tether USD) stabila valutatransaktioner. I denna handledning kommer vi att fokusera uteslutande på användningen av Speed Wallet på Bitcoin-nätverken.



Speed Wallet finns som en mobilapplikation på plattformarna Android (Google Play Store) och iOS (Apple Store), men också som ett Chrome-tillägg som gör att du kan genomföra transaktioner från din dator.



Du hittar länkar till nedladdningsplattformar på den officiella webbplatsen [Speed Wallet] (https://speed.app).



⚠️ **VIKTIGT**: Det är viktigt att ladda ner dina plånböcker från officiella nedladdningsplattformar för att garantera applikationens äkthet.



I den här handledningen fokuserar vi på appens Android-plattform, men alla processer som kommer att presenteras i den här handledningen är lika giltiga på iOS.



Speed Wallet kräver att du skapar ett användarkonto. Du kan skapa ett från ditt Google-konto eller definiera en stark e-postadress och ett lösenord.

### Gör din första transaktion med Speed Wallet



Om du redan har erfarenhet av Bitcoin-plånböcker kommer du inte att gå ur steget alls. Speed Wallet erbjuder dig en intuitiv Interface för att hjälpa dig att komma igång.



På hemsidan för **Wallet** hittar du :





- Ett avsnitt för den valuta som du vill arbeta med.





- Ditt saldo i den valda valutan.





- En möjlighet att skapa och anpassa din Lightning Address för att ta emot Lightning-betalningar.



![lightning-address](assets/fr/03.webp)



⚠️ **NOTE**: En Lightning Address är ett unikt e-postalias som förenklar läsningen av din Lightning URL. Var noga med att ange rätt Lightning Address när du vill ta emot Lightning-betalningar.





- Mottagande av betalningar på Speed Wallet :**



Klicka på knappen **Receive** och välj sedan den Layer som du vill ta emot och transaktionsbeloppet för att dela din Invoice med din avsändare.



Du kan också helt enkelt använda din Lightning Address när du vill ge din avsändare flexibiliteten att definiera transaktionsbeloppet.



![receive-bitcoins](assets/fr/04.webp)





- Skicka bitcoins med Speed Wallet** :



Från **Sänd**-knappen kan du skicka bitcoins med din mottagares Lightning Address, Lightning Invoice eller Bitcoin (Mainnet) Address.



Speed Wallet kommer att upptäcka, baserat på ditt Address-format, det nätverk som den ska fungera på, vilket ger dig en smidigare upplevelse.



![send-btc](assets/fr/05.webp)



### Utforska mer Hastighet Wallet



En av de första funktionerna i Speed Wallet är att den låter dig köpa och Exchange valutor utan att lämna applikationen. På detta sätt kan du få bitcoins från saldot för en annan valuta i din Wallet. Du kan till exempel köpa Bitcoin med USDT som du redan har i din Wallet, med lägre Exchange-avgifter.



Alternativen **Buy** och **Swap** låter dig växla 43 bitcoins mot de andra valutorna som finns tillgängliga i Speed.





- Köp Bitcoin med ditt kreditkort**: Speed Wallet gör det enkelt att förvärva Bitcoin från de fiatvalutor du använder varje dag. Den innehåller en betalningsaggregator som gör att du kan betala bitcoins med ditt kreditkort.



![credit-card-buy-btc](assets/fr/06.webp)





- Köp Bitcoin från andra kryptovalutor**: Du kan Exchange din USDT, USDC för bitcoins i din Wallet och vice versa. Genom detta alternativ förenklar Speed Wallet processen att köpa och sälja Bitcoin utan att hänvisa till externa Exchange-plattformar. Så du kan handla för så lite som 20 000 satoshis, cirka 20 dollar till nuvarande kurs, utan att lämna din Speed Wallet.

## Tjäna belöningar



Förutom de inbyggda applikationerna implementerar Speed Wallet en evolutionär process baserad på din användning av Wallet. Beroende på hur vanliga dina transaktioner är tjänar du Speed-tokens som gör dig berättigad till en belöning som du kan få i bitcoins.



Speed Wallet har också ett inbjudningssystem som låter dig tjäna bitcoins varje gång någon använder din länk för att skapa sin Speed Wallet Wallet.



För att kvalificera dig måste du uppfylla följande kriterier:





- Sponsringsprogrammet finns tillgängligt i ditt land.





- Ditt konto är minst 24 timmar gammalt.





- Du har minst **1000 Satoshis eller 1 USDT** i din portfölj.





- Inbjudan och belöningar är inte begränsade till ditt konto.

## Hastighet Wallet för detaljhandeln



Speed Wallet lägger särskild vikt vid integration och användning av Bitcoin och stablecoins i företag runt om i världen.



Från [Speed Business] (https://www.tryspeed.com/) har du en enhetlig betalningsaggregator för att acceptera Bitcoin, som drivs av Lightning Network som du kan använda i din butik, online eller fysisk.



I huvudsak inriktad på betalningar, hittar du följande alternativ:





- Betalningar online**: Med det här alternativet kan du acceptera Bitcoin som betalningsmedel för dina digitala produkter, via betalningslänkar, fakturering och prenumerationer.





- Betalningar på plats**: För att ta emot betalningar i din butik.





- Omedelbara betalningar**: Ett alternativ som gör att du kan hantera ersättningar, uttag, utgifter och lönebesked för anställda direkt från Speed Business Interface.





- Betalningar till plattformar**: Anslut ditt Speed Business-konto till externa verktyg som du använder för att göra överföringar och betalningar till dessa plattformar.
