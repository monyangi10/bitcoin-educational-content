## 什么是速度 Wallet？



对于比特币新手来说，选择用于存储比特币的 Wallet 和 Exchange 至关重要，会影响到你的资产安全。

- 通过销售点应用程序在您的企业中接受 Bitcoin。

## 开始使用 Speed Wallet



Speed Wallet 是一种多货币托管 Wallet，支持 Bitcoin 主网络上的交易、Lightning Network 交易以及 USDT (Tether USD) 稳定货币交易。在本教程中，我们将专注于在 Bitcoin 网络上使用 Speed Wallet。



Speed Wallet 是安卓（Google Play 商店）和 iOS（苹果商店）平台上的移动应用程序，也是 Chrome 浏览器的扩展程序，可让您在电脑上进行交易。



您可以在 [Speed Wallet] 官方网站 (https://speed.app) 上找到下载平台的链接。



⚠️ **重要**：请务必从官方下载平台下载钱包，以确保应用程序的真实性。



在本教程中，我们将重点介绍应用程序的 Android 平台，但本教程中介绍的所有流程在 iOS 平台上同样适用。



Speed Wallet 需要创建一个用户账户。您可以通过谷歌账户创建一个账户，或者定义一个强大的电子邮件和密码。



https://planb.network/tutorials/computer-security/authentication/bitwarden-0532f569-fb00-4fad-acba-2fcb1bf05de9

![download](assets/fr/01.webp)



创建账户后，我们建议您设置双重验证系统或 PIN/生物指纹验证。这些配置增加了额外的安全级别，允许您在访问 Wallet 之前和从 Speed Wallet 进行 Bitcoin 交易期间进行身份验证。



https://planb.network/tutorials/computer-security/authentication/authy-a76ab26b-71b0-473c-aa7c-c49153705eb7

为此，请进入应用程序设置，然后激活双重认证和生物识别验证。



![2fa-biometry](assets/fr/02.webp)



您还可以在 "设置 "选项中选择当地货币，自定义余额的显示方式。



### 使用极速 Wallet 进行首次交易



如果您已经拥有使用 Bitcoin 钱包的经验，那么您完全不会感到不适应。Speed Wallet 为您提供直观的 Interface 来帮助您入门。



在 **Wallet** 主页上，您可以找到 ：





- 您希望使用的货币部分。





- 您所选货币的余额。





- 创建和个性化您的闪电 Address 以接收闪电付款的选项。



![lightning-address](assets/fr/03.webp)



⚠️ **注意**：Lightning Address是一个独特的电子邮件别名，可简化您的Lightning URL的读取。当您希望接收 Lightning 付款时，请务必提供正确的 Lightning Address。





- 接收速度 Wallet 上的付款 :**



点击**接收**按钮，然后选择要接收的 Layer 和交易金额，以便与发件人共享 Invoice。



如果您想让发件人灵活定义交易金额，也可以直接使用 Lightning Address。



![receive-bitcoins](assets/fr/04.webp)





- 使用 **Speed Wallet 发送比特币**：



通过**发送**按钮，您可以使用收件人的闪电 Address、闪电 Invoice 或 Bitcoin (Mainnet) Address 发送比特币。



Speed Wallet 将根据您的 Address 格式检测它应该运行的网络，为您带来更流畅的体验。



![send-btc](assets/fr/05.webp)



### 探索更多速度 Wallet



Speed Wallet 的首要功能之一是让您无需离开应用程序即可购买货币和 Exchange 货币。这样，您就可以从 Wallet 中的另一种货币余额中获得比特币。例如，您可以用 Wallet 中已有的 USDT 购买 Bitcoin，Exchange 费用较低。



通过**购买**和**交换**选项，您可以用 Exchange 比特币兑换 Speed 中的其他货币。





- 用信用卡购买 **Bitcoin**：通过 Speed Wallet，您可以轻松地用日常使用的法定货币购买 Bitcoin。它包含一个支付聚合器，允许您用信用卡支付比特币。



![credit-card-buy-btc](assets/fr/06.webp)





- 从其他加密货币购买 **Bitcoin**：您可以在 Wallet 中用 USDT 和 USDC 购买比特币，反之亦然。通过此选项，Speed Wallet 简化了买卖 Bitcoin 的过程，无需参考外部 Exchange 平台。因此，您可以在不离开 Speed Wallet 的情况下，以低至 20,000 萨托希（按当前汇率约合 20 美元）的价格进行交易。

## 赚取奖励



除内置应用程序外，Speed Wallet 还根据您对 Wallet 的使用情况实施进化程序。根据您的交易规律，您将获得 Speed 代币，从而有资格获得比特币奖励。



Speed Wallet 还设有一个邀请系统，每次有人使用您的链接创建他们的 Speed Wallet Wallet，您就可以赚取比特币。



要获得资格，您必须满足以下条件：





- 您所在的国家可提供赞助计划。





- 您的账户至少已使用 24 小时。





- 您的投资组合中至少有 **1000 Satoshis 或 1 USDT**。





- 邀请和奖励不限于您的账户。

## 零售业的 Wallet 速度



速度 Wallet 特别强调在世界各地的企业中整合和使用 Bitcoin 和稳定币。



通过 [Speed Business](https://www.tryspeed.com/)，您可以使用由 Lightning Network 支持的统一支付聚合器来接受 Bitcoin，您可以在您的在线或实体商店中使用该聚合器。



主要集中在支付方面，您可以找到以下选项：





- 在线支付：使用此选项，您可以通过付款链接、发票和订阅接受 Bitcoin 作为数字产品的付款方式。





- **现场付款**：在店内收款。





- **即时支付**：该选项可让您直接从 Speed Business Interface 管理报销、提款、费用和员工工资单。





- 平台支付：将您的 "极速商务" 账户连接到外部工具，以便向这些平台进行转账和付款。
