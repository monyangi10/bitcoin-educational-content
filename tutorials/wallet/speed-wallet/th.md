## Speed Wallet คืออะไร?



สำหรับผู้เริ่มต้นใช้งานบิตคอยน์ การเลือก wallet ที่จะใช้ในการเก็บและแลกเปลี่ยนบิตคอยน์ของคุณเป็นสิ่งสำคัญและสามารถส่งผลต่อความปลอดภัยของสินทรัพย์ของคุณได้



ด้วยเหตุนี้ Speed Wallet จึงเป็นระบบนิเวศที่ใช้งานง่ายและปลอดภัยที่ช่วยให้คุณทำทั้งสองอย่าง:





- จัดเก็บและแลกเปลี่ยนบิตคอยน์ของคุณ





- ซื้อบริการดิจิทัลผ่านการรวม Bitrefill





- ยอมรับ Bitcoin ในธุรกิจของคุณผ่านแอปพลิเคชันจุดขาย



ในบทแนะนำนี้ เราจะพาคุณผ่านแต่ละแง่มุมเหล่านี้เพื่อทำให้ประสบการณ์ของคุณกับ Speed Wallet ง่ายที่สุดเท่าที่จะเป็นไปได้

## เริ่มต้นใช้งาน Speed Wallet



Speed Wallet เป็น wallet ที่รองรับหลายสกุลเงินและรองรับธุรกรรมบนเครือข่ายหลัก Bitcoin, ธุรกรรมเครือข่าย Lightning และยังรองรับธุรกรรมสกุลเงินเสถียร USDT (Tether USD) อีกด้วย ในบทแนะนำนี้ เราจะมุ่งเน้นเฉพาะการใช้ Speed Wallet บนเครือข่าย Bitcoin เท่านั้น



Speed Wallet มีให้บริการในรูปแบบแอปพลิเคชันมือถือบนแพลตฟอร์ม Android (Google Play Store) และ iOS (Apple Store) แต่ยังมีในรูปแบบส่วนขยายของ Chrome ที่ให้คุณทำธุรกรรมจากคอมพิวเตอร์ของคุณได้ด้วย



คุณสามารถค้นหาลิงก์เพื่อดาวน์โหลดแพลตฟอร์มได้ที่เว็บไซต์ทางการ [Speed Wallet](https://speed.app)



⚠️ **สำคัญ**: จำเป็นต้องดาวน์โหลดกระเป๋าเงินของคุณจากแพลตฟอร์มดาวน์โหลดอย่างเป็นทางการเพื่อรับประกันความถูกต้องของแอปพลิเคชัน



ในบทแนะนำนี้ เราจะมุ่งเน้นไปที่แพลตฟอร์ม Android ของแอป แต่กระบวนการทั้งหมดที่จะแสดงในบทแนะนำนี้สามารถใช้ได้เช่นเดียวกันบน iOS

### การทำธุรกรรมครั้งแรกของคุณกับ Speed Wallet



หากคุณมีประสบการณ์กับกระเป๋าเงิน Bitcoin อยู่แล้ว คุณจะไม่รู้สึกไม่เข้ากันเลย Speed Wallet มอบ Interface ที่ใช้งานง่ายเพื่อช่วยให้คุณเริ่มต้นได้



บนหน้าแรกของ **Wallet** คุณจะพบ:





- ส่วนสำหรับสกุลเงินที่คุณต้องการดำเนินการ





- ยอดเงินของคุณในสกุลเงินที่เลือก





- ตัวเลือกในการสร้างและปรับแต่งที่อยู่ Lightning ของคุณเพื่อรับการชำระเงินผ่าน Lightning



![lightning-address](assets/fr/03.webp)



⚠️ **หมายเหตุ**: ที่อยู่ Lightning เป็นนามแฝงอีเมลที่ไม่ซ้ำกันซึ่งทำให้ง่ายต่อการอ่าน URL ของ Lightning ของคุณ อย่าลืมระบุที่อยู่ Lightning ที่ถูกต้องเมื่อคุณต้องการรับการชำระเงินผ่าน Lightning





- การรับชำระเงินบน Speed Wallet:



คลิกที่ปุ่ม **รับ** จากนั้นเลือกเลเยอร์ที่คุณต้องการรับและจำนวนธุรกรรมเพื่อแชร์ใบแจ้งหนี้ของคุณกับผู้ส่งของคุณ



คุณยังสามารถใช้ที่อยู่ Lightning ของคุณได้อย่างง่ายดายเมื่อคุณต้องการให้ผู้ส่งมีความยืดหยุ่นในการกำหนดจำนวนธุรกรรม



![receive-bitcoins](assets/fr/04.webp)





- ส่งบิตคอยน์ด้วย **Speed Wallet**:



จากปุ่ม **ส่ง** คุณสามารถส่งบิตคอยน์โดยใช้ที่อยู่ Lightning ของผู้รับ ใบแจ้งหนี้ Lightning หรือที่อยู่ Bitcoin (Mainnet)



Speed Wallet จะตรวจจับตามรูปแบบที่อยู่ของคุณว่าเครือข่ายใดที่ควรทำงาน เพื่อให้คุณได้รับประสบการณ์ที่ราบรื่นยิ่งขึ้น



![send-btc](assets/fr/05.webp)



### สำรวจเพิ่มเติม Speed Wallet



หนึ่งในคุณสมบัติแรกของ Speed Wallet คือช่วยให้คุณซื้อและแลกเปลี่ยนสกุลเงินโดยไม่ต้องออกจากแอปพลิเคชัน ด้วยวิธีนี้ คุณสามารถรับบิตคอยน์จากยอดคงเหลือของสกุลเงินอื่นใน wallet ของคุณได้ ตัวอย่างเช่น คุณสามารถซื้อ Bitcoin ด้วย USDT ที่คุณมีอยู่แล้วใน wallet ของคุณ โดยมีค่าธรรมเนียมการแลกเปลี่ยนที่ต่ำกว่า



ตัวเลือก **ซื้อ** และ **แลกเปลี่ยน** ช่วยให้คุณแลกเปลี่ยนบิตคอยน์เป็นสกุลเงินอื่น ๆ ที่มีใน Speed.

## Speed Wallet สำหรับขายปลีก



Speed Wallet ให้ความสำคัญเป็นพิเศษกับการบูรณาการและการใช้ Bitcoin และ stablecoins ในธุรกิจทั่วโลก



จาก [Speed Business](https://www.tryspeed.com/), คุณมีตัวรวบรวมการชำระเงินแบบรวมศูนย์เพื่อรับ Bitcoin ซึ่งขับเคลื่อนโดย Lightning Network ที่คุณสามารถใช้ในร้านค้าของคุณ ทั้งออนไลน์หรือร้านค้าจริง



โดยเน้นที่การชำระเงินเป็นหลัก คุณจะพบตัวเลือกต่อไปนี้:





- การชำระเงินออนไลน์**: ด้วยตัวเลือกนี้ คุณสามารถยอมรับ Bitcoin เป็นวิธีการชำระเงินสำหรับผลิตภัณฑ์ดิจิทัลของคุณ ผ่านลิงก์การชำระเงิน การออกใบแจ้งหนี้ และการสมัครสมาชิก





- การชำระเงินในสถานที่**: เพื่อเก็บเงินในร้านค้าของคุณ





- การชำระเงินทันที**: ตัวเลือกที่ให้คุณจัดการการคืนเงิน การถอนเงิน ค่าใช้จ่าย และสลิปเงินเดือนพนักงานโดยตรงจาก Speed Business Interface





- การชำระเงินบนแพลตฟอร์ม**: เชื่อมต่อบัญชี Speed Business ของคุณกับเครื่องมือภายนอกที่คุณใช้เพื่อทำการโอนและชำระเงินไปยังแพลตฟอร์มเหล่านี้
