Передача биткоинов в случае смерти или потери трудоспособности представляет собой серьезную проблему для любого владельца криптоактивов. Без соответствующего плана наследования эти активы становятся невозвратными для ваших близких.



Heritage предлагает элегантный ответ, реализовав механизм переключения "мертвого человека" непосредственно в блокчейне Bitcoin. Этот блокчейн с открытым исходным кодом wallet позволяет настраивать условия наследования on-chain: если владелец не совершает больше никаких транзакций в течение определенного периода, заранее назначенные альтернативные ключи могут освободить средства.



## Что такое "Наследие"?



Heritage - это портфель Bitcoin, в который встроен механизм наследования через скрипты Taproot. Разработанное под лицензией MIT Жереми Родоном, это программное обеспечение с открытым исходным кодом гарантирует прозрачность и долговечность.



Heritage использует скрипты Taproot, закодированные в адресах Bitcoin. Каждый UTXO объединяет два типа условий расходования средств:





- Основной путь** : Владелец может потратить свои биткоины в любое время с помощью своего первичного ключа
- Альтернативные пути**: Для каждого назначенного наследника сценарий объединяет его открытый ключ с временным замком



Каждая сделка с владельцем автоматически переносит дату активации условий наследования. В случае длительного бездействия (смерть, недееспособность) условия срабатывают автоматически.



## Служба наследия (по желанию)



Heritage предлагает два варианта использования:



**Сделай сам (бесплатно)**: Только приложение с открытым исходным кодом. Вы управляете всем автономно с помощью собственного узла. Этот вариант предлагает встроенный резервный доступ, встроенное наследование и эксклюзивный контроль над вашими биткоинами. Однако вам необходимо создать собственные оповещения (календарь, напоминания), чтобы не забыть обновить свои таймлоки, а автоматических уведомлений для наследников не предусмотрено.



**Пользуйтесь сервисом (0,05% в год)** : Сервис btc-heritage.com добавляет функции для упрощения управления:




- Автоматические напоминания до истечения установленных сроков
- Автоматические уведомления для наследников, чтобы направить их в процессе восстановления
- Приоритетная поддержка
- Упрощенное управление дескрипторами



Комиссия: 0,05% от управляемой суммы в год, минимум 0,5 mBTC/год. Первый год бесплатно.



Услуга не требует хранения: ваши закрытые ключи никогда не покидают ваше устройство. Наследие не может получить доступ к вашим средствам.



## Heritage CLI



Для опытных пользователей, предпочитающих терминал, Heritage предлагает инструмент командной строки (CLI) для гранулированного управления и работы машины с воздушной заслонкой.



![Page Heritage CLI](assets/fr/03.webp)



Полная документация по CLI доступна по адресу [btc-heritage.com/heritage-cli](https://btc-heritage.com/heritage-cli). Здесь вы найдете инструкции по загрузке, подключению к сервису, созданию кошельков (с Ledger или локальными ключами), управлению наследниками и транзакциями.



Это руководство посвящено приложению Desktop, которое более доступно для большинства пользователей.

### Установка в Linux



В Linux приложение распространяется в формате AppImage. Прежде чем запустить его, необходимо установить зависимость `libfuse2`:



```bash
sudo apt install libfuse2
```

Затем сделайте файл исполняемым и запустите его:



```bash

./Heritage-GUI-vX.X.X.AppImage
```

## Лучшие практики



### Сохранение дескрипторов



Дескрипторы wallet необходимы для восстановления адресов вашего наследия. **Без дескрипторов, даже с помощью мнемонической фразы, вы не сможете найти свои фонды.



### Безопасность ключей





- Используйте Ledger для главного ключа, если это возможно
- Никогда не храните приговоры наследников в том же месте, что и свои собственные
- Распространяйте информацию в различных средствах массовой информации и местах



### Документация для ваших близких



Напишите четкие инструкции, объясняющие каждый шаг процесса восстановления. В критический момент ваши наследники могут быть не знакомы с Bitcoin.
