การส่งต่อบิตคอยน์ในกรณีที่เสียชีวิตหรือไร้ความสามารถเป็นความท้าทายสำคัญสำหรับผู้ถือครองสินทรัพย์คริปโต หากไม่มีแผนการสืบทอดที่เหมาะสม สินทรัพย์เหล่านี้จะไม่สามารถกู้คืนได้สำหรับคนที่คุณรัก



Heritage ให้คำตอบที่หรูหราโดยการใช้กลไกสวิตช์คนตายโดยตรงบนบล็อกเชน Bitcoin โอเพ่นซอร์ส wallet นี้ช่วยให้สามารถกำหนดเงื่อนไขการสืบทอด on-chain ได้: หากเจ้าของไม่ทำธุรกรรมเพิ่มเติมในช่วงเวลาที่กำหนด คีย์ทางเลือกที่กำหนดไว้ล่วงหน้าสามารถปล่อยเงินทุนได้



## มรดกคืออะไร?



Heritage เป็นพอร์ตโฟลิโอ Bitcoin ที่ผสานรวมกลไกการสืบทอดผ่านสคริปต์ Taproot โดยกำเนิด พัฒนาโดยใช้ใบอนุญาต MIT โดย Jérémie Rodon ซอฟต์แวร์โอเพ่นซอร์สนี้รับประกันความโปร่งใสและความทนทาน



มรดกใช้สคริปต์ Taproot ที่เข้ารหัสในที่อยู่ Bitcoin แต่ละ UTXO รวมเงื่อนไขการใช้จ่ายสองประเภท:





- เส้นทางหลัก** : เจ้าของสามารถใช้จ่ายบิตคอยน์ของเขาได้ตลอดเวลาด้วยกุญแจหลักของเขา
- เส้นทางทางเลือก**: สำหรับทายาทที่ได้รับการแต่งตั้งแต่ละคน สคริปต์จะรวมคีย์สาธารณะของพวกเขากับการล็อกเวลา



ธุรกรรมของเจ้าของแต่ละรายจะเลื่อนวันที่เริ่มต้นของข้อสืบทอดออกไปโดยอัตโนมัติ ในกรณีที่ไม่มีการใช้งานเป็นเวลานาน (เสียชีวิต, ไร้ความสามารถ) เงื่อนไขจะถูกกระตุ้นโดยอัตโนมัติ



## บริการมรดก (ไม่บังคับ)



Heritage มีตัวเลือกการใช้งานสองแบบ:



**ทำด้วยตัวเอง (ฟรี)**: แอปพลิเคชันโอเพนซอร์สเพียงอย่างเดียว คุณจัดการทุกอย่างด้วยตัวเองด้วยโหนดของคุณเอง ตัวเลือกนี้มีการเข้าถึงการสำรองข้อมูลในตัว การสืบทอดในตัว และการควบคุมบิตคอยน์ของคุณอย่างเต็มที่ อย่างไรก็ตาม คุณจำเป็นต้องสร้างการแจ้งเตือนของคุณเอง (ปฏิทิน, การเตือนความจำ) เพื่อไม่ให้ลืมต่ออายุการล็อกเวลา และไม่มีการแจ้งเตือนอัตโนมัติสำหรับทายาทของคุณ



**ใช้บริการ (0.05% ต่อปี)** : บริการ btc-heritage.com เพิ่มคุณสมบัติเพื่อให้ง่ายต่อการจัดการ:




- การแจ้งเตือนอัตโนมัติก่อนที่กำหนดเวลาของคุณจะหมดอายุ
- การแจ้งเตือนอัตโนมัติถึงทายาทเพื่อแนะนำพวกเขาผ่านกระบวนการกู้คืน
- การสนับสนุนลำดับความสำคัญ
- การจัดการตัวบรรยายแบบง่าย



ค่าธรรมเนียม: 0.05% ของจำนวนเงินที่จัดการต่อปี ขั้นต่ำ 0.5 mBTC/ปี ปีแรกฟรี



บริการยังคงเป็นแบบไม่รับฝาก: กุญแจส่วนตัวของคุณจะไม่ออกจากอุปกรณ์ของคุณ Heritage ไม่สามารถเข้าถึงเงินทุนของคุณได้



## Heritage CLI



สำหรับผู้ใช้ขั้นสูงที่ชอบใช้เทอร์มินัล Heritage มีเครื่องมือบรรทัดคำสั่ง (CLI) สำหรับการควบคุมอย่างละเอียดและการทำงานของเครื่องที่แยกออกจากเครือข่าย



![Page Heritage CLI](assets/fr/03.webp)



เอกสาร CLI ฉบับเต็มสามารถดูได้ที่ [btc-heritage.com/heritage-cli](https://btc-heritage.com/heritage-cli) ที่นี่คุณจะพบคำแนะนำสำหรับการดาวน์โหลด การเชื่อมต่อกับบริการ การสร้างกระเป๋าเงิน (ด้วย Ledger หรือคีย์ท้องถิ่น) การจัดการทายาทและธุรกรรมต่างๆ



บทแนะนำนี้มุ่งเน้นไปที่แอปพลิเคชันเดสก์ท็อป ซึ่งเข้าถึงได้ง่ายกว่าสำหรับผู้ใช้ส่วนใหญ่



## Heritage Desktop



Heritage Desktop เป็นแอปพลิเคชันกราฟิกที่มีอินเทอร์เฟซที่ใช้งานง่าย ช่วยแนะนำผู้ใช้ในทุกขั้นตอนของกระบวนการกำหนดค่า



### ดาวน์โหลด



ไปที่ [btc-heritage.com](https://btc-heritage.com) และคลิกที่ "ดาวน์โหลดแอปพลิเคชัน"



![Page d'accueil Heritage](assets/fr/01.webp)



เลือกเวอร์ชันที่ตรงกับระบบปฏิบัติการของคุณ (Linux 64bits หรือ Windows 64bits) ไฟล์ไบนารีไม่ได้รับการลงนามแบบดิจิทัล ซึ่งอาจทำให้เกิดคำเตือนด้านความปลอดภัย



![Page de téléchargement](assets/fr/02.webp)



### การติดตั้งบน Linux



บน Linux แอปพลิเคชันถูกแจกจ่ายในรูปแบบ AppImage ก่อนที่คุณจะสามารถรันได้ คุณจำเป็นต้องติดตั้งการพึ่งพา `libfuse2`:



```bash
sudo apt install libfuse2
```



![Installation libfuse2](assets/fr/04.webp)



จากนั้นทำให้ไฟล์สามารถรันได้และเรียกใช้:



```bash
chmod +x Heritage-GUI-vX.X.X.AppImage
./Heritage-GUI-vX.X.X.AppImage
```



### เปิดตัวครั้งแรก



เมื่อเปิดใช้งานครั้งแรก วิซาร์ดการเริ่มต้นจะเสนอทางเลือกสามทางให้คุณ:



![Onboarding initial](assets/fr/05.webp)

- สำรวจด้วยตัวเอง**: สำรวจฟังก์ชันโดยไม่ต้องขอความช่วยเหลือ



เลือก "ตั้งค่า Heritage Wallet" เพื่อสร้าง wallet แรกของคุณ



### การเชื่อมต่อเครือข่าย Bitcoin



เลือกวิธีการเชื่อมต่อกับเครือข่าย Bitcoin:



![Choix connexion](assets/fr/06.webp)





- การใช้บริการ Heritage**: โครงสร้างพื้นฐานที่มีการจัดการ ง่ายขึ้นสำหรับทายาท
- การใช้โหนดของฉันเอง**: เชื่อมต่อกับโหนด Bitcoin Core หรือ Electrum ของคุณเอง



สำหรับบทแนะนำนี้ เรากำลังใช้โหนดของเราเอง



### การจัดการกุญแจส่วนตัว



เลือกวิธีการจัดการคีย์ส่วนตัวของคุณ:



![Gestion des clés](assets/fr/07.webp)





- ด้วยอุปกรณ์ฮาร์ดแวร์ Ledger** : ความปลอดภัยสูงสุดด้วยฮาร์ดแวร์ wallet (แนะนำ)
- ที่เก็บข้อมูลในเครื่องพร้อมรหัสผ่าน**: คีย์ที่เก็บในเครื่องพร้อมการป้องกันด้วยรหัสผ่าน
- กู้คืน wallet** ที่มีอยู่: กู้คืนจาก seed ที่มีอยู่



### การกำหนดค่าโหนด



หากคุณกำลังใช้โหนดของคุณเอง แอปพลิเคชันจะแนะนำคุณเกี่ยวกับการกำหนดค่า ตรวจสอบให้แน่ใจว่าโหนด Bitcoin หรือ Electrum ของคุณคือ :




- ติดตั้งและกำลังทำงาน
- ซิงโครไนซ์กับเครือข่าย Bitcoin
- กำหนดค่าให้ยอมรับการเชื่อมต่อ RPC (สำหรับ Bitcoin Core)



![Configuration nœud](assets/fr/08.webp)



คลิกที่ "My Bitcoin node is up and running" เมื่อโหนดของคุณพร้อมใช้งานแล้ว



### แผงสถานะ



คลิกที่ "Status" ที่มุมขวาบน จากนั้นเลือก "Open Configuration" เพื่อเข้าถึงพารามิเตอร์การเชื่อมต่อ



![Panneau Status](assets/fr/09.webp)



ตั้งค่า URL ของเซิร์ฟเวอร์ Electrum ของคุณ (เช่น `umbrel.local:50001` หากคุณใช้ Umbrel)



![Configuration Electrum](assets/fr/10.webp)



### การสร้าง wallet



เมื่อการเชื่อมต่อได้รับการสร้างขึ้นแล้ว ให้คลิกที่ "Create Wallet" ในแท็บ WALLETS



![Créer wallet](assets/fr/11.webp)



ป๊อปอัพอธิบายสถาปัตยกรรมแบบแยกส่วนของ Heritage :



![Architecture split](assets/fr/12.webp)



1. **Key Provider (Offline)**: จัดการคีย์ส่วนตัวของคุณและลงนามธุรกรรม สามารถเป็นซอฟต์แวร์ Ledger หรือ wallet ได้


2. **ออนไลน์ Wallet**: จัดการการซิงโครไนซ์กับเครือข่าย Bitcoin การสร้างที่อยู่และการกระจายธุรกรรม



กรอกแบบฟอร์มการสร้าง :



![Formulaire création wallet](assets/fr/13.webp)





- Wallet ชื่อ**: ชื่อเฉพาะเพื่อระบุ wallet ของคุณ
- ผู้ให้บริการหลัก**: เลือกการจัดเก็บคีย์ในเครื่องสำหรับบทแนะนำนี้
- ใหม่/กู้คืน**: เลือก "ใหม่" เพื่อ generate seed ใหม่
- จำนวนคำ**: 24 คำแนะนำสำหรับความปลอดภัยสูงสุด



ป้อนรหัสผ่านที่รัดกุมและเลือกตัวเลือกสำหรับ Online Wallet :



![Options Online Wallet](assets/fr/14.webp)





- โหนดท้องถิ่น**: ใช้โหนด Core Electrum หรือ Bitcoin ของคุณเอง
- บริการ Heritage**: ใช้บริการ Heritage (แนะนำสำหรับฟังก์ชันการแจ้งเตือน)



คลิกที่ "Create Wallet" เพื่อเสร็จสิ้นการสร้าง



### Interface จาก wallet

![Interface wallet](assets/fr/15.webp)

### สร้างทายาท



ไปที่แท็บ "HEIRS" เพื่อสร้างทายาทของคุณ



![Page Heirs](assets/fr/16.webp)



ป๊อปอัพข้อมูลอธิบายว่า:




- ทายาทคือคีย์สาธารณะ Bitcoin ที่เชื่อมโยงกับบุคคล
- ทายาททุกคนมีวลีช่วยจำของตนเอง
- ทายาทคนแรกควรเป็น "สำรอง" สำหรับตัวคุณเอง (ในกรณีที่สูญเสีย wallet หลักของคุณ)



#### การสร้างทายาทสำรอง



คลิกที่ "Create Heir" และตั้งชื่อว่า "Backup"



![Création héritier Backup](assets/fr/17.webp)



ป๊อปอัพอธิบายว่าทำไมประโยค 12 คำที่ไม่มี passphrase ถึงปลอดภัยสำหรับทายาท:


1. **ไม่มีการเข้าถึงทันที**: กุญแจทายาทไม่สามารถเข้าถึงเงินทุนได้จนกว่าการล็อคเวลาจะหมดอายุ


2. **การตรวจจับการประนีประนอม**: หากมีผู้เข้าถึงวลีนี้ คุณสามารถอัปเดตการกำหนดค่า Heritage ได้


3. **การเข้าถึงในระยะยาว**: passphrase อาจถูกลืมหลังจากผ่านไปหลายปี



กำหนดทายาท :



![Configuration héritier](assets/fr/18.webp)





- Key Provider** : Local Key Storage
- ใหม่**: สร้าง seed ใหม่
- จำนวนคำ**: 12 คำ



สรุปการสร้าง :



![Finalisation héritier](assets/fr/19.webp)





- ประเภททายาท**: คีย์สาธารณะที่ขยายออก
- ส่งออกไปยังบริการ**: ตัวเลือก, เปิดใช้งานการแจ้งเตือนอัตโนมัติแก่ทายาท



ทายาทสำรองถูกสร้างขึ้นแล้ว:



![Héritier créé](assets/fr/20.webp)



#### การบันทึกวลีช่วยจำของทายาท



คลิกที่ตัวสำรองทายาทแล้วคลิกที่ "แสดง Mnemonic" :



![Afficher mnemonic](assets/fr/21.webp)



**สำคัญ: จดบันทึกคำ 12 คำนี้และเก็บไว้ในที่ปลอดภัย นี่คือกุญแจสำคัญในการกู้คืนเงินทุน



![Phrase mnémotechnique](assets/fr/22.webp)



#### กำลังลบ seed ออกจากแอปพลิเคชัน



เมื่อคุณเขียนวลีลงไปแล้ว ให้เข้าถึงพารามิเตอร์ของทายาท (ไอคอนรูปเฟือง):



![Paramètres héritier](assets/fr/23.webp)



ใช้ "Strip Heir Seed" เพื่อลบคีย์ส่วนตัวออกจากแอปพลิเคชัน ยืนยันว่าคุณได้บันทึกวลีแล้ว



![Strip Heir Seed](assets/fr/24.webp)



นี่คือมาตรการรักษาความปลอดภัย: มีเพียงกุญแจสาธารณะที่ยังคงอยู่ในแอปพลิเคชัน ซึ่งเพียงพอสำหรับการกำหนดค่าการสืบทอด



#### การสร้างทายาทคนที่สอง



ทำซ้ำกระบวนการเพื่อสร้างทายาทคนที่สอง (เช่น "Satoshi") ตอนนี้คุณจะมีทายาทสองคน:

- สำรองข้อมูล**: กุญแจฉุกเฉินส่วนตัวของคุณ
- Satoshi**: ทายาทที่ได้รับการแต่งตั้ง

## แนวทางปฏิบัติที่ดีที่สุด



### บันทึกตัวอธิบาย



ตัวบ่งชี้ wallet มีความสำคัญอย่างยิ่งในการสร้างที่อยู่ Heritage ของคุณใหม่ **หากไม่มีตัวบ่งชี้ แม้ว่าคุณจะมีวลีช่วยจำ คุณก็จะไม่สามารถค้นหาเงินทุนของคุณได้**



### ความปลอดภัยของกุญแจ





- ใช้ Ledger เป็นกุญแจหลักถ้าเป็นไปได้
- อย่าเก็บประโยคของทายาทไว้ในที่เดียวกับของคุณเอง
- กระจายข้อมูลผ่านสื่อและสถานที่หลายแห่ง



### เอกสารสำหรับคนที่คุณรัก



เขียนคำแนะนำที่ชัดเจนโดยอธิบายแต่ละขั้นตอนของกระบวนการกู้คืน ทายาทของคุณอาจไม่คุ้นเคยกับ Bitcoin ในช่วงเวลาสำคัญ
