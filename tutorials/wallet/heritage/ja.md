死亡または能力喪失時にビットコインを受け継ぐことは、暗号資産保有者にとって大きな課題です。適切な相続プランがなければ、これらの資産はあなたの愛する人にとって回復不可能となります。



ヘリテージは、Bitcoinブロックチェーンに直接デッドマンスイッチメカニズムを実装することで、エレガントな回答を提供している。このオープンソースのwalletは、on-chainの継承条件を設定することを可能にする：所有者が定義された期間、それ以上の取引を行わない場合、事前に指定された代替キーが資金を解放することができる。



## ヘリテージとは何か？



Heritageは、Taprootスクリプトによる継承メカニズムをネイティブに統合したBitcoinポートフォリオです。ジェレミー・ロドンによってMITライセンスの下で開発されたこのオープンソースソフトウェアは、透明性と耐久性を保証します。



ヘリテージはBitcoinアドレスにエンコードされたTaprootスクリプトを使用する。各UTXOは2種類の支出条件を統合している：





- プライマリパス** ：所有者はプライマリ・キーでいつでもビットコインを使うことができる。
- 代替パス**：指定された各相続人について、スクリプトがその公開鍵とタイムロックを組み合わせる。



各所有者の取引により、相続条項の発動日が自動的に延期される。長期不在（死亡、無能力）の場合、条件は自動的に発動される。



## ヘリテージ・サービス（オプション）



ヘリテージには2つの使用方法がある：



**自分でやる（無料）**：オープンソースのアプリケーションのみ。あなた自身のノードで自律的にすべてを管理します。このオプションは、組み込みのバックアップアクセス、組み込みの継承、ビットコインの排他的なコントロールを提供します。しかし、タイムロックの更新を忘れないように、自分でアラート（カレンダー、リマインダー）を作成する必要があり、相続人への自動通知はありません。



**サービスをご利用ください（年間0.05%）** ：btc-heritage.comサービスは、管理を簡素化するための機能を追加します：




- 期限切れの前に自動リマインダー
- 相続人への自動通知で復旧プロセスを案内
- 優先サポート
- 簡素化された記述子管理



手数料：年間管理額の0.05%、最低0.5mBTC/年。初年度無料。



秘密鍵がお客様のデバイスから離れることはありません。ヘリテージはあなたの資金にアクセスできません。

ターミナルを好む上級ユーザーのために、ヘリテージは、きめ細かなコントロールとエアギャップマシン操作のためのコマンドラインツール（CLI）を提供しています。

Linux では、アプリケーションは AppImage 形式で配布されています。実行する前に、依存関係 `libfuse2` をインストールする必要がある：



```bash

```

そして、そのファイルを実行可能ファイルにして実行する：



```bash

./Heritage-GUI-vX.X.X.AppImage
```

## ベストプラクティス



### ディスクリプタの保存



walletディスクリプタは、あなたのヘリテージアドレスを再構築するために不可欠です。 **ディスクリプターがなければ、たとえニーモニックフレーズがあっても、資金を見つけることはできない。



### キーセキュリティ





- 可能であれば、メインキーにはLedgerを使用する。
- 相続人の文章を自分の文章と同じ場所に保管しない。
- 複数のメディアや場所に情報を広める



### 大切な人のためのドキュメント



リカバリープロセスの各ステップを説明する明確な説明書を書いてください。相続人は肝心な時にBitcoinのことをよく知らないかもしれません。

## 結論



ヘリテージは、デスクトップ・アプリケーションを通じて、Bitcoinの後継者育成を支援します。実施には、適切な期間と秘密の確保について熟考する必要があります。相続人への継承をお忘れなく：




- 彼らの12語のニーモニック・フレーズ
- ディスクリプタのバックアップファイル
- リカバリーの指示
