Gutanga bitcoins mu gihe umuntu apfuye canke atagira ubushobozi, bigereranya ingorane ikomeye ku muntu wese afise crypto-asset. Iyo ata mugambi ukwiye w’iragi, ivyo bintu bica bihinduka ibidashobora gusubirwamwo ku bakunzi bawe.



Iragi ritanga inyishu nziza cane mu gushiramwo uburyo bwo guhindura umuntu yapfuye ataco akora ku rubuga rwa Bitcoin. Iyi wallet y’inkomoko yuguruye ishoboza ivyangombwa vyo gukurikirana on-chain gutunganirizwa: iyo nyen’ivyo bikoresho ata bindi bikorwa akora mu kiringo kinaka, imfunguruzo zindi zategekanijwe imbere y’igihe zirashobora kurekura ayo mahera.



## Iragi ni iki?



Iragi ni igitabu ca Bitcoin gishiramwo uburyo bwo kuragwa biciye mu nyandiko za Taproot. Iyi porogarama yakozwe n’uruhusha rwa MIT na Jérémie Rodon, ikaba itanga icemeza ko ikora neza kandi ikagumaho.



Heritage ikoresha inyandiko za Taproot zishizwe mu maderesi ya Bitcoin. Buri UTXO ishiramwo ubwoko bubiri bw’ibintu umuntu akoresha:





- Inzira ya mbere** : Nyir’iyi bitcoins ashobora gukoresha amafaranga yiwe igihe cose akoresheje urufunguzo rwiwe rwa mbere .
- Inzira zindi**: Ku muragwa wese yashinzwe, inyandiko ifatanya urufunguzo rwayo rwa bose n'igihe co gufunga



Igikorwa cose c’umunyabubasha gica gisubiramwo itariki y’ugutangura gukora kw’ingingo z’iragi. Iyo umuntu amaze igihe kirekire ataco akora (urupfu, ubumuga), ivyo bintu biraca bivyuka.



## Ibikorwa vy'iragi (ntibikenewe)



Iragi ritanga uburyo bubiri bwo gukoresha:



**Bikore wewe nyene (ku buntu)**: Porogaramu y'inkomoko yuguruye yonyene. Ucungera vyose wigenga n’urudodo rwawe bwite. Iyi nzira itanga uburyo bwo gushikira ububiko bwubatswemwo, iragi ryubatswemwo n’ubugenzuzi bwihariye bw’amahera yawe ya bitcoins. Ariko rero, ukeneye kwihingurira ivyibutsa (ikirangamisi, ivyibutsa) kugira ngo ntukibagire gusubiramwo amasaha yawe, kandi nta bimenyetso vyihuta vy’abazungu bawe.



**Koresha serivisi (0.05% ku mwaka)** : Serivisi ya btc-heritage.com yongerako ibintu vyo kworohereza uburongozi:




- Ivyibutsa vyihuta imbere y'uko iherezo ryawe rihera
- Amatangazo yihuta ku baragwa kugira ngo abayobore mu nzira yo gusubirana
- Infashanyo y'imbere
- Ubuyobozi bworoshe bw'ibisobanuro



Amafaranga: 0.05% y’amahera acungiwe ku mwaka, n’agaciro gatoyi 0.5 mBTC/umwaka. Umwaka wa mbere ku buntu.



Iryo shirahamwe riguma ritagira ububiko: imfunguruzo zawe z’ibanga ntizigera ziva ku gikoresho cawe. Iragi ntishobora gushika ku mahera yawe.



## Iragi CLI



Ku bakoresha bateye imbere bakunda terminal, Heritage itanga igikoresho c’umurongo w’amabwirizwa (CLI) co kugenzura ivyuma n’ugukoresha imashini ikoresheje umuyaga.



![Page Heritage CLI](assets/fr/03.webp)



Ivyanditswe vyose vya CLI biraboneka kuri [btc-iragi.com/iragi-cli](https://iragi.com/iragi-cli). Aha uzosanga amabwirizwa yo gukura, gufatanya na serivisi, gukora ama wallet (afise Ledger canke imfunguruzo zo mu karere), gucunga abazungu n’ibikorwa.



Iyi nyigisho yibanze ku gikorwa co kuri Desktop, gishobora gukoreshwa n’abakoresha benshi.



## Ibiro vy'iragi

### Gukurura



Genda kuri [https://iragi.com) hanyuma ukande kuri "Gukuraho ubusabe".



![Page d'accueil Heritage](assets/fr/01.webp)



Hitamwo verisiyo ihuye n’ivyo ukoresha (Linux 64bits canke Windows 64bits). Binaries ntizishirwako umukono mu buryo bwa digitale, ivyo bishobora gutuma haba imburi z’umutekano.



![Page de téléchargement](assets/fr/02.webp)



### Gushiramwo kuri Linux



Kuri Linux, porogaramu ikwiragizwa mu buryo bwa AppImage. Imbere y'uko ushobora kuyikoresha, ukeneye gushiramwo `libfuse2`:



```bash
sudo apt install libfuse2
```



![Installation libfuse2](assets/fr/04.webp)



Hanyuma ushire dosiye ishobora gukorwa maze uyikoreshe:



```bash
chmod +x Heritage-GUI-vX.X.X.AppImage
./Heritage-GUI-vX.X.X.AppImage
```

## Ibikorwa vyiza



### Kubika ibisobanuro



Ibisobanuro vya wallet ni ngirakamaro mu gusubira kwubaka amaderesi yawe y’Iragi. **Utagira abadondora, mbere n'ijambo ryawe ry'ukwibuka, ntuzoshobora kuronka amahera yawe.



### Umutekano w'urufunguzo





- Koresha Ledger ku rufunguzo nyamukuru nimba bishoboka
- Ntukigere ubika amajambo y'abazungu ahantu hamwe n'ayawe
- Gukwiragiza amakuru mu binyamakuru vyinshi no mu bibanza vyinshi



### Inyandiko z'abakunzi bawe



Wandike amabwirizwa atomoye asigura intambwe yose y’ugukira. Abazungu bawe boshobora kuba batamenyereye Bitcoin mu gihe gikomeye.
