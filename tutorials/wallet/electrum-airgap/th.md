## Cold Wallet



ในบทแนะนำนี้ ฉันจะอธิบายวิธีการสร้างอุปกรณ์ลงนามแบบแอร์แก๊ปเครื่องแรกของคุณ โดยไม่เชื่อมต่อกับอินเทอร์เน็ต แม้จะไม่มี Hardware Wallet โดยเฉพาะ สิ่งที่คุณต้องมีคือคอมพิวเตอร์สองเครื่อง:




- อุปกรณ์เก่าที่จะถูกป้องกันไม่ให้เชื่อมต่อกับอินเทอร์เน็ตตลอดไป;
- คอมพิวเตอร์ที่คุณใช้ในชีวิตประจำวัน



การกำหนดค่านี้ช่วยให้มีความปลอดภัยในระดับที่สูงกว่าการใช้ `Hot Wallet` แบบคลาสสิก: คอมพิวเตอร์เครื่องเก่าที่ไม่มีการเชื่อมต่อเครือข่ายจะเป็นผู้เก็บรักษาคีย์ส่วนตัวของคุณ ซึ่งจะไม่ถูกเปิดเผยบนอินเทอร์เน็ต แต่จะถูกเก็บไว้แบบออฟไลน์ ("airgap" หรือ "Cold")



แทนที่จะทำเช่นนั้น คุณจะติดตั้งจอแสดงผล Wallet ("ดูอย่างเดียว") บนคอมพิวเตอร์ที่คุณใช้ประจำวัน ซึ่งเชื่อมต่อกับเครือข่ายและคุณสามารถใช้เพื่อตรวจสอบยอดคงเหลือและเตรียมธุรกรรมการรับเงินได้



## Wallet Airgap: คืออะไรและทำอย่างไร



โดยการทำตามขั้นตอนในคู่มือนี้ เราจะติดตั้ง Software Wallet และ Electrum บนคอมพิวเตอร์สองเครื่องที่แตกต่างกัน และสุดท้ายสร้าง Wallet สองอันที่มี keystores ต่างกัน: Wallet airgap จะใช้ลำดับชั้นทั้งหมดของ Wallet HD ในขณะที่ Wallet display จะถูกสร้างขึ้นด้วย master public key



กระเป๋าเงินสองใบนี้จะมีความแตกต่างกันอย่างสิ้นเชิงในทุกๆ ด้าน สิ่งเดียวที่พวกมันจะมีเหมือนกัน, อย่างที่เราจะเห็น, คือที่อยู่:




- gW-13 บนคอมพิวเตอร์ที่แยกออกจากเครือข่ายสามารถลงนามได้เท่านั้น แต่เมื่อถูกตัดการเชื่อมต่อจากเครือข่าย จะไม่ทราบยอดคงเหลือและที่อยู่ที่ใช้;
- Wallet บนคอมพิวเตอร์รายวันจะสามารถเตรียมและเผยแพร่ธุรกรรมได้เท่านั้น โดยไม่สามารถจัดการค่าใช้จ่ายได้ หากไม่มีคีย์ส่วนตัว



## การเตรียมการเบื้องต้น



ในการดาวน์โหลด Electrum ฉันขอแนะนำให้คุณทำตามขั้นตอนแรกในบทแนะนำนี้:



https://planb.academy/tutorials/wallet/desktop/electrum-efec9166-46b5-4937-8cee-6bc310975177

หลังจากดาวน์โหลดแล้ว ให้ตรวจสอบการเผยแพร่ก่อนติดตั้งเสมอ จากนั้นดำเนินการกำหนดค่า "One Server" ตามที่คุณจะพบในส่วนช่วยเหลือ ภายใต้ `Start with a Dummy Wallet`.



การกำหนดค่า "One Server" จำเป็นเฉพาะสำหรับ Wallet ที่ติดตั้งบนคอมพิวเตอร์ที่ใช้งานประจำวันเท่านั้น เนื่องจากคอมพิวเตอร์เครื่องอื่นจะออฟไลน์เสมอ



การดำเนินการต่อไปนี้เกี่ยวข้องกับการฝึกฝนบนคอมพิวเตอร์สองเครื่องที่แตกต่างกัน (และกระเป๋าเงิน) ดังนั้นเพื่อความสะดวกและความมุ่งมั่น ฉันเลือกตั้งค่า Wallet airgap ด้วยธีมสีอ่อน ในขณะที่หน้าจอ Wallet มีธีมสีเข้ม



## Wallet การสร้างช่องว่างอากาศ



หลังจากดาวน์โหลดและตรวจสอบการดาวน์โหลด Electrum แล้ว ให้ทำสำเนาไฟล์ปฏิบัติการและนำไปยังคอมพิวเตอร์ของคุณที่ออฟไลน์ จากนั้นเปิดและติดตั้ง Electrum



ดับเบิลคลิกเพื่อเริ่ม Electrum: คอมพิวเตอร์ที่คุณจะใช้ Wallet นี้ออฟไลน์อยู่ ให้ละเว้นการตั้งค่าเครือข่ายและไปที่การสร้าง Wallet ซึ่งในคู่มือนี้เราจะเรียกว่า `airgap`.

## การสร้าง Wallet ของการแสดงภาพ



ตอนนี้ที่ Wallet ของคุณมีคีย์ส่วนตัวแบบออฟไลน์แล้ว คุณจำเป็นต้องตั้งค่าแสดงผล Wallet หรือ `watch-only` ซึ่งจะช่วยให้คุณสามารถดูยอดคงเหลือ รวมถึงเตรียมธุรกรรมการรับเพื่อสะสม Sats อย่างปลอดภัยต่อไป



จาก Wallet ที่อยู่บนอุปกรณ์ออฟไลน์ เลือกเมนู _Wallet_ -> _Information_



![image](assets/en/08.webp)



หน้าต่างที่มีข้อมูล Wallet ทั้งหมดของคุณจะปรากฏขึ้น ซึ่งคุณสามารถตรวจสอบ `derivation path` และ `master fingerprint` ได้ เช่น เพื่อทำเครื่องหมายไว้ข้างคำในประโยคช่วยจำ (แนะนำอย่างยิ่ง)



![image](assets/en/09.webp)



โปรดจำไว้ว่าคุณกำลังนำข้อมูลนี้จากคอมพิวเตอร์ที่ไม่ได้เชื่อมต่อ ดังนั้นคุณจะต้องคัดลอก/วาง `zpub` ไปยังไฟล์ข้อความและบันทึกลงใน USB stick



ตอนนี้คุณสามารถย้ายไปที่คอมพิวเตอร์ที่เชื่อมต่อกับอินเทอร์เน็ต เพื่อเปิดตัว Electrum และสร้าง Wallet ใหม่ได้แล้ว



จากเมนู _File_ เลือก _New/Restore_



![image](assets/en/10.webp)



Wallet ใหม่เป็นแบบดูอย่างเดียว ดังนั้นสำหรับคู่มือนี้เราจะเรียกมันว่า `watch-only`.



![image](assets/en/12.webp)



บนหน้าจอถัดไปเลือก _Standard wallet_ และดำเนินการต่อโดยคลิก _Next_.



![image](assets/en/13.webp)



ในการเลือก `Keystore` โปรดระวัง: เพื่อสร้างการแสดงผล Wallet ให้เลือก _Use a master key_ จากนั้นดำเนินการต่อด้วย _Next_



![image](assets/en/14.webp)



วาง `zpub` ที่คัดลอกจาก Wallet ออฟไลน์และที่คุณนำมายังคอมพิวเตอร์นี้ผ่านสื่อ USB ที่นี่



![image](assets/en/15.webp)



สรุปโดยการตั้งรหัสผ่านที่แข็งแกร่งสำหรับ Wallet ด้วย ซึ่งอาจจะแตกต่างจากรหัสผ่านที่เลือกสำหรับ Cold



คุณจะเห็นการแสดงผล Wallet ปรากฏขึ้นพร้อมกับคำเตือน ข้อความเตือนคุณว่านี่เป็นเพียงการแสดงผล Wallet เท่านั้นและคุณไม่สามารถใช้จ่ายเงินที่เกี่ยวข้องกับมันได้



**หมายเหตุ**: **คุณจะต้องมีคีย์ส่วนตัวเสมอเพื่อจัดการกับ UTXOs ของ Wallet นี้** ด้วยระบบสำรองข้อมูลที่ดี คุณจะสามารถครอบครอง Bitcoins ของคุณได้อย่างเต็มที่โดยไม่ยากเย็น



![image](assets/en/16.webp)



คำเตือนนี้จะปรากฏทุกครั้งที่คุณเปิด Wallet นี้ คลิก _ตกลง_ และไปยังขั้นตอนการตรวจสอบต่อไป



## การตรวจสอบ Wallet สองรายการ



ตามที่เราได้เรียนรู้ในตอนต้นของคู่มือนี้ Wallet airgap และหน้าจอแสดงผล Wallet เป็นกระเป๋าเงินสองใบที่มีความสามารถต่างกันแต่**ใช้ที่อยู่เดียวกัน**



หากเราดูที่กระเป๋าเงินทั้งสองใบเคียงข้างกัน เราจะสังเกตเห็นว่าในช่องว่างอากาศของ Wallet มีสัญลักษณ์ "seed" ในขณะที่ในแบบดูอย่างเดียวไม่มี แม้แต่รายละเอียดนี้ก็จะช่วยให้คุณจำได้ว่า Wallet แสดง Wallet ไม่มีคีย์ส่วนตัว



![image](assets/en/17.webp)



ในการตรวจสอบครั้งแรกอย่างแม่นยำ ให้เลือกเมนู `Addresses` ในทั้งสอง Wallets: เนื่องจากพวกเขาใช้ที่อยู่เดียวกัน รายการที่อยู่ควรจะเหมือนกันสำหรับทั้งสองฝ่าย



![image](assets/en/18.webp)



⚠️ **ATTENTION**: **ไม่มีทางสายกลาง; ที่อยู่ต้องเหมือนกัน ในกรณีที่แตกต่างกัน จำเป็นต้องลบงานทั้งหมดที่ทำไปแล้วและเริ่มต้นใหม่**.



ตอนนี้คุณสามารถดำเนินการตรวจสอบสองแบบที่แตกต่างกันได้แล้ว ขั้นแรก ลองลบกระเป๋าเงินทั้งสองและกู้คืนจากศูนย์ โดยแต่ละกระเป๋าเงินบนคอมพิวเตอร์ที่เหมาะสม ในกรณีที่คุณดำเนินการตรวจสอบนี้ ขั้นตอนสำหรับการแสดงผล Wallet จะเหมือนกับที่ระบุไว้ข้างต้น



สำหรับ Wallet airgap อย่างไรก็ตาม บนหน้าจอ `keystore` คุณจะต้องเลือก _I already have a seed_ และป้อนคำโดยการคัดลอกจากการสำรองข้อมูลบนกระดาษของคุณ



หลังจากการทดลอง "ไม่มีโหลด" สิ้นสุดลง คุณสามารถลองทำธุรกรรมจำนวนเล็กน้อยและใช้จ่ายทันที



## การรับและการใช้จ่ายธุรกรรม



ในการเริ่มต้นใช้งาน Electrum airgap ของคุณ คุณสามารถทำธุรกรรมรับเงินด้วยจำนวนเล็กน้อย จากนั้นใช้จ่ายไปยังที่อยู่ของคุณเอง คุณสามารถทำความคุ้นเคยกับขั้นตอนนี้ โดยตรวจสอบว่าคุณมีการควบคุมเงินทุนอย่างเต็มที่



**หมายเหตุ**: ฉันไม่แนะนำให้คุณฝากเงินจำนวนมากบน Wallet ก่อนที่คุณจะมั่นใจว่าคุณสามารถดำเนินการทั้งหมดได้อย่างราบรื่น



ขั้นตอนที่อธิบายด้านล่างนี้อาจดูซับซ้อนในตอนแรก อย่าปล่อยให้สิ่งนี้ทำให้คุณท้อใจ: เมื่อคุณได้ลองทำครั้งแรกแล้ว คุณจะพบว่ามันใช้เวลาเพียงไม่กี่นาทีในการทำให้เสร็จสิ้น



ในการรับเงิน คุณจำเป็นต้องใช้หน้าจอ Wallet ที่อยู่บนคอมพิวเตอร์ของคุณซึ่งเชื่อมต่อกับอินเทอร์เน็ต จากเมนู `Receive` คลิกที่ _Create request_ เพื่อให้ Electrum generate ที่อยู่แรกที่ใช้ได้และใช้มันเพื่อส่ง Satss ให้เราเล็กน้อย



![image](assets/en/19.webp)



![image](assets/en/20.webp)



เมื่อการทำธุรกรรมถูกเผยแพร่แล้ว คุณจะเห็นได้ว่า - ตามธรรมชาติ - มันจะปรากฏเฉพาะบนหน้าจอ Wallet และไม่ปรากฏบน Wallet airgap



![image](assets/en/21.webp)



หลังจากที่ธุรกรรมของคุณได้รับการยืนยันบางส่วนแล้ว คุณสามารถเตรียมค่าใช้จ่ายและลองขั้นตอนการลงนามจาก Wallet ที่อยู่นอกเครือข่าย จากนั้นเตรียมธุรกรรมบน watch-only และกด _Preview_ เพื่อตรวจสอบ



![image](assets/en/22.webp)



คุณจะได้รับหน้าต่างธุรกรรมขั้นสูงที่คุณสามารถเห็นว่า:




- ธุรกรรมไม่ได้ลงนาม (`Status: Unsigned);
- คำสั่ง `Sign` และ `Broadcast` ถูกยับยั้ง



สิ่งเดียวที่คุณสามารถทำได้คือการส่งออกธุรกรรมตามที่เป็นอยู่ เพื่อนำไปยัง Wallet airgap และลงนาม



แนะนำให้เสียบแฟลชไดร์ฟ USB เข้ากับคอมพิวเตอร์ของคุณ และจากเมนูที่มุมล่างซ้าย เลือก _Share_



![image](assets/en/23.webp)



หลังจากนั้นเลือก _บันทึกลงไฟล์_



![image](assets/en/24.webp)



บันทึกธุรกรรมลงในแฟลชไดร์ฟ USB.



คุณจะสังเกตเห็นว่า Electrum ให้ชื่อไฟล์ที่มีตัวเลขแรกของ transaction ID และนามสกุลไฟล์คือ `.PSBT` ซึ่งหมายถึง `Partially Signed Bitcoin Transaction`



แยกสื่อที่มีไฟล์ `.PSBT` และเชื่อมต่อกับคอมพิวเตอร์แบบออฟไลน์



จาก Wallet airgap ให้เลือกเมนู _Tools_ จากนั้นเลือก _Load transaction_ และตามด้วย From file_



![image](assets/en/25.webp)

ตอนนี้ที่การทำธุรกรรมได้ลงนามแล้ว จำไว้ว่าระบบ Wallet ของคุณอยู่บนเครื่องออฟไลน์ ดังนั้น-แม้ว่าคุณจะเห็นคำสั่ง `Broadcast` ทำงานอยู่-ระบบ Wallet ของคุณจะไม่สามารถเผยแพร่การทำธุรกรรมไปยังเครือข่าย Bitcoin ได้



สิ่งที่คุณต้องทำตอนนี้คือทำการส่งออกธุรกรรมที่ลงนามแล้วไปยัง USB stick ซ้ำอีกครั้ง เพื่อที่คุณจะสามารถนำเข้าไปยังคอมพิวเตอร์ที่เชื่อมต่อกับอินเทอร์เน็ตและเผยแพร่ได้

## บทสรุป



การทดสอบของคุณเสร็จสิ้นแล้ว หากคุณทำตามคู่มือและได้ผลลัพธ์เดียวกัน คุณได้สร้าง Wallet Cold ด้วย Electrum บนคอมพิวเตอร์สองเครื่องที่แตกต่างกัน ซึ่งคุณสามารถใช้ในรูปแบบ airgap เพื่อเก็บบิตคอยน์ของคุณได้



สิ่งเดียวที่คุณจะต้องให้ความสนใจอย่างใกล้ชิดมีสองอย่าง:


1) **อย่าใช้ Wallet airgap กับที่อยู่รับ generate** เนื่องจากมันออฟไลน์ มันจะเสนอที่อยู่แรกให้คุณเสมอ ซึ่งตรงกับที่อยู่ที่คุณเพิ่งใช้ทำธุรกรรมทดสอบ;



![image](assets/en/33.webp)



_ตามที่คุณเห็นจากภาพด้านบน Wallet ออฟไลน์ไม่ทราบประวัติที่อยู่ของตัวเอง มันตาบอดสนิทในเรื่องนี้ **งานเดียวที่มันสามารถทำให้คุณได้คือการเก็บกุญแจออฟไลน์ของคุณและเซ็นธุรกรรมของคุณ**_.



2) ใช้แฟลชไดร์ฟ USB ที่มีไว้สำหรับวัตถุประสงค์นี้เท่านั้น **อย่าใช้สื่อที่คุณใช้บ่อยๆ** เครื่องมือที่ใช้ในชีวิตประจำวันมีโอกาสถูกโจมตีทางไซเบอร์มากกว่า และโดยไม่ตั้งใจ คุณอาจโจมตีแม้กระทั่งคอมพิวเตอร์ที่คุณเก็บไว้ไม่ให้เชื่อมต่อกับเครือข่าย แฟลชไดร์ฟ USB ที่คุณใช้เฉพาะสำหรับวัตถุประสงค์นี้มีโอกาสน้อยมากที่จะติดต่อกับพีซีของคุณทางออนไลน์ โดยเฉพาะถ้าคุณเป็น hodler ที่ไม่ต้องใช้จ่าย จึงลดโอกาสในการรับและส่งไวรัส มัลแวร์ ฯลฯ
