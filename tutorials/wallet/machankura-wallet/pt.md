A África está a avançar, a inovar e a construir, mas continua a enfrentar numerosos desafios, nomeadamente a conetividade à Internet e o acesso a serviços financeiros básicos.


De acordo com a União Internacional das Telecomunicações, em 2024, a taxa de penetração da Internet em África (o rácio entre o número de pessoas que utilizam a Internet e a população total do continente) será de 38%, em comparação com 68% a nível mundial.



No que diz respeito ao acesso aos serviços financeiros, os serviços de transferência de dinheiro (dinheiro móvel) registaram um boom no continente, na sequência das dificuldades de acesso aos serviços de crédito bancário.


Perante esta situação, Kgothatso Ngako, um programador sul-africano, criou uma solução revolucionária que permite a este segmento da sociedade africana sem banco e sem Internet beneficiar do Bitcoin com o seu projeto chamado **Machankura**.



## O QUE É MACHANKURA?



Machankura, calão sul-africano para "dinheiro", é um projeto cuja visão é ligar a África rural ao ecossistema Bitcoin.



Trata-se de um Wallet de custódia que permite enviar e receber bitcoins no Lightning Network através da tecnologia USSD. O Machankura funciona em todos os telemóveis, mesmo os mais básicos (Symbian), que utilizem a rede GSM tradicional.


O **8333.mobi** que normalmente aparece na parte inferior do logótipo é um aceno subtil. De facto, **8333** é a porta predefinida utilizada pelos nós Bitcoin para comunicarem entre si. Machankura mostra que África não está à margem desta revolução e é uma terra de inovação neste domínio.



O domínio **.mobi** foi concebido para serviços móveis que funcionam em telefones básicos através de USSD. O **8333.mobi** resume, portanto, as caraterísticas especiais da solução Machankura.



Antes de continuar com este tutorial, vamos dar uma vista de olhos ao USSD, a tecnologia que permite que o Machankura seja utilizado sem a Internet.



## USSD



A tecnologia USSD (*Unstructured supplementary Service Data*) é geralmente utilizada pelos operadores móveis GSM. Trata-se de uma funcionalidade que permite comunicar com um serviço remoto, mesmo sem Internet, utilizando códigos específicos e selecionando uma opção de acordo com as possibilidades oferecidas pelo serviço.



Num processo de comunicação USSD, imagine-se a digitar um código especial como `*123#` no seu telefone. Ao executar este código USSD, envia um pedido específico à sua rede GSM para aceder ao serviço associado a este código.



Assim, pode utilizar o USSD para realizar acções como consultar o seu pacote Internet, o saldo do crédito de chamadas, etc.



É através deste mesmo sistema de comunicação que a banca móvel e as transferências de dinheiro móvel, como o M-PESA, o MTN Mobile Money, etc., se desenvolveram no continente.


Como dissemos anteriormente, desenvolvido especificamente para África, este serviço funciona em todos os telefones, sem necessidade de qualquer configuração técnica complexa ou ligação à Internet.



## Inovação Machankura



Ao contrário de uma solução Wallet convencional, a Machankura conseguiu utilizar a tecnologia USSD para resolver um problema real enfrentado pela população de África: a cobertura da Internet.



O Machankura é um serviço desenvolvido e associado a um código GSM que permite aos utilizadores receber e enviar bitcoins sem Internet. É possível utilizar o serviço nos seguintes países, utilizando o código USSD associado:



| PAYS           | CODE USSD              |
| -------------- | ---------------------- |
| Ghana          | `*920*8333#`           |
| Kenya          | `*483*8333#`           |
| Malawi         | `*384*8333#`           |
| Namibie        | `*142*8333#`           |
| Nigeria        | `*347*8333#`           |
| Afrique du Sud | `54052.co.za`          |
| Tanzanie       | `SMS +255 679 066 977` |
| Ouganda        | `SMS +256 744 830 624` |
| Zambie         | `*384*8333#`           |

Com base nesta tabela, podemos ver que países como a Tanzânia, o Uganda e a África do Sul não têm um código USSD específico para o serviço.



No entanto, a Machankura está a resolver este problema alargando a sua funcionalidade através do seu sítio Web, de mensagens SMS e do WhatsApp.



Para ser informado sobre os novos países onde o serviço estará disponível, consulte regularmente o [sítio Web](https://8333.mobi).



Este tutorial passo a passo explica como utilizar o Machankura, primeiro num telemóvel básico sem Internet e depois num smartphone.



## Funcionamento sem smartphone (via USSD)



### Criar o seu portefólio



Para a sua primeira ligação :




- Introduza o código USSD correspondente ao seu país;
- Prima o botão de chamada para iniciar.



O sistema pedir-lhe-á que crie um código PIN de 5 dígitos




- Escolha um código PIN seguro (5 dígitos).
- Confirme o seu código PIN.
- A sua carteira Bitcoin é criada instantaneamente.



Este Bitcoin Wallet está associado ao seu número de telefone. O código PIN cuidadosamente escolhido encriptará o seu Wallet e será também utilizado para confirmar todas as suas futuras transacções em Machankura.


Uma vez criada a sua carteira, acede ao menu principal com as seguintes opções:



1. Enviar bitcoins


2. Receber bitcoins


3. Detalhes da conta


4. Compra de bens/serviços


5. Alterar/redefinir o código PIN


6. Saída



Para aceder a qualquer opção do menu principal, basta premir o número de sequência a que está associada.


Para aceder aos "detalhes da conta", selecionar a opção **3** e premir a tecla de chamada para iniciar.



### Segurança e boas práticas



Machankura é um Lightning Wallet de custódia, pelo que os seus bitcoins são geridos através de nós Machankura. A sua conta é protegida pelo seu código PIN escolhido.





- Não defina um código PIN associado a uma data importante da sua vida.
- Nunca partilhe o seu código PIN com ninguém.
- Memorize o seu código PIN.
- Verifique sempre os números de telefone antes de os enviar.
- Manter um equilíbrio razoável para a utilização quotidiana.



O seu Wallet está associado ao seu número de telefone:




- Se perder o seu telefone, contacte o apoio Machankura;
- Mantenha um registo das suas transacções importantes.



### Receber bitcoins





- Selecionar o número de ordem da opção "Receber Bitcoins" no menu principal.
- O seu número de telefone funciona como o seu Address público.
- Partilhe o seu número com a pessoa que quer enviar-lhe bitcoins.



#### Relâmpago personalizado Address



A cada conta Machankura é atribuído um Lightning Address com base no seu número Machankura


`votrenumero@8333.mobi`


Exemplo: um número +2371234567890 tornar-se-á `+2371234567890@8333.mobi`


No entanto, pode escolher um nome de utilizador personalizado para substituir o número (por exemplo, `Satoshi@8333.mobi`).


Qualquer pessoa com o seu Lightning Address pode enviar-lhe bitcoins sem saber o seu número de telefone.



#### Recarregar a sua carteira Machankura



Para além de receber bitcoins de outro Lightning Wallet, pode reabastecer o seu Machankura Wallet com **Azteco** e **1Voucher** do **Flash Group**.



**Azteco** e **Flash Group** são duas empresas que oferecem **vouchers** de Bitcoin, ou seja, serviços de vouchers pré-pagos que podem ser comprados online ou a revendedores para obter Bitcoin sem passar por uma plataforma de Exchange. Estes cupões funcionam como **cartões de oferta**. Existem **vales On-Chain** e **Lightning**.



![azteco](assets/fr/01.webp)



O processo é simples:





- Compra-se um vale para um montante fixo;
- Recebe um código de **16 dígitos** por correio eletrónico ou num pequeno Invoice.



![bonazteco](assets/fr/02.webp)





- Introduza o código USSD MACHANKURA do seu país no seu telefone.
- Aguarde que apareça o menu principal.
- Selecione o número de ordem da opção "Receber bitcoins".
- Introduza o código de 16 dígitos (o código de referência do vale Azteco na parte inferior do vale ou o código PIN do 1Voucher no Invoice).



Bitcoins equivalentes ao valor do voucher comprado são adicionados diretamente ao seu Machankura Wallet.



### Enviar Bitcoins





- Aceder ao menu principal.
- Selecione a opção "Enviar Bitcoin" introduzindo o seu número de encomenda.
- Introduza o número de telefone do destinatário, o Lightning Address ou o nome de utilizador Machankura.
- Em seguida, introduzir o montante a enviar em Sats.
- Confirme com o seu código PIN.



Machankura envia então os bitcoins via Lightning Network numa questão de segundos. O remetente também pode enviar bitcoins noutras moedas, incluindo moedas locais.



Se o número do destinatário ainda não tiver sido registado como utilizador, a Machankura efectuará um pré-registo e creditará a conta do destinatário.



É enviada automaticamente uma mensagem para o número pré-registado e o utilizador recebe os seus bitcoins depois de confirmar o seu registo no serviço.

### Verificar o seu saldo





- Selecionar o número de ordem da opção "Saldo" no menu principal.
- O seu saldo é apresentado em satoshis e equivalentes na moeda local.



### Histórico de transacções





- Selecione o número de sequência da opção ''Histórico'' no menu principal.

## Utilizar com um smartphone



A Machankura escolheu o WhatsApp como plataforma intermediária para a utilização de smartphones.



Há duas razões principais para isso. A primeira é tornar a solução mais acessível aos africanos, sem os obrigar a instalar uma aplicação: uma aplicação que poderiam facilmente desinstalar se o telemóvel ficasse sem memória de armazenamento.



A segunda, que é uma extensão lógica da primeira, é ser uma solução inclusiva e local, utilizando uma aplicação que é amplamente utilizada pelos africanos nas suas trocas diárias: O WhatsApp.
