아프리카는 발전과 혁신, 건설을 거듭하고 있지만 인터넷 연결과 기본적인 금융 서비스에 대한 접근성 등 수많은 도전에 직면해 있습니다.


국제전기통신연합에 따르면 2024년까지 아프리카의 인터넷 보급률(대륙 전체 인구 대비 인터넷 사용 인구 비율)은 전 세계 68%에 비해 38%에 그칠 것으로 예상됩니다.



금융 서비스 접근성과 관련해서는 은행 신용 서비스 이용이 어려워지면서 송금 서비스(모바일 머니)가 아프리카 대륙에서 붐을 이루고 있습니다.


이러한 상황에 직면한 남아프리카의 개발자 Kgothatso Ngako는 **마찬쿠라**라는 프로젝트를 통해 은행이 없고 인터넷이 없는 아프리카 사회의 일부가 Bitcoin의 혜택을 받을 수 있는 혁신적인 솔루션을 만들었습니다.



## 마찬쿠라란 무엇인가요?



'돈'을 뜻하는 남아공 속어인 마찬쿠라는 아프리카 농촌과 Bitcoin 생태계를 연결하는 것을 비전으로 하는 프로젝트입니다.



마찬쿠라는 USSD 기술을 통해 Lightning Network에서 비트코인을 주고받을 수 있는 커스터디 Wallet입니다. 마찬쿠라는 기존 GSM 네트워크를 사용하는 가장 기본적인 휴대폰(심비안 휴대폰)을 포함한 모든 휴대폰에서 작동합니다.


보통 로고 하단에 표시되는 **8333.mobi**는 미묘하게 고개를 끄덕이게 합니다. 실제로 **8333**은 Bitcoin 노드가 서로 통신할 때 사용하는 기본 포트입니다. 마찬쿠라는 아프리카가 이 혁명의 변방이 아니며, 이 분야에서 혁신의 땅이라는 것을 보여줍니다.



.mobi** 도메인은 USSD를 통해 기본 휴대폰에서 실행되는 모바일 서비스를 위해 설계되었습니다. 따라서 **8333.mobi**는 마찬쿠라 솔루션의 특별한 기능을 요약합니다.



이 튜토리얼을 계속 진행하기 전에 인터넷 없이도 마찬쿠라를 사용할 수 있게 해주는 기술인 USSD에 대해 살펴보겠습니다.



## USSD



USSD(*비구조화 보충 서비스 데이터*) 기술은 일반적으로 GSM 이동 통신사에서 사용합니다. 인터넷 없이도 특정 코드를 사용하고 서비스에서 제공하는 가능성에 따라 옵션을 선택하여 원격 서비스와 통신할 수 있는 기능입니다.



USSD 통신 과정에서 휴대폰에 `*123#`과 같은 특수 코드를 입력한다고 상상해 보세요. 이 USSD 코드를 실행하면 이 코드에 연결된 서비스에 액세스하기 위한 특정 요청을 GSM 네트워크에 전송합니다.



따라서 USSD를 사용하여 인터넷 패키지 상담, 통화 크레딧 잔액 조회 등의 작업을 수행할 수 있습니다.



이러한 통신 시스템을 통해 아프리카 대륙에서 모바일 뱅킹과 M-PESA, MTN 모바일 머니 등의 모바일 송금이 발전할 수 있었습니다.


앞서 말했듯이 아프리카를 위해 특별히 개발된 이 서비스는 복잡한 기술 구성이나 인터넷 연결 없이도 모든 휴대폰에서 작동합니다.



## 마찬쿠라 혁신



기존의 Wallet 솔루션과 달리 Machankura는 USSD 기술을 사용하여 아프리카 사람들이 직면한 실제 문제인 인터넷 커버리지를 Address로 전환할 수 있었습니다.



마찬쿠라는 사용자가 인터넷 없이도 비트코인을 주고받을 수 있도록 개발한 후 GSM 코드에 연결한 서비스입니다. 다음 국가에서는 연결된 USSD 코드를 사용하여 서비스를 이용할 수 있습니다:

하지만 마찬쿠라는 웹사이트, SMS 메시지, WhatsApp을 통해 기능을 확장하여 이 문제를 해결하고 있습니다.

## 스마트폰 없이 작동(USSD를 통해)



### 포트폴리오 만들기



첫 번째 연결의 경우 :




- 해당 국가에 해당하는 USSD 코드를 입력합니다;
- 시작하려면 통화 버튼을 누릅니다.



시스템에서 5자리 PIN 코드를 생성하라는 메시지가 표시됩니다




- 보안 PIN 코드(5자리)를 선택합니다.
- PIN 코드를 확인합니다.
- Bitcoin 포트폴리오는 즉시 생성됩니다.



이 Bitcoin Wallet은 전화번호와 연결되어 있습니다. 신중하게 선택한 PIN 코드는 Wallet을 암호화하며 향후 Machankura에서의 모든 거래를 확인하는 데에도 사용됩니다.


포트폴리오가 생성되면 다음 옵션이 있는 기본 메뉴에 액세스하게 됩니다:



1. 비트코인 보내기


2. 비트코인 받기


3. 계정 세부 정보


4. 상품/서비스 구매


5. PIN 코드 변경/재설정


6. 출력



메인 메뉴의 옵션에 액세스하려면 해당 옵션이 연결된 시퀀스 번호를 누르기만 하면 됩니다.


'계정 세부정보'에 액세스하려면 **3** 옵션을 선택하고 통화 키를 눌러 실행합니다.



### 안전 및 모범 사례



마찬쿠라는 라이트닝 Wallet 커스터디이므로 비트코인은 마찬쿠라 노드를 통해 관리됩니다. 회원님의 계정은 회원님이 선택한 PIN 코드로 보호됩니다.





- 인생의 중요한 날짜와 관련하여 비밀번호를 설정하지 마세요.
- 비밀번호를 다른 사람과 공유하지 마세요.
- 비밀번호를 외우세요.
- 보내기 전에 항상 전화번호를 확인하세요.
- 일상적인 사용을 위해 합리적인 균형을 유지하세요.



Wallet는 휴대폰 번호에 연결되어 있습니다:




- 휴대폰을 분실한 경우 Machankura 지원팀에 문의하세요;
- 중요한 거래를 추적하세요.



### 비트코인 받기





- 메인 메뉴에서 '비트코인 받기' 옵션의 주문 번호를 선택합니다.
- 휴대폰 번호는 공개 Address으로 사용됩니다.
- 비트코인을 보내려는 상대방과 번호를 공유합니다.



#### 커스텀 라이트닝 Address



각 마찬쿠라 계정에는 마찬쿠라 번호에 따라 라이트닝 Address가 할당됩니다


`votrenumero@8333.mobi`


예: +2371234567890 번호는 `+2371234567890@8333.mobi`이 됩니다


그러나 사용자 지정 사용자 이름을 선택하여 번호를 대체할 수 있습니다(예: `Satoshi@8333.mobi`).


라이트닝 Address만 있으면 누구나 전화번호를 몰라도 비트코인을 보낼 수 있습니다.



#### 마찬쿠라 포트폴리오 재로드



다른 라이트닝 Wallet에서 비트코인을 받는 것 외에도, **플래시 그룹**에서 **아즈테코**와 **1바우처**로 마찬쿠라 Wallet을 보충할 수 있습니다.



**아즈테코**와 **플래시 그룹**은 Bitcoin **바우처**, 즉 온라인 또는 리셀러를 통해 구매하여 Exchange 플랫폼을 거치지 않고 Bitcoin을 획득할 수 있는 선불 바우처 서비스를 제공하는 두 회사입니다. 이 바우처는 **기프트 카드**처럼 작동합니다. On-Chain**과 **라이트닝** 바우처가 있습니다.



![azteco](assets/fr/01.webp)



절차는 간단합니다:





- 정해진 금액의 바우처를 구매합니다;
- 16자리** 코드를 이메일 또는 작은 Invoice로 받게 됩니다.



![bonazteco](assets/fr/02.webp)





- 휴대폰에 해당 국가의 USSD 마찬쿠라 코드를 입력합니다.
- 메인 메뉴가 나타날 때까지 기다립니다.
- '비트코인 받기' 옵션의 주문 번호를 선택합니다.
- 16자리 코드(바우처 하단의 아즈테코 바우처 참조 코드 또는 GW-1의 23Voucher PIN 코드)를 입력합니다.



구매한 바우처 금액에 해당하는 비트코인이 마찬쿠라 Wallet에 직접 추가됩니다.



### 비트코인 보내기





- 메인 메뉴로 이동합니다.
- 주문 번호를 입력하여 'Bitcoin 보내기' 옵션을 선택합니다.
- 받는 사람의 전화번호, Lightning Address 또는 Machankura 사용자 아이디를 입력합니다.
- 그런 다음 Sats에 송금할 금액을 입력합니다.
- 비밀번호로 확인합니다.



그런 다음 마찬쿠라는 Lightning Network을 통해 몇 초 만에 비트코인을 전송합니다. 발신자는 현지 통화를 포함한 다른 통화로 비트코인을 보낼 수도 있습니다.



받는 사람의 번호가 아직 사용자로 등록되지 않은 경우, Machankura가 사전 등록한 다음 받는 사람의 계정에 크레딧을 적립합니다.



미리 등록한 번호로 메시지가 자동으로 전송되며, 사용자가 서비스 등록을 확인하면 비트코인을 받게 됩니다.

## 스마트폰과 함께 사용



마찬쿠라는 스마트폰 사용을 위한 중개 플랫폼으로 WhatsApp을 선택했습니다.



여기에는 두 가지 주요 이유가 있습니다. 첫 번째 단계는 휴대폰의 저장 메모리가 부족할 경우 쉽게 제거할 수 있는 애플리케이션을 강제로 설치하지 않고도 아프리카 사람들이 솔루션에 더 쉽게 액세스할 수 있도록 하기 위해서입니다.



첫 번째의 논리적 확장인 두 번째는 아프리카 사람들이 일상적인 교류에서 널리 사용하는 애플리케이션을 사용하여 포괄적이고 현지화된 솔루션이 되는 것입니다: WhatsApp입니다.
