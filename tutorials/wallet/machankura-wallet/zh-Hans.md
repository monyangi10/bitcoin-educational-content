非洲正在不断进步、创新和建设，但仍面临诸多挑战，特别是互联网连接和基本金融服务的获取。


根据国际电信联盟的数据，到 2024 年，非洲的互联网普及率（使用互联网的人数与非洲大陆总人口的比率）将达到 38%，而全球的这一比率为 68%。



在获取金融服务方面，继难以获得银行信贷服务之后，转账服务（移动支付）在非洲大陆蓬勃发展。


面对这种情况，南非开发商 Kgothatso Ngako 创造了一个革命性的解决方案，通过他的项目**Machankura**，让非洲社会中这部分没有银行账户、没有互联网的人也能从 Bitcoin 中受益。



## 什么是马千仓？



Machankura 是南非俚语，意为 "钱"，这个项目的愿景是将非洲农村与 Bitcoin 生态系统连接起来。



它是一种托管型 Wallet，可让您通过 USSD 技术在 Lightning Network 上发送和接收比特币。Machankura 适用于所有使用传统 GSM 网络的手机，即使是最基本的手机（塞班手机）。


通常出现在徽标底部的**8333.mobi**是一个微妙的点头。事实上，**8333** 是 Bitcoin 节点用于相互通信的默认端口。马钱库拉表明，非洲并没有置身于这场革命之外，而是在这一领域不断创新。



**.mobi**域名专为通过 USSD 在基本手机上运行的移动服务而设计。因此，**8333.mobi**概括了马钱库拉解决方案的特色。



在继续本教程之前，让我们先来了解一下 USSD，它是使 Machankura 可以在没有互联网的情况下使用的技术。



## USSD



USSD （*非结构化补充服务数据*）技术通常由 GSM 移动运营商使用。即使没有互联网，您也可以使用特定代码并根据服务提供的可能性选择选项，与远程服务进行通信。



在 USSD 通信过程中，想象一下在手机上键入一个特殊的代码，如 `*123#`。通过执行这个 USSD 代码，您可以向 GSM 网络发送一个特定请求，以访问与该代码相关联的服务。



因此，您可以使用 USSD 执行各种操作，如查询互联网套餐、话费余额等。



正是通过这一通信系统，M-PESA、MTN Mobile Money 等移动银行和移动转账业务在非洲大陆得到了发展。


正如我们前面所说，这项服务专为非洲开发，适用于所有手机，无需复杂的技术配置或互联网连接。

## 免智能手机操作（通过 USSD）



### 创建您的投资组合



对于您的首次连接 ：




- 输入您所在国家的 USSD 代码；
- 按呼叫按钮开始。



系统将要求您创建一个 5 位数的 PIN 码




- 选择安全 PIN 码（5 位数）。
- 确认 PIN 码。
- 您的 Bitcoin 投资组合即刻创建。



此 Bitcoin Wallet 与您的电话号码相关联。精心选择的 PIN 码将为您的 Wallet 加密，也将用于确认您今后在 Machankura 上的所有交易。


创建投资组合后，您将进入主菜单，其中有以下选项：



1.发送比特币


2.接收比特币


3.账户详情


4.购买商品/服务


5.更改/重置 PIN 码


6.输出



要访问主菜单中的任何选项，只需按下与之相关的序列号即可。


要访问 "账户详情"，请选择选项 **3**，然后按呼叫键启动。



### 安全和最佳做法



Machankura 是一个托管闪电 Wallet，因此您的比特币通过 Machankura 节点进行管理。您的账户由您选择的 PIN 码保护。





- 不要将 PIN 码与人生中的重要日子联系起来设置。
- 切勿与任何人共享您的 PIN 码。
- 记住您的 PIN 码。
- 发送前一定要核对电话号码。
- 保持日常使用的合理平衡。



您的 Wallet 与您的电话号码相连：




- 如果您丢失了手机，请联系 Machankura 支持中心；
- 跟踪您的重要交易。



### 接收比特币





- 在主菜单中选择 "接收比特币 "选项的订单号。
- 您的电话号码是您的公开 Address。
- 把你的号码告诉想给你发送比特币的人。



#### 定制闪电 Address



每个 Machankura 帐户都会根据您的 Machankura 编号分配一个 Lightning Address


`votrenumero@8333.mobi`


例如："+2371234567890 "将变为 "+2371234567890@8333.mobi"。


不过，您可以选择一个自定义用户名来代替数字（如 `Satoshi@8333.mobi`）。


任何拥有你的 Lightning Address 的人都可以向你发送比特币，而无需知道你的电话号码。



#### 重装您的马钱库拉投资组合



除了从另一个闪电 Wallet 接收比特币外，您还可以用 **Azteco** 和 **1Voucher** 从 **Flash Group** 补充您的 Machankura Wallet。



**Azteco**和**Flash Group**是两家提供 Bitcoin **代金券**的公司，即预付代金券服务，您可以在线购买或从经销商处购买，无需通过 Exchange 平台即可获得 Bitcoin。这些代金券的作用类似于**礼品卡**。有**On-Chain**和**闪电**券。



![azteco](assets/fr/01.webp)



过程很简单：





- 您购买的是固定金额的代金券；
- 您会通过电子邮件或小 Invoice 收到一个**16 位**码。



![bonazteco](assets/fr/02.webp)





- 在手机上输入您所在国家的 USSD MACHANKURA 代码。
- 等待主菜单出现。
- 选择 "接收比特币 "选项的订单号。
- 输入 16 位代码（凭单底部的 Azteco 凭单参考代码或 Invoice 上的 1Voucher PIN 码）。



与所购代金券金额等值的比特币将直接存入您的 Machankura Wallet。



### 发送比特币





- 进入主菜单。
- 输入订单号，选择 "发送 Bitcoin "选项。
- 输入收件人的电话号码、Lightning Address 或 Machankura 用户名。
- 然后在 Sats 中输入要发送的金额。
- 使用 PIN 码确认。



然后，Machankura 在几秒钟内通过 Lightning Network 发送比特币。发送者还可以发送其他货币的比特币，包括当地货币。



如果收件人的号码尚未注册为用户，马赞库拉将进行预注册，然后记入收件人的账户。



一条信息会自动发送到预先注册的号码上，用户一旦确认注册该服务，就会收到比特币。

## 与智能手机一起使用



Machankura 选择 WhatsApp 作为使用智能手机的中介平台。



这主要有两个原因。第一步是让非洲人更容易获得该解决方案，而不是强迫他们安装一个应用程序：如果手机存储内存不足，他们可以很容易地卸载该应用程序。



第二种解决方案是第一种解决方案的合理延伸，即使用非洲人在日常交流中广泛使用的应用程序，提供一种包容性的本地解决方案：WhatsApp.
