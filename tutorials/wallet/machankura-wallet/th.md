ตามข้อมูลของสหภาพโทรคมนาคมระหว่างประเทศ ภายในปี 2024 อัตราการเข้าถึงอินเทอร์เน็ตของแอฟริกา (อัตราส่วนของจำนวนคนที่ใช้อินเทอร์เน็ตต่อประชากรทั้งหมดของทวีป) อยู่ที่ 38% เมื่อเทียบกับ 68% ทั่วโลก



เกี่ยวกับการเข้าถึงบริการทางการเงิน บริการโอนเงิน (เงินมือถือ) ได้รับความนิยมอย่างมากในทวีปนี้ หลังจากที่มีความยากลำบากในการเข้าถึงบริการสินเชื่อธนาคาร


เมื่อต้องเผชิญกับสถานการณ์นี้ Kgothatso Ngako นักพัฒนาชาวแอฟริกาใต้ ได้สร้างโซลูชันที่ปฏิวัติวงการซึ่งช่วยให้กลุ่มคนในสังคมแอฟริกาที่ไม่มีบัญชีธนาคารและไม่มีอินเทอร์เน็ตสามารถได้รับประโยชน์จาก Bitcoin ด้วยโครงการของเขาที่เรียกว่า **Machankura**.



## MACHANKURA คืออะไร?



Machankura, คำสแลงของแอฟริกาใต้ที่หมายถึง "เงิน", เป็นโครงการที่มีวิสัยทัศน์ในการเชื่อมต่อชนบทของแอฟริกากับระบบนิเวศ Bitcoin



มันคือ wallet แบบดูแลรักษาที่ให้คุณส่งและรับบิตคอยน์บน Lightning Network ผ่านเทคโนโลยี USSD Machankura ทำงานบนโทรศัพท์ทุกเครื่อง แม้กระทั่งเครื่องที่พื้นฐานที่สุด (โทรศัพท์ Symbian) โดยใช้เครือข่าย GSM แบบดั้งเดิม


**8333.mobi** ที่มักปรากฏอยู่ด้านล่างของโลโก้เป็นการบอกใบ้ที่ละเอียดอ่อน จริงๆ แล้ว **8333** เป็นพอร์ตเริ่มต้นที่ใช้โดยโหนด Bitcoin ในการสื่อสารกัน Machankura แสดงให้เห็นว่าแอฟริกาไม่ได้อยู่ข้างสนามของการปฏิวัตินี้ และเป็นดินแดนแห่งนวัตกรรมในด้านนี้



โดเมน **.mobi** ถูกออกแบบมาสำหรับบริการมือถือที่ทำงานบนโทรศัพท์พื้นฐานผ่าน USSD ดังนั้น **8333.mobi** จึงสรุปคุณสมบัติพิเศษของโซลูชัน Machankura



ก่อนที่จะดำเนินการต่อกับบทแนะนำนี้ มาดูที่ USSD เทคโนโลยีที่ทำให้ Machankura สามารถใช้งานได้โดยไม่ต้องใช้อินเทอร์เน็ต



## USSD



เทคโนโลยี USSD (*Unstructured supplementary Service Data*) มักถูกใช้โดยผู้ให้บริการโทรศัพท์มือถือระบบ GSM เป็นฟีเจอร์ที่ช่วยให้คุณสามารถสื่อสารกับบริการระยะไกลได้ แม้ไม่มีอินเทอร์เน็ต โดยใช้รหัสเฉพาะและเลือกตัวเลือกตามความเป็นไปได้ที่บริการนั้นเสนอให้



ในกระบวนการสื่อสาร USSD ลองจินตนาการถึงการพิมพ์รหัสพิเศษเช่น `*123#` บนโทรศัพท์ของคุณ โดยการดำเนินการรหัส USSD นี้ คุณจะส่งคำขอเฉพาะไปยังเครือข่าย GSM ของคุณเพื่อเข้าถึงบริการที่เชื่อมโยงกับรหัสนี้



ดังนั้น คุณสามารถใช้ USSD เพื่อดำเนินการต่างๆ เช่น ตรวจสอบแพ็กเกจอินเทอร์เน็ต ยอดคงเหลือเครดิตการโทร เป็นต้น



เป็นผ่านระบบการสื่อสารเดียวกันนี้ที่การธนาคารผ่านมือถือและการโอนเงินผ่านมือถือเช่น M-PESA, MTN Mobile Money เป็นต้น ได้พัฒนาขึ้นในทวีปนี้


ตามที่เราได้กล่าวไว้ก่อนหน้านี้ บริการนี้ได้รับการพัฒนาเฉพาะสำหรับแอฟริกา ทำงานได้บนโทรศัพท์ทุกเครื่อง โดยไม่ต้องมีการตั้งค่าทางเทคนิคที่ซับซ้อนหรือการเชื่อมต่ออินเทอร์เน็ต

## การใช้งานโดยไม่ใช้สมาร์ทโฟน (ผ่าน USSD)



### สร้าง wallet ของคุณ



สำหรับการเชื่อมต่อครั้งแรกของคุณ :




- ป้อนรหัส USSD ที่ตรงกับประเทศของคุณ;
- กดปุ่มโทรเพื่อเริ่มต้น



ระบบจะขอให้คุณสร้างรหัส PIN 5 หลัก




- เลือก PIN โค้ดที่ปลอดภัย (5 หลัก)
- ยืนยันรหัส PIN ของคุณ
- Bitcoin wallet ของคุณถูกสร้างขึ้นทันที



Bitcoin wallet นี้เชื่อมโยงกับหมายเลขโทรศัพท์ของคุณ รหัส PIN ที่เลือกอย่างระมัดระวังจะเข้ารหัส wallet ของคุณและจะใช้เพื่อยืนยันธุรกรรมทั้งหมดในอนาคตของคุณบน Machankura


เมื่อสร้าง wallet ของคุณแล้ว คุณจะเข้าสู่เมนูหลักพร้อมตัวเลือกต่อไปนี้:



1. ส่งบิตคอยน์


2. รับบิตคอยน์


3. รายละเอียดบัญชี


4. การซื้อสินค้า/บริการ


5. เปลี่ยน/รีเซ็ตรหัส PIN


6. เอาต์พุต



ในการเข้าถึงตัวเลือกใด ๆ ในเมนูหลัก เพียงกดหมายเลขลำดับที่เชื่อมโยงกับตัวเลือกนั้น


ในการเข้าถึง "รายละเอียดบัญชี" ให้เลือกตัวเลือก **3** และกดปุ่มโทรเพื่อเปิดใช้งาน



### ความปลอดภัยและแนวทางปฏิบัติที่ดีที่สุด



Machankura เป็น Lightning wallet แบบดูแลรักษา ดังนั้นบิตคอยน์ของคุณจะถูกจัดการผ่านโหนด Machankura บัญชีของคุณได้รับการป้องกันด้วยรหัส PIN ที่คุณเลือก





- อย่าตั้งรหัส PIN ที่เกี่ยวข้องกับวันที่สำคัญในชีวิตของคุณ
- อย่าเปิดเผยรหัส PIN ของคุณให้ใครทราบ
- จดจำรหัส PIN ของคุณให้ดี
- ตรวจสอบหมายเลขโทรศัพท์ก่อนส่งเสมอ
- รักษายอดคงเหลือที่เหมาะสมสำหรับการใช้งานในชีวิตประจำวัน



wallet ของคุณเชื่อมโยงกับหมายเลขโทรศัพท์ของคุณ:




- หากคุณทำโทรศัพท์หาย ติดต่อฝ่ายสนับสนุน Machankura;
- ติดตามธุรกรรมสำคัญของคุณ



### รับบิตคอยน์





- เลือกหมายเลขคำสั่งของตัวเลือก "รับบิตคอยน์" ในเมนูหลัก
- หมายเลขโทรศัพท์ของคุณทำหน้าที่เป็นที่อยู่สาธารณะของคุณ
- แบ่งปันหมายเลขของคุณกับบุคคลที่ต้องการส่งบิตคอยน์ให้คุณ

อย่างไรก็ตาม คุณสามารถเลือกชื่อผู้ใช้ที่กำหนดเองเพื่อแทนที่หมายเลข (เช่น `Satoshi@8333.mobi`)


ใครก็ตามที่มีที่อยู่ Lightning ของคุณสามารถส่งบิตคอยน์ให้คุณได้โดยไม่ต้องรู้หมายเลขโทรศัพท์ของคุณ



#### กำลังโหลดใหม่ Machankura wallet



นอกเหนือจากการรับบิตคอยน์จาก Lightning wallet อื่น ๆ คุณสามารถเติม Machankura wallet ของคุณด้วย **Azteco** และ **1Voucher** จาก **Flash Group** ได้

### ส่งบิตคอยน์





- ไปที่เมนูหลัก
- เลือกตัวเลือก "ส่ง Bitcoin" โดยป้อนหมายเลขคำสั่งซื้อของมัน
- ป้อนหมายเลขโทรศัพท์ของผู้รับ ที่อยู่ Lightning หรือชื่อผู้ใช้ Machankura
- จากนั้นป้อนจำนวนเงินที่จะส่งใน Sats.
- ยืนยันด้วยรหัส PIN ของคุณ



Machankura จากนั้นส่งบิตคอยน์ผ่าน Lightning Network ในเวลาเพียงไม่กี่วินาที ผู้ส่งยังสามารถส่งบิตคอยน์ในสกุลเงินอื่น ๆ รวมถึงสกุลเงินท้องถิ่นได้อีกด้วย



หากหมายเลขของผู้รับยังไม่ได้ลงทะเบียนเป็นผู้ใช้ Machankura จะทำการลงทะเบียนล่วงหน้าและจากนั้นจะเติมเครดิตให้กับบัญชีของผู้รับของคุณ

## ใช้กับสมาร์ทโฟน



Machankura ได้เลือก WhatsApp เป็นแพลตฟอร์มตัวกลางสำหรับการใช้งานสมาร์ทโฟน



มีสองเหตุผลหลักสำหรับเรื่องนี้ ขั้นตอนแรกคือการทำให้โซลูชันนี้เข้าถึงได้ง่ายขึ้นสำหรับชาวแอฟริกัน โดยไม่บังคับให้พวกเขาติดตั้งแอปพลิเคชัน: แอปพลิเคชันที่พวกเขาสามารถถอนการติดตั้งได้ง่ายหากโทรศัพท์ของพวกเขามีหน่วยความจำเก็บข้อมูลไม่เพียงพอ



ประการที่สอง ซึ่งเป็นการขยายความจากประการแรก คือการเป็นโซลูชันที่ครอบคลุมและท้องถิ่น โดยใช้แอปพลิเคชันที่ชาวแอฟริกันใช้กันอย่างแพร่หลายในชีวิตประจำวัน: WhatsApp.
