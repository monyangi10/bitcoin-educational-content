非洲正在不斷進步、創新和建設，但仍面臨眾多挑戰，尤其是網際網路連線和基本金融服務的取得。


根據國際電信聯盟的資料，到 2024 年，非洲的網際網路滲透率 (使用網際網路的人數與非洲總人口的比率) 為 38%，而全球則為 68%。



在金融服務的取得方面，繼難以取得銀行信貸服務之後，匯款服務 (行動貨幣) 在非洲大陸蓬勃發展。


面對這種情況，南非開發人員 Kgothatso Ngako 創造了一個革命性的解決方案，讓非洲社會中這群沒有銀行、沒有網路的人也能從 Bitcoin 中獲益，他的專案名為 **Machankura**。



## 什麼是 Machankura？



Machankura 是南非俚語「錢」的意思，這個專案的願景是要將非洲農村與 Bitcoin 生態系統連結起來。



它是一個保管 Wallet，可讓您透過 USSD 技術在 Lightning Network 上傳送和接收比特幣。Machankura 適用於所有使用傳統 GSM 網路的手機，即使是最基本的手機 (Symbian 手機)。


通常出現在標誌底部的**8333.mobi**是一個微妙的點擊。事實上，**8333** 是 Bitcoin 節點彼此通訊時使用的預設連接埠。Machankura 顯示非洲在這場革命中並非旁門左道，而是該領域的創新之地。



**.mobi**網域專為透過USSD在基本電話上執行的行動服務所設計。因此，**8333.mobi** 總結了 Machankura 解決方案的特殊功能。



在繼續本教學之前，讓我們先來看看 USSD，它是讓 Machankura 可以在沒有網際網路的情況下使用的技術。



## USSD



USSD (*Unstructured supplementary Service Data*) 技術一般由 GSM 行動運營商使用。即使沒有網際網路，只要使用特定代碼並根據服務提供的可能性選擇選項，就能與遠端服務進行通訊。



在 USSD 通訊過程中，想像一下在手機上輸入一個特殊的代碼，例如 `*123#`。透過執行此 USSD 代碼，您會向 GSM 網路傳送特定請求，以存取與此代碼連結的服務。



因此，您可以使用 USSD 執行各種動作，例如查詢您的網際網路套裝、通話點數餘額等。



M-PESA 和 MTN Mobile Money 等移動銀行和移動貨幣轉帳正是通過這一通信系統在非洲大陸發展起來的。


正如我們之前所說，這項專為非洲開發的服務適用於所有手機，不需要複雜的技術設定或網際網路連線。

## 免智慧型手機操作（透過 USSD）



### 建立您的投資組合



對於您的第一次連接 ：




- 輸入與您國家對應的 USSD 代碼；
- 按下通話按鈕開始。



系統會要求您建立一個 5 位數的 PIN 碼




- 選擇安全 PIN 碼（5 位數）。
- 確認您的 PIN 碼。
- 您的 Bitcoin 產品組合即時建立。



此Bitcoin Wallet與您的電話號碼相關聯。精心選擇的 PIN 碼將加密您的 Wallet，也將用於確認您未來在 Machankura 上的所有交易。


建立您的投資組合後，您將可進入主選單，其中包含下列選項：



1.發送比特幣


2.接收比特幣


3.帳戶詳細資料


4.購買商品/服務


5.變更/重設 PIN 碼


6.輸出



若要存取主功能表中的任何選項，只需按下與該選項相關的順序編號即可。


若要存取「帳戶詳細資料」，請選取選項 **3**，然後按下通話鍵啟動。



### 安全與最佳實務



Machankura是一個托管的Lightning Wallet，所以您的比特幣是通過Machankura節點管理的。您的帳戶由您選擇的 PIN 碼保護。





- 不要將 PIN 碼設定在您生命中的重要日期。
- 切勿與任何人分享您的 PIN 碼。
- 記住您的 PIN 碼。
- 發送前請務必檢查電話號碼。
- 保持日常使用的合理平衡。



您的 Wallet 與您的電話號碼相連：




- 如果您遺失了手機，請聯絡 Machankura 支援；
- 記錄您的重要交易。



### 接收比特幣





- 在主功能表中選擇「接收 Bitcoins」選項的訂單號碼。
- 您的電話號碼是您的公開 Address。
- 將您的號碼分享給要寄比特幣給您的人。



#### 客製化閃電 Address



每個 Machankura 帳戶都會根據您的 Machankura 編號分配一個 Lightning Address


`votrenumero@8333.mobi`


例如：一個 +2371234567890 號碼會變成 `+2371234567890@8333.mobi`


不過，您可以選擇自訂使用者名稱來取代號碼 (例如：`Satoshi@8333.mobi`)。


任何持有您 Lightning Address 的人都可以在不知道您電話號碼的情況下向您發送比特幣。



#### 重裝您的 Machankura 產品組合



除了從另一個 Lightning Wallet 接收比特幣外，您也可以從 **Flash Group** 使用 **Azteco** 和 **1Voucher** 來補充您的 Machankura Wallet。



**Azteco**和**Flash Group**是兩家提供 Bitcoin **代金券**的公司，也就是預付代金券服務，您可以線上購買或向經銷商購買，不需透過 Exchange 平台即可取得 Bitcoin。這些代金券的作用類似**禮物卡**。有**On-Chain**和**Lightning**券。



![azteco](assets/fr/01.webp)



過程很簡單：





- 您購買固定金額的代用券；
- 您會透過電子郵件或小 Invoice 收到一個 **16 位元的**代碼。



![bonazteco](assets/fr/02.webp)





- 在手機上輸入您國家的 USSD MACHANKURA 代碼。
- 等待主功能表出現。
- 選擇「接收 bitcoins」選項的訂單號碼。
- 輸入 16 位數的代碼（代碼底部的 Azteco 代碼或 Invoice 上的 1Voucher PIN 代碼）。



與所購買的代金券金額等值的比特幣會直接加入您的 Machankura Wallet。



### 發送比特幣





- 前往主功能表。
- 輸入 Bitcoin 的訂單號，選擇「寄送 Bitcoin」選項。
- 輸入收件人的電話號碼、Lightning Address 或 Machankura 使用者名稱。
- 然後在 Sats 中輸入要寄送的金額。
- 使用 PIN 碼確認。



然後，Machankura 會在幾秒鐘內透過 Lightning Network 傳送比特幣。發件人也可以其他貨幣（包括當地貨幣）發送比特幣。



如果收件人的號碼尚未註冊為使用者，Machankura 將會預先註冊，然後將款項存入您的收件人帳戶。



訊息會自動傳送至預先註冊的號碼，使用者一旦確認註冊該服務，就會收到他的 bitcoins。

## 搭配智慧型手機使用



Machankura 選擇 WhatsApp 作為智慧型手機使用的中介平台。



這主要有兩個原因。第一步是讓非洲人更容易獲得解決方案，而不強迫他們安裝應用程式：如果手機儲存記憶體用完，他們可以輕鬆卸載這個應用程式。



第二個方案是第一個方案的邏輯延伸，使用非洲人在日常交流中廣泛使用的應用程式，成為一個包容性的當地解決方案：WhatsApp.
