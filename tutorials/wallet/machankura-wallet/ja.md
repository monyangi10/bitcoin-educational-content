アフリカは進歩し、革新し、発展しているが、インターネット接続や基本的な金融サービスへのアクセスなど、多くの課題に直面し続けている。


国際電気通信連合によると、2024年までにアフリカのインターネット普及率（大陸の総人口に対するインターネット利用者数の割合）は38％となり、世界の68％を上回るという。



金融サービスへのアクセスに関しては、銀行の信用サービスへのアクセスが困難になったため、大陸では送金サービス（モバイルマネー）がブームになっている。


このような状況に直面し、南アフリカの開発者であるKgothatso Ngakoは、**Machankura**と呼ばれる彼のプロジェクトによって、アフリカ社会の銀行口座を持たず、インターネットも利用しない層がBitcoinの恩恵を受けられるようにする画期的なソリューションを生み出した。



## マッチャンクラとは何か？



南アフリカのスラングで "お金 "を意味する「マチャンクラ」は、アフリカの農村とBitcoinのエコシステムをつなぐことをビジョンとするプロジェクトだ。



これは、USSDテクノロジーを介してLightning Networkでビットコインを送受信することができる保管型Walletだ。マチャンクラは、最も基本的な携帯電話（シンビアン携帯）であっても、従来のGSMネットワークを使用するすべての携帯電話で動作する。


通常ロゴの下に表示される**8333.mobi**は、さりげなくうなずける。実際、**8333**はBitcoinのノードが相互に通信するために使用するデフォルトのポートである。マチャンクラは、アフリカがこの革命の傍観者ではなく、この分野における革新の地であることを示している。



.mobi**ドメインは、USSDを介して基本的な携帯電話で実行されるモバイルサービス用に設計されています。したがって、**8333.mobi**は、Machankuraソリューションの特別な機能を要約しています。



このチュートリアルを続ける前に、まちゃんくらをインターネットなしで使えるようにする技術であるUSSDについて見てみよう。



## USSD



USSD（*Unstructured supplementary Service Data*）技術は、一般的にGSM携帯電話会社によって使用されています。これは、インターネットがなくても、特定のコードを使用して、サービスが提供する可能性に応じてオプションを選択することで、リモートサービスと通信できる機能です。



USSD通信プロセスでは、`*123#`のような特別なコードを携帯電話で入力することを想像してください。このUSSDコードを実行することで、このコードにリンクされたサービスにアクセスするための特定のリクエストをGSMネットワークに送信します。



そのため、USSDを使用して、インターネット・パッケージや通話クレジットの残高などを確認することができます。



M-PESAやMTNモバイルマネーなどのモバイルバンキングやモバイル送金が大陸で発展したのも、この同じ通信システムを利用したものである。


先に述べたように、アフリカ向けに特別に開発されたこのサービスは、複雑な技術的設定やインターネット接続を必要とせず、すべての携帯電話で利用できる。



## マチャンクラ・イノベーション



従来のWalletソリューションとは異なり、マチャンクラはUSSDテクノロジーを使って、アフリカの人々が直面している現実的な問題、すなわちインターネットのカバレッジをAddressにすることができた。



マチャンクラは、インターネットなしでビットコインの送受信を可能にするために開発され、GSMコードにリンクされたサービスである。関連するUSSDコードを使って、以下の国でサービスを利用できる：



| PAYS           | CODE USSD              |
| -------------- | ---------------------- |
| Ghana          | `*920*8333#`           |
| Kenya          | `*483*8333#`           |
| Malawi         | `*384*8333#`           |
| Namibie        | `*142*8333#`           |
| Nigeria        | `*347*8333#`           |
| Afrique du Sud | `54052.co.za`          |
| Tanzanie       | `SMS +255 679 066 977` |
| Ouganda        | `SMS +256 744 830 624` |
| Zambie         | `*384*8333#`           |
| Côte d’Ivoire  | `*9141#`               |

この表から、タンザニア、ウガンダ、南アフリカなどの国には、このサービス用のUSSDコードがないことがわかる。



しかし、まっちゃんくらぶは、ウェブサイト、SMSメッセージ、WhatsAppを通じて機能を拡張することで、この問題に対処している。



サービスが利用可能になる新しい国の情報は、同社の[ウェブサイト](https://8333.mobi)を定期的に参照してください。



このステップバイステップのチュートリアルでは、まずインターネットのない基本的な携帯電話で、次にスマートフォンで、「まちゃんくら」の使い方を説明します。



## スマートフォン不要の操作（USSD経由）



### ポートフォリオの作成



最初の接続のために：




- あなたの国に対応するUSSDコードを入力してください；
- 通話ボタンを押してスタート。



システムは5桁のPINコードを作成するよう要求します。




- 安全なPINコード（5桁）を選択してください。
- PINコードを確認してください。
- Bitcoinポートフォリオは即座に作成されます。



このBitcoin Walletは、あなたの電話番号と関連付けられています。慎重に選ばれたPINコードは、あなたのWalletを暗号化し、また、今後まっちゃんくらぶで行われるすべてのお取引の確認に使用されます。


ポートフォリオが作成されると、メインメニューに以下のオプションが表示されます：



1.ビットコインを送る


2.ビットコインを受け取る


3.口座詳細


4.商品/サービスの購入


5.PINコードの変更/リセット


6.出力



メインメニューのどのオプションにもアクセスするには、そのオプションに関連付けられているシーケンス番号を押すだけです。


アカウント詳細」にアクセスするには、オプション**3**を選択し、通話キーを押して起動します。



### 安全性とベストプラクティス



MachankuraはLightning Walletのカストディアンであるため、あなたのビットコインはMachankuraノードを通じて管理されます。あなたのアカウントは、選択したPINコードによって保護されます。





- 人生の重要な日にPINコードを設定しないこと。
- PINコードは絶対に他人に教えないでください。
- PINコードを覚える。
- 送信前に必ず電話番号を確認すること。
- 普段使いの適度なバランスを保つ。



あなたのWalletは電話番号とリンクしています：




- 携帯電話を紛失された場合は、まっちゃんくらサポートまでご連絡ください；
- 重要な取引を記録



### ビットコインを受け取る





- メインメニューの「ビットコインを受け取る」オプションの注文番号を選択します。
- あなたの電話番号は、あなたのパブリックAddressとして機能します。
- ビットコインを送りたい人と番号を共有する。



#### カスタムライトニング Address



まっちゃんくらぶのアカウントには、まっちゃんくらぶ番号に基づき、ライトニングAddressが割り当てられます。


`votrenumero@8333.mobi`


例: +2371234567890 の番号は `+2371234567890@8333.mobi` になります。


ただし、番号の代わりにカスタムユーザー名を選択することができます（例：`Satoshi@8333.mobi`）。


あなたのLightning Addressを持っている人なら、電話番号を知らなくてもビットコインを送ることができる。



#### まっちゃん倶楽部のポートフォリオをリロード



他のライトニングWalletからビットコインを受け取るだけでなく、**フラッシュグループ**から**アズテコ**と**1バウチャー**を使用して、まっちゃんくらWalletにビットコインを補充することができます。



**Azteco**と**Flash Group**は、Bitcoinの**バウチャー**を提供する2つの会社です。つまり、Exchangeのプラットフォームを経由せずにBitcoinを入手するために、オンラインまたは再販業者から購入できるプリペイド・バウチャー・サービスです。これらのバウチャーは**ギフトカード**のように機能する。On-Chain**バウチャーと**Lightning**バウチャーがあります。



![azteco](assets/fr/01.webp)



やり方は簡単だ：





- 一定額のバウチャーを購入する；
- 16桁の**コードを電子メールまたは小さなInvoiceで受け取る。



![bonazteco](assets/fr/02.webp)





- あなたの国のUSSD MACHANKURAコードを携帯電話に入力してください。
- メインメニューが表示されるまで待つ。
- ビットコインを受け取る」オプションの注文番号を選択します。
- 16桁のコード（バウチャー下部のAztecoバウチャー参照コードまたはInvoiceの1Voucher PINコード）を入力する。



購入されたバウチャーの金額と同額のビットコインが、お客様の「まっちゃんくらWallet」に直接加算されます。



### ビットコインを送る





- メインメニューへ。
- Bitcoinを送る」オプションを選択し、注文番号を入力する。
- 受信者の電話番号、Lightning Address、またはまっちゃんくらユーザー名のいずれかを入力してください。
- 次にSatsに送金額を入力する。
- PINコードで確認してください。



その後、まっちゃんくらはLightning Network経由でビットコインを数秒で送る。送金者は、現地通貨を含む他の通貨でビットコインを送ることもできる。



お受取人様のマイナンバーが未登録の場合、まっちゃんくらぶが事前登録を行い、お受取人様の口座に入金いたします。



事前に登録された番号に自動的にメッセージが送信され、ユーザーはサービスへの登録を確認するとビットコインを受け取る。

### 残高の確認





- メインメニューの "Balance "オプションの注文番号を選択する。
- 残高はサトシと現地通貨建てで表示されます。



### 取引履歴





- メインメニューの''History''オプションのシーケンス番号を選択する。
- 最新の取引（出荷と領収書）を表示します。
- 各取引の詳細を確認する。



### その他の特徴



まっちゃんくらは単なるWalletではありません。アプリから直接、サトシを商品やサービス（ビットリフィルやライトニングワッツのギフトカードなど）と交換することができる。



https://planb.academy/tutorials/exchange/centralized/bitrefill-8c588412-1bfc-465b-9bca-e647a647fbc1

また、"**クラン**"機能もあり、複数署名による協力体制を管理することができます。クランメンバーは各支出を承認する必要があり、クランメンバー間で自動的に資金を分配することができます。



## スマートフォンでの使用



まっちゃんくらぶは、スマートフォンの仲介プラットフォームとしてWhatsAppを選んだ。



これには主に2つの理由がある。第一のステップは、アフリカの人々にアプリケーションのインストールを強いることなく、ソリューションをより身近なものにすることだ。



2つ目は、1つ目の論理的延長線上にあるもので、アフリカ人が日常的に広く使っているアプリケーションを使った、包括的でローカルなソリューションである：WhatsAppである。
