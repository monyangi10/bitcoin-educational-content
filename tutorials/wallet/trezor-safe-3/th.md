*เครดิตภาพ: [Trezor.io](https://trezor.io/)*



Trezor Safe 3 เป็น Hardware Wallet ที่ออกแบบโดย SatoshiLabs และสร้างขึ้นในปี 2023 เป็นรุ่นที่มีขนาดกะทัดรัดและน้ำหนักเบามาก (14 กรัม) ออกแบบมาสำหรับผู้เริ่มต้นและผู้ใช้ระดับกลาง เป็นผู้สืบทอดของ Model One ที่มีชื่อเสียง โดยมีการเพิ่มเติมที่สำคัญ ในขณะที่ยังคงรักษาแนวทางโอเพ่นซอร์สของแบรนด์ที่ทำให้แตกต่างจากคู่แข่งหลัก Ledger Safe 3 มีราคาอยู่ที่ €79 ดังนั้นจึงอยู่ในกลุ่มฮาร์ดแวร์ระดับกลาง wallet ที่แข่งขันโดยตรงกับ Ledger Nano S Plus



Safe 3 ไม่มีแบตเตอรี่และทำงานผ่านการเชื่อมต่อ USB-C เท่านั้น ใช้สำหรับทั้งพลังงานและการสื่อสาร มีหน้าจอ OLED ขาวดำขนาดเล็ก 0.96 นิ้ว และปุ่มกดสองปุ่ม



![Image](assets/fr/01.webp)



The Safe 3 มีคุณสมบัติที่จำเป็นทั้งหมดที่คาดหวังจาก Hardware Wallet ที่ดี รวมถึงการผสานรวมที่ยอดเยี่ยมของ passphrase BIP39 อย่างไรก็ตาม มันยังไม่รองรับ Miniscript.



รุ่นนี้เหมาะสำหรับผู้เริ่มต้นเป็นพิเศษ และอาจจะเป็น Hardware Wallet ที่ฉันจะแนะนำให้กับผู้ใช้ใหม่ นอกจากนี้ยังเหมาะกับผู้ใช้ระดับกลาง ในทางกลับกัน อาจไม่ตรงตามความคาดหวังทั้งหมดของผู้ใช้ขั้นสูงที่มองหาคุณสมบัติเฉพาะเพิ่มเติม ซึ่งมีอยู่ในอุปกรณ์เช่น Coldcard อย่างไรก็ตาม หากคุณไม่ต้องการตัวเลือกขั้นสูงเหล่านี้ Trezor Safe 3 อาจเป็นตัวเลือกที่ยอดเยี่ยม



## Trezor Safe 3 รุ่นความปลอดภัย



Trezor Safe 3 ได้รับการติดตั้ง **Secure Element** ที่ได้รับการรับรอง EAL6+ ซึ่งเป็นความก้าวหน้าที่สำคัญจากรุ่นก่อนหน้า เช่น Model One และ Model T นี่คือชิป OPTIGA Trust M V3 ซึ่งไม่ได้เก็บ seed โดยตรง แต่ทำหน้าที่เป็นส่วนประกอบการเข้ารหัสเพื่อรักษาความปลอดภัยในการเข้าถึง Secure Element จะเก็บรักษาความลับที่สามารถเข้าถึงได้ก็ต่อเมื่อผู้ใช้ป้อน PIN อย่างถูกต้อง ความลับนี้จะถูกใช้ในการถอดรหัส seed ซึ่งถูกเก็บไว้ในหน่วยความจำหลักของอุปกรณ์ในรูปแบบที่เข้ารหัส



ระบบรักษาความปลอดภัยแบบไฮบริดนี้มีการป้องกันทางกายภาพที่ดีขึ้น โดยเฉพาะอย่างยิ่งต่อการโจมตีแบบสกัดหรือการวิเคราะห์ที่รุกราน ซึ่งเป็นปัญหาที่ Model One มีแนวโน้มจะเกิดขึ้น โดยเฉพาะในการจัดการ PIN ช่องโหว่เหล่านี้ได้รับการแก้ไขแล้วด้วยการใช้ Secure Element รุ่นนี้ยังคงรักษาสถาปัตยกรรมซอฟต์แวร์แบบโอเพ่นซอร์ส: โค้ดที่จัดการการสร้างและการใช้คีย์ส่วนตัวยังคงสามารถเข้าถึงและตรวจสอบได้อย่างเต็มที่ ชิป OPTIGA จัดการเฉพาะรหัส PIN ซึ่งเป็นองค์ประกอบภายนอกต่อการจัดการคีย์ Bitcoin wallet มันจะปล่อยความลับที่สามารถใช้ถอดรหัส seed ได้เท่านั้น นอกจากนี้ ชิป OPTIGA Trust M V3 ยังได้รับประโยชน์จากใบอนุญาตที่ค่อนข้างเสรี ซึ่งอนุญาตให้ SatoshiLabs เผยแพร่ช่องโหว่ที่อาจเกิดขึ้นได้อย่างอิสระ



โมเดลความปลอดภัยนี้เป็นหนึ่งในทางเลือกที่ดีที่สุดในตลาดปัจจุบันในความเห็นของฉัน มันรวมข้อดีของ Secure Element เข้ากับการจัดการซอฟต์แวร์แบบโอเพนซอร์ส ก่อนหน้านี้ ผู้ใช้ต้องเลือกระหว่างความปลอดภัยทางกายภาพที่เพิ่มขึ้นด้วยชิปและความโปร่งใสด้วยโอเพนซอร์ส; แต่กับ Trezor Safe 3 มันเป็นไปได้ที่จะได้รับประโยชน์จากทั้งสองอย่าง



ในบทแนะนำนี้ เราจะแสดงวิธีการตั้งค่าและใช้งาน Trezor Safe 3 ของคุณอย่างปลอดภัย



## แกะกล่อง Trezor Safe 3



เมื่อคุณได้รับ Safe 3 ของคุณ ตรวจสอบให้แน่ใจว่ากล่องและซีลยังคงอยู่ในสภาพสมบูรณ์เพื่อยืนยันว่าพัสดุไม่ได้ถูกเปิด นอกจากนี้ยังจะมีการตรวจสอบซอฟต์แวร์เพื่อยืนยันความถูกต้องและความสมบูรณ์ของอุปกรณ์เมื่อมีการตั้งค่าในภายหลัง



เนื้อหาในกล่องประกอบด้วย :




- Trezor Safe 3;
- กระเป๋าที่บรรจุกระดาษแข็งสำหรับบันทึกวลีช่วยจำ สติกเกอร์ และคำแนะนำ;
- สาย USB-C เป็น USB-C



![Image](assets/fr/02.webp)



เมื่อเปิดออกมา Trezor Safe 3 ของคุณควรได้รับการป้องกันด้วยพลาสติกป้องกันและพอร์ต USB-C ควรถูกปิดด้วยซีลโฮโลแกรม ตรวจสอบให้แน่ใจว่ามีอยู่



![Image](assets/fr/03.webp)



การนำทางบนอุปกรณ์เป็นเรื่องง่าย: ใช้ปุ่มขวาเพื่อเลื่อนไปทางขวา และปุ่มซ้ายเพื่อเลื่อนไปทางซ้าย กดปุ่มทั้งสองพร้อมกันเพื่อยืนยันการทำงาน



![Image](assets/fr/04.webp)



## ข้อกำหนดเบื้องต้น



สำหรับบทแนะนำนี้ ฉันจะแสดงวิธีการใช้ Trezor Safe 3 กับ [ซอฟต์แวร์การจัดการ Sparrow Wallet wallet](https://sparrowwallet.com/download/) หากคุณยังไม่ได้ติดตั้งซอฟต์แวร์นี้ กรุณาทำการติดตั้งตอนนี้ หากคุณต้องการความช่วยเหลือ เราก็มีบทแนะนำรายละเอียดเกี่ยวกับการกำหนดค่า Sparrow Wallet :



https://planb.academy/tutorials/wallet/desktop/sparrow-c674e2ac-d46f-4c82-92a7-7d1b0e262f5d

คุณจะต้องใช้ซอฟต์แวร์ Trezor Suite เพื่อกำหนดค่า Safe 3 ตรวจสอบความถูกต้องและติดตั้งเฟิร์มแวร์ เราจะใช้ซอฟต์แวร์นี้เฉพาะสำหรับสิ่งนั้น และหลังจากนั้นจะต้องใช้เฉพาะสำหรับการอัปเดตเฟิร์มแวร์เท่านั้น สำหรับการจัดการ wallet ในชีวิตประจำวัน เราจะใช้ Sparrow Wallet เท่านั้น เนื่องจากมันถูกปรับให้เหมาะสมสำหรับ Bitcoin และใช้งานง่าย แม้แต่สำหรับผู้เริ่มต้น (Sparrow รองรับเฉพาะ Bitcoin ไม่รองรับ altcoins)



[ดาวน์โหลด Trezor Suite จากเว็บไซต์ทางการ](https://trezor.io/trezor-suite)

วลีช่วยจำนี้ให้การเข้าถึงบิตคอยน์ทั้งหมดของคุณอย่างเต็มที่และไม่มีข้อจำกัด ใครก็ตามที่ครอบครองวลีนี้สามารถขโมยเงินของคุณได้ แม้จะไม่มีการเข้าถึงทางกายภาพไปยัง Trezor Safe 3 ของคุณก็ตาม



วลี 12 คำจะช่วยกู้คืนการเข้าถึงบิตคอยน์ของคุณในกรณีที่สูญหาย ถูกขโมย หรือเสียหายของ Hardware Wallet ของคุณ ดังนั้นจึงเป็นสิ่งสำคัญมากที่จะต้องบันทึกไว้อย่างระมัดระวังและเก็บไว้ในที่ปลอดภัย



คุณสามารถเขียนมันลงบนกระดาษแข็งที่มีให้ในกล่อง หรือเพื่อความปลอดภัยเพิ่มเติม ฉันแนะนำให้สลักมันลงบนฐานสแตนเลสเพื่อป้องกันจากไฟไหม้ น้ำท่วม หรือการพังทลาย



ยืนยันคำแนะนำ จากนั้นคลิกที่ปุ่ม "*Create Wallet backup*"

Safe 3 จะสร้างวลีช่วยจำของคุณโดยใช้ตัวสร้างตัวเลขสุ่มของมัน ตรวจสอบให้แน่ใจว่าไม่มีใครกำลังดูคุณในระหว่างการดำเนินการนี้ จดคำที่แสดงบนหน้าจอลงบนสื่อทางกายภาพที่คุณเลือก ขึ้นอยู่กับกลยุทธ์ความปลอดภัยของคุณ คุณอาจพิจารณาทำสำเนาทางกายภาพของวลีทั้งหมดหลายชุด (แต่เหนือสิ่งอื่นใด อย่าแบ่งมัน) สิ่งสำคัญคือต้องเก็บคำเหล่านั้นไว้ตามลำดับหมายเลขและตามลำดับต่อเนื่อง

## การตั้งรหัส PIN



ขั้นตอนต่อไปคือขั้นตอนรหัส PIN รหัส PIN จะปลดล็อก Trezor ของคุณ ดังนั้นจึงให้การป้องกันการเข้าถึงทางกายภาพโดยไม่ได้รับอนุญาต รหัส PIN นี้ไม่ได้มีส่วนเกี่ยวข้องในการสร้างคีย์การเข้ารหัสของ wallet ของคุณ ดังนั้นแม้จะไม่มีการเข้าถึงรหัส PIN การครอบครองวลีช่วยจำ 12 คำของคุณจะช่วยให้คุณสามารถเข้าถึงบิตคอยน์ของคุณได้อีกครั้ง



บน Trezor Suite คลิกที่ "*Continue to PIN*" จากนั้นคลิกที่ปุ่ม "*Set PIN*"

## การตั้งค่า wallet บน Sparrow Wallet



เริ่มต้นด้วยการดาวน์โหลดและติดตั้ง Sparrow Wallet [จากเว็บไซต์ทางการ](https://sparrowwallet.com/) บนคอมพิวเตอร์ของคุณ หากคุณยังไม่ได้ทำเช่นนั้น



เมื่อคุณเปิด Sparrow Wallet แล้ว ให้แน่ใจว่าซอฟต์แวร์เชื่อมต่อกับโหนด Bitcoin ซึ่งจะแสดงด้วยเครื่องหมายถูกที่มุมล่างขวาของ Interface หากคุณมีปัญหาในการเชื่อมต่อ Sparrow ฉันขอแนะนำให้อ่านตอนต้นของบทแนะนำนี้:
