Elysium Wallet 是瑞士新創公司 Elysium Labs 推出的第一款非監控型 Software Wallet。


有了創新的金鑰管理系統，您可以使用日常生活中的 Elements 來存取您的數位資產：您的使用者名稱、通行鑰、密碼或通行碼。沒錯：不再嚴格地需要使用 seed Phrase 才能重新存取您的數位資產。這種簡化可能會加速 Bitcoin 在全球的普及。


## 如何開設帳戶？


從 Apple Store 或 Google Play 下載 Elysium Wallet 應用程式，然後在裝置上開啟您下載的 Elysium Wallet 應用程式。點選「建立新的 Wallet」，此時會出現使用條款與條件畫面。若要接受並繼續建立您的帳戶，請點選「開始設定」，然後輸入您的使用者名稱；請注意個人檔案圖片是可自訂的：從提供的選項中選擇一張，您可以拍照或從裝置上傳圖片。選擇完成後，點選「繼續」。


![image](assets/en/02.webp)


Elysium 因其創新的多因素演算法而脫穎而出，該演算法結合了 Passkey、PassCode 和 PassWord。PassKeys 是強制性的。它們允許您使用裝置內建的安全功能，例如臉部 ID 或指紋掃描，快速且安全地進行驗證。它們是您的主要 Layer 保護，可確保快速安全的存取。


![image](assets/en/03.webp)


選擇您的第二層級：PassCode 或 PassWord；接下來，您需要選擇第二層安全性：



- PassCode：易於記憶的 6 位數密碼。最適合增加額外的 Layer 保護。
- 密碼： 建立至少 8 個字元的強大密碼，以增加安全性。


您必須結合 PassCode 或 PassWord 使用通行鑰。


![image](assets/en/04.webp)

**註：** 要設定您的帳戶，您至少需要 2 個因素，其中一個必須是通行鑰匙。


為了進一步提高安全性，您可以增加第三個 Layer 保護 (Passkey + PassCode + PassWord)。


多層組合，提供最高安全性

您永遠會使用 Passkey 作為主要因素。對於第二個 Layer，請選擇 PassCode 或 PassWord。

如果您已選擇 PassCode 作為第二個因素，則可加入 PassWord 作為第三個 Layer，反之亦然。這種靈活的方式可確保您的資產依照您的喜好受到保護。

您可以在設定階段 (請參閱圖片) 或稍後加入第三個安全因素，方法是進入設定 > 改善安全性。


![image](assets/en/05.webp)


但是，如果您忘記了其中一個因素，請注意：


如果您已設定所有三個因素，您可以隨時從設定中變更或重設。

不幸的是，如果您只設定了兩個因素，卻忘記了其中一個，就沒有復原選項。


我們強烈建議從一開始就設定這三個因素，以獲得最大的安全性和彈性。


## 如何接收交易？


開啟 Elysium 應用程式，進入主選單，然後點選「接收」。


![image](assets/en/06.webp)


現在，選擇您要收取付款的連鎖店（Bitcoin 或 Polygon），您只需將您的極樂世界 Wallet 的 QR 代碼分享給需要付款的人，他們就會處理剩下的事情。


## 如何在 Lightning Network 中接收交易？


**步驟 1：** 點選「請求付款」，即表示您透過 Lightning Network 請求 Bitcoin 付款。


![image](assets/en/07.webp)


**步驟 2：** 輸入您要申請的金額、選擇您要接收的貨幣，必要時加入說明。


![image](assets/en/08.webp)


**註：** 首次 Lightning Network (LN) 付款開通 LN 通道需要支付少量費用。之後，之後的付款都是免費的。


## 如何傳送交易？


**步驟 1：** 進入主功能表，點選「傳送」。


![image](assets/en/09.webp)


**步驟 2：** 從收件人的 Elysium Wallet 掃描其 QR 代碼，即可自動將其聯繫人儲存至您的 Address 手冊。或者，手動複製他們的 Address 並貼到收件人欄位。選擇收件人或將其加入 Address 帳簿後，點選「傳送付款」。


![image](assets/en/10.webp)


已經有聯絡人了嗎？直接從 Address 手冊中選擇。


![image](assets/en/11.webp)


**步驟 3：** 輸入您要傳送的金額，並選擇您要傳送的資產。

對於 BTC 交易，您可以選擇您偏好的網路速度和費用（如第三張圖所示）


![image](assets/en/12.webp)


您的交易已經提交！您可以輕鬆查看您的 Elysium Wallet 的最新餘額和交易狀態。


## 如何在 Lightning Network 中傳送交易？


![image](assets/en/13.webp)


**步驟 1：** 點選「掃瞄」開啟掃瞄器。


**步驟 2：** 掃描 LN QR 代碼進行付款。


**步驟 3：** 檢視付款詳細資料，並確認一切正確無誤。


**步驟 4：** 點選「確認」完成交易。


## 如何查看 seed Phrase？


進入主功能表，點選「Hub」。選擇「設定」並點選「提取私人密碼匙


![image](assets/en/14.webp)


使用密碼匙登入，並輸入密碼和/或通行碼。seed 詞組將以 24 字格式顯示。


![image](assets/en/15.webp)


不要與任何人分享！
