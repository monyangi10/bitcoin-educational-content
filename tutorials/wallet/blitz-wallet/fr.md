L'expérience utilisateur représente l'un des facteurs décisifs lorsque vous souhaiter débuter avec un portefeuille. Dans ce tutoriel, nous vous présenterons un portefeuille qui a fait de son expérience utilisateur, un facteur décisif : Blitz Wallet vous propose le portefeuille Bitcoin le plus simple et l'un des plus complet que vous pouvez trouver.

## Qu'est-ce que Blitz Wallet ?

Blitz Wallet est un portefeuille Bitcoin self-custody (d'auto détention) dont le code source est disponible (Open Source), qui mise sur votre souveraineté et son expérience utilisateur qui vous permet de facilement prendre en main son utilisation.

[Blitz Wallet](https://blitz-wallet.com/) est un portefeuille mobile disponible sur Android (Play Store) et iOS (App Store).

⚠️**IMPORTANT** : Télécharger un portefeuille Bitcoin sur une plateforme officielle est importante pour vérifier l'authenticité de l'application et par extension, renforcer la sécurité de vos fonds.

Dans ce tutoriel, nous nous baserons sur la version Android de Blitz Wallet mais tous les processus qui seront présentés ci-dessous sont également valable sur iOS.

## Débuter avec Blitz

Effectuer des transactions avec Blitz est plus intuitif que dans la plupart des autres portefeuilles Bitcoin.

Dans le menu Wallet, vous avez une interface minimaliste et centrée uniquement sur les actions principales :

### Recevoir des bitcoins

Pour recevoir des bitcoins sur votre portefeuille Blitz, cliquez sur l'icône "Flèche bas", insérez le montant de satoshis que vous souhaitez recevoir puis le portefeuille vous créera une facture que vous partagerez à votre expéditeur.

 ⚠️ **NOTES** : le satoshi (ou "sat") représente la plus petite unité de bitcoin : 1 bitcoin = 100.000.000 satoshis

L'une des particularités de Blitz Wallet est qu'il supporte différents réseaux et chaines de l'écosystème Bitcoin :

- **Lightning Network** : L'une des surcouches Bitcoin qui vous permet d'effectuer les microtransactions instantanément.

- **Bitcoin Mainnet** : La chaîne principale du protocole Bitcoin, adaptée pour les transactions de sommes conséquentes.

- **Liquid Network** : Une chaîne parallèle au Bitcoin Mainnet développé par BlockStream qui utilise des Liquid Bitcoin pour effectuer des transactions rapides et confidentielles.

https://planb.network/tutorials/wallet/mobile/blockstream-app-liquid-b3e4fb82-902e-4782-ad2b-a61ab05a543a

Par défaut, toutes vos transactions seront sur le Liquid Network mais Blitz vous permet de définir le réseau sur lequel vous souhaitez recevoir des satoshis en cliquant sur le buton **Choisir le format**.

![receive-sats](assets/fr/03.webp)

### Créer des contacts avec Blitz

Blitz Wallet vous facilite le processus d'envoi des bitcoins depuis son portefeuille.

Dans le menu **Contacts**, vous pouvez enregistrer les noms d'utilisateurs Blitz ou les Lightning URL avec lesquelles vous interagissez le plus.

Vous pourrez ainsi envoyer des satoshis à ces adresses facilement en outrepassant l'étape de scannage et de saisie manuelle d'adresse.

![add-contacts](assets/fr/04.webp)

### Envoyez des bitcoins

Outres les méthodes classiques d'envoi de bitcoin (Code QR, saisie manuelle), en utilisant les contacts pré-enregistrés dans votre portefeuille, vous pourrez envoyer en trois clics, des sats à votre destinataire.

Dans le menu **Wallet**, cliquez sur le bouton "Flèche Haut", choisissez la méthode pour envoyer des bitcoins puis insérez le montant à envoyer et procédez à la confirmation.

Le montant minimal pour effectuer un envoi de bitcoin dans Blitz Wallet est actuellement de 1000 satoshis.

![send-bitcoin](assets/fr/05.webp)

## Blitz Wallet derrière la scène : Aller plus loin

Derrière la simplicité de l'utilisation de Blitz Wallet, se trouve une puissance et une personnalisation bien pensée.

Comme nous l'avons souligné plus tôt, tous les bitcoins que vous recevez sont par défaut sur Liquid Network.

Blitz utilise les micros échanges de Liquid Network afin de vous présenter votre solde en satoshi lorsque vous avez un solde de moins de 500.000 satoshis.

Cette approche se justifie par le souhait de faciliter l'expérience de démarrage et d'aider les nouveaux utilisateurs d'effectuer le plus simplement possible des transactions sur le Lightning Network avec de petits montants.

https://planb.network/tutorials/wallet/mobile/aqua-8e6d7dd3-8c03-45cc-90dd-fe3899a7d125

Vous pourrez constater la répartition de votre solde dans le menu **Paramètres>Balance Info**.

![balance](assets/fr/09.webp)

Blitz Wallet vous donne néanmoins la flexibilité d'activer le mode Lightning qui ouvrira automatiquement un canal de paiement pour vous lorsque vous aurez atteint un solde de 500.000 satoshis.

Pour activer le mode Lightning, vous pouvez vous rendre dans les **Paramètres** puis dans la section **Paramètres Techniques**, cliquez sur l'option **Node Info**.

![enable-lightning](assets/fr/10.webp)

En activant le mode Lightning, une fois la condition principale remplie (solde de 500.000 satoshis ou 0.005 bitcoin), vous pourrez effectuer des transactions sur Lightning Network et n'aurez plus à passer par le Liquid Network de BlockStream.

- **Accepter bitcoin dans votre commerce** :

L'intégration des paiements bitcoin dans les commerces est encore à la phase expérimentale avec Blitz Wallet. Nous vous recommandons de l'utiliser avec modération.

Dans le menu **Paramètres>Point-of-sale**, vous pouvez paramétrer l'identifiant unique associé à votre magasin et la devise fiduciaire locale dans laquelle vous souhaitez recevoir les paiements.
