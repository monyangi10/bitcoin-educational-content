## Blitz Wallet คืออะไร?



Blitz Wallet เป็น Bitcoin การดูแลตนเอง wallet ที่มีซอร์สโค้ดให้ใช้งาน (โอเพ่นซอร์ส) ซึ่งมุ่งเน้นที่อธิปไตยของคุณและประสบการณ์ผู้ใช้ที่ทำให้เข้าใจง่าย



[Blitz Wallet](https://blitz-Wallet.com/) เป็น wallet บนมือถือที่สามารถใช้งานได้บน Android (Play Store) และ iOS (App Store)



⚠️**สำคัญ**: การดาวน์โหลด Bitcoin wallet บนแพลตฟอร์มอย่างเป็นทางการเป็นสิ่งสำคัญสำหรับการตรวจสอบความถูกต้องของแอปพลิเคชันและโดยการขยายเพื่อเสริมสร้างความปลอดภัยของเงินทุนของคุณ



ในบทแนะนำนี้ เราจะอ้างอิงจากเวอร์ชัน Android ของ Blitz Wallet แต่กระบวนการทั้งหมดที่นำเสนอด้านล่างนี้สามารถใช้ได้เช่นกันบน iOS

## เริ่มต้นกับ Blitz



การซื้อขายกับ Blitz นั้นใช้งานง่ายกว่ากระเป๋าเงิน Bitcoin ส่วนใหญ่



ในเมนู Wallet คุณมี Interface ที่เรียบง่ายซึ่งเน้นเฉพาะการกระทำหลัก:



### รับบิตคอยน์



ในการรับบิตคอยน์บน Blitz wallet ของคุณ ให้คลิกที่ไอคอน "ลูกศรลง" ใส่จำนวนซาโตชิที่คุณต้องการรับ และ wallet จะสร้างใบแจ้งหนี้ให้คุณแชร์กับผู้ส่งของคุณ



⚠️ **หมายเหตุ**: Satoshi (หรือ "sat") แทนหน่วยที่เล็กที่สุดของ Bitcoin: 1 Bitcoin = 100,000,000 satoshis



คุณสมบัติพิเศษอย่างหนึ่งของ Blitz Wallet คือการรองรับเครือข่ายและช่องทางที่แตกต่างจากระบบนิเวศ Bitcoin:





- Lightning Network**: หนึ่งในโอเวอร์เลย์ Bitcoin ที่ให้คุณทำธุรกรรมขนาดเล็กได้ทันที





- Bitcoin Mainnet**: สายหลักของโปรโตคอล Bitcoin เหมาะสำหรับการทำธุรกรรมมูลค่าสูง





- Liquid Network**: เชนคู่ขนานกับ Bitcoin Mainnet ที่พัฒนาโดย BlockStream ซึ่งใช้ Liquid Bitcoins เพื่อดำเนินการธุรกรรมที่รวดเร็วและเป็นความลับ



https://planb.academy/tutorials/wallet/mobile/blockstream-app-liquid-b3e4fb82-902e-4782-ad2b-a61ab05a543a

ตามค่าเริ่มต้น การทำธุรกรรมทั้งหมดของคุณจะอยู่บน Liquid Network แต่ Blitz ให้คุณกำหนดเครือข่ายที่คุณต้องการรับ satoshis โดยการคลิกที่ปุ่ม **Choose format**



![receive-sats](assets/fr/03.webp)



### สร้างรายชื่อผู้ติดต่อด้วย Blitz



Blitz Wallet ทำให้ง่ายต่อการส่งบิตคอยน์จาก wallet ของคุณ



ในเมนู **Contacts** คุณสามารถลงทะเบียนชื่อผู้ใช้ Blitz หรือ URL Lightning ที่คุณโต้ตอบด้วยบ่อยที่สุด



ซึ่งหมายความว่าคุณสามารถส่งซาโตชิไปยังที่อยู่เหล่านี้ได้อย่างง่ายดาย โดยข้ามขั้นตอนการสแกนและการป้อนที่อยู่ด้วยตนเอง



![add-contacts](assets/fr/04.webp)



### ส่งบิตคอยน์



นอกเหนือจากวิธีการส่งบิตคอยน์แบบคลาสสิก (QR code, การป้อนข้อมูลด้วยตนเอง) โดยใช้รายชื่อที่ลงทะเบียนล่วงหน้าใน wallet ของคุณ คุณสามารถส่ง Satss ให้กับผู้รับของคุณได้เพียงแค่สามคลิก



ในเมนู **Wallet** คลิกที่ปุ่ม "ลูกศรขึ้น" เลือกวิธีการส่งบิตคอยน์ จากนั้นใส่จำนวนเงินที่จะส่งและดำเนินการยืนยัน



จำนวนขั้นต่ำในการส่ง Bitcoin ใน Blitz Wallet ปัจจุบันคือ 1,000 ซาโตชิ



![send-bitcoin](assets/fr/05.webp)

## Wallet Blitz เบื้องหลัง: ก้าวไปไกลกว่าเดิม



เบื้องหลังความเรียบง่ายของการทำงานของ Blitz Wallet ซ่อนพลังและการปรับแต่งที่หลากหลาย



ตามที่เราได้กล่าวไว้ก่อนหน้านี้ บิตคอยน์ทั้งหมดที่คุณได้รับจะถูกตั้งค่าเริ่มต้นเป็น Liquid Network



Blitz ใช้ไมโครเอ็กซ์เชนจ์ Liquid Network เพื่อแสดงยอดคงเหลือของคุณใน Satoshi เมื่อคุณมียอดคงเหลือน้อยกว่า 500,000 ซาโตชิ



วิธีการนี้มีเหตุผลมาจากความต้องการที่จะอำนวยความสะดวกในการเริ่มต้นใช้งานและช่วยให้ผู้ใช้ใหม่สามารถดำเนินการธุรกรรมบน Lightning Network ด้วยจำนวนเงินที่น้อยที่สุดได้อย่างง่ายดายที่สุด



https://planb.academy/tutorials/wallet/mobile/aqua-8e6d7dd3-8c03-45cc-90dd-fe3899a7d125

คุณสามารถดูรายละเอียดของยอดเงินของคุณได้ในเมนู **การตั้งค่า>ข้อมูลยอดเงิน**.



![balance](assets/fr/09.webp)



Blitz Wallet อย่างไรก็ตาม ให้คุณมีความยืดหยุ่นในการเปิดใช้งานโหมด Lightning ซึ่งจะเปิดช่องทางการชำระเงินให้คุณโดยอัตโนมัติเมื่อคุณมียอดคงเหลือถึง 500,000 ซาโตชิ



หากต้องการเปิดใช้งานโหมด Lightning ให้ไปที่ **การตั้งค่า** จากนั้นในส่วน **การตั้งค่าทางเทคนิค** คลิกที่ตัวเลือก **ข้อมูลโหนด**



![enable-lightning](assets/fr/10.webp)



โดยการเปิดโหมด Lightning เมื่อเงื่อนไขหลักได้รับการตอบสนอง (ยอดคงเหลือ 500,000 satoshis หรือ 0.005 Bitcoin) คุณจะสามารถดำเนินการธุรกรรมบน Lightning Network ได้และไม่จำเป็นต้องผ่าน Liquid Network ของ BlockStream อีกต่อไป
