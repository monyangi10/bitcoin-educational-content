## Blitz Wallet 是什麼？



Blitz Wallet 是一款 Bitcoin 自攜式 Wallet，其原始碼已公開 (開放原始碼)，它著重於您的主權以及讓您輕鬆上手的使用者經驗。



[Blitz Wallet](https://blitz-Wallet.com/) 是可在 Android (Play Store) 和 iOS (App Store) 上使用的行動 Wallet。



⚠️** 重要**：在官方平台下載 Bitcoin Wallet 對於驗證應用程式的真實性非常重要，從而加強您資金的安全性。



在本教程中，我們將以 Blitz Wallet 的 Android 版本為基礎，但以下介紹的所有程序在 iOS 上同樣有效。

## 開始使用 Blitz



使用 Blitz 進行交易比使用其他大多數 Bitcoin 電子錢包更加直觀。



在 Wallet 功能表中，您可以看到簡約的 Interface 只專注於主要動作：



### 接收比特幣



若要在 Blitz Wallet 上接收比特幣，請點選「向下箭頭」圖示，插入您想要接收的金額，Wallet 就會製作一個 Invoice 讓您與寄件者分享。



⚠️ **註**：Satoshi（或 "sat"）代表 Bitcoin 的最小單位：1 Bitcoin = 100,000,000 薩托希



Blitz Wallet 的特點之一是支援與 Bitcoin 生態系統不同的網路和頻道：





- Lightning Network** ：Bitcoin 覆蓋層之一，可讓您立即執行微交易。





- Bitcoin Mainnet** ：Bitcoin 協定的主鏈，適用於大額交易。





- Liquid Network**：BlockStream 開發的 Bitcoin Mainnet 的平行鏈，使用 Liquid 比特幣進行快速、Confidential Transactions。



https://planb.network/tutorials/wallet/mobile/blockstream-app-liquid-b3e4fb82-902e-4782-ad2b-a61ab05a543a

預設情況下，您所有的交易都會在 Liquid Network 上進行，但 Blitz 可讓您透過點選 ** 選擇格式** 按鈕，定義希望接收 Satoshis 的網路。



![receive-sats](assets/fr/03.webp)



### 使用 Blitz 建立連絡人



Blitz Wallet 可讓您輕鬆地從其 Wallet 傳送比特幣。



在**聯絡人**選單中，您可以註冊與您互動最多的 Blitz 使用者名稱或 Lightning URL。



這表示您可以輕鬆地寄送 Satoshis 到這些地址，繞過掃描和手動輸入 Address 的階段。



![add-contacts](assets/fr/04.webp)



### 發送比特幣



除了傳送 Bitcoins 的經典方法 (QR 代碼、手動輸入)，透過 Wallet 中預先註冊的聯絡人，您只需按三下即可傳送 Satss 給收件人。



在**Wallet**菜單中，點擊 「向上箭頭 」按鈕，選擇發送比特幣的方法，然後輸入要發送的金額，並進行確認。



在 Blitz Wallet 中傳送 Bitcoin 的最低金額目前為 1,000 薩托希。



![send-bitcoin](assets/fr/05.webp)

## Wallet Blitz 背後的故事：更進一步



在 Blitz Wallet 簡單的操作背後，隱藏著豐富的功能和客製化設定。



正如我們之前指出的，您收到的所有比特幣預設為 Liquid Network。



當您的餘額少於 500,000 Satoshis 時，Blitz 會使用 Liquid Network 微型交換器以 Satoshi 顯示您的餘額。



這種做法的理由是希望促進新手經驗，並幫助新使用者盡可能簡單地在 Lightning Network 上進行小額交易。



https://planb.network/tutorials/wallet/mobile/aqua-8e6d7dd3-8c03-45cc-90dd-fe3899a7d125

您可以在功能表 ** 設定 > 結餘資訊** 中查看結餘的明細。



![balance](assets/fr/09.webp)



不過，Blitz Wallet 也提供您啟動「閃電」模式的彈性，一旦您的餘額達到 500,000 薩托希，就會自動為您開啟付款通道。



若要啟動 Lightning 模式，請前往 **Settings**，然後在 **Technical Settings**（技術設定）部分，按一下 **Node Info** 選項。



![enable-lightning](assets/fr/10.webp)



啟動「閃電」模式後，一旦滿足主要條件（餘額達到 500,000 Satoshis 或 0.005 Bitcoin），您就可以在 Lightning Network 上進行交易，而無需再經過 BlockStream 的 Liquid Network。

在 Blitz Wallet 中，商店中 Bitcoin 付款的整合仍處於實驗階段。我們建議您少用。



在 ** 設定 > 銷售點** 功能表中，您可以設定與您商店相關的唯一識別碼，以及您希望接收付款的當地法定貨幣。
