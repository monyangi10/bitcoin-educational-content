BitBox02 (https://bitbox.swiss/) เป็นอุปกรณ์ wallet ที่ผลิตในสวิตเซอร์แลนด์ซึ่งออกแบบมาเพื่อรักษาความปลอดภัยให้กับ Bitcoins ของคุณโดยเฉพาะ คุณสมบัติหลักบางประการของมันรวมถึงการสำรองข้อมูลและการกู้คืนที่ง่ายดายด้วยการใช้ microSD card, การออกแบบที่เรียบง่ายและไม่สะดุดตา, และการสนับสนุนอย่างครอบคลุมสำหรับ Bitcoin.


![device](assets/1.webp)


มันนำเสนอความปลอดภัยล้ำสมัยที่ออกแบบโดยผู้เชี่ยวชาญ มีการออกแบบชิปคู่ที่รวมถึงชิปที่ปลอดภัย โค้ดต้นฉบับได้รับการตรวจสอบอย่างเต็มที่โดยนักวิจัยด้านความปลอดภัยและเป็นโอเพ่นซอร์สทั้งหมด BitBox02 มาพร้อมกับ BitBoxApp ที่เรียบง่ายแต่ทรงพลัง ซึ่งให้การจัดการ Bitcoin ของคุณอย่างปลอดภัย รองรับ full node สำหรับ Bitcoin และรับรองการสื่อสารที่เข้ารหัสจากต้นทางถึงปลายทางระหว่างแอปและอุปกรณ์ ผลิตในสวิตเซอร์แลนด์ BitBox02 ได้รับชื่อเสียงในเชิงบวกจากผู้ใช้


![video](https://youtu.be/sB4b2PbYaj0)


นี่คือสเปคของ wallet:



- การเชื่อมต่อ: USB-C
- ความเข้ากันได้: Windows 7 และใหม่กว่า, macOS 10.13 และใหม่กว่า, Linux, Android
- อินพุต: เซ็นเซอร์สัมผัสแบบ capacitive
- ไมโครคอนโทรลเลอร์: ATSAMD51J20A; 120 Mhz 32-bit Cortex-M4F; ตัวสร้างตัวเลขสุ่มจริง
- ชิปความปลอดภัย: ATECC608B; เครื่องสร้างตัวเลขสุ่มแท้จริง (NIST SP 800-90A/B/C)
- หน้าจอ: 128 x 64 px OLED สีขาว
- วัสดุ: โพลีคาร์บอเนต
- ขนาด: 54.5 x 25.4 x 9.6 มม. รวมปลั๊ก USB-C
- น้ำหนัก: อุปกรณ์ 12g; พร้อมบรรจุภัณฑ์และอุปกรณ์เสริม 160g


ดาวน์โหลดแผ่นข้อมูลได้ที่เว็บไซต์ของพวกเขา https://bitbox.swiss/bitbox02/


## วิธีใช้ BitBox02 Hardware Wallet


### การตั้งค่า BitBox02


BitBox02 มีการเชื่อมต่อ USB-C ติดอยู่กับตัวเครื่อง หากคอมพิวเตอร์ของคุณใช้พอร์ต USB ปกติ คุณจะต้องใช้อะแดปเตอร์ที่มาพร้อมกับอุปกรณ์


เชื่อมต่อกับคอมพิวเตอร์ของคุณและอุปกรณ์จะเปิดเครื่อง (อย่าทำตอนนี้)


มีเซ็นเซอร์ด้านบนและด้านล่าง และจะเตือนให้คุณสัมผัสด้านบนหรือด้านล่างเพื่อปรับหน้าจอในทิศทางที่คุณต้องการ

### ไม่เร็วขนาดนั้น!!


มันค่อนข้างแปลก แต่ BitBox02 กำลังบอกเราว่าเราพร้อมที่จะใช้เครื่องมือนี้แล้ว แต่ยังไม่ได้แจ้งให้เราจดคำ seed! การสำรองข้อมูลเพียงอย่างเดียวที่เรามีคือไฟล์ที่บันทึกลงในการ์ด microSD ซึ่งไม่เพียงพอ อุปกรณ์จัดเก็บข้อมูลเหล่านี้ไม่ได้คงอยู่ตลอดไป (เนื่องจาก "bit rot") เราต้องการการสำรองข้อมูลบนกระดาษ และสำเนาเก็บไว้ในตู้เซฟ (ตามที่อธิบายไว้ในคู่มือการใช้กระเป๋าเงินฮาร์ดแวร์ทั่วไป)


ในการรับวลี seed ของเราและเขียนลงไป ให้ไปที่แท็บ “จัดการอุปกรณ์” ทางด้านซ้าย จากนั้นคลิก “แสดงคำกู้คืน”


![image](assets/16.webp)


จากนั้นคุณสามารถดำเนินการยืนยัน และอุปกรณ์จะนำเสนอคำให้คุณ เขียนคำเหล่านั้นลงอย่างเรียบร้อย และอย่าให้ใครเห็นคำเหล่านั้นเด็ดขาด


![image](assets/17.webp)


หลังจากนั้น คุณสามารถคลิกที่แท็บ Bitcoin ทางด้านซ้ายเพื่อรับที่อยู่สำหรับการรับของคุณ


![image](assets/18.webp)


มันแสดงทีละรายการ แต่ก็ยังให้คุณเลือกที่อยู่ที่จะใช้จาก 20 รายการแรก:


![image](assets/19.webp)


การคลิกปุ่มสีน้ำเงินจะแสดงที่อยู่เต็ม และคุณจะได้รับการแจ้งเตือนให้ตรวจสอบว่าที่อยู่ตรงกับที่แสดงบนหน้าจอหรือไม่ นี่เป็นวิธีปฏิบัติที่ดีในการยืนยันว่าไม่มีมัลแวร์บนคอมพิวเตอร์ของคุณที่หลอกให้คุณส่งบิตคอยน์ไปยังที่อยู่ของผู้โจมตี


![image](assets/20.webp)


ในการส่งบิทคอยน์ไปยัง wallet นี้ คุณสามารถคัดลอกที่อยู่และวางลงในหน้าถอนของการแลกเปลี่ยนที่เหรียญของคุณอยู่ ฉันแนะนำให้คุณส่งจำนวนเล็กน้อยเพื่อทดสอบก่อน จากนั้นฝึกใช้จ่ายมันกลับไปยังการแลกเปลี่ยนหรือไปยังที่อยู่ที่สองใน wallet ของคุณ


สำหรับจำนวนที่มากขึ้น ฉันแนะนำให้คุณสร้าง passphrase (ดูด้านล่าง) wallet ดั้งเดิม (ไม่มี passphrase) สามารถใช้เป็น wallet หลอกของคุณได้ (จะต้องมีจำนวนที่สมเหตุสมผลเพื่อให้ดูน่าเชื่อถือ)


### เชื่อมต่อกับโหนด


BitBox02 จะเชื่อมต่อกับโหนดโดยอัตโนมัติ มาดูกันว่ามันกำลังเชื่อมต่อไปที่ไหน คลิกที่แท็บการตั้งค่าทางด้านซ้าย แล้วเลือก “เชื่อมต่อ full node ของคุณเอง”


![image](assets/21.webp)


และที่นี่เราสามารถเห็นว่ามันกำลังเชื่อมต่อกับโหนดของ shiftcrypto ไม่ดีเลย เราได้เปิดเผยที่อยู่บิตคอยน์ทั้งหมดของเราให้กับพวกเขา และที่อยู่ IP ของเรา (แน่นอนว่าไม่ใช่ seed; พวกเขาสามารถเห็นที่อยู่/ยอดคงเหลือของเรา แต่ไม่สามารถใช้จ่ายได้) เราสามารถป้อนรายละเอียดโหนดของเราเองในหน้านี้ (เกินขอบเขตของคู่มือเฉพาะนี้) หรือเราสามารถใช้ซอฟต์แวร์ที่ดีกว่าเช่น Sparrow Bitcoin Wallet, Electrum Desktop Wallet, หรือ Specter Desktop ฉันจะแสดงตัวอย่าง Sparrow Bitcoin Wallet ในคู่มือนี้ภายหลัง


![image](assets/22.webp)


เพิ่ม passphrase


ตอนนี้ที่เราได้ตั้งค่าอุปกรณ์ด้วยแอป BitBox02 (และเปิดเผยที่อยู่ของเรา ซึ่งหลีกเลี่ยงไม่ได้กับฮาร์ดแวร์ wallet นี้) เราสามารถเพิ่ม passphrase ลงในวลี seed ของเราได้ สิ่งนี้จะช่วยให้เราสร้าง wallet ใหม่โดยใช้ seed เดียวกัน และ ShiftCrypto จะไม่เห็นที่อยู่ใหม่ของเรา เราจะเชื่อมต่อ wallet นี้กับซอฟต์แวร์ของเราเองเท่านั้น
