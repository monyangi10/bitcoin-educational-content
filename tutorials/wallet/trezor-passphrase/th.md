![cover](assets/cover.webp)



passphrase BIP39 เป็นรหัสผ่านที่เลือกใช้ได้ ซึ่งเมื่อรวมกับวลีช่วยจำ จะให้ชั้นความปลอดภัยเพิ่มเติมสำหรับกระเป๋าเงิน Bitcoin ที่กำหนดและมีลำดับขั้น ในบทแนะนำนี้ เราจะค้นพบไปด้วยกันว่าจะตั้งค่า passphrase บน Bitcoin wallet ที่ปลอดภัยของคุณบน Trezor (Safe 3, Safe 5 และ Model One) ได้อย่างไร



![Image](assets/fr/01.webp)



ก่อนเริ่มต้นบทแนะนำนี้ หากคุณไม่คุ้นเคยกับแนวคิดของ passphrase วิธีการทำงานและผลกระทบต่อ Bitcoin wallet ของคุณ ฉันขอแนะนำอย่างยิ่งให้คุณปรึกษาบทความทฤษฎีอื่นที่ฉันอธิบายทุกอย่าง (นี่เป็นสิ่งสำคัญมาก เพราะการใช้ passphrase โดยไม่เข้าใจวิธีการทำงานอย่างถ่องแท้อาจทำให้บิตคอยน์ของคุณตกอยู่ในความเสี่ยง) :



https://planb.academy/tutorials/wallet/backup/passphrase-a26a0220-806c-44b4-af14-bafdeb1adce7

passphrase บน Trezor ถูกจัดการในวิธีคลาสสิกหากคุณเลือกใช้มาตรฐาน BIP39 ระหว่างการกำหนดค่า (ซึ่งฉันแนะนำหากคุณไม่ต้องการ *Multi-share Backup*) สิ่งที่พิเศษเกี่ยวกับ Trezor คือคุณสามารถป้อน passphrase ได้โดยตรงบน Hardware Wallet หรือผ่านแป้นพิมพ์ของคอมพิวเตอร์โดยใช้ซอฟต์แวร์ Trezor Suite ตัวเลือกที่สองนี้มีความปลอดภัยน้อยกว่ามาก เนื่องจากคอมพิวเตอร์มีพื้นผิวการโจมตีที่ใหญ่กว่ามากเมื่อเทียบกับ Hardware Wallet อย่างไรก็ตาม การพิมพ์ passphrase ที่ซับซ้อนอาจเร็วกว่าในแป้นพิมพ์ทั่วไปมากกว่าบน Hardware Wallet ซึ่งอาจกระตุ้นให้ใช้รหัสผ่านที่แข็งแกร่ง ดังนั้นจึงดีกว่าเสมอที่จะใช้ passphrase แม้ว่าจะต้องพิมพ์ก็ตาม ดีกว่าไม่ใช้เลย อย่างไรก็ตาม สิ่งสำคัญคือต้องตระหนักถึงความเสี่ยงที่เพิ่มขึ้นของการโจมตีทางตัวเลขที่สิ่งนี้บ่งบอกถึง



ตัวเลือกเหล่านี้ไม่สามารถใช้ได้กับซอฟต์แวร์การจัดการ wallet ที่เข้ากันได้กับ Trezor ทั้งหมด ตัวอย่างเช่น สำหรับ Model One สามารถป้อน passphrase ผ่านแป้นพิมพ์บน Sparrow Wallet ได้ สำหรับ Model T, รุ่น Safe 3 และ Safe 5 คุณต้องใช้ Trezor Suite หรือป้อน passphrase โดยตรงบน Hardware Wallet เนื่องจากตัวเลือกการป้อนผ่าน Sparrow ถูกปิดใช้งานโดย HWI เมื่อไม่กี่ปีที่ผ่านมา



![Image](assets/fr/02.webp)



ใน Trezor Suite คุณมีสองวิธีที่แตกต่างกันในการจัดการความต้องการ passphrase คุณสามารถเปิดใช้งานตัวเลือก "*passphrase*" ในแท็บ "*Device*" หากเปิดใช้งาน Trezor Suite และซอฟต์แวร์การจัดการ wallet อื่น ๆ ทั้งหมดจะขอให้คุณป้อน passphrase ของคุณทุกครั้งที่เริ่มต้น หากคุณต้องการวิธีการที่ไม่เด่นชัดในการใช้ passphrase คุณสามารถตั้งค่าไว้ที่ "*Standard*" ในกรณีนี้ คุณจะต้องเข้าถึงเมนูของ Hardware Wallet ของคุณด้วยตนเองที่มุมซ้ายบน และคลิกที่ปุ่ม "*+ passphrase*" ทุกครั้งที่คุณเริ่มต้นใช้งาน



ก่อนเริ่มต้นบทแนะนำนี้ โปรดตรวจสอบให้แน่ใจว่าคุณได้เริ่มต้น Trezor ของคุณและสร้างวลีช่วยจำของคุณแล้ว หากคุณยังไม่ได้ทำ และ Trezor ของคุณเป็นของใหม่ ให้ทำตามบทแนะนำเฉพาะรุ่นที่มีอยู่ใน Plan ₿ Academy เมื่อคุณทำขั้นตอนนี้เสร็จแล้ว คุณสามารถกลับมาที่บทแนะนำนี้ได้

## การเพิ่ม passphrase ไปยัง Safe 3 หรือ Safe 5



เมื่อคุณสร้าง wallet ของคุณแล้ว บันทึกคำช่วยจำและตั้งค่า PIN คุณจะถูกนำไปยังเมนูหลักของ Trezor Suite ที่มุมซ้ายบน ควรมีหน้าต่างปรากฏขึ้นเชิญให้คุณเปิดใช้งาน passphrase BIP39



![Image](assets/fr/03.webp)



หากหน้าต่างนี้ไม่ปรากฏขึ้น คุณจะต้องเปิดใช้งานตัวเลือก "*passphrase*" ในแท็บการตั้งค่า "*Device*" ด้วยตนเอง



![Image](assets/fr/04.webp)



หน้าต่างนี้ขอให้คุณป้อน passphrase ของคุณ เลือก passphrase ที่แข็งแกร่งและทำการสำรองข้อมูลทางกายภาพทันที บนสื่อเช่นกระดาษหรือโลหะ ในตัวอย่างนี้ ฉันได้เลือก passphrase: `fH3&kL@9mP#2sD5qR!82` นี่เป็นเพียงตัวอย่าง; อย่างไรก็ตาม ฉันแนะนำให้คุณเลือก passphrase ที่ยาวกว่านี้เล็กน้อย ระหว่าง 30 ถึง 40 ตัวอักษรจะเป็นอุดมคติ (เหมือนรหัสผ่านที่ดี)



แน่นอน คุณไม่ควรแชร์ passphrase ของคุณบนอินเทอร์เน็ต เช่นเดียวกับที่ฉันทำในบทแนะนำนี้ ตัวอย่าง wallet นี้จะถูกใช้เฉพาะบน Testnet และจะถูกลบเมื่อสิ้นสุดบทแนะนำ



สำหรับคำแนะนำเฉพาะเจาะจงเพิ่มเติมเกี่ยวกับการเลือก passphrase ของคุณ ฉันขอเชิญคุณอีกครั้งให้ปรึกษาบทความอื่นนี้:



https://planb.academy/tutorials/wallet/backup/passphrase-a26a0220-806c-44b4-af14-bafdeb1adce7

หากคุณต้องการป้อน passphrase ของคุณผ่านแป้นพิมพ์คอมพิวเตอร์ ให้ป้อนในช่องที่ให้ไว้ จากนั้นคลิกที่ "*Access passphrase Wallet*"



![Image](assets/fr/05.webp)



Hardware Wallet ของคุณจะแสดง passphrase ของคุณ จากนั้นตรวจสอบให้แน่ใจว่าตรงกับการสำรองข้อมูลทางกายภาพของคุณ (กระดาษหรือโลหะ) ก่อนคลิกบนหน้าจอเพื่อดำเนินการต่อ



![Image](assets/fr/06.webp)



สิ่งนี้จะทำให้คุณสามารถเข้าถึง passphrase ที่ได้รับการป้องกัน wallet ของคุณได้



![Image](assets/fr/07.webp)



หากคุณต้องการเพิ่มความปลอดภัยโดยการป้อน passphrase ของคุณเฉพาะบน Trezor ของคุณ เมื่อได้รับการแจ้งเตือน ให้คลิกที่ "*Enter passphrase on Trezor*"



![Image](assets/fr/08.webp)



แป้นพิมพ์ T9 จะปรากฏบน Trezor ของคุณ ช่วยให้คุณป้อน passphrase ของคุณ เมื่อคุณป้อนเสร็จแล้ว ให้คลิกที่เครื่องหมายถูกสีเขียวเพื่อใช้ passphrase กับ wallet ของคุณ



![Image](assets/fr/09.webp)



คุณจะสามารถเข้าถึง passphrase ที่ปลอดภัย wallet ของคุณได้



![Image](assets/fr/10.webp)



ในการใช้ Sparrow Wallet ขั้นตอนจะคล้ายกัน แต่สำหรับรุ่น T, Safe 3 และ Safe 5 ต้องป้อน passphrase บน Hardware Wallet และไม่ผ่านทางแป้นพิมพ์คอมพิวเตอร์



เมื่อใดก็ตามที่ Sparrow Wallet ต้องการเข้าถึง Trezor ของคุณ และ passphrase ยังไม่ได้ถูกใช้ตั้งแต่เริ่มต้นครั้งล่าสุด คุณจะต้องป้อนมันโดยใช้แป้นพิมพ์ T9



![Image](assets/fr/11.webp)



## การเพิ่ม passphrase ไปยัง Model One



บน Model One การใช้ passphrase BIP39 เป็นสิ่งที่แทบจะขาดไม่ได้ เนื่องจากอุปกรณ์นี้ไม่ได้รวม Secure Element ไว้ จึงค่อนข้างง่ายที่จะดึงข้อมูลที่ละเอียดอ่อนออกมา ดังนั้นจึงไม่ทนทานต่อการโจมตีทางกายภาพ อย่างไรก็ตาม เนื่องจาก passphrase จะไม่ถูกเก็บไว้ในอุปกรณ์หลังจากที่ปิดเครื่อง การใช้ passphrase ที่แข็งแกร่ง (ไม่สามารถถูกโจมตีด้วยวิธี brute force) สามารถปกป้องคุณจากการโจมตีทางกายภาพที่รู้จักส่วนใหญ่บนรุ่นนี้ได้



บน Model One ไม่สามารถป้อน passphrase โดยตรงบน Hardware Wallet ได้ คุณจะต้องป้อนผ่านแป้นพิมพ์คอมพิวเตอร์ของคุณ



เมื่อคุณสร้าง wallet ของคุณแล้ว บันทึกคำช่วยจำและตั้งค่า PIN คุณจะถูกนำไปยังเมนูหลักของ Trezor Suite ที่มุมซ้ายบน ควรปรากฏหน้าต่างเชิญให้คุณเปิดใช้งาน passphrase BIP39



![Image](assets/fr/12.webp)



หากหน้าต่างนี้ไม่ปรากฏขึ้น คุณจำเป็นต้องเปิดใช้งานตัวเลือก "*passphrase*" ในแท็บ "*Device*" ของการตั้งค่า



![Image](assets/fr/13.webp)



หน้าต่างนี้ขอให้คุณป้อน passphrase ของคุณ เลือก passphrase ที่แข็งแกร่งและทำการสำรองข้อมูลทางกายภาพทันที บนสื่อเช่นกระดาษหรือโลหะ ในตัวอย่างนี้ ฉันได้เลือก passphrase: `fH3&kL@9mP#2sD5qR!82` นี่เป็นเพียงตัวอย่าง อย่างไรก็ตาม ฉันแนะนำให้คุณเลือก passphrase ที่ยาวกว่านี้เล็กน้อย ระหว่าง 30 ถึง 40 ตัวอักษรจะเหมาะสมที่สุด (เหมือนรหัสผ่านที่ดี)



สำหรับคำแนะนำเฉพาะเจาะจงเพิ่มเติมเกี่ยวกับการเลือก passphrase ของคุณ ฉันขอเชิญคุณอีกครั้งให้ปรึกษาบทความอื่นนี้:



https://planb.academy/tutorials/wallet/backup/passphrase-a26a0220-806c-44b4-af14-bafdeb1adce7

ป้อน passphrase ของคุณในช่องที่ให้ไว้ จากนั้นคลิกที่ปุ่ม "*Access passphrase Wallet*"



![Image](assets/fr/14.webp)



Hardware Wallet ของคุณจะแสดง passphrase ของคุณ ตรวจสอบให้แน่ใจว่าตรงกับการสำรองข้อมูลทางกายภาพของคุณ (กระดาษหรือโลหะ) จากนั้นคลิกที่ปุ่มด้านขวาเพื่อดำเนินการต่อ



![Image](assets/fr/15.webp)



สิ่งนี้จะนำคุณไปยัง passphrase ที่ได้รับการป้องกัน wallet ของคุณ



![Image](assets/fr/16.webp)



ในการใช้ Sparrow Wallet หลังจากนั้น ขั้นตอนยังคงเหมือนเดิม ทุกครั้งที่ Sparrow ต้องการเข้าถึง Hardware Wallet ของคุณ และ passphrase ยังไม่ได้ถูกป้อนตั้งแต่เริ่มต้นอุปกรณ์ครั้งล่าสุด คุณจะต้องป้อนมัน



![Image](assets/fr/17.webp)



ยินดีด้วย คุณได้เรียนรู้การใช้ passphrase BIP39 บนกระเป๋าเงินฮาร์ดแวร์ Trezor แล้ว หากคุณต้องการเพิ่มความปลอดภัยของ wallet ไปอีกขั้น ลองดูบทแนะนำนี้เกี่ยวกับระบบสำรองข้อมูล *Multi-share* ของ Trezor (*Shamir's Secret Sharing Scheme*):



https://planb.academy/tutorials/wallet/backup/trezor-shamir-backup-7f98b593-face-48fb-a643-0e811b87c94e

หากคุณพบว่าบทแนะนำนี้มีประโยชน์ ฉันจะรู้สึกขอบคุณมากหากคุณจะกดนิ้วโป้งสีเขียวด้านล่างนี้ อย่าลังเลที่จะแบ่งปันบทความนี้บนเครือข่ายสังคมของคุณ ขอบคุณมาก!
