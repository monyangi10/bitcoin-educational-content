## 🚀 Добро пожаловать в Alby Go - самый простой в использовании мобильный Wallet


**Alby Go** - это простое в использовании мобильное приложение с открытым исходным кодом, которое выполняет функции молниевых узлов и кошельков Wallet Interface - Bitcoin. Вот как начать работу и извлечь максимум пользы.


**✅ Поддерживаемые и известные совместимые кошельки/ноды:**



- [Alby Hub](https://albyhub.com/) **(рекомендуется)**
- **Umbrel**, **Start9**, **RaspiBlitz** (via **Alby Hub**)
- **Coinos** *(не проверено)*
- **Primal** *(не проверено)*
- **Минибиты** *(не проверено)*


Большинство сервисов с поддержкой NWC должны работать. Если вы тестируете новый сервис, сообщите сообществу о своих результатах!

## 🔌 Подключите Wallet


Alby Go подключается к узлу с поддержкой молнии или Wallet с помощью секрета NWC (Nostr Wallet Connect). Вы можете связать один или несколько кошельков, чтобы легко переключаться между ними.

Шаги:

1. Откройте Настройки → Кошельки → Подключить Wallet

2. Отсканируйте QR-код или вставьте секрет подключения NWC (например, nostr+walletconnect://...)

3. Дайте своему соединению пользовательское имя


После подключения ваш Wallet или узел готов к отправке и приему молниеносных платежей Bitcoin через Alby Go.


✅ Совет: Вы можете подключить несколько кошельков и переключаться между ними в любое время.




## ⚡ Отправить Bitcoin


Чтобы передать Sats через Lightning Network:

1. Нажмите большую желтую кнопку **ОТПРАВИТЬ**.

2. Выберите один из следующих вариантов:


 - Сканирование QR-кода молнии Invoice
 - Вставить молнию Invoice из буфера обмена
 - Ручной ввод молнии Address


Вы также можете выбрать получателя из книги Address, в которой можно сохранять адреса молний для удобства повторного использования.




## 💸 Получение Bitcoin

Получить Sats с Элби Гоу:

1. Нажмите большую желтую кнопку **ПРИНЯТЬ**.

2. Выберите один из следующих вариантов:


 - Поделитесь Вашими впечатлениями от Молния Address
 - Выберите "Сумма" для generate молнии Invoice с пользовательскими суммами
 - Нажмите "Redeem", чтобы отсканировать QR-коды для вывода средств LNURL


Для удобства отправителя доступны как QR-коды, так и строки Invoice.
