## 🚀歡迎使用 Alby Go - 最容易使用的行動 Wallet


**Alby Go** 是一個開放原始碼、易於使用的行動應用程式，可作為 Wallet Interface 到 Bitcoin 的閃電節點和錢包。以下是如何開始使用並充分利用您的體驗。

- [Alby Hub](https://albyhub.com/) **(建議)**

- **Coinos** *（未測試）*

## 🔌 連接 Wallet


Alby Go 可使用 NWC (Nostr Wallet Connect) 秘訣連接至您啟用閃光燈的節點或 Wallet。您可以連結一個或多個錢包，輕鬆切換。

步驟：

1.開啟設定 → 錢包 → 連接 Wallet

2.掃描 QR 代碼或貼上 NWC Connection Secret (例如 nostr+walletconnect://...)

3.自訂連線名稱


連線後，您的 Wallet 或節點就可以透過 Alby Go 傳送和接收 Bitcoin 閃電付款。


✅ 提示：您可以連接多個錢包，並隨時在它們之間切換。




## 發送 Bitcoin


透過 Lightning Network 傳送 Sats：

1.點選黃色 **SEND 按鈕**。

2.請選擇下列其中一項：


 - 掃描閃電 Invoice QR 代碼
 - 從剪貼簿貼上閃電 Invoice
 - 手動輸入閃電 Address


您也可以從 Address Book 中選擇收件人，您可以儲存閃電地址，方便再次使用。




## 接收 Bitcoin

與 Alby Go 一起接收 Sats：

1.點選黃色 **RECEIVE 按鈕**。

2.請選擇下列其中一項：


 - 分享您所顯示的閃電 Address
 - 選擇「金額」至 generate a 自訂金額的閃電 Invoice
 - 按一下「Redeem」以掃瞄 LNURL 提款 QR 代碼


QR 代碼和 Invoice 字串都可用，以方便寄件者。
