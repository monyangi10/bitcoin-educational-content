## 开始使用 Tando



Tando 是一款可在 Android 和 iOS 平台上使用的移动应用程序，让您可以在肯尼亚的任何地方使用 Bitcoin。在本教程中，我们将重点介绍安卓平台。不过，下面详细介绍的过程也适用于 iOS 平台。



![download](assets/fr/01.webp)



Tando 是一种基于两种技术的 Exchange 介质：





- Lightning Network：Bitcoin Layer，可实现即时、几乎免费的支付。
- M-Pesa：肯尼亚最流行的移动支付。



它不需要用户提供个人信息，也不需要身份验证。有了 Tando，您可以在进行交易时保护自己的隐私。



在肯尼亚，大多数商家、超市、服务提供商和金融交易都可以使用 M-Pesa。通过弥合 Lightning Network 和 M-Pesa 之间的差距，Tando 允许您在任何接受 M-Pesa 的地方使用比特币，收款人收到的是肯尼亚先令（KES）。



有了 Tando，您就可以使用 Bitcoin 来 .NET 和 .NET 技术：




- 支付货物和服务
- 汇款
- 支付账单



![menu](assets/fr/02.webp)



通过直观、简约的 Interface，Tando 可让您使用任何 Lightning Wallet 以肯尼亚先令（KES）进行支付。这为来肯尼亚的外国人提供了一个机会，他们可以不带现金，使用自己的 Lightning Wallet 正常生活。Tando还为Bitcoin提供了相关的使用框架，鼓励肯尼亚社区采用Bitcoin。



## 使用 Tando



Tando 让您可以用比特币在肯尼亚购买任何 M-Pesa 可以购买的东西。此外，与本地 M-Pesa 交易相比，交易费用几乎为零。使用 Tando 时，您可以用闪电 Wallet 支付等值的肯尼亚先令，Tando 将使用其基础设施将您的闪电支付转换成肯尼亚先令，然后将金额发送到您指定的 M-Pesa 号码。

- 扫描付款** ：



扫描支付是应用程序的自动支付选项之一。扫描商家提供的 M-Pesa 代码，然后输入要支付的金额，接着支付为您生成的闪电 Invoice。



![scan](assets/fr/03.webp)





- 向肯尼亚汇款** ：



通过 Tando 的汇款选项，您可以从世界任何地方向肯尼亚汇款。因此，您可以使用 Lightning Wallet 向肯尼亚进行跨境和洲际交易，而无需支付高昂的费用。



选择 "**汇款**"选项，然后输入收款人号码。输入要发送的金额（15 至 50,000 肯尼亚先令之间），然后创建与此付款相关的闪电 Invoice。



![send-money](assets/fr/04.webp)



使用 Lightning Wallet 支付账单，Tando 会将其兑换成肯尼亚先令 (KES) 并发送给指定收件人。



![pay](assets/fr/05.webp)





- 支付账单** ：



输入您要支付的 Invoice 编号，然后继续支付相关的闪电 Invoice。



![bills](assets/fr/06.webp)





- 购买商品** ：



直接使用 Tando 购物。输入商户的商户编号和总购物金额（肯尼亚先令），然后用 generate 闪电 Invoice 付款。



![buy](assets/fr/07.webp)



在应用程序的菜单中，您还可以找到有关 Tando 的所有信息，从而更好地了解他们的愿景和定价。



![about](assets/fr/08.webp)
