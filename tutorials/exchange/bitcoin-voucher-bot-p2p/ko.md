SEPA 은행 송금을 통해 [KYC](https://planb.academy/resources/glossary/kyc-know-your-customer)없이 Bitcoin를 구매하기 위해 만들어진 Telegram 봇인 BitcoinVoucherBot에 대한 문의가 여전히 이어지고 있습니다. 안타깝게도 2025년 11월부터 중앙화된 형태의 BitcoinVoucherBot은 더 이상 KYC 없이 서비스를 이용할 수 없습니다.



이 가이드에서는 사용자가 새로운 P2P(P2P) 마켓플레이스에서 KYC 없이 Bitcoin을 직접 사고 팔 수 있도록 하는 새로운 구현이 어떻게 작동하는지 살펴보겠습니다. 디지털 자유와 프라이버시를 점점 더 위협하는 새로운 규제에 대응하기 위해 개발자들은 이 확장 기능을 개발하여 사용자가 P2P(P2P)을 통해 높은 수준의 익명성으로 Bitcoin을 사고 팔 수 있는 기능을 제공하게 되었습니다. 이 새로운 교환 방법이 어떻게 작동하는지 함께 살펴보겠습니다.



이 서비스를 이용하려면 Lightning Network을 통해 이체해야 합니다. 따라서 이 프로토콜을 지원하는 wallet이 있고 'LNURL' 또는 'Lightning Address'를 사용하여 자금을 송금할 수 있는지 확인하세요.

## Bitcoin 구매 주문 생성



마켓플레이스에 들어가기: 기본 탐색 모음에서 장바구니 기호인 "마켓플레이스"("마켓플레이스")를 클릭하여 주문서를 엽니다. 새 주문을 시작하기: "새 주문" 버튼("새 주문")을 눌러 프로세스를 시작합니다.



![image](assets/it/13.webp)





- 주문 세부 정보를 설정합니다:
- "Bitcoin 구매"("Bitcoin 구매") 옵션을 선택합니다.
- 원하는 sats의 양을 입력합니다.
- 가격 마진(시장가 대비 -20%에서 +20% 사이)을 정의합니다.
- 결제 방법(즉시 결제 등)을 선택합니다.
- 기본 통화를 표시합니다.
- 주문 확인: '주문 생성'('주문 확인')을 클릭하여 제출 단계로 진행합니다.



![image](assets/it/14.webp)



보증금 필요: 주문을 활성화하려면 총 금액의 10%에 해당하는 보증금과 서비스 수수료가 필요합니다.




- 보증금 결제: 주문이 생성되면 시스템에서 자동으로 라이트닝 인보이스를 생성합니다. 보증금은 일시적인 것이며 주문이 완료되면 환불됩니다.
- 주요 기능:
- 보증금: 주문 금액의 10%.
- 서비스 수수료: 플랫폼 사용 비용입니다.
- 시간 제한: 5분 이내에 결제를 완료해야 하며, 그렇지 않으면 거래가 만료됩니다.



![image](assets/it/15.webp)



결제가 완료되면 주문이 북에 표시되고 모든 사용자가 주문을 선택하고 수락할 수 있습니다. 판매 주문을 생성하려면 "Bitcoin 판매"("Bitcoin 판매")를 클릭하고 판매하고자 하는 satoshi의 수량을 입력한 후 마진을 설정하고 결제 방법과 통화를 선택한 다음 보증금으로 10%를 입금하기만 하면 됩니다. 완료되면 매치가 목록에 표시됩니다.



![image](assets/it/16.webp)



## 주문 수락 방법



1. 판매자는 책에서 사용 가능한 모든 주문 목록을 볼 수 있습니다.


2. 세부 정보 확인: 각 주문에는 다음과 같은 정보가 표시됩니다:




  - Bitcoin 수량,
  - 가격 마진,
  - 결제 방법,
  - 사용자 평판.



![image](assets/it/17.webp)



3. 주문을 클릭하면 모든 정보가 포함된 전체 시트를 열 수 있습니다.


4. 제안을 수락하려면 "Sell Bitcoin"("Bitcoin 판매")을 누릅니다.



![image](assets/it/18.webp)



## 판매자가 요구하는 보증금



주문이 수락되면 시스템에서 결제 인보이스를 생성합니다. 보증금에는 다음이 포함됩니다:



- 주문의 총 금액입니다,



- 서비스 수수료.



정해진 기한 내에 입금을 완료하지 않으면 거래가 확정되지 않습니다.



![image](assets/it/19.webp)



## 결제 지침 보내기



입금이 완료되면 판매자의 개인 대시보드에 거래가 표시되며, 판매자는 법정 화폐로 결제를 완료하기 위해 구매자에게 세부 정보를 제공해야 합니다.



1. 판매자는 패널에 활성 거래를 표시합니다.


2. "결제 지침 제출"을 클릭합니다


3. 법정 화폐 결제에 필요한 모든 정보(IBAN, 수취인, 주소, 결제 사유 등)를 입력합니다.


4. 구매자에게 데이터를 전송하려면 "메시지 보내기"("메시지 보내기")를 클릭합니다.



![image](assets/it/20.webp)



## 결제 절차



구매자는 플랫폼 채팅 내에서 법정 화폐로 결제하는 데 필요한 모든 데이터가 포함된 메시지를 받게 됩니다:




- 은행 세부 정보: 판매자 계좌 소유자의 이름과 주소가 포함된 IBAN입니다.
- 정확한 금액: 이체할 정확한 법정 화폐 금액입니다.
- 인과관계/설명: 거래에 포함할 텍스트입니다.
- 시간 제한: 결제를 완료해야 하는 기한입니다.

## 결제 법정화폐 확인



이 단계는 구매자에게 매우 중요하며 실제로 결제가 송금되었는지 확인한 후에만 수행해야 합니다.



1. 데이터 수신: 구매자가 판매자로부터 결제 지침을 받았습니다.


2. 결제 실행: 은행 계좌에서 법정 화폐를 이체합니다.


3. 확인: 작업이 올바르게 처리되었는지 확인합니다.


4. 플랫폼에서 확인: 거래 페이지에서 "법정화폐 결제 확인"("법정화폐 결제 확인")을 클릭합니다.


'결제 법정화폐 확인' 버튼은 거래 섹션에 표시되며, 결제가 실제로 송금되었는지 확인한 후에만 사용해야 합니다.



![image](assets/it/22.webp)
