KaleidoSwap เป็นแอปพลิเคชันเดสก์ท็อปโอเพ่นซอร์สที่ซับซ้อนซึ่งเชื่อมช่องว่างระหว่างโปรโตคอล RGB และ Lightning Network มันทำหน้าที่เป็นอินเทอร์เฟซที่ครอบคลุมสำหรับการจัดการ RGB Lightning Nodes, การโต้ตอบกับ RGB Lightning Service Providers (LSPs) ผ่านสเปค LSPS1, และการดำเนินการสวอปอะตอมของสินทรัพย์ RGB


ในฐานะโซลูชันที่ไม่ต้องมีการควบคุม KaleidoSwap ช่วยให้ผู้ใช้สามารถควบคุมกุญแจและข้อมูลของตนได้อย่างเต็มที่ โดยการใช้รูปแบบการตรวจสอบฝั่งไคลเอนต์ของ RGB มันช่วยให้สามารถใช้สัญญาอัจฉริยะที่เป็นส่วนตัวและขยายได้บน Bitcoin บทแนะนำนี้เจาะลึกคุณสมบัติขั้นสูงของ KaleidoSwap โดยแนะนำคุณผ่านความซับซ้อนของการจัดการ "Colored" UTXO สภาพคล่องของช่องทางสำหรับสินทรัพย์เฉพาะ และรูปแบบการซื้อขายแบบผู้รับ-ผู้สร้าง เพื่อให้แน่ใจว่าคุณสามารถใช้ประโยชน์จากโปรโตคอลการแลกเปลี่ยนแบบกระจายศูนย์ที่ทรงพลังนี้ได้อย่างเต็มที่


## การติดตั้ง


KaleidoSwap มีไบนารีที่สร้างไว้ล่วงหน้าสำหรับระบบปฏิบัติการหลัก ๆ แต่สำหรับผู้ใช้ขั้นสูง การสร้างจากซอร์สโค้ดจะช่วยให้มั่นใจได้ว่าคุณกำลังใช้งานโค้ดล่าสุดพร้อมการกำหนดค่าที่เฉพาะเจาะจงของคุณเอง


### ดาวน์โหลดไบนารี


คุณสามารถดาวน์โหลดรุ่นล่าสุดสำหรับระบบปฏิบัติการของคุณได้จาก [เว็บไซต์ทางการ](https://kaleidoswap.com/) หรือ [หน้าปล่อยบน GitHub](https://github.com/kaleidoswap/desktop-app/releases)


### วิธีการติดตั้ง


1.  **Windows**: ดาวน์โหลดไฟล์ติดตั้ง `.exe` และเรียกใช้งาน.

2.  **macOS**: ดาวน์โหลดไฟล์ `.dmg` เปิดไฟล์ และลาก KaleidoSwap ไปยังโฟลเดอร์ Applications ของคุณ

3.  **Linux**: ดาวน์โหลดไฟล์ `.AppImage` หรือ `.deb` และติดตั้ง/เรียกใช้งานมัน



## ตัวเลือกการตั้งค่าโหนด


เมื่อคุณเปิด KaleidoSwap เป็นครั้งแรก คุณจะพบกับ **หน้าจอการเชื่อมต่อ** นี่เป็นขั้นตอนแรกในการกำหนดค่าการใช้งานของคุณ


![Node Selection Screen](assets/en/01.webp)


คุณต้องเลือกวิธีการเชื่อมต่อกับ RGB Lightning Network การเลือกนี้มีผลต่อระดับการควบคุมและความเป็นส่วนตัวของคุณ


### ตัวเลือกที่ 1: โหนดท้องถิ่น (แนะนำสำหรับการดูแลตนเอง)


**สำหรับการควบคุมและความเป็นส่วนตัวอย่างสมบูรณ์** รันโหนดโดยตรงบนเครื่องของคุณ ดูข้อดีด้านล่าง:


- Self-Custodial**: คุณถือกุญแจ ไม่มีบุคคลที่สามสามารถระงับเงินของคุณหรือเซ็นเซอร์การทำธุรกรรมของคุณได้
- ความเป็นส่วนตัว**: ข้อมูลของคุณจะอยู่บนอุปกรณ์ของคุณเท่านั้น
- อิสรภาพ**: ไม่มีการพึ่งพาผู้ให้บริการภายนอก.


หากคุณเลือก **Local Node** คุณจะถูกนำไปยังหน้าจอตั้งค่าที่คุณสามารถสร้าง wallet ใหม่หรือกู้คืนที่มีอยู่แล้วได้


![Local Node Setup Screen](assets/en/02.webp)


### ตัวเลือกที่ 2: โหนดระยะไกล


เชื่อมต่อกับ RGB Lightning Node ระยะไกล (โฮสต์ด้วยตนเองบน VPS หรือผู้ให้บริการโฮสต์)


- ข้อดี**: ไม่มีการใช้ทรัพยากรในท้องถิ่น, พร้อมใช้งานตลอด 24/7.
- การแลกเปลี่ยน**: ต้องเชื่อถือโฮสต์หรือจัดการเซิร์ฟเวอร์ระยะไกล


![Remote Node Setup Screen](assets/en/03.webp)


**KaleidoSwap ถูกออกแบบมาเพื่อเสริมสร้างการควบคุมตนเอง** เราขอแนะนำอย่างยิ่งให้คุณรันโหนดของคุณเอง—ไม่ว่าจะเป็นในเครื่อง (ตัวเลือกที่ 1) หรือโดยการโฮสต์โหนดระยะไกลด้วยตนเอง—เพื่อใช้ประโยชน์จากคุณสมบัติต้านทานการเซ็นเซอร์ของ Bitcoin และ RGB อย่างเต็มที่


## สร้าง Wallet


KaleidoSwap จัดการสินทรัพย์ทั้ง Bitcoin และ RGB กระบวนการสร้าง wallet จะเริ่มต้น keystore ที่รักษาความปลอดภัยให้กับเงิน on-chain ของคุณและสถานะช่องทาง Lightning ของคุณ


นี่คือกระบวนการโดยละเอียด:

1. เปิด KaleidoSwap และเลือก **Local Node**.

2. คลิกที่ **Create New Wallet**.

3. **การตั้งค่าบัญชี**: ป้อน **ชื่อบัญชี** และเลือก **เครือข่าย** (เช่น Bitcoin Mainnet, Testnet, Signet).

4. **การกำหนดค่าขั้นสูง** (ไม่บังคับ): หากคุณเป็นผู้ใช้ระดับสูง คุณสามารถกำหนดค่าเอนด์พอยต์ RPC ที่กำหนดเอง, URL ของ Indexer หรือการตั้งค่า Proxy ภายใต้ "การตั้งค่าขั้นสูง"

5. คลิก **Continue**.

6. **รหัสผ่าน**: ตั้งรหัสผ่านที่รัดกุมเพื่อเข้ารหัสไฟล์ wallet ของคุณในเครื่อง.

7. **Recovery Phrase**: จดวลี seed ของคุณและเก็บไว้อย่างปลอดภัย


    - วิกฤติ**: วลีนี้จำเป็นสำหรับการกู้คืนเงิน on-chain และตัวตนของโหนดของคุณ
    - คำเตือน**: **สถานะช่องทาง Lightning ไม่สามารถกู้คืนได้อย่างสมบูรณ์จาก seed เพียงอย่างเดียว** คุณต้องรักษาการสำรองข้อมูลช่องทางแบบสถิต (SCB) เพื่อกู้คืนเงินที่ถูกล็อคในช่องทางด้วย


![Wallet Creation Screen](assets/en/04.webp)

## การระดมทุน Wallet ของคุณ


ในการดำเนินการกับสินทรัพย์ RGB คุณจำเป็นต้องเติมเงินให้กับ wallet ของคุณ KaleidoSwap รองรับการฝากสินทรัพย์ทั้ง Bitcoin และ RGB ผ่านธุรกรรม on-chain หรือ Lightning Network


### การฝากเงิน Bitcoin


1. คลิก **Deposit** ในเมนู Quick Actions

2. เลือก **BTC** จากรายการสินทรัพย์


![Select BTC Asset](assets/en/06.webp)


3. เลือกวิธีการฝากของคุณ: **On-chain** หรือ **Lightning**.


![BTC Deposit Options](assets/en/07.webp)



- On-chain**: สแกนรหัส QR หรือคัดลอกที่อยู่เพื่อส่ง Bitcoin จาก wallet อื่น
- Lightning**: สร้างใบแจ้งหนี้สำหรับจำนวนเงินที่ต้องการ


![BTC On-chain Deposit](assets/en/08.webp)


### การฝากสินทรัพย์ RGB & UTXOs ที่มีสี


ในการรับสินทรัพย์ RGB (เช่น USDT) คุณจำเป็นต้องมี UTXOs เฉพาะที่พร้อมสำหรับการ "ระบายสี" (กำหนดสินทรัพย์)


1. คลิก **Deposit** และเลือกสินทรัพย์ RGB (เช่น USDT) **สำคัญ**: หากนี่เป็น **ครั้งแรก** ที่โหนดของคุณได้รับสินทรัพย์นี้ **ปล่อยให้ช่อง Asset ID ว่างไว้** การใส่ ID สำหรับสินทรัพย์ที่ไม่รู้จักจะทำให้โหนดส่งคืนข้อผิดพลาดเนื่องจากยังไม่รู้จักสินทรัพย์นั้น

2. เลือก **On-chain** หรือ **Lightning**.


![USDT Deposit Options](assets/en/09.webp)


#### โหมดการรับแบบ On-chain: พยาน vs. ปิดบัง


เมื่อได้รับสินทรัพย์ RGB on-chain คุณมีโหมดความเป็นส่วนตัวสองโหมด:



- Blinded Receive (แนะนำสำหรับความเป็นส่วนตัว)**: คุณให้ "blinded" UTXO แก่ผู้ส่ง คุณกำลังขอให้ผู้ส่งส่งสินทรัพย์ไปยัง UTXO ที่มีอยู่ที่คุณเป็นเจ้าของ แต่คุณปิดบังตัวระบุ UTXO ที่แท้จริง สิ่งนี้ให้ความเป็นส่วนตัวที่ดียิ่งขึ้น
- พยานรับ**: คุณให้ที่อยู่ Bitcoin มาตรฐาน คุณกำลังขอให้ผู้ส่งสร้าง UTXO *ใหม่* สำหรับคุณโดยการส่งสินทรัพย์ไปยังที่อยู่นั้น ซึ่งจะทำให้สินทรัพย์ RGB ถูกแนบโดยตรงกับ UTXO ใหม่ที่สร้างขึ้นโดยการทำธุรกรรม


#### เงินฝากสายฟ้า


สำหรับการฝาก Lightning เพียงแค่ generate ใบแจ้งหนี้ สินทรัพย์ RGB จะถูกส่งไปยังคุณผ่านช่องทางที่เปิดอยู่ของคุณ


![USDT Lightning Invoice](assets/en/10.webp)



## เปิดช่องทางด้วยสินทรัพย์ RGB


ในการกำหนดเส้นทางสินทรัพย์ RGB ผ่าน Lightning Network คุณจำเป็นต้องมีช่องทางที่มีสภาพคล่องและการจัดสรรสินทรัพย์เพียงพอ วิธีที่ง่ายที่สุดในการเริ่มต้นคือ **ซื้อช่องทาง** จาก LSP (ผู้ให้บริการสายฟ้า)


### การซื้อช่องทางจาก Kaleido LSP


1. ไปที่แท็บ **Channels** คุณจะเห็นตัวเลือก "Open Channel" (manual) หรือ "Buy Channel" (LSP)

2. คลิก **ซื้อช่อง**.


![Channels Dashboard](assets/en/11.webp)


3. **เชื่อมต่อกับ LSP**: แอปจะเชื่อมต่อกับ Kaleido LSP เริ่มต้น ผู้ให้บริการนี้มีสภาพคล่องขาเข้าและรองรับการกำหนดเส้นทางสินทรัพย์ RGB


![Connect to LSP](assets/en/12.webp)


4. **กำหนดค่าช่อง**:


    - ความจุ**: เลือกความจุรวม Bitcoin สำหรับช่องทางนี้
    - RGB Allocation**: เลือกสินทรัพย์ RGB (เช่น USDT) ที่คุณต้องการรับหรือส่ง LSP จะทำให้แน่ใจว่าช่องทางถูกกำหนดค่าให้รองรับสินทรัพย์นี้


![Configure Channel](assets/en/13.webp)



    - การจัดสรร RGB**: เลือกสินทรัพย์ RGB (เช่น USDT) ที่คุณต้องการรับหรือส่ง LSP จะทำให้แน่ใจว่าช่องทางถูกกำหนดค่าเพื่อรองรับสินทรัพย์นี้


![RGB Allocation](assets/en/14.webp)


5.  **การชำระเงิน**: คุณต้องจ่ายค่าธรรมเนียมให้กับ LSP สำหรับการเปิดช่องทางและการให้สภาพคล่อง คุณสามารถชำระเงินโดยใช้ **Lightning** หรือ **On-chain** Bitcoin การชำระเงินสามารถทำได้จากภายใน KaleidoSwap wallet ของคุณหรือจากภายนอก wallet


![Complete Payment](assets/en/15.webp)


6. เมื่อยืนยันการชำระเงินแล้ว LSP จะเริ่มการทำธุรกรรมเปิดช่องทาง คุณจะเห็นหน้าจอ **Order Completed**


![Order Completed](assets/en/16.webp)


7. หลังจากการยืนยันบนบล็อกเชน ช่องของคุณจะพร้อมใช้งานและพร้อมสำหรับการโอน RGB



## การซื้อขาย: โมเดล Taker-Maker


เครื่องยนต์การซื้อขายของ KaleidoSwap ทำงานบน **โมเดล Taker-Maker** คุณสามารถแลกเปลี่ยนสินทรัพย์ด้วยตนเองกับเพื่อนหรือใช้ Market Maker (LSP)


### การแลกเปลี่ยนกับผู้ดูแลสภาพคล่อง (LSP)


นี่เป็นวิธีการซื้อขายที่พบมากที่สุด คุณทำหน้าที่เป็น **Taker** ดำเนินการคำสั่งซื้อขายกับสภาพคล่องที่มีอยู่ซึ่งจัดหาโดย LSP (**Maker**)


1. ไปที่แท็บ **Trade** และเลือก **Market Maker**.

2. **Enter Amount**: ป้อนจำนวน Bitcoin ที่คุณต้องการส่ง (หรือสินทรัพย์ที่คุณต้องการรับ) อินเทอร์เฟซจะแสดงอัตราแลกเปลี่ยนและค่าธรรมเนียมโดยประมาณ


![Market Maker Swap](assets/en/17.webp)


3. **ยืนยันการสลับ**: ตรวจสอบรายละเอียด รวมถึงอัตราแลกเปลี่ยนและจำนวนที่คุณจะได้รับ คลิก **ยืนยันการสลับ**.


![Confirm Swap](assets/en/18.webp)


4. **การประมวลผล**: การสลับจะดำเนินการแบบอะตอมมิกบน Lightning Network คุณจะเห็นหน้าจอสถานะที่แสดงว่าการสลับกำลังรอดำเนินการ


![Swap Pending](assets/en/19.webp)


5. **ความสำเร็จ**: เมื่อ HTLCs ถูกชำระเรียบร้อยแล้ว การแลกเปลี่ยนก็เสร็จสมบูรณ์ และสินทรัพย์จะอยู่ในช่องของคุณ


![Swap Success](assets/en/20.webp)



## นักพัฒนา API


สำหรับนักพัฒนาที่สร้างบน KaleidoSwap แอปพลิเคชันนี้เปิดเผย API ที่รองรับ:



- RGB LSPS1**: สำหรับบริการสภาพคล่องอัตโนมัติ
- Swap API**: สำหรับการซื้อขายเชิงโปรแกรมและการสร้างตลาด
- WebSocket**: สำหรับการสมัครรับข้อมูลตลาดแบบเรียลไทม์

KaleidoSwap ช่วยให้คุณสามารถใช้ประโยชน์จากความเป็นส่วนตัวและความสามารถในการขยายของสินทรัพย์ RGB บน Lightning Network ได้ โดยการทำความเข้าใจ Colored UTXOs และการจัดสรรสินทรัพย์ในช่องทาง คุณสามารถใช้ประโยชน์จากโปรโตคอลการแลกเปลี่ยนแบบกระจายศูนย์ที่ทรงพลังนี้ได้อย่างเต็มที่
