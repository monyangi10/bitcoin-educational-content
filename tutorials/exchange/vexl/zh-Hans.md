世界各地都出现了使 Bitcoin 的使用更方便、更安全的解决方案。在这一领域，Vexl 因其方法和愿景而脱颖而出：让 Bitcoin 重新回到人们的手中。



Vexl 是一个点对点社交网络，将世界各地的 Bitcoin 买家和卖家联系在一起。它无需身份验证，不会披露您的任何敏感信息，从而提高了您的隐私保护。



## 使用 Vexl 的第一步



开始使用 Vexl 非常简单：访问 [官方网站](https://vexl.it)，然后将移动应用程序下载到您的 iOS 或 Android 手机上。



⚠️ 从官方网站下载移动应用程序可防止平台被欺诈性复制，并保证数据的完整性。



在本教程中，我们将主要在 Android 平台上进行操作，但下面介绍的整个过程也适用于 iOS。



![app](assets/fr/01.webp)



安装应用程序后，请通过验证您的电话号码来创建您的帐户。Vexl 要求您提供尽可能少的信息，以保护您的匿名性并加强您的保密性。



![verify](assets/fr/02.webp)



验证您的账户后，开始查找联系人中的网络成员。导入联系人并开始使用 Vexl。



![contacts](assets/fr/03.webp)



Vexl 以点对点市场的形式运作，为您提供多种选择：





- 用现金购买 Bitcoin。
- 出售 Bitcoin 换取现金。
- 购买带有 Bitcoin 的产品。
- 用 Exchange 出售产品，换取比特币。
- 查看您所在地区的所有 Bitcoin 销售优惠。
- 查看您所在地区的所有 Bitcoin 购买优惠。
- 寻找特别的东西



![options](assets/fr/04.webp)



在 Vexl 主页上，一旦注册，您就会看到有关如何执行特定操作和配置的建议，使您的用户体验更加流畅。



点击 "**聊天**"图标，您就会看到您与联系人的讨论历史以及与您交流过的人。



![suggests](assets/fr/05.webp)



### 下订单



Vexl 为您提供了一个相当可定制的生态系统，用于配置您想要进行的交换。



![initiate](assets/fr/07.webp)



选择所需的订单列表类型，然后指定是多头头寸还是空头头寸。



在本教程中，我们将重点介绍 Bitcoin 交换。



![rate](assets/fr/08.webp)



选择您想交易的货币，然后选择您想买入或卖出的等值金额。您可以从多个网络成员处进行买卖，直到达到您想要的金额。您还可以为 Exchange 设置到期日。



![préférences](assets/fr/09.webp)



根据您的方便程度，您可以在您找到的地理区域内进行 Exchange 实际交易，或者留在 Vexl 上，与联系人联系并在线进行 Exchange。说到 Bitcoin 兑换，Vexl 同时支持 Lightning Network 和主链交易，前者是小额快速支付的理想选择，后者则涉及大量比特币。



![arguments](assets/fr/10.webp)



此外，请说明您希望使用的交流语言，并至少给出一个合理的理由，让其他网络成员认真对待您的提议。



![friends](assets/fr/11.webp)



在你的联系人中寻找可以交换的人，或者更进一步，将交换扩大到朋友的朋友。



![done](assets/fr/12.webp)



检查报价的不同配置，然后验证以保存报价。




![checkup](assets/fr/13.webp)



![confirmed](assets/fr/14.webp)



### 保密性



在应用程序设置中，您可以找到与您的帐户相关联的配置。在 Vexl 上，您的用户名不是静态的，这提高了您的保密性：其他用户无法根据您的用户名将您的交易与您的账户联系起来。



![settings](assets/fr/15.webp)



### 社区



Vexl 可让您加入俱乐部，在那里您可以与更多人进行 Exchange 交流，但与您的朋友和他们的朋友相比，这种方式不太安全。
