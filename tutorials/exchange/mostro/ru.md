Как приобрести или продать биткоины, не ставя под угрозу свой финансовый суверенитет? Централизованные платформы навязывают инвазивные процедуры KYC, раскрывающие ваши личные данные, с возможностью произвольного замораживания счета или государственного наблюдения.



**Mostro P2P** - это децентрализованная альтернатива, сочетающая в себе Lightning Network, протокол Nostr и счета-фактуры. Его главное новшество: автоматизированная система условного депонирования, при которой ваши биткоины остаются под вашим контролем на протяжении всего времени работы биржи, что исключает риск кражи, банкротства биржи или произвольной конфискации.



## Что такое Mostro P2P?



Mostro P2P - это протокол обмена Bitcoin с открытым исходным кодом, созданный **grunch**, разработчиком популярного Telegram-бота **lnp2pbot**, запущенного в 2021 году. Хотя lnp2pbot уже поддерживал обмен P2P без KYC через Lightning, он представлял собой серьезную уязвимость: **Telegram представляет собой централизованную точку отказа**, подверженную цензуре со стороны правительств.



Mostro представляет собой **децентрализованную эволюцию** этой концепции: заменив Telegram на протокол **Nostr** (не поддающуюся измерению сеть распределенных реле), Mostro устраняет этот системный риск. Протокол сочетает в себе Lightning Network для мгновенных транзакций, Nostr для коммуникации, устойчивой к цензуре, и **hold invoices** для создания по-настоящему необязательного автоматизированного депонирования.



### Техническая архитектура



Mostro работает по трем направлениям:




- Демон Mostrod**: координирует обмен данными между пользователями и Lightning Network
- Узел Lightning**: создает счета-фактуры с удержанием (~24 часа криптографического депонирования)
- Клиенты Mostro**: пользовательские интерфейсы (CLI, Mobile, Web)



Ордера публикуются на Nostr как публичные события (тип 38383), а частные сделки передаются с помощью сквозных зашифрованных сообщений (NIP-59, NIP-44). Каждый инстанс Mostro устанавливает свои комиссии (обычно от 0,3 до 1 %) и лимиты на сделки (от нескольких тысяч до нескольких сотен тысяч sats, в зависимости от инстанса).



### Решающие преимущества



**Устойчив к цензуре**: Ни одно правительство не сможет отключить Mostro, пока в мире существуют ретрансляторы Nostr. В отличие от уязвимого lnp2pbot через Telegram, Mostro позволяет обмениваться **цензура-защита**.



**Эскроу без хранения**: Lightning hold invoices блокирует ваши биткоины, не передавая их на постоянное хранение. Ваши средства остаются под вашим контролем и автоматически возвращаются вам в случае возникновения проблем (~24 часа).



**Конфиденциальность по дизайну**: Доступны два режима, чтобы удовлетворить ваши потребности. Режим репутации** связывает вашу репутацию с ключом Nostr для создания прочного доверия. Полностью приватный режим** предпочитает абсолютную анонимность с одноразовыми эфемерными ключами.



## Основные характеристики



**Децентрализованная коммуникация**: Все заказы публикуются на Nostr через криптографически подписанные события. Частные переговоры передаются через сквозные зашифрованные сообщения, что гарантирует абсолютную конфиденциальность.



**Репутационная система**: 5-звездочный рейтинг с итеративным подсчетом постоянно хранится на Nostr. Позволяет постепенно создавать репутацию надежного трейдера.



**Децентрализованный арбитраж**: В случае возникновения спора беспристрастный арбитр изучает доказательства и принимает решение на основе прозрачных критериев. Каждый спор генерирует уникальный token для отслеживания.



**Гибкость платежей**: Поддержка многих фиатных валют с помощью сервиса обменных курсов yadio.io. Принимаются SEPA-переводы, PayPal, Revolut, наличные и любые способы, согласованные между сторонами.



## Клиенты Mostro доступны



Mostro предлагает два основных операционных клиента для ваших бирж P2P.



### Mostro CLI - клиент командной строки



**Mostro CLI** - наиболее зрелый и функциональный клиент. Написанный на языке Rust, он предлагает весь спектр возможностей Mostro через интерфейс командной строки. Совместим с macOS, Linux и Windows.



**Основные элементы управления** :




- `listorders`: Отображение всех доступных заказов
- `neworder` : Создать новый ордер на покупку или продажу
- `takesell` / `takebuy`: Принимать ордер на покупку или продажу
- `fiatsent`: Подтвердить отправку фиатного платежа
- `release`: Разблокировать депонирование и завершить обмен
- `getdm`: Просмотр прямых сообщений
- `rate` : Оцените своего контрагента после обмена



Идеально подходит для технических пользователей, автоматизации и сред, требующих максимальной безопасности.



### Mostro Mobile - приложение для смартфонов



Приложение **mobile app** в альфа-версии позволяет торговать на P2P прямо со смартфона. Интуитивно понятный графический интерфейс Interface с навигацией по вкладкам, просмотром ордеров, расширенными фильтрами и интегрированным чатом для общения с контрагентами.



Доступная для **Android** (через APK на GitHub), она является оптимальным выбором для пользователей, предпочитающих простоту и периодическую мобильную торговлю.



### Mostro-web - Interface web (в разработке)



Interface продвинутое веб-приложение на JavaScript, находящееся в стадии активной разработки. Предназначено для предоставления полного пользовательского опыта с широкими возможностями торговли и управления портфелем. Доступ через браузер без необходимости установки.

### Установка Mostro CLI



#### На macOS



**Шаг 1: проверка Rust**



Убедитесь, что установлен Rust (требуется версия 1.64+):

```bash
rustc --version
```



Если Rust не установлен :

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source $HOME/.cargo/env
```

```bash

```

Скопируйте и отредактируйте файл :



```bash
cp .env-sample .env
```



Откройте файл `.env` и настройте параметры:



```bash
# Public key of the Mostro instance (choose an instance)
# Main mainnet instance (recommended):
MOSTRO_PUBKEY='npub1ykvsmrmw2hk7jgxgy64zr8tfkx4nnjhq9eyfxdlg3caha3ph0skq6jr3z0'

# MOSTRO_PUBKEY='npub19m9laul6k463czdacwx5ta4ap43nlf3lr0p99mqugnz8mdz7wtvskkm5wg'
```

**Важно для синхронизации CLI/Mobile**: Чтобы получить доступ к одним и тем же заказам на CLI и в мобильном приложении, вы должны использовать **одинаковый Mostro Pubkey** и **одинаковые ретрансляторы Nostr** в обоих клиентах. Проверьте эти параметры в Настройках мобильного приложения.

Компиляция занимает 1-2 минуты. Исполняемый файл `mostro-cli` будет установлен в `~/.cargo/bin/`.

Какую бы платформу вы ни использовали:





- Выделенный ключ**: Используйте отдельный идентификатор Nostr для торговли
- Небольшие суммы**: Начните с менее чем sats 10 000 для начала работы
- Несколько реле**: Настройка 3-5 географически различных реле
- Защита сети**: Активируйте VPN или Tor, чтобы скрыть свой IP-адрес

## Используйте с CLI



В этом разделе демонстрируется покупка биткоинов через Mostro CLI в соответствии с реальным случаем использования.



### Шаг 1: Составьте список доступных заказов



Команда `listorders` отображает все активные заказы:



```bash
mostro-cli listorders
```



Вы увидите таблицу с подробной информацией о каждом заказе: ID, тип (покупка/продажа), сумма, валюта, способ оплаты.



![Liste des ordres disponibles](assets/fr/05.webp)



В этом примере виден ордер на продажу 5 EUR через Revolut (ID: `305a59d0-dbee-4880-9b9a-44a60486ba4a`).



### Шаг 2: Принятие заказа



Чтобы купить эти биткоины, примите заказ с помощью `takesell`:



```bash
mostro-cli takesell -o 305a59d0-dbee-4880-9b9a-44a60486ba4a
```



Mostro попросит вас предоставить **Lightning invoice** для получения BTC. В нем будет указана точная сумма в сатоши (здесь: 4715 sats).



![Prise d'ordre de vente](assets/fr/06.webp)



### Шаг 3: Предоставьте счет-фактуру Lightning



Сформируйте счет-фактуру Lightning из wallet (Phoenix, Breez и т. д.) на точную сумму, затем отправьте его :



```bash
mostro-cli takesell -o 305a59d0-dbee-4880-9b9a-44a60486ba4a --invoice lnbc47150n1p...
```



Система подтверждает отправку и говорит вам, что нужно подождать, пока продавец оплатит счет-фактуру, прежде чем активировать условное депонирование.



![Envoi de la Lightning invoice](assets/fr/07.webp)



### Шаг 4: Свяжитесь с продавцом



После активации эскроу используйте `dmtouser` для запроса платежных реквизитов у продавца:



```bash
mostro-cli dmtouser --pubkey 7100aed1b44819555b34f90a6ca8dbb7263526e0c580f5ee35fe20d7b0644ae4 \
--orderid 305a59d0-dbee-4880-9b9a-44a60486ba4a \
--message "Hey what's your revtag ?"
```



![Communication avec le vendeur](assets/fr/08.webp)



### Шаг 5: Получение ответа



Проверьте прямые сообщения, чтобы узнать ответ продавца:



```bash
mostro-cli getdm
```



Продавец сообщает вам свой платежный идентификатор (здесь его Revtag: `@satoshi`).



![Réception de la réponse](assets/fr/09.webp)



### Шаг 6: Осуществление фиатного платежа



Отправьте платеж согласованным способом (в данном примере Revolut) на указанные контактные данные. **Сохраните все подтверждающие документы** (скриншоты, ссылки на транзакции).



### Шаг 7: Подтвердите отправку платежа



После оплаты сообщите об этом в компанию Mostro :



```bash
mostro-cli fiatsent -o 305a59d0-dbee-4880-9b9a-44a60486ba4a
```



### Шаг 8: Получение биткоинов



Как только продавец подтвердит получение фиатных средств и разблокирует условное депонирование командой `release`, вы мгновенно получите свои биткоины на указанный вами Lightning-счет.



### Шаг 9: Оценка



Оцените продавца, чтобы внести свой вклад в децентрализованную репутацию:



```bash
mostro-cli rate -o 305a59d0-dbee-4880-9b9a-44a60486ba4a -r 5
```



### Полезные команды



**Отменить заказ** (до того, как он будет принят) :


```bash
mostro-cli cancel -o <order-id>
```



**Создайте новый заказ на продажу** :


```bash
mostro-cli neworder -k sell -c eur -f 20 -m "Revolut" -p 2
```



**Открыть спор** :


```bash
mostro-cli dispute -o <order-id>
```



## Использование с мобильным приложением



В этом разделе демонстрируется продажа биткоинов через Mostro Mobile на примере реального случая.



### Interface главный



Приложение имеет 3 основные вкладки:





- Книга заказов**: Просмотр доступных ордеров на покупку (BUY BTC) и продажу (SELL BTC)
- Мои сделки**: Просмотр активных и исторических ордеров
- Чат**: Общайтесь с контрагентами с помощью цифр



![Interface principale](assets/fr/14.webp)



### Рекомендуемая конфигурация



Перед началом торговли настройте несколько параметров через меню (☰) > **Настройки** :





- Молния Address** (необязательно): Чтобы получать платежи напрямую
- Релеи**: Добавьте несколько ретрансляторов Nostr для устойчивости (например, `wss://nos.lol`, `wss://relay.damus.io`)
- Mostro Pubkey**: Проверьте открытый ключ используемого экземпляра Mostro



![Paramètres de l'application](assets/fr/16.webp)



**Важно для синхронизации CLI и мобильного приложения**: Если вы используете и CLI, и мобильное приложение, настройте **одно и то же реле Nostr и Mostro Pubkey** в обоих клиентах. Без такой идентичной конфигурации вы не увидите одинаковые заказы, доступные на обоих интерфейсах. Реле и Mostro Pubkey, отображаемые в Настройках (скриншот выше), должны совпадать с теми, что указаны в файле `.env' вашего CLI.



### Шаг 1: Создайте ордер на продажу



Нажмите зеленую кнопку **"+"** внизу справа, затем выберите **ПРОДАТЬ** (красная кнопка).



![Boutons de création d'ordre](assets/fr/17.webp)



Заполните форму создания:



1. **Валюта**: Выберите валюту, в которой вы хотите получать деньги (EUR, USD и т. д.)


2. **Сумма** : Введите сумму в фиатах (например, от 1 до 3 EUR)


3. **Способы оплаты** : Выберите способ (например, "Revolut")


4. **Тип цены**: Выберите "Рыночная цена"


5. **Премия**: Настройте премию (ползунок от -10% до +10%, рекомендуется: 0% для быстрой продажи)



Нажмите **Submit**, чтобы опубликовать заказ.



### Шаг 2: Подтверждение публикации



Ваш заказ опубликован! Он будет доступен в течение 24 часов. Вы можете отменить его в любой момент, пока покупатель не забрал его, выполнив команду `cancel`.



![Ordre publié](assets/fr/18.webp)



Ордер появляется на вкладке **Мои сделки** со статусом "Отложенный" и значком "Создан вами".



### Шаг 3: Покупатель принимает ваш заказ



Когда покупатель принимает ваш заказ, вы получаете уведомление. Подробности см. в разделе **Мои сделки**.



![Ordre pris par un acheteur](assets/fr/19.webp)



**Важное указание**: Теперь вы должны **оплатить выставленный счет на удержание**, чтобы заблокировать ваши биткоины (здесь 4674 sats за 1-5 EUR) в условном депонировании. У вас есть **15 минут максимум**, иначе обмен будет отменен.



Скопируйте счет на оплату и оплатите его с вашего wallet Lightning (Phoenix, Breez и т.д.).



### Шаг 4: Общайтесь с покупателем



После активации эскроу нажмите **CONTACT**, чтобы открыть зашифрованный чат с покупателем.



Покупатель (здесь "anonymous-gloriaZhao") свяжется с вами, чтобы запросить ваши платежные реквизиты. Пожалуйста, ответьте, указав свои реквизиты (Revtag, IBAN и т.д.).



![Chat avec l'acheteur](assets/fr/20.webp)



### Шаг 5: Получение фиатного платежа



Дождитесь получения фиатного платежа на ваш счет в Revolut (или другим согласованным способом). **Внимательно проверьте**:




- Точное количество
- Отправитель
- Ссылка, если требуется



Покупатель сообщит вам в чате, что он отправил платеж. Mostro также отобразит сообщение, подтверждающее, что фиат был отправлен вам.



![Confirmation d'envoi du paiement](assets/fr/20.webp)



### Шаг 6: Разблокировка условного депонирования



Как только вы **подтвердите поступление** фиатного платежа на ваш счет, нажмите зеленую кнопку **RELEASE** и подтвердите отправку сатса покупателю.



![Libération de l'escrow](assets/fr/20.webp)



Биткоины мгновенно переводятся покупателю через его Lightning-счет.



### Шаг 7: Оцените соображения



После освобождения нажмите **RATE**, чтобы оценить покупателя. Выберите от 1 до 5 звезд (здесь 5/5) и нажмите **Отправить оценку**.



![Évaluation de la contrepartie](assets/fr/21.webp)



Обмен окончен!



### Покупайте биткоины с помощью мобильного приложения



Процесс **покупки** биткоинов аналогичен, но обратный:



1. Просмотрите вкладку **Книга заказов** > **BUY BTC**, чтобы просмотреть ордера на продажу


2. Нажмите на интересный заказ, чтобы просмотреть подробности


3. Нажмите **Принять заказ**


4. Предоставьте счет-фактуру Lightning (сгенерированный из вашего wallet)


5. Дождитесь, пока продавец активирует эскроу


6. Свяжитесь с продавцом через **КОНТАКТ** для получения деталей оплаты


7. Отправьте фиатный платеж (сохраните доказательство)


8. Продавец освобождает эскроу после проверки


9. Получайте биткоины мгновенно на свой счет Lightning


10. Оцените продавца (1-5 звезд)



### Управление проблемами



**Отмена ордера**: В разделе **Мои сделки** нажмите на свой ордер, а затем на кнопку **ОТМЕНИТЬ** (доступна только до его взятия).



**Открыть спор**: Если возникли разногласия, нажмите **DISPUTE** в деталях заказа. Приложите все доказательства (скриншоты чата, ссылки на банковские операции).



## Преимущества и ограничения



### Преимущества



**Финансовый суверенитет**: Ваши биткоины никогда не выйдут из-под вашего непосредственного контроля благодаря механизму удержания счетов, что исключает риск банкротства или пиратства биржи.



**Устойчивость к цензуре**: Ни один орган власти не может отключить сеть или подвергнуть цензуре ваши приказы. Система работает до тех пор, пока реле Nostr в рабочем состоянии.



**Анонимность по умолчанию**: Только псевдонимный ключ Nostr идентифицирует вас, без KYC или личных данных. Зашифрованные коммуникации от конца до конца.



**Гибкость оплаты**: Возможен любой взаимоприемлемый способ оплаты (переводы, мобильные приложения, наличные, бартер).



### Ограничения



**Ранняя разработка**: Требуются элементарные интерфейсы и техническое обучение.



**Ограниченная ликвидность**: Ограниченное количество ордеров, особенно на крупные суммы или редкие валюты.



**Технические требования**: Требуется базовое понимание Lightning и Nostr.



**Индивидуальная ответственность**: Отсутствие централизованной поддержки в случае возникновения проблем, требуется осторожность.



## Заключение



Mostro P2P представляет собой многообещающую альтернативу централизованным биржам, объединяя Lightning Network, Nostr и автоматизированное депонирование для действительно децентрализованной торговли. Несмотря на раннюю разработку и ограниченную ликвидность, платформа уже предлагает финансовый суверенитет, устойчивость к цензуре и анонимность по умолчанию.



Для биткоинщиков, предпочитающих автономность и конфиденциальность, Mostro - это жизнеспособный вариант, достойный постепенного изучения. Его децентрализованная архитектура гарантирует развитие сообщества, а не коммерции, прокладывая путь к будущему действительно свободных бирж Bitcoin.
