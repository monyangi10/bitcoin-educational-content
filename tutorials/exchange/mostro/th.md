**Mostro P2P** นำเสนอทางเลือกแบบกระจายศูนย์ที่รวม Lightning Network, โปรโตคอล Nostr และการถือใบแจ้งหนี้ นวัตกรรมหลักของมัน: ระบบเอสโครว์อัตโนมัติที่บิตคอยน์ของคุณยังคงอยู่ภายใต้การควบคุมของคุณตลอดการแลกเปลี่ยน ขจัดความเสี่ยงจากการโจรกรรม การล้มละลายของการแลกเปลี่ยน หรือการยึดทรัพย์สินโดยพลการ



## Mostro P2P คืออะไร?



Mostro P2P เป็นโปรโตคอลแลกเปลี่ยน Bitcoin แบบโอเพ่นซอร์สที่สร้างขึ้นโดย **grunch** ผู้พัฒนาบอท Telegram ยอดนิยม **lnp2pbot** ที่เปิดตัวในปี 2021 แม้ว่า lnp2pbot จะช่วยให้สามารถแลกเปลี่ยน P2P ได้โดยไม่ต้องใช้ KYC ผ่าน Lightning แต่ก็มีช่องโหว่สำคัญ: **Telegram เป็นจุดศูนย์กลางที่อาจล้มเหลว** ซึ่งเสี่ยงต่อการถูกเซ็นเซอร์โดยรัฐบาล



Mostro แสดงถึง **วิวัฒนาการแบบกระจายศูนย์** ของแนวคิดนี้: โดยการแทนที่ Telegram ด้วยโปรโตคอล **Nostr** (เครือข่ายรีเลย์แบบกระจายที่ไม่สามารถเซ็นเซอร์ได้) Mostro กำจัดความเสี่ยงเชิงระบบนี้ โปรโตคอลนี้รวม Lightning Network สำหรับการทำธุรกรรมทันที, Nostr สำหรับการสื่อสารที่ต้านทานการเซ็นเซอร์, และ **ใบแจ้งหนี้ถือครอง** เพื่อสร้างเอสโครว์อัตโนมัติที่ไม่ต้องมีผู้ดูแลอย่างแท้จริง



### สถาปัตยกรรมทางเทคนิค



Mostro ทำงานผ่านสามองค์ประกอบ:




- Daemon Mostrod**: ประสานการแลกเปลี่ยนระหว่างผู้ใช้และ Lightning Network
- Lightning** node: สร้างใบแจ้งหนี้แบบถือ (~24h การฝากทรัพย์สินด้วยการเข้ารหัส)
- ลูกค้า Mostro**: ส่วนติดต่อผู้ใช้ (CLI, มือถือ, เว็บ)



คำสั่งซื้อถูกเผยแพร่บน Nostr ในรูปแบบเหตุการณ์สาธารณะ (ประเภท 38383) ในขณะที่การซื้อขายส่วนตัวถูกส่งผ่านข้อความเข้ารหัสแบบ end-to-end (NIP-59, NIP-44) แต่ละอินสแตนซ์ของ Mostro กำหนดค่าธรรมเนียมของตัวเอง (โดยทั่วไปอยู่ระหว่าง 0.3% ถึง 1%) และขีดจำกัดการทำธุรกรรม (ตั้งแต่ไม่กี่พันถึงหลายแสน sats ขึ้นอยู่กับอินสแตนซ์)



### ข้อได้เปรียบที่เด็ดขาด



**ทนทานต่อการเซ็นเซอร์**: ไม่มีรัฐบาลใดสามารถปิด Mostro ได้ตราบใดที่ยังมี Nostr รีเลย์อยู่ที่ไหนสักแห่งในโลก แตกต่างจาก lnp2pbot ที่เปราะบางผ่าน Telegram, Mostro อนุญาตให้มีการแลกเปลี่ยนที่ **ป้องกันการเซ็นเซอร์**.



**Escrow non-custodial**: ใบแจ้งหนี้ Lightning hold ล็อคบิตคอยน์ของคุณโดยไม่โอนย้ายอย่างถาวร เงินของคุณยังคงอยู่ภายใต้การควบคุมของคุณและจะถูกส่งคืนให้คุณโดยอัตโนมัติในกรณีที่เกิดปัญหา (~24h)



**การรักษาความลับโดยการออกแบบ**: มีสองโหมดให้เลือกตามความต้องการของคุณ โหมดชื่อเสียง** เชื่อมโยงชื่อเสียงของคุณกับกุญแจ Nostr เพื่อสร้างความไว้วางใจที่ยั่งยืน โหมดส่วนตัวเต็มรูปแบบ** เน้นการไม่เปิดเผยตัวตนอย่างสมบูรณ์ด้วยกุญแจชั่วคราวที่ใช้ครั้งเดียว



## คุณสมบัติหลัก



**การสื่อสารแบบกระจายศูนย์**: คำสั่งทั้งหมดจะถูกเผยแพร่บน Nostr ผ่านเหตุการณ์ที่ลงนามด้วยการเข้ารหัสลับ การเจรจาส่วนตัวจะถูกส่งผ่านข้อความที่เข้ารหัสจากต้นทางถึงปลายทาง เพื่อรับประกันความลับอย่างสมบูรณ์



**ระบบชื่อเสียง**: การให้คะแนน 5 ดาวพร้อมการคำนวณแบบวนซ้ำที่จัดเก็บถาวรบน Nostr ช่วยให้คุณสร้างชื่อเสียงในฐานะผู้ค้าที่ยอดเยี่ยมได้อย่างต่อเนื่อง



**การอนุญาโตตุลาการแบบกระจายศูนย์**: ในกรณีที่เกิดข้อพิพาท ผู้ตัดสินที่เป็นกลางจะตรวจสอบหลักฐานและตัดสินใจตามเกณฑ์ที่โปร่งใส แต่ละข้อพิพาทจะสร้าง token ที่ไม่ซ้ำกันเพื่อการติดตามได้



**ความยืดหยุ่นในการชำระเงิน**: รองรับสกุลเงินเฟียตหลายสกุลผ่านบริการอัตราแลกเปลี่ยนของ yadio.io ยอมรับการโอนเงินผ่าน SEPA, PayPal, Revolut, เงินสด และวิธีการใด ๆ ที่ตกลงกันระหว่างคู่สัญญา



## ลูกค้า Mostro ที่มีอยู่



Mostro มีลูกค้าปฏิบัติการหลักสองรายสำหรับการแลกเปลี่ยน P2P ของคุณ



### Mostro CLI - ไคลเอนต์บรรทัดคำสั่ง



**Mostro CLI** เป็นไคลเอนต์ที่มีความสมบูรณ์และใช้งานได้ดีที่สุด เขียนด้วย Rust มันมีฟีเจอร์ครบถ้วนของ Mostro ผ่านทางอินเทอร์เฟซบรรทัดคำสั่ง รองรับ macOS, Linux และ Windows



**Main controls** :




- `listorders`: แสดงคำสั่งซื้อทั้งหมดที่มีอยู่
- `neworder` : สร้างคำสั่งซื้อหรือขายใหม่
- `takesell` / `takebuy`: รับคำสั่งซื้อหรือขาย
- `fiatsent`: ยืนยันการส่งการชำระเงินด้วยเงินตรา
- `release`: ปล่อยเงินเอสโครว์และสรุปการแลกเปลี่ยน
- `getdm`: ดูข้อความโดยตรง
- `rate` : ประเมินคู่สัญญาของคุณหลังจากการแลกเปลี่ยน



เหมาะสำหรับผู้ใช้ทางเทคนิค, การทำงานอัตโนมัติ และสภาพแวดล้อมที่ต้องการความปลอดภัยสูงสุด



### Mostro Mobile - แอปพลิเคชันสมาร์ทโฟน



แอปพลิเคชัน **มือถือ** ในเวอร์ชันอัลฟ่าช่วยให้การซื้อขาย P2P โดยตรงจากสมาร์ทโฟนของคุณ กราฟิก Interface ที่ใช้งานง่ายพร้อมการนำทางแบบแท็บ การดูคำสั่งซื้อ ฟิลเตอร์ขั้นสูง และแชทในตัวเพื่อสื่อสารกับคู่ค้าของคุณ



พร้อมใช้งานสำหรับ **Android** (ผ่าน APK บน GitHub) เป็นตัวเลือกที่เหมาะสมที่สุดสำหรับผู้ใช้ที่ชื่นชอบความเรียบง่ายและการซื้อขายผ่านมือถือเป็นครั้งคราว



### Mostro-web - Interface web (อยู่ระหว่างการพัฒนา)



Interface แอปพลิเคชันเว็บ JavaScript ขั้นสูงที่กำลังพัฒนาอย่างต่อเนื่อง ออกแบบมาเพื่อมอบประสบการณ์ผู้ใช้ที่สมบูรณ์พร้อมด้วยฟังก์ชันการซื้อขายและการจัดการ wallet ที่ครอบคลุม เข้าถึงผ่านเบราว์เซอร์โดยไม่ต้องติดตั้ง

#### บน macOS



**ขั้นตอนที่ 1: ตรวจสอบ Rust**



ตรวจสอบว่า Rust ได้รับการติดตั้งแล้ว (ต้องใช้เวอร์ชัน 1.64+):



```bash
rustc --version
```



หากไม่ได้ติดตั้ง Rust :



```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source $HOME/.cargo/env
```


#### บน Linux (Ubuntu/Debian)



การติดตั้งเหมือนกับ macOS โดยมีการเพิ่ม:



```bash
sudo apt update
sudo apt install -y cmake build-essential pkg-config
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source $HOME/.cargo/env
git clone https://github.com/MostroP2P/mostro-cli.git
cd mostro-cli
cargo build --release
```



จากนั้นทำตามขั้นตอนที่ 3 และ 4 ในส่วนของ macOS



#### บน Windows



ขั้นแรกติดตั้ง Rust ผ่านทาง [rustup.rs](https://rustup.rs/), จากนั้นใช้ PowerShell :



```powershell
git clone https://github.com/MostroP2P/mostro-cli.git
cd mostro-cli
cargo build --release
```



การกำหนดค่าที่เหมือนกัน: คัดลอก `.env-sample` ไปยัง `.env` และกรอกพารามิเตอร์ของคุณลงไป

### การกำหนดค่าที่แนะนำ

- Lightning Address** (ตัวเลือก): เพื่อรับการชำระเงินโดยตรง
- รีเลย์**: เพิ่มรีเลย์ Nostr หลายตัวเพื่อความยืดหยุ่น (เช่น `wss://nos.lol`, `wss://relay.damus.io`)

### ขั้นตอนที่ 1: สร้างคำสั่งขาย



กดปุ่ม **"+"** สีเขียวที่มุมล่างขวา จากนั้นเลือก **SELL** (ปุ่มสีแดง)



![Boutons de création d'ordre](assets/fr/17.webp)



กรอกแบบฟอร์มการสร้าง :



1. **สกุลเงิน**: เลือกสกุลเงินที่คุณต้องการรับ (EUR, USD, ฯลฯ)


2. **Amount** : ป้อนจำนวนเงินในสกุลเงินเฟียต (เช่น 1 ถึง 3 EUR)


3. **วิธีการชำระเงิน** : เลือกวิธีการ (เช่น "Revolut")


4. **ประเภทของราคา**: เลือก "ราคาตลาด"


5. **พรีเมียม**: ปรับพรีเมียม (แถบเลื่อนจาก -10% ถึง +10%, แนะนำ: 0% เพื่อขายอย่างรวดเร็ว)



กด **Submit** เพื่อเผยแพร่คำสั่งซื้อของคุณ



### ขั้นตอนที่ 2: ยืนยันการเผยแพร่



คำสั่งซื้อของคุณได้รับการเผยแพร่แล้ว! จะสามารถใช้งานได้เป็นเวลา 24 ชั่วโมง คุณสามารถยกเลิกได้ตลอดเวลาก่อนที่ผู้ซื้อจะรับคำสั่งซื้อโดยการใช้คำสั่ง `cancel`



![Ordre publié](assets/fr/18.webp)



คำสั่งซื้อปรากฏในแท็บ **การซื้อขายของฉัน** พร้อมสถานะ "รอดำเนินการ" และตราสัญลักษณ์ "สร้างโดยคุณ"



### ขั้นตอนที่ 3: ผู้ซื้อรับคำสั่งซื้อของคุณ



เมื่อผู้ซื้อรับคำสั่งซื้อของคุณ คุณจะได้รับการแจ้งเตือน ดูรายละเอียดใน **การซื้อขายของฉัน**



![Ordre pris par un acheteur](assets/fr/19.webp)



**คำแนะนำที่สำคัญ**: คุณต้อง **ชำระใบแจ้งหนี้ที่ถืออยู่** ที่แสดงเพื่อล็อคบิทคอยน์ของคุณ (ที่นี่ 4674 sats สำหรับ 1-5 EUR) ในเอสโครว์ คุณมีเวลา **สูงสุด 15 นาที** มิฉะนั้นการแลกเปลี่ยนจะถูกยกเลิก



คัดลอกใบแจ้งหนี้ที่ถือไว้และชำระจาก wallet Lightning ของคุณ (Phoenix, Breez, ฯลฯ)



### ขั้นตอนที่ 4: สื่อสารกับผู้ซื้อ



เมื่อเอสโครว์ถูกเปิดใช้งานแล้ว ให้กด **CONTACT** เพื่อเปิดแชทที่เข้ารหัสกับผู้ซื้อ



ผู้ซื้อ (ที่นี่ "anonymous-gloriaZhao") จะติดต่อคุณเพื่อขอรายละเอียดการชำระเงินของคุณ กรุณาตอบกลับพร้อมรายละเอียดของคุณ (Revtag, IBAN, ฯลฯ)



![Chat avec l'acheteur](assets/fr/20.webp)



### ขั้นตอนที่ 5: การรับชำระเงินด้วยเงินตรา



รอจนกว่าคุณจะได้รับการชำระเงินด้วยเงินเฟียตในบัญชี Revolut ของคุณ (หรือวิธีการอื่นที่ตกลงกันไว้) **ตรวจสอบอย่างรอบคอบ**:




- จำนวนที่แน่นอน
- ผู้ส่ง
- การอ้างอิงหากมีการร้องขอ



ผู้ซื้อจะแจ้งให้คุณทราบผ่านทางแชทว่าเขาได้ส่งการชำระเงินแล้ว Mostro จะแสดงข้อความยืนยันด้วยว่าเงินเฟียตได้ถูกส่งถึงคุณแล้ว



![Confirmation d'envoi du paiement](assets/fr/20.webp)



### ขั้นตอนที่ 6: ปล่อยเอสโครว์



เมื่อคุณได้ **ยืนยันการรับ** การชำระเงินด้วยเงินเฟียตในบัญชีของคุณแล้ว ให้กดปุ่มสีเขียว **RELEASE** และยืนยันเพื่อส่ง satss ให้กับผู้ซื้อ



![Libération de l'escrow](assets/fr/20.webp)



Bitcoins ถูกโอนให้กับผู้ซื้อทันทีผ่านใบแจ้งหนี้ Lightning ของเขา



### ขั้นตอนที่ 7: ประเมินการพิจารณา



หลังจากปล่อยแล้ว กด **RATE** เพื่อให้คะแนนผู้ซื้อ เลือกจาก 1 ถึง 5 ดาว (ที่นี่ 5/5) และกด **Submit Rating**.



![Évaluation de la contrepartie](assets/fr/21.webp)



การแลกเปลี่ยนสิ้นสุดลงแล้ว!



### ซื้อบิตคอยน์ด้วยแอปมือถือ



กระบวนการในการ **ซื้อ** บิตคอยน์มีความคล้ายคลึงกันแต่กลับกัน:



1. เรียกดู **Order Book** > แท็บ **BUY BTC** เพื่อดูคำสั่งขาย


2. คลิกที่คำสั่งซื้อที่น่าสนใจเพื่อดูรายละเอียด


3. กด **Take Order**


4. โปรดระบุใบแจ้งหนี้ Lightning ของคุณ (ที่สร้างจาก wallet ของคุณ)


5. รอให้ผู้ขายเปิดใช้งานเอสโครว์


6. ติดต่อผู้ขายผ่านทาง **CONTACT** เพื่อขอรายละเอียดการชำระเงิน


7. ส่งการชำระเงินด้วยเงินสด (เก็บหลักฐาน)


8. ผู้ขายปล่อยเงินเอสโครว์หลังการตรวจสอบ


9. รับบิตคอยน์ทันทีบนใบแจ้งหนี้ Lightning ของคุณ


10. ให้คะแนนผู้ขาย (1-5 ดาว)



### การจัดการปัญหา

**ความยืดหยุ่นในการชำระเงิน**: สามารถใช้วิธีการชำระเงินใด ๆ ที่ตกลงร่วมกันได้ (โอนเงิน, แอปพลิเคชันมือถือ, เงินสด, แลกเปลี่ยนสินค้า)



### ข้อจำกัด



**การพัฒนาในช่วงแรก**: อินเทอร์เฟซขั้นพื้นฐานและความจำเป็นในการเรียนรู้ทางเทคนิคที่ซับซ้อน



**สภาพคล่องจำกัด**: จำนวนคำสั่งซื้อขายที่จำกัด โดยเฉพาะสำหรับจำนวนมากหรือสกุลเงินที่หายาก



**ข้อกำหนดทางเทคนิค**: ต้องมีความเข้าใจพื้นฐานเกี่ยวกับ Lightning และ Nostr



**ความรับผิดชอบส่วนบุคคล**: ไม่มีการสนับสนุนจากศูนย์กลางในกรณีที่เกิดปัญหา ต้องใช้ความระมัดระวัง



## บทสรุป



Mostro P2P แสดงถึงทางเลือกที่มีศักยภาพต่อการแลกเปลี่ยนแบบรวมศูนย์ โดยการรวม Lightning Network, Nostr และการค้ำประกันอัตโนมัติสำหรับการซื้อขายที่กระจายอำนาจอย่างแท้จริง แม้จะอยู่ในช่วงพัฒนาเบื้องต้นและมีสภาพคล่องจำกัด แพลตฟอร์มนี้ก็ยังคงเสนออธิปไตยทางการเงิน การต้านทานการเซ็นเซอร์ และความเป็นนิรนามโดยปริยาย



สำหรับผู้ที่ชื่นชอบบิทคอยน์ที่ให้ความสำคัญกับความเป็นอิสระและความลับ Mostro เป็นตัวเลือกที่มีศักยภาพที่ควรค่าแก่การสำรวจอย่างต่อเนื่อง สถาปัตยกรรมแบบกระจายศูนย์ของมันรับประกันการพัฒนาของชุมชนมากกว่าการพาณิชย์ เปิดทางสู่อนาคตของการแลกเปลี่ยน Bitcoin ที่เป็นอิสระอย่างแท้จริง
