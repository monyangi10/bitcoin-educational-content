## 开始使用 BitSpenda



BitSpenda 是一个基于网络的平台，您可以在此平台上进行 Exchange Bitcoin 和移动货币交易，同时还能为您保密。事实上，BitSpenda 是一个无需账户、无需个人数据、无需身份验证的兑换商：只有您和您的交易。



![hero](assets/fr/01.webp)



使用 BitSpenda 进行交易相当直观，只需几个步骤。在 BitSpenda [官方网站](https://bitspenda.app) 上，点击 "开始 "按钮，就会跳转到 Interface Exchange。



BitSpenda 目前覆盖三个国家：




- 加纳，Bitcoin 兑换移动货币号码。
- 尼日利亚，通过银行转账。
- 肯尼亚通过 M-pesa 移动支付。



![countries](assets/fr/02.webp)



有了这些选项，您就可以更方便地通过 Lightning Network 使用 Bitcoin 向这些国家汇款，并在不使用当地货币的情况下进行日常购物。



### 尼日利亚的银行



非洲的银行间交易会产生足够多的摩擦，而且通常需要相当长的处理时间。通过 BitSpenda，您可以在尼日利亚免费即时进行 Exchange、向任何地方汇款、银行转账。BitSpenda 覆盖尼日利亚 50 多家银行和新银行。



在 BitSpenda 应用程序的第一页，选择尼日利亚作为目的地国家。

然后检查您填写的信息，确认您的 Exchange。



![confirm](assets/fr/09.webp)

### 肯尼亚的 M-Pesa



BitSpenda 基于两项金融和技术创新：Bitcoin 和移动货币。前者使用户能够在世界任何地方通过 Lightning Network Layer 即时进行匿名交易，而无需中央机构的中介，从而重新获得金融主权。至于第二种，它解决了非洲社会银行渗透率极低的问题。



BitSpenda 通过肯尼亚流行的移动支付服务 M-Pesa 为肯尼亚提供服务。


得益于 BitSpenda 等举措，如今您可以使用 Bitcoin 在商店付款、在餐馆用餐、支付房租和账单。



选择肯尼亚作为目的地国家，然后填写收件人的 M-pesa 号码和收件人应收到的金额。

## 在非洲普及 Bitcoin



通过 BitSpenda，[Bitcoin Dua](https://www.bitcoindua.org/) 旨在为非洲采用 Bitcoin 建立一个理想的框架。BitSpenda 脱颖而出的主要原因是.....：




- 保密性**：您无需在平台上创建账户，无需验证身份，也无需提供任何个人信息。
- 自由**：您可以在任何国家使用 BitSpenda 向加纳、尼日利亚和肯尼亚进行交易。
