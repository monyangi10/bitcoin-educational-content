BitSpenda 是一個貨幣 Exchange 平台，可讓您在 Bitcoin 和 Mobile Money 之間進行轉換，無需任何驗證、無需任何個人資訊，也無需任何費用。它的出現是為了解決一個在加納以及許多非洲國家常見的特殊問題：跨國支付。使用 BitSpenda，您可以進行快速、匿名的交換，並可在日常業務中使用。BitSpenda 的主要目標是讓 Bitcoin 在非洲社區的全球採用能夠順利交易。



## 開始使用 BitSpenda



BitSpenda 是一個基於 Web 的平台，您可以在此平台上進行 Exchange Bitcoin 和移動貨幣交易，同時為您保密。事實上，BitSpenda 是一個不需要帳戶、個人資料和身份驗證的交換器：只有您和您的交易。



![hero](assets/fr/01.webp)



使用 BitSpenda 進行交易相當直觀，只需要幾個步驟。在 BitSpenda [官方網站](https://bitspenda.app)，點擊「開始」按鈕，您會被重定向到 Interface Exchange。



BitSpenda 目前覆蓋三個國家：




- 加納與 Bitcoin 交換至行動貨幣號碼。
- 奈及利亞透過銀行轉帳。
- 肯亞透過 M-pesa 行動貨幣。



![countries](assets/fr/02.webp)



有了這些選項，您可以更輕鬆地透過 Lightning Network 使用 Bitcoin 向這些國家匯款，並在不使用當地貨幣的情況下進行日常購物。



### 奈及利亞的銀行



非洲的跨行交易造成了足夠的摩擦，通常有相當長的處理時間。透過 BitSpenda，您可以在尼日利亞免費、即時地進行 Exchange、隨處寄錢、銀行轉帳。BitSpenda 涵蓋尼日利亞 50 多家銀行和新銀行。



在 BitSpenda 應用程式的第一頁，選擇尼日利亞為您的目的地國家。



![nigeria](assets/fr/07.webp)



選擇收件人的銀行並輸入其帳號。每個 Exchange 可傳送 1,000 到 2,000,000 NGN（尼日爾奈拉）。



![banks](assets/fr/08.webp)



然後檢查您填寫的資訊，確認您的 Exchange。



![confirm](assets/fr/09.webp)



交易確認後，請支付相關的 Lightning Invoice。您的付款一經完成，銀行轉帳即自動完成。

### 肯亞的 M-Pesa



BitSpenda 基於兩項金融和技術創新：Bitcoin 和移動貨幣。第一種創新讓使用者可以透過 Lightning Network Layer 即時在世界任何地方進行假名交易，無需中央機構的中介，重新獲得金融主權。至於第二種，它解決了非洲社會銀行滲透率非常低的問題。



BitSpenda 透過肯亞流行的行動貨幣服務 M-Pesa 提供服務。


感謝 BitSpenda 等計畫，今天您可以使用 Bitcoin 在商店付款、在餐廳用餐、支付房租和帳單。



選擇肯亞為目的地國家，然後填寫收件人的 M-pesa 號碼和收件人應收到的金額。



![mpesa](assets/fr/11.webp)



繼續確認，檢查您輸入的資訊。



![confirmation](assets/fr/12.webp)



支付「閃電」Invoice 以驗證您的 Exchange，並完成 Exchange。



![payment](assets/fr/13.webp)

## 在非洲普及 Bitcoin



透過 BitSpenda，[Bitcoin Dua](https://www.bitcoindua.org/) 旨在為 Bitcoin 在非洲的應用建立一個理想的框架。BitSpenda 的突出之處主要在於 ：




- **保密性**：您不需要在平台上建立帳戶，不需要驗證身份，也不需要提供任何個人資訊。
- **自由**：您可以在任何國家使用 BitSpenda 向加納、奈及利亞和肯亞進行交易。
- **速度**：依靠閃電支付，BitSpenda 可以縮短處理時間，使交易瞬間完成。
- **透明度**：您在 BitSpenda 上進行的所有交易都與獨特的識別碼相關聯，使您可以追蹤和追蹤這些交易的進度。您也可以使用這些識別碼聯絡 [X 上的應用程式支援](https://x.com/bitspenda)。



現在您可以在 BitSpenda 服務的國家每天使用 Bitcoin。
