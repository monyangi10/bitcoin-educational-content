- (b)** Swaps instantanés en ligne
- (e)** GAB Bitcoin

## Comment acheter des BTC pour des VND


![BitcoinVN.io](assets/en/02.webp)


### Étape 1 : Sélectionner la devise à envoyer


Dans la section "Vous envoyez" - le grand carré avec l'icône VND rouge, choisissez votre méthode de paiement préférée pour payer les BTC dans le menu déroulant. Dans ce cas, le VND Bank Transfer (paiement en VND via les banques locales) est sélectionné. Nous supportons **67 banques locales** au Vietnam.


### Étape 2 : Choisir la monnaie de réception


Dans la section "Vous recevez" - le grand carré avec le logo Bitcoin, sélectionnez BTC (Bitcoin blockchain/mainchain) - c'est généralement la valeur par défaut ou facile à trouver dans la liste déroulante.


### Étape 3 : Saisir le montant en VND et vérifier les détails de l'opération


Saisissez manuellement le montant que vous souhaitez échanger et utilisez le [lien vers les limites quotidiennes](https://bitcoinvn.io/info) ou les boutons Min/Max pour vérifier votre capacité si vous n'avez pas de compte. Le système calcule instantanément le montant final de BTC que vous recevrez après avoir pris en compte les [frais d'échange](https://support.bitcoinvn.io/help/en-us/4-orders/12-fees) et le taux de change à ce moment précis. Vérifiez le montant final en BTC pour voir si vous êtes satisfait de l'accord.


#### Conseils


N'oubliez pas que **les taux de transaction fluctuent**, vérifiez donc toujours le prix final avant de confirmer.


**Un compte n'est pas nécessaire avec** des transactions quotidiennes limitées à 250 000 VND (~ 8 USD). De plus amples informations sont disponibles dans nos [politiques de transfert bancaire](https://support.bitcoinvn.io/help/en-us/5-deposit-settle-methods/17-vnd-bank-transfer).


**Attention** : les frais de réseau sont généralement moins élevés lorsque vous choisissez la méthode Lightning Network par rapport à la méthode de transaction onchain.


### Étape 4 : Saisissez votre Bitcoin Address


Copiez et collez l'adresse wallet de réception des BTC, ou scannez le code QR dans le champ d'adresse (surligné en rouge dans la capture d'écran). Vérifiez-la deux fois, car c'est là que vos BTC seront envoyés.


### Étape 5 : Procéder au paiement de votre choix


Cliquez sur le gros bouton jaune Swap pour accéder à une page de paiement avec les coordonnées bancaires de BitcoinVN accompagnées d'un code QR (pour les applications financières locales comme Momo et VNPAY), et le montant exact en VND. Transférez le montant exact depuis votre compte bancaire/application vietnamien(ne) dans le délai imparti, sinon l'opération expirera et vous devrez recommencer à partir de l'étape 3.


### Étape 6 : Vérifier votre paiement et recevoir l'actif


Après avoir effectué le paiement, téléchargez une capture d'écran/réception du reçu de paiement pour vérification (généralement 1 à 10 minutes les jours ouvrables). Une fois la vérification confirmée, les BTC sont envoyés instantanément à votre wallet. Vous recevrez des mises à jour par courriel et pourrez suivre le numéro de commande.


## Comment vendre des BTC pour des VND


![BitcoinVN.io](assets/en/03.webp)


### Etape 1, 2 et 3 : Sélectionner Monnaie d'envoi, Monnaie de réception, saisir les montants des swaps


Comme pour le processus d'achat ci-dessus, cliquez sur la section "Vous envoyez" pour choisir l'actif que vous souhaitez échanger via la blockchain, par exemple BTC (bitcoin on-chain), ou BTC via le réseau Lightning (une solution off-chain) dans la liste déroulante.


Ensuite, dans la section "Vous recevez", sélectionnez l'actif que vous voulez recevoir et la manière dont vous voulez le recevoir ; par exemple, par un transfert bancaire en VND (recevoir par l'intermédiaire de votre banque).


Saisissez le montant de BTC que vous souhaitez échanger. L'application calcule instantanément le montant en VND que vous recevrez. Vérifiez les frais de réseau, les montants finaux reçus et la note de règlement (peut être retardée en dehors des heures d'ouverture).


### Étape 4 : Vérifier et entrer les détails de votre paiement


Sélectionnez l'une des 67 banques locales prises en charge dans la liste déroulante, puis indiquez le nom complet du titulaire du compte, le numéro de compte et ajoutez une note si vous le souhaitez.


### Étape 5 : Envoyer l'actif et recevoir votre paiement


Cliquez sur le bouton Swap pour afficher l'adresse de dépôt BTC de BitcoinVN et un code QR. Envoyez exactement le montant de BTC spécifié de votre wallet à l'adresse indiquée en scannant le code QR ou en copiant l'adresse Bitcoin.


Prévoyez un délai de confirmation (secondes sur Lightning, minutes sur on-chain). Les VND seront transférés sur votre compte bancaire, généralement dans un délai de 1 à 2 heures pendant les heures d'ouverture. Un courriel et un numéro de commande vous seront envoyés pour le suivi.
