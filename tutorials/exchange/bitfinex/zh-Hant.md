Bitfinex 成立於 2012 年，是最早的 Bitcoin 和 Altcoin Exchange 平台之一。該平台最初專注於 P2P Bitcoin 交換，後來迅速擴展其服務，包括保證金交易、P2P 融資、衍生品交易，以及用於大量交易的 OTC（「*場外交易*」）市場。


今天，Bitfinex 已經是一個完整的平台，既可以簡單地購買比特幣，也可以使用先進的交易功能與風險管理工具。它有網頁版，對於簡單的交易，也有易用的移動應用程式。


Bitfinex 積極支持 Bitcoin 的發展，投資於 Lightning Network 等技術，並推廣點對點解決方案，實現財務自由。

## 2 - 帳戶驗證


Bitfinex 採用符合當前法規的驗證流程 (KYC)。在 "Basic 「模式下，無法進行存款、取款或交易，因此必須至少獲得 」Basic Plus 「驗證等級，必要時甚至可以獲得 」Intermediate 「或 」Full "驗證等級：




- Basic* 是僅顯示模式，可讓您查詢平台而不執行任何操作；
- Basic Plus* 提供基本功能，包括現貨市場交易和特定存取款選項；
- 中級*鎖定進階服務，如保證金交易、貸款、P2P 和 OTC 交易，並縮短存款期限；
- Full* 可讓您使用國際銀行轉帳及平台的所有功能。


每個等級都需要額外的文件進行驗證。對大多數使用者而言，*Basic Plus* 或 *Intermediate* 帳戶一般已足夠。


建立帳戶後，會彈出視窗提示您驗證帳戶。按一下「*驗證*」。


![BITFINEX](assets/fr/07.webp)


如果此視窗未出現，請前往 Interface 右上方的個人資料，然後按一下「*驗證*」。


![BITFINEX](assets/fr/08.webp)


在「*帳戶類型*」下，選擇「*個人*」。在我的情況下，我要升級到 "*Basic Plus*「，所以我選擇 」*Upgrade to Basic Plus*"。


![BITFINEX](assets/fr/09.webp)


通過提供所要求的信息和文件來完成步驟。一旦您的申請完成並提交，Bitfinex 將驗證您的帳戶以解鎖相關功能。


## 3 - Bitfinex Interface 快速介紹


登入後，您會在 Interface 的上方看到一個導覽列，上面有「*交易*」、「*衍生品*」、「*資金*」、"*OTC*"、"*P2P*"、"*Wallet*"等。Bitfinex 提供廣泛的功能，包括：




- 交易**：您可以下單買賣加密貨幣（包括 Bitcoin）的「*基本*」市場 ；
- OTC**：場外交易服務 (Over-The-Counter service)：在公開買賣盤之外，直接與其他參與者進行大量交易；
- 資金**：專用於貸款和保證金融資的區域；
- 衍生產品**：衍生性金融商品（期貨等）部分，適合有經驗的交易員；
- P2P**：允許您以點對點方式向其他使用者購買或出售加密貨幣。


對於標準使用（買/賣比特幣和存/取款），您主要使用「*交易*」標籤，以及「*Wallet*」、「*存款*」和「*取款*」部分。


![BITFINEX](assets/fr/11.webp)

## 4 - 存取歐元（法幣）


![bitfinex](https://youtu.be/z2YlJr9sF20)


若要將歐元存入您的 Bitfinex 帳戶，請存取 Interface 最上方「*Wallet*」功能表中的「*存款*」子功能表。


![BITFINEX](assets/fr/12.webp)


選擇「*銀行轉帳*」或「*信用卡/借記卡*」以歐元（或任何其他法定貨幣）存款。


![BITFINEX](assets/fr/13.webp)


選擇要傳送的法定貨幣，例如歐元。如果您只使用基本的「*交易*」功能，請點選「*Exchange*」。此外，請註明您希望存入的金額以及您的銀行所在國家。


![BITFINEX](assets/fr/14.webp)


從您的銀行帳戶轉帳到 Bitfinex 指定的銀行帳戶。


若要提款，程序與此類似：前往「*提款*」子功能表。


![BITFINEX](assets/fr/15.webp)


按一下「*銀行轉帳*」。


![BITFINEX](assets/fr/16.webp)


選擇您要提取的法定貨幣、要在 Bitfinex 扣款的帳戶（如果您只使用基本功能，則選擇 "*Exchange*"）以及要提取的金額。


![BITFINEX](assets/fr/17.webp)


基於合規原因，Bitfinex 在接受轉帳前可能會要求驗證您的銀行帳戶。


![BITFINEX](assets/fr/18.webp)


程序啟動後，Bitfinex 會將資金轉入您的銀行帳戶。


## 5 - 存入和提取比特幣


若要在 Bitfinex 上存入 Bitcoin，請存取「*存款*」子功能表。


![BITFINEX](assets/fr/19.webp)


按一下「*加密貨幣*」。


![BITFINEX](assets/fr/13.webp)


選擇「*BTC*」。將會出現一個接收 Address。複製此 Address，並使用它從您的自我保管 Wallet 或其他平台發送您的 BTC。


![BITFINEX](assets/fr/20.webp)


若要撤回 Bitcoin，請前往「*撤回*」子功能表。


![BITFINEX](assets/fr/21.webp)


按一下「*加密貨幣*」。


![BITFINEX](assets/fr/22.webp)


選擇 "*BTC*"。選擇提款時要扣款的 Bitfinex 帳戶（基本功能為 "*Exchange*"）。輸入比特幣的金額和目的地Address。請務必檢查提款Address以防止任何錯誤。


![BITFINEX](assets/fr/23.webp)


在您確認之後，您的 bitcoins 將被轉移。請注意，費用和延遲可能因 Mempool 擁擠程度而異。


Bitfinex 也透過 Lightning Network 提供存款和提款選項，讓交易更快速、更便宜。


![BITFINEX](assets/fr/24.webp)


如果您對 Lightning Network 感興趣，我們也有完整的訓練課程，幫助您瞭解其運作方式：


https://planb.network/courses/34bd43ef-6683-4a5c-b239-7cb1e40a4aeb

在購買您的第一個比特幣之後，您顯然可以選擇把它們留在平台上，但我強烈建議您把它們轉移到您自己的Wallet中自我保管。如果您還不知道如何使用 Bitcoin Wallet，我邀請您探索 [Plan ₿ Network 上的 "Wallet" 部分](https://planb.network/tutorials/Wallet)。


## 6 - 在 Bitfinex 上買賣比特幣


Bitfinex 提供多種交易模式。為了方便使用，請選擇經典的現貨市場，也稱為 「*交易*」或 "*Exchange*"。在這裡，您可以按市價下買入或賣出訂單，或設定限價。


在頂部功能表中，按一下「*交易*」。


![BITFINEX](assets/fr/25.webp)


例如，如果您想以 Exchange 買入或賣出 BTC 以換取歐元，請選擇「*BTC/EUR*」交易對。


![BITFINEX](assets/fr/26.webp)


Interface 中部顯示價格圖，底部顯示訂單簿，左側顯示訂單輸入模組。在訂單輸入部分，您可以選擇 "*Market*「訂單（以最佳可用價格立即執行）或 」*Limit*"訂單（由您定義價格）。指出要買入或賣出的 BTC 數量，或選擇您餘額的百分比。然後點擊 "*Buy*「買入或 」*Sell*"賣出。


![BITFINEX](assets/fr/27.webp)


您可以在 Interface 的下半部分查看已執行訂單的歷史記錄。


![BITFINEX](assets/fr/28.webp)


您也可以設定從應用程式定期購買比特幣，讓您的取得價格隨時間平滑調整 (*DCA*)。


![bitfinex](https://youtu.be/8uoBacYSn08)
