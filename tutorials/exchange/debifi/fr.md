## Introduction


Dans ce tutoriel vidéo étape par étape, vous verrez comment utiliser Debifi pour débloquer la valeur de votre BTC par le biais d'un prêt garanti par Bitcoin. Vous apprendrez à créer un compte, à le sécuriser avec 2FA, à parcourir les offres et même à utiliser une Coldcard Q dans une configuration Multisig pour s'assurer que votre Bitcoin ne peut pas être prêté à nouveau.


Qu'il s'agisse de financer une opportunité commerciale, de faire face à une situation d'urgence ou simplement de disposer de liquidités à court terme sans déclencher d'événement imposable, ce guide vous donne les outils nécessaires pour bien faire les choses.
