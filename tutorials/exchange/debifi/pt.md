## Introdução


Neste tutorial em vídeo passo a passo, você verá como usar o Debifi para desbloquear o valor do seu BTC por meio de um empréstimo apoiado pelo Bitcoin. Você aprenderá como criar uma conta, protegê-la com 2FA, navegar pelas ofertas e até mesmo usar um Coldcard Q em uma configuração Multisig para garantir que seu Bitcoin não possa ser reemprestado.


Quer esteja a financiar uma oportunidade de negócio, a lidar com uma emergência, ou apenas pretenda liquidez a curto prazo sem desencadear um evento tributável - este guia dá-lhe as ferramentas para o fazer corretamente.
