## Introducción


En este videotutorial paso a paso, verás cómo usar Debifi para desbloquear el valor de tu BTC a través de un préstamo respaldado por Bitcoin. Aprenderás cómo crear una cuenta, asegurarla con 2FA, navegar por las ofertas e incluso utilizar una Coldcard Q en una configuración Multisig para asegurarte de que tu Bitcoin no puede ser re-prestado.


Tanto si desea financiar una oportunidad de negocio, hacer frente a una emergencia o simplemente obtener liquidez a corto plazo sin provocar un hecho imponible, esta guía le ofrece las herramientas necesarias para hacerlo correctamente.
