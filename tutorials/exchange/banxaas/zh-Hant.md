Lightning Network 的出現標誌著 Bitcoin 主流的開始，它提供了比 Bitcoin 通訊協定的主要網路 (Mainnet) 更快、更便宜的可能性。


如今，在不發達國家和/或新興國家，取得 Bitcoin 是一項創新挑戰，以彌補當地金融體系的某種失敗。


在本教程中，我們將發現 **Banxaas**，一個讓塞內加爾人更接近 Bitcoin 交換的易物平台。


## 開始使用 Banxaas



Banxaas 來自塞內加爾方言 (Wolof)，意思是分支。在意識形態上，我們可以將 Banxaas 視為連結塞內加爾人使用 Bitcoin 的分支。一個您可以信賴的分支，保護您的儲蓄不受通貨膨脹和審查的影響。Banxaas](https://banxaas.com) 由一家塞內加爾新創公司（Yité Technologies）開發，借助 Lightning Network 的力量，在 Bitcoin 和非洲金融共同体法郎 (XOF) 之間提供即時 Exchange 服務，反之亦然。Banxaas 在西非次區域的 Exchange 平台上採用了不同尋常的方法。




- Banxaas 是一個非監護平台：您不需要在平台上創建帳戶或擁有 Wallet。您仍然可以控制您的資金，而且您的匿名性得到了加強。

- 透明會計：Banxaas 倡導透明度，同時強化金融交易的匿名性。

### 首次與 Banxaas 進行交易



在 [banxaas](https://banxaas.com) 上要做的第一件事就是定義 Exchange 的方向：





- 您擁有 CFA Franc 並希望將 Exchange 轉換成 Bitcoin。



![sens1](assets/fr/01.webp)





- 您有比特幣，想兌換成非洲法郎



![sens2](assets/fr/02.webp)



Banxaas 網頁平台簡約直觀，讓您在一分鐘內完成交易。在定義了您的Exchange的方向後，在表格的第一個文字欄位中填入您想要發送的金額，或者填入您想要接收的金額。





- 從 CFA 法郎到 Satoshis** ：



Banxaas 的服務範圍覆蓋整個塞內加爾領土，擁有兩家主要的行動運營商。在其支付集成商 (DexchangePay) 的協助下，該平台主要支援 .NET 和 .NET 2.0：




- 橘子錢
- 波浪
- Wizall 錢



當您啟動從 CFA 法郎到 Bitcoin 的轉換時，您會在目的地欄位中添加您的 Lightning Address 或從 Lightning Wallet 中創建的不含金額的 Invoice。



https://planb.network/tutorials/wallet/mobile/blitz-wallet-794bdac4-1af4-49d5-9ea5-abb8228ca196

https://planb.network/tutorials/wallet/mobile/speed-wallet-8715e454-1720-4a7f-8c1d-3da02cf67312

Banxaas 可讓您按一下 ** 驗證目的地** 按鈕，檢查 Bitcoin 接收 Address 的準確性。



![payxof](assets/fr/03.webp)



點擊 **Troc** 按鈕確認您的交易，從您的 CFA Franc 付款中購買比特幣。



![confirm1](assets/fr/04.webp)





- 從 Satoshis 到 CFA 法郎**



Banxaas 允許您將比特幣兌換成非洲法郎，並在塞內加爾的兩家運營商上接收。





- Orange Money 塞內加爾
- 波浪塞內加爾



按一下 **Troc** 按鈕，支付將會產生的 Lightning Invoice，以便在您的 Mobile Money 上收到等值金額。



![pay2](assets/fr/05.webp)



Banxaas 不只是一個兌換平台，更是一項可讓您使用比特幣作為 Exchange 方式的服務。透過該平台，您可以使用薩托希支付餐費、購物和支付設備費用，方法是將**薩托希換成非洲法郎**，並提供賣家的 Money 手機號碼。


### 流程核心的透明度



透明度是 Banxaas 的重要支柱，以盡量減少審查，並讓解決方案的使用者能夠判斷其可靠性。


在交易選單中，你可以查看最近的交易，而無需存取使用者資訊：Banxaas只要求你提供電話號碼或Lightning Address。每筆交易都是唯一的，並由UUID（通用唯一識別碼）識別。



![transactions](assets/fr/06.webp)



![transactions](assets/fr/07.webp)
