Перевод средств между Bitcoin on-chain и Lightning Network обычно требует либо ручного открытия каналов Lightning (что технически сложно и дорого), либо использования централизованных своп-платформ с KYC. SwapMarket предлагает альтернативу: бездоверительные атомарные свопы через конкурентных провайдеров без KYC.



Инновации: несмотря на то, что провайдеры являются посредниками, HTLC (*Hash Time Locked Contracts*) математически гарантирует, что ваши средства остаются под вашим контролем. Объединение нескольких провайдеров (Boltz, ZEUS Swaps, Eldamar, Middle Way) создает ценовую конкуренцию. Interface web open-source self-hostable.



## Что такое SwapMarket?



Агрегатор с открытым исходным кодом, запущенный в 2024 году, SwapMarket функционирует как компаратор поставщиков свопов Bitcoin/Lightning. Пользователь мгновенно сравнивает условия (комиссии, ликвидность, лимиты) и выбирает оптимального поставщика.



### Техническая архитектура



**Фронтенд клиентской части**: 100% клиентское приложение (fork Boltz Web App), размещенное на GitHub Pages. Код запускается в браузере без бэкенд-сервера. История хранится локально (куки/кэш). Публичный и проверяемый исходный код.



**Открытие провайдера** : Жестко закодированный список в `src/configs/mainnet.ts`. Новые провайдеры добавляются через Pull Request или по электронной почте.



**Независимые бэкенды**: Каждый провайдер управляет своим собственным бэкендом Boltz. Интерфейс запрашивает API в режиме реального времени, чтобы мгновенно сравнивать котировки.



**HTLC Атомарные свопы**: Контракты Hash с временной блокировкой гарантируют атомарность: либо своп исполняется, либо каждая сторона возвращает свои средства. Риск контрагента математически исключен.



### Философия



SwapMarket снижает централизацию, создавая конкуренцию между провайдерами за комиссии и ликвидность. Отсутствие KYC, самодостаточный код с открытым исходным кодом, множество независимых операторов, чтобы избежать единых точек отказа.



## Основные характеристики



### Рынок поставщиков



В интерфейсе отображаются все активные провайдеры: название провайдера, применяемые комиссии (процентные и/или фиксированные), минимальные/максимальные доступные суммы и поддерживаемые типы свопов. Приложение напрямую запрашивает API каждого провайдера, указанного в файле конфигурации, чтобы получить котировки в режиме реального времени. Конкуренция между провайдерами гарантирует оптимальные ставки, как правило, около 0,5 % для стандартных свопов.



### Двунаправленные свопы



**Обмен (on-chain → Lightning)**: Конвертируйте BTC on-chain в сатоши Lightning. Вариант использования: питание мобильного wallet Lightning, получение входящей мощности на узле или мгновенная ликвидность.



**Обмен (Lightning → on-chain)**: Конвертирует сатоши Lightning в BTC on-chain. Сфера применения: опустошение wallet Lightning в холодное хранилище или восстановление баланса ликвидности между слоями.



### Безопасность и восстановление



**Trustless атомный обмен**: HTLC гарантируют, что либо обмен будет завершен полностью, либо каждая сторона вернет свою ставку. Риск контрагента математически исключен.



**Механизм расплаты**: Каждый своп имеет временную блокировку. Если своп не удался, средства автоматически возвращаются по истечении срока действия. У пользователя всегда остается возможность вернуть свои биткоины.



** Ключи восстановления**: SwapMarket позволяет экспортировать ключи восстановления для текущих свопов. В случае возникновения проблем эти ключи можно использовать для завершения или отмены свопа с любого устройства.



## Установка и доступ



### Interface web



SwapMarket не требует установки. Доступ осуществляется через браузер по адресу https://swapmarket.github.io. Для максимальной конфиденциальности используйте Brave, Firefox с расширениями против слежки или LibreWolf. Для обеспечения сетевой анонимности рекомендуется использовать браузер Tor.



Регистрация, электронная почта или проверка личности не требуются.



### Самостоятельный хостинг (опционально)



Для технических пользователей, желающих избавиться от зависимости от официального домена GitHub Pages, SwapMarket можно запустить локально:



**Via npm** :


```
git clone https://github.com/SwapMarket/swapmarket.github.io.git
cd swapmarket.github.io
npm install
npm run dev
```



**Via Docker** :


```
docker run -p 3000:80 ghcr.io/swapmarket/swapmarket:latest
```



Приложение будет доступно по адресу `http://localhost:3000`. Самостоятельный хостинг гарантирует полный контроль над интерфейсом, исключает риск цензуры официального домена и позволяет проверять исходный код перед исполнением.



### Начальная конфигурация



**Wallet Lightning**: Убедитесь, что у вас есть действующий wallet Lightning (Phoenix, Zeus, BlueWallet и т.д.). При подмене вы получите счет generate Lightning. При обмене вы оплачиваете счет Lightning.



**Wallet on-chain**: Для своп-ин вам понадобится wallet Bitcoin on-chain для отправки средств. Для своп-аутов подготовьте адрес получения Bitcoin.



**Опциональная конфигурация**: SwapMarket сохраняет историю свопов и предпочтения в cookies браузера. Создание учетной записи не требуется.

## Создание свопа шаг за шагом



### Замена: Молния → Bitcoin



В этом первом примере показано, как конвертировать сатоши Lightning в биткоины on-chain.



**Шаг 1: Подмена конфигурации



На главной странице выберите форму обмена:




- LIGHTNING** (верхнее поле): Введите сумму, которую вы хотите отправить в sats Lightning (пример: 30 000 sats)
- BITCOIN** (нижнее поле): Сумма, которую вы получите, автоматически отображается после вычета комиссии (пример: 29 320 sats)



В нижнем поле вставьте свой **адрес получателя Bitcoin**, на который вы хотите получить средства. Внимательно проверьте этот адрес.



Поставщиком по умолчанию обычно является Boltz Exchange. Сетевые тарифы и тарифы провайдера четко отображаются.



![Configuration swap-out](assets/fr/03.webp)



**Шаг 2: Выбор поставщика**



Нажмите на выпадающее меню поставщика (по умолчанию: "Boltz Exchange"), чтобы отобразить всех доступных поставщиков ликвидности.



Откроется модальное окно, в котором будет показана сравнительная таблица:




- Статус**: Индикатор Green, если провайдер активен
- Псевдоним**: Имя поставщика (Boltz Exchange, Middle Way, Eldamar, ZEUS Swaps)
- Плата**: Плата, взимаемая поставщиком услуг (обычно от 0,49% до 0,5%)
- Максимальный своп**: Максимальная сумма, принимаемая для свопа



Сравните тарифы и максимальные суммы, а затем выберите подходящего поставщика.



**Обратите внимание**: В интерфейсе выбора поставщика не отображаются **минимальные суммы** для каждого поставщика. Эта информация появляется только в интерфейсе создания свопа, после выбора поставщика. Минимальные и максимальные суммы могут отличаться у разных провайдеров и могут меняться со временем. **Всегда проверяйте эти лимиты во время свопа**: если сумма, которую вы хотите обменять, выходит за рамки лимитов провайдера, вы можете выбрать другого, более подходящего для вашей операции.



![Sélection du provider](assets/fr/04.webp)



**Шаг 3: Создание свопа и молниеносная** оплата



Нажмите на желтую кнопку **"CREATE ATOMIC SWAP "**. SwapMarket выставит generate счет **Lightning** (BOLT11) для оплаты с вашего wallet Lightning.



На странице отображается :




- Идентификатор свопа**: Уникальный идентификатор свопа (пример: J4ymFIMVR6Hm)
- Статус**: "swap.created" (своп создан, ожидает оплаты)
- QR-код**: Отсканируйте его с помощью вашего wallet Lightning
- Invoice Lightning**: Строка символов, начинающаяся с "lnbc" (пример: lnbc300u1p50whiv...gn5dk2szgqkvfkzc)



Оплатите этот счет с вашего wallet Lightning (Phoenix, Zeus, BlueWallet и т.д.). Отображается точная сумма к оплате (пример: 30 000 sats).



![Paiement Lightning](assets/fr/05.webp)



**Шаг 4: Подтверждение и принятие**



После подтверждения платежа Lightning SwapMarket мгновенно получает ваш платеж, а провайдер транслирует транзакцию Bitcoin на ваш адрес.



Статус меняется на **"invoice.settled "** (счет оплачен), и отображается подтверждающее сообщение.



Ваши биткоины on-chain будут доступны сразу же после подтверждения транзакции (обычно это занимает от нескольких минут до нескольких часов, в зависимости от комиссии mining, выбранной провайдером).



![Confirmation swap-out](assets/fr/06.webp)



Вы можете нажать на **"OPEN CLAIM TRANSACTION "**, чтобы просмотреть транзакцию Bitcoin в браузере блокчейна.



### Подмена: Bitcoin → Молния



Второй пример показывает, как конвертировать биткоины on-chain в сатоши Lightning.



**Шаг 1: Подмена конфигурации



На главной странице выберите форму обмена:




- BITCOIN** (верхнее поле): Введите сумму, которую вы хотите отправить в sats Bitcoin (пример: 63 400 sats)
- LIGHTNING** (нижнее поле): Сумма, которую вы получите, автоматически отображается после вычета комиссионных (пример: 62 884 sats)



В нижнее поле вставьте счет-фактуру Lightning** (BOLT11), созданную в wallet Lightning, или используйте адрес LNURL, если wallet его поддерживает.



![Configuration swap-in](assets/fr/07.webp)



**Шаг 2: Проверка ключа спасения**



После нажатия на **"CREATE ATOMIC SWAP "** появится модальное окно, в котором вам будет предложено подтвердить ваш ключ спасения.



![Modal Rescue Key](assets/fr/08.webp)



**Boltz Rescue Key**: Поскольку вы уже загрузили ключ восстановления во время первоначальной настройки (см. предыдущий раздел), нажмите на кнопку **"VERIFY EXISTING KEY "**, чтобы импортировать сохраненный ключ.



Выберите ранее загруженный файл Rescue Key. После успешной проверки интерфейс автоматически переключится на следующий шаг.



**Шаг 3: Bitcoin** адрес вклада



SwapMarket теперь генерирует **уникальный адрес Bitcoin**, содержащий контракт HTLC, связанный с вашим счетом-фактурой Lightning.



На странице отображается :




- Идентификатор подмены**: Уникальный идентификатор (пример: 1kGmB6JyGqU4)
- Статус**: "invoice.set" (счет выставлен, ожидает оплаты Bitcoin)
- QR-код**: Адрес склада Bitcoin
- Адрес Bitcoin**: Обычно начинается с "bc1p..." (пример: bc1p5mvtwxapjkds...9d4n9f)
- Предупреждение желтого цвета** : "Убедитесь, что ваша сделка подтверждена в течение ~24 часов после создания этого свопа!"



Этот период в ~24 часа является **тайм-аутом** контракта HTLC. Если ваша транзакция Bitcoin не будет подтверждена в течение этого периода времени, своп будет провален, и вам придется использовать ваш Rescue Key, чтобы вернуть свои средства.



![Adresse de dépôt Bitcoin](assets/fr/09.webp)



Вы можете скопировать адрес, нажав на кнопку **"АДРЕС "**, или отсканировать QR-код прямо с вашего wallet on-chain.



**Шаг 4: Отправка биткоинов**



С вашего wallet Bitcoin on-chain, отправьте **точно** указанную сумму (например, 63 400 sats) на сгенерированный адрес.



**Важно**: Используйте соответствующие комиссии mining, чтобы гарантировать быстрое подтверждение. Если комиссия слишком мала и транзакция остается в mempool дольше таймаута (~24 часа), своп будет неудачным.



После отправки транзакции SwapMarket обнаруживает, что она находится в mempool, и отображает :




- Статус** : "transaction.mempool
- Сообщение**: "Транзакция находится в mempool - ожидание подтверждения для завершения свопа"



![Transaction en mempool](assets/fr/10.webp)



**Шаг 5: Подтверждение и молниеносный** прием



Как только транзакция Bitcoin получает первое подтверждение, провайдер автоматически оплачивает ваш счет Lightning. Вы мгновенно получаете сатоши на свой wallet Lightning.



Статус меняется на **"transaction.claim.pending "**, затем отображается подтверждающее сообщение:



![Confirmation swap-in](assets/fr/11.webp)



Ваши сатоши Молнии сразу же становятся доступны в вашем wallet.



## Преимущества и ограничения



### Преимущества



**Конкуренция тарифов**: Объединение провайдеров создает естественную конкуренцию, снижая тарифы (от 0,49% до 0,5%).



**Конфиденциальность**: Отсутствие KYC, 100% клиентский интерфейс (без передачи персональных данных), совместимость с Tor Browser.



**Без опеки**: HTLC математически гарантирует исключительный контроль над вашими средствами. Либо обмен пройдет успешно, либо вы получите свои биткоины обратно.



**Open-source self-hostable**: проверяемый публичный код, развертываемый локально для максимальной устойчивости к цензуре.



### Ограничения



**Ограниченная ликвидность**: Ограниченное количество активных провайдеров (Boltz, Eldamar, MiddleWay в зависимости от периода). Максимальные суммы могут быть ограничены.



**Время истечения**: Время истечения от 24 до 48 часов. Если транзакция on-chain не подтверждена до истечения срока действия, требуется ручное восстановление.



**Централизация Interface**: Несмотря на возможность самостоятельного хостинга, официальный интерфейс размещен на GitHub Pages. Если GitHub подвергнет репо цензуре, доступ через swapmarket.github.io будет заблокирован (решение: самостоятельный хостинг).



**Следы on-chain**: Скрипты HTLC потенциально могут быть идентифицированы с помощью продвинутого анализа блокчейна.



## Лучшие практики



### Безопасная конфигурация



**Загрузите ключ спасения**: Перед первым обменом загрузите ключ спасения из Настроек (см. специальный раздел выше). Этот уникальный ключ будет работать для всех ваших будущих обменов, позволяя вам восстановить средства в случае возникновения проблем.



**Используйте браузер Tor**: Для обеспечения максимальной конфиденциальности заходите на SwapMarket через браузер Tor, чтобы скрыть свой IP-адрес.



**Рассмотрите возможность самостоятельного хостинга**: Для технических пользователей запуск собственного экземпляра SwapMarket устраняет зависимость от официального домена GitHub Pages.



### Оптимизация подкачки



**Следите за пулом памяти**: Проверяйте mempool.space перед заменой. Выбирайте время низкой активности, чтобы минимизировать затраты на mining.



**Проверьте адреса**: При обмене тщательно проверяйте адрес получателя. Используйте копирование и вставку и проверяйте первые 5 и последние 5 символов.



**Протестируйте небольшие суммы**: Начните с минимально допустимого (25 000 - 50 000 sats). Постепенно увеличивайте, когда освоите процесс.



**Документируйте свои свопы**: Записывайте идентификатор каждого свопа, адрес выкупа и дату истечения срока действия. Эта информация облегчает отслеживание и восстановление в случае технических проблем.



### Стратегия использования



**Сбалансируйте свой денежный поток**: Используйте SwapMarket, чтобы регулировать распределение между on-chain (сбережения, долгосрочная безопасность) и Lightning (ежедневные расходы, мгновенные платежи) в соответствии с вашими реальными потребностями.



**Рассчитайте рентабельность**: Для постоянных потребностей в ликвидности Lightning сравните совокупную стоимость повторных свопов с открытием канала Lightning напрямую. SwapMarket отлично подходит для разовых корректировок, но не обязательно для больших регулярных потоков.



## SwapMarket против Boltz: В чем разница?



### Больц: Технология против сервиса



**Boltz - это технология с открытым исходным кодом** (`boltz-backend` на GitHub), которая реализует атомарный обмен через HTLC между Bitcoin, Lightning и Liquid.



**Критический момент**: Все провайдеры SwapMarket (Boltz Exchange, ZEUS Swaps, Eldamar, Middle Way) развертывают свой собственный экземпляр бэкенда Boltz. Таким образом, базовая технология идентична. Уязвимость в бэкенде Boltz потенциально может затронуть всех провайдеров, однако открытый исходный код системы позволяет проводить аудит сообществом.



**Boltz Exchange** - это единый сервис, управляемый командой Boltz, а **SwapMarket** объединяет несколько провайдеров, использующих технологию Boltz, создавая конкурентную ценовую среду.



Более подробную информацию вы найдете в наших руководствах по обмену Boltz и Zeus:

**Преимущества SwapMarket**: Ценовая конкуренция, диверсификация бэкэнд-инстансов, сравнение в реальном времени.



**Технологические альтернативы** (не совместимые с SwapMarket): Lightning Loop (Lightning Labs), Muun Wallet, NLoop, Breez Wallet. Эти решения используют собственные реализации подводных свопов.



**Рекомендация**: Используйте Boltz Exchange для простоты или SwapMarket для оптимизации затрат за счет конкуренции. Оба варианта эквивалентны по безопасности (HTLC не имеет ограничений).



## Заключение



SwapMarket облегчает проведение обменов Bitcoin/Lightning, объединяя нескольких провайдеров в единый интерфейс. Архитектура HTLC гарантирует некастодиальный характер свопов, отсутствие KYC сохраняет конфиденциальность, а самодостаточный код с открытым исходным кодом усиливает устойчивость к цензуре.



Конкуренция между провайдерами повышает ставки и увеличивает источники ликвидности. Для оптимизации двухуровневого управления (экономия on-chain, расходы на Lightning) SwapMarket является практичным инструментом, сохраняющим финансовый суверенитет и конфиденциальность.
