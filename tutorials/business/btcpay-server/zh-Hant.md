BTCPay Server 是一款由 Nicolas Dorier 創建的免費開源軟體套件，可讓比特幣付款在無中介的情況下被接受。其設計目的是提供自主權和金融主權，它安裝在自己的伺服器上，並提供完整的交易管理，從開票到 on-chain 或 Lightning Network 付款的驗證，以及會計。它可輕鬆與電子商務網站 (WooComerce、Shopify 等) 整合，也可用作實體商店的付款終端機 (*POS*)。



對於希望接受比特幣的商家來說，BTCPay Server 無疑是最先進的解決方案。在安全性、主權性和保密性方面，它是最全面、最穩健的軟體。另一方面，它也是安裝和維護最複雜的。也有更簡單的替代方案：有些是完全托管的，如 OpenNode，而有些則在易用性和主權之間提供了有趣的折衷，如 Swiss Bitcoin Pay ：

## BTCPay 伺服器功能



集中式 Bitcoin 銷售點解決方案，例如 *OpenNode*，提供易用性，但依賴於第三方公司，因為它們無法自行託管，而且在大多數情況下是專有的。儘管這些解決方案能讓您更輕鬆地設定付款，但卻會收取佣金，而且在資金保管和機密性方面，使用者會比 BTCPay Server 等解決方案面臨更大的風險。



BTCPay Server 主要針對希望接受比特幣捐款的線上或實體商家、協會和非營利組織。對於尋求社區直接支持的專案擁有者和開發者而言，它也是一個理想的解決方案。



BTCPay 伺服器的特殊功能包括 ：




- 其**完全自主**、
- 缺乏**KYC**程序、
- 完全控制資金**、
- **取消平台費用**。



成為您自己的付款處理器後，您就可以消除您與客戶之間對中央第三方的任何依賴。您可以直接接受比特幣付款和 generate 付款發票。這可以確保您和您的公司都不會被其他人禁止。您同時扮演銀行和支付處理商的角色，而無需為每筆交易向中介支付佣金。



**on-chain**的交易費用維持不變，但可透過使用**Liquid**或**Lightning**網路來降低費用。



此外：




- 完全客製化的介面和發票；
- 原生 **Tor** 支援可增強機密性；
- 支援**集資**、**POS**和**付款按鈕**；
- 相容於多種貨幣 ；
- Bitcoin 直接和點對點付款 ；
- 完全控制您的私密金鑰；
- 增強隱私 ；
- 加強安全性 ；
- 自託管軟體 ；
- 支援 **SegWit** 和 **Lightning 網路** ；
- 內部、以節點為基礎的組合，並整合硬體組合。



## 安裝和設定 BTCPay 伺服器



### 選擇您的寄存模式



BTCPay 伺服器可以用多種不同的方式安裝。根據您的需求和資源，主要有三種選擇：




- 由第三方託管的BTCPay伺服器**：您使用一個為您託管服務的平台。這很簡單，但通常不是免費的。
- BTCPay 伺服器自行託管於雲端伺服器** (例如透過 [btcpayprovider](https://btcpayprovider.com/)、[Bitcoin People](http://bitcoinpeople.it/) 或任何其他供應商)。這是推薦給大多數新手商家的解決方案。
- BTCPay 伺服器安裝在您自己的硬體（本地）**：電腦、迷你電腦或 Umbrel。此方法技術性較高，但提供完全的獨立性。

## 安全與最佳實務



BTCPay伺服器賦予的自主權（對您的資金擁有完全的主權）是一個真正的優勢。但隨著這種自由，在安全方面的責任也更大。通過管理您自己的支付，您就扮演了您自己的銀行的角色。這就是為什麼必須採用最佳實務來保護您的資金、資料和基礎設施。以下是需要考慮的要點。



### 安全存取您的伺服器





- 使用強大的密碼：結合大小楷字母、數字和特殊字符。避免重複使用現有的密碼。
- 啟動雙重認證 (2FA) 以存取您的 BTCPay 介面。
- 定期更新您的作業系統、您的 BTCPay 伺服器實例和您的依賴項目 (Docker、Bitcoin 節點、Lightning 節點...)。更新通常會修正安全漏洞。



### 管理和備份私人密碼匙





- 離線保存您的私密金鑰和種子密碼在實體媒體 (紙張或金屬) 上。
- 最好使用 wallet 硬體 wallet。
- 在不同的受保護實體位置保存多份備份。



### 安全支付和保密





- 使用 Tor 網路或 VPN 隱藏伺服器的 IP 位址，保護您的隱私。
- 停用伺服器上不必要的連接埠，並限制 SSH 連線只能使用受信任的位址。
- 啟動 HTTPS (SSL 證書) 用於所有連接到您的 BTCPay 網頁介面。
- 切勿與未受過 Bitcoin 組合管理訓練的人員共用您的管理介面。



### 實施 Lightning 網路的最佳作法



您的商店連接到 Voltage 上託管的**Lightning 節點**，這大大簡化了網路的技術管理。儘管如此，採用**良好的監控和安全措施**仍然非常重要：





- 儲存您的 Voltage 節點的 API** 登入和存取金鑰 (可讓 BTCPay 連線)。
- 使用雙重認證和強大的密碼保護您的 Voltage 帳戶**。
