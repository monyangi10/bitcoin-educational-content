Voltz 平台的创意来自于一群比特币爱好者，他们希望提供自己的比特币 wallet 服务。其结果是建立了一个完整的基础设施，用于在零售店接受比特币。在本教程中，我们将带您参观 Voltz 以及该平台提供的比特币整合可能性。



## 开始使用 Voltz



Voltz 是一个完整的平台，提供众多扩展功能，可将比特币作为销售点或市场整合到您的业务中。



访问 [Voltz] 平台 (https://www.lnvoltz.xyz/en) 并点击 "立即尝试 "按钮创建账户。



![voltz](assets/fr/01.webp)



![login](assets/fr/02.webp)



⚠️ 重要的是设置一个强大的字母数字密码，以增加抵御暴力破解攻击的机会，并检查您是否确实在 Voltz 官方网站上填写登录信息，以防范网络钓鱼。



Voltz 的界面与 LnBits 平台非常相似。



https://planb.academy/tutorials/business/others/lnbits-cdfe1e38-069a-4df9-a86b-ce01ef28f4c2

事实上，Voltz 是在 LnBits 服务器上构建的；请查看我们的教程，了解如何安装自己的 LnBits 服务器并在其上构建解决方案。



https://planb.academy/tutorials/business/others/lnbits-server-6a406046-3cef-4a64-a82b-8d8f0f82a192

通过该平台，您可以有效地管理多个投资组合。默认情况下，当您注册时，您会自动拥有一个 "闪电 "投资组合。不过，您也可以添加其他投资组合。



![wallets](assets/fr/03.webp)



### 便携性



Voltz 不局限于网络界面：在**移动接入**部分，您可以将激活的 Voltz wallet 连接到 Zeus 或 Blue Wallet 等应用程序。



https://planb.academy/tutorials/wallet/mobile/zeus-embedded-c67fa8bb-9ff5-430d-beee-80919cac96b9

https://planb.academy/tutorials/wallet/mobile/blue-wallet-2f4093da-6d03-4f26-8378-b9351d0dbc90

为此，您需要在平台上安装并激活**LndHub**扩展。



![lndhub](assets/fr/04.webp)



选择您正在使用的 Voltz 产品组合，然后根据您希望授予的权限，扫描相应的 QR 码。




- 发票二维码只允许您阅读投资组合历史记录和 generate 新发票。
- 通过管理二维码，您可以查看历史记录、generate 发票以及支付闪电发票。



![qrs](assets/fr/05.webp)



在本教程中，我们将把 Voltz wallet 连接到 Blue Wallet 应用程序。



![connect](assets/fr/06.webp)



一旦我们的产品组合连接起来，所有执行的操作都将在 Blue Wallet 和 Voltz 之间同步。例如，当您在 Blue Wallet 上开具 generate 发票时，您在 Voltz 上也会有历史记录。



![sync](assets/fr/07.webp)



在**投资组合配置**部分，您可以找到最基本的参数，如自定义投资组合名称和接收付款的货币。



![config](assets/fr/08.webp)



### 扩展



Voltz 平台的特点之一是模块化，您可以激活所需的扩展功能。扩展功能列表可在**扩展功能**菜单中找到。



![extensions](assets/fr/09.webp)



在这些扩展中，TPoS 是一种销售点终端，您可以用它来保存库存和向客户收款。



![tpos](assets/fr/10.webp)



通过销售终端，您可以 ：




- 获得可与客户和合作伙伴共享的网页；
- 管理产品库存；
- 生成闪电发票；
- 通过 Bolt 卡支付现金。



该扩展有免费版和可提供更多高级功能的付费版。要创建 POS 终端，请点击扩展徽标下方的**打开**按钮，然后点击**新建 POS**。



![new_tpos](assets/fr/11.webp)



定义您的销售点名称，然后将 Voltz wallet 与您的终端连接，即可收款。您可以通过勾选**激活小费**框来激活小费。如果您希望向客户开具收据，还可以激活发票打印选项（该功能仍在开发中，因此可能会出现故障）。



销售点终端还包括税务配置，可直接对产品征收本国税款。



![tpos](assets/fr/12.webp)



一旦创建了 POS 终端，您就可以添加预配置的产品和服务，以确保您和您的客户获得流畅的支付和会计体验。



![product](assets/fr/13.webp)



通过定义产品名称、添加图片和设定售价来创建产品。  您可以对产品进行分类，以便于跟踪。您可以直接对特定产品征税。如果产品没有应用税项，则会自动应用在销售点终端创建阶段配置的税项。



![products](assets/fr/14.webp)



点击**导入/导出**按钮，即可从 JSON 格式自动导入产品列表。



![exports](assets/fr/15.webp)



点击**新标签**图标，通过链接进入结账区，或点击**QR 码**图标，通过 QR 码与客户共享您的 POS 平台。



![lien](assets/fr/16.webp)



![qr](assets/fr/17.webp)



客户可以通过该界面查阅产品目录并付款。



![pos](assets/fr/18.webp)



![chose](assets/fr/19.webp)



![pay](assets/fr/20.webp)



![checkout](assets/fr/21.webp)



在可用的扩展工具组中，您还可以找到**Invoice**和**支付链接**等工具，这些工具可让您在 generate 发票上注明特定产品，从而无需通过 POS 终端即可收取隔离付款。



## 跟踪您的付款



在**付款**菜单中，Voltz 可让您概览向您的各种投资组合付款的情况。


您可以通过 ：




- 状态
- 标签：例如，**TPOS** 用于销售点支付，**lnhub** 通过 Zeus 和 Blue Wallet 钱包的移动连接。
- 收藏品组合。
- 入账和出账总额。
- 一个明确界定的时期。



![payments](assets/fr/22.webp)



## 更多选择



Voltz 也是一个基础架构，您可以在此基础上构建自己的解决方案。在**扩展**部分，您将找到在 Voltz 和 LnBits 生态系统内开发自己的扩展的指南。



![build](assets/fr/23.webp)



如果您想在生态系统之外开发解决方案，但仍想使用其基础架构，在**节点网址**部分，您将找到 API 密钥和文档信息，并附有使用这些数据的示例。



您可以使用 Voltz wallet 从应用程序创建 Lightning 发票、支付 Lightning 发票、解码和验证发票。
