ในขณะที่หลายคนยังคงลด Bitcoin ให้เป็นเครื่องมือการลงทุนและการเก็งกำไร ผู้สร้างของมัน Satoshi Nakamoto ได้คิดค้นมันขึ้นมาเป็นระบบการชำระเงินอิเล็กทรอนิกส์แบบ Peer-to-Peer เป็นหลัก อย่างไรก็ตาม มันยากที่จะรู้ว่าคุณสามารถใช้จ่ายบิตคอยน์ของคุณได้ที่ไหน แต่ทั้งหมดนั้นเกิดขึ้นก่อน BTC Map



จาก OSM (OpenStreetMap) ซึ่งเป็นเครื่องมือการทำแผนที่แบบมีส่วนร่วมที่เป็นโอเพ่นซอร์ส BTC Map นำเสนอวิธีง่ายๆ ในการแสดงรายการสถานประกอบการที่รับชำระเงินด้วย BTC, Lightning หรือ On-Chain ฐานข้อมูลระดับโลกที่ยังอยู่ในช่วงเริ่มต้นแต่มีความสำคัญแล้ว ซึ่งถูกเติมเต็มโดยและสำหรับผู้ใช้บิตคอยน์



ไปที่ [btcmap.org](https://btcmap.org/):



![image](assets/fr/01.webp)



มีจำหน่ายที่:




- iOS ผ่าน [AppStore](https://apps.apple.com/app/btc-world-map/id6443604345)
- Android ผ่าน [Play Store](https://play.google.com/store/apps/details?id=org.btcmap.app)
- [F-Droid](https://f-droid.org/en/packages/org.btcmap/) แคตตาล็อกแอปโอเพนซอร์สสำหรับแอนดรอยด์
- โดยการดาวน์โหลด [apk android](https://github.com/teambtcmap/btcmap-android/releases/latest) โดยตรงจาก github ของโปรเจค
- ผ่านเว็บแอปจากเบราว์เซอร์ของคุณ (มุมมองของเราวันนี้)



วันนี้เราจะมุ่งเน้นไปที่เวอร์ชันเว็บ สำหรับตอนนี้มันเป็นภาษาอังกฤษทั้งหมด ดังนั้นเรามาดูส่วนต่างๆ ด้วยกัน และหากปุ่มหรือลิงก์ไม่ทำงาน ให้คลิกขวา -> เปิดในแท็บอื่น



มาเริ่มกันเลย ไอคอนรูปพระจันทร์หรือพระอาทิตย์ที่มุมขวาบนใช้สลับระหว่างธีมสว่างและธีมมืด



![image](assets/fr/02.webp)



แท็บต่างๆ ที่ด้านบนของไซต์ จากซ้ายไปขวา:




- โลโก้ BTC Map: กลับไปที่หน้าแรก
- "Maps": หัวใจของผลิตภัณฑ์ พร้อมด้วยแผนที่สองแบบ (สิ่งอำนวยความสะดวกและชุมชน)
- "แอป: สื่อที่แตกต่างกันในการติดตั้ง BTC Map
- "Areas": การนำเสนอและสถิติของชุมชน แยกตามทวีปและประเทศ
- "Maintain": มีส่วนร่วมในการอัปเดตและเพิ่มคุณค่าให้กับแผนที่
- "Wiki: the project's GitHub
- "สนับสนุนเรา": การบริจาคและการสนับสนุนเป็นแหล่งรายได้เพียงอย่างเดียวของโครงการ



โดยบังเอิญ แท็บเหล่านี้จะเป็นสารบัญของบทแนะนำนี้

- 2: ซูมออก
- 3: หน้าจอขนาดใหญ่

- 6: (แผนที่ร้านค้า) แสดงเฉพาะสถานประกอบการที่ได้รับการส่งเสริม

- 8: (แผนที่ร้านค้า) เพิ่มร้านค้า

คุณจะสังเกตเห็นว่ามีปุ่มสำหรับเปลี่ยนเลเยอร์แผนที่อยู่ที่มุมขวาบนของแต่ละแผนที่ ในบทแนะนำนี้ฉันเลือกที่จะใช้เวอร์ชันสีเข้ม

หากคุณคลิกที่ศิลปินภูมิทัศน์เท็กซัสนี้ จะมีแทรกปรากฏขึ้น:

## แท็บไซต์อื่น ๆ



**แอป**: หน้านี้เตือนคุณเกี่ยวกับสื่อที่มี BTC Map ให้บริการ



**Stats** เสนอข้อมูลสถิติต่างๆ เกี่ยวกับแอปพลิเคชันให้คุณ:




- แดชบอร์ด: สถิติบนฟีดฐานข้อมูล เช่น จำนวนสถานที่ที่ระบุไว้หรือจำนวนการยืนยันล่าสุดที่ทำขึ้น
- กระดานผู้นำ Tagger: ตารางของผู้ใช้ที่จัดอันดับตามปริมาณการมีส่วนร่วม (เข้าร่วมกับพวกเขา!)
- กระดานผู้นำชุมชน: การจัดอันดับชุมชน
- กระดานผู้นำประเทศ



**Areas (zones)** ให้ข้อมูลอื่นๆ นอกเหนือจากแผนที่:




- ชุมชน: แสดงรายการชุมชนที่ลงทะเบียนไว้แล้ว, แบบฟอร์มสำหรับการลงทะเบียนใหม่, และสถิติบางส่วน, ทั้งหมดจัดกลุ่มตามทวีป
- ประเทศ: สถิติเล็กน้อยที่จัดกลุ่มตามประเทศแสดงให้คุณเห็นจำนวนสถานประกอบการ, ผู้เข้าร่วม, ไฟล์ที่ต้องอัปเดต...



**รักษา (รักษา)**




- เพิ่มสถานที่: เพิ่มสถานที่ที่รับชำระเงิน Bitcoin
- ยืนยันตำแหน่ง: อัปเดต/แก้ไขข้อมูลเกี่ยวกับสถานที่ที่มีอยู่แล้วในรายการ
- เพิ่มชุมชน: เพิ่มชุมชน (มีการพิมพ์ผิดใน URL, เพื่อเข้าถึงแบบฟอร์ม, ไปที่ พื้นที่ -> ชุมชน -> เพิ่มชุมชน)
- ตั๋วที่เปิด: นี่คือที่ที่คำขอสำหรับการเพิ่ม การตรวจสอบ สถานที่ และชุมชนมาถึง เพื่อให้ผู้ที่อาสาสมัครดำเนินการ
- กิจกรรมการติดแท็ก: แสดงการดำเนินการล่าสุดที่ดำเนินการโดยผู้เข้าร่วมโครงการ (ผู้ใช้ใด ๆ เช่นคุณหรือฉันสามารถเป็นผู้เข้าร่วมได้) เช่น สถานที่ล่าสุดที่เพิ่ม อัปเดต หรือแม้กระทั่งลบ...
- ปัญหาการติดแท็ก: นี่คือที่ที่ผู้ใช้ระบุข้อผิดพลาดในการติดแท็กทั้งหมด



**Wiki**: ลิงก์นี้จะพาคุณไปยังหน้า GitHub ของโครงการ



**สนับสนุนเรา**: หน้านี้บอกวิธีการบริจาค (ใน Sats) หรือเป็นผู้สนับสนุนโครงการ



## "เพิ่มชุมชน



BTC Map ช่วยให้คุณสามารถ [เพิ่มชุมชนของคุณเอง](https://btcmap.org/communities/add/) มาดูวิธีการทีละขั้นตอน:




![image](assets/fr/10.webp)





- 1: โซนที่สอดคล้องกับชุมชนของคุณ
- 2: ชื่อชุมชนของคุณ
- 3: เว็บไซต์ URL
- 4: ที่อยู่ Lightning ที่สามารถส่งทิปได้
- 5: การอ้างอิงถึงเครือข่ายสังคมที่ชุมชนของคุณมีอยู่
- 6: ที่อยู่อีเมลของคุณเพื่อให้แพลตฟอร์มสามารถขอข้อมูลเพิ่มเติมจากคุณได้หากจำเป็น
- 7: คำอธิบายโดยย่อ (เช่น ชุมชนที่พูดภาษาเยอรมัน, ภูมิภาคแฟรงก์เฟิร์ต)
- 8: กรอก captcha (คำนึงถึงตัวพิมพ์เล็ก-ใหญ่)
- 9: คลิกที่ "Submit Community" เพื่อส่งแบบฟอร์ม



## "เพิ่มสถานที่
