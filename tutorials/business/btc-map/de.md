Während viele Bitcoin immer noch auf ein Investitions- und Spekulationsinstrument reduzieren, hat sein Schöpfer Satoshi Nakamoto es in erster Linie als ein elektronisches Peer-to-Peer-Zahlungssystem konzipiert. Allerdings ist es Hard wichtig zu wissen, wo man seine Bitcoins ausgeben kann. Aber das war vor BTC Map.



BTC Map basiert auf OSM (OpenStreetMap), einem Open-Source-Tool für partizipative Kartierung, und bietet eine einfache Möglichkeit, Einrichtungen aufzulisten, die Zahlungen in BTC, Lightning oder On-Chain akzeptieren. Eine globale Datenbank, die noch in den Kinderschuhen steckt, aber bereits unverzichtbar ist und von und für Bitcoiners befüllt wird.

## Händlerkarte



![image](assets/fr/04.webp)



Die Händlerkarte listet Einrichtungen weltweit auf, die Zahlungen in BTC akzeptieren. Sie können verschiedene Arten von Symbolen sehen, die einen ersten Hinweis auf die Art der Einrichtung geben (ein Pint für eine Bar, ein Diamant für ein Juweliergeschäft...). Ich nehme eins nach dem Zufallsprinzip.



Wenn Sie auf diesen texanischen Landschaftsmaler klicken, erscheint ein Insert:



![image](assets/fr/05.webp)





- Der Name in Blau ist der Name der Einrichtung
- Der Address und die Öffnungszeiten sind unten angegeben, sofern auf OSM verfügbar
- mit "Navigieren" können Sie eine Route zwischen Ihrer Position und der Einrichtung festlegen
- mit "Bearbeiten" können Sie Änderungen an der OSM-Datei vorschlagen (erfordert ein kostenloses OSM-Konto), z. B. Name, Kontaktangaben, Fahrpläne usw
- "Teilen" führt Sie zu einer erweiterten Beschreibung der Website
- unter "Mehr" finden Sie die folgenden Optionen:



![image](assets/fr/06.webp)





- "Anrufen" ist eine Abkürzung auf dem Telefon, um die Einrichtung anzurufen
- "Website" führt Sie zur Website des Unternehmens
- "Show Tags" zeigt auf OSM eingegebene Elemente an, ohne Hyperlinks oder Anwendungen auszulösen
- "Tag Issues" und "Map Legend" sind derzeit defekte Links
- "OSM anzeigen" fordert Sie auf, die Position der Anlage in der OSM-Anwendung zu öffnen



Endlich,





- "Last Surveyed" gibt das Datum an, an dem die Datei zuletzt erstellt oder aktualisiert wurde
- mit "Boost" können Sie Ihrer Website gegen eine Gebühr von Sats einen zeitlich begrenzten Sichtbarkeitsvorteil bieten
- mit "Standort überprüfen" gelangen Sie schließlich zu einem BTC Map-Formular, mit dem Sie eine Aktualisierung des Eintrags vorschlagen können. Schauen wir mal, wie es aussieht:



![image](assets/fr/07.webp)





- 1: Markieren Sie dieses Kästchen, wenn Sie nur bestätigen möchten, dass die Informationen in der Datei aktuell sind (in diesem Fall gehen Sie direkt zu Schritt 4)
- 2: Geben Sie hier an, welche Informationen ggf. falsch sind und welche Korrektur Sie vorschlagen
- 3: Beschreiben Sie hier, wie Sie die Informationen erhalten haben (Besuch, Anruf...)
- 4: Weiter zur Captcha-Prüfung (Groß- und Kleinschreibung beachten)
- 5: Klicken Sie auf "Bericht einreichen", um Ihren Vorschlag abzuschicken



## Gemeinschaft Karte



![image](assets/fr/08.webp)



Mit der "Community Map" können Sie die verschiedenen Bitcoin-Gemeinschaften auf der ganzen Welt entdecken, indem Sie sie aufteilen und geografisch darstellen.



![image](assets/fr/09.webp)



Sie werden sofort die orangefarbenen Bereiche bemerken. Wie Sie sehen können, handelt es sich dabei um die verschiedenen Bitcoin-Gemeinschaften, die auf der BTC-Karte aufgeführt sind. Wenn Sie mit der linken Maustaste auf eine davon klicken, wird eine kleine Einblendung mit allen bereitgestellten Links angezeigt, z. B. die Website und die Konten der sozialen Netzwerke. Wer weiß, vielleicht befinden Sie sich gerade inmitten einer aktiven Bitcoin-Gemeinschaft und sind nur ein paar Klicks davon entfernt, ihr beizutreten, wenn Sie Lust dazu haben!



## Andere Registerkarten der Website



**Apps**: Auf dieser Seite finden Sie Informationen zu den Medien, auf denen BTC Map verfügbar ist.



**Stats** bietet Ihnen verschiedene Statistiken über die Anwendung:




- Dashboard: Statistiken über den Datenbank-Feed, z. B. die Anzahl der aufgelisteten Standorte oder die Anzahl der kürzlich durchgeführten Überprüfungen
- Tagger-Leaderboard: Tabelle mit Nutzern, die nach der Anzahl ihrer Beiträge geordnet sind (schließen Sie sich ihnen an!)
- Community-Rangliste: Community-Ranglisten
- Länder-Rangliste



**Gebiete (Zonen)** liefert Ihnen andere Informationen als Karten:




- Gemeinschaften: listet die verschiedenen registrierten Gemeinschaften auf, das Formular für die Registrierung einer neuen Gemeinschaft und einige Statistiken, alle gruppiert nach Kontinenten
- Länder: einige nach Ländern gruppierte Statistiken zeigen Ihnen die Anzahl der Einrichtungen, Teilnehmer, zu aktualisierende Dateien...



**Maintain (pflegen)**




- Standort hinzufügen: Fügen Sie einen Standort hinzu, der Bitcoin-Zahlungen annimmt
- Standort überprüfen: Aktualisierung/Korrektur von Informationen zu einer bereits aufgelisteten Einrichtung
- Gemeinschaft hinzufügen: Fügen Sie eine Gemeinschaft hinzu (es gibt einen Tippfehler in der Url, um das Formular aufzurufen, gehen Sie zu Bereiche -> Gemeinschaften -> Gemeinschaft hinzufügen)
- Offene Tickets: Hier landen Anfragen für Ergänzungen, Überprüfungen, Standorte und Gemeinschaften, die von Freiwilligen bearbeitet werden
- Tagging-Aktivitäten: Zeigt Ihnen die letzten Aktionen der Projektteilnehmer (jeder Benutzer wie Sie oder ich kann ein Teilnehmer sein), wie z. B. die letzten hinzugefügten, aktualisierten oder sogar gelöschten Orte...
- Kennzeichnungsprobleme: Hier listen die Benutzer alle Kennzeichnungsfehler auf



**Wiki**: Dieser Link führt Sie zur GitHub-Seite des Projekts.

## "Gemeinschaft hinzufügen



BTC Map ermöglicht es Ihnen, [Ihre eigene Gemeinschaft hinzuzufügen] (https://btcmap.org/communities/add/), lassen Sie uns einen Blick darauf werfen, Schritt für Schritt:




![image](assets/fr/10.webp)





- 1: Das Ihrer Gemeinde entsprechende Gebiet
- 2: Name Ihrer Gemeinde
- 3: URL der Website
- 4: Ein Lightning Address, an den Tipps gesendet werden können
- 5: Verweise auf die sozialen Netzwerke, in denen Ihre Gemeinschaft präsent ist
- 6: Ihre E-Mail Address, damit die Plattform Sie bei Bedarf um weitere Informationen bitten kann
- 7: Eine kurze Beschreibung (z. B. deutschsprachige Gemeinschaft, Region Frankfurt)
- 8: Füllen Sie das Captcha aus (Groß- und Kleinschreibung beachten)
- 9: Klicken Sie auf "Submit Community", um das Formular abzuschicken



## "Standort hinzufügen



[Diese Seite](https://btcmap.org/add-location/) zeigt Ihnen, wie Sie mit Open Street Map selbst ein Bitcoin-konformes Immobilienverzeichnis hinzufügen können. Wenn Sie Probleme haben, können Sie ein Formular ausfüllen, damit jemand anderes den Eintrag für Sie erstellen kann (dies kann einige Wochen dauern). Schauen wir uns das mal an:



![image](assets/fr/11.webp)





- 1: Der Name der Einrichtung
- 2: Physischer Address (obligatorisch, Sie benötigen einen Address für die Straße)
- 3: Markieren Sie den genauen Punkt auf der Karte
- 4: In welche Kategorie fällt die Pflanze?
- 5: Welche Bitcoin-Zahlungsmedien sind verfügbar (BTC, Lightning, kontaktlos)?



![image](assets/fr/12.webp)





- 6: Website-URL (optional)
- 7: Eine Telefonnummer (fakultativ)
- 8: Öffnungszeiten (fakultativ)
- 9: Das X-Konto (Twitter) der Schule, dann Ihr eigenes (optional)
- 10: Jedes Detail, das Sie für angebracht halten
- 11: Ihre Rolle
  - "Ich bin der Geschäftsinhaber": Ich bin der Manager des Unternehmens
  - "Ich habe diese Einrichtung als Kunde besucht": Ich habe diese Einrichtung als Kunde besucht
  - "andere Methode
- 12: Ihre E-Mail Address für den Fall, dass die Plattform weitere Informationen benötigt
- 13: Captcha-Prüfung (Groß- und Kleinschreibung beachten)
- 14: Klicken Sie auf "Standort einreichen", um Ihren Eintrag zu senden
