尽管许多人仍将 Bitcoin 简化为一种投资和投机工具，但其创造者 Satoshi Nakamoto 主要将其设想为一种点对点电子支付系统。然而，要知道比特币可以花在哪里，这是 Hard。但那是在 BTC 地图之前。



BTC Map 基于 OSM（OpenStreetMap）（OpenStreetMap 本身就是一个开源的参与式制图工具），提供了一种简单的方式，列出接受 BTC、闪电币或 On-Chain 支付的机构。这是一个由比特币使用者创建并为比特币使用者服务的全球数据库，虽然还处于起步阶段，但已经非常重要。



请访问 [btcmap.org](https://btcmap.org/)：



![image](assets/fr/01.webp)



可在 ：




- iOS 通过 [AppStore](https://apps.apple.com/app/btc-world-map/id6443604345)
- 通过 [Play 商店](https://play.google.com/store/apps/details?id=org.btcmap.app) 安装安卓系统
- [F-Droid](https://f-droid.org/en/packages/org.btcmap/)安卓开源应用程序目录
- 直接从项目的 github 下载 [apk android](https://github.com/teambtcmap/btcmap-android/releases/latest)。
- 通过浏览器的网络应用程序（我们今天的观点）



今天，我们将重点介绍网络版。目前，它完全是英文版，所以让我们一起来看看不同的部分。如果按钮或链接不起作用，请右击 -> 在另一个标签页中打开。



让我们开始吧。右上角的月亮或太阳小图标可以在明暗主题之间切换。



![image](assets/fr/02.webp)



网站顶部的各种标签，从左到右依次为




- BTC 地图徽标： 返回主页

## 地图



该网站提供两张目标不同的卡片。不过，让我们先来介绍一下出现在每张卡片左侧边缘的工具：



![image](assets/fr/03.webp)





- 1: 放大
- 2: 放大
- 3：大屏幕
- 4: 找到我（如果您的浏览器允许的话）
- 5：（商家地图）按名称搜索商店
- 6：（商户地图）只显示已升级的商户
- 7: 返回主页
- 8: （商家地图）添加一家商店
- 9: 从 "社区 "卡切换到 "商户 "卡，反之亦然



*注：获得提升的机构是那些收到满意游客或用户提供的小费的机构，可在一段时间内提高其可见度*。



您还会注意到，在每张地图的右上角，都有一个用于更改地图 Layer 的按钮。在本教程中，我选择保留暗色版本。

![image](assets/fr/07.webp)

## 社区地图



![image](assets/fr/08.webp)



通过 "社区地图"，您可以发现世界各地不同的 Bitcoin 社区，并将它们按地理位置进行划分和展示。



![image](assets/fr/09.webp)



您会立即注意到橙色区域。如您所见，这些是 BTC 地图上列出的各种 Bitcoin 社区。左键点击其中任何一个，都会出现一个小插件，显示所提供的所有链接，如网站和社交网络账户。谁知道呢，说不定您就在一个活跃的 Bitcoin 社区之中，只需点击几下就能加入其中！



## 其他网站标签



**应用程序**：本页提醒您可以使用 BTC 地图的媒体。



**Stats** 为您提供有关应用程序的多项统计数据：




- 仪表板：数据库馈送的统计数据，例如列出的地点数量或最近进行的验证数量
- 标签排行榜：按贡献量排名的用户表（加入他们！）。
- 社区排行榜：社区排名
- 国家排行榜



**区域（zone）** 为您提供地图以外的信息：




- 社群：列出各种已注册的社群、注册新社群的表格以及一些统计数据，所有数据均按大洲分类
- 国家：按国家分类的一些统计数据可显示机构数量、参与者数量、待更新文件数量......



**维护（保持）**




- 添加地点：添加接受 Bitcoin 支付的地点
- 验证地点：更新/更正已列出设施的信息
- 添加社区：添加社区（网址有错别字，要访问该表单，请转到区域 -> 社区 -> 添加社区）
- 开放票据：这是添加、核实、地点和社区请求的地方，由志愿者处理
- 标记活动：显示项目参与者（任何用户，如你或我，都可以是参与者）采取的最新行动，例如添加、更新甚至删除的最新位置...
- 标记问题：用户在此列出所有标记错误



**维基**：通过此链接可进入项目的 GitHub 页面。



**支持我们**：本页告诉您如何捐款（Sats）或成为项目赞助商。



## "添加社区



BTC Map 允许您 [添加自己的社区](https://btcmap.org/communities/add/)，让我们一步步来看看：




![image](assets/fr/10.webp)





- 1: 与您的社区相对应的区域
- 2: 社区名称
- 3: 网站 URL
- 4: 可向其发送提示的闪电 Address
- 5: 提及您的社区所在的社交网络
- 6: 您的电子邮件 Address，以便平台在必要时向您询问更多信息
- 7: 简短描述（如德语社区、法兰克福地区）
- 8: 填写验证码（区分大小写）
- 9： 点击 "提交社区 "提交表格



## "添加位置



[本页](https://btcmap.org/add-location/) 向您展示了如何使用 Open Street Map 自行添加符合 Bitcoin 标准的房产列表。如果您遇到困难，可以填写一份表格，让其他人为您创建列表（这可能需要几周时间）。让我们来看看：



![image](assets/fr/11.webp)





- 1: 机构名称
- 2：实体 Address（必填，您需要一个街道 Address）。
- 3: 在地图上标出精确点
- 4: 该植物属于哪一类？
- 5: 有哪些 Bitcoin 支付媒介（BTC、闪电、非接触式）？



![image](assets/fr/12.webp)





- 6: 网站 URL（可选）
- 7：电话号码（可选）
- 8: 开放时间（可选）
- 9：学校的 X（twitter）账户，然后是您自己的账户（可选）
- 10: 任何你认为合适的细节
- 11: 你的角色
  - "我是企业主我是企业经理
  - "我以顾客身份来过"：我曾作为顾客光顾过这家店
  - "其他方法
- 12: 如果平台需要进一步信息，您的电子邮件 Address
- 13: 验证码检查（区分大小写）
- 14: 点击 "提交位置 "发送您的列表
