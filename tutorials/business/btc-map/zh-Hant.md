雖然許多人仍將 Bitcoin 簡化為一種投資和投機工具，但其創造者 Satoshi Nakamoto 主要將其構想為一種點對點電子支付系統。然而，要知道比特幣可以花在哪裡是 Hard 的事。但那是在 BTC 地圖之前。



BTC Map 基於 OSM (OpenStreetMap)，本身就是一個開放源碼的參與性地圖工具，它提供了一個簡單的方法來列出接受 BTC、Lightning 或 On-Chain 付款的機構。這是一個全球資料庫，雖然還處於起步階段，但已經非常重要，它是由比特幣使用者建立的，也是為了比特幣使用者而建立的。



前往 [btcmap.org](https://btcmap.org/)：

- 透過瀏覽器的網頁應用程式 (我們今天的觀點)



今天我們將專注於網頁版本。目前，它完全是英文版本，所以讓我們一起來看看不同的部分。如果按鈕或連結無法運作，請按一下滑鼠右鍵 -> 在其他標籤頁開啟。

## 商家地圖



![image](assets/fr/04.webp)



Merchant Map 列出了全球接受 BTC 付款的機構。您可以看到不同類型的圖標，這些圖標是機構性質的第一條線索（一品脫代表酒吧，鑽石代表珠寶店......）。我隨機選一個。



如果您按一下這位德州風景藝術家，就會出現一個插頁：



![image](assets/fr/05.webp)





- 藍色的名稱是機構的名稱
- Address 和開放時間如下，若 OSM 上有提供
- "導航」可在您的位置與設施之間建立路線
- "編輯」可讓您建議修改 OSM 檔案 (需要免費 OSM 帳戶)，例如姓名、聯絡方式、時間表等。
- "分享」會帶您到延伸的網站說明
- 「更多」會展開下列選項：



![image](assets/fr/06.webp)





- 「呼叫 」是電話上呼叫設施的捷徑
- 「網站」會帶您到公司網站
- "顯示標籤」可顯示在 OSM 上輸入的項目，而不會觸發超連結或應用程式
- 目前「標籤問題」和「地圖圖例」是已損壞的連結
- "檢視 OSM」會提示您在 OSM 應用程式上開啟工廠的位置



最後





- "最後測量」表示檔案最後建立或更新的日期
- "提升」讓您只需支付 Sats 的費用，即可為您的網站提供有時限的能見度優勢
- 最後，「驗證位置」會帶您到 BTC 地圖表單，以建議更新列名。讓我們看看它的外觀：



![image](assets/fr/07.webp)





- 1: 如果您只想確認檔案中的資訊是最新的，請勾選此方塊（在此情況下，請直接前往步驟 4）
- 2: 請在此說明哪些資訊（如有）不正確，以及您建議的修正方式
- 3: 在此描述您如何獲得資訊（訪問、電話...)
- 4: 進入驗證碼（區分大小寫）
- 5: 按一下「提交報告」以傳送您的建議

## 其他網站標籤



**應用程式**：本頁提醒您 BTC 地圖可用的媒體。



**Stats** 提供您關於應用程式的多種統計資料：




- 儀表板：資料庫饋送的統計資料，例如列出的地點數量或最近進行的驗證數量
- 標籤排行榜：依貢獻量排名的使用者表（加入他們！）。
- 社群排行榜：社群排名
- 國家排行榜



**Areas (zones) ** 可為您提供地圖以外的資訊：




- 社群：列出各種已註冊的社群、註冊新社群的表格，以及一些統計資料，所有資料都依大洲分類。
- 國家：按國家分類的幾項統計資料可顯示機構、參與者、待更新檔案的數量...



**Maintain (維護) **




- 新增地點：新增接受 Bitcoin 付款的地點
- 驗證地點：更新/更正已列出設施的資訊
- 新增社群：新增社群 (網址有誤，若要存取表單，請前往區域 -> 社群 -> 新增社群)
- Open Tickets：這是新增、驗證、位置和社群請求的位置，由志工處理。
- 標籤活動： 顯示專案參與者（任何使用者如你或我都可以是參與者）的最新動作，例如最新新增、更新甚至刪除的位置...
- 標籤問題：使用者在此列出所有標籤錯誤



**維基**：此連結會帶您到該專案的 GitHub 頁面。



**支持我們**：本頁面會告訴您如何捐款 (以 Sats 為單位) 或成為專案贊助者。



## 「新增社群



BTC Map 允許您 [新增您自己的社群](https://btcmap.org/communities/add/)，讓我們一步一步來看看：




![image](assets/fr/10.webp)





- 1: 與您社群對應的區域
- 2: 社區名稱
- 3: 網站 URL
- 4: 可傳送提示的 Lightning Address
- 5: 參考您社群所在的社群網路
- 6: 您的電子郵件 Address，以便平台在必要時向您索取更多資訊
- 7: 簡明說明（例如：德語社群、法蘭克福地區）
- 8: 填寫驗證碼（區分大小寫）
- 9: 按一下「提交社群」以提交表格



## 「新增位置



[This page](https://btcmap.org/add-location/) 展示如何使用 Open Street Map 自行新增符合 Bitcoin 標準的房產清單。如果您有問題，可以填寫一份表格，讓其他人為您建立房產清單（這可能需要數週時間）。讓我們來看看：



![image](assets/fr/11.webp)





- 1: 機構名稱
- 2: 實體 Address（必須，您需要街頭 Address）
- 3: 在地圖上標記精確的地點
- 4: 該植物屬於哪一類？
- 5: 有哪些 Bitcoin 支付媒介可用（BTC、Lightning、非接觸式）？



![image](assets/fr/12.webp)





- 6: 網站網址（選擇性）
- 7：一個電話號碼（可選）
- 8: 開放時間（選擇性）
- 9： 學校的 X (twitter) 帳戶，然後是您自己的 (可選)
- 10: 任何您認為合適的細節
- 11: 您的角色
  - 「我是企業老闆」：我是企業經理
  - "我以顧客身份造訪我曾以顧客身份造訪此機構
  - "其他方法
- 12: 您的電子郵件 Address，以備平台需要進一步資訊時使用
- 13: 驗證碼檢查（區分大小寫）
- 14: 按一下「提交位置」以傳送您的列表
