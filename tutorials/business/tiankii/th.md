ในระบบนิเวศ Bitcoin การยอมรับการชำระเงินแบบเรียลไทม์ยังคงเป็นความท้าทายหลักสำหรับธุรกิจและบุคคลทั่วไป โซลูชันแบบดั้งเดิมมักต้องการความรู้ทางเทคนิคขั้นสูง โครงสร้างพื้นฐานที่ซับซ้อนในการบำรุงรักษา หรือจำเป็นต้องให้เงินทุนถูกถือโดยตัวกลาง สถานการณ์นี้กำลังยับยั้งการยอมรับ Bitcoin ในวงกว้างในฐานะสกุลเงินที่ใช้ในชีวิตประจำวัน แม้จะมีความก้าวหน้าทางเทคโนโลยีของ Lightning Network ก็ตาม



Tiankii บริษัทจากเอลซัลวาดอร์ที่ก่อตั้งขึ้นในปี 2021 ตอบสนองต่อปัญหานี้ด้วยการนำเสนอระบบโครงสร้างพื้นฐาน Bitcoin ที่เข้าถึงได้และเป็นโมดูล แทนที่จะบังคับให้ใช้ระบบปิด Tiankii นำเสนอชุดเครื่องมือที่เชื่อมต่อกันซึ่งช่วยให้ทุกคนสามารถรวม Bitcoin Lightning เข้ากับธุรกิจของตนได้โดยไม่ต้องเสียการควบคุมเงินทุนของตนเอง



## Tiankii คืออะไร?



### ต้นกำเนิดและปรัชญา



Tiankii - คำในภาษานาวาตล์ที่หมายถึง "ตลาดกลางแจ้งที่ทุกคนเข้าถึงได้" - เป็นสตาร์ทอัพ Bitcoin แห่งแรกของเอลซัลวาดอร์ที่มีความเป็น 100% ก่อตั้งโดย Darvin Otero หลังจากการยอมรับ Bitcoin เป็นเงินตราที่ถูกกฎหมายของประเทศ บริษัทมีเป้าหมายที่จะเปลี่ยน Bitcoin จากการเป็นที่เก็บมูลค่าให้กลายเป็นสกุลเงินที่สามารถใช้ทำธุรกรรมสำหรับการซื้อขายในชีวิตประจำวันได้ แตกต่างจากแพลตฟอร์มที่มีการควบคุมเงิน Tiankii ใช้วิธีการที่ไม่ควบคุมเงิน ซึ่งผู้ใช้จะยังคงควบคุมเงินของตนเอง โดยมีโครงสร้างพื้นฐานทำหน้าที่เป็นเพียงตัวกลางทางเทคนิคเท่านั้น



### สถาปัตยกรรมทางเทคนิค



Tiankii ถูกวางตำแหน่งเป็นผู้ให้บริการโครงสร้างพื้นฐาน Bitcoin-as-a-Service ที่อิงจาก Lightning Network เทคโนโลยีชั้นที่สองนี้ช่วยให้การทำธุรกรรมเกิดขึ้นได้เกือบจะทันทีด้วยต้นทุนที่น้อยมาก ทำให้การชำระเงินขนาดเล็กและการซื้อในชีวิตประจำวันเป็นไปได้



สถาปัตยกรรมนี้มีพื้นฐานอยู่บนสามเสาหลัก:



**Lightning-first**: ให้ความสำคัญกับเครือข่าย Lightning อย่างเป็นระบบเพื่อความรวดเร็วและต้นทุนที่ต่ำกว่า ในขณะที่ยังคงรองรับการทำธุรกรรม on-chain สำหรับจำนวนเงินที่มาก



**มาตรฐานเปิด**: การใช้ LNURL เพื่อทำให้การร้องขอการชำระเงินเป็นอัตโนมัติ, Lightning Address สำหรับอีเมลไอดีที่อ่านได้, และ Bolt Card สำหรับการชำระเงิน NFC ที่สามารถทำงานร่วมกันได้.



**wallet-agnostic modularity**: ความเป็นไปได้ในการเชื่อมต่อพอร์ตโฟลิโอสายฟ้าที่แตกต่างกัน (LNbits, Blink, BlueWallet) หรือโหนดของคุณเอง มอบความยืดหยุ่นสูงสุดขึ้นอยู่กับระดับของความเชี่ยวชาญและความเป็นอิสระที่ต้องการ



## เครื่องมือระบบนิเวศ Tiankii



### Bitcoin POS - เครื่องชำระเงินในร้านค้า



แอปพลิเคชันนี้เปลี่ยนสมาร์ทโฟนหรือแท็บเล็ตให้เป็นเทอร์มินัล Lightning ผู้ขายป้อนจำนวนเงินในสกุลเงินท้องถิ่น และแอปจะสร้างรหัส QR ของ Lightning หรือรับบัตร Bolt ธุรกรรมจะได้รับเครดิตทันทีโดยไม่มีค่าคอมมิชชั่น พร้อมการแปลงอัตโนมัติในกว่า 150 สกุลเงิน การจัดการทิป และประวัติการขาย



### แดชบอร์ดผู้ค้า - การจัดการการขายแบบรวมศูนย์



Interface เว็บศูนย์กลางเพื่อเชื่อมต่อกับ wallet Lightning ติดตามธุรกรรมแบบเรียลไทม์ ออกใบแจ้งหนี้และรายงานบัญชี generate แพลตฟอร์มรวบรวมทุกช่องทาง: การชำระเงินในร้านค้า (POS) การขายออนไลน์ (ปลั๊กอินอีคอมเมิร์ซ) หรือการรวม API การชำระเงินจะรวมกันที่ wallet ที่เลือก



### Bitcoin บัตรไร้สัมผัส (Bolt บัตร)



บัตร NFC Bolt แสดงถึงนวัตกรรมสำคัญของ Tiankii ในการทำให้ Bitcoin เข้าถึงได้สำหรับประชาชนทั่วไป ทำงานเหมือนบัตรธนาคารแบบไร้สัมผัส ช่วยให้คุณชำระเงินสำหรับการซื้อ Bitcoin Lightning ได้ง่ายๆ เพียงแตะที่เครื่องรับที่รองรับ



ไม่เหมือนกับโซลูชันการดูแลแบบดั้งเดิม การ์ดนี้ปฏิบัติตามมาตรฐานเปิดที่รับประกันการทำงานร่วมกันได้ ผู้ใช้สามารถเชื่อมโยงกับ wallet ของตนเองผ่านแอปพลิเคชัน IBI ทำให้สามารถเก็บรักษาคีย์ส่วนตัวและควบคุมเงินที่เกี่ยวข้องได้อย่างเต็มที่ การชำระเงินใช้เวลาเพียงหนึ่งวินาที โดยไม่จำเป็นต้องนำสมาร์ทโฟนออกมาหรือมีการเชื่อมต่ออินเทอร์เน็ตในขณะชำระเงิน



โซลูชันนี้มีความครอบคลุมเป็นพิเศษสำหรับผู้ที่ไม่คุ้นเคยกับสมาร์ทโฟน โดยเสนอทางเข้าที่เข้าถึงได้สู่เศรษฐกิจ Bitcoin



### IBI App - Interface individual Bitcoin



แอปพลิเคชัน IBI (Individual Bitcoin Interface) ถูกออกแบบมาสำหรับบุคคลที่ต้องการใช้ Bitcoin Lightning ในชีวิตประจำวัน ข้อได้เปรียบหลักของมันอยู่ที่การสร้าง Address Lightning ส่วนบุคคล ซึ่งเป็นตัวระบุการชำระเงินที่สามารถอ่านได้ในรูปแบบอีเมล (ตัวอย่าง: alice@ibi.me)



ตัวระบุนี้ช่วยให้การรับชำระเงินง่ายขึ้นอย่างมาก: ไม่จำเป็นต้อง generate ใบแจ้งหนี้ใหม่สำหรับแต่ละธุรกรรม ผู้ส่งสามารถป้อนที่อยู่ Lightning ได้อย่างง่ายดาย อินเทอร์เฟซยังช่วยให้คุณจัดการ Bolt Card ของคุณ (การเปิดใช้งาน การปิดใช้งาน ขีดจำกัดการใช้จ่าย) เชื่อมต่อกระเป๋าเงิน Lightning ต่างๆ และชำระเงินโดยการสแกนรหัส QR



### ปลั๊กอินอีคอมเมิร์ซ



การผสานรวมที่พร้อมใช้งานสำหรับ WooCommerce, Shopify และ Cloudbeds ติดตั้งในไม่กี่นาทีเพื่อเพิ่ม Bitcoin Lightning ในการชำระเงิน ข้อดี: ไม่มีค่าคอมมิชชั่น (เทียบกับบัตรเครดิต 2-3%), การชำระเงินทันที, การเข้าถึงทั่วโลก, การกำจัดการเรียกเก็บเงินคืน การชำระเงินจะมาถึงโดยตรงบน wallet ที่เชื่อมต่อของผู้ค้า



### Bitcoin API และเครื่องมือสำหรับนักพัฒนา



Tiankii SDK ทำให้สามารถผสานรวม Bitcoin Lightning เข้ากับแอปพลิเคชันที่มีอยู่โดยไม่ต้องดูแลโหนดของตัวเอง API จัดการการสร้างใบแจ้งหนี้ การยืนยันการชำระเงิน และการส่งจดหมายจำนวนมากผ่านโครงสร้างพื้นฐานที่แข็งแกร่งซึ่งโฮสต์บน Google Cloud Command Center ทำให้ข้อเสนอสมบูรณ์สำหรับองค์กรที่มี Address Lightning บนโดเมนที่กำหนดเอง การชำระเงินจำนวนมาก และการจัดการศูนย์กลางของเทอร์มินัล NFC และบัตร



## การติดตั้งและขั้นตอนแรก



### ข้อกำหนดเบื้องต้นที่จำเป็น



การใช้ Tiankii ไม่จำเป็นต้องมีความรู้ทางเทคนิคที่ซับซ้อน แอปพลิเคชันทำงานผ่านเว็บเบราว์เซอร์บนสมาร์ทโฟน แท็บเล็ต หรือคอมพิวเตอร์ ไม่จำเป็นต้องติดตั้งแอปพลิเคชันพื้นเมือง: สิ่งที่คุณต้องมีคือการเข้าถึงอินเทอร์เน็ตและเบราว์เซอร์ที่ทันสมัยเพื่อเข้าถึง IBI หรือ Merchant Dashboard



**สำหรับผู้ใช้ส่วนตัว (แอป IBI)**: ไม่จำเป็นต้องมี wallet Lightning ล่วงหน้า เมื่อคุณสร้างบัญชีของคุณ Tiankii จะกำหนดค่า Address Lightning ที่ใช้งานได้โดยอัตโนมัติผ่าน [Breez SDK Liquid](https://sdk-doc-liquid.breez.technology/guide/about_breez_sdk_liquid.html) ซึ่งเป็นการใช้งานแบบไม่มีโหนดที่ใช้เครือข่าย Liquid ในพื้นหลัง คุณสามารถรับและส่งการชำระเงินได้ทันทีโดยไม่ต้องมีการกำหนดค่าทางเทคนิค โซลูชันนี้ช่วยให้การเข้าถึงสำหรับผู้เริ่มต้นง่ายขึ้นอย่างมาก ในขณะที่ยังคงการควบคุมด้วยตนเอง



**สำหรับผู้ค้า (แดชบอร์ดผู้ค้า)** : การเชื่อมต่อ wallet Lightning ที่มีอยู่เป็นสิ่งจำเป็นในการใช้เครื่องรับชำระเงินและรับการชำระเงิน Tiankii รองรับหลายโซลูชัน: กระเป๋าเงินมือถือ (Blink, Strike), โซลูชันที่โฮสต์เอง (LNbits, LND/CLN node), หรือกระเป๋าเงินเว็บ (Alby) คู่มือการเชื่อมต่อโดยละเอียดมีอยู่ในส่วนทรัพยากรของบทเรียนนี้

หลังจากเลือก "การใช้งานธุรกิจ" คุณจะถูกเปลี่ยนเส้นทางไปยัง Merchant Dashboard (pay.tiankii.com) โดยอัตโนมัติ ซึ่งจะนำคุณไปยังอินเทอร์เฟซการจัดการธุรกิจที่มีการติดตามรายได้ การทำธุรกรรม และเครื่องมือการชำระเงิน เพื่อเริ่มรับการชำระเงิน คุณต้องเชื่อมต่อ wallet Lightning ของคุณก่อนโดยคลิกที่ลิงก์ที่ด้านบนของหน้า (ดูลูกศรในภาพ)

## ประโยชน์และข้อจำกัด



### ไฮไลท์



**การดูแลตนเอง**: คุณเก็บรักษากุญแจส่วนตัวและควบคุมเงินทุนของคุณอย่างเต็มที่ ไม่มีความเสี่ยงจากการถูกระงับบัญชีหรือการล้มละลายของแพลตฟอร์มบุคคลที่สาม



**ความเรียบง่าย**: Lightning Address เป็นที่อยู่อีเมล, การชำระเงิน NFC ด้วยการแตะเพียงครั้งเดียว, อินเทอร์เฟซที่ใช้งานง่ายโดยไม่ต้องมีความเชี่ยวชาญทางเทคนิค



**ระบบนิเวศที่สมบูรณ์**: เครื่องมือสำหรับทุกโปรไฟล์ (บุคคล ร้านค้า นักพัฒนา) พร้อมการผสานรวมแบบโมดูลาร์เพื่อตอบสนองความต้องการของคุณ



**การปฏิบัติตามข้อกำหนดทางวิชาชีพ**: การโฮสต์บนคลาวด์ที่ปลอดภัย, การปฏิบัติตามมาตรฐาน PCI-DSS, การปฏิบัติตามข้อกำหนดของกฎหมายเอลซัลวาดอร์



### ข้อจำกัด



**ข้อจำกัดของ Lightning**: ต้องการการเชื่อมต่อ wallet ถาวร, การจัดการสภาพคล่องสำหรับปริมาณมาก, ความเสี่ยงของความล้มเหลวหากผู้รับออฟไลน์ Tiankii ลดผลกระทบเหล่านี้ด้วย LSPs ที่รวมอยู่ในระบบ



**การพึ่งพา SaaS**: หาก Tiankii ไม่พร้อมใช้งาน การสร้างใบแจ้งหนี้จะเป็นไปไม่ได้ชั่วคราว (เงินของคุณยังคงปลอดภัย)



**ความเป็นส่วนตัว**: Tiankii สามารถสังเกตข้อมูลเมตาของธุรกรรม (จำนวนเงิน, วันที่) มีความเป็นส่วนตัวน้อยกว่าการโฮสต์ด้วยตัวเอง เช่น BTCPay Server.



## บทสรุป



Tiankii แสดงให้เห็นว่าโครงสร้างพื้นฐานที่ออกแบบมาอย่างดีสามารถขจัดอุปสรรคทางเทคนิคที่ขัดขวางการยอมรับ Bitcoin ในวงกว้างในฐานะสกุลเงินที่ใช้ในชีวิตประจำวันได้อย่างไร โดยการผสานพลังของ Lightning Network เข้ากับแนวทางที่ไม่ต้องมีผู้ดูแลและเครื่องมือที่เข้าถึงได้ ระบบนิเวศนี้จึงเสนอเส้นทางที่สมดุลระหว่างอธิปไตยทางการเงินและความสะดวกในการใช้งาน



สำหรับพ่อค้า Tiankii เป็นโอกาสในการลดต้นทุนการทำธุรกรรมอย่างมากในขณะที่เข้าถึงฐานลูกค้าใหม่ สำหรับผู้บริโภค Lightning Address และบัตร NFC เปลี่ยน Bitcoin ให้เป็นสกุลเงินที่ใช้งานได้จริง โดยไม่มีความซับซ้อนทางเทคนิค



แม้ว่าการยอมรับอย่างแพร่หลายของ Bitcoin ยังคงเป็นความท้าทายที่ต้องการการศึกษาและเวลา โครงสร้างพื้นฐานเช่น Tiankii แสดงให้เห็นว่าเครื่องมือทางเทคนิคมีอยู่แล้ว ภารกิจในการทำให้การชำระเงิน Bitcoin ง่ายขึ้นไม่ใช่วิสัยทัศน์ที่ห่างไกลอีกต่อไป แต่เป็นความจริงที่ทุกคนที่พร้อมจะก้าวกระโดดสามารถเข้าถึงได้
