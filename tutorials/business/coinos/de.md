## Einführung



In einer Welt, in der Zahlungen immer digitaler werden, etabliert sich Bitcoin allmählich als eine Alternative. Doch für viele Neulinge kann die Verwaltung eines wallet oder die Annahme von Bitcoin-Zahlungen kompliziert erscheinen.



Hier kommt Coinos ins Spiel, eine Plattform von Adam Soltys, einem Webentwickler, der die erste Version entwickelt hat, um Händlern in seiner Region zu helfen, Bitcoin-Zahlungen zu akzeptieren.



Coinos ist eine einfache, intuitive Plattform, die leicht zu bedienen ist. Sie ermöglicht das Senden, Empfangen und Akzeptieren von Zahlungen in Bitcoin, Liquid oder Lightning, direkt von Ihrem Browser aus, ohne technische Installation. Coinos ist für alle zugänglich und kombiniert die Vorteile von Bitcoin mit der Einfachheit einer Webanwendung (*Progressive Web App*, oder PWA), ideal für Privatpersonen, Händler und Neugierige.

## Erste Schritte mit Coinos



Sie brauchen keine technischen oder tiefgreifenden Kenntnisse, um **Coinos** in die Hand zu nehmen. Ein Grundverständnis von Bitcoin, on chain, Lightning und Liquid Transaktionen ist jedoch von Vorteil.

- und andere, vor allem über das **Liquid**-Netz und das **Ecash**-Protokoll.

## Bitcoins erhalten



Um Bitcoins auf Coinos zu empfangen, klicken Sie entweder auf **Empfangen** oder auf das Symbol **Empfangsschnittstelle**. Sobald Sie auf der Empfangsschnittstelle sind, fragen Sie Ihren Absender, **an welches Netzwerk er die Bitcoins senden soll**, um zu bestimmen, **an welche Adresse er sie senden soll**.



### Empfang auf dem Hauptkanal



Klicken Sie auf **Bitcoin**.



![screen|220](assets/fr/27.webp)



Es werden dann automatisch ein QR-Code und eine Lieferadresse generiert.



![screen|220](assets/fr/28.webp)



Sie können sie personalisieren, indem Sie einen bestimmten Betrag und einen Vermerk (Etikett) festlegen.



![screen|220](assets/fr/29.webp)



![screen|220](assets/fr/30.webp)



Ihr **QR-Code** und Ihre **personalisierte Bitcoin-Adresse** sind nun fertig; Sie müssen sie nur noch an Ihren Absender weiterleiten, um Ihre Bitcoins zu erhalten.



![screen|220](assets/fr/31.webp)



### Empfang über Liquid



Klicken Sie auf **Mehr Optionen**, dann auf **Liquid**.



![screen|220](assets/fr/32.webp)



![screen|220](assets/fr/33.webp)



Es werden auch ein QR-Code und eine Liquid-Empfangsadresse generiert.



![screen|220](assets/fr/34.webp)



Coinos erinnert Sie daran, dass Sie nur Liquid** (**L-BTC**) Bitcoins an diese Adresse oder über diesen QR-Code senden dürfen.



Sie können sie mit einem bestimmten Betrag und einem Etikett personalisieren.



![screen|220](assets/fr/35.webp)



Geben Sie es an Ihren Absender weiter und erhalten Sie Ihre **L-BTC**.



![screen|220](assets/fr/36.webp)



### Lightning Network-Empfang



Standardmäßig öffnet sich die Empfangsschnittstelle mit der Lightning-Option. Sie haben dann Ihren QR-Code und Ihre Lightning-Adresse, die Sie mit einem bestimmten Betrag und einem Vermerk personalisieren können.



![screen|220](assets/fr/37.webp)



![screen|220](assets/fr/38.webp)



Ihr personalisierter **QR-Code** und Ihre **Blitzadresse** sind jetzt fertig. Schicke sie an deinen Absender und erhalte deine Satoshis sofort.



![screen|220](assets/fr/39.webp)



Um Zahlungen über **Lightning** zu erhalten, können Sie das kleine Dropdown-Menü auf der Hauptschnittstelle des wallet verwenden. Damit können Sie Ihre Lightning-Adresse anzeigen: Sie können sie dann kopieren oder zu ihrem QR-Code wechseln, um sie einfach zu teilen.



![screen|220](assets/fr/40.webp)



Jedes Mal, wenn Sie einen Betrag eingeben, unabhängig vom verwendeten Netzwerk, zeigt Coinos automatisch den Gegenwert in Satoshis an.



## Bitcoins senden



Wenn Sie Bitcoins von Ihrem Coinos wallet aus versenden möchten, klicken Sie auf das Symbol **Senden**. Auf der entsprechenden Schnittstelle haben Sie mehrere Optionen:




- Scannen Sie einen vom Empfänger bereitgestellten QR-Code;
- Fügen Sie eine kopierte Adresse in die Zwischenablage ein;
- Wählen Sie im Bereich **Kontakte** direkt den Benutzernamen einer Person aus, mit der Sie bereits Transaktionen getätigt haben;
- Geben Sie eine Bitcoin-Adresse (on-chain, Liquid oder Bolt LN), eine Lightning-Rechnung oder den Coinos-Benutzernamen des Empfängers in das entsprechende Feld ein.



![screen|220](assets/fr/41.webp)



Nachdem Sie die Adresse eingegeben oder eingefügt haben, drücken Sie **Weiter**, geben Sie den Transaktionsbetrag ein und drücken Sie dann erneut **Weiter**.



![screen|220](assets/fr/42.webp)



![screen|220](assets/fr/43.webp)



Der letzte Schritt vor dem Versand ist die Überprüfung der Informationen und die Anpassung der Netzentgelte an Ihre Bedürfnisse.



![screen|220](assets/fr/44.webp)



Um die Gebühren anzupassen, drücken Sie auf das kleine Dreieck und wählen dann die gewünschte Gebührenstufe entsprechend der gewünschten Ausführungsgeschwindigkeit. Je schneller die Transaktion bestätigt werden muss, desto höher ist die Netzwerkgebühr. Beachten Sie auch, dass Coinos eine kleine Servicegebühr erhebt. Nachdem Sie alle Parameter überprüft und Ihren Wünschen entsprechend angepasst haben, klicken Sie auf **Senden**.



![screen|220](assets/fr/45.webp)

## Sicherheit und bewährte Praktiken



Sicherheit ist ein zentrales Thema beim Ausgeben von Bitcoins. Auch wenn Coinos die Erfahrung einfach und nahtlos macht, ist es wichtig, Ihre Bitcoins zu schützen.





- Verwenden Sie ein sicheres Passwort: Wählen Sie immer ein eindeutiges, langes und schwer zu erratendes Passwort.
- Aktivieren Sie die doppelte Authentifizierung (2FA).
- Behalten Sie Ihr Login für sich: Geben Sie Ihr Passwort niemals an Dritte weiter. Coinos wird Sie niemals danach fragen, weder per Nachricht noch per E-Mail.
- Lagern Sie keine großen Beträge auf der gehosteten Instanz: Coinos ist praktisch für tägliche Zahlungen, bleibt aber eine Verwahrungslösung. Bewahren Sie nur das auf, was Sie für die täglichen Transaktionen benötigen. Entscheiden Sie sich bei größeren Beträgen für ein nicht verwahrtes wallet.
