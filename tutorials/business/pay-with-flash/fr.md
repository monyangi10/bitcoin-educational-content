L'adoption de Bitcoin est devenue massive tout autour du globe grâce aux possibilités offertes par le Lightning Network. Plusieurs solutions existent désormais pour vous permettre de faire partie de cette vague d'adoption, et intégrer le bitcoin comme moyen de paiement afin de donner à votre service, votre commerce, une envergure globale afin de recevoir des paiements instantanés de partout dans le monde. Dans cette approche, Flash, une solution self-custody se démarque particulièrement par ses fonctionnalités et son expérience utilisateur.

## Débuter avec Flash

Flash est une plateforme qui vous permet en tant que commerçant, entrepreneur d'encaisser des paiements en vous basant sur la surcouche Lightning du protocole Bitcoin. La plateforme est assez intuitive et vous présente dès le début les étapes pour configurer votre organisation.

### Créer un point de vente

Vous avez un magasin physique déjà implanté dans votre région et vous souhaitez accepter les paiements en bitcoin ? L'option **Point de vente** de Flash est la solution qui vous correspond le mieux.

A la création de votre point de vente, vous obtenez un lien web qui constitue l'interface à utiliser pour effectuer des actions dans le point de vente.

![pos-link](assets/fr/08.webp)


Sur la page de votre tableau de bord, cliquez sur **Commencez** dans le cadran **Point de Vente**. Vous serez redirigé sur l'interface de configuration de votre point de vente. Remplissez les informations correspondantes à votre point de vente et associez-y le portefeuille que vous avez précédemment connecté puis soumettez vos données.

![pdv](assets/fr/09.webp)


Avant d'aller sur cette interface pour commencer à encaisser des paiements en bitcoin, nous vous recommandons, pour une meilleure traçabilité, de créer des produits disponibles dans votre magasin afin d'avoir un inventaire sur Flash.

Sur le tableau de bord, cliquez sur l'icône **Plus**, puis cliquez sur **Nouveau produit**.

![products](assets/fr/10.webp)

Remplissez les différentes informations concernant votre produit puis reliez ce produit à votre point de vente.

![link_product](assets/fr/11.webp)

Vous pouvez accéder à votre menu Point de Vente en cliquant sur le nom de votre point de vente dans la section **Services actifs**.

![active_service](assets/fr/12.webp)

Copiez le lien de l'interface de votre point de vente puis accédez-y. Dans la section **Produits**, vous retrouverez la liste de tous les produits que vous avez associé à ce point de vente. Vous pouvez alors générer des factures bitcoin (Lightning) afin de faire payer vos clients.

![pos1](assets/fr/13.webp)

### Les liens de paiements

Flash vous permet d'accepter des paiements de partout dans le monde via bitcoin non seulement via un point de vente mais également via un service de lien de paiement. Vous pouvez générer des liens de paiement en étant un freelance, un promoteur de projet ou peu importe votre statut. Ces liens sont faciles à partager avec votre destinataire et vous permettent de recevoir des bitcoins directement dans votre portefeuille self-custody et ceci, sans intermédiaire.

Sur le tableau de bord Flash, cliquez sur **Lien de paiement** puis remplissez le formulaire pour générer votre lien de paiement.

### Les souscriptions

Les souscriptions représentent un service idéal lorsque vous avez des abonnés ou que vous souhaitez recevoir de façon récurrente, un certain montant.

Dans le menu **Souscriptions**, cliquez sur l'option **Nouveau plan de souscription**. Définissez les différents plans que vous souhaitez mettre en place puis configurez les paramètres de votre souscription.

### Créer sa boutique

Vous êtes un nouveau marchand ou vous n'avez pas de site web sur lequel intégrer Flash ? La plateforme vous propose une option de construction de site qui vous permet de créer des boutiques sur Flash et de vendre vos produits tout en acceptant les paiements en bitcoin.

Dans le menu, accédez à l'interface de l'option **Boutique** puis cliquez sur **Nouvelle boutique**.

Configurez les détails de votre boutique en y renseignant un nom unique visible uniquement par vous, un nom public pour vos visiteurs, la description de votre boutique et les visuels d'identité de votre marque (logo, affiche de couverture). Sélectionnez la devise dans laquelle vous souhaitez configurer les produits sur votre site web. Les prix de ces produits seront convertis en bitcoin suivant le taux de conversion courant sur le marché et connectez le portefeuille bitcoin sur lequel vous souhaitez recevoir les paiements.
