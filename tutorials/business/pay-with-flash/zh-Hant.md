由於 Lightning Network 所提供的可能性，Bitcoin 已在全球廣泛採用。目前已有多種解決方案可讓您成為這股採用浪潮的一員，並整合 Bitcoin 作為一種付款方式，讓您的服務、您的業務邁向全球，從世界任何地方接收即時付款。在這種方式中，Flash 這種自我保管解決方案因其功能性和使用者體驗而脫穎而出。



## 開始使用 Flash



Flash 是一個平台，可讓身為商家或企業家的您，根據 Bitcoin 通訊協定的 Lightning 疊層收取付款。這個平台相當直覺，一開始就會帶您完成建立組織的步驟。



### 建立您的組織



利用 Flash 的簡易性，只需三個步驟即可建立您的組織。




- 填寫您組織的詳細資料：


Flash 平台接受來自各大洲的企業，包括非洲 (奈及利亞、加納)、歐洲、美洲和亞洲 (越南、中國)。



![organizations](assets/fr/01.webp)





- 平台上不需要任何身份證明。只需輸入您的名字和姓氏，然後輸入您希望從哪個國家進行操作，然後繼續流程。



![infos](assets/fr/02.webp)





- 一種自我託管服務：雖然 Flash 是一個現金收付平台，但它給您自由和控制您的資金。您可以選擇連接您最常用的 Lightning Wallet，或者連接 Flash 開發的專用非保管 Wallet。



![connect_wallet](assets/fr/03.webp)




視您使用的 Lightning Wallet 而定，您會發現下列選項：





- Alby Hub** 適用於 Alby Go 錢包，是瀏覽器的 Alby 延伸功能。



https://planb.network/tutorials/wallet/mobile/alby-go-40202802-b346-4a3c-9863-465c3bde9903



- Coinos** 連接 Bitcoin 網路 CoinOS Wallet。
- Flash Wallet** 用於連接 Flash Wallet。
- LN 連結**



![connection_type](assets/fr/04.webp)



在本教程中，我們將使用 Blitz Wallet 搭配 Nostr Wallet Connect 連線，將 Wallet 連接到 Flash 上的組織。



https://planb.network/tutorials/wallet/mobile/blitz-wallet-794bdac4-1af4-49d5-9ea5-abb8228ca196

![wallet_name](assets/fr/05.webp)



![connected](assets/fr/06.webp)



連接 Bitcoin Wallet 後，請指定您感興趣的功能，以及您想在平台上使用的功能。這些功能包括




- 訂閱管理：最適合管理訂閱者的訂閱，在特定期間內收取固定金額。
- 付款連結至 generate 頁面，讓您的客戶可以支付您的發票。
- 捐款支持您的專案。
- 電子商務，可讓您在線上配置並銷售實體和數位產品。
- 銷售點提供您管理店內付款的介面。



![about](assets/fr/07.webp)



完成這些初始設定後，您就可以在儀表板右上方的 Coin 檢查您的帳戶是否正常運作。



### 建立銷售點



您是否已經在當地擁有實體商店，並希望接受 Bitcoin 付款？Flash 的 ** 銷售點** 選項是最適合您的解決方案。



當您建立銷售點時，會給您一個網路連結，這是您在銷售點中執行動作的 Interface。



![pos-link](assets/fr/08.webp)




在您的儀表板頁面，按一下**銷售點**象限中的**Start**。您將被重新導向至 Interface，以設定您的銷售點。填寫與您的銷售點相對應的資訊，並將其與您先前連接的組合相聯繫，然後傳送您的資料。



![pdv](assets/fr/09.webp)




在您前往此 Interface 開始在 Bitcoin 中收取付款之前，我們建議您在商店中建立可用的產品，以獲得更好的可追蹤性，以便在 Flash 上建立庫存。



在儀表板上，按一下 **More** 圖示，然後按一下 **New product**。



![products](assets/fr/10.webp)



填寫您的產品資訊，並連結到您的銷售點。



![link_product](assets/fr/11.webp)



您可以按一下 **Active services** 區段中的銷售點名稱，存取您的銷售點功能表。



![active_service](assets/fr/12.webp)



複製您銷售點的 Interface 連結並存取。在**產品**部分，您會找到與此銷售點相關聯的所有產品清單。然後您就可以 generate Bitcoin (Lightning) 發票來向您的客戶收費。



![pos1](assets/fr/13.webp)



### 付款連結



Flash 讓您不僅可以透過 Bitcoin 銷售點，也可以透過付款連結服務，接受來自世界任何地方的付款。您可以以自由工作者、專案開發者或任何身份使用 generate 付款連結。這些連結很容易與您的收款人分享，讓您可以直接將比特幣收到您的自我保管 Wallet，不需要任何中介。



在 Flash 面板上，按一下 ** 付款連結**，然後填寫表單以 generate 您的付款連結。



![payment_link](assets/fr/14.webp)



![checkouts](assets/fr/15.webp)



![link_created](assets/fr/16.webp)



### 捐款



在 **Checkouts** 功能表中，您可以建立捐款外掛程式，放在您的網站上或社群網路中，讓人們可以支持您的專案。



按一下 ** 新捐款外掛程式** 選項，然後設定您的詳細資訊



![donation](assets/fr/17.webp)



一旦您建立了捐款外掛程式，您就會得到一個 javascript 程式碼，您可以將它整合到您的網站上，開始接受捐款。




![dons](assets/fr/18.webp)



### 訂閱



當您有訂閱者或希望定期收到一定金額時，訂閱是一項理想的服務。



在 **Subscriptions** 功能表中，按一下 **New subscription plan**。定義您想要設定的不同計劃，然後設定您的訂閱參數。



![souscription](assets/fr/19.webp)



![plan](assets/fr/20.webp)



### 整合



Flash 不只是一個封閉的生態系統：感謝**Integrations**選單，您可以將您的業務連結到您已經在使用的工具。根據說明文件，您可以將您的 Flash 組織與 .NET 和 Java 整合：




- WooCommerce
- Shopify
- Wix
- Vimeo OTT (即將推出)



![integrations](assets/fr/21.webp)



### 建立您的商店



您是新的商家，或是沒有可以整合 Flash 的網站？這個平台提供您一個建站選項，讓您可以建立 Flash 商店並銷售產品，同時接受 Bitcoin 付款。



在功能表中，前往 Interface 的 ** 商店**選項，然後按一下 ** 新建商店**。



設定您商店的詳細資訊，輸入僅對您可見的唯一名稱、對您的訪客公開的名稱、您的商店描述和您的品牌視覺識別（標誌、封面海報）。選擇您希望在網站上設定產品的貨幣。這些產品的價格將使用當前的市場換算率換算為 Bitcoin，並連接您希望收取付款的 Bitcoin Wallet。
