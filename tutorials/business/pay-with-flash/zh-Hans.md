得益于 Lightning Network 提供的可能性，Bitcoin 已在全球范围内得到广泛采用。现在有几种解决方案可以让您成为这股应用浪潮的一部分，并将 Bitcoin 整合为一种支付手段，让您的服务、您的业务遍及全球，从世界任何地方接收即时付款。在这种方法中，Flash 作为一种自我保管解决方案，因其功能性和用户体验而脱颖而出。



## 开始使用 Flash



Flash 是一个平台，可以让商家或企业家基于 Bitcoin 协议的闪电覆盖层进行收款。该平台相当直观，从一开始就会引导您完成建立组织的相关步骤。



### 建立你的组织



利用 Flash 的简便性，只需三个步骤即可建立您的组织。




- 填写贵组织的详细信息：


Flash 平台接受来自各大洲的企业，包括非洲（尼日利亚、加纳）、欧洲、美洲和亚洲（越南、中国）。



![organizations](assets/fr/01.webp)





- 在平台上无需身份验证。只需输入您的名字和姓氏，然后输入您希望在哪个国家开展业务，然后继续操作即可。



![infos](assets/fr/02.webp)





- 自我保管服务：虽然 Flash 是一个现金平台，但它为您提供了对资金的自由和控制。您可以选择连接您最常用的 Lightning Wallet，或连接 Flash 开发的专用非托管 Wallet。



![connect_wallet](assets/fr/03.webp)

- Coinos** 连接 Bitcoin 网络 CoinOS Wallet。

连接好 Bitcoin Wallet 后，请指定您感兴趣并希望在平台上使用的功能。这些功能包括




- 订阅管理：通过在一定时期内收取固定金额来管理用户订阅的理想工具。
- 指向 generate 页面的付款链接，客户可在此支付发票。
- 捐款支持您的项目。
- 电子商务，让您可以在线配置和销售实体产品和数字产品。

完成这些初始配置后，您就可以在仪表板右上方的 Coin 检查账户是否已启动并运行。



### 创建销售点



您是否已经在当地拥有一家实体店，并希望接受 Bitcoin 支付？Flash 的**销售点**选项是最适合您的解决方案。



当您创建销售点时，您会得到一个网络链接，该链接是您在销售点中执行操作时使用的 Interface。



![pos-link](assets/fr/08.webp)




在仪表板页面上，单击**销售点**象限中的**开始**。您将被重定向到用于配置销售点的 Interface。填写与您的销售点相对应的信息，并将其与您之前连接的投资组合相关联，然后提交您的数据。



![pdv](assets/fr/09.webp)




在使用 Interface 开始在 Bitcoin 中收款之前，我们建议您在商店中创建可用产品，以便更好地跟踪，从而在 Flash 上建立库存。



在仪表板上，点击**更多**图标，然后点击**新产品**。



![products](assets/fr/10.webp)



填写产品信息并将其链接到销售点。



![link_product](assets/fr/11.webp)



点击**活动服务**部分的销售点名称，即可进入销售点菜单。



![active_service](assets/fr/12.webp)



复制销售点的 Interface 链接并进入。在**产品**部分，您会看到与该销售点相关联的所有产品列表。然后，您就可以使用 generate Bitcoin（闪电）发票向客户收费。



![pos1](assets/fr/13.webp)



### 付款链接



Flash 使您不仅可以通过 Bitcoin 销售点，还可以通过付款链接服务接受来自世界任何地方的付款。您可以以自由职业者、项目开发者或任何身份使用 generate 支付链接。这些链接很容易与收款人分享，并能让您直接将比特币接收到自我保管的 Wallet 中，无需任何中介。



在 Flash 仪表板上，点击 **付款链接**，然后填写表格以获得 generate 付款链接。



![payment_link](assets/fr/14.webp)



![checkouts](assets/fr/15.webp)



![link_created](assets/fr/16.webp)



### 捐赠



在 "**结账**"菜单中，您可以创建捐赠插件，将其放在网站或社交网络上，以便人们支持您的项目。



点击 "**新捐款插件**"选项，然后配置您的捐款插件的详细信息。



![donation](assets/fr/17.webp)



创建捐赠插件后，您将获得一个 javascript 代码，将其整合到您的网站中即可开始接收捐赠。




![dons](assets/fr/18.webp)



### 订阅



如果您有订阅者或希望定期收到一定数额的邮件，订阅是一种理想的服务。



在**订阅**菜单中，点击**新订阅计划**。定义要设置的不同计划，然后配置订阅参数。



![souscription](assets/fr/19.webp)

### 集成



Flash 不仅仅是一个封闭的生态系统：借助**集成**菜单，您可以将您的业务与您已经在使用的工具连接起来。根据文档，您可以将您的 Flash 组织与 .NET Framework 3.0 集成：




- WooCommerce
- Shopify
- Wix
- Vimeo OTT（即将推出）



![integrations](assets/fr/21.webp)



### 创建您的商店



您是新商户，还是没有可整合 Flash 的网站？该平台为您提供了一个建站选项，让您可以创建 Flash 商店并销售产品，同时接受 Bitcoin 付款。



在菜单中，转到 Interface 的**商店**选项，然后点击**新商店**。



通过输入仅对您可见的唯一名称、供访客使用的公开名称、商店描述和品牌识别视觉效果（徽标、封面海报）来配置商店的详细信息。选择您希望在网站上设置产品所使用的货币。这些产品的价格将按照当前的市场换算率换算成 Bitcoin，并连接您希望接收付款的 Bitcoin Wallet。
