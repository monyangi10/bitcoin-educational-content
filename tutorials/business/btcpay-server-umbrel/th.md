ในระบบนิเวศ Bitcoin การยอมรับการชำระเงินถือเป็นความท้าทายสำคัญสำหรับพ่อค้าและธุรกิจต่างๆ โซลูชันแบบดั้งเดิม ไม่ว่าจะเป็นธนาคาร (บัตรเครดิต, Stripe, PayPal) หรือแม้แต่ Bitcoin (BitPay, Coinbase Commerce) มักมีตัวกลางที่เรียกเก็บค่าธรรมเนียมสูง เก็บข้อมูลธุรกิจที่ละเอียดอ่อนของคุณ และสามารถบล็อกหรือเซ็นเซอร์ธุรกรรมของคุณตามอำเภอใจ การพึ่งพานี้ขัดแย้งกับหลักการพื้นฐานของ Bitcoin ที่เน้นการกระจายอำนาจ ความลับ และอธิปไตยทางการเงิน



BTCPay Server กำลังกลายเป็นคำตอบแบบโอเพ่นซอร์สสำหรับปัญหานี้ ตัวประมวลผลการชำระเงินที่โฮสต์ด้วยตัวเองนี้เปลี่ยนโหนด Bitcoin ของคุณให้กลายเป็นโครงสร้างพื้นฐานระดับมืออาชีพ โดยไม่มีคนกลาง ไม่มีค่าธรรมเนียมการประมวลผลเพิ่มเติม และไม่มีการประนีประนอมด้านความเป็นส่วนตัว พัฒนาโดยชุมชนผู้ร่วมให้ข้อมูลทั่วโลกตั้งแต่ปี 2017 BTCPay Server ช่วยให้คุณรับการชำระเงิน Bitcoin และ Lightning ได้โดยตรงเข้าสู่กระเป๋าเงินของคุณ โดยคงการควบคุมเงินของคุณอย่างเต็มที่ตลอดเวลา



ตามธรรมเนียมแล้ว การติดตั้ง BTCPay Server ต้องการทักษะทางเทคนิคขั้นสูง: การกำหนดค่าของเซิร์ฟเวอร์ Linux, ความชำนาญใน Docker, การจัดการใบรับรอง SSL และความปลอดภัยของเครือข่าย Umbrel ปฏิวัติวิธีการนี้ด้วยการติดตั้งเพียงคลิกเดียวที่รวมเข้ากับ Bitcoin และโหนด Lightning ของคุณโดยตรง การทำให้เรียบง่ายนี้ทำให้สิ่งที่เคยสงวนไว้สำหรับช่างเทคนิคที่มีประสบการณ์สามารถเข้าถึงได้สำหรับทุกคน



**สิ่งสำคัญที่ต้องเข้าใจ**: BTCPay Server บน Umbrel ทำงานโดยค่าเริ่มต้นเฉพาะในเครือข่ายภายในของคุณเท่านั้น คุณสามารถสร้างใบแจ้งหนี้ รับชำระเงินผ่าน Lightning และ Bitcoin และจัดการบัญชีของคุณจากอุปกรณ์ใดๆ ที่เชื่อมต่อกับเครือข่ายภายในบ้านของคุณ (คอมพิวเตอร์, สมาร์ทโฟน, แท็บเล็ต) การตั้งค่านี้เหมาะสำหรับการเรียกเก็บเงินสำหรับบริการที่ให้บริการแบบตัวต่อตัว จัดการการชำระเงินแบบพบหน้า หรือใช้ BTCPay Server จากเครือข่ายภายในของคุณ ในทางกลับกัน หากต้องการรวม BTCPay Server เข้ากับร้านค้าออนไลน์ที่สามารถเข้าถึงได้สาธารณะบนอินเทอร์เน็ต จะต้องมีการตั้งค่าเพิ่มเติมเพื่อเปิดเผยสู่สาธารณะ (เราจะครอบคลุมประเด็นนี้ในตอนท้ายของบทแนะนำ)



บทแนะนำนี้จะพาคุณผ่านการติดตั้ง BTCPay Server บน Umbrel อย่างสมบูรณ์ การกำหนดค่า Bitcoin wallet และโหนด Lightning ของคุณ การสร้างและชำระเงินใบแจ้งหนี้ และการจัดการรายงานทางบัญชี คุณจะได้เรียนรู้วิธีใช้ BTCPay Server อย่างมีประสิทธิภาพบนเครือข่ายท้องถิ่นของคุณ จากนั้นเราจะพิจารณาวิธีแก้ปัญหาสำหรับการแสดงผลสาธารณะหากคุณต้องการรวมเข้ากับเว็บไซต์อีคอมเมิร์ซ



## ข้อกำหนดเบื้องต้น



ในการติดตามบทแนะนำนี้ คุณจำเป็นต้องติดตั้งและกำหนดค่า Umbrel อย่างถูกต้อง หากคุณยังไม่ได้ดำเนินการดังกล่าว โปรดดูบทแนะนำของเราเกี่ยวกับการติดตั้ง Umbrel



https://planb.academy/tutorials/node/bitcoin/umbrel-8b0e3b5b-d3cf-4a1e-8bb8-1ad2db4dd848

โหนด Bitcoin Core ของคุณต้องซิงโครไนซ์กับบล็อกเชนอย่างสมบูรณ์ (100% ในแอปพลิเคชัน Bitcoin ของ Umbrel) การซิงโครไนซ์เริ่มต้นนี้มักใช้เวลาระหว่าง 3 วันถึง 2 สัปดาห์ ขึ้นอยู่กับฮาร์ดแวร์และการเชื่อมต่ออินเทอร์เน็ตของคุณ



ในการยอมรับการชำระเงินแบบทันทีด้วย Lightning คุณจะต้องติดตั้ง LND (Lightning Network Daemon) บน Umbrel ด้วย ดูบทแนะนำของเราเกี่ยวกับการติดตั้งและการกำหนดค่า LND บน Umbrel หากคุณต้องการเปิดใช้งานฟีเจอร์นี้



https://planb.academy/tutorials/node/lightning-network/umbrel-lnd-b12e0b5b-12ff-45f1-978e-62f4b4a8ba16

อนุญาตให้มีพื้นที่ว่างบนดิสก์อย่างน้อย 50 GB สำหรับ BTCPay Server, ฐานข้อมูลและข้อมูล Lightning การเชื่อมต่ออินเทอร์เน็ตที่เสถียรผ่านสาย Ethernet ขอแนะนำอย่างยิ่งเพื่อหลีกเลี่ยงการตัดการเชื่อมต่อ

จากอินเทอร์เฟซ Umbrel (`umbrel.local`) เข้าถึง App Store และค้นหา "BTCPay Server" ในหมวดหมู่ Bitcoin

คลิกติดตั้ง Umbrel จะตรวจสอบโดยอัตโนมัติว่าได้ติดตั้ง Bitcoin Core และ LND แล้ว จากนั้นเริ่มการปรับใช้ (2-5 นาที)



![Dépendances requises pour BTCPay Server](assets/fr/02.webp)



เมื่อติดตั้งแล้ว ให้เปิดแอปพลิเคชัน คุณจะต้องสร้างบัญชีผู้ดูแลระบบด้วยข้อมูลประจำตัวที่แข็งแกร่ง



![Création du compte administrateur BTCPay Server](assets/fr/03.webp)



เมื่อสร้างบัญชีของคุณเสร็จสิ้น BTCPay Server จะขอให้คุณตั้งค่าร้านค้าของคุณทันที เลือกชื่อธุรกิจและเลือกสกุลเงินอ้างอิง (EUR, USD หรือ BTC)



![Création du premier magasin BTCPay Server](assets/fr/04.webp)



## เข้าถึง BTCPay Server บนเครือข่ายท้องถิ่นของคุณ



BTCPay Server สามารถเข้าถึงได้จากอุปกรณ์ใด ๆ บนเครือข่ายท้องถิ่นของคุณ (WiFi หรือ Ethernet) เข้าถึงจากเบราว์เซอร์ของคุณที่ :



```url
http://umbrel.local
```



หรือส่งตรงไปที่ :



```url
http://umbrel.local:3003
```



**การเข้าถึงระยะไกลด้วย Tailscale**: เพื่อเข้าถึง BTCPay Server จากทุกที่ในโลก ใช้ Tailscale VPN ที่ปลอดภัยนี้ช่วยให้คุณเชื่อมต่อกับ Umbrel ของคุณราวกับว่าคุณอยู่ในเครือข่ายท้องถิ่นของคุณ ดูบทแนะนำของเราที่อุทิศให้กับ Tailscale บน Umbrel

## กำลังตั้งค่า Bitcoin wallet ของคุณ



ในการรับชำระเงิน คุณจำเป็นต้องกำหนดค่า Bitcoin wallet BTCPay Server จะแสดงตัวเลือกการกำหนดค่าในแดชบอร์ด



![Tableau de bord avec options de configuration de portefeuille](assets/fr/05.webp)



ในการกำหนดค่า wallet Bitcoin ให้ไปที่ "Wallets" > "Bitcoin"



คุณมีสองตัวเลือก: สร้าง wallet ใหม่โดยตรงใน BTCPay หรือ นำเข้า wallet ที่มีอยู่แล้ว สำหรับการนำเข้า มีหลายวิธีที่สามารถใช้ได้:




- เชื่อมต่อฮาร์ดแวร์ wallet** (แนะนำ): นำเข้าคีย์สาธารณะของคุณผ่านแอปพลิเคชัน Vault
- นำเข้าไฟล์ wallet** (แนะนำ): อัปโหลดไฟล์ที่ส่งออกจาก wallet ของคุณ
- ป้อนคีย์สาธารณะที่ขยาย**: ป้อน XPub/YPub/ZPub ของคุณด้วยตนเอง
- สแกนรหัส QR wallet** : สแกนรหัส QR จาก BlueWallet, Cobo Vault, Passport หรือ Specter DIY
- ป้อน wallet seed** (ไม่แนะนำ) : ป้อนวลีการกู้คืน 12 หรือ 24 คำของคุณ



![Options de création de portefeuille](assets/fr/06.webp)



สำหรับบทแนะนำนี้ เราจะสร้าง Hot wallet ใหม่: ดังนั้นคีย์ส่วนตัวจะถูกเก็บไว้บนเซิร์ฟเวอร์ Umbrel ของเรา ในกรณีนี้ เราขอแนะนำอย่างยิ่งให้คุณย้ายเงินทุนไปยัง wallet แบบเย็นเป็นประจำเพื่อหลีกเลี่ยงการเก็บเงินจำนวนมากบนเซิร์ฟเวอร์



![Choix entre Hot wallet et Watch-only wallet](assets/fr/07.webp)



เมื่อกำหนดค่าเสร็จสิ้น BTCPay Server ยืนยันว่า wallet ของคุณพร้อมที่จะรับการชำระเงิน on-chain แล้ว



![Portefeuille Bitcoin configuré avec succès](assets/fr/08.webp)



## เปิดใช้งาน Lightning Network



หากต้องการยอมรับการชำระเงิน Lightning ทันที ให้ไปที่ กระเป๋าเงิน > Lightning จากนั้น เนื่องจากโหนด LND ของคุณได้ติดตั้งอยู่บน Umbrel แล้ว เพียงคลิกที่ปุ่ม "บันทึก" เพื่อยืนยันการเชื่อมต่อระหว่าง BTCPay Server ของคุณและโหนด Lightning ของคุณ



![Configuration du nœud Lightning](assets/fr/09.webp)



## สร้างและชำระใบแจ้งหนี้



ในอินเทอร์เฟซของ BTCPay Server ไปที่ ใบแจ้งหนี้ > สร้าง Invoice ป้อนจำนวนเงิน เพิ่มคำอธิบายเพิ่มเติม (ถ้ามี) แล้วคลิก สร้าง



![Création d'une nouvelle facture](assets/fr/10.webp)



จากนั้นคุณสามารถคลิกที่ปุ่ม "Checkout" เพื่อแสดงใบแจ้งหนี้ BTCPay จะสร้างใบแจ้งหนี้พร้อมรหัส QR แบบรวม (BIP21) ที่มีที่อยู่ Bitcoin และใบแจ้งหนี้ Lightning



![Détails de la facture générée](assets/fr/11.webp)



ลูกค้าของคุณสามารถสแกน QR โค้ดด้วย wallet ที่รองรับได้



![Page de paiement avec QR code](assets/fr/12.webp)



เมื่อชำระเงินแล้ว ใบแจ้งหนี้จะกลายเป็น "ชำระแล้ว" ในเวลาเพียงไม่กี่วินาทีสำหรับ Lightning.



![Confirmation de paiement réussi](assets/fr/13.webp)

## การกำหนดค่าร้านค้า



BTCPay Server ช่วยให้คุณจัดการร้านค้าหลายแห่งด้วยพารามิเตอร์ที่แตกต่างกัน แต่ละร้านค้าเป็นตัวแทนของหน่วยธุรกิจที่แยกจากกัน: ร้านค้าอีคอมเมิร์ซ, จุดขายทางกายภาพ, หรือการเรียกเก็บค่าบริการ



ในการตั้งค่าร้านค้า คุณจะพบกับหลายส่วนสำคัญ:



![Paramètres du magasin](assets/fr/15.webp)





- การตั้งค่าทั่วไป**: ชื่อร้านค้า, สกุลเงินอ้างอิง (BTC, EUR, USD), เวลาหมดอายุของใบแจ้งหนี้ (ค่าเริ่มต้น 15 นาที), จำนวนการยืนยันบล็อกเชนที่ต้องการ
- อัตรา**: การกำหนดค่าแหล่งที่มาของอัตราแลกเปลี่ยนและการแปลงสกุลเงินเฟียต/Bitcoin
- ลักษณะการชำระเงิน**: ปรับแต่งลักษณะของหน้าชำระเงินของคุณ (โลโก้, สี, ข้อความส่วนตัว)
- การตั้งค่าอีเมล**: การกำหนดค่าการแจ้งเตือนทางอีเมลสำหรับการชำระเงินที่ได้รับ
- โทเค็นการเข้าถึง**: การจัดการโทเค็น API สำหรับการผสานรวมอีคอมเมิร์ซ (WooCommerce, Shopify, ฯลฯ)
- ผู้ใช้**: จัดการการเข้าถึงของผู้ใช้ในร้านค้าด้วยระดับการอนุญาตที่แตกต่างกัน (เจ้าของ, ผู้เยี่ยมชม)
- Webhooks**: การกำหนดค่า Webhook สำหรับการซิงโครไนซ์แบบเรียลไทม์กับระบบบัญชีหรือ ERP ของคุณ



BTCPay Server ยังมีส่วนของปลั๊กอินเพื่อขยายฟังก์ชันการทำงานด้วยการผสานรวมอีคอมเมิร์ซ ระบบจุดขาย และเครื่องมือเพิ่มเติม



![Gestion des plugins](assets/fr/16.webp)



## ข้อดีและข้อจำกัดของการใช้งานในท้องถิ่น



**ข้อดีของ BTCPay Server เหนือกว่า Umbrel** :




- อธิปไตยทั้งหมด: การควบคุมกุญแจส่วนตัวและเงินทุนอย่างเต็มที่ ไม่มีบุคคลที่สามสามารถระงับหรือเซ็นเซอร์การชำระเงินของคุณได้
- ประหยัดได้มาก: เฉพาะค่าใช้จ่ายเครือข่าย Bitcoin (ไม่กี่เซ็นต์บน Lightning) เทียบกับ 2-3% บนตัวประมวลผลแบบดั้งเดิม
- ความลับสูงสุด: ไม่มีการลงทะเบียน การยืนยันตัวตน หรือการแบ่งปันข้อมูลกับบริษัทบุคคลที่สาม
- สถาปัตยกรรมโอเพ่นซอร์สรับประกันความโปร่งใส ความสามารถในการตรวจสอบ และความยั่งยืนผ่านชุมชนนักพัฒนาขนาดใหญ่
- การติดตั้งง่ายผ่าน Umbrel โดยไม่จำเป็นต้องมีทักษะทางเทคนิคขั้นสูง



**ข้อจำกัดที่สำคัญ** :




- เฉพาะเครือข่ายภายในเท่านั้น**: BTCPay Server บน Umbrel สามารถเข้าถึงได้เฉพาะจากเครือข่ายภายในบ้านของคุณ เหมาะสำหรับการเรียกเก็บเงินแบบพบหน้า, บริการฟรีแลนซ์ หรือธุรกิจขนาดเล็กที่มีหน้าร้านจริง แต่ไม่เหมาะสำหรับร้านค้าออนไลน์ที่เข้าถึงได้สาธารณะ
- ความรับผิดชอบทางเทคนิคทั้งหมด: การบำรุงรักษาโหนด, การสำรองข้อมูลเป็นประจำ, การตรวจสอบการเชื่อมต่อ
- การจัดการสภาพคล่องของ Lightning: การเปิดและจัดการช่องทางด้วยความจุขาเข้าที่เพียงพอ
- การสนับสนุนจำกัดเฉพาะเอกสารชุมชนและฟอรัม ซึ่งต้องการความเป็นอิสระมากกว่าฝ่ายบริการลูกค้าเชิงพาณิชย์



ข้อจำกัดของเครือข่ายท้องถิ่นนี้เป็นอุปสรรคหลักในการรวม BTCPay Server เข้ากับร้านค้าอีคอมเมิร์ซ ซึ่งลูกค้าจำเป็นต้องสามารถเข้าถึงหน้าการชำระเงินจากที่ใดก็ได้บนอินเทอร์เน็ต



## แนวทางปฏิบัติที่ดีที่สุดและความปลอดภัย



เปิดใช้งานการสำรองข้อมูลอัตโนมัติ Umbrel และจัดเก็บสำเนาไว้ในสื่อภายนอก (USB stick, hard disk, encrypted cloud) เก็บเมล็ดพันธุ์ Bitcoin ของคุณ (วลีการกู้คืน) ไว้ในที่ปลอดภัย แยกออกจากกันทางกายภาพ สำรองไฟล์ channel.backup ของ LND สำหรับการกู้คืน Lightning



ตรวจสอบการซิงโครไนซ์ Bitcoin Core, ช่องทาง Lightning และการตอบสนองของ BTCPay Server อย่างสม่ำเสมอ การทดสอบง่ายๆ รายสัปดาห์: generate และชำระบิลด้วย satoshis ไม่กี่รายการ รักษา Umbrel ให้ทันสมัย (แพตช์ความปลอดภัย, การปรับปรุง) ทำการสำรองข้อมูลก่อนการอัปเดตใหญ่ สำหรับการใช้งานระดับมืออาชีพ พิจารณาการตรวจสอบภายนอก (UptimeRobot) พร้อมการแจ้งเตือนทางอีเมล/SMS



## การเปิดเผย BTCPay Server สู่สาธารณะสำหรับร้านค้าออนไลน์



ในการผสานรวม BTCPay Server เข้ากับร้านค้าอีคอมเมิร์ซบนเว็บ (WooCommerce, Shopify, ฯลฯ) ลูกค้าของคุณจำเป็นต้องสามารถเข้าถึงหน้าการชำระเงินจากที่ใดก็ได้ ไม่ใช่แค่เครือข่ายภายในของคุณเท่านั้น



**โซลูชัน: Nginx Proxy Manager**



คุณสามารถเปิดเผย BTCPay Server สู่สาธารณะโดยใช้ Nginx Proxy Manager (มีให้ใน Umbrel App Store) โซลูชันนี้ต้องการ:




- ชื่อโดเมน (แบบคลาสสิกหรือฟรีผ่าน DuckDNS, No-IP, Afraid.org)
- กำหนดค่าการส่งต่อพอร์ต (พอร์ต 80 และ 443) บนเราเตอร์ของคุณ
- การติดตั้ง Nginx Proxy Manager ซึ่งจัดการใบรับรอง SSL โดยอัตโนมัติ



การกำหนดค่านี้ทำให้เซิร์ฟเวอร์ของคุณเปิดเผยต่ออินเทอร์เน็ตและต้องการความระมัดระวังเป็นพิเศษ (รหัสผ่านที่แข็งแกร่ง, 2FA, การอัปเดตเป็นประจำ) เราจะเตรียมบทแนะนำเฉพาะที่มีรายละเอียดขั้นตอนทั้งหมดนี้



## บทสรุป



BTCPay Server บน Umbrel ผสานพลังของโหนด Bitcoin เข้ากับความเรียบง่ายของ Umbrel เพื่อสร้างโครงสร้างพื้นฐานการชำระเงินแบบมืออาชีพที่โฮสต์ด้วยตนเองซึ่งทุกคนสามารถเข้าถึงได้ อธิปไตยทางการเงินนี้มาพร้อมกับความรับผิดชอบในการบำรุงรักษา แต่ Umbrel ช่วยลดภาระการดำเนินงานได้อย่างมากเมื่อเทียบกับประโยชน์ที่ได้รับ: การกำจัดค่าธรรมเนียมการประมวลผล การปกป้องความเป็นส่วนตัวของคุณ การต่อต้านการเซ็นเซอร์ และการควบคุมเงินทุนของคุณอย่างเต็มที่



การใช้งานเครือข่ายท้องถิ่นครอบคลุมการใช้งานที่หลากหลายอยู่แล้ว: การเรียกเก็บเงินสำหรับบริการฟรีแลนซ์, การชำระเงินแบบตัวต่อตัว, ร้านค้าขนาดเล็ก, หรือเพียงแค่การเรียนรู้และทดลองกับ Bitcoin และ Lightning ในสภาพแวดล้อมที่ควบคุมได้ สำหรับความต้องการอีคอมเมิร์ซที่ต้องการการเปิดเผยต่อสาธารณะ มีโซลูชัน Nginx Proxy Manager แต่ต้องการการกำหนดค่าทางเทคนิคเพิ่มเติม ซึ่งเราจะอธิบายในบทเรียนเฉพาะทาง



ไม่ว่าคุณจะดำเนินธุรกิจ โครงการที่เพิ่งเริ่มต้น หรือเพียงแค่ทดลองใช้ BTCPay Server บน Umbrel มอบอิสระทางการเงินอย่างสมบูรณ์ เส้นทางเริ่มต้นด้วยร้านค้าของคุณครั้งแรก ใบแจ้งหนี้ครั้งแรก การชำระเงินครั้งแรกที่ได้รับโดยตรงเข้าสู่โครงสร้างพื้นฐานอธิปไตยของคุณ
