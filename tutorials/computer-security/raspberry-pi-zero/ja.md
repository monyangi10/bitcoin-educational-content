## 必要条件


**1** [Raspberry PI Zero](https://www.raspberrypi.com/products/raspberry-pi-zero/): PI Zero（後続の記号なし）は、最小限の性能を持つコンピュータを構築するための基盤ですが、何よりも Wi-Fi と Bluetooth カードを欠いており、この演習の目的に不可欠な要件です。





- 費用**：このチュートリアルを書いている時点（2025年8月）で約15ドル。
- 生産の継続性**：ラズベリーは2030年1月まで生産を保証する。



Wi-FiとBluetoothを搭載していないPI Zeroは、残念ながらほとんど入手できなくなってしまった。PI Zero WとPI Zero 2Wの代替品をもっと簡単に見つけることができるかもしれません。この場合、設定ファイルを変更することで、接続機能を無効にすることができます。オペレーティングシステムをインストールした後、これらの項目をコンフィグに追加する必要があります：



```
dtoverlay=disable-wifi
dtoverlay=disable-bt
```



このガイドのセクションでは、その方法と場所を紹介します。しかし、本当に確実を期すのであれば、電子基板を加工するのに適した小型のカッターでWi-Fiチップを取り外すチュートリアルをウェブ上でいくつか見つけることができます。



**2** Raspberry PI Zero用のスターターキット：ラズベリーの世界では一般的なことだが、外付けのケースはない。さらに、このような小さなボードの限られたリソースが、外部との接続の可能性を制限している。



そこで、PI Zeroのポテンシャルをフルに活かすためのアクセサリーが満載の[このキット](https://www.amazon.it/-/en/GeeekPi-Raspberry-Aluminum-Passive-Heatsink/dp/B0BJ1WWHGF?crid=1NAFFVHG3IFBU&sprefix=raspberry+pi+zero+kit+geeek+pi%2Caps%2C88&sr=8-65)を見つけた。実際、キットにはUSB A -> micro USB電源Supply、小型USBハブ、mini-HDMI -> HDMIアダプター、銅製ヒートシンク、アルミ製アウターケースが含まれています。また、PI Zeroを新しいケースに入れるために必要なネジと六角レンチもキットに含まれている。





- コスト**： 19.99ユーロ



**3** このチュートリアルでは、エアギャップ・コンピューターに多額の予算をかける必要はありません。ただし、USBキーボードとマウス（有線に限る、Bluetoothは避けること）、モニターが必要であることは知っておく必要がある。モニターへの入力によっては、PI Zeroで利用可能な唯一の出力であるmini-HDMIからのアダプターが必要になるかもしれない。最後に、ワイヤレスではないキーボードとマウスが家のどこかにあるという事実にHardを見てほしい。

## オペレーティング・システムのインストール


PI Zeroをケースにしまう前に、OSをインストールすることをお勧めします。そうすれば、すぐに動作を示すLEDを確認することができます。



オペレーティング・システムを選んで焼くために、私は最も簡単な方法を選んだ。



![img](assets/it/01.webp)



[RaspberryのGithub](https://github.com/raspberrypi/rpi-imager/releases) にアクセスして、Imagerの最新リリースをダウンロードし、あなたのオペレーティングシステムに最も適したものを選択してください（執筆時点で v. 1.9.6）。各アセットの横に対応するファイルのハッシュがあることに気づくでしょう。これは検証に役立ちます。



![img](assets/it/02.webp)



個人的な安心のために、airgapコンピュータで使用するオペレーティングシステムを確認することをお勧めします。そうすることで、正規の（悪意のない）バージョンのイメージャー、ひいてはラズパイOSで操作しているという確信が得られます。



ダウンロード後すぐに、普段使用しているマシンをインターネットに接続した状態で検証を行う。



それからLinuxのターミナルを開き、ダウンロードの準備をし、作業するのに便利な`raspios`ディレクトリを作成する。



![img](assets/it/19.webp)



wget`を使用して、Linuxディストリビューション用のイメージャをダウンロードする。



![img](assets/it/20.webp)



最後に、`sha256sum`コマンドを実行し、HashとRaspberryがGithubで提供しているHashを比較する。



![img](assets/it/21.webp)



あるいは、ウィンドウズを使っている場合は、パワーシェルを開き、以下のコマンドを入力する：



```
Get-FileHash -Path <yourpath>\imager-1.9.6.exe
```



![img](assets/it/04.webp)



Hashは、ラズベリーのGithubで公開されているものと一致していなければなりません。



ダウンロードを確認したら、普段使っているコンピューターにImagerをインストールして開きます。Imagerは、PI Zeroの「システムディスク」となるマイクロSDにオペレーティングシステムを書き込むために必要なツールです。



手順は非常に簡単で、まず使用するラズベリー・デバイスを選択し（ラズパイゼロの**モデルに注意）、次にOSのバージョンを選択し、最後にOSをフラッシュするマイクロSDカードのマウントポイントを選択する。

## 最初の構成


ラズパイOSの最初の起動は、実際のインストール作業をいくつもこなさなければならないため、通常より少し遅い。しかし、すべてがうまくいっていれば、ようこそ画面が表示されます。



![img](assets/it/14.webp)



特に正しいキーボードを読み込むために、_Next_をクリックして地理的な地域を設定する。特に後者に注意してください。



![img](assets/it/15.webp)



次へ」をクリックすると、ユーザーを作成する画面が表示されます。



![img](assets/it/16.webp)



ウィザードは、デフォルトのウェブブラウザをChromiumとFirefoxから選択するよう尋ねます。また、Zero Wまたは2W PIで作業している場合は、Wi-Fiネットワークの設定について尋ねられることもあります。先に進み、_Skip_をクリックします。



ある時点で、オンボードソフトウェアとオペレーティングシステムをアップグレードするよう促されます。Skip_を選択してください。この演習の目的上、私たちはオフラインのマシンを構築しており、この時点ではオフラインのままでなければなりません。



最後に、`ssh`を有効にするよう尋ねてくるかもしれないが、ゼロエアギャップのIPなので、このステップも断ること。



![img](assets/it/17.webp)



これ以上設定することはありません。設定が終わったら、Raspberryを再起動して変更を有効にする。



![img](assets/it/18.webp)



## 新しいコンピュータの隙間


再起動後、新しいエアギャップコンピュータの準備が整いました。PI Zeroには新しいデスクトップが表示され、GUIまたはコマンドラインから操作できます。



![img](assets/it/22.webp)



### PIゼロWまたは2Wへの第一歩


Raspberry PI Zero Wまたは2Wシリーズをお使いの場合、ボードにWi-FiとBluetooth用のチップが搭載されています。最初のセットアップ時に接続設定をスキップしたため、PI Zeroはインターネットに接続されていません。ソフトウェアでこの2つのチップを永久に無効にすることができます。



実際、ラズパイOSには`config.txt`ファイルが用意されており、好きなように編集することができる。config`は`boot`パーティションの`firmware`ディレクトリにあり、`root`権限で編集・保存できる。



左上のアイコンからターミナルを開くと`root`になる(1)



![img](assets/it/23.webp)



(1) もし Raspi OS が前のステップで `root` パスワードを作らせなかったのなら、今 `passwd` コマンドで作ることをお勧めします。root` ユーザーを個人ユーザーから独立させておくと、リカバリーの際に非常に便利です。



ターミナルで`config.txt`ファイルを確認し、`nano`エディターで編集できるように準備しておく。



![img](assets/it/24.webp)



編集は`config`全体の一番下、`[All]`の後に行う。この時点で、チュートリアルの最初に示した`dtoverlay`コマンドを追加することになる。



![img](assets/it/25.webp)



保存して閉じ、再起動する。次のステップでは、小さなラズベリーを探索します。

## 用途

- まずメインメニューに行き、_Add/Remove software_パネルでインスピレーションを得てください。ターミナルからもできますが、常に `root` 権限が必要であることを覚えておいてください。

## 結論


このセットアップには、おそらく1つ大きな弱点がある。それは、マイクロSDが問題を引き起こす可能性のあるメディアであるということです。Zero airgap IPで使いたいソフトウェアをすべてインストールした後、手持ちのラズパイOSツールを使って、大切なマイクロSDのバックアップをしっかり取ってください。



![img](assets/it/34.webp)



ニーズが高まり、予算の可能性が広がれば、他のラズベリーや類似のソリューションを検討することができる。例えばメモリについて言えば、PI 5ではM2 NVMEドライブを組み立てることができる。



オフラインで使おうとしているユーティリティソフトのインストールの全ステップをメモし、文書化することをお忘れなく。 **遅かれ早かれ、ラズパイOSのアップデートが行われるでしょう。その時は、すべてを削除して、もう一度やり直す必要があります（おそらく新しいマイクロSDで😂）。



私たちが今行った練習は、あなたにとっても初めての実験であると仮定すれば、長く記憶に残ることだろう。時々電源を入れてチェックするために、エアギャップされたマシンへの注意を怠ることなく、オフラインで`組み込み`操作にハードウェアを割くことは必ずしも可能ではない。
