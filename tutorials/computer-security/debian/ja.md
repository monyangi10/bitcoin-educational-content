Debian はフリーな GNU/Linux ディストリビューションであり、 その堅牢性と信頼性で有名です。その Linux カーネルとすべてのパッケージは、揺るぎない安定性と高水準のセキュリ ティを保証するために厳しくテストされています。サーバにもワークステーションにも適しており、Debian は使いやすく、 膨大なソフトウェアカタログを提供しています。普段使いの安全なシステムを探しているのであれ、 生産環境を探しているのであれ、Debian は正しい選択です。



## なぜ Debian を選ぶのですか?





- フリーでオープン**：Debian は完全にオープンソースであり、透明性とライセンス料が保証されています。
- **安定性とセキュリティ**: 全てのリリースは徹底的なテストプロセスを経ており、 Debian は市場で最も信頼性が高く安全なディストリビューションの一つとなっています。
- **活発なコミュニティ**: 広範なコミュニティと広範なドキュメントが、必要なときにいつでもあなたをサポートします。
- **軽量でスケーラブル**: 良好なパフォーマンスを維持しながら、そこそこのリソースのマシンに Debian をインストールできます。
- **豊富なソフトウェアカタログ**：50,000以上の公式パッケージがリポジトリから入手可能。



## Interfaceのグラフィックを選ぶ



Debian は、あなたのニーズに合わせていくつかのデスクトップ環境を提供しています：





- **GNOME**: モダンで直感的なInterface、初心者に最適です。アプリケーションにアクセスするための流麗で使いやすいグラフィカル・メニューを提供します。
- **XFCE**：軽量で高速、性能の低いマシンに最適。
- **KDE Plasma**: Windowsライクな外観で、高度にカスタマイズ可能。
- **シナモン**：Windowsにインスパイアされた、シンプルでエレガントなInterface。
- **LXDE / LXQt**: 超軽量、古いコンピュータに適しています。
- **MATE**：シンプルでクラシック、昔のGNOMEに近い。



💡 快適で握りやすい体験のために、**GNOMEを強くお勧めします**。



## Debianのインストールと設定


### ハードウェア要件



インストールを開始する前に、以下の機器が揃っていることを確認してください：





- USBキー：ブート可能なISOイメージを保存するため、最低8GB。
- ランダム・アクセス・メモリー（RAM）：スムーズなインストールと操作のために4GB。
- **ディスク容量**: システムとアップデートのために最低15GBの空き容量が必要です。



### ダウンロード



Debian イメージの選択は、あなたのプロセッサアーキテクチャに依存します：





- **AMD64**: [download] リストから "live hybrid" エディションをダウンロードする (https://debian.obspm.fr/debian-cd/12.11.0-live/amd64/iso-hybrid/)。
- **ARM64**: [Debian] 公式ウェブサイト (https://debian.obspm.fr/debian-cd/12.11.0/arm64/iso-dvd/) から DVD イメージを入手してください。
- **その他のアーキテクチャ**: お使いのアーキテクチャに対応するISOを[こちら](https://debian.obspm.fr/debian-cd/12.11.0/)からお探しください。



![download](assets/fr/01.webp)



### ブータブルUSBキーの作成



適切なISOイメージをダウンロードしたら、インストールメディアの作成に進みます：




- **Balena Etcher**を[公式サイト](https://etcher.balena.io/)からダウンロードし、お使いのシステムに合ったバイナリを入手してインストールする。



![etcher](assets/fr/02.webp)





- **Etcher** を起動: ソフトウェアを開き、以前にダウンロードした Debian ISO イメージを選択します。
- USBキー**を選択**：ターゲットとしてキー（8GB以上）を指定します。
- Start flash**: **Flash!**をクリックし、プロセスが完了するまで待つ。



![flash](assets/fr/03.webp)



これであなたの USB キーは Debian のインストールを開始する準備ができました。



## Debian をシステムにインストールする



### USBキーからの起動



USBキーからインストールを開始するには




- コンピュータの電源を完全に切る。
- **再起動**を行い、すぐに `ESC`、`F2`、`F11`（またはブランドによっては専用キー）を押して BIOS/UEFI にアクセスする。
- ブートメニューで、ブートデバイスとしてUSBキーを選択します。
- Enter キーで**確認**して、Debian イメージを起動します。



### インストールの開始



スタート画面：



![starting](assets/fr/04.webp)



USB メモリから起動すると、Debian のウェルカムスクリーンにいくつかのオプションが表示されます：




- **ライブシステム**: Debian をインストールせずに起動します。
- **Start Installer**: Hard ディスクに直接インストールを開始します。
- **高度なインストールオプション**：カスタマイズされたインストールモードにアクセスできます。



ライブモードで Debian を調べるには、**Live System** を選択し、**Enter** で確定します。ライブ環境で **Install Debian** をクリックすると、インストールを開始できます。



![system](assets/fr/05.webp)





- 言語選択**（オプション）**



リストから Debian システムの主要言語を選択し、OK をクリックします。



![language](assets/fr/06.webp)





- **タイムゾーン** (GMT)



地理的なゾーンを選択すると、自動的に日付と時刻が設定されます。



![timezone](assets/fr/07.webp)





- キーボードレイアウト



キーボードの言語とレイアウトを選択します。内蔵のテストフィールドを使用して、各キーが期待される文字を生成することを確認します。



![keyboard](assets/fr/08.webp)



### ディスク・パーティショニング





- **ディスクの消去**：専用パーティションがある場合、このオプションはその内容をすべて消去します。
- **手動パーティション分割**：このオプションを選択すると、必要に応じてパーティションの作成、サイズ変更、削除ができます。



![disk](assets/fr/09.webp)





- ユーザーアカウントの作成



セッションの安全性を確保するため、フルネーム、アカウント名、強力なパスワードを入力してください。



![user](assets/fr/10.webp)





- **パラメータ概要**



各項目をチェックし、必要であれば戻って修正する。



![settings](assets/fr/11.webp)





- インストールの開始



インストール**をクリックしてファイルのコピーとシステムの設定を開始し、プロセスが完了するまで待ちます。**



![install](assets/fr/12.webp)





- リスタート



インストールが完了したら、コンピュータを再起動してすべての設定を適用し、新しい Debian システムにアクセスしてください。



![restart](assets/fr/13.webp)



起動時に、システムにアクセスするためのユーザー名とパスワードを入力します。



![login](assets/fr/14.webp)



## システム・アップデート



システムを使い始める前に、アップデートを行うことが重要です。これにより、最新のソフトウェア・パッチや最新のリポジトリ、場合によってはオペレーティング・システム自体のセキュリティ・パッチを利用することができます。



### オプション1: グラフィカルInterface (GNOME)経由でのアップデート



GNOME デスクトップ環境で Debian をインストールしている場合、グラフィカルに アップデートを実行できます。これを行うには、**ソフトウェア** アプリケーションを開き、**更新** タブを開きます。そして、**Restart and update** をクリックして処理を開始します。



### オプション2：ターミナル経由でのアップデート（推奨）



この方法は、より完全なコントロールを提供する。リポジトリ、ソフトウェアパッケージ、そして必要であればカーネルをアップデートすることができる。




- Ctrl + Alt + T`のショートカットを使ってターミナルを開く。
- 以下のコマンドで利用可能なパッケージのリストを更新する：



```shell
sudo apt update
```



プロンプトが表示されたらパスワードを入力してください（入力中に文字が表示されませんが、これは正常です）。





- 利用可能なアップデートをインストールするには：



```shell
sudo apt full-upgrade
```



## 基本タスクの発見



### インターネットの閲覧



Debian のデフォルトのウェブブラウザは **Firefox** です。他のブラウザを使いたい場合は、Debian のリポジトリで利用可能なもの (Chromium や Brave など) をインストールしてください。



### ワープロ



Debianには**LibreOffice**スイートがデフォルトでインストールされています。





- 文書を作成するには、**Microsoft Wordに相当する** **LibreOffice Writer** を使用します。
- 表計算に関しては、**LibreOffice Calc**がExcelの代わりとして機能する。
- 最後に、**LibreOffice Impress**では、PowerPointのようにプレゼンテーションを作成することができます。



## アプリケーションのインストール



Debianにアプリケーションをインストールするには、2つの方法があります：



### グラフィカルな方法：



ソフトウェアマネージャー（グラフィカルな**Interface**からアクセス可能）を使用して、簡単にアプリケーションを検索し、インストールすることができます。



### コマンドライン方式 ：



探しているアプリケーションがグラフィカルなInterfaceに表示されない場合、またはターミナルを好む場合は、次のコマンドを使用する：



```shell
sudo apt install <name>
```



<name>`をパッケージ名に置き換えてください。例えば、`curl` をインストールするには ：



```shell
sudo apt install curl
```



### 手動でダウンロードしたパッケージのインストール ：



.deb`ファイル（Debianパッケージ）をダウンロードした場合は、以下のコマンドでインストールできる：



```shell
sudo apt install ./name.deb
```

これで Debian システムはインストールされ、日々の作業に使えるようになり ました。
