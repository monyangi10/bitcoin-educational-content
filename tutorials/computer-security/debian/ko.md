Debian은 견고함과 안정성으로 유명한 무료 GNU/Linux 배포판입니다. Linux 커널과 모든 패키지는 엄격한 테스트를 거쳐 견고한 안정성과 높은 수준의 보안을 보장합니다. 서버와 워크스테이션 모두에 적합한 Debian은 사용하기 쉬운 환경과 방대한 소프트웨어 카탈로그를 제공합니다. 일상적인 사용을 위한 안전한 시스템을 찾고 있든 프로덕션 환경을 위한 시스템을 찾고 있든, Debian은 올바른 선택입니다.



## 왜 데비안을 선택해야 하나요?





- 무료 개방형**: Debian은 완전히 오픈 소스이며 투명성을 보장하고 라이선스 비용이 없습니다.
- 안정성 및 보안**: 모든 릴리스는 철저한 테스트 과정을 거치므로 Debian은 시중에서 가장 안정적이고 안전한 배포판 중 하나입니다.
- 활발한 커뮤니티**: 방대한 커뮤니티와 방대한 문서가 필요할 때마다 여러분을 지원합니다.
- 가볍고 확장 가능**: 리소스가 많지 않은 머신에도 우수한 성능을 유지하면서 Debian을 설치할 수 있습니다.
- 광범위한 소프트웨어 카탈로그**: 50,000개 이상의 공식 패키지가 리포지토리를 통해 제공됩니다.



## Interface 그래픽 선택



Debian은 사용자의 필요에 맞는 여러 데스크톱 환경을 제공합니다:





- GNOME**: 현대적이고 직관적인 Interface로 초보자에게 이상적입니다. 애플리케이션에 액세스하기 위한 유동적이고 사용하기 쉬운 그래픽 메뉴를 제공합니다.
- XFCE**: 가볍고 빠르며 성능이 낮은 머신에 적합합니다.
- KDE 플라즈마**: 고도로 사용자 정의할 수 있으며 Windows와 유사한 외관을 제공합니다.
- Cinnamon**: Windows에서 영감을 받은 심플하고 우아한 Interface.
- LXDE/LXQt**: 초경량으로 구형 컴퓨터에 적합합니다.
- MATE**: 단순하고 클래식하며 이전 GNOME에 가깝습니다.



💡 편안하고 잡기 쉬운 경험을 원한다면 **GNOME**을 적극 추천합니다**.



## Debian 설치 및 구성


### 하드웨어 요구 사항



설치를 시작하기 전에 다음 장비가 준비되어 있는지 확인하세요:





- USB 키**: 부팅 가능한 ISO 이미지를 저장하려면 최소 8GB가 필요합니다.
- RAM(랜덤 액세스 메모리)**: 원활한 설치 및 작동을 위한 4GB.
- 디스크 공간**: 시스템 및 업데이트를 위한 최소 15GB의 여유 공간이 필요합니다.



### 다운로드



Debian 이미지의 선택은 프로세서 아키텍처에 따라 다릅니다:





- AMD64**: [다운로드] 목록(https://debian.obspm.fr/debian-cd/12.11.0-live/amd64/iso-hybrid/)에서 "라이브 하이브리드" 에디션을 다운로드하세요.
- ARM64**: 공식 [Debian] 웹사이트(https://debian.obspm.fr/debian-cd/12.11.0/arm64/iso-dvd/)에서 DVD 이미지를 다운로드합니다.
- 기타 아키텍처**: 사용 중인 아키텍처에 해당하는 ISO를 [여기](https://debian.obspm.fr/debian-cd/12.11.0/)에서 찾아보세요.



![download](assets/fr/01.webp)



### 부팅 가능한 USB 키 만들기



적절한 ISO 이미지를 다운로드했으면 설치 미디어 생성을 진행합니다:




- 공식 웹사이트](https://etcher.balena.io/)에서 **발레나 에처**를 다운로드한 다음, 시스템에 맞는 바이너리를 받아 설치하세요.



![etcher](assets/fr/02.webp)





- Etcher** 실행: 소프트웨어를 열고 이전에 다운로드한 Debian ISO 이미지를 선택합니다.
- USB 키 선택: 키(8GB 이상)를 대상으로 지정합니다.
- 플래시 시작**: **플래시!**를 클릭하고 프로세스가 완료될 때까지 기다립니다.



![flash](assets/fr/03.webp)



이제 USB 키가 준비되었으므로 데비안 설치를 시작할 수 있습니다.



## 시스템에 Debian 설치하기



### USB 키로 부팅하기



USB 키에서 설치를 시작하려면 다음과 같이 하세요:




- 컴퓨터를 완전히 끄세요**.
- 재부팅**한 다음 즉시 `ESC`, `F2`, `F11`(또는 브랜드에 따라 전용 키)을 눌러 BIOS/UEFI에 액세스합니다.
- 부팅 메뉴에서 부팅 장치로 USB 키를 **선택**합니다.
- Enter 키를 눌러** 확인**하여 Debian 이미지에서 시작하면 설치 관리자의 시작 화면으로 이동합니다.



### 설치 시작하기



시작 화면:



![starting](assets/fr/04.webp)



USB 스틱에서 부팅할 때 Debian 시작 화면은 몇 가지 옵션을 제공합니다:




- 라이브 시스템**: Debian을 설치하지 않고 실행하므로 환경 테스트에 이상적입니다.
- 설치 프로그램 시작**: Hard 디스크에서 바로 설치를 시작합니다.
- 고급 설치 옵션**: 사용자 지정 설치 모드에 액세스할 수 있습니다.



라이브 모드에서 Debian을 탐색하려면 **라이브 시스템**을 선택하고 **입력**으로 확인합니다. 그런 다음 라이브 환경에서 **데비안 설치**를 클릭하여 설치를 시작할 수 있습니다.



![system](assets/fr/05.webp)





- 언어 선택** (선택 사항)



목록에서 Debian 시스템의 기본 언어를 선택한 다음 확인을 클릭합니다.



![language](assets/fr/06.webp)





- 시간대** (GMT)



날짜와 시간을 자동으로 설정하려면 지역을 선택하세요.



![timezone](assets/fr/07.webp)





- 키보드 레이아웃



키보드 언어와 레이아웃을 선택합니다. 내장된 테스트 필드를 사용하여 각 키가 예상되는 문자를 생성하는지 확인합니다.



![keyboard](assets/fr/08.webp)

## 시스템 업데이트



시스템을 사용하기 전에 반드시 업데이트해야 합니다. 이를 통해 최신 소프트웨어 패치, 최신 리포지토리, 경우에 따라 운영 체제 자체의 보안 패치의 혜택을 누릴 수 있습니다.



### 옵션 1: 그래픽 Interface(GNOME)를 통한 업데이트



GNOME 데스크톱 환경과 함께 Debian을 설치한 경우 그래픽으로 업데이트를 수행할 수 있습니다. 이렇게 하려면 **소프트웨어** 애플리케이션을 연 다음 **업데이트** 탭으로 이동합니다. 그런 다음 **다시 시작 및 업데이트**를 클릭하여 프로세스를 시작합니다.



### 옵션 2: 터미널을 통한 업데이트(권장)



이 방법은 보다 완벽한 제어를 제공합니다. 이 방법을 사용하면 리포지토리, 소프트웨어 패키지 및 필요한 경우 커널을 업데이트할 수 있습니다.




- 단축키 'Ctrl + Alt + T'를 사용하여 터미널을 엽니다.
- 다음 명령어로 사용 가능한 패키지 목록을 업데이트합니다:



```shell
sudo apt update
```



메시지가 표시되면 비밀번호를 입력합니다(입력할 때 문자가 표시되지 않는 것은 정상입니다).





- 사용 가능한 업데이트를 설치하려면 다음과 같이 하세요:



```shell
sudo apt full-upgrade
```

## 애플리케이션 설치



데비안에 애플리케이션을 설치하는 방법에는 두 가지가 있습니다:



### 그래픽 방식:



소프트웨어 관리자**(그래픽 Interface를 통해 액세스할 수 있음)를 사용하여 애플리케이션을 쉽게 검색하고 설치할 수 있습니다.



### 명령줄 메서드:



찾고 있는 애플리케이션이 그래픽 Interface에 나타나지 않거나 터미널을 선호하는 경우 다음 명령을 사용하세요:



```shell
sudo apt install <name>
```



'<name>'을 패키지 이름으로 바꿉니다. 예를 들어 `curl`을 설치하려면:



```shell
sudo apt install curl
```



### 수동으로 다운로드한 패키지를 설치합니다:



.deb` 파일(데비안 패키지)을 다운로드한 경우 다음 명령어로 설치할 수 있습니다:



```shell
sudo apt install ./name.deb
```
