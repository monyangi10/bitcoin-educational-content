Debian เป็นการแจกจ่าย GNU/Linux ฟรีที่มีชื่อเสียงในด้านความแข็งแกร่งและความน่าเชื่อถือ เคอร์เนล Linux และแพ็คเกจทั้งหมดได้รับการทดสอบอย่างเข้มงวดเพื่อให้มั่นใจถึงความเสถียรที่มั่นคงและระดับความปลอดภัยสูง เหมาะสำหรับทั้งเซิร์ฟเวอร์และเวิร์กสเตชัน Debian มอบประสบการณ์ที่ใช้งานง่ายและแคตตาล็อกซอฟต์แวร์ที่หลากหลาย ไม่ว่าคุณจะมองหาระบบที่ปลอดภัยสำหรับการใช้งานในชีวิตประจำวันหรือสภาพแวดล้อมการผลิต Debian คือตัวเลือกที่เหมาะสม



## ทำไมถึงเลือก Debian?





- ฟรีและเปิดกว้าง**: Debian เป็นโอเพ่นซอร์สทั้งหมด รับประกันความโปร่งใสและไม่มีค่าธรรมเนียมใบอนุญาต
- ความเสถียรและความปลอดภัย**: ทุกการปล่อยเวอร์ชันจะผ่านกระบวนการทดสอบอย่างละเอียด ทำให้ Debian เป็นหนึ่งในดิสทริบิวชันที่น่าเชื่อถือและปลอดภัยที่สุดในตลาด
- ชุมชนที่มีชีวิตชีวา**: มีชุมชนขนาดใหญ่และเอกสารประกอบมากมายพร้อมที่จะสนับสนุนคุณทุกเมื่อที่คุณต้องการ
- น้ำหนักเบาและปรับขนาดได้**: คุณสามารถติดตั้ง Debian บนเครื่องที่มีทรัพยากรจำกัดในขณะที่ยังคงรักษาประสิทธิภาพที่ดี
- แคตตาล็อกซอฟต์แวร์ที่กว้างขวาง**: มีแพ็คเกจอย่างเป็นทางการมากกว่า 50,000 รายการที่สามารถเข้าถึงได้ผ่านที่เก็บข้อมูล



## เลือกกราฟิก Interface



Debian มีสภาพแวดล้อมเดสก์ท็อปหลายแบบเพื่อตอบสนองความต้องการของคุณ:





- GNOME**: ทันสมัย, ใช้งานง่าย Interface, เหมาะสำหรับผู้เริ่มต้น มันมีเมนูกราฟิกที่ลื่นไหลและใช้งานง่ายสำหรับการเข้าถึงแอปพลิเคชัน
- XFCE**: เบาและเร็ว เหมาะสำหรับเครื่องที่มีประสิทธิภาพน้อยกว่า
- KDE Plasma**: ปรับแต่งได้สูง พร้อมรูปลักษณ์คล้าย Windows
- Cinnamon**: เรียบง่าย สง่างาม Interface ได้แรงบันดาลใจจาก Windows.
- LXDE / LXQt**: เบามาก เหมาะสำหรับคอมพิวเตอร์รุ่นเก่า
- MATE**: เรียบง่ายและคลาสสิก ใกล้เคียงกับ GNOME แบบเก่า



💡 สำหรับประสบการณ์ที่สะดวกสบายและจับถนัดมือ **แนะนำ GNOME เป็นอย่างยิ่ง**



## การติดตั้งและกำหนดค่า Debian


### ข้อกำหนดฮาร์ดแวร์



ก่อนเริ่มการติดตั้ง โปรดตรวจสอบว่าคุณมีอุปกรณ์ดังต่อไปนี้:





- USB key**: 8 GB ขั้นต่ำเพื่อเก็บภาพ ISO ที่สามารถบูตได้
- หน่วยความจำเข้าถึงโดยสุ่ม (RAM)**: 4 GB สำหรับการติดตั้งและการทำงานที่ราบรื่น
- พื้นที่ดิสก์**: อย่างน้อย 15 GB ของพื้นที่ว่างสำหรับระบบและการอัปเดต



### ดาวน์โหลด



การเลือกภาพ Debian ขึ้นอยู่กับสถาปัตยกรรมโปรเซสเซอร์ของคุณ:





- AMD64**: download the "live hybrid" edition from the [download] list (https://debian.obspm.fr/debian-cd/12.11.0-live/amd64/iso-hybrid/).
- ARM64**: รับภาพ DVD จากเว็บไซต์ทางการ [Debian] (https://debian.obspm.fr/debian-cd/12.11.0/arm64/iso-dvd/).
- สถาปัตยกรรมอื่นๆ**: ค้นหา ISO ที่ตรงกับสถาปัตยกรรมของคุณ [ที่นี่](https://debian.obspm.fr/debian-cd/12.11.0/).



![download](assets/fr/01.webp)



### การสร้าง USB Key ที่สามารถบูตได้



เมื่อคุณดาวน์โหลดไฟล์ ISO ที่เหมาะสมแล้ว ให้ดำเนินการสร้างสื่อการติดตั้งของคุณ:




- ดาวน์โหลด **Balena Etcher** จาก [เว็บไซต์ทางการ](https://etcher.balena.io/), จากนั้นรับไบนารีสำหรับระบบของคุณและติดตั้งมัน



![etcher](assets/fr/02.webp)





- เปิด Etcher**: เปิดซอฟต์แวร์และเลือกไฟล์ภาพ Debian ISO ที่ดาวน์โหลดมาก่อนหน้านี้
- เลือกคีย์ USB: ระบุคีย์ของคุณ (8 GB+) เป็นเป้าหมาย
- เริ่มแฟลช**: คลิกที่ **Flash!** และรอจนกว่ากระบวนการจะเสร็จสิ้น



![flash](assets/fr/03.webp)



USB ของคุณพร้อมที่จะเริ่มติดตั้ง Debian แล้ว



## การติดตั้ง Debian บนระบบของคุณ



### การบูตจาก USB key



ในการเริ่มการติดตั้งจาก USB ของคุณ:




- ปิด**คอมพิวเตอร์อย่างสมบูรณ์
- รีบูต** จากนั้นเข้าถึง BIOS/UEFI โดยกด `ESC`, `F2`, `F11` (หรือปุ่มเฉพาะขึ้นอยู่กับยี่ห้อของคุณ) ทันที
- ในเมนูบูต **เลือก USB key ของคุณ** เป็นอุปกรณ์บูต
- ยืนยัน** ด้วยปุ่ม Enter เพื่อเริ่มต้นที่ภาพ Debian: ซึ่งจะนำคุณไปยังหน้าจอต้อนรับของตัวติดตั้ง



### กำลังเริ่มการติดตั้ง



หน้าจอเริ่มต้น:



![starting](assets/fr/04.webp)



เมื่อบูตจาก USB stick หน้าจอต้อนรับ Debian จะมีตัวเลือกหลายอย่าง:

- เริ่มตัวติดตั้ง**: เริ่มการติดตั้งโดยตรงบนฮาร์ดดิสก์
- ตัวเลือกการติดตั้งขั้นสูง**: ให้คุณเข้าถึงโหมดการติดตั้งที่กำหนดเองได้



หากต้องการสำรวจ Debian ในโหมดสด ให้เลือก **Live System** และยืนยันด้วย **Enter** จากนั้นคุณสามารถเริ่มการติดตั้งได้โดยคลิกที่ **Install Debian** ในสภาพแวดล้อมสด

- การเลือกภาษา** (ไม่บังคับ)



เลือกภาษาหลักของระบบ Debian ของคุณจากรายการ จากนั้นคลิกตกลง

- เขตเวลา** (GMT)



เลือกเขตภูมิศาสตร์ของคุณเพื่อกำหนดวันที่และเวลาโดยอัตโนมัติ

- รูปแบบแป้นพิมพ์

![keyboard](assets/fr/08.webp)



### การแบ่งพาร์ติชันดิสก์





- ลบข้อมูลในดิสก์**: หากคุณมีพาร์ติชันเฉพาะ ตัวเลือกนี้จะลบเนื้อหาทั้งหมดของพาร์ติชันนั้น

![user](assets/fr/10.webp)





- สรุปพารามิเตอร์**



สรุปตัวเลือกของคุณจะแสดง: ตรวจสอบแต่ละรายการและกลับไปแก้ไขหากจำเป็น



![settings](assets/fr/11.webp)





- กำลังเริ่มการติดตั้ง



คลิกที่ **ติดตั้ง** เพื่อเริ่มคัดลอกไฟล์และกำหนดค่าระบบ จากนั้นรอจนกว่ากระบวนการจะเสร็จสิ้น



![install](assets/fr/12.webp)

![restart](assets/fr/13.webp)

## อัปเดตระบบ



ก่อนที่คุณจะเริ่มใช้งานระบบของคุณ สิ่งสำคัญคือต้องอัปเดตระบบก่อน การทำเช่นนี้จะช่วยให้คุณได้รับประโยชน์จากแพตช์ซอฟต์แวร์ล่าสุด ที่เก็บข้อมูลที่ทันสมัย และในบางกรณี แพตช์ความปลอดภัยสำหรับระบบปฏิบัติการเอง



### ตัวเลือกที่ 1: อัปเดตผ่านกราฟิก Interface (GNOME)



หากคุณได้ติดตั้ง Debian พร้อมกับสภาพแวดล้อมเดสก์ท็อป GNOME คุณสามารถทำการอัปเดตผ่านกราฟิกได้ ในการทำเช่นนี้ ให้เปิดแอปพลิเคชัน **Software** จากนั้นไปที่แท็บ **Updates** จากนั้นคลิกที่ **Restart and update** เพื่อเริ่มกระบวนการ



### ตัวเลือกที่ 2: อัปเดตผ่านเทอร์มินัล (แนะนำ)



วิธีนี้มอบการควบคุมที่สมบูรณ์ยิ่งขึ้น ช่วยให้คุณสามารถอัปเดตที่เก็บ ซอฟต์แวร์แพ็คเกจ และหากจำเป็น เคอร์เนล




- เปิดเทอร์มินัลโดยใช้ทางลัด `Ctrl + Alt + T`
- อัปเดตรายการแพ็คเกจที่มีอยู่ด้วยคำสั่งต่อไปนี้:



```shell
sudo apt update
```



ป้อนรหัสผ่านของคุณเมื่อได้รับการแจ้งเตือน (โปรดทราบว่าจะไม่มีการแสดงอักขระใด ๆ ขณะที่คุณพิมพ์ - นี่เป็นเรื่องปกติ)





- เมื่อต้องการติดตั้งการอัปเดตที่มีอยู่:



```shell
sudo apt full-upgrade
```



## ค้นพบงานพื้นฐาน



### การท่องอินเทอร์เน็ต



เว็บเบราว์เซอร์เริ่มต้นบน Debian คือ **Firefox** หากคุณต้องการใช้เบราว์เซอร์อื่น คุณสามารถติดตั้งเบราว์เซอร์อื่นได้ โดยมีเงื่อนไขว่าต้องมีอยู่ในที่เก็บของ Debian (เช่น Chromium, Brave...)



### การประมวลผลคำ



ชุดโปรแกรม **LibreOffice** ถูกติดตั้งโดยค่าเริ่มต้นบน Debian.





- ในการเขียนเอกสาร ให้ใช้ **LibreOffice Writer** ซึ่งเป็นโปรแกรมที่เทียบเท่ากับ Microsoft Word
- สำหรับสเปรดชีต **LibreOffice Calc** ทำหน้าที่เป็นทางเลือกแทน Excel.
- ในที่สุด, **LibreOffice Impress** ก็ให้คุณสร้างงานนำเสนอได้ เช่นเดียวกับ PowerPoint.



## การติดตั้งแอปพลิเคชัน



มีสองวิธีในการติดตั้งแอปพลิเคชันบน Debian:



### วิธีกราฟิก:



คุณสามารถใช้ **ตัวจัดการซอฟต์แวร์** (เข้าถึงได้ผ่านทางกราฟิก Interface) เพื่อค้นหาและติดตั้งแอปพลิเคชันได้อย่างง่ายดาย



### วิธีบรรทัดคำสั่ง:



หากแอปพลิเคชันที่คุณกำลังมองหาไม่ปรากฏในกราฟิก Interface หรือหากคุณชอบใช้เทอร์มินัล ให้ใช้คำสั่งต่อไปนี้:



```shell
sudo apt install <name>
```



แทนที่ `<name>` ด้วยชื่อแพ็กเกจ ตัวอย่างเช่น เพื่อติดตั้ง `curl`:



```shell
sudo apt install curl
```



### การติดตั้งแพ็กเกจที่ดาวน์โหลดด้วยตนเอง:



หากคุณได้ดาวน์โหลดไฟล์ `.deb` (แพ็กเกจ Debian) คุณสามารถติดตั้งได้ด้วยคำสั่งต่อไปนี้:



```shell
sudo apt install ./name.deb
```

ระบบ Debian ของคุณได้รับการติดตั้งและพร้อมใช้งานสำหรับงานประจำวันของคุณแล้ว


ขอบคุณสภาพแวดล้อมเดสก์ท็อป **GNOME** คุณสามารถเข้าถึงแอปพลิเคชันหลากหลายผ่าน Interface กราฟิกที่ใช้งานง่าย เหมาะสำหรับผู้เริ่มต้นและผู้ใช้ขั้นสูงเช่นกัน
