Debian ni ubuntu bwa GNU/Linux, buzwi cane kubera ubukomezi bwayo n'ukwizigirwa. Linux kernel yayo n’ibindi vyose biyikoresha birageragezwa cane kugira ngo bibe bikomeye nk’urutare kandi bibe bifise umutekano mwinshi. Ibereye kuri serveri n'ibibanza vy'akazi, Debian itanga ubumenyi bworoshe gukoresha n'urutonde runini rwa porogaramu. Waba uriko urarondera uburyo butekanye bwo gukoresha buri musi canke ahantu ho gukorera, Debian ni ihitamwo ryiza.



## Ni kuki uhisemwo Debian?





- Ku buntu kandi yuguruye**: Debian ni inkomoko yuguruye yose, itanga icemezo c'uguseruka kandi nta mahera y'uruhusha.
- Gushikama n’umutekano**: igisohoka cose gica mu nzira y’igerageza ryinshi, bikaba bituma Debian iba imwe mu nzira zishobora kwizigirwa kandi zitekanye ku isoko.
- Umuryango ukora**: umuryango munini n’inyandiko nyinshi ziraboneka kugira ngo zigufashe igihe cose ubikeneye.
- Iremereye kandi ishobora guhindurwa**: ushobora gushiramwo Debian ku mashini zifise uburyo buke mu gihe uguma ukora neza.
- Urutonde rwa porogarama nyinshi**: amapaki arenga 50.000 yemewe araboneka biciye ku bubiko.



## Hitamwo igishushanyo ca Interface



Debian itanga ibidukikije vyinshi vy'ibiro bihuye n'ivyo ukeneye:





- GNOME**: Interface y’ubu, ishobora gutahurwa, ni nziza ku batangura. Itanga urutonde rw’ibishushanyo rugenda neza kandi rworoshe gukoresha kugira ngo umuntu ashobore gushika ku bikorwa.
- XFCE**: yoroshe kandi yihuta, ibereye ku mashini zidakomeye cane.
- KDE Plasma**: ishobora guhindurwa cane, isa n’iya Windows.
- Cinnamon**: yoroshe, nziza cane Interface, yahumekewe na Windows.
- LXDE / LXQt**: umuco mwinshi cane, ubereye mudasobwa za kera.
- MATE**: yoroshe kandi ya kera, hafi ya GNOME ya kera.



💡 Kugira ngo ubone ubumenyi bwiza, bworoshe gufata, **GNOME iraremeshwa cane**.



## Gushiramwo no gutunganya Debian


### Ibisabwa ku bikoresho



Imbere yo gutangura gushiramwo, urabe neza ko ufise ibi bikoresho bikurikira:





- Urufunguzo rwa USB**: 8 GB n’imiburiburi kugira ngo ufate ishusho ya ISO ishobora gufungurwa.
- Ubuhinga bwo kwibuka (RAM)**: 4 GB kugira ngo ushobore gushiramwo no gukoresha neza.
- Disk space**: nibura 15 GB y’umwanya w’ubuntu wo gukoresha muri sisitemu n’ibindi bishasha.



### Gukurura



Guhitamwo ishusho ya Debian bivana n'ubwubatsi bwa processeur yawe:





- AMD64**: gukuraho igitabu "cy'ubuzima" ku rutonde [gukuraho] (https://debian.obspm.fr/debian-cd/12.11.0-ubuzima/amd64/iso-hybrid/).
- ARM64**: ubone ishusho ya DVD ku rubuga rwemewe rwa Debian (Debian.obspm.fr/12.11.0/arm64/iso-dvd/).
- Ibindi vyubatswe**: rondera ISO ihuye n'ubwubatsi bwawe [hano](https://debian.obspm.fr/debian-cd/12.11.0/).



![download](assets/fr/01.webp)



### Gukora urufunguzo rwa USB rushobora gufungurwa



Umaze gukuraho ishusho ISO ibereye, genda ureme ubuhinga bwawe bwo gushiramwo:




- Gukura **Balena Etcher** ku [urubuga rwemewe](https://etcher.balena.io/), hanyuma utore binaire ya sisitemu yawe maze uyishiremwo.



![etcher](assets/fr/02.webp)





- Launch Etcher**: gufungura porogaramu maze uhitemwo ishusho ya Debian ISO yari yarakuweho mbere.
- Hitamwo urufunguzo rwa USB: vuga urufunguzo rwawe (8 GB+) nk’intumbero.
- Tangira flash**: ukande kuri **Flash!** hanyuma urindire gushika igikorwa kirangiye.



![flash](assets/fr/03.webp)



Urufunguzo rwawe rwa USB ubu rwiteguye gutangura gushiramwo Debian.



## Gushiramwo Debian kuri sisitemu yawe



### Gufungura bivuye ku rufunguzo rwa USB



Kugira ngo utangure gushiramwo ukoresheje urufunguzo rwawe rwa USB:




- Zima** mudasobwa yose.
- Reboot** hanyuma winjire muri BIOS/UEFI ukanda `ESC`, `F2`, `F11` (canke urufunguzo rwihariye bivanye n’ibara ryawe).
- Mu nzira y'ugutangura, **hitamwo urufunguzo rwawe rwa USB** nk'igikoresho co gufungura.
- Wemeze** n'urufunguzo Enter kugira ngo utangure ku ishusho ya Debian: ibi bizokujana ku mugaragaro w'akabazo w'umushiramwo.



### Gutanguza gushiramwo



Igishushanyo co gutangura:



![starting](assets/fr/04.webp)



Igihe ufunguye ukoresheje inkoni ya USB, igicapo c'akabazo ca Debian gitanga amahitamwo menshi:




- Live System**: itanguza Debian itayishizemwo, ni nziza cane mu kugerageza ibidukikije.
- Gutangura gushiramwo**: itangura gushiramwo ataco ihinduye kuri disiki ya Hard.
- Amahitamwo yo gushiramwo**: aguha uburenganzira bwo gukoresha uburyo bwo gushiramwo bugenewe.



Kugira ngo ubone Debian mu buryo bubaho, hitamwo **Uburyo bubaho** maze wemeze na **Injira**. Ushobora rero gutanguza gushiramwo ukanda kuri **Shiraho Debian** mu bidukikije bibaho.



![system](assets/fr/05.webp)





- Guhitamwo ururimi** (ntibikenewe)



Hitamwo ururimi nyamukuru rwa sisitemu yawe ya Debian muri urwo rutonde, hanyuma ukande OK.



![language](assets/fr/06.webp)





- Igihe** (GMT)



Hitamwo akarere kawe kugira ngo ushireho itariki n’isaha.



![timezone](assets/fr/07.webp)





- Imiterere ya klavye



Hitamwo ururimi rwa klavye yawe n’ingene iteye. Koresha umwanya w’ikigeragezo wubakiwemwo kugira ngo usuzume ko urufunguzo rwose rutanga ikimenyetso citezwe.



![keyboard](assets/fr/08.webp)



### Gucapura disiki





- Erase disk**: niba ufise igice kigenewe, iyi nzira izofuta ibirimwo vyose.
- Gucapura n'amaboko**: hitamwo iyi nzira kugira ngo ureme, uhindure ingano canke usibe ibice nk'uko bisabwa.



![disk](assets/fr/09.webp)





- Gukora konti y'ukoresha

## Kuvugurura sisitemu



Imbere y’uko utangura gukoresha sisitemu yawe, ni ngombwa ko uyihindura. Ivyo bituma wungukira ku bikoresho bishasha vya porogarama, ku bibanza vy’ububiko bigezweho, rimwe na rimwe na ho, ku bikoresho vy’umutekano vy’ivyo bikoresho ubwavyo.



### Ihitamwo rya 1: Kuvugurura biciye ku gishushanyo Interface (GNOME)



Niba warashizeho Debian n'ibidukikije vya GNOME, ushobora gukora ivugurura mu buryo bw'ibishushanyo. Kugira ngo ubikore, fungura porogarama ya **Software**, hanyuma ugende ku rubuga rwa **Updates**. Hanyuma ukande kuri **Restart and update** kugira ngo utangure igikorwa.



### Amahitamwo ya 2: Kuvugurura biciye kuri terminal (ni vyiza)



Ubwo buryo buratuma umuntu ashobora kugenzura neza cane. Igufasha guhindura ububiko, amapaki ya porogarama n’ibindi, iyo bikenewe, n’inkomoko.




- Gufungura urubuga ukoresheje inzira ngufi `Ctrl + Alt + T`.
- Kuvugurura urutonde rw'amapaki ariho n'itegeko rikurikira:



```shell
sudo apt update
```



Injira ijambobanga ryawe iyo usabwe (menya ko ata nyuguti zizogaragara uko wandika - ivyo ni ibisanzwe).





- Kugira ngo ushiremwo ivyagezwe biriho:



```shell
sudo apt full-upgrade
```



## Tora ibikorwa vy'ishimikiro



### Gusura kuri Internet



Umucukumbuzi w'urubuga kuri Debian ni **Firefox**. Niba ushaka uwundi mucukumbuzi, ushobora gushiramwo uwundi, igihe cose azoba ari mu bubiko bwa Debian (nka Chromium, Brave...).



### Gutunganya amajambo



**LibreOffice** ishizweho ku buryo busanzwe kuri Debian.





- Kugira ngo wandike inyandiko, koresha **Umwanditsi wa LibreOffice**, bingana n'Ijambo rya Microsoft.
- Ku mpapuro z'imibare, **LibreOffice Calc** ikora nk'iyindi nzira kuri Excel.
- Ubwa nyuma, **LibreOffice Impress** iragufasha gukora ibiganiro, nka PowerPoint.



## Gushiramwo porogaramu



Hari uburyo bubiri bwo gushiramwo porogaramu kuri Debian:



### Uburyo bw'ibishushanyo:



Ushobora gukoresha **umucungerezi wa porogarama** (ushobora kuyironka biciye ku gishushanyo Interface) kugira ngo uronde kandi ushiremwo porogarama mu buryo bworoshe.



### Uburyo bw'umurongo w'itegeko:



Niba porogaramu uriko urarondera itaboneka mu gishushanyo ca Interface, canke niba ukunda terminal, koresha iri tegeko:



```shell
sudo apt install <name>
```



Subiriza `<izina>` n'izina ry'umuzigo. Akarorero, gushiramwo `curl`:



```shell
sudo apt install curl
```



### Gushiramwo umuzigo wakuweho n'amaboko:



Niba warakuyeho dosiye `.deb` (Ipaki ya Debian), ushobora kuyishiramwo ukoresheje iri tegeko rikurikira:



```shell
sudo apt install ./name.deb
```
