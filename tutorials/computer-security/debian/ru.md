Debian - это свободный дистрибутив GNU/Linux, известный своей прочностью и надежностью. Его ядро Linux и все пакеты проходят строгое тестирование, чтобы обеспечить стабильность и высокий уровень безопасности. Подходящий как для серверов, так и для рабочих станций, Debian предлагает простоту в использовании и обширный каталог программного обеспечения. Если вы ищете безопасную систему для повседневного использования или для производственной среды, Debian - это правильный выбор.



## Почему стоит выбрать Debian?





- Свободный и открытый**: Debian - это полностью открытый исходный код, что гарантирует прозрачность и отсутствие лицензионных платежей.
- Стабильность и безопасность**: каждый выпуск проходит через процесс тщательного тестирования, что делает Debian одним из самых надёжных и безопасных дистрибутивов на рынке.
- Активное сообщество**: обширное сообщество и подробная документация готовы оказать вам поддержку в любой момент.
- Лёгкий и масштабируемый**: вы можете установить Debian на машины со скромными ресурсами, сохраняя при этом высокую производительность.
- Обширный каталог программного обеспечения**: в репозиториях доступно более 50 000 официальных пакетов.



## Выберите графику Interface



Debian предлагает несколько окружений рабочего стола для удовлетворения ваших потребностей:





- GNOME**: современный, интуитивно понятный Interface, идеально подходящий для новичков. Он предлагает плавное и простое в использовании графическое меню для доступа к приложениям.
- XFCE**: легкий и быстрый, идеально подходит для менее мощных машин.
- KDE Plasma**: очень настраиваемый, с внешним видом, похожим на Windows.
- Cinnamon**: простой, элегантный Interface, вдохновленный Windows.
- LXDE / LXQt**: ультралегкий, подходит для старых компьютеров.
- MATE**: простой и классический, близкий к старому GNOME.



💡 Для удобства и простоты захвата настоятельно рекомендуется использовать **GNOME**.



## Установка и настройка Debian


### Требования к аппаратному обеспечению



Перед началом установки убедитесь, что у вас есть следующее оборудование:





- USB-носитель**: минимум 8 ГБ для хранения загрузочного ISO-образа.
- Случайная память (RAM)**: 4 ГБ для плавной установки и работы.
- Дисковое пространство**: не менее 15 ГБ свободного места для системы и обновлений.



### Скачать



Выбор образа Debian зависит от архитектуры вашего процессора:





- AMD64**: загрузите "живое гибридное" издание из списка [download] (https://debian.obspm.fr/debian-cd/12.11.0-live/amd64/iso-hybrid/).
- ARM64**: получите образ DVD с официального сайта [Debian] (https://debian.obspm.fr/debian-cd/12.11.0/arm64/iso-dvd/).
- Другие архитектуры**: найдите ISO, соответствующий вашей архитектуре [здесь](https://debian.obspm.fr/debian-cd/12.11.0/).



![download](assets/fr/01.webp)



### Создание загрузочного USB-носителя



После загрузки соответствующего ISO-образа перейдите к созданию установочного носителя:




- Загрузите Balena Etcher** с [официального сайта](https://etcher.balena.io/), затем получите бинарник для вашей системы и установите его.



![etcher](assets/fr/02.webp)





- Запустите Etcher**: откройте программу и выберите загруженный ранее ISO-образ Debian.
- Выберите USB-носитель**: укажите в качестве цели ваш носитель (8 ГБ+).
- Запустите прошивку**: нажмите на **Прошивка!** и дождитесь завершения процесса.



![flash](assets/fr/03.webp)



Теперь ваш USB-носитель готов к установке Debian.



## Установка Debian на вашу систему



### Загрузка с USB-носителя



Чтобы запустить установку с USB-носителя, выполните следующие действия:




- Полностью выключите** компьютер.
- Перезагрузитесь**, затем войдите в BIOS/UEFI, сразу же нажав `ESC`, `F2`, `F11` (или специальную клавишу в зависимости от марки).
- В меню загрузки **выберите USB-носитель** в качестве загрузочного устройства.
- Подтвердите** клавишей Enter, чтобы начать установку образа Debian: это приведёт вас к приветственному экрану программы установки.



### Запуск установки



Начальный экран:



![starting](assets/fr/04.webp)



При загрузке с USB-накопителя экран приветствия Debian предлагает несколько вариантов:




- Live System**: запускает Debian без установки, идеально подходит для тестирования среды.
- Start Installer**: запуск установки непосредственно на диск Hard.
- Advanced Install Options**: предоставляет доступ к настраиваемым режимам установки.



Чтобы изучить Debian в живом режиме, выберите **Живая система** и подтвердите выбор нажатием **Enter**. Затем вы можете запустить установку, нажав на **Установить Debian** в живой среде.



![system](assets/fr/05.webp)





- Выбор языка** (опционально)



Выберите основной язык вашей системы Debian из списка, затем нажмите OK.



![language](assets/fr/06.webp)





- Часовой пояс** (GMT)



Выберите географическую зону, чтобы автоматически установить дату и время.



![timezone](assets/fr/07.webp)





- Раскладка клавиатуры



Выберите язык и раскладку клавиатуры. Используйте встроенное тестовое поле, чтобы проверить, что каждая клавиша выдает ожидаемый символ.



![keyboard](assets/fr/08.webp)



### Разбиение диска на разделы





- Стереть диск**: если у вас есть выделенный раздел, эта опция удалит все его содержимое.
- Ручная разбивка**: выберите этот вариант, чтобы создавать, изменять размер или удалять разделы по мере необходимости.



![disk](assets/fr/09.webp)





- Создание учетной записи пользователя



Введите свое полное имя, имя учетной записи и надежный пароль, чтобы обеспечить безопасность сеанса.



![user](assets/fr/10.webp)





- Сводка параметров**



На экране появится сводка выбранных вами вариантов: отметьте каждый пункт и при необходимости вернитесь назад, чтобы внести изменения.



![settings](assets/fr/11.webp)





- Запуск установки



Нажмите на **Install**, чтобы начать копирование файлов и настройку системы, затем дождитесь завершения процесса.



![install](assets/fr/12.webp)





- Перезапуск**



После завершения установки перезагрузите компьютер, чтобы применить все конфигурации и получить доступ к новой системе Debian.



![restart](assets/fr/13.webp)



При запуске введите имя пользователя и пароль для доступа к системе.



![login](assets/fr/14.webp)



## Обновление системы



Прежде чем начать пользоваться системой, необходимо ее обновить. Это позволит вам воспользоваться последними исправлениями программного обеспечения, обновленными репозиториями, а в некоторых случаях и исправлениями безопасности для самой операционной системы.



### Вариант 1: Обновление через графический Interface (GNOME)



Если вы установили Debian с окружением рабочего стола GNOME, вы можете выполнять обновления графически. Для этого откройте приложение **Программное обеспечение**, затем перейдите на вкладку **Обновления**. Затем нажмите на **Перезапустить и обновить**, чтобы начать процесс.



### Вариант 2: Обновление через терминал (рекомендуется)



Этот метод обеспечивает более полный контроль. Он позволяет обновлять репозитории, пакеты программ и, при необходимости, ядро.




- Откройте терминал с помощью сочетания клавиш `Ctrl + Alt + T`.
- Обновите список доступных пакетов с помощью следующей команды:



```shell
sudo apt update
```



Введите пароль при появлении запроса (обратите внимание, что при вводе символов не будет отображаться - это нормально).





- Чтобы установить доступные обновления:



```shell
sudo apt full-upgrade
```



## Узнайте об основных задачах



### Просмотр Интернета



Веб-браузером по умолчанию в Debian является **Firefox**. Если вы предпочитаете другой браузер, вы можете установить его, если он доступен в репозиториях Debian (например, Chromium, Brave...).



### Работа с текстом



В Debian по умолчанию установлен пакет **LibreOffice**.





- Для написания документов используйте **LibreOffice Writer**, аналог Microsoft Word.
- Для работы с электронными таблицами в качестве альтернативы Excel выступает **LibreOffice Calc**.
- Наконец, **LibreOffice Impress** позволяет создавать презентации, как и PowerPoint.



## Установка приложений



Существует два способа установки приложений в Debian:



### Графический метод:



Вы можете использовать **менеджер программного обеспечения** (доступен через графический интерфейс Interface) для удобного поиска и установки приложений.



### Метод командной строки:



Если нужное вам приложение не отображается в графическом интерфейсе Interface или вы предпочитаете терминал, воспользуйтесь следующей командой:



```shell
sudo apt install <name>
```



Замените `<имя>` на имя пакета. Например, чтобы установить `curl`:



```shell
sudo apt install curl
```



### Установка загруженного вручную пакета:



Если вы загрузили файл `.deb` (пакет Debian), вы можете установить его с помощью следующей команды:



```shell
sudo apt install ./name.deb
```
