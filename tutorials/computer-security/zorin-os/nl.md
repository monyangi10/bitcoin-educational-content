## Inleiding



Een besturingssysteem (OS) is de fundamentele software waarmee een computer kan werken: het beheert hardware, software, beveiliging en de gebruikersinterface.


Zorin OS is een Linux-distributie die speciaal is ontworpen om de overgang van Windows te vergemakkelijken en tegelijkertijd de voordelen van open source software te bieden: veiligheid, stabiliteit, privacy en prestaties.



Zorin OS is gebaseerd op Ubuntu LTS en combineert een hoge softwarecompatibiliteit met een vertrouwde, aanpasbare interface, waardoor het een geloofwaardig en toegankelijk alternatief is voor Windows.



## Waarom Zorin OS?





- Interface vertrouwd**: Windows-achtig uiterlijk (startmenu, taakbalk)
- Gemakkelijke overgang**: ontworpen voor gebruikers die van Windows komen
- Verbeterde beveiliging**: Linux-architectuur, minder blootgesteld aan virussen
- Respect voor privacy**: geen opdringerige gegevensverzameling
- Geoptimaliseerde prestaties**: werkt goed op bescheiden machines
- Ubuntu LTS** basis: stabiliteit, regelmatige updates en brede compatibiliteit
- Geavanceerde personalisatie**: via de Zorin Appearance tool.



## Installatie en configuratie



### 1. Vereisten



**vereiste uitrusting:**





- Een USB-sleutel van minstens **8 GB** (12 GB aanbevolen);
- Een computer met ten minste **25 GB vrije schijfruimte**
- Internetverbinding (aanbevolen).

### Toepassingen



Er zijn verschillende manieren om applicaties te installeren:





- Software**, de applicatiewinkel van Zorin OS. Applicaties komen van verschillende bronnen: apt, Flatpak en Snap.



![capture](assets/fr/38.webp)



![capture](assets/fr/39.webp)





- apt** installeren (opdrachtregel) :



```bash
sudo apt install gparted
```



![capture](assets/fr/40.webp)



Voor meer informatie over het installeren van applicaties in Zorin OS, zie deze pagina: [Apps installeren (zorin.com)](https://zorin.com/help/install-apps/).



### Windows-toepassingen



Om Windows applicaties te installeren, begin je met het installeren van het **zorin-windows-app-support** pakket via Terminal :



```bash
sudo apt install zorin-windows-app-support
```



Voor een lijst met compatibele Windows-toepassingen en hun compatibiliteitsniveaus, zie de pagina [Wine Application Database] (https://appdb.winehq.org/). Daar vind je de volgende badges, die overeenkomen met het compatibiliteitsniveau (van best naar slechtst): Platinum, Gold, Silver, Bronze en Garbage.



Om een Windows .exe- of .msi-toepassing te installeren, heb je twee opties:





- Open **PlayOnLinux** en klik op de knop **Installeren** om door compatibele applicaties en games te bladeren.



![capture](assets/fr/41.webp)





- Dubbelklik op het **.exe of .msi** bestand van de toepassing en laat je leiden door het installatieprogramma.
