## บทนำ



ระบบปฏิบัติการ (OS) คือซอฟต์แวร์พื้นฐานที่ทำให้คอมพิวเตอร์ทำงานได้: มันจัดการฮาร์ดแวร์ ซอฟต์แวร์ ความปลอดภัย และส่วนติดต่อผู้ใช้


Zorin OS เป็นการกระจายตัวของ Linux ที่ออกแบบมาโดยเฉพาะเพื่อให้ง่ายต่อการเปลี่ยนจาก Windows ในขณะเดียวกันก็เสนอประโยชน์ของซอฟต์แวร์โอเพนซอร์ส: ความปลอดภัย, ความเสถียร, ความเป็นส่วนตัว และประสิทธิภาพ



จาก Ubuntu LTS, Zorin OS ผสมผสานความเข้ากันได้ของซอฟต์แวร์สูงกับอินเทอร์เฟซที่คุ้นเคยและปรับแต่งได้ ทำให้เป็นทางเลือกที่น่าเชื่อถือและเข้าถึงได้สำหรับ Windows



## ทำไมต้อง Zorin OS?





- Interface คุ้นเคย**: ลักษณะคล้าย Windows (เมนูเริ่ม, แถบงาน)
- การเปลี่ยนแปลงที่ง่ายดาย**: ออกแบบมาสำหรับผู้ใช้ที่มาจาก Windows
- การรักษาความปลอดภัยที่เพิ่มขึ้น**: สถาปัตยกรรม Linux, มีความเสี่ยงต่อไวรัสน้อยกว่า
- เคารพความเป็นส่วนตัว**: ไม่มีการเก็บข้อมูลที่ล่วงล้ำ
- ประสิทธิภาพที่ปรับแต่ง**: ทำงานได้ดีบนเครื่องที่มีสเปคไม่สูง
- Ubuntu LTS** ฐาน: ความเสถียร, การอัปเดตเป็นประจำ และความเข้ากันได้อย่างกว้างขวาง
- การปรับแต่งขั้นสูง**: ผ่านเครื่องมือ Zorin Appearance

### แอปพลิเคชัน



มีหลายวิธีในการติดตั้งแอปพลิเคชัน:





- Software**, ร้านค้าแอปพลิเคชันของ Zorin OS แอปพลิเคชันมาจากหลายแหล่ง: apt, Flatpak และ Snap.



![capture](assets/fr/38.webp)



![capture](assets/fr/39.webp)





- apt** ติดตั้ง (บรรทัดคำสั่ง) :



```bash
sudo apt install gparted
```



![capture](assets/fr/40.webp)



สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการติดตั้งแอปพลิเคชันใน Zorin OS ดูหน้านี้: [Install Apps (zorin.com)](https://zorin.com/help/install-apps/).



### แอปพลิเคชัน Windows



ในการติดตั้งแอปพลิเคชัน Windows ให้เริ่มต้นด้วยการติดตั้งแพ็กเกจ **zorin-windows-app-support** ผ่านทาง Terminal :



```bash
sudo apt install zorin-windows-app-support
```



สำหรับรายชื่อแอปพลิเคชัน Windows ที่เข้ากันได้และระดับความเข้ากันได้ ดูที่หน้า [Wine Application Database] (https://appdb.winehq.org/) ที่นั่นคุณจะพบกับตราสัญลักษณ์ต่อไปนี้ ซึ่งสอดคล้องกับระดับความเข้ากันได้ (จากดีที่สุดไปแย่ที่สุด): Platinum, Gold, Silver, Bronze และ Garbage.



ในการติดตั้งแอปพลิเคชัน Windows .exe หรือ .msi คุณมีสองตัวเลือก:





- เปิด **PlayOnLinux** และคลิกที่ปุ่ม **Install** เพื่อเรียกดูแอปพลิเคชันและเกมที่เข้ากันได้



![capture](assets/fr/41.webp)





- ดับเบิลคลิกที่ไฟล์ **.exe หรือ .msi** ของแอปพลิเคชันและให้โปรแกรมติดตั้งนำทางคุณ
