## 소개



운영 체제(OS)는 하드웨어, 소프트웨어, 보안 및 사용자 인터페이스를 관리하는 등 컴퓨터가 작동할 수 있도록 하는 기본 소프트웨어입니다.


Zorin OS는 오픈 소스 소프트웨어의 장점인 보안, 안정성, 개인 정보 보호 및 성능을 제공하면서 Windows에서 쉽게 전환할 수 있도록 특별히 설계된 Linux 배포판입니다.



Ubuntu LTS를 기반으로 하는 Zorin OS는 높은 소프트웨어 호환성과 친숙하고 사용자 정의 가능한 인터페이스를 결합하여 Windows를 대체할 수 있는 신뢰할 수 있고 접근하기 쉬운 대안입니다.



## 왜 조린 OS인가?





- Interface 익숙한**: Windows와 유사한 모양(시작 메뉴, 작업 표시줄)
- 간편한 전환**: Windows 사용자를 위한 설계
- 강화된 보안**: 바이러스에 덜 노출되는 Linux 아키텍처
- 프라이버시 존중**: 침입적 데이터 수집 금지
- 최적화된 성능**: 적당한 사양의 컴퓨터에서 잘 작동합니다
- Ubuntu LTS** 기반: 안정성, 정기적인 업데이트 및 광범위한 호환성
- 고급 개인화**: 조린 모양 도구를 통해.

## 키 부팅 및 BIOS 액세스



Zorin OS를 설치하려는 컴퓨터의 전원을 끈 다음 USB 키를 연결합니다.


컴퓨터가 시작되면 BIOS(브랜드에 따라 `ESC`, `F9` 또는 `F11`)에 액세스하여 USB 키를 부팅 장치로 선택한 다음 `Enter`를 눌러 부팅 프로세스를 시작하세요.





- 시작 시 **조린 OS 시도 또는 설치**를 선택합니다.



![capture](assets/fr/08.webp)





- NVIDIA 그래픽 카드가 있는 경우 **Zorin OS(최신 NVIDIA 드라이버)** 시도 또는 설치를 선택합니다.
- 파일을 확인하는 동안 잠시 기다려주세요.



![capture](assets/fr/09.webp)





- 조린 OS 설치 관리자에서 **프랑스어** 언어를 선택한 다음 **조린 OS 설치**를 클릭합니다.



![capture](assets/fr/10.webp)





- 키보드 레이아웃을 선택합니다.



![capture](assets/fr/11.webp)





- Zorin OS 설치 중 업데이트 다운로드** 및 **그래픽 및 Wi-Fi 하드웨어와 추가 미디어 포맷을 위한 타사 소프트웨어 설치** 확인란을 선택합니다.



![capture](assets/fr/12.webp)





- 전체 디스크에 Zorin OS를 설치하려면: **디스크 지우기 및 Zorin OS 설치**를 선택합니다.

### 애플리케이션



애플리케이션을 설치하는 방법에는 여러 가지가 있습니다:





- Zorin OS 애플리케이션 스토어인 Software**. 애플리케이션은 apt, Flatpak 및 Snap 등 여러 소스에서 제공됩니다.



![capture](assets/fr/38.webp)



![capture](assets/fr/39.webp)





- apt** 설치 (명령줄) :



```bash
sudo apt install gparted
```



![capture](assets/fr/40.webp)



Zorin OS에서 애플리케이션 설치에 대한 자세한 내용은 이 페이지를 참조하세요: [앱 설치하기(zorin.com)](https://zorin.com/help/install-apps/)를 참조하세요.



### Windows 애플리케이션



Windows 애플리케이션을 설치하려면 먼저 터미널을 통해 **zorin-windows-app-support** 패키지를 설치하세요:



```bash
sudo apt install zorin-windows-app-support
```



호환되는 Windows 애플리케이션 목록과 호환성 수준은 [와인 애플리케이션 데이터베이스] 페이지(https://appdb.winehq.org/)를 참조하세요. 호환성 수준에 따라 다음과 같은 배지가 표시됩니다(최고부터 최악까지): 플래티넘, 골드, 실버, 브론즈, 쓰레기.



Windows .exe 또는 .msi 애플리케이션을 설치하려면 두 가지 옵션이 있습니다:





- 플레이온리눅스**를 열고 **설치** 버튼을 클릭하여 호환되는 애플리케이션과 게임을 찾아보세요.



![capture](assets/fr/41.webp)





- 애플리케이션의 **.exe 또는 .msi** 파일을 더블 클릭하고 설치 프로그램의 안내에 따라 진행합니다.
