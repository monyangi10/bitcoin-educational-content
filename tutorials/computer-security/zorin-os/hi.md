## परिचय



ऑपरेटिंग सिस्टम (ओएस) वह मूलभूत सॉफ्टवेयर है जो कंप्यूटर को कार्य करने में सक्षम बनाता है: यह हार्डवेयर, सॉफ्टवेयर, सुरक्षा और उपयोगकर्ता इंटरफ़ेस का प्रबंधन करता है।


ज़ोरिन ओएस एक लिनक्स वितरण है जिसे विशेष रूप से विंडोज से लिनक्स में आसानी से बदलाव लाने के लिए डिज़ाइन किया गया है, साथ ही यह ओपन सोर्स सॉफ़्टवेयर के लाभ भी प्रदान करता है: सुरक्षा, स्थिरता, गोपनीयता और प्रदर्शन।



Ubuntu LTS पर आधारित, Zorin OS उच्च सॉफ्टवेयर अनुकूलता को एक परिचित, अनुकूलन योग्य इंटरफ़ेस के साथ जोड़ता है, जिससे यह विंडोज का एक विश्वसनीय और सुलभ विकल्प बन जाता है।



## Zorin OS क्यों?





- Interface से परिचित**: विंडोज़ जैसी दिखावट (स्टार्ट मेनू, टास्कबार)
- आसान बदलाव**: विंडोज से आने वाले उपयोगकर्ताओं के लिए डिज़ाइन किया गया है
- बेहतर सुरक्षा**: लिनक्स आर्किटेक्चर, वायरस से कम प्रभावित।
- निजता का सम्मान**: किसी भी प्रकार का दखलंदाजी वाला डेटा संग्रह नहीं किया जाएगा
- बेहतर प्रदर्शन**: सामान्य मशीनों पर भी अच्छा काम करता है
- Ubuntu LTS** का आधार: स्थिरता, नियमित अपडेट और व्यापक अनुकूलता
- उन्नत वैयक्तिकरण**: ज़ोरिन अपीयरेंस टूल के माध्यम से।

## मुख्य बूटिंग और BIOS एक्सेस



जिस कंप्यूटर पर आप ज़ोरिन ओएस इंस्टॉल करना चाहते हैं, उसे बंद कर दें, फिर यूएसबी कुंजी को उसमें प्लग इन करें।


जब आपका कंप्यूटर चालू हो जाए, तो BIOS में जाएं (`ESC`, `F9` या `F11`, ब्रांड के आधार पर) और USB कुंजी को बूट डिवाइस के रूप में चुनें, फिर बूट प्रक्रिया शुरू करने के लिए `Enter` दबाएं।





- स्टार्टअप के समय, **ज़ोरिन ओएस को आज़माएं या इंस्टॉल करें** चुनें।



![capture](assets/fr/08.webp)





- यदि आपके पास एनवीडिया ग्राफिक्स कार्ड है, तो **ज़ोरिन ओएस (आधुनिक एनवीडिया ड्राइवर) को आज़माएं या इंस्टॉल करें** चुनें।
- कृपया प्रतीक्षा करें, फाइलों की जांच की जा रही है।



![capture](assets/fr/09.webp)





- Zorin OS इंस्टॉलर में, भाषा **फ्रेंच** चुनें और फिर Install **Zorin OS** पर क्लिक करें।



![capture](assets/fr/10.webp)

![capture](assets/fr/11.webp)

### आवेदन



एप्लिकेशन इंस्टॉल करने के कई तरीके हैं:





- Software**, ज़ोरिन OS का एप्लिकेशन स्टोर है। एप्लिकेशन कई स्रोतों से आते हैं: apt, Flatpak और Snap।



![capture](assets/fr/38.webp)



![capture](assets/fr/39.webp)





- apt** इंस्टॉल (कमांड लाइन):



```bash
sudo apt install gparted
```



![capture](assets/fr/40.webp)



Zorin OS में एप्लिकेशन इंस्टॉल करने के बारे में अधिक जानकारी के लिए, यह पृष्ठ देखें: [ऐप्स इंस्टॉल करें (zorin.com)](https://zorin.com/help/install-apps/)।



### विंडोज एप्लिकेशन



विंडोज एप्लिकेशन इंस्टॉल करने के लिए, सबसे पहले टर्मिनल के माध्यम से **zorin-windows-app-support** पैकेज इंस्टॉल करें:



```bash
sudo apt install zorin-windows-app-support
```



संगत विंडोज़ एप्लिकेशन और उनकी संगतता स्तरों की सूची के लिए, [वाइन एप्लिकेशन डेटाबेस] पृष्ठ (https://appdb.winehq.org/) देखें। वहां आपको संगतता स्तर के अनुसार निम्नलिखित बैज मिलेंगे (सर्वोत्तम से सबसे खराब क्रम में): प्लेटिनम, गोल्ड, सिल्वर, ब्रॉन्ज़ और गार्बेज।



विंडोज .exe या .msi एप्लिकेशन इंस्टॉल करने के लिए आपके पास दो विकल्प हैं:





- **PlayOnLinux** खोलें और संगत एप्लिकेशन और गेम ब्राउज़ करने के लिए **इंस्टॉल** बटन पर क्लिक करें।



![capture](assets/fr/41.webp)





- एप्लिकेशन की **.exe या .msi** फ़ाइल पर डबल-क्लिक करें और इंस्टॉलेशन प्रोग्राम को आपको निर्देशित करने दें।
