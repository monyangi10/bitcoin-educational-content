## Introduzione



Un sistema operativo (OS) è il software fondamentale che consente il funzionamento di un computer: gestisce l'hardware, il software, la sicurezza e l'interfaccia utente.


Zorin OS è una distribuzione Linux progettata appositamente per facilitare la transizione da Windows, offrendo al contempo i vantaggi del software open source: sicurezza, stabilità, privacy e prestazioni.



Basato su Ubuntu LTS, Zorin OS combina un'elevata compatibilità software con un'interfaccia familiare e personalizzabile, rendendolo un'alternativa credibile e accessibile a Windows.



## Perché Zorin OS?





- Interface familiare**: Aspetto simile a Windows (menu di avvio, barra delle applicazioni)
- Transizione facile**: progettato per gli utenti che provengono da Windows
- Maggiore sicurezza**: Architettura Linux, meno esposta ai virus
- Rispetto della privacy**: nessuna raccolta di dati invasiva
- Prestazioni ottimizzate**: funziona bene su macchine modeste
- Base Ubuntu LTS**: stabilità, aggiornamenti regolari e ampia compatibilità
- Personalizzazione avanzata**: tramite lo strumento Zorin Appearance.



## Installazione e configurazione



### 1. Prerequisiti



**Attrezzatura richiesta:**





- Una chiave USB di almeno **8 GB** (12 GB consigliati);
- Un computer con almeno **25 GB di spazio libero su disco**
- Connessione a Internet (consigliata).



### 2. Scaricare Zorin OS





- Visita il sito web ufficiale: [https://zorin.com/os](https://zorin.com/os)



![Page de téléchargement Balena Etcher](assets/fr/03.webp)





- Scegliere **Zorin OS Core** (si consiglia la versione gratuita)



![Page de téléchargement Balena Etcher](assets/fr/04.webp)





- Scarica l'immagine ISO



Zorin OS offre anche :





- Zorin OS Lite** (computer più vecchi)
- Zorin OS Pro** (a pagamento, con funzioni avanzate e supporto)



## Creare una chiave USB avviabile



È possibile utilizzare diversi strumenti, come Balena Etcher :





- Scaricare e installare [Balena Etcher](https://etcher.balena.io/).
- Aprire Balena Etcher, quindi selezionare l'immagine ISO di Zorin.
- Selezionare la chiave USB come supporto di destinazione.
- Fare clic su Flash e attendere il completamento del processo.



![Utilisation de Balena Etcher](assets/fr/05.webp)



## Avvio con chiave e accesso al BIOS



Spegnere il computer su cui si desidera installare Zorin OS, quindi inserire la chiave USB.


All'avvio del computer, accedere al BIOS (`ESC`, `F9` o `F11` a seconda della marca) e selezionare la chiave USB come dispositivo di avvio, quindi premere `Invio` per avviare il processo di avvio.





- All'avvio, selezionare **Prova o installa Zorin OS**.



![capture](assets/fr/08.webp)





- Se si dispone di una scheda grafica NVIDIA, selezionare **Prova o installa Zorin OS (driver NVIDIA moderni)**.
- Si prega di attendere mentre i file vengono controllati.



![capture](assets/fr/09.webp)





- Nel programma di installazione di Zorin OS, selezionare la lingua **Francese** e fare clic su Installa **Zorin OS**.



![capture](assets/fr/10.webp)





- Selezionare il layout della tastiera.



![capture](assets/fr/11.webp)





- Selezionare le caselle **Scaricare gli aggiornamenti durante l'installazione di Zorin OS** e **Installare software di terze parti per l'hardware grafico e Wi-Fi e formati multimediali aggiuntivi**.



![capture](assets/fr/12.webp)





- Per installare Zorin OS sull'intero disco: selezionare **Cancella disco e installa Zorin OS**.



![capture](assets/fr/14.webp)



Per installare Zorin OS accanto a Windows (dual-boot) :





- Selezionare **Installa Zorin OS accanto a Windows Boot Manager**.



![capture](assets/fr/15.webp)





- Se non avete partizionato il disco, selezionate lo spazio che desiderate assegnare a Zorin OS, quindi fate clic su **Installa ora**.



![capture](assets/fr/16.webp)

![capture](assets/fr/18.webp)

### Applicazioni Windows



Per installare le applicazioni Windows, iniziare installando il pacchetto **zorin-windows-app-support** tramite il Terminale :



```bash
sudo apt install zorin-windows-app-support
```



Per un elenco delle applicazioni Windows compatibili e dei loro livelli di compatibilità, consultare la pagina [Wine Application Database] (https://appdb.winehq.org/). Lì troverete i seguenti badge, corrispondenti al livello di compatibilità (dal migliore al peggiore): Platinum, Gold, Silver, Bronze e Garbage.



Per installare un'applicazione Windows .exe o .msi, avete due opzioni:





- Aprite **PlayOnLinux** e fate clic sul pulsante **Install** per sfogliare le applicazioni e i giochi compatibili.



![capture](assets/fr/41.webp)





- Fare doppio clic sul file **.exe o .msi** dell'applicazione e lasciarsi guidare dal programma di installazione.
