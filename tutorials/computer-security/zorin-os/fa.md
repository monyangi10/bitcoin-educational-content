## مقدمه



یک سیستم‌عامل (OS) نرم‌افزار اساسی است که به یک کامپیوتر امکان عملکرد می‌دهد: آن مدیریت سخت‌افزار، نرم‌افزار، امنیت و رابط کاربری را بر عهده دارد.


Zorin OS یک توزیع لینوکس است که به‌طور خاص برای تسهیل انتقال از ویندوز طراحی شده است، در حالی که مزایای نرم‌افزار منبع باز را ارائه می‌دهد: امنیت، پایداری، حریم خصوصی و عملکرد.



بر اساس Ubuntu LTS، Zorin OS ترکیبی از سازگاری بالای نرم‌افزاری با یک رابط کاربری آشنا و قابل تنظیم است که آن را به یک جایگزین معتبر و قابل دسترس برای ویندوز تبدیل می‌کند.



## چرا Zorin OS؟





- Interface آشنا**: ظاهر شبیه به ویندوز (منوی شروع، نوار وظیفه)
- انتقال آسان**: طراحی شده برای کاربرانی که از ویندوز می‌آیند
- امنیت پیشرفته**: معماری لینوکس، کمتر در معرض ویروس‌ها
- احترام به حریم خصوصی**: جمع‌آوری داده‌های مزاحم انجام نمی‌شود
- عملکرد بهینه**: به خوبی روی ماشین‌های متوسط کار می‌کند
- پایه Ubuntu LTS**: ثبات، به‌روزرسانی‌های منظم و سازگاری گسترده
- شخصی‌سازی پیشرفته**: از طریق ابزار Zorin Appearance.

## دسترسی به کلید بوت و BIOS

وقتی کامپیوتر شما روشن می‌شود، به BIOS دسترسی پیدا کنید (`ESC`، `F9` یا `F11` بسته به برند) و کلید USB را به عنوان دستگاه بوت انتخاب کنید، سپس `Enter` را فشار دهید تا فرآیند بوت آغاز شود.

### برنامه‌های ویندوز



برای نصب برنامه‌های ویندوز، ابتدا بسته **zorin-windows-app-support** را از طریق ترمینال نصب کنید:



```bash
sudo apt install zorin-windows-app-support
```



برای فهرستی از برنامه‌های سازگار با ویندوز و سطح سازگاری آن‌ها، به صفحه [پایگاه داده برنامه‌های Wine] (https://appdb.winehq.org/) مراجعه کنید. در آنجا نشان‌های زیر را خواهید یافت که با سطح سازگاری (از بهترین به بدترین) مطابقت دارند: پلاتین، طلا، نقره، برنز و زباله.



برای نصب یک برنامه .exe یا .msi ویندوز، دو گزینه دارید:





- برنامه **PlayOnLinux** را باز کنید و روی دکمه **Install** کلیک کنید تا برنامه‌ها و بازی‌های سازگار را مرور کنید.



![capture](assets/fr/41.webp)





- روی فایل **.exe یا .msi** برنامه دوبار کلیک کنید و اجازه دهید برنامه نصب شما را راهنمایی کند.
