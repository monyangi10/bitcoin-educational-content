## Imenyekanisha



Operating system (OS) ni porogaramu y’ishimikiro ishoboza mudasobwa gukora: icungera ibikoresho, porogaramu, umutekano n’ivyo umuntu akoresha.


Zorin OS ni urubuga rwa Linux rwakozwe canecane kugira ngo rworohereza umuntu kuva kuri Windows, mu gihe rutanga inyungu z’ubuhinga bufunguye: umutekano, ugushikama, ubuzima bwite n’ubushobozi.



Ishingiye kuri Ubuntu LTS, Zorin OS ihuriza hamwe ubuhinga buhambaye bwo gukoresha porogarama n’ubuhinga bumenyerewe, bushobora guhindurwa, bikaba biyigira ubundi buryo bwo kwizigirwa kandi bushoboka kuruta Windows.



## Kubera iki Zorin OS?





- Interface imenyerewe**: Igaragara nk'i Windows (intango y'ibikubiyemo, umurongo w'ibikorwa)
- Impinduka yoroshe**: yagenewe abakoresha bava kuri Windows
- Umutekano wongerewe**: Ubwubatsi bwa Linux, budashobora gushikirwa na virusi
- Kwubaha ubuzima bwite**: nta gukusanya amakuru yinjira mu mutima
- Ibikorwa vyiza**: bikora neza ku mashini zisanzwe
- Ubuntu LTS** ishingiro: ugushikama, guhindura ubudasiba n'uguhuza kwagutse
- Guhindura umuntu ku giti ciwe**: biciye ku gikoresho ca Zorin Appearance.

## Urufunguzo rwo gufungura no gushika ku BIOS



Zima mudasobwa wipfuza gushiramwo Zorin OS, hanyuma ushiremwo urufunguzo rwa USB.


Iyo mudasobwa yawe itanguye, genda muri BIOS (`ESC`, `F9` canke `F11` bivanye n’ibara) hanyuma uhitemwo urufunguzo rwa USB nk’igikoresho co gufungura, hanyuma ukande `Enter` kugira ngo utangure igikorwa co gufungura.





- Ku gutangura, hitamwo **Gerageza canke Shiramwo Zorin OS**.



![capture](assets/fr/08.webp)





- Niba ufise ikarita y’ibishushanyo ya NVIDIA, hitamwo **Gerageza canke Shiraho Zorin OS (abashoferi ba NVIDIA bo muri iki gihe)**.
- Urasabwa kurindira amadosiye asuzumwa.



![capture](assets/fr/09.webp)





- Mu gikoresho co gushiramwo Zorin OS, uhitemwo ururimi **Igifaransa** hanyuma ukande kuri Install **Zorin OS**.



![capture](assets/fr/10.webp)





- Hitamwo uko klavye iteye.

### Ibikoresho vya Windows



Kugira ngo ushiremwo porogaramu za Windows, tangura ushiremwo **zorin-amadirisha-porogaramu-ifasha** biciye kuri Terminal :



```bash
sudo apt install zorin-windows-app-support
```



Kugira ngo ubone urutonde rw’ibikoresho vya Windows bihuye n’ingero z’uguhuza kwavyo, raba urupapuro [Urutonde rw’Ibikoresho vy’Umuvinyu] (https://appdb.winehq.org/). Aho uzosanga ibimenyetso bikurikira, bihuye n’urugero rw’uguhuza (kuva ku vyiza gushika ku bibi): Platine, Zahabu, Ifeza, Umuringa n’Imyanda.



Kugira ngo ushiremwo porogaramu ya Windows .exe canke .msi, ufise uburyo bubiri:





- Ugurure **PlayOnLinux** hanyuma ukande kuri buto ya **Install** kugira ngo ubone porogarama n’imikino bihuye.



![capture](assets/fr/41.webp)





- Fyonda kabiri kuri dosiye **.exe canke .msi** y’iyo porogarama maze ureke porogarama yo gushiramwo iyo porogarama ikuyobore.
