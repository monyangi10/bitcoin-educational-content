## Úvod



Operační systém (OS) je základní software, který umožňuje fungování počítače: spravuje hardware, software, zabezpečení a uživatelské rozhraní.


Zorin OS je linuxová distribuce navržená speciálně pro usnadnění přechodu z Windows a zároveň nabízí výhody open source softwaru: bezpečnost, stabilitu, soukromí a výkon.



Systém Zorin OS je založen na platformě Ubuntu LTS a kombinuje vysokou kompatibilitu softwaru se známým, přizpůsobitelným rozhraním, což z něj činí důvěryhodnou a dostupnou alternativu k systému Windows.



## Proč Zorin OS?





- Interface známý**: Vzhled podobný systému Windows (nabídka Start, hlavní panel)
- Snadný přechod**: navrženo pro uživatele přicházející ze systému Windows
- Zvýšené zabezpečení**: Linuxová architektura, méně náchylná k virům
- Respektování soukromí**: žádné rušivé shromažďování údajů
- Optimalizovaný výkon**: funguje i na skromných strojích
- Základ Ubuntu LTS**: stabilita, pravidelné aktualizace a široká kompatibilita
- Pokročilá personalizace**: pomocí nástroje Zorin Appearance.



## Instalace a konfigurace



### 1. Předpoklady



**Potřebné vybavení:**





- Klíč USB o velikosti alespoň **8 GB** (doporučuje se 12 GB);
- Počítač s alespoň **25 GB volného místa na disku**
- Připojení k internetu (doporučeno).



### 2. Stáhnout Zorin OS





- Navštivte oficiální webové stránky: [https://zorin.com/os](https://zorin.com/os)



![Page de téléchargement Balena Etcher](assets/fr/03.webp)





- Vyberte **Zorin OS Core** (doporučujeme bezplatnou verzi)



![Page de téléchargement Balena Etcher](assets/fr/04.webp)





- Stažení obrazu ISO



Zorin OS nabízí také :





- Zorin OS Lite** (starší počítače)
- Zorin OS Pro** (placený, s pokročilými funkcemi a podporou)

## Zavedení systému a přístup do systému BIOS pomocí kláves



Vypněte počítač, do kterého chcete nainstalovat systém Zorin OS, a poté připojte klíč USB.


Po spuštění počítače vstupte do systému BIOS (`ESC`, `F9` nebo `F11` v závislosti na značce) a vyberte klíč USB jako spouštěcí zařízení, poté stiskněte `Enter` pro spuštění spouštěcího procesu.

### Aplikace systému Windows



Chcete-li nainstalovat aplikace pro systém Windows, začněte instalací balíčku **zorin-windows-app-support** pomocí Terminálu :



```bash
sudo apt install zorin-windows-app-support
```



Seznam kompatibilních aplikací systému Windows a jejich úrovně kompatibility naleznete na stránce [Databáze aplikací Wine] (https://appdb.winehq.org/). Tam najdete následující odznaky odpovídající úrovni kompatibility (od nejlepší po nejhorší): Platinum, Gold, Silver, Bronze a Garbage.
