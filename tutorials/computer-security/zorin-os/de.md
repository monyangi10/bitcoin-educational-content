## Einführung



Ein Betriebssystem (OS) ist die grundlegende Software, die das Funktionieren eines Computers ermöglicht: Es verwaltet Hardware, Software, Sicherheit und die Benutzeroberfläche.


Zorin OS ist eine Linux-Distribution, die speziell entwickelt wurde, um den Umstieg von Windows zu erleichtern, und gleichzeitig die Vorteile von Open-Source-Software bietet: Sicherheit, Stabilität, Datenschutz und Leistung.



Zorin OS basiert auf Ubuntu LTS und kombiniert eine hohe Softwarekompatibilität mit einer vertrauten, anpassbaren Benutzeroberfläche, was es zu einer glaubwürdigen und zugänglichen Alternative zu Windows macht.



## Warum Zorin OS?





- Interface vertraut**: Windows-ähnliches Aussehen (Startmenü, Taskleiste)
- Einfacher Übergang**: Entwickelt für Benutzer, die von Windows kommen
- Verbesserte Sicherheit**: Linux-Architektur, weniger anfällig für Viren
- Achtung der Privatsphäre**: keine aufdringliche Datenerfassung
- Optimierte Leistung**: funktioniert auch auf bescheidenen Maschinen
- Ubuntu LTS** Basis: Stabilität, regelmäßige Updates und breite Kompatibilität
- Erweiterte Personalisierung**: über das Tool Zorin Appearance.



## Installation und Konfiguration



### 1. Voraussetzungen



**Erforderliche Ausrüstung:**





- Ein USB-Stick mit mindestens **8 GB** (12 GB empfohlen);
- Ein Computer mit mindestens **25 GB freiem Speicherplatz**
- Internetverbindung (empfohlen).



### 2. Zorin OS herunterladen





- Besuchen Sie die offizielle Website: [https://zorin.com/os](https://zorin.com/os)

### Windows-Anwendungen



Um Windows-Anwendungen zu installieren, installieren Sie zunächst das Paket **zorin-windows-app-support** über Terminal :



```bash
sudo apt install zorin-windows-app-support
```



Eine Liste der kompatiblen Windows-Anwendungen und ihrer Kompatibilitätsstufen finden Sie auf der Seite [Wine Application Database] (https://appdb.winehq.org/). Dort finden Sie die folgenden Abzeichen, die der Kompatibilitätsstufe entsprechen (von der besten bis zur schlechtesten): Platin, Gold, Silber, Bronze und Müll.



Um eine Windows .exe- oder .msi-Anwendung zu installieren, haben Sie zwei Möglichkeiten:





- Öffnen Sie **PlayOnLinux** und klicken Sie auf die Schaltfläche **Installieren**, um kompatible Anwendungen und Spiele zu suchen.



![capture](assets/fr/41.webp)





- Doppelklicken Sie auf die Datei **.exe oder .msi** der Anwendung und lassen Sie sich vom Installationsprogramm führen.
