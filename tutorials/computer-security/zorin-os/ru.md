## Введение



Операционная система (ОС) - это фундаментальное программное обеспечение, обеспечивающее функционирование компьютера: она управляет аппаратным и программным обеспечением, безопасностью и пользовательским интерфейсом.


Zorin OS - это дистрибутив Linux, разработанный специально для облегчения перехода с Windows и предлагающий преимущества программного обеспечения с открытым исходным кодом: безопасность, стабильность, конфиденциальность и производительность.



Основанная на Ubuntu LTS, Zorin OS сочетает в себе высокую совместимость программного обеспечения с привычным, настраиваемым интерфейсом, что делает ее надежной и доступной альтернативой Windows.



## Почему именно Zorin OS?





- Interface знаком**: Внешний вид, похожий на Windows (стартовое меню, панель задач)
- Легкий переход**: предназначен для пользователей, перешедших с Windows
- Повышенная безопасность**: Архитектура Linux, менее подверженная воздействию вирусов
- Уважение к частной жизни**: никакого навязчивого сбора данных
- Оптимизированная производительность**: хорошо работает на скромных машинах
- База Ubuntu LTS**: стабильность, регулярные обновления и широкая совместимость
- Расширенная персонализация**: с помощью инструмента Zorin Appearance.



## Установка и настройка



### 1. Пререквизиты



**Необходимое оборудование:**





- USB-носитель объемом не менее **8 ГБ** (рекомендуется 12 ГБ);
- Компьютер с не менее чем **25 ГБ свободного дискового пространства**
- Подключение к Интернету (рекомендуется).



### 2. Скачать Zorin OS





- Посетите официальный сайт: [https://zorin.com/os](https://zorin.com/os)



![Page de téléchargement Balena Etcher](assets/fr/03.webp)





- Выберите **Zorin OS Core** (рекомендуется бесплатная версия)



![Page de téléchargement Balena Etcher](assets/fr/04.webp)





- Загрузите образ ISO



Zorin OS также предлагает :





- Zorin OS Lite** (старые компьютеры)
- Zorin OS Pro** (платная, с расширенными возможностями и поддержкой)



## Создание загрузочного USB-носителя



Вы можете использовать несколько инструментов, например, Balena Etcher :





- Скачайте и установите [Balena Etcher](https://etcher.balena.io/).
- Откройте Balena Etcher, затем выберите образ Zorin ISO.
- Выберите USB-носитель в качестве целевого носителя.
- Нажмите Flash и дождитесь окончания процесса.

## Загрузка с помощью клавиш и доступ к BIOS



Выключите компьютер, на который вы хотите установить Zorin OS, затем подключите USB-носитель.


Когда компьютер загрузится, войдите в BIOS (`ESC`, `F9` или `F11` в зависимости от марки) и выберите USB-носитель в качестве загрузочного устройства, затем нажмите `Enter`, чтобы начать процесс загрузки.





- При запуске выберите **Попробовать или установить Zorin OS**.



![capture](assets/fr/08.webp)





- Если у вас видеокарта NVIDIA, выберите **Попробовать или установить Zorin OS (современные драйверы NVIDIA)**.
- Пожалуйста, подождите, пока файлы будут проверены.



![capture](assets/fr/09.webp)





- В программе установки Zorin OS выберите язык **Французский** и нажмите на кнопку Установить **Zorin OS**.



![capture](assets/fr/10.webp)





- Выберите раскладку клавиатуры.



![capture](assets/fr/11.webp)





- Установите флажки **Загружать обновления во время установки Zorin OS** и **Устанавливать стороннее программное обеспечение для графического и Wi-Fi оборудования и дополнительных форматов носителей**.



![capture](assets/fr/12.webp)





- Чтобы установить Zorin OS на весь диск, выберите **Стереть диск и установить Zorin OS**.



![capture](assets/fr/14.webp)

### Приложения для Windows



Чтобы установить приложения Windows, начните с установки пакета **zorin-windows-app-support** через Терминал :



```bash
sudo apt install zorin-windows-app-support
```



Список совместимых приложений для Windows и уровни их совместимости можно найти на странице [База данных приложений Wine] (https://appdb.winehq.org/). Там вы найдете следующие значки, соответствующие уровню совместимости (от лучшего к худшему): Platinum, Gold, Silver, Bronze и Garbage.
