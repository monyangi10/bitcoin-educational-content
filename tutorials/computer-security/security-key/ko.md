오늘날 2단계 인증(2FA)은 무단 액세스로부터 온라인 계정의 보안을 강화하는 데 필수적인 요소가 되었습니다. 사이버 공격이 증가함에 따라 비밀번호에만 의존해 계정을 보호하는 것은 때때로 불충분할 수 있습니다.


2단계 인증은 기존 비밀번호 외에 두 번째 인증 수단을 요구함으로써 보안을 한층 더 강화합니다. 이 인증은 SMS를 통해 전송된 코드, 전용 앱에서 생성된 동적 코드, 물리적 보안 키 사용 등 다양한 형태로 이루어질 수 있습니다. 2단계 인증을 사용하면 비밀번호를 도난당한 경우에도 계정이 유출될 위험이 크게 줄어듭니다.


다른 튜토리얼에서 TOTP 2FA 애플리케이션을 설정하고 사용하는 방법에 대해 설명했습니다:


https://planb.network/tutorials/computer-security/authentication/authy-a76ab26b-71b0-473c-aa7c-c49153705eb7

여기에서는 모든 계정에 대한 두 번째 인증 요소로 물리적 보안 키를 사용하는 방법을 살펴봅니다.


## 물리적 보안 키란 무엇인가요?


물리적 보안 키는 2단계 인증(2FA)을 통해 온라인 계정의 보안을 강화하는 데 사용되는 장치입니다. 이러한 장치는 종종 컴퓨터의 포트에 삽입하여 연결을 시도하는 사용자가 실제로 정당한 사용자인지 확인해야 하는 작은 USB 키와 비슷합니다.

![SECURITY KEY 2FA](assets/notext/01.webp)

2단계 인증으로 보호되는 계정에 로그인하고 물리적 보안 키를 사용할 때는 평소 사용하는 비밀번호를 입력할 뿐만 아니라 물리적 보안 키를 컴퓨터에 삽입하고 버튼을 눌러 인증을 확인해야 합니다. 따라서 이 방법을 사용하면 누군가 비밀번호를 알아내더라도 물리적으로 키를 소지하지 않으면 계정에 액세스할 수 없으므로 보안이 1단계 더 강화됩니다.


물리적 보안 키는 지식 증명(비밀번호)과 소유 증명(물리적 키)이라는 두 가지 유형의 인증 요소를 결합하기 때문에 특히 효과적입니다.


하지만 이 2단계 인증 방법에도 단점이 있습니다. 첫째, 계정에 액세스하려면 항상 보안 키를 사용할 수 있어야 합니다. 키 체인에 보안 키를 추가해야 할 수도 있습니다. 둘째, 다른 2FA 방법과 달리 물리적 보안 키를 사용하려면 소형 장치를 구입해야 하므로 초기 비용이 발생합니다. 보안 키의 가격은 일반적으로 선택한 기능에 따라 30유로에서 100유로 사이입니다.


## 어떤 물리적 보안 키를 선택해야 하나요?


보안 키를 선택하려면 몇 가지 기준을 고려해야 합니다.

무엇보다도 장치에서 지원하는 프로토콜을 확인해야 합니다. 최소한 OTP, FIDO2 및 U2F를 지원하는 키를 선택하는 것이 좋습니다. 이러한 세부 사항은 일반적으로 제조업체에서 제품 설명에 강조 표시합니다. 각 키의 호환성을 확인하려면 [dongleauth.com](https://www.dongleauth.com/dongles/)에서도 확인할 수 있습니다.

또한, 유비키와 같이 잘 알려진 브랜드는 일반적으로 널리 사용되는 모든 시스템을 지원하지만, 키가 운영 체제와 호환되는지 확인하세요.


또한 컴퓨터 또는 스마트폰에서 사용 가능한 포트 유형에 따라 키를 선택해야 합니다. 예를 들어 컴퓨터에 USB-C 포트만 있는 경우 USB-C 커넥터가 있는 키를 선택하세요. 일부 키는 블루투스 또는 NFC를 통한 연결 옵션도 제공합니다.

![SECURITY KEY 2FA](assets/notext/02.webp)

키의 모양과 크기뿐만 아니라 방수 및 Dust 저항과 같은 추가 기능을 기준으로 기기를 비교할 수도 있습니다.


보안 키 브랜드로는 유비코가 [유비키 디바이스](https://www.yubico.com/)로 가장 잘 알려져 있으며, 개인적으로도 사용하고 추천합니다. 구글도 [Titan 보안 키](https://store.google.com/fr/product/titan_security_key) 기기를 제공합니다. 오픈 소스 대안의 경우, [SoloKeys](https://solokeys.com/)(비 OTP)와 [NitroKey](https://www.nitrokey.com/products/nitrokeys)가 흥미로운 옵션이지만 테스트해 볼 기회는 없었습니다.


## 물리적 보안 키는 어떻게 사용하나요?


보안 키를 받은 후에는 특별한 설정이 필요하지 않습니다. 키는 일반적으로 수령 즉시 사용할 수 있습니다. 이 유형의 인증을 지원하는 온라인 계정을 보호하는 데 즉시 사용할 수 있습니다. 예를 들어, 이 물리적 보안 키를 사용하여 내 Proton 메일 계정을 보호하는 방법을 보여드리겠습니다.

![SECURITY KEY 2FA](assets/notext/03.webp)

계정 설정의 '*비밀번호*' 또는 '*보안*' 섹션에서 2FA를 활성화할 수 있는 옵션을 찾을 수 있습니다. 실제 키로 2FA를 활성화할 수 있는 확인란 또는 버튼을 클릭합니다.

![SECURITY KEY 2FA](assets/notext/04.webp)

컴퓨터에 키를 연결합니다.

![SECURITY KEY 2FA](assets/notext/05.webp)

보안 키의 버튼을 터치하여 인증합니다.

![SECURITY KEY 2FA](assets/notext/06.webp)

사용한 키를 기억하기 위해 이름을 입력합니다.

![SECURITY KEY 2FA](assets/notext/07.webp)

이제 계정의 2단계 인증을 위한 보안 키가 성공적으로 추가되었습니다.

![SECURITY KEY 2FA](assets/notext/08.webp)

![SECURITY KEY 2FA](assets/notext/09.webp)

그런 다음 두 번째 인증 요소로 보안 키를 연결하라는 메시지가 표시됩니다.

![SECURITY KEY 2FA](assets/notext/10.webp)

![SECURITY KEY 2FA](assets/notext/11.webp)

이러한 방식으로 보호하려는 모든 온라인 계정, 특히 이메일 계정, 비밀번호 관리자, 클라우드 및 온라인 저장소 서비스, 금융 계정과 같은 중요한 계정에 대해 이 작업을 반복합니다.
