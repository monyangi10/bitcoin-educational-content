ทุกวันนี้ การยืนยันตัวตนแบบสองปัจจัย (2FA) ได้กลายเป็นสิ่งจำเป็นสำหรับการเพิ่มความปลอดภัยของบัญชีออนไลน์จากการเข้าถึงโดยไม่ได้รับอนุญาต ด้วยการเพิ่มขึ้นของการโจมตีทางไซเบอร์ การพึ่งพาเพียงรหัสผ่านเพื่อรักษาความปลอดภัยของบัญชีของคุณบางครั้งอาจไม่เพียงพอ


2FA เพิ่มชั้นความปลอดภัยเพิ่มเติมโดยการต้องการรูปแบบการยืนยันตัวตนที่สองนอกเหนือจากรหัสผ่านแบบดั้งเดิม การยืนยันนี้สามารถมีรูปแบบต่าง ๆ เช่น รหัสที่ส่งผ่าน SMS, รหัสไดนามิกที่สร้างโดยแอปเฉพาะ หรือการใช้กุญแจความปลอดภัยทางกายภาพ การใช้ 2FA ช่วยลดความเสี่ยงที่บัญชีของคุณจะถูกบุกรุกได้อย่างมาก แม้ในกรณีที่รหัสผ่านของคุณถูกขโมย


ในบทแนะนำอื่น ฉันได้อธิบายวิธีการตั้งค่าและใช้งานแอปพลิเคชัน TOTP 2FA:

## กุญแจความปลอดภัยทางกายภาพคืออะไร?


กุญแจความปลอดภัยทางกายภาพเป็นอุปกรณ์ที่ใช้เพื่อเพิ่มความปลอดภัยให้กับบัญชีออนไลน์ของคุณผ่านการยืนยันตัวตนแบบสองปัจจัย (2FA) อุปกรณ์เหล่านี้มักมีลักษณะคล้ายกับกุญแจ USB ขนาดเล็กที่ต้องเสียบเข้ากับพอร์ตของคอมพิวเตอร์เพื่อยืนยันว่าเป็นผู้ใช้ที่ถูกต้องที่พยายามเชื่อมต่อจริงๆ

![SECURITY KEY 2FA](assets/notext/01.webp)

เมื่อคุณเข้าสู่ระบบบัญชีที่มีการป้องกันด้วย 2FA และใช้กุญแจความปลอดภัยแบบกายภาพ คุณจะต้องไม่เพียงแค่ป้อนรหัสผ่านปกติของคุณ แต่ยังต้องเสียบกุญแจความปลอดภัยแบบกายภาพเข้ากับคอมพิวเตอร์ของคุณและกดปุ่มเพื่อยืนยันการตรวจสอบสิทธิ์ วิธีนี้จึงเพิ่มชั้นความปลอดภัยเพิ่มเติม เพราะแม้ว่ามีคนสามารถได้รหัสผ่านของคุณ พวกเขาก็จะไม่สามารถเข้าถึงบัญชีของคุณได้หากไม่มีการครอบครองกุญแจนั้นทางกายภาพ


กุญแจความปลอดภัยทางกายภาพมีประสิทธิภาพเป็นพิเศษเพราะมันรวมปัจจัยการยืนยันตัวตนสองประเภทที่แตกต่างกัน: หลักฐานของความรู้ (รหัสผ่าน) และหลักฐานของการครอบครอง (กุญแจทางกายภาพ)


อย่างไรก็ตาม วิธีการ 2FA นี้ก็มีข้อเสียเช่นกัน ประการแรก คุณต้องมีคีย์ความปลอดภัยพร้อมใช้งานเสมอหากคุณต้องการเข้าถึงบัญชีของคุณ คุณอาจต้องเพิ่มมันลงในพวงกุญแจของคุณ ประการที่สอง ไม่เหมือนกับวิธีการ 2FA อื่น ๆ การใช้คีย์ความปลอดภัยแบบกายภาพมีค่าใช้จ่ายเริ่มต้นเนื่องจากคุณต้องซื้ออุปกรณ์ขนาดเล็กนี้ ราคาของคีย์ความปลอดภัยมักจะแตกต่างกันระหว่าง €30 ถึง €100 ขึ้นอยู่กับคุณสมบัติที่เลือก


## เลือกกุญแจความปลอดภัยทางกายภาพแบบไหน?


ในการเลือกกุญแจรักษาความปลอดภัยของคุณ จำเป็นต้องพิจารณาหลายเกณฑ์

ก่อนอื่นเลย คุณจำเป็นต้องตรวจสอบโปรโตคอลที่อุปกรณ์รองรับ อย่างน้อยที่สุด ฉันแนะนำให้เลือกคีย์ที่รองรับ OTP, FIDO2 และ U2F รายละเอียดเหล่านี้มักจะถูกเน้นโดยผู้ผลิตในคำอธิบายสินค้า เพื่อยืนยันความเข้ากันได้ของแต่ละคีย์ คุณยังสามารถเยี่ยมชม [dongleauth.com](https://www.dongleauth.com/dongles/) ได้อีกด้วย

นอกจากนี้ ให้แน่ใจว่ากุญแจนั้นเข้ากันได้กับระบบปฏิบัติการของคุณ แม้ว่ายี่ห้อที่เป็นที่รู้จักดีอย่าง Yubikey จะรองรับระบบที่ใช้กันอย่างแพร่หลายทั้งหมดก็ตาม


คุณควรเลือกกุญแจตามประเภทของพอร์ตที่มีอยู่บนคอมพิวเตอร์หรือสมาร์ทโฟนของคุณด้วย ตัวอย่างเช่น หากคอมพิวเตอร์ของคุณมีเพียงพอร์ต USB-C ให้เลือกกุญแจที่มีตัวเชื่อมต่อ USB-C กุญแจบางรุ่นยังมีตัวเลือกการเชื่อมต่อผ่าน Bluetooth หรือ NFC อีกด้วย

![SECURITY KEY 2FA](assets/notext/02.webp)

คุณยังสามารถเปรียบเทียบอุปกรณ์ตามคุณสมบัติเพิ่มเติม เช่น ความต้านทานต่อน้ำและฝุ่น รวมถึงรูปทรงและขนาดของปุ่มได้อีกด้วย


เกี่ยวกับแบรนด์กุญแจรักษาความปลอดภัย Yubico เป็นที่รู้จักมากที่สุดกับ [อุปกรณ์ YubiKey](https://www.yubico.com/) ซึ่งฉันใช้และแนะนำเป็นการส่วนตัว Google ก็มีอุปกรณ์ที่ชื่อว่า [Titan Security Key](https://store.google.com/fr/product/titan_security_key) สำหรับทางเลือกแบบโอเพ่นซอร์ส [SoloKeys](https://solokeys.com/) (ไม่ใช่ OTP) และ [NitroKey](https://www.nitrokey.com/products/nitrokeys) เป็นตัวเลือกที่น่าสนใจ แต่ฉันยังไม่มีโอกาสได้ทดสอบพวกมันเลย


## วิธีใช้กุญแจความปลอดภัยทางกายภาพ?


เมื่อคุณได้รับกุญแจความปลอดภัยของคุณแล้ว จะไม่มีการตั้งค่าเฉพาะที่จำเป็น กุญแจนี้พร้อมใช้งานทันทีเมื่อได้รับ คุณสามารถใช้มันเพื่อรักษาความปลอดภัยให้กับบัญชีออนไลน์ของคุณที่รองรับการยืนยันประเภทนี้ได้ทันที ตัวอย่างเช่น ฉันจะแสดงให้คุณเห็นวิธีการรักษาความปลอดภัยให้กับบัญชี Proton mail ของฉันด้วยกุญแจความปลอดภัยทางกายภาพนี้

![SECURITY KEY 2FA](assets/notext/03.webp)

คุณจะพบตัวเลือกในการเปิดใช้งาน 2FA ในการตั้งค่าบัญชีของคุณ ซึ่งมักจะอยู่ภายใต้ส่วน "*รหัสผ่าน*" หรือ "*ความปลอดภัย*". คลิกที่ช่องทำเครื่องหมายหรือปุ่มที่อนุญาตให้คุณเปิดใช้งาน 2FA ด้วยกุญแจทางกายภาพ.

![SECURITY KEY 2FA](assets/notext/04.webp)

![SECURITY KEY 2FA](assets/notext/08.webp)
