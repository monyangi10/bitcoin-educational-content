## Matrix ni iki?



Matrix ni **umurongo w’itumanaho wegerejwe** wagenewe gutuma haba uguhana ubutumwa, amadosiye n’amajwi/amasanamu hagati y’abakoresha n’ibikoresho, ata kwizigira ikigo gikuru. Udakunze ubutumwa bwa kera, Matrix ni **ibikorwa remezo vyuguruye**, bigereranywa na email: umuntu wese arashobora guhitamwo server canke akayikoresha we nyene, mu gihe agumana ubushobozi bwo guhanahana n’ibindi bice vy’urubuga.



Matrix itandukanywa n’ingingo ngenderwako zitatu z’ishimikiro:



### Itegeko, si ugusaba



Matrix si porogaramu nka WhatsApp canke Telegram.


Ni ururimi rusanzwe rushobora gukoreshwa n'amaporogarama menshi.


Mu yandi majambo, ubwoko bwinshi bwa porogarama za Element, FluffyChat, Cinny, Nheko n’izindi, zitanga uburyo bwo gushika ku rubuga rumwe rwa Matrix.



Ivyo bitanga umwidegemvyo wose: guhindura ibikorwa ata gutakaza abo mubonana, ubudasa bw’imirongo, kwigenga ku muguzi umwe.



![capture](assets/fr/03.webp)



### Urubuga rwegerejwe, ruhurikiye hamwe



Ubuhinga bwa Matrix bushingiye kuri **federation**, uburyo aho abakozi benshi bigenga bakorana.


Serveri yose (yitwa _homeserver_) irashobora kwakira abakoresha, ivyumba vy’ibiganiro no guhuza ubutumwa n’izindi serveri ziri ku rubuga.


Gutyo :





- nta kintu na kimwe kigenzura urutonde rwose;
- server irashobora kuzimangana ataco ikoze ku bindi bice vy’urubuga;
- buri shirahamwe canke umuntu wese arashobora gutunganya ikibanza ciwe.



Ico kigereranyo kiratuma haba **ukwihangana gukomeye** kandi kigaragaza agaciro k’ubusegaba bw’ubuhinga.



![capture](assets/fr/03.webp)



### Uburyo butekanye, bushizwemwo amakuru



Matrix ishigikira **ugushiramwo amakuru kuva ku mpera kugeza ku mpera (E2EE)** ku bijanye n’uguhanahana amakuru y’abantu ku giti cabo n’imigwi yashizwemwo amakuru.


Ubutumwa bushobora gusomwa gusa n’abaje mu nama, atari n’abakozi bo hagati.


Ubwo buryo buratuma umuntu ashobora kuvugana ataco amenyesheje uwundi muntu, mu gihe aguma afise uguserukira kw’itegeko n’ubushobozi bwo kwakira server yiwe bwite.



![capture](assets/fr/05.webp)



### Gukorana bidasanzwe



Kimwe mu bintu bikomeye vya Matrix ni ubushobozi bwayo bwo gukora nk'**ikiraro hagati y'uburyo butandukanye bwo guhanahana amakuru**. Ushimiye _ibiraro_, birashoboka guhuza :





- Telegram.
- WhatsApp
- Signal.
- Intumwa
- Guteba
- Ukutumvikana
- IRC, XMPP, n'ibindi.



Ivyo bituma bishoboka guhuza imiryango itatanijwe ku mbuga zitandukanye, mu gihe iguma igenzura ibikorwa remezo.



![capture](assets/fr/06.webp)



## None Matrix ikora gute?



Iki gice kirerekana imiterere y’imbere y’urubuga rwa Matrix kugira ngo umuntu atahure ingene abakoresha, abakozi n’ibikoresho bikorana muri iyo nzira y’ibidukikije yegerejwe. Matrix ishingiye ku bintu bitatu vy'ingenzi: _abakozi bo mu nzu_, akaranga, na _abaguzi_ bakoreshwa mu guhanahana amakuru.



### Abakozi: abakozi bo muhira



Matrix ikora ku maserukira yigenga yitwa _amaserukira yo mu nzu_.


Buri serveri yo mu nzu icungera :





- konti y'ukoresha yakira,
- ibiyago vy’ibanga n’ibibanza vyo kwiruhukiramwo abo bakoresha bagiramwo uruhara,
- guhuza n'izindi serveri z'urubuga.



Ama homeserver yose ahuye n’urubuga rwa Matrix ahita ahana ubutumwa n’ibintu biva mu vyumba vy’uburaro rusangi. Nk'akarorero:





- umukoresha yanditswe kuri server A ashobora kuvugana n'uwukoresha kuri server B,
- salon ishobora gukwiragizwa ku ba serveri amajana,
- nta n’umwe afise ububasha ku salon canke ku kibano cose.



Ico kigereranyo kirakomeye cane kandi kigatuma ishirahamwe canke umuntu wese ashobora gucungera ibikorwa remezo vyiwe.



### Ibiranga ibimenyetso



Buri mukoresha afise ikimenyetso kidasanzwe, citwa **MXID** (_Matrix ID_), kimeze nk'aderesi:



```bash
@nomdutilisateur:serveur.xyz
```



Gigizwe na :





- izina ry'ukoresha, ritangurirwa na **@**
- izina rya server iyo konti yaremeweko, ritangurirwa na **:**



Ingero:





- `@Alisi:Ikigereranyo.org`
- `@bened: umucungezi.bj`



Ico kimenyetso kirashoboza guhanahana amakuru n'uwundi muntu wese akoresha Matrix, ataravye kuri server ikomoka.



### Abaguzi ba Matrix (ibisabwa)



Kugira ngo ukoreshe Matrix, ukeneye kwifatanya n'ikoreshwa ryitwa **client Matrix**.



Abazwi cane ni :





- Ikintu (urubuga, telefone ngendanwa, mudasobwa)
- FluffyChat (kugendagenda)
- Cinny (urubuga ruto/ibiro)
- Nheko (ibiro)



Izo porogaramu ni interfaces gusa za :





- kugira ngo ubone ubutumwa,
- kohereza umwandiko, amashusho canke amadosiye,
- winjire canke ureme amaserukiramuco y'ubudandaji,
- guhamagara amajwi/amasanamu.



Porogaramu zose zivugana n’abakozi biciye ku nzira imwe nyene.



### Ivyumba n'ubutumwa bwihariye (DM)



Muri Matrix, uguhanahana bibera mu **vyumba** :





- Icumba gishobora kuba ica bose canke icabo
- irashobora kwakira abantu babiri canke ibihumbi .
- Bishobora gusangizwa hagati y'abakozi benshi
- ifise ikimenyetso kidasanzwe gitangura na **!**



Ubutumwa bwihariye ni ivyumba vyo gusangiriramwo abantu babiri, akenshi vyitwa **DM (Ubutumwa butaziguye)**.



Guhuza ivy’i Salon bibera mu gihe nyaco hagati y’abakozi bari muri iyo nama, bikaba bituma habaho ubumenyi butagira umurongo mu gihe haguma habaho ugusenyura ubutegetsi.



## Ni kuki ukoresha Matrix?



Matrix si ubundi buryo gusa bwo gukoresha ubutumwa buhuriweko: ni ubuhinga buhuye n’ivyo umuntu akeneye vy’ukuri mu bijanye n’ubusegaba bwa digitale, umutekano n’ugukorana neza. Hari impamvu nyinshi zituma abantu benshi, amashirahamwe n’inzego zihitamwo iyo protocole kugira ngo bavugane.



### Subira ugenzure ubutumwa bwawe



Ivyuma vyinshi vy’ubutumwa bikora ku buryo bumwe: umukinyi umwe ni we agenzura ama server, ukuronka, amakuru n’amategeko y’ikoreshwa. Ico kigereranyo kivuga ko umuntu yizigira rwose uwumuha ivyo akeneye.


Matrix afata uburyo butandukanye.


Umuntu wese arashobora guhitamwo aho azokwakira konti yiwe, canke mbere agashiraho server yiwe bwite. Nta kigo kiri mu rwego rwo kubuza umukoresha, gusaba ikimenyetso c’umuntu canke gutegeka guhindura amategeko.


Ubwo nyubakwa burasubiza ubwigenge ku bantu ku giti cabo no ku mashirahamwe. Ivy’itumanaho ntibigishingiye ku kwizigira ishirahamwe, ahubwo bishingiye ku masezerano yuguruye, yanditswe kandi ashobora kugenzurwa.



### Uguhanahana amakuru gutekanye, gushizwemwo amakuru



Matrix ishigikira ububiko bw’iherezo kugeza ku mpera ku biganiro vy’ibanga n’imigwi. Ubwo buryo butuma abaje mu nama ari bo bonyene bashobora gusoma ubutumwa, naho bwoba buca ku ba serveri b’abandi bo muri iyo federasiyo.



Iryo tegeko rikoresha ubuhinga bwa Megolm/Olm, bwagenewe canecane gutanga umutekano ukomeye mu bidukikije vy’ibikoresho vyinshi.



Ivyo bituma bishoboka ku :





- gukingira ibiyago bihambaye,
- kubuza kwinjira ata wemerewe (mbere no kuva kuri server y'umushitsi),
- kubungabunga ibanga mu kiringo kirekire.



Gushiramwo amakuru si uburyo bwo guhitamwo: ni igice nyamukuru c'amasezerano.



### Ntibigishingiye ku gusaba kumwe



Matrix si igikorwa, ahubwo ni umurongo.



Ubwo butandukanye bw'abaguzi buratanga :





- amahitamwo ahuye n’ivyo umuntu akeneye,
- ubushobozi bwo gukoresha Matrix ku bwoko bwose bw'igikoresho,
- nta kwizigira porogarama imwe.



Iyo umukiriya adakwiriye canke ahagarika kwitaho, uhitemwo gusa uwundi: konti irabandanya gukora nk’uko bisanzwe.



### Guhuza no guhuza imiryango itandukanye



Federation iremeza ko ama server atandukanye akorana mu gihe arongorwa yigenga.


Gutyo :





- ishirahamwe rishobora gucunga umukozi waryo bwite,
- abantu ku giti cabo bashobora kwifatanya n'abakozi ba bose,
- bose barashobora kuvugana nk’aho boba bari ku rubuga rumwe.



Ukwo guhinduranya bituma bishoboka kurema ibibanza vyo guhanahana amakuru bihuye n’ivyo umuntu akeneye vyose: amashirahamwe, amashirahamwe, imiryango, inzego canke imigambi y’inkomoko yuguruye.



Matrix irazwi cane cane mu nzego z’ubuhinga, amashirahamwe y’abaharanira uburenganzira bw’abantu, abashakashatsi, intwaro, no mu miryango ya Bitcoin.



### Gukorana bidasanzwe mu bijanye n'ubutumwa



Kimwe mu bintu bikomeye vya Matrix ni ubushobozi bwayo bwo **kwagura** ubucuruzi kubera ibiraro bishobora guhuza :





- WhatsApp
- Telegram.
- Signal
- Guteba
- Ukutumvikana
- IRC
- XMPP
- n'izindi nzira nyinshi .



Matrix rero iba igice gihuza abantu mu bijanye n’itumanaho, igahuza imiryango myinshi itatanye ku bikorwa bitandukanye.



Ukwo gukorana neza kuragabanya uguca ibice kandi kworosha ugukorana.



### Amasezerano y'ubuntu, yuguruye kandi aramba



Itegeko rya Matrix ryose ni inkomoko yuguruye kandi ryateguwe mu buryo buboneye.


Ivyo biremeza ivyiza vyinshi:





- ugutera imbere gukomeza kw’ingingo mfatirwako,
- ubushobozi umuntu wese afise bwo kugenzura kode,
- kwigenga ku mahinduka y’ubudandaji canke ya politike,
- ukwihangana kw’igihe kirekire.



Mu buryo butandukanye n’uburyo bwo gutanga ubutumwa bw’abanyagihugu, kazoza ka Matrix ntikavana n’ishirahamwe rimwe, ahubwo kavana n’umuryango w’isi yose n’ingingo ngenderwako yuguruye.



## None nokora gute konti ya Matrix?



Gukora konti ya Matrix biroroshe kandi ntibisaba ubuhinga bwo gukora. Abakoresha barashobora kwinjira muri server iriho, bagakora login maze bagatangura kuvugana ubwo nyene. Iki gice kiradondora intambwe z’ingenzi.



### Hitamwo server (ya bose canke yigenga)



Matrix ni urubuga rw’ubumwe: hariho ama server menshi (homeservers) arongowe n’imiryango itandukanye, imiryango canke abantu ku giti cabo. Guhitamwo umukozi bigena gusa _aho_ konti icumbikiwe, ariko ntibibuza guhanahana amakuru n'urubuga rwose.


**Hariho uburyo bubiri:**



### - Koresha umukozi wa bose



Uwo ni wo muti woroshe kuruta iyindi yose.


Ingero z'abakozi bazwi cane:





- _matrix.org_ (izwi cane)
- _uruzitiro.urubuga_
- abakozi b'umuryango (ubuhinga, ubuzima bwite, inkomoko yuguruye...)



Izo server zirabereye abakoresha bashasha bashaka kwiyandikisha ningoga.



### - Koresha umukozi w'ibanga



Ivyiza kuri :





- ishirahamwe,
- umuryango,
- umugambi w'inkomoko yuguruye,
- ikipe y'akazi,
- canke kugira ngo umuntu akoreshe mu buryo bw’ubusegaba, bwite.



Muri ivyo, umuntu ategerezwa gutwara umukozi (Synapse, Dendrite, Conduit...).


Utitaye kuri server uhisemwo, abakoresha barashobora kuvugana kubera federation.



### Rema konti intambwe ku yindi



Kubera ko Matrix ari porotokole yuguruye, ishobora gushikirwa n’ibikorwa vyinshi.


Nk’uko vyavuzwe haruguru, zitanga interfaces zitandukanye n’imikorere bivanye n’ibisabwa:





- Element**: umukiriya yuzuye kuruta abandi bose, aboneka ku mbuga zose.
- FluffyChat**: yoroshe, igezweho kandi ibereye ku matelefone ngendanwa.
- Nheko**: umukiriya w’uburemere buke, w’ubuhinga bwiza ku ma PC.
- SchildiChat**: Ibintu bitandukanye bifise ivyiza vy'ubuhinga.
- NeoChat**: yinjijwe mu bidukikije vya KDE.



Guhitamwo umukiriya ntaco bihindura kuri konti: vyose bikorana na server iyo ari yo yose ya Matrix.



### Intambwe za kera :





- Ugurure porogarama wahisemwo. Mu gihe cacu, ivyo tuzobikora dukoresheje [Ikintu](app.ikintu.io).
- Hitamwo "Rema konti".



![cover-kali](assets/fr/10.webp)



Kugira ngo bibe vyoroshe, ushobora gukora konti ya Matrix ukoresheje **Google, Facebook, Apple, GitHub canke GitLab**. Izo serivisi zizomenya gusa ko konti yabo yakoreshejwe kugira ngo bashobore gushika kuri Matrix: ivyo bizwi nka **uguhuza abantu**.



Kugira ngo ibanga ribe ryinshi, urashobora kandi kwiyandikisha n'amaboko uhisemwo **izina ry'ukoresha**, **ijambobanga** na **aderesi ya e-mail**.



Bivanye na server yatowe, **captcha** ishobora gusabwa, hamwe n'ukwemera **amabwirizwa y'ikoreshwa**.



Iyo umuntu amaze kwemezwa, haca harungikwa ubutumwa bwo kwemeza.


Fyonda gusa ku nzira kugira ngo ufungure konti yawe maze winjire kuri porogarama y’urubuga (Element) kugira ngo winjire mu biganiro vyawe vya mbere vya Matrix.

Ubu ufise konti kandi ukoresha verisiyo y’urubuga ya Element.

### Kwongerako uwo guhamagara



Kugira ngo utumire abagenzi mu kiganiro c'umugwi wawe, fyonda ku nziga ya `i` iri hejuru iburyo. Ivyo bica bifungura urupapuro rw’iburyo. Fyonda kuri "Abantu" kugira ngo ubone urutonde rw'abanywanyi bari muri iki cumba: ni wewe wenyene ukwiye kuba uriho muri iki gihe.



![cover-kali](assets/fr/12.webp)



Fyonda kuri "Invite to this room" iri hejuru y'urutonde rw'abantu hanyuma hazogufungukira ubutumwa kugira ngo ushobore gutumira abagenzi bawe ngo baze baze muri Matrix. Niba bamaze kwinjira muri Matrix, shiramwo ID yabo ya Matrix. Niba batariko, shiramwo aderesi yabo ya e-mail bazotumirwa ngo baze badufashe.



Nta nzira y'"umugenzi" iriho: umuntu ashobora guhura na we ni umuntu gusa bavuganye na we.



![cover-kali](assets/fr/13.webp)



Umuntu watumiye arashobora kwemera canke kwanka ubutumire. Nibemera, ukwiye kubabona bifatanya n’icumba. Uko vyinshi niko bigenda birahimbarwa!

### Gutegura server yawe bwite



Matrix vy’ukuri iza mu vyo yayo iyo ikoreshejwe hamwe n’umukozi w’umuntu ku giti ciwe.


Gukoresha homeserver yawe bwite bigufasha :





- kuguma ugenzura neza amakuru,
- gusobanura amategeko yayo bwite y’ikoreshwa,
- kwakira amakonti menshi (abagenzi, ikipe, umuryango),
- kandi n’ugutuma habaho ukwihangana gukomeye mu gihe hoba hariho amategeko canke ugucengera.



**Inyishu zihari:**





- Synapse**: ugushirwa mu ngiro kw’akahise kandi kwuzuye kuruta kwose.
- Dendrite**: iraremereye, irakomeye cane kandi yateguwe ku bw’akazoza k’umurongo.
- Conduit**: uburyo bukeyi bworoshe gukoresha.



**Ibisabwa:**





- izina ry'indangarubuga,
- imashini canke VPS,
- ubuhinga bukeyi bwo gutwara uburyo.



Naho vyosaba guhindura gatoyi, gucunga server yawe bwite bihindura Matrix igikoresho gikomeye kandi kiramvye.



### Kwifatanya n'amaserukiramuco yawe yambere y'ubudandaji



Matrix yizigira cane _ivyumba_ (ivyumba vy’uburaro).


Hariho amaserukiramuco y’ubudandaji ya Leta, ay’abikorera ku giti cabo, ay’abanyagihugu, ay’ubuhinga, ay’igihugu n’ay’isi yose.



**Uburyo butatu bwo kwinjira muri salon:**



1. **Biciye ku nzira y'ubutumire** (kenshi mu buryo bwa `matrix.to` URL).


2. **Kurondera izina rya salon** mu gisata.


3. **Mu kwinjiza ID yuzuye y’iyerekanwa**, nk’akarorero. :


`#amafaranga y'amafaranga: matrix.org`


`#abanyagihugu-benin:monsrv.bj`



Iyo umaze kwinjira, chatroom yigenza nk’umugwi w’amakuru wa kera, ufise amateka, ubuhinga bwo gupfuka, amadosiye, inyishu n’uguhamagara kw’amajwi/amasanamu, bivanye n’umukiriya akoresha.

## Kuja kure



Iyo umaze kumenya neza ivy’ishimikiro, Matrix iratanga uburyo bwinshi buteye imbere. Waba ushaka guhuza izindi nzira zo gutanga ubutumwa, kwakira server yawe canke gutunganya umuryango, iyo nzira y’ibidukikije iratunze cane.



### Ibiraro (Whatsap, Telegram, Signal, n'ibindi)



Ikiraro gihuza Matrix n’izindi nzira zo gutanga ubutumwa.


Nayo, ushobora kohereza no kwakira ubutumwa buva kuri :





- WhatsApp
- Telegram**
- Signal**
- Ubutumwa bwa Facebook
- Ukutumvikana
- Guteba**
- IRC** (IRC)
- n'abandi benshi .



### Ivyo ibiraro bishobora gukora





- Gushira hamwe ibiyago vyawe vyose muri Matrix
- Gutanga urubuga rwuguruye rwo gukorana n'ibikorwa vy'abanyagihugu
- Gucungera umuryango w'abantu benshi uri ahantu hamwe



Ibiraro bimwe bimwe ni ivy’ubutegetsi, ibindi ni ivy’abanyagihugu.


Bivanye n’igisata, bashobora gusaba :





- umukozi w'umuntu ku giti ciwe,
- inyishu y'inyongera,
- canke gukoresha ikiraro ca bose kiriho.



### Gukoresha Matrix ku muryango wa Bitcoin, umuryango canke umugambi



Matrix si igikoresho c’umuntu ku giti ciwe gusa.


Ishobora gukoreshwa mu gutunganya imigwi y’akazi, gutunganya imiryango yo mu karere canke gucunga ivy’itumanaho ry’imigambi.



**Ingero z'ikoreshwa:**





- Imiryango y'inkomoko yuguruye
- Bitcoin n'imigambi y'ibidukikije y'umuravyo
- Amatsinda y'abanyeshure canke abateza imbere
- Imiryango y'abanyagihugu
- Itangazamakuru ryigenga
- Imigwi n'amashirahamwe yo mu karere



**Kubera iki ivyo bishimishije?





- 100% inkomoko yuguruye** igikoresho
- Itangazamakuru ry'ubusegaba n'iry'intara**
- Ivyumba vyatunganijwe mu **vyumba vyo kwiruhukiramwo**, **imigwi mitomito**, **ivyumba vyo kwiruhukiramwo vy'abantu ku giti cabo**, n'ibindi.
- Kwinjirana n'igicu gikurikira, GitLab, ikintu gihambaye, canke ubuhinga busanzwe
- Uburongozi bwiza bw'uburenganzira n'ugucungera



Matrix rero iba inkingi y’itumanaho ku ntumbero iyo ari yo yose yipfuza kuguma yigenga ku bibanza binini binini bihurikiye hamwe.



## Iciyumviro



Matrix iserukira umuti w’ubuhinga bwa none, wuguruye kandi utekanye wo guhanahana amakuru mu gihe nyaco, utanga uburyo bundi bwo guhanahana amakuru bushingiye ku nzira za kera. Kubera ubuhinga bwayo bwo gukorana n’abandi be n’imirongo ngenderwako y’ugushiramwo amakuru, iratuma abakoresha bashobora kuguma bagenzura amakuru yabo mu gihe bariko bararonka ubuhinga bwo gukorana n’abandi. Yaba ari iyo gukoresha ku giti cawe, ku kazi canke ku kibano, Matrix itanga uburyo bukomeye kandi bushobora gukoreshwa mu kwubaka ibidukikije vyo guhanahana amakuru bihuye n’ivyo abantu bakeneye muri iki gihe.
