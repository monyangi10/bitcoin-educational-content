在这个数字监控无处不在的世界，保护您的网络隐私从未像现在这样重要。公司使用复杂的技术来跟踪你：





- 第三方 cookie：由外部网站存入的小文件，用于从一个网站跟踪您到另一个网站
- 指纹：收集您的浏览器和设备的独特特征（屏幕分辨率、安装的字体、插件等），以便在没有 cookie 的情况下识别您的身份。
- **跟踪脚本**：分析你的浏览行为（点击、滚动、花费时间）的隐形 JavaScript 代码
- **IP Address 分析**：您的互联网服务提供商的地理位置和标识



然后，这些数据会被组合起来，形成你上网行为的详细资料，并在你不知情的情况下被货币化。这一现实提出了一个基本问题：如何才能在匿名和保密的情况下上网？



答案主要在于您选择的网络浏览器。这个我们每天用来获取信息、购物或交流的工具，在保护我们的个人数据方面起着决定性的作用。不幸的是，谷歌 Chrome 浏览器等流行浏览器（约占全球市场的 65%）的设计都是基于大量收集用户数据的商业模式。

面对这一挑战，新的参与者正在崛起，他们的理念与众不同：将隐私放在设计的核心位置。其中，Mullvad 浏览器脱颖而出，成为将最佳隐私保护与流畅、易用的浏览体验相结合的创新解决方案。



传统的浏览器会通过收集用户数据来实现个性化体验，而 Mullvad 浏览器则与之相反：让所有用户在网站上看起来一模一样，从而几乎不可能实现个性化跟踪。



在本教程中，我们将一起了解 Mullvad 浏览器如何改变您浏览互联网的方式，在不牺牲易用性的情况下为您提供强大的监控保护。




## 介绍 Mullvad 浏览器



**Mullvad浏览器**是瑞典公司Mullvad VPN与Tor项目合作开发的一款注重隐私的网络浏览器，由Mullvad VPN免费发布。它于2023年4月推出，自称是**"没有Tor网络的Tor浏览器 "**，旨在最大限度地减少在线跟踪和指纹识别，同时允许用户通过可信的VPN而不是Tor网络进行浏览。



Mullvad 浏览器是一款免费的开源浏览器，基于火狐 ESR（Mozilla Firefox 的长效版本），并由 Tor 项目专家进行了加固。具体来说，它包含了 Tor 浏览器的大部分**保护功能，但不通过 Tor 网络流量**。相反，用户可以（也应该）将其链接到可信的加密 VPN，以匿名化其 IP Address。



在用户体验方面，Mullvad 浏览器类似于经典浏览器，提供流畅的导航。它可在 Windows、macOS 和 Linux（无手机版）上免费使用。你不需要成为 Mullvad VPN 用户就可以使用它；不过，**在不屏蔽 IP 的情况下使用 Mullvad 浏览器并不能提供完全的匿名**，因此强烈建议与可靠的 VPN 结合使用。



### 目标：隐私和反跟踪



Mullvad 浏览器的设计有一个主要目标： **保护用户在线隐私**，抵制常见的跟踪和特征分析技术。其主要目标包括





- 大幅减少网站和广告公司的广告跟踪和追踪。默认情况下，Mullvad 浏览器会阻止可识别你身份的第三方跟踪器、跟踪 cookie 和指纹脚本。





- 将浏览器的指纹**标准化，以"融入人群"**。指纹就像一张独一无二的 "身份证"，由浏览器的所有特征组合而成。Mullvad 浏览器可确保所有用户都拥有完全相同的 "身份证"，从而无法将他们单独区分开来。





- 无需额外扩展即可提供即时保护。Mullvad 浏览器采用 "即用型" 配置：用户无需安装一系列扩展程序或修改任何设置即可获得保护。





- 不要牺牲任何必要的性能或人体工学。在没有 Tor 路由的情况下，Mullvad 浏览器的浏览速度比 Tor 浏览器快得多，接近标准浏览器和 VPN 的性能。



### Mullvad 浏览器的主要功能



Mullvad 浏览器包含一系列**安全和隐私功能**，其灵感直接来源于 Tor 浏览器：





- 始终保持私密浏览：**私密浏览模式默认为激活状态，无法停用。**从一个会话到下一个会话，不会存储历史记录、cookies 或缓存。关闭 Mullvad 浏览器后，所有浏览数据将被删除。





- 增强的指纹识别保护：**浏览器采用严格的设置来阻止数字指纹识别。这包括**
- 用户代理**和浏览器版本**标准化
- 所有用户的时区设置为 **UTC**
- **信箱**：这是一种在网页周围自动添加灰色边距的技术，以统一显示尺寸，防止根据屏幕尺寸进行识别
- 阻止指纹 API：禁用画布（2D 绘图）、WebGL（3D 图形）和 AudioContext（音频处理）技术，因为它们会泄露您硬件的独特细节
- 标准化系统字体，避免被已安装的字体识别





- 阻止跟踪器和广告：**Mullvad 浏览器原生集成了 uBlock Origin 扩展（预装）和附加保护列表，可阻止第三方跟踪器、广告脚本和其他恶意内容**。这种保护措施还伴有**第一方隔离**：一种为每个网站单独 "存储 "cookie的技术，可防止一个网站读取另一个网站存储的cookie。





- 会话重置按钮：与 Tor 浏览器的 "新身份 "按钮一样，Mullvad 浏览器也提供了一个按钮，用于**快速重启浏览器，使用新的空白会话**。





- 可调整的安全级别：**您可以在设置中调整安全级别（*正常*、*较安全*、*最安全*），就像在 Tor 浏览器中一样。**



## 默认内置扩展



Mullvad 浏览器包含三个预装扩展程序，它们构成了反跟踪保护的核心。**切勿删除或修改它们的配置**，因为这将使你成为独一无二的 Mullvad 浏览器用户：



### **uBlock Origin**


此广告和跟踪器拦截扩展预配置了**优化的过滤列表**，可拦截.NET广告和跟踪器：




- 侵扰性广告
- 收集您数据的第三方跟踪器
- 恶意脚本
- 行为跟踪 Elements



Mullvad 浏览器中的 uBlock Origin 使用标准化参数，确保所有用户屏蔽完全相同的 Elements，从而保持数字足迹的统一性。



### **NoScript**


NoScript 在后台运行，管理浏览器的**安全级别**。此：




- 根据所选级别（正常/最安全/最安全）控制 **JavaScript** 的执行
- 自动过滤 **XSS（跨站脚本）攻击**
- 阻止非 HTTPS 网站上**危险的活动内容**
- 其图标默认情况下是隐藏的，但可通过 "自定义工具栏 "显示



### **Mullvad浏览器**扩展


根据你是否是 Mullvad VPN 客户，这个 Mullvad 专用扩展提供不同的功能：



#### **未订阅 Mullvad VPN：**




- **基本连接检查**：显示您当前的公共 IP 和一些连接信息
- **隐私建议**：改进安全设置的提示（DNS、仅 HTTPS、搜索引擎）
- **WebRTC** 控制：启用/禁用，以防止 IP Address 泄漏
- 如果不使用 Mullvad VPN，可以在不影响**正常使用**的情况下删除您的数字足迹



#### **订阅 Mullvad VPN：**


该扩展通过先进的功能充分展现了其潜力：





- 集成 SOCKS5 代理：一键连接到 Mullvad VPN 代理服务器
- 固定 IP Address：VPN 可以更改其 IP Address，而代理则不同，它始终保证输出相同的 Address
- 自动关闭开关：如果 VPN 断开连接，浏览器流量会立即被阻止
- 支持 **IPv6**：即使您的 VPN 连接未启用 IPv6，也能实现 IPv6 连接





- 多跳（双 VPN）**：能够更改代理位置，在隧道内创建隧道**
 - 您的流量首先通过您的 VPN 服务器，然后 "跳转 "到另一个 Mullvad 服务器
 - 仅对浏览器使用不同的本地化语言





- **高级连接监控**：实时监控 VPN 状态、连接的服务器和 DNS 泄露检测





- 访问 **Mullvad Leta**：为用户保留的私人搜索引擎（但由于与您的帐户相关，Mullvad 并不推荐使用该搜索引擎）



这三种扩展共同创建了一个连贯的保护生态系统，每个用户都能从完全相同的防御系统中获益，而不会因为定制而损害集体匿名性。



## Mullvad 浏览器的优缺点



### 益处





- 默认情况下的出色隐私保护：**Mullvad 浏览器从一开始就应用了非常严格的隐私设置，无需手动配置。**





- 性能优于 Tor 浏览器：** 在没有洋葱路由的情况下，Mullvad 浏览器的**速度和响应速度明显优于 Tor 浏览器。





- 熟悉的 Interface 简洁性：**Mullvad 浏览器基于 Firefox 的 Interface。如果你用惯了火狐甚至 Tor 浏览器，就不会感到不适应。**





- 值得信赖的合作和经过审核的代码：**Mullvad 浏览器得益于 Tor 项目的专业技术，所有源代码都可供外部审核。**



### 缺点





- 没有 VPN 就无法实现网络匿名：** 最重要的一点是，**Mullvad 浏览器不会自行隐藏您的 IP Address**（与所有其他浏览器一样，Tor 浏览器除外）。您的 IP Address 就像您在互联网上的 "邮政 Address"：它会显示您的位置和您的 ISP。因此，它在很大程度上依赖于 VPN**（虚拟专用网络）来隐藏这一关键信息。





- 没有手机版本：**目前，Mullvad 浏览器只在 PC（Windows、Mac 和 Linux）上提供。**





- 与某些习惯不兼容：**永久私人模式**意味着您无法在一次使用中保持会话。无法从一个会话连接到下一个会话。





- 受限功能：**为保持指纹的统一性，Mullvad 浏览器禁用了火狐浏览器中的若干功能**，且不用于自定义。

## 配置和首次使用



首次启动 Mullvad 浏览器时，你会看到一个与 Tor 浏览器非常相似的 Interface。该浏览器已为保护隐私进行了预配置，无需进行特殊修改。



![MULLVAD BROWSER](assets/fr/04.webp)



*Interface Mullvad 浏览器主页带扩展功能，扫帚图标改为 generate 新标识，右上方为应用程序菜单。*



**重要提示：** Mullvad 浏览器默认不屏蔽您的 IP Address。为提供全面保护，我们强烈建议同时使用 VPN。您可以使用 Mullvad VPN 或任何其他可信的 VPN 服务。



该浏览器还包含**DNS-over-HTTPS（DoH）**，使用 Mullvad 的 DNS 服务：该技术会对您的 DNS 请求（将网站名称转换为 IP 地址）进行加密，以防止您的 ISP 监控您访问的网站。



### 安全设置



您可以点击右上角的**应用程序菜单**（三个横条），然后点击**"设置 "**，再点击**"隐私与安全 "**选项卡来调整安全级别。向下滚动到**"安全 "**部分：



![MULLVAD BROWSER](assets/fr/05.webp)



*安全级别选择：箭头表示从应用程序菜单到 "隐私与安全 "选项卡再到安全选项的路径*。



Mullvad 浏览器提供三个安全级别：





- 正常**（当前默认级别）**：启用所有浏览器和网站功能





- **更安全**：禁用通常具有危险性的网站功能，这可能会导致某些网站失去功能：
 - 非 HTTPS 网站禁用 JavaScript
 - 禁用某些字体和数学符号
 - 声音和视频（HTML5 媒体）以及 WebGL 均可 "点击播放"。





- **最安全**：只允许使用静态网站和基本服务所需的网站功能：
 - 所有网站默认禁用 JavaScript
 - 禁用某些字体、图标、图像和数学符号
 - 声音和视频（HTML5 媒体）以及 WebGL 均可 "点击播放"。



### 新建会话按钮



要在不关闭浏览器的情况下重新启动空白会话，请单击扫帚图标或使用应用程序菜单 > **"新建会话 "**。



![MULLVAD BROWSER](assets/fr/06.webp)



*重置身份 "功能可使用全新会话重启浏览器*。



## 日常使用



### 正常导航



Mullvad 浏览器在日常浏览中的表现与经典浏览器无异。您可以像往常一样访问所有网站，同时还能增强防跟踪功能。



### Cookie 管理和登录



由于采用永久保密模式，因此每次登录时都必须重新连接账户。这就是为获得最大保密性而付出的代价。



### 扩展



从技术上讲，Mullvad 浏览器允许你从 Firefox 目录中安装额外的扩展，但**重要的是要了解**的影响。每个新增扩展都会改变你的数字足迹，并将你与其他 Mullvad 浏览器用户区分开来，这有悖于浏览器的基本原则：让所有用户看起来完全一样。



正如穆尔瓦德所解释的那样 *"有时，没有特定的防御措施比有防御措施更好。如果你想增加网络隐私，你安装的扩展程序最终会让你变得更加显眼。"*



如果你还是选择安装扩展，请注意你正在创建一个独一无二的指纹，它可以用来在各个网站之间追踪你。为了获得最大程度的保护，最好坚持使用三个预装扩展，这三个扩展对所有用户都是一样的。



## 使用 Mullvad 浏览器的最佳实践



1. **始终使用 VPN：Mullvad 浏览器不会屏蔽你的 IP。要实现完全匿名，必须使用 VPN。**



2. **不要自定义浏览器**：避免更改设置或添加扩展程序，因为这可能会使您的身份被识别。



3. **使用新会话按钮**：在不同的活动之间，使用重置功能来分割会话。



4. **选择最适合您需求的安全级别** ：




- 正常（推荐）：适合日常浏览。已提供出色的保护，同时保持网站功能。这对 95% 的用户来说是最佳平衡。
- 更安全：如果您访问的是未知或有潜在危险的网站，或在公共 Wi-Fi 网络上需要额外保护。某些网站可能会出现故障。
- 最安全：用于高风险情况（新闻调查、敏感通信、敌对环境）。大多数现代网站都会被破坏，但这是最高安全性的代价。



5. **定期检查更新**：使用最新的安全补丁更新浏览器。



6. **使用隐私友好型搜索引擎**：选择 DuckDuckGo、Startpage 或 Searx 而不是 Google。



7. **只启用 HTTPS 模式**：在设置中，确保启用 "仅限 HTTPS "模式，以强制安全连接。



8. **请勿更改任何高级设置**：与其他浏览器不同，Mullvad 浏览器的设计使所有用户拥有完全相同的配置。修改 `about:config` 中的设置或更改 uBlock Origin/NoScript 设置会使你成为独一无二的用户，并完全破坏浏览器的匿名性。



## 建议的 DNS 配置



Mullvad 浏览器自动集成了 Mullvad DNS-over-HTTPS。如果你正在使用 Mullvad VPN，扩展会建议你**禁用 Mullvad DoH**，因为使用 VPN 服务器的 DNS 服务器会更快。如果你没有使用 Mullvad VPN，请继续启用 Mullvad DoH，以避免 ISP 监控 DNS。



## 结论



Mullvad 浏览器是一款优秀的解决方案，适合那些寻求隐私友好型网络浏览而又不受 Tor 网络性能限制的用户。与高质量的 VPN 结合使用，它能提供强大的保护，防止网络跟踪和监控。



Mullvad 浏览器虽然有一定的局限性（没有移动版本、非持续会话），但对于希望重新获得数字隐私控制权的人来说，它是一个非常有价值的工具。它的易用性和默认配置使其成为注重隐私的用户的明智之选，无论是初学者还是资深用户。
