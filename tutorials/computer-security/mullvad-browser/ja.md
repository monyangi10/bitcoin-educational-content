その答えは、ウェブブラウザの選択にあります。情報へのアクセス、買い物、コミュニケーションに毎日使うこのツールは、個人データの保護に決定的な役割を果たす。残念ながら、グーグル・クローム（世界市場の約65％を占める）のような人気ブラウザは、ユーザーデータの大量収集に基づくビジネスモデルに基づいて設計されている。

## Mullvadブラウザのご紹介



**Mullvadブラウザ**は、Torプロジェクトと共同で開発され、スウェーデンの会社Mullvad VPNによって無料で配布されているプライバシーに焦点を当てたウェブブラウザです。2023年4月に発表されたこのブラウザは、**"TorネットワークなしのTorブラウザ "**であり、ユーザーがTorネットワークではなく、信頼できるVPN経由でブラウジングできるようにしながら、オンライントラッキングとフィンガープリンティングを最小限に抑えるように設計されています。



Mullvad Browserは、Firefox ESR (Mozilla Firefoxの長寿バージョン)をベースに、Torプロジェクトの専門家によって強化された無料のオープンソースブラウザです。具体的には、Tor Browserの**保護機能のほとんど**を含みますが、**Torネットワーク**を経由してトラフィックをルーティングしません。その代わりに、ユーザーは自分のIP Addressを匿名化するために、信頼できる暗号化されたVPNとリンクさせることができます（そしてそうすべきです）。



ユーザーエクスペリエンスの面では、Mullvad Browserはクラシックなブラウザに似ており、流れるようなナビゲーションを提供する。Windows、macOS、Linuxで無料で利用できます（モバイル版はありません）。Mullvad VPNに加入していなくても使用できますが、**IPをマスキングせずにMullvad Browserを使用しても完全な匿名性は得られません** - そのため、信頼できるVPNと組み合わせて使用することを強くお勧めします。



### 目的：プライバシーと追跡防止



Mullvadブラウザは、1つの主な目標を念頭に置いて設計されています： **ユーザーのプライバシー**をオンラインで保護し、一般的なトラッキングやプロファイリング技術に対抗することです。その主な目的は以下の通りです：





- ウェブサイトや広告代理店による広告追跡やトラッキング**を大幅に削減します。デフォルトでは、Mullvad Browserは、サードパーティのトラッカー、トラッキングクッキー、およびあなたを特定する可能性のあるフィンガープリントスクリプトをブロックします。





- ブラウザのフィンガープリント**を標準化し、「群衆に溶け込む」ようにしましょう。フィンガープリントは、ブラウザのすべての特徴を組み合わせて作成されるユニークな「IDカード」のようなものです。Mullvad Browserは全てのユーザーが全く同じ「IDカード」を持つことを保証し、個人を区別することを不可能にします。**





- 拡張機能を追加することなく、即座に保護を提供します。Mullvad Browserは**すぐに使える**コンフィギュレーションで提供されます：ユーザーは保護されるために拡張機能の数々をインストールしたり、設定を変更したりする必要はありません。





- パフォーマンスや人間工学**を**必要以上に犠牲にしないでください。Torルーティングがない場合、Mullvad BrowserはTor Browserよりもはるかに高速なブラウジングを提供し、VPNと組み合わせた標準的なブラウザのパフォーマンスに近づきます。



### Mullvadブラウザの主な機能



Mullvad Browserには、Tor Browserに直接インスパイアされた一連の**セキュリティとプライバシー機能**が含まれています：





- 常にプライベートブラウジング:**プライベートブラウジングモードはデフォルトで有効になっており、無効にすることはできません。 **履歴、クッキー、キャッシュはセッションごとに保存されません。Mullvad Browserを閉じると同時に、すべての閲覧データは削除されます。





- **フィンガープリンティングに対する強化された保護:** ブラウザは、デジタル・フィンガープリンティングを阻止するための厳格な設定を適用します。これには以下が含まれます：
- ユーザーエージェント**とブラウザのバージョンの標準化**
- 全ユーザーのタイムゾーンを**UTC**に設定
- レターボックス：ウェブページの周囲にグレーの余白を自動的に追加することで、表示サイズを標準化し、画面寸法による識別を防止する技術。
- ブロック・フィンガープリンティング**API**：Canvas（2D描画）、WebGL（3Dグラフィックス）、AudioContext（オーディオ処理）テクノロジーは、ハードウェアに関する固有の詳細を明らかにする可能性があるため、無効化されています。
- 標準化されたシステムフォント**により、インストールされたフォントによる識別を回避**





- **トラッカーと広告のブロック:** Mullvad Browserは、**サードパーティのトラッカー、広告スクリプト、およびその他の悪意のあるコンテンツ**をブロックするための追加保護リストと**uBlock Origin**拡張機能（プリインストール）をネイティブに統合しています。この保護は、**First-Party Isolation**（ファーストパーティ・アイソレーション）を伴っています。





- **セッションリセットボタン:** Tor Browserの "New Identity "ボタンのように、Mullvad Browserは**新しい空白のセッション**でブラウザを素早く再起動するためのボタンを提供します。





- **調整可能なセキュリティレベル:** Tor Browserと同様に、設定でセキュリティレベル(*Normal*, *Safer*, *Safest*)を調整できます。



## デフォルトの内蔵エクステンション



Mullvad Browserには、トラッキング防止の核となる**3つの拡張機能がプレインストールされています。 **Mullvad Browserユーザーの中で唯一無二の存在となるため、これらを削除したり、設定を変更したりしないことが重要です：



### **ブロック・オリジン**


この広告とトラッカーをブロックする拡張機能には、.NET Frameworkをブロックするための**最適化されたフィルターリスト**があらかじめ設定されています：




- 押し付けがましい広告
- お客様のデータを収集する第三者トラッカー
- 悪意のあるスクリプト
- 行動追跡 Elements



Mullvad BrowserのuBlock Originは、標準化されたパラメータを使用し、すべてのユーザーが全く同じElementsをブロックすることを保証し、デジタルフットプリントの均一性を保ちます。



### **ノースクリプト**


NoScript はバックグラウンドで実行され、ブラウザの **セキュリティレベル** を管理します。この：




- **JavaScript**の実行を、選択したレベル（Normal/Most Secure/Most Secure）に従って制御する。
- **XSS（クロスサイトスクリプティング）攻撃**を自動的にフィルタリングします。
- 非HTTPSサイトの危険な**アクティブコンテンツ**をブロックする
- アイコンはデフォルトでは非表示ですが、"ツールバーのカスタマイズ "で表示できます。



### **マルバッド・ブラウザ**拡張機能


このMullvad専用の拡張機能は、あなたがMullvad VPNの顧客であるかどうかによって異なる機能を提供します：



#### **Mullvad VPNに加入していない場合:**。




- **基本的な接続チェック**：現在のパブリックIPといくつかの接続情報を表示します。
- **プライバシーに関する推奨事項**：セキュリティ設定を改善するためのヒント（DNS、HTTPSのみ、検索エンジン）
- **WebRTC**コントロール：IP Addressのリークを防ぐために有効/無効を切り替える
- Mullvad VPNを使用しない場合、デジタルフットプリントへの影響**なく**削除することができます。



#### **Mullvad VPN のサブスクリプションの場合:**。


このエクステンションは、高度な機能によってその潜在能力をフルに発揮します：





- 統合SOCKS5プロキシ**：ワンクリックでMullvad VPNサーバープロキシに接続
- **固定IP Address**: IP Addressを変更できるVPNとは異なり、プロキシは常に同じ出力Addressを保証する。
- **自動キルスイッチ**：VPNが切断された場合、ブラウザのトラフィックは即座にブロックされます。
- **IPv6サポート**：VPN接続でIPv6が有効になっていなくても接続可能





- マルチホップ（2重VPN）**：プロキシの場所を変えてトンネル内にトンネルを作る機能**
 - トラフィックはまずVPNサーバーを通過し、その後別のMullvadサーバーに「ジャンプ」します。
 - ブラウザのみ異なるローカライズを使用する





- **高度な接続監視**：VPNステータス、接続サーバー、DNSリーク検出をリアルタイムで監視します。





- **Mullvad Leta**へのアクセス：加入者専用のプライベート検索エンジン（Mullvadはアカウントとの関連性から推奨していませんが）



これら3つの拡張機能が連携することで、集団の匿名性を損なうようなカスタマイズの可能性なしに、すべてのユーザーがまったく同じ防御の恩恵を受けられる、首尾一貫した保護のエコシステムが構築される。



## Mullvadブラウザの長所と短所



### メリット





- デフォルトで優れたプライバシー保護: **Mullvad Browserは、手動で設定する必要がなく、最初から非常に厳格なプライバシー設定を適用します。**





- Tor Browserよりも優れたパフォーマンス:** オニオンルーティングがない場合、Mullvad BrowserはTor Browserよりも**クラシックなウェブブラウジングにおいて**際立って速く、応答性が高い**。





- **使い慣れたInterfaceのシンプルさ:** Mullvad BrowserはFirefoxのInterfaceをベースにしています。FirefoxやTor Browserに慣れていれば、違和感を感じることはないでしょう。





- 信頼できる共同作業と監査されたコード: **Mullvad BrowserはTor Projectの専門知識の恩恵を受けており、すべてのソースコードは外部監査のために利用可能です。**



### デメリット





- VPNなしではネットワークの匿名性はありません：**最も重要な点は、Mullvad Browserはそれ自体ではあなたのIP Addressを隠さない**ということです（Tor Browserを除く他のブラウザのように）。あなたのIP Addressはインターネット上の "郵便Address "のようなものです：それはあなたの場所とあなたのISPを明らかにします。したがって、この重要な情報を隠すには、**VPN**（仮想プライベートネットワーク）に大きく依存します。





- モバイル版はありません：**現在、Mullvad BrowserはPC（Windows、Mac、Linux）でのみ利用可能です。**





- 特定の習慣との相性が悪い: **常時プライベートモード**は、1回の使用から次の使用までセッションを維持できないことを意味します。あるセッションから次のセッションまでウェブアカウントに接続し続けることは不可能です。





- 制限された機能:** フィンガープリントの均一性を保つため、Mullvad Browser は Firefox に存在するいくつかの機能**を無効にしており、カスタマイズを意図していません。

 - 一部のフォント、アイコン、画像、数学記号は使用できません。
 - サウンドとビデオ（HTML5メディア）、WebGLは "クリックで再生"

## 普段使い



### 通常のナビゲーション



Mullvad Browserは、日常的なブラウジングのための古典的なブラウザのように動作します。すべてのウェブサイトは通常通りアクセスでき、さらにトラッキングからの保護が強化されています。



### クッキーの管理とログイン



永続的なプライベートモードのため、ログオンするたびにアカウントに再接続する必要がある。これは、最大限の機密性を得るために支払う代償です。



### エクステンション



Mullvad Browser は技術的には Firefox カタログから追加の拡張機能をインストールできますが、**その意味を理解することが重要です**。追加された拡張機能はそれぞれ、あなたのデジタルフットプリントを変更し、他のMullvad Browserユーザーとあなたを区別します。これは、ブラウザの基本原則に反しています。



マルバッドが説明する： *「特定の防御がないほうが、あるよりいいこともある。オンライン上のプライバシーを高めようとするあまり、拡張機能をインストールすることになり、結果的に自分の存在がより見えやすくなってしまう。"*



いずれにせよ、拡張機能をインストールすることを選択した場合、サイト間であなたを追跡するために使用することができるユニークな指紋を作成することに注意してください。最大限の保護のためには、すべてのユーザーにとって同じである3つのプリインストールされた拡張機能に固執するのが最善です。



## Mullvadブラウザのベストプラクティス



1. **常にVPNを使用してください：Mullvad BrowserはあなたのIPをマスクしません。完全な匿名性を保つためにはVPNが不可欠です。**



2. **ブラウザをカスタマイズしないでください**：ブラウザの設定を変更したり、拡張機能を追加したりしないでください。



3. **新しいセッションボタンを使用してください**：異なるアクティビティ間で、リセット機能を使用してセッションを分割します。



4. **お客様のニーズに合ったセキュリティレベルをお選びください：**




- ノーマル（推奨）：日常的なブラウジング向け。ウェブサイトを機能的に保ちながら、すでに優れた保護機能を提供しています。95%のユーザーにとって最適なバランスです。
- より安全**に**：未知のサイトや危険な可能性のあるサイトにアクセスする場合、または公共のWi-Fiネットワークでの保護を強化する場合。サイトによっては誤動作する場合があります。
- 最も安全：リスクの高い状況（調査報道、機密通信、敵対的な環境）のために確保されている。ほとんどの最新サイトは壊れるが、それが最大限のセキュリティの代償である。



5. **定期的にアップデートを確認しましょう**：ブラウザを最新のセキュリティパッチに保つ。



6. **プライバシーに配慮した検索エンジン**を使いましょう：GoogleよりもDuckDuckGo、Startpage、Searxを選びましょう。



7. **HTTPSのみモードを有効にする**：設定で、"HTTPS only "モードがセキュアな接続を強制するために有効になっていることを確認してください。



8. **詳細設定は変更しないでください：他のブラウザとは異なり、Mullvad Browserはすべてのユーザーが全く同じ設定を持つように設計されています。`about:config`の設定を変更したり、uBlock Origin/NoScriptの設定を変更したりすると、ユニークなブラウザになり、匿名性が完全に失われます。**



## 推奨DNS設定



Mullvad Browser は自動的に Mullvad DNS-over-HTTPS を統合します。Mullvad VPNを使用している場合、拡張機能はMullvad DoHを無効にすることを推奨します。Mullvad VPNを使用していない場合は、ISPによるDNS監視を避けるためにMullvad DoHを有効にしておいてください。



## 結論



Mullvad Browserは、Torネットワークのパフォーマンスの制約を受けることなく、プライバシーに配慮したウェブブラウジングをお求めの方に最適なソリューションです。質の高いVPNと組み合わせることで、オンライン追跡や監視からの強固な保護を提供します。



Mullvad Browserは一定の制限（モバイル版なし、非永続的セッション）はありますが、デジタルプライバシーのコントロールを取り戻したい人にとっては貴重なツールです。その使いやすさとデフォルトの設定は、初心者であろうと経験者であろうと、プライバシーに敏感なユーザーにとって賢明な選択です。
