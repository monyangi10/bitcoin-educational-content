SimpleX 于 2021 年推出，是一款免费的即时通讯应用程序，其隐私保护方式与众不同。与 WhatsApp、Signal 和其他集中式信息服务不同，SimpleX 在用户管理方面独树一帜：没有用户标识符、假名、号码或可见的公钥。由于完全没有标识符，因此几乎不可能对用户进行关联，从而保证了高度的隐私性。



与大多数需要账户或电话号码的应用程序不同，SimpleX 可让您通过共享链接或短暂的二维码来启动对话。每个链接都会创建一个独一无二的加密通道，如果没有明确的 Exchange，联系人就无法找到或重新联系发件人。信息从头到尾都是加密的，并通过中继服务器发送，服务器会在发送后删除信息，并且不会看到发送方、接收方或他们的密钥。



![Image](assets/fr/01.webp)



网络架构完全分散，没有联盟：服务器之间互不相识，不保存全局目录，也不托管任何用户配置文件。更妙的是，每个用户都可以部署和使用自己的中继服务器，同时与公共网络上的服务器保持互通。



SimpleX 完全开源（客户端、协议和服务器），可在 Android、iOS、Linux、Windows 和 macOS 上使用。它的本地存储是加密和便携式的，因此您可以将配置文件从一台设备传输到另一台设备，而无需依赖集中式服务器。



SimpleX 整合了消息应用程序的所有经典功能。不过，它的人体工学设计仍不如 WhatsApp 或 Signal 那样流畅。它的使用限制也更多，尤其是在添加联系人时。因此，在我看来，它是 WhatsApp 或 Signal 的替代品，适合将保密性放在首位，并为此准备在日常使用舒适度方面做出一些让步的用户。

## 安装 SimpleX 聊天应用程序



SimpleX Chat 适用于所有平台。您可以直接从手机的应用程序商店下载该应用程序：




- [Google Play]（https://play.google.com/store/apps/details?id=chat.simplex.app）；
- [App Store](https://apps.apple.com/us/app/simplex-chat-secure-messenger/id1605771084)；
- [F-Droid](https://simplex.chat/fdroid/).



在 Android 上，也可以 [通过 APK 安装](https://github.com/simplex-chat/simplex-chat/releases)。



在本教程中，我们将重点介绍移动版本，但请注意，[桌面版本也可用](https://simplex.chat/downloads/) （MacOS、Linux 和 Windows）。可以将桌面版链接到现有的移动用户配置文件，但要实现同步，两台设备必须连接到同一个本地网络。



![Image](assets/fr/02.webp)



## 在 SimpleX Chat 上创建账户



首次启动应用程序时，点击 "*创建个人资料*"按钮。



![Image](assets/fr/03.webp)



选择一个用户名，可以是真名，也可以是笔名，然后点击 "*创建*"。



![Image](assets/fr/04.webp)



接下来，设置应用程序检查新信息的频率。如果手机电池寿命不是问题，选择 "*即时*"即可实时接收信息。如果你希望节省电池并防止应用程序在后台运行，请选择 "*当应用程序运行时*"：这样你就只能在应用程序打开时接收信息。一个可能的折衷办法是选择每 10 分钟定期检查一次。



做出选择后，点击 "*使用聊天*"。



![Image](assets/fr/05.webp)



现在您已连接到 SimpleX Chat，可以开始聊天了。



![Image](assets/fr/06.webp)



## 设置 SimpleX 聊天



首先，点击右下角的个人照片，然后点击 "*设置*"进入设置。



![Image](assets/fr/07.webp)



默认设置通常适合大多数用户。不过，我建议您转到 "*数据库 passphrase 和导出*"菜单。在这里您可以导出 SimpleX 账户数据库，以便备份。



您还可以修改用于加密该数据库的 passphrase。默认情况下，passphrase 是随机生成并存储在本地设备上的。如果您愿意，可以定义自己的 passphrase，并删除本地 passphrase 备份：这样，每次打开应用程序以及迁移到其他设备时，都必须手动输入 passphrase。



**请注意**：如果选择此选项，丢失 passphrase 将导致永久丢失所有 SimpleX 配置文件和信息。



![Image](assets/fr/08.webp)



我还建议你进入 "*隐私与安全*"菜单，在这里你可以激活 "*SimpleX 锁*"选项。这可以通过锁定密码保护对应用程序的访问。



![Image](assets/fr/09.webp)



最后，"*通知*"和 "*外观*"菜单可让您根据自己的喜好自定义 SimpleX Chat。



![Image](assets/fr/10.webp)



## 使用 SimpleX 聊天工具发送信息



要在 SimpleX 上与他人建立联系，您有两个选择：




- 使用一次性链接；
- 使用可重复使用的静态 Address。



静态 Address 允许任何知道它的人在 SimpleX 上与您联系。这是一个持久的 Address，不同的人可以多次使用，没有时间限制。正是这种持久性使它更容易受到垃圾邮件的攻击。不过，与其他信息应用程序不同的是，删除 SimpleX Address 就足以阻止所有垃圾邮件，而不会影响现有对话。事实上，Address 只用于建立初始连接，Exchange 开始后就不再需要了。



而一次性链接只能被任何用户使用一次。一旦有联系人使用，链接就会失效。您需要为每一个新的连接创建一个新的 generate 链接。



### 静态 Address



如果您希望使用 Address，请单击 Interface 左下方的个人资料图片，然后选择 "*创建 SimpleX Address*"。然后再次点击 "*创建 SimpleX Address*"。



![Image](assets/fr/11.webp)



您的可重复使用 Address 已经创建。您可以与希望与您取得联系的人分享，向他们展示二维码或向他们发送链接即可。



![Image](assets/fr/12.webp)



点击 "**Address 设置**"按钮。在这里，您可以配置与 Address 相关的权限。**与联系人共享**选项可让您的 Address 在您的 SimpleX 个人档案中可见。这样，您的联系人就可以查阅并转发给希望与您联系的其他人。



自动接受*"选项会自动接受通过您的 Address 发送的连接。这意味着，除非您激活 "*隐身接受*"选项，否则任何使用您 Address 的人都可以看到您的个人资料并给您发送信息。当建立新连接时，该选项会隐藏您的姓名和个人照片，取而代之的是一个随机的化名，每个对话者都是不同的。



![Image](assets/fr/13.webp)



### 带可重复使用的链接



与某人联系的第二种方法是创建一次性链接。为此，请单击 Interface 右下角的铅笔图标，然后选择 "*创建一次性链接*"。



如果联系人向您发送了链接，请点击 "*扫描/粘贴链接*"进行扫描或粘贴。



![Image](assets/fr/14.webp)



然后，SimpleX 会生成一个一次性链接。您可以通过任何方式将其转发给联系人：实体 Exchange、其他信息等。



![Image](assets/fr/15.webp)



您还可以选择将哪个个人资料与此邀请链接关联。为此，请单击二维码下方的个人资料。然后您就可以 ：




- 选择一个现有的配置文件（我们将在下一节了解如何创建多个配置文件）；
- 或选择 "*隐身*"模式，该模式会隐藏你的姓名和个人照片，并为你的联系人随机生成一个假名。



在这里，我选择了 "*隐身*"模式。



![Image](assets/fr/16.webp)



我的联系人使用了该链接。他没有激活 "*隐身*"模式，因此我看到了他的个人资料名称 "*Bob*"。另一方面，鲍勃看到的不是我的真名 "*Loïc Morel*"，而是一个随机的假名，这里是 "*RealSynergy*"。



![Image](assets/fr/17.webp)



我可以立即开始聊天，但首先我要确认我是在和鲍勃聊天，而不是在和某个可能截获了链接或实施了 MITM 攻击的恶意人员聊天。



为此，我们将在应用程序**外**检查我们的安全链接。这一点很重要，因为一旦发生 MITM 攻击，内部验证就会泄露。因此，请使用另一个安全信息传输系统，或者亲自检查密码，效果会更好。



在聊天中，点击鲍勃的照片，然后点击 "*验证安全码*"。鲍勃这边也要这样做。



![Image](assets/fr/18.webp)



如果您是远程工作，请比较另一个安全信息系统上的代码（它们必须完全相同）。或者，如果你们能面对面，点击 "*扫描代码*"扫描联系人的二维码，效果会更好。



![Image](assets/fr/19.webp)



如果验证成功，您的联系人姓名旁边会出现一个带复选标记的盾牌图标。这是您与鲍勃交换信息的保证。如果验证不成功，则会出现 "*安全代码不正确！*"的提示。



![Image](assets/fr/20.webp)



您现在可以与鲍勃自由 Exchange 信息、通话和文件，具体取决于您为此会话设置的权限。



## 自定义您的 SimpleX 聊天配置文件



SimpleX 最强大的功能之一是可以管理多个完全独立的用户配置文件，所有这些配置文件都可以从同一个本地账户访问。这样，您就可以整齐地分隔不同的身份，而不会使信息管理复杂化。



例如，您可以创建 .NET Framework 2.0：




- 在个人简介中使用真实姓名和真实照片，以便进行专业交流；
- 在个人档案中填写您的真实姓名，并为您的家庭交流上传一张有趣的照片；
- 使用假照片和假名进行个人对话的个人资料；
- 另一个用于与陌生人聊天的匿名配置文件。



这就是我们要做的。我首先要配置我的主配置文件，也就是与我真实身份相关联的配置文件。为此，我点击右下角的个人照片，然后点击我的用户名。



![Image](assets/fr/21.webp)



然后，我点击我的个人照片进行更改并添加新照片。



![Image](assets/fr/22.webp)



要添加其他配置文件，请单击 "*您的聊天配置文件*"菜单。



![Image](assets/fr/23.webp)



在这里，您将看到您的所有个人资料。点击 "*添加个人资料*"创建新的个人资料。



![Image](assets/fr/24.webp)



然后选择新个人资料的信息：姓名、照片等。在这里，我使用化名和不同的形象，以便在某些交流中隐藏我的真实身份。



![Image](assets/fr/25.webp)



在个人资料上按住手指，即可将其隐藏。这将使其在应用程序中与所有相关会话一起隐形。您还可以选择 "*静音*"，以停止接收通知。



![Image](assets/fr/26.webp)



创建个人档案后，您可以独立管理它们。在主页上，只需选择要显示的活动个人档案即可。



![Image](assets/fr/27.webp)



创建邀请链接或静态 Address 时，您现在可以选择与之关联的个人资料。例如，如果我选择个人资料 "*Satoshi Nakamoto*"来 generate 链接并发送给爱丽丝，她将只能看到我的假名身份 "*Satoshi Nakamoto*"，而不会知道我的真实身份 "*Loïc Morel*"。相反，如果我向她提供我真实个人资料中的链接，她就无法链接到我的假名个人资料。



![Image](assets/fr/28.webp)
