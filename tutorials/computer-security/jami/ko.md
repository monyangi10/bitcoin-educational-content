![cover](assets/cover.webp)



Jami는 오랜 역사와 다양한 기능을 갖춘 무료 오픈소스 P2P 메시징 프로그램입니다. 그 역사는 2004년 12월 몬트리올에 본사를 둔 *Savoir-faire Linux*가 오픈 소스 기술을 기반으로 한 기업용 디지털 전화 프로젝트인 SFLPhone을 출시했을 때로 거슬러 올라갑니다. 통신 표준(SIP 및 IAX)을 준수하는 이 소프트웨어는 많은 수의 회선과 통화를 처리할 수 있는 능력이 돋보였습니다.



2015년, SFLPhone은 Ring으로 이름을 바꾸고 더 이상 중앙 서버가 필요 없는 분산 아키텍처를 통합했습니다. 이듬해에는 Ring이 GNU 프로젝트에 공식적으로 합류하여 자유 소프트웨어 생태계에서의 입지를 강화했습니다. 마침내 2018년 12월, "*Ring*"이라는 영어 용어를 사용하는 상용 제품과의 혼동을 피하기 위해 현재의 이름인 Jami를 채택했습니다. 그 이후로 무료, 탈중앙화, 개인정보 보호 친화적인 커뮤니케이션 플랫폼으로 계속 발전하고 있습니다.



오늘날 Jami는 다양한 시스템에서 사용할 수 있습니다. 성능, 유동성, 사용 편의성으로 잘 알려져 있습니다. 인스턴트 메시징, 음성 통화 또는 영상 통화를 통해 커뮤니케이션할 수 있으며, 종단 간 암호화를 통해 대화의 기밀성을 보장합니다. 간단한 설치와 다양한 기능으로 일상에서 쉽고 편리하게 사용할 수 있는 완벽한 커뮤니케이션 애플리케이션입니다.

## 왜 Jami를 사용하나요?





- 오픈 소스이며 완전히 무료**이므로 무료로 사용할 수 있습니다.
- 유용한 기능**: 스토어에서 손쉽게 플러그인을 추가할 수 있는 기능 덕분에 다양한 옵션을 활용할 수 있습니다. 필요에 더 적합한 기능을 위해 자신만의 확장 기능을 만들 수도 있습니다.
- 사용하기 쉽고 직관적인 Interface**: Jami는 많은 기능을 제공함에도 불구하고 여전히 매우 쉽게 사용할 수 있습니다.
- 강력한 보안**: Jami는 사용자의 개인 정보를 존중하면서 통신의 보안을 보장하는 고급 암호화 알고리즘을 통합하고 있습니다.
- 높은 접근성과 빠른 속도**: 대역폭이 제한되어 있어도 쉽게 커뮤니케이션할 수 있어 사용자 편의성이 향상됩니다.



## Jami는 어떻게 설치하나요?



Jami를 설치하기 전에 이 애플리케이션은 다양한 운영 체제에서 사용할 수 있다는 점을 알아두세요. 따라서 사용 중인 시스템에 따라 설치 방법이 달라집니다.



![0_01](assets/fr/01.webp)



### Android 또는 iOS 사용자의 경우



이 애플리케이션은 App Store 또는 Play 스토어에서 바로 다운로드할 수 있습니다. 검색창에서 검색한 다음 설치를 실행하기만 하면 됩니다.



### Windows 또는 macOS 사용자의 경우



디바이스에 Jami를 설치하려면 먼저 공식 Jami 웹사이트를 방문해야 합니다. 기본적으로 사이트에는 디바이스의 운영 체제에 해당하는 소프트웨어가 표시되며, 다운로드 버튼을 직접 클릭하여 실행할 수 있습니다. 그러나 [다운로드 페이지](https://jami.net/download-jami-windows/)에서 Windows 실행 파일을 직접 다운로드할 수도 있습니다.



![0_02](assets/fr/02.webp)



MacOS 사용자의 경우 [macOS 다운로드 페이지](https://jami.net/download-jami-macos/)에서도 파일을 다운로드할 수 있습니다.



![0_03](assets/fr/03.webp)



실행 파일을 다운로드했으면 더블클릭하여 설치 프로세스를 시작합니다. 사용 약관에 동의한 다음 설치를 시작하고 완료될 때까지 기다립니다.



![0_04](assets/fr/04.webp)



![0_05](assets/fr/05.webp)



### Linux 사용자의 경우



Linux에 Jami를 설치하려면 명령줄을 사용하는 것이 가장 좋습니다. Jami는 다양한 Linux 배포판에서 사용할 수 있다는 점에 유의하세요. Jami 설치를 시작하기 전에 시스템에 적합한 배포판을 선택했는지 확인하세요.



배포판을 선택했으면 시스템을 설치할 수 있습니다. Linux OS에서 Jami를 실행하는 데 필요한 종속성을 설치해야 합니다. 명령어는 [이 페이지](https://jami.net/download-jami-linux/)에서 직접 확인할 수 있습니다.



![0_06](assets/fr/06.webp)



우분투**에 Jami를 설치하려면 다음과 같이 하면 됩니다:



```shell
sudo apt install gnupg dirmngr ca-certificates curl --no-install-recommends
```



이 명령은 GPG 키(gnupg 및 dirmngr), SSL 인증서(ca-certificates) 및 curl 다운로드 도구를 관리하는 데 필요한 도구를 설치합니다.



https://planb.network/tutorials/computer-security/operating-system/ubuntu-78a3be56-5d51-4ec3-8629-0dd27c352ab5

```shell
curl -s https://dl.jami.net/public-key.gpg | sudo tee /usr/share/keyrings/jami-archive-keyring.gpg > /dev/null
```



여기서 curl은 Jami의 GPG 공개키를 다운로드하는 데 사용됩니다. Dev/null로 리디렉션하면 원시 키가 화면에 표시되는 것을 방지할 수 있습니다.



```shell
sudo sh -c "echo 'deb [signed-by=/usr/share/keyrings/jami-archive-keyring.gpg] https://dl.jami.net/stable/ubuntu_25.04/ jami main' > /etc/apt/sources.list.d/jami.list"
```



이 명령은 공식 Jami 리포지토리를 APT 소스 목록에 추가합니다.



```shell
sudo apt-get update && sudo apt-get install jami
```



마지막으로 `apt-get update`로 사용 가능한 패키지 목록을 업데이트한 다음 `apt-get install jami`로 공식 리포지토리에서 직접 Jami를 설치합니다.



## Jami 기본 구성



Jami가 시스템에 설치되면 시스템 메뉴에서 바로 실행할 수 있습니다.



애플리케이션을 시작하면 계정을 새로 만들거나 이미 만든 계정을 계속 사용할 수 있습니다.



![0_07](assets/fr/07.webp)



### 계정 만들기



Jami 계정 생성은 비교적 간단합니다. 이메일 Address이나 전화번호가 필요하지 않습니다: Jami는 최소한의 정보만 수집합니다. 원하는 경우 *Jami ID*(암호화 지문)를 가리키는 사용자 이름(별칭)을 등록할 수 있습니다. 별칭 ↔ Jami ID* 연결은 기본 네임 서버(교체 가능/자체 호스팅)에 게시되므로 별칭이 반드시 필요한 것은 아닙니다.



![0_08](assets/fr/08.webp)



로컬에서 데이터를 보호하기 위해 디바이스에서 프로필과 백업을 암호화하는 비밀번호를 설정할 수 있습니다. 이 비밀번호는 선택 사항이며 기본적으로 활성화되어 있는 통신의 종단 간 암호화에는 영향을 미치지 않습니다. 이 로컬 보호 기능을 활성화하려면 길고 임의의 고유한 비밀번호를 선택한 다음 유효성을 검사하세요.



![0_09](assets/fr/09.webp)



계정을 암호화한 후 전체 이름을 설정합니다.



![0_10](assets/fr/10.webp)



![0_11](assets/fr/11.webp)



## 기존 계정에 로그인



Jami는 **식별자**를 사용하지 않으며 계정에 연결하기 위한 데이터베이스를 보유하지 않습니다. 모든 데이터는 기기에 직접 저장됩니다. 이전 계정에 연결하려면 이전 계정의 **백업**을 수행해야 합니다.



설정**, **계정**, **계정 관리**로 이동합니다. 하단으로 스크롤하여 **계정 백업**을 수행합니다. 백업 파일을 저장할 위치를 선택하고 계정을 만들 때 설정한 **비밀번호**를 입력한 다음 인증합니다.



![0_12](assets/fr/12.webp)



이 백업 파일을 통해 계정에 다시 연결할 수 있습니다.



![0_13](assets/fr/13.webp)



### 아카이브를 통한 연결



이렇게 하려면 **백업 아카이브에서 가져오기** 버튼을 클릭하고 Jami 계정의 백업 파일을 선택합니다.



![0_14](assets/fr/14.webp)



계정을 만들 때 비밀번호를 설정한 경우 비밀번호를 입력하고 확인합니다. 데이터와 메시지가 자동으로 표시됩니다.



![0_15](assets/fr/15.webp)



계정을 정기적으로 백업하여 데이터를 최신 상태로 유지하고 최신 메시지를 잃어버리지 않도록 하는 것이 중요합니다.



### 장치를 통한 연결



이미 다른 디바이스에서 Jami 계정으로 로그인한 경우 **계정**으로 이동한 다음 **연결된 디바이스** 섹션으로 이동하여 **새 디바이스 연결**을 클릭합니다. 연결 코드를 입력할 수 있는 필드와 함께 QR코드 리더가 나타납니다.



![0_16](assets/fr/16.webp)



새 기기에서 Jami를 실행하고 **이미 계정이 있습니다**를 선택한 다음 **다른 기기에서 가져오기**를 선택합니다. 그러면 **QR 코드**와 로그인 코드가 표시됩니다. 연결하려면 QR코드 스캐너를 사용해 다른 장치에 표시된 코드를 스캔하면 됩니다. 하지만 연결 코드를 복사하여 제공된 필드에 직접 붙여넣는 것이 더 간편할 때가 많습니다.



![0_17](assets/fr/17.webp)



계정을 만들 때 비밀번호를 설정한 경우 계속하려면 새 디바이스에서 비밀번호를 입력해야 합니다.



![417](assets/fr/18.webp)



계속하려면 이미 연결된 디바이스에서 연결을 확인합니다.



![0_19](assets/fr/19.webp)



비밀번호를 입력하면 디바이스가 자동으로 계정에 연결되고 동기화됩니다. 그러면 모든 디바이스에서 메시지를 주고받을 수 있습니다.



## Jami에 확장 프로그램 추가



Jami의 흥미로운 기능 중 하나는 확장 기능(플러그인)을 통해 새로운 옵션을 통합할 수 있다는 점입니다. 플러그인은 네이티브 모듈(C/C++)이며, SDK는 플러그인을 만드는 데 도움이 되는 도구와 스크립트(특히 Python)를 제공합니다. 일부 플러그인은 [여기](https://jami.net/extensions/)에서 직접 사용할 수 있습니다.



확장 프로그램을 설치하려면 데스크톱에서 통합 확장 프로그램 스토어를 열고 해당 플러그인을 다운로드한 다음 **설정 → 확장 프로그램 → 설치**로 이동하여 활성화하세요. Android에서는 스토어가 통합되어 있지 않으므로 '.jpl' 파일을 다운로드한 다음 **설정 → 확장 프로그램 → 설치**에서 수동으로 가져오세요. 가져오기는 자동으로 이루어지며, 필요한 경우 활성화하고 설정을 조정할 수 있습니다.



![0_20](assets/fr/20.webp)



나만의 플러그인을 개발하려면 **[자미 플러그인 SDK를 살펴보고 나만의 플러그인 만들기](https://jami.net/plugins-sdk/) 블로그 포스팅을 확인하세요.



## 고급 기능



Jami는 또한 애플리케이션을 더 자세히 구성하고 사용하고자 하는 사용자를 위해 **고급 기능**을 제공합니다. 이러한 옵션에는 다음이 포함됩니다:





- 랑데부 포인트** 만들기: 이 기능을 사용하면 커뮤니케이션을 위한 **랑데부 지점**을 만들 수 있어 여러 사용자 간의 보안 세션 또는 교환을 구성하는 데 유용합니다.
- Jami 서버에 연결**: Jami를 **Jami 서버**에 연결하면 특히 전문적인 환경에서 커뮤니케이션의 성능이나 가용성을 향상시킬 수 있습니다.
- SIP 계정 설정**: SIP 계정**(세션 개시 프로토콜)을 설정하여 Jami를 기존 전화 시스템과 통합하거나 전화 통화를 할 수 있습니다.



![0_21](assets/fr/21.webp)



이러한 옵션은 Jami 환경을 사용자 지정하고 특정 요구에 맞는 추가 기능을 활용하고자 하는 고급 사용자를 위해 설계되었습니다.



요컨대, Jami는 완벽하고 안전하며 유연한 커뮤니케이션 솔루션으로, **확장 기능**을 통해 개인용과 보다 구체적인 요구사항 모두에 적합합니다. 간단한 설치, 직관적인 조작, **암호화**, **다중 장치 동기화**, **자체 플러그인 생성** 기능과 같은 고급 기능으로 **개인정보**를 보호하면서 커뮤니케이션을 관리할 수 있는 강력한 도구입니다.



엔드투엔드 암호화(E2E), 공개 키 및 기타 여러 알고리즘을 결합하여 다양한 클라이언트를 통해 기밀을 보호하는 최적의 통신을 제공하는 탈중앙화 프로토콜인 Tox를 알아보세요.
