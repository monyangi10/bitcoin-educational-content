Jami es un programa de mensajería peer-to-peer gratuito y de código abierto con una larga historia y una amplia gama de funciones. Su historia se remonta a diciembre de 2004, cuando *Savoir-faire Linux*, con sede en Montreal, lanzó SFLPhone, un proyecto de telefonía digital para empresas basado íntegramente en tecnologías de código abierto. El software, que cumplía los estándares de telecomunicaciones (SIP e IAX), destacaba por su capacidad para gestionar un gran número de líneas y llamadas.



En 2015, SFLPhone pasó a llamarse Ring e integró una arquitectura distribuida que ya no requiere un servidor central. Al año siguiente, Ring se unió oficialmente al proyecto GNU, reforzando su posición en el ecosistema del software libre. Finalmente, en diciembre de 2018, para evitar confusiones con productos comerciales que utilizan el término inglés "*Ring*", el software adoptó su nombre actual: Jami. Desde entonces, ha seguido evolucionando como plataforma de comunicación libre, descentralizada y respetuosa con la privacidad.



Hoy en día, Jami está disponible en muchos sistemas. Es famoso por su rendimiento, fluidez y facilidad de uso. Te permite comunicarte mediante mensajería instantánea, llamadas de audio o videollamadas, garantizando al mismo tiempo la confidencialidad de tus conversaciones gracias al cifrado de extremo a extremo. Su sencilla instalación y sus numerosas funciones la convierten en una aplicación de comunicación completa, fácil y cómoda de usar a diario.

## ¿Por qué utilizar Jami?





- Es de código abierto y totalmente gratuito**, por lo que puedes utilizarlo sin coste alguno.
- Completo con funciones útiles**: este software le permite aprovechar numerosas opciones, gracias a la posibilidad de añadir plugins fácilmente desde la tienda. Incluso es posible crear tus propias extensiones para obtener funciones más adaptadas a tus necesidades.
- Fácil de usar e intuitivo**: a pesar de sus numerosas funciones, Jami es muy fácil de manejar.
- Fuerte seguridad**: Jami incorpora un avanzado algoritmo de encriptación que garantiza la seguridad de tus comunicaciones respetando tu privacidad.
- Muy accesible y rápido**: ofrece una comunicación sencilla incluso cuando el ancho de banda es limitado, para una mayor facilidad de uso.



## ¿Cómo se instala Jami?



Antes de pasar a la instalación de Jami, es importante tener en cuenta que la aplicación está disponible en diferentes sistemas operativos. Por lo tanto, su instalación dependerá del sistema que estés utilizando.



![0_01](assets/fr/01.webp)



### Para usuarios de Android o iOS



La aplicación está disponible directamente en App Store o Play Store. Basta con buscarla en la barra de búsqueda e iniciar la instalación.



### Para usuarios de Windows o macOS



Para instalar Jami en tu dispositivo, primero tienes que visitar el sitio web oficial de Jami. Por defecto, el sitio muestra el software correspondiente al sistema operativo de tu dispositivo, y puedes hacer clic directamente en el botón de descarga para iniciarlo. Sin embargo, también puedes descargar el ejecutable para Windows directamente desde la [página de descargas](https://jami.net/download-jami-windows/).



![0_02](assets/fr/02.webp)



Para los usuarios de macOS, el archivo también está disponible en [página de descargas de macOS](https://jami.net/download-jami-macos/).



![0_03](assets/fr/03.webp)



Una vez descargado el ejecutable, inicia el proceso de instalación haciendo doble clic sobre él. Acepta las condiciones de uso, inicia la instalación y espera a que finalice.



![0_04](assets/fr/04.webp)



![0_05](assets/fr/05.webp)



### Para usuarios de Linux



Para instalar Jami en Linux, lo mejor es utilizar la línea de comandos. Es importante tener en cuenta que Jami está disponible para diferentes distribuciones de Linux. Antes de empezar a instalar Jami, asegúrate de elegir la distribución adecuada para tu sistema.



Una vez que hayas seleccionado tu distribución, puedes instalar el sistema. Tendrás que instalar las dependencias necesarias para ejecutar Jami en tu sistema operativo Linux. Los comandos están disponibles directamente en [esta página](https://jami.net/download-jami-linux/).



![0_06](assets/fr/06.webp)



Para instalar Jami en **Ubuntu**, puedes hacerlo de la siguiente manera:



```shell
sudo apt install gnupg dirmngr ca-certificates curl --no-install-recommends
```



Este comando instala las herramientas necesarias para gestionar claves GPG (gnupg y dirmngr), certificados SSL (ca-certificates) y la herramienta de descarga curl.



https://planb.network/tutorials/computer-security/operating-system/ubuntu-78a3be56-5d51-4ec3-8629-0dd27c352ab5

```shell
curl -s https://dl.jami.net/public-key.gpg | sudo tee /usr/share/keyrings/jami-archive-keyring.gpg > /dev/null
```



Aquí se utiliza curl para descargar la clave pública GPG de Jami. La redirección a /dev/null evita mostrar la clave sin procesar en la pantalla.



```shell
sudo sh -c "echo 'deb [signed-by=/usr/share/keyrings/jami-archive-keyring.gpg] https://dl.jami.net/stable/ubuntu_25.04/ jami main' > /etc/apt/sources.list.d/jami.list"
```



Este comando añade el repositorio oficial de Jami a la lista de fuentes de APT.



```shell
sudo apt-get update && sudo apt-get install jami
```



Por último, actualiza la lista de paquetes disponibles con `apt-get update`, y luego instala Jami directamente desde el repositorio oficial con `apt-get install jami`.



## Configuración básica de Jami



Una vez instalado Jami en tu sistema, puedes iniciarlo directamente desde el menú del sistema.



Una vez iniciada la aplicación, tendrá la opción de crear una cuenta o continuar con una ya creada.



![0_07](assets/fr/07.webp)



### Crear una cuenta



Crear una cuenta Jami es relativamente sencillo. No necesitas ni correo electrónico ni número de teléfono: Jami sólo recoge el mínimo de información. Si lo desea, puede registrar un nombre de usuario (alias) que apunte a su *Jami ID* (huella digital criptográfica). La asociación *alias ↔ Jami ID* se publica en un servidor de nombres por defecto (reemplazable / autoalojable), por lo que el alias no es necesariamente obligatorio.



![0_08](assets/fr/08.webp)



Para proteger tus datos localmente, puedes establecer una contraseña para cifrar tu perfil y las copias de seguridad en el dispositivo. Esta contraseña es opcional y no tiene ningún efecto sobre el cifrado de extremo a extremo de las comunicaciones, que está activo por defecto. Si activas esta protección local, elige una contraseña larga, aleatoria y única, y luego valídala.



![0_09](assets/fr/09.webp)



Después de cifrar tu cuenta, establece tu nombre completo.



![0_10](assets/fr/10.webp)



![0_11](assets/fr/11.webp)



## Acceder a una cuenta existente



Jami no utiliza **identificadores** y no tiene ninguna base de datos para conectarse a tu cuenta. Todos tus datos se almacenan directamente en tu dispositivo. Para conectarte a tu antigua cuenta, necesitas realizar una **copia de seguridad** de tu antigua cuenta.



Vaya a **Configuración**, luego a **Cuenta**, luego a **Gestión de cuenta**. Desplácese hasta la parte inferior y realice una **copia de seguridad de su cuenta**. Elija la ubicación donde se guardará el archivo de copia de seguridad, introduzca la **contraseña** definida al crear su cuenta y, a continuación, valide.



![0_12](assets/fr/12.webp)



Este archivo de copia de seguridad le permitirá volver a conectarse a su cuenta.



![0_13](assets/fr/13.webp)



### Conexión mediante archivo



Para ello, haga clic en el botón **Importar desde un archivo de copia de seguridad** y seleccione el archivo de copia de seguridad de su cuenta de Jami.



![0_14](assets/fr/14.webp)



Si estableció una contraseña al crear su cuenta, introdúzcala y confírmela. Tus datos y mensajes aparecerán automáticamente.



![0_15](assets/fr/15.webp)



Es importante hacer copias de seguridad periódicas de tu cuenta para mantener tus datos actualizados y asegurarte de que no pierdes tus últimos mensajes.



### Conexión a través de un dispositivo



Si ya has iniciado sesión en otro dispositivo con tu cuenta de Jami, ve a **Cuenta** y luego a la sección **Dispositivo conectado** y haz clic en **Conectar un nuevo dispositivo**. Aparecerá el lector de códigos QR y un campo para introducir un código de conexión.



![0_16](assets/fr/16.webp)



Inicia Jami en el nuevo dispositivo y selecciona **Ya tengo una cuenta** y luego **Importar desde otro dispositivo**. Obtendrás entonces un **código QR** y un código de conexión. Para conectarte, puedes utilizar el escáner de códigos QR para escanear el código que aparece en el otro dispositivo. Sin embargo, a menudo es más sencillo copiar el código de conexión y pegarlo directamente en el campo previsto a tal efecto.



![0_17](assets/fr/17.webp)



Si estableciste una contraseña al crear tu cuenta, tendrás que introducirla en el nuevo dispositivo para continuar.



![417](assets/fr/18.webp)



Confirme la conexión en el dispositivo ya conectado para continuar.



![0_19](assets/fr/19.webp)



Una vez introducida la contraseña, el dispositivo se conectará automáticamente a la cuenta y se sincronizará. A continuación, podrás enviar y recibir mensajes desde cualquiera de tus dispositivos.



## Añadir una extensión a Jami



Una de las características interesantes de Jami es su capacidad para integrar nuevas opciones a través de extensiones (plugins). Los plugins son módulos nativos (C/C++); el SDK proporciona herramientas y scripts (especialmente en Python) para ayudar a crearlos. Algunos plugins están disponibles directamente [aquí](https://jami.net/extensions/).



Para instalar una extensión, en Escritorio abra la tienda de extensiones integrada, descargue el plugin correspondiente y vaya a **Configuración → Extensiones → Instalar** y actívelo. En Android, la tienda no está integrada: descargue el archivo `.jpl` e impórtelo manualmente desde **Configuración → Extensiones → Instalar**; la importación es automática y, a continuación, puede activarla y ajustar su configuración si es necesario.



![0_20](assets/fr/20.webp)



Para desarrollar tus propios plugins, consulta la entrada del blog **[Descubre el SDK de plugins de Jami y crea tus propios plugins](https://jami.net/plugins-sdk/)**.



## Funciones avanzadas



Jami también ofrece **funciones avanzadas** para los usuarios que deseen ir más allá en la configuración y el uso de la aplicación. Estas opciones incluyen:





- Crear un punto de encuentro**: Esta función te permite crear un **punto de encuentro** para tus comunicaciones, útil para organizar sesiones o intercambios seguros entre varios usuarios.
- Conectarse a un servidor Jami**: Puede conectar Jami a un **servidor Jami**, lo que puede mejorar el rendimiento o la disponibilidad de las comunicaciones, especialmente en entornos profesionales.
- Crear una cuenta SIP**: Puede crear una cuenta **SIP** (Session Initiation Protocol), que le permitirá integrar Jami con los sistemas telefónicos existentes o realizar llamadas telefónicas.



![0_21](assets/fr/21.webp)



Estas opciones están diseñadas para usuarios más avanzados que desean personalizar su experiencia con Jami y aprovechar funciones adicionales para necesidades específicas.



En definitiva, Jami es una solución de comunicaciones completa, segura y flexible, apta tanto para uso personal como para necesidades más específicas gracias a sus **extensiones**. Su sencilla instalación, manejo intuitivo y funciones avanzadas como **cifrado**, **sincronización multidispositivo** y la posibilidad de **crear tus propios plugins** lo convierten en una potente herramienta para gestionar tus comunicaciones preservando tu **privacidad**.
