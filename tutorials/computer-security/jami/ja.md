Jamiは、長い歴史と幅広い機能を持つ、フリーでオープンソースのピアツーピアメッセージングプログラムです。その歴史は2004年12月に遡る。モントリオールを拠点とする*Savoir-faire Linux*が、完全にオープンソース技術に基づいた企業向けデジタルテレフォニープロジェクトであるSFLPhoneを立ち上げたのだ。電気通信標準（SIPとIAX）に準拠したこのソフトウェアは、大量の回線と通話を処理する能力で際立っていた。



2015年、SFLPhoneはRingと改名し、中央サーバーを必要としない分散アーキテクチャを統合した。翌年、Ringは正式にGNUプロジェクトに参加し、フリーソフトウェアのエコシステムにおける足場を固めた。最終的に2018年12月、「*Ring*」という英語表記を使用する商用製品との混同を避けるため、ソフトウェアは現在の名称である「Jami」を採用した。それ以来、自由で分散型でプライバシーに配慮したコミュニケーション・プラットフォームとして進化を続けている。



今日、Jamiは多くのシステムで利用できる。Jamiは、そのパフォーマンス、流動性、使いやすさで有名です。エンドツーエンドの暗号化により会話の機密性を確保しながら、インスタントメッセージ、音声通話、ビデオ通話でコミュニケーションすることができます。シンプルなインストールと豊富な機能により、日常的に使いやすく便利な完全なコミュニケーション・アプリケーションとなっています。

## なぜジャミを使うのか？





- オープンソースで完全無料**なので、無料で使うことができる。
- 便利な機能が満載**: このソフトウェアは、ストアから簡単にプラグインを追加できるため、数多くのオプションを利用できます。あなたのニーズに合った機能のために独自の拡張機能を作成することも可能です。
- 使いやすく、Interfaceの直感的な操作**：Jamiは多くの機能を備えているにもかかわらず、非常に簡単に使いこなすことができる。
- 強力なセキュリティ**：Jamiは、あなたのプライバシーを尊重しながら、通信のセキュリティを保証する高度な暗号化アルゴリズムを組み込んでいます。
- 高いアクセス性と高速性**：帯域幅が制限されている場合でも簡単に通信でき、使いやすさが向上しています。



## ジャミのインストール方法は？



Jamiのインストールに進む前に、このアプリケーションが異なるオペレーティングシステムで利用可能であることに注意することが重要です。そのため、インストールはお使いのシステムによって異なります。



![0_01](assets/fr/01.webp)



### AndroidまたはiOSユーザーの方へ



アプリケーションはApp StoreまたはPlay Storeから直接入手できます。検索バーで検索し、インストールを起動してください。



### WindowsまたはmacOSユーザーの方へ



お使いのデバイスにJamiをインストールするには、まずJamiの公式ウェブサイトにアクセスする必要があります。デフォルトでは、あなたのデバイスのオペレーティングシステムに対応するソフトウェアが表示され、ダウンロードボタンを直接クリックして起動することができます。しかし、[ダウンロードページ](https://jami.net/download-jami-windows/)からWindowsの実行ファイルを直接ダウンロードすることもできます。



![0_02](assets/fr/02.webp)



macOSユーザーの方は、[macOSダウンロードページ](https://jami.net/download-jami-macos/)からもダウンロードできます。



![0_03](assets/fr/03.webp)



実行ファイルをダウンロードしたら、ダブルクリックしてインストールを開始します。利用規約に同意してからインストールを開始し、終了するまで待ちます。



![0_04](assets/fr/04.webp)



![0_05](assets/fr/05.webp)



### Linuxユーザーの方へ



LinuxにJamiをインストールするには、コマンドラインを使うのが一番だ。Jamiは異なるLinuxディストリビューションで利用可能であることに注意することが重要だ。Jamiのインストールを始める前に、あなたのシステムに合ったディストリビューションを選んでください。



ディストリビューションを選択したら、システムをインストールすることができます。あなたのLinux OS上でJamiを実行するために必要な依存関係をインストールする必要があります。コマンドは[このページ](https://jami.net/download-jami-linux/)に直接あります。



![0_06](assets/fr/06.webp)



Jamiを**Ubuntu**にインストールするには、以下のようにします：



```shell
sudo apt install gnupg dirmngr ca-certificates curl --no-install-recommends
```



このコマンドは、GPG鍵（gnupgとdirmngr）、SSL証明書（ca-certificates）、curlダウンロードツールの管理に必要なツールをインストールする。



https://planb.network/tutorials/computer-security/operating-system/ubuntu-78a3be56-5d51-4ec3-8629-0dd27c352ab5

```shell
curl -s https://dl.jami.net/public-key.gpg | sudo tee /usr/share/keyrings/jami-archive-keyring.gpg > /dev/null
```



ここでは、curlを使ってJamiのGPG公開鍵をダウンロードしている。dev/nullにリダイレクトすることで、生の鍵が画面に表示されるのを防いでいる。



```shell
sudo sh -c "echo 'deb [signed-by=/usr/share/keyrings/jami-archive-keyring.gpg] https://dl.jami.net/stable/ubuntu_25.04/ jami main' > /etc/apt/sources.list.d/jami.list"
```



このコマンドはJami公式リポジトリをAPTソースリストに追加する。



```shell
sudo apt-get update && sudo apt-get install jami
```



最後に、`apt-get update`で利用可能なパッケージのリストを更新し、`apt-get install jami`でJamiを公式リポジトリから直接インストールする。



## ジャミの基本構成



Jamiがシステムにインストールされたら、システムメニューから直接起動できる。



アプリケーションを開始すると、アカウントを作成するか、すでに作成したアカウントを継続するかを選択できます。



![0_07](assets/fr/07.webp)



### アカウントの作成



Jamiアカウントの作成は比較的簡単です。EメールAddressや電話番号は必要ありません：Jamiは最小限の情報のみを収集します。ご希望であれば、*Jami ID*（暗号指紋）を指すユーザー名（エイリアス）を登録することができます。エイリアス↔Jami ID*の関連付けは、デフォルトのネームサーバー（交換可能/セルフホスト）で公開されるため、エイリアスは必ずしも必須ではありません。



![0_08](assets/fr/08.webp)



ローカルでデータを保護するために、パスワードを設定してデバイス上のプロファイルとバックアップを暗号化することができます。このパスワードはオプションであり、デフォルトで有効になっている通信のエンドツーエンド暗号化には影響しません。このローカル保護を有効にする場合は、長くてランダムな一意のパスワードを選択し、検証してください。



![0_09](assets/fr/09.webp)



アカウントを暗号化した後、フルネームを設定します。



![0_10](assets/fr/10.webp)



![0_11](assets/fr/11.webp)



## 既存のアカウントにログインする



Jamiは**識別子**を使用せず、あなたのアカウントに接続するためのデータベースも持っていません。全てのデータはあなたのデバイスに直接保存されます。古いアカウントに接続するには、古いアカウントの**バックアップ**を実行する必要があります。



Settings（設定）***、次に**Account（アカウント）***、次に**Account Management（アカウント管理）***に進みます。一番下までスクロールし、**アカウントのバックアップ**を実行します。バックアップファイルが保存される場所を選択し、アカウントが作成されたときに定義された**パスワード**を入力し、検証します。



![0_12](assets/fr/12.webp)



このバックアップファイルにより、アカウントへの再接続が可能になります。



![0_13](assets/fr/13.webp)



### アーカイブ経由の接続



これを行うには、**Import from a backup archive** ボタンをクリックし、Jamiアカウントのバックアップファイルを選択します。



![0_14](assets/fr/14.webp)



アカウント作成時にパスワードを設定した場合は、パスワードを入力し、確認してください。あなたのデータとメッセージが自動的に表示されます。



![0_15](assets/fr/15.webp)



データを最新の状態に保ち、最新のメッセージを失わないようにするためには、アカウントのバックアップを定期的に取ることが重要です。



### デバイス経由の接続



すでにJamiアカウントで他のデバイスにログインしている場合は、**アカウント**から**接続デバイス**セクションに移動し、**新しいデバイスを接続する**をクリックします。接続コードを入力する欄とともに、QRコードリーダーが表示されます。



![0_16](assets/fr/16.webp)



新しいデバイスでJamiを起動し、**すでにアカウントを持っています**を選択し、**他のデバイスからインポートします**。すると、**QRコード**とログインコードが表示されます。接続するには、QRコードスキャナーを使って他のデバイスに表示されたコードをスキャンします。しかし、接続コードをコピーして、指定されたフィールドに直接貼り付ける方が簡単な場合が多いです。



![0_17](assets/fr/17.webp)



アカウント作成時にパスワードを設定した場合、続行するには新しいデバイスでパスワードを入力する必要があります。



![417](assets/fr/18.webp)



すでに接続されているデバイスで接続を確認して続行します。



![0_19](assets/fr/19.webp)



パスワードが入力されると、デバイスは自動的にアカウントに接続し、同期します。その後、どのデバイスからでもメッセージの送受信が可能になります。



## ジャミに拡張機能を追加する



Jamiの興味深い特徴のひとつは、拡張機能（プラグイン）を使って新しいオプションを統合できることだ。プラグインはネイティブモジュール(C/C++)であり、SDKはその作成を支援するツールとスクリプト(特にPython)を提供する。いくつかのプラグインは[こちら](https://jami.net/extensions/)から直接入手できる。



拡張機能をインストールするには、デスクトップでは、統合された拡張機能ストアを開き、適切なプラグインをダウンロードし、**設定 → 拡張機能 → インストール**でアクティベートします。Androidの場合、ストアは統合されていません。`.jpl`ファイルをダウンロードし、**設定 → 拡張機能 → インストール**から手動でインポートしてください。



![0_20](assets/fr/20.webp)



独自のプラグインを開発するには、ブログ記事 **[Discover the Jami Plugin SDK and create your own plugins](https://jami.net/plugins-sdk/)** をご覧ください。



## 高度な機能



Jamiはまた、アプリケーションの設定や使用において、より高度なことを望むユーザーのために**高度な機能**を提供しています。これらのオプションには以下が含まれます：





- ランデブーポイントの作成**：この機能により、コミュニケーション用の**ランデブーポイント**を作成することができ、複数のユーザー間で安全なセッションや交換を行うのに便利です。
- Jamiサーバー**に接続します：Jamiを**Jamiサーバー**に接続することで、特にプロフェッショナルな環境において、通信のパフォーマンスや可用性を向上させることができます。
- SIPアカウント**の設定SIPアカウント**（セッション開始プロトコル）を設定することで、Jamiを既存の電話システムと統合したり、電話をかけたりすることができます。



![0_21](assets/fr/21.webp)



これらのオプションは、Jami体験をカスタマイズし、特定のニーズに合わせて追加機能を利用したい上級ユーザー向けに設計されています。



一言で言えば、Jamiは完全で安全かつ柔軟なコミュニケーションソリューションであり、個人的な使用にも、**拡張機能**のおかげでより特殊なニーズにも適しています。簡単なインストール、直感的な操作、そして**暗号化**、**マルチデバイス同期**、*独自のプラグインを作成する**能力**などの高度な機能により、**プライバシー**を守りながらコミュニケーションを管理するための強力なツールとなっています。
