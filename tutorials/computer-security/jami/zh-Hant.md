Jami 是一個免費的開放原始碼點對點通訊程式，歷史悠久，功能眾多。其歷史可追溯至 2004 年 12 月，當時位於蒙特婁的 *Savoir-faire Linux* 推出 SFLPhone，這是一個完全以開源技術為基礎的企業數位電話專案。該軟體符合電信標準 (SIP 和 IAX)，因其處理大量線路和通話的能力而脫穎而出。



2015 年，SFLPhone 更名為 Ring，並整合了分散式架構，不再需要中央伺服器。次年，Ring 正式加入 GNU 項目，鞏固了其在自由軟體生態系統中的地位。最後，在 2018 年 12 月，為了避免與使用英文「*Ring*」的商業產品混淆，該軟體採用了目前的名稱：Jami。自此，它繼續發展成為一個自由、去中心化和隱私友好的通訊平台。



如今，Jami 可在許多系統上使用。它以性能、流暢性和易用性著稱。它可讓您透過即時通訊、音訊通話或視訊通話進行溝通，同時透過端對端加密來確保對話內容的機密性。簡易的安裝方式和眾多的功能，讓它成為一個完整的通訊應用程式，日常使用簡單方便。

## 為什麼使用 Jami？





- 它是開放原始碼且完全免費，因此您可以免費使用。
- **完整實用的功能**：本軟體可讓您利用眾多選項，這要歸功於可從商店輕鬆新增外掛程式。您甚至可以建立自己的擴充套件，讓功能更符合您的需求。
- 易於使用且 **Interface 直覺**：儘管 Jami 提供了許多功能，但它仍然非常容易上手。
- **強大的安全性**：Jami 整合了先進的加密演算法，在尊重您隱私的同時，保證您通訊的安全性。
- 高度方便、快速：即使在頻寬有限的情況下，也能提供簡易的通訊方式，增進使用者的便利性。



## 如何安裝 Jami？



在繼續安裝 Jami 之前，請務必注意該應用程式適用於不同的作業系統。因此，其安裝將取決於您使用的系統。



![0_01](assets/fr/01.webp)



### 適用於 Android 或 iOS 使用者



此應用程式可直接從 App Store 或 Play Store 取得。只需在搜尋列中搜尋，然後啟動安裝即可。



### 適用於 Windows 或 macOS 使用者



要在您的設備上安裝 Jami，首先需要訪問 Jami 官方網站。預設情況下，該網站會顯示與您的裝置作業系統相對應的軟體，您可以直接點擊下載按鈕啟動。不過，您也可以直接從 [下載頁面](https://jami.net/download-jami-windows/) 下載 Windows 可執行檔。



![0_02](assets/fr/02.webp)



對於 macOS 使用者，該檔案也可從 [macOS 下載頁面](https://jami.net/download-jami-macos/) 下載。



![0_03](assets/fr/03.webp)



下載執行檔後，按兩下執行檔開始安裝程序。接受使用條款，然後開始安裝，等待安裝完成。



![0_04](assets/fr/04.webp)



![0_05](assets/fr/05.webp)



### 適用於 linux 使用者



要在 Linux 上安裝 Jami，最好使用命令列。需要注意的是，Jami 適用於不同的 Linux 發行版本。在您開始安裝 Jami 之前，請確定您選擇了適合您系統的發行版。



一旦你選擇了你的發行版，你就可以安裝系統了。您需要安裝在 Linux 作業系統上執行 Jami 所需的相依性。這些指令可直接在 [this page](https://jami.net/download-jami-linux/) 上找到。



![0_06](assets/fr/06.webp)



若要在 **Ubuntu ** 上安裝 Jami，您可以按以下方式進行：



```shell
sudo apt install gnupg dirmngr ca-certificates curl --no-install-recommends
```



此指令會安裝管理 GPG 金鑰 (gnupg 和 dirmngr)、SSL 憑證 (ca-certificates) 和 curl 下載工具所需的工具。



https://planb.network/tutorials/computer-security/operating-system/ubuntu-78a3be56-5d51-4ec3-8629-0dd27c352ab5

```shell
curl -s https://dl.jami.net/public-key.gpg | sudo tee /usr/share/keyrings/jami-archive-keyring.gpg > /dev/null
```



這裡使用 curl 下載 Jami 的 GPG 公鑰。重定向到 /dev/null 可以避免在螢幕上顯示原始金鑰。



```shell
sudo sh -c "echo 'deb [signed-by=/usr/share/keyrings/jami-archive-keyring.gpg] https://dl.jami.net/stable/ubuntu_25.04/ jami main' > /etc/apt/sources.list.d/jami.list"
```



此指令會將官方 Jami 套件庫加入 APT 來源清單。



```shell
sudo apt-get update && sudo apt-get install jami
```



最後，使用 `apt-get update` 更新可用套件清單，再使用 `apt-get install jami` 直接從官方套件庫安裝 Jami。



## Jami 基本配置



Jami 安裝到系統後，您可以直接從系統功能表啟動它。



開始申請後，您可以選擇建立帳號或繼續使用已建立的帳號。



![0_07](assets/fr/07.webp)



### 建立帳號



建立 Jami 帳戶相對簡單。您不需要電子郵件 Address 或電話號碼：Jami只收集最低限度的信息。如果您願意，您可以註冊一個用戶名（別名），指向您的*Jami ID*（加密指紋）。*alias ↔ Jami ID* 關聯會在預設的名稱伺服器上公佈（可替換/自託管），因此別名不一定是強制性的。



![0_08](assets/fr/08.webp)



若要在本機保護您的資料，您可以設定密碼來加密裝置上的設定檔和備份。此密碼是可選的，對於預設啟動的通訊端對端加密沒有影響。如果啟動此本機保護，請選擇一個長的、隨機且唯一的密碼，然後驗證。



![0_09](assets/fr/09.webp)



加密帳戶後，設定您的全名。



![0_10](assets/fr/10.webp)



![0_11](assets/fr/11.webp)



## 登入現有帳戶



Jami 不使用**識別碼**，也沒有資料庫可連接到您的帳戶。您的所有資料都直接儲存在您的裝置上。為了連線到您的舊帳號，您需要對您的舊帳號進行**備份**。



移至 **設定**，然後選取 **帳號**，然後選取 **帳號管理**。捲動至底部，執行**帳戶備份**。選擇備份檔案的儲存位置，輸入建立帳號時定義的**密碼**，然後驗證。



![0_12](assets/fr/12.webp)



此備份檔案可讓您重新連線至您的帳戶。



![0_13](assets/fr/13.webp)



### 透過檔案連接



若要執行此動作，請按一下 ** 從備份存檔匯入** 按鈕，然後選擇您 Jami 帳戶的備份檔案。



![0_14](assets/fr/14.webp)



如果您在建立帳戶時設定了密碼，請輸入密碼並確認。您的資料和訊息將會自動顯示。



![0_15](assets/fr/15.webp)



定期備份您的帳戶以保持資料更新並確保您不會遺失最新的訊息是非常重要的。



### 透過裝置連接



如果您已經使用 Jami 帳戶在其他裝置上登入，請前往 ** 帳戶**，然後到 ** 連線裝置** 區段，點選 ** 連線新裝置**。QR 碼閱讀器將會出現，同時會出現輸入連線碼的欄位。



![0_16](assets/fr/16.webp)



在新裝置上啟動 Jami，然後選擇 **我已有帳號**，再選擇 **從其他裝置匯入**。接著您會收到一個 ** QR 碼** 和登入碼。若要連線，您可以使用 QR 代碼掃描器掃描其他裝置上顯示的代碼。不過，複製連線代碼並直接貼到所提供的欄位通常會比較簡單。



![0_17](assets/fr/17.webp)



如果您在建立帳號時設定了密碼，您需要在新裝置上輸入密碼才能繼續。



![417](assets/fr/18.webp)



在已連接的裝置上確認連線，以繼續進行。



![0_19](assets/fr/19.webp)



輸入密碼後，裝置會自動連線至帳戶並進行同步處理。之後您就可以從任何裝置傳送和接收訊息。



## 為 Jami 新增擴充套件



Jami 其中一個有趣的功能是它能夠透過擴充套件 (外掛) 整合新的選項。外掛是原生模組 (C/C++)；SDK 提供工具和腳本 (主要是 Python) 來協助建立外掛。有些外掛可直接使用 [這裡](https://jami.net/extensions/)。



若要安裝擴充套件，在桌上型電腦上開啟整合的擴充套件商店，下載適當的外掛程式，然後到 **設定 → 擴充套件 → 安裝** 並將其啟用。在 Android 上，商店未整合：下載 `.jpl` 檔案，然後從 ** 設定 → 擴充套件 → 安裝** 手動匯入；匯入是自動進行的，然後您可以啟用它，並視需要調整其設定。



![0_20](assets/fr/20.webp)



若要開發您自己的外掛，請參閱部落格文章 **[探索 Jami Plugin SDK 並建立您自己的外掛](https://jami.net/plugins-sdk/)**。



## 進階功能



Jami 也提供**進階功能**給希望進一步設定和使用應用程式的使用者。這些選項包括 ：





- 建立會合點：此功能可讓您為通訊建立**會合點**，有助於組織安全的會話或多位使用者之間的交換。
- 連接至 **Jami 伺服器**：您可以將 Jami 連接到 **Jami 伺服器**，這樣可以提高通訊效能或可用性，尤其是在專業環境中。
- 建立SIP帳戶：您可以設定**SIP 帳戶** (Session Initiation Protocol)，讓您可以將 Jami 整合至現有的電話系統或撥打電話。



![0_21](assets/fr/21.webp)



這些選項專為希望自訂 Jami 體驗和利用額外功能以滿足特定需求的進階使用者而設計。
