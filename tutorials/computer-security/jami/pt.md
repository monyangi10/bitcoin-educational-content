O Jami é um programa de mensagens peer-to-peer gratuito e de código aberto com uma longa história e uma vasta gama de funcionalidades. A sua história remonta a dezembro de 2004, quando a *Savoir-faire Linux*, sediada em Montreal, lançou o SFLPhone, um projeto de telefonia digital para empresas baseado inteiramente em tecnologias de código aberto. O software, que cumpria as normas de telecomunicações (SIP e IAX), destacava-se pela sua capacidade de gerir um grande número de linhas e chamadas.



Em 2015, o SFLPhone passou a chamar-se Ring e integrou uma arquitetura distribuída que já não necessita de um servidor central. No ano seguinte, o Ring aderiu oficialmente ao projeto GNU, reforçando a sua posição no ecossistema do software livre. Finalmente, em dezembro de 2018, para evitar confusão com produtos comerciais que utilizam o termo inglês "*Ring*", o software adoptou o seu nome atual: Jami. Desde então, tem continuado a evoluir como uma plataforma de comunicação livre, descentralizada e amiga da privacidade.



Atualmente, o Jami está disponível em muitos sistemas. É conhecido pelo seu desempenho, fluidez e facilidade de utilização. Permite-lhe comunicar através de mensagens instantâneas, chamadas áudio ou videochamadas, assegurando simultaneamente a confidencialidade das suas conversas através de encriptação de ponta a ponta. A instalação simples e uma série de funcionalidades fazem dela uma aplicação de comunicação completa, fácil e cómoda de utilizar diariamente.

## Porquê utilizar a Jami?





- É de código aberto e totalmente gratuito**, pelo que pode utilizá-lo sem qualquer custo.
- Completo com funcionalidades úteis**: este software permite-lhe tirar partido de inúmeras opções, graças à possibilidade de adicionar facilmente plug-ins a partir da loja. É mesmo possível criar as suas próprias extensões para obter funcionalidades mais adaptadas às suas necessidades.
- Fácil de utilizar e intuitivo**: apesar das muitas funcionalidades que o Jami tem para oferecer, continua a ser muito fácil de manusear.
- Segurança forte**: O Jami incorpora um algoritmo de encriptação avançado que garante a segurança das suas comunicações, respeitando a sua privacidade.
- Altamente acessível e rápido**: oferece uma comunicação fácil, mesmo quando a largura de banda é limitada, para uma maior facilidade de utilização.



## Como é que instalo o Jami?



Antes de passar à instalação do Jami, é importante notar que a aplicação está disponível em diferentes sistemas operativos. A sua instalação dependerá, portanto, do sistema que estiver a utilizar.



![0_01](assets/fr/01.webp)



### Para utilizadores de Android ou iOS



A aplicação está disponível diretamente na App Store ou na Play Store. Basta procurá-la na barra de pesquisa e, em seguida, iniciar a instalação.



### Para utilizadores do Windows ou do macOS



Para instalar o Jami no seu dispositivo, primeiro precisa de visitar o site oficial do Jami. Por predefinição, o site apresenta o software correspondente ao sistema operativo do seu dispositivo e pode clicar diretamente no botão de transferência para o iniciar. No entanto, também pode transferir o executável para Windows diretamente da [página de transferência] (https://jami.net/download-jami-windows/).



![0_02](assets/fr/02.webp)



Para os utilizadores do macOS, o ficheiro também está disponível na [página de transferência do macOS] (https://jami.net/download-jami-macos/).



![0_03](assets/fr/03.webp)



Depois de ter descarregado o executável, inicie o processo de instalação fazendo duplo clique sobre ele. Aceite os termos de utilização, inicie a instalação e aguarde que esta termine.



![0_04](assets/fr/04.webp)



![0_05](assets/fr/05.webp)



### Para utilizadores de linux



Para instalar o Jami no Linux, é melhor utilizar a linha de comandos. É importante notar que o Jami está disponível para diferentes distribuições Linux. Antes de começar a instalar o Jami, certifique-se de que escolhe a distribuição correta para o seu sistema.



Depois de ter selecionado a sua distribuição, pode instalar o sistema. Terá de instalar as dependências necessárias para executar o Jami no seu sistema operativo Linux. Os comandos estão diretamente disponíveis em [esta página](https://jami.net/download-jami-linux/).



![0_06](assets/fr/06.webp)



Para instalar o Jami no **Ubuntu**, pode fazê-lo da seguinte forma:



```shell
sudo apt install gnupg dirmngr ca-certificates curl --no-install-recommends
```



Este comando instala as ferramentas necessárias para gerir chaves GPG (gnupg e dirmngr), certificados SSL (ca-certificates) e a ferramenta de download curl.



https://planb.network/tutorials/computer-security/operating-system/ubuntu-78a3be56-5d51-4ec3-8629-0dd27c352ab5

```shell
curl -s https://dl.jami.net/public-key.gpg | sudo tee /usr/share/keyrings/jami-archive-keyring.gpg > /dev/null
```



Aqui, o curl é usado para descarregar a chave pública GPG de Jami. O redireccionamento para /dev/null evita a apresentação da chave bruta no ecrã.



```shell
sudo sh -c "echo 'deb [signed-by=/usr/share/keyrings/jami-archive-keyring.gpg] https://dl.jami.net/stable/ubuntu_25.04/ jami main' > /etc/apt/sources.list.d/jami.list"
```



Este comando adiciona o repositório oficial Jami à lista de fontes APT.



```shell
sudo apt-get update && sudo apt-get install jami
```



Finalmente, atualize a lista de pacotes disponíveis com `apt-get update`, e então instale o Jami diretamente do repositório oficial com `apt-get install jami`.



## Configuração básica do Jami



Depois de instalar o Jami no seu sistema, pode iniciá-lo diretamente a partir do menu do sistema.



Depois de iniciar a aplicação, terá a opção de criar uma conta ou de continuar com uma conta já criada.



![0_07](assets/fr/07.webp)



### Criar uma conta



Criar uma conta Jami é relativamente simples. Não é necessário um endereço de correio eletrónico Address ou um número de telefone: O Jami recolhe apenas o mínimo de informações. Se desejar, pode registar um nome de utilizador (alias) que aponta para o seu *Jami ID* (impressão digital criptográfica). A associação *alias ↔ Jami ID* é publicada num servidor de nomes predefinido (substituível / auto-hospedagem), pelo que o alias não é necessariamente obrigatório.



![0_08](assets/fr/08.webp)



Para proteger os seus dados localmente, pode definir uma palavra-passe para encriptar o seu perfil e as cópias de segurança no dispositivo. Esta palavra-passe é opcional e não tem qualquer efeito na encriptação de ponta a ponta das comunicações, que está ativa por predefinição. Se ativar esta proteção local, escolha uma palavra-passe longa, aleatória e única e, em seguida, valide-a.



![0_09](assets/fr/09.webp)



Depois de encriptar a sua conta, defina o seu nome completo.



![0_10](assets/fr/10.webp)



![0_11](assets/fr/11.webp)



## Aceder a uma conta existente



A Jami não utiliza **identificadores** e não tem uma base de dados para ligar à sua conta. Todos os seus dados são armazenados diretamente no seu dispositivo. Para se ligar à sua conta antiga, é necessário efetuar uma **cópia de segurança** da sua conta antiga.



Aceda a **Definições**, depois **Conta**, depois **Gestão de contas**. Desloque-se para a parte inferior e efectue uma **cópia de segurança da sua conta**. Escolha o local onde o ficheiro de cópia de segurança será guardado, introduza a **palavra-passe** definida aquando da criação da sua conta e, em seguida, valide.



![0_12](assets/fr/12.webp)



Este ficheiro de cópia de segurança permitir-lhe-á voltar a ligar-se à sua conta.



![0_13](assets/fr/13.webp)



### Ligação através de arquivo



Para o fazer, clique no botão **Importar de um arquivo de cópia de segurança** e selecione o ficheiro de cópia de segurança da sua conta Jami.



![0_14](assets/fr/14.webp)



Se definiu uma palavra-passe quando criou a sua conta, introduza-a e confirme. Os seus dados e mensagens aparecerão automaticamente.



![0_15](assets/fr/15.webp)



É importante fazer cópias de segurança regulares da sua conta para manter os dados actualizados e garantir que não perde as mensagens mais recentes.



### Ligação através de um dispositivo



Se já tiver iniciado sessão noutro dispositivo com a sua conta Jami, aceda a **Conta** e, em seguida, à secção **Dispositivo ligado** e clique em **Ligar um novo dispositivo**. Aparecerá o leitor de código QR, juntamente com um campo para introduzir um código de ligação.



![0_16](assets/fr/16.webp)



Inicie o Jami no novo dispositivo e selecione **Já tenho uma conta** e, em seguida, **Importar de outro dispositivo**. Receberá então um **código QR** e um código de início de sessão. Para estabelecer a ligação, pode utilizar o leitor de código QR para ler o código apresentado no outro dispositivo. No entanto, muitas vezes é mais simples copiar o código de ligação e colá-lo diretamente no campo fornecido.



![0_17](assets/fr/17.webp)



Se definiu uma palavra-passe quando criou a sua conta, terá de a introduzir no novo dispositivo para continuar.



![417](assets/fr/18.webp)



Confirme a ligação no dispositivo já ligado para continuar.



![0_19](assets/fr/19.webp)



Assim que a palavra-passe for introduzida, o dispositivo liga-se automaticamente à conta e sincroniza-se. Pode então enviar e receber mensagens a partir de qualquer um dos seus dispositivos.



## Adicionar uma extensão a Jami



Uma das caraterísticas interessantes do Jami é a sua capacidade de integrar novas opções através de extensões (plugins). Os plugins são módulos nativos (C/C++); o SDK fornece ferramentas e scripts (nomeadamente em Python) para ajudar a criá-los. Alguns plugins estão disponíveis diretamente [aqui](https://jami.net/extensions/).



Para instalar uma extensão, no ambiente de trabalho, abra a loja de extensões integrada, transfira o plug-in adequado e, em seguida, vá a **Configurações → Extensões → Instalar** e active-o. No Android, a loja não está integrada: descarregue o ficheiro `.jpl`, depois importe-o manualmente a partir de **Configurações → Extensões → Instalar**; a importação é automática e pode depois activá-lo e ajustar as suas definições, se necessário.



![0_20](assets/fr/20.webp)



Para desenvolver os seus próprios plug-ins, consulte a publicação no blogue **[Descubra o Jami Plugin SDK e crie os seus próprios plug-ins](https://jami.net/plugins-sdk/)**.



## Funcionalidades avançadas



O Jami também oferece **caraterísticas avançadas** para os utilizadores que pretendam ir mais longe na configuração e utilização da aplicação. Estas opções incluem:





- Criar um ponto de encontro**: Esta função permite-lhe criar um **ponto de encontro** para as suas comunicações, útil para organizar sessões seguras ou intercâmbios entre vários utilizadores.
- Ligar a um servidor Jami**: Pode ligar o Jami a um **servidor Jami**, o que pode melhorar o desempenho ou a disponibilidade das comunicações, especialmente em ambientes profissionais.
- Criar uma conta SIP**: Pode configurar uma **conta SIP** (Session Initiation Protocol), que lhe permite integrar o Jami nos sistemas telefónicos existentes ou fazer chamadas telefónicas.



![0_21](assets/fr/21.webp)



Estas opções foram concebidas para utilizadores mais avançados que pretendam personalizar a sua experiência Jami e tirar partido de funcionalidades adicionais para necessidades específicas.



Em suma, o Jami é uma solução de comunicação completa, segura e flexível, adequada tanto para uso pessoal como para necessidades mais específicas graças às suas **extensões**. A sua instalação simples, o seu manuseamento intuitivo e as suas funcionalidades avançadas, como a **encriptação**, a **sincronização entre vários dispositivos** e a possibilidade de **criar os seus próprios plugins**, fazem dele uma ferramenta poderosa para gerir as suas comunicações, preservando a sua **privacidade**.
