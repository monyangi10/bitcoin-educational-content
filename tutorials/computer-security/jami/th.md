Jami เป็นโปรแกรมส่งข้อความแบบ peer-to-peer ที่ฟรีและโอเพ่นซอร์สซึ่งมีประวัติยาวนานและมีฟีเจอร์หลากหลาย ประวัติของมันย้อนกลับไปในเดือนธันวาคม 2004 เมื่อ *Savoir-faire Linux* ซึ่งตั้งอยู่ในมอนทรีออลได้เปิดตัว SFLPhone ซึ่งเป็นโครงการโทรศัพท์ดิจิทัลสำหรับธุรกิจที่อิงจากเทคโนโลยีโอเพ่นซอร์สทั้งหมด ซอฟต์แวร์นี้ซึ่งเป็นไปตามมาตรฐานโทรคมนาคม (SIP และ IAX) โดดเด่นด้วยความสามารถในการจัดการสายและการโทรจำนวนมาก



ในปี 2015 SFLPhone ได้เปลี่ยนชื่อเป็น Ring และรวมสถาปัตยกรรมแบบกระจายที่ไม่ต้องการเซิร์ฟเวอร์กลางอีกต่อไป ในปีถัดมา Ring ได้เข้าร่วมโครงการ GNU อย่างเป็นทางการ เสริมสร้างฐานที่มั่นในระบบนิเวศซอฟต์แวร์เสรี สุดท้ายในเดือนธันวาคม 2018 เพื่อหลีกเลี่ยงความสับสนกับผลิตภัณฑ์เชิงพาณิชย์ที่ใช้คำภาษาอังกฤษว่า "*Ring*" ซอฟต์แวร์ได้เปลี่ยนชื่อเป็น Jami ตั้งแต่นั้นมา มันได้พัฒนาอย่างต่อเนื่องในฐานะแพลตฟอร์มการสื่อสารที่เสรี กระจาย และเป็นมิตรกับความเป็นส่วนตัว



วันนี้ Jami มีให้ใช้งานบนหลายระบบ เป็นที่รู้จักในด้านประสิทธิภาพ ความลื่นไหล และความง่ายในการใช้งาน ช่วยให้คุณสื่อสารผ่านการส่งข้อความทันที การโทรด้วยเสียง หรือการโทรผ่านวิดีโอ พร้อมทั้งรับรองความลับของการสนทนาของคุณด้วยการเข้ารหัสแบบ end-to-end การติดตั้งที่ง่ายและฟีเจอร์มากมายทำให้เป็นแอปพลิเคชันการสื่อสารที่ครบถ้วน ใช้งานง่ายและสะดวกในชีวิตประจำวัน

## ทำไมต้องใช้ Jami?





- มันเป็น **โอเพ่นซอร์สและฟรีทั้งหมด** ดังนั้นคุณสามารถใช้งานได้โดยไม่มีค่าใช้จ่าย
- ครบครันด้วยฟีเจอร์ที่มีประโยชน์**: ซอฟต์แวร์นี้ให้คุณใช้ประโยชน์จากตัวเลือกมากมาย ด้วยความสามารถในการเพิ่มปลั๊กอินจากร้านค้าได้อย่างง่ายดาย นอกจากนี้ยังสามารถสร้างส่วนขยายของคุณเองสำหรับฟีเจอร์ที่เหมาะสมกับความต้องการของคุณมากขึ้น
- ใช้งานง่ายและ Interface ใช้งานง่าย**: แม้ว่า Jami จะมีฟีเจอร์มากมายให้ใช้งาน แต่ก็ยังคงใช้งานได้ง่ายมาก
- ความปลอดภัยที่แข็งแกร่ง**: Jami รวมเอาอัลกอริธึมการเข้ารหัสขั้นสูงที่รับประกันความปลอดภัยของการสื่อสารของคุณในขณะที่เคารพความเป็นส่วนตัวของคุณ
- เข้าถึงได้ง่ายและรวดเร็ว**: มอบการสื่อสารที่ง่ายดายแม้ในขณะที่แบนด์วิธจำกัด เพื่อเพิ่มความสะดวกสบายให้กับผู้ใช้



## ฉันจะติดตั้ง Jami ได้อย่างไร?



ก่อนที่จะดำเนินการติดตั้ง Jami สิ่งสำคัญคือต้องทราบว่าแอปพลิเคชันนี้มีให้ใช้งานบนระบบปฏิบัติการที่แตกต่างกัน ดังนั้นการติดตั้งจะขึ้นอยู่กับระบบที่คุณใช้งานอยู่



![0_01](assets/fr/01.webp)



### สำหรับผู้ใช้ Android หรือ iOS



แอปพลิเคชันสามารถดาวน์โหลดได้โดยตรงจาก App Store หรือ Play Store เพียงค้นหาในแถบค้นหาแล้วเริ่มการติดตั้ง



### สำหรับผู้ใช้ Windows หรือ macOS



ในการติดตั้ง Jami บนอุปกรณ์ของคุณ คุณต้องไปที่เว็บไซต์ทางการของ Jami ก่อน โดยปกติแล้ว เว็บไซต์จะแสดงซอฟต์แวร์ที่ตรงกับระบบปฏิบัติการของอุปกรณ์ของคุณ และคุณสามารถคลิกที่ปุ่มดาวน์โหลดเพื่อเริ่มการติดตั้งได้โดยตรง อย่างไรก็ตาม คุณยังสามารถดาวน์โหลดไฟล์ปฏิบัติการของ Windows ได้โดยตรงจาก [หน้าดาวน์โหลด](https://jami.net/download-jami-windows/)

### สำหรับผู้ใช้ลินุกซ์



ในการติดตั้ง Jami บน Linux ควรใช้บรรทัดคำสั่ง สิ่งสำคัญคือต้องทราบว่า Jami มีให้สำหรับการแจกจ่าย Linux ที่แตกต่างกัน ก่อนที่คุณจะเริ่มติดตั้ง Jami ให้แน่ใจว่าคุณเลือกการแจกจ่ายที่ถูกต้องสำหรับระบบของคุณ



เมื่อคุณเลือกการกระจายของคุณแล้ว คุณสามารถติดตั้งระบบได้ คุณจะต้องติดตั้งการพึ่งพาที่จำเป็นในการรัน Jami บน Linux OS ของคุณ คำสั่งมีให้โดยตรงบน [หน้านี้](https://jami.net/download-jami-linux/)



![0_06](assets/fr/06.webp)



ในการติดตั้ง Jami บน **Ubuntu** คุณสามารถทำได้ดังนี้:



```shell
sudo apt install gnupg dirmngr ca-certificates curl --no-install-recommends
```



คำสั่งนี้ติดตั้งเครื่องมือที่จำเป็นสำหรับจัดการคีย์ GPG (gnupg และ dirmngr), ใบรับรอง SSL (ca-certificates) และเครื่องมือดาวน์โหลด curl.



https://planb.academy/tutorials/computer-security/operating-system/ubuntu-78a3be56-5d51-4ec3-8629-0dd27c352ab5

```shell
curl -s https://dl.jami.net/public-key.gpg | sudo tee /usr/share/keyrings/jami-archive-keyring.gpg > /dev/null
```



ที่นี่ curl ถูกใช้เพื่อดาวน์โหลด GPG public key ของ Jami การเปลี่ยนเส้นทางไปยัง /dev/null ช่วยหลีกเลี่ยงการแสดงคีย์ดิบบนหน้าจอ



```shell
sudo sh -c "echo 'deb [signed-by=/usr/share/keyrings/jami-archive-keyring.gpg] https://dl.jami.net/stable/ubuntu_25.04/ jami main' > /etc/apt/sources.list.d/jami.list"
```



คำสั่งนี้เพิ่มที่เก็บ Jami อย่างเป็นทางการไปยังรายการแหล่งที่มาของ APT



```shell
sudo apt-get update && sudo apt-get install jami
```



สุดท้ายนี้ อัปเดตรายการแพ็กเกจที่มีอยู่ด้วย `apt-get update` จากนั้นติดตั้ง Jami โดยตรงจากที่เก็บข้อมูลอย่างเป็นทางการด้วย `apt-get install jami`.



## Jami การกำหนดค่าพื้นฐาน



เมื่อ Jami ได้รับการติดตั้งบนระบบของคุณแล้ว คุณสามารถเปิดใช้งานได้โดยตรงจากเมนูระบบ



เมื่อคุณเริ่มต้นแอปพลิเคชันแล้ว คุณจะมีตัวเลือกในการสร้างบัญชีใหม่หรือดำเนินการต่อด้วยบัญชีที่คุณได้สร้างไว้แล้ว



![0_07](assets/fr/07.webp)



### สร้างบัญชี



การสร้างบัญชี Jami นั้นค่อนข้างง่าย คุณไม่จำเป็นต้องมีที่อยู่อีเมลหรือหมายเลขโทรศัพท์: Jami เก็บรวบรวมข้อมูลเพียงขั้นต่ำเท่านั้น หากคุณต้องการ คุณสามารถลงทะเบียนชื่อผู้ใช้ (นามแฝง) ที่ชี้ไปยัง *Jami ID* (ลายนิ้วมือเข้ารหัส) การเชื่อมโยง *นามแฝง ↔ Jami ID* จะถูกเผยแพร่บนเซิร์ฟเวอร์ชื่อเริ่มต้น (สามารถเปลี่ยนได้ / โฮสต์เอง) ดังนั้นนามแฝงจึงไม่จำเป็นต้องมีเสมอไป



![0_08](assets/fr/08.webp)



เพื่อปกป้องข้อมูลของคุณในเครื่อง คุณสามารถตั้งรหัสผ่านเพื่อเข้ารหัสโปรไฟล์และการสำรองข้อมูลบนอุปกรณ์ได้ รหัสผ่านนี้เป็นทางเลือกและไม่มีผลต่อการเข้ารหัสแบบ end-to-end ของการสื่อสาร ซึ่งเปิดใช้งานโดยค่าเริ่มต้น หากคุณเปิดใช้งานการป้องกันในเครื่องนี้ ให้เลือกรหัสผ่านที่ยาว สุ่ม และไม่ซ้ำกัน จากนั้นยืนยัน



![0_09](assets/fr/09.webp)



หลังจากเข้ารหัสบัญชีของคุณ ให้ตั้งชื่อเต็มของคุณ

## เข้าสู่ระบบบัญชีที่มีอยู่



Jami ไม่ใช้ **ตัวระบุ** และไม่มีฐานข้อมูลเพื่อเชื่อมต่อกับบัญชีของคุณ ข้อมูลทั้งหมดของคุณจะถูกเก็บไว้โดยตรงบนอุปกรณ์ของคุณ เพื่อเชื่อมต่อกับบัญชีเก่าของคุณ คุณจำเป็นต้องทำการ **สำรองข้อมูล** ของบัญชีเก่าของคุณ



ไปที่ **การตั้งค่า** จากนั้น **บัญชี** แล้วเลือก **การจัดการบัญชี** เลื่อนลงไปที่ด้านล่างและทำการ **สำรองข้อมูลบัญชีของคุณ** เลือกตำแหน่งที่ไฟล์สำรองข้อมูลจะถูกบันทึก ป้อน **รหัสผ่าน** ที่กำหนดเมื่อสร้างบัญชีของคุณ จากนั้นยืนยัน



![0_12](assets/fr/12.webp)



ไฟล์สำรองนี้จะช่วยให้คุณเชื่อมต่อกับบัญชีของคุณได้อีกครั้ง



![0_13](assets/fr/13.webp)



### การเชื่อมต่อผ่านที่เก็บถาวร



ในการทำเช่นนี้ ให้คลิกที่ปุ่ม **นำเข้าจากไฟล์สำรองข้อมูล** และเลือกไฟล์สำรองข้อมูลสำหรับบัญชี Jami ของคุณ



![0_14](assets/fr/14.webp)



หากคุณตั้งรหัสผ่านเมื่อสร้างบัญชี ให้ป้อนและยืนยัน ข้อมูลและข้อความของคุณจะปรากฏโดยอัตโนมัติ



![0_15](assets/fr/15.webp)



สิ่งสำคัญคือต้องทำการสำรองข้อมูลบัญชีของคุณเป็นประจำเพื่อให้ข้อมูลของคุณเป็นปัจจุบันและมั่นใจว่าคุณจะไม่สูญเสียข้อความล่าสุดของคุณ



### การเชื่อมต่อผ่านอุปกรณ์



หากคุณได้ลงชื่อเข้าใช้ด้วยบัญชี Jami บนอุปกรณ์อื่นแล้ว ให้ไปที่ **บัญชี** จากนั้นไปที่ส่วน **อุปกรณ์ที่เชื่อมต่อ** และคลิกที่ **เชื่อมต่ออุปกรณ์ใหม่** เครื่องอ่าน QR code จะปรากฏขึ้น พร้อมกับช่องสำหรับป้อนรหัสการเชื่อมต่อ



![0_16](assets/fr/16.webp)



เปิดตัว Jami บนอุปกรณ์ใหม่และเลือก **ฉันมีบัญชีอยู่แล้ว** จากนั้นเลือก **นำเข้าจากอุปกรณ์อื่น** คุณจะได้รับ **QR code** และรหัสเข้าสู่ระบบ เพื่อเชื่อมต่อ คุณสามารถใช้เครื่องสแกน QR code เพื่อสแกนโค้ดที่แสดงบนอุปกรณ์อื่น อย่างไรก็ตาม มักจะง่ายกว่าที่จะคัดลอกรหัสการเชื่อมต่อและวางลงในช่องที่ให้ไว้โดยตรง



![0_17](assets/fr/17.webp)



หากคุณตั้งรหัสผ่านเมื่อคุณสร้างบัญชี คุณจะต้องป้อนรหัสผ่านนั้นบนอุปกรณ์ใหม่เพื่อดำเนินการต่อ



![417](assets/fr/18.webp)



ยืนยันการเชื่อมต่อบนอุปกรณ์ที่เชื่อมต่อแล้วเพื่อดำเนินการต่อ



![0_19](assets/fr/19.webp)



เมื่อป้อนรหัสผ่านแล้ว อุปกรณ์จะเชื่อมต่อกับบัญชีและซิงโครไนซ์โดยอัตโนมัติ จากนั้นคุณสามารถส่งและรับข้อความจากอุปกรณ์ใด ๆ ของคุณได้



## เพิ่มส่วนขยายให้กับ Jami



คุณสมบัติที่น่าสนใจอย่างหนึ่งของ Jami คือความสามารถในการรวมตัวเลือกใหม่ผ่านส่วนขยาย (ปลั๊กอิน) ปลั๊กอินเป็นโมดูลเนทีฟ (C/C++); SDK มีเครื่องมือและสคริปต์ (โดยเฉพาะใน Python) เพื่อช่วยในการสร้างปลั๊กอินเหล่านี้ ปลั๊กอินบางตัวสามารถใช้งานได้โดยตรง [ที่นี่](https://jami.net/extensions/)



เมื่อต้องการติดตั้งส่วนขยาย บนเดสก์ท็อปให้เปิดร้านค้าส่วนขยายที่รวมอยู่ ดาวน์โหลดปลั๊กอินที่เหมาะสม จากนั้นไปที่ **การตั้งค่า → ส่วนขยาย → ติดตั้ง** และเปิดใช้งาน บน Android ร้านค้าไม่ได้ถูกรวมอยู่: ดาวน์โหลดไฟล์ `.jpl` จากนั้นนำเข้าไฟล์ด้วยตนเองจาก **การตั้งค่า → ส่วนขยาย → ติดตั้ง**; การนำเข้าจะเป็นไปโดยอัตโนมัติและคุณสามารถเปิดใช้งานและปรับการตั้งค่าได้หากจำเป็น



![0_20](assets/fr/20.webp)



หากต้องการพัฒนาปลั๊กอินของคุณเอง โปรดดูบล็อกโพสต์ **[ค้นพบ Jami Plugin SDK และสร้างปลั๊กอินของคุณเอง](https://jami.net/plugins-sdk/)**



## คุณสมบัติขั้นสูง



Jami ยังมี **คุณสมบัติขั้นสูง** สำหรับผู้ใช้ที่ต้องการก้าวไปอีกขั้นในการกำหนดค่าและการใช้งานแอปพลิเคชัน ตัวเลือกเหล่านี้รวมถึง:





- สร้างจุดนัดพบ**: ฟีเจอร์นี้ช่วยให้คุณสร้าง **จุดนัดพบ** สำหรับการสื่อสารของคุณ ซึ่งมีประโยชน์สำหรับการจัดการเซสชันที่ปลอดภัยหรือการแลกเปลี่ยนระหว่างผู้ใช้หลายคน
- เชื่อมต่อกับเซิร์ฟเวอร์ Jami**: คุณสามารถเชื่อมต่อ Jami กับ **เซิร์ฟเวอร์ Jami** ซึ่งสามารถปรับปรุงประสิทธิภาพหรือความพร้อมใช้งานของการสื่อสารได้ โดยเฉพาะในสภาพแวดล้อมทางวิชาชีพ
- ตั้งค่าบัญชี SIP**: คุณสามารถตั้งค่า **บัญชี SIP** (Session Initiation Protocol) เพื่อให้คุณสามารถรวม Jami เข้ากับระบบโทรศัพท์ที่มีอยู่หรือโทรออกได้



![0_21](assets/fr/21.webp)



ตัวเลือกเหล่านี้ถูกออกแบบมาสำหรับผู้ใช้ขั้นสูงที่ต้องการปรับแต่งประสบการณ์การใช้งาน Jami ของตนและใช้ประโยชน์จากฟีเจอร์เพิ่มเติมเพื่อตอบสนองความต้องการเฉพาะเจาะจง



โดยสรุปแล้ว Jami เป็นโซลูชันการสื่อสารที่ครบถ้วน ปลอดภัย และยืดหยุ่น เหมาะสำหรับการใช้งานส่วนตัวและความต้องการเฉพาะทางด้วย **ส่วนขยาย** การติดตั้งที่ง่ายดาย การใช้งานที่เข้าใจง่าย และคุณสมบัติขั้นสูง เช่น **การเข้ารหัส** การ **ซิงโครไนซ์หลายอุปกรณ์** และความสามารถในการ **สร้างปลั๊กอินของคุณเอง** ทำให้เป็นเครื่องมือที่ทรงพลังสำหรับการจัดการการสื่อสารของคุณในขณะที่รักษา **ความเป็นส่วนตัว** ของคุณ



ค้นพบ Tox โปรโตคอลแบบกระจายศูนย์ที่รวมการเข้ารหัสแบบปลายทางถึงปลายทาง (E2E), กุญแจสาธารณะ และอัลกอริธึมอื่น ๆ อีกมากมายเพื่อมอบการสื่อสารที่ดีที่สุดที่ปกป้องความลับของคุณผ่านลูกค้าต่าง ๆ ของมัน
