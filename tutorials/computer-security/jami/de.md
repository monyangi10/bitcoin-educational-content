Jami ist ein freies, quelloffenes Peer-to-Peer-Messaging-Programm mit einer langen Geschichte und einer breiten Palette von Funktionen. Seine Geschichte geht auf den Dezember 2004 zurück, als das in Montreal ansässige Unternehmen *Savoir-faire Linux* SFLPhone ins Leben rief, ein digitales Telefonieprojekt für Unternehmen, das vollständig auf Open-Source-Technologien basiert. Die Software, die den Telekommunikationsstandards (SIP und IAX) entsprach, zeichnete sich durch ihre Fähigkeit aus, eine große Anzahl von Leitungen und Anrufen zu verwalten.



Im Jahr 2015 wurde SFLPhone in Ring umbenannt und integrierte eine verteilte Architektur, die keinen zentralen Server mehr benötigt. Im darauffolgenden Jahr trat Ring offiziell dem GNU-Projekt bei und stärkte damit seine Position im Ökosystem der freien Software. Um Verwechslungen mit kommerziellen Produkten, die den englischen Begriff "*Ring*" verwenden, zu vermeiden, nahm die Software im Dezember 2018 ihren aktuellen Namen an: Jami. Seitdem hat sie sich als freie, dezentralisierte und datenschutzfreundliche Kommunikationsplattform weiterentwickelt.



Heute ist Jami auf vielen Systemen verfügbar. Es ist bekannt für seine Leistung, Flüssigkeit und Benutzerfreundlichkeit. Mit Jami können Sie über Instant Messaging, Audio- oder Videoanrufe kommunizieren, wobei die Vertraulichkeit Ihrer Gespräche durch eine Ende-zu-Ende-Verschlüsselung gewährleistet ist. Die einfache Installation und eine Vielzahl von Funktionen machen es zu einer kompletten Kommunikationsanwendung, die einfach und bequem im Alltag zu nutzen ist.

## Warum Jami verwenden?





- Es ist quelloffen und völlig kostenlos**, Sie können es also kostenlos nutzen.
- Voller nützlicher Funktionen**: Mit dieser Software können Sie dank der Möglichkeit, Plugins einfach über den Store hinzuzufügen, von zahlreichen Optionen profitieren. Es ist sogar möglich, eigene Erweiterungen für Funktionen zu erstellen, die Ihren Bedürfnissen besser entsprechen.
- Einfach zu bedienen und Interface intuitiv**: Trotz der vielen Funktionen, die Jami zu bieten hat, ist es immer noch sehr einfach zu handhaben.
- Starke Sicherheit**: Jami verfügt über einen fortschrittlichen Verschlüsselungsalgorithmus, der die Sicherheit Ihrer Kommunikation garantiert und gleichzeitig Ihre Privatsphäre schützt.
- Hohe Zugänglichkeit und Schnelligkeit**: Die Kommunikation ist auch bei eingeschränkter Bandbreite einfach und benutzerfreundlich.



## Wie kann ich Jami installieren?



Bevor wir mit der Installation von Jami fortfahren, ist es wichtig zu wissen, dass die Anwendung auf verschiedenen Betriebssystemen verfügbar ist. Die Installation hängt daher von dem System ab, das Sie verwenden.



![0_01](assets/fr/01.webp)



### Für Android- oder iOS-Benutzer



Die Anwendung ist direkt im App Store oder Play Store erhältlich. Suchen Sie einfach in der Suchleiste danach und starten Sie dann die Installation.



### Für Benutzer von Windows oder macOS



Um Jami auf Ihrem Gerät zu installieren, müssen Sie zunächst die offizielle Jami-Website besuchen. Standardmäßig wird auf der Website die Software angezeigt, die dem Betriebssystem Ihres Geräts entspricht, und Sie können direkt auf die Schaltfläche zum Herunterladen klicken, um sie zu starten. Sie können jedoch auch die ausführbare Windows-Datei direkt von der [Download-Seite](https://jami.net/download-jami-windows/) herunterladen.



![0_02](assets/fr/02.webp)



Für macOS-Benutzer ist die Datei auch auf der [macOS-Downloadseite] (https://jami.net/download-jami-macos/) verfügbar.



![0_03](assets/fr/03.webp)



Sobald Sie die ausführbare Datei heruntergeladen haben, starten Sie den Installationsvorgang durch einen Doppelklick darauf. Akzeptieren Sie die Nutzungsbedingungen, starten Sie dann die Installation und warten Sie, bis sie abgeschlossen ist.



![0_04](assets/fr/04.webp)



![0_05](assets/fr/05.webp)



### Für Linux-Benutzer



Um Jami unter Linux zu installieren, ist es am besten, die Kommandozeile zu benutzen. Es ist wichtig zu beachten, dass Jami für verschiedene Linux-Distributionen verfügbar ist. Bevor Sie mit der Installation von Jami beginnen, stellen Sie sicher, dass Sie die richtige Distribution für Ihr System auswählen.



Sobald du deine Distribution ausgewählt hast, kannst du das System installieren. Du musst die Abhängigkeiten installieren, die nötig sind, um Jami auf deinem Linux Betriebssystem laufen zu lassen. Die Befehle sind direkt auf [dieser Seite](https://jami.net/download-jami-linux/) verfügbar.



![0_06](assets/fr/06.webp)



Um Jami auf **Ubuntu** zu installieren, kannst du wie folgt vorgehen:



```shell
sudo apt install gnupg dirmngr ca-certificates curl --no-install-recommends
```



Dieser Befehl installiert die Werkzeuge zur Verwaltung von GPG-Schlüsseln (gnupg und dirmngr), SSL-Zertifikaten (ca-certificates) und das Download-Tool curl.



https://planb.network/tutorials/computer-security/operating-system/ubuntu-78a3be56-5d51-4ec3-8629-0dd27c352ab5

```shell
curl -s https://dl.jami.net/public-key.gpg | sudo tee /usr/share/keyrings/jami-archive-keyring.gpg > /dev/null
```



Hier wird curl verwendet, um den öffentlichen GPG-Schlüssel von Jami herunterzuladen. Die Umleitung nach /dev/null verhindert, dass der rohe Schlüssel auf dem Bildschirm angezeigt wird.



```shell
sudo sh -c "echo 'deb [signed-by=/usr/share/keyrings/jami-archive-keyring.gpg] https://dl.jami.net/stable/ubuntu_25.04/ jami main' > /etc/apt/sources.list.d/jami.list"
```



Dieser Befehl fügt das offizielle Jami-Repository in die Liste der APT-Quellen ein.



```shell
sudo apt-get update && sudo apt-get install jami
```



Zum Schluss aktualisieren Sie die Liste der verfügbaren Pakete mit `apt-get update`, dann installieren Sie Jami direkt aus dem offiziellen Repository mit `apt-get install jami`.



## Jami Grundkonfiguration



Sobald Jami auf Ihrem System installiert ist, können Sie es direkt über das Systemmenü starten.



Nach dem Start der Anwendung haben Sie die Möglichkeit, ein Konto zu erstellen oder mit einem bereits erstellten Konto fortzufahren.



![0_07](assets/fr/07.webp)



### Ein Konto erstellen



Die Einrichtung eines Jami-Kontos ist relativ einfach. Du brauchst weder eine E-Mail Address noch eine Telefonnummer: Jami sammelt nur das Minimum an Informationen. Wenn Sie möchten, können Sie einen Benutzernamen (Alias) registrieren, der auf Ihre *Jami-ID* (kryptographischer Fingerabdruck) verweist. Die Verknüpfung *Alias ↔ Jami ID* wird auf einem Standard-Namensserver veröffentlicht (austauschbar / selbsthostend), so dass der Alias nicht unbedingt erforderlich ist.



![0_08](assets/fr/08.webp)



Um Ihre Daten lokal zu schützen, können Sie ein Kennwort festlegen, um Ihr Profil und Ihre Sicherungen auf dem Gerät zu verschlüsseln. Dieses Passwort ist optional und hat keinen Einfluss auf die Ende-zu-Ende-Verschlüsselung der Kommunikation, die standardmäßig aktiviert ist. Wenn Sie diesen lokalen Schutz aktivieren, wählen Sie ein langes, zufälliges und eindeutiges Kennwort und bestätigen Sie es dann.



![0_09](assets/fr/09.webp)



Nachdem Sie Ihr Konto verschlüsselt haben, legen Sie Ihren vollständigen Namen fest.



![0_10](assets/fr/10.webp)



![0_11](assets/fr/11.webp)



## Anmeldung bei einem bestehenden Konto



Jami verwendet keine **Identifikatoren** und hat keine Datenbank, um sich mit deinem Konto zu verbinden. Alle deine Daten werden direkt auf deinem Gerät gespeichert. Um sich mit deinem alten Konto zu verbinden, musst du ein **Backup** deines alten Kontos durchführen.



Gehen Sie zu **Einstellungen**, dann **Konto**, dann **Kontoverwaltung**. Scrollen Sie bis zum Ende und führen Sie ein **Backup Ihres Kontos** durch. Wählen Sie den Speicherort für die Sicherungsdatei, geben Sie das **Passwort** ein, das bei der Erstellung Ihres Kontos festgelegt wurde, und bestätigen Sie es.



![0_12](assets/fr/12.webp)



Mit dieser Sicherungsdatei können Sie sich wieder mit Ihrem Konto verbinden.



![0_13](assets/fr/13.webp)



### Verbindung über Archiv



Klicken Sie dazu auf die Schaltfläche **Importieren aus einem Sicherungsarchiv** und wählen Sie die Sicherungsdatei für Ihr Jami Konto aus.



![0_14](assets/fr/14.webp)



Wenn Sie bei der Einrichtung Ihres Kontos ein Passwort festgelegt haben, geben Sie es ein und bestätigen Sie es. Ihre Daten und Nachrichten werden automatisch angezeigt.



![0_15](assets/fr/15.webp)



Es ist wichtig, regelmäßig Sicherungskopien Ihres Kontos zu erstellen, um Ihre Daten auf dem neuesten Stand zu halten und sicherzustellen, dass Sie Ihre letzten Nachrichten nicht verlieren.



### Verbindung über ein Gerät



Wenn Sie bereits auf einem anderen Gerät mit Ihrem Jami-Konto eingeloggt sind, gehen Sie zu **Konto**, dann zum Abschnitt **Verbundenes Gerät** und klicken Sie auf **Neues Gerät verbinden**. Der QR-Code-Reader erscheint zusammen mit einem Feld zur Eingabe eines Verbindungscodes.



![0_16](assets/fr/16.webp)



Starten Sie Jami auf dem neuen Gerät und wählen Sie **Ich habe bereits ein Konto** und dann **Importieren von einem anderen Gerät**. Sie erhalten dann einen **QR-Code** und einen Anmeldecode. Um eine Verbindung herzustellen, können Sie den QR-Code-Scanner verwenden, um den auf dem anderen Gerät angezeigten Code zu scannen. Oft ist es jedoch einfacher, den Verbindungscode zu kopieren und direkt in das dafür vorgesehene Feld einzufügen.



![0_17](assets/fr/17.webp)



Wenn Sie bei der Erstellung Ihres Kontos ein Passwort festgelegt haben, müssen Sie es auf dem neuen Gerät eingeben, um fortzufahren.



![417](assets/fr/18.webp)



Bestätigen Sie die Verbindung auf dem bereits angeschlossenen Gerät, um fortzufahren.



![0_19](assets/fr/19.webp)



Nach Eingabe des Kennworts wird das Gerät automatisch mit dem Konto verbunden und synchronisiert. Sie können dann von jedem Ihrer Geräte aus Nachrichten senden und empfangen.



## Eine Erweiterung zu Jami hinzufügen



Eines der interessanten Merkmale von Jami ist die Möglichkeit, neue Optionen über Erweiterungen (Plugins) zu integrieren. Plugins sind native Module (C/C++); das SDK bietet Werkzeuge und Skripte (vor allem in Python), um sie zu erstellen. Einige Plugins sind direkt verfügbar [hier](https://jami.net/extensions/).



Um eine Erweiterung zu installieren, öffnen Sie auf dem Desktop den integrierten Erweiterungsspeicher, laden Sie das entsprechende Plugin herunter, gehen Sie dann zu **Einstellungen → Erweiterungen → Installieren** und aktivieren Sie es. Auf Android ist der Store nicht integriert: Laden Sie die `.jpl'-Datei herunter und importieren Sie sie manuell unter **Einstellungen → Erweiterungen → Installieren**; der Import erfolgt automatisch und Sie können sie dann aktivieren und ihre Einstellungen anpassen, falls erforderlich.



![0_20](assets/fr/20.webp)



Um Ihre eigenen Plugins zu entwickeln, lesen Sie den Blogbeitrag **[Entdecken Sie das Jami Plugin SDK und erstellen Sie Ihre eigenen Plugins](https://jami.net/plugins-sdk/)**.



## Erweiterte Funktionen



Jami bietet auch **erweiterte Funktionen** für Benutzer, die bei der Konfiguration und Verwendung der Anwendung weiter gehen wollen. Diese Optionen umfassen:





- Einen Rendezvous-Punkt** erstellen: Mit dieser Funktion können Sie einen **Rendezvouspunkt** für Ihre Kommunikation einrichten, der für die Organisation sicherer Sitzungen oder den Austausch zwischen mehreren Benutzern nützlich ist.
- Verbindung mit einem Jami-Server**: Sie können Jami mit einem **Jami-Server** verbinden, was die Leistung oder Verfügbarkeit der Kommunikation verbessern kann, insbesondere in professionellen Umgebungen.
- Einrichten eines SIP-Kontos**: Sie können einen **SIP-Account** (Session Initiation Protocol) einrichten, der es Ihnen ermöglicht, Jami in bestehende Telefonanlagen zu integrieren oder zu telefonieren.



![0_21](assets/fr/21.webp)



Diese Optionen sind für fortgeschrittene Benutzer gedacht, die ihre Jami-Erfahrung anpassen und zusätzliche Funktionen für spezielle Bedürfnisse nutzen möchten.



Kurz gesagt, Jami ist eine komplette, sichere und flexible Kommunikationslösung, die sowohl für den persönlichen Gebrauch als auch für spezielle Bedürfnisse dank seiner **Erweiterungen** geeignet ist. Die einfache Installation, die intuitive Bedienung und die fortschrittlichen Funktionen wie **Verschlüsselung**, **Synchronisation mit mehreren Geräten** und die Möglichkeit, **eigene Plugins** zu erstellen, machen es zu einem leistungsstarken Werkzeug für die Verwaltung Ihrer Kommunikation unter Wahrung Ihrer **Privatsphäre**.
