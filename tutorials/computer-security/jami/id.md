Jami adalah aplikasi pesan peer-to-peer gratis dan open-source dengan sejarah panjang dan beragam fitur. Sejarahnya dimulai pada Desember 2004, ketika perusahaan yang berbasis di Montreal, _Savoir-faire Linux_, meluncurkan SFLPhone, sebuah proyek telepon digital untuk bisnis yang sepenuhnya berbasis pada teknologi open-source. Perangkat lunak ini, yang mematuhi standar telekomunikasi (SIP dan IAX), menonjol karena kemampuannya menangani sejumlah besar jalur dan panggilan.

Pada tahun 2015, SFLPhone berganti nama menjadi Ring, dan mengintegrasikan arsitektur terdistribusi yang tidak lagi memerlukan server pusat. Tahun berikutnya, Ring secara resmi bergabung dengan proyek GNU, memperkuat pijakannya di ekosistem free software . Akhirnya, pada Desember 2018, untuk menghindari kebingungan dengan produk komersial yang menggunakan istilah bahasa Inggris "_Ring_", perangkat lunak ini mengadopsi nama saat ini: Jami. Sejak itu, ia terus berkembang sebagai platform komunikasi yang bebas, terdesentralisasi, dan ramah privasi.

Saat ini, Jami tersedia di banyak sistem. Ini dikenal karena performa, kelancaran, dan kemudahan penggunaannya. Aplikasi ini memungkinkan Anda berkomunikasi melalui pesan instan, panggilan audio, atau panggilan video, sambil memastikan kerahasiaan percakapan Anda melalui enkripsi end-to-end. Instalasi yang sederhana dan sejumlah besar fitur menjadikannya aplikasi komunikasi yang lengkap, mudah, dan nyaman untuk digunakan setiap hari.

*E2EE = Enkripsi end-to-end*

## Mengapa menggunakan Jami?

- Aplikasi ini **open source dan sepenuhnya gratis**, jadi Anda dapat menggunakannya tanpa biaya.
- **Lengkap dengan fitur-fitur yang berguna**: perangkat lunak ini memungkinkan Anda memanfaatkan banyak opsi, berkat kemampuan untuk menambahkan plugin dengan mudah dari toko aplikasi. Bahkan dimungkinkan untuk membuat ekstensi Anda sendiri untuk fitur yang lebih sesuai dengan kebutuhan Anda.
- **Mudah digunakan dan Interface intuitif**: terlepas dari banyak fitur yang ditawarkan Jami, ini masih sangat mudah untuk dipahami.
- **Keamanan yang kuat**: Jami menggabungkan algoritma enkripsi canggih yang menjamin keamanan komunikasi Anda sambil menghormati privasi Anda.
- **Sangat mudah diakses dan cepat**: Jami menawarkan komunikasi yang mudah bahkan ketika bandwidth terbatas, untuk kemudahan penggunaan yang ditingkatkan.

## Bagaimana cara menginstal Jami?

Sebelum beralih ke pemasangan Jami, penting untuk dicatat bahwa aplikasi ini tersedia di berbagai sistem operasi. Oleh karena itu, penginstalannya akan bergantung pada sistem yang Anda gunakan.

![0_01](assets/fr/01.webp)

### Untuk pengguna Android atau iOS

Aplikasi ini tersedia langsung dari App Store atau Play Store. Cukup mencarinya di kolom pencarian, lalu luncurkan penginstalan.

### Untuk pengguna Windows atau macOS

Untuk memasang Jami di perangkat Anda, pertama-tama Anda perlu mengunjungi situs web resmi Jami. Secara default, situs ini menampilkan perangkat lunak yang sesuai dengan sistem operasi perangkat Anda, dan Anda dapat mengklik langsung tombol unduh untuk meluncurkannya. Namun, Anda juga dapat mengunduh file yang dapat dieksekusi Windows langsung dari [halaman unduhan](https://jami.net/download-jami-windows/).

![0_02](assets/fr/02.webp)

Untuk pengguna macOS, file ini juga tersedia dari [halaman unduhan macOS](https://jami.net/download-jami-macos/).

![0_03](assets/fr/03.webp)

Setelah Anda mengunduh file yang dapat dieksekusi, mulailah proses instalasi dengan mengklik dua kali. Setujui persyaratan penggunaan, lalu mulai penginstalan dan tunggu hingga selesai.

![0_04](assets/fr/04.webp)

![0_05](assets/fr/05.webp)

### Untuk pengguna linux

Untuk menginstal Jami di Linux, yang terbaik adalah menggunakan command line. Penting untuk dicatat bahwa Jami tersedia untuk berbagai distribusi Linux. Sebelum Anda mulai menginstal Jami, pastikan Anda memilih distribusi yang tepat untuk sistem Anda.

Setelah Anda memilih distribusi, Anda dapat menginstal sistem. Anda perlu menginstal dependensi yang diperlukan untuk menjalankan Jami pada OS Linux Anda. Perintah-perintahnya tersedia secara langsung di [halaman ini](https://jami.net/download-jami-linux/).

![0_06](assets/fr/06.webp)

Untuk menginstal Jami di **Ubuntu**, Anda dapat melakukannya sebagai berikut:

```shell
sudo apt install gnupg dirmngr ca-certificates curl --no-install-recommends
```

Perintah ini menginstal aplikasi yang diperlukan untuk mengelola kunci GPG (gnupg dan dirmngr), sertifikat SSL (ca-certificates), dan alat unduh curl.

https://planb.network/tutorials/computer-security/operating-system/ubuntu-78a3be56-5d51-4ec3-8629-0dd27c352ab5

```shell
curl -s https://dl.jami.net/public-key.gpg | sudo tee /usr/share/keyrings/jami-archive-keyring.gpg > /dev/null
```

Di sini, curl digunakan untuk mengunduh kunci publik GPG Jami. Pengalihan ke /dev/null menghindari menampilkan kunci mentah pada layar.

```shell
sudo sh -c "echo 'deb [signed-by=/usr/share/keyrings/jami-archive-keyring.gpg] https://dl.jami.net/stable/ubuntu_25.04/ jami main' > /etc/apt/sources.list.d/jami.list"
```

Perintah ini menambahkan repositori Jami resmi ke daftar sumber APT.

```shell
sudo apt-get update && sudo apt-get install jami
```

Terakhir, perbarui daftar paket yang tersedia dengan `apt-get update`, lalu instal Jami langsung dari repositori resmi dengan `apt-get install jami`.

## Konfigurasi dasar Jami

Setelah Jami terinstal di sistem Anda, Anda dapat menjalankan langsung dari menu sistem.

Setelah Anda memulai aplikasi, Anda akan memiliki opsi untuk membuat akun atau melanjutkan dengan akun yang sudah Anda buat.

![0_07](assets/fr/07.webp)

### Membuat akun

Membuat akun Jami relatif sederhana. Anda tidak memerlukan alamat e-mail atau nomor telepon: Jami hanya mengumpulkan informasi minimum. Jika Anda mau, Anda dapat mendaftarkan nama pengguna (alias) yang menunjuk ke _Jami ID_ Anda (cryptographic fingerprint). Asosiasi _alias_ ↔ Jami ID dipublikasikan di name server default (dapat diganti/dihosting sendiri), sehingga alias tidak selalu wajib.

![0_08](assets/fr/08.webp)

Untuk melindungi data Anda secara lokal, Anda dapat menetapkan kata sandi untuk mengenkripsi profil dan cadangan Anda di perangkat. Kata sandi ini bersifat opsional dan tidak memengaruhi enkripsi end-to-end komunikasi, yang aktif secara default. Jika Anda mengaktifkan perlindungan lokal ini, pilih kata sandi yang panjang, acak, dan unik, lalu validasi.

![0_09](assets/fr/09.webp)

Setelah mengenkripsi akun Anda, tetapkan nama lengkap Anda.

![0_10](assets/fr/10.webp)

![0_11](assets/fr/11.webp)

## Masuk ke akun yang sudah ada

Jami tidak menggunakan **identifier** dan tidak memiliki database untuk menghubungkan ke akun Anda. Semua data Anda disimpan langsung di perangkat Anda. Agar dapat terhubung ke akun lama Anda, Anda perlu melakukan cadangan dari akun lama Anda.

Buka **Settings**, lalu **Account**, kemudian **Account Management**. Geser ke bawah dan lakukan **pencadangan akun Anda**. Pilih lokasi di mana file cadangan akan disimpan, masukkan **kata sandi** yang ditentukan saat akun Anda dibuat, lalu validasi.

![0_12](assets/fr/12.webp)

File cadangan ini akan memungkinkan Anda untuk terhubung kembali ke akun Anda.

![0_13](assets/fr/13.webp)

### Koneksi melalui arsip

Untuk melakukan ini, klik tombol **Import from a backup archive**, dan pilih file cadangan untuk akun Jami Anda.

![0_14](assets/fr/14.webp)

Jika Anda menetapkan kata sandi saat membuat akun Anda, masukkan dan konfirmasi. Data dan pesan Anda akan muncul secara otomatis.

![0_15](assets/fr/15.webp)

Penting untuk membuat cadangan akun Anda secara teratur agar data Anda tetap mutakhir dan memastikan Anda tidak kehilangan pesan-pesan terbaru.

### Koneksi melalui perangkat

Jika Anda sudah masuk di perangkat lain dengan akun Jami Anda, buka **Account** lalu ke bagian **Connected device** dan klik **Connect a new device**. Pembaca kode QR akan muncul, bersama dengan kolom untuk memasukkan kode koneksi.

![0_16](assets/fr/16.webp)

Luncurkan Jami di perangkat baru dan pilih **I already have an account** lalu **Import from another device**. Anda kemudian akan mendapatkan QR code dan kode login. Untuk terhubung, Anda dapat menggunakan pemindai kode QR untuk memindai kode yang ditampilkan di perangkat lain. Namun, seringkali lebih sederhana untuk menyalin kode koneksi dan menempelkannya langsung ke kolom yang tersedia.

![0_17](assets/fr/17.webp)

Jika Anda menetapkan kata sandi saat membuat akun, Anda harus memasukkannya pada perangkat baru untuk melanjutkan.

![417](assets/fr/18.webp)

Konfirmasikan koneksi pada perangkat yang sudah tersambung untuk melanjutkan.

![0_19](assets/fr/19.webp)

Setelah kata sandi dimasukkan, perangkat akan secara otomatis terhubung ke akun dan melakukan sinkronisasi. Anda kemudian dapat mengirim dan menerima pesan dari perangkat mana pun.

## Menambahkan ekstensi ke Jami

Salah satu fitur menarik Jami adalah kemampuannya untuk mengintegrasikan opsi baru melalui ekstensi (plugins). Plugins adalah modul native (C/C++); SDK menyediakan alat dan script (terutama dalam Python) untuk membantu pembuatannya. Beberapa plugins tersedia langsung [di sini](https://jami.net/extensions/).

Untuk memasang ekstensi, pada Desktop buka toko ekstensi terintegrasi, unduh plugin yang sesuai, lalu buka **Settings → Extensions → Install** dan aktifkan. Pada Android, toko tidak terintegrasi: unduh file `.jpl`, lalu impor secara manual dari **Settings → Extensions → Install**; impor akan otomatis dan Anda kemudian dapat mengaktifkannya serta menyesuaikan pengaturannya jika perlu.

![0_20](assets/fr/20.webp)

Untuk mengembangkan plugins Anda sendiri, lihat postingan blog **[Discover the Jami Plugin SDK and create your own plugins](https://jami.net/plugins-sdk/)**.

## Fitur lanjutan


Jami juga menawarkan **fitur tingkat lanjut** bagi pengguna yang ingin melangkah lebih jauh dalam mengonfigurasi dan menggunakan aplikasi. Opsi-opsi ini meliputi:

- **Membuat rendezvous point**: Fitur ini memungkinkan Anda membuat titik temu (rendezvous point) untuk komunikasi Anda, berguna untuk mengatur sesi atau pertukaran yang aman antara beberapa pengguna.
- **Terhubung ke server Jami**: Anda dapat menghubungkan Jami ke server Jami, yang dapat meningkatkan kinerja atau ketersediaan komunikasi, terutama di lingkungan profesional.
- **Menyiapkan akun SIP**: Anda dapat menyiapkan akun **SIP** (Session Initiation Protocol), yang memungkinkan Anda mengintegrasikan Jami dengan sistem telepon yang ada atau melakukan panggilan telepon.

Opsi-opsi ini dirancang untuk pengguna yang lebih mahir yang ingin menyesuaikan pengalaman Jami mereka dan memanfaatkan fitur-fitur tambahan untuk kebutuhan tertentu.

![0_21](assets/fr/21.webp)

Singkatnya, Jami adalah solusi komunikasi yang lengkap, aman, dan fleksibel, cocok untuk penggunaan pribadi maupun kebutuhan yang lebih spesifik berkat ekstensinya. Instalasi yang sederhana, penanganan yang intuitif, dan fitur-fitur canggih seperti **enkripsi**, **sinkronisasi multi-perangkat**, dan kemampuan untuk **membuat plugins Anda sendiri** menjadikannya alat yang kuat untuk mengelola komunikasi Anda sambil menjaga **privasi** Anda.
