Jami 是一款历史悠久、功能丰富的免费开源点对点消息程序。它的历史可以追溯到 2004 年 12 月，当时总部位于蒙特利尔的*Savoir-faire Linux* 推出了 SFLPhone，这是一个完全基于开源技术的企业数字电话项目。该软件符合电信标准（SIP 和 IAX），因其能够处理大量线路和呼叫而脱颖而出。



2015 年，SFLPhone 更名为 Ring，并集成了分布式架构，不再需要中央服务器。次年，Ring 正式加入 GNU 项目，巩固了其在自由软件生态系统中的地位。最后，在 2018 年 12 月，为了避免与使用英文术语 "*Ring*"的商业产品混淆，该软件采用了现在的名称：Jami。从那时起，它作为一个自由、去中心化和隐私友好的通信平台不断发展。



如今，Jami 可在许多系统上使用。它以性能、流畅性和易用性而闻名。它可让您通过即时消息、音频通话或视频通话进行通信，同时通过端到端加密确保通话保密。简单的安装和丰富的功能使其成为一个完整的通信应用程序，日常使用简单方便。

## 为什么使用 Jami？





- 它是开源的，完全免费**，因此您可以免费使用。
- 功能齐全**：由于可以从商店轻松添加插件，这款软件可以让您利用众多选项。您甚至可以创建自己的扩展程序，以获得更适合您需求的功能。
- 易于使用，Interface 直观**：尽管 Jami 提供了许多功能，但它仍然非常容易上手。
- 强大的安全性**：Jami 采用先进的加密算法，在保证通信安全的同时尊重您的隐私。
- 高度方便快捷**：即使在带宽有限的情况下，它也能提供便捷的通信，从而提高用户友好性。



## 如何安装 Jami？



在继续安装 Jami 之前，需要注意的是，该应用程序适用于不同的操作系统。因此，它的安装将取决于你使用的系统。



![0_01](assets/fr/01.webp)



### 适用于安卓或 iOS 用户



该应用程序可直接从 App Store 或 Play Store 下载。只需在搜索栏中搜索，然后启动安装即可。



### 适用于 Windows 或 macOS 用户



要在设备上安装 Jami，首先需要访问 Jami 官方网站。默认情况下，网站会显示与你的设备操作系统相对应的软件，你可以直接点击下载按钮启动它。不过，你也可以直接从 [下载页面](https://jami.net/download-jami-windows/) 下载 Windows 可执行文件。



![0_02](assets/fr/02.webp)



对于 macOS 用户，该文件也可从 [macOS 下载页面](https://jami.net/download-jami-macos/) 获取。



![0_03](assets/fr/03.webp)



下载可执行文件后，双击该文件启动安装程序。接受使用条款，然后开始安装并等待完成。



![0_04](assets/fr/04.webp)



![0_05](assets/fr/05.webp)



### 针对 Linux 用户



要在 Linux 上安装 Jami，最好使用命令行。需要注意的是，Jami 适用于不同的 Linux 发行版。在开始安装 Jami 之前，请确保为你的系统选择了正确的发行版。



选定发行版后，就可以安装系统了。你需要安装在 Linux 操作系统上运行 Jami 所需的依赖项。这些命令可直接在 [this page](https://jami.net/download-jami-linux/) 上找到。



![0_06](assets/fr/06.webp)



在**Ubuntu**上安装Jami的步骤如下：



```shell
sudo apt install gnupg dirmngr ca-certificates curl --no-install-recommends
```



该命令安装管理 GPG 密钥（gnupg 和 dirmngr）、SSL 证书（ca-certificates）和 curl 下载工具所需的工具。



https://planb.network/tutorials/computer-security/operating-system/ubuntu-78a3be56-5d51-4ec3-8629-0dd27c352ab5

```shell
curl -s https://dl.jami.net/public-key.gpg | sudo tee /usr/share/keyrings/jami-archive-keyring.gpg > /dev/null
```



这里使用 curl 下载 Jami 的 GPG 公钥。重定向到 /dev/null 可以避免在屏幕上显示原始密钥。



```shell
sudo sh -c "echo 'deb [signed-by=/usr/share/keyrings/jami-archive-keyring.gpg] https://dl.jami.net/stable/ubuntu_25.04/ jami main' > /etc/apt/sources.list.d/jami.list"
```



此命令将 Jami 官方软件源添加到 APT 源列表中。



```shell
sudo apt-get update && sudo apt-get install jami
```



最后，使用 "apt-get update "更新可用软件包列表，然后使用 "apt-get install jami "直接从官方软件仓库安装 Jami。



## 雅米基本配置



Jami 安装到系统后，可以直接从系统菜单启动。



开始申请后，您可以选择创建账户或继续使用已创建的账户。



![0_07](assets/fr/07.webp)



### 创建账户



创建 Jami 帐户相对简单。您不需要电子邮件 Address 或电话号码：Jami 只收集最基本的信息。如果你愿意，可以注册一个用户名（别名），指向你的*Jami ID*（加密指纹）。别名 ↔ Jami ID* 关联信息发布在默认名称服务器上（可替换/自托管），因此别名并不一定是强制性的。



![0_08](assets/fr/08.webp)



为保护本地数据，您可以设置密码来加密设备上的配置文件和备份。此密码为可选项，对端到端通信加密没有影响，默认情况下已激活。如果激活本地保护，请选择一个长的、随机的、唯一的密码，然后进行验证。



![0_09](assets/fr/09.webp)



加密账户后，设置您的全名。



![0_10](assets/fr/10.webp)



![0_11](assets/fr/11.webp)



## 登录现有账户



Jami 不使用**标识符**，也没有连接到您账户的数据库。您的所有数据都直接存储在您的设备上。要连接到您的旧账户，您需要对旧账户进行**备份。



转到**设置**，然后转到**账户**，再转到**账户管理**。滚动到底部，执行**账户备份**。选择保存备份文件的位置，输入创建账户时定义的**密码**，然后验证。



![0_12](assets/fr/12.webp)



有了这个备份文件，您就可以重新连接账户。



![0_13](assets/fr/13.webp)



### 通过档案连接



为此，请单击**从备份存档导入**按钮，然后选择 Jami 账户的备份文件。



![0_14](assets/fr/14.webp)



如果您在创建账户时设置了密码，请输入并确认。您的数据和信息将自动显示。



![0_15](assets/fr/15.webp)



定期备份账户非常重要，这样可以保持数据最新，确保不会丢失最新信息。



### 通过设备连接



如果你已经用 Jami 帐户登录了其他设备，请进入**帐户**，然后进入**连接设备**部分，点击**连接新设备**。这时会出现二维码阅读器，并有一个输入连接码的区域。



![0_16](assets/fr/16.webp)



在新设备上启动 Jami，选择**我已经有一个账户**，然后选择**从其他设备导入**。然后，您会收到一个**二维码**和一个登录代码。要进行连接，您可以使用 QR 码扫描仪扫描另一台设备上显示的代码。不过，复制连接代码并直接粘贴到提供的字段中通常会更简单。



![0_17](assets/fr/17.webp)



如果您在创建账户时设置了密码，则需要在新设备上输入密码才能继续。



![417](assets/fr/18.webp)



在已连接的设备上确认连接，继续。



![0_19](assets/fr/19.webp)



输入密码后，设备将自动连接到账户并进行同步。然后，您就可以在任何设备上收发信息了。



## 为 Jami 添加扩展名



Jami 的一个有趣功能是它能通过扩展（插件）集成新的选项。插件是本地模块（C/C++）；SDK 提供了帮助创建插件的工具和脚本（主要是 Python）。有些插件可以直接获取 [此处](https://jami.net/extensions/)。



要安装扩展，请在桌面上打开集成的扩展商店，下载相应插件，然后转到**设置 → 扩展 → 安装**并激活。在安卓系统上，商店未集成：下载`.jpl`文件，然后从**设置 → 扩展工具 → 安装**手动导入；导入是自动的，然后可以激活它，必要时调整其设置。



![0_20](assets/fr/20.webp)



要开发自己的插件，请查看博文**[探索 Jami 插件 SDK 并创建自己的插件](https://jami.net/plugins-sdk/)**。



## 高级功能



Jami 还为希望进一步配置和使用应用程序的用户提供**高级功能**。这些选项包括





- 创建会合点**：该功能可让您为通信创建一个**会合点**，这对组织多个用户之间的安全会话或交流非常有用。
- 连接到 Jami 服务器**：您可以将 Jami 连接到**Jami 服务器**，这可以提高通信性能或可用性，尤其是在专业环境中。
- 设置 SIP 帐户**：您可以设置一个**SIP 帐户**（会话启动协议），以便将 Jami 与现有电话系统集成或拨打电话。



![0_21](assets/fr/21.webp)



这些选项专为希望定制 Jami 体验并利用附加功能满足特定需求的高级用户设计。
