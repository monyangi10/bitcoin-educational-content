## I.プレゼンテーション



このチュートリアルでは、OPNsense オープンソースファイアウォールを取り上げます。この FreeBSD ベースのソリューションの主な機能、 前提条件、インストール方法について見ていきます。



始める前に知っておくべきことは、**OPNsense と pfSense はどちらも FreeBSD ベースのオープンソース・ファイアウォール**だということだ。pfSense は OPNsense の兄のようなもので、後者は 2015 年に作成された Fork です。最後に、2017年以降、**OPNsenseはFreeBSD**の代わりにHardenedBSDに切り替えたことを指摘しておきたい。HardenedBSD は FreeBSD の強化版で、高度なセキュリティ機能を備えています。



OPNsense は、よりモダンなユーザー Interface と **より頻繁な更新頻度** で際立っています。実際、OPNsense のアップデートスケジュールには年に 2 回のメジャーリリースが含まれており、2 週間ごとにアップデートされる（結果的にマイナーリリースになる）。これらのソリューションのコミュニティ版を見ると、このフォローアップは pfSense と比較して非常に興味深い。



![Image](assets/fr/050.webp)



## II.OPNsenseの特徴



OPNsenseは、ファイアウォールとルーターとして機能するように設計されたオペレーティング・システムであるが、その機能は多数あり、追加パッケージをインストールすることで拡張することができる。本番環境での使用に適しており、主にネットワーク・セキュリティとフロー管理に使用される。



### A.主な特徴



OPNsenseの主な機能を紹介しよう：





- ファイアウォールと NAT**：OPNsense は、ステートフル・フィルタリングによる高度なステートフル・ファイアウォール機能と、ネットワーク Address トランスレーション（NAT）機能を提供します。





- DNS/DHCP**：OPNsense は、ネットワーク上の DNS と DHCP サービスを管理するように設定できます。DHCPサーバーとして機能するだけでなく、ローカルネットワーク上のマシンのDNSリゾルバとしても使用できます。Dnsmasqもデフォルトで統合されています。





- VPN**：OPNsenseは、IPsec、OpenVPN、WireGuardを含む複数のVPNプロトコルをサポートしており、モバイルワークステーションへのリモートアクセスやサイト間接続のためのセキュアな接続を可能にします。





- ウェブ・プロキシ**：OPNsenseには、インターネットアクセスを制御およびフィルタリングするためのWebプロキシが含まれています。また、コンテンツのフィルタリングやネットワークアクセスの管理にも使用できます。





- 帯域幅管理 (QoS)**：OPNsenseは、ネットワーク・トラフィックを優先し、ネットワーク帯域幅をよりよく管理するためのQoS（Quality of Service）管理機能を提供します。





- キャプティブポータル**：この機能により、認証ページ（ローカルベース、バウチャーなど）を介してネットワークへのユーザーアクセスを管理できます。公衆Wi-Fiネットワークによく導入されている機能です。





- IDS/IPS**：OPNsenseはSuricataを統合し、攻撃からネットワークを保護する侵入検知・防御（IDS/IPS）機能を提供します。





- 高可用性 (CARP)**：OPNsense は、複数の OPNsense ファイアウォール間の高可用性を実現する CARP (*Common Address Redundancy Protocol*) をサポートしており、ハードウェアの障害時にもサービスが有効であることを保証します。





- レポートとモニタリング**：OPNsense は、ログを作成することにより、ネットワーク・パフォーマンスを追跡し（NetFlow を使用）、潜在的な問題を検出するためのリアルタイムのレポートおよびモニタリング・ツールを提供します。これにはグラフィックも含まれます。MonitツールはOPNsenseに統合されており、ファイアウォール自体の監視を可能にします。



### B.追加パッケージ



これはOPNsenseが提供する機能の概要に過ぎません。さらに、OPNsenseの管理Interfaceからアクセスできる**パッケージ・カタログ**では、ファイアウォールを**追加機能**で充実させることができます。これにはACMEクライアント、Wazuhエージェント、NTP Chronyサービス、リバースプロキシとしてのCaddyが含まれます。

## IV.結論



このガイドに従えば、OPNsense をインストールしてすぐに使い始めることができます。OPNsense は、ネットワーク・トラフィックを効率的に保護および管理するための幅広い機能を備えており、プロフェッショナルな環境での使用に適しています。
