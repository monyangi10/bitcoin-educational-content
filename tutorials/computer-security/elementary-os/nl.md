Elementary OS is een op Ubuntu gebaseerd besturingssysteem, ontworpen om eenvoudig, snel en stabiel te zijn voor veel alledaags gebruik. Het is een evenwichtig Linux-alternatief voor MacOS en Windows. De vloeiende, intuïtieve en overzichtelijke grafische Interface maakt het makkelijk te leren, zelfs voor beginners. Het richt zich ook op ergonomie, veiligheid en prestaties.



https://planb.network/tutorials/computer-security/operating-system/ubuntu-78a3be56-5d51-4ec3-8629-0dd27c352ab5

## Waarom kiezen voor Elementary OS





- Eenvoud en gebruiksgemak**: De grafische Interface van Elementary OS houdt het midden tussen die van MacO's en Windows. Deze vertrouwdheid maakt het makkelijk te gebruiken, zelfs voor onervaren gebruikers.





- Beveiliging**: Net als de meeste Linux-distributies profiteert Elementary OS van een hoog beveiligingsniveau. Regelmatige updates, rechtenbeheer en de afwezigheid van veelvoorkomende virussen maken het een betrouwbaar systeem.





- Snelheid**: Elementary OS is een lichtgewicht distributie. Het heeft weinig bronnen nodig, waardoor het snel is en geschikt voor computers met bescheiden configuraties.





- Gratis**: Het systeem is helemaal gratis. Als je het downloadt, kun je echter een donatie doen om de ontwikkelaars te steunen.





- Actieve gemeenschap**: De gemeenschap rond Elementary OS is divers en reageert snel. Als je tegen problemen aanloopt, kun je gemakkelijk hulp vinden op de forums of sociale netwerken.



## Installatie en configuratie


### Hardwarevereisten



Controleer voordat u met de installatie begint of u over de volgende apparatuur beschikt:





- Een **USB-sleutel** van minstens 12 GB
- RAM** geheugen van minstens 4 GB
- Een **Hard schijf van 20 GB** of meer voor comfortabel gebruik



## ISO-afbeelding downloaden



Ga naar de officiële website van het besturingssysteem [elementary] (https://elementary.io/) en kies een bedrag om het project te steunen. Deze stap is optioneel.


Als je het ISO-image gratis wilt downloaden, voer dan 0 in in het veld **"Other"** en start het downloaden van het ISO-image van het systeem.



![0_01](assets/fr/01.webp)



## Een opstartbare USB-sleutel maken



Zodra je het ISO-image hebt gedownload, moet je het opstartbaar maken op een USB-sleutel om verder te gaan met de installatie.



Download software zoals [Balena Etcher] (https://etcher.balena.io/) of een vergelijkbaar hulpprogramma en start de software.


Selecteer de eerder gedownloade **Elementary OS** ISO image en stel je USB-sleutel in als doel.



Klik op de knop **flash** om het proces te starten en wacht tot het proces is voltooid voordat u de USB-stekker verwijdert.



![0_02](assets/fr/02.webp)



## Toets opstarten en BIOS-toegang



Schakel de computer uit waarop je Elementary OS wilt installeren en sluit vervolgens de USB-stick aan.


Wanneer je computer opstart, ga dan naar het BIOS (`ESC`, `F9` of `F11` afhankelijk van het merk) en selecteer de USB-stick als opstartapparaat, druk dan op `Enter` om het opstartproces te starten.

## Systeem bijwerken



Voor langdurig gebruik is het belangrijk om het systeem bij te werken met de nieuwste patches.


### Optie 1: Bijwerken via Interface afbeeldingen



Ga naar **AppCenter** en klik op het updatepictogram in de rechterbovenhoek. Wacht tot de updates in de lijst staan en klik dan op **"Alles bijwerken"** om de installatie te starten.



![0_20](assets/fr/20.webp)



![0_21](assets/fr/21.webp)



### Optie 2: Bijwerken via terminal



Je kunt de update ook uitvoeren vanaf de opdrachtregel via je terminal: een aanbevolen optie vanwege de nauwkeurigheid.



```shell
sudo apt update
```



```shell
sudo apt full-upgrade
```



Voor je eerste updates heeft het besturingssysteem je gebruikerswachtwoord en bevestiging nodig om softwarepakketten bij te werken, zelfs in de Elementaire kernel.



## Werkomgeving configureren



Elementary OS bevat alleen de essentiële tools. Om je omgeving aan te passen aan je behoeften, vooral als je een ontwikkelaar bent, raden we aan om extra tools te installeren.





- Je kunt nuttige afhankelijkheden toevoegen met het volgende commando (aan te passen aan je behoeften):



```shell
sudo apt update && sudo apt install -y git python3 python3-pip build-essential wget curl zsh make snapd && sudo snap install code --classic
```



Dit commando installeert **Git**, **Python 3**, **pip**, **compiler tools**, **wget**, **curl**, **zsh**, **make**, **snapd** en **vscode** om een basis ontwikkelomgeving voor te bereiden.
