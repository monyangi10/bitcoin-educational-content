سیستم‌عامل Elementary OS یک سیستم‌عامل مبتنی بر اوبونتو است که برای استفاده‌های روزمره به صورت ساده، سریع و پایدار طراحی شده است. این سیستم‌عامل به عنوان یک جایگزین متعادل لینوکس برای MacOS و ویندوز محسوب می‌شود. رابط گرافیکی Interface آن که روان، شهودی و بدون شلوغی است، یادگیری را حتی برای مبتدیان آسان می‌کند. همچنین بر ارگونومی، امنیت و عملکرد تمرکز دارد.

## چرا Elementary OS را انتخاب کنیم؟





- سادگی و سهولت استفاده**: رابط گرافیکی Interface در سیستم‌عامل Elementary OS در میانه‌ی رابط‌های MacOs و Windows قرار دارد. این آشنایی باعث می‌شود که حتی کاربران بی‌تجربه نیز به راحتی آن را بپذیرند.





- امنیت**: مانند اکثر توزیع‌های لینوکس، Elementary OS از سطح بالایی از امنیت بهره‌مند است. به‌روزرسانی‌های منظم، مدیریت حقوق و عدم وجود ویروس‌های رایج، آن را به سیستمی قابل اعتماد تبدیل می‌کند.





- سرعت**: Elementary OS یک توزیع سبک وزن است. این سیستم به منابع کمی نیاز دارد، که آن را سریع و مناسب برای کامپیوترهایی با پیکربندی‌های متوسط می‌سازد.





- رایگان**: این سیستم کاملاً رایگان است. با این حال، هنگامی که آن را دانلود می‌کنید، می‌توانید برای حمایت از توسعه‌دهندگان، کمک مالی کنید.





- جامعه فعال**: جامعه پیرامون Elementary OS متنوع و پاسخگو است. اگر با مشکلی مواجه شدید، می‌توانید به راحتی در انجمن‌ها یا شبکه‌های اجتماعی کمک پیدا کنید.



## نصب و پیکربندی


### پیش‌نیازهای سخت‌افزاری



قبل از شروع نصب، لطفاً اطمینان حاصل کنید که تجهیزات زیر را دارید:





- یک **کلید USB** با حداقل ۱۲ گیگابایت
- حافظه RAM** حداقل 4 گیگابایت
- یک **دیسک Hard با ظرفیت 20 گیگابایت** یا بیشتر برای استفاده راحت



## دانلود تصویر ISO



به وب‌سایت رسمی سیستم‌عامل [elementary](https://elementary.io/) بروید و مبلغی را برای حمایت از پروژه انتخاب کنید. این مرحله اختیاری است.


اگر می‌خواهید تصویر ISO را به‌صورت رایگان دانلود کنید، عدد 0 را در قسمت **"Other "** وارد کرده و دانلود تصویر ISO سیستم را شروع کنید.



![0_01](assets/fr/01.webp)



## ایجاد یک کلید USB قابل بوت



پس از دانلود تصویر ISO، باید آن را روی یک کلید USB قابل بوت کنید تا بتوانید با نصب ادامه دهید.



نرم‌افزارهایی مانند [Balena Etcher](https://etcher.balena.io/) یا ابزار مشابهی را دانلود کرده و سپس نرم‌افزار را اجرا کنید.


تصویر ISO **Elementary OS** که قبلاً دانلود شده است را انتخاب کنید و کلید USB خود را به عنوان هدف تنظیم کنید.



روی دکمه **فلش** کلیک کنید تا فرآیند شروع شود و قبل از خارج کردن کلید USB منتظر بمانید تا فرآیند کامل شود.



![0_02](assets/fr/02.webp)



## کلید بوت و دسترسی به BIOS



رایانه‌ای را که می‌خواهید Elementary OS را روی آن نصب کنید خاموش کنید، سپس کلید USB را وصل کنید.


وقتی کامپیوتر شما روشن می‌شود، به BIOS دسترسی پیدا کنید (`ESC`، `F9` یا `F11` بسته به برند) و کلید USB را به عنوان دستگاه بوت انتخاب کنید، سپس `Enter` را فشار دهید تا فرآیند بوت آغاز شود.

## به‌روزرسانی سیستم



قبل از استفاده طولانی‌مدت، مهم است که سیستم را با آخرین وصله‌ها به‌روزرسانی کنید.


### گزینه 1: به‌روزرسانی از طریق گرافیک Interface



وارد **AppCenter** شوید و روی آیکون به‌روزرسانی در گوشه بالا سمت راست کلیک کنید. منتظر بمانید تا به‌روزرسانی‌ها فهرست شوند، سپس روی **"Update All "** کلیک کنید تا نصب آغاز شود.



![0_20](assets/fr/20.webp)



![0_21](assets/fr/21.webp)



### گزینه ۲: به‌روزرسانی از طریق ترمینال



شما همچنین می‌توانید به‌روزرسانی را از طریق خط فرمان در ترمینال خود انجام دهید: گزینه‌ای توصیه‌شده به دلیل دقت آن.



```shell
sudo apt update
```



```shell
sudo apt full-upgrade
```



برای اولین به‌روزرسانی‌ها، سیستم‌عامل نیاز به رمز عبور کاربر و تأیید شما برای به‌روزرسانی بسته‌های نرم‌افزاری دارد، حتی در هسته Elementary.



## پیکربندی محیط کار



سیستم‌عامل Elementary OS فقط ابزارهای ضروری را شامل می‌شود. برای تطبیق محیط خود با نیازهایتان، به‌ویژه اگر توسعه‌دهنده هستید، توصیه می‌کنیم ابزارهای اضافی نصب کنید.





- می‌توانید وابستگی‌های مفید را با فرمان زیر اضافه کنید (با توجه به نیازهای خود تنظیم کنید):



```shell
sudo apt update && sudo apt install -y git python3 python3-pip build-essential wget curl zsh make snapd && sudo snap install code --classic
```



این دستور **Git**، **Python 3**، **pip**، **ابزارهای کامپایلر**، **wget**، **curl**، **zsh**، **make**، **snapd** و **vscode** را نصب می‌کند تا یک محیط توسعه پایه را آماده کند.



سیستم‌عامل Elementary OS اکنون بر روی دستگاه شما در حال اجرا است. فلسفه‌ی سادگی، سبکی و ظرافت آن را به انتخابی عالی برای استفاده شخصی و حرفه‌ای تبدیل کرده است. شما یک سیستم پایدار، روان و بدون شلوغی دریافت می‌کنید که آماده است تا بر اساس ترجیحات شما سفارشی‌سازی شود. چه برای توسعه، استفاده اداری یا مرور روزانه، همه چیز برای ساخت یک محیط کاری کارآمد، شهودی و دلپذیر در جای خود قرار دارد. همچنین به آموزش ما درباره Fedora، یک توزیع لینوکس به همان اندازه ساده، قوی و ماژولار، نگاهی بیندازید.
