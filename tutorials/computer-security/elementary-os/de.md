Elementary OS ist ein auf Ubuntu basierendes Betriebssystem, das für viele alltägliche Anwendungen einfach, schnell und stabil ist. Es stellt eine ausgewogene Linux-Alternative zu MacOS und Windows dar. Seine flüssige, intuitive und übersichtliche grafische Interface macht es auch für Anfänger leicht zu erlernen. Es konzentriert sich auch auf Ergonomie, Sicherheit und Leistung.



https://planb.network/tutorials/computer-security/operating-system/ubuntu-78a3be56-5d51-4ec3-8629-0dd27c352ab5

## Warum Elementary OS wählen?





- Einfachheit und Benutzerfreundlichkeit**: Das grafische Interface von Elementary OS liegt in der Mitte zwischen dem von Mac OS und Windows. Diese Vertrautheit macht es auch für unerfahrene Benutzer leicht zu übernehmen.





- Sicherheit**: Wie die meisten Linux-Distributionen profitiert auch Elementary OS von einem hohen Sicherheitsniveau. Regelmäßige Updates, Rechteverwaltung und das Fehlen gängiger Viren machen es zu einem zuverlässigen System.





- Geschwindigkeit**: Elementary OS ist eine leichtgewichtige Distribution. Es benötigt nur wenige Ressourcen, wodurch es schnell und für Computer mit bescheidenen Konfigurationen geeignet ist.





- Kostenlos**: Das System ist völlig kostenlos. Wenn Sie es herunterladen, können Sie jedoch eine Spende machen, um die Entwickler zu unterstützen.





- Aktive Gemeinschaft**: Die Gemeinschaft rund um Elementary OS ist vielfältig und hilfsbereit. Wenn Sie auf Schwierigkeiten stoßen, können Sie in den Foren oder sozialen Netzwerken leicht Hilfe finden.



## Installation und Konfiguration


### Hardware-Voraussetzungen



Bevor Sie mit der Installation beginnen, vergewissern Sie sich bitte, dass Sie über die folgenden Geräte verfügen:





- Ein **USB-Stick** mit mindestens 12 GB
- RAM**-Speicher von mindestens 4 GB
- Eine **Hard-Festplatte mit 20 GB** oder mehr für eine komfortable Nutzung



## ISO-Abbild herunterladen



Besuchen Sie die offizielle Website des Betriebssystems [elementar] (https://elementary.io/) und wählen Sie einen Betrag zur Unterstützung des Projekts. Dieser Schritt ist freiwillig.


Wenn Sie das ISO-Abbild kostenlos herunterladen möchten, geben Sie in das Feld **"Andere "** eine 0 ein und beginnen Sie mit dem Herunterladen des ISO-Abbilds des Systems.



![0_01](assets/fr/01.webp)



## Erstellen eines bootfähigen USB-Sticks



Sobald Sie das ISO-Image heruntergeladen haben, müssen Sie es auf einem USB-Stick bootfähig machen, um mit der Installation fortfahren zu können.



Laden Sie eine Software wie [Balena Etcher] (https://etcher.balena.io/) oder ein ähnliches Programm herunter und starten Sie die Software.


Wählen Sie das zuvor heruntergeladene **Elementary OS** ISO-Image aus und legen Sie Ihren USB-Stick als Ziel fest.



Klicken Sie auf die Schaltfläche **flash**, um den Vorgang zu starten, und warten Sie, bis der Vorgang abgeschlossen ist, bevor Sie den USB-Stick entfernen.



![0_02](assets/fr/02.webp)



## Tastenbooten und BIOS-Zugriff



Schalten Sie den Computer aus, auf dem Sie Elementary OS installieren möchten, und schließen Sie dann den USB-Stick an.


Wenn Ihr Computer hochfährt, rufen Sie das BIOS auf (`ESC`, `F9` oder `F11`, je nach Marke) und wählen Sie den USB-Stick als Boot-Gerät aus, dann drücken Sie `Enter`, um den Boot-Prozess zu starten.

## Aktualisierung des Systems



Vor längerem Gebrauch ist es wichtig, das System mit den neuesten Patches zu aktualisieren.


### Option 1: Aktualisierung über Interface-Grafiken



Öffnen Sie das **AppCenter** und klicken Sie auf das Update-Symbol in der oberen rechten Ecke. Warten Sie, bis die Updates aufgelistet werden, und klicken Sie dann auf **"Alle aktualisieren "**, um die Installation zu starten.



![0_20](assets/fr/20.webp)



![0_21](assets/fr/21.webp)



### Option 2: Aktualisierung über das Terminal



Sie können die Aktualisierung auch von der Befehlszeile aus über Ihr Terminal durchführen: eine empfohlene Option wegen ihrer Genauigkeit.



```shell
sudo apt update
```



```shell
sudo apt full-upgrade
```



Bei den ersten Aktualisierungen verlangt das Betriebssystem Ihr Benutzerkennwort und eine Bestätigung, um Softwarepakete zu aktualisieren, selbst im elementaren Kernel.



## Konfiguration der Arbeitsumgebung



Elementary OS enthält nur die wichtigsten Werkzeuge. Um Ihre Umgebung an Ihre Bedürfnisse anzupassen, insbesondere wenn Sie Entwickler sind, empfehlen wir die Installation zusätzlicher Tools.





- Sie können nützliche Abhängigkeiten mit dem folgenden Befehl hinzufügen (der an Ihre Bedürfnisse angepasst werden muss):



```shell
sudo apt update && sudo apt install -y git python3 python3-pip build-essential wget curl zsh make snapd && sudo snap install code --classic
```



Dieser Befehl installiert **Git**, **Python 3**, **pip**, **Compiler-Tools**, **wget**, **curl**, **zsh**, **make**, **snapd** und **vscode**, um eine grundlegende Entwicklungsumgebung vorzubereiten.
