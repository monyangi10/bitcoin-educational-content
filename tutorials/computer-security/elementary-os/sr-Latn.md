Elementary OS je operativni sistem zasnovan na Ubuntu-u, dizajniran da bude jednostavan, brz i stabilan za mnoge svakodnevne upotrebe. Predstavlja uravnoteženu Linux alternativu za MacOS i Windows. Njegov fluidan, intuitivan i pregledan grafički Interface čini ga lakim za učenje, čak i za početnike. Takođe se fokusira na ergonomiju, sigurnost i performanse.

## Zašto izabrati Elementary OS





- Jednostavnost i lakoća korišćenja**: Grafički Interface operativnog sistema Elementary OS je negde između onih kod MacOs-a i Windows-a. Ova poznatost olakšava usvajanje, čak i za neiskusne korisnike.





- Bezbednost**: Kao većina Linux distribucija, Elementary OS ima visok nivo bezbednosti. Redovna ažuriranja, upravljanje pravima i odsustvo uobičajenih virusa čine ga pouzdanim sistemom.





- Brzina**: Elementary OS je lagana distribucija. Zahteva malo resursa, što je čini brzom i pogodnom za računare sa skromnim konfiguracijama.





- Besplatno**: Sistem je potpuno besplatan. Međutim, kada ga preuzmete, možete donirati kako biste podržali programere.





- Aktivna zajednica**: Zajednica oko Elementary OS-a je raznolika i odgovorna. Ako naiđete na poteškoće, lako možete pronaći pomoć na forumima ili društvenim mrežama.



## Instalacija i konfiguracija


### Hardverski preduslovi



Pre nego što započnete instalaciju, molimo vas da se uverite da imate sledeću opremu:





- USB ključ od najmanje 12 GB
- RAM** memorija od najmanje 4 GB
- **Hard disk od 20 GB** ili više za udobno korišćenje



## Preuzmi ISO sliku



Idite na zvaničnu veb stranicu operativnog sistema [elementary](https://elementary.io/) i izaberite iznos za podršku projektu. Ovaj korak je opcionalan.


Ako želite da preuzmete ISO sliku besplatno, unesite 0 u polje **"Ostalo "** i započnite preuzimanje ISO slike sistema.



![0_01](assets/fr/01.webp)



## Kreiranje USB ključa za pokretanje sistema



Kada preuzmete ISO sliku, potrebno je da je učinite bootabilnom na USB ključu kako biste nastavili sa instalacijom.



Preuzmite softver kao što je [Balena Etcher](https://etcher.balena.io/) ili sličan alat, zatim pokrenite softver.


Odaberite prethodno preuzetu ISO sliku **Elementary OS** i postavite vaš USB ključ kao cilj.



Kliknite na dugme **flash** da biste započeli proces i sačekajte da se proces završi pre nego što uklonite USB ključ.



![0_02](assets/fr/02.webp)



## Ključ za pokretanje i pristup BIOS-u



Isključite računar na koji želite da instalirate Elementary OS, a zatim priključite USB ključ.


When your computer starts up, access the BIOS (`ESC`, `F9` or `F11` depending on the brand) and select the USB key as the boot device, then press `Enter` to start the boot process.

## Ažuriranje sistema



Pre nego što se koristi duže vreme, važno je ažurirati sistem najnovijim zakrpama.


### Opcija 1: Ažuriranje putem Interface grafike



Uđite u **AppCenter** i kliknite na ikonu za ažuriranje u gornjem desnom uglu. Sačekajte da se ažuriranja prikažu, zatim kliknite na **"Update All "** da započnete instalaciju.



![0_20](assets/fr/20.webp)



![0_21](assets/fr/21.webp)



### Opcija 2: Ažuriranje putem terminala



Ažuriranje možete izvršiti i putem komandne linije preko vašeg terminala: preporučena opcija zbog svoje tačnosti.



```shell
sudo apt update
```



```shell
sudo apt full-upgrade
```



Za vaše prve ažuriranja, operativni sistem zahteva vašu korisničku lozinku i potvrdu za ažuriranje softverskih paketa, čak i u Elementary kernelu.



## Konfiguracija radnog okruženja



Elementary OS uključuje samo osnovne alate. Da biste prilagodili svoje okruženje svojim potrebama, posebno ako ste programer, preporučujemo instaliranje dodatnih alata.





- Možete dodati korisne zavisnosti sledećom komandom (prilagodite je svojim potrebama):



```shell
sudo apt update && sudo apt install -y git python3 python3-pip build-essential wget curl zsh make snapd && sudo snap install code --classic
```



Ova komanda instalira **Git**, **Python 3**, **pip**, **alatke za kompajliranje**, **wget**, **curl**, **zsh**, **make**, **snapd** i **vscode** kako bi se pripremilo osnovno razvojno okruženje.
