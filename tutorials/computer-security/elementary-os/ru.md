Elementary OS - это операционная система на базе Ubuntu, разработанная как простая, быстрая и стабильная система для повседневного использования. Она представляет собой сбалансированную Linux-альтернативу MacOS и Windows. Ее плавная, интуитивно понятная и незагроможденная графика Interface делает ее легкой в освоении даже для новичков. Кроме того, особое внимание уделяется эргономике, безопасности и производительности.

## Почему стоит выбрать Elementary OS





- Простота и удобство использования**: Графический интерфейс Elementary OS Interface находится посередине между MacOs и Windows. Благодаря этому ее легко освоить даже неопытным пользователям.





- Безопасность**: Как и большинство дистрибутивов Linux, Elementary OS обладает высоким уровнем безопасности. Регулярные обновления, управление правами и отсутствие распространенных вирусов делают ее надежной системой.





- Скорость**: Elementary OS - это легкий дистрибутив. Он требует мало ресурсов, что делает его быстрым и подходящим для компьютеров со скромной конфигурацией.





- Бесплатно**: Система полностью бесплатна. Однако, загрузив ее, вы можете сделать пожертвование в поддержку разработчиков.





- Активное сообщество**: Сообщество вокруг Elementary OS разнообразно и отзывчиво. Если у вас возникнут трудности, вы легко найдете помощь на форумах или в социальных сетях.



## Установка и настройка


### Необходимое оборудование



Перед началом установки убедитесь, что у вас есть следующее оборудование:





- Ключ **USB** объемом не менее 12 ГБ
- Оперативная память** не менее 4 ГБ
- Диск **Hard объемом 20 Гб** или более для комфортного использования



## Загрузите образ ISO



Перейдите на официальный сайт операционной системы [elementary](https://elementary.io/) и выберите сумму для поддержки проекта. Этот шаг необязателен.


Если вы хотите загрузить ISO-образ бесплатно, введите 0 в поле **"Другое "** и начните загрузку ISO-образа системы.



![0_01](assets/fr/01.webp)



## Создание загрузочного USB-носителя



После загрузки ISO-образа вам нужно будет сделать его загрузочным на USB-носителе, чтобы продолжить установку.



Загрузите программное обеспечение, например [Balena Etcher](https://etcher.balena.io/) или аналогичный инструмент, затем запустите его.


Выберите загруженный ранее ISO-образ **Elementary OS** и установите USB-носитель в качестве целевого.



Нажмите на кнопку **flash**, чтобы начать процесс, и дождитесь его завершения, прежде чем извлекать USB-носитель.



![0_02](assets/fr/02.webp)



## Загрузка с помощью клавиш и доступ к BIOS



Выключите компьютер, на который вы хотите установить Elementary OS, затем подключите USB-носитель.


Когда компьютер загрузится, войдите в BIOS (`ESC`, `F9` или `F11` в зависимости от марки) и выберите USB-носитель в качестве загрузочного устройства, затем нажмите `Enter`, чтобы начать процесс загрузки.

### Режим тестирования



Элементарная ОС позволяет протестировать систему перед ее установкой. В этом режиме вы можете изучить Interface, не изменяя ничего на диске Hard.



### Запуск установки





- Нажмите на **"Стереть диск и установить "**, чтобы установить программу непосредственно на весь диск.
- Нажмите на **"Выборочная установка "**, если вы хотите управлять разделами вручную.



![0_07](assets/fr/07.webp)



### Выбор диска



Выберите диск, на который будет установлена Elementary OS, и нажмите кнопку Продолжить.

### Шифрование диска



Опция позволяет зашифровать диск, чтобы **защитить ваши данные**. Для активации этой защиты необходимо установить надежный пароль. Впрочем, это необязательно.

## Обновление системы



Перед длительным использованием важно обновить систему последними патчами.


### Вариант 1: Обновление через графический интерфейс Interface



Войдите в **AppCenter** и нажмите на значок обновления в правом верхнем углу. Дождитесь появления списка обновлений, затем нажмите **"Обновить все "**, чтобы начать установку.



![0_20](assets/fr/20.webp)



![0_21](assets/fr/21.webp)



### Вариант 2: Обновление через терминал



Вы также можете выполнить обновление из командной строки через терминал: этот вариант рекомендуется из-за его точности.



```shell
sudo apt update
```



```shell
sudo apt full-upgrade
```



При первых обновлениях операционная система требует пароль пользователя и подтверждение для обновления программных пакетов, даже в ядре Elementary.



## Настройка рабочей среды



В состав Elementary OS входят только самые необходимые инструменты. Чтобы адаптировать среду под свои нужды, особенно если вы разработчик, мы рекомендуем установить дополнительные инструменты.





- Вы можете добавить полезные зависимости с помощью следующей команды (ее можно адаптировать под свои нужды):



```shell
sudo apt update && sudo apt install -y git python3 python3-pip build-essential wget curl zsh make snapd && sudo snap install code --classic
```



Эта команда устанавливает **Git**, **Python 3**, **pip**, **compiler tools**, **wget**, **curl**, **zsh**, **make**, **snapd** и **vscode** для подготовки базовой среды разработки.
