Elementary OS ni ubuhinga bushingiye kuri Ubuntu, bugenewe kuba bworoshe, bwihuta kandi bushikamye ku bikorwa vyinshi vya misi yose. Igereranya Linux iri ku rugero rubereye ishobora gusubirira MacOS na Windows. Igishushanyo caco Interface gifise amazi, gisanzwe kandi kitagiramwo ibintu vyinshi gituma vyoroha kwiga, mbere n’abatangura. Ishimikiye kandi ku bijanye n’ubuhinga bwo gukora neza, umutekano n’ubushobozi.

## Kubera iki uhisemwo OS y'intango





- Ukworohereza no gukoresha neza**: Igishushanyo ca Interface ca Elementary OS kiri hagati y’ivyo MacOs na Windows. Ukwo kumenyera bituma vyoroha kubikoresha, mbere n’ababikoresha batazi utuntu n’utundi.





- Umutekano**: Cokimwe n’ibindi bikoresho vyinshi vya Linux, Elementary OS ivyungukira ku mutekano wo hejuru. Guhindura ibintu bihoraho, gucunga uburenganzira n’ukubura ama virusi asanzwe bituma iba uburyo bwo kwizigirwa.





- Umuvuduko**: OS y’intango ni ugusangira kworoshe. Bisaba ibikoresho bike, ivyo bikaba bituma vyihuta kandi bibereye kuri orodinateri zifise imiterere mikeyi.





- Ubuntu**: Sisitemu ni ubuntu rwose. Ariko rero, iyo uyikuye kuri Internet, urashobora gutanga intererano kugira ngo ushigikire abayiteguye.





- Umuryango ukora**: Umuryango ukikuje Elementary OS ni uwutandukanye kandi urasubiza. Iyo ushikiwe n’ingorane, urashobora kuronka imfashanyo bitagoranye ku mahuriro canke ku mbuga ngurukanabumenyi.



## Gushiramwo no gutunganya


### Ibisabwa mu bikoresho



Imbere yo gutangura gushiramwo, urabe neza ko ufise ibi bikoresho bikurikira:





- **Urufunguzo rwa USB** rw'imiburiburi 12 GB
- RAM** ububiko bwa 4 GB
- Disiki **Hard ya 20 GB** canke irenga kugira ngo ukoreshe neza



## Gukuraho ishusho ya ISO



Genda ku rubuga rwemewe rw’ubuhinga bwo gukoresha [elementary](https://elementary.io/) uhitemwo amahera yo gufasha umugambi. Iyi ntambwe ni ubusabe.


Niba wipfuza gukuraho ishusho ya ISO ku buntu, shiramwo 0 mu kibanza **"Ibindi "** maze utangure gukuraho ishusho ya ISO ya sisitemu.



![0_01](assets/fr/01.webp)



## Gukora urufunguzo rwa USB rushobora gufungurwa



Uhejeje gukuraho ishusho ya ISO, uzokenera kuyigira bootable ku rufunguzo rwa USB kugira ngo ukomeze gushiramwo.



Gukuraho porogarama nka [Balena Etcher](https://etcher.balena.io/) canke igikoresho gisa n’ico, hanyuma utangure iyo porogarama.


Hitamwo ishusho ISO ya **Elementary OS** yashizweho mbere maze ushireho urufunguzo rwawe rwa USB nk'intumbero.



Fyonda kuri buto ya **flash** kugira utangure igikorwa, hanyuma urindire gushika igikorwa kirangiye imbere yo gukuraho urufunguzo rwa USB.



![0_02](assets/fr/02.webp)



## Urufunguzo rwo gufungura no gushika ku BIOS



Zima mudasobwa wipfuza gushiramwo Elementary OS, hanyuma ushiremwo urufunguzo rwa USB.


Iyo mudasobwa yawe itanguye, genda muri BIOS (`ESC`, `F9` canke `F11` bivanye n’ibara) hanyuma uhitemwo urufunguzo rwa USB nk’igikoresho co gufungura, hanyuma ukande `Enter` kugira ngo utangure igikorwa co gufungura.

## Kuvugurura sisitemu



Imbere y’uko ukoresha igihe kirekire, birahambaye ko ukoresha ubuhinga bushasha n’ibintu bishasha.


### Ihitamwo rya 1: Kuvugurura biciye ku bishushanyo vya Interface



Injira muri **AppCenter** hanyuma ukande ku kimenyetso c’ivugurura kiri hejuru iburyo. Rindira ko ivyahinduwe biza ku rutonde, hanyuma ukande kuri **"Update All "** kugira ngo utangure gushiramwo.



![0_20](assets/fr/20.webp)



![0_21](assets/fr/21.webp)



### Amahitamwo ya 2: Kuvugurura biciye ku kibanza



Ushobora kandi gukora ivugurura kuva ku murongo w'amabwirizwa biciye ku nzira yawe: uburyo bwiza kubera ukuri kwayo.



```shell
sudo apt update
```



```shell
sudo apt full-upgrade
```



Ku bijanye n’ivugurura ryawe rya mbere, sisitemu ikoresha isaba ijambobanga ryawe ry’ukoresha n’ukwemeza kugira ngo uvugurure amapaki ya porogarama, mbere no mu nzira y’intango.



## Gutunganya ibidukikije vy'akazi



OS y’intango irimwo ibikoresho vy’ingenzi gusa. Kugira ngo uhuze ibidukikije vyawe n'ivyo ukeneye, cane cane iyo uri umuhinga mu bijanye n'ubuhinga, turagusavye gushiramwo ibindi bikoresho.





- Ushobora kwongerako ibishingiye ku ngirakamaro n'itegeko rikurikira (kugira ngo bihuzwe n'ivyo ukeneye):



```shell
sudo apt update && sudo apt install -y git python3 python3-pip build-essential wget curl zsh make snapd && sudo snap install code --classic
```



Iri tegeko rishiramwo **Git**, **Python 3**, **pip**, **ibikoresho vy'umukozi**, **wget**, **curl**, **zsh**, **make**, **snapd** na **vscode** kugira ngo utegure ibidukikije vy'iterambere ry'ishimikiro.



OS y’intango ubu irakora ku mashine yawe. Filozofiya yayo y’ukworohereza, yoroshe n’uburanga ituma iba ihitamwo ryiza cane ryo gukoresha ku giti cawe no ku kazi. Uronka uburyo buhamye, bugenda buhinduka kandi butagiramwo ibintu vyinshi, bwiteguye guhindurwa bivanye n’ivyo ukunda. Yaba iyo guteza imbere, gukoresha ibiro canke gusoma ku musi ku musi, vyose biri mu kibanza kugira ngo wubake ahantu heza ho gukorera, horoshe kandi haryoshe. Raba kandi inyigisho yacu kuri Fedora, uburyo bwo gutanga Linux bworoshe, bukomeye kandi bufise ibice.
