Elementary OS เป็นระบบปฏิบัติการที่พัฒนาบนพื้นฐานของ Ubuntu ออกแบบมาให้เรียบง่าย รวดเร็ว และเสถียรสำหรับการใช้งานในชีวิตประจำวันหลายๆ ด้าน มันเป็นทางเลือกที่สมดุลของ Linux เมื่อเทียบกับ MacOS และ Windows อินเทอร์เฟซกราฟิก Interface ที่ลื่นไหล ใช้งานง่าย และไม่ซับซ้อน ทำให้เรียนรู้ได้ง่าย แม้แต่สำหรับผู้เริ่มต้น นอกจากนี้ยังเน้นที่การยศาสตร์ ความปลอดภัย และประสิทธิภาพ

## ทำไมถึงเลือก Elementary OS





- ความเรียบง่ายและใช้งานง่าย**: Elementary OS's graphical Interface อยู่กึ่งกลางระหว่าง MacOs และ Windows ความคุ้นเคยนี้ทำให้สามารถนำไปใช้ได้ง่าย แม้กระทั่งสำหรับผู้ใช้ที่ไม่มีประสบการณ์





- ความปลอดภัย**: เช่นเดียวกับการแจกจ่าย Linux ส่วนใหญ่ Elementary OS มีความปลอดภัยในระดับสูง การอัปเดตเป็นประจำ การจัดการสิทธิ์ และการไม่มีไวรัสทั่วไปทำให้มันเป็นระบบที่เชื่อถือได้





- Speed**: Elementary OS เป็นดิสทริบิวชันที่มีน้ำหนักเบา ต้องการทรัพยากรเพียงเล็กน้อย ทำให้รวดเร็วและเหมาะสำหรับคอมพิวเตอร์ที่มีการกำหนดค่าที่ไม่สูงมาก





- ฟรี**: ระบบนี้ฟรีทั้งหมด อย่างไรก็ตาม เมื่อคุณดาวน์โหลด คุณสามารถบริจาคเพื่อสนับสนุนนักพัฒนาได้





- ชุมชนที่มีชีวิตชีวา**: ชุมชนรอบๆ Elementary OS มีความหลากหลายและตอบสนองได้ดี หากคุณพบปัญหา คุณสามารถหาความช่วยเหลือได้ง่ายๆ บนฟอรัมหรือเครือข่ายสังคมออนไลน์



## การติดตั้งและการกำหนดค่า


### ข้อกำหนดเบื้องต้นของฮาร์ดแวร์



ก่อนเริ่มการติดตั้ง โปรดตรวจสอบว่าคุณมีอุปกรณ์ดังต่อไปนี้:





- **USB key** ขนาดอย่างน้อย 12 GB
- หน่วยความจำ RAM** อย่างน้อย 4 GB
- ฮาร์ดดิสก์ **ขนาด 20 GB** หรือมากกว่าสำหรับการใช้งานที่สะดวกสบาย



## ดาวน์โหลดไฟล์ภาพ ISO



ไปที่เว็บไซต์ระบบปฏิบัติการอย่างเป็นทางการ [elementary](https://elementary.io/) และเลือกจำนวนเงินเพื่อสนับสนุนโครงการ ขั้นตอนนี้เป็นทางเลือก


หากคุณต้องการดาวน์โหลดไฟล์ ISO โดยไม่เสียค่าใช้จ่าย ให้ใส่ 0 ในช่อง **"อื่น ๆ "** และเริ่มดาวน์โหลดไฟล์ ISO ของระบบ



![0_01](assets/fr/01.webp)



## สร้าง USB ที่สามารถบูตได้



เมื่อคุณดาวน์โหลดไฟล์ ISO เสร็จแล้ว คุณจะต้องทำให้มันสามารถบูตได้บน USB key เพื่อดำเนินการติดตั้งต่อไป



ดาวน์โหลดซอฟต์แวร์เช่น [Balena Etcher](https://etcher.balena.io/) หรือเครื่องมือที่คล้ายกัน จากนั้นเปิดซอฟต์แวร์


เลือกภาพ ISO **Elementary OS** ที่ดาวน์โหลดมาก่อนหน้านี้และตั้งค่า USB key ของคุณเป็นเป้าหมาย



คลิกที่ปุ่ม **flash** เพื่อเริ่มกระบวนการ และรอจนกว่ากระบวนการจะเสร็จสิ้นก่อนที่จะถอดกุญแจ USB ออก



![0_02](assets/fr/02.webp)



## การบูตคีย์และการเข้าถึง BIOS



ปิดคอมพิวเตอร์ที่คุณต้องการติดตั้ง Elementary OS จากนั้นเสียบกุญแจ USB เข้าไป


เมื่อคอมพิวเตอร์ของคุณเริ่มทำงาน ให้เข้าถึง BIOS (`ESC`, `F9` หรือ `F11` ขึ้นอยู่กับยี่ห้อ) และเลือก USB key เป็นอุปกรณ์บูต จากนั้นกด `Enter` เพื่อเริ่มกระบวนการบูต

## อัปเดตระบบ

### ตัวเลือกที่ 1: อัปเดตผ่านกราฟิก Interface



เข้าสู่ **AppCenter** และคลิกที่ไอคอนอัปเดตที่มุมขวาบน รอให้รายการอัปเดตแสดงขึ้นมา จากนั้นคลิกที่ **"Update All "** เพื่อเริ่มการติดตั้ง



![0_20](assets/fr/20.webp)

### ตัวเลือกที่ 2: อัปเดตผ่านเทอร์มินัล



คุณยังสามารถดำเนินการอัปเดตจากบรรทัดคำสั่งผ่านทางเทอร์มินัลของคุณ: เป็นตัวเลือกที่แนะนำสำหรับความแม่นยำของมัน



```shell
sudo apt update
```



```shell
sudo apt full-upgrade
```



สำหรับการอัปเดตครั้งแรก ระบบปฏิบัติการต้องการรหัสผ่านผู้ใช้ของคุณและการยืนยันเพื่ออัปเดตแพ็คเกจซอฟต์แวร์ แม้กระทั่งในเคอร์เนล Elementary



## การกำหนดค่าบรรยากาศการทำงาน



Elementary OS รวมเฉพาะเครื่องมือที่จำเป็นเท่านั้น เพื่อปรับสภาพแวดล้อมของคุณให้ตรงกับความต้องการ โดยเฉพาะถ้าคุณเป็นนักพัฒนา เราแนะนำให้ติดตั้งเครื่องมือเพิ่มเติม





- คุณสามารถเพิ่มการพึ่งพาที่มีประโยชน์ด้วยคำสั่งต่อไปนี้ (ปรับให้เหมาะสมกับความต้องการของคุณ):



```shell
sudo apt update && sudo apt install -y git python3 python3-pip build-essential wget curl zsh make snapd && sudo snap install code --classic
```



คำสั่งนี้ติดตั้ง **Git**, **Python 3**, **pip**, **compiler tools**, **wget**, **curl**, **zsh**, **make**, **snapd** และ **vscode** เพื่อเตรียมสภาพแวดล้อมการพัฒนาขั้นพื้นฐาน



Elementary OS กำลังทำงานบนเครื่องของคุณแล้ว ปรัชญาของความเรียบง่าย เบา และสง่างามทำให้มันเป็นตัวเลือกที่ยอดเยี่ยมสำหรับการใช้งานทั้งส่วนตัวและมืออาชีพ คุณจะได้รับระบบที่เสถียร ลื่นไหล และไม่รก พร้อมที่จะปรับแต่งตามความต้องการของคุณ ไม่ว่าจะเป็นสำหรับการพัฒนา การใช้งานสำนักงาน หรือการท่องเว็บในชีวิตประจำวัน ทุกอย่างพร้อมที่จะสร้างสภาพแวดล้อมการทำงานที่มีประสิทธิภาพ ใช้งานง่าย และน่าพึงพอใจ อย่าลืมตรวจสอบบทแนะนำของเราเกี่ยวกับ Fedora ซึ่งเป็นการกระจาย Linux ที่เรียบง่าย แข็งแกร่ง และโมดูลาร์เช่นกัน
