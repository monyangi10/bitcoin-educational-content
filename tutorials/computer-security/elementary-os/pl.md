Elementary OS to system operacyjny oparty na Ubuntu, zaprojektowany tak, aby był prosty, szybki i stabilny w wielu codziennych zastosowaniach. Stanowi on zrównoważoną linuksową alternatywę dla MacOS i Windows. Jego płynny, intuicyjny i przejrzysty graficzny Interface sprawia, że jest łatwy do nauczenia, nawet dla początkujących. Skupia się również na ergonomii, bezpieczeństwie i wydajności.

## Dlaczego warto wybrać Elementary OS





- **Prostota i łatwość użytkowania**: Graficzny Interface systemu Elementary OS znajduje się w połowie drogi między systemami MacOs i Windows. To sprawia, że jest on łatwy w obsłudze nawet dla niedoświadczonych użytkowników.





- **Bezpieczeństwo**: Podobnie jak większość dystrybucji Linuksa, Elementary OS korzysta z wysokiego poziomu bezpieczeństwa. Regularne aktualizacje, zarządzanie prawami i brak powszechnych wirusów sprawiają, że jest to niezawodny system.





- **Szybkość**: Elementary OS jest lekką dystrybucją. Wymaga niewielu zasobów, dzięki czemu jest szybki i odpowiedni dla komputerów o skromnych konfiguracjach.





- **Darmowy**: System jest całkowicie darmowy. Jednak po jego pobraniu możesz przekazać darowiznę, aby wesprzeć programistów.





- **Aktywna społeczność**: Społeczność skupiona wokół Elementary OS jest zróżnicowana i elastyczna. Jeśli napotkasz trudności, możesz łatwo znaleźć pomoc na forach lub w sieciach społecznościowych.



## Instalacja i konfiguracja


### Wymagania sprzętowe



Przed rozpoczęciem instalacji upewnij się, że posiadasz następujący sprzęt:





- **Klucz USB** o pojemności co najmniej 12 GB
- Pamięć **RAM** o pojemności co najmniej 4 GB
- Dysk **Hard o pojemności 20 GB** lub większej do wygodnego użytkowania



## Pobierz obraz ISO



Wejdź na oficjalną stronę systemu operacyjnego [elementary](https://elementary.io/) i wybierz kwotę, którą chcesz wesprzeć projekt. Ten krok jest opcjonalny.


Jeśli chcesz pobrać obraz ISO bezpłatnie, wpisz 0 w polu **"Inne "** i rozpocznij pobieranie obrazu ISO systemu.



![0_01](assets/fr/01.webp)



## Tworzenie bootowalnego klucza USB



Po pobraniu obrazu ISO należy go uruchomić na kluczu USB, aby kontynuować instalację.



Pobierz oprogramowanie takie jak [Balena Etcher](https://etcher.balena.io/) lub podobne narzędzie, a następnie uruchom je.


Wybierz wcześniej pobrany obraz ISO **Elementary OS** i ustaw klucz USB jako cel.



Kliknij przycisk **flash**, aby rozpocząć proces i poczekaj na jego zakończenie przed odłączeniem klucza USB.



![0_02](assets/fr/02.webp)



## Uruchamianie za pomocą klawiszy i dostęp do systemu BIOS



Wyłącz komputer, na którym chcesz zainstalować system Elementary OS, a następnie podłącz klucz USB.


Po uruchomieniu komputera wejdź do BIOS-u (`ESC`, `F9` lub `F11` w zależności od marki) i wybierz klucz USB jako urządzenie rozruchowe, a następnie naciśnij `Enter`, aby rozpocząć proces rozruchu.



## Instalacja podstawowego systemu operacyjnego



Instalacja rozpocznie się automatycznie, jeśli klucz USB jest poprawnie skonfigurowany.



### Wybór języka



Wybierz język, w którym chcesz zainstalować system. Możesz także wybrać wariant regionalny.

![0_04](assets/fr/04.webp)



### Układ klawiatury



Wybierz układ odpowiadający posiadanej klawiaturze. Dostępne jest pole umożliwiające sprawdzenie, czy klawisze generują prawidłowe znaki.



![0_05](assets/fr/05.webp)



![0_06](assets/fr/06.webp)



### Tryb testowy



Elementary OS pozwala przetestować system przed jego instalacją. Ten tryb pozwala poznać Interface bez modyfikowania czegokolwiek na dysku Hard.



### Uruchamianie instalacji





- Kliknij **"Wymaż dysk i zainstaluj "**, aby zainstalować bezpośrednio na całym dysku.
- Kliknij **"Instalacja niestandardowa "**, jeśli chcesz ręcznie zarządzać partycjami.



![0_07](assets/fr/07.webp)



### Wybór dysku



Wybierz dysk, na którym ma zostać zainstalowany system Elementary OS, a następnie kliknij przycisk Continue.



![0_08](assets/fr/08.webp)



### Szyfrowanie dysku



Opcja umożliwia zaszyfrowanie dysku w celu **zabezpieczenia danych**. Aby aktywować tę ochronę, należy ustawić silne hasło. Jest to jednak opcjonalne.



![0_09](assets/fr/09.webp)



![0_10](assets/fr/10.webp)



### Uruchamianie instalacji



Przed uruchomieniem instalacji można autoryzować automatyczną instalację dodatkowych sterowników sprzętowych, w zależności od kompatybilności urządzenia.





- Kliknij "Usuń i zainstaluj", aby rozpocząć instalację.
- Poczekaj na zakończenie procesu.



![0_11](assets/fr/11.webp)



![0_12](assets/fr/12.webp)

Po zakończeniu kliknij przycisk **enter**, aby uruchomić ponownie, a następnie wyjmij klucz USB w odpowiednim momencie, aby uniknąć ponownego uruchomienia instalacji.

## Aktualizacja systemu



Przed dłuższym użytkowaniem ważne jest, aby zaktualizować system o najnowsze poprawki.


### Opcja 1: Aktualizacja za pomocą grafiki Interface



Wejdź do **AppCenter** i kliknij ikonę aktualizacji w prawym górnym rogu. Poczekaj, aż aktualizacje zostaną wyświetlone na liście, a następnie kliknij **"Aktualizuj wszystko "**, aby rozpocząć instalację.



![0_20](assets/fr/20.webp)



![0_21](assets/fr/21.webp)



### Opcja 2: Aktualizacja przez terminal



Aktualizację można również przeprowadzić z wiersza poleceń za pośrednictwem terminala: jest to zalecana opcja ze względu na jej dokładność.



```shell
sudo apt update
```



```shell
sudo apt full-upgrade
```



W przypadku pierwszych aktualizacji system operacyjny wymaga hasła użytkownika i potwierdzenia, aby zaktualizować pakiety oprogramowania, nawet w jądrze Elementary.



## Konfiguracja środowiska pracy



Elementary OS zawiera tylko niezbędne narzędzia. Aby dostosować środowisko do swoich potrzeb, zwłaszcza jeśli jesteś programistą, zalecamy zainstalowanie dodatkowych narzędzi.





- Przydatne zależności można dodać za pomocą poniższego polecenia (które należy dostosować do własnych potrzeb):



```shell
sudo apt update && sudo apt install -y git python3 python3-pip build-essential wget curl zsh make snapd && sudo snap install code --classic
```



To polecenie instaluje **Git**, **Python 3**, **pip**, **narzędzia kompilatora**, **wget**, **curl**, **zsh**, **make**, **snapd** i **vscode** w celu przygotowania podstawowego środowiska programistycznego.
