Elementary OS는 우분투 기반 운영 체제로, 일상적인 용도에 맞게 간단하고 빠르며 안정적으로 설계되었습니다. MacOS와 Windows에 대한 균형 잡힌 Linux 대안입니다. 유동적이고 직관적이며 깔끔한 그래픽의 Interface은 초보자도 쉽게 배울 수 있습니다. 또한 인체공학, 보안 및 성능에 중점을 두고 있습니다.



https://planb.network/tutorials/computer-security/operating-system/ubuntu-78a3be56-5d51-4ec3-8629-0dd27c352ab5

## Elementary OS를 선택해야 하는 이유





- 단순함과 사용 편의성**: Elementary OS의 그래픽 Interface은 MacO와 Windows의 중간 수준입니다. 이러한 친숙함 덕분에 경험이 없는 사용자도 쉽게 적응할 수 있습니다.





- 보안**: 대부분의 Linux 배포판과 마찬가지로 Elementary OS는 높은 수준의 보안을 제공합니다. 정기적인 업데이트, 권한 관리, 일반적인 바이러스의 부재로 인해 신뢰할 수 있는 시스템입니다.





- 속도**: 기본 OS는 경량 배포판입니다. 리소스가 거의 필요하지 않으므로 속도가 빠르며 적당한 구성의 컴퓨터에 적합합니다.





- 무료**: 이 시스템은 완전 무료입니다. 하지만 다운로드하면 개발자를 지원하기 위해 기부할 수 있습니다.





- 활발한 커뮤니티**: Elementary OS 관련 커뮤니티는 다양하고 반응이 좋습니다. 문제가 발생하면 포럼이나 소셜 네트워크에서 쉽게 도움을 찾을 수 있습니다.



## 설치 및 구성


### 하드웨어 사전 요구 사항



설치를 시작하기 전에 다음 장비가 준비되어 있는지 확인하세요:





- 최소 12GB의 **USB 키** 제공
- 최소 4GB의 RAM** 메모리
- 쾌적한 사용을 위한 20GB 이상의 **Hard 디스크** 사용



## ISO 이미지 다운로드



공식 운영 체제 웹사이트 [초등](https://elementary.io/)으로 이동하여 프로젝트 후원 금액을 선택합니다. 이 단계는 선택 사항입니다.


ISO 이미지를 무료로 다운로드하려면 **"기타"** 필드에 0을 입력하고 시스템 ISO 이미지 다운로드를 시작하세요.



![0_01](assets/fr/01.webp)



## 부팅 가능한 USB 키 만들기



ISO 이미지를 다운로드한 후에는 USB 키로 부팅할 수 있도록 만들어야 설치를 계속할 수 있습니다.



Balena Etcher](https://etcher.balena.io/) 또는 유사한 도구와 같은 소프트웨어를 다운로드한 다음 소프트웨어를 실행합니다.


이전에 다운로드한 **Elementary OS** ISO 이미지를 선택하고 USB 키를 대상으로 설정합니다.



플래시** 버튼을 클릭하여 프로세스를 시작하고 프로세스가 완료될 때까지 기다렸다가 USB 키를 제거하세요.



![0_02](assets/fr/02.webp)



## 키 부팅 및 BIOS 액세스



Elementary OS를 설치하려는 컴퓨터의 전원을 끈 다음 USB 키를 연결합니다.


컴퓨터가 시작되면 BIOS(브랜드에 따라 `ESC`, `F9` 또는 `F11`)에 액세스하여 USB 키를 부팅 장치로 선택한 다음 `Enter`를 눌러 부팅 프로세스를 시작하세요.

## 시스템 업데이트



장기간 사용하기 전에 최신 패치로 시스템을 업데이트하는 것이 중요합니다.


### 옵션 1: Interface 그래픽을 통한 업데이트



앱센터**로 들어가서 오른쪽 상단의 업데이트 아이콘을 클릭합니다. 업데이트가 나열될 때까지 기다린 다음 **"모두 업데이트"**를 클릭하여 설치를 시작합니다.



![0_20](assets/fr/20.webp)



![0_21](assets/fr/21.webp)



### 옵션 2: 터미널을 통한 업데이트



터미널을 통해 명령줄에서 업데이트를 수행할 수도 있는데, 정확성을 위해 권장되는 옵션입니다.



```shell
sudo apt update
```



```shell
sudo apt full-upgrade
```



첫 번째 업데이트의 경우 운영 체제에서 소프트웨어 패키지를 업데이트하려면 사용자 비밀번호와 확인이 필요하며, 이는 Elementary 커널에서도 마찬가지입니다.



## 작업 환경 구성



Elementary OS에는 필수 도구만 포함되어 있습니다. 특히 개발자의 경우 필요에 따라 환경을 조정하려면 추가 도구를 설치하는 것이 좋습니다.





- 다음 명령을 사용하여 유용한 종속성을 추가할 수 있습니다(필요에 따라 조정 가능):



```shell
sudo apt update && sudo apt install -y git python3 python3-pip build-essential wget curl zsh make snapd && sudo snap install code --classic
```



이 명령은 **Git**, **Python 3**, **pip**, **컴파일러 도구**, **wget**, **curl**, **zsh**, **make**, **snapd** 및 **vscode**를 설치하여 기본 개발 환경을 준비합니다.



이제 컴퓨터에서 Elementary OS를 실행할 수 있습니다. 단순함, 가벼움, 우아함이라는 철학으로 개인용과 업무용 모두에 탁월한 선택이 될 것입니다. 안정적이고 유동적이며 깔끔한 시스템을 원하는 대로 커스터마이징할 수 있습니다. 개발, 사무용 또는 일상적인 브라우징 등 효율적이고 직관적이며 쾌적한 작업 환경을 구축할 수 있는 모든 것이 준비되어 있습니다. 또한 똑같이 간단하고 강력하며 모듈식으로 구성된 Linux 배포판인 Fedora에 대한 튜토리얼도 확인해 보세요.
