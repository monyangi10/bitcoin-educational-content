Elementary OSはUbuntuベースのオペレーティング・システムで、多くの日常的な用途に対応できるよう、シンプル、高速、安定するように設計されています。MacOSやWindowsに代わるバランスの取れたLinuxです。流れるような直感的ですっきりしたグラフィカルInterfaceにより、初心者でも簡単に学ぶことができます。また、人間工学、セキュリティ、パフォーマンスにも重点を置いています。

## エレメンタリーOSを選ぶ理由





- **シンプルさと使いやすさ**：エレメンタリーOSのグラフィカルなInterfaceは、マックOSとウィンドウズの中間的なものです。この親しみやすさが、経験の浅いユーザーでも簡単に導入できる理由です。





- セキュリティ：ほとんどのLinuxディストリビューションと同様、Elementary OSは高レベルのセキュリティの恩恵を受けています。定期的なアップデート、権限管理、一般的なウイルスの不在により、信頼性の高いシステムとなっています。





- **スピード**：Elementary OSは軽量なディストリビューションです。リソースをほとんど必要としないため高速で、控えめな構成のコンピュータに適しています。





- **無料**：システムは完全に無料です。ただし、ダウンロードする際に、開発者を支援するための寄付をすることができます。





- **活発なコミュニティ**：Elementary OSを取り巻くコミュニティは多様性に富んでおり、反応が良い。困ったことがあれば、フォーラムやソーシャルネットワークで簡単に助けを見つけることができます。



## インストールと設定


### ハードウェアの前提条件



インストールを開始する前に、以下の機器が揃っていることを確認してください：





- 12GB以上のUSBキー
- 4GB以上の**RAM**メモリ
- 快適に使用できる20GB**以上の**Hardディスク



## ISOイメージのダウンロード



オフィシャル・オペレーティング・システムのウェブサイト[elementary](https://elementary.io/)にアクセスし、支援金額を選択する。このステップは任意です。


ISOイメージを無料でダウンロードしたい場合は、**"Other "**フィールドに0を入力し、システムISOイメージのダウンロードを開始します。



![0_01](assets/fr/01.webp)



## ブータブルUSBキーの作成



ISOイメージをダウンロードしたら、インストールを進めるためにUSBキーで起動可能にする必要がある。



Balena Etcher](https://etcher.balena.io/)などのソフトをダウンロードし、起動してください。


先にダウンロードした**Elementary OS** ISOイメージを選択し、USBキーをターゲットに設定します。



フラッシュ**ボタン**をクリックしてプロセスを開始し、プロセスが完了するまで待ってからUSBキーを取り外してください。



![0_02](assets/fr/02.webp)



## キーブートとBIOSアクセス



Elementary OSをインストールするコンピュータの電源を切り、USBキーを差し込みます。


コンピュータが起動したら、BIOSにアクセスし（ブランドにより`ESC`、`F9`または`F11`）、USBキーをブートデバイスとして選択し、`Enter`を押してブートプロセスを開始します。

## システム・アップデート



長期間使用する前に、最新のパッチでシステムをアップデートすることが重要です。


### オプション1：Interfaceグラフィックスによるアップデート



AppCenter**に入り、右上のアップデートアイコンをクリックします。アップデートの一覧が表示されるまで待ち、**"Update All "**をクリックしてインストールを開始します。



![0_20](assets/fr/20.webp)



![0_21](assets/fr/21.webp)



### オプション2：ターミナル経由でアップデートする



ターミナルからコマンドラインでアップデートを実行することもできる。



```shell
sudo apt update
```



```shell
sudo apt full-upgrade
```



初めてのアップデートでは、エレメンタリーカーネルであっても、ソフトウェアパッケージをアップデートするために、オペレーティングシステムはユーザーパスワードと確認を要求します。



## 作業環境設定



Elementary OSには必要不可欠なツールしか含まれていません。特に開発者であれば、自分のニーズに合わせて環境を適応させるために、追加のツールをインストールすることをお勧めします。





- 以下のコマンドで便利な依存関係を追加することができる（必要に応じて変更してください）：



```shell
sudo apt update && sudo apt install -y git python3 python3-pip build-essential wget curl zsh make snapd && sudo snap install code --classic
```



このコマンドは**Git**、**Python 3**、**pip**、**コンパイラツール**、**wget**、**curl**、**zsh**、**make**、**snapd**、**vscode**をインストールし、基本的な開発環境を準備します。
