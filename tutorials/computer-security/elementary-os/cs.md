Elementary OS je operační systém založený na Ubuntu, který je navržen tak, aby byl jednoduchý, rychlý a stabilní pro mnoho každodenních použití. Představuje vyváženou linuxovou alternativu k MacOS a Windows. Díky plynulé, intuitivní a nenáročné grafice Interface se jej snadno naučí i začátečníci. Zaměřuje se také na ergonomii, bezpečnost a výkon.



https://planb.network/tutorials/computer-security/operating-system/ubuntu-78a3be56-5d51-4ec3-8629-0dd27c352ab5

## Proč si vybrat systém Elementary OS





- **Jednoduchost a snadnost použití**: Grafický systém Elementary OS Interface je na půli cesty mezi systémy MacOs a Windows. Díky této známosti si jej snadno osvojí i nezkušení uživatelé.





- **Bezpečnost**: Stejně jako většina linuxových distribucí i Elementary OS těží z vysoké úrovně zabezpečení. Pravidelné aktualizace, správa práv a absence běžných virů z něj činí spolehlivý systém.





- **Rychlost**: Elementary OS je lehká distribuce. Vyžaduje málo prostředků, takže je rychlá a vhodná pro počítače se skromnými konfiguracemi.





- **Zdarma**: Systém je zcela zdarma. Při jeho stažení však můžete přispět na podporu vývojářů.





- **Aktivní komunita**: Komunita kolem Elementary OS je rozmanitá a vnímavá. Pokud narazíte na potíže, můžete snadno najít pomoc na fórech nebo sociálních sítích.



## Instalace a konfigurace


### Hardwarové předpoklady



Před zahájením instalace se ujistěte, že máte následující vybavení:





- Klíč **USB** o velikosti nejméně 12 GB
- **Paměť RAM** alespoň 4 GB
- **Disk Hard o velikosti 20 GB** nebo více pro pohodlné používání



## Stažení obrazu ISO



Přejděte na oficiální webové stránky operačního systému [elementární](https://elementary.io/) a vyberte částku na podporu projektu. Tento krok je nepovinný.


Pokud si přejete stáhnout obraz ISO zdarma, zadejte do pole **"Ostatní "** hodnotu 0 a spusťte stahování obrazu ISO systému.



![0_01](assets/fr/01.webp)



## Vytvoření spouštěcího klíče USB



Po stažení obrazu ISO jej musíte spustit na klíči USB, abyste mohli pokračovat v instalaci.



Stáhněte si software, například [Balena Etcher](https://etcher.balena.io/) nebo podobný nástroj, a spusťte jej.


Vyberte dříve stažený obraz ISO **Elementary OS** a jako cíl nastavte klíč USB.



Kliknutím na tlačítko **flash** spusťte proces a před vyjmutím klíče USB vyčkejte na jeho dokončení.



![0_02](assets/fr/02.webp)



## Zavedení systému a přístup do systému BIOS pomocí kláves



Vypněte počítač, do kterého chcete nainstalovat systém Elementary OS, a poté připojte klíč USB.


Po spuštění počítače přejděte do systému BIOS (`ESC`, `F9` nebo `F11` v závislosti na značce) a vyberte klíč USB jako spouštěcí zařízení, poté stiskněte `Enter` pro spuštění spouštěcího procesu.

## Aktualizace systému



Před delším používáním je důležité aktualizovat systém nejnovějšími záplatami.


### Možnost 1: Aktualizace prostřednictvím grafiky Interface



Vstupte do **AppCenter** a klikněte na ikonu aktualizace v pravém horním rohu. Počkejte, až se zobrazí seznam aktualizací, a kliknutím na **"Aktualizovat vše "** spusťte instalaci.



![0_20](assets/fr/20.webp)



![0_21](assets/fr/21.webp)



### Možnost 2: Aktualizace přes terminál



Aktualizaci můžete provést také z příkazového řádku prostřednictvím terminálu: doporučená možnost pro její přesnost.



```shell
sudo apt update
```



```shell
sudo apt full-upgrade
```



Při prvních aktualizacích vyžaduje operační systém vaše uživatelské heslo a potvrzení aktualizace softwarových balíčků, a to i v základním jádře.



## Konfigurace pracovního prostředí



Elementární operační systém obsahuje pouze základní nástroje. Chcete-li přizpůsobit prostředí svým potřebám, zejména pokud jste vývojář, doporučujeme nainstalovat další nástroje.





- Užitečné závislosti můžete přidat pomocí následujícího příkazu (upravte podle svých potřeb):



```shell
sudo apt update && sudo apt install -y git python3 python3-pip build-essential wget curl zsh make snapd && sudo snap install code --classic
```



Tento příkaz nainstaluje **Git**, **Python 3**, **pip**, **kompilátorové nástroje**, **wget**, **curl**, **zsh**, **make**, **snapd** a **vscode** a připraví základní vývojové prostředí.
