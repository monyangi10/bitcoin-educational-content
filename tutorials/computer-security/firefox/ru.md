## Введение



Мы все проводим часы в Интернете, часто не осознавая, что именно раскрывает о нас наш браузер. Каждый клик, каждый поиск, каждый сайт, который мы посещаем, питает огромную индустрию сбора персональных данных.



![Statistiques navigateurs 2024](assets/fr/01.webp)


*Доля рынка веб-браузеров: Доминирует Chrome с 65 % рынка, за ним следуют Safari и Edge. Источник: [gs.statcounter.com](https://gs.statcounter.com/browser-market-share)*



Как видно на этом графике, Google Chrome занимает доминирующее положение - более 65 % пользователей во всем мире. Эта гегемония означает, что большинство пользователей Интернета доверяют свои данные о просмотре сайтов компании Google, чья бизнес-модель основана на целевой рекламе. Firefox, занимающий всего 3 % рынка, представляет собой альтернативу, разработанную Mozilla, некоммерческой организацией, не заинтересованной в использовании ваших данных в коммерческих целях.



Но выбор Firefox - это только первый шаг. По умолчанию даже Firefox требует настроек для максимальной защиты. В этом руководстве вы шаг за шагом, от самого простого к самому продвинутому, сможете превратить Firefox в настоящий щит от слежки, сохранив при этом приятные впечатления от просмотра веб-страниц.



### Почему именно Firefox?





- Свободный и открытый исходный код** (движок Gecko): проверяемый, прозрачный код
- Некоммерческая организация**: Mozilla Foundation, миссия общего интереса
- Встроенные встроенные средства защиты**: Enhanced Tracking Protection (ETP), Total Cookie Protection (TCP), State Partitioning, режим HTTPS-only, DNS over HTTPS (DoH)
- Расширенная настройка**: в отличие от Chrome, Firefox позволяет глубоко изменять свое поведение



### Важные принципы перед началом работы





- Универсального рецепта нет**: чем больше вы вносите изменений, тем больше рискуете выделиться (отпечатки пальцев). Цель - быть лучше защищенным, не выделяясь из толпы.
- Пошаговое продвижение**: Измените настройки, протестируйте привычные сайты, а затем продолжите. Нет необходимости менять все сразу.
- Личный баланс**: Найдите свой компромисс между конфиденциальностью и простотой использования.

## Защиты уже активированы по умолчанию (обнадеживает)





- Изоляция сайтов (Fission)**: при прогрессивном развертывании. Эта функция запускает каждый сайт в отдельном процессе, чтобы предотвратить доступ одной вредоносной вкладки к данным другой. Проверьте ее статус в разделе `about:support` (поиск по слову "Fission"). Если функция не включена, вы можете вручную активировать ее в `about:config` с помощью `fission.autostart = true`.
- Total Cookie Protection (TCP)**: активна по умолчанию. Файлы cookie и другие хранилища хранятся только на сайте первой стороны (по одному "банку" на сайт), что нейтрализует межсайтовое отслеживание. При необходимости делаются временные исключения через Storage Access API (встроенные кнопки входа).
- Защита от отслеживания отказов/перенаправлений**: Firefox автоматически обнаруживает и очищает файлы cookie, оставленные на сайтах-отказниках (ссылки, которые перенаправляют вас через трекер до пункта назначения), сокращая этот канал отслеживания без каких-либо действий с вашей стороны.



## Уровень 1 - основной (≤ 10 минут)



Цель: значительное повышение уровня конфиденциальности без разрушения сети. Для 90% пользователей.



Чтобы перейти к настройкам, нажмите на меню ☰ в правом верхнем углу, затем **"Настройки "**:



![Paramètres généraux](assets/fr/07.webp)


*Страница настроек Firefox - вкладка "Общие". Здесь вы настраиваете большинство параметров конфиденциальности*



**Защита от слежения (ЭТП)**




- Переключите **ETP** на **Strict**. Вы блокируете больше трекеров (межсайтовые куки, отпечатки пальцев, криптомайнинг, социальные виджеты...).
- Если сайт ломается (видео, кнопка входа...), отключите защиту только для этого сайта через щит 🛡️, а затем обновите вкладку.



Вот различные уровни безопасности ЭТП:




- Стандарт** (сбалансированный, максимальная совместимость)
  - Блокирует: социальные трекеры, межсайтовые куки (все окна), отслеживание содержимого в приватном просмотре, криптовалютные майнеры, детекторы отпечатков пальцев.
  - Включает **Total Cookie Protection** (TCP): один кувшин на сайт.
- Строгий** (рекомендуется для обеспечения конфиденциальности)
  - Также блокирует следящее содержимое во всех окнах + известные и подозреваемые отпечатки пальцев.
  - Может нарушить работу некоторых сайтов; используйте щит 🛡️ для локального исключения.
- Пользовательский** (расширенный)
  - Тонкая настройка: куки, отслеживание содержимого, несовершеннолетние, отпечатки пальцев (известные/подозреваемые).



![Paramètres protection contre le pistage](assets/fr/06.webp)



**Куки и данные сайта




- Включите **"Удалять куки и данные сайта при закрытии "** для чистого перезапуска при каждом перезапуске.
- Добавьте **Исключения** для 2-3 важных сайтов, если хотите (электронная почта, банк).


**Автоматический ввод данных, предложения и домашняя страница**




- Отключите **автозаполнение** (идентификаторы, адреса, карты). Вместо этого используйте менеджер паролей.
- Поиск**: отключить **"Показывать предложения по поиску "**.
- Address bar**: вырезать **"Спонсорские предложения "** и **"Контекстные предложения "**.
- Главная**: отключите **Карман** и **спонсируемый контент**.



![Paramètres cookies et mots de passe](assets/fr/08.webp)



** Только по протоколу HTTPS**




- Активируйте **"Режим HTTPS только во всех окнах "**.


![Configuration DNS over HTTPS](assets/fr/09.webp)



**Телеметрия и рекламные измерения




- В разделе "Сбор данных Firefox" **снимите все флажки**.
- Отключите **"Меры по обеспечению конфиденциальности рекламы "** (PPA).
- Безопасный просмотр**: включите эту функцию (рекомендуется). Firefox проверяет сайты по спискам угроз с помощью хэшированных запросов и локальных проверок, защищая от фишинга и вредоносного ПО с минимальным влиянием на конфиденциальность.



**Глобальный контроль конфиденциальности (опция)**




- Активируйте **GPC**, чтобы сообщить о своем отказе от продажи/обмена данными.



** Поисковая система




- Переключитесь на **DuckDuckGo**, **Startpage**, **Qwant** или **Brave Search** (Настройки → Поиск).



![Configuration moteur de recherche DuckDuckGo](assets/fr/11.webp)



**Частная навигация




- Приватные окна (Ctrl/Cmd+Shift+P) для разовых сеансов (подарки, вторичные учетные записи...). Избегайте режима "всегда приватно": расширения могут быть неактивны, а исключения из cookie менее полезны.



**Основные расширения** (устанавливаются из официального каталога)




- uBlock Origin**: блокирует рекламу и текущее отслеживание, легкий.
- Privacy Badger**: учится блокировать то, что следует за вами; отправляет Do Not Track / GPC.
- ClearURLs** (необязательно): Firefox (ETP Strict) и uBO уже многое очищают; сохраните его, если вы все еще видите "грязные" URL (utm, fbclid).
- Firefox Multi-Account Containers**: **изолирует куки/сессии и хранилища для каждого контейнера; параллельное использование нескольких аккаунтов; меньше межсайтового отслеживания**. Официальное расширение: `https://addons.mozilla.org/fr/firefox/addon/multi-account-containers/`.



![Extension Multi-Account Containers](assets/fr/12.webp)



**Пароли и 2FA**




- Используйте специальный менеджер паролей** (Bitwarden, KeePassXC). **Избегайте** хранения паролей в браузере. **Включите 2FA** везде, где это возможно.



## Уровень 2 - Усиленный (компартментализация и сеть)



Цель: разделение деятельности и уменьшение утечки информации в сеть.



**DNS over HTTPS (DoH)**




- Состояние по умолчанию**: Автоматически активируется в некоторых регионах (США, Канада, Россия, Украина). В других регионах требуется ручная активация.
- Конфигурация**: Настройки → Общие → Настройки сети → **Включить DoH** → **Cloudflare** или **Quad9** → **Максимальная защита**.
- Максимальная защита = только TRR** (без возврата к системному DNS). Если корпоративная сеть/отель блокирует, переключитесь обратно на **Стандарт** или отключите DoH.
- Избыточность**: Если вы уже используете надежную VPN с собственным безопасным DNS, DoH может оказаться лишним.
- Тест на верификацию**: `https://www.dnsleaktest.com/` должен отображать только выбранного провайдера DoH.



![Sélection fournisseur DNS Cloudflare](assets/fr/10.webp)



** Разделение с помощью контейнеров и профилей




- Контейнеры для нескольких аккаунтов**: создавайте пространства (Личное, Рабочее, Финансовое, Социальные сети, Покупки, Одноразовое). Настройте **"Всегда открывать в этом контейнере "** для повторяющихся сайтов. Официальное расширение: `https://addons.mozilla.org/fr/firefox/addon/multi-account-containers/`.
- Зачем их использовать?
  - Строгая изоляция** файлов cookie/сессий/хранилищ по пространству.
  - Меньше межсайтового отслеживания**: ограничьтесь гигантами (Facebook, Google).
  - Одновременное использование нескольких учетных записей** на одном сервисе.
  - Меньше риска CSRF/XSS** между сегментированными идентификаторами.
  - Совет: как минимум, выделите контейнеры для социальных сетей/Google, работы, финансов.
- Facebook Container** (опционально): упрощенная версия, предназначенная для FB/Instagram.
- Раздельные профили**: через `about:profiles` (основной профиль, минимальный "ультрабезопасный" профиль, тестовый профиль). Полное разделение данных и расширений.



**Дополнительные расширения** (будет зарезервировано)




- Cookie AutoDelete**: удаляет файлы cookie сайта, как только вкладка закрывается (полезно, если Firefox открыт долгое время).
- LocalCDN**: обслуживает текущие библиотеки локально (сокращает обращения к Google/Microsoft). Частичная совместимость.



**Мобильный (Android)**




- Firefox Android + uBlock Origin**: аналогичная защита в пути.



## Уровень 3 - Эксперт (about:config & Arkenfox)



Цель: продвинутая закалка с допустимыми компромиссами. Рекомендуется на **отдельном профиле**.



Выберите только один из двух следующих подходов:



**Подход A - ручные изменения**: Несколько целевых настроек через `about:config` (более простой и точный контроль)


**Прием B - Arkenfox user.js**: Полная автоматическая настройка (более сложная, максимальная защита)



➡️ **Arkenfox уже включает в себя ВСЕ изменения в about:config, упомянутые ниже** + сотни других. Если вы выбрали Arkenfox, игнорируйте раздел about:config.



### Подход A: Внесение изменений вручную через about:config



Введите `about:config` в строке Address → Примите риск.



![Avertissement about:config](assets/fr/13.webp)



![Interface about:config](assets/fr/14.webp)



![Préférences about:config](assets/fr/15.webp)





- Устойчивость к отпечаткам пальцев (унаследовано от Tor Browser)


```text
privacy.resistFingerprinting = true
```


Эффекты: Часовой пояс UTC, **letterboxing** (стандартные размеры окон), стандартизированные User-Agent/политики, ограничения Canvas/WebGL/AudioContext. Больше единообразия, но несколько "причуд" (смещенное время, иногда немного английского).





- Отключите WebRTC (предотвращает утечку IP-адресов; ломает Web visio)


```text
media.peerconnection.enabled = false
```





- Referer plus compatible (по умолчанию)


```text
network.http.referer.XOriginPolicy = 1
network.http.referer.trimOnCrossOrigin = true
```


Строгий вариант (может нарушить платежи/SSO):


```text
network.http.referer.XOriginPolicy = 2
```





- Ограничение болтливых API и спекуляций


```text
dom.battery.enabled = false
device.sensors.enabled = false
beacon.enabled = false
geo.enabled = false
media.navigator.enabled = false
network.prefetch-next = false
browser.urlbar.speculativeConnect.enabled = false
network.http.speculative-parallel-limit = 0
```



Золотое правило: если что-то сломалось, вернитесь к последнему изменению.



### Подход B: Arkenfox user.js (полностью автоматизированная настройка)



Проект **Arkenfox** предоставляет поддерживаемый сообществом файл `user.js`, который автоматически применяет сотни настроек Firefox, ориентированных на конфиденциальность и безопасность. При перезапуске Firefox считывает этот файл из вашего профиля и применяет эти настройки.





- В чем смысл? Начните с **последовательной базы** без "щелчков повсюду"; сократите количество ошибок; сэкономьте время.
- Что это меняет (примеры): сокращение телеметрии, усиление cookies/cache/referrer/HTTPS, **RFP** + letterboxing, **WebRTC отключен**, регулировка DoH/TLS, ограничение чатти API.
- Когда его использовать: если вам нужен Firefox, упрочненный за 10 минут и допускающий несколько исключений (DRM/streaming, Web visio, SSO/payments).
- Преимущества: быстрый, последовательный, **обновляемый** (ESR-aligned), очень хорошо **документированный** (wiki + комментарии), **настраиваемый** с помощью переопределений.
- Ограничения: совместимость (некоторые веб-приложения), удобство (UTC, размеры окон), и напоминание: **это не Tor** (нет сетевой анонимности).



Установка (в идеале на **выделенный профиль**)


1. Сохраните профиль/избранные сайты и список сайтов с исключениями из cookie.


2. Загрузите `user.js` с сайта `https://github.com/arkenfox/user.js` (ESR/стабильная версия).


3. Найдите папку своего профиля через `about:profiles`:




   - Windows: `%APPDATA%/Mozilla/Firefox/Profiles/...`
   - Linux: `~/.mozilla/firefox/...`
   - macOS: `~/Library/Application Support/Firefox/Profiles/...`


4. Закройте Firefox и переместите файл `user.js` в корень папки профиля.


5. Перезапуск; настройка через `about:config` или файл переопределений.



Обновления




- Следуйте релизам Arkenfox (выровненным по ESR), замените `user.js`, перезапустите Firefox; прочитайте примечания к выпуску.



**Настройка с помощью переопределений**



По умолчанию Arkenfox намеренно ограничен. Чтобы адаптировать некоторые настройки под свои нужды (потоковое вещание, visio, определенные сайты), вы можете создать файл `user-overrides.js` в той же папке, что и `user.js`. Этот файл позволяет "отменить" определенные настройки Arkenfox, не изменяя основной файл.



Создайте файл `user-overrides.js` и добавьте в него свои настройки:


```javascript
// DRM/streaming
user_pref("media.eme.enabled", true);

// Safe Browsing (si vous préférez le garder)
user_pref("browser.safebrowsing.phishing.enabled", true);
user_pref("browser.safebrowsing.malware.enabled", true);

// Historique moins restrictif
user_pref("places.history.expiration.max_pages", 200000);

// Synchronisation Firefox
user_pref("identity.fxaccounts.enabled", true);

// WebRTC (si visio nécessaire)
user_pref("media.peerconnection.enabled", true);

// Referer plus compatible
user_pref("network.http.referer.XOriginPolicy", 1);
user_pref("network.http.referer.trimOnCrossOrigin", true);
```



Лучшие практики




- Используйте отдельный профиль **"Arkenfox "** и сохраните "обычный" профиль для удобства.
- Минимизируйте расширения (uBlock Origin OK), чтобы ограничить площадь атаки и уникальность.
- При необходимости добавьте исключения для каждого сайта (защита 🛡️, uBO, NoScript, если используется).
- Тестируйте после каждого изменения: Утечки WebRTC/DNS, заметание следов, CreepJS.



## Лучшие практики





- Обновления**: Firefox и расширения обновлены.
- Продление**: разумно и надежно; остерегайтесь "сомнительных" погашений.
- Загрузки**: осторожно; проверьте чувствительные файлы на VirusTotal.
- Пароли**: **выделенный менеджер** (Bitwarden, KeePassXC); **2FA** включен; избегайте хранения в браузере.
- Гигиена**: ограничьте Google/Facebook контейнерами; регулярно закрывайте/открывайте их, чтобы "сбросить" контекст.



## Ограничения и альтернативы





- Усиленный браузер ≠ анонимность в сети: без **VPN** ваш IP остается видимым; даже с ним корреляция остается возможной.
- Слишком сильные изменения могут сделать вас **уникальным**. **RFP** стандартизирует; инструменты рандомизации (например, Chameleon) могут... выделить вас из толпы. Тестируйте, сравнивайте, корректируйте.
- Альтернативы/дополнения:
 - Tor Browser: анонимность в сети через Tor; медленнее. См. наше полное руководство по установке и настройке**:



https://planb.network/tutorials/computer-security/communication/tor-browser-a847e83c-31ef-4439-9eac-742b255129bb



 - Браузер Mullvad: "Tor без Tor", сочетающийся с VPN; стандартизированный след. Узнайте, как установить его в нашем специальном руководстве**:



https://planb.network/tutorials/computer-security/communication/mullvad-browser-a16c13d6-8bf9-4cb5-9aa0-85411a9cda0e



- Рекомендуемые комбинации: Firefox (уровень 2) + VPN для повседневного использования; Tor/Mullvad для конфиденциальной деятельности; отдельные профили для разделения.



## Заключение



Следуя этому пошаговому руководству, вы превратите Firefox в настоящий оплот против цифровой слежки. От основных настроек первого уровня до продвинутых конфигураций Arkenfox - каждое изменение укрепляет вашу конфиденциальность без ущерба для работы в браузере.



**Ваша конфиденциальность теперь лучше защищена**: рекламные трекеры заблокированы, файлы cookie разделены, утечки IP Address нейтрализованы, телеметрия отключена. Firefox больше не просто браузер, а инструмент цифровой защиты, созданный с учетом ваших потребностей.



**Помните: конфиденциальность никогда не является само собой разумеющимся фактом. Регулярно проверяйте свою защиту, обновляйте настройки и не стесняйтесь корректировать конфигурацию по мере изменения ваших привычек. Ваша анонимность в Интернете зависит не только от ваших инструментов, но и от вашей практики.
