Keet เป็นแอปพลิเคชันส่งข้อความโต้ตอบแบบทันทีที่ออกแบบมาให้ทำงานโดยไม่ต้องใช้เซิร์ฟเวอร์ เปิดตัวในปี 2022 โดย Holepunch (บริษัทที่ได้รับการสนับสนุนทางการเงินจาก Tether และ Bitfinex) แอปพลิเคชันนี้มีพื้นฐานอยู่บนเครือข่ายแบบเพียร์ทูเพียร์ทั้งหมดและมีแนวทางการกระจายศูนย์อย่างสุดขั้ว: ข้อความ, การโทร และไฟล์ถูกแลกเปลี่ยนโดยตรงระหว่างผู้ใช้ โดยไม่มีตัวกลางใดๆ



Keet เข้ารหัสการสื่อสารทั้งหมดแบบ end-to-end และไม่ขอข้อมูลส่วนบุคคล การลงทะเบียนเป็นแบบไม่ระบุตัวตน โดยไม่ต้องใช้หมายเลขโทรศัพท์หรืออีเมล นอกจากการแลกเปลี่ยนข้อความแล้ว Keet ยังมีการโทรวิดีโอคุณภาพสูงมาก รวมถึงการแชร์ไฟล์ได้ไม่จำกัด แอปพลิเคชันนี้จึงสามารถใช้งานได้ทั้งในรูปแบบส่วนตัวและมืออาชีพ



![Image](assets/fr/01.webp)



ณ ขณะนี้ (เมษายน 2025) Keet ยังไม่เป็นโอเพ่นซอร์สทั้งหมด โค้ดบางส่วนสามารถเข้าถึงได้ใน [GitHub repository ของ Holepunch](https://github.com/holepunchto) โดยเฉพาะส่วนประกอบด้านการเข้ารหัสและเครือข่าย แต่ไคลเอนต์ Interface ยังไม่เปิดให้ใช้งาน อย่างไรก็ตาม Holepunch ได้ประกาศเจตนารมณ์ที่จะทำให้แอปพลิเคชันทั้งหมดเป็นโอเพ่นซอร์สในที่สุด ขึ้นอยู่กับว่าคุณค้นพบบทแนะนำนี้เมื่อใด สิ่งนี้อาจได้ทำไปแล้วก็ได้

## ติดตั้ง Keet



Keet พร้อมใช้งานบนทุกแพลตฟอร์ม คุณสามารถดาวน์โหลดแอปพลิเคชันได้โดยตรงจากร้านค้าแอปบนโทรศัพท์ของคุณ:




- [Google Play](https://play.google.com/store/apps/details?id=io.keet.app&pli=1);
- [App Store](https://apps.apple.com/us/app/keet-by-holepunch/id6443880549);



บน Android ก็สามารถ [ติดตั้งผ่าน APK](https://github.com/holepunchto/keet-mobile-releases/releases) ได้เช่นกัน



ในบทแนะนำนี้ เราจะมุ่งเน้นไปที่เวอร์ชันมือถือ แต่โปรดทราบว่า [เวอร์ชันคอมพิวเตอร์ก็มีให้บริการเช่นกัน](https://keet.io/) (MacOS, Linux และ Windows) นอกจากนี้ยังสามารถซิงโครไนซ์บัญชีของคุณบนอุปกรณ์หลายเครื่องได้อีกด้วย



## สร้างบัญชีบน Keet



เมื่อเปิดใช้งานครั้งแรก คุณสามารถละเว้นหน้าจอการนำเสนอได้



![Image](assets/fr/02.webp)



คลิกที่ปุ่ม "*ฉันเป็นผู้ใช้ใหม่*"



![Image](assets/fr/03.webp)



ยอมรับเงื่อนไขการใช้งาน จากนั้นคลิกที่ "*การตั้งค่าอย่างรวดเร็ว*"



![Image](assets/fr/04.webp)



เลือกชื่อหรือชื่อเล่น จากนั้นคลิกที่ "*เสร็จสิ้นการตั้งค่า*".



![Image](assets/fr/05.webp)



โปรไฟล์ของคุณถูกสร้างขึ้นแล้ว คลิก "*Finish setup*" อีกครั้งเพื่อเสร็จสิ้นการตั้งค่า



![Image](assets/fr/06.webp)



คุณสามารถปรับแต่งโปรไฟล์ของคุณได้ตลอดเวลาจากแท็บ "*โปรไฟล์*"



## บันทึกบัญชี Keet ของคุณ



สิ่งแรกที่ต้องทำกับบัญชี Keet ใหม่ของคุณคือการบันทึกวลีการกู้คืนของคุณ นี่คือชุดคำ 24 คำที่จะช่วยให้คุณสามารถกู้คืนการเข้าถึงบัญชีของคุณในกรณีที่สูญหายหรือเปลี่ยนอุปกรณ์ วลีนี้ให้การเข้าถึงบัญชีของคุณอย่างเต็มที่แก่ใครก็ตามที่รู้ ดังนั้นจึงเป็นสิ่งสำคัญที่จะต้องทำการสำรองข้อมูลที่เชื่อถือได้และไม่เปิดเผยวลีนี้แก่ผู้อื่น



ในการทำเช่นนี้ ให้คลิกที่แท็บ "*Profile*" ที่มุมล่างขวาของ Interface



![Image](assets/fr/07.webp)



จากนั้นเข้าถึงเมนู "*Settings*"



![Image](assets/fr/08.webp)



เลือก "*ความเป็นส่วนตัวและความปลอดภัย*"



![Image](assets/fr/09.webp)



จากนั้นคลิกที่ "*Recovery phrase*"



![Image](assets/fr/10.webp)



กดปุ่ม "*ดูวลี*" เพื่อแสดงวลีการกู้คืนของคุณ คัดลอกอย่างระมัดระวังและเก็บไว้ในที่ปลอดภัย



![Image](assets/fr/11.webp)



## ส่งข้อความด้วย Keet



ในการสื่อสารบน Keet คุณจำเป็นต้องสร้าง "*Rooms*" เพื่อทำสิ่งนี้ ให้คลิกที่ไอคอนรูปดินสอที่มุมขวาบนของ Interface



![Image](assets/fr/12.webp)



เลือก "*แชทกลุ่มใหม่*"



![Image](assets/fr/13.webp)



เลือกชื่อและคำอธิบายสำหรับ "*ห้อง*" ของคุณ จากนั้นคลิกที่ "*สร้างกลุ่มแชท*".



![Image](assets/fr/14.webp)



"*Room*" ของคุณถูกสร้างขึ้นแล้ว คลิกที่ไอคอน "*+*" ที่มุมขวาบนเพื่อเชิญผู้เข้าร่วม



![Image](assets/fr/15.webp)



กำหนดสิทธิ์ที่คุณมอบให้กับสมาชิกใหม่ผ่านลิงก์เชิญ รวมถึงระยะเวลาที่ลิงก์มีผลบังคับใช้ จากนั้นคลิกที่ "*generate invite*"



![Image](assets/fr/16.webp)



Keet จะ generate ลิงก์เพื่อเข้าร่วม "*Room*" ของคุณ คุณสามารถคัดลอกและแชร์ลิงก์นั้น หรือให้คนที่คุณต้องการเชิญสแกน QR code ของลิงก์นั้นก็ได้



![Image](assets/fr/17.webp)



คุณสามารถเริ่มแลกเปลี่ยนข้อความและไฟล์มัลติมีเดียได้แล้ว หากต้องการเริ่มการโทร ให้คลิกที่ไอคอนโทรศัพท์ที่มุมขวาบน



![Image](assets/fr/18.webp)



จากกลุ่มนี้ คุณยังสามารถส่งข้อความส่วนตัวไปยังสมาชิกเฉพาะได้อีกด้วย คลิกที่รูปโปรไฟล์ของกลุ่ม จากนั้นเลือกสมาชิกที่ต้องการในส่วน "*สมาชิก*"



![Image](assets/fr/19.webp)



คลิกที่ปุ่ม "*ส่งคำขอ DM*" และป้อนข้อความของคุณ



![Image](assets/fr/20.webp)



เมื่อคำขอ DM ได้รับการยอมรับแล้ว คุณจะพบผู้ติดต่อคนนี้บนหน้าแรก ซึ่งคุณสามารถพูดคุยกับเขาเป็นการส่วนตัวได้



![Image](assets/fr/21.webp)



## ซิงโครไนซ์บัญชีของคุณบนอุปกรณ์หลายเครื่อง



ตอนนี้ที่คุณรู้วิธีใช้ Keet และมีบัญชีแล้ว คุณยังสามารถซิงโครไนซ์มันบนอุปกรณ์อื่น เช่น คอมพิวเตอร์ได้อีกด้วย ในการทำเช่นนี้ ให้เปิดแอปพลิเคชันบนมือถือของคุณ จากนั้นคลิกที่ "*โปรไฟล์*" และเข้าถึง "*การตั้งค่า*"



![Image](assets/fr/22.webp)



จากนั้นไปที่เมนู "*อุปกรณ์ของฉัน*"



![Image](assets/fr/23.webp)



คลิกที่ "*เพิ่มอุปกรณ์*". Keet จะ generate ลิงก์เพื่อซิงโครไนซ์อุปกรณ์ใหม่ คัดลอกลิงก์นี้



![Image](assets/fr/24.webp)



บนอุปกรณ์ที่สองของคุณ ติดตั้ง Keet บนหน้าจอหลัก เลือกตัวเลือก "*ฉันเป็นผู้ใช้ปัจจุบัน*"



![Image](assets/fr/25.webp)



จากนั้นคลิกที่ "*เชื่อมโยงอุปกรณ์*".



![Image](assets/fr/26.webp)



วางลิงก์ที่ได้รับจากอุปกรณ์แรกของคุณ จากนั้นคลิกที่ "*เริ่มการซิงค์*".



![Image](assets/fr/27.webp)



บนอุปกรณ์แรกของคุณ คลิกที่ "*ยืนยันการเชื่อมโยงอุปกรณ์*" เพื่ออนุญาตการเชื่อมต่อ



![Image](assets/fr/28.webp)



บนอุปกรณ์ที่สอง ให้ทำกระบวนการให้เสร็จสิ้นโดยคลิกที่ "*เชื่อมโยงอุปกรณ์*".



![Image](assets/fr/29.webp)



คุณสามารถเข้าถึง "*Room*" และการสนทนาทั้งหมดของคุณจากอุปกรณ์ใหม่นี้ได้แล้ว



![Image](assets/fr/30.webp)
