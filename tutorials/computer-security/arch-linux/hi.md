आर्क लिनक्स एक ऐसा वितरण है जो अपनी मज़बूती, प्रदर्शन और अनुकूलनशीलता के लिए जाना जाता है, खासकर विकास उद्देश्यों के लिए। यह उत्कृष्ट स्थिरता और अनुकूलन के लिए अनुकूल वातावरण प्रदान करता है, जो एक अत्यंत तेज़ और विश्वसनीय पैकेज मैनेजर द्वारा समर्थित है। इसका दर्शन **KISS** (*Keep It Simple, Stupid*) सिद्धांत पर आधारित है: एक हल्का, सरल, तेज़ और सुव्यवस्थित वितरण प्रदान करना, जबकि उपयोगकर्ता को पर्याप्त स्वतंत्रता प्रदान करना।



## आर्क लिनक्स क्यों चुनें?





- मुफ़्त और खुला स्रोत**: ज़्यादातर लिनक्स वितरणों की तरह, आर्क लिनक्स भी पूरी तरह मुफ़्त है। इसमें कोई लाइसेंस शुल्क नहीं है, जो इसे छात्रों, फ्रीलांसरों या उत्साही लोगों के लिए एक बेहतरीन विकल्प बनाता है।
- KISS** का दर्शन: आर्क को सरल, हल्का और कुशल बनाने के लिए डिज़ाइन किया गया है। यह केवल आवश्यक सुविधाएँ प्रदान करता है, जिससे आप अपनी पसंद का वातावरण बना सकते हैं।
- पैकमैन** पैकेज मैनेजर: पैकमैन एक तेज़, विश्वसनीय और अच्छी तरह से डिज़ाइन किया गया पैकेज मैनेजर है। यह सॉफ़्टवेयर की कुशल स्थापना और अद्यतन को सक्षम बनाता है, और निर्भरताओं को सटीकता से प्रबंधित करता है।
- व्यापक दस्तावेज़ीकरण और एक सक्रिय समुदाय**: [आर्क विकी](https://wiki.archlinux.org) शायद लिनक्स की दुनिया के सर्वश्रेष्ठ तकनीकी दस्तावेज़ीकरणों में से एक है। यह समझने के लिए एक सोने की खान है कि आप क्या कर रहे हैं। यह समुदाय, जो ज़्यादातर अनुभवी लोगों से बना है, बहुत सक्रिय है और अगर आप किसी समस्या में फँस जाते हैं तो आपकी मदद कर सकता है, बशर्ते आपने पहले से थोड़ी खोजबीन कर ली हो।

## आर्क लिनक्स स्थापित करना



## USB कुंजी पर बूट करना





- अपना कंप्यूटर पूरी तरह से बंद कर दें
- बूट करने योग्य USB कुंजी प्लग इन करें
- रीबूट करें और **F1**, **Escape**, **F9**, आदि दबाकर BIOS/UEFI में प्रवेश करें (आपके मॉडल के आधार पर)
- बूट मेनू में, बूट डिवाइस के रूप में USB कुंजी चुनें। अगर सब कुछ सही तरीके से सेट हो गया है, तो आपको Arch Linux Interface बूट स्क्रीन पर ले जाया जाएगा।



## स्थापना प्रारंभ करना

### कीबोर्ड कॉन्फ़िगरेशन



आप उपलब्ध लेआउट को निम्न के साथ प्रदर्शित कर सकते हैं:



```shell
localectl list-keymaps
```



![0_6](assets/fr/06.webp)



फिर एक लेआउट लोड करें:



```shell
loadkeys nom-disposition
```



डिफ़ॉल्ट रूप से, कीबोर्ड अंग्रेजी (qwerty) में है, लेकिन आप `loadkeys fr` के साथ `azarty` पर स्विच कर सकते हैं।



### दिनांक और समय निर्धारित करना



आर्क लिनक्स सिस्टम घड़ी को प्रबंधित करने के लिए `timedatectl` टूल का उपयोग करता है।



![0_7](assets/fr/07.webp)





- अपना समय क्षेत्र निर्धारित करें:


```shell
timedatectl set-timezone Europe/Paris
```





- जाँच करें कि स्वचालित सिंक्रनाइज़ेशन इसके साथ सक्षम है:


```shell
timedatectl status
```





- यदि आवश्यक हो तो इसे सक्रिय करें:


```shell
timedatectl set-ntp true
```




यह NTP को सक्रिय करता है, जो समय सर्वरों के साथ स्वचालित सिंक्रनाइज़ेशन के लिए एक प्रोटोकॉल है। पैकेज इंस्टॉल करते समय या बाद में SSL प्रमाणपत्र कॉन्फ़िगर करते समय दिनांक त्रुटियों से बचने के लिए यह चरण महत्वपूर्ण है।



### डिस्क विभाजन





- जांचें कि आपका सिस्टम **UEFI** या **BIOS** में बूट होता है या नहीं:



```shell
ls /sys/firmware/efi
```



यदि फ़ाइल मौजूद है, तो आप **UEFI** में हैं। अन्यथा, आप **BIOS (विरासत)** में हैं।




- उपलब्ध डिस्क की सूची बनाएं:


```shell
lsblk
```



![0_8](assets/fr/08.webp)





- विभाजन प्रबंधक प्रारंभ करें:



```shell
cfdisk /dev/nom-du-disque
```



यदि आप UEFI में हैं तो **GPT** चुनें, यदि आप BIOS में हैं तो **DOS** चुनें।



![0_9](assets/fr/09.webp)



#### बनाने के लिए स्कोर




- UEFI** मोड में



| Point de montage sur le système installé | Partition                 | Type de partition       | Taille suggérée |
| ---------------------------------------- | ------------------------- | ----------------------- | --------------- |
| /boot1                                   | /dev/efi_system_partition | Partition système EFI   | 1 Go            |
| [SWAP]                                   | /dev/swap_partition       | Espace d’échange (swap) | Au moins 4 Go   |
| /                                        | /dev/root_partition       | Racine Linux x86-64 (/) | Reste du disque |

- BIOS में



| Point de montage sur le système installé | Partition           | Type de partition       | Taille suggérée |
| ---------------------------------------- | ------------------- | ----------------------- | --------------- |
| [SWAP]                                   | /dev/swap_partition | Espace d’échange (swap) | Au moins 4 Go   |
| /                                        | /dev/root_partition | Linux                   | Reste du disque |

### विभाजनों को स्वरूपित करना





- यूईएफआई**:



```shell
mkfs.fat -F32 /dev/sda1
mkswap /dev/sda2
swapon /dev/sda2
mkfs.ext4 /dev/sda3
```





- BIOS**:



```shell
mkswap /dev/sda1
swapon /dev/sda1
mkfs.ext4 /dev/sda2
```

### बुनियादी प्रणाली स्थापना



**रूट** विभाजन को माउंट करें:





- BIOS पर:


```shell
mount /dev/sda2 /mnt
```




- UEFI पर:


```shell
mount /dev/sda3 /mnt
```



फिर आवश्यक पैकेज स्थापित करें:



```shell
pacstrap -K /mnt base linux linux-firmware
```



![0_12](assets/fr/12.webp)



generate **fstab** फ़ाइल, जो ऑपरेटिंग सिस्टम को प्रत्येक बूट पर, बिना किसी मैन्युअल हस्तक्षेप के, स्वचालित रूप से विभाजन माउंटिंग का प्रबंधन करने में सक्षम बनाती है:



```shell
genfstab -U /mnt >> /mnt/etc/fstab
```



**Chroot** वातावरण में प्रवेश करें:



```shell
arch-chroot /mnt
```

- रूट पासवर्ड सेट करें:



```shell
passwd
```

### GRUB स्थापित करना



स्थापित करें:



```shell
pacman -S grub
```



![0_16](assets/fr/16.webp)



एक बार डाउनलोड हो जाने के बाद, आपको इसे डिस्क विभाजन प्रारूप के अनुसार स्थापित करना होगा।




- **BIOS** के लिए:



```shell
grub-install --target=i386-pc /dev/sda
grub-mkconfig -o /boot/grub/grub.cfg
```



![0_17](assets/fr/17.webp)





- **UEFI** के लिए:



```shell
pacman -S efibootmgr
mkdir /boot/efi
mount /dev/sda1 /boot/efi
grub-install --target=x86_64-efi --efi-directory=/boot/efi --bootloader-id=GRUB
grub-mkconfig -o /boot/grub/grub.cfg
```

## रीबूट के बाद नेटवर्क कनेक्शन



ऐसा हो सकता है कि रीबूट करने पर कोई नेटवर्क कनेक्शन सक्रिय न हो। आप इंटरफ़ेस को इस प्रकार सूचीबद्ध कर सकते हैं:



```shell
ip link
```



फिर टर्मिनल में निम्नलिखित पाठ दर्ज करके Interface नेटवर्क को कॉन्फ़िगर करें



```shell
cat <<EOF > /etc/systemd/network/20-wired.network
[Match]
Name=nom_de_l_interface

[Network]
DHCP=yes
EOF
```



## Interface ग्राफ़िक्स (GNOME)



डिफ़ॉल्ट रूप से, **आर्क लिनक्स** में कोई ग्राफ़िकल Interface नहीं होता। इसे जोड़ने के लिए:



सिस्टम को अपडेट करें:



```shell
pacman -Syu
```



निम्नलिखित कमांड के साथ **Xorg** स्थापित करें और डिफ़ॉल्ट विकल्पों को बनाए रखने के लिए हर बार एंटर दबाएं:



```shell
pacman -S xorg
```



![0_19](assets/fr/19.webp)



निम्नलिखित कमांड के साथ **GNOME** स्थापित करें और डिफ़ॉल्ट विकल्पों को बनाए रखने के लिए हर बार एंटर करें:



```shell
pacman -S gnome gnome-extra
```



![0_20](assets/fr/20.webp)



**सत्र प्रबंधक** को सक्रिय करें:



```shell
systemctl enable gdm
systemctl start gdm
```



सिस्टम अपने आप रीबूट हो जाएगा और आपको Interface ग्राफ़िकल लॉगिन मिल जाएगा। रूट यूज़रनेम और पासवर्ड से लॉग इन करें।



![0_21](assets/fr/21.webp)



## उपयोगकर्ता बनाना



**Interface GNOME** में प्रवेश करने के बाद, आपको अधिक सुरक्षा और सुरक्षित, जोखिम-मुक्त उपयोग के लिए एक नया उपयोगकर्ता बनाना होगा। एप्लिकेशन दर्ज करें और टर्मिनल लॉन्च करने के लिए "कंसोल" विकल्प चुनें।





- उपयोगकर्ता जोड़ें:



```shell
useradd -m -G wheel -s /bin/bash nom_utilisateur
passwd nom_utilisateur
```



![0_22](assets/fr/22.webp)





- **sudo** स्थापित करें:


```shell
pacman -S sudo
```





- **sudo** अधिकार सक्रिय करें:



```shell
EDITOR=nano visudo
```





- फिर पंक्ति को अनकमेंट करें:



```shell
%wheel ALL=(ALL:ALL) ALL
```





- सिस्टम को पुनः आरंभ करें और अपने उपयोगकर्ता नाम से लॉग इन करें।



![0_23](assets/fr/23.webp)

## सॉफ़्टवेयर स्थापित करना



चूँकि Arch Linux न्यूनतम है, इसलिए बहुत सारे सॉफ़्टवेयर डिफ़ॉल्ट रूप से इंस्टॉल नहीं होते। अपनी ज़रूरत के अनुसार सॉफ़्टवेयर जोड़ने के लिए, निम्न कमांड टाइप करें:



```shell
pacman -S nom_du_paquet_a_installe
```



उदाहरण के लिए, **नैनो** टेक्स्ट एडिटर स्थापित करने के लिए, आप टाइप कर सकते हैं:



```shell
pacman -S nano
```



`फ़ायरफ़ॉक्स` जैसे हल्के वेब ब्राउज़र को स्थापित करने के लिए, उपयोग करें:



```shell
pacman -S firefox
```



अंत में, यदि आप `net-tools` जैसे आवश्यक नेटवर्क उपकरण जोड़ना चाहते हैं, तो कमांड होगा:



```shell
pacman -S net-tools
```



यह न भूलें कि आप एक ही कमांड में कई पैकेजों को अलग-अलग सूचीबद्ध करके उन्हें स्थापित कर सकते हैं:



```shell
pacman -S vim firefox net-tools
```



आर्क लिनक्स अपनी उल्लेखनीय स्थिरता, न्यूनतम दर्शन और मज़बूती के लिए जाना जाता है, जो इसे विकास परिवेशों के लिए एक आदर्श विकल्प बनाता है। केवल आवश्यक सुविधाएँ प्रदान करके, यह एक हल्का, उच्च-प्रदर्शन आधार प्रदान करता है जिसे आपकी विशिष्ट आवश्यकताओं के अनुसार आसानी से अनुकूलित किया जा सकता है। यह न्यूनतम दृष्टिकोण सिस्टम पर बेहतर नियंत्रण को भी बढ़ावा देता है, जिससे हमले की संभावना सीमित करके सुरक्षा को मज़बूत किया जा सकता है। अपने सक्रिय समुदाय और विस्तृत दस्तावेज़ीकरण के साथ, आर्क लिनक्स आपको व्यावसायिक विकास के लिए अनुकूलित एक सुरक्षित, लचीला परिवेश बनाने में मदद कर सकता है।



यदि आपको आर्क लिनक्स के साथ शुरुआत करने में आनंद आया है, तो आपको **फेडोरा ओएस** पर हमारा ट्यूटोरियल पसंद आएगा, जो एक मॉड्यूलर, सुरक्षित और मजबूत ऑपरेटिंग सिस्टम है जो आपकी आवश्यकताओं और उपयोगों के अनुकूल है।
