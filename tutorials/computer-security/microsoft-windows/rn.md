## Ukwikoresha!


Kuri iyi ntambwe, turakeneye guhindura uburyo bwo gushiramwo Windows busanzwe. Muri iyi ntambwe, dukoresheje Unattended install, turamenya ibintu bizoshirwamwo, ataco umukoresha yinjije inyuma. Nkako, muri ubwo buryo, dosiye ya XML irakoreshwa mu gutunganya intambwe zo gushiramwo n’ibikorwa vyashizwe muri Windows. Mu yandi majambo, gukoresha dosiye Unattended.xml bituma habaho uburyo bwo kwikoresha mu gihe co gushiramwo, bikabuza guhitamwo uburyo bwinshi no kwirinda intambwe ziruhije zisanzwe zikenewe mu gihe co gushiramwo. Ubu buryo ni uburyo budasanzwe ariko busanzwe bwashizweho na Microsoft. Ibindi bisobanuro biraboneka kuri [urubuga rwemewe rwa Microsoft](https://kwiga.microsoft.com/ru-us/amadirisha-ibikoresho/uguhingura/ibiro/guhindura-amadirisha-n'inyandiko-gukora-dosiye-yawe-inyishu-idirisha-sxs=view1).


Hariho ibikoresho bitandukanye biri kuri internet vyo gukora amadosiye atagira uwuyicungera. Bamwe muri bo bari kuri internet, abandi na bo ntibari kuri internet. Kimwe mu bikoresho vyo kuri internet vyo gukora iyi dosiye ni [uru rubuga](https://schneegans.de/amadirisha/umuhinguzi-utagira uwuwucungera). Tumaze kuyifungura, turaronka urupapuro rukurikira:


![Image](assets/en/06.webp)


Nk’uko twabibonye hejuru kuri iyo paji, ubu buryo burashobora gukoreshwa mu gushiramwo Windows 10 na 11. Mu ntambwe ya mbere, duhitamwo ururimi rwa Windows. Niba dukeneye kwongerako ururimi rwa kabiri canke mbere rwa gatatu ku rutonde rw’indimi zikoreshwa kuri Windows n’izikoreshwa kuri klavye, turashobora gukoresha akazu kari aha hepfo:


![Image](assets/en/07.webp)


Mu ntambwe ikurikira, turahitamwo aho twipfuza.


![Image](assets/en/08.webp)


Kuri iyi ntambwe, turashobora kandi gusobanura ubuhinga bwa processeur ya mudasobwa. Muri iyi ntambwe, turashobora:

1. Fata ingingo yo kwirengagiza ibintu vy’umutekano vya Windows, nka TPM na Secure Boot. Igikoresho ca Secure Boot kiratuma iyo amadosiye ya Windows yose ahinduwe mu gihe c’ugufungura, ikibazo kimenyekana kandi n’ugushirwa mu ngiro kwavyo kukabuzwa. Ivyo birafasha kandi gukingira iyo sisitemu gushiramwo ibintu bibi kuri Windows. Gutuma umuntu ashobora guhitamwo guca muri ivyo bintu rimwe na rimwe ntibishobora kwirindwa kuri mudasobwa zimwezimwe, canecane izo mudasobwa za kera. Ariko rero, muri rusangi ni vyiza ko uguma ukoresha ibintu nka Secure Boot.

2. Kwirengagiza igisabwa co gukoresha internet kugira ngo urangize iyo nzira. Ivyo ni ngirakamaro mu bihe aho LAN idafise umurongo ushobora gukoreshwa, kuko kenshi, ikarita y’umurongo itaramenyekana mu gihe co gushiramwo Windows, kandi bisaba ko umuntu akoresha internet akoresheje umurongo. Gukoresha iyi nzira bitorera umuti ibibazo bijanye n'iyi ntambwe.


Mu ntambwe ikurikira, turashobora guhitamwo izina ry’iyo mudasobwa.


![Image](assets/en/09.webp)


Turashobora kandi kwemera ko Windows ihitamwo izina ry’ivyo bikoresho. Muri iyi ntambwe, turashobora guhitamwo ubwoko bwa Windows, yaba ipfutse canke idapfutse, canke tukareka Windows igahitamwo verisiyo ibereye bivanye n’ivyo mudasobwa ivuga. Igihe co gukoreramwo na co nyene gishobora gushirwaho kuri iyo ntambwe.


Intambwe ikurikira irimwo imiterere y'imigabane:

Mu guhitamwo uburyo bwa mbere, uguhitamwo n’ugucapura bisubirwamwo ku gihe co gushiramwo Windows, kandi mu gihe co gushiramwo, ivyo bibazo bizobazwa nk’uko bigenda mu buryo busanzwe bwo gushiramwo.

## Ivyo kubona konti


Kuri iyi ntambwe:


1. Turashobora gusobanura izina n’ijambobanga rya konti ya admin. Birashoboka kandi gukora amakonti menshi y’abakoresha canke y’abarongozi.

2. Aha, turadondora konti yo kwinjiramwo ubwa mbere inyuma yo gushiramwo Windows. Amahitamwo atandukanye y’iki gice yerekanwa ku ishusho.

3. Niba udashaka ko habaho amakonti yose, usukure amakonti yose, hanyuma uhitemwo iyi nzira. Muri ivyo, umaze gushiramwo Windows, uzoca winjira muri konti y’Umuyobozi wa Windows.


Intambwe ikurikira ni ugutunganya ijambobanga n'imiterere ya dosiye y'umushitsi:


![Image](assets/en/13.webp)


Muri iyi ntambwe, turamenya nimba amajambo y’ibanga akwiye kugira igihe co guhera. Ikindi, iki gice kirimwo ivyerekeye umutekano bijanye n’ukugerageza kwinjira mu rubuga rwa interineti, bishobora gukoreshwa canke guhagarara bivanye n’ivyo ukeneye.


Mu mpera z’iki gice, hariho amategeko yo kwerekana dosiye. Nta n’imwe muri izo nzira iboneka mu gihe co gushiramwo Windows isanzwe kandi itegerezwa gutunganirizwa inyuma yo gushiramwo. Mu buryo butandukanye n’ubwo, iyo umuntu akoresheje uburyo bwo gushiramwo ibintu ata wubicungera, ivyo bintu biraboneka bitagoranye.


Intambwe ikurikira ni ugutunganya amategeko y'umutekano wa Windows:

## Amagenamiterere y'umutekano


Kuri iyi ntambwe:


1. Windows Defender ishobora gufungurwa canke gukurwaho. Ico gikoresho gikora nk’iporogarama y’umutekano muri Windows kandi gifasha gukingira ugushirwa mu ngiro kw’amadosiye y’ububisha, ibitero bimwebimwe vyo ku rubuga, n’ibindi.

2. Ivyiyumviro bishasha vya Windows birashobora guhagarara. Ivyo ni bimwe mu bibazo bikunze gushikira abakoresha Windows!

3. Iki gice kiremera gukoresha canke guhagarika UAC (Igenzura rya Konti y’Ukoresha). Iki kintu kibuza porogaramu zikekwa gukora zifise uruhusha rwo gusoma no kwandika rwo hejuru.

4. Iyi nzira ikoreshwa na Windows kugira ngo imenye porogarama zishobora gutera ingorane.

5. Gushoboza canke guhagarika ubufasha bw’inzira ndende mu bikorwa vya Windows, nka PowerShell n’ibindi.

6. Gukoresha canke guhagarika Remote Desktop kugira ngo ushobore gushika kuri sisitemu uri kure.


Bivanye n’uko Windows ikoreshwa, bimwe muri ivyo bintu birashobora gushigikirwa canke ntibishobore gushigikirwa.


Intambwe ikurikira ni ugutunganya ibishushanyo:


![Image](assets/en/15.webp)


Muri iki gice:


1. Ibishushanyo vyo ku biro birashirwa ku rutonde, bishobora kwongerwako canke gukurwaho uko bikenewe.

2. Ivyerekana urutonde rw’intango birashirwa ku rutonde, navyo bishobora kwongerwako canke gukurwaho bivanye n’ibisabwa.

3. Iki gice kiremesha gutunganya nimba ibikoresho bijanye n’ivy’ubuhinga bwa none bishizwemwo canke bitashizweho. Iyi nzira ni iya Windows 11 kandi ntikoreshwa kuri Windows 10.

Nk’uko twabibonye, ​​kenshi na kenshi, ikarita ya Wi-Fi ntimenyekana mu gihe co gushiramwo Windows, rero gufatanya mu gihe co gushiramwo akenshi ntibishoboka. Ariko rero, mu gutunganya iki gice, iyo ikarita y’ivyuma ibonetse, iyo sisitemu irashobora gukorana na interineti.

## Porogaramu mburabuzi gushoboza/guhagarika


Muri iki gice, turashobora guhitamwo porogarama iyo ari yo yose tudashaka ko ishirwamwo ku buryo busanzwe. Nk’akarorero, turashobora guhitamwo kudashiramwo Cortana canke Copilot.


Intambwe ikurikira irimwo imiterere y'umutekano ijanye n'ugushirwa mu ngiro kw'ikoreshwa:


![Image](assets/en/19.webp)


Mu gukoresha amategeko ya WDAC, ugushirwa mu ngiro kw’ibikorwa bimwebimwe birashobora kubuzwa.


Ubwanyuma, umaze gukoresha amategeko wipfuza, dosiye ya XML yavutse irashobora gukurwako:


![Image](assets/en/20.webp)


Mu gufyonda kuri Gukuraho Dosiye ya XML, dosiye autounattend.xml irakurwako. Kugira ngo ukoreshe iyo dosiye, ushobora gusa gushiramwo ISO yakuwe kuri USB, ugashira dosiye autounattend.xml mu bubiko bw’imizi, hanyuma ugakomeza gushiramwo Windows.


Kimwe mu bikoresho biriho vyo gukora USB drive ishobora gufunguka ni Rufus. Rufus ashobora gukora flash drive yo gushiramwo Windows, ifise dosiye ISO y’ugushiramwo Windows. Ni vyihuta kandi vyoroshe, ushobora kubikura [hano](https://rufus.ie/ru/#kubikura)


![Image](assets/en/21.webp)


Muri iyo porogarama, tumaze guhitamwo USB twipfuza be na dosiye ISO ibereye, duca dufyonda kuri Start.


![Image](assets/en/22.webp)


Kuri iyi ntambwe, turazimya amahitamwo yose, kuko kuyakoresha bishobora gutuma haba amakimbirane igihe ukoresha dosiye ya Unattend yavutse. Inyuma y'aho amadosiye akopiwe kuri USB, dushira dosiye autounattend.xml mu bubiko bw'umuzi:


![Image](assets/en/23.webp)


Muri iki gihe, iyo USB irateguwe gukoreshwa mu gushiramwo Windows ubwayo, kandi gushiramwo birashobora gutangura hakoreshejwe iyo drive.


## Guhindura ISO


Niba ukeneye gushiramwo Windows ku mashini igaragara, urashobora gukoresha porogarama kugira ngo ureme kandi uhindure amadosiye ya ISO. Imwe muri izo porogarama ni AnyBurn. Amaze gukura ibiri muri dosiye ISO yakuwe ku rubuga rwa Microsoft, shira dosiye autounattend.xml mu bubiko bw’imizi. Hanyuma, ukoresheje AnyBurn, ureme ISO nshasha irimwo ibirimwo vyavuguruwe.


AnyBurn ni porogaramu ikora vyinshi mu gukorana n’amadosiye ya ISO. Itanga ibintu bitandukanye vyo gukoresha amadosiye ya ISO, kimwe muri vyo ni uguhingura amashusho ya ISO ashobora gutangura; [hano](https://www.anyburn.com/gukuraho.php) ni urubuga rw’umwimerere.


Kuri paji nyamukuru ya porogaramu, hitamwo "Rema ishusho muri dosiye/dosiye":


![Image](assets/en/24.webp)


Ku rupapuro rukurikira, hitamwo amadosiye yose yakuwe muri ISO hamwe n’idosiye autounattend.xml.


![Image](assets/en/25.webp)


Muri iyi ntambwe, turatunganya amategeko kugira ngo dosiye ISO ishobore gutangura:


![Image](assets/en/26.webp)


Kuri iyi ntambwe, inzira ija kuri dosiye bootfix.bin itegerezwa gushirwaho kugira ngo ISO ishobore gutangura. Iyi dosiye iri mu muzi wa ISO, imbere muri dosiye y'intango. Ni vyiza kandi gukoresha amahitamwo yose ISO9660 na UDF mu gice c'Imiterere.


![Image](assets/en/27.webp)
