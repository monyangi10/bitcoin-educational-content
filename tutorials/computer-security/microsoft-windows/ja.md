このチュートリアルでは、Windowsの標準的なインストールプロセス以外の方法で、Windows 11を自動的にインストールする方法を学びます。


## ダウンロード


まず必要なのはインストールファイルだ。最も安全で信頼できるのは、マイクロソフトの公式サイトから直接ダウンロードする方法だ。


以下のリンクにアクセスし、指示に従って[Windows 11 ISOファイル](https://www.microsoft.com/en-us/software-download/windows11)をダウンロードしてください。


![Image](assets/en/02.webp)


ダウンロードページに入ったら、ISOファイルをダウンロードするセクションまでスクロールダウンしてください。


![Image](assets/en/01.webp)


そして適切なバージョンを選びなさい。


![Image](assets/en/03.webp)


Windows 11を選択後、確認ボタンをクリックします。


このステップでは、リクエストの処理に数秒かかる場合があります：


![Image](assets/en/04.webp)


リクエストを確認した後、ご希望の言語を選択してください。


![Image](assets/en/05.webp)


言語を選択し、確認ボタンをクリックすると、リクエストは処理されます。このステップには数秒かかる場合があります。


リクエストが正常に処理されると、.isoファイルのダウンロードリンクのページが表示されます。64-bit Downloadボタンをクリックしてダウンロードを開始します。


ファイルサイズは約5.5GBで、生成されたリンクは24時間有効です。


## オートメーション！


この段階では、標準のWindowsインストールに変更を加える必要がある。この段階では、無人インストールを使用し、ユーザーの入力なしに、インストールされる項目を決定する。実際、この方法では、XMLファイルを使用して、インストール手順とWindowsにインストールされるサービスを設定する。つまり、Unattended.xml ファイルを使用することで、インストール時に自動化プロセスが作成され、複数のオプションを選択する必要がなくなり、セットアップ時に通常必要とされる面倒な手順を避けることができます。この方法は珍しいが、マイクロソフトが導入した標準的な方法である。詳細は[マイクロソフトの公式サイト](https://learn.microsoft.com/en-us/windows-hardware/manufacture/desktop/update-windows-settings-and-scripts-create-your-own-answer-file-sxs?view=windows-11)に掲載されています。

この段階で、コンピュータのプロセッサ・アーキテクチャを指定することもできる。このステップでは

1.TPMやセキュアブートなどのWindowsセキュリティ機能を無視するかどうかを決定します。セキュアブート機能は、ブートプロセス中にWindowsのコアファイルが改ざんされた場合、その問題が検出され、実行が阻止されるようにします。この機能はまた、Windowsに悪意のある更新プログラムがインストールされないようにシステムを保護するのにも役立ちます。これらの機能をバイパスするオプションを有効にすることは、特定のコンピュータ、特に古いモデルでは避けられない場合があります。しかし、一般的にはセキュアブートのような機能は有効にしておくことをお勧めします。

2.プロセスを完了するために、インターネット接続の要件を無視します。これは、有線LAN接続が利用できない場合に便利です。ほとんどの場合、Windowsのインストール中にワイヤレスカードがまだ認識されず、ケーブル経由でのインターネット接続が必要になるからです。このオプションを有効にすると、このステップに関する問題が解決されます。

## セキュリティ設定


この段階では：


1.Windows Defenderは有効にも無効にもできます。この機能はWindowsのセキュリティソフトのように動作し、悪意のあるファイルの実行や特定のネットワーク攻撃などを防ぐのに役立ちます。

2.Windowsの自動アップデートは無効にできるこれはWindowsユーザーが直面する一般的な課題の1つである！

3.このセクションでは、UAC（ユーザーアカウント制御）を有効または無効にすることができます。この機能により、不審なアプリケーションが昇格した読み書き権限で実行されるのを防ぎます。

4.この機能は、Windowsが潜在的に有害なソフトウェアを検出するために使用される。

5.PowerShellなどのWindowsアプリケーションでロングパスのサポートを有効または無効にする。

6.システムにリモートアクセスするためのリモートデスクトップを有効または無効にします。


使用しているWindowsのバージョンによっては、これらの機能がサポートされている場合とサポートされていない場合があります。

ブート可能なUSBドライブを作成するツールの1つにRufusがあります。Rufusは与えられたWindowsインストールISOファイルを使って、起動可能なWindowsインストールフラッシュドライブを作成することができます。ダウンロードは[こちら](https://rufus.ie/en/#download)から。

![Image](assets/en/22.webp)

![Image](assets/en/23.webp)

## ISO編集


仮想マシンにWindowsをインストールする必要がある場合、ISOファイルを作成・編集するソフトウェアを使うことができる。そのようなソフトウェアの1つがAnyBurnです。MicrosoftのウェブサイトからダウンロードしたISOファイルの内容を解凍した後、autounattend.xmlファイルをルート・ディレクトリに置きます。次に、AnyBurnを使用して、更新された内容で新しいISOを作成します。


AnyBurnはISOファイルを扱うための多機能ソフトウェアです。AnyBurnはISOファイルを扱うための様々な機能を提供しており、その中の一つにブート可能なISOイメージを作成する機能があります。[こちら](https://www.anyburn.com/download.php)はオリジナルのウェブサイトです。


ソフトウェアのメインページで、「ファイル/フォルダから画像を作成」を選択します：


![Image](assets/en/24.webp)


次のページで、autounattend.xmlファイルとともにISOから抽出したすべてのファイルを選択します。


![Image](assets/en/25.webp)


このステップでは、ISOファイルをブート可能にするための設定を行う：
