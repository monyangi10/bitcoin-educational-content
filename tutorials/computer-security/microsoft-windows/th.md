ในบทแนะนำนี้ เราจะเรียนรู้วิธีการติดตั้ง Windows 11 โดยอัตโนมัติด้วยวิธีการอื่นที่ไม่ใช่กระบวนการติดตั้ง Windows มาตรฐาน


## ดาวน์โหลด!


สิ่งแรกที่คุณต้องการคือไฟล์การติดตั้ง สถานที่ที่ปลอดภัยและเชื่อถือได้มากที่สุดในการดาวน์โหลดคือจากเว็บไซต์ทางการของ Microsoft โดยตรง


เพียงแค่เยี่ยมชมลิงก์ที่ให้ไว้ด้านล่างและทำตามคำแนะนำเพื่อดาวน์โหลด [Windows 11 ISO file](https://www.microsoft.com/en-us/software-download/windows11)


![Image](assets/en/02.webp)


เมื่อคุณอยู่ในหน้าดาวน์โหลดแล้ว ให้เลื่อนลงไปที่ส่วนสำหรับดาวน์โหลดไฟล์ ISO


![Image](assets/en/01.webp)


และเลือกเวอร์ชันที่เหมาะสม


![Image](assets/en/03.webp)


หลังจากเลือก Windows 11 แล้ว ให้คลิกปุ่มยืนยัน


ในขั้นตอนนี้ อาจใช้เวลาสักครู่ในการประมวลผลคำขอ จากนั้นคุณจะเห็นหน้าต่อไปนี้:


![Image](assets/en/04.webp)


หลังจากยืนยันคำขอแล้ว คุณต้องเลือกภาษาที่คุณต้องการ


![Image](assets/en/05.webp)


หลังจากเลือกภาษาและคลิกปุ่มยืนยัน คำขอจะถูกดำเนินการ ขั้นตอนนี้อาจใช้เวลาสองสามวินาที


เมื่อคำขอได้รับการประมวลผลเรียบร้อยแล้ว คุณจะเห็นหน้าที่มีลิงก์ดาวน์โหลดสำหรับไฟล์ .iso คลิกปุ่มดาวน์โหลด 64-bit เพื่อเริ่มการดาวน์โหลด


ขนาดไฟล์ประมาณ 5.5 GB และลิงก์ที่สร้างขึ้นจะใช้ได้เป็นเวลา 24 ชั่วโมง


## อัตโนมัติ!


ในขั้นตอนนี้ เราจำเป็นต้องทำการเปลี่ยนแปลงการติดตั้ง Windows มาตรฐาน ในขั้นตอนนี้ โดยใช้การติดตั้งแบบ Unattended เราจะกำหนดว่ารายการใดบ้างที่จะถูกติดตั้ง โดยไม่ต้องมีการป้อนข้อมูลจากผู้ใช้หลังจากนั้น ในความเป็นจริง ในวิธีนี้จะใช้ไฟล์ XML เพื่อกำหนดขั้นตอนการติดตั้งและบริการที่ติดตั้งใน Windows กล่าวอีกนัยหนึ่ง การใช้ไฟล์ Unattended.xml จะสร้างกระบวนการอัตโนมัติระหว่างการติดตั้ง ป้องกันความจำเป็นในการเลือกหลายตัวเลือกและหลีกเลี่ยงขั้นตอนที่น่าเบื่อที่มักจะต้องทำในระหว่างการตั้งค่า วิธีนี้เป็นวิธีที่ไม่ธรรมดาแต่เป็นมาตรฐานที่ Microsoft ได้แนะนำ ข้อมูลเพิ่มเติมสามารถดูได้ที่ [เว็บไซต์ทางการของ Microsoft](https://learn.microsoft.com/en-us/windows-hardware/manufacture/desktop/update-windows-settings-and-scripts-create-your-own-answer-file-sxs?view=windows-11).


มีเครื่องมือต่าง ๆ มากมายบนอินเทอร์เน็ตสำหรับการสร้างไฟล์ Unattended บางส่วนเป็นแบบออนไลน์ ในขณะที่บางส่วนเป็นแบบออฟไลน์ หนึ่งในเครื่องมือออนไลน์สำหรับสร้างไฟล์นี้คือ [เว็บไซต์นี้](https://schneegans.de/windows/unattend-generator) หลังจากเปิดแล้ว เราจะพบกับหน้าต่อไปนี้:


![Image](assets/en/06.webp)


ตามที่กล่าวไว้ที่ด้านบนของหน้า วิธีนี้สามารถใช้สำหรับการติดตั้ง Windows 10 และ 11 ได้ ในขั้นตอนแรก เราเลือกภาษาของ Windows หากเราต้องการเพิ่มภาษาที่สองหรือแม้แต่ภาษาที่สามในรายการภาษาการแสดงผลและแป้นพิมพ์ของ Windows เราสามารถใช้กล่องด้านล่าง:


![Image](assets/en/07.webp)


ในขั้นตอนถัดไป เราเลือกตำแหน่งที่ต้องการ


![Image](assets/en/08.webp)


ในขั้นตอนนี้ เราสามารถระบุสถาปัตยกรรมของโปรเซสเซอร์สำหรับคอมพิวเตอร์ได้ ในขั้นตอนนี้ เราสามารถ:

1. ตัดสินใจว่าจะละเลยคุณสมบัติความปลอดภัยของ Windows เช่น TPM และ Secure Boot หรือไม่ คุณสมบัติ Secure Boot ช่วยให้มั่นใจได้ว่าหากมีการแก้ไขไฟล์หลักของ Windows ใด ๆ ในระหว่างกระบวนการบูต ปัญหาจะถูกตรวจพบและป้องกันการดำเนินการของไฟล์เหล่านั้น คุณสมบัตินี้ยังช่วยป้องกันระบบจากการติดตั้งการอัปเดตที่เป็นอันตรายบน Windows การเปิดใช้งานตัวเลือกเพื่อข้ามคุณสมบัติเหล่านี้บางครั้งเป็นสิ่งที่หลีกเลี่ยงไม่ได้ในคอมพิวเตอร์บางเครื่อง โดยเฉพาะรุ่นเก่า อย่างไรก็ตาม โดยทั่วไปแนะนำให้เปิดใช้งานคุณสมบัติเช่น Secure Boot ไว้

2. ไม่สนใจข้อกำหนดสำหรับการเชื่อมต่ออินเทอร์เน็ตเพื่อทำกระบวนการให้เสร็จสมบูรณ์ สิ่งนี้มีประโยชน์ในสถานการณ์ที่ไม่มีการเชื่อมต่อ LAN แบบมีสาย เพราะในกรณีส่วนใหญ่ การ์ดไร้สายยังไม่ได้รับการจดจำระหว่างการติดตั้ง Windows และต้องการการเข้าถึงอินเทอร์เน็ตผ่านสายเคเบิล การเปิดใช้งานตัวเลือกนี้จะแก้ไขปัญหาที่เกี่ยวข้องกับขั้นตอนนี้


ในขั้นตอนถัดไป เราสามารถเลือกชื่อสำหรับคอมพิวเตอร์ได้


![Image](assets/en/09.webp)


เรายังสามารถให้ Windows เลือกชื่อสำหรับระบบได้อีกด้วย ในขั้นตอนนี้ เราสามารถเลือกประเภทของ Windows ได้ว่าจะเป็นแบบบีบอัดหรือไม่บีบอัด หรือให้ Windows กำหนดเวอร์ชันที่เหมาะสมตามสเปคของคอมพิวเตอร์ โซนเวลายังสามารถตั้งค่าได้ในขั้นตอนนี้เช่นกัน


ขั้นตอนถัดไปเกี่ยวข้องกับการตั้งค่าพาร์ทิชัน:


![Image](assets/en/10.webp)


ในขั้นตอนนี้ เราสามารถระบุประเภทพาร์ติชันสำหรับการติดตั้ง Windows รวมถึงการตั้งค่าที่จำเป็นสำหรับการติดตั้ง Windows Recovery Environment ได้ โดยการเลือกตัวเลือกแรก การเลือกพาร์ติชันและการแบ่งพาร์ติชันจะถูกเลื่อนออกไปจนถึงเวลาติดตั้ง Windows และในระหว่างการตั้งค่า คำถามเหล่านี้จะถูกถามเช่นเดียวกับวิธีการติดตั้งปกติ

## การตั้งค่าบัญชี


ในขั้นตอนนี้:


1. เราสามารถกำหนดชื่อและรหัสผ่านสำหรับบัญชีผู้ดูแลระบบได้ นอกจากนี้ยังสามารถสร้างบัญชีผู้ใช้หรือบัญชีผู้ดูแลระบบหลายบัญชีได้อีกด้วย

2. ที่นี่ เราระบุว่าควรเข้าสู่ระบบบัญชีใดเป็นครั้งแรกหลังจากการติดตั้ง Windows ตัวเลือกต่างๆ สำหรับส่วนนี้จะแสดงในภาพ

3. หากคุณไม่ต้องการให้มีการสร้างบัญชีใด ๆ ให้ล้างบัญชีทั้งหมดและเลือกตัวเลือกนี้ ในกรณีนี้ หลังจากการติดตั้ง Windows คุณจะเข้าสู่ระบบบัญชีผู้ดูแลระบบของ Windows โดยอัตโนมัติ


ขั้นตอนถัดไปเกี่ยวข้องกับการกำหนดค่ารหัสผ่านและการตั้งค่าไฟล์โฮสต์:


![Image](assets/en/13.webp)


ในขั้นตอนนี้ เราจะพิจารณาว่ารหัสผ่านควรมีระยะเวลาหมดอายุหรือไม่ นอกจากนี้ ส่วนนี้ยังรวมถึงการตั้งค่าความปลอดภัยที่เกี่ยวข้องกับความพยายามในการเข้าสู่ระบบที่ล้มเหลว ซึ่งสามารถเปิดหรือปิดใช้งานได้ตามความต้องการของคุณ


ที่ด้านล่างของส่วนนี้ มีการตั้งค่าสำหรับการแสดงผลไฟล์ ตัวเลือกเหล่านี้ไม่มีให้ในระหว่างการติดตั้ง Windows มาตรฐานและต้องกำหนดค่าหลังการติดตั้ง ในทางตรงกันข้าม ด้วยวิธีการติดตั้งแบบ Unattended การตั้งค่าเหล่านี้สามารถเข้าถึงได้อย่างง่ายดาย


ขั้นตอนต่อไปเกี่ยวข้องกับการกำหนดค่าการตั้งค่าความปลอดภัยของ Windows:


![Image](assets/en/14.webp)


## การตั้งค่าความปลอดภัย


ในขั้นตอนนี้:


1. Windows Defender สามารถเปิดหรือปิดใช้งานได้ ฟีเจอร์นี้ทำหน้าที่เหมือนซอฟต์แวร์รักษาความปลอดภัยใน Windows และช่วยป้องกันการทำงานของไฟล์ที่เป็นอันตราย การโจมตีเครือข่ายบางประเภท และอื่นๆ

2. การอัปเดต Windows อัตโนมัติสามารถปิดใช้งานได้ นี่เป็นหนึ่งในความท้าทายทั่วไปที่ผู้ใช้ Windows ต้องเผชิญ!

3. ส่วนนี้อนุญาตให้เปิดหรือปิดใช้งาน UAC (User Account Control) ฟีเจอร์นี้ป้องกันไม่ให้แอปพลิเคชันที่น่าสงสัยทำงานด้วยสิทธิ์ที่สูงขึ้นสำหรับการอ่านและเขียน

4. คุณลักษณะนี้ถูกใช้โดย Windows เพื่อตรวจจับซอฟต์แวร์ที่อาจเป็นอันตราย

5. เปิดหรือปิดการสนับสนุนสำหรับเส้นทางยาวในแอปพลิเคชัน Windows เช่น PowerShell และอื่นๆ

6. เปิดหรือปิดใช้งาน Remote Desktop สำหรับการเข้าถึงระบบจากระยะไกล


ขึ้นอยู่กับเวอร์ชันของ Windows ที่ใช้งานอยู่ ฟีเจอร์เหล่านี้บางอย่างอาจได้รับการสนับสนุนหรือไม่ก็ได้


ขั้นตอนถัดไปเกี่ยวข้องกับการกำหนดค่าไอคอน:


![Image](assets/en/15.webp)


ในส่วนนี้:


1. ไอคอนเดสก์ท็อปจะแสดงรายการ ซึ่งสามารถเพิ่มหรือลบได้ตามต้องการ

2. ไอคอนเมนูเริ่มต้นจะแสดงรายการ ซึ่งสามารถเพิ่มหรือลบได้ตามความต้องการ

3. ส่วนนี้อนุญาตให้กำหนดค่าการติดตั้งเครื่องมือที่เกี่ยวข้องกับการจำลองเสมือนหรือไม่ ตัวเลือกนี้เฉพาะสำหรับ Windows 11 และไม่ใช้กับ Windows 10


ขั้นตอนถัดไปเกี่ยวข้องกับการกำหนดค่า Wi-Fi:


![Image](assets/en/16.webp)


ในส่วนนี้ สามารถกำหนดค่าการตั้งค่าเครือข่าย Wi-Fi ได้ ตามที่กล่าวไว้ก่อนหน้านี้ ในกรณีส่วนใหญ่ การ์ด Wi-Fi จะไม่ถูกตรวจพบระหว่างการติดตั้ง Windows ดังนั้นการเชื่อมต่อระหว่างการตั้งค่าจึงมักจะไม่สามารถทำได้ อย่างไรก็ตาม โดยการกำหนดค่าส่วนนี้ หากตรวจพบการ์ดไร้สาย ระบบจะสามารถเชื่อมต่อกับอินเทอร์เน็ตได้


ขั้นตอนถัดไปเกี่ยวข้องกับการตั้งค่าที่สำคัญ:


![Image](assets/en/17.webp)


ในส่วนนี้ เราระบุว่าควรส่งข้อมูลปัญหาของระบบไปยัง Microsoft หรือไม่


ขั้นตอนถัดไปเกี่ยวข้องกับการกำหนดค่าแอปพลิเคชันเริ่มต้น:

## เปิด/ปิดใช้งานซอฟต์แวร์เริ่มต้น


ในส่วนนี้ เราสามารถเลือกแอปพลิเคชันใดก็ได้ที่เราไม่ต้องการให้ติดตั้งโดยอัตโนมัติ ตัวอย่างเช่น เราสามารถเลือกไม่ติดตั้ง Cortana หรือ Copilot ได้


ขั้นตอนถัดไปเกี่ยวข้องกับการตั้งค่าความปลอดภัยที่เกี่ยวข้องกับการดำเนินการของแอปพลิเคชัน:


![Image](assets/en/19.webp)


โดยการใช้การตั้งค่า WDAC สามารถป้องกันการทำงานของแอปพลิเคชันบางตัวได้


ในที่สุด หลังจากตั้งค่าที่ต้องการแล้ว ไฟล์ XML ที่สร้างขึ้นสามารถดาวน์โหลดได้:


![Image](assets/en/20.webp)


โดยการคลิกที่ ดาวน์โหลดไฟล์ XML ไฟล์ autounattend.xml จะถูกดาวน์โหลด เพื่อใช้ไฟล์นี้ เพียงแค่เมานต์ ISO ที่ดาวน์โหลดลงในไดรฟ์ USB วางไฟล์ autounattend.xml ไว้ในไดเรกทอรีราก จากนั้นดำเนินการติดตั้ง Windows ต่อไป


หนึ่งในเครื่องมือที่ใช้สร้าง USB ไดรฟ์ที่สามารถบูตได้คือ Rufus Rufus สามารถสร้างแฟลชไดรฟ์ติดตั้ง Windows ที่สามารถบูตได้ ด้วยไฟล์ ISO การติดตั้ง Windows ที่กำหนด มันรวดเร็วและง่ายดาย คุณสามารถดาวน์โหลดได้ [ที่นี่](https://rufus.ie/en/#download)


![Image](assets/en/21.webp)


ในซอฟต์แวร์นี้ หลังจากเลือกไดรฟ์ USB ที่ต้องการและไฟล์ ISO ที่เหมาะสมแล้ว เราคลิกที่ เริ่มต้น


![Image](assets/en/22.webp)


ในขั้นตอนนี้ เราจะปิดใช้งานตัวเลือกทั้งหมด เนื่องจากการเปิดใช้งานอาจทำให้เกิดความขัดแย้งเมื่อใช้ไฟล์ Unattend ที่สร้างขึ้น หลังจากที่ไฟล์ถูกคัดลอกไปยังไดรฟ์ USB แล้ว เราจะวางไฟล์ autounattend.xml ไว้ในไดเรกทอรีราก:


![Image](assets/en/23.webp)


ณ จุดนี้ ไดรฟ์ USB พร้อมใช้งานสำหรับการติดตั้ง Windows โดยอัตโนมัติ และสามารถเริ่มการติดตั้งโดยใช้ไดรฟ์นี้ได้


## การแก้ไข ISO


หากคุณต้องการติดตั้ง Windows บนเครื่องเสมือน คุณสามารถใช้ซอฟต์แวร์เพื่อสร้างและแก้ไขไฟล์ ISO ได้ ซอฟต์แวร์หนึ่งที่สามารถใช้ได้คือ AnyBurn หลังจากที่คุณแยกเนื้อหาของไฟล์ ISO ที่ดาวน์โหลดจากเว็บไซต์ของ Microsoft แล้ว ให้วางไฟล์ autounattend.xml ไว้ในไดเรกทอรีราก จากนั้นใช้ AnyBurn เพื่อสร้างไฟล์ ISO ใหม่ที่มีเนื้อหาที่อัปเดตแล้ว


AnyBurn เป็นซอฟต์แวร์อเนกประสงค์สำหรับการทำงานกับไฟล์ ISO มีฟีเจอร์ต่างๆ สำหรับจัดการไฟล์ ISO หนึ่งในนั้นคือการสร้างภาพ ISO ที่สามารถบูตได้; [ที่นี่](https://www.anyburn.com/download.php) คือเว็บไซต์ต้นฉบับ


บนหน้าหลักของซอฟต์แวร์ เลือก "สร้างภาพจากไฟล์/โฟลเดอร์":


![Image](assets/en/24.webp)


ในหน้าถัดไป ให้เลือกไฟล์ทั้งหมดที่แยกออกมาจาก ISO พร้อมกับไฟล์ autounattend.xml


![Image](assets/en/25.webp)


ในขั้นตอนนี้ เราจะกำหนดการตั้งค่าเพื่อทำให้ไฟล์ ISO สามารถบูตได้:


![Image](assets/en/26.webp)


ในขั้นตอนนี้ ต้องตั้งค่าเส้นทางไปยังไฟล์ bootfix.bin เพื่อทำให้ ISO สามารถบูตได้ ไฟล์นี้ตั้งอยู่ในรากของ ISO ภายในโฟลเดอร์ boot นอกจากนี้ยังแนะนำให้เปิดใช้งานทั้งตัวเลือก ISO9660 และ UDF ในส่วน Properties


![Image](assets/en/27.webp)


หลังจากขั้นตอนนี้ การคลิกถัดไปจะสร้างไฟล์ ISO ไฟล์นี้สามารถใช้ในซอฟต์แวร์การจำลองเสมือน เช่น Oracle VirtualBox ด้านล่างนี้คุณจะพบกับบทแนะนำเกี่ยวกับ VirtualBox:
