## บทนำ


ระบบปฏิบัติการ (OS) เป็นซอฟต์แวร์หลักที่จัดการทรัพยากรทั้งหมดของคอมพิวเตอร์ของคุณ การเลือกใช้ระบบปฏิบัติการทางเลือกเช่น Ubuntu สามารถให้ข้อดีหลายประการในด้านความปลอดภัย ค่าใช้จ่าย และความเป็นส่วนตัว


### ทำไมต้อง Ubuntu?




- ความปลอดภัยที่เพิ่มขึ้น**: การแจกจ่าย Linux มีชื่อเสียงในด้านความปลอดภัยและความแข็งแกร่ง
- ไม่มีค่าใช้จ่าย**: Ubuntu และการแจกจ่าย Linux ส่วนใหญ่ไม่มีค่าใช้จ่าย
- ชุมชนขนาดใหญ่**: ชุมชนของผู้ใช้ที่พร้อมให้ความช่วยเหลือผ่านฟอรัมและบทแนะนำ
- เคารพความเป็นส่วนตัว**: ระบบโอเพ่นซอร์สเพื่อความโปร่งใสมากขึ้น
- ความเรียบง่าย**: อินเทอร์เฟซที่ใช้งานง่ายและสะดวกในการใช้งาน
- ระบบนิเวศที่หลากหลาย**: แคตตาล็อกซอฟต์แวร์โอเพ่นซอร์สที่กว้างขวาง
- การสนับสนุนปกติ**: รับการอัปเดตที่ปลอดภัยจาก Canonical


## การติดตั้งและการกำหนดค่า


### 1. ข้อกำหนดเบื้องต้น


**อุปกรณ์ที่จำเป็น:**




- USB key ขนาดอย่างน้อย 12 GB
- คอมพิวเตอร์ที่มีพื้นที่ว่างอย่างน้อย 25 GB


### 2. ดาวน์โหลด




- ไปที่ [ubuntu.com/download](https://ubuntu.com/download)
- เลือกเวอร์ชันที่เสถียร (แนะนำ LTS)
- ดาวน์โหลดไฟล์อิมเมจ ISO


![Page de téléchargement Ubuntu](assets/fr/01.webp)


![Sélection de la version Ubuntu](assets/fr/02.webp)


### 3. สร้าง USB key ที่สามารถบูตได้


คุณสามารถใช้เครื่องมือหลายอย่าง เช่น Balena Etcher:




- ดาวน์โหลดและติดตั้ง [Balena Etcher](https://etcher.balena.io/)


![Page de téléchargement Balena Etcher](assets/fr/03.webp)

- คลิกที่ Flash และรอให้กระบวนการเสร็จสิ้น


![Utilisation de Balena Etcher](assets/fr/05.webp)


### 4. การติดตั้งและรักษาความปลอดภัย Ubuntu


**4.1 การบูตจาก USB memory stick** (เป็นภาษาฝรั่งเศส)




- ปิดคอมพิวเตอร์
- เสียบกุญแจ USB (ที่มี Ubuntu)
- เปิดคอมพิวเตอร์ของคุณ บนพีซีรุ่นใหม่ ระบบควรจะรู้จัก USB boot key โดยอัตโนมัติ หากไม่เป็นเช่นนั้น ให้รีบูตโดยกดปุ่มเข้าถึง BIOS/UEFI ค้างไว้ (ปกติคือ F2, F12 หรือ Delete ขึ้นอยู่กับยี่ห้อ)
 - ในเมนู BIOS/UEFI เลือก USB key ของคุณเป็นอุปกรณ์บูต
 - บันทึกและเริ่มใหม่


**4.2 การเริ่มต้นการติดตั้ง** (เป็นภาษาฝรั่งเศส)


**หน้าจอเริ่มต้น**


เมื่อบูตจาก USB key คุณจะเห็นหน้าจอนี้ ซึ่งจะช่วยให้คุณเริ่มต้น Ubuntu ได้


![Écran de démarrage Ubuntu](assets/fr/06.webp)


**ตัวเลือกภาษา**

**ตัวเลือกการเข้าถึง**


กำหนดค่าตัวเลือกการเข้าถึงหากจำเป็น (โปรแกรมอ่านหน้าจอ, ความคมชัดสูง, ฯลฯ)

![Configuration du clavier](assets/fr/09.webp)


**การเชื่อมต่อเครือข่าย**


เชื่อมต่อกับเครือข่าย Wi-Fi หรือเครือข่ายแบบใช้สายเพื่อดาวน์โหลดการอัปเดตระหว่างการติดตั้ง

![Sélection des applications](assets/fr/13.webp)


**แอปพลิเคชันของบุคคลที่สาม**

![Résumé de l'installation](assets/fr/18.webp)


**4.3 การอัปเกรด Ubuntu หลังการติดตั้ง** (เป็นภาษาฝรั่งเศส)


การอัปเดตระบบของคุณเป็นขั้นตอนสำคัญหลังจากการติดตั้งใหม่ คุณมีสองตัวเลือก:


**ตัวเลือกที่ 1: ผ่านทางอินเทอร์เฟซผู้ใช้แบบกราฟิก**




- ค้นหา "ซอฟต์แวร์และการอัปเดต" ในเมนูแอปพลิเคชัน
- แอปพลิเคชันจะตรวจสอบการอัปเดตที่มีอยู่โดยอัตโนมัติ
- ทำตามคำแนะนำบนหน้าจอเพื่อติดตั้งการอัปเดต


**ตัวเลือกที่ 2: ผ่านทางเทอร์มินัล**




- เปิดเทอร์มินัล (Ctrl + Alt + T)
- พิมพ์คำสั่งต่อไปนี้เพื่อตรวจสอบการอัปเดตที่มีอยู่:


```bash
sudo apt update
```




- กรุณาใส่รหัสผ่านเมื่อระบบร้องขอ
- ในการติดตั้งการอัปเดต ให้พิมพ์:


```bash
sudo apt upgrade
```




- ยืนยันการติดตั้งโดยพิมพ์ 'Y' แล้วกด Enter


### 5. การค้นพบงานพื้นฐาน


**5.1 การท่องอินเทอร์เน็ต**


ตามค่าเริ่มต้น คุณมักจะพบ Firefox ในแถบเปิดตัว


เปิดตัว Firefox และเริ่มการท่องเว็บ


เบราว์เซอร์อื่นๆ (Chrome, Brave, ฯลฯ) สามารถติดตั้งได้ผ่านทาง Software Manager หรือผ่านทางแพ็กเกจ .deb


**5.2 การประมวลผลคำ**


Ubuntu มาพร้อมกับชุดโปรแกรม LibreOffice (Writer สำหรับการประมวลผลคำ)


ในการเปิด: กิจกรรม > ค้นหา "LibreOffice Writer" หรือคลิกที่ไอคอนหากปรากฏในแถบ.


คุณสามารถสร้าง แก้ไข และบันทึกเอกสารในรูปแบบต่างๆ (รวมถึง .docx)


**5.3 การติดตั้งแอปพลิเคชัน**


ผู้จัดการซอฟต์แวร์ (เรียกว่า "Ubuntu Software"): อินเทอร์เฟซกราฟิกสำหรับการค้นหาและติดตั้งแอปพลิเคชัน


จาก Terminal, ใช้คำสั่ง:


```bash
sudo apt install nom-du-paquet
```


ตัวอย่าง:


```bash
sudo apt install vlc
```
