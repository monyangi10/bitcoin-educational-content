## Imenyekanisha


Sisitemu ikoresha (OS) ni porogaramu nyamukuru icungera ibikoresho vyose vya mudasobwa yawe. Guhitamwo ubundi buryo bwo gukoresha nka Ubuntu birashobora gutanga inyungu nyinshi mu bijanye n’umutekano, igiciro n’ubuzima bwite.


### Kuki Ubuntu?




- Umutekano wongerewe**: Ivyiyumviro vya Linux birazwi cane kubera umutekano wavyo n'ubukomezi bwavyo
- Igiciro ntaco kigura**: Ubuntu n'ibindi bikoresho vyinshi vya Linux ni ubuntu
- Umuryango munini**: Umuryango w'abakoresha witeguriye gufasha biciye mu mahuriro n'inyigisho
- Kwubaha ubuzima bwite**: Uburyo bufunguye kugira ngo habeho uguserukira abantu
- Ukworohereza**: Interface ikoreshwa neza kandi yoroshe gukoresha
- Ubutunzi bw'ibidukikije**: Urutonde rwinshi rwa porogaramu zifunguye
- Infashanyo idasanzwe**: Ivyagezwe biva kuri Canonical

### 4. Gushiramwo no gukingira Ubuntu


**4.1 Gufungura kuri USB** (mu gifaransa)




- Zima mudasobwa
- Shira urufunguzo rwa USB (rurimwo Ubuntu)
- Hindukiza mudasobwa yawe. Ku ma PC aherutse gusohoka, iyo sisitemu ikwiye kumenya ubwayo urufunguzo rwo gufungura USB. Niba bitameze gutyo, subira utangure ufashe urufunguzo rwo gushika kuri BIOS/UEFI (kenshi F2, F12, canke Delete, bivanye n’ibara)
 - Mu bimenyetso vya BIOS/UEFI, hitamwo urufunguzo rwawe rwa USB nk'igikoresho co gutangura
 - Bika kandi usubire gutangura

**4.3 Gusubiramwo Ubuntu inyuma yo gushirwaho** (mu gifaransa)


Guhindura sisitemu yawe ni intambwe ihambaye inyuma yo gushiramwo ibintu bishasha. Ufise uburyo bubiri:


**Ihitamwo rya 1: Biciye ku mukoresha w'igishushanyo Interface**




- Rondera "Porogaramu n'ivyahinduwe" mu bikubiyemo vy'ibikorwa
- Porogaramu izohita igenzura ko hariho ivyagezwe bishasha
- Kurikiza amabwirizwa ari ku rubuga kugira ngo ushiremwo ivyo bishasha


**Ihitamwo rya 2: Biciye ku nzira**




- Gufungura Ikibanza (Ctrl + Alt + T)
- Andika itegeko rikurikira kugira ngo ugenzure ibivuguruwe biriho:


```bash
sudo apt update
```




- Injira ijambobanga ryawe iyo usabwe
- Kugira ngo ushiremwo ivyagezwe, wandike:


```bash
sudo apt upgrade
```




- Wemeze ko washizeho wanditse 'Y' hanyuma winjize


### 5. Kuvumbura ibikorwa vy’ishimikiro


**5.1 Gusura kuri Internet**


Ku mburabuzi, uzosanga kenshi Firefox mu mwanya wo gutangura.


Tangaza Firefox maze utangure gusesa.


Ibindi bikoresho (Chrome, Brave, n’ibindi) bishobora gushirwamwo biciye ku Mucungerezi wa Porogarama canke biciye ku bikoresho vya .deb.


**5.2 Gutunganya amajambo**


Ubuntu izana n'umugwi wa LibreOffice (Umwanditsi wo gutunganya amajambo).


Kugira ngo uyifungure: Ibikorwa > Rondera "Umwanditsi wa LibreOffice" canke ukande ku kimenyetso nimba kiboneka mu kabati.


Ushobora kurema, guhindura no kubika inyandiko mu buryo butandukanye (harimwo .docx).


**5.3 Gushiramwo porogaramu**


Umuyobozi wa porogaramu (yitwa "Porogaramu ya Ubuntu"): igishushanyo Interface co kurondera no gushiramwo porogaramu.


Kuva kuri Terminal, koresha itegeko:


```bash
sudo apt install nom-du-paquet
```


Akarorero:


```bash
sudo apt install vlc
```
