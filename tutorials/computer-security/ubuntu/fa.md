## مقدمه


یک سیستم‌عامل (OS) نرم‌افزار اصلی است که تمام منابع کامپیوتر شما را مدیریت می‌کند. انتخاب یک سیستم‌عامل جایگزین مانند اوبونتو می‌تواند مزایای زیادی از نظر امنیت، هزینه و حریم خصوصی ارائه دهد.


### چرا اوبونتو؟




- امنیت پیشرفته**: توزیع‌های لینوکس به خاطر امنیت و استحکام خود مشهور هستند
- هزینه صفر**: اوبونتو و اکثر توزیع‌های لینوکس رایگان هستند
- جامعه بزرگ**: جامعه‌ای از کاربران آماده کمک از طریق انجمن‌ها و آموزش‌ها
- احترام به حریم خصوصی**: سیستم متن‌باز برای شفافیت بیشتر
- سادگی**: Interface کاربرپسند و سهولت در استفاده
- اکوسیستم غنی**: کاتالوگ گسترده‌ای از نرم‌افزارهای متن‌باز
- پشتیبانی منظم**: دریافت به‌روزرسانی‌های امن از Canonical

### ۴. نصب و ایمن‌سازی اوبونتو


**4.1 راه‌اندازی از طریق حافظه USB**




- کامپیوتر را خاموش کنید
- کلید USB (حاوی اوبونتو) را وصل کنید
- رایانه خود را روشن کنید. در رایانه‌های شخصی جدید، سیستم باید به‌طور خودکار کلید بوت USB را شناسایی کند. اگر اینطور نیست، با نگه داشتن کلید دسترسی به BIOS/UEFI (معمولاً F2، F12 یا Delete، بسته به برند) دوباره راه‌اندازی کنید.
 - در منوی BIOS/UEFI، کلید USB خود را به عنوان دستگاه بوت انتخاب کنید
 - ذخیره و راه‌اندازی مجدد

**4.3 ارتقاء اوبونتو پس از نصب**


به‌روزرسانی سیستم شما یک گام مهم پس از نصب جدید است. شما دو گزینه دارید:


**گزینه 1: از طریق رابط کاربری گرافیکی Interface**




- در منوی برنامه‌ها به دنبال "نرم‌افزار و به‌روزرسانی‌ها" بگردید.
- برنامه به‌طور خودکار برای به‌روزرسانی‌های موجود بررسی خواهد کرد
- دستورالعمل‌های روی صفحه را دنبال کنید تا به‌روزرسانی‌ها نصب شوند.


**گزینه ۲: از طریق ترمینال




- ترمینال را باز کنید (Ctrl + Alt + T)
- برای بررسی به‌روزرسانی‌های موجود، دستور زیر را تایپ کنید:


```bash
sudo apt update
```




- رمز عبور خود را هنگام درخواست وارد کنید
- برای نصب به‌روزرسانی‌ها، تایپ کنید:


```bash
sudo apt upgrade
```




- نصب را با تایپ 'Y' و سپس Enter تأیید کنید


### ۵. کشف وظایف پایه


**5.1 مرور اینترنت


به طور پیش‌فرض، اغلب فایرفاکس را در نوار راه‌انداز خواهید یافت.


فایرفاکس را راه‌اندازی کنید و مرور را آغاز کنید.


مرورگرهای دیگر (Chrome، Brave و غیره) می‌توانند از طریق مدیر نرم‌افزار یا از طریق بسته‌های .deb نصب شوند.


**5.2 پردازش کلمه**


اوبونتو با مجموعه LibreOffice (Writer برای پردازش کلمه) ارائه می‌شود.


برای باز کردن آن: فعالیت‌ها > جستجو برای "LibreOffice Writer" یا روی آیکون کلیک کنید اگر در نوار ظاهر شد.


شما می‌توانید اسناد را در قالب‌های مختلف (از جمله .docx) ایجاد، ویرایش و ذخیره کنید.


**5.3 نصب برنامه‌ها


مدیر نرم‌افزار (به نام "نرم‌افزار اوبونتو"): Interface گرافیکی برای جستجو و نصب برنامه‌ها.


از ترمینال، از دستور زیر استفاده کنید:


```bash
sudo apt install nom-du-paquet
```


مثال:


```bash
sudo apt install vlc
```
