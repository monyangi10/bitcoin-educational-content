## परिचय


ऑपरेटिंग सिस्टम (OS) मुख्य सॉफ़्टवेयर है जो आपके कंप्यूटर के सभी संसाधनों का प्रबंधन करता है। उबंटू जैसे वैकल्पिक ऑपरेटिंग सिस्टम को चुनने से सुरक्षा, लागत और गोपनीयता के मामले में कई लाभ मिल सकते हैं।


### उबंटू क्यों?




- उन्नत सुरक्षा**: लिनक्स वितरण अपनी सुरक्षा और मजबूती के लिए प्रसिद्ध हैं
- शून्य लागत**: उबंटू और अधिकांश लिनक्स वितरण निःशुल्क हैं
- बड़ा समुदाय**: मंचों और ट्यूटोरियल के माध्यम से मदद करने के लिए तैयार उपयोगकर्ताओं का समुदाय
- गोपनीयता का सम्मान**: अधिक पारदर्शिता के लिए ओपन सोर्स प्रणाली
- सरलता**: उपयोगकर्ता-अनुकूल Interface और उपयोग में आसानी
- समृद्ध पारिस्थितिकी तंत्र**: ओपन सोर्स सॉफ्टवेयर की विस्तृत सूची
- नियमित समर्थन**: Canonical से सुरक्षित अपडेट

### 4. उबंटू को स्थापित करना और सुरक्षित करना


**4.1 USB मेमोरी स्टिक से बूट करना** (फ्रेंच में)




- कंप्यूटर बंद करें
- USB कुंजी (जिसमें Ubuntu है) प्लग करें
- अपने कंप्यूटर को चालू करें। हाल ही के पीसी पर, सिस्टम को USB बूट कुंजी को स्वचालित रूप से पहचान लेना चाहिए। यदि ऐसा नहीं है, तो BIOS/UEFI एक्सेस कुंजी (आमतौर पर F2, F12, या Delete, ब्रांड के आधार पर) को दबाकर रीबूट करें
 - BIOS/UEFI मेनू में, बूट डिवाइस के रूप में अपनी USB कुंजी का चयन करें
 - सहेजें और पुनः आरंभ करें


**4.2 इंस्टॉलेशन शुरू करना** (फ्रेंच में)


**स्टार्ट-अप स्क्रीन**

**भाषा का चयन

![Sélection de la langue](assets/fr/07.webp)


**अभिगम्यता के विकल्प


यदि आवश्यक हो तो पहुँच-योग्यता विकल्प कॉन्फ़िगर करें (स्क्रीन रीडर, उच्च कंट्रास्ट, आदि).

![Configuration du clavier](assets/fr/09.webp)


**नेटवर्क कनेक्शन**

**4.3 स्थापना के बाद उबंटू को अपग्रेड करना** (फ्रेंच में)


नए इंस्टॉलेशन के बाद अपने सिस्टम को अपडेट करना एक महत्वपूर्ण कदम है। आपके पास दो विकल्प हैं:


**विकल्प 1: ग्राफ़िकल यूजर Interface के माध्यम से**




- एप्लिकेशन मेनू में "सॉफ़्टवेयर और अपडेट" खोजें
- एप्लिकेशन स्वचालित रूप से उपलब्ध अपडेट की जांच करेगा
- अपडेट इंस्टॉल करने के लिए ऑन-स्क्रीन निर्देशों का पालन करें


**विकल्प 2: टर्मिनल के माध्यम से




- टर्मिनल खोलें (Ctrl + Alt + T)
- उपलब्ध अद्यतनों की जांच करने के लिए निम्नलिखित कमांड टाइप करें:


```bash
sudo apt update
```




- संकेत मिलने पर अपना पासवर्ड दर्ज करें
- अद्यतन स्थापित करने के लिए, टाइप करें:


```bash
sudo apt upgrade
```




- 'Y' टाइप करके फिर Enter लिखकर इंस्टॉलेशन की पुष्टि करें


### 5. बुनियादी कार्यों की खोज


**5.1 इंटरनेट ब्राउज़ करना


डिफ़ॉल्ट रूप से, आपको अक्सर फ़ायरफ़ॉक्स लॉन्च बार में मिलेगा।


फ़ायरफ़ॉक्स लॉन्च करें और ब्राउज़िंग शुरू करें।


अन्य ब्राउज़रों (क्रोम, ब्रेव, आदि) को सॉफ्टवेयर मैनेजर या .deb पैकेज के माध्यम से स्थापित किया जा सकता है।


**5.2 वर्ड प्रोसेसिंग


उबंटू लिबरऑफिस सुइट (वर्ड प्रोसेसिंग के लिए राइटर) के साथ आता है।


इसे खोलने के लिए: गतिविधियाँ > "लिब्रे ऑफिस राइटर" खोजें या यदि यह बार में दिखाई दे तो आइकन पर क्लिक करें।


आप विभिन्न प्रारूपों (.docx सहित) में दस्तावेज़ बना सकते हैं, संपादित कर सकते हैं और सहेज सकते हैं।


**5.3 एप्लिकेशन इंस्टॉल करना


सॉफ्टवेयर प्रबंधक (जिसे "उबंटू सॉफ्टवेयर" कहा जाता है): अनुप्रयोगों को खोजने और स्थापित करने के लिए ग्राफिकल Interface.


टर्मिनल से, निम्न कमांड का उपयोग करें:


```bash
sudo apt install nom-du-paquet
```


उदाहरण:


```bash
sudo apt install vlc
```


### 6. निष्कर्ष और अतिरिक्त संसाधन


अब आप दैनिक आधार पर उबंटू का उपयोग करने के लिए तैयार हैं: अपने सिस्टम को सुरक्षित रखें, ब्राउज़ करें, कार्यालय का काम करें, सॉफ्टवेयर इंस्टॉल करें और अपने ओएस को अद्यतन रखें!
