信箱是您線上活動的核心，通常在您的 IT 安全中扮演重要角色。如果攻擊者成功入侵您的信箱，他就可以透過「*忘記密碼*」功能輕鬆存取您的其他帳戶。這可讓他控制您的社交網路、銀行帳戶和其他線上服務，因為現在電子郵件 Address 通常被用來作為您線上身分的唯一識別碼。因此，保護您的郵箱安全對於保護自己免受攻擊非常重要。



為了確保您信箱的安全性，採取一些簡單的最佳作法是很重要的，我們將在本電腦初學者教學中探討這些作法。選擇一個提供進階保護選項和健全隱私政策的安全電子郵件供應商也很重要。這就是我在本教程中推薦 ProtonMail 的原因。即使您不想使用此供應商，這裡介紹的最佳實務也可應用於任何郵箱，以提高其安全性。



## 為什麼使用 ProtonMail？



ProtonMail 是一個相當安全的電子郵件解決方案，這要歸功於它的幾個功能。首先，ProtonMail 為您的電子郵件提供端對端加密，這意味著只有發件人和收件人才能閱讀郵件內容。理論上，即使是 ProtonMail 也無法讀取用戶的電郵。這種加密方式是自動應用的，用戶不需要具備特定的技術技能。



更重要的是，ProtonMail 採用了先進的技術保護您的隱私，包括封鎖某些追蹤系統和遮罩您的 IP Address。由於 Proton 位於瑞士，因此可受惠於一些其他國家所沒有的資料保護法。更重要的是，ProtonMail 是開放源碼的，允許獨立專家自由審核軟件代碼。



Proton 的商業模式是以訂閱制度為基礎，這讓人感到安心，因為這表示該公司在不剝削用戶資料的情況下為自己提供資金。在本教程中，我們將探討如何使用 ProtonMail 的免費版本，但也有多個訂閱級別提供更多功能。這種商業模式比起完全免費的系統更為可取，這可能會讓您懷疑我們的個人資料是否被用來牟利。幸運的是，ProtonMail 的情況似乎並非如此。



## 建立 Proton 帳戶



請造訪質子官方網站：https://proton.me/



![proton](assets/notext/01.webp)



按一下「*建立帳戶*」按鈕：



![proton](assets/notext/02.webp)



您可以根據自己的需要選擇不同的套餐。要開始使用，您可以選擇免費帳戶，讓您測試 ProtonMail 的基本服務。之後，如果您想使用其他功能和其他 Proton 軟件，例如日曆、VPN 或密碼管理器，您可以考慮訂購付費套餐。



![proton](assets/notext/03.webp)



然後，您將會進入帳戶建立頁面。



![proton](assets/notext/04.webp)



您可以按一下小箭頭，選擇您希望在電子郵件 Address 上使用的網域名稱。這個選擇對接下來發生的事情沒有影響。



![proton](assets/notext/05.webp)



同時選擇您的電子郵件 Address 識別碼。



![proton](assets/notext/06.webp)



接下來會要求您設定密碼。在此階段選擇一個強大的密碼是很重要的，因為這樣才能存取您的信箱。一個強大的密碼應盡量長，使用多種字符，並隨機選擇。在 2024 年，安全密碼的最低建議為 13 個字元，包括數字、大小寫字母和符號，前提是密碼是真正隨機的。但是，我建議選擇至少 20 個字元的密碼，包括所有可能的字元類型，以保證其安全性更長的時間。



使用密碼管理器是非常好的做法。它不僅可以讓您安全地儲存密碼，而無需記住密碼，還可以為您 generate 長而隨機的密碼。人類確實很不擅長創建隨機序列，隨機性不足的密碼很容易受到暴力攻擊。我也建議您參考我們關於設定密碼管理器的全面教學，以瞭解更多相關細節：



https://planb.network/tutorials/computer-security/authentication/bitwarden-0532f569-fb00-4fad-acba-2fcb1bf05de9

![proton](assets/notext/07.webp)



按一下「*建立帳戶*」按鈕。



![proton](assets/notext/08.webp)



解決 CAPTCHA。



![proton](assets/notext/09.webp)



選擇顯示名稱。這是您傳送電子郵件時會顯示給收件者的名稱。選擇您的真名或假名。



![proton](assets/notext/10.webp)



Proton 還提供您設定恢復帳戶方法的選項，您可以透過電話號碼或其他電子郵件 Address。請務必瞭解，此選項可能會增加您信箱的攻擊面。對您來說，這是額外的安全保障，讓您可以在忘記密碼時重新取得帳號存取權限，但對駭客來說，這是嘗試入侵您帳號的額外機會。您不一定要選擇此復原選項，但如果您決定不選擇，請務必保留一份安全的密碼副本。否則，如果您遺失密碼，將無法恢復信箱的存取權。



![proton](assets/notext/11.webp)



## 設定您的 Proton 信箱



恭喜您，您的 Proton 信箱已經建立！開始選擇您的信箱主題顏色。



![proton](assets/notext/12.webp)



如果您願意，您也可以設定一個郵件轉發器，從您舊的 Gmail 信箱轉發到您新的 ProtonMail 信箱。



![proton](assets/notext/13.webp)



一旦您進入 Interface 信箱，我建議您到設定中自訂。按一下右上角的齒輪圖示。



![proton](assets/notext/14.webp)



然後按一下「*所有設定*」按鈕。



![proton](assets/notext/15.webp)



在 「*儀表板*」標籤中，您可以找到您的帳戶資訊。再往下瀏覽此部分，您可以選擇同意接收來自 Proton 的電子郵件類型。如果您不希望收到促銷或資訊通知，您可以選擇全部取消勾選。



![proton](assets/notext/16.webp)



在「*升級方案*」標籤中，您可以選擇具有新功能的付費套裝。



![proton](assets/notext/17.webp)



在「*復原*」標籤中，您可以新增或修改您的復原方法。



![proton](assets/notext/18.webp)



在「*帳號和密碼*」標籤中，您可以變更使用者名稱以及帳號安全方法。



![proton](assets/notext/19.webp)



目前，您的郵箱只有密碼保護。我建議您至少使用應用程式新增雙因素驗證保護。若要這樣做，請按一下核取標記。



![proton](assets/notext/20.webp)



確認您的密碼。



![proton](assets/notext/21.webp)



然後使用您的 2FA 應用程式掃描 QR 代碼。



![proton](assets/notext/22.webp)



如需詳細資訊，請參閱我們關於如何使用 2FA 應用程式的教學。



https://planb.network/tutorials/computer-security/authentication/authy-a76ab26b-71b0-473c-aa7c-c49153705eb7

在「*語言和時間*」標籤中，您可以變更 Interface 的語言和時區。



![proton](assets/notext/23.webp)



在「*外觀*」標籤中，您可以變更 Interface 的顏色。



![proton](assets/notext/24.webp)



在「*安全與隱私*」標籤中，您可以存取各種安全選項。其中有些選項只有付費套裝才能使用。您也可以選擇禁止 Proton 公司收集您的資料，Proton 公司會將這些資料用於診斷和解決錯誤。



![proton](assets/notext/25.webp)



在 「*導入*」標籤中，您可以管理舊郵件遷移至新的 ProtonMail 信箱。如果您希望從一個全新的信箱開始，而不匯入您的舊郵件，您可以選擇忽略此選項。



![proton](assets/notext/26.webp)



透過「*取得應用程式*」標籤，您可以下載 Proton 的行動應用程式和桌面軟體，在這些平台上管理您的信箱。如果您願意，可以繼續只使用目前使用的網頁版信箱，因為它提供相同的功能。



![proton](assets/notext/27.webp)



在「*訊息與撰寫*」標籤中，您的信箱有無數的個人化選項。



![proton](assets/notext/28.webp)



在「*電子郵件隱私*」標籤中，您可以選擇電子郵件隱私選項。



![proton](assets/notext/29.webp)



在「*身分與地址*」標籤中，您可以自訂電子郵件簽名。如果您有付費帳號，您也可以建立數個不同的電子郵件地址，全部由同一個帳號管理。這對於區分您的不同用途非常有用。



![proton](assets/notext/30.webp)



在「*資料夾和標籤*」標籤中，您可以建立資料夾和標籤來組織您的信箱。



![proton](assets/notext/31.webp)



*篩選器*」標籤可讓您管理收到的電子郵件的篩選器。



![proton](assets/notext/32.webp)



轉寄與自動回覆」標籤可讓您管理電子郵件的轉寄與自動回覆。



![proton](assets/notext/33.webp)



在「*網域名稱*」標籤中，您可以選擇使用自己的網域設定電子郵件 Address，如果您擁有自己的網站，這個選項會很有用。對於個人使用，不一定需要使用此功能。



![proton](assets/notext/34.webp)



* 加密和金鑰*」標籤可讓您管理電子郵件加密選項。對於新手使用者，一般不需要修改此部分的設定。



![proton](assets/notext/35.webp)



最後，「*IMAP/SMTP*」標籤可讓您設定一個橋樑來使用 ProtonMail 與 Outlook 或 Apple Mail 等電子郵件程式。



![proton](assets/notext/36.webp)



若要返回信箱首頁，請按一下左上角的「*信箱*」按鈕。



![proton](assets/notext/37.webp)

收件匣*」標籤包含您收到的電子郵件。
