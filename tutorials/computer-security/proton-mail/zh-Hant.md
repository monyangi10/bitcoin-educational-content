電子郵件信箱是您線上活動的核心元素，通常在您的電腦安全中扮演重要角色。如果攻擊者成功入侵您的電子郵件信箱，他們就可以透過「*忘記密碼*」功能輕鬆存取您的其他帳戶。這可讓他們控制您的社交網路、銀行帳戶和其他線上服務，因為現在電子郵件 Address 通常被用作您線上身分的唯一識別碼。因此，保護您的電子郵箱安全對於保護自己免受攻擊是非常重要的。


為了確保您電子郵件信箱的安全性，採取一些簡單的良好作法是很重要的，我們將在本教學中針對電腦初學者進行研究。選擇一個提供進階保護選項和健全隱私權保護政策的安全電子郵件供應商也很重要。這就是為什麼我在本教程中推薦使用 ProtonMail。即使您不希望使用此供應商，這裡介紹的良好作法也可應用於任何電子郵件信箱，以加強其安全性。


## 為什麼使用 ProtonMail？


ProtonMail 是一個相當安全的訊息傳輸解決方案，這要歸功於它的幾項功能。首先，ProtonMail 可確保電郵的端對端加密，這意味著只有寄件者和收件者才能閱讀電郵內容。理論上，即使是 ProtonMail 也無法存取用戶的電郵。這種加密是自動應用的，不需要用戶有任何特定的技術技能。


此外，ProtonMail 整合了先進的技術來保護您的隱私，包括封鎖某些追蹤系統和遮蔽您的 IP Address。由於 Proton 公司位於瑞士，因此可受惠於一些其他國家所沒有的資料保護法。此外，ProtonMail 是開放源碼的，獨立專家可以自由審核軟體程式碼。


Proton 的商業模式以訂閱系統為基礎，這讓人放心，因為這表示該公司的資金來源不一定是剝削用戶的資料。在本教程中，我們將探討如何使用 ProtonMail 的免費版本，但也有多個訂閱級別提供更多功能。這種商業模式比起完全免費的系統更為可取，因為完全免費的系統可能會讓人擔心我們的個人資料是否被用來牟利。幸好，ProtonMail 的情況似乎並非如此。


## 建立 Proton 帳戶


請造訪質子官方網站：https://proton.me/


![proton](assets/notext/01.webp)


按一下「*建立帳戶*」按鈕：

您可以根據自己的需要選擇不同的計劃。開始時，您可以選擇免費帳戶，讓您測試 ProtonMail 的基本服務。之後，如果您想使用其他功能和其他 Proton 軟件，如日曆、VPN 或密碼管理器，您可以考慮訂購付費計劃。


然後您就會到達帳戶建立頁面。


您可以按一下小箭頭，為您的電子郵件 Address 選擇您喜歡的網域名稱。此選擇對接下來的內容沒有影響。


此外，請選擇電子郵件 Address 的使用者名稱。


接著會要求您設定密碼。在此階段選擇一個強大的密碼非常重要，因為這將允許存取您的信箱。穩健的密碼應盡量長，使用多種字符，並隨機選擇。在 2024 年，安全密碼的最低建議為 13 個字元，包括數字、小寫和大寫字母以及符號，前提是密碼是真正隨機的。但是，我建議選擇至少 20 個字元的密碼，包括所有可能的字元類型，以確保其安全性更長久。


使用密碼管理器是非常好的做法。它不僅可以讓您安全地儲存密碼，而無需記住密碼，還可以為您 generate 長而隨機的密碼。人類確實很不擅長創建隨機序列，而不夠隨機的密碼很容易受到暴力攻擊。我也建議您參考我們關於設定密碼管理器的完整教學，以瞭解此主題的更多詳細資訊：


https://planb.network/tutorials/computer-security/authentication/bitwarden-0532f569-fb00-4fad-acba-2fcb1bf05de9

按一下「*建立帳戶*」按鈕。


解決 CAPTCHA。


選擇顯示名稱。這是您傳送電子郵件時會顯示給收件者的名稱。選擇您的真名或暱稱。

Proton 還提供您設定復原帳號方法的選項，可透過您的電話號碼或其他電子郵件 Address。請務必瞭解，此選項可能會增加您電子郵件收件箱的攻擊面。對您來說，這是額外的安全措施，讓您在忘記密碼的情況下重新取得帳號的存取權，但對駭客而言，這是額外試圖入侵您帳號的機會。您不需要選擇此復原選項，但如果您決定不選擇此復原選項，請務必保留一份安全的密碼副本。如果沒有它，一旦遺失密碼，您將無法恢復電子郵件收件箱的存取權。

![proton](assets/notext/12.webp)

![proton](assets/notext/13.webp)


一旦進入您信箱的 Interface，我建議您看一下自訂的設定。按一下右上角的齒輪圖示。


![proton](assets/notext/14.webp)


然後按一下「*所有設定*」按鈕。


![proton](assets/notext/15.webp)


在 「*儀表板*」標籤中，您可以找到與您帳戶相關的資訊。在此部分向下滾動，您可以選擇您願意收到的來自 Proton 的電子郵件類型。如果您不願意收到促銷或資訊通知，您可以選擇全部取消選擇。


![proton](assets/notext/16.webp)


在「*升級方案*」標籤中，您可以選擇具有新功能的付費方案。


![proton](assets/notext/17.webp)


在「*復原*」標籤中，您可以新增或修改您的復原方法。


![proton](assets/notext/18.webp)


在「*帳號和密碼*」標籤中，您可以變更使用者名稱，以及保護帳號安全的方法。


![proton](assets/notext/19.webp)


目前，您的信箱只能使用密碼保護。我建議您至少使用應用程式新增雙因素驗證保護。若要這樣做，請按一下核取方塊。

在「*語言和時間*」標籤中，您可以變更 Interface 語言以及時區。


![proton](assets/notext/23.webp)

*Filters*" 標籤可讓您管理所收到電子郵件的篩選條件。

![proton](assets/notext/39.webp)

![proton](assets/notext/40.webp)


寫下您的訊息。

![proton](assets/notext/42.webp)


然後，您可以在「*已傳送」標籤中找到已傳送的訊息。

收件匣*」標籤包含您已收到的電子郵件。
