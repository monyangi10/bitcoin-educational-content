Im digitalen Zeitalter der Datenerfassung ist der Online-Datenschutz für uns Internetnutzer zu einem wichtigen Thema geworden. Zwischen der Verfolgung durch Werbung, den Risiken des Hackens über öffentliche Netzwerke und geografischen Beschränkungen wenden sich immer mehr Nutzer an VPNs (*Virtual Private Networks*), um ihr Surfen zu sichern. Unter den zahlreichen Optionen, die ihnen zur Verfügung stehen, sticht ein Dienst der Mozilla-Stiftung, die für ihr Commitment für ein freies und ethisches Internet bekannt ist, hervor. In diesem Lernprogramm werfen wir einen Blick auf Mozilla VPN, um die Kontrolle über Ihre Internet-Privatsphäre zu übernehmen.



## Was ist Mozilla VPN?



Ein ***Virtuelles Privates Netzwerk*** (VPN) ist ein System zur Herstellung einer direkten Verbindung zwischen entfernten Computern, die an verschiedene lokale Netzwerke angeschlossen sind. Mit anderen Worten, es ist ein System, das Ihren Datenaustausch vom restlichen Internetverkehr isoliert und verschlüsselt. Wenn Sie mehr über VPNs, ihre Verwendung und ihre Vorteile erfahren möchten, schauen Sie sich den Kurs SCU 101 an:



https://planb.network/courses/99c46148-7080-4915-a7e0-9df0e145cd47

Basierend auf diesem Prinzip ist [Mozilla VPN] (https://www.mozilla.org/fr/products/vpn/download/) ein Open-Source-VPN-Dienst, der 2020 von der Mozilla Foundation entwickelt wurde. Er ist verfügbar auf:





- Android,
- iOS,
- Mac,
- Linux,
- Fenster,
- und auch als Erweiterung für Mozillas Firefox-Browser.



![download](assets/fr/01.webp)



Es ist in mehr als 30 Ländern verfügbar und verfügt über mehr als 500 Server, die für die Maskierung Ihrer IP Address verantwortlich sind, um Sie umzusiedeln und gleichzeitig die Vertraulichkeit Ihrer Interaktionen im Internet zu gewährleisten. Mozilla VPN zeichnet sich aus durch:





- Benutzerfreundlichkeit: eine schlanke, minimalistische Interface-Grafik, die Ihnen die wichtigsten Server und Länder zeigt, aus denen Sie wählen können.





- WireGuard-Technologie: ein Kommunikationsprotokoll und eine Open-Source-Software, die modernste Kryptografie zur Erstellung verschlüsselter Tunnel verwendet und eine leichtgewichtige, einfacher zu implementierende Alternative mit einer kleineren Codebasis und einem Schwerpunkt auf Geschwindigkeit und Sicherheit bietet.





- Transparente Preise: rund 10 Euro für eine monatliche Gebühr und 5 Euro pro Monat für ein Jahresabonnement.





- Mehrere verbundene Geräte: Verbinden Sie bis zu 5 Geräte gleichzeitig mit Ihrem Mozilla VPN-Konto.



## Erste Schritte mit Mozilla VPN



Sie können [Mozilla VPN] (https://www.mozilla.org/fr/products/vpn/download/) je nach Betriebssystem herunterladen. In diesem Tutorial werden wir uns Mozilla VPN unter dem Windows-Betriebssystem ansehen.



⚠️ Da Mozilla VPN in einigen Ländern nicht verfügbar ist, kann diese Meldung erscheinen, wenn Sie die Mozilla VPN-Anwendung aus dem Google Play Store herunterladen.



![playstore](assets/fr/02.webp)



Nach erfolgreicher Installation klicken Sie auf die Schaltfläche **Registrieren**, um ein neues Konto zu erstellen.



![inscription](assets/fr/03.webp)



Geben Sie Ihr Passwort ein und bestätigen Sie Ihr Konto, indem Sie den OTP-Code eingeben, der an Ihre E-Mail Address gesendet wird. Da Mozilla VPN ein kostenpflichtiger Dienst der Mozilla Foundation ist, benötigen Sie ein Abonnement, um Mozilla VPN in vollem Umfang nutzen zu können. Klicken Sie auf "Jetzt abonnieren", um auf die Mozilla VPN-Preisseite weitergeleitet zu werden.



![tarifs](assets/fr/04.webp)



Wählen Sie dann den für Sie passenden Abonnementplan.



![plans](assets/fr/05.webp)



Fahren Sie dann mit der Zahlung per PayPal oder Kreditkarte fort.



![paiement](assets/fr/06.webp)



Sobald Sie Ihr Abonnement aktiviert haben, öffnen Sie die Mozilla VPN-Software und erlauben Sie Mozilla VPN, technische Daten über Ihr Surfverhalten zu sammeln oder nicht.



![collecte](assets/fr/07.webp)



Sie können auch Optionen zur Werbeunterdrückung, Browserüberwachung und Malware-Blockierung aktivieren.



![privacy](assets/fr/08.webp)



Sobald der Konfigurationsprozess abgeschlossen ist, sieht Mozilla VPN Interface wie folgt aus:



![home](assets/fr/09.webp)



Sie können das VPN aktivieren, indem Sie auf die Optionsschaltfläche unten klicken, wodurch die IP Address Ihres Computers auf eine Reihe von IP-Adressen an dem von Ihnen gewählten Standort verlegt wird. Sie können die Liste der Geräte, die mit Ihrem Mozilla VPN-Konto verbunden sind, auch direkt auf der Startseite einsehen.



![activate](assets/fr/10.webp)



Mit Mozilla VPN können Sie Ihren Standort in zwei Formaten auswählen:





- Single-Hop: Die IP Address Ihres Computers wird an einen Server in einer bestimmten Region, in unserem Beispiel Sofia in Weißrussland, verlegt und die Daten werden verschlüsselt.





- Multi-Hop: stellt eine verschlüsselte Verbindung von Ihrem Computer zu zwei entfernten Servern her. Dabei handelt es sich um eine doppelte Verschlüsselung: Ihre Daten werden über Server A verschlüsselt, von Server A aus werden die Daten dann erneut zu Server B verschlüsselt.



![hops](assets/fr/11.webp)



Wenn Sie auf das Symbol **Einstellungen** klicken, können Sie auf die verschiedenen Anpassungsoptionen zugreifen, die Mozilla VPN bietet. Im Menü **Netzwerkeinstellungen** können Sie Mozilla VPN für alle Ihre Anwendungen konfigurieren oder die Anwendungen auswählen, für die Sie Mozilla VPN nicht verwenden möchten. Diese Option ist eine der Innovationen, die Mozilla VPN von den meisten anderen VPNs abhebt: Tunnelsplitting.



![permissions](assets/fr/12.webp)



Split-Tunneling ist eine praktische Funktion, mit der Sie einen Teil Ihres Internetverkehrs über ein VPN und einen Teil ohne VPN während derselben Sitzung leiten können. Dies kann für Online-Banking oder anderen Internetverkehr nützlich sein, der mit einem VPN nicht richtig funktioniert.



⚠️Mozilla VPN bietet Split-Tunneling auf allen Produkten außer MacOs und iOS. Apple-Nutzer können diese Funktion leider nicht nutzen. Es ist wahrscheinlich, dass sie es tun werden, da sie diese Funktion nur über ihre Android-App zugelassen haben und sie nun auf andere Betriebssysteme ausgeweitet wird.



Mozilla VPN verfügt über ein **Kill Switch**-System, das es ermöglicht, Ihre Internetverbindung zu unterbrechen, wenn das VPN aus irgendeinem Grund zusammenbricht. Dies schützt Ihre IP Address und andere persönliche Informationen.
