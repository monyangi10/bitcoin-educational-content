Fedora ni ubuhinga bwo gukoresha Linux ku buntu, bufunguye, bwatangujwe mu mwaka w’2003, bwateguwe n’umuryango wa **Fedora Project** kandi bushigikiwe na **Red Hat Linux**. Irazwi cane kubera ukuntu idahinduka, ikora neza kandi yoroshe gukoresha, ivyo bikaba bituma iba ihitamwo ryiza cane ku batangura n’abateye imbere. Iyo sisitemu ikoreshwa ku bikoresho vyinshi vy’ubuhinga bwa none, ivyo bikaba bituma vyoroha kuyishira kuri orodinateri hafi iyo ari yo yose. Fedora iraboneka kandi mu bitabu vyinshi vyatunganijwe imbere y'igihe, vyitwa "Fedora Spins" canke "Fedora Editions", vyateguwe ku bikenewe vyihariye (imikino ya videwo, ivy'inyenyeri, iterambere...).



## Ubwubatsi bwa Fedora Linux



Nk’uko mwabisomye mbere, Fedora ni ubuhinga bwo gukoresha ku buntu bushingiye ku ntumbero ya Linux. Linux kernel ni igice ca sisitemu ikoresha kivugana n’ibikoresho vya mudasobwa kandi kigacungera ibikoresho vya sisitemu nk’ubuhinga bwo kwibuka n’ubushobozi bwo gukora.



Fedora Linux irimwo ibikoresho bitandukanye vy’ubuhinga n’ibikoresho bisabwa kugira ngo ukoreshe ubuhinga bwo gukoresha hejuru y’umubiri wa Linux. Ubwubatsi bwa Fedora bufise ibice bisigura ko bugizwe ahanini n’ibice bimwebimwe bishobora kwongerwako, gukurwako canke gusubirwamwo bitagoranye uko bisabwa. Ivyo bituma ushobora guhingura ubuhinga bwo gukoresha ukoresheje gusa ivyo ukeneye.



Fedora kandi irimwo ibidukikije vy’ibiro, ari vyo Interface abakoresha bakoresha mu gukora ibikorwa no gushika ku bikoresho. Igikoresho ca Fedora ni GNOME, gikoreshwa neza, gikoreshwa neza kandi gishobora guhindurwa cane.



## Ni kuki uhisemwo Fedora?



Mu bikoresho vyinshi vya Linux biriho, Fedora iratandukanye cane cane kubera:





- Modularity**: Ihuye n’imiterere itandukanye y’ibikoresho, Fedora ishobora gushirwa kuri mudasobwa nyinshi, mbere n’izikoresha ubushobozi buke, igahuye neza n’ivyo ukeneye.





- **Interface** yoroshe kandi igaragara: Fedora ifatanya Interface y’iki gihe n’umurongo w’amabwirizwa ukomeye Interface, bikaba bituma vyoroha gukoresha ku mirongo yose.





- Kernel stability**: Ishingiye kuri Red Hat, Fedora izwi cane kubera ukwizigirwa kw’ivyo ihindura, cane cane ivyo ihindura, ivyo bikaba bikorwa ata bibazo bikomeye bivuye ku ntererano z’ubuntu zivuye mu muryango munini.





- Gushiramwo vyihuta, vyoroshe**: n’ubunini bw’ishusho bwa 3 GB gusa, gushiramwo biranyaruka kandi biroroshe, mbere no ku mashini zifise uburyo buke.



## Ivyasohowe vya Fedora



Bivanye n’ivyo ukora n’ingene ukoresha, Fedora itanga ibitabu bihuye n’ivyo ukeneye. Uzosanga ahanini:





- Fedora Workstation**: Ni nziza cane ku mudasobwa zawe bwite no/canke mu kazi, iyi nteguro ishizwemwo ibikoresho rusangi nk’ibikoresho vyo gucukura, ibiro (abahindura inyandiko), n’ibikoresho vyo gusubiramwo amakuru.





- Fedora Server**: Iyi nteguro ni iyo gucunga server. Fedora Server irimwo ibikoresho bitandukanye bigufasha gukoresha no gucunga ama server ku rugero rwawe bwite.





- Fedora CoreOS**: Ushaka gukoresha no gukoresha mu buryo bworoshe ibikorwa vyo mu gicu? Fedora CoreOS ni igitabu kiguha ibikoresho vyo kurema no gucunga amashusho ukoresheje Docker na Kubernets, nk’akarorero.



Muri iyi nyigisho, tuzoba turiko turatwara igitabu ca Fedora Workstation. Ariko rero, inzira zivugwa mu buryo burambuye aha hepfo ni nk’izo nyene ku zindi nteguro.

### Gushiramwo Fedora



Niwaheza gufungura urufunguzo rwawe rwa USB, uzime mudasobwa yawe.


Hindukiza mudasobwa yawe, hanyuma winjire muri BIOS mu gihe co gutangura ukanda urufunguzo `F2`, `F12` canke `ESC`, bivanye n’umudasobwa wawe.



Mu mahitamwo yo gufungura, hitamwo urufunguzo rwawe rwa USB nk’igikoresho nyamukuru co gufungura. Mu kwemeza iryo hitamwo, mudasobwa yawe izosubira gukora maze ihite ifungura **Fedora installer** iri ku rufunguzo rwa USB.



Iyo mudasobwa yawe imaze gufunguka kuri USB, **GRUB screen** iraboneka.



Kuri iyi ntambwe, ufise amahitamwo akurikira:




- Test media**: Iyi nzira ishobora kugufasha kugenzura ubutungane bw’inkoni ya USB no kumenya neza ko ibintu vyose bisabwa kugira ngo ushiremwo neza biriho. Iyi ni intambwe y’ubusa, ariko iraremeshwa nimba ufise amakenga ku bijanye n’inkoni ya USB.



![install](assets/fr/06.webp)



![testing](assets/fr/07.webp)





- Gutangura Fedora**: Ivyo bica bitanguza Fedora mu buryo "ubuzima", ata gushiramwo.



Ku biro vya Fedora (uburyo bwo kubaho), fyonda ku **Shiraho Fedora** (canke Shiraho kuri disiki ya Hard) kugira ngo utangure igikorwa co gushiramwo. Ushobora guhitamwo gushiramwo mu nyuma maze ukagerageza Fedora utabwirizwa kuyishiramwo.

Iyo disiki iri ubusa, Fedora izoca ikoresha umwanya wose uriho. Ahandiho, urashobora guhindura uko ugabanya (n’amaboko canke n’ivyo ukora).

Kuri iyi ntambwe, Fedora asaba gushiramwo amakuru kuri disiki yawe kugira ngo wongereko Layer y’umutekano. Ivyo bituma amakuru yawe ashobora gusomwa gusa na sisitemu yawe ya Fedora.

## Tora ibikorwa vy'ishimikiro



### Gusura kuri Internet


Mucukumbuzi mburabuzi ya Fedora ni **Firefox**. Biroroshe gukoresha kandi birabereye ku bikenewe vyinshi.


Niba ushaka uwundi mucukumbuzi, ushobora kuwushiramwo ukoresheje **umucungerezi wa porogaramu** canke ukoresheje **terminal**.


### Gutunganya amajambo


Fedora irimwo **LibreOffice** ibiro vy'ibiro ku buryo busanzwe, bitanga ibikoresho vyinshi vy'ingirakamaro:




- Umwanditsi** wo gutunganya amajambo.
- Calc** ku mpapuro z'ibarabara.
- Impress** kugira ngo ureme ibiganiro.


## Gushiramwo porogaramu


Kugira ngo ushiremwo porogaramu nshasha, ushobora gukoresha **umuyobozi wa porogaramu** ya Fedora (yitwa _Software_), ivyo bikaba bituma gushiramwo porogaramu vyoroha kandi bigaragara.  Ariko rero, gukoresha **terminal** kenshi biranyaruka kandi bikaba ari ukuri.



Imbere yo gushiramwo porogarama iyo ari yo yose, wama wibuka guhindura **ububiko** kugira ngo umenye neza ko ushobora kuronka verisiyo nshasha ziriho.



Hanyuma ukoreshe itegeko rikurikira kugira ngo utangure gushiramwo porogaramu wipfuza:


sudo dnf gushiramwo porogaramu_izina`


## Kuvugurura ubuhinga bwawe bwo gukoresha


Inyuma yo gushiramwo, birahambaye guhindura Fedora kugira ngo ukoreshe ivyiza bishasha vy’umutekano n’ivy’ubuhinga bushasha.


### Ihitamwo rya 1: Biciye ku gishushanyo ca Interface




- Ugurure Fedora **Ivyagezwe**, hanyuma ugende ku gice ca **Sisitemu**.
- Fyonda kuri **Ivugurura rya porogaramu**.
- Tanga gukuraho ibintu bishasha maze urindire gushika igikorwa kirangiye.



**Gusubiramwo** bishobora gukenerwa iyo ivyagezwe bimaze gushirwaho.


### Amahitamwo ya 2: Biciye ku kibanza




- Gufungura terminal maze utangure guhindura **ububiko** kugira ngo umenye neza ko ufise verisiyo nshasha za:



```shell
sudo dnf check-update
```





- Ibikurikira, hindura porogaramu zose zashizwemwo ukoresheje iri tegeko rikurikira:



```shell
sudo dnf upgrade
```
