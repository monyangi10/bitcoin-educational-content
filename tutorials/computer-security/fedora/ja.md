Fedoraは、**Fedora Project**コミュニティによって開発され、**Red Hat Linux**によってサポートされている、2003年に開始されたフリーでオープンソースのLinuxベースのオペレーティングシステムです。Fedoraは、その安定性、優れたパフォーマンス、使いやすさで有名で、初心者にも上級者にも最適です。このシステムは、ほとんどの最新のプロセッサー・アーキテクチャーで動作するため、事実上どんなコンピューターにも簡単にインストールできます。Fedoraは、特定のニーズ（ビデオゲーム、天文学、開発...）のために設計された、"Fedoraスピン "または "Fedoraエディション "と呼ばれる、いくつかの設定済みエディションでも利用可能です。



## Fedora Linuxアーキテクチャ



先にお読みいただいたように、FedoraはLinuxカーネルをベースとしたフリーのオペレーティング・システムです。Linuxカーネルは、コンピュータのハードウェアと通信し、メモリや処理能力などのシステムリソースを管理するオペレーティングシステムの一部です。



Fedora Linuxには、Linuxカーネル上でオペレーティングシステムを実行するために必要な様々なソフトウェアツールとアプリケーションが含まれています。Fedora のモジュラー・アーキテクチャーは、主に個々のコンポーネントの集まりから構成されていることを意味し、必要に応じて簡単に追加、削除、交換することができます。これにより、必要なリソースだけを使用してオペレーティングシステムを形成することができます。



Fedora にはデスクトップ環境も含まれており、ユーザーがタスクを実行し、アプリケーションにアクセスするための Interface です。Fedoraのデフォルトのデスクトップ環境はGNOMEで、ユーザーフレンドリーで使いやすく、高度にカスタマイズ可能なデスクトップ環境です。



## Fedoraを選ぶ理由



数多くのLinuxディストリビューションがある中で、Fedoraは特に：





- **モジュール性**：異なるプロセッサーアーキテクチャーと互換性があるため、Fedoraはほとんどのコンピューターにインストールすることができます。





- シンプルで直感的な**Interface**：Fedoraは、現代的なグラフィカルInterfaceとパワフルなコマンドラインInterfaceを組み合わせ、すべてのプロファイルで使いやすくしています。





- **カーネルの安定性**：Red HatをベースとするFedoraは、そのアップデートの信頼性で有名で、特にカーネルアップデートは、大規模なコミュニティからの無償の貢献のおかげで大きなバグもなく実施されています。





- **迅速で簡単なインストール**：イメージサイズはわずか3GBで、リソースが限られたマシンでも迅速かつ簡単にインストールできます。



## Fedoraエディション



Fedoraは、あなたのプロファイルと使用方法に応じて、あなたのニーズに合ったエディションを提供しています。主に ：





- **Fedora Workstation**：このエディションには、ブラウザ、オフィススイート（テキストエディタ）、メディア再生ソフトウェアなどの一般的なユーティリティがインストールされています。





- **Fedora Server**：このエディションはサーバー管理に特化しています。Fedora Serverには、独自のスケールでサーバーを展開・管理するのに役立つ様々なツールが含まれています。





- **Fedora CoreOS**：クラウド・アプリケーションを簡単に実行、デプロイしたいですか？Fedora CoreOSは、例えばDockerやKubernetsでイメージを作成・管理するためのツールを提供するエディションです。



このチュートリアルでは、Fedora Workstation エディションを担当します。しかし、以下に詳述するプロセスは他のエディションでも同様です。



## Fedora Workstationのインストールと設定



Fedora Workstation をインストールするには、以下のハードウェア構成が必要です：




- オペレーティング・システムを起動するための少なくとも**8 GB**のUSBキー。
- パソコンのHardディスクに**40GB以上の空き容量**があること。
- 4GBのRAM**でスムーズな体験を。**



### Fedora Workstationのダウンロード



Fedora Workstation] エディション (https://fedoraproject.org/fr/workstation/download) は Fedora プロジェクトの公式ウェブサイトからダウンロードできます。そして、あなたのプロセッサー・アーキテクチャー（32-bit - 64-bit）に対応するバージョンを選択し、**Download**アイコンをクリックします。



![download](assets/fr/01.webp)



![telecharger](assets/fr/02.webp)


### ブート可能なUSBキーを作成する



Fedoraをインストールするには、[Balena Etcher](https://etcher.balena.io/)のようなソフトウェアを使ってブータブルUSBキーを作成する必要があります。



![flashOs](assets/fr/03.webp)



![flash](assets/fr/04.webp)



Balena Etcherのインストールが終わったら、アプリケーションを開き、ダウンロードしたFedora Workspace ISOイメージを選択します。保存先メディアとしてUSBキーを選択し、**Flash**ボタンをクリックしてブータブルキーの作成を開始します。



![boot](assets/fr/05.webp)


### Fedoraのインストール



USBキーの起動が完了したら、コンピュータの電源を切ります。


コンピュータの電源を入れ、お使いのコンピュータに応じて `F2`、`F12` または `ESC` キーを押して起動中に BIOS にアクセスします。



ブートオプションで、USB キーをプライマリブートデバイスとして選択します。この選択を確定すると、コンピューターが再起動し、USBキーにあるFedoraインストーラーが自動的に起動します。



コンピュータがUSBスティックから起動すると、**GRUB画面**が表示されます。



この段階では、以下の選択肢がある：




- テストメディア：このオプションにより、USB メモリの完全性をチェックし、正しいインストールに必要な依存関係がすべて存在することを確認できます。これはオプションのステップですが、USB メモリに疑問がある場合に推奨します。



![install](assets/fr/06.webp)



![testing](assets/fr/07.webp)





- **Fedora** を起動します：インストールせずに、「ライブ」モードで Fedora を起動します。



Fedora デスクトップ（ライブモード）で、**Install Fedora** （または Install on Hard disk）をクリックしてインストールプロセスを開始します。Fedora をインストールせずに、後でインストールしてテストすることもできます。



![install-live](assets/fr/08.webp)



最初のステップは Fedora の **インストール言語** と **キーボードレイアウト** を選択することです。



![language](assets/fr/10.webp)





- インストールディスクの選択 ：



Fedora をインストールする Hard ディスクを選択します。



ディスクが空の場合、Fedora は自動的に利用可能なスペースを全て使用します。そうでなければ、パーティション分割をカスタマイズできます（手動または自動）。



![disk](assets/fr/11.webp)





- 暗号化：



この段階で、Fedora はさらに Layer のセキュリティを追加するためにディスクを暗号化することを提案します。これは、あなたのデータがあなたの Fedora システムによってのみ読み取られることを保証します。



![chiffrement](assets/fr/12.webp)



インストールを開始する前に、Fedora はあなたの選択の概要を表示します。確認し、インストールボタンをクリックして Fedora のインストールを開始します。



![resume](assets/fr/13.webp)



インストール中、Fedora はファイルをコピーし、システムを設定します。終了すると、コンピューターは自動的に再起動します。



![installation](assets/fr/14.webp)

## 基本タスクの発見



### インターネットの閲覧


Fedoraのデフォルトブラウザは**Firefox**です。使いやすく、ほとんどのニーズに適しています。


他のブラウザーをお好みの場合は、**ソフトウェア・マネージャー**または**ターミナル**からインストールできます。


### ワープロ


Fedoraには**LibreOffice**オフィススイートがデフォルトで含まれており、いくつかの便利なツールを提供しています：




- ワープロ用ライター。
- 表計算ソフトの**Calc**。
- **Impress**でプレゼンテーションを作成。


## アプリケーションのインストール


新しいアプリケーションをインストールするには、Fedora の **ソフトウェアマネージャー** （_Software_と呼ばれます）を使用することができます。  しかし、** ターミナル** を使用する方がより速く、より正確なことがよくあります。



ソフトウェアをインストールする前に、常に**リポジトリ**を更新し、利用可能な最新バージョンにアクセスできるようにしてください。



次に、以下のコマンドを使用して、目的のアプリケーションのインストールを開始する：


sudo dnf install software_name`.


## オペレーティングシステムのアップデート


インストール後、最新のセキュリティパッチとソフトウェアアップデートを利用するために Fedora をアップデートすることが重要です。


### オプション1：Interfaceグラフィック経由




- Fedora **Settings**を開き、**System**セクションに進みます。
- **ソフトウェアアップデート**をクリックしてください。
- アップデートのダウンロードを開始し、プロセスが完了するまで待ちます。



アップデートがインストールされると、**再起動**が必要になる場合があります。


### オプション2：ターミナル経由




- ターミナルを開き、.NET Frameworkの最新版があることを確認するため、**リポジトリ**の更新から始める：



```shell
sudo dnf check-update
```





- 次に、インストールされているすべてのソフトウェアを以下のコマンドでアップデートする：



```shell
sudo dnf upgrade
```
