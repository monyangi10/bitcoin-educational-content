Fedora เป็นระบบปฏิบัติการแบบโอเพ่นซอร์สที่ใช้ Linux ฟรี เปิดตัวในปี 2003 พัฒนาโดยชุมชน **Fedora Project** และได้รับการสนับสนุนจาก **Red Hat Linux** มีชื่อเสียงในด้านความเสถียร ประสิทธิภาพที่ดี และใช้งานง่าย ทำให้เป็นตัวเลือกที่ยอดเยี่ยมสำหรับผู้เริ่มต้นและผู้ใช้ขั้นสูง ระบบสามารถทำงานบนสถาปัตยกรรมโปรเซสเซอร์สมัยใหม่ส่วนใหญ่ ทำให้ง่ายต่อการติดตั้งบนคอมพิวเตอร์แทบทุกเครื่อง Fedora ยังมีให้เลือกในหลายรุ่นที่กำหนดค่าไว้ล่วงหน้า เรียกว่า "Fedora Spins" หรือ "Fedora Editions" ซึ่งออกแบบมาเพื่อตอบสนองความต้องการเฉพาะ (วิดีโอเกม ดาราศาสตร์ การพัฒนา...)



## สถาปัตยกรรม Fedora Linux



ตามที่คุณได้อ่านก่อนหน้านี้ Fedora เป็นระบบปฏิบัติการฟรีที่พัฒนาขึ้นบนเคอร์เนลของลินุกซ์ เคอร์เนลของลินุกซ์เป็นส่วนหนึ่งของระบบปฏิบัติการที่ทำหน้าที่สื่อสารกับฮาร์ดแวร์ของคอมพิวเตอร์และจัดการทรัพยากรของระบบ เช่น หน่วยความจำและพลังการประมวลผล



Fedora Linux รวมเครื่องมือซอฟต์แวร์และแอปพลิเคชันหลากหลายที่จำเป็นสำหรับการรันระบบปฏิบัติการบนเคอร์เนลของ Linux สถาปัตยกรรมแบบโมดูลาร์ของ Fedora หมายความว่ามันประกอบด้วยส่วนประกอบแต่ละชิ้นที่สามารถเพิ่ม ลบ หรือแทนที่ได้อย่างง่ายดายตามที่ต้องการ สิ่งนี้ช่วยให้คุณสามารถปรับแต่งระบบปฏิบัติการโดยใช้เฉพาะทรัพยากรที่คุณต้องการเท่านั้น



Fedora ยังรวมถึงสภาพแวดล้อมเดสก์ท็อป ซึ่งคือ Interface ที่ผู้ใช้ใช้ในการทำงานและเข้าถึงแอปพลิเคชัน สภาพแวดล้อมเดสก์ท็อปเริ่มต้นของ Fedora คือ GNOME ซึ่งเป็นสภาพแวดล้อมเดสก์ท็อปที่ใช้งานง่ายและปรับแต่งได้สูง



## ทำไมต้องเลือก Fedora?



ท่ามกลางการกระจายตัวของลีนุกซ์ที่มีอยู่มากมาย Fedora โดดเด่นเป็นพิเศษสำหรับ:





- โมดูลาร์ลิตี้**: เข้ากันได้กับสถาปัตยกรรมโปรเซสเซอร์ต่างๆ, Fedora สามารถติดตั้งบนคอมพิวเตอร์ส่วนใหญ่ได้ แม้แต่เครื่องที่มีพลังงานต่ำ ปรับให้เข้ากับความต้องการของคุณได้อย่างสมบูรณ์แบบ





- Interface ที่เรียบง่ายและเข้าใจง่าย **Interface**: Fedora ผสมผสาน Interface แบบกราฟิกสมัยใหม่เข้ากับ Interface แบบบรรทัดคำสั่งที่ทรงพลัง ทำให้ง่ายต่อการใช้งานสำหรับทุกโปรไฟล์





- ความเสถียรของเคอร์เนล**: ตามข้อมูลจาก Red Hat, Fedora มีชื่อเสียงในด้านความน่าเชื่อถือของการอัปเดต โดยเฉพาะการอัปเดตเคอร์เนล ซึ่งดำเนินการโดยไม่มีบั๊กใหญ่ ๆ ขอบคุณการสนับสนุนฟรีจากชุมชนขนาดใหญ่





- การติดตั้งที่รวดเร็วและง่ายดาย**: ด้วยขนาดภาพเพียง 3 GB การติดตั้งจึงรวดเร็วและง่ายดาย แม้ในเครื่องที่มีทรัพยากรจำกัด



## Fedora editions



ขึ้นอยู่กับโปรไฟล์และการใช้งานของคุณ Fedora มีรุ่นต่างๆ ที่เหมาะกับความต้องการของคุณ คุณจะพบหลักๆ ว่า:





- Fedora Workstation**: เหมาะสำหรับการใช้งานส่วนตัวและ/หรือการใช้งานในระดับมืออาชีพบนคอมพิวเตอร์ของคุณ รุ่นนี้ติดตั้งด้วยโปรแกรมอรรถประโยชน์ทั่วไป เช่น เบราว์เซอร์ ชุดสำนักงาน (โปรแกรมแก้ไขข้อความ) และซอฟต์แวร์เล่นสื่อ





- Fedora Server**: รุ่นนี้ทุ่มเทให้กับการจัดการเซิร์ฟเวอร์ Fedora Server รวมเครื่องมือหลากหลายเพื่อช่วยคุณในการปรับใช้และจัดการเซิร์ฟเวอร์ในขนาดของคุณเอง





- Fedora CoreOS**: ต้องการรันและปรับใช้แอปพลิเคชันบนคลาวด์ได้อย่างง่ายดายหรือไม่? Fedora CoreOS คือรุ่นที่ให้คุณมีเครื่องมือในการสร้างและจัดการอิมเมจด้วย Docker และ Kubernets เป็นต้น



ในบทแนะนำนี้ เราจะดูแลรุ่น Fedora Workstation อย่างไรก็ตาม กระบวนการที่อธิบายด้านล่างมีความคล้ายคลึงกันสำหรับรุ่นอื่นๆ



## การติดตั้งและกำหนดค่า Fedora Workstation



การติดตั้ง Fedora Workstation ต้องการการกำหนดค่าฮาร์ดแวร์ดังต่อไปนี้:




- คีย์ USB ขนาดอย่างน้อย **8 GB** เพื่อบูตระบบปฏิบัติการ

## ค้นพบงานพื้นฐาน



### การท่องอินเทอร์เน็ต


เบราว์เซอร์เริ่มต้นของ Fedora คือ **Firefox** ใช้งานง่ายและเหมาะสำหรับความต้องการส่วนใหญ่


หากคุณต้องการเบราว์เซอร์อื่น คุณสามารถติดตั้งได้จาก **ตัวจัดการซอฟต์แวร์** หรือผ่านทาง **เทอร์มินัล**


### การประมวลผลคำ


Fedora รวมชุดโปรแกรมสำนักงาน **LibreOffice** โดยค่าเริ่มต้น ซึ่งมีเครื่องมือที่มีประโยชน์หลายอย่าง:




- Writer** สำหรับประมวลผลคำ
- Calc** สำหรับสเปรดชีต
- Impress** เพื่อสร้างงานนำเสนอ


## การติดตั้งแอปพลิเคชัน


ในการติดตั้งแอปพลิเคชันใหม่ คุณสามารถใช้ **ตัวจัดการซอฟต์แวร์** ของ Fedora (เรียกว่า _Software_) ซึ่งทำให้การติดตั้งง่ายและมีภาพ อย่างไรก็ตาม การใช้ **เทอร์มินัล** มักจะเร็วกว่าและแม่นยำกว่า



ก่อนติดตั้งซอฟต์แวร์ใด ๆ อย่าลืมอัปเดต **repositories** เสมอเพื่อให้แน่ใจว่าคุณสามารถเข้าถึงเวอร์ชันล่าสุดที่มีอยู่ได้



จากนั้นใช้คำสั่งต่อไปนี้เพื่อเริ่มการติดตั้งแอปพลิเคชันที่ต้องการ:


sudo dnf install software_name`


## กำลังอัปเดตระบบปฏิบัติการของคุณ


หลังการติดตั้ง สิ่งสำคัญคือต้องอัปเดต Fedora เพื่อใช้ประโยชน์จากแพตช์ความปลอดภัยล่าสุดและการอัปเดตซอฟต์แวร์

- คลิกที่ **Software update**.
- เริ่มดาวน์โหลดการอัปเดตและรอจนกว่ากระบวนการจะเสร็จสิ้น



อาจจำเป็นต้อง **รีสตาร์ท** หลังจากติดตั้งการอัปเดตแล้ว


### ตัวเลือกที่ 2: ผ่านทางเทอร์มินัล




- เปิดเทอร์มินัลและเริ่มต้นด้วยการอัปเดต **repositories** เพื่อให้แน่ใจว่าคุณมีเวอร์ชันล่าสุดของ:



```shell
sudo dnf check-update
```





- ถัดไป อัปเดตซอฟต์แวร์ที่ติดตั้งทั้งหมดด้วยคำสั่งต่อไปนี้:



```shell
sudo dnf upgrade
```



ตอนนี้ระบบ Fedora ของคุณได้รับการอัปเดตและพร้อมใช้งานสำหรับงานประจำวันทั้งหมดของคุณแล้ว ค้นหาบทแนะนำของเราเกี่ยวกับ Linux Mint ซึ่งเป็นการแจกจ่าย Linux อีกตัวหนึ่ง และวิธีการตั้งค่าสภาพแวดล้อมที่ปลอดภัยและมีสุขภาพดีสำหรับการทำธุรกรรม Bitcoin ของคุณ
