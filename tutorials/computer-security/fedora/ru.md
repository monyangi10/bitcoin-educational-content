Fedora - это бесплатная операционная система на базе Linux с открытым исходным кодом, выпущенная в 2003 году, разработанная сообществом **Fedora Project** и поддерживаемая **Red Hat Linux**. Она известна своей стабильностью, высокой производительностью и простотой использования, что делает ее отличным выбором как для новичков, так и для опытных пользователей. Система работает на большинстве современных процессорных архитектур, что позволяет легко установить ее практически на любой компьютер. Fedora также доступна в нескольких предварительно сконфигурированных редакциях, называемых "Fedora Spins" или "Fedora Editions", предназначенных для конкретных нужд (видеоигры, астрономия, разработка...).



## Архитектура Fedora Linux



Как вы уже читали, Fedora - это свободная операционная система, основанная на ядре Linux. Ядро Linux - это часть операционной системы, которая взаимодействует с аппаратным обеспечением компьютера и управляет системными ресурсами, такими как память и вычислительная мощность.



Fedora Linux включает в себя множество программных инструментов и приложений, необходимых для работы операционной системы на базе ядра Linux. Модульная архитектура Fedora означает, что она состоит в основном из набора отдельных компонентов, которые можно легко добавлять, удалять или заменять по мере необходимости. Это позволяет формировать операционную систему, используя только те ресурсы, которые вам необходимы.



Fedora также включает среду рабочего стола, которая представляет собой Interface, с помощью которой пользователи выполняют задачи и получают доступ к приложениям. Среда рабочего стола Fedora по умолчанию - GNOME, удобная, простая в использовании и очень настраиваемая среда рабочего стола.



## Почему стоит выбрать Fedora?



Среди множества доступных дистрибутивов Linux, Fedora выделяется, в частности, тем, что:





- Модульность**: Совместимая с различными архитектурами процессоров, Fedora может быть установлена на большинство компьютеров, даже маломощных, идеально адаптируясь к вашим потребностям.





- Простой, интуитивно понятный Interface**: Fedora сочетает в себе современный графический Interface и мощную командную строку Interface, что делает ее простой в использовании для всех профилей.





- Стабильность ядра**: Основанная на Red Hat, Fedora известна надежностью своих обновлений, особенно обновлений ядра, которые выполняются без серьезных ошибок благодаря бесплатному вкладу большого сообщества.





- Быстрая и простая установка**: размер образа составляет всего 3 ГБ, поэтому установка выполняется быстро и легко даже на машинах с ограниченными ресурсами.



## Издания Fedora



В зависимости от вашего профиля и особенностей использования, Fedora предлагает редакции, соответствующие вашим потребностям. В основном вы найдете:





- Fedora Workstation**: Идеально подходит для личного и/или профессионального использования на ваших компьютерах. В этой редакции установлены общие утилиты, такие как браузеры, офисный пакет (текстовые редакторы) и программы для воспроизведения мультимедиа.





- Fedora Server**: Это издание посвящено управлению серверами. Fedora Server включает в себя множество инструментов, которые помогут вам развернуть и управлять серверами собственного масштаба.





- Fedora CoreOS**: Хотите легко запускать и развертывать облачные приложения? Fedora CoreOS - это издание, которое предоставляет вам инструменты для создания и управления образами, например, с помощью Docker и Kubernets.



В этом руководстве мы будем работать с редакцией Fedora Workstation. Однако описанные ниже процессы аналогичны и для других редакций.



## Установка и настройка Fedora Workstation



Для установки Fedora Workstation требуется следующая конфигурация оборудования:




- USB-носитель объемом не менее **8 ГБ** для загрузки операционной системы.
- Не менее **40 ГБ свободного места** на диске Hard вашего компьютера.
- 4 ГБ оперативной памяти** для плавной работы.



### Загрузите рабочую станцию Fedora



Вы можете загрузить редакцию [Fedora Workstation] (https://fedoraproject.org/fr/workstation/download) с официального сайта проекта Fedora. Затем выберите версию, соответствующую архитектуре вашего процессора (32-бит - 64-бит), и нажмите на значок **Загрузить**.

Чтобы установить Fedora, необходимо создать загрузочный USB-носитель с помощью такого программного обеспечения, как [Balena Etcher](https://etcher.balena.io/).

### Установка Fedora



После завершения загрузки USB-носителя выключите компьютер.


Включите компьютер, затем войдите в BIOS во время запуска, нажав клавишу `F2`, `F12` или `ESC`, в зависимости от типа компьютера.



В параметрах загрузки выберите USB-носитель в качестве основного загрузочного устройства. Подтвердив этот выбор, вы перезагрузите компьютер и автоматически запустите программу установки Fedora**, находящуюся на USB-носителе.



После загрузки компьютера с USB-накопителя появится экран **GRUB**.



На этом этапе у вас есть следующие возможности:




- Проверить носитель**: Эта опция позволяет проверить целостность USB-накопителя и убедиться в наличии всех зависимостей, необходимых для корректной установки. Это необязательный шаг, но он рекомендуется, если у вас есть какие-либо сомнения относительно USB-накопителя.



![install](assets/fr/06.webp)



![testing](assets/fr/07.webp)





- Запустить Fedora**: Запускает Fedora в "живом" режиме, без установки.



На рабочем столе Fedora (в режиме реального времени) нажмите **Install Fedora** (или Install on Hard disk), чтобы начать процесс установки. Вы можете выбрать более позднюю установку и протестировать Fedora без необходимости ее установки.



![install-live](assets/fr/08.webp)



Первым шагом будет выбор **языка установки** и **разметки клавиатуры** Fedora



![language](assets/fr/10.webp)





- Выбор установочного диска:



Выберите диск Hard, на который вы хотите установить Fedora.



Если диск пуст, Fedora автоматически использует все доступное пространство. В противном случае вы можете настроить разметку (вручную или автоматически).



![disk](assets/fr/11.webp)





- Шифрование:



На этом этапе Fedora предлагает зашифровать ваш диск, чтобы добавить дополнительный Layer безопасности. Это гарантирует, что ваши данные могут быть прочитаны только системой Fedora.



![chiffrement](assets/fr/12.webp)



Перед запуском установки Fedora отобразит краткую информацию о выбранных вами вариантах. Подтвердите его и нажмите на кнопку установки, чтобы начать установку Fedora.



![resume](assets/fr/13.webp)



Во время установки Fedora копирует файлы и настраивает систему. После завершения установки компьютер автоматически перезагружается.

## Узнайте об основных задачах



### Просмотр Интернета


Браузер по умолчанию в Fedora - **Firefox**. Он прост в использовании и подходит для большинства нужд.


Если вы предпочитаете другой браузер, вы можете установить его из **менеджера программного обеспечения** или через **терминал**.


### Работа с текстом


Fedora по умолчанию включает офисный пакет **LibreOffice**, который предлагает несколько полезных инструментов:




- Writer** для обработки текстов.
- Calc** для работы с электронными таблицами.
- Impress** для создания презентаций.


## Установка приложений


Для установки новых приложений вы можете использовать **менеджер программного обеспечения** (называется _Software_) Fedora, который делает установку простой и наглядной.  Однако использование **терминала** зачастую быстрее и точнее.



Перед установкой любого программного обеспечения всегда помните об обновлении **репозиториев**, чтобы убедиться, что у вас есть доступ к последним версиям.



Затем выполните следующую команду, чтобы запустить установку нужного приложения:


sudo dnf install software_name`


## Обновление операционной системы


После установки важно обновить Fedora, чтобы воспользоваться последними исправлениями безопасности и обновлениями программного обеспечения.


### Вариант 1: График Interface




- Откройте Fedora **Настройки**, затем перейдите в раздел **Система**.
- Нажмите на **Обновление программного обеспечения**.
- Начните загрузку обновлений и дождитесь завершения процесса.



После установки обновлений может потребоваться **перезагрузка**.


### Вариант 2: Через терминал




- Откройте терминал и начните с обновления **репозиториев**, чтобы убедиться, что у вас есть последние версии:



```shell
sudo dnf check-update
```





- Затем обновите все установленное программное обеспечение с помощью следующей команды:



```shell
sudo dnf upgrade
```
