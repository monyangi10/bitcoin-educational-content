## I.介绍



我们在启动最喜欢的浏览器时都会这样做：安装一个**广告拦截器**（ad blocker）。然而，在使用电视浏览器或安卓设备等时......要找到有效的拦截器就比较麻烦了。而且，如果家里有不止一台设备，那么每个浏览器都要重复操作！



在本教程中，我们将解决一个简单的问题：为网络上的所有机器提供广告拦截器，并对其进行集中管理。



为此，我们将使用专门开发的工具： **皮孔**



Pi-Hole 是一个 DNS 天坑。它将利用你的设备发出的 DNS 请求来验证或拒绝流量，从而保护你免受已知的传播广告、恶意软件等的地址和域的影响。



DNS 是域名系统的缩写。那么什么是域名呢？it-connect.fr "就是一个例子。域名是一个或多个资源的唯一标识符，通常由一个实体管理。



机器名称加上域名称为 FQDN，即 "完全合格域名"。它允许你通过 "调用 "特定的机器来达到目的。例如，当你输入 "www.trucmachin.com "时，你实际上是在调用属于 "trucmachin.com "域名的机器 "www"。



只不过，我们的计算机不懂人类语言，只懂二进制，所以它们需要一个 IP Address（相当于电话号码）才能访问网站。



因此，每次在浏览器中输入网站名称或点击链接时，电脑首先会向 DNS 服务器询问与该名称相对应的 IP Address。



**然后，Pi-Hole 将检查这些请求（每天都有数百个请求！），并自动阻止那些已知包含广告甚至恶意文件的请求。**



## II.安装 Pi-Hole



有了 Pi-Hole 这样一个名字，你可能会正确地认为你需要一个 Raspberry-Pi...但事实并非如此。**Pi-Hole可以安装在任何Linux电脑上（Debian、Fedora、Rocky、Ubuntu等）**。



另一方面，您需要记住，**这个设备必须每天 24 小时运行，原因很简单：没有 DNS，就没有互联网！**因此，树莓派是个好主意，因为它几乎不消耗能源。



安装时，只需通过 SSH 连接到 Linux 机器，然后以 "*root*"身份输入以下命令即可：



```
curl -sSL https://install.pi-hole.net | bash
```



> **注意**：在正常情况下，在不知道脚本的功能之前，不建议 "黑 "它。如果不确定，请使用浏览器访问页面或将内容下载为文件。
>


> **注意：在 Debian 11 的最小版本中，Curl 没有安装，因此需要在输入上述命令前使用** `apt-get install curl` **命令手动安装。**

## III.DHCP 配置



为了正常运行，Pi-Hole 需要 "解析 "客户端的 DNS 请求，因此客户端必须知道它是发送请求的服务器。有几种方法可以做到这一点：





- 修改 DHCP 服务器（如 Box）中的 DNS 设置
- 禁用该服务器，使用 Pi-Hole 提供的服务器
- 手动修改每个设备，使其使用 Pi-Hole 作为 DNS



我个人选择第一种解决方案。你所在的地方**可能已经有一个 DHCP 服务器（通常是你的机箱）**。所以没必要麻烦。



由于不同运营商的盒子（我并不了解所有运营商的盒子）和拥有自己路由器的用户之间存在大量可能性，我就不提供这些修改的截图了。无论如何，你都需要进入 DHCP 设置，修改 "DNS "参数，加入 Pi-Hole 的 IP Address。



完成此操作后，如果有任何设备之前已开启，它们将保留旧的设置，因此您需要重新启动配置请求。



在 Windows 工作站上，使用命令提示符 ：



```
ipconfig /renew
```



在 Linux 工作站上 ：



```
dhclient
```



对于所有其他设备，必须关闭并重新开启。



因此，他们应该获得正确的参数，以进行检查：



```
ipconfig /all
```



在 DNS 字段中，您应该输入 Pi-Hole 的 Address，我的情况是 192.168.1.42：

## IV.使用 Interface 网络 Pi-Hole



为便于管理，**Pi-Hole** 采用了精心设计的 Interface 网络 Interface。它界面友好、易于访问，可让您 ：





- 实时查看请求数、被阻止的请求数等。
- 管理白名单和黑名单
- 添加静态条目、别名等。
- 添加列表
- 还有许多其他功能！



就我而言，我将添加一个拦截列表。如上所述，与 Soft 同时安装的只有一个列表。广告网站的列表有很多，但最好至少选择一个针对你所在国家的列表。最著名的列表之一是**EasyList**，其中一个专门针对法国：[EasyList-ListFR](https://raw.githubusercontent.com/deathbybandaid/piholeparser/master/Subscribable-Lists/ParsedBlacklists/EasyList-Liste-FR.txt)
