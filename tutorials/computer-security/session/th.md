Session เป็นแอปพลิเคชันส่งข้อความที่เข้ารหัสซึ่งสร้างขึ้นในปี 2020 ออกแบบมาเพื่อให้มีความลับสูงกว่าการส่งข้อความแบบดั้งเดิม พัฒนาขึ้นครั้งแรกโดย *Oxen Privacy Tech Foundation* จากนั้นโดย *Session Technology Foundation*



Session มีคุณสมบัติทางเทคนิคที่น่าสนใจบางประการ: การเข้ารหัสข้อความแบบ end-to-end, เครือข่ายแบบกระจายศูนย์ที่จัดระเบียบเพื่อรับประกันความพร้อมใช้งานและความซ้ำซ้อน, และการกำหนดเส้นทางแบบหัวหอมที่ได้รับแรงบันดาลใจจาก Tor นอกจากนี้ แตกต่างจาก WathsApp หรือ Signal ซึ่งต้องใช้หมายเลขโทรศัพท์ในการลงทะเบียน Session ไม่ขอข้อมูลส่วนบุคคลใดๆ (ไม่มีหมายเลข, ไม่มีอีเมล, เพียงแค่คู่ของกุญแจเข้ารหัส)



Session ช่วยให้คุณส่งข้อความ ไฟล์ ข้อความเสียง การโทรด้วยเสียง รวมถึงกลุ่มที่มีสมาชิกสูงสุด 100 คน (และชุมชนที่มากกว่านั้น) ในขณะที่ลดการรั่วไหลของข้อมูลเมตาให้น้อยที่สุด

## ติดตั้งแอปพลิเคชัน Session



Session มีให้บริการบนทุกแพลตฟอร์ม คุณสามารถดาวน์โหลดแอปพลิเคชันได้โดยตรงจากร้านค้าแอปพลิเคชันบนโทรศัพท์ของคุณ:




- [Google Play](https://play.google.com/store/apps/details?id=network.loki.messenger);
- [App Store](https://apps.apple.com/us/app/session-private-messenger/id1470168868);
- [F-Droid](https://fdroid.getsession.org/).



บน Android ก็สามารถ [ติดตั้งผ่าน APK](https://github.com/session-foundation/session-android/releases) ได้เช่นกัน



ในบทแนะนำนี้ เราจะมุ่งเน้นไปที่เวอร์ชันมือถือ แต่โปรดทราบว่า [เวอร์ชันคอมพิวเตอร์ก็มีให้บริการเช่นกัน](https://getsession.org/download) (MacOS, Linux และ Windows) ต่อไป เราจะดูวิธีการซิงโครไนซ์บัญชีในหลายอุปกรณ์



## สร้างบัญชีบน Session



ในการเปิดตัวครั้งแรก ให้คลิกที่ "*สร้างบัญชี*"



![Image](assets/fr/02.webp)



เลือกชื่อที่จะแสดงสำหรับโปรไฟล์ของคุณ อาจเป็นนามแฝงหรือชื่อจริงของคุณก็ได้



![Image](assets/fr/03.webp)



จากนั้นคุณจะต้องเลือกโหมดการจัดการการแจ้งเตือนระหว่างสองโหมด:





- โหมดรวดเร็ว ("*Firebase Cloud Messaging/Apple Push Notification Service*"): ช่วยให้คุณได้รับการแจ้งเตือนข้อความเกือบจะทันที ด้วยบริการแจ้งเตือนที่จัดให้โดย Google หรือ Apple (ขึ้นอยู่กับระบบของคุณ) เพื่อให้สิ่งนี้ทำงานได้ ที่อยู่ IP ของคุณและรหัสการแจ้งเตือนที่ไม่ซ้ำกันจะถูกส่งไปยัง Google หรือ Apple และรหัสบัญชี Session ก็จะถูกลงทะเบียนกับเซิร์ฟเวอร์ STF (ผ่าน Tor) โหมดนี้เกี่ยวข้องกับการเปิดเผยข้อมูลเมตา (แม้ว่าจะน้อยมาก) แต่ไม่กระทบต่อเนื้อหาข้อความหรือรายชื่อผู้ติดต่อ และไม่อนุญาตให้ติดตามกิจกรรมจริงของคุณ โหมดนี้จึงมีประสิทธิภาพมากกว่าในแง่ของการตอบสนอง แต่พึ่งพาโครงสร้างพื้นฐานแบบรวมศูนย์และมีประสิทธิภาพน้อยกว่าเล็กน้อยในแง่ของความลับ





- โหมดช้า (*การสำรวจพื้นหลัง*): แอปพลิเคชัน Session ยังคงทำงานอยู่ในพื้นหลัง โดยสำรวจเครือข่ายเป็นระยะเพื่อหาข้อความใหม่ วิธีการนี้รับประกันความลับที่มากกว่าวิธีแรก เนื่องจากไม่มีการส่งข้อมูลไปยังเซิร์ฟเวอร์ของบุคคลที่สาม; ทั้ง Google, Apple หรือเซิร์ฟเวอร์ STF ไม่ได้รับข้อมูลใด ๆ ในทางกลับกัน โหมดนี้มีข้อเสียสองประการ: การแจ้งเตือนอาจล่าช้า (นานถึงหลายนาที) และการใช้พลังงานโดยทั่วไปจะสูงขึ้นเนื่องจากกิจกรรมของแอปพลิเคชันในพื้นหลัง



![Image](assets/fr/04.webp)



คุณเชื่อมต่อกับแอปพลิเคชัน Session แล้วและสามารถเริ่มแลกเปลี่ยนข้อความได้



![Image](assets/fr/05.webp)



## บันทึกบัญชี Session ของคุณ



สิ่งแรกที่ต้องทำก่อนที่คุณจะเริ่มใช้ Session คือการบันทึกบัญชีของคุณเพื่อให้คุณสามารถกู้คืนได้หากคุณทำอุปกรณ์หาย ในการทำเช่นนี้ ให้คลิกที่ปุ่ม "*Continue*" ถัดจาก "*Save your recovery password*".



![Image](assets/fr/06.webp)



Session จะแสดงวลีช่วยจำ ให้คัดลอกอย่างระมัดระวังและเก็บไว้ในที่ปลอดภัย วลีนี้ให้การเข้าถึงบัญชี Session ของคุณอย่างเต็มรูปแบบ ดังนั้นจึงสำคัญมากที่จะไม่เปิดเผย คุณจะต้องใช้มันเพื่อเข้าถึงบัญชีของคุณบนอุปกรณ์อื่น โดยเฉพาะอย่างยิ่งหากโทรศัพท์ปัจจุบันของคุณสูญหายหรือถูกเปลี่ยน



![Image](assets/fr/07.webp)



วลีนี้ทำงานในลักษณะเดียวกับวลีช่วยจำที่ใช้ในกระเป๋าเงิน Bitcoin ดังนั้นฉันขอแนะนำให้คุณดูบทแนะนำอื่นนี้ ซึ่งฉันอธิบายแนวทางปฏิบัติที่ดีที่สุดในการบันทึกวลีช่วยจำ:



https://planb.academy/tutorials/wallet/backup/backup-mnemonic-22c0ddfa-fb9f-4e3a-96f9-46e2a7954270

**โปรดทราบ**: ไม่เหมือนกับวลีช่วยจำที่ใช้ในกระเป๋าเงิน Bitcoin สำหรับ Session, **คุณต้องบันทึกแต่ละคำให้ครบถ้วน**. แค่ 4 ตัวอักษรแรกไม่เพียงพอ!



## การตั้งค่าแอปพลิเคชัน Session



ในการเข้าถึงการตั้งค่าแอปพลิเคชัน ให้คลิกที่รูปโปรไฟล์ของคุณที่มุมซ้ายบนของ Interface ที่นี่คือที่ที่คุณสามารถเพิ่มรูปโปรไฟล์ได้



![Image](assets/fr/08.webp)



ในเมนู "*Privacy*" คุณสามารถเปิดหรือปิดคุณสมบัติต่างๆ ได้ (ระวัง บางอย่างอาจเปิดเผยที่อยู่ IP ของคุณ) ฉันยังแนะนำให้เปิดใช้งานตัวเลือก "*Lock App*" ซึ่งต้องมีการยืนยันตัวตนเพื่อเข้าถึงแอปพลิเคชัน



![Image](assets/fr/09.webp)



ในเมนู "*Notification*" คุณจะพบตัวเลือกระหว่าง "*Fast Mode*" และ "*Slow Mode*" (ดูส่วนก่อนหน้าของบทแนะนำ) คุณยังสามารถปรับแต่งการแจ้งเตือนให้เหมาะกับความต้องการของคุณได้อีกด้วย



![Image](assets/fr/10.webp)



สุดท้าย ให้ไปที่เมนู "*Appearance*" เพื่อปรับแต่ง Interface ตามที่คุณต้องการ เมนู "*Recovery Password*" ช่วยให้คุณดึงวลีช่วยจำของคุณกลับมาได้ หากคุณต้องการสร้างการสำรองข้อมูลใหม่



![Image](assets/fr/11.webp)



## การส่งข้อความด้วย Session



หากต้องการติดต่อผู้อื่น ให้คลิกที่ปุ่ม "*+*" บนหน้าแรก



![Image](assets/fr/12.webp)



มีหลายตัวเลือกให้เลือก หากคุณต้องการสร้างหรือเข้าร่วมกลุ่ม ให้เลือก "*Create Group*" หรือ "*Join Community*".



![Image](assets/fr/13.webp)



หากคุณต้องการให้ใครบางคนเพิ่มคุณเป็นผู้ติดต่อ คุณสามารถให้พวกเขาสแกนรหัส Session ของคุณเป็น QR โค้ดได้



![Image](assets/fr/14.webp)



หากต้องการส่งข้อมูลการเข้าสู่ระบบของคุณจากระยะไกล ให้คลิกที่ "*เชิญเพื่อน*". จากนั้นคุณสามารถคัดลอก Session ID ของคุณและส่งผ่านช่องทางการสื่อสารอื่น ๆ ได้ คุณยังสามารถดึงข้อมูลนี้ได้โดยคลิกที่รูปโปรไฟล์ของคุณจากหน้าแรก



![Image](assets/fr/15.webp)



หากคุณมีรหัส Session ของบุคคลอื่นและต้องการเพิ่ม ให้คลิกที่ "*ข้อความใหม่*"



![Image](assets/fr/16.webp)



คุณสามารถวางตัวระบุในข้อความ หรือสแกนโดยตรงหากคุณมีเป็น QR โค้ด



![Image](assets/fr/17.webp)



จากนั้นส่งข้อความเริ่มต้นไปยังบุคคลนี้



![Image](assets/fr/18.webp)



ทันทีที่บุคคลนั้นยอมรับคำขอของคุณ คุณจะเห็นชื่อผู้ใช้ของพวกเขาปรากฏขึ้น และคุณจะสามารถแชทกับพวกเขาได้อย่างอิสระ



![Image](assets/fr/19.webp)



## ซิงโครไนซ์ซอฟต์แวร์เดสก์ท็อป



ในการซิงโครไนซ์บัญชีของคุณบนคอมพิวเตอร์ คุณจำเป็นต้องติดตั้งซอฟต์แวร์ [ดาวน์โหลดจากเว็บไซต์ทางการ](https://getsession.org/download) ฉันแนะนำให้คุณตรวจสอบความถูกต้องและความสมบูรณ์ของมันก่อนที่จะติดตั้ง



![Image](assets/fr/20.webp)



เมื่อเปิดใช้งานครั้งแรก ให้คลิกที่ "*ฉันมีบัญชี*"



![Image](assets/fr/21.webp)



ป้อนวลีช่วยจำของคุณ โดยเว้นวรรคระหว่างแต่ละคำ



![Image](assets/fr/22.webp)



คุณสามารถเข้าถึงการสนทนาของคุณจากคอมพิวเตอร์ได้แล้ว



![Image](assets/fr/23.webp)



ยินดีด้วย คุณเข้าใจการใช้การส่งข้อความ Session แล้ว ซึ่งเป็นทางเลือกที่ยอดเยี่ยมแทน WathsApp!
