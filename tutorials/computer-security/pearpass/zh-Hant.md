當每個人都管理數十甚至數百個線上帳號時，登入的安全性已成為 IT 安全的核心問題。社交網路、訊息系統、專業服務、金融平台：每一項存取都依賴於一個秘密，一旦外洩，可能會對您的生活造成嚴重後果。



然而，儘管攻擊次數不斷增加，不良做法仍在大眾中廣泛存在：弱密碼、重複使用的密碼、以明文形式儲存的密碼，或僅憑模糊記憶記住的密碼。為了在不增加日常生活複雜性的情況下解決這些問題，解決方案是使用密碼管理器。



目前已有數十種密碼管理器，Plan ₿ Academy 也提供了大部分密碼管理器的教學。但在本教程中，我想向您介紹一個在運作方式上顯然與眾不同的密碼管理器： **PearPass**。



**PearPass 是一款點對點、本地優先且開源的密碼管理器，旨在將對資料的完全控制權交還給使用者。**



![Image](assets/fr/01.webp)



## 為什麼選擇 PearPass？



密碼管理器是一種軟體程式，能以安全的方式產生、儲存及整理您所有的密碼。與其記憶或重複使用密碼，您只需要保護一個秘密：主密碼，它會加密您的整個保險箱。這種方式可讓每個服務都使用獨特、長且隨機的密碼，既可降低遺忘和外洩的風險，又可限制任何可能外洩的影響。如今，這是每個人都應該使用的基本 IT 工具。



密碼管理器有兩大類。一方面是本機專用軟體，由於資料不會儲存在中央伺服器上，因此非常安全，但不太實用，因為它無法讓您輕鬆地在多個裝置（電腦、智慧型手機、平板電腦......）之間同步憑證。另一方面，雲端密碼管理器會將您的憑證儲存在伺服器上，並在您所有的裝置上同步處理。雲端密碼管理器的主要優點是方便，但卻會在安全性上打折扣，因為您的密碼會儲存在您無法控制的基礎架構上。



PearPass 故意打破這兩種模式。它的定位介於本機管理器和雲端解決方案之間：它可以同步您的密碼，同時保證它們絕不會儲存在遠程服務器上。因此，您的所有憑證都儲存在您的本地設備上，多台機器之間的同步完全是點對點的。此架構消除了與集中式資料庫相關的單點故障：沒有伺服器可以入侵，也沒有第三方基礎架構可以存取您的憑證。您的裝置之間的通訊是端對端加密的，確保只有您所持有的金鑰才允許存取資料。



![Image](assets/fr/02.webp)



為了實現這一目標，正如其名所示，PearPass 依賴於 **Pears**，這是一個對等技術生態系統，專門用於創建和執行無伺服器應用程式。Pears 提供執行環境、散佈機制和網路層，以執行完全分散的應用程式，能夠在對等者之間直接同步，不需要任何中央基礎架構。就 PearPass 而言，Pears 提供了設備發現、加密連接建立以及機器之間的密碼庫同步。此方法可確保 PearPass 保持功能、彈性及獨立於任何外部權限。

## 如何安裝 PearPass？



PearPass 適用於所有平台：Windows、Linux、macOS、Android、iOS 和瀏覽器擴展。不需要在您的機器上安裝 Pears：PearPass 可自行運作。



### 在 Windows 上安裝



在 Windows 下，PearPass 是以經典安裝程式的形式提供的。前往 [官方下載頁面](https://pass.pears.com/download)，然後點選 `Download Windows installer` 按鈕。



檔案下載完成後，開啟安裝程式並依照精靈指示的步驟進行。然後可以從「開始功能表」或透過桌面捷徑存取應用程式。



![Image](assets/fr/03.webp)



### 在 macOS 上安裝



在 macOS 上，PearPass 以磁碟映像檔 (`.dmg`)的形式發行。前往 [官方下載頁面](https://pass.pears.com/download) 並選擇與您的 Mac 架構 (Intel 或 Apple Silicon) 相對應的版本。下載完成後，打開 `.dmg` 檔案並從 `Applications` 資料夾中啟動應用程式。



首次啟動時，macOS 會顯示安全訊息，指出應用程式來自網際網路：只需確認即可繼續。



### 在 Linux 上安裝



在 Linux 上，PearPass 是以`.AppImage`格式提供的，它保證了與大多數發行版本的廣泛兼容性，而不需要任何特定的相依性。從 [官方下載頁面](https://pass.pears.com/download) 下載`.AppImage`檔，然後直接雙擊啟動。



根據您的環境，您可能需要通過文件屬性（右鍵單擊）或使用 `chmod +x` 命令使文件可執行。一旦授權，PearPass 就會作為一個獨立的應用程序啟動。



### 安裝瀏覽器擴充套件



PearPass 提供瀏覽器擴充套件，可讓您在瀏覽網頁時自動登入並快速存取您的保險箱。該擴展目前適用於 Google Chrome 和相容的瀏覽器。要安裝它，請前往 [官方下載頁面](https://chromewebstore.google.com/detail/pearpass/pdeffakfmcdnjjafophphgmddmigpejh)。



![Image](assets/fr/04.webp)



添加後，您可以將其固定到工具欄上以便快速訪問。啟動擴展後，需要與本機安裝在您電腦上的 PearPass 應用程式建立連結（我們稍後會在教學中再說到）。



### 在 iOS 和 Android 上安裝



在 iPhone 和 Android 上，只需從應用程式商店下載應用程式即可：




- [Google Play 商店](https://play.google.com/store/apps/details?id=com.pears.pass)；
- [App Store](https://apps.apple.com/us/app/pearpass/id6752954830).



![Image](assets/fr/05.webp)



除了這些經典的安裝方法之外，也可以直接從 GitHub 套件庫下載軟體，每個 .NET 版本都可以從 GitHub 套件庫下載：




- [Desktop](https://github.com/tetherto/pearpass-app-desktop)；
- [Mobile](https://github.com/tetherto/pearpass-app-mobile)；
- [瀏覽器延伸功能](https://github.com/tetherto/pearpass-app-browser-extension)。



一旦在一個或多個平台上安裝了 PearPass，您就可以開始進行初始配置了。在本教程中，我們將首先在桌面配置軟件，然後與瀏覽器擴展和移動應用同步。



## 我如何建立一個 PearPass 保險箱？



當您第一次在電腦上啟動 PearPass 時，應用程式會引導您完成兩個步驟：設定您的主密碼，然後創建您的第一個保險箱。



### 設定主密碼



當應用程式第一次在桌面上啟動時，按一下 `Skip` 按鈕，然後按一下 `Continue` 按鈕，即可進入介紹畫面，並到達主密碼設定階段。



![Image](assets/fr/06.webp)



接下來是選擇主密碼的重要步驟。正如我們在介紹中所看到的，這個密碼非常重要，因為它可以讓您存取管理器中儲存的所有其他密碼。技術上來說，它是用來衍生加密資料的密碼鑰匙。



主密碼存在兩大主要風險：遺失與洩露。如果您失去對該密碼的存取權限，便無法再存取您的憑證。事實上，PearPass 從不儲存您的主密碼：**一旦遺失，您的憑證將永久遺失**。不存在任何復原機制。相反地，若此密碼遭到洩露，且攻擊者取得您其中一部裝置的存取權限，便能存取您所有的帳戶。



為了限制遺失的風險，您可以將主密碼做實體備份，例如寫在紙上，並將其保存在安全的地方。理想的做法是將此備份密封在信封中，以便您可以定期檢查它是否被存取。另一方面，切勿將此密碼做成數位備份。



為了降低外洩風險，您的主密碼必須強大。它應該盡可能長，包含多種字符，並且以真正隨機的方式選擇。在 2025 年，最低建議密碼至少包含 13 個字元，包括大小寫字母、數字和符號，前提是密碼是隨機的。不過，我建議密碼至少要有 20 個字元，甚至更多，並包含所有類型的字元，以確保更持久的安全等級。



在提供的欄位中輸入您的主密碼，再確認一次，然後按一下「繼續」按鈕。



![Image](assets/fr/07.webp)



然後，您會被重新導向到登入畫面：輸入您的主密碼，檢查一切運作是否正常。



![Image](assets/fr/08.webp)



### 建立您的第一個保險箱



登入後，PearPass 會提示您建立您的第一個保險箱。保險箱是一個加密的容器，你的密碼、ID、安全筆記和其他資訊都儲存在裡面。每個保險箱都是獨立的，可以用一個獨特的名字來識別，讓您可以根據您的用途（個人、專業、特定項目......）來組織您的資料。



按一下「建立新保管庫」按鈕。



![Image](assets/fr/09.webp)



為儲存庫選擇一個名稱，然後再按一下 ` 建立新儲存庫 ` 以完成建立。



![Image](assets/fr/10.webp)



您的保險箱立即可供使用。您可以立即開始新增密碼、登入帳號或安全備註。



![Image](assets/fr/11.webp)



## 如何在 PearPass 中加入登入帳號？



一旦你建立了你的保險箱，你就可以開始在裡面儲存你的物品了。PearPass 支援多種類型物品的註冊：




- 登入網站或服務 ；
- 身份：您的個人資訊可以快速填寫表格，也可以直接在 PearPass 中儲存身份證明文件；
- 信用卡：您的信用卡號碼，以便在線上購物時快速付款；
- Wi-Fi：您的 Wi-Fi 網路的密碼 ；
- PassPhrase：由多個單字組成的秘密詞組（警告：我強烈建議不要將其用於 wallet Bitcoin 記憶詞組）；
- 注意：建立安全筆記 ；
- 自訂：此功能可讓您建立自訂元素類型，以符合您的特定需求。



首先打開 PearPass 並使用您的主密碼登入。



![Image](assets/fr/12.webp)



選擇您要儲存此識別碼的保險箱。



![Image](assets/fr/13.webp)



在首頁中，依據您想要記錄的資訊類型，按一下新增項目按鈕。在我的例子中，我想要為我在 Plan ₿ Academy 網站的帳號新增登入帳號：因此我點選「建立登入帳號」按鈕。



![Image](assets/fr/14.webp)



一旦您選擇了要新增的項目類型，就會出現一個表格，讓您輸入與相關服務相關的資訊。根據您的需求，您可以輸入服務名稱、登入帳號、密碼，如果需要，也可以輸入網站位址和其他備註。



PearPass 還有一個密碼生成器，允許您直接從表格中創建一個強大的密碼。要使用它，點擊`密碼`欄位中代表三個小圓點的圖標，選擇所需的長度，然後點擊`插入密碼`。



填寫所有欄位後，按一下「儲存」按鈕，將識別碼儲存在保險箱中。



![Image](assets/fr/15.webp)



現在已儲存識別碼。它會出現在所選保險箱的項目清單中，按一下即可檢視。



![Image](assets/fr/16.webp)



您可以按一下元素，然後按一下「編輯」按鈕，輕鬆修改該元素。您也可以按一下介面右上方的三個小圓點，然後按一下`刪除元素`來刪除它。



![Image](assets/fr/22.webp)



在手機上，雖然介面有所調整，但邏輯仍然相同。登入後，選擇所需的保險箱，點選 `+`按鈕，選擇要建立的項目類型，然後填寫必要資訊。



![Image](assets/fr/17.webp)



## 如何組織 PearPass？



正如我們在前面的章節中所看到的，PearPass 可以讓您創建幾個不同的保險庫。這可以分開不同的用途，並構成您密碼管理器的第一層組織。在首頁，點擊介面左上方的箭頭，您可以輕鬆地從一個儲存區切換到另一個儲存區。



![Image](assets/fr/18.webp)



另一種組織密碼的方法是建立資料夾，即使是在儲存庫內。要做到這一點，請在介面左側欄中，按一下「所有資料夾」旁邊的 `+` 符號，然後輸入您要建立的資料夾名稱。



![Image](assets/fr/19.webp)



然後，您就可以儲存您所選擇的識別碼，可以在建立項目時直接儲存，也可以稍後點選 `編輯`。您只需在表格左上角選擇所需的資料夾。



![Image](assets/fr/20.webp)



開啟項目 (例如登入) 之後，您可以按一下介面右上方的星星圖示，將其加入我的最愛。除了基本資料夾外，還可在專屬資料夾中快速找到我的最愛。



![Image](assets/fr/21.webp)



最後，介面頂端有一個搜尋列，即使您的儲存庫包含許多識別碼，您也可以快速找到要找的項目。



## 我如何在手機上同步 PearPass？



現在您有一個可運作的儲存庫，裡面的項目都儲存在您的電腦上，您可能想要從您的智慧型手機或其他裝置存取您的密碼。PearPass 允許您使用 Pears 安全地同步多台機器上的管理員。讓我們來看看如何操作。



首先，在來源機器 (例如您的電腦) 上，使用您的主密碼登入您的儲存庫。進入首頁後，按一下介面右下方的「新增裝置」按鈕。



![Image](assets/fr/23.webp)



PearPass 隨後會產生一個邀請連結，也可以 QR 代碼的方式，將所選的儲存庫同步到您所選擇的裝置上。



![Image](assets/fr/24.webp)



如果您想在行動裝置上進行同步處理，請先安裝應用程式，然後啟動它。您會被要求為行動管理員建立一個主密碼。您可以選擇使用與電腦上相同的密碼，或使用不同的密碼。在所有情況下，請遵循相同的建議：將強大且隨機的密碼儲存在實體媒體上。



![Image](assets/fr/25.webp)



然後，如果您願意的話，可以啟動生物辨識驗證，避免每次解鎖手機時都要手動輸入主密碼。



![Image](assets/fr/26.webp)



重新輸入您的主密碼，然後按一下「繼續」按鈕。



![Image](assets/fr/27.webp)



選擇 `Load a vault` 選項，然後點擊 `Scan QR Code` 並掃描 PearPass 在您的來源機器（電腦）上顯示的邀請 QR 代碼。



![Image](assets/fr/28.webp)



您在電腦和手機上的保險庫現在可以同步。在一台裝置上新增的每個 ID 都會安全地儲存並在另一台裝置上複製。



![Image](assets/fr/29.webp)



在行動裝置上，您也可以依需要啟用欄位的自動填寫。為此，請前往 `Settings > Advanced`，然後在 `Autofill` 區段中點擊 `Set as Default` 按鈕。



![Image](assets/fr/30.webp)



## 如何使 PearPass 與瀏覽器擴展功能同步？



讓密碼管理器在電腦和智慧型手機之間同步已經非常實用了，但直接將它整合到瀏覽器中就更實用了。要做到這一點，首先 [將 PearPass 官方擴展功能添加到您的瀏覽器中](https://chromewebstore.google.com/detail/pearpass/pdeffakfmcdnjjafophphgmddmigpejh)。



![Image](assets/fr/31.webp)



從安裝在本地機器上的 PearPass 軟體，進入`設定 > 進階`，然後啟動`啟用瀏覽器擴展功能`選項。



![Image](assets/fr/32.webp)



PearPass 會產生一個 token 配對檔案。複製它。



![Image](assets/fr/33.webp)



然後在瀏覽器中開啟 PearPass 擴充套件，貼入 token 配對，然後按一下「驗證」按鈕，接著按「完成」。



![Image](assets/fr/34.webp)



您的密碼管理器現在已與瀏覽器擴充套件同步。



![Image](assets/fr/35.webp)



現在您可以使用它輕鬆連線到您的各種網路帳戶。



![Image](assets/fr/36.webp)
