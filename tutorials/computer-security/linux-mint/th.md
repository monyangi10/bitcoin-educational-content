## มีอะไรผิดถ้าคุณใช้คอมพิวเตอร์ทั่วไป?


เมื่อทำธุรกรรม Bitcoin ควรให้คอมพิวเตอร์ของคุณไม่มีมัลแวร์ เห็นได้ชัดเจน


หากคุณเก็บวลี Bitcoin seed ของคุณ (โดยปกติจะมี 12 หรือ 24 คำ) ไว้นอกคอมพิวเตอร์ด้วยอุปกรณ์เซ็นชื่อ (เช่น ฮาร์ดแวร์ wallet – ซึ่งเป็นวัตถุประสงค์หลักของมัน) คุณอาจคิดว่าการมีคอมพิวเตอร์ที่ “สะอาด” ไม่สำคัญ – ไม่จริง


คอมพิวเตอร์ที่ติดมัลแวร์อาจอ่านที่อยู่ Bitcoin ของคุณ ทำให้ผู้โจมตีเห็นยอดเงินของคุณได้ – พวกเขาไม่สามารถเอาบิทคอยน์จากการรู้ที่อยู่เพียงอย่างเดียวได้ แต่พวกเขาสามารถเห็นว่าคุณมีเท่าไหร่ และคำนวณจากนั้นว่าคุณเป็นเป้าหมายที่คุ้มค่าหรือไม่ พวกเขาอาจหาวิธีรู้ว่าคุณอาศัยอยู่ที่ไหน เช่น การดึงเล็บหรือเด็กจากคุณเพื่อให้คุณจ่ายค่าไถ่


## ทางออกคืออะไร?


ฉันสนับสนุนให้ผู้ใช้ Bitcoin ส่วนใหญ่ใช้คอมพิวเตอร์ที่ปราศจากมัลแวร์โดยเฉพาะ (ที่มีการเชื่อมต่ออินเทอร์เน็ต) สำหรับการทำธุรกรรม Bitcoin ฉันแนะนำให้ผู้คนใช้ระบบปฏิบัติการโอเพ่นซอร์สอย่าง Linux Mint แต่ถ้าจำเป็นก็ใช้ Windows หรือ Mac – นั่นดีกว่าการใช้คอมพิวเตอร์ทั่วไปที่มีการใช้งานบ่อยซึ่งมักจะมีมัลแวร์ซ่อนอยู่ในนั้น


อุปสรรคหนึ่งที่ผู้คนพบเจอคือการติดตั้งระบบปฏิบัติการใหม่บนคอมพิวเตอร์เช่นนี้ คู่มือนี้มีไว้เพื่อช่วยเหลือในเรื่องนั้น


มีลีนุกซ์หลากหลายประเภทและฉันได้ลองใช้หลายตัว คำแนะนำของฉันสำหรับผู้ใช้ Bitcoin คือ Linux Mint เพราะติดตั้งง่าย เร็วมาก (โดยเฉพาะตอนบูตและปิดเครื่อง) ไม่บวม (ซอฟต์แวร์เพิ่มเติมทุกชิ้นเป็นความเสี่ยง) และแทบไม่เคยล่มหรือทำงานแปลก ๆ กับฉัน (เมื่อเทียบกับเวอร์ชันอื่น ๆ เช่น Ubuntu และ Debian)


บางคนอาจจะต่อต้านระบบปฏิบัติการใหม่อย่างมาก โดยชอบใช้ Windows หรือ Mac OS มากกว่า ฉันเข้าใจ แต่ระบบปฏิบัติการของ Windows และ Apple เป็นแบบปิด ดังนั้นเราต้องเชื่อใจในสิ่งที่พวกเขาทำ ฉันไม่คิดว่านั่นเป็นนโยบายที่ดี แต่ก็ไม่ใช่ทั้งหมดหรือไม่มีอะไรเลย ฉันอยากให้คนใช้คอมพิวเตอร์ Windows หรือ Mac OS ที่ติดตั้งใหม่โดยเฉพาะมากกว่าคอมพิวเตอร์ที่ใช้งานมานาน (ซึ่งไม่รู้ว่ามีมัลแวร์อะไรสะสมอยู่บ้าง) อีกขั้นหนึ่งที่ดีกว่าคือการใช้คอมพิวเตอร์ Linux ที่ติดตั้งใหม่ ซึ่งเป็นสิ่งที่ฉันจะแสดงให้ดู


หากคุณรู้สึกกังวลเกี่ยวกับการใช้ Linux เพราะความไม่รู้ นั่นเป็นเรื่องธรรมชาติ แต่การใช้เวลาเรียนรู้ก็เป็นเรื่องธรรมชาติด้วยเช่นกัน มีข้อมูลมากมายที่สามารถหาได้ทางออนไลน์ นี่คือวิดีโอสั้น ๆ ที่ยอดเยี่ยมที่แนะนำพื้นฐานของบรรทัดคำสั่งที่ฉันขอแนะนำอย่างยิ่ง

เลือกคอมพิวเตอร์


ฉันจะเริ่มด้วยสิ่งที่ฉันคิดว่าเป็นตัวเลือกที่ดีที่สุด จากนั้นฉันจะให้ความคิดเห็นเกี่ยวกับทางเลือกอื่น ๆ


ตัวเลือกในอุดมคติ:


คำแนะนำของฉันคือ ถ้าคุณสามารถจ่ายได้ และถ้าขนาดของบิตคอยน์ที่คุณมีอยู่สมเหตุสมผล ควรซื้อแล็ปท็อประดับเริ่มต้นใหม่เอี่ยม รุ่นพื้นฐานที่สุดที่สร้างขึ้นในปัจจุบันดีพอที่จะรองรับการใช้งานที่ตั้งใจไว้ สเปคของโปรเซสเซอร์และ RAM ไม่สำคัญ เพราะทั้งหมดจะดีพออยู่แล้ว


หลีกเลี่ยง:



- แท็บเล็ตคอมโบใด ๆ รวมถึง Surface Pro
- Chromebooks – มักจะมีความจุในการจัดเก็บข้อมูลต่ำเกินไป
- คอมพิวเตอร์ใด ๆ ที่มีไดรฟ์ eMMC; ถ้ามีไดรฟ์ SSD นั่นก็สมบูรณ์แบบ
- Macs – พวกมันมีราคาแพง และฮาร์ดแวร์ไม่ค่อยเข้ากันได้ดีกับระบบปฏิบัติการ Linux ตามประสบการณ์ของฉัน

## ตัวเลือกอื่นๆ – Tails


Tails เป็นระบบปฏิบัติการที่บูตจากไดรฟ์ USB และเข้าควบคุมฮาร์ดแวร์ของคอมพิวเตอร์ชั่วคราว ใช้การเชื่อมต่อ Tor เท่านั้น ดังนั้นคุณต้องรู้สึกสบายใจในการใช้ Tor ข้อมูลใด ๆ ที่คุณเขียนลงในหน่วยความจำระหว่างเซสชันของคุณจะไม่ถูกบันทึกลงในไดรฟ์ (เริ่มใหม่ทุกครั้ง) เว้นแต่คุณจะปรับแต่งการตั้งค่าและสร้างตัวเลือกการจัดเก็บถาวร (บนไดรฟ์ USB) ซึ่งคุณล็อกด้วยรหัสผ่าน


มันไม่ใช่ตัวเลือกที่แย่และมันฟรี แต่สำหรับวัตถุประสงค์ของเราแล้วมันค่อนข้างเทอะทะ การติดตั้งซอฟต์แวร์ใหม่บนมันไม่ใช่เรื่องง่าย หนึ่งในคุณสมบัติที่ดีคือมันมาพร้อมกับ Electrum แต่ข้อเสียของเรื่องนี้คือคุณไม่ได้ติดตั้งมันเอง ตรวจสอบให้แน่ใจว่าไดรฟ์ USB ที่คุณใช้นั้นมีขนาดอย่างน้อย 8Gb


ความยืดหยุ่นของคุณจะลดลงหากคุณใช้ Tails คุณอาจไม่สามารถปฏิบัติตามคู่มือต่างๆ เพื่อกำหนดค่าที่คุณต้องการและทำให้มันทำงานได้อย่างถูกต้อง ตัวอย่างเช่น หากคุณปฏิบัติตามคู่มือของฉันในการติดตั้ง Bitcoin Core จะมีการปรับเปลี่ยนที่จำเป็นเพื่อให้มันทำงานได้ ฉันไม่คิดว่าจะทำคู่มือเฉพาะสำหรับ Tails ดังนั้นคุณจะต้องพัฒนาทักษะของคุณและทำมันด้วยตัวเอง


ฉันก็ไม่แน่ใจเหมือนกันว่า hardware wallets จะทำงานร่วมกับระบบปฏิบัติการนี้ได้ดีแค่ไหน


เมื่อกล่าวถึงทั้งหมดนี้ คอมพิวเตอร์ Tails สำหรับธุรกรรม Bitcoin เป็นตัวเลือกเพิ่มเติมที่ดี และจะช่วยพัฒนาทักษะความเป็นส่วนตัวโดยรวมของคุณในการเรียนรู้การใช้ Tails อย่างแน่นอน


## ตัวเลือกอื่น ๆ – Live OS Boot


นี่คล้ายกับ Tails มาก ยกเว้นระบบปฏิบัติการไม่ได้เน้นเรื่องความเป็นส่วนตัว วิธีพื้นฐานในการใช้งานคือการแฟลชไดรฟ์ USB ด้วยระบบปฏิบัติการ Linux ที่คุณเลือกและทำให้คอมพิวเตอร์บูตจากนั้นแทนที่จะเป็นไดรฟ์ภายใน วิธีการทำนี้จะอธิบายในภายหลัง


ข้อดีคือคุณมีข้อจำกัดน้อยลงและสิ่งต่างๆ จะทำงานได้โดยไม่ต้องปรับแต่งขั้นสูง


ฉันไม่แน่ใจว่าระบบดังกล่าวสามารถแยกมัลแวร์บนคอมพิวเตอร์ที่มีอยู่จากไดรฟ์บูต USB ที่คุณใช้ซึ่งมีระบบปฏิบัติการใหม่ได้ดีเพียงใด มันอาจจะทำงานได้ดีและอาจจะไม่ดีเท่า Tails เพราะฉันไม่รู้ ความชอบของฉันคือแล็ปท็อปเฉพาะทาง

ตัวเลือกอื่น ๆ – แล็ปท็อปหรือคอมพิวเตอร์ตั้งโต๊ะมือสองของคุณเอง


การใช้คอมพิวเตอร์มือสองไม่ใช่ตัวเลือกที่ดีนัก ส่วนใหญ่เพราะฉันไม่รู้เกี่ยวกับการทำงานภายในของมัลแวร์ที่ซับซ้อน หรือว่าการล้างไดรฟ์จะเพียงพอที่จะกำจัดมันหรือไม่ อาจจะเพียงพอแต่ฉันไม่อยากประเมินความฉลาดของแฮกเกอร์ที่ไม่หวังดีต่ำเกินไป คุณสามารถตัดสินใจได้ ฉันไม่อยากผูกมัด


หากคุณเลือกใช้เดสก์ท็อปเก่าแทนที่จะเป็นแล็ปท็อปเก่า นี่จะไม่เป็นปัญหา ยกเว้นว่ามันจะใช้พื้นที่ถาวรสำหรับการทำธุรกรรมบิตคอยน์ที่อาจจะเกิดขึ้นไม่บ่อย คุณไม่ควรใช้มันเพื่อสิ่งอื่น ในขณะที่แล็ปท็อป คุณสามารถเก็บมันไว้และแม้กระทั่งซ่อนมันเพื่อความปลอดภัยเพิ่มเติม


## การติดตั้ง Linux Mint บนคอมพิวเตอร์ใด ๆ


นี่คือคำแนะนำในการลบระบบปฏิบัติการใด ๆ ออกจากแล็ปท็อปใหม่ของคุณและติดตั้ง Linux Mint แต่คุณสามารถปรับใช้เพื่อติดตั้งลินุกซ์เวอร์ชันใดก็ได้บนคอมพิวเตอร์เกือบทุกเครื่อง


เราจะใช้คอมพิวเตอร์ใดก็ได้ในการแฟลชระบบปฏิบัติการไปยังเมมโมรี่สติ๊กชนิดใดก็ได้ ไม่สำคัญว่าเป็นเมมโมรี่สติ๊กแบบไหน ตราบใดที่สามารถใช้งานร่วมกับพอร์ต USB ได้ และฉันแนะนำให้ใช้ขนาดขั้นต่ำ 16Gb

เมื่อดาวน์โหลดแล้ว คุณสามารถแฟลชไฟล์ไปยังอุปกรณ์หน่วยความจำแบบพกพาและทำให้สามารถบูตได้ วิธีที่ง่ายที่สุดในการทำเช่นนี้คือใช้ Balena Etcher ดาวน์โหลดและติดตั้งหากคุณยังไม่มี

คลิกแฟลชจากไฟล์ และเลือกไฟล์ LinuxMint ที่คุณดาวน์โหลด


จากนั้นคลิก เลือกเป้าหมาย ตรวจสอบให้แน่ใจว่าอุปกรณ์หน่วยความจำเสียบอยู่และตรวจสอบให้แน่ใจว่าคุณกำลังเลือกไดรฟ์ที่ถูกต้อง มิฉะนั้นคุณอาจทำลายเนื้อหาของไดรฟ์ที่ผิดได้!


หลังจากนั้น ให้เลือก Flash! คุณอาจจำเป็นต้องใส่รหัสผ่านของคุณ เมื่อเสร็จสิ้น ไดรฟ์อาจจะไม่สามารถอ่านได้โดยคอมพิวเตอร์ Windows หรือ Mac ของคุณเพราะมันถูกแปลงเป็นอุปกรณ์ Linux แล้ว แค่ดึงมันออกมา

กำลังเตรียมคอมพิวเตอร์เป้าหมาย


เปิดแล็ปท็อปเครื่องใหม่ และขณะที่มันกำลังเปิดเครื่อง ให้กดปุ่ม BIOS ค้างไว้ โดยทั่วไปจะเป็น F2 แต่ก็อาจจะเป็น F1, F8, F10, F11, F12 หรือ Delete ลองกดแต่ละปุ่มจนกว่าจะได้ หรือค้นหาทางอินเทอร์เน็ตสำหรับรุ่นคอมพิวเตอร์ของคุณและถามคำถามที่ถูกต้อง


เช่น “BIOS key Dell laptops”.


คอมพิวเตอร์แต่ละเครื่องจะมีเมนู BIOS ที่แตกต่างกัน สำรวจและค้นหาว่าเมนูใดที่อนุญาตให้คุณกำหนดลำดับการบูต สำหรับวัตถุประสงค์ของเรา เราต้องการให้คอมพิวเตอร์พยายามบูตจากอุปกรณ์ที่เชื่อมต่อผ่าน USB (หากมีการเชื่อมต่อ) ก่อนที่จะพยายามบูตจากฮาร์ดไดรฟ์ภายใน (มิฉะนั้น Windows จะโหลด) เมื่อคุณตั้งค่านั้นแล้ว คุณอาจจำเป็นต้องบันทึกก่อนออก หรืออาจบันทึกโดยอัตโนมัติ


รีบูตเครื่องคอมพิวเตอร์และมันควรจะโหลดจากอุปกรณ์หน่วยความจำ USB เราไม่สามารถติดตั้ง Linux บนไดรฟ์ภายในและ Windows จะถูกลบออกอย่างถาวร


เมื่อคุณไปถึงหน้าจอต่อไปนี้ ให้เลือก “OEM install (for manufacturers)”. หากคุณเลือก “Start Linux Mint” แทน คุณจะได้เซสชัน Linux Mint ที่โหลดจากอุปกรณ์หน่วยความจำ แต่เมื่อคุณปิดคอมพิวเตอร์ ข้อมูลของคุณจะไม่ถูกบันทึก – มันเป็นเซสชันชั่วคราวเพื่อให้คุณลองใช้งานเท่านั้น


![image](assets/10.webp)


คุณจะถูกนำทางผ่านวิซาร์ดแบบกราฟิกซึ่งจะถามคำถามหลายข้อที่ควรจะตรงไปตรงมา หนึ่งในนั้นจะเป็นการตั้งค่าภาษา อีกข้อหนึ่งจะเป็นการเชื่อมต่อเครือข่ายอินเทอร์เน็ตที่บ้านของคุณและรหัสผ่าน หากมีการแจ้งให้ติดตั้งซอฟต์แวร์เพิ่มเติม ให้ปฏิเสธ เมื่อคุณไปถึงคำถามเกี่ยวกับประเภทการติดตั้ง บางคนอาจลังเล – คุณต้องเลือก “ลบข้อมูลในดิสก์และติดตั้ง Linux Mint” นอกจากนี้ อย่าเข้ารหัสไดรฟ์และอย่าเลือก LVM


ในที่สุดคุณจะไปถึงเดสก์ท็อป ในจุดนี้ คุณยังไม่เสร็จสิ้นจริงๆ คุณกำลังทำหน้าที่เป็นผู้ผลิต (เช่น คนที่สร้างคอมพิวเตอร์และติดตั้ง Linux ให้กับลูกค้า) คุณต้องดับเบิลคลิกที่ไอคอนเดสก์ท็อป “Install Linux Mint” เพื่อทำให้เสร็จสิ้น


![image](assets/11.webp)


อย่าลืมถอดเมมโมรี่สติ๊กออก แล้วรีบูต หลังจากรีบูต คุณจะได้ใช้ระบบปฏิบัติการเป็นครั้งแรกในฐานะผู้ใช้ใหม่ ยินดีด้วย


หนึ่งในสิ่งแรกที่ต้องทำ (และต้องทำเป็นประจำ) คือการรักษาระบบให้ทันสมัยอยู่เสมอ


เปิดแอปพลิเคชัน Terminal และพิมพ์ดังต่อไปนี้:


```bash
sudo apt-get update
```


กด <enter>, ยืนยันการเลือกของคุณ, แล้วใช้คำสั่งนี้:


```bash
sudo apt-get upgrade
```


กด <enter> และยืนยันการเลือกของคุณ


ปล่อยให้มันทำงาน อาจใช้เวลาหลายนาที


ถัดไป ฉันต้องการติดตั้ง Tor (คำนึงถึงตัวพิมพ์ใหญ่และเล็ก):


```bash
sudo apt-get install tor
```


**เคล็ดลับมือโปร:** คุณยังสามารถรัน Linux Mint boot จาก “OEM install” (ตรวจสอบให้แน่ใจว่าคุณเชื่อมต่อกับอินเทอร์เน็ต มิฉะนั้นคุณอาจพบข้อผิดพลาด) หากคุณทำเช่นนี้ คุณจำเป็นต้องคลิกที่ไอคอน “ship to end user” ซึ่งควรอยู่บนเดสก์ท็อป จากนั้นคุณรีบูตและเริ่มระบบปฏิบัติการเหมือนกับว่าคุณกำลังเปิดคอมพิวเตอร์เป็นครั้งแรก


คู่มือนี้อธิบายว่าทำไมคุณอาจต้องการคอมพิวเตอร์เฉพาะสำหรับการทำธุรกรรม Bitcoin และวิธีการติดตั้งระบบปฏิบัติการ Linux Mint ใหม่บนคอมพิวเตอร์นั้น
