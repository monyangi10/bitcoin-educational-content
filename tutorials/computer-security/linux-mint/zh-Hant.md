## 使用一般電腦有什麼問題？


進行 Bitcoin 交易時，最理想的情況是您的電腦沒有惡意軟體。很明顯


如果您使用簽名裝置 (例如 Hardware Wallet - 其主要用途) 將 Bitcoin seed 詞組 (通常為 12 或 24 個字) 置於電腦之外，那麼您可能會認為擁有一台「乾淨」的電腦並不那麼重要 - 事實並非如此。


受惡意軟體感染的電腦可能會讀取您的 Bitcoin 位址，將您的餘額暴露給攻擊者 - 他們無法僅藉由知道 Address 就取得 Bitcoin，但他們可以看到您有多少錢，並據此計算您是否是值得攻擊的目標。例如，他們也可能以某種方式找出您的住址，並從您身上提取指甲或小孩，讓您支付贖金。


## 解決方案是什麼？


我鼓勵大多數的比特幣使用者使用專用的無惡意軟體的電腦（可以上網）進行Bitcoin交易。我建議大家使用像Linux Mint這樣的開源作業系統，但如果你必須使用的話，也可以使用Windows或Mac - 這總比使用普通的、用得很好的電腦要好，因為普通的電腦中總是隱藏著惡意軟件。


人們會遇到的一個障礙是在這類電腦上安裝新的作業系統。本指南將協助您解決這個問題。


Linux 有很多種，我也嘗試過幾種。我給比特幣玩家的推薦是 Linux Mint，因為它容易安裝、速度非常快（尤其是開機和關機時）、不臃腫（每個額外的軟體都是風險），而且很少當機或出現奇怪的行為（與 Ubuntu 和 Debian 等其他版本相比）。


有些人可能會非常抗拒新的作業系統，偏好 Windows 或 Mac OS。我了解，但 Windows 和 Apple 作業系統都是封閉源碼的，所以我們必須相信他們的作法；我不認為這是個好政策，但也不是全有或全無。我更希望人們使用專用的新安裝 Windows 或 Mac OS 電腦，而不是使用多年的電腦 (誰知道上面累積了什麼惡意軟體)。更好的做法是使用新安裝的 Linux 電腦，這也是我要示範的。


如果您因為未知而對使用 Linux 感到緊張，這是很自然的，但花一些時間去學習也是很自然的。網上有很多資訊。以下是介紹命令列基本知識的精彩短片，我強力推薦。

選擇電腦


我會先說我認為最好的選擇。然後，我會提出我對替代方案的意見。


理想的選擇：


我的建議是，如果您負擔得起，而且 Bitcoin 堆疊的大小也符合您的需求，就買一台全新的入門級筆記型電腦。現在最基本的機型就足以應付它要用來做的事。處理器和記憶體規格並不重要，因為它們都已經很好了。


避免：



- 任何平板電腦組合，包括 Surface Pro
- Chromebook - 通常儲存容量太小
- 任何配備 eMMC 硬碟機的電腦；如果配備 SSD 硬碟機，那就再好不過了
- Mac - 價格昂貴，而且根據我的經驗，硬體與 Linux 作業系統相容性不佳。
- 任何翻新機或二手機 (但並非絕對不能接受)

## 其他選項 - 尾巴


Tails 是一個從 USB 隨身碟開機的作業系統，可以暫時接管任何電腦的硬體。它只使用 Tor 連線，因此您需要熟悉使用 Tor。除非您調整設定並建立永久儲存選項 (在 USB 隨身碟上)，否則您在會話期間寫入記憶體的任何資料都不會儲存到硬碟機 (每次都會從新開始) - 您可以使用密碼鎖定該選項。


它不是一個壞的選擇，而且是免費的，但對於我們的目的來說，它有點笨拙。在它上面安裝新軟體並不輕鬆。它的一個好處是自帶 Electrum，但它的壞處是您沒有自己安裝。請確定您使用的 USB 隨身碟至少有 8Gb。


如果您使用 Tails，您的彈性就會降低。您可能無法依照各種指南來設定您需要的東西，並讓它正常運作。例如，如果您依照我的指南來安裝 Bitcoin Core，就需要做一些修改才能讓它正常運作。我想我不會製作專屬於 Tails 的指南，所以您需要培養自己的技能，獨力完成。


我也不確定硬體錢包與此作業系統的互動性如何。


說了這麼多，用來進行 Bitcoin 交易的 Tails 電腦是個不錯的額外選擇，而且肯定會對您學習使用 Tails 的整體隱私技能有所幫助。


## 其他選項 - Live OS Boot


這與 Tails 非常類似，只是作業系統並非專用於隱私。基本的使用方法是將您選擇的 Linux 作業系統閃存到 USB 隨身碟，然後讓電腦從 USB 隨身碟開機，而不是從內部硬碟開機。稍後會說明如何執行。


這樣做的好處是您受到的限制較少，不需要進階的調整就能運作。


我不確定這樣的系統能多好地隔離現有電腦上的惡意軟體與您使用的 USB 開機磁碟機上的新作業系統。它可能做得很好，也可能比不上 Tails。因為我不知道，所以我的偏好是專用筆記型電腦。

其他選項 - 您自己的二手筆記型電腦或桌上型電腦


使用二手電腦並不理想，主要是因為我不瞭解複雜的惡意軟體的內部運作，也不知道抹除硬碟是否足以擺脫惡意軟體。也許可以，但我不想低估惡意駭客的聰明。您可以決定，我不想承諾。


如果您選擇使用舊的桌上型電腦，而不是舊的筆記型電腦，這樣也沒問題，只是它會永久佔用您可能稀有的 Bitcoin 交易空間；您不應該用它來做其他事情。而使用筆記型電腦，您可以直接將它收起來，甚至藏起來以增加安全性。


## 在任何電腦上安裝 Linux Mint


這些是從您的新筆記型電腦抹除任何作業系統並安裝 Linux Mint 的說明，但您可以將它調整為在任何電腦上安裝任何 Linux 版本。


我們要使用任何電腦將作業系統快閃到某種記憶棒上。只要能與 USB 連接埠相容，哪一種記憶卡都沒關係，我建議最少使用 16Gb。

接下來，瀏覽 linuxmint.com

將滑鼠移至頂端的「下載」功能表，然後點選連結「Linux Mint 20.3」或任何您執行此動作時所推薦的最新版本。


![image](assets/6.webp)


有幾種「口味」可供選擇。請選擇「Cinnamon」以跟隨本指南。按一下「下載」按鈕。

在下一頁，您可以向下捲動來查看鏡像 (鏡像是指持有我們想要的檔案複本的各種伺服器)。您可以使用 SHA256 和 gpg（推薦）來驗證下載，但我在此不作說明，因為我已經寫了相關的指南。


![image](assets/8.webp)


選擇離您最近的鏡像，然後按一下其連結 (鏡像列中的 Green 文字)。檔案會開始下載 - 我下載的版本是 2.1 gigabytes。


下載完成後，您可以將檔案快閃到可攜式記憶體裝置，使其可開機。要做到這一點，最簡單的方法是使用 Balena Etcher。如果您沒有，請下載並安裝。


然後執行它：


![image](assets/9.webp)


按一下 flash from file，然後選擇您下載的 LinuxMint 檔案。


然後按一下選擇目標。確保記憶體裝置已插入，並確認選擇正確的磁碟機，否則可能會破壞錯誤磁碟機的內容！


之後，選擇 Flash！您可能需要輸入密碼。完成後，您的 Windows 或 Mac 電腦可能無法讀取硬碟機，因為它已轉換成 Linux 裝置。只要將它拔出即可。

準備目標電腦


開啟新筆記型電腦，並在開機時按住 BIOS 鍵。這通常是 F2，但也可能是 F1、F8、F10、F11、F12 或 Delete。逐一嘗試，直到找到為止，或在網際網路上搜尋您電腦的型號，然後提出正確的問題。


例如 「BIOS key Dell 筆記型電腦」。


每台電腦都有不同的 BIOS 功能表。探索並找出允許您設定開機順序的功能表。就我們的目的而言，我們希望電腦先嘗試從 USB 連接裝置 (如果有連接的話) 開機，然後再嘗試從內部 Hard 磁碟機開機 (否則 Windows 將會載入)。設定完成後，您可能需要先儲存再退出，也可能會自動儲存。


重新啟動電腦，它應該會從 USB 記憶體裝置載入。我們無法在內接式硬碟機上安裝 Linux，Windows 會永久移除。


當你進入以下畫面時，選擇「OEM 安裝 (給製造商)」。如果您選擇「Start Linux Mint」，您就會從記憶體裝置載入一個 Linux Mint 會話，但是一旦您關閉電腦，您的任何資訊都不會被儲存 - 基本上這是一個臨時的會話，讓您可以試試看。


![image](assets/10.webp)


您會看到一個圖形精靈，它會問您一些簡單的問題。其中一個問題是 Languange 設定，另一個問題是您的家庭網際網路連線和密碼。如果提示您安裝其他軟體，請拒絕。當您問到安裝類型時，有些人可能會猶豫 - 您需要選擇「刪除磁碟並安裝 Linux Mint」。此外，不要加密磁碟機，也不要選擇 LVM。


您最終會到達桌面。此時，您還沒完全完成。你實際上是扮演製造商的角色 (也就是有人製造電腦並為客戶設定 Linux)。您需要按兩下桌面上的圖示「Install Linux Mint」，才能完成。


![image](assets/11.webp)


記得移除記憶棒，然後再重新開機。重新開機後，您將以新使用者的身份第一次使用作業系統。恭喜您


首先要做的事情之一 (而且要定期做)，就是保持系統的最新狀態。


開啟終端應用程式，並輸入下列內容：


```bash
sudo apt-get update
```


按下 <enter>，確認您的選擇，然後按下此指令：


```bash
sudo apt-get upgrade
```


按下 <enter> 並確認您的選擇。


讓它做它的事，可能需要幾分鐘。


接下來，我喜歡安裝 Tor (區分大小寫)：


```bash
sudo apt-get install tor
```


**Pro Tip:** 您也可以從「OEM 安裝」執行 Linux Mint 開機（請確認您已連線至網路，否則可能會出錯）。如果您這樣做，您稍後需要按一下桌面上的「傳送給最終使用者」圖示。然後重新開機並啟動作業系統，就像第一次開啟電腦一樣。
