## 일반 컴퓨터를 사용한다면 무엇이 문제인가요?


Bitcoin 거래를 할 때는 컴퓨터에 멀웨어가 없는 것이 가장 이상적입니다. 당연하죠.


서명 장치(예: Hardware Wallet - 주 목적)를 사용하여 컴퓨터에서 Bitcoin seed 문구(보통 12단어 또는 24단어)를 지우면 컴퓨터를 "깨끗한" 상태로 유지하는 것이 그다지 중요하지 않다고 생각할 수 있지만, 이는 사실이 아닙니다.


멀웨어에 감염된 컴퓨터가 Bitcoin 주소를 읽으면 공격자에게 잔액이 노출될 수 있습니다. 공격자는 Address를 아는 것만으로 Bitcoin를 탈취할 수는 없지만, 사용자가 얼마나 가지고 있는지 확인하고 이를 통해 사용자가 가치 있는 표적이 될 수 있는지 계산할 수 있습니다. 예를 들어, 공격자는 사용자의 거주지를 알아내어 손톱이나 자녀의 정보를 빼내 몸값을 지불하도록 유도할 수도 있습니다.


## 해결책은 무엇인가요?


저는 대부분의 비트코인 사용자에게 멀웨어가 없는 전용 컴퓨터(인터넷 접속 가능)를 사용해 Bitcoin 거래를 할 것을 권장합니다. 리눅스 민트와 같은 오픈 소스 운영 체제를 사용하되, 꼭 사용해야 한다면 윈도우나 맥을 사용하는 것이 멀웨어가 항상 숨겨져 있는 일반 컴퓨터를 사용하는 것보다 낫습니다.


이러한 컴퓨터에 새 운영체제를 설치하는 데 어려움을 겪는 사람들이 있습니다. 이 가이드는 이를 돕기 위한 것입니다.


리눅스에는 다양한 종류가 있으며 저도 여러 가지를 사용해 보았습니다. 비트코인 사용자들에게 리눅스 민트를 추천하는 이유는 설치가 쉽고, 매우 빠르며(특히 부팅과 종료 시), 부피가 크지 않고(모든 추가 소프트웨어는 위험), (우분투나 데비안 같은 다른 버전에 비해) 충돌이나 이상한 동작이 거의 발생하지 않았기 때문입니다.


일부 사용자는 새로운 운영체제에 대한 거부감이 커서 Windows나 Mac OS를 선호할 수 있습니다. 이해는 하지만 Windows와 Apple 운영체제는 비공개 소스이기 때문에 그 운영체제가 하는 일을 신뢰해야 하고, 그것이 좋은 정책이라고 생각하지는 않지만 그렇다고 해서 전부가 아닌 것도 아닙니다. 저는 사람들이 (어떤 멀웨어가 설치되어 있을지 모르는) 잘 사용하는 컴퓨터보다는 새로 설치한 Windows 또는 Mac OS 전용 컴퓨터를 사용하는 것을 훨씬 더 선호합니다. 한 단계 더 나은 방법은 새로 설치한 Linux 컴퓨터를 사용하는 것이며, 제가 시연할 것입니다.


리눅스 사용에 대해 잘 몰라서 불안한 것은 당연한 일이지만, 배우기 위해 시간을 투자하는 것도 마찬가지입니다. 온라인에서 많은 정보를 얻을 수 있습니다. 다음은 명령줄의 기본 사항을 소개하는 훌륭한 짧은 동영상으로, 적극 추천합니다.

## 기타 옵션 - 테일즈


Tails는 USB 썸 드라이브에서 부팅하는 운영 체제로, 모든 컴퓨터의 하드웨어를 일시적으로 점유합니다. 토르 연결만을 사용하므로 토르 사용에 익숙해야 합니다. 설정을 조정하고 비밀번호로 잠그는 영구 저장 옵션(USB 썸 드라이브에)을 만들지 않는 한, 세션 중에 메모리에 쓴 데이터는 드라이브에 저장되지 않습니다(매번 새로 시작됩니다).


나쁘지 않은 옵션이고 무료이지만 저희의 목적에는 조금 투박합니다. 새로운 소프트웨어를 설치하는 것도 쉬운 일이 아닙니다. 한 가지 좋은 기능은 Electrum과 함께 제공된다는 것이지만, 직접 설치하지 않는다는 단점이 있습니다. 사용하는 USB 드라이브가 8Gb 이상인지 확인하세요.


Tails를 사용하면 유연성이 떨어집니다. 다양한 가이드를 따라 필요한 사항을 설정하고 제대로 작동하지 않을 수도 있습니다. 예를 들어, Bitcoin core 설치 가이드를 따르는 경우 작동을 위해 수정해야 할 사항이 있습니다. 저는 테일즈 전용 가이드를 만들지 않을 것이므로 혼자서 기술을 익혀야 할 것입니다.


또한 하드웨어 지갑이 이 OS와 얼마나 잘 작동할지도 잘 모르겠습니다.


이 모든 것을 고려했을 때, Bitcoin 거래용 테일즈 컴퓨터는 좋은 추가 옵션이며, 전반적인 개인 정보 보호 기술을 배우는 데 확실히 도움이 될 것입니다.


## 기타 옵션 - 라이브 OS 부팅


운영 체제가 개인 정보 보호 전용이 아니라는 점을 제외하면 테일즈와 매우 유사합니다. 이 기능을 사용하는 기본적인 방법은 원하는 Linux 운영 체제가 설치된 USB 드라이브를 플래시하고 내부 드라이브 대신 이 드라이브에서 컴퓨터를 부팅하는 것입니다. 이 방법은 나중에 설명합니다.


제한이 적고 고급 조정 없이도 모든 것이 작동한다는 장점이 있습니다.


이러한 시스템이 기존 컴퓨터의 멀웨어를 새 운영 체제가 들어 있는 USB 부팅 드라이브에서 얼마나 잘 격리하는지 잘 모르겠습니다. 아마도 잘 작동하지만 Tails만큼 좋지는 않을 것입니다. 그래서 저는 전용 노트북을 선호합니다.

기타 옵션 - 개인 중고 노트북 또는 데스크톱 컴퓨터


중고 컴퓨터는 정교한 멀웨어의 내부 작동 방식을 알지 못하거나 드라이브를 지우는 것만으로 멀웨어를 제거할 수 있는지도 모르기 때문에 이상적이지 않습니다. 그럴 수도 있지만 해커가 얼마나 영리한지 과소평가하고 싶지 않습니다. 결정할 수 있지만, 저는 저지르고 싶지 않습니다.


구형 노트북 대신 구형 데스크톱을 사용하는 경우, 드물게 발생할 수 있는 Bitcoin 트랜잭션을 위한 공간을 영구적으로 차지한다는 점을 제외하면 괜찮지만 다른 용도로 사용해서는 안 됩니다. 반면 노트북을 사용하면 보안을 강화하기 위해 그냥 치워두거나 숨겨둘 수도 있습니다.

그런 다음 플래시를 선택하세요! 비밀번호를 입력해야 할 수도 있습니다. 완료되면 드라이브가 Linux 장치로 변환되었기 때문에 Windows 또는 Mac 컴퓨터에서 읽을 수 없을 가능성이 높습니다. 그냥 꺼내세요.

대상 컴퓨터 준비하기


새 노트북을 켜고 전원이 켜지는 동안 BIOS 키를 길게 누릅니다. 이 키는 일반적으로 F2이지만 F1, F8, F10, F11, F12 또는 Delete일 수도 있습니다. 익숙해질 때까지 하나씩 시도하거나 인터넷에서 컴퓨터 모델을 검색하여 올바른 질문을 하세요.


예: "BIOS 키 Dell 노트북".


컴퓨터마다 BIOS 메뉴가 다릅니다. 부팅 순서를 구성할 수 있는 메뉴를 탐색하고 찾아보세요. 여기서는 컴퓨터가 내부 Hard 드라이브에서 부팅을 시도하기 전에 USB에 연결된 장치(연결된 장치가 있는 경우)에서 부팅을 시도하도록 하고 싶습니다(그렇지 않으면 Windows가 로드됩니다). 설정한 후에는 종료하기 전에 저장해야 하거나 자동으로 저장될 수 있습니다.


컴퓨터를 재부팅하면 USB 메모리 장치에서 로드됩니다. 내부 드라이브에 Linux를 설치할 수 없으며 Windows는 영원히 제거됩니다.


다음 화면이 나타나면 "OEM 설치(제조업체용)"를 선택합니다. 대신 "Linux Mint 시작"을 선택하면 메모리 장치에서 Linux Mint 세션이 로드되지만 컴퓨터를 종료하면 어떤 정보도 저장되지 않으며, 기본적으로 임시 세션이므로 시험해 볼 수 있습니다.


![image](assets/10.webp)


그래픽 마법사를 통해 몇 가지 간단한 질문을 하게 될 것입니다. 그중 하나는 언어 설정이고 다른 하나는 홈 인터넷 네트워크 연결 및 비밀번호입니다. 추가 소프트웨어를 설치하라는 메시지가 표시되면 거부하세요. 설치 유형에 대한 질문이 나오면 일부 사람들은 망설일 수 있습니다. "디스크 지우기 및 Linux Mint 설치"를 선택해야 합니다. 또한 드라이브를 암호화하지 말고 LVM을 선택하지 마세요.


결국 데스크톱으로 이동하게 됩니다. 이 시점에서는 아직 완료된 것이 아닙니다. 여러분은 실제로 제조업체(즉, 컴퓨터를 만들고 고객을 위해 Linux를 설정하는 사람)의 역할을 하고 있는 것입니다. 완료하려면 바탕 화면 아이콘인 "Linux Mint 설치"를 두 번 클릭해야 합니다.


![image](assets/11.webp)


메모리 스틱을 제거한 다음 재부팅하는 것을 잊지 마세요. 재부팅하면 새 사용자로서 처음으로 운영 체제를 사용하게 됩니다. 축하합니다.


가장 먼저 해야 할 일(그리고 정기적으로 해야 할 일) 중 하나는 시스템을 최신 상태로 유지하는 것입니다.


터미널 애플리케이션을 열고 다음을 입력합니다:


```bash
sudo apt-get update
```


<입력> 키를 누르고 선택을 확인한 다음 이 명령을 입력합니다:


```bash
sudo apt-get upgrade
```


<입력>을 누르고 선택을 확인합니다.
