## Начало работы с PureOS



PureOS - это операционная система на базе Debian, разработанная компанией Purism. PureOS подходит как для ИТ-специалистов, так и для пользователей, которым нужен простой и удобный в использовании дистрибутив. Она уникальна тем, что является *свободным программным обеспечением* и фокусируется на защите личных данных своих пользователей, устанавливая рамки, основанные на конфиденциальности, свободе, безопасности и стабильности, предлагаемых Debian.



### Почему стоит выбрать PureOS?





- Простой, интуитивно понятный Interface**: GNOME предлагает понятный рабочий стол Interface, разработанный для простоты использования даже для тех, кто не умеет работать с командной строкой.





- Бесплатно**: как и большинство дистрибутивов Linux, PureOS полностью бесплатна для использования. Однако для поддержки разработчиков доступна ежемесячная подписка.





- Безопасность и стабильность**: Архитектура и режим работы PureOS делают его высокобезопасным дистрибутивом, гарантирующим защиту данных и стабильность системы.





- Документация и активное сообщество**: PureOS имеет четкую, доступную документацию и активное сообщество, что позволяет легко решать проблемы и изучать систему шаг за шагом.



## Установка и настройка



Установка и настройка PureOS на вашем компьютере потребует следующих минимальных возможностей:




- USB-носитель объемом не менее 8 ГБ для прошивки системы.
- 4 ГБ ОПЕРАТИВНОЙ ПАМЯТИ.
- 30 ГБ свободного места на диске Hard.



Перейдите на [официальный сайт PureOS](https://pureos.net/), затем загрузите ISO-образ операционной системы в соответствии с архитектурой вашей машины.



Чтобы запустить установку PureOS, необходимо создать загрузочный USB-носитель с помощью программного обеспечения для флэш-памяти, например [Balena Etcher](https://www.balena.io/etcher).



За три простых шага вы получите USB-накопитель с загруженной операционной системой PureOS.





- Подключите USB-носитель и запустите загруженное программное обеспечение Balena.
- Затем выберите ISO-образ PureOS
- Выберите USB-носитель в качестве устройства вывода, затем нажмите на кнопку **Flash** и дождитесь окончания процесса.



![0_01](assets/fr/01.webp)



После загрузки USB-носителя перезагрузите компьютер, на который вы хотите установить PureOS.



При загрузке войдите в BIOS, нажав клавишу `ESC`, `F9` или `F11`, в зависимости от типа вашего компьютера. Выберите USB-носитель в качестве загрузочного устройства, затем нажмите `ENTER` для подтверждения.

## Обновления



Прежде чем начать использовать PureOS, необходимо обновить систему. Это позволит вам воспользоваться новейшими функциями и исправлениями безопасности, а также обеспечит большую стабильность.





- Обновление через графику Interface**:


Откройте приложение **Программное обеспечение**, затем перейдите на вкладку **Обновления**. Доступные обновления отображаются автоматически. Нажмите **Загрузить**, а затем **Установить** после завершения загрузки.





- Обновление через терминал**:


Откройте терминал и введите следующую команду, чтобы обновить список доступных пакетов:



```shell
sudo apt update
```



Введите пароль и подтвердите его. Затем установите обновления с помощью:



```shell
sudo apt full-upgrade
```



## PureOS для разработки



По умолчанию PureOS не включает в себя все инструменты, необходимые для разработки.


Вы можете быстро установить их с помощью следующей команды:



```shell
sudo apt install build-essential git curl
```



Это позволит установить инструменты компиляции **Git** и **Curl**, полезные в большинстве сред разработки.



## Среда PureOS



PureOS отличается инновационным подходом к истинной конвергенции: единая операционная система обеспечивает плавную и бесперебойную работу на различных устройствах, включая ноутбуки, планшеты, мини-ПК и смартфоны.



Приложения PureOS разработаны как адаптивные: они автоматически подстраиваются под размер экрана и режим ввода (сенсорный или клавиатура/мышь). Например, веб-браузер GNOME динамически адаптирует свой Interface, чтобы обеспечить оптимальную работу как на мобильных, так и на настольных устройствах.



В состав PureOS также входит офисный пакет **LibreOffice**, который включает в себя:





- Writer**: полноценный текстовый процессор для создания и редактирования документов.
- Calc**: мощная программа электронных таблиц для управления данными и расчетами.
- Impress**: инструмент для создания профессиональных презентаций.



Этот бесплатный пакет позволяет эффективно работать, не завися от проприетарного программного обеспечения.



PureOS предлагает единую, безопасную и гибкую среду, основанную на дистрибутиве со 100% открытым исходным кодом, который гарантирует полный контроль и строгое соблюдение конфиденциальности. Ее истинная конвергенция облегчает создание приложений, совместимых с различными типами устройств, такими как компьютеры, планшеты и смартфоны, без необходимости сложной адаптации.



Благодаря встроенному доступу к необходимым инструментам, надежным менеджерам пакетов и богатой экосистеме открытых исходных кодов PureOS упрощает разработку, тестирование и развертывание приложений в безопасной среде. Стабильная архитектура и конфиденциальность Commitment делают ее надежной платформой для различных применений, включая разработку Blockchain, быстрое создание прототипов или производственные среды.
