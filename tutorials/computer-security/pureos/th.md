## เริ่มต้นใช้งาน PureOS



PureOS เป็นระบบปฏิบัติการที่พัฒนาจาก Debian โดย Purism PureOS เหมาะสำหรับทั้งผู้เชี่ยวชาญด้านไอทีและผู้ใช้ที่มองหาการกระจายที่เรียบง่ายและใช้งานง่าย ความพิเศษของมันคือเป็น *ซอฟต์แวร์เสรี* และมุ่งเน้นไปที่การปกป้องข้อมูลส่วนบุคคลของผู้ใช้โดยการตั้งค่ากรอบงานที่อิงตามความเป็นส่วนตัว เสรีภาพ ความปลอดภัย และความเสถียรที่ Debian มอบให้



### ทำไมต้องเลือก PureOS?





- Simple, intuitive Interface**: GNOME นำเสนอเดสก์ท็อป Interface ที่ชัดเจน ออกแบบมาให้ใช้งานง่าย แม้กระทั่งสำหรับผู้ที่ไม่คุ้นเคยกับการใช้บรรทัดคำสั่ง





- ฟรี**: เช่นเดียวกับลินุกซ์ดิสทริบิวชันส่วนใหญ่ PureOS สามารถใช้งานได้ฟรีทั้งหมด อย่างไรก็ตาม มีการสมัครสมาชิกแบบรายเดือนเพื่อสนับสนุนนักพัฒนา





- ความปลอดภัยและเสถียรภาพ**: สถาปัตยกรรมและโหมดการทำงานของ PureOS' ทำให้เป็นการกระจายที่มีความปลอดภัยสูง รับประกันการปกป้องข้อมูลและเสถียรภาพของระบบ





- เอกสารและชุมชนที่มีชีวิตชีวา**: PureOS มีเอกสารที่ชัดเจนและเข้าถึงได้ง่าย พร้อมด้วยชุมชนที่มุ่งมั่นและตอบสนอง ทำให้ง่ายต่อการแก้ปัญหาและเรียนรู้ระบบทีละขั้นตอน



## การติดตั้งและการกำหนดค่า



การติดตั้งและกำหนดค่า PureOS บนคอมพิวเตอร์ของคุณจะต้องมีคุณสมบัติต่อไปนี้:




- USB key ขนาดอย่างน้อย 8GB สำหรับแฟลชระบบ
- 4 GB RAM.
- พื้นที่ว่าง 30 GB บนฮาร์ดดิสก์ของคุณ



ไปที่ [เว็บไซต์ทางการของ PureOS](https://pureos.net/) จากนั้นดาวน์โหลดไฟล์ ISO ของระบบปฏิบัติการตามสถาปัตยกรรมของเครื่องคุณ



ในการเปิดตัวการติดตั้ง PureOS คุณจำเป็นต้องสร้าง USB ที่สามารถบูตได้โดยใช้ซอฟต์แวร์แฟลช เช่น [Balena Etcher](https://www.balena.io/etcher)



ในสามขั้นตอนง่ายๆ คุณจะได้รับ USB ที่บูตด้วยระบบปฏิบัติการ PureOS





- เสียบ USB key และเรียกใช้ซอฟต์แวร์ Balena ที่ดาวน์โหลดมาแล้ว
- จากนั้นเลือกไฟล์ภาพ ISO PureOS
- เลือกกุญแจ USB เป็นอุปกรณ์เอาท์พุต จากนั้นคลิกที่ปุ่ม **Flash** และรอให้กระบวนการเสร็จสิ้น



![0_01](assets/fr/01.webp)



เมื่อ USB key ได้ถูกบูตแล้ว ให้รีสตาร์ทคอมพิวเตอร์ที่คุณต้องการติดตั้ง PureOS



เมื่อเริ่มต้นระบบ ให้เข้าถึง BIOS โดยกดปุ่ม `ESC`, `F9` หรือ `F11` ขึ้นอยู่กับเครื่องของคุณ เลือก USB key เป็นอุปกรณ์บูต จากนั้นกด `ENTER` เพื่อยืนยัน

## อัปเดต



ก่อนที่คุณจะเริ่มใช้ PureOS สิ่งสำคัญคือต้องอัปเดตระบบของคุณ นี่จะช่วยให้คุณได้รับประโยชน์จากฟีเจอร์ล่าสุดและแพตช์ความปลอดภัย และรับรองเสถียรภาพที่ดียิ่งขึ้น





- อัปเดตผ่านกราฟิก Interface:


เปิดแอปพลิเคชัน **Software** จากนั้นไปที่แท็บ **Updates** การอัปเดตที่มีอยู่จะแสดงโดยอัตโนมัติ คลิก **Download** แล้วคลิก **Install** เมื่อการดาวน์โหลดเสร็จสิ้น





- อัปเดตผ่านเทอร์มินัล**:


เปิดเทอร์มินัล จากนั้นป้อนคำสั่งต่อไปนี้เพื่ออัปเดตรายการแพ็กเกจที่มีอยู่:



```shell
sudo apt update
```



ป้อนรหัสผ่านของคุณและยืนยัน จากนั้นติดตั้งการอัปเดตด้วย:



```shell
sudo apt full-upgrade
```



## PureOS สำหรับการพัฒนา



ตามค่าเริ่มต้น PureOS ไม่ได้รวมเครื่องมือทั้งหมดที่จำเป็นสำหรับการพัฒนา


คุณสามารถติดตั้งได้อย่างรวดเร็วด้วยคำสั่งต่อไปนี้:



```shell
sudo apt install build-essential git curl
```



สิ่งนี้จะติดตั้งเครื่องมือคอมไพล์ **Git** และ **Curl** ซึ่งมีประโยชน์ในสภาพแวดล้อมการพัฒนาส่วนใหญ่



## สภาพแวดล้อม PureOS



PureOS โดดเด่นด้วยแนวทางนวัตกรรมในการบรรจบกันอย่างแท้จริง: ระบบปฏิบัติการเดียวที่รับประกันการทำงานที่ราบรื่นและไร้รอยต่อในอุปกรณ์หลากหลายประเภท รวมถึงแล็ปท็อป แท็บเล็ต มินิพีซี และสมาร์ทโฟน



แอปพลิเคชัน PureOS ถูกออกแบบมาให้ปรับตัวได้: พวกมันจะปรับขนาดหน้าจอและโหมดการป้อนข้อมูล (สัมผัสหรือคีย์บอร์ด/เมาส์) โดยอัตโนมัติ ตัวอย่างเช่น เบราว์เซอร์ GNOME Web จะปรับ Interface ของมันแบบไดนามิกเพื่อมอบประสบการณ์ที่ดีที่สุดทั้งบนอุปกรณ์มือถือและเดสก์ท็อป



PureOS ยังรวมถึงชุดโปรแกรมสำนักงาน **LibreOffice** ซึ่งประกอบด้วย:





- Writer**: โปรแกรมประมวลผลคำที่สมบูรณ์สำหรับการสร้างและแก้ไขเอกสาร
- Calc**: โปรแกรมสเปรดชีตที่ทรงพลังสำหรับการจัดการข้อมูลและการคำนวณของคุณ
- Impress**: เครื่องมือสำหรับสร้างงานนำเสนอแบบมืออาชีพ



ชุดโปรแกรมฟรีนี้ช่วยให้คุณทำงานได้อย่างมีประสิทธิภาพโดยไม่ต้องพึ่งพาซอฟต์แวร์ที่มีลิขสิทธิ์



PureOS นำเสนอสิ่งแวดล้อมที่เป็นหนึ่งเดียว ปลอดภัย และยืดหยุ่น โดยอิงจากการแจกจ่ายแบบโอเพ่นซอร์ส 100% ที่รับประกันการควบคุมอย่างสมบูรณ์และการเคารพความเป็นส่วนตัวอย่างเคร่งครัด การบรรจบกันอย่างแท้จริงของมันช่วยอำนวยความสะดวกในการสร้างแอปพลิเคชันที่เข้ากันได้กับอุปกรณ์ประเภทต่างๆ เช่น คอมพิวเตอร์ แท็บเล็ต และสมาร์ทโฟน โดยไม่จำเป็นต้องปรับเปลี่ยนที่ซับซ้อน



ด้วยการเข้าถึงเครื่องมือที่จำเป็นอย่างเต็มรูปแบบ, ตัวจัดการแพ็คเกจที่แข็งแกร่ง และระบบนิเวศโอเพ่นซอร์สที่หลากหลาย, PureOS ทำให้การพัฒนา, การทดสอบ และการปรับใช้แอปพลิเคชันในสภาพแวดล้อมที่ปลอดภัยเป็นเรื่องง่าย สถาปัตยกรรมที่เสถียรและความมุ่งมั่นในความลับทำให้เป็นแพลตฟอร์มที่เชื่อถือได้สำหรับการใช้งานที่หลากหลาย รวมถึงการพัฒนา Blockchain, การสร้างต้นแบบอย่างรวดเร็ว หรือสภาพแวดล้อมการผลิต
