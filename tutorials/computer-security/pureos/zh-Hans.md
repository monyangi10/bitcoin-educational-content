## 开始使用 PureOS



PureOS 是 Purism 开发的基于 Debian 的操作系统。PureOS 既适合 IT 专业人士，也适合寻求简单易用的发行版的用户。它的独特之处在于它是*自由软件*，并以 Debian 提供的隐私、自由、安全和稳定为基础建立了一个框架，重点保护用户的个人数据。



### 为什么选择 PureOS？





- 简单、直观的 **Interface**：GNOME 提供清晰的 Interface 桌面，即使是不熟悉命令行的人也能轻松使用。





- **免费**：与大多数 Linux 发行版一样，PureOS 的使用完全免费。不过，为支持开发者，可按月订购。





- **安全性和稳定性**：PureOS 的架构和运行模式使其成为一个高度安全的发行版，保证了数据保护和系统稳定性。





- **文档和活跃的社区**：PureOS 有清晰易懂的文档，还有一个热心、积极响应的社区，让您可以轻松解决问题，逐步学习系统。



## 安装和配置



在计算机上安装和配置 PureOS 需要以下最基本功能：




- 至少 8GB 的 USB 密钥，用于闪存系统。
- 4 GB 内存
- Hard 磁盘上有 30 GB 的可用空间。



访问 [PureOS 官方网站](https://pureos.net/)，然后根据机器的架构下载操作系统的 ISO 映像。



要启动 PureOS 安装程序，需要使用闪存软件（如 [Balena Etcher](https://www.balena.io/etcher)）创建可启动 USB 密钥。



只需三个简单的步骤，你就能让 USB 盘启动 PureOS 操作系统。





- 插入 USB 密钥，运行下载的 Balena 软件。
- 然后选择 PureOS ISO 映像
- 选择 USB 密钥作为输出设备，然后点击 **Flash** 按钮，等待程序完成。



![0_01](assets/fr/01.webp)



USB 密钥启动后，重新启动要安装 PureOS 的计算机。



启动时，根据不同的机器，按 `ESC`、`F9` 或 `F11` 键进入 BIOS。选择 USB 密钥作为启动设备，然后按 `ENTER` 键确认。

点击**活动**菜单，查看系统中预装的应用程序和工具。点击 ** 安装 PureOS** 开始安装



![0_09](assets/fr/09.webp)



根据需要设置系统语言和键盘布局，然后配置 Hard 磁盘分区模式。

您有两种选择来对 Hard 磁盘进行分区：




- 擦除磁盘：完全安装 PureOS，删除 Hard 磁盘上的所有现有数据。



![0_14](assets/fr/14.webp)





- 手动分区，创建自己的乐谱



⚠️ 为操作系统手动创建分区时，确保为 PureOS 操作分配至少 2 GB 的分区，然后为数据分配另一个分区。



![0_15](assets/fr/15.webp)



如果要确保数据安全，请激活**磁盘加密**。输入一个强大、复杂的密码。



通过定义用户名和至少 20 个字符的字母数字密码，将用户与操作系统关联起来，以加强数据的安全性。



![0_16](assets/fr/16.webp)



查看您所做的设置，必要时进行修改。



![0_17](assets/fr/17.webp)



点击 "**安装**"，然后点击 "**立即安装**"，确认 PureOS 已安装到您的计算机上。



![0_18](assets/fr/18.webp)



![0_19](assets/fr/19.webp)

- 键盘布局
- 您的 NextCloud 帐户（可选）。

## 更新



在开始使用 PureOS 之前，更新系统至关重要。这将使您受益于最新的功能和安全补丁，并确保更高的稳定性。





- 通过 **Interface 图形更新**：


打开**软件**应用程序，然后转到**更新**选项卡。系统会自动显示可用的更新。点击**下载**，下载完成后点击**安装**。





- 通过终端更新** ：


打开终端，然后输入以下命令更新可用软件包列表：



```shell
sudo apt update
```



输入密码并确认。然后用 .NET Framework 安装更新：



```shell
sudo apt full-upgrade
```



## 用于开发的 PureOS



默认情况下，PureOS 不包含开发所需的所有工具。


您可以使用以下命令快速安装它们：



```shell
sudo apt install build-essential git curl
```



这将安装**Git**和**Curl**编译工具，在大多数开发环境中都很有用。



## PureOS 环境



PureOS 以其实现真正融合的创新方法脱颖而出：单一操作系统可确保在笔记本电脑、平板电脑、迷你 PC 和智能手机等各种设备上流畅、无缝地运行。



PureOS 应用程序的设计是自适应的：它们会根据屏幕尺寸和输入模式（触摸或键盘/鼠标）自动调整。例如，GNOME 网页浏览器会动态调整其 Interface，以便在移动设备和桌面设备上提供最佳体验。



PureOS 还包含**LibreOffice**办公套件，其中包括 .NET 和 .NET Framework：





- **Writer**：用于创建和编辑文档的完整文字处理器。
- **Calc**：功能强大的电子表格程序，用于管理数据和计算。
- **Impress**：制作专业演示文稿的工具。



这套免费软件可让您高效工作，而无需依赖专有软件。



PureOS 提供了一个统一、安全和灵活的环境，基于 100% 开放源码的发行版，保证了全面控制和对隐私的严格尊重。其真正的融合性有助于创建与不同类型设备（如电脑、平板电脑和智能手机）兼容的应用程序，而无需进行复杂的适配。



PureOS 可通过本地访问基本工具、强大的软件包管理器和丰富的开源生态系统，在安全的环境中简化应用程序开发、测试和部署。其稳定的架构和 Commitment 保密性使其成为一个可靠的平台，可用于各种用途，包括 Blockchain 开发、快速原型或生产环境。
