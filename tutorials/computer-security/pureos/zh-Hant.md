## 開始使用 PureOS



PureOS 是 Purism 開發的 Debian 作業系統。PureOS 適合 IT 專業人士和尋求簡單易用發行版的使用者。它的獨特之處在於它是*自由軟體*，並透過建立基於 Debian 所提供的隱私、自由、安全與穩定的架構，著重於保護使用者的個人資料。



### 為何選擇 PureOS？





- 簡單、直覺的 Interface**：GNOME 提供了一個清晰的 Interface 桌面，即使是不熟悉命令列的人也能輕鬆使用。





- 免費**：與大多數 Linux 發行版一樣，PureOS 完全免費使用。不過，您可以每月訂閱以支援開發人員。





- 安全性與穩定性**：PureOS 的架構和作業模式使其成為高度安全的發行版，可保證資料保護和系統穩定。





- 文件與活躍的社群**：PureOS 擁有清楚易懂的說明文件，以及致力於回應的社群，讓您可以輕鬆解決問題，並逐步學習系統。



## 安裝與設定



在您的電腦上安裝和設定 PureOS，需要以下極簡的功能：




- 至少 8GB 的 USB 密鑰，用來閃存系統。
- 4 GB RAM。
- Hard 磁碟上有 30 GB 的可用空間。



前往 [PureOS 官方網站](https://pureos.net/)，然後根據您的機器架構下載作業系統的 ISO 映像檔。



若要啟動 PureOS 安裝，您需要使用 Flash 軟體，例如 [Balena Etcher](https://www.balena.io/etcher) 建立可開機的 USB key。



只需三個簡單步驟，您就能讓 USB 隨身碟啟動 PureOS 作業系統。





- 插入 USB 密鑰並執行下載的 Balena 軟體。
- 然後選擇 PureOS ISO 映像
- 選擇 USB 隨身碟為輸出裝置，然後按一下 **Flash** 按鈕，等待程序完成。



![0_01](assets/fr/01.webp)



USB 密鑰啟動後，請重新啟動要安裝 PureOS 的電腦。



開機時，視您的機器而定，按 `ESC`、`F9` 或 `F11` 鍵進入 BIOS。選擇 USB 鑰匙作為開機裝置，然後按 `ENTER` 鍵確認。

### 啟動安裝



按一下 **Activities** 功能表，探索系統上預先安裝的應用程式和工具。按一下 ** 安裝 PureOS** 開始安裝



![0_09](assets/fr/09.webp)



根據需要設定系統語言和鍵盤配置，然後設定 Hard 磁碟分割模式。



![0_10](assets/fr/10.webp)



![0_11](assets/fr/11.webp)



![0_12](assets/fr/12.webp)

您有兩個選項來分割 Hard 磁碟：




- 刪除磁碟**：用於完整安裝 PureOS，刪除 Hard 磁碟上所有先前存在的資料。



![0_14](assets/fr/14.webp)





- 手動分區**建立您自己的樂譜



⚠️ 手動為作業系統建立磁碟分割時，請確定您已為 PureOS 作業分配至少 2 GB 的磁碟分割，然後再為資料分配另一個磁碟分割。



![0_15](assets/fr/15.webp)



如果要保護資料安全，請啟動 ** 磁碟加密**。輸入強大、複雜的密碼。



透過定義使用者名稱和至少 20 個字元的密碼，將使用者與作業系統聯繫起來，以加強資料的安全性。



![0_16](assets/fr/16.webp)



檢視您所做的設定，必要時進行修改。



![0_17](assets/fr/17.webp)



按一下 ** 安裝**，然後按一下 ** 立即安裝**，確認 PureOS 已安裝在您的電腦上。



![0_18](assets/fr/18.webp)



![0_19](assets/fr/19.webp)



安裝完成後，勾選 **Restart now** 方塊以重新啟動電腦，然後確認：




- 語言。
- 鍵盤配置。
- 您的 NextCloud 帳戶 (選用)。

## 更新



在您開始使用 PureOS 之前，更新您的系統是非常重要的。這將讓您受惠於最新的功能和安全修補程式，並確保更高的穩定性。





- 透過 Interface 圖形更新** ：


開啟 **Software** 應用程式，然後轉到 **Updates** 標籤。可用的更新會自動顯示。按一下 **下載**，下載完成後再按一下 **安裝**。





- 透過終端機更新** ：


開啟終端機，然後輸入以下指令更新可用套件清單：



```shell
sudo apt update
```



輸入密碼並確認。然後用 .NET Framework 安裝更新：



```shell
sudo apt full-upgrade
```



## 用於開發的 PureOS



根據預設，PureOS 並未包含開發所需的所有工具。


您可以使用下列指令快速安裝：



```shell
sudo apt install build-essential git curl
```



這會安裝**Git**和**Curl**編譯工具，對大多數的開發環境都很有用。



## PureOS 環境



PureOS 以其真正融合的創新方法脫穎而出：單一作業系統可確保在筆記型電腦、平板電腦、迷你電腦和智慧型手機等各種裝置上順暢無縫地操作。



PureOS 應用程式的設計是自適應性的：它們會自動調整以適應螢幕尺寸和輸入模式 (觸控或鍵盤/滑鼠)。例如，GNOME Web 瀏覽器會動態調整其 Interface，以在行動和桌上型裝置上提供最佳體驗。



PureOS 還包含 **LibreOffice** 辦公室套件，其中包括 .NET 和 Java：





- Writer**：完整的文字處理器，可用於建立和編輯文件。
- Calc**: 一個功能強大的試算表程式，用來管理您的資料和計算。
- Impress**：用於製作專業簡報的工具。



此免費套裝軟體可讓您有效率地工作，而無需依賴專屬軟體。



PureOS 提供統一、安全且彈性的環境，以 100% 的開放原始碼散佈為基礎，可保證完全控制並嚴格尊重隱私。其真正的融合性有助於建立與不同類型裝置 (如電腦、平板電腦和智慧型手機) 相容的應用程式，而無需複雜的適配。



PureOS 原生可存取基本工具、強大的套件管理員和豐富的開放原始碼生態系統，可在安全的環境中簡化應用程式開發、測試和部署。其穩定的架構和 Commitment 的保密性使其成為一個可靠的平台，適用於各種用途，包括 Blockchain 開發、快速原型或生產環境。
