## PureOSを使い始める



PureOS は Purism が開発した Debian ベースのオペレーティングシステムです。PureOSは、ITプロフェッショナルにも、シンプルで使いやすいディストリビューションをお探しのユーザーにも適しています。フリーソフトウェアであり、Debianが提供するプライバシー、自由、セキュリティ、安定性に基づいたフレームワークを設定することで、ユーザーの個人情報の保護に重点を置いている点が特徴です。



### PureOSを選ぶ理由





- シンプルで直感的な **Interface**：GNOME は明快な Interface デスクトップを提供し、コマンドラインが苦手な人でも簡単に使えるように設計されています。





- **無料**: ほとんどのLinuxディストリビューションと同様、PureOSは完全に無料で使用できます。ただし、開発者をサポートするための月額サブスクリプションが用意されています。





- **セキュリティと安定性**：PureOSのアーキテクチャとオペレーティングモードは、データ保護とシステムの安定性を保証し、非常に安全なディストリビューションです。





- ドキュメントと活発なコミュニティ：PureOSには、わかりやすくアクセスしやすいドキュメントと、熱心で応答性の高いコミュニティがあり、問題を簡単に解決し、システムを段階的に学ぶことができます。



## インストールと設定



お使いのコンピューターにPureOSをインストールして設定するには、以下の最小限の機能が必要です：




- システムをフラッシュするための8GB以上のUSBキー。
- 4 GB RAM。
- Hardディスクに30GBの空き容量。



PureOS公式ウェブサイト](https://pureos.net/)にアクセスし、お使いのマシンのアーキテクチャに応じたオペレーティングシステムのISOイメージをダウンロードします。



PureOSのインストールを開始するには、[Balena Etcher](https://www.balena.io/etcher)などのフラッシュソフトを使ってブータブルUSBキーを作成する必要があります。



簡単な3ステップで、PureOSオペレーティング・システムで起動したUSBメモリを入手できる。





- USBキーを差し込み、ダウンロードしたBalenaソフトウェアを実行する。
- 次にPureOS ISOイメージを選択します。
- 出力デバイスとしてUSBキーを選択し、**Flash**ボタンをクリックし、プロセスが終了するのを待ちます。



![0_01](assets/fr/01.webp)



USBキーが起動したら、PureOSをインストールするコンピュータを再起動します。



起動するときは、マシンによって異なるが、`ESC`、`F9`、`F11`キーを押してBIOSにアクセスする。ブートデバイスとしてUSBキーを選択し、`ENTER`キーを押して確定する。

Hardディスクのパーティションには2つのオプションがあります：




- ディスクの消去PureOSを完全にインストールするために、Hardディスク上の既存のデータをすべて削除します。



![0_14](assets/fr/14.webp)





- **マニュアル・パーティショニング**で独自のスコアを作成



⚠️ オペレーティングシステムのパーティションを手動で作成する場合は、PureOS操作用に最低2GBのパーティションを割り当て、さらにデータ用に別のパーティションを割り当ててください。



![0_15](assets/fr/15.webp)



データを保護したい場合は、**ディスク暗号化**を有効にしてください。強力で複雑なパスワードを入力してください。



データのセキュリティを強化するため、ユーザー名と20文字以上の英数字のパスワードを定義して、オペレーティング・システムにユーザーを関連付けます。



![0_16](assets/fr/16.webp)



設定した内容を確認し、必要であれば修正してください。



![0_17](assets/fr/17.webp)

![0_18](assets/fr/18.webp)

## 更新情報



PureOSを使い始める前に、システムのアップデートが不可欠です。これにより、最新の機能とセキュリティパッチを利用できるようになり、安定性が向上します。





- **Interfaceグラフィックによる更新**：


ソフトウェア**アプリケーションを開き、**アップデート**タブを開きます。利用可能なアップデートが自動的に表示されます。ダウンロード**をクリックし、ダウンロードが完了したら**インストール**をクリックします。





- **ターミナル経由での更新**：


ターミナルを開き、以下のコマンドを入力して利用可能なパッケージのリストを更新する：



```shell
sudo apt update
```



パスワードを入力し、確認する。その後、.NET Frameworkでアップデートをインストールしてください：



```shell
sudo apt full-upgrade
```



## 開発用PureOS



デフォルトでは、PureOSには開発に必要なすべてのツールが含まれていません。


以下のコマンドですぐにインストールできる：



```shell
sudo apt install build-essential git curl
```



これは、ほとんどの開発環境で役立つ**Git**と**Curl**コンパイル・ツールをインストールします。



## PureOS環境



PureOSは、真のコンバージェンスへの革新的なアプローチで際立っています。単一のオペレーティング・システムにより、ラップトップ、タブレット、ミニPC、スマートフォンなど、さまざまなデバイスでスムーズでシームレスな操作が可能になります。



PureOSのアプリケーションは、画面サイズや入力モード（タッチまたはキーボード/マウス）に自動的に適応するように設計されています。例えば、GNOMEウェブブラウザは、モバイルデバイスとデスクトップデバイスの両方で最適なエクスペリエンスを提供するために、Interfaceを動的に適応させます。



PureOSには**LibreOffice**オフィススイートも含まれています：





- **Writer**：文書の作成と編集のための完全なワードプロセッサ。
- **Calc**: データと計算を管理するための強力なスプレッドシートプログラムです。
- **Impress**：プロフェッショナルなプレゼンテーションを作成するためのツール。



この無料スイートを使えば、プロプライエタリなソフトウェアに依存することなく、効率的に作業を進めることができる。



PureOSは、100％オープンソースのディストリビューションに基づき、統合された安全で柔軟な環境を提供します。その真のコンバージェンスは、複雑な適応を必要とせずに、コンピュータ、タブレット、スマートフォンなどの異なるタイプのデバイスと互換性のあるアプリケーションの作成を容易にします。



必須ツールへのネイティブアクセス、堅牢なパッケージマネージャ、豊富なオープンソースエコシステムを備えたPureOSは、セキュアな環境でのアプリケーション開発、テスト、デプロイを簡素化します。安定したアーキテクチャとCommitmentの機密性により、Blockchainの開発、ラピッドプロトタイピング、本番環境など、さまざまな用途に対応する信頼性の高いプラットフォームです。
