## PureOS 시작하기



PureOS는 퓨리즘에서 개발한 데비안 기반 운영체제입니다. PureOS는 간단하고 사용하기 쉬운 배포판을 찾는 IT 전문가와 사용자 모두에게 적합합니다. 이 운영체제는 *자유 소프트웨어*라는 점에서 독특하며, Debian이 제공하는 개인 정보 보호, 자유, 보안 및 안정성을 기반으로 프레임워크를 설정하여 사용자의 개인 데이터 보호에 중점을 둡니다.



### 왜 PureOS를 선택해야 하나요?





- 간단하고 직관적인 Interface**: GNOME은 명령줄에 익숙하지 않은 사용자도 쉽게 사용할 수 있도록 설계된 깔끔한 Interface 데스크톱을 제공합니다.





- 무료**: 대부분의 Linux 배포판과 마찬가지로 PureOS는 완전히 무료로 사용할 수 있습니다. 그러나 개발자를 지원하기 위해 월간 구독을 이용할 수 있습니다.





- 보안 및 안정성**: PureOS의 아키텍처와 운영 모드는 데이터 보호와 시스템 안정성을 보장하는 매우 안전한 배포판입니다.





- 문서와 활발한 커뮤니티**: PureOS는 명확하고 접근하기 쉬운 문서와 헌신적이고 반응이 빠른 커뮤니티를 갖추고 있어 문제를 쉽게 해결하고 시스템을 단계별로 학습할 수 있습니다.



## 설치 및 구성



컴퓨터에 PureOS를 설치하고 구성하려면 다음과 같은 최소한의 기능이 필요합니다:




- 시스템 플래시를 위한 최소 8GB의 USB 키가 필요합니다.
- 4GB RAM.
- gW-1 디스크에 30GB의 여유 공간이 있습니다.



PureOS 공식 웹사이트](https://pureos.net/)로 이동한 다음 사용 중인 머신의 아키텍처에 따라 운영 체제의 ISO 이미지를 다운로드하세요.



PureOS 설치를 시작하려면 [Balena Etcher](https://www.balena.io/etcher)와 같은 플래시 소프트웨어를 사용하여 부팅 가능한 USB 키를 만들어야 합니다.



간단한 3단계로 PureOS 운영 체제로 부팅된 USB 스틱을 얻을 수 있습니다.





- USB 키를 연결하고 다운로드한 Balena 소프트웨어를 실행합니다.
- 그런 다음 PureOS ISO 이미지를 선택합니다
- 출력 장치로 USB 키를 선택한 다음 **플래시** 버튼을 클릭하고 프로세스가 완료될 때까지 기다립니다.



![0_01](assets/fr/01.webp)



USB 키가 부팅되면 PureOS를 설치하려는 컴퓨터를 재시작합니다.



부팅할 때 기기에 따라 `ESC`, `F9` 또는 `F11` 키를 눌러 BIOS에 액세스합니다. USB 키를 부팅 장치로 선택한 다음 `ENTER`를 눌러 확인합니다.

## 업데이트



PureOS를 사용하기 전에 시스템을 업데이트하는 것이 필수적입니다. 이렇게 하면 최신 기능과 보안 패치의 이점을 누리고 안정성을 높일 수 있습니다.





- Interface 그래픽을 통해 업데이트합니다:


소프트웨어** 애플리케이션을 연 다음 **업데이트** 탭으로 이동합니다. 사용 가능한 업데이트가 자동으로 표시됩니다. 다운로드가 완료되면 **다운로드**를 클릭한 다음 **설치**를 클릭합니다.





- 터미널**을 통해 업데이트하세요:


터미널을 연 다음 다음 명령을 입력하여 사용 가능한 패키지 목록을 업데이트합니다:



```shell
sudo apt update
```



비밀번호를 입력하고 확인합니다. 그런 다음 업데이트를 설치합니다:



```shell
sudo apt full-upgrade
```



## 개발용 PureOS



기본적으로 PureOS에는 개발에 필요한 모든 도구가 포함되어 있지 않습니다.


다음 명령을 사용하여 빠르게 설치할 수 있습니다:



```shell
sudo apt install build-essential git curl
```



이렇게 하면 대부분의 개발 환경에서 유용한 **Git** 및 **Curl** 컴파일 도구가 설치됩니다.



## PureOS 환경



단일 운영 체제로 노트북, 태블릿, 미니 PC, 스마트폰 등 다양한 기기에서 원활하고 원활한 작동을 보장하는 퓨어OS는 진정한 컨버전스를 위한 혁신적인 접근 방식이 돋보이는 제품입니다.



PureOS 애플리케이션은 적응형으로 설계되어 화면 크기와 입력 모드(터치 또는 키보드/마우스)에 따라 자동으로 조정됩니다. 예를 들어, GNOME 웹 브라우저는 모바일과 데스크톱 기기 모두에서 최적의 경험을 제공하기 위해 Interface를 동적으로 조정합니다.



PureOS에는 **LibreOffice** 오피스 제품군도 포함되어 있습니다:





- Writer**: 문서 작성 및 편집을 위한 완벽한 워드 프로세서입니다.
- Calc**: 데이터와 계산을 관리할 수 있는 강력한 스프레드시트 프로그램입니다.
- Impress**: 전문적인 프레젠테이션을 만들기 위한 도구입니다.



이 무료 제품군을 사용하면 독점 소프트웨어에 의존하지 않고도 효율적으로 작업할 수 있습니다.



PureOS는 100% 오픈 소스 배포를 기반으로 하는 통합적이고 안전하며 유연한 환경을 제공하며, 완벽한 제어와 엄격한 개인 정보 보호를 보장합니다. 진정한 컨버전스를 통해 복잡한 조정 없이도 컴퓨터, 태블릿, 스마트폰 등 다양한 유형의 디바이스와 호환되는 애플리케이션을 쉽게 만들 수 있습니다.



필수 도구, 강력한 패키지 관리자 및 풍부한 오픈 소스 에코시스템에 대한 기본 액세스를 통해 PureOS는 안전한 환경에서 애플리케이션 개발, 테스트 및 배포를 간소화합니다. 안정적인 아키텍처와 기밀성을 보장하는 Commitment은 Blockchain 개발, 신속한 프로토타이핑 또는 프로덕션 환경 등 다양한 용도로 사용할 수 있는 신뢰할 수 있는 플랫폼입니다.
