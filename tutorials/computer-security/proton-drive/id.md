Sangatlah penting di era sekarang untuk menetapkan strategi guna memastikan aksesibilitas, keamanan, dan pencadangan file pribadi Anda, seperti dokumen personal, foto, atau proyek penting. Kehilangan data-data ini dapat berakibat fatal.

Untuk mencegah masalah ini, saya menyarankan untuk menjaga beberapa salinan cadangan file Anda pada media yang berbeda. Sebuah strategi yang umum digunakan dalam komputasi adalah strategi pencadangan "3-2-1", yang menjamin proteksi berkas Anda:
- **3** salinan file Anda;
- Disimpan setidaknya pada **2** jenis media yang berbeda;
- Dengan setidaknya **1** salinan disimpan di lokasi lain.

Dengan kata lain, sangat disarankan untuk menyimpan berkas Anda di 3 lokasi berbeda, menggunakan jenis media yang bervariasi, seperti komputer Anda, hard drive eksternal, USB stick, atau layanan penyimpanan online. Dan terakhir, memiliki salinan di lokasi lain berarti Anda harus menyimpan cadangan di luar rumah atau tempat bisnis Anda. Poin terakhir ini membantu menghindari kerugian total berkas Anda jika terjadi bencana lokal seperti kebakaran atau banjir. Salinan eksternal, yang jauh dari rumah atau bisnis Anda, memastikan data Anda akan tetap bertahan terlepas dari risiko lokal.

Untuk memfasilitasi implementasi strategi pencadangan 3-2-1, Anda dapat memanfaatkan layanan penyimpanan online. Solusi ini, yang secara umum disebut "cloud", menawarkan perlindungan tambahan dengan menyimpan data Anda pada server yang aman dan dapat diakses dari perangkat mana pun. Istilah "cloud" sendiri merujuk pada penyimpanan data pada server eksternal.

Banyak orang menggunakan solusi penyimpanan dari perusahaan digital besar: Google Drive, Microsoft OneDrive, atau Apple iCloud.
![PROTON DRIVE](assets/notext/01.webp)

Solusi penyimpanan yang disebutkan tadi memang nyaman untuk penggunaan sehari-hari dan menjamin data Anda mudah diakses, namun tidak menjamin kerahasiaannya. Dalam tutorial ini, saya akan mengajak Anda mengenal solusi lain yang sama mudahnya digunakan dengan sarana penyimpanan dari perusahaan teknologi besar, tetapi dengan langkah-langkah tambahan untuk melindungi privasi Anda. Solusi ini adalah Proton Drive, sarana penyimpanan online dari perusahaan Swiss bernama Proton. Kita juga akan melihat bagaimana cara mudah menerapkan strategi 3-2-1 yang cocok untuk penggunaan sehari-hari.

## Pengenalan ke Proton Drive

Proton Drive adalah solusi penyimpanan online yang menarik karena menggabungkan kemudahan penggunaan dengan keamanan untuk file-file Anda. Tidak seperti layanan cloud tradisional dari raksasa teknologi, Proton Drive menerapkan langkah-langkah untuk melindungi privasi Anda. Ini memastikan enkripsi end-to-end untuk semua file Anda, yang berarti bahkan tim Proton sendiri tidak dapat mengakses data Anda. Selain itu, Proton Drive bersifat open-source, memungkinkan para ahli independen untuk dengan bebas mengaudit kode perangkat lunak tersebut.
![PROTON DRIVE](assets/notext/02.webp)

Model bisnis Proton didasarkan pada sistem langganan, yang memberikan keyakinan karena menunjukkan bahwa perusahaan didanai tanpa harus mengeksploitasi data penggunanya. Dalam tutorial ini, saya akan menjelaskan cara menggunakan versi gratis Proton Drive, namun tersedia juga beberapa tingkatan langganan yang menawarkan lebih banyak fitur. Model bisnis ini lebih disukai dibandingkan sistem gratis ala raksasa teknologi, yang mungkin menimbulkan pertanyaan apakah data pribadi kita digunakan untuk keuntungan. Hal ini tampaknya tidak terjadi pada Proton.

Proton Drive menawarkan lebih dari sekadar opsi penyimpanan sederhana; ia juga memungkinkan berbagi, mengedit, dan berkolaborasi pada dokumen secara online dengan sarana pengeditan, serupa dengan kumpulan perangkat lunak Google.

Mengenai [penetapan harga](https://proton.me/pricing), versi gratis menawarkan penyimpanan hingga 5 GB dan mencakup fitur-fitur dasar. Untuk memperluas kapasitas penyimpanan hingga 200 GB, langganan spesifik Proton Drive tersedia dengan biaya €4 per bulan. Paket Proton Unlimited, di sisi lain, menawarkan ruang penyimpanan hingga 500 GB di Proton Drive dengan biaya €10 per bulan, di samping mencakup semua layanan berbayar Proton, seperti VPN dan pengelola kata sandi, serta manfaat tambahan pada sarana gratis (email dan kalender).
![PROTON DRIVE](assets/notext/03.webp)

## Bagaimana cara menginstal perangkat lunak Proton Drive?

Proton Drive juga menyediakan perangkat lunak yang memungkinkan sinkronisasi file lokal Anda dengan ruang penyimpanan online Anda. Fitur ini mempermudah dan mengotomatiskan penerapan strategi cadangan 3-2-1 kita. Dengan perangkat lunak Proton Drive, Anda akan mendapatkan 2 salinan file yang tersinkronisasi: satu di komputer Anda dan satu lagi di server Proton. Ini secara otomatis memenuhi kriteria 2 jenis media dan cadangan beda lokasi. Anda hanya perlu membuat salinan ketiga, yang akan kita atur nanti.

Untuk menggunakan perangkat lunak, klik pada tab "*Computers / Komputer*" di akun Proton Drive Anda dan pilih tombol yang sesuai dengan sistem operasi Anda untuk melanjutkan dengan pengunduhan.
![PROTON DRIVE](assets/notext/25.webp)

Setelah terinstal, Anda perlu masuk untuk membuka akun Anda, kemudian klik pada "*Sign in / Masuk*".
![PROTON DRIVE](assets/notext/26.webp)

Pilih file lokal yang ingin Anda sinkronkan dengan Proton Drive Anda.
![PROTON DRIVE](assets/notext/27.webp)

Sebagai contoh, saya hanya memilih folder "*Proton Backup*". Kemudian klik tombol "*Continue / Lanjutkan*".
![PROTON DRIVE](assets/notext/28.webp)

Anda kemudian akan ada pada tampilan perangkat lunak, yang serupa dengan aplikasi web.
![PROTON DRIVE](assets/notext/29.webp)

Mulai sekarang, Anda akan memiliki folder bernama "*Proton Drive*" secara lokal di komputer Anda. Folder ini akan mengumpulkan semua dokumen Anda yang tersimpan di Proton secara online. Jika Anda menambahkan file ke folder ini dari komputer Anda, Anda akan secara otomatis menemukannya di halaman utama aplikasi web Proton Drive, dan begitu pula sebaliknya. Untuk folder-folder yang Anda pilih untuk disinkronkan selama instalasi perangkat lunak, Anda juga dapat menemukannya secara online dengan membuka bagian "Computers" di Proton Drive, lalu memilih komputer Anda.
![PROTON DRIVE](assets/notext/30.webp)

Dengan demikian, semua file Anda dicadangkan dan disinkronkan baik secara lokal di komputer Anda maupun di server online Proton Drive.

## Bagaimana cara membuat cadangan dari Proton Drive?

Jika Anda sudah mengikuti langkah-langkah sebelumnya, Anda sekarang memiliki 2 lokasi cadangan yang berbeda untuk file penting Anda. Untuk melengkapi strategi cadangan 3-2-1 kita, kita perlu menambahkan salinan ketiga.

Saya menyarankan Anda untuk melakukan cadangan tambahan ini pada media eksternal, seperti hard drive eksternal atau USB stick. Tergantung seberapa sering Anda menggunakan file tersebut, tentukan frekuensi pembaruan cadangan yang sesuai (mingguan, bulanan, setengah tahun sekali, atau lainnya). Pada setiap interval yang Anda pilih, Anda perlu mengunduh seluruh isi Proton Drive Anda untuk mencadangkan data ke media eksternal yang sudah Anda pilih. Dengan cara ini, bahkan jika komputer Anda dicuri dan server Proton hancur secara bersamaan, Anda tetap akan memiliki akses aman ke file Anda berkat salinan yang ada di USB stick.

Untuk melakukan ini, pergi ke Proton Drive Anda.
![PROTON DRIVE](assets/notext/31.webp)

Pilih semua file Anda.
![PROTON DRIVE](assets/notext/32.webp)

Kemudian klik pada panah kecil untuk mengunduhnya.
![PROTON DRIVE](assets/notext/33.webp)

Kita kemudian akan mengulangi operasi dengan file kita yang disinkronkan dari komputer kita.
![PROTON DRIVE](assets/notext/34.webp)

Anda kemudian akan menemukan file .zip di unduhan Anda. Cukup hubungkan media eksternal pilihan Anda ke komputer, dan kemudian transfer file-file ini ke sana.
![PROTON DRIVE](assets/notext/35.webp)

Jika Anda khawatir USB stick ini mungkin dicuri, pertimbangkan untuk mengenkripsinya dengan perangkat lunak seperti VeraCrypt (kami akan membuat tutorial tentang perangkat lunak ini segera).

Dengan memilih Proton Drive untuk cadangan online Anda, Anda juga mendapatkan manfaat enkripsi end-to-end yang menjamin perlindungan privasi Anda.
