Arch Linux jest popularnym systemem operacyjnym w wielu dziedzinach, dzięki swojej solidności i stabilności. Jednak może być trudny do opanowania dla początkujących użytkowników. Właśnie w celu rozwiązania tego problemu powstał **Manjaro**: aby zaoferować moc Arch Linux, ale z prostszym, bardziej dostępnym doświadczeniem, opartym na intuicyjnym, łatwym do nauczenia Interface.



## Pierwsze kroki z Manjaro



Jednym z największych atutów Manjaro jest **prosty i wydajny system aktualizacji**. Nie trzeba zarządzać nimi ręcznie: Manjaro zajmuje się nimi za ciebie! Ikona w obszarze powiadomień (lokalizacja różni się w zależności od edycji) powiadamia o dostępności aktualizacji. Wszystko, co musisz zrobić, to postępować zgodnie z instrukcjami, a proces jest szybki i łatwy.



Manjaro oferuje również **bogaty katalog aplikacji**. Ponieważ Manjaro bazuje na Arch Linux, korzysta z bezpośredniego dostępu do jego oficjalnych repozytoriów, bogatych w różnorodne oprogramowanie, w tym aplikacje własnościowe. Manjaro nieznacznie opóźnia niektóre aktualizacje Arch Linux w celu przeprowadzenia dodatkowych testów; poprawia to stabilność kosztem niewielkiego opóźnienia w nowych wydaniach. A jeśli ten wybór ci nie wystarcza, możesz również uzyskać dostęp do **AUR (*Arch User Repository*)**, ogromnej biblioteki zarządzanej przez społeczność. Jeśli jakiegoś programu nie ma w oficjalnych repozytoriach, istnieje duże prawdopodobieństwo, że jest on dostępny w AUR.



Kolejną zaletą Manjaro jest to, że jest **znacznie mniej zasobożerny** niż systemy takie jak Windows czy macOS. Zużywa mniej pamięci RAM i mocy obliczeniowej, co czyni go idealnym wyborem dla starszych lub mniej wydajnych komputerów: twoja maszyna zyskuje na płynności i szybkości, odzyskując drugą młodość.



Co więcej, Manjaro jest **całkowicie darmowy**. W przeciwieństwie do Windows lub macOS, nie musisz nic płacić, aby go zainstalować i w pełni wykorzystać jego możliwości. Wreszcie, oferuje **zwiększone bezpieczeństwo** dzięki regularnym, szybkim aktualizacjom, ograniczając narażenie na luki w zabezpieczeniach. Aktywna społeczność zapewnia również, że wszelkie problemy są szybko rozwiązywane i proponowane są skuteczne rozwiązania.



## Odkryj Manjaro OS



Przed rozpoczęciem instalacji Manjaro warto wiedzieć, że dystrybucja ta jest dostępna w kilku wersjach. Każda z tych edycji oferuje unikalne środowisko graficzne, które wpływa zarówno na przepływ pracy, jak i zużycie zasobów systemowych. Istnieją trzy główne oficjalne edycje Manjaro.



![0_01](assets/fr/01.webp)





- Edycja **KDE Plasma** jest najbardziej konfigurowalna. Jeśli szukasz systemu, który jest zarówno elegancki wizualnie, jak i wydajny, KDE Plasma jest doskonałym wyborem. To stabilne środowisko graficzne jest idealne dla użytkowników, którzy chcą mieć całkowitą kontrolę i dostosowane do swoich potrzeb doświadczenie.





- Dla maszyn z bardziej ograniczonymi zasobami, idealnym rozwiązaniem jest edycja **Xfce**. Jego Interface jest lekki i intuicyjny, gwarantując płynną pracę nawet na starszych komputerach. Co więcej, jego układ przypomina Windows, dzięki czemu przejście na Linuksa jest łatwiejsze dla nowych użytkowników.





- Wreszcie, edycja **GNOME** oferuje zupełnie inne podejście. Uproszczona konstrukcja kładzie nacisk na produktywność i organizację poprzez wirtualne obszary robocze. Ten skoncentrowany na działaniach przepływ pracy jest szczególnie atrakcyjny dla użytkowników już zaznajomionych z Linuksem i poszukujących minimalistycznego, wysoce wydajnego środowiska.



### Inne wydania Manjaro



![0_02](assets/fr/02.webp)



Oprócz oficjalnych wydań, społeczność Manjaro oferuje również inne wersje. Te alternatywne edycje zostały zaprojektowane w celu zaspokojenia określonych potrzeb i oferują różnorodne środowiska graficzne.



Edycja **Cinnamon** to doskonały wybór, jeśli dopiero zaczynasz i szukasz znajomego Interface. Został zaprojektowany tak, aby był łatwy w użyciu, zachowując klasyczne konwencje tradycyjnych środowisk biurowych.



Dla bardziej zaawansowanych użytkowników dostępne są wersje takie jak **i3 Window Manager** lub **Sway**. Znacznie lżejsze i szybsze, wymagają jednak dobrego opanowania wiersza poleceń, aby można je było w pełni skonfigurować i wykorzystać. Środowiska te są idealne dla tych, którzy chcą mieć całkowitą kontrolę nad swoim systemem i stawiają wydajność ponad wszystko.



## Wymagania techniczne



Aby Manjaro działało optymalnie, komputer musi spełniać kilka minimalnych wymagań. Zalecamy posiadanie co najmniej :





- Procesor 64-bitowy (x86_64)**
- zalecane 4 GB pamięci RAM (minimum 2 GB)** (patrz poniżej)
- 30 GB miejsca na dysku (więcej, jeśli utworzysz dedykowaną partycję `/home`)**



## Instalacja Manjaro



Aby pobrać plik, przejdź do [oficjalnej strony Manjaro] (https://manjaro.org/) i wybierz edycję, która najlepiej odpowiada Twoim potrzebom. Po pobraniu pliku należy utworzyć bootowalny klucz USB z obrazem ISO Manjaro.



Następnie przejdź do strony oprogramowania [Rufus] (https://rufus.ie/fr/) i pobierz je. Uruchom program, podłącz klucz USB, wybierz obraz ISO Manjaro i rozpocznij flashowanie. Poczekaj na zakończenie procesu przed wyjęciem klucza. Następnie można ponownie uruchomić komputer.



![0_03](assets/fr/03.webp)



Aby zainstalować Manjaro na komputerze, należy go najpierw całkowicie wyłączyć. Następnie podłącz klucz USB, włącz komputer ponownie i uzyskaj dostęp do menu rozruchowego lub oprogramowania układowego UEFI/BIOS, naciskając **F2, F10, F12, Escape** lub **Delete** (w zależności od producenta).



Następnie wybierz klucz USB jako urządzenie rozruchowe, aby rozpocząć proces instalacji systemu operacyjnego.



### Ekran startowy



Przy pierwszym uruchomieniu Manjaro z klucza USB zostaniesz przeniesiony bezpośrednio do menu instalacji. Przed uruchomieniem instalacji można zmienić układ klawiatury lub język systemu.



Następnie wybierz opcję **Boot with open source drivers**, aby rozpocząć instalację Manjaro. Te sterowniki open source są kompatybilne z większością sprzętu i wystarczą w większości przypadków. Jeśli na przykład posiadasz kartę graficzną NVIDIA lub potrzebujesz wyższej wydajności graficznej, możesz wybrać **Boot with proprietary drivers**, który wykorzystuje sterowniki własnościowe.



![0_04](assets/fr/04.webp)



System uruchomi się w **domyślnym trybie live**. Pozwoli ci to przetestować funkcjonalność Manjaro, aby sprawdzić, czy odpowiada twoim potrzebom, zanim zainstalujesz go na stałe. Gdy wszystko będzie gotowe, kliknij opcję **Install Manjaro Linux**.



Wybierz żądany język instalacji, a następnie kliknij **Next**.



![0_06](assets/fr/06.webp)



Następnie wybierz swoją lokalizację, aby ustawić prawidłową strefę czasową. Na tym etapie można również zmienić język i format daty.



![0_07](assets/fr/07.webp)



Wybierz układ klawiatury. Dostępne jest pole testowe umożliwiające sprawdzenie, czy wpisywane klawisze odpowiadają oczekiwanym znakom.



![0_08](assets/fr/08.webp)



### Podział dysku na partycje


Dostępne są dwie opcje partycjonowania dysku.





- Pierwszym i najprostszym sposobem jest wymazanie całego dysku i zainstalowanie na nim Manjaro.
- Drugi umożliwia **ręczne partycjonowanie**.



![0_09](assets/fr/09.webp)



Aby zapewnić czystą instalację, zalecamy utworzenie co najmniej trzech partycji:





- Pierwsza partycja **516 MB** (podstawowa) dla **boot**.
- Druga partycja **2 GB** (logiczna) dla **swap**.
- Trzecia partycja na **dane osobowe**.



Jeśli chcesz równolegle zainstalować inny system, musisz podzielić tę ostatnią partycję i przydzielić tylko jedną część Manjaro (co najmniej **15 GB**, aby zagwarantować prawidłowe działanie systemu).


### Tworzenie konta użytkownika



Utwórz konto użytkownika w systemie, wypełniając wymagane informacje. Na koniec ustaw hasło administratora (**root**). Ten administrator jest **superużytkownikiem** z pełnymi prawami w systemie i możliwością wykonywania zaawansowanych poleceń.



![0_10](assets/fr/10.webp)



### Wybierz odpowiedni pakiet biurowy



Manjaro pozwala wybrać pomiędzy **FreeOffice** i **LibreOffice**.





- LibreOffice** jest bardziej kompletny, z szerszym zakresem oprogramowania i zaawansowanymi funkcjami.
- Z drugiej strony FreeOffice** jest lżejszy i zawiera tylko niezbędne elementy: **TextMaker**, PlanMaker** i Presentations**.



![0_11](assets/fr/11.webp)



### Podsumowanie konfiguracji



Ekran podsumowania pokazuje wszystkie ustawione parametry. W razie potrzeby można cofnąć się, aby wprowadzić zmiany, bez wpływu na inne już wprowadzone ustawienia.



Następnie kliknij **Install** i potwierdź, aby rozpocząć właściwą instalację.



![0_12](assets/fr/12.webp)

Pod koniec procesu, zaznacz pole **Restart now**, a następnie kliknij **Done**. System uruchomi się ponownie i **Manjaro będzie gotowe do użycia**.

## Pierwsze kroki z Manjaro



Instalacja Manjaro to tylko pierwszy krok. Aby jak najlepiej wykorzystać swój system, oto kilka przydatnych rzeczy, które warto wiedzieć.



### Aktualizacja systemu



Manjaro znacznie upraszcza aktualizacje. Aby zaktualizować pakiety :



```shell
sudo pacman -Syu
```



To polecenie pobiera i instaluje najnowsze dostępne wersje. Zalecamy jego regularne uruchamianie, aby utrzymać **bezpieczny i stabilny** system.



### Konfigurowanie środowiska programistycznego



Aby tworzyć aplikacje internetowe na Manjaro, zainstaluj niezbędne narzędzia za pomocą jednego polecenia:



```shell
sudo pacman -S nodejs npm git yarn
```



To polecenie instaluje Node.js i npm do uruchamiania i zarządzania projektami JavaScript i TypeScript, Git do zarządzania wersjami oraz Yarn jako alternatywny menedżer pakietów.



### Instalacja Bitcoin Wallet



Aby zarządzać swoimi bitcoinami na Manjaro, możesz zainstalować **Electrum**, lekki i bezpieczny Wallet :



```shell
sudo pacman -S electrum  # Install Electrum
```



Electrum pozwala z łatwością **odbierać i wysyłać bitcoiny**, oferując jednocześnie zaawansowane funkcje, takie jak zarządzanie wieloma Wallet i ochrona passphrase. Aby uzyskać kompletny przewodnik po korzystaniu z Electrum, zapoznaj się z naszym dedykowanym samouczkiem, który wyjaśnia, jak utworzyć Wallet, zabezpieczyć swoje środki i przeprowadzać transakcje.

## Zabezpieczanie systemu Manjaro



Bezpieczeństwo jest kluczowym aspektem każdej instalacji systemu Linux. Ważne jest, aby chronić poufność i integralność danych.



### Konfiguracja zapory sieciowej



Manjaro zawiera UFW (*Uncomplicated Firewall*), program do zarządzania zaporami sieciowymi, ale trzeba go aktywować ręcznie:



```bash
# Installation if not present
sudo pacman -S ufw

# Firewall activation
sudo systemctl enable ufw.enable

sudo ufw enable

# Allow SSH connections (optional)
sudo ufw allow ssh

# Check the status
sudo ufw status verbose
```



![verbose](assets/fr/15.webp)



### Zarządzanie uprawnieniami użytkowników



1. **Utworzenie nieuprzywilejowanego użytkownika**



```shell
sudo useradd -m username
sudo passwd username
```



2. **Konfiguracja pliku Sudoers**



```shell
sudo EDITOR=nano visudo
```

Manjaro łączy w sobie **stabilność, szybkość i bezpieczeństwo**, pozostając jednocześnie **całkowicie darmowym**, co czyni go idealnym wyborem zarówno dla początkujących, jak i zaawansowanych użytkowników.
