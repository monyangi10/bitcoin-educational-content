Arch Linux es un sistema operativo popular en muchos campos, gracias a su robustez y estabilidad. Sin embargo, puede resultar difícil de manejar para los usuarios novatos. Precisamente para Address este problema se creó **Manjaro**: para ofrecer la potencia de Arch Linux, pero con una experiencia más sencilla y accesible, basada en un Interface intuitivo y fácil de aprender.



## Primeros pasos con Manjaro



Uno de los mayores activos de Manjaro es su **simple y eficiente sistema de actualizaciones**. No hay necesidad de gestionarlas manualmente: ¡Manjaro se encarga de ellas por ti! Un icono en el área de notificación (la ubicación varía según la edición) te avisa cuando hay una actualización disponible. Todo lo que tienes que hacer es seguir las instrucciones, y el proceso es rápido y sin esfuerzo.



Manjaro también ofrece un **vastísimo catálogo de aplicaciones**. Dado que Manjaro está basado en Arch Linux, se beneficia del acceso directo a sus repositorios oficiales, ricos en una gran variedad de software, incluyendo aplicaciones propietarias. Manjaro retrasa ligeramente algunas actualizaciones de Arch Linux para realizar pruebas adicionales; esto mejora la estabilidad a costa de un ligero retraso en las nuevas versiones. Y si esta opción no es suficiente para ti, también puedes acceder al **AUR (*Arch User Repository*)**, una enorme biblioteca gestionada por la comunidad. Si un programa no existe en los repositorios oficiales, lo más probable es que esté disponible en el AUR.



Otra ventaja de Manjaro es que **consume muchos menos recursos** que sistemas como Windows o macOS. Consume menos RAM y potencia de cálculo, lo que lo convierte en una opción ideal para ordenadores antiguos o menos potentes: tu máquina gana en fluidez y velocidad, recuperando una segunda juventud.



Además de todo eso, Manjaro es **totalmente gratis**. A diferencia de Windows o macOS, no tienes que pagar nada para instalarlo y sacarle el máximo partido. Por último, ofrece **seguridad mejorada** gracias a actualizaciones rápidas y regulares, que limitan la exposición a vulnerabilidades. Su activa comunidad también garantiza que cualquier problema se corrija rápidamente y se propongan soluciones eficaces.



## Descubre Manjaro OS



Antes de empezar a instalar Manjaro, es importante saber que esta distribución está disponible en varias ediciones. Cada una de estas ediciones ofrece un entorno de escritorio único que influye tanto en tu flujo de trabajo como en el consumo de recursos del sistema. Hay tres ediciones oficiales principales de Manjaro.



![0_01](assets/fr/01.webp)





- La edición **KDE Plasma** es la más personalizable. Si buscas un sistema que sea a la vez visualmente elegante y de alto rendimiento, KDE Plasma es una excelente elección. Este entorno de escritorio estable es ideal para los usuarios que desean un control total y una experiencia a medida.





- Para máquinas con recursos más limitados, la edición **Xfce** es la solución ideal. Su Interface es ligero e intuitivo, lo que garantiza un funcionamiento fluido incluso en ordenadores antiguos. Además, su diseño recuerda al de Windows, lo que facilita la transición a Linux a los nuevos usuarios.





- Por último, la edición **GNOME** ofrece un enfoque totalmente distinto. Su diseño aerodinámico hace hincapié en la productividad y la organización a través de espacios de trabajo virtuales. Este flujo de trabajo centrado en la actividad resulta especialmente atractivo para los usuarios ya familiarizados con Linux y que buscan un entorno minimalista y altamente eficiente.



### Otras ediciones de Manjaro



![0_02](assets/fr/02.webp)



Además de las ediciones oficiales, la comunidad Manjaro también ofrece otras versiones. Estas ediciones alternativas están diseñadas para satisfacer necesidades específicas y ofrecen una gran variedad de entornos de escritorio.



La edición **Cinnamon** es una opción excelente si estás empezando y buscas un Interface familiar. Se ha diseñado para que sea fácil de usar, conservando las convenciones clásicas de los entornos de oficina tradicionales.



Para usuarios más avanzados, existen ediciones como **i3 Window Manager** o **Sway**. Mucho más ligeras y rápidas, requieren no obstante un buen dominio de la línea de comandos para ser plenamente configuradas y explotadas. Estos entornos son ideales para quienes desean un control total sobre su sistema y anteponen la eficacia a todo lo demás.



## Requisitos técnicos



Para que Manjaro funcione de forma óptima, tu ordenador debe cumplir unos requisitos mínimos. Te recomendamos que tengas al menos :





- Un procesador de 64 bits (x86_64)
- se recomiendan **4 GB de RAM (mínimo 2 GB)** (véase más abajo)
- 30 GB de espacio en disco (más si creas una partición `/home` dedicada)



## Instalación de Manjaro



Para descargarlo, visita [el sitio web oficial de Manjaro](https://manjaro.org/) y elige la edición que mejor se adapte a tus necesidades. Una vez descargado el archivo, tendrás que crear una llave USB de arranque con la imagen ISO de Manjaro.



Luego ve al sitio web del programa [Rufus] (https://rufus.ie/fr/) y descárgalo. Ejecuta el programa, conecta tu llave USB, selecciona la imagen ISO de Manjaro y empieza a flashear. Espera a que termine el proceso antes de retirar la llave. A continuación, puedes reiniciar el ordenador.



![0_03](assets/fr/03.webp)



Para instalar Manjaro en tu ordenador, primero apágalo completamente. A continuación, conecta la llave USB, vuelve a encender la máquina y accede al menú de arranque o al firmware UEFI/BIOS pulsando **F2, F10, F12, Escape** o **Supr** (según el fabricante).



A continuación, elija la llave USB como dispositivo de arranque para iniciar el proceso de instalación del sistema operativo.



### Pantalla de inicio



La primera vez que inicies Manjaro desde la llave USB, accederás directamente al menú de instalación. Antes de iniciar la instalación, puedes cambiar la distribución del teclado o el idioma del sistema.



A continuación, seleccione la opción **Boot with open source drivers** para iniciar la instalación de Manjaro. Estos controladores de código abierto son compatibles con la mayoría del hardware y serán suficientes en la mayoría de los casos. Si tienes una tarjeta gráfica NVIDIA, por ejemplo, o necesitas un mayor rendimiento gráfico, puedes elegir **Arrancar con controladores propietarios**, que utiliza controladores propietarios.



![0_04](assets/fr/04.webp)



El sistema se iniciará en **modo por defecto**. Esto te permite probar la funcionalidad de Manjaro para ver si se adapta a tus necesidades antes de instalarlo permanentemente. Una vez listo, haz clic en la opción **Instalar Manjaro Linux**.



Seleccione el idioma deseado para su instalación y haga clic en **Siguiente**.



![0_06](assets/fr/06.webp)



A continuación, elige tu ubicación para establecer la zona horaria correcta. En esta fase, también puedes cambiar el idioma y el formato de la fecha.



![0_07](assets/fr/07.webp)



Seleccione la distribución del teclado. Dispone de un campo de prueba para comprobar que las teclas tecleadas corresponden a los caracteres esperados.



![0_08](assets/fr/08.webp)



### Partición de discos


Tienes dos opciones para particionar tu disco.





- La primera, y más sencilla, es borrar todo el disco e instalar Manjaro en él.
- El segundo permite la **partición manual**.



![0_09](assets/fr/09.webp)



Para una instalación limpia, recomendamos crear al menos tres particiones:





- Una primera partición de **516 MB** (primaria) para el **boot**.
- Una segunda partición (lógica) de **2 GB** para **swap**.
- Una tercera partición para tus **datos personales**.



Si deseas instalar otro sistema en paralelo, necesitas dividir esta última partición y asignar sólo una parte a Manjaro (al menos **15 GB** para garantizar el correcto funcionamiento del sistema).


### Crear una cuenta de usuario



Cree una cuenta de usuario en el sistema rellenando la información requerida. Por último, establece la contraseña del administrador (**root**). Este administrador es un **superusuario** con plenos derechos sobre el sistema y capacidad para ejecutar comandos avanzados.



![0_10](assets/fr/10.webp)



### Elija el paquete ofimático adecuado



Manjaro te permite elegir entre **FreeOffice** y **LibreOffice**.





- **LibreOffice** es más completo, con una gama más amplia de programas y funciones avanzadas.
- **LibreOffice**, en cambio, es más ligero e incluye sólo lo esencial: **TextMaker**, **PlanMaker** y **Presentaciones**.



![0_11](assets/fr/11.webp)



### Resumen de la configuración



Una pantalla de resumen te muestra todos los parámetros que has configurado. Puedes volver atrás para hacer cambios si es necesario, sin que ello afecte a otros ajustes que ya hayas hecho.



A continuación, haga clic en **Instalar** y confirme para iniciar la instalación propiamente dicha.



![0_12](assets/fr/12.webp)



![0_13](assets/fr/13.webp)



### Fin de la instalación



Al final del proceso, marca la casilla **Reiniciar ahora** y haz clic en **Hecho**. El sistema se reiniciará y **Manjaro estará listo para su uso**.



![0_13](assets/fr/14.webp)



## Primeros pasos con Manjaro



Instalar Manjaro es sólo el primer paso. Para sacar el máximo partido a tu sistema, aquí tienes algunas cosas útiles que debes saber.



### Actualizar el sistema



Manjaro simplifica enormemente las actualizaciones. Para actualizar sus paquetes :



```shell
sudo pacman -Syu
```



Este comando descarga e instala las últimas versiones disponibles. Le recomendamos que lo ejecute regularmente para mantener su sistema **seguro y estable**.



### Configurar un entorno de desarrollo



Para desarrollar aplicaciones Web en Manjaro, instala las herramientas esenciales en un solo comando:



```shell
sudo pacman -S nodejs npm git yarn
```



Este comando instala Node.js y npm para ejecutar y gestionar tus proyectos JavaScript y TypeScript, Git para la gestión de versiones y Yarn como gestor de paquetes alternativo.



### Instalación de una Bitcoin Wallet



Para gestionar tus bitcoins en Manjaro, puedes instalar **Electrum**, un Wallet ligero y seguro:



```shell
sudo pacman -S electrum  # Install Electrum
```



Electrum te permite **recibir y enviar bitcoins** con facilidad, a la vez que te ofrece funciones avanzadas como la gestión de múltiples Wallet y la protección de passphrase. Para obtener una guía completa sobre el uso de Electrum, consulte nuestro tutorial dedicado que explica cómo crear un Wallet, asegurar sus fondos y realizar transacciones.

## Proteja su sistema Manjaro



La seguridad es un aspecto crucial de cualquier instalación Linux. Es importante que protejas la confidencialidad e integridad de tus datos.



### Configuración del cortafuegos



Manjaro incluye UFW (*Uncomplicated Firewall*), un programa para gestionar cortafuegos con filtro de red, pero tienes que activarlo manualmente:



```bash
# Installation if not present
sudo pacman -S ufw

# Firewall activation
sudo systemctl enable ufw.enable

sudo ufw enable

# Allow SSH connections (optional)
sudo ufw allow ssh

# Check the status
sudo ufw status verbose
```



![verbose](assets/fr/15.webp)



### Gestión de los permisos de los usuarios



1. **Crear un usuario sin privilegios**



```shell
sudo useradd -m username
sudo passwd username
```



2. **Configuración del archivo *Sudoers***



```shell
sudo EDITOR=nano visudo
```
