Arch Linux är ett populärt operativsystem inom många områden tack vare sin robusthet och stabilitet. Det kan dock vara svårt för nybörjare att få grepp om det. Det är just för att lösa detta problem som **Manjaro** skapades: att erbjuda kraften i Arch Linux, men med en enklare och mer tillgänglig upplevelse, baserad på en intuitiv, lättlärd Interface.



## Komma igång med Manjaro



En av Manjaros största tillgångar är dess **enkla och effektiva uppdateringssystem**. Du behöver inte hantera dem manuellt: Manjaro tar hand om dem åt dig! En ikon i meddelandefältet (placeringen varierar beroende på utgåva) varnar dig när en uppdatering är tillgänglig. Allt du behöver göra är att följa instruktionerna, och processen är snabb och enkel.



Manjaro erbjuder också en **stor katalog av applikationer**. Eftersom Manjaro är baserat på Arch Linux har det direkt tillgång till dess officiella arkiv, som innehåller en mängd olika programvaror, inklusive proprietära applikationer. Manjaro försenar vissa Arch Linux-uppdateringar något för ytterligare testning; detta förbättrar stabiliteten på bekostnad av en liten fördröjning av nya utgåvor. Och om det här valet inte räcker för dig kan du också komma åt **AUR (*Arch User Repository*)**, ett stort bibliotek som hanteras av samhället. Om ett program inte finns i de officiella arkiven är chansen stor att det finns tillgängligt i AUR.



En annan fördel med Manjaro är att det är **mycket mindre resurskrävande** än system som Windows eller macOS. Det förbrukar mindre RAM-minne och datorkraft, vilket gör det till ett perfekt val för äldre eller mindre kraftfulla datorer: din maskin blir smidigare och snabbare och återfår en andra ungdom.



Utöver allt detta är Manjaro **helt gratis**. Till skillnad från Windows eller macOS behöver du inte betala något för att installera det och få ut det mesta av det. Slutligen erbjuder det **förbättrad säkerhet** tack vare regelbundna, snabba uppdateringar, vilket begränsar exponeringen för sårbarheter. Det aktiva communityt ser också till att eventuella problem snabbt åtgärdas och att effektiva lösningar föreslås.



## Upptäck Manjaro OS



Innan du börjar installera Manjaro är det viktigt att veta att den här distributionen finns i flera utgåvor. Var och en av dessa utgåvor erbjuder en unik skrivbordsmiljö som påverkar både ditt arbetsflöde och förbrukningen av systemresurser. Det finns tre huvudsakliga officiella utgåvor av Manjaro.



![0_01](assets/fr/01.webp)





- Utgåvan **KDE Plasma** är den mest anpassningsbara. Om du letar efter ett system som är både visuellt elegant och högpresterande är KDE Plasma ett utmärkt val. Denna stabila skrivbordsmiljö är idealisk för användare som vill ha total kontroll och en skräddarsydd upplevelse.





- För maskiner med mer begränsade resurser är **Xfce**-utgåvan den perfekta lösningen. Dess Interface är lätt och intuitivt och garanterar smidig användning även på äldre datorer. Dessutom påminner dess layout om Windows, vilket gör övergången till Linux enklare för nya användare.





- Slutligen erbjuder **GNOME**-utgåvan ett helt annat tillvägagångssätt. Dess strömlinjeformade design betonar produktivitet och organisation genom virtuella arbetsytor. Detta aktivitetsfokuserade arbetsflöde är särskilt tilltalande för användare som redan är bekanta med Linux och som vill ha en minimalistisk och mycket effektiv miljö.



### Andra Manjaro-utgåvor



![0_02](assets/fr/02.webp)



Förutom de officiella utgåvorna erbjuder Manjaro-gemenskapen även andra versioner. Dessa alternativa utgåvor är utformade för att möta specifika behov och erbjuder en mängd olika skrivbordsmiljöer.



Utgåvan **Cinnamon** är ett utmärkt val om du precis har börjat och letar efter en bekant Interface. Den har utformats för att vara enkel att använda, samtidigt som den behåller de klassiska konventionerna för traditionella kontorsmiljöer.

Dessa miljöer är idealiska för dem som vill ha total kontroll över sitt system och som sätter effektivitet framför allt annat.



## Tekniska krav



För att Manjaro ska fungera optimalt måste din dator uppfylla vissa minimikrav. Vi rekommenderar att du har minst :





- En 64-bitars (x86_64)** processor
- 4 GB RAM rekommenderas (minst 2 GB)** (se nedan)
- 30 GB diskutrymme (mer om du skapar en särskild `/home`-partition)**



## Installation av Manjaro



För att ladda ner, gå till [den officiella Manjaro-webbplatsen] (https://manjaro.org/) och välj den utgåva som bäst passar dina behov. När du har hämtat filen måste du skapa ett startbart USB-minne med Manjaros ISO-bild.



Gå sedan till webbplatsen för [Rufus] programvara (https://rufus.ie/fr/) och ladda ner den. Kör programmet, anslut din USB-nyckel, välj Manjaro ISO-bilden och börja blinka. Vänta tills processen är klar innan du tar ut nyckeln. Du kan sedan starta om datorn.



![0_03](assets/fr/03.webp)



För att installera Manjaro på din dator måste du först stänga av den helt. Koppla sedan in USB-nyckeln, slå på datorn igen och öppna startmenyn eller UEFI/BIOS-firmware genom att trycka på **F2, F10, F12, Escape** eller **Delete** (beroende på tillverkare).



Välj sedan USB-nyckeln som startenhet för att starta OS-installationsprocessen.



### Startskärm



Första gången du startar Manjaro från USB-minnet kommer du direkt till installationsmenyn. Innan du startar installationen kan du ändra tangentbordslayouten eller systemspråket.



Välj sedan alternativet **Starta med drivrutiner med öppen källkod** för att starta Manjaro-installationen. Dessa drivrutiner med öppen källkod är kompatibla med den mesta hårdvaran och räcker i de flesta fall. Om du till exempel har ett NVIDIA-grafikkort eller behöver högre grafikprestanda kan du välja **Starta med proprietära drivrutiner**, som använder proprietära drivrutiner.



![0_04](assets/fr/04.webp)



Systemet kommer att starta i **standard live-läge**. Detta gör att du kan testa Manjaros funktionalitet för att se om det passar dina behov innan du installerar det permanent. När du är klar klickar du på alternativet **Installera Manjaro Linux**.



Välj önskat språk för din installation och klicka sedan på **Nästa**.



![0_06](assets/fr/06.webp)



Välj sedan din plats för att ställa in rätt tidszon. I det här skedet kan du också ändra språk och datumformat.



![0_07](assets/fr/07.webp)



Välj tangentbordslayout. Ett testfält finns tillgängligt för att kontrollera att de tangenter som skrivs motsvarar de förväntade tecknen.



![0_08](assets/fr/08.webp)



### Diskpartitionering


Du har två alternativ för partitionering av din disk.





- Det första, och enklaste, är att radera hela disken och installera Manjaro på den.
- Den andra tillåter **manuell partitionering**.



![0_09](assets/fr/09.webp)



För en ren installation rekommenderar vi att du skapar minst tre partitioner:





- En första partition på **516 MB** (primär) för **boot**.
- En andra **2 GB** (logisk) partition för **swap**.
- En tredje partition för dina **personuppgifter**.



Om du vill installera ett annat system parallellt måste du dela upp den sista partitionen och endast tilldela en del till Manjaro (minst **15 GB** för att garantera att systemet fungerar korrekt).

## Första stegen med Manjaro



Att installera Manjaro är bara det första steget. För att du ska få ut mesta möjliga av ditt system finns det några saker som är bra att veta.



### Uppdatera systemet



Manjaro förenklar uppdateringar avsevärt. För att uppdatera dina paket :



```shell
sudo pacman -Syu
```



Detta kommando hämtar och installerar de senaste tillgängliga versionerna. Vi rekommenderar att du kör det regelbundet för att hålla ditt system **säkert och stabilt**.



### Konfigurera en utvecklingsmiljö



Om du vill utveckla webbapplikationer på Manjaro installerar du de viktigaste verktygen med ett enda kommando:



```shell
sudo pacman -S nodejs npm git yarn
```



Detta kommando installerar Node.js och npm för att köra och hantera dina JavaScript- och TypeScript-projekt, Git för versionshantering och Yarn som en alternativ pakethanterare.



### Installera en Bitcoin Wallet



För att hantera dina bitcoins på Manjaro kan du installera **Electrum**, en lättviktig och säker Wallet :



```shell
sudo pacman -S electrum  # Install Electrum
```



Med Electrum kan du enkelt **ta emot och skicka bitcoins**, samtidigt som du erbjuder avancerade funktioner som multipel Wallet-hantering och passphrase-skydd. För en komplett guide till hur du använder Electrum, kolla in vår dedikerade handledning som förklarar hur du skapar en Wallet, säkrar dina medel och utför transaktioner.

## Säkra ditt Manjaro-system



Säkerhet är en viktig aspekt av alla Linux-installationer. Det är viktigt för dig att skydda sekretessen och integriteten för dina data.



### Konfiguration av brandvägg



Manjaro innehåller UFW (*Uncomplicated Firewall*), ett program för hantering av brandväggar med nätverksfilter, men du måste aktivera det manuellt:



```bash
# Installation if not present
sudo pacman -S ufw

# Firewall activation
sudo systemctl enable ufw.enable

sudo ufw enable

# Allow SSH connections (optional)
sudo ufw allow ssh

# Check the status
sudo ufw status verbose
```



![verbose](assets/fr/15.webp)



### Hantera användarbehörigheter



1. **Skapa en icke-privilegierad användare**



```shell
sudo useradd -m username
sudo passwd username
```



2. **Konfiguration av Sudoers-fil**



```shell
sudo EDITOR=nano visudo
```
