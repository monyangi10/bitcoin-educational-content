Arch Linux adalah sistem operasi yang populer di berbagai bidang, berkat ketangguhan dan stabilitasnya. Namun, bisa jadi sulit bagi pengguna pemula untuk menguasainya. Justru untuk mengatasi masalah ini, **Manjaro** diciptakan: untuk menawarkan kekuatan Arch Linux, tetapi dengan pengalaman yang lebih sederhana dan lebih mudah diakses, berdasarkan Interface yang intuitif dan mudah dipelajari.



## Memulai dengan Manjaro



Salah satu aset terbesar Manjaro adalah sistem pembaruannya yang **sederhana dan efisien**. Tidak perlu mengelolanya secara manual: Manjaro akan mengurusnya untuk Anda! Sebuah ikon di area notifikasi (lokasi bervariasi menurut edisi) memberi tahu Anda ketika pembaruan tersedia. Anda hanya perlu mengikuti petunjuknya, dan prosesnya cepat dan mudah.



Manjaro juga menawarkan **katalog aplikasi yang luas**. Karena Manjaro berbasis Arch Linux, Manjaro mendapatkan keuntungan dari akses langsung ke repositori resminya, yang kaya akan berbagai perangkat lunak, termasuk aplikasi berpemilik. Manjaro sedikit menunda beberapa pembaruan Arch Linux untuk pengujian tambahan; ini meningkatkan stabilitas dengan mengorbankan sedikit penundaan dalam rilis baru. Dan jika pilihan ini tidak cukup untuk Anda, Anda juga dapat mengakses **AUR (*Arch User Repository*)**, sebuah perpustakaan besar yang dikelola oleh komunitas. Jika sebuah program tidak ada di repositori resmi, kemungkinan besar program tersebut tersedia di AUR.



Keuntungan lain dari Manjaro adalah **jauh lebih sedikit membutuhkan sumber daya** daripada sistem seperti Windows atau macOS. Manjaro mengkonsumsi lebih sedikit RAM dan daya komputasi, menjadikannya pilihan ideal untuk komputer yang lebih tua atau kurang bertenaga: komputer Anda mendapatkan kelancaran dan kecepatan, mendapatkan kembali masa mudanya.



Di atas semua itu, Manjaro **sepenuhnya gratis**. Tidak seperti Windows atau macOS, Anda tidak perlu membayar apa pun untuk menginstalnya dan memaksimalkannya. Terakhir, Manjaro menawarkan **keamanan yang ditingkatkan** berkat pembaruan yang teratur dan cepat, sehingga membatasi paparan terhadap kerentanan. Komunitasnya yang aktif juga memastikan bahwa masalah apa pun dengan cepat diperbaiki dan solusi yang efektif diusulkan.



## Temukan Manjaro OS



Sebelum Anda mulai menginstal Manjaro, penting untuk mengetahui bahwa distribusi ini tersedia dalam beberapa edisi. Masing-masing edisi ini menawarkan lingkungan desktop yang unik yang memengaruhi alur kerja dan konsumsi sumber daya sistem Anda. Ada tiga edisi resmi utama Manjaro.



![0_01](assets/fr/01.webp)





- Edisi **KDE Plasma** adalah edisi yang paling mudah dikustomisasi. Jika Anda mencari sistem yang elegan secara visual dan berkinerja tinggi, KDE Plasma adalah pilihan yang tepat. Lingkungan desktop yang stabil ini sangat ideal bagi pengguna yang menginginkan kontrol penuh dan pengalaman yang disesuaikan.





- Untuk mesin dengan sumber daya yang lebih terbatas, edisi **Xfce** adalah solusi yang ideal. Interface-nya ringan dan intuitif, menjamin pengoperasian yang lancar bahkan pada komputer yang lebih tua. Terlebih lagi, tata letaknya mengingatkan pada Windows, membuat transisi ke Linux lebih mudah bagi pengguna baru.





- Terakhir, edisi **GNOME** menawarkan pendekatan yang sama sekali berbeda. Desainnya yang ramping menekankan produktivitas dan organisasi melalui ruang kerja virtual. Alur kerja yang berfokus pada aktivitas ini sangat menarik bagi pengguna yang sudah terbiasa dengan Linux dan mencari lingkungan yang minimalis dan sangat efisien.



### Edisi Manjaro lainnya



![0_02](assets/fr/02.webp)



Selain edisi resmi, komunitas Manjaro juga menawarkan versi lain. Edisi alternatif ini dirancang untuk memenuhi kebutuhan khusus dan menawarkan berbagai lingkungan desktop.



Edisi **Cinnamon** adalah pilihan yang sangat baik jika Anda baru memulai dan mencari Interface yang sudah dikenal. Kamera ini telah didesain agar mudah digunakan, namun tetap mempertahankan konvensi klasik lingkungan kantor tradisional.



Untuk pengguna yang lebih mahir, ada edisi seperti **i3 Window Manager** atau **Sway**. Jauh lebih ringan dan lebih cepat, namun demikian, keduanya memerlukan penguasaan baris perintah yang baik untuk dapat dikonfigurasi dan dieksploitasi sepenuhnya. Lingkungan ini sangat ideal bagi mereka yang menginginkan kontrol penuh atas sistem mereka, dan yang menempatkan efisiensi di atas segalanya.



## Persyaratan teknis



Agar Manjaro dapat bekerja secara optimal, komputer Anda harus memenuhi beberapa persyaratan minimum. Kami menyarankan agar Anda memiliki setidaknya file :





- Prosesor 64-bit (x86_64)
- **RAM 4 GB direkomendasikan (minimum 2 GB)** (lihat di bawah)
- ruang disk sebesar 30 GB (lebih besar jika Anda membuat partisi khusus `/home`)



## Instalasi Manjaro



Untuk mengunduh, kunjungi [situs web resmi Manjaro](https://manjaro.org/) dan pilih edisi yang paling sesuai dengan kebutuhan Anda. Setelah mengunduh file, Anda harus membuat kunci USB yang dapat di-booting dengan image ISO Manjaro.



Kemudian kunjungi situs web perangkat lunak [Rufus](https://rufus.ie/fr/) dan unduh. Jalankan program, colokkan kunci USB Anda, pilih image ISO Manjaro dan mulai flashing. Tunggu hingga prosesnya selesai sebelum melepas kunci. Anda kemudian dapat memulai ulang komputer Anda.



![0_03](assets/fr/03.webp)



Untuk menginstal Manjaro di komputer Anda, pertama-tama matikan komputer sepenuhnya. Kemudian colokkan kunci USB, nyalakan kembali mesin dan akses menu boot atau firmware UEFI/BIOS dengan menekan **F2, F10, F12, Escape** atau **Delete** (tergantung pada produsen).



Kemudian pilih kunci USB sebagai perangkat boot untuk memulai proses instalasi OS.



### Layar pengaktifan



Pertama kali Anda menjalankan Manjaro dari kunci USB, Anda akan dibawa langsung ke menu instalasi. Sebelum meluncurkan instalasi, Anda dapat mengubah tata letak papan ketik atau bahasa sistem.



Kemudian pilih opsi **Boot dengan driver sumber terbuka** untuk memulai instalasi Manjaro. Driver sumber terbuka ini kompatibel dengan sebagian besar perangkat keras dan akan mencukupi dalam banyak kasus. Jika Anda memiliki kartu grafis NVIDIA, misalnya, atau membutuhkan kinerja grafis yang lebih tinggi, Anda dapat memilih **Boot with proprietary drivers**, yang menggunakan driver berpemilik.



![0_04](assets/fr/04.webp)



Sistem akan diluncurkan dalam **mode live default**. Ini memungkinkan Anda menguji fungsionalitas Manjaro untuk mengetahui apakah sesuai dengan kebutuhan Anda sebelum menginstalnya secara permanen. Setelah siap, klik opsi **Instal Manjaro Linux**.



Pilih bahasa yang diinginkan untuk penginstalan Anda, lalu klik **Next**.



![0_06](assets/fr/06.webp)



Kemudian pilih lokasi Anda untuk mengatur zona waktu yang benar. Pada tahap ini, Anda juga dapat mengubah bahasa dan format tanggal.



![0_07](assets/fr/07.webp)



Pilih tata letak keyboard. Bidang uji tersedia untuk memeriksa apakah tombol yang diketik sesuai dengan karakter yang diharapkan.



![0_08](assets/fr/08.webp)



### Partisi disk


Anda memiliki dua opsi untuk mempartisi disk Anda.





- Yang pertama, dan yang paling sederhana, adalah menghapus seluruh disk dan menginstal Manjaro di dalamnya.
- Yang kedua memungkinkan **pemisahan secara manual**.



![0_09](assets/fr/09.webp)



Untuk instalasi yang bersih, kami sarankan untuk membuat setidaknya tiga partisi:





- Partisi pertama sebesar **516 MB** (primer) untuk **boot**.
- Partisi **2 GB** (logis) kedua untuk **swap**.
- Partisi ketiga untuk **data pribadi** Anda.



Jika Anda ingin menginstal sistem lain secara paralel, Anda harus membagi partisi terakhir ini dan mengalokasikan hanya satu bagian untuk Manjaro (setidaknya **15 GB** untuk menjamin operasi sistem yang tepat).


### Membuat akun pengguna



Buat akun pengguna pada sistem dengan mengisi informasi yang diperlukan. Terakhir, tetapkan kata sandi administrator (**root**). Administrator ini adalah **pengguna super** dengan hak penuh pada sistem dan kemampuan untuk menjalankan perintah tingkat lanjut.



![0_10](assets/fr/10.webp)



### Pilih office suite yang tepat



Manjaro memungkinkan Anda memilih antara **FreeOffice** dan **LibreOffice**.





- **LibreOffice** lebih lengkap, dengan perangkat lunak yang lebih luas dan fitur-fitur canggih.
- **FreeOffice**, di sisi lain, lebih ringan dan hanya mencakup hal-hal yang penting: **TextMaker**, **PlanMaker**, dan **Presentasi**.

Layar ringkasan menunjukkan kepada Anda semua parameter yang sudah Anda tetapkan. Anda dapat kembali untuk membuat perubahan jika perlu, tanpa memengaruhi pengaturan lain yang sudah Anda buat.



Kemudian klik **Install**, dan konfirmasikan untuk memulai instalasi yang sesungguhnya.

Pada akhir proses, centang kotak **Restart now**, lalu klik **Done**. Sistem akan melakukan boot ulang dan **Manjaro akan siap digunakan**.

## Langkah pertama dengan Manjaro



Menginstal Manjaro hanyalah langkah pertama. Untuk mendapatkan hasil maksimal dari sistem Anda, berikut ini beberapa hal yang perlu diketahui.



### Memperbarui sistem



Manjaro sangat menyederhanakan pembaruan. Untuk memperbarui paket Anda:



```shell
sudo pacman -Syu
```



Perintah ini mengunduh dan menginstal versi terbaru yang tersedia. Kami sarankan Anda menjalankannya secara teratur untuk menjaga sistem Anda tetap **aman dan stabil**.



### Mengonfigurasi lingkungan pengembangan



Untuk mengembangkan aplikasi Web di Manjaro, instal alat bantu penting dalam satu perintah:



```shell
sudo pacman -S nodejs npm git yarn
```



Perintah ini menginstal Node.js dan npm untuk menjalankan dan mengelola proyek JavaScript dan TypeScript Anda, Git untuk manajemen versi, dan Yarn sebagai manajer paket alternatif.



### Memasang Bitcoin Wallet



Untuk mengelola bitcoin Anda di Manjaro, Anda bisa memasang **Electrum**, sebuah Wallet yang ringan dan aman:



```shell
sudo pacman -S electrum  # Install Electrum
```



Electrum memungkinkan Anda untuk **menerima dan mengirim bitcoin** dengan mudah, sambil menawarkan fitur-fitur canggih seperti manajemen beberapa Wallet dan perlindungan passphrase. Untuk panduan lengkap menggunakan Electrum, lihat tutorial khusus kami yang menjelaskan cara membuat Wallet, mengamankan dana, dan melakukan transaksi.

## Mengamankan sistem Manjaro Anda



Keamanan adalah aspek penting dari instalasi Linux. Penting bagi Anda untuk melindungi kerahasiaan dan integritas data Anda.



### Konfigurasi firewall



Manjaro menyertakan UFW (*Uncomplicated Firewall*), sebuah program untuk mengelola firewall filter jaringan, tetapi Anda harus mengaktifkannya secara manual:



```bash
# Installation if not present
sudo pacman -S ufw

# Firewall activation
sudo systemctl enable ufw.enable

sudo ufw enable

# Allow SSH connections (optional)
sudo ufw allow ssh

# Check the status
sudo ufw status verbose
```
