Arch Linux ni ubuhinga buzwi cane mu bikorwa vyinshi, kubera ubukomezi bwayo n’ugushikama kwayo. Ariko rero, birashobora kugora abakoresha bashasha kumenya neza. Ni kuri Address nyene iki kibazo **Manjaro** yaremewe: gutanga ububasha bwa Arch Linux, ariko n'ubumenyi bworoshe, bushikira, bushingiye ku buryo bworoshe, bworoshe kwiga Interface.



## Gutangura na Manjaro



Kimwe mu bintu bikomeye vya Manjaro ni **uburyo bwo guhindura amakuru bworoshe kandi bukora neza**. Ntibikenewe ko ubicungera n’amaboko: Manjaro aravyitaho! Igishushanyo kiri mu karere k’imenyekanisha (ahantu hatandukanye bivanye n’igitabu) kirakumenyesha igihe hariho ihinduka rishasha. Ico ubwirizwa gukora n’ugukurikiza amabwirizwa, kandi iyo nzira iranyaruka kandi nta ngorane.



Manjaro kandi itanga **urutonde runini rw'ibikoresho**. Kubera ko Manjaro ishingiye kuri Arch Linux, iraheza ikaronka inyungu mu gushika ku bubiko bwayo buzwi, bufise ubuhinga butandukanye, harimwo n’ibikoresho vy’ubuhinga. Manjaro iracerezwa gatoyi n’ibindi bishasha vya Arch Linux kugira ngo hakorwe ibindi bigeragezo; ivyo bituma habaho ugushikama ku giciro c’ugucerezwa gatoyi mu bishasha bisohoka. Kandi nimba iyo mahitamwo idahagije kuri wewe, urashobora kandi gushika kuri **AUR (*Ububiko bw'abakoresha bwa Arch*)**, ububiko bunini bw'ibitabu bucungiwe n'abanyagihugu. Iyo porogarama itaboneka mu bubiko buzwi, amahirwe ni menshi ko iboneka muri AUR.



Ikindi ciza ca Manjaro ni uko **idakoresha cane amafaranga** kuruta ama systems nka Windows canke macOS. Ikoresha RAM nke n’ububasha bwo gukoresha mudasobwa, bikaba biyigira ihitamwo ryiza kuri mudasobwa za kera canke zidakomeye cane: imashini yawe iraronka ubushobozi bwo gukora neza no kwihuta, igasubira kuronka ubuto bwa kabiri.



Hejuru y’ivyo vyose, Manjaro **ni umwidegemvyo wose**. Udakunze Windows canke macOS, ntaco ubwirizwa kuriha kugira ngo uyishiremwo maze uyironkemwo vyinshi. Ubwa nyuma, itanga **umutekano mwiza** bivuye ku guhindura ubudasiba, vyihuta, bikagabanya ukuntu umuntu ashobora gushikirwa n’ingorane. Umuryango wayo ukora kandi urabona ko ingorane iyo ari yo yose ishobora gukosorwa ningoga kandi hakaboneka inyishu nziza.



## Kubona Manjaro OS



Imbere y'uko utangura gushiramwo Manjaro, birahambaye kumenya ko iyo distribution iboneka mu nteguro nyinshi. Imwe muri izo nteguro itanga ibidukikije vy’ibiro bidasanzwe bigira ico bikoze ku bijanye n’ingene ukora be n’ingene ukoresha ibikoresho vya sisitemu. Hariho integuro zitatu nyamukuru z’ikinyamakuru Manjaro zizwi ku mugaragaro.



![0_01](assets/fr/01.webp)





- Igitabo ca **KDE Plasma** ni co gishobora guhindurwa cane. Niba uriko urarondera uburyo buboneka neza kandi bukora neza cane, KDE Plasma ni ihitamwo ryiza cane. Iyi nzira y’ibiro ihamye ni nziza ku bakoresha bashaka ubugenzuzi bwose n’ubumenyi bubereye.





- Ku mashini zifise uburyo buke, **Xfce** ni umuti mwiza. Interface yayo iraremereye kandi irakoreshwa neza, igatuma ikora neza no kuri orodinateri za kera. Ikindi, uburyo bwayo bwibutsa Windows, bituma guhindukira ukaja kuri Linux vyoroha ku bashasha bakoresha.





- Ubwa nyuma, igitabu ca **GNOME** kiratanga uburyo butandukanye rwose. Igishushanyo caco giteye neza gishimika ku gutanga umusaruro no ku gutunganya ibintu biciye ku bibanza vy’akazi vy’ubuhinga bwa none. Iyi nzira y’ibikorwa yibanze ku bikorwa iraryohera cane cane abakoresha bamaze kumenya Linux kandi barondera ibidukikije bifise ubushobozi buke kandi bikora neza cane.



### Ibindi bitabu vy'ikimanjaro

Uretse izo nteguro y’ibitabu yemewe, umuryango w’Abamanjaro uratanga n’izindi nteguro. Izo nteguro zindi zigenewe gushitsa ivyo umuntu akeneye vyihariye kandi zitanga ibidukikije bitandukanye vyo gukoresha ku biro.



Igitabo ca **Cinnamon** ni amahitamwo meza cane nimba uriko uratangura kandi uriko urarondera Interface umenyereye. Yahinguwe kugira ngo yorohe gukoresha, mu gihe igumya amategeko ya kera y’ibibanza vy’ibiro vya kera.



Ku bakoresha bateye imbere cane, hariho insiguro nka **i3 Umuyobozi w'Idirisha** canke **Sway**. Biroroshe cane kandi vyihuta, naho biri ukwo bisaba ubuhinga bwiza bwo gukoresha umurongo w’amabwirizwa kugira ngo bishobore gutunganirizwa neza no gukoreshwa. Ivyo bidukikije ni vyiza ku bashaka kugenzura bimwe bishitse ubuhinga bwabo, kandi bashira ubushobozi hejuru y’ibindi vyose.



## Ibisabwa mu vy'ubuhinga



Kugira ngo Manjaro ikore neza, mudasobwa yawe itegerezwa kuba ifise ibisabwa bikeyi. Turagusavye ko wogira nibura :





- Igikoresho c'ibice 64 (x86_64)
- 4 GB RAM irasabwa (nibura 2 GB)** (raba munsi)
- 30 GB y'umwanya kuri disiki (ibindi iyo uremye umugabane wihariye `/home`)



## Gushiramwo Manjaro



Kugira ngo uyironke, genda kuri [urubuga rwemewe rwa Manjaro](https://manjaro.org/) maze uhitemwo igitabu kijanye n’ivyo ukeneye. Uhejeje gukuraho dosiye, uzokenera gukora urufunguzo rwa USB rushobora gufungurwa rufise ishusho ya Manjaro ISO.



Hanyuma ugende ku rubuga rwa porogarama ya [Rufus] (https://rufus.ie/fr/) maze uyikuremwo. Gukoresha porogarama, ushiremwo urufunguzo rwawe rwa USB, uhitemwo ishusho ya Manjaro ISO maze utangure guca ibibatsi. Rindira ko igikorwa kirangira imbere y’uko ukuraho urufunguzo. Ushobora rero gusubira gufungura mudasobwa yawe.



![0_03](assets/fr/03.webp)



Kugira ngo ushire Manjaro kuri mudasobwa yawe, banza uyizimye burundu. Hanyuma ushiremwo urufunguzo rwa USB, usubire gufungura imashini maze winjire muri menu ya boot canke muri firmware ya UEFI/BIOS ukanda **F2, F10, F12, Escape** canke **Delete** (bivanye n’uwayikoze).



Hanyuma uhitemwo urufunguzo rwa USB nk’igikoresho co gutangura kugira ngo utangure igikorwa co gushiramwo OS.



### Igishushanyo co gutangura



Igihe ca mbere ufunguye Manjaro ukoresheje urufunguzo rwa USB, uzoca uja ku rutonde rwo gushiramwo. Imbere yo gutangura gushiramwo, urashobora guhindura uko klavye iteye canke ururimi rwa sisitemu.



Hanyuma uhitemwo **Boot n'abashoferi b'inkomoko yuguruye** kugira ngo utangure gushiramwo Manjaro. Ivyo bikoresho vy’ubuhinga bwa none bihuye n’ibikoresho vyinshi kandi bizoba bihagije mu bihe vyinshi. Niba ufise ikarita y’ibishushanyo ya NVIDIA, nk’akarorero, canke usaba ubushobozi bw’ibishushanyo bwinshi, ushobora guhitamwo **Boot n’abashoferi b’abanyagihugu**, ikoresha abashoferi b’abanyagihugu.



![0_04](assets/fr/04.webp)



Sisitemu izotangura mu **uburyo busanzwe**. Ivyo bigufasha kugerageza ingene Manjaro ikora kugira ngo urabe nimba ihuye n’ivyo ukeneye imbere y’uko uyishiramwo ubudasiba. Iyo umaze kwitegura, ukande kuri **Shiraho Manjaro Linux**.



Hitamwo ururimi wipfuza kugira ngo ushiremwo, hanyuma ukande **Ibikurikira**.



![0_06](assets/fr/06.webp)



Hanyuma uhitemwo aho uri kugira ngo ushireho isaha ibereye. Kuri iyi ntambwe, urashobora kandi guhindura ururimi n’itariki.



![0_07](assets/fr/07.webp)



Hitamwo uko klavye iteye. Hariho umwanya wo kugerageza kugira ngo ugenzure ko imfunguruzo zanditswe zihuye n'inyuguti zitezwe.



![0_08](assets/fr/08.webp)



### Gucapura disiki


Ufise uburyo bubiri bwo gucapura disiki yawe.





- Ica mbere, kandi coroshe, ni ugukuraho disiki yose hanyuma ukayishirako Manjaro.
- Ica kabiri kiremera **ugucapura n'amaboko**.



![0_09](assets/fr/09.webp)



Kugira ngo ukoreshe neza, turagusavye gukora nibura ibice bitatu:





- Igice ca mbere ca **516 MB** (ica mbere) ku **boot**.
- Igipande ca kabiri **2 GB** (gihuje n'ubwenge) ku **swap**.
- Igice ca gatatu c'amakuru yawe bwite**.



Niba wipfuza gushiramwo iyindi systeme mu buryo bumwe, ukeneye kugabanya iyo partition ya nyuma maze ugaha Manjaro igice kimwe gusa (nibura **15 GB** kugira ngo systeme ikore neza).

## Intambwe za mbere na Manjaro



Gushiramwo Manjaro ni intambwe ya mbere gusa. Kugira ngo ubone inyungu nyinshi muri sisitemu yawe, ng’ibi ibintu bikeyi vy’ingirakamaro womenya.



### Kuvugurura sisitemu



Manjaro yorosha cane ivyo guhindura. Kugira ngo uhindure amapaki yawe :



```shell
sudo pacman -Syu
```



Iri tegeko rikuraho rikashiramwo verisiyo nshasha zihari. Turagusavye ko uyikoresha ubudasiba kugira ngo system yawe igume **itekanye kandi ihagaze neza**.



### Gutunganya ibidukikije vy'iterambere



Kugira ngo ukore porogarama z’urubuga kuri Manjaro, shiramwo ibikoresho vy’ingenzi mw’itegeko rimwe:



```shell
sudo pacman -S nodejs npm git yarn
```



Iri tegeko rishiramwo Node.js na npm kugira ngo ukoreshe kandi ucunge imigambi yawe ya JavaScript na TypeScript, Git ku gucunga verisiyo, na Yarn nk'umucungerezi w'amapaki.



### Gushiramwo Bitcoin Wallet.



Kugira ngo ushobore gucunga amafaranga yawe kuri Manjaro, ushobora gushiramwo **Electrum**, Wallet yoroshe kandi itekanye :



```shell
sudo pacman -S electrum  # Install Electrum
```



Electrum iragufasha **kwakira no kohereza bitcoins** mu buryo bworoshe, mu gihe itanga ibintu biteye imbere nk’uburongozi bwinshi bwa Wallet n’uburinzi bwa passphrase. Kugira ngo ubone ubuyobozi bushitse bwo gukoresha Electrum, raba inyigisho yacu yihariye isigura ingene wokora Wallet, ukingira amahera yawe no gukora ibikorwa.

## Gukingira sisitemu yawe ya Manjaro



Umutekano ni ikintu gihambaye cane mu gushiramwo Linux iyo ari yo yose. Ni ngombwa ko ukingira ibanga n’ubutungane bw’amakuru yawe.



### Imiterere y'uruhome rw'umuriro



Manjaro irimwo UFW (*Uncomplicated Firewall*), porogarama yo gucunga ivyuma vy'umuriro vy'iyunguruzo ry'urubuga, ariko utegerezwa kuyikoresha n'amaboko:



```bash
# Installation if not present
sudo pacman -S ufw

# Firewall activation
sudo systemctl enable ufw.enable

sudo ufw enable

# Allow SSH connections (optional)
sudo ufw allow ssh

# Check the status
sudo ufw status verbose
```



![verbose](assets/fr/15.webp)



### Gucungera uruhusha rw'ukoresha



1. **Rema umukoresha atagira uburenganzira**



```shell
sudo useradd -m username
sudo passwd username
```



2. **Itunganywa rya dosiye ya Sudoers**



```shell
sudo EDITOR=nano visudo
```
