Arch Linux ist dank seiner Robustheit und Stabilität in vielen Bereichen ein beliebtes Betriebssystem. Allerdings kann es für unerfahrene Benutzer schwierig sein, damit zurechtzukommen. Genau um dieses Problem zu lösen, wurde **Manjaro** entwickelt: Es bietet die Leistung von Arch Linux, aber mit einer einfacheren, leichter zugänglichen Erfahrung, basierend auf einem intuitiven, leicht zu erlernenden Interface.



## Erste Schritte mit Manjaro



Einer der größten Vorteile von Manjaro ist sein **einfaches und effizientes Update-System**. Sie müssen diese nicht manuell verwalten: Manjaro kümmert sich um sie für Sie! Ein Symbol im Infobereich (die Position variiert je nach Edition) informiert Sie, wenn ein Update verfügbar ist. Alles, was Sie tun müssen, ist den Anweisungen zu folgen, und der Prozess ist schnell und mühelos.



Manjaro bietet auch einen **großen Katalog von Anwendungen**. Da Manjaro auf Arch Linux basiert, profitiert es vom direkten Zugriff auf dessen offizielle Repositories, die eine Vielzahl von Software, einschließlich proprietärer Anwendungen, enthalten. Manjaro verzögert einige Arch-Linux-Updates leicht, um zusätzliche Tests durchzuführen; dies verbessert die Stabilität auf Kosten einer leichten Verzögerung bei neuen Versionen. Und wenn Ihnen diese Auswahl nicht ausreicht, können Sie auch auf das **AUR (*Arch User Repository*)** zugreifen, eine riesige, von der Community verwaltete Bibliothek. Wenn ein Programm nicht in den offiziellen Repositories vorhanden ist, stehen die Chancen gut, dass es im AUR verfügbar ist.



Ein weiterer Vorteil von Manjaro ist, dass es **viel weniger ressourcenhungrig** ist als Systeme wie Windows oder macOS. Es verbraucht weniger Arbeitsspeicher und Rechenleistung, was es zu einer idealen Wahl für ältere oder weniger leistungsstarke Computer macht: Ihr Rechner gewinnt an Flüssigkeit und Geschwindigkeit und gewinnt eine zweite Jugend zurück.



Obendrein ist Manjaro **völlig kostenlos**. Im Gegensatz zu Windows oder macOS müssen Sie nichts bezahlen, um es zu installieren und das Beste daraus zu machen. Und schließlich bietet es dank regelmäßiger, schneller Updates eine **verbesserte Sicherheit**, die die Anfälligkeit für Schwachstellen einschränkt. Die aktive Community sorgt außerdem dafür, dass Probleme schnell behoben und wirksame Lösungen vorgeschlagen werden.



## Entdecken Sie Manjaro OS



Bevor Sie mit der Installation von Manjaro beginnen, ist es wichtig zu wissen, dass diese Distribution in verschiedenen Editionen erhältlich ist. Jede dieser Editionen bietet eine einzigartige Desktop-Umgebung, die sowohl Ihren Arbeitsablauf als auch den Verbrauch von Systemressourcen beeinflusst. Es gibt drei offizielle Haupteditionen von Manjaro.



![0_01](assets/fr/01.webp)





- Die **KDE Plasma** Edition ist die am besten anpassbare. Wenn Sie ein System suchen, das sowohl visuell elegant als auch leistungsstark ist, ist KDE Plasma eine ausgezeichnete Wahl. Diese stabile Desktop-Umgebung ist ideal für Benutzer, die volle Kontrolle und ein maßgeschneidertes Erlebnis wünschen.





- Für Rechner mit begrenzteren Ressourcen ist die **Xfce** Edition die ideale Lösung. Ihr Interface ist leichtgewichtig und intuitiv und garantiert einen reibungslosen Betrieb auch auf älteren Computern. Darüber hinaus erinnert das Layout an Windows, was neuen Benutzern den Einstieg in Linux erleichtert.





- Die **GNOME**-Edition schließlich bietet einen völlig anderen Ansatz. Ihr schlankes Design legt den Schwerpunkt auf Produktivität und Organisation durch virtuelle Arbeitsbereiche. Dieser auf Aktivitäten ausgerichtete Arbeitsablauf ist besonders für Benutzer interessant, die bereits mit Linux vertraut sind und eine minimalistische, hocheffiziente Umgebung suchen.



### Andere Manjaro-Editionen



![0_02](assets/fr/02.webp)



Zusätzlich zu den offiziellen Editionen bietet die Manjaro-Community auch andere Versionen an. Diese alternativen Editionen wurden entwickelt, um spezifische Bedürfnisse zu erfüllen und bieten eine Vielzahl von Desktop-Umgebungen.



Die **Cinnamon** Edition ist eine ausgezeichnete Wahl, wenn Sie gerade erst anfangen und ein vertrautes Interface suchen. Sie wurde so konzipiert, dass sie einfach zu bedienen ist und gleichzeitig die klassischen Konventionen traditioneller Büroumgebungen beibehält.



Für fortgeschrittene Benutzer gibt es Editionen wie **i3 Window Manager** oder **Sway**. Sie sind viel leichter und schneller, erfordern aber dennoch eine gute Beherrschung der Befehlszeile, um vollständig konfiguriert und genutzt werden zu können. Diese Umgebungen sind ideal für diejenigen, die die volle Kontrolle über ihr System haben wollen und denen Effizienz wichtiger ist als alles andere.



## Technische Anforderungen



Damit Manjaro optimal funktioniert, muss Ihr Computer einige Mindestanforderungen erfüllen. Wir empfehlen, dass Sie mindestens eine :





- Ein 64-Bit-Prozessor (x86_64)
- **4 GB RAM empfohlen (mindestens 2 GB)** (siehe unten)
- 30 GB Festplattenspeicher (mehr, wenn Sie eine eigene "Home"-Partition erstellen)



## Manjaro-Installation



Gehen Sie zum Herunterladen auf [die offizielle Manjaro-Website](https://manjaro.org/) und wählen Sie die Edition, die Ihren Anforderungen am besten entspricht. Sobald Sie die Datei heruntergeladen haben, müssen Sie einen bootfähigen USB-Stick mit dem Manjaro ISO-Image erstellen.



Gehen Sie dann auf die [Rufus]-Software-Website (https://rufus.ie/fr/) und laden Sie das Programm herunter. Starten Sie das Programm, schließen Sie Ihren USB-Stick an, wählen Sie das Manjaro-ISO-Image aus und beginnen Sie mit dem Flashen. Warten Sie, bis der Vorgang abgeschlossen ist, bevor Sie den Schlüssel abziehen. Anschließend können Sie Ihren Computer neu starten.



![0_03](assets/fr/03.webp)



Um Manjaro auf Ihrem Computer zu installieren, schalten Sie ihn zunächst vollständig aus. Stecken Sie dann den USB-Stick ein, schalten Sie den Rechner wieder ein und rufen Sie das Boot-Menü oder die UEFI/BIOS-Firmware auf, indem Sie **F2, F10, F12, Escape** oder **Delete** (je nach Hersteller) drücken.



Wählen Sie dann den USB-Stick als Boot-Gerät aus, um den Installationsprozess des Betriebssystems zu starten.



### Startbildschirm



Wenn Sie Manjaro zum ersten Mal vom USB-Stick aus starten, gelangen Sie direkt zum Installationsmenü. Bevor Sie die Installation starten, können Sie das Tastaturlayout oder die Systemsprache ändern.



Wählen Sie dann die Option **Boot mit Open-Source-Treibern**, um die Manjaro-Installation zu starten. Diese Open-Source-Treiber sind mit der meisten Hardware kompatibel und reichen in den meisten Fällen aus. Wenn Sie z. B. eine NVIDIA-Grafikkarte haben oder eine höhere Grafikleistung benötigen, können Sie **Boot mit proprietären Treibern** wählen, das proprietäre Treiber verwendet.



![0_04](assets/fr/04.webp)



Das System wird im **Standard-Live-Modus** gestartet. So können Sie die Funktionalität von Manjaro testen, um zu sehen, ob es Ihren Anforderungen entspricht, bevor Sie es dauerhaft installieren. Sobald Sie bereit sind, klicken Sie auf die Option **Manjaro Linux installieren**.



Wählen Sie die gewünschte Sprache für Ihre Installation aus, und klicken Sie dann auf **Weiter**.



![0_06](assets/fr/06.webp)



Wählen Sie dann Ihren Standort, um die richtige Zeitzone einzustellen. In diesem Stadium können Sie auch die Sprache und das Datumsformat ändern.



![0_07](assets/fr/07.webp)



Wählen Sie das Tastaturlayout. Ein Testfeld ist verfügbar, um zu prüfen, ob die eingegebenen Tasten den erwarteten Zeichen entsprechen.



![0_08](assets/fr/08.webp)



### Partitionierung der Festplatte


Sie haben zwei Möglichkeiten, Ihre Festplatte zu partitionieren.





- Die erste und einfachste Möglichkeit ist, die gesamte Festplatte zu löschen und Manjaro darauf zu installieren.
- Die zweite ermöglicht eine **manuelle Partitionierung**.



![0_09](assets/fr/09.webp)



Für eine saubere Installation empfehlen wir, mindestens drei Partitionen zu erstellen:





- Eine erste Partition von **516 MB** (primär) für den **Boot**.
- Eine zweite **2 GB** (logische) Partition für **Swap**.
- Eine dritte Partition für Ihre **persönlichen Daten**.



Wenn Sie parallel ein anderes System installieren möchten, müssen Sie diese letzte Partition aufteilen und nur einen Teil für Manjaro reservieren (mindestens **15 GB**, um den ordnungsgemäßen Betrieb des Systems zu gewährleisten).


### Erstellen eines Benutzerkontos



Erstellen Sie ein Benutzerkonto auf dem System, indem Sie die erforderlichen Informationen eingeben. Legen Sie schließlich das Passwort des Administrators (**root**) fest. Dieser Administrator ist ein **Superuser** mit vollen Rechten auf dem System und der Fähigkeit, erweiterte Befehle auszuführen.



![0_10](assets/fr/10.webp)



### Wählen Sie die richtige Office-Suite



Manjaro lässt Ihnen die Wahl zwischen **FreeOffice** und **LibreOffice**.





- **LibreOffice** ist umfassender und bietet eine größere Auswahl an Software und erweiterten Funktionen.
- **FreeOffice** hingegen ist leichter und enthält nur das Wesentliche: **TextMaker**, **PlanMaker** und **Presentations**.



![0_11](assets/fr/11.webp)



### Zusammenfassung der Konfiguration



Ein Übersichtsbildschirm zeigt Ihnen alle von Ihnen eingestellten Parameter an. Sie können bei Bedarf zurückgehen und Änderungen vornehmen, ohne dass sich dies auf andere, bereits vorgenommene Einstellungen auswirkt.



Klicken Sie dann auf **Installieren**, und bestätigen Sie, um die eigentliche Installation zu starten.



![0_12](assets/fr/12.webp)



![0_13](assets/fr/13.webp)



### Ende der Installation



Aktivieren Sie am Ende des Vorgangs das Kontrollkästchen **Neustart jetzt** und klicken Sie dann auf **Fertig**. Das System wird neu gestartet und **Manjaro ist jetzt einsatzbereit**.



![0_13](assets/fr/14.webp)



## Erste Schritte mit Manjaro



Die Installation von Manjaro ist nur der erste Schritt. Um Ihr System optimal nutzen zu können, sollten Sie einige nützliche Dinge wissen.



### Aktualisierung des Systems



Manjaro vereinfacht die Aktualisierung erheblich. Um Ihre Pakete zu aktualisieren:



```shell
sudo pacman -Syu
```



Mit diesem Befehl werden die neuesten verfügbaren Versionen heruntergeladen und installiert. Wir empfehlen Ihnen, ihn regelmäßig auszuführen, um Ihr System **sicher und stabil** zu halten.



### Konfigurieren einer Entwicklungsumgebung



Um Webanwendungen unter Manjaro zu entwickeln, installieren Sie die wichtigsten Tools mit einem einzigen Befehl:



```shell
sudo pacman -S nodejs npm git yarn
```



Dieser Befehl installiert Node.js und npm zur Ausführung und Verwaltung Ihrer JavaScript- und TypeScript-Projekte, Git für die Versionsverwaltung und Yarn als alternativen Paketmanager.



### Installieren eines Bitcoin Wallet



Um Ihre Bitcoins unter Manjaro zu verwalten, können Sie **Electrum** installieren, ein leichtes und sicheres Wallet :



```shell
sudo pacman -S electrum  # Install Electrum
```



Mit Electrum können Sie **Bitcoins** ganz einfach empfangen und versenden. Gleichzeitig bietet Electrum fortschrittliche Funktionen wie die Verwaltung mehrerer Wallet und den passphrase-Schutz. Eine vollständige Anleitung zur Verwendung von Electrum finden Sie in unserem speziellen Tutorial, in dem erklärt wird, wie Sie einen Wallet erstellen, Ihr Geld sichern und Transaktionen durchführen.

## Ihr Manjaro-System absichern



Sicherheit ist ein entscheidender Aspekt jeder Linux-Installation. Es ist wichtig, dass Sie die Vertraulichkeit und Integrität Ihrer Daten schützen.



### Firewall-Konfiguration



Manjaro enthält UFW (*Uncomplicated Firewall*), ein Programm zur Verwaltung von Netzwerkfilter-Firewalls, das Sie jedoch manuell aktivieren müssen:



```bash
# Installation if not present
sudo pacman -S ufw

# Firewall activation
sudo systemctl enable ufw.enable

sudo ufw enable

# Allow SSH connections (optional)
sudo ufw allow ssh

# Check the status
sudo ufw status verbose
```



![verbose](assets/fr/15.webp)



### Verwalten von Benutzerberechtigungen



1. **Anlegen eines nicht-privilegierten Benutzers**



```shell
sudo useradd -m username
sudo passwd username
```



2. **Konfiguration der Sudoers-Datei**



```shell
sudo EDITOR=nano visudo
```
