Arch Linux เป็นระบบปฏิบัติการที่ได้รับความนิยมในหลายสาขา เนื่องจากความทนทานและเสถียรภาพของมัน อย่างไรก็ตาม มันอาจจะยากสำหรับผู้ใช้มือใหม่ที่จะเข้าใจ นี่คือเหตุผลที่ **Manjaro** ถูกสร้างขึ้นมา: เพื่อมอบพลังของ Arch Linux แต่ด้วยประสบการณ์ที่ง่ายและเข้าถึงได้มากขึ้น โดยอิงจาก Interface ที่ใช้งานง่ายและเรียนรู้ได้ไม่ยาก



## เริ่มต้นใช้งาน Manjaro



หนึ่งในทรัพย์สินที่ยิ่งใหญ่ที่สุดของ Manjaro คือ **ระบบอัปเดตที่เรียบง่ายและมีประสิทธิภาพ** ไม่จำเป็นต้องจัดการด้วยตนเอง: Manjaro จะดูแลให้คุณ! ไอคอนในพื้นที่การแจ้งเตือน (ตำแหน่งจะแตกต่างกันไปตามรุ่น) จะแจ้งเตือนคุณเมื่อมีการอัปเดต สิ่งที่คุณต้องทำคือทำตามคำแนะนำ และกระบวนการนี้รวดเร็วและไม่ยุ่งยาก



Manjaro ยังมี **แคตตาล็อกแอปพลิเคชันที่หลากหลาย** เนื่องจาก Manjaro พัฒนามาจาก Arch Linux จึงได้รับประโยชน์จากการเข้าถึงที่เก็บข้อมูลอย่างเป็นทางการโดยตรง ซึ่งเต็มไปด้วยซอฟต์แวร์หลากหลายประเภท รวมถึงแอปพลิเคชันที่เป็นกรรมสิทธิ์ Manjaro มีการหน่วงเวลาการอัปเดตบางอย่างของ Arch Linux เล็กน้อยเพื่อการทดสอบเพิ่มเติม ซึ่งช่วยเพิ่มความเสถียรแม้จะต้องแลกกับการล่าช้าเล็กน้อยในการเปิดตัวใหม่ และหากตัวเลือกนี้ยังไม่เพียงพอสำหรับคุณ คุณยังสามารถเข้าถึง **AUR (*Arch User Repository*)** ซึ่งเป็นห้องสมุดขนาดใหญ่ที่จัดการโดยชุมชนได้อีกด้วย หากโปรแกรมใดไม่อยู่ในที่เก็บข้อมูลอย่างเป็นทางการ มีโอกาสสูงที่จะมีอยู่ใน AUR



ข้อดีอีกประการของ Manjaro คือมัน **ใช้ทรัพยากรน้อยกว่ามาก** เมื่อเทียบกับระบบอย่าง Windows หรือ macOS มันใช้ RAM และพลังการประมวลผลน้อยลง ทำให้เป็นตัวเลือกที่เหมาะสมสำหรับคอมพิวเตอร์ที่เก่าหรือมีประสิทธิภาพน้อยกว่า: เครื่องของคุณจะทำงานได้อย่างลื่นไหลและรวดเร็วขึ้น ราวกับได้ชีวิตใหม่อีกครั้ง



เหนือสิ่งอื่นใด Manjaro นั้น **ฟรีทั้งหมด** ไม่เหมือนกับ Windows หรือ macOS ที่คุณต้องจ่ายเงินเพื่อที่จะติดตั้งและใช้ประโยชน์สูงสุดจากมัน สุดท้ายนี้ มันยังมี **ความปลอดภัยที่เพิ่มขึ้น** ด้วยการอัปเดตที่รวดเร็วและสม่ำเสมอ ซึ่งช่วยลดความเสี่ยงจากช่องโหว่ต่างๆ ชุมชนที่มีความกระตือรือร้นยังช่วยให้มั่นใจได้ว่าปัญหาต่างๆ จะได้รับการแก้ไขอย่างรวดเร็วและมีการเสนอวิธีแก้ปัญหาที่มีประสิทธิภาพ



## ค้นพบ Manjaro OS



ก่อนที่คุณจะเริ่มติดตั้ง Manjaro สิ่งสำคัญคือต้องทราบว่าการแจกจ่ายนี้มีให้ในหลายรุ่น แต่ละรุ่นเหล่านี้มีสภาพแวดล้อมเดสก์ท็อปที่ไม่ซ้ำกันซึ่งมีผลต่อทั้งการทำงานของคุณและการใช้ทรัพยากรของระบบ Manjaro มีรุ่นหลักอย่างเป็นทางการสามรุ่น



![0_01](assets/fr/01.webp)





- **KDE Plasma** edition เป็นรุ่นที่ปรับแต่งได้มากที่สุด หากคุณกำลังมองหาระบบที่ทั้งสวยงามและมีประสิทธิภาพสูง KDE Plasma เป็นตัวเลือกที่ยอดเยี่ยม สภาพแวดล้อมเดสก์ท็อปที่เสถียรนี้เหมาะสำหรับผู้ใช้ที่ต้องการการควบคุมอย่างเต็มที่และประสบการณ์ที่ปรับแต่งได้





- สำหรับเครื่องที่มีทรัพยากรจำกัดมากขึ้น รุ่น **Xfce** เป็นทางออกที่เหมาะสมที่สุด Interface ของมันมีน้ำหนักเบาและใช้งานง่าย รับประกันการทำงานที่ราบรื่นแม้บนคอมพิวเตอร์รุ่นเก่า ยิ่งไปกว่านั้น การจัดวางของมันยังคล้ายกับ Windows ทำให้การเปลี่ยนไปใช้ Linux ง่ายขึ้นสำหรับผู้ใช้ใหม่





- ในที่สุด **GNOME** edition ก็เสนอแนวทางที่แตกต่างอย่างสิ้นเชิง การออกแบบที่เรียบง่ายของมันเน้นที่การเพิ่มประสิทธิภาพและการจัดระเบียบผ่านพื้นที่ทำงานเสมือนจริง กระบวนการทำงานที่เน้นกิจกรรมนี้มีความน่าสนใจเป็นพิเศษสำหรับผู้ใช้ที่คุ้นเคยกับ Linux อยู่แล้วและกำลังมองหาสภาพแวดล้อมที่เรียบง่ายและมีประสิทธิภาพสูง



### รุ่นอื่นๆ ของ Manjaro



![0_02](assets/fr/02.webp)



นอกเหนือจากฉบับทางการแล้ว ชุมชน Manjaro ยังมีฉบับอื่น ๆ ให้เลือกอีกด้วย ฉบับทางเลือกเหล่านี้ถูกออกแบบมาเพื่อตอบสนองความต้องการเฉพาะและมีสภาพแวดล้อมเดสก์ท็อปที่หลากหลาย



รุ่น **Cinnamon** เป็นตัวเลือกที่ยอดเยี่ยมถ้าคุณเพิ่งเริ่มต้นและกำลังมองหา Interface ที่คุ้นเคย มันถูกออกแบบมาให้ง่ายต่อการใช้งาน ในขณะที่ยังคงรักษาขนบธรรมเนียมแบบดั้งเดิมของสภาพแวดล้อมสำนักงานคลาสสิก



สำหรับผู้ใช้ขั้นสูง มีรุ่นต่าง ๆ เช่น **i3 Window Manager** หรือ **Sway** ซึ่งเบาและเร็วกว่า แต่ยังคงต้องการความชำนาญในการใช้คำสั่งบรรทัดเพื่อการตั้งค่าและใช้งานอย่างเต็มที่ สภาพแวดล้อมเหล่านี้เหมาะสำหรับผู้ที่ต้องการควบคุมระบบของตนอย่างเต็มที่ และให้ความสำคัญกับประสิทธิภาพเหนือสิ่งอื่นใด



## ข้อกำหนดทางเทคนิค



สำหรับ Manjaro เพื่อให้ทำงานได้อย่างเต็มประสิทธิภาพ คอมพิวเตอร์ของคุณต้องมีคุณสมบัติตรงตามข้อกำหนดขั้นต่ำบางประการ เราแนะนำให้คุณมีอย่างน้อย:





- โปรเซสเซอร์ 64 บิต (x86_64)
- แนะนำ RAM 4 GB (ขั้นต่ำ 2 GB)** (ดูด้านล่าง)
- พื้นที่ดิสก์ 30 GB (มากกว่านี้หากคุณสร้างพาร์ติชัน `/home` โดยเฉพาะ)



## การติดตั้ง Manjaro



ในการดาวน์โหลด ไปที่ [เว็บไซต์ทางการของ Manjaro](https://manjaro.org/) และเลือกเวอร์ชันที่เหมาะสมกับความต้องการของคุณ เมื่อคุณดาวน์โหลดไฟล์เสร็จแล้ว คุณจะต้องสร้าง USB ที่สามารถบูตได้ด้วยภาพ ISO ของ Manjaro



จากนั้นไปที่เว็บไซต์ซอฟต์แวร์ [Rufus] (https://rufus.ie/fr/) และดาวน์โหลดมัน เรียกใช้โปรแกรม เสียบกุญแจ USB ของคุณ เลือกภาพ ISO ของ Manjaro และเริ่มการแฟลช รอให้กระบวนการเสร็จสิ้นก่อนที่จะถอดกุญแจออก จากนั้นคุณสามารถรีสตาร์ทคอมพิวเตอร์ของคุณได้



![0_03](assets/fr/03.webp)



ในการติดตั้ง Manjaro บนคอมพิวเตอร์ของคุณ ให้ปิดเครื่องอย่างสมบูรณ์ก่อน จากนั้นเสียบ USB key เปิดเครื่องอีกครั้งและเข้าถึงเมนูบูตหรือเฟิร์มแวร์ UEFI/BIOS โดยกด **F2, F10, F12, Escape** หรือ **Delete** (ขึ้นอยู่กับผู้ผลิต)



จากนั้นเลือกคีย์ USB เป็นอุปกรณ์บูตเพื่อเริ่มกระบวนการติดตั้งระบบปฏิบัติการ



### หน้าจอเริ่มต้น



เมื่อคุณเปิดตัว Manjaro จาก USB key เป็นครั้งแรก คุณจะถูกนำไปยังเมนูการติดตั้งโดยตรง ก่อนที่จะเริ่มการติดตั้ง คุณสามารถเปลี่ยนรูปแบบแป้นพิมพ์หรือภาษาของระบบได้



จากนั้นเลือกตัวเลือก **Boot with open source drivers** เพื่อเริ่มการติดตั้ง Manjaro ไดรเวอร์โอเพ่นซอร์สเหล่านี้เข้ากันได้กับฮาร์ดแวร์ส่วนใหญ่และเพียงพอในกรณีส่วนใหญ่ หากคุณมีการ์ดกราฟิก NVIDIA ตัวอย่างเช่น หรือจำเป็นต้องใช้ประสิทธิภาพกราฟิกที่สูงขึ้น คุณสามารถเลือก **Boot with proprietary drivers** ซึ่งใช้ไดรเวอร์ที่เป็นกรรมสิทธิ์ได้



![0_04](assets/fr/04.webp)



ระบบจะเปิดตัวใน **โหมดสดเริ่มต้น** ซึ่งจะช่วยให้คุณทดสอบการทำงานของ Manjaro เพื่อดูว่าตรงกับความต้องการของคุณหรือไม่ก่อนที่จะติดตั้งอย่างถาวร เมื่อพร้อมแล้ว ให้คลิกที่ตัวเลือก **ติดตั้ง Manjaro Linux**



เลือกภาษาที่ต้องการสำหรับการติดตั้งของคุณ จากนั้นคลิก **ถัดไป**



![0_06](assets/fr/06.webp)



จากนั้นเลือกตำแหน่งของคุณเพื่อตั้งค่าเขตเวลาที่ถูกต้อง ในขั้นตอนนี้ คุณยังสามารถเปลี่ยนภาษาและรูปแบบวันที่ได้อีกด้วย



![0_07](assets/fr/07.webp)



เลือกเค้าโครงแป้นพิมพ์ มีช่องทดสอบให้ตรวจสอบว่าปุ่มที่พิมพ์ตรงกับตัวอักษรที่คาดหวังหรือไม่



![0_08](assets/fr/08.webp)



### การแบ่งพาร์ติชันดิสก์


คุณมีสองตัวเลือกสำหรับการแบ่งพาร์ติชันดิสก์ของคุณ





- วิธีแรกและง่ายที่สุดคือการลบข้อมูลทั้งดิสก์และติดตั้ง Manjaro ลงไป
- ตัวที่สองอนุญาตให้มีการ **แบ่งพาร์ติชันแบบแมนนวล**



![0_09](assets/fr/09.webp)



สำหรับการติดตั้งใหม่ที่สะอาด เราแนะนำให้สร้างพาร์ติชันอย่างน้อยสามพาร์ติชัน:





- พาร์ติชันแรกขนาด **516 MB** (หลัก) สำหรับ **บูต**.
- พาร์ติชันที่สอง **2 GB** (เชิงตรรกะ) สำหรับ **swap**.
- พาร์ติชันที่สามสำหรับ **ข้อมูลส่วนตัว** ของคุณ



หากคุณต้องการติดตั้งระบบอื่นควบคู่กันไป คุณจำเป็นต้องแบ่งพาร์ติชันสุดท้ายนี้และจัดสรรเพียงส่วนเดียวให้กับ Manjaro (อย่างน้อย **15 GB** เพื่อรับประกันการทำงานของระบบที่เหมาะสม)


### การสร้างบัญชีผู้ใช้



สร้างบัญชีผู้ใช้บนระบบโดยกรอกข้อมูลที่จำเป็น สุดท้าย ตั้งรหัสผ่านของผู้ดูแลระบบ (**root**) ผู้ดูแลระบบนี้เป็น **super-user** ที่มีสิทธิ์เต็มรูปแบบบนระบบและสามารถดำเนินการคำสั่งขั้นสูงได้



![0_10](assets/fr/10.webp)



### เลือกชุดโปรแกรมสำนักงานที่เหมาะสม



Manjaro ให้คุณเลือกได้ระหว่าง **FreeOffice** และ **LibreOffice**.





- LibreOffice** มีความสมบูรณ์มากกว่า ด้วยซอฟต์แวร์ที่หลากหลายและฟีเจอร์ขั้นสูง
- FreeOffice** ในทางกลับกัน มีน้ำหนักเบาและรวมเฉพาะสิ่งจำเป็นเท่านั้น: **TextMaker**, **PlanMaker** และ **Presentations**.



![0_11](assets/fr/11.webp)



### สรุปการกำหนดค่า



หน้าจอสรุปจะแสดงพารามิเตอร์ทั้งหมดที่คุณได้ตั้งค่าไว้ คุณสามารถย้อนกลับไปทำการเปลี่ยนแปลงได้หากจำเป็น โดยไม่ส่งผลกระทบต่อการตั้งค่าอื่น ๆ ที่คุณได้ทำไว้แล้ว



จากนั้นคลิกที่ **ติดตั้ง** และยืนยันเพื่อเริ่มการติดตั้งจริง



![0_12](assets/fr/12.webp)



![0_13](assets/fr/13.webp)



### สิ้นสุดการติดตั้ง



เมื่อสิ้นสุดกระบวนการ ให้ทำเครื่องหมายที่ช่อง **Restart now** จากนั้นคลิก **Done** ระบบจะรีบูตและ **Manjaro จะพร้อมใช้งาน**



![0_13](assets/fr/14.webp)



## ขั้นตอนแรกกับ Manjaro



การติดตั้ง Manjaro เป็นเพียงขั้นตอนแรกเท่านั้น เพื่อให้ได้ประโยชน์สูงสุดจากระบบของคุณ นี่คือสิ่งที่มีประโยชน์บางประการที่ควรรู้



### อัปเดตระบบ



Manjaro ช่วยให้อัปเดตง่ายขึ้นมาก ในการอัปเดตแพ็คเกจของคุณ :



```shell
sudo pacman -Syu
```



คำสั่งนี้จะดาวน์โหลดและติดตั้งเวอร์ชันล่าสุดที่มีอยู่ เราแนะนำให้คุณเรียกใช้เป็นประจำเพื่อให้ระบบของคุณ **ปลอดภัยและเสถียร**

ในการพัฒนาแอปพลิเคชันเว็บบน Manjaro ให้ติดตั้งเครื่องมือที่จำเป็นทั้งหมดด้วยคำสั่งเดียว:



```shell
sudo pacman -S nodejs npm git yarn
```



คำสั่งนี้ติดตั้ง Node.js และ npm เพื่อรันและจัดการโปรเจกต์ JavaScript และ TypeScript ของคุณ, Git สำหรับการจัดการเวอร์ชัน, และ Yarn เป็นตัวจัดการแพ็คเกจทางเลือก



### ติดตั้ง Bitcoin wallet



ในการจัดการบิตคอยน์ของคุณบน Manjaro คุณสามารถติดตั้ง **Electrum** ซึ่งเป็น wallet ที่มีน้ำหนักเบาและปลอดภัย :



```shell
sudo pacman -S electrum  # Install Electrum
```



Electrum ช่วยให้คุณ **รับและส่งบิตคอยน์** ได้อย่างง่ายดาย พร้อมทั้งมีฟีเจอร์ขั้นสูง เช่น การจัดการ wallet หลายรายการและการป้องกัน passphrase สำหรับคู่มือการใช้งาน Electrum อย่างครบถ้วน โปรดดูที่บทแนะนำเฉพาะของเราซึ่งอธิบายวิธีการสร้าง wallet ปกป้องเงินทุนของคุณ และดำเนินการทำธุรกรรม

## การรักษาความปลอดภัยระบบ Manjaro ของคุณ



ความปลอดภัยเป็นส่วนสำคัญของการติดตั้ง Linux ทุกครั้ง การปกป้องความลับและความสมบูรณ์ของข้อมูลของคุณเป็นสิ่งสำคัญ



### การกำหนดค่าไฟร์วอลล์



Manjaro รวมถึง UFW (*Uncomplicated Firewall*), โปรแกรมสำหรับจัดการไฟร์วอลล์กรองเครือข่าย แต่คุณต้องเปิดใช้งานด้วยตนเอง:



```bash
# Installation if not present
sudo pacman -S ufw

# Firewall activation
sudo systemctl enable ufw.enable

sudo ufw enable

# Allow SSH connections (optional)
sudo ufw allow ssh

# Check the status
sudo ufw status verbose
```
