ระบบปฏิบัติการแบบพกพาและลืมข้อมูลที่ปกป้องคุณจากการเฝ้าระวังและการเซ็นเซอร์


## ทำไมต้องมี USB key ที่ติดตั้ง Tails?


Tails (https://tails.boum.org/) เป็นวิธีที่ง่ายที่สุดในการมีคอมพิวเตอร์ที่ปลอดภัยไว้ใช้งานตลอดเวลา ซึ่งจะไม่ทิ้งร่องรอยใดๆ บนคอมพิวเตอร์ที่คุณใช้ร่วมด้วย


ในการใช้ Tails ให้ปิดคอมพิวเตอร์ที่คุณสามารถเข้าถึงได้ (ที่บ้านพ่อแม่, ที่บ้านเพื่อน, ในร้านอินเทอร์เน็ต...) และเริ่มต้นด้วย USB key ของ Tails แทนที่จะเป็น Windows, macOS, หรือ Linux


หลังจากนั้น คุณจะมีพื้นที่ทำงานและสภาพแวดล้อมการสื่อสารที่เป็นอิสระจากระบบปฏิบัติการทั่วไปและไม่เคยใช้ฮาร์ดไดรฟ์เลย


Tails ไม่เคยเขียนลงในฮาร์ดไดรฟ์และใช้เฉพาะ RAM ของคอมพิวเตอร์ในการทำงาน หน่วยความจำนี้จะถูกลบออกทั้งหมดเมื่อ Tails ถูกปิดลง ดังนั้นจึงลบล้างร่องรอยทั้งหมดที่เป็นไปได้


## กรณีการใช้งานคอนกรีตบางประการ


เพื่อให้คุณมีแนวคิดที่ชัดเจนเกี่ยวกับประโยชน์ของการมี USB key พร้อมกับ Tails อยู่เสมอ นี่คือรายการเล็ก ๆ ที่ไม่ครบถ้วนซึ่งรวบรวมโดยทีมงาน Agora256:



- เชื่อมต่อกับอินเทอร์เน็ตและ Tor ในวิธีที่ไม่ถูกเซ็นเซอร์และไม่ระบุตัวตนเพื่อท่องเว็บไซต์โดยไม่ทิ้งร่องรอย;
- เปิดไฟล์ PDF จากเว็บไซต์ที่น่าสงสัย;
- ทดสอบการสำรองข้อมูลคีย์ส่วนตัว Bitcoin ของคุณด้วย Electrum wallet;
- ใช้ชุดโปรแกรมสำนักงาน (LibreOffice) และทำงานบนคอมพิวเตอร์ที่ไม่ใช่ของคุณ;
- เริ่มต้นก้าวแรกของคุณในสภาพแวดล้อม Linux เพื่อเรียนรู้วิธีการออกจากโลกของ Microsoft และ Apple


## วิธีการเชื่อถือ Tails?


การใช้ซอฟต์แวร์มักมีองค์ประกอบของความไว้วางใจเสมอ แต่ไม่จำเป็นต้องเป็นการไว้วางใจแบบไม่ลืมหูลืมตา เครื่องมืออย่าง Tails ต้องพยายามมอบวิธีการให้ผู้ใช้สามารถไว้วางใจได้ สำหรับ Tails นี่หมายถึง:



- ซอร์สโค้ดสาธารณะ: https://gitlab.tails.boum.org/;
- โครงการที่อิงจากโครงการที่มีชื่อเสียง: Tor และ Debian;
- ดาวน์โหลดที่สามารถตรวจสอบและทำซ้ำได้;
- คำแนะนำโดยบุคคลและองค์กรต่างๆ


## คู่มือการติดตั้งและการใช้งาน


วัตถุประสงค์ของคู่มือติดตั้งนี้คือการแนะนำคุณผ่านแต่ละขั้นตอนของการติดตั้ง เราจะไม่อธิบายการดำเนินการที่ต้องทำมากกว่าคู่มือทางการ แต่เราจะชี้แนะแนวทางที่ถูกต้องพร้อมให้คำแนะนำและเคล็ดลับแก่คุณ


ด้วยเหตุผลด้านประสบการณ์จริง เคล็ดลับเหล่านี้จะเน้นไปที่แพลตฟอร์ม macOS และ Linux

🛠️

ก่อนเริ่มขั้นตอนนี้ โปรดตรวจสอบให้แน่ใจว่าคุณมี USB key ที่มีความเร็วในการอ่านขั้นต่ำ 150 MB/s และมีความจุอย่างน้อย 8 GB โดยควรเป็น USB 3.0


ข้อกำหนดเบื้องต้น:



- 1 คีย์ USB, เฉพาะสำหรับ Tails, มีความจุอย่างน้อย 8 GB
- คอมพิวเตอร์ที่เชื่อมต่อกับอินเทอร์เน็ตด้วย Linux, macOS, (หรือ Windows)
- เวลาว่างประมาณหนึ่งชั่วโมง ขึ้นอยู่กับความเร็วในการเชื่อมต่ออินเทอร์เน็ตของคุณ รวมถึง ½ ชั่วโมงสำหรับการติดตั้ง (ไฟล์ขนาด 1.3 GB สำหรับดาวน์โหลด)


## ขั้นตอนที่ 1: ดาวน์โหลด Tails จากคอมพิวเตอร์ของคุณ


![image](assets/1.webp)


🔗 **Official Tails section:** https://tails.boum.org/install/linux/index.fr.html#download


การดาวน์โหลดไฟล์ติดตั้งที่มีนามสกุล .img อาจใช้เวลาสักครู่ ขึ้นอยู่กับความเร็วในการดาวน์โหลดอินเทอร์เน็ตของคุณ ดังนั้นควรวางแผนล่วงหน้า ด้วยการเชื่อมต่อที่ทันสมัยและมีประสิทธิภาพ จะใช้เวลาน้อยกว่า 5 นาที


บันทึกไฟล์ในโฟลเดอร์ที่รู้จัก เช่น ดาวน์โหลด เนื่องจากจะจำเป็นสำหรับขั้นตอนถัดไป


## ขั้นตอนที่ 2: ตรวจสอบการดาวน์โหลดของคุณ


![image](assets/2.webp)


🔗 **Official Tails section:** https://tails.boum.org/install/linux/index.fr.html#verify


การตรวจสอบการดาวน์โหลดช่วยให้มั่นใจได้ว่ามันถูกออกโดยนักพัฒนา Tails และไม่ได้ถูกทำลายหรือถูกดักฟังระหว่างการดาวน์โหลด


เป็นไปได้ที่จะตรวจสอบด้วยตนเองว่าไฟล์ที่คุณเพิ่งดาวน์โหลดเป็นไฟล์ที่คาดหวังไว้โดยใช้ PGP แต่หากไม่มีความรู้ขั้นสูง การตรวจสอบนี้จะให้ระดับความปลอดภัยเท่ากับการตรวจสอบ JavaScript บนหน้าดาวน์โหลด ในขณะที่มีความซับซ้อนมากกว่าและมีแนวโน้มที่จะเกิดข้อผิดพลาดมากกว่า


ในการตรวจสอบไฟล์ ให้ใช้ปุ่ม "เลือกการดาวน์โหลดของคุณ..." ที่มีให้ในส่วนทางการ!


## ขั้นตอนที่ 3: ติดตั้ง Tails บน USB key ของคุณ


![image](assets/3.webp)


🔗 **ส่วนอย่างเป็นทางการ Tails:**



- Linux:** https://tails.boum.org/install/linux/index.fr.html#install
- macOS:** https://tails.boum.org/install/mac/index.fr.html#etcher และ https://tails.boum.org/install/mac/index.fr.html#install


ขั้นตอนการติดตั้ง Tails บน USB key ของคุณเป็นขั้นตอนที่ยากที่สุดในคู่มือทั้งหมด โดยเฉพาะอย่างยิ่งหากคุณไม่เคยทำมาก่อน จุดที่สำคัญที่สุดคือการเลือกขั้นตอนที่ถูกต้องในส่วนทางการสำหรับระบบปฏิบัติการของคุณ: Linux หรือ macOS


เมื่อเครื่องมือถูกติดตั้งและเตรียมพร้อมตามคำแนะนำแล้ว ไฟล์ที่มีนามสกุล .img สามารถคัดลอกไปยังคีย์ของคุณ (ลบข้อมูลที่มีอยู่ทั้งหมด) เพื่อทำให้มัน "บูตได้" อย่างอิสระ


โชคดี! แล้วพบกันที่ขั้นตอนที่ 4


## ขั้นตอนที่ 4: รีสตาร์ทบนคีย์ USB Tails ของคุณ


![image](assets/4.webp)


🔗 **Official Tails section:** https://tails.boum.org/install/linux/index.en.html#restart


ถึงเวลาที่จะเริ่มต้นคอมพิวเตอร์เครื่องหนึ่งของคุณโดยใช้ USB stick ใหม่ของคุณ เสียบมันเข้ากับพอร์ต USB หนึ่งในนั้นและรีสตาร์ท!


**หมายเหตุ💡: คอมพิวเตอร์ส่วนใหญ่จะไม่บูตจาก USB stick รุ่น Tails โดยอัตโนมัติ แต่คุณสามารถกดปุ่มเมนูบูตเพื่อแสดงรายการอุปกรณ์ที่สามารถบูตได้**

## ขั้นตอนที่ 5: ยินดีต้อนรับสู่ Tails!


![image](assets/6.webp)


🔗 **Official Tails section:** https://tails.boum.org/install/linux/index.en.html#tails


หนึ่งหรือสองนาทีหลังจากโปรแกรมบูตโหลดเดอร์และหน้าจอโหลด หน้าจอต้อนรับจะปรากฏขึ้น


![image](assets/7.webp)


ในหน้าจอต้อนรับ ให้เลือกภาษาของคุณและรูปแบบแป้นพิมพ์ในส่วนภาษาและภูมิภาค คลิกที่ เริ่ม Tails


![image](assets/8.webp)


หากคอมพิวเตอร์ของคุณไม่ได้เชื่อมต่อกับเครือข่ายของคุณ โปรดดูคำแนะนำอย่างเป็นทางการของ Tails เพื่อช่วยคุณเชื่อมต่อกับเครือข่ายของคุณโดยไม่ใช้ Wi-Fi (ส่วน "ทดสอบ Wi-Fi ของคุณ")


เมื่อเชื่อมต่อกับเครือข่ายท้องถิ่นแล้ว ตัวช่วยการเชื่อมต่อ Tor จะปรากฏขึ้นเพื่อช่วยคุณเชื่อมต่อกับเครือข่าย Tor


![image](assets/9.webp)


คุณสามารถเริ่มท่องเว็บแบบไม่ระบุตัวตน สำรวจตัวเลือกและซอฟต์แวร์ที่รวมอยู่ใน Tails สนุกกับตัวเองได้เลย คุณมีพื้นที่มากมายสำหรับข้อผิดพลาด เพราะไม่มีอะไรถูกแก้ไขใน USB stick... การรีสตาร์ทครั้งถัดไปของคุณจะลืมประสบการณ์ทั้งหมดของคุณ!


## ในคู่มืออนาคต...


เมื่อคุณได้ทดลองใช้งานกับ Tails USB stick ของคุณเองมากขึ้นแล้ว เราจะสำรวจหัวข้อขั้นสูงอื่นๆ ในบทความถัดไป เช่น:



- อัปเดตคีย์ด้วย **เวอร์ชันล่าสุดของ Tails**;
- กำหนดค่าและใช้ **ที่เก็บข้อมูลถาวร**;
- ติดตั้ง **ซอฟต์แวร์เพิ่มเติม**.


จนกว่าจะถึงตอนนั้น เช่นเคย หากคุณมีคำถามใด ๆ อย่าลังเลที่จะแบ่งปันกับชุมชน Agora256 เรากำลังเรียนรู้ร่วมกันเพื่อให้วันพรุ่งนี้ดีกว่าวันนี้!
