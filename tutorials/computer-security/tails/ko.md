## 테일즈가 설치된 USB 키가 필요한 이유는 무엇인가요?


Tails(https://tails.boum.org/)는 사용 중인 컴퓨터에 흔적을 남기지 않는 가장 쉬운 방법으로 항상 컴퓨터를 안전하게 보호할 수 있는 서비스입니다.


테일즈를 사용하려면 부모님 집, 친구 집, 인터넷 카페 등에서 액세스할 수 있는 컴퓨터를 끄고 Windows, macOS 또는 Linux 대신 테일즈 USB 키로 시작하세요.


그 후에는 일반적인 운영 체제와 독립적인 작업 공간과 커뮤니케이션 환경을 갖게 되며, Hard 드라이브를 사용하지 않습니다.


Tails는 Hard 드라이브에 절대 쓰지 않으며 컴퓨터의 RAM만 사용하여 작동합니다. 이 메모리는 Tails가 종료되면 완전히 지워지므로 가능한 모든 흔적이 제거됩니다.


## 몇 가지 구체적인 사용 사례


Tails와 함께 USB 키를 항상 가지고 있으면 어떤 이점이 있는지 구체적으로 알려드리기 위해 아고라256 팀이 정리한 전체 목록은 다음과 같습니다:



- 무수정 익명의 방법으로 인터넷에 연결하고 Tor에 접속하여 흔적을 남기지 않고 웹사이트를 브라우징하세요;
- 의심스러운 웹사이트에서 PDF를 엽니다;
- Electrum Wallet로 Bitcoin 개인 키 백업을 테스트하세요;
- 오피스 제품군(LibreOffice)을 사용하고 본인 소유가 아닌 컴퓨터에서 작업하세요;
- Linux 환경에서 첫발을 내딛고 Microsoft와 Apple의 세계에서 벗어나는 방법을 알아보세요.


## 테일즈를 신뢰하는 방법?


소프트웨어 사용에는 항상 신뢰라는 요소가 필요하지만, 맹목적일 필요는 없습니다. Tails와 같은 도구는 사용자에게 신뢰할 수 있는 수단을 제공하기 위해 노력해야 합니다. 테일즈의 경우, 이는 곧



- 공개 소스 코드: https://gitlab.tails.boum.org/;
- 평판이 좋은 프로젝트에 기반한 프로젝트: 토르와 데비안;
- 검증 가능하고 재현 가능한 다운로드;
- 다양한 개인 및 조직에서 추천합니다.


## 설치 및 사용 가이드


이 설치 가이드의 목적은 설치의 각 단계를 안내하는 것입니다. 공식 가이드보다 더 많은 조치를 설명하지는 않지만, 팁과 요령을 제공하면서 올바른 방향으로 안내해 드릴 것입니다.


실제 경험을 위해 이 팁은 macOS 및 Linux 플랫폼에 중점을 두고 설명합니다.

🛠️

이 절차를 시작하기 전에 최소 읽기 속도가 150MB/s이고 용량이 8GB 이상인 USB 키(USB 3.0이 이상적)가 준비되어 있는지 확인하세요.


전제 조건:



- 테일즈 전용 USB 키 1개, 용량 8GB 이상
- Linux, macOS(또는 Windows)로 인터넷에 연결된 컴퓨터
- 인터넷 연결 속도에 따라 약 1시간의 여유 시간(설치에 소요되는 ½시간 포함, 1.3GB 파일 다운로드)


## 1단계: 컴퓨터에서 Tails 다운로드


![image](assets/1.webp)


공식 테일즈 섹션:** https://tails.boum.org/install/linux/index.fr.html#download


확장자가 .img인 설치 파일을 다운로드하는 데는 인터넷 다운로드 속도에 따라 다소 시간이 걸릴 수 있으므로 미리 계획하세요. 최신의 효율적인 연결을 사용하면 5분도 채 걸리지 않습니다.


다음 단계에 필요하므로 다운로드와 같은 알려진 폴더에 파일을 저장합니다.


## 2단계: 다운로드 확인


![image](assets/2.webp)


공식 테일즈 섹션:** https://tails.boum.org/install/linux/index.fr.html#verify


다운로드를 확인하면 Tails 개발자가 발행한 것이며 다운로드 중에 손상되거나 가로채지 않았는지 확인할 수 있습니다.


PGP를 사용하여 방금 다운로드한 파일이 예상한 파일인지 수동으로 확인할 수 있지만, 고급 지식이 없으면 이 인증은 다운로드 페이지의 JavaScript 인증과 동일한 수준의 보안을 제공하는 반면 훨씬 더 복잡하고 오류가 발생하기 쉽습니다.


파일을 확인하려면 공식 섹션에 제공된 "다운로드 선택..." 버튼을 사용하세요!


## 3단계: USB 키에 테일즈 설치하기


![image](assets/3.webp)


공식 테일즈 섹션 :**



- Linux:** https://tails.boum.org/install/linux/index.fr.html#install
- macOS:** https://tails.boum.org/install/mac/index.fr.html#etcher 및 https://tails.boum.org/install/mac/index.fr.html#install


USB 키에 Tails를 설치하는 이 단계는 전체 가이드에서 가장 어려운 단계이며, 특히 이전에 한 번도 해본 적이 없는 경우 더욱 그렇습니다. 가장 중요한 점은 운영 체제의 공식 섹션에서 올바른 절차를 선택하는 것입니다: Linux 또는 macOS.


권장하는 대로 도구를 설치하고 준비하면 확장자가 .img인 파일을 키에 복사하여(기존 데이터를 모두 지움) 독립적으로 '부팅 가능'하게 만들 수 있습니다.


행운을 빕니다! 4단계에서 뵙겠습니다.


## 4단계: Tails USB 키에서 다시 시작하기


![image](assets/4.webp)


공식 테일즈 섹션:** https://tails.boum.org/install/linux/index.en.html#restart


이제 새 USB 스틱을 사용하여 컴퓨터 중 하나를 시작할 차례입니다. USB 포트 중 하나에 삽입하고 다시 시작하세요!


**참고💡: 대부분의 컴퓨터는 Tails USB 스틱에서 자동으로 부팅되지 않지만 부팅 메뉴 키를 누르면 부팅 가능한 장치 목록을 표시할 수 있습니다**


부팅 메뉴에서 일반적인 Hard 드라이브 대신 USB 스틱을 선택할 수 있도록 하기 위해 어떤 키를 눌러야 하는지 확인하려면 제조업체별로 전체 목록이 나와 있습니다:

## 5단계: 테일즈에 오신 것을 환영합니다!


![image](assets/6.webp)


공식 테일즈 섹션:** https://tails.boum.org/install/linux/index.en.html#tails


부팅 로더와 로딩 화면이 끝나고 1~2분 후에 시작 화면이 나타납니다.


![image](assets/7.webp)


시작 화면의 언어 및 지역 섹션에서 언어와 키보드 레이아웃을 선택합니다. 꼬리 시작을 클릭합니다.


![image](assets/8.webp)


컴퓨터가 네트워크에 연결되어 있지 않은 경우, Wi-Fi 없이 네트워크에 연결하는 데 도움이 되는 공식 Tails 지침을 참조하세요('Wi-Fi 테스트하기' 섹션).


로컬 네트워크에 연결되면 토르 연결 마법사가 나타나 토르 네트워크에 연결할 수 있도록 도와줍니다.


![image](assets/9.webp)


익명으로 브라우징을 시작하고 Tails에 포함된 옵션과 소프트웨어를 살펴볼 수 있습니다. USB 스틱에는 아무것도 수정되지 않으므로 오류가 발생할 여지가 충분하니 마음껏 즐기세요... 다음에 재시작하면 모든 경험이 잊혀질 것입니다!


## 향후 가이드에서...


자신만의 테일즈 USB 스틱으로 조금 더 실험을 해본 후에는 다른 글에서 다른 고급 주제에 대해 살펴보도록 하겠습니다:



- 최신 버전의 **테일즈**로 키를 업데이트하세요;
- 영구 저장소**를 구성하고 사용하세요;
- 추가 소프트웨어**를 설치합니다.


그때까지 항상 그렇듯이 궁금한 점이 있으면 언제든지 Agora256 커뮤니티에 공유해 주세요. 우리는 오늘보다 나은 내일이 되기 위해 함께 배우고 있습니다!
