## บทนำ - การควบคุมการแชทและความเป็นส่วนตัว



ในช่วงไม่กี่ปีที่ผ่านมามีการพูดคุยกันมากขึ้นเกี่ยวกับ Chat Control ซึ่งเป็นข้อเสนอด้านกฎระเบียบที่มุ่งหมายจะแนะนำการสแกนอัตโนมัติของข้อความส่วนตัวบนแพลตฟอร์มการสื่อสารหลัก ๆ เป้าหมายที่ระบุไว้คือการต่อสู้กับเนื้อหาที่ผิดกฎหมาย ปัญหาคือกลไกนี้จะเกี่ยวข้องกับการสอดส่องมวลชน ซึ่งจะบ่อนทำลายการเข้ารหัสแบบ end-to-end และดังนั้นความเป็นส่วนตัวของผู้ใช้ทั้งหมด ไม่ใช่แค่ผู้กระทำผิดเท่านั้น



ความเสี่ยงที่แท้จริงคือการที่การสนทนากลายเป็นสภาพแวดล้อมที่ถูกควบคุม ซึ่งทุกข้อความ รูปภาพ หรือไฟล์แนบอาจถูกตรวจสอบก่อนที่จะถึงผู้รับ และนี่คือที่มาของวิธีแก้ปัญหาที่เป็นไปได้วิธีหนึ่ง: ย้ายออกจากแพลตฟอร์มแบบรวมศูนย์และไปสู่ระบบการส่งข้อความแบบกระจายศูนย์ ซึ่งไม่ขึ้นอยู่กับผู้ให้บริการรายเดียวและไม่สามารถถูกตรวจสอบได้ง่ายๆ



หนึ่งในวิธีแก้ปัญหาดังกล่าวจะถูกนำเสนอในบทแนะนำนี้: Delta Chat. เครื่องมือที่มีความสมบูรณ์และพร้อมใช้งานแล้ว




## ทำไมต้อง Delta Chat และมันทำงานอย่างไร



Delta Chat เป็นโซลูชันการส่งข้อความที่ดีมากสำหรับการใช้งานในชีวิตประจำวัน: มีประโยชน์มากสำหรับการพูดคุยกับเพื่อน ครอบครัว และคนอื่น ๆ เช่นเดียวกับ WhatsApp อย่างแท้จริง



มันเป็นระบบส่งข้อความแบบกระจายศูนย์ที่อิงอยู่บนอีเมลทั้งหมด โดยพื้นฐานแล้วมันใช้โครงสร้างพื้นฐานของอีเมลแบบดั้งเดิม แต่สร้างอินเทอร์เฟซและประสบการณ์ของโปรแกรมส่งข้อความทันสมัยขึ้นมาใหม่ บนพื้นฐานนี้อาจดูเหมือนเป็นการด้นสดเล็กน้อย แต่จริงๆ แล้วมันทำงานได้ดีมากและมีความทนทานอย่างน่าประหลาดใจ คุณสามารถใช้เซิร์ฟเวอร์เมลเฉพาะที่เรียกว่า ChatMail แต่ก็สามารถทำงานร่วมกับเซิร์ฟเวอร์อีเมลทั่วไปได้อย่างราบรื่น ซึ่งหมายความว่าคุณสามารถเข้าสู่ระบบด้วยบัญชีที่มีอยู่แล้วหากต้องการ โดยไม่ต้องสร้างอะไรใหม่



อีกหนึ่งไฮไลท์คือการรองรับ WebXDCs ซึ่งเป็นแอปพลิเคชันเว็บขนาดเล็กที่สามารถใช้งานได้โดยตรงภายในแชท คล้ายกับแอปขนาดเล็กใน Telegram ความแตกต่างที่สำคัญคือแอปเหล่านี้ไม่มีการเข้าถึงอินเทอร์เน็ต ดังนั้นจึงไม่สามารถติดตามผู้ใช้หรือส่งข้อมูลออกไปภายนอกได้



จากมุมมองด้านความปลอดภัย Delta Chat ใช้การเข้ารหัสแบบ end-to-end ที่ผ่านการตรวจสอบแล้ว โดยอิงจาก PGP แต่มีการขยายที่ทันสมัยซึ่งทำให้มีระดับการป้องกันที่เทียบเท่ากับ Signal ข้อบกพร่องในปัจจุบันเพียงอย่างเดียวคือการขาด Perfect Forward Secrecy แต่เป็นแง่มุมที่กำลังพัฒนา



การอิงอยู่บนอีเมลเพียงอย่างเดียว, Delta Chat หลีกเลี่ยงมันทั้งหมด:




- หมายเลขโทรศัพท์ที่จำเป็น
- ID แบบรวมศูนย์
- การลงทะเบียนที่เชื่อมโยงกับบริการเดียว



และนั่นคือสิ่งที่ทำให้เครื่องมือนี้ทนทานต่อกฎระเบียบที่ล่วงล้ำเช่น Chat Control

## การสร้างโปรไฟล์



เมื่อเปิดแอป Delta Chat จะถามว่าต้องการสร้างโปรไฟล์ใหม่หรือใช้โปรไฟล์ที่มีอยู่แล้ว



![image](assets/it/02.webp)



โดยการสร้างโปรไฟล์ใหม่คุณสามารถเข้า:




- ชื่อ
- ภาพ (ไม่บังคับ)



เซิร์ฟเวอร์ ChatMail ถูกเสนอโดยค่าเริ่มต้น แต่เป็นไปได้:




- เลือกเซิร์ฟเวอร์ ChatMail อื่น
- ใช้บัญชีอีเมลแบบคลาสสิก
- กำหนดค่า IMAP และ SMTP ด้วยตนเอง
- ลงทะเบียนโดยใช้รหัสเชิญของผู้ใช้อื่น



หลังจากไม่กี่วินาที โปรไฟล์ก็พร้อมและคุณสามารถเริ่มใช้งานแอปได้



![image](assets/it/03.webp)




## Interface และแชท



อินเทอร์เฟซนั้นเรียบง่ายและตรงไปตรงมา:




- ข้อความของอุปกรณ์ ซึ่งเป็นการสื่อสารภายในท้องถิ่น
- บันทึกข้อความที่บันทึกไว้ คล้ายกับใน Telegram และสามารถซิงโครไนซ์ระหว่างอุปกรณ์ได้



![image](assets/it/04.webp)



ในการเพิ่มผู้ติดต่อ เพียงแค่:




- แสดงรหัส QR ของคุณ
- สแกนบุคคลอื่น
- เชิญผ่านลิงก์ (แชร์ลิงก์เชิญ)



![image](assets/it/05.webp)



เมื่อการเชื่อมต่อถูกสร้างขึ้น การเข้ารหัสแบบ end-to-end จะถูกกำหนดค่าโดยอัตโนมัติ ส่วนติดต่อผู้ใช้แชทมีความคล้ายคลึงกับ WhatsApp มาก:




- ข้อความและข้อความเสียง
- รูปภาพ วิดีโอ และไฟล์
- การตอบกลับข้อความ
- ปฏิกิริยา
- ข้อความป๊อปอัพ
- การแจ้งเตือนที่ปรับแต่งได้



![image](assets/it/06.webp)



## WebXDC: แอปในแชท:



Delta Chat อนุญาตให้ใช้ WebXDC ซึ่งเป็นแอปพลิเคชันขนาดเล็กที่ฝังอยู่ในการสนทนา นี่คือรายการสั้น ๆ ของแอปที่มีประโยชน์ที่สุดที่ระบุได้:




- แบบสำรวจ
- กระดานวาดภาพ
- แชทส่วนตัวชั่วคราว
- เกมที่มีการแชร์คะแนนในแชท



เกมที่ซับซ้อนมากขึ้นก็สามารถเริ่มต้นได้เช่นกัน ซึ่งแสดงให้เห็นถึงความยืดหยุ่นของเครื่องมือนี้



![image](assets/it/07.webp)



## กลุ่ม ช่อง และฟีเจอร์ขั้นสูง



คุณสามารถสร้างกลุ่ม ใช้สติกเกอร์ (โดยเฉพาะบนเดสก์ท็อป) และโดยการเปิดใช้งานตัวเลือกทดลอง แม้กระทั่งช่องทางที่คล้ายกับใน Telegram



ในการตั้งค่าขั้นสูงคุณสามารถเปิด:




- การโทรด้วยเสียง (ยังอยู่ในขั้นทดลอง)
- การจัดการโปรไฟล์อีเมลขั้นสูง
- การสำรองข้อมูลแบบเต็มรูปแบบ



![image](assets/it/08.webp)



## หลายอุปกรณ์และการสำรองข้อมูล



Delta Chat รองรับการใช้งานหลายอุปกรณ์อย่างเต็มรูปแบบ:




- คุณสามารถเพิ่มอุปกรณ์ที่สองผ่าน QR code
- คุณสามารถทำการโอนย้ายทั้งหมดผ่านการสำรองข้อมูลได้



ในไม่กี่วินาทีคุณจะพบแชท กลุ่ม และประวัติทั้งหมดของคุณอีกครั้ง โดยไม่ต้องพึ่งพาเซิร์ฟเวอร์กลาง



![image](assets/it/09.webp)




## บทสรุป



ในช่วงเวลาที่มีการพูดคุยเพิ่มขึ้นเกี่ยวกับการควบคุมการสื่อสารส่วนตัว Delta Chat เป็นคำตอบที่ชัดเจน: การส่งข้อความแบบกระจายศูนย์และเข้ารหัสที่สามารถใช้งานได้จริงในทุกๆ วัน



มันคือโซลูชันที่ในบรรดาทั้งหมดที่ฉันได้ลองมา ได้ทำให้ฉันเชื่อมั่นมากที่สุดในเรื่องของความเรียบง่าย ความเป็นส่วนตัว และเสรีภาพ หากคุณต้องการ คุณสามารถติดต่อฉันผ่าน Delta Chat ได้ทาง [ลิงก์เชิญ](https://i.delta.chat/#38824F04DD40600D5D4F079C1F5E0EBA875A6D7E&i=GStGfNW5LMIXhwQMiuQWj3QU&s=cVi5izRJ9NsbIcPlU8yC_SeB&a=9l4la5imj%40nine.testrun.org&n=SatoSats)



หากคุณชอบคู่มือนี้ คุณสามารถสนับสนุนฉันได้โดยการบริจาคและกดถูกใจ และจำไว้ว่า: การใช้และสำรวจ Delta Chat ทั้งจากมือถือและเดสก์ท็อปเท่านั้นที่จะทำให้คุณค้นพบฟังก์ชันการทำงานทั้งหมดของมันอย่างแท้จริง
