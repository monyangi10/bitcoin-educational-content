## บทนำ



ในบริบทที่เบราว์เซอร์ส่วนใหญ่เก็บรวบรวมข้อมูลส่วนบุคคลของเราอย่างมหาศาล การเลือกเบราว์เซอร์ที่เป็นมิตรต่อความเป็นส่วนตัวจึงกลายเป็นสิ่งสำคัญ Chrome ครองตลาดโลกด้วยส่วนแบ่ง 65% แต่โมเดลธุรกิจของมันขึ้นอยู่กับการใช้ประโยชน์จากข้อมูลการท่องเว็บของคุณ Safari แม้จะถูกรวมเข้ากับระบบนิเวศของ Apple แต่ก็ขาดคุณสมบัติการป้องกันขั้นสูงและไม่รองรับส่วนขยายของบุคคลที่สามอย่างยืดหยุ่น



![Répartition du marché des navigateurs](assets/fr/01.webp)


*การแบ่งส่วนตลาดเว็บเบราว์เซอร์: Chrome ครองตลาดด้วยส่วนแบ่งมากกว่า 65% ตามมาด้วย Safari, Edge และ Firefox*



**Orion Browser** นำเสนอตัวเองเป็นทางเลือกใหม่สำหรับผู้ใช้ Apple พัฒนาโดย Kagi เบราว์เซอร์นี้ผสมผสานความเร็วของเครื่องยนต์ WebKit กับปรัชญาไร้การติดตามข้อมูล ไม่เหมือนกับคู่แข่ง Orion ไม่ส่งข้อมูลไปยังเซิร์ฟเวอร์ระยะไกลและบล็อกโฆษณาและตัวติดตาม 99.9% โดยธรรมชาติ รวมถึง YouTube



คุณสมบัติที่โดดเด่นของมัน? Orion เป็นเบราว์เซอร์ **WebKit เพียงหนึ่งเดียว** ที่สามารถติดตั้งส่วนขยายของ Chrome **และ** Firefox ได้โดยตรง ซึ่งมอบสิ่งที่ดีที่สุดจากทั้งสองโลก ความเข้ากันได้นี้ รวมกับการใช้หน่วยความจำที่ต่ำกว่าเบราว์เซอร์อื่น 2 ถึง 3 เท่า และการผสานรวมอย่างไร้รอยต่อกับระบบนิเวศของ Apple (iCloud, Keychain) ทำให้มันเป็นตัวเลือกที่เหมาะสมที่สุดสำหรับผู้ใช้ Mac และ iPhone ที่ใส่ใจเรื่องความเป็นส่วนตัว



## ทำไมถึงเลือก Orion Browser?



### ประโยชน์หลัก



**การป้องกันสูงสุดตั้งแต่แกะกล่อง**: Orion บล็อกโฆษณา 99.9% (รวมถึง YouTube) และตัวติดตามจากบุคคลที่หนึ่งและบุคคลที่สามทั้งหมดโดยค่าเริ่มต้น เทคโนโลยีของมันผสมผสานการป้องกันการติดตามอัจฉริยะของ WebKit กับรายการ EasyPrivacy เพื่อประสิทธิภาพสูงสุด คุณสมบัติพิเศษ: Orion บล็อกสคริปต์การพิมพ์ลายนิ้วมือ **ก่อนที่จะถูกดำเนินการ** ทำให้การติดตามเป็นไปไม่ได้จริง ๆ - เป็นวิธีการที่รุนแรงกว่าที่เบราว์เซอร์อื่น ๆ ซึ่งพยายามเพียงแค่ "ปกปิด" ข้อมูลเท่านั้น



**ไม่มีการตรวจวัดทางไกลที่ตรวจสอบได้**: Orion ใช้วิธีการที่แปลกใหม่ในการรักษาความเป็นส่วนตัว โดยออกแบบให้ไม่มีการตรวจวัดทางไกล แตกต่างจากเบราว์เซอร์อื่น ๆ ที่มีการร้องขอเครือข่ายหลายร้อยครั้งเมื่อเริ่มต้น (IP exponent, ลายนิ้วมือเบราว์เซอร์, ข้อมูลส่วนบุคคล) Orion ไม่เคย "โทรกลับบ้าน" ความแตกต่างพื้นฐานนี้ช่วยขจัดความเสี่ยงของการรั่วไหลของข้อมูลโดยไม่ตั้งใจอย่างสิ้นเชิง



**ประสิทธิภาพที่ยอดเยี่ยม**: ด้วยการใช้เวอร์ชันที่ปรับแต่งของ WebKit, Orion มีความเร็วเทียบเท่าหรือแม้กระทั่งเหนือกว่า Safari บน Mac. การทดสอบ Speedometer 2.0/2.1 มักจะจัดอันดับให้มันเป็นที่หนึ่งบนโปรเซสเซอร์ Apple Silicon. การบล็อกโฆษณาแบบเนทีฟยังช่วยเร่งการโหลดหน้าเว็บได้อีก 20 ถึง 40%.



**รองรับส่วนขยายสากล**: นวัตกรรมสำคัญ, Orion ช่วยให้คุณติดตั้งส่วนขยายจาก Chrome Web Store **และ** Mozilla Add-ons ได้ การรองรับ WebExtensions ยังอยู่ในขั้นทดลอง โดยมีเป้าหมายความเข้ากันได้ 100% ในการเปิดตัวเบต้า คุณสามารถใช้ส่วนขยายยอดนิยมหลายตัว เช่น uBlock Origin, Bitwarden แม้กระทั่งบน iPhone - เป็นครั้งแรกในโลกบน iOS แม้ว่าบางตัวอาจทำงานไม่สมบูรณ์แบบก็ตาม



### ข้อจำกัดที่ควรทราบ





- ความพร้อมใช้งานจำกัด**: ปัจจุบันสงวนไว้สำหรับ macOS และ iOS/iPadOS เวอร์ชัน Linux กำลังถึงจุดพัฒนาสำคัญ (Milestone 2 ในปี 2025) แต่ยังไม่มีการสร้างสาธารณะ Windows และ Android ไม่ได้อยู่ในระหว่างการพัฒนาเนื่องจากขาดทรัพยากร
- โค้ดปิด**: แม้ว่าบางส่วนจะเป็นโอเพ่นซอร์ส แต่ Orion ยังคงเป็นซอฟต์แวร์ที่มีกรรมสิทธิ์เป็นหลัก ซึ่งเป็นประเด็นถกเถียงในชุมชนด้านความเป็นส่วนตัว
- ส่วนขยายทดลอง**: การสนับสนุนส่วนขยายยังคงอยู่ในช่วงเบต้า โดยมีความไม่เข้ากันบ่อยครั้ง ส่วนขยายอาจส่งผลต่อประสิทธิภาพ และบางส่วนขยายไม่ทำงานเลย
- ความปลอดภัยของ WebKit**: ไม่เหมือนกับ Chromium, WebKit ไม่ได้มีการแยกกระบวนการต่อไซต์ที่แข็งแกร่งเท่า ซึ่งอาจก่อให้เกิดความเสี่ยงด้านความปลอดภัยในบางสถานการณ์
- การทดสอบการบล็อก**: Orion ทำผลงานได้ไม่ดีในแบบทดสอบโฆษณาออนไลน์ (26-35%) เนื่องจาก Kagi พิจารณาว่าแบบทดสอบเหล่านี้ "มีข้อบกพร่องโดยพื้นฐาน" ประสิทธิภาพที่แท้จริงในการใช้งานทุกวันนั้นเหนือกว่ามาก



## การติดตั้ง Orion Browser



### บน macOS



![Page d'accueil Kagi avec Orion Browser](assets/fr/02.webp)


*หน้าแรกของ Kagi นำเสนอ Orion Browser ว่าเป็น "เบราว์เซอร์ที่ไม่มีโฆษณาพร้อมการปกป้องความเป็นส่วนตัวอย่างสมบูรณ์และการสนับสนุนส่วนขยายที่ครอบคลุม"*





- ไปที่ [kagi.com/orion](https://kagi.com/orion/)
- คลิกที่ "**ดาวน์โหลด Orion สำหรับ macOS**"



![Page de téléchargement d'Orion Browser](assets/fr/03.webp)


*หน้าดาวน์โหลด Orion Browser แสดงความพร้อมใช้งานสำหรับ macOS และ iOS พร้อมลิงก์ไปยัง App Store*





- เปิดไฟล์ `.dmg` ที่ดาวน์โหลด
- ลากแอปพลิเคชัน Orion ไปยังโฟลเดอร์ Applications
- เมื่อเปิดใช้งานครั้งแรก macOS จะขอให้คุณยืนยันการเปิด



**ทางเลือกโฮมบรูว์**:


```bash
brew install --cask orion
```



### บน iPhone/iPad





- เปิด **App Store**
- ค้นหา "**Orion Browser by Kagi**"
- ติดตั้งแอปฟรี (รองรับ iOS 15+)



### การกำหนดค่าเริ่มต้น



เมื่อเปิดใช้งานครั้งแรก Orion จะนำคุณผ่านหลายขั้นตอน:



**1. หน้าจอต้อนรับ**


![Écran de bienvenue d'Orion](assets/fr/04.webp)


*หน้าจอต้อนรับ Orion Browser เน้นคุณสมบัติเด่น: การท่องเว็บที่เร็วขึ้น, ไม่มีการเก็บข้อมูลการใช้งาน, การบล็อกโฆษณา และการรองรับส่วนขยาย*



**2. Interface** การปรับแต่ง


![Options de personnalisation](assets/fr/05.webp)


*หน้าจอการปรับแต่งช่วยให้คุณกำหนดลักษณะของแท็บและ Interface ให้เหมาะกับความต้องการของคุณ*





- นำเข้าข้อมูล**: โอนย้ายรายการโปรดและรหัสผ่านจาก Safari, Chrome หรือ Firefox ได้อย่างง่ายดาย
- iCloud sync**: เปิดใช้งานเพื่อค้นหารายการโปรดและแท็บของคุณบนอุปกรณ์ Apple ทั้งหมดของคุณ



**3. การติดตั้งบนอุปกรณ์เคลื่อนที่**


![Installation sur iOS](assets/fr/06.webp)


*หน้าจอติดตั้งบน iOS แสดงรหัส QR เพื่อดาวน์โหลด Orion Browser จาก App Store อย่างรวดเร็ว*



**4. Interface ยินดีต้อนรับและเครื่องมือที่จำเป็น**



![Page d'accueil Orion](assets/fr/07.webp)


*Orion Browser Interface หน้าแรก: ลูกศรแสดงถึงเครื่องมือหลักสามอย่างที่สามารถเข้าถึงได้โดยตรงจากแถบที่อยู่*



เมื่อการกำหนดค่าเสร็จสมบูรณ์ คุณจะพบกับ Interface ของ Orion ที่มี **เครื่องมือสำคัญสามอย่าง** (ระบุด้วยลูกศร):





- โล่ 🛡️**: แสดงรายงานความเป็นส่วนตัวพร้อมจำนวนรายการที่ถูกบล็อกในหน้าปัจจุบัน
- แปรง 🖌️**: ปรับแต่งการแสดงผลของหน้า (ธีม, ฟอนต์, ลบองค์ประกอบที่รบกวน)
- เกียร์ ⚙️**: กำหนดค่าพารามิเตอร์เฉพาะเว็บไซต์ (การอนุญาต, การบล็อก, ฯลฯ)



เครื่องมือเหล่านี้พร้อมใช้งานเสมอและช่วยให้คุณควบคุมประสบการณ์การท่องเว็บของคุณได้ตามแต่ละเว็บไซต์



**สำคัญ**: Orion ฟรีและไม่ต้องการการลงทะเบียนหรือสร้างบัญชีเพื่อใช้งาน



![Orion+ dans les préférences](assets/fr/08.webp)


*หน้าจอการสมัครสมาชิก Orion+ ในการตั้งค่า เสนอการสมัครสมาชิกแบบเลือกได้เพื่อสนับสนุนการพัฒนา*



**Orion+ (ตัวเลือก)**: เพื่อสนับสนุนการพัฒนาโครงการ Kagi เสนอ Orion+ ($5/เดือน, $50/ปี, หรือ $150 ตลอดชีพ) การสมัครสมาชิกโดยสมัครใจนี้ช่วยให้:




- สื่อสารโดยตรงกับทีมพัฒนา
- มีอิทธิพลต่อวิวัฒนาการของเบราว์เซอร์ตามความต้องการของคุณ
- เข้าถึงเวอร์ชัน Nightly พร้อมคุณสมบัติทดลองล่าสุด
- รับประโยชน์จากเอนจิน WebKit ล่าสุด
- รับตราสัญลักษณ์ที่โดดเด่นในฟอรัมข้อเสนอแนะ



Orion+ รับประกันความเป็นอิสระของโครงการ: "การสนับสนุนทางการเงินของคุณช่วยให้เรายังคงเป็นอิสระและรักษาสัญญาที่จะเป็นเบราว์เซอร์ที่ดีที่สุดสำหรับผู้ใช้ของเรา" เป็นโมเดลการระดมทุนจากผู้ใช้ที่ทำให้ Orion ปราศจากโฆษณาและการติดตามข้อมูล



## การกำหนดค่าสำหรับความลับสูงสุด



### พารามิเตอร์ที่จำเป็น



เข้าถึงการตั้งค่าผ่าน **Orion → Preferences** (หรือ ⌘,):



**1. ค้นหา - เครื่องมือค้นหาส่วนตัว**



![Configuration du moteur de recherche](assets/fr/09.webp)


*การตั้งค่าเครื่องมือค้นหาเริ่มต้น: เลือก DuckDuckGo เพื่อความเป็นส่วนตัวสูงสุด*





- เครื่องมือค้นหาเริ่มต้น**: เลือก **DuckDuckGo**, **Startpage** หรือ **Kagi** เพื่อความเป็นส่วนตัวสูงสุด (หลีกเลี่ยง Google/Bing)
- คำแนะนำการค้นหา**: ปิดใช้งานเพื่อป้องกันการรั่วไหลของการกดแป้นพิมพ์ไปยังเซิร์ฟเวอร์ของเครื่องมือค้นหา



**2. ความเป็นส่วนตัว - ทั่วไป** การป้องกัน



![Content Blocker dans les préférences](assets/fr/12.webp)


*การตั้งค่าความเป็นส่วนตัวของ Orion แสดง Content Blocker ที่มี 119,156 กฎที่ใช้งานอยู่, ตัวเลือกการลบตัวติดตาม และตัวแทนผู้ใช้ที่กำหนดเอง*



**ตัวบล็อกเนื้อหาทำงานโดยค่าเริ่มต้น**:




- EasyList**: 119k+ กฎการบล็อกโฆษณา
- EasyPrivacy**: การป้องกันการติดตาม
- จัดการรายการตัวกรอง**: เพิ่มรายการเพิ่มเติม (แนะนำโดย Hagezi)



**ตัวเลือกความเป็นส่วนตัว**:




- ลบตัวติดตามออกจาก URL**: "สำหรับการท่องเว็บแบบส่วนตัวเท่านั้น" ทำความสะอาดลิงก์ที่คัดลอก
- แชร์รายงานการขัดข้อง**: "หลังจากขออนุมัติ" เคารพการยินยอมของคุณ
- ตัวแทนผู้ใช้ที่กำหนดเอง**: สามารถแก้ไขเพื่อหลีกเลี่ยงการบล็อกบางอย่าง



![YouTube avec Privacy Report](assets/fr/10.webp)


*ตัวอย่างของ YouTube ที่ดูด้วย Orion: ไม่มีโฆษณาให้เห็นและรายงานความเป็นส่วนตัวแสดงองค์ประกอบหลายอย่างที่ถูกบล็อก*



**3. การตั้งค่าเว็บไซต์ - ควบคุมโดยเว็บไซต์**



![Website Settings pour YouTube](assets/fr/11.webp)


*การตั้งค่าเว็บไซต์สำหรับ YouTube แสดงตัวเลือกความเข้ากันได้ การบล็อกเนื้อหา และการอนุญาตเฉพาะไซต์*



**การเข้าถึงอย่างรวดเร็ว**: คลิกที่รูปเฟือง ⚙️ ในแถบที่อยู่เพื่อปรับ:




- โหมดความเข้ากันได้**: แก้ปัญหาการแสดงผลโดยการระงับส่วนขยาย
- ตัวบล็อกเนื้อหา**: ปิดการบล็อกสำหรับเว็บไซต์เฉพาะหากจำเป็น
- JavaScript/Cookies**: การควบคุมอย่างละเอียดตามเว็บไซต์
- การอนุญาต**: กล้อง, ไมโครโฟน, ตำแหน่งที่ตั้ง กำหนดค่าแยกกัน



**4. ตัวกรองแบบกำหนดเองขั้นสูง** (ดูด้านล่าง)



**สร้างตัวกรองแบบกำหนดเอง** (ความเป็นส่วนตัว → จัดการรายการตัวกรอง → ตัวกรองแบบกำหนดเอง):



**ไวยากรณ์ที่เรียบง่าย** (เข้ากันได้กับ Adblock Plus):




- `reddit.com##.promotedlink`: ซ่อนโพสต์ที่ได้รับการสนับสนุนบน Reddit
- `||ads.example.com^`: บล็อกโดเมนโฆษณาอย่างสมบูรณ์
- `@@||site-utile.com^`: สร้างข้อยกเว้นสำหรับเว็บไซต์



**เคล็ดลับ: เยี่ยมชม [FilterLists.com](https://filterlists.com) สำหรับรายการเฉพาะทางที่พร้อมใช้งานนับพันรายการ**



### ส่วนขยายที่แนะนำ



Orion รองรับส่วนขยาย Chrome และ Firefox โดยตรง ติดตั้งได้จากร้านค้าอย่างเป็นทางการ:



**สิ่งจำเป็น**:




- uBlock Origin**: เพิ่มการควบคุมอย่างละเอียดให้กับตัวบล็อกเนทีฟ
- Bitwarden**: ตัวจัดการรหัสผ่านแบบโอเพนซอร์ส
- ClearURLs**: ลบพารามิเตอร์การติดตามใน URL



**ตัวเลือก**:




- LocalCDN**: ให้บริการไลบรารีที่ใช้ร่วมกันในเครื่อง
- Cookie AutoDelete**: ลบคุกกี้โดยอัตโนมัติหลังจากปิดแท็บ
- NoScript**: ควบคุมการทำงานของ JavaScript ได้อย่างสมบูรณ์ (สำหรับผู้ใช้ขั้นสูง)



เพื่อติดตั้ง:




- เยี่ยมชม [chrome.google.com/webstore](https://chrome.google.com/webstore) หรือ [addons.mozilla.org](https://addons.mozilla.org)
- คลิกที่ "เพิ่มใน Chrome/Firefox"
- Orion จะสกัดกั้นและติดตั้งส่วนขยายโดยอัตโนมัติ



**คำเตือน**: เนื่องจากการรองรับส่วนขยายยังอยู่ในขั้นทดลอง ส่วนขยายหลายตัวอาจทำงานไม่ถูกต้องหรืออาจส่งผลต่อประสิทธิภาพการทำงาน ในกรณีที่เกิดปัญหา (เว็บไซต์ไม่ทำงานอีกต่อไป, ความช้าลง) ให้ปิดการใช้งานส่วนขยายทีละตัวเพื่อระบุแหล่งที่มา



## การใช้งานประจำวัน



### Interface และคุณสมบัติพิเศษ




![Outil de personnalisation pinceau](assets/fr/13.webp)


*เมนูแปรง Orion สำหรับปรับแต่งการแสดงผล: ขนาดตัวอักษร, ธีม (สว่าง/มืด), การปิดใช้งานส่วนหัวที่ติดหนึบ และการลบองค์ประกอบที่รบกวน*



**เครื่องมือแปรง: การปรับแต่งขั้นสูง**



เครื่องมือ **แปรง** ของ Orion เป็นคุณสมบัติพิเศษที่ให้คุณปรับแต่งการแสดงผลของแต่ละเว็บไซต์:



**ตัวเลือกธีม**:




- สลับระหว่างธีมสว่างและมืดสำหรับแต่ละเว็บไซต์
- การปรับอัตโนมัติตามการตั้งค่าระบบของคุณ



**การควบคุมการพิมพ์**:




- ขนาดตัวอักษร**: ปรับความอ่านง่ายด้วยปุ่ม A- และ A+
- รูปแบบตัวอักษร**: เปลี่ยนฟอนต์ (ค่าเริ่มต้นหรือกำหนดเอง)



**การทำความสะอาด Interface**:




- ปิดใช้งานส่วนหัวที่ติดหนึบ**: ลบส่วนหัวที่ยังคงติดอยู่ด้านบนเมื่อเลื่อนหน้า
- ลบ Elements**: ลบองค์ประกอบที่น่ารำคาญออกอย่างถาวร (โฆษณา, ป๊อปอัพ, แบนเนอร์คุกกี้)
  - คลิกที่ "+ ลบ" จากนั้นเลือกไอเท็มที่ต้องการซ่อน
  - มีประโยชน์มากสำหรับเว็บไซต์ที่มีโฆษณาถาวรหรือองค์ประกอบการติดตามภาพ



**ความคงอยู่**: การตั้งค่าทั้งหมดนี้จะถูกบันทึกต่อโดเมนและนำมาใช้ใหม่โดยอัตโนมัติในครั้งถัดไปที่คุณเข้าชม



**การจัดการแท็บขั้นสูง**:




- แท็บแนวตั้ง**: เปิดใช้งานผ่านแถบเมนู (ฟังก์ชันแท็บด้านข้าง)
- แท็บแบบกะทัดรัด**: ในการตั้งค่า → แท็บ → เลย์เอาต์ "กะทัดรัด" เพื่อประหยัดพื้นที่
- กลุ่มแท็บ**: จัดระเบียบเซสชันของคุณตามธีม
- โปรไฟล์หลายรายการ**: สร้างตัวตนแยกต่างหากผ่านแถบเมนู (ฟังก์ชันโปรไฟล์) โดยมีข้อมูลที่แยกออกจากกันอย่างสิ้นเชิง



**โหมดพลังงานต่ำ**: ได้แรงบันดาลใจจาก iPhone โหมดนี้จะระงับแท็บที่ไม่ใช้งานโดยอัตโนมัติหลังจาก 5 นาที และสามารถลดการใช้พลังงานได้ถึง 90% เปิดใช้งานผ่านแถบเมนูของ Orion บน Mac หรือในการตั้งค่าบน iOS



**เครื่องมือในตัว** (เมนูแก้ไขและอื่นๆ):




- แก้ไขข้อความบนหน้า**: แก้ไขข้อความชั่วคราว (เมนูแก้ไข)
- อนุญาตให้คัดลอกและวาง**: ข้ามข้อจำกัดการคัดลอก (เมนูแก้ไข)
- คัดลอกลิงก์ที่สะอาด**: คลิกขวาที่ลิงก์เพื่อลบพารามิเตอร์การติดตาม
- โหมดโฟกัส**: การนำทางแบบเต็มหน้าจอ ปราศจากสิ่งรบกวน
- Picture-in-Picture**: ดูวิดีโอในหน้าต่างลอย
- เปิดใน Internet Archive**: เข้าถึงเวอร์ชันที่เก็บถาวรโดยตรง
- รายงานความเป็นส่วนตัว**: คลิกที่โล่ 🛡️ เพื่อดูรายการที่ถูกบล็อกโดยหน้า



### การจัดการการท่องเว็บแบบส่วนตัว



การนำทางส่วนตัวของ Orion (⌘⇧N) มีให้:




- การแยกคุกกี้และเซสชันอย่างสมบูรณ์
- การลบอัตโนมัติเมื่อปิด
- ประวัติและการปิดใช้งานแคช
- การป้องกันที่เพิ่มขึ้นจากการติดตามลายนิ้วมือ



**เคล็ดลับ**: สำหรับการแบ่งส่วนขั้นสูง ให้สร้าง **โปรไฟล์แยกต่างหาก** ผ่านแถบเมนู (ฟังก์ชันโปรไฟล์) แต่ละโปรไฟล์จะปรากฏเป็นแอปแยกใน Dock โดยมีการตั้งค่า ส่วนขยาย และข้อมูลของตัวเองที่แยกออกจากกันอย่างสมบูรณ์



### การเพิ่มประสิทธิภาพและความเป็นส่วนตัว



เพื่อให้ Orion รวดเร็วและเป็นส่วนตัว:




- ส่วนขยาย**: จำกัดให้เหลือน้อยที่สุด (อาจลดประสิทธิภาพ)
- โหมดพลังงานต่ำ**: เปิดใช้งานสำหรับการใช้งานเป็นเวลานาน (ประหยัดได้ถึง 90%)
- รายงานความเป็นส่วนตัว**: คลิกที่โล่ 🛡️ เพื่อดูการบล็อกแบบเรียลไทม์
- การปรับแต่งภาพ**: ใช้แปรง 🖌️ เพื่อปรับการแสดงผลและลบองค์ประกอบที่รบกวน
- คัดลอกลิงก์สะอาด**: คลิกขวาเพื่อคัดลอกลิงก์โดยไม่มีตัวติดตาม
- โปรไฟล์แยกต่างหาก**: ใช้โปรไฟล์เฉพาะเพื่อแยกกิจกรรมของคุณออกจากกัน
- การตั้งค่าเว็บไซต์**: คลิกที่รูปเฟือง ⚙️ เพื่อปรับสิทธิ์ตามเว็บไซต์
- การทำความสะอาดเป็นประจำ**: ล้างแคชผ่าน Orion → ล้างข้อมูลการท่องเว็บ

## กรณีการใช้งานที่แนะนำ



**เหมาะสำหรับ**:




- ผู้ใช้ Apple ที่ต้องการความเป็นส่วนตัวมากกว่า Safari
- คนที่ต้องการบล็อกโฆษณาทั้งหมด (รวมถึง YouTube) โดยไม่ใช้ส่วนขยาย
- นักพัฒนาที่ต้องการ WebKit devtools พร้อมการปกป้องความเป็นส่วนตัวในตัว
- ผู้ใช้ iPhone ต้องการส่วนขยายเดสก์ท็อปบนมือถือ (นวัตกรรมที่ไม่เหมือนใคร)
- มืออาชีพที่ต้องการแยกกิจกรรมของตนออกเป็นส่วน ๆ (โปรไฟล์หลายอัน)
- ผู้ใช้มือถือที่มองหาการจัดการแบตเตอรี่ที่ดียิ่งขึ้น (โหมดพลังงานต่ำ)



**หลีกเลี่ยงหาก**:




- คุณใช้ Windows/Linux เป็นหลัก (ไม่มีเวอร์ชันที่ใช้ได้)
- โอเพ่นซอร์สเต็มรูปแบบเป็นสิ่งจำเป็นสำหรับโมเดลภัยคุกคามของคุณ
- คุณพึ่งพาส่วนขยายเฉพาะที่อาจไม่ทำงาน
- คุณต้องการการซิงโครไนซ์ข้ามแพลตฟอร์มนอกเหนือจากระบบนิเวศของ Apple
- คุณชอบโซลูชันที่ได้รับการพิสูจน์แล้วและมีความเสถียร (สถานะเบต้าถาวรตั้งแต่ปี 2021)



## จุดที่ต้องให้ความสนใจและความปลอดภัย



### คุณสมบัติด้านความปลอดภัยที่ไม่เหมือนใคร



**การป้องกันลายนิ้วมือที่ปฏิวัติวงการ**: Orion เป็นเบราว์เซอร์เดียวที่สามารถบล็อกการทำงานของสคริปต์ลายนิ้วมือได้อย่างสมบูรณ์ก่อนที่พวกมันจะสามารถสแกนระบบของคุณได้ วิธีการ "ไม่มีการรันสคริปต์ = ไม่มีการลายนิ้วมือ" นี้มีประสิทธิภาพเหนือกว่าวิธีการปกปิดแบบดั้งเดิมที่ใช้โดยเบราว์เซอร์อื่น ๆ



**Whitelist ที่โปร่งใส**: Orion รักษารายการสาธารณะขนาดเล็กของเว็บไซต์ (browserbench.org, wizzair.com) ที่การบล็อกถูกปิดใช้งานโดยอัตโนมัติเพื่อหลีกเลี่ยงการทำงานผิดพลาด ความโปร่งใสนี้ช่วยให้ผู้ใช้เข้าใจได้อย่างชัดเจนว่าเมื่อใดและทำไมการบล็อกจึงถูกผ่อนคลาย



**ส่วนขยายที่ไม่ได้ตรวจสอบ**: การสนับสนุนสำหรับส่วนขยาย Chrome/Firefox เพิ่มความเสี่ยงเพิ่มเติม เนื่องจากส่วนขยายเหล่านี้ไม่ได้ออกแบบมาสำหรับ WebKit และไม่ได้รับการตรวจสอบเฉพาะสำหรับสภาพแวดล้อมนี้



### การบำรุงรักษาและการอัปเดต





- อัปเดตอัตโนมัติ**: Orion อัปเดตอัตโนมัติบน macOS ผ่าน Sparkle
- การติดตามช่องโหว่**: ตรวจสอบหมายเหตุการเผยแพร่อย่างสม่ำเสมอสำหรับแพตช์ความปลอดภัย
- การรายงานข้อบกพร่อง**: ใช้ [orionfeedback.org](https://orionfeedback.org) เพื่อรายงานปัญหา




## บทสรุป



Orion Browser เป็นก้าวสำคัญสำหรับความเป็นส่วนตัวบน macOS และ iOS ด้วยแนวทางที่ไม่มีการเก็บข้อมูลการใช้งาน รวมกับตัวบล็อกเนทีฟที่มีประสิทธิภาพสูงและการสนับสนุนพิเศษสำหรับส่วนขยายสากล ทำให้เป็นตัวเลือกที่ยอดเยี่ยมสำหรับผู้ใช้ Apple ที่ใส่ใจในความเป็นส่วนตัว



**สำคัญ**: Orion ยังคงอยู่ในสถานะเบต้าถาวรตั้งแต่ปี 2021 โดยมีข้อจำกัดด้านความเข้ากันได้ของส่วนขยายและความเสี่ยงที่มีอยู่ใน WebKit ประเมินการแลกเปลี่ยนเหล่านี้ตามรูปแบบภัยคุกคามของคุณ



สำหรับการใช้งานในชีวิตประจำวันบน Mac หรือ iPhone มันอาจจะเป็นทางเลือกที่ดีที่สุดระหว่างความเป็นส่วนตัว, ประสิทธิภาพ และความง่ายในการใช้งานที่มีอยู่ในระบบนิเวศของ Apple โดยมีเงื่อนไขว่าคุณยอมรับข้อจำกัดของมัน



โปรดจำไว้ว่า: การปกป้องความเป็นส่วนตัวของคุณไม่ได้ขึ้นอยู่กับเบราว์เซอร์ของคุณเท่านั้น ผสาน Orion กับแนวทางปฏิบัติที่ดีที่สุด (รหัสผ่านที่แข็งแกร่ง, 2FA, VPN หากจำเป็น) เพื่อความปลอดภัยออนไลน์ที่ดีที่สุด
