Signal เป็นแอปพลิเคชันส่งข้อความที่เข้ารหัสแบบ end-to-end ออกแบบมาเพื่อให้ความเป็นส่วนตัวที่ดีโดยค่าเริ่มต้น ทุกข้อความ การโทร และไฟล์ได้รับการปกป้องโดยโปรโตคอล Signal ซึ่งได้รับการยอมรับว่าเป็นหนึ่งในโปรโตคอลการส่งข้อความที่แข็งแกร่งที่สุด มันถูกนำมาใช้ซ้ำโดยแอปพลิเคชันอื่นๆ อีกมากมาย รวมถึง WathsApp, Facebook Messenger, Skype และ Google Messages สำหรับการสื่อสาร RCS



Signal เปิดตัวในปี 2014 โดย Moxie Marlinspike (นามแฝง) และได้รับการพัฒนาตั้งแต่ปี 2018 โดยมูลนิธิ Signal ซึ่งเป็นองค์กรไม่แสวงหาผลกำไรที่ก่อตั้งขึ้นด้วยการสนับสนุนจาก Brian Acton (ผู้ร่วมก่อตั้ง WhatsApp)



![Image](assets/fr/01.webp)



เมื่อเปรียบเทียบกับ WhatsApp, Signal โดดเด่นในเรื่องความโปร่งใส: โค้ดของแอปพลิเคชันทั้งฝั่งลูกค้าและฝั่งเซิร์ฟเวอร์เป็นโอเพ่นซอร์สทั้งหมด สิ่งนี้ทำให้ใครก็ตามสามารถตรวจสอบได้ และโดยเฉพาะอย่างยิ่งเพื่อตรวจสอบว่าการเข้ารหัสถูกนำมาใช้ตามที่โฆษณาไว้



อย่างไรก็ตาม Signal อาศัยการใช้หมายเลขโทรศัพท์ ซึ่งเป็นจุดอ่อนหลักเมื่อพูดถึงการไม่เปิดเผยตัวตนเมื่อเทียบกับโซลูชันอื่น ๆ ถึงกระนั้น แอปพลิเคชันนี้ก็เป็นหนึ่งในแอปพลิเคชันที่น่าเชื่อถือที่สุดในแง่ของความปลอดภัยและความเป็นส่วนตัวในความเห็นของฉัน เนื่องจากสถาปัตยกรรมที่เปิดกว้างทั้งหมดและโปรโตคอลการเข้ารหัสที่ได้รับการยอมรับอย่างกว้างขวาง และดังนั้นจึงได้รับการทดสอบและตรวจสอบแล้ว ซึ่งแตกต่างจากแอปพลิเคชันอื่น ๆ ที่มีความสำคัญน้อยกว่า

## ติดตั้งแอปพลิเคชัน Signal



Signal พร้อมใช้งานบนทุกแพลตฟอร์ม คุณสามารถดาวน์โหลดแอปพลิเคชันได้โดยตรงจากร้านค้าแอปพลิเคชันของโทรศัพท์ของคุณ:




- [Google Play](https://play.google.com/store/apps/details?id=org.thoughtcrime.securesms);
- [App Store](https://apps.apple.com/us/app/signal-private-messenger/id874139669);



บน Android ก็สามารถ [ติดตั้งผ่าน APK](https://github.com/signalapp/Signal-Android/releases) ได้เช่นกัน



ในบทแนะนำนี้ เราจะมุ่งเน้นไปที่เวอร์ชันมือถือ แต่โปรดทราบว่า [เวอร์ชันเดสก์ท็อปก็มีให้บริการเช่นกัน](https://signal.org/fr/download/) (MacOS, Linux และ Windows) อย่างไรก็ตาม คุณจำเป็นต้องตั้งค่าแอปพลิเคชันมือถือก่อน ก่อนที่คุณจะสามารถซิงโครไนซ์บัญชีของคุณกับเวอร์ชันเดสก์ท็อปได้



## สร้างบัญชีบน Signal



เมื่อคุณเปิดแอปพลิเคชันเป็นครั้งแรก ให้คลิกที่ปุ่ม "*Continue*"



![Image](assets/fr/02.webp)



ป้อนหมายเลขโทรศัพท์ของคุณ จากนั้นคลิกที่ "*ถัดไป*"



![Image](assets/fr/03.webp)



รหัสยืนยันจะถูกส่งไปยังคุณทาง SMS ป้อนรหัสนี้ในแอปพลิเคชัน Signal



![Image](assets/fr/04.webp)



เลือกรหัส PIN เพื่อรักษาความปลอดภัยให้กับบัญชี Signal ของคุณ รหัสนี้จะเข้ารหัสข้อมูลของคุณ และสามารถใช้เพื่อกู้คืนการเข้าถึงบัญชีของคุณหากอุปกรณ์ของคุณสูญหาย ดังนั้นจึงเป็นสิ่งสำคัญที่จะเลือกรหัส PIN ที่แข็งแกร่งที่สุดเท่าที่จะเป็นไปได้ และเก็บบันทึกที่เชื่อถือได้ของรหัสนั้นไว้



![Image](assets/fr/05.webp)



ยืนยันรหัส PIN นี้อีกครั้ง



![Image](assets/fr/06.webp)



คุณสามารถปรับแต่งโปรไฟล์ผู้ใช้ของคุณได้แล้ว เลือกรูปภาพ ใส่ชื่อหรือชื่อเล่นของคุณ ในขั้นตอนนี้ คุณยังสามารถกำหนดได้ว่าใครสามารถค้นหาคุณบน Signal ผ่านหมายเลขของคุณ เลือก "*Everyone*" หากคุณต้องการให้มองเห็นได้ หรือ "*Nobody*" เพื่อไม่ให้สามารถติดตามได้ผ่านหมายเลขโทรศัพท์ (จากนั้นคุณสามารถถูกเพิ่มได้ด้วย "*Username*" ของคุณเท่านั้น) เมื่อคุณทำการเลือกเสร็จแล้ว คลิกที่ "*Next*".



![Image](assets/fr/07.webp)



คุณเชื่อมต่อกับ Signal แล้วและพร้อมที่จะแลกเปลี่ยนข้อความ



![Image](assets/fr/08.webp)



## การตั้งค่าบัญชี Signal ของคุณ



คลิกที่รูปโปรไฟล์ของคุณที่มุมซ้ายบนเพื่อเข้าถึงการตั้งค่าแอปพลิเคชัน



![Image](assets/fr/09.webp)



เมนู "*Account*" ให้คุณจัดการการตั้งค่าโปรไฟล์ของคุณ ฉันแนะนำให้คุณเก็บการตั้งค่าเริ่มต้นไว้ คุณยังสามารถเปิดใช้งานตัวเลือก "*Registration Lock*" ซึ่งปกป้องบัญชีของคุณจากการโจมตีบางประเภท เมนูนี้ยังมีตัวเลือกที่คุณต้องการเพื่อโอนย้ายบัญชีของคุณไปยังอุปกรณ์ใหม่



![Image](assets/fr/10.webp)



การคลิกที่รูปโปรไฟล์ของคุณในหน้าการตั้งค่าอีกครั้งจะนำคุณไปยังตัวเลือกสำหรับการปรับแต่งโปรไฟล์ของคุณ ฉันแนะนำให้คุณตั้งค่า "*ชื่อผู้ใช้*": ซึ่งจะช่วยให้คุณติดต่อกับผู้อื่นได้โดยไม่ต้องเปิดเผยหมายเลขโทรศัพท์ของคุณ



![Image](assets/fr/11.webp)



โดยการเลือก "*QR code หรือ ลิงก์*" คุณจะได้รับข้อมูลที่คุณต้องการแชร์กับคนที่ต้องการเพิ่มคุณใน Signal



![Image](assets/fr/12.webp)



เมนู "*ความเป็นส่วนตัว*" มีความสำคัญเป็นพิเศษ ที่นี่คุณจะพบตัวเลือกสำหรับควบคุมการมองเห็นหมายเลขของคุณ การจัดการข้อความของคุณกับผู้ติดต่อ รวมถึงการอนุญาตต่างๆ ที่ได้รับบนแอปพลิเคชัน



![Image](assets/fr/13.webp)



และอย่าลังเลที่จะสำรวจเมนู "*Appearance*", "*Chats*" และ "*Notifications*" เพื่อปรับแต่ง Interface และการทำงานของแอปพลิเคชันให้ตรงกับความต้องการส่วนตัวของคุณ



## เชื่อมต่อแอปพลิเคชันเดสก์ท็อป



ในการเชื่อมต่อแอปพลิเคชันเดสก์ท็อป ให้เริ่มต้นด้วยการติดตั้งซอฟต์แวร์บนคอมพิวเตอร์ของคุณ (ดูส่วนแรกของบทแนะนำนี้) จากนั้น บนโทรศัพท์ของคุณ ไปที่การตั้งค่าและเปิดส่วน "*อุปกรณ์ที่เชื่อมโยง*"



![Image](assets/fr/14.webp)



คลิกที่ปุ่ม "*เชื่อมโยงอุปกรณ์ใหม่*"



![Image](assets/fr/15.webp)



บนคอมพิวเตอร์ของคุณ เปิดซอฟต์แวร์ จากนั้นสแกน QR code ที่แสดงบนหน้าจอโดยใช้โทรศัพท์ของคุณ หากคุณต้องการนำเข้าบทสนทนาของคุณ ให้เลือกตัวเลือก "*โอนประวัติข้อความ*"



![Image](assets/fr/16.webp)



อุปกรณ์ของคุณซิงโครไนซ์เรียบร้อยแล้ว



![Image](assets/fr/17.webp)



## การส่งข้อความด้วย Signal



ในการติดต่อกับใครบางคนบน Signal คุณต้องเพิ่มพวกเขาเป็นผู้ติดต่อก่อน มีหลายตัวเลือก: คุณสามารถเพิ่มพวกเขาผ่านหมายเลขโทรศัพท์ของพวกเขา (หากบุคคลนั้นได้เปิดใช้งานตัวเลือกให้สามารถค้นหาได้ด้วยวิธีนี้) หรือใช้ Signal ID ของพวกเขา



คลิกที่ไอคอนรูปดินสอที่มุมล่างขวาของ Interface.



![Image](assets/fr/18.webp)



ในกรณีของฉัน ฉันต้องการเพิ่มบุคคลโดยใช้ชื่อผู้ใช้ ดังนั้นฉันคลิกที่ "*ค้นหาตามชื่อผู้ใช้*"



![Image](assets/fr/19.webp)



คุณสามารถวางข้อมูลการเข้าสู่ระบบหรือสแกน QR code ของมันได้



![Image](assets/fr/20.webp)



ส่งข้อความถึงเขาเพื่อสร้างการติดต่อ



![Image](assets/fr/21.webp)



การสนทนาจะปรากฏบนหน้าแรก จากนั้นหากบุคคลนั้นยอมรับคำขอการติดต่อของคุณ คุณจะเห็นชื่อและรูปโปรไฟล์ของพวกเขา



![Image](assets/fr/22.webp)



ยินดีด้วย ตอนนี้คุณทันสมัยกับการใช้การส่งข้อความ Signal แล้ว ซึ่งเป็นทางเลือกที่ยอดเยี่ยมสำหรับ WathsApp! หากคุณพบว่าบทแนะนำนี้มีประโยชน์ ฉันจะขอบคุณมากถ้าคุณจะให้คะแนนนิ้วโป้งสีเขียวด้านล่าง อย่าลังเลที่จะแชร์บทแนะนำนี้บนเครือข่ายสังคมของคุณ ขอบคุณมาก!
