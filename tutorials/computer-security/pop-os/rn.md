## Imenyekanisha



Pop!OS ni ubuhinga bushingiye kuri Linux bwakozwe na **System76**, ishirahamwe ry’abanyamerika ryitaho cane imashini z’abahinguzi, abahinguzi n’abakoresha bateye imbere.



Pop!OS yagenewe gutanga ibidukikije vy’ubuhinga bwa none, bishikamye, bikora neza cane, itandukanye n’ubumenyi bworoshe, ibikoresho bikomeye n’ukwibanda cane ku gutanga umusaruro.



### Sisitemu76 ni nde?



System76 ni ishirahamwe ry’abanyamerika ryashinzwe mu mwaka w’2005, rikaba rifise icicaro i Denver, ryitaho gukora amakaye, ama desktop n’ama server yagenewe cane cane Linux.



Udakunze abahinguzi ba kera, System76 ikora imashini zigenewe gufunguka, gusanurwa kandi zijanye n’umwidegemvyo wa porogarama.



System76 ntikora mudasobwa gusa.



Iyo sosiyete kandi irategura :




- Pop!OS**, ni ubuhinga bwayo bwite bushingiye kuri Linux;
- COSMIC**, ikibanza ca none, gikora neza cane gikoreshwa na Pop!OS ;
- Gufungura Porogaramu**, porogaramu y'inkomoko yuguruye ishingiye kuri Coreboot ;
- ibikoresho vy’abahinguzi n’abahinguzi.



Intumbero ni ugutanga ubuhinga bwo gukorana n’ibikoresho vy’ubuhinga bwa none, bigereranywa n’ubuhinga bwa Apple, ariko bikaba vyuguruye vyose kandi bishingiye kuri Linux.



## Uburyo bugezweho, buhamye kandi bushikira



Pop!OS yubatswe ku mishinge ya Ubuntu, ikayiha ugushikama kwiza cane, ubushobozi bwo gukorana n’ibikoresho vyinshi n’ugushika ku bidukikije vyinshi vy’ubuhinga.


Itanga interface nziza cane, COSMIC desktop, yakozwe kugira ngo ikore neza, ishobore gukoreshwa neza kandi ishobora guhindurwa, mbere n’abakoresha bashasha.



## Ihitamwo ryiza ku bategura, abahinguzi n'abakoresha basaba



Pop!OS irashimwa cane na :





- abategura (ibikoresho vyashizweho mbere, uburongozi buteye imbere bw'amabuye),
- abakoresha bafise amakarata y'ibishushanyo ya Nvidia canke AMD,
- abanyeshure n’abahinga barondera uburyo bwo kwizigirwa,
- abakoresha amadirisha bipfuza guhindura yoroshe.



Harimwo ivyuma vy’amabuye vyikora, ikibanza gitomoye c’amaporogarama n’ibikoresho vy’umwimbu bihuriweko kugira ngo gukoresha ku musi ku musi vyorohe.



## Ivyiza vya Pop!OS





- Ibikorwa vyiza** bivuye ku guhindurwa kwamaho.
- Verisiyo zibiri za ISO ziraboneka**: zisanzwe n’izitunganijwe na Nvidia.
- Umutekano wongerewe** (ububiko bwa LUKS buraboneka iyo ushizeho).
- Interface COSMIC** ivy’ubuhinga n’ivy’ubuhinga bwa none.
- Ihuye cane** na porogarama ya Ubuntu na Flatpak.



## Kuvanaho POP!OS mu mutekano



### 1. Ibisabwa



Imbere yo gukura no gushiramwo POP!OS, hari ibintu bikeyi ukeneye gukora kugira ngo utegure neza aho uzoshiramwo.



### Ibikoresho bisabwa





- Mudasobwa ihuye**: Igikoresho ca Intel canke AMD, GPU ya Intel / AMD / Nvidia.
- Nibura 4 GB RAM** (8 GB nivyo bikenewe kugira ngo ukoreshe neza).
- 20 GB y’umwanya w’ubuntu ni mutoyi** (40 GB canke n’ibindi birakenewe).
- Urufunguzo rwa USB rwa 4 GB** kugira ngo ureme ubuhinga bwo gushiramwo.



### Ihuriro rya interineti



Ihuzwa rihamye ni ngirakamaro kuri :





- gukuraho ishusho ya ISO,
- gushiramwo ivyagezwe inyuma yo gushiramwo.



Pop!OS ishobora gukora ata n’imwe ihuriyeko, ariko gushiramwo biraryoshe cane kuri Internet.



### Ububiko bw'amakuru



Nimba Pop!OS izosubirizwa canke ibe kumwe n’iyindi sisitemu (Windows, Ubuntu, n’ibindi), ni vyiza ko ukora backup y’amadosiye ahambaye imbere y’uko ukomeza.



### Suzuma ko hariho GPU ya Nvidia (niba ari ngombwa)



Ku mudasobwa zifise ikarita y’ibishushanyo ya Nvidia, uzokenera gukuraho ishusho idasanzwe ya ISO irimwo abashoferi ba Nvidia.


Iyi nkuru yoroshe cane:





- mu kuraba ibisobanuro vya PC,
- canke mu kurondera ivyerekanwa vy’ikarata y’ibishushanyo mu mitunganyirize ya sisitemu.

## Gushiramwo no gukingira Pop!OS



### Gufungura bivuye ku rufunguzo rwa USB





- Zima mudasobwa.
- Nimushiremwo urufunguzo rwa USB (rurimwo Pop!OS).
- Hindukiza mudasobwa yawe. Ku ma PC aherutse gusohoka, iyo sisitemu ikwiye kumenya ubwayo urufunguzo rwo gufungura USB. Niba bitameze gutyo, subira gufungura ufashe urufunguzo rwo gushika kuri BIOS/UEFI (kenshi F2, F12 canke Delete, bivanye n’ibara).
  - Mu BIOS/UEFI, hitamwo urufunguzo rwawe rwa USB nk’igikoresho co gutangura.
  - Bika kandi wongere utangure.



### Gutanguza gushiramwo

### Ibishushanyo vya NVIDIA



Niba warashizeho uhereye kuri Intel/AMD ISO kandi system yawe ifise ikarita y’ibishushanyo ya NVIDIA itandukanye, canke niwaba wongeyeko imwe mu nyuma, uzokenera gushiramwo n’amaboko abashoferi b’ikarita yawe kugira ngo ushobore gukora neza. Gukoresha itegeko rikurikira mu nzira y'itegeko kugira ngo ushiremwo umushoferi:



```bash
sudo apt install system76-driver-nvidia
```



Ushobora kandi gushiramwo abashoferi b'ibishushanyo vya NVIDIA bavuye muri Pop!_Shop.

## Kuvugurura sisitemu



### Ihitamwo rya 1: Biciye ku kigaragaza umukoresha (GUI)



Pop!OS ifise umucungerezi w'ivugurura ry'ibishushanyo vyoroshe, vyumvikana.



1. Fyonda kuri **menu nyamukuru** (icon iri hasi ibubamfu).


2. Gufungura **"Pop!_Iduka "**.


3. Muri Pop!_Shop, fyonda ku **"Ivyashizwe ku rubuga "**.


4. Sisitemu izokwisuzuma ubwayo nimba hariho ibintu bishasha biriho.


5. Fyonda kuri **"Update all "** kugira utangure gushiramwo ivyagezwe.


6. Shiramwo ijambo banga ryawe iyo usabwe.


7. Reka igikorwa kirangire, hanyuma wongere utangure nimba bikenewe.



### Amahitamwo ya 2: Biciye ku kibanza



Gufungura terminal wandike :



```bash
# Mettre à jour la liste des paquets et le système
sudo apt update && sudo apt full-upgrade -y

# Nettoyer les paquets inutiles
sudo apt autoremove -y && sudo apt autoclean
```



### Ubuyobozi bw'abakoresha



Turahimiriza gukoresha konti y'ukoresha isanzwe ifise uburenganzira bwa sudo ku bikorwa vya misi yose.



Kugira ngo ureme umukoresha mushasha :



```bash
sudo adduser votrenom && sudo usermod -aG sudo votrenom
```



Sohoka, hanyuma usubire kwinjira n'uwo mukoresha mushasha.



### Ubuyobozi bw'umushoferi w'ibishushanyo





- Ku makarita ya Nvidia, suzuma ko abashoferi b'abanyagihugu bashizwemwo:



```bash
sudo apt install system76-driver-nvidia
```





- Ku AMD/Intel, abashoferi muri rusangi barashirwamwo ku buryo busanzwe.



### Gukoresha uruhome rw'umuriro (UFW)



Pop!OS ntibuza uruja n'uruza rw'urubuga ku buryo busanzwe. Gukoresha UFW kugira ngo wongere umutekano:



```bash
sudo ufw enable && sudo ufw status verbose
```



### Kugena ivyagezwe vyihuta



Kugira ngo urutonde rugume ruri ku rugero rushasha ataco umuntu akora:



```bash
sudo apt install unattended-upgrades && sudo dpkg-reconfigure --priority=low unattended-upgrades
```



### Guhindura ukuntu usa n'inyifato





- Gufungura **Ivyagezwe vya sisitemu** → **Ibigaragara** kugira uhitemwo insanganyamatsiko y’umuco canke y’umwiza.
- Gutegura imfuruka zikora, amashusho n'ivyo kwagura biciye ku muyobozi wa COSMIC.
- Tunganya imiterere y'ibiro kugira ngo ukore neza urugendo rwawe.



### Gutunganya ububiko bwihuta



Pop!OS ishiramwo ibikoresho nka Deja Dup kugira ngo ukore ububiko:





- Gutanguza **Ivyiyumviro** mu bimenyetso.
- Hitamwo umuduga wo hanze canke ahantu h'urubuga.
- Gutegura ivy’ugusubiza inyuma ubudasiba.



### Gushiramwo ivy'ingirakamaro vya GNOME/COSMIC



Aha hariho ivyungura bikeyi vy'impanuro vyo kwongereza ubumenyi bw'abakoresha:





- Dash to Dock**: umurongo w'ibikorwa wama uboneka.
- GSConnect**: guhuza na Android.
- Ikimenyetso c'ibarabara**: uburongozi bw'ibarabara buteye imbere.



Bishiremwo biciye kuri :



```bash
sudo apt install gnome-shell-extensions
```



Hanyuma uzikoreshe mu ma settings.



### Gutuma ububiko bugenda neza no guhindura ubuyobozi



Suzuma uko uhinduye:



```bash
swapon --show
```



Niba ari ngombwa, wongere ubunini bw'uguhindura canke utunganye dosiye y'uguhindura :



```bash
sudo fallocate -l 4G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab
```



Wongere kuri dosiye `/etc/fstab` kugira ngo uyishireko ubwayo.



## Ubuyobozi bw'amapaki n'ububiko



### Gutahura inkomoko y'amapaki



Pop!OS, ishingiye kuri Ubuntu, ikoresha cane cane :





- Ububiko bwa Ubuntu** buzwi: ku maporogarama menshi ahamye.
- Ububiko bwa System76**: bw’abashoferi, porogarama n’ibikoresho vyihariye.
- Flatpak**: gushika ku bikoresho vyinshi vy’umusenyi.
- Snap** (Ihitamwo): ubundi buryo bw'amapaki y'isi yose.

### Kwongera no gucunga ububiko bwa PPA



Kugira ngo ushiremwo porogarama zigenda zivugururwa, hariho ama PPA (Ububiko bw’Ibintu vy’Umuntu ku giti ciwe) ashobora kwongerwako:



```bash
sudo add-apt-repository ppa:nom/ppa
sudo apt update
```



## Iciyumviro



Pop!OS ni ubuhinga bwa Linux bugezweho, bushikamye, bubereye abatangura n’abateye imbere.



Kubera interface yayo COSMIC n’ibikoresho vyinjijwe, itanga ubumenyi bugenda neza kandi bufise akamaro, haba mu guteza imbere, mu kurema canke mu gukoresha buri musi.



Iyi nyigisho ivuga ku ntambwe nyamukuru: gutegura, gukuraho, gushiramwo, gutegura ivy’intango n’ibikoresho vy’ingenzi.



Ushobora gutohoza Pop!OS kurushirije no guhindura ibidukikije vyawe kugira ngo ubone inyungu nyinshi.
