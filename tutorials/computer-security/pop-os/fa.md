## مقدمه



Pop!OS یک سیستم‌عامل مبتنی بر لینوکس است که توسط **System76**، یک تولیدکننده آمریکایی که در ساخت ماشین‌هایی برای توسعه‌دهندگان، طراحان و کاربران پیشرفته تخصص دارد، توسعه یافته است.



طراحی شده برای ارائه یک محیط مدرن، پایدار و با عملکرد بالا، Pop!OS با تجربه‌ای ساده، ابزارهای قدرتمند و تمرکز قوی بر بهره‌وری متمایز می‌شود.



### ‏System76 کیست؟



سیستم۷۶ یک شرکت آمریکایی است که در سال ۲۰۰۵ تأسیس شده و در دنور مستقر است و در زمینه تولید نوت‌بوک‌ها، دسکتاپ‌ها و سرورهایی که به‌طور خاص برای لینوکس طراحی شده‌اند، تخصص دارد.



برخلاف تولیدکنندگان سنتی، System76 ماشین‌هایی را توسعه می‌دهد که به‌گونه‌ای طراحی شده‌اند که باز، قابل تعمیر و به سمت آزادی نرم‌افزار گرایش دارند.



‏System76 فقط کامپیوتر نمی‌سازد.



این شرکت همچنین توسعه می‌دهد:




- Pop!OS**، سیستم‌عامل مبتنی بر لینوکس خود؛
- COSMIC**، محیط دسکتاپ مدرن و با عملکرد بالا که توسط Pop!OS استفاده می‌شود؛
- Open Firmware**، یک سیستم‌افزار متن‌باز مبتنی بر Coreboot؛
- ابزارهایی برای توسعه‌دهندگان و طراحان.



هدف این است که یکپارچگی سخت‌افزار و نرم‌افزار با کیفیت بالا ارائه شود، که قابل مقایسه با اکوسیستم اپل باشد، اما کاملاً باز و متمرکز بر لینوکس باشد.



## یک سیستم مدرن، پایدار و قابل دسترس



Pop!OS بر پایه‌های Ubuntu ساخته شده است و به آن پایداری عالی، سازگاری گسترده با سخت‌افزار و دسترسی به یک اکوسیستم نرم‌افزاری بزرگ را می‌دهد.


این یک رابط کاربری زیبا، دسکتاپ COSMIC، ارائه می‌دهد که به گونه‌ای طراحی شده است که روان، شهودی و قابل تنظیم باشد، حتی برای کاربران جدید.



## یک انتخاب ایده‌آل برای توسعه‌دهندگان، طراحان و کاربران پرتوقع



Pop!OS به‌طور خاص توسط :





- توسعه‌دهندگان (ابزارهای از پیش نصب‌شده، مدیریت پیشرفته کاشی‌بندی)،
- کاربران با کارت‌های گرافیک Nvidia یا AMD،
- دانشجویان و حرفه‌ای‌هایی که به دنبال یک سیستم قابل اعتماد هستند،
- کاربران ویندوز که مایل به انجام یک انتقال ساده هستند.



این شامل کاشی‌کاری خودکار، یک مرکز نرم‌افزاری واضح و ابزارهای بهره‌وری یکپارچه است تا استفاده روزانه را آسان‌تر کند.



## نکات برجسته Pop!OS





- عملکرد بهینه‌شده** به لطف به‌روزرسانی‌های منظم.
- دو نسخه ISO موجود است**: استاندارد و بهینه‌سازی‌شده برای Nvidia.
- امنیت پیشرفته** (رمزگذاری LUKS در زمان نصب در دسترس است).
- Interface COSMIC** ارگونومیک و مدرن.
- بسیار سازگار** با نرم‌افزار Ubuntu و Flatpak.

## نصب و ایمن‌سازی Pop!OS



### راه‌اندازی از طریق کلید USB





- کامپیوتر را خاموش کنید.
- کلید USB (حاوی Pop!OS) را وصل کنید.
- رایانه خود را روشن کنید. در رایانه‌های شخصی جدید، سیستم باید به‌طور خودکار کلید بوت USB را شناسایی کند. اگر اینطور نیست، با نگه داشتن کلید دسترسی به BIOS/UEFI (معمولاً F2، F12 یا Delete، بسته به برند) دوباره راه‌اندازی کنید.
  - در منوی BIOS/UEFI، کلید USB خود را به عنوان دستگاه بوت انتخاب کنید.
  - ذخیره و راه‌اندازی مجدد.

درایو می‌تواند با استفاده از رمز عبور ورود شما رمزگذاری شود با انتخاب گزینه `رمز عبور رمزگذاری همان رمز عبور حساب کاربری است`. همچنین می‌توانید این گزینه را بردارید و در پایین `تنظیم رمز عبور` را انتخاب کنید. برای نادیده گرفتن فرآیند رمزگذاری دیسک، گزینه `رمزگذاری نکنید` را انتخاب کنید.

### انویدیا گرافیک



اگر از ISO اینتل/ای‌ام‌دی نصب کرده‌اید و سیستم شما دارای کارت گرافیک مجزای انویدیا است، یا اگر در تاریخ بعدی یکی اضافه کرده‌اید، باید به صورت دستی درایورهای کارت خود را نصب کنید تا به عملکرد بهینه دست یابید. برای نصب درایور، فرمان زیر را در یک ترمینال فرمان اجرا کنید:

### نصب از طریق Pop! Shop (رابط گرافیکی)





- باز کردن **Pop!_Shop** از منوی اصلی.
- از نوار جستجو برای پیدا کردن برنامه‌های مورد نظر خود استفاده کنید (برای مثال، "Brave").
- برای هر برنامه روی **نصب** کلیک کنید.
- Pop!_Shop به‌طور خودکار وابستگی‌ها و به‌روزرسانی‌ها را مدیریت می‌کند.



## به‌روزرسانی سیستم



### گزینه 1: از طریق رابط کاربری گرافیکی (GUI)



Pop!OS دارای یک مدیر به‌روزرسانی گرافیکی ساده و شهودی است.



۱. روی **منوی اصلی** (آیکون پایین سمت چپ) کلیک کنید.


2. **"Pop!_Shop "** را باز کنید.


۳. در Pop!_Shop، روی زبانه **"Updates "** کلیک کنید.


۴. سیستم به‌طور خودکار برای به‌روزرسانی‌های موجود بررسی خواهد کرد.


۵. روی **"Update all "** کلیک کنید تا نصب به‌روزرسانی‌ها آغاز شود.


۶. در صورت درخواست، گذرواژه خود را وارد کنید.


۷. بگذارید فرآیند به پایان برسد، سپس در صورت لزوم مجدداً راه‌اندازی کنید.



### گزینه 2: از طریق ترمینال



یک ترمینال باز کنید و تایپ کنید :



```bash
# Mettre à jour la liste des paquets et le système
sudo apt update && sudo apt full-upgrade -y

# Nettoyer les paquets inutiles
sudo apt autoremove -y && sudo apt autoclean
```



### مدیریت کاربران



ما توصیه می‌کنیم از یک حساب کاربری استاندارد با حقوق sudo برای عملیات روزمره استفاده کنید.



برای ایجاد یک کاربر جدید :



```bash
sudo adduser votrenom && sudo usermod -aG sudo votrenom
```



خروج کنید، سپس با این کاربر جدید دوباره وارد شوید.



### مدیریت درایور گرافیک





- برای کارت‌های Nvidia، بررسی کنید که درایورهای اختصاصی نصب شده‌اند:



```bash
sudo apt install system76-driver-nvidia
```





- برای AMD/Intel، درایورها به طور پیش‌فرض گنجانده شده‌اند.



### فایروال (UFW) را فعال کنید



Pop!OS به‌طور پیش‌فرض ترافیک شبکه را مسدود نمی‌کند. برای افزایش امنیت، UFW را فعال کنید:



```bash
sudo ufw enable && sudo ufw status verbose
```



### پیکربندی به‌روزرسانی‌های خودکار



برای به‌روز نگه‌داشتن سیستم بدون دخالت دستی:



```bash
sudo apt install unattended-upgrades && sudo dpkg-reconfigure --priority=low unattended-upgrades
```



### سفارشی‌سازی ظاهر و رفتار





- باز کنید **تنظیمات سیستم** → **ظاهر** تا یک تم روشن یا تاریک انتخاب کنید.
- گوشه‌های فعال، انیمیشن‌ها و افزونه‌ها را از طریق مدیر COSMIC پیکربندی کنید.
- طرح‌بندی دسکتاپ را برای بهینه‌سازی جریان کار خود تنظیم کنید.

Pop!OS ابزارهایی مانند Deja Dup را برای پشتیبان‌گیری ادغام می‌کند:





- راه‌اندازی **Backup** از منو.
- یک درایو خارجی یا یک مکان شبکه را انتخاب کنید.
- برنامه‌ریزی برای پشتیبان‌گیری‌های منظم.



### نصب افزونه‌های مفید GNOME/COSMIC



در اینجا چند افزونه پیشنهادی برای بهبود تجربه کاربری آورده شده است:





- Dash to Dock**: نوار برنامه همیشه قابل مشاهده.
- GSConnect**: همگام‌سازی با اندروید.
- **Clipboard Indicator**: مدیریت پیشرفته کلیپ‌بورد.



آن‌ها را نصب کنید از طریق:



```bash
sudo apt install gnome-shell-extensions
```



سپس آن‌ها را در تنظیمات فعال کنید.



### بهینه‌سازی مدیریت حافظه و تعویض



وضعیت مبادله خود را بررسی کنید:



```bash
swapon --show
```



در صورت لزوم، اندازه swap را افزایش دهید یا یک فایل swap پیکربندی کنید :



```bash
sudo fallocate -l 4G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab
```



آن را به فایل `/etc/fstab` اضافه کنید تا به صورت خودکار مونت شود.



## مدیریت بسته و مخزن



### درک منابع بسته‌بندی



پاپ! او‌اس، مبتنی بر Ubuntu، عمدتاً از :





- مخازن رسمی Ubuntu**: برای پایدارترین نرم‌افزار.
- مخازن System76**: برای درایورها، فریمور و ابزارهای خاص.
- Flatpak**: به طیف گسترده‌ای از برنامه‌های سندباکس شده دسترسی پیدا کنید.
- Snap** (اختیاری): یک فرمت بسته‌بندی جهانی دیگر.



---

### افزودن و مدیریت مخازن PPA



برای نصب نرم‌افزارهایی که به‌طور مکرر به‌روزرسانی می‌شوند، می‌توان برخی از PPAها (آرشیوهای بسته شخصی) را اضافه کرد:



```bash
sudo add-apt-repository ppa:nom/ppa
sudo apt update
```



## نتیجه‌گیری



Pop!OS یک توزیع لینوکس مدرن و پایدار است که برای مبتدیان و کاربران پیشرفته مناسب است.



با تشکر از رابط کاربری شهودی COSMIC و ابزارهای یکپارچه آن، تجربه‌ای روان و پربار را ارائه می‌دهد، چه برای توسعه، خلق یا استفاده روزمره.



این آموزش مراحل کلیدی را پوشش می‌دهد: آماده‌سازی، دانلود، نصب، تنظیمات اولیه و ابزارهای ضروری.



در استفاده از Pop!OS برای کاوش بیشتر و سفارشی‌سازی محیط خود آزاد باشید تا بیشترین بهره را از آن ببرید.
