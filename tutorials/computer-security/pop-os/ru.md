## Введение



Pop!OS - это операционная система на базе Linux, разработанная **System76**, американским производителем, специализирующимся на машинах для разработчиков, дизайнеров и продвинутых пользователей.



Разработанная для создания современной, стабильной и высокопроизводительной среды, Pop!OS отличается простотой использования, мощными инструментами и сильной ориентацией на производительность.



### Кто такой System76?



System76 - американская компания, основанная в 2005 году и базирующаяся в Денвере, специализирующаяся на производстве ноутбуков, настольных компьютеров и серверов, разработанных специально для Linux.



В отличие от традиционных производителей, System76 разрабатывает открытые, ремонтопригодные машины, ориентированные на свободу программного обеспечения.



System76 не просто производит компьютеры.



Компания также разрабатывает :




- Pop!OS**, собственная операционная система на базе Linux;
- COSMIC**, современная, высокопроизводительная среда рабочего стола, используемая в Pop!OS;
- Open Firmware**, прошивка с открытым исходным кодом, основанная на Coreboot;
- инструменты для разработчиков и дизайнеров.



Цель - предложить высококачественную интеграцию аппаратного и программного обеспечения, сравнимую с экосистемой Apple, но полностью открытую и основанную на Linux.



## Современная, стабильная и доступная система



Pop!OS опирается на фундамент Ubuntu, обеспечивая ему отличную стабильность, широкую совместимость с оборудованием и доступ к огромной экосистеме программного обеспечения.


Он предоставляет элегантный интерфейс - рабочий стол COSMIC, разработанный для того, чтобы быть плавным, интуитивно понятным и настраиваемым даже для новых пользователей.



## Идеальный выбор для разработчиков, дизайнеров и требовательных пользователей



Pop!OS особенно ценится :





- разработчиков (предустановленные инструменты, расширенное управление плиткой),
- пользователи с видеокартами Nvidia или AMD,
- студенты и профессионалы, которым нужна надежная система,
- пользователи windows, желающие осуществить простой переход.



Он включает в себя автоматическую укладку плитки, понятный программный центр и интегрированные инструменты повышения производительности, облегчающие ежедневную работу.



## Основные моменты Pop!OS





- Оптимизированная производительность** благодаря регулярным обновлениям.
- Доступны две версии ISO**: стандартная и оптимизированная для Nvidia.
- Повышенная безопасность** (шифрование LUKS доступно при установке).
- Interface COSMIC** эргономичный и современный.
- Высокая совместимость** с программным обеспечением Ubuntu и Flatpak.



## Скачать POP!OS безопасно



### 1. Пререквизиты



Перед загрузкой и установкой POP!OS необходимо выполнить несколько действий, чтобы правильно подготовить среду установки.



### Необходимое оборудование





- Совместимый компьютер**: Процессор Intel или AMD, графический процессор Intel / AMD / Nvidia.
- Не менее 4 ГБ оперативной памяти** (для комфортного использования рекомендуется 8 ГБ).
- минимум 20 ГБ свободного места** (рекомендуется 40 ГБ или более).
- USB-носитель** объемом не менее 4 ГБ для создания установочного носителя.



### Подключение к Интернету



Стабильное соединение полезно для :





- загрузите образ ISO,
- устанавливайте обновления после установки.



Pop!OS может работать и без подключения, но установка через Интернет проходит гораздо более гладко.



### Резервное копирование данных



Если Pop!OS будет заменять или сосуществовать с другой системой (Windows, Ubuntu и т.д.), рекомендуется создать резервную копию важных файлов перед началом работы.



### Проверьте наличие графического процессора Nvidia (если применимо)



Для компьютеров с видеокартой Nvidia необходимо загрузить специальный ISO-образ, содержащий драйверы Nvidia.


Эта проверка очень проста:





- обратитесь к спецификации ПК,
- или узнать модель видеокарты в системных настройках.



### Скачать с официального сайта



ISO-образ Pop!OS следует загружать непосредственно с официальной страницы System76: [system76.com/pop](https://system76.com/pop/).



На этой странице всегда предлагается самая последняя версия, адаптированная к вашему оборудованию.



![capture](assets/fr/03.webp)



### Выберите версию: Стандартная или Nvidia, или Raspberry Pi (ARM64)



Pop!OS доступен в трех вариантах:



### Стандартная версия



Рекомендуется для компьютеров, использующих :





- процессор intel или AMD;
- интегрированный графический процессор Intel или AMD;
- графическая карта AMD Radeon.



![Utilisation de Balena Etcher](assets/fr/04.webp)



### Версия Nvidia



Рекомендуется для компьютеров, оснащенных видеокартами Nvidia.

### Версия Raspberry Pi (ARM64)



Для Raspberry Pi 4 и 400 (процессор ARM).


Адаптированная к архитектуре ARM, эта версия специально предназначена для этих мини-компьютеров.

## Установка и защита Pop!OS



### Загрузка с USB-носителя





- Выключите компьютер.
- Подключите USB-носитель (содержащий Pop!OS).
- Включите компьютер. На последних моделях ПК система должна автоматически распознать загрузочный USB-носитель. Если это не так, перезагрузитесь, удерживая клавишу доступа к BIOS/UEFI (обычно F2, F12 или Delete, в зависимости от марки).
  - В меню BIOS/UEFI выберите USB-носитель в качестве загрузочного устройства.
  - Сохраните и перезапустите.



### Запуск установки



После выбора загрузочного USB-носителя в качестве устройства запуска ваш компьютер загрузится в среду Pop!OS Live.



Выберите язык.



![Capture](assets/fr/10.webp)



Выберите местоположение.



![Capture](assets/fr/11.webp)



Выберите язык для ввода с клавиатуры.



![Capture](assets/fr/12.webp)



Выберите раскладку клавиатуры.



![Capture](assets/fr/13.webp)



Выберите опцию `Чистая установка` для стандартной установки. Это лучший вариант для начинающих пользователей Linux, но имейте в виду, что при этом будет удалено все содержимое целевого диска. В качестве альтернативы вы можете выбрать `Попробовать демо-режим`, чтобы продолжить тестирование Pop!OS в реальной среде.



![Capture](assets/fr/14.webp)



Выберите `Настройка (Дополнительно)`, чтобы перейти к GParted. Этот инструмент позволяет настроить дополнительные функции, такие как двойная загрузка, создание отдельного раздела `/home` или размещение раздела `/tmp` на другом диске.



Нажмите `Стереть и установить`, чтобы установить Pop!OS на выбранный диск.



![Capture](assets/fr/15.webp)



### Конфигурация учетной записи пользователя



В следующем разделе программы установки вам будет предложено создать учетную запись пользователя, чтобы вы могли войти в новую операционную систему.



Укажите полное имя (оно может содержать любой текст по вашему выбору, в верхнем или нижнем регистре), а также имя пользователя (оно должно быть в нижнем регистре):

### Полное шифрование диска



Шифрование системного диска не обязательно, но оно гарантирует безопасность пользовательских данных в случае, если кто-то получит несанкционированный физический доступ к устройству.



Диск можно зашифровать с помощью пароля для входа в систему, установив флажок `Пароль шифрования совпадает с паролем учетной записи пользователя`. Вы также можете снять этот флажок и выбрать `Set Password` внизу. Выберите `Не шифровать`, чтобы проигнорировать процесс шифрования диска.



![Capture](assets/fr/18.webp)



Если вы выбрали кнопку `Установить пароль`, появится дополнительное предложение задать пароль шифрования.



Перейдите к следующему шагу программы установки. Pop!OS начнет установку на диск.



![Capture](assets/fr/19.webp)



После завершения установки перезагрузите компьютер и войдите в систему, чтобы завершить процесс настройки учетной записи пользователя.



Если вы изменили порядок загрузки, чтобы при запуске приоритет отдавался Live USB-накопителю, полностью выключите компьютер и извлеките установочный USB-накопитель. Если вы находитесь в режиме двойной загрузки, нажмите соответствующие клавиши, чтобы получить доступ к конфигурации и выбрать диск, на котором установлена Pop!OS.



![Capture](assets/fr/20.webp)



### Графика NVIDIA



Если вы установили систему из Intel/AMD ISO, а в вашей системе есть дискретная видеокарта NVIDIA, или если вы добавили ее позже, вам нужно вручную установить драйверы для вашей карты, чтобы достичь оптимальной производительности. Выполните следующую команду в командном терминале, чтобы установить драйвер:



```bash
sudo apt install system76-driver-nvidia
```



Вы также можете установить графические драйверы NVIDIA из Pop!_Shop.



![Capture](assets/fr/20.webp)

## Обновление системы



### Вариант 1: Через графический интерфейс пользователя (GUI)



Pop!OS имеет простой и интуитивно понятный графический менеджер обновлений.



1. Нажмите на **главное меню** (значок внизу слева).


2. Откройте **"Pop!_Shop "**.


3. В магазине Pop!_Shop перейдите на вкладку **"Обновления "**.


4. Система автоматически проверит наличие доступных обновлений.


5. Нажмите на **"Обновить все "**, чтобы начать установку обновлений.


6. Введите пароль, если потребуется.


7. Дайте процессу завершиться, затем при необходимости перезапустите его.



### Вариант 2: Через терминал



Откройте терминал и введите :



```bash
# Mettre à jour la liste des paquets et le système
sudo apt update && sudo apt full-upgrade -y

# Nettoyer les paquets inutiles
sudo apt autoremove -y && sudo apt autoclean
```



### Управление пользователями



Мы рекомендуем использовать стандартную учетную запись пользователя с правами sudo для повседневных операций.



Чтобы создать нового пользователя :



```bash
sudo adduser votrenom && sudo usermod -aG sudo votrenom
```



Выйдите из системы, а затем снова войдите в нее под этим новым пользователем.



### Управление графическими драйверами





- Для карт Nvidia проверьте, установлены ли фирменные драйверы:



```bash
sudo apt install system76-driver-nvidia
```





- Для AMD/Intel драйверы обычно включены по умолчанию.



### Активируйте брандмауэр (UFW)



Pop!OS не блокирует сетевой трафик по умолчанию. Активируйте UFW для повышения безопасности:



```bash
sudo ufw enable && sudo ufw status verbose
```



### Настройка автоматических обновлений



Чтобы поддерживать систему в актуальном состоянии без ручного вмешательства:



```bash
sudo apt install unattended-upgrades && sudo dpkg-reconfigure --priority=low unattended-upgrades
```



### Настройте внешний вид и поведение





- Откройте **Настройки системы** → **Внешний вид**, чтобы выбрать светлую или темную тему.
- Настройте активные углы, анимацию и расширения с помощью менеджера COSMIC.
- Настройте макет рабочего стола, чтобы оптимизировать рабочий процесс.



### Настройка автоматического резервного копирования



В Pop!OS интегрированы такие инструменты, как Deja Dup для резервного копирования:





- Запустите **Backup** из меню.
- Выберите внешний диск или сетевое расположение.
- Планируйте регулярное резервное копирование.



### Установка полезных расширений GNOME/COSMIC



Вот несколько рекомендуемых расширений для повышения удобства работы:





- Dash to Dock**: панель приложений всегда видна.
- GSConnect**: синхронизация с Android.
- Индикатор буфера обмена**: расширенное управление буфером обмена.



Установите их с помощью :



```bash
sudo apt install gnome-shell-extensions
```



Затем активируйте их в настройках.



### Оптимизация управления памятью и подкачкой



Проверьте состояние своего свопа:



```bash
swapon --show
```



При необходимости увеличьте размер файла подкачки или настройте файл подкачки:



```bash
sudo fallocate -l 4G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab
```



Добавьте его в файл `/etc/fstab` для автоматического монтирования.



## Управление пакетами и репозиториями



### Понимание источников пакетов



Pop!OS, основанная на Ubuntu, использует в основном :





- Официальные репозитории Ubuntu**: для наиболее стабильного программного обеспечения.
- Репозитории System76**: для драйверов, микропрограмм и специальных инструментов.
- Flatpak**: доступ к широкому спектру приложений в песочнице.
- Snap** (необязательно): еще один универсальный формат пакетов.



---

### Добавление и управление репозиториями PPA



Чтобы установить часто обновляемое программное обеспечение, можно добавить определенные PPA (Personal Package Archives):



```bash
sudo add-apt-repository ppa:nom/ppa
sudo apt update
```
