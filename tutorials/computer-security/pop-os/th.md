## บทนำ



Pop!OS เป็นระบบปฏิบัติการที่ใช้ Linux ซึ่งพัฒนาโดย **System76** ผู้ผลิตจากอเมริกาที่เชี่ยวชาญในการผลิตเครื่องสำหรับนักพัฒนา นักออกแบบ และผู้ใช้ขั้นสูง



ออกแบบมาเพื่อมอบสภาพแวดล้อมที่ทันสมัย มั่นคง และมีประสิทธิภาพสูง Pop!OS โดดเด่นด้วยประสบการณ์ที่เรียบง่าย เครื่องมือที่ทรงพลัง และการมุ่งเน้นที่ประสิทธิภาพในการทำงาน



### System76 คือใคร?



System76 เป็นบริษัทอเมริกันที่ก่อตั้งขึ้นในปี 2005 และมีฐานอยู่ในเดนเวอร์ เชี่ยวชาญในการผลิตโน้ตบุ๊ก เดสก์ท็อป และเซิร์ฟเวอร์ที่ออกแบบมาโดยเฉพาะสำหรับลินุกซ์



ไม่เหมือนกับผู้ผลิตแบบดั้งเดิม System76 พัฒนาระบบที่ออกแบบมาให้เปิดกว้าง ซ่อมแซมได้ และมุ่งเน้นไปที่เสรีภาพของซอฟต์แวร์



System76 ไม่ได้แค่ผลิตคอมพิวเตอร์เท่านั้น



บริษัทยังพัฒนา :




- Pop!OS**, ระบบปฏิบัติการที่ใช้ Linux ของตัวเอง;
- COSMIC** สภาพแวดล้อมเดสก์ท็อปที่ทันสมัยและมีประสิทธิภาพสูงที่ใช้โดย Pop!OS ;
- Open Firmware**, เฟิร์มแวร์โอเพ่นซอร์สที่พัฒนาบนพื้นฐานของ Coreboot ;
- เครื่องมือสำหรับนักพัฒนาและนักออกแบบ



เป้าหมายคือการนำเสนอการผสานรวมฮาร์ดแวร์และซอฟต์แวร์คุณภาพสูง ซึ่งเปรียบได้กับระบบนิเวศของ Apple แต่เปิดกว้างอย่างสมบูรณ์และมุ่งเน้นที่ Linux



## ระบบที่ทันสมัย มั่นคง และเข้าถึงได้



Pop!OS สร้างขึ้นบนพื้นฐานของ Ubuntu ทำให้มีความเสถียรที่ยอดเยี่ยม ความเข้ากันได้กับฮาร์ดแวร์ที่หลากหลาย และการเข้าถึงระบบนิเวศซอฟต์แวร์ขนาดใหญ่


มันมีอินเทอร์เฟซที่หรูหรา เดสก์ท็อป COSMIC ที่ออกแบบมาให้ลื่นไหล ใช้งานง่าย และปรับแต่งได้ แม้กระทั่งสำหรับผู้ใช้ใหม่



## ตัวเลือกที่เหมาะสำหรับนักพัฒนา นักออกแบบ และผู้ใช้ที่ต้องการ



Pop!OS ได้รับความนิยมเป็นพิเศษจาก :





- นักพัฒนา (เครื่องมือที่ติดตั้งล่วงหน้า, การจัดการการจัดวางขั้นสูง),
- ผู้ใช้ที่มีการ์ดจอ Nvidia หรือ AMD,
- นักเรียนและมืออาชีพที่กำลังมองหาระบบที่เชื่อถือได้,
- ผู้ใช้ Windows ที่ต้องการเปลี่ยนแปลงอย่างง่ายดาย



ประกอบด้วยการจัดเรียงอัตโนมัติ ศูนย์ซอฟต์แวร์ที่ชัดเจน และเครื่องมือเพิ่มประสิทธิภาพที่รวมเข้าด้วยกันเพื่อให้ง่ายต่อการใช้งานในชีวิตประจำวัน



## ไฮไลท์ของ Pop!OS





- ประสิทธิภาพที่เพิ่มขึ้น** ขอบคุณการอัปเดตอย่างสม่ำเสมอ
- มีสองเวอร์ชัน ISO ให้เลือก**: มาตรฐานและปรับแต่งสำหรับ Nvidia
- การรักษาความปลอดภัยที่เพิ่มขึ้น** (การเข้ารหัส LUKS มีให้ในการติดตั้ง)
- Interface COSMIC** ออกแบบตามหลักสรีรศาสตร์และทันสมัย
- **เข้ากันได้อย่างสูง** กับซอฟต์แวร์ Ubuntu และ Flatpak



## ดาวน์โหลด POP!OS อย่างปลอดภัย



### 1. ข้อกำหนดเบื้องต้น



ก่อนดาวน์โหลดและติดตั้ง POP!OS มีบางสิ่งที่คุณต้องทำเพื่อเตรียมสภาพแวดล้อมการติดตั้งให้ถูกต้อง



### อุปกรณ์ที่จำเป็น





- คอมพิวเตอร์ที่เข้ากันได้**: โปรเซสเซอร์ Intel หรือ AMD, GPU Intel / AMD / Nvidia.
- RAM อย่างน้อย 4 GB** (แนะนำ 8 GB เพื่อการใช้งานที่สะดวกสบาย).
- พื้นที่ว่างขั้นต่ำ 20 GB** (แนะนำให้มี 40 GB หรือมากกว่า)
- ต้องใช้ USB key ขนาดขั้นต่ำ 4 GB** เพื่อสร้างสื่อการติดตั้ง



### การเชื่อมต่ออินเทอร์เน็ต



การเชื่อมต่อที่เสถียรมีประโยชน์สำหรับ :





- ดาวน์โหลดไฟล์ ISO,
- ติดตั้งการอัปเดตหลังการติดตั้ง



Pop!OS สามารถทำงานได้โดยไม่ต้องเชื่อมต่ออินเทอร์เน็ต แต่การติดตั้งจะราบรื่นกว่าหากเชื่อมต่ออินเทอร์เน็ต



### การสำรองข้อมูล



หาก Pop!OS จะถูกแทนที่หรือใช้งานร่วมกับระบบอื่น (Windows, Ubuntu, ฯลฯ) ขอแนะนำให้สำรองไฟล์สำคัญก่อนดำเนินการ



### ตรวจสอบการมีอยู่ของ Nvidia GPU (ถ้ามี)



สำหรับคอมพิวเตอร์ที่ติดตั้งการ์ดจอ Nvidia คุณจะต้องดาวน์โหลดไฟล์ ISO พิเศษที่รวมไดรเวอร์ Nvidia ไว้ด้วย


การตรวจสอบนี้ง่ายมาก:





- โดยการตรวจสอบสเปคของ PC,
- หรือโดยการค้นหารุ่นการ์ดกราฟิกในการตั้งค่าระบบ



### ดาวน์โหลดจากเว็บไซต์ทางการ



ไฟล์ภาพ ISO ของ Pop!OS ควรดาวน์โหลดโดยตรงจากหน้าทางการของ System76: [system76.com/pop](https://system76.com/pop/)



หน้านี้มักจะมีเวอร์ชันล่าสุดที่ปรับให้เข้ากับฮาร์ดแวร์ของคุณเสมอ

### เลือกเวอร์ชัน: Standard หรือ Nvidia หรือ Raspberry Pi (ARM64)



Pop!OS มีให้เลือกสามรูปแบบ:



### เวอร์ชันมาตรฐาน



แนะนำสำหรับคอมพิวเตอร์ที่ใช้ :





- intel หรือ AMD processor;
- กราฟิกการ์ดแบบรวมของ Intel หรือ AMD;
- การ์ดจอ AMD Radeon



![Utilisation de Balena Etcher](assets/fr/04.webp)



### เวอร์ชันของ Nvidia



แนะนำสำหรับคอมพิวเตอร์ที่ติดตั้งการ์ดจอ Nvidia


รูปภาพนี้มีไดรเวอร์ Nvidia อยู่แล้ว ทำให้การติดตั้งง่ายขึ้นและลดปัญหากราฟิก

## การติดตั้งและการรักษาความปลอดภัย Pop!OS



### การบูตจาก USB key





- ปิดคอมพิวเตอร์
- เสียบกุญแจ USB (ที่มี Pop!OS)
- เปิดคอมพิวเตอร์ของคุณ บนพีซีรุ่นใหม่ ระบบควรจะรู้จัก USB boot key โดยอัตโนมัติ หากไม่เป็นเช่นนั้น ให้รีบูตโดยกดปุ่มเข้าถึง BIOS/UEFI ค้างไว้ (ปกติคือ F2, F12 หรือ Delete ขึ้นอยู่กับยี่ห้อ)
  - ในเมนู BIOS/UEFI ให้เลือก USB key ของคุณเป็นอุปกรณ์บูต
  - บันทึกและเริ่มใหม่อีกครั้ง

### การเข้ารหัสดิสก์ทั้งหมด



การเข้ารหัสดิสก์ระบบไม่จำเป็นต้องทำ แต่จะรับประกันความปลอดภัยของข้อมูลผู้ใช้ในกรณีที่มีผู้เข้าถึงอุปกรณ์โดยไม่ได้รับอนุญาตทางกายภาพ



ไดรฟ์สามารถเข้ารหัสโดยใช้รหัสผ่านเข้าสู่ระบบของคุณได้โดยการเลือก `Encryption password is the same as user account password` คุณยังสามารถยกเลิกการเลือกช่องนี้และเลือก `Set Password` ที่ด้านล่าง เลือก `Don't Encrypt` เพื่อไม่สนใจขั้นตอนการเข้ารหัสดิสก์



![Capture](assets/fr/18.webp)



หากคุณเลือกปุ่ม `Set Password` คุณจะเห็นข้อความเพิ่มเติมเพื่อกำหนดรหัสผ่านการเข้ารหัสของคุณ



ดำเนินการไปยังขั้นตอนถัดไปในโปรแกรมการติดตั้ง Pop!OS จะเริ่มการติดตั้งบนดิสก์แล้ว



![Capture](assets/fr/19.webp)



เมื่อติดตั้งเสร็จสิ้น ให้รีสตาร์ทคอมพิวเตอร์ของคุณและเข้าสู่ระบบเพื่อดำเนินการตั้งค่าบัญชีผู้ใช้ให้เสร็จสมบูรณ์



หากคุณได้เปลี่ยนลำดับการบูตเพื่อให้ความสำคัญกับ Live USB key ของคุณในขณะเริ่มต้น ให้ปิดคอมพิวเตอร์อย่างสมบูรณ์และถอด USB key การติดตั้งออก หากคุณอยู่ในโหมด dual-boot ให้กดปุ่มที่เหมาะสมเพื่อเข้าถึงการกำหนดค่าและเลือกไดรฟ์ที่มีการติดตั้ง Pop!OS

### กราฟิก NVIDIA



หากคุณได้ติดตั้งจาก Intel/AMD ISO และระบบของคุณมีการ์ดกราฟิก NVIDIA แยกต่างหาก หรือหากคุณได้เพิ่มการ์ดกราฟิกในภายหลัง คุณจะต้องติดตั้งไดรเวอร์สำหรับการ์ดของคุณด้วยตนเองเพื่อให้ได้ประสิทธิภาพสูงสุด รันคำสั่งต่อไปนี้ในเทอร์มินัลคำสั่งเพื่อติดตั้งไดรเวอร์:



```bash
sudo apt install system76-driver-nvidia
```



คุณยังสามารถติดตั้งไดรเวอร์กราฟิก NVIDIA จาก Pop!_Shop ได้เช่นกัน



![Capture](assets/fr/20.webp)



## การติดตั้งเครื่องมือที่จำเป็น



Pop!OS มีซอฟต์แวร์หลากหลายให้เลือกใช้ผ่านทาง Pop!Shop แต่เครื่องมือที่จำเป็นหลายอย่างก็สามารถติดตั้งได้ผ่านทาง terminal ด้วย `apt` หรือ `flatpak` นี่คือวิธีการติดตั้งเครื่องมือสำคัญสำหรับสภาพแวดล้อมการทำงานที่ครบถ้วน

### การติดตั้งผ่าน Pop! Shop (อินเทอร์เฟซแบบกราฟิก)





- เปิด **Pop!_Shop** จากเมนูหลัก
- ใช้แถบค้นหาเพื่อค้นหาแอปพลิเคชันที่คุณต้องการ (เช่น "Brave")
- คลิก **ติดตั้ง** สำหรับแต่ละแอปพลิเคชัน
- Pop!_Shop จัดการการพึ่งพาและการอัปเดตโดยอัตโนมัติ



## อัปเดตระบบ



### ตัวเลือกที่ 1: ผ่านทางส่วนติดต่อผู้ใช้แบบกราฟิก (GUI)



Pop!OS มีตัวจัดการอัปเดตกราฟิกที่เรียบง่ายและใช้งานง่าย



1. คลิกที่ **เมนูหลัก** (ไอคอนด้านล่างซ้าย)


2. เปิด **"Pop!_Shop "**.


3. ใน Pop!_Shop คลิกที่แท็บ **"Updates "**


4. ระบบจะตรวจสอบการอัปเดตที่มีอยู่โดยอัตโนมัติ


5. คลิกที่ **"อัปเดตทั้งหมด "** เพื่อเริ่มการติดตั้งการอัปเดต


6. ป้อนรหัสผ่านของคุณหากมีการร้องขอ


7. ปล่อยให้กระบวนการเสร็จสิ้น จากนั้นรีสตาร์ทหากจำเป็น



### ตัวเลือกที่ 2: ผ่านทางเทอร์มินัล



เปิดเทอร์มินัลและพิมพ์ :



```bash
# Mettre à jour la liste des paquets et le système
sudo apt update && sudo apt full-upgrade -y

# Nettoyer les paquets inutiles
sudo apt autoremove -y && sudo apt autoclean
```



### การจัดการผู้ใช้



เราแนะนำให้ใช้บัญชีผู้ใช้มาตรฐานที่มีสิทธิ์ sudo สำหรับการดำเนินงานในแต่ละวัน



ในการสร้างผู้ใช้ใหม่ :



```bash
sudo adduser votrenom && sudo usermod -aG sudo votrenom
```



ออกจากระบบ แล้วเข้าสู่ระบบอีกครั้งด้วยผู้ใช้ใหม่นี้



### การจัดการไดรเวอร์กราฟิก

```bash
sudo apt install system76-driver-nvidia
```





- สำหรับ AMD/Intel ไดรเวอร์มักจะถูกรวมไว้โดยค่าเริ่มต้น



### เปิดใช้งานไฟร์วอลล์ (UFW)



Pop!OS ไม่บล็อกการรับส่งข้อมูลเครือข่ายโดยค่าเริ่มต้น เปิดใช้งาน UFW เพื่อเพิ่มความปลอดภัย:



```bash
sudo ufw enable && sudo ufw status verbose
```



### กำหนดค่าการอัปเดตอัตโนมัติ



เพื่อให้ระบบทันสมัยโดยไม่ต้องมีการแทรกแซงด้วยตนเอง:



```bash
sudo apt install unattended-upgrades && sudo dpkg-reconfigure --priority=low unattended-upgrades
```



### ปรับแต่งลักษณะและพฤติกรรม





- เปิด **การตั้งค่าระบบ** → **ลักษณะที่ปรากฏ** เพื่อเลือกธีมแบบสว่างหรือมืด
- กำหนดค่ามุมที่ใช้งานอยู่, แอนิเมชัน และส่วนขยายผ่านตัวจัดการ COSMIC
- ปรับแต่งเลย์เอาต์เดสก์ท็อปเพื่อเพิ่มประสิทธิภาพการทำงานของคุณ



### กำหนดค่าการสำรองข้อมูลอัตโนมัติ



Pop!OS รวมเครื่องมือต่างๆ เช่น Deja Dup สำหรับการสำรองข้อมูล:





- เปิด **สำรองข้อมูล** จากเมนู
- เลือกไดรฟ์ภายนอกหรือที่ตั้งเครือข่าย
- กำหนดเวลาการสำรองข้อมูลเป็นประจำ



### การติดตั้งส่วนขยาย GNOME/COSMIC ที่มีประโยชน์



นี่คือส่วนขยายที่แนะนำเพื่อเพิ่มประสบการณ์ของผู้ใช้:





- Dash to Dock**: แถบแอปพลิเคชันที่มองเห็นได้ตลอดเวลา
- GSConnect**: การซิงโครไนซ์กับ Android.
- Clipboard Indicator**: การจัดการคลิปบอร์ดขั้นสูง



ติดตั้งผ่าน:



```bash
sudo apt install gnome-shell-extensions
```



จากนั้นเปิดใช้งานในการตั้งค่า



### การเพิ่มประสิทธิภาพการจัดการหน่วยความจำและการสลับ



ตรวจสอบสถานะการแลกเปลี่ยนของคุณ:



```bash
swapon --show
```



หากจำเป็น ให้เพิ่มขนาด swap หรือกำหนดค่าไฟล์ swap :



```bash
sudo fallocate -l 4G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab
```



เพิ่มลงในไฟล์ `/etc/fstab` เพื่อการเมานต์อัตโนมัติ



## การจัดการแพ็กเกจและที่เก็บข้อมูล



### ทำความเข้าใจแหล่งที่มาของแพ็คเกจ



Pop!OS, based on Ubuntu, uses mainly :





- ที่เก็บข้อมูล Ubuntu** อย่างเป็นทางการ: สำหรับซอฟต์แวร์ที่เสถียรที่สุด
- ที่เก็บข้อมูล System76**: สำหรับไดรเวอร์ เฟิร์มแวร์ และเครื่องมือเฉพาะ
- Flatpak**: เข้าถึงแอปพลิเคชันที่หลากหลายภายใต้สภาพแวดล้อมที่แยกออกจากกัน
- Snap** (optional): รูปแบบแพ็คเกจสากลอีกแบบหนึ่ง



---

### เพิ่มและจัดการที่เก็บ PPA



ในการติดตั้งซอฟต์แวร์ที่มีการอัปเดตบ่อยครั้ง สามารถเพิ่ม PPA (Personal Package Archives) บางรายการได้:



```bash
sudo add-apt-repository ppa:nom/ppa
sudo apt update
```



## บทสรุป



Pop!OS เป็นระบบปฏิบัติการลินุกซ์ที่ทันสมัยและเสถียร เหมาะสำหรับผู้เริ่มต้นและผู้ใช้ขั้นสูง



ด้วยอินเทอร์เฟซ COSMIC ที่ใช้งานง่ายและเครื่องมือที่ผสานรวมกัน มันมอบประสบการณ์ที่ราบรื่นและมีประสิทธิภาพ ไม่ว่าจะเป็นสำหรับการพัฒนา การสร้างสรรค์ หรือการใช้งานในชีวิตประจำวัน



บทแนะนำนี้ครอบคลุมขั้นตอนสำคัญ: การเตรียมการ, การดาวน์โหลด, การติดตั้ง, การตั้งค่าเริ่มต้น และเครื่องมือที่จำเป็น



รู้สึกอิสระที่จะสำรวจ Pop!OS เพิ่มเติมและปรับแต่งสภาพแวดล้อมของคุณเพื่อให้ได้ประโยชน์สูงสุดจากมัน
