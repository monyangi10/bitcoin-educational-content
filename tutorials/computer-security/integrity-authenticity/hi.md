![cover](assets/cover.webp)


सॉफ़्टवेयर डाउनलोड करते समय, यह सुनिश्चित करना बहुत महत्वपूर्ण है कि इसमें कोई बदलाव नहीं किया गया है और यह वास्तव में आधिकारिक स्रोत से आता है। यह Bitcoin से संबंधित सॉफ़्टवेयर के लिए विशेष रूप से सच है, जैसे कि Wallet सॉफ़्टवेयर, जो आपको अपने फंड तक पहुँच प्रदान करने वाली कुंजियों को सुरक्षित करने की अनुमति देता है। इस ट्यूटोरियल में, हम देखेंगे कि इसे इंस्टॉल करने से पहले सॉफ़्टवेयर की अखंडता और प्रामाणिकता को कैसे सत्यापित किया जाए। हम एक उदाहरण के रूप में स्पैरो Wallet का उपयोग करेंगे, जो बिटकॉइनर्स के बीच एक पसंदीदा Wallet सॉफ़्टवेयर है, लेकिन प्रक्रिया किसी भी अन्य सॉफ़्टवेयर के लिए समान होगी।


अखंडता की पुष्टि करने में यह सुनिश्चित करना शामिल है कि डाउनलोड की गई फ़ाइल को उसके डिजिटल फ़िंगरप्रिंट (यानी, उसके Hash) की तुलना आधिकारिक डेवलपर द्वारा प्रदान किए गए फ़िंगरप्रिंट से करके संशोधित नहीं किया गया है। यदि दोनों मेल खाते हैं, तो इसका मतलब है कि फ़ाइल मूल के समान है और किसी हमलावर द्वारा दूषित या संशोधित नहीं की गई है।


दूसरी ओर, प्रामाणिकता सत्यापित करने से यह सुनिश्चित होता है कि फ़ाइल वास्तव में आधिकारिक डेवलपर से आई है, न कि किसी धोखेबाज़ से। यह डिजिटल हस्ताक्षर को सत्यापित करके किया जाता है। यह हस्ताक्षर साबित करता है कि सॉफ़्टवेयर वैध डेवलपर की निजी कुंजी के साथ हस्ताक्षरित किया गया था।


यदि ये जाँच नहीं की जाती हैं, तो मैलवेयर इंस्टॉल होने का जोखिम होता है जिसमें संशोधित कोड हो सकता है। यह कोड या तो आपकी निजी कुंजी जैसी जानकारी चुरा सकता है या आपकी फ़ाइलों तक पहुँच को अवरुद्ध कर सकता है। इस प्रकार का हमला काफी आम है, खासकर ओपन-सोर्स सॉफ़्टवेयर के संदर्भ में जहाँ नकली संस्करण वितरित किए जा सकते हैं।


इस सत्यापन के लिए, हम दो उपकरणों का उपयोग करेंगे: अखंडता को सत्यापित करने के लिए हैशिंग फ़ंक्शन, और प्रामाणिकता को सत्यापित करने के लिए GnuPG, एक ओपन-सोर्स उपकरण जो PGP प्रोटोकॉल को कार्यान्वित करता है।


## आवश्यक शर्तें


यदि आप **Linux** पर हैं, तो अधिकांश डिस्ट्रीब्यूशन पर GPG पहले से इंस्टॉल है। यदि नहीं, तो आप इसे निम्न कमांड से इंस्टॉल कर सकते हैं:


```bash
sudo apt install gnupg
```


**macOS** के लिए, यदि आपने पहले से होमब्रू पैकेज मैनेजर स्थापित नहीं किया है, तो निम्नलिखित कमांड के साथ ऐसा करें:


```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```


```bash
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
```


```bash
eval "$(/opt/homebrew/bin/brew shellenv)"
```


फिर इस कमांड के साथ GPG स्थापित करें:


```bash
brew install gnupg
```

**विंडोज** के लिए, यदि आपके पास GPG नहीं है, तो आप [Gpg4win](https://www.gpg4win.org/) सॉफ़्टवेयर इंस्टॉल कर सकते हैं।

![GnuPG](assets/notext/01.webp)


## दस्तावेज़ डाउनलोड करना


शुरू करने के लिए, हमें विभिन्न दस्तावेजों की आवश्यकता होगी। ["*डाउनलोड*" अनुभाग में स्पैरो Wallet की आधिकारिक साइट पर जाएँ](https://sparrowwallet.com/download/)। यदि आप किसी अन्य सॉफ़्टवेयर को सत्यापित करना चाहते हैं, तो उस सॉफ़्टवेयर की वेबसाइट पर जाएँ।


![GnuPG](assets/notext/02.webp)


आप [प्रोजेक्ट के GitHub रिपॉजिटरी पर भी जा सकते हैं](https://github.com/sparrowwallet/sparrow/releases).


![GnuPG](assets/notext/03.webp)


अपने ऑपरेटिंग सिस्टम के अनुरूप सॉफ़्टवेयर का इंस्टॉलर डाउनलोड करें।


![GnuPG](assets/notext/04.webp)


आपको फ़ाइल के Hash की भी आवश्यकता होगी, जिसे अक्सर "*SHA256SUMS*" या "*MANIFEST*" कहा जाता है।


![GnuPG](assets/notext/05.webp)


फ़ाइल का PGP हस्ताक्षर भी डाउनलोड करें। यह दस्तावेज़ `.asc` प्रारूप में है।


![GnuPG](assets/notext/06.webp)


निम्नलिखित चरणों के लिए इन सभी फ़ाइलों को एक ही फ़ोल्डर में रखना सुनिश्चित करें।


अंत में, आपको डेवलपर की सार्वजनिक कुंजी की आवश्यकता होगी, जिसका उपयोग हम PGP हस्ताक्षर को सत्यापित करने के लिए करेंगे। यह कुंजी अक्सर सॉफ़्टवेयर की वेबसाइट पर, प्रोजेक्ट के GitHub रिपॉजिटरी पर, कभी-कभी डेवलपर के सोशल मीडिया पर या Keybase जैसी विशेष साइटों पर उपलब्ध होती है। स्पैरो Wallet के मामले में, आप डेवलपर क्रेग रॉ की सार्वजनिक कुंजी [Keybase पर](https://keybase.io/craigraw) पा सकते हैं। इसे सीधे टर्मिनल से डाउनलोड करने के लिए, कमांड निष्पादित करें:


```bash
curl https://keybase.io/craigraw/pgp_keys.asc | gpg --import
```


![GnuPG](assets/notext/07.webp)


## हस्ताक्षर का सत्यापन


हस्ताक्षर को सत्यापित करने की प्रक्रिया **Windows**, **macOS**, और **Linux** पर समान है। आम तौर पर, आपने पिछले चरण के दौरान पहले से ही सार्वजनिक कुंजी आयात कर ली है, लेकिन यदि नहीं, तो कमांड के साथ ऐसा करें:


```bash
gpg --import [key path]
```


`[कुंजी पथ]` को डेवलपर की सार्वजनिक कुंजी फ़ाइल के स्थान से प्रतिस्थापित करें।


![GnuPG](assets/notext/08.webp)


निम्नलिखित आदेश से हस्ताक्षर सत्यापित करें:


```bash
gpg --verify [file.asc]
```


`[file.asc]` को हस्ताक्षर फ़ाइल के पथ से बदलें। स्पैरो के मामले में, इस फ़ाइल को संस्करण 2.0.0 के लिए "*sparrow-2.0.0-manifest.txt.asc*" कहा जाता है।


![GnuPG](assets/notext/09.webp)


यदि हस्ताक्षर वैध है, तो GPG आपको यह संकेत देगा। फिर आप अगले चरण पर जा सकते हैं, क्योंकि यह फ़ाइल की प्रामाणिकता की पुष्टि करता है।


![GnuPG](assets/notext/10.webp)


## Hash का सत्यापन

अब जबकि सॉफ़्टवेयर की प्रामाणिकता की पुष्टि हो गई है, तो इसकी अखंडता को सत्यापित करना भी आवश्यक है। हम सॉफ़्टवेयर के Hash की तुलना डेवलपर द्वारा प्रदान किए गए Hash से करेंगे। यदि दोनों मेल खाते हैं, तो यह गारंटी देता है कि सॉफ़्टवेयर कोड में कोई बदलाव नहीं किया गया है।


विंडोज़ पर, टर्मिनल खोलें और निम्नलिखित कमांड निष्पादित करें:


```bash
CertUtil -hashfile [file path] SHA256 | findstr /v "hash"
```


`[फ़ाइल पथ]` को इंस्टॉलर के स्थान से प्रतिस्थापित करें।


![GnuPG](assets/notext/11.webp)


टर्मिनल डाउनलोड किए गए सॉफ्टवेयर का Hash लौटाएगा।


![GnuPG](assets/notext/12.webp)


ध्यान रखें, कुछ सॉफ़्टवेयर के लिए, SHA256 से अलग Hash फ़ंक्शन का उपयोग करना आवश्यक हो सकता है। इस मामले में, कमांड में बस Hash फ़ंक्शन का नाम बदलें।


फिर परिणाम की तुलना फ़ाइल "*sparrow-2.0.0-manifest.txt*" में संबंधित मान से करें।


![GnuPG](assets/notext/13.webp)


मेरे मामले में, हम देखते हैं कि दोनों हैश पूरी तरह से मेल खाते हैं।


**macOS** और **Linux** पर, Hash सत्यापन प्रक्रिया स्वचालित है। विंडोज की तरह दो हैश के बीच मिलान की मैन्युअल रूप से जाँच करना आवश्यक नहीं है।


**macOS** पर बस इस कमांड को निष्पादित करें:


```bash
shasum --check [file name] --ignore-missing
```


`[फ़ाइल नाम]` को इंस्टॉलर के नाम से बदलें। उदाहरण के लिए, स्पैरो Wallet के लिए:


```bash
shasum --check sparrow-2.0.0-manifest.txt --ignore-missing
```


यदि हैश मेल खाते हैं, तो आपको निम्नलिखित आउटपुट दिखाई देगा:


```bash
Sparrow-2.0.0.dmg: OK
```


**लिनक्स** पर, कमांड समान है:


```bash
sha256sum --check [file name] --ignore-missing
```


और यदि हैश मेल खाते हैं, तो आपको निम्नलिखित आउटपुट दिखाई देगा:


```bash
sparrow_2.0.0-1_amd64.deb: OK
```


अब आप आश्वस्त हैं कि आपने जो सॉफ़्टवेयर डाउनलोड किया है वह प्रामाणिक और सही है। आप अपनी मशीन पर इसकी स्थापना के साथ आगे बढ़ सकते हैं।


अगर आपको यह ट्यूटोरियल मददगार लगा, तो मैं नीचे दिए गए अंगूठे के निशान की सराहना करूंगा। इस लेख को अपने सोशल नेटवर्क पर शेयर करना न भूलें। आपका बहुत-बहुत धन्यवाद!


मैं वेराक्रिप्ट पर इस अन्य ट्यूटोरियल को भी देखने की सलाह देता हूं, यह एक सॉफ्टवेयर है जो आपको स्टोरेज डिवाइस को एन्क्रिप्ट और डिक्रिप्ट करने की अनुमति देता है।
