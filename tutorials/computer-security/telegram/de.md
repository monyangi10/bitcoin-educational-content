## Warum Telegramm

Telegram ist viel mehr als eine Messaging-App und geht über das Konzept der sozialen Medien hinaus. Im Vergleich zu vielen seiner Konkurrenten hat es viele Funktionen, die es zu einem Tool machen, das man kennen sollte.


Mit Telegram können Sie nicht nur Nachrichten austauschen, sondern auch Video- und Sprachanrufe tätigen, Nachrichten bearbeiten oder löschen, nachdem sie gesendet wurden, große Dateien ohne Speicherplatzbeschränkung versenden und vieles mehr. Wir hoffen, dass dieses Tutorial dazu beitragen kann, das Erlernen und vor allem die Teilnahme an den zahlreichen Bitcoiner-Communities auf Telegram zu erleichtern.

# Einstellungen für Sicherheit und Datenschutz


## Konfiguration des Benutzernamens


Der Telegram-Benutzername - manchmal auch `Handle` genannt - ist viel mehr als eine bloße Fantasie. Das Handle ist in der Tat **einzigartig für jeden Benutzer**.

Auf Telegram ist es leicht, Betrügern zu begegnen, die privat schreiben und vorgeben, jemand zu sein, der sie nicht sind. Jeder Nutzer kann in die Falle tappen und persönliche Informationen preisgeben, in dem Glauben, dass er mit einem Kontakt chattet, dem er voll und ganz vertraut. **Wir werden sehen, dass der Handle die beste Verteidigung ist, um diese Art von Gefahren zu entschärfen.

Wählen Sie im Hauptmenü _Mein Profil_.


![image](assets/en/33.webp)


Wählen Sie auf dem nächsten Bildschirm das "Bleistift"-Symbol oben rechts, um das Menü zur Profilbearbeitung aufzurufen.


![image](assets/en/12.webp)


Sie werden alle sensiblen Details Ihres Kontos sehen, einschließlich Ihrer Telefonnummer und zwei leeren Feldern: _Bio_ und _Benutzername_.

**Durch Anklicken der einzelnen Felder können Sie diese mit Ihrer Auswahl füllen**.

Beim Einrichten des _Benutzernamens_ meldet Telegram, ob das Handle verfügbar ist oder nicht.

(Dieser Screenshot wurde ebenfalls von einem Telefon aufgenommen, bei dem der Benutzername bereits eingestellt war).


Klicken Sie auf _Benutzernamen festlegen_ (hier _Benutzernamen ändern_ aus dem eben genannten Grund)


![image](assets/en/13.webp)


und konfigurieren Sie Ihr Handle, dann speichern Sie, indem Sie oben rechts auf das Häkchen ✅ klicken.



![image](assets/en/14.webp)


In den meisten Telegram-Gruppen und -Kanälen wird dieser Benutzername als Voraussetzung für den Zugang verlangt. Für die Administratoren solcher Gruppen ist dies eine der Möglichkeiten, Bots und Spam in Schach zu halten.


⚠️ Sie sollten immer den Handle von jedem, der Sie privat kontaktiert, überprüfen und niemals vertrauliche Informationen wie Passwörter oder Mnemonic-Sätze an jemanden weitergeben, selbst wenn dieser behauptet, offizieller Support zu sein oder Hilfe anzubieten (vielleicht von Ihnen angefordert). Blockieren Sie Benutzer, die Sie ohne Ihre Erlaubnis kontaktieren, denn sie tun dies mit Sicherheit in betrügerischer Absicht.


Wie nimmt ein Hochstapler die Identität einer anderen Person an?

Dank der Einzigartigkeit des Benutzernamens können sie das nicht.

**Sie können einen "ähnlichen" Benutzernamen verwenden, der sich nur leicht verändert (ein Buchstabe oder eine Zahl), so dass nur ein aufmerksames Auge erkennen kann, dass es sich um einen Betrüger handelt**. Achten Sie immer genau auf den Benutzernamen, und Sie werden sehen, dass Betrüger kein leichtes Spiel haben werden.


## Datenschutz


Eine weitere wichtige Vorsichtsmaßnahme, die Sie treffen können, ist die Beschränkung der Informationen, die Sie über Ihr neu erstelltes Konto herausgeben.


Gehen Sie zurück zum Hauptmenü und dann in _Einstellungen_:


![image](assets/en/19.webp)


Wählen Sie nun _Datenschutz und Sicherheit_


![image](assets/en/35.webp)


Hier finden Sie eine ganze Reihe von wichtigen Parametern, die Sie anpassen können, je nachdem, wie Sie Ihr Telegram-Konto nutzen möchten.


![image](assets/en/36.webp)


Achten Sie auf die Einstellung:


- _Telefonnummer_ an "Niemand"
- _Anrufe_ bei "Meine Kontakte"
- _Einladungen_ an "Niemand"

Diese Maßnahmen verhindern die Preisgabe Ihrer Telefonnummer, so dass Sie keine unerwünschten Anrufe erhalten oder unwissentlich zu Gruppen zweifelhafter Herkunft hinzugefügt werden. Später können Sie alle anderen Parameter nach Belieben anpassen.

Nun, da Ihr Telegram-Konto eingerichtet ist und Sie ein wenig Privatsphäre gewonnen haben, können Sie es nutzen.


## Hinzufügen von Kontakten und Chats


Wenn Ihr Konto gerade erst erstellt wurde, ist die Hauptseite wahrscheinlich noch völlig leer.


Hier sehen Sie bereits die 2 Hauptfunktionen, die Sie für die Nachrichtenübermittlung verwenden werden:


- einen Suchbefehl, oben rechts;
- das Symbol mit dem Bleistift unten rechts, um das Dashboard zu öffnen, in dem Sie neue Nachrichten verwalten können.


![image](assets/en/37.webp)


Wenn Sie auf Letzteres klicken, bittet Telegram zunächst um die Erlaubnis, auf die Kontakte in Ihrem Address-Buch zuzugreifen, die Sie je nach Bedarf erteilen oder verweigern können. Wenn Sie die Genehmigung erteilen, können Sie die ersten Freunde erreichen, die die App bereits heruntergeladen haben.


![image](assets/en/38.webp)


Daraufhin werden die Kontakte auf dem Hauptbildschirm angezeigt.


![image](assets/en/40.webp)


Wenn Sie auf das Symbol mit dem Bleistift unten rechts klicken, wird der Bildschirm zum Hinzufügen weiterer Kontakte aktiviert, aber nicht nur.


Telegram bietet die Möglichkeit, nach **Gruppen** zu bestimmten Themen zu suchen, die ähnlich wie Foren sind, in denen sich verschiedene Nutzer versammeln, um über ein bestimmtes Thema zu sprechen, oder **Kanäle**, die in der Regel als Informationsmittel genutzt werden, in denen nur Administratoren posten können und Follower den Inhalt konsumieren können.


![image](assets/en/44.webp)


Wenn Sie das Profilbild eines Kontakts in der Liste auswählen, erhalten Sie Zugriff auf ein umfangreiches Menü, in dem Sie interessante Aktionen durchführen können:


- sehen Sie alle Details des Kontakts;
- einen Videoanruf starten (**a**);
- einen Sprachanruf starten (**b**);
- einen Chat starten (**c**);
- benachrichtigungen anpassen (**d**).


![image](assets/en/41.webp)


Sie können auf ein sehr fortgeschrittenes Menü zugreifen, indem Sie auf die 3 Punkte oben rechts klicken, um:


- einen Timer für die automatische Löschung von Nachrichten einstellen;
- den Kontakt freigeben, sperren oder bearbeiten;
- ein Geschenk senden (normalerweise _Telegram Premium_);
- einen geheimen Chat starten, der eine der schönsten Funktionen von Telegram ist: **Ein geheimer Chat ist ein Raum, in dem es zum Beispiel nicht möglich ist, Screenshots zu machen, es ist ein sehr privater Chat und ist nur auf dem Handy aktiv**;
- den Kontakt zum Startbildschirm hinzufügen.

Standardmäßig wird jeder, von einzelnen Benutzern bis hin zu thematischen Kanälen, durch sein Handle identifiziert. Wenn Sie nach jemandem oder etwas suchen, genügt es, das At-Symbol "@" gefolgt von einem Namen einzugeben.

⚠️Attention: **Vermeiden Sie es, Gruppen und Kanälen beizutreten, ohne deren Authentizität zu überprüfen**. Um den offiziellen Telegram-Kanal/die offizielle Telegram-Gruppe eines Unternehmens oder eines Themas zu finden, dem Sie folgen möchten, wenden Sie sich an den Abschnitt _Kontakte_ der offiziellen Websites oder an äußerst zuverlässige Quellen.


### Erweiterte Messaging-Funktionen


Telegram ermöglicht es Ihnen, einzigartige erweiterte Funktionen für den Austausch von Nachrichten zu nutzen. Betreten Sie einen Chat und klicken Sie auf den Hintergrund neben einer Nachricht von einem anderen Absender.


Es erscheint eine Reihe von Optionen, die Sie nutzen können:


- nachrichten anpinnen (_pin_ **A**), um wichtige Nachrichten schnell zu finden;
- einen Anruf starten (**B**);
- reaktionen einfügen (**C**);
- weiterleiten, Kopieren, Löschen der Nachricht (**D**);
- mehr als eine Nachricht für mehrere Aktionen auswählen.


![image](assets/en/45.webp)


Wenn Sie stattdessen den gleichen Vorgang an einer Ihrer Nachrichten durchführen, **werden Sie feststellen, dass Sie Ihre eigenen Nachrichten bearbeiten können, auch die bereits gesendeten**.


Sie können auch große Dateien anhängen und so problemlos "schwere" Medien austauschen, viel mehr als bei allen anderen Anwendungen dieser Art.


![image](assets/en/46.webp)


### Persönliche Wolke


Unter den vielen unglaublichen Funktionen von Telegram gibt es auch eine persönliche Cloud, die - zum Zeitpunkt der Erstellung dieses Artikels - **unbegrenzt** ist.


Wir sprechen über die berühmten "Gespeicherten Nachrichten", oder _Saved Messages_ von Telegram. Es ist ein Chat, in dem man fast **(1)** jede Art von Information senden kann, zum Beispiel die Übertragung von Dateien vom PC zum Handy und umgekehrt.


Um auf die _Gespeicherten Nachrichten_ Ihres Kontos zuzugreifen, gehen Sie zum Hauptmenü und wählen Sie den entsprechenden Punkt unter den Optionen, die auf dem Bildschirm erscheinen.


![image](assets/en/29.webp)


Der Chat erscheint im Vordergrund und kann benutzt werden.


![image](assets/en/30.webp)


***

**(1)** _Nutzen Sie die Telegram-Cloud nicht für vertrauliche Informationen wie Passwörter, Pins, Mnemonics und Daten dieser Art_.

***


## Planen von Nachrichten und stummes Senden


Weitere nützliche erweiterte Funktionen ermöglichen den Versand von Nachrichten unter Wahrung der Privatsphäre der Empfänger, die Wahl zwischen stillem Versand und die Planung der Nachrichten zu geeigneten Zeiten und Tagen.

Sie müssen nur die Nachricht schreiben, aber anstatt sie sofort zu senden, halten Sie das Senden-Symbol einige Sekunden lang gedrückt. Was normalerweise zu einer gesendeten Nachricht wird, weicht einem neuen Bildschirm, aus dem Sie wählen können:


- die Nachricht planen (Datum und Uhrzeit)
- nur senden, wenn der Kontakt online ist
- stillschweigend senden, um die Benachrichtigungen des Empfängers nicht zu aktivieren.


![image](assets/en/48.webp)


### Löschen des Cache Ihres Telefons


Eine weitere nützliche Methode, um Ihr Telefon effizient zu betreiben, ist, den Telegram-Cache von Zeit zu Zeit zu löschen. Je nachdem, wie vielen Gruppen und Kanälen Sie folgen, können sich die Informationen und Medien aus diesen Quellen im Cache ansammeln und Ihr Telefon langsam machen.


Rufen Sie erneut die Hauptmenüs auf, indem Sie auf die drei Balken oben rechts klicken, und wählen Sie erneut _Mein Profil_. Diesmal wählen Sie jedoch die 3 Punkte oben rechts.


![image](assets/en/34.webp)


Es öffnet sich ein Dropdown-Menü, aus dem Sie _Abmelden_ wählen müssen.


⚠️ **Sie werden sich nicht abmelden, keine Sorge: wir wählen dieses Menü nur, um auf die Funktionalität zuzugreifen, über die wir sprechen**.


![image](assets/en/24.webp)


Wählen Sie unter den Optionen _Cache leeren_.


![image](assets/en/25.webp)


Das Gerät beginnt, den belegten Speicherplatz zu ermitteln. Sobald die Berechnung abgeschlossen ist, erscheint die Schaltfläche _Cache löschen_.


![image](assets/en/26.webp)


Wenn Sie darauf klicken, wird ein Bestätigungsbildschirm angezeigt, auf dem Sie erneut _Cache löschen_ wählen müssen, um fortzufahren.


![image](assets/en/27.webp)


Sobald der Vorgang abgeschlossen ist, zeigt Telegram einen Bildschirm an, auf dem unter dem Ergebnis der Bereinigung auch eine interessante Einstellung erscheint, nämlich die Möglichkeit zu wählen, wie viel Cache-Speicherplatz für Medien verwendet werden soll.

Ich empfehle, den Speicherplatz für Fotos und Videos nicht unbegrenzt zu halten, sondern der App zu erlauben, schwere Dateien zu entfernen, sobald diese Grenze erreicht ist.

Auf dem folgenden Foto können Sie sehen, wo diese Einstellung zu finden ist.

## Telegramm-Desktop


Telegram kann von Ihrem Computer aus genutzt werden, so dass es mit dem auf Ihrem Telefon angezeigten Konto synchronisiert wird. Sie können sich dafür entscheiden, die Anwendung nicht herunterzuladen und sie nur über das Internet zu nutzen. Diese Version hat jedoch Einschränkungen im Vergleich zu der auf dem Computer ausgeführten Version, daher empfehle ich, sie herunterzuladen und zu installieren, um das Beste aus diesem leistungsstarken Tool herauszuholen.


Alle Optionen, die Sie bisher mit dem mobilen Modell gesehen haben, können Sie auf genau dieselbe Weise von Ihrem Computer aus nutzen. Gehen Sie auch für diese Installation auf die offizielle Website [telegram.org] (https://telegram.org). Wählen Sie auf der Homepage _Telegram für PC/Linux_.

Klicken Sie in dem sich öffnenden Fenster auf , um die für Ihr Betriebssystem geeignete ausführbare Datei herunterzuladen.


![image](assets/en/15.webp)


Installieren Sie Telegram und starten Sie es, so dass Sie sofort den ersten Bildschirm finden, auf dem Sie auf _Start Messaging_ klicken.


![image](assets/en/16.webp)


Auf dem Bildschirm wird ein QR-Code angezeigt, den Sie mit Ihrem mobilen Gerät, auf dem Telegram bereits aktiv ist, scannen müssen: So können Sie das Konto über den Desktop nutzen.


![image](assets/en/17.webp)


Öffnen Sie die App auf Ihrem Handy und gehen Sie zum Hauptmenü (die drei Zeilen oben links).


![image](assets/en/31.webp)


Wählen Sie _Einstellungen_


![image](assets/en/19.webp)


und dann unmittelbar nach _Geräte_.


![image](assets/en/20.webp)


Wählen Sie nun _Link Desktop Device_


![image](assets/en/32.webp)


Die Kamera Ihres Telefons wird aktiviert. Bei der ersten Verwendung wird Ihr Android-System wahrscheinlich nach Berechtigungen fragen: Erteilen Sie diese.


![image](assets/en/21.webp)


Scannen Sie nun den QR-Code, der zuvor auf dem Computerbildschirm erschienen ist.


![image](assets/en/22.webp)


Eine Benachrichtigung auf Ihrem Telefon bestätigt, dass das neue Gerät erfolgreich hinzugefügt wurde.


![image](assets/en/23.webp)


Insbesondere ist Telegram auch auf Ihrem Desktop-Computer aktiv und nutzbar.


### Gruppenruf


Wenn Sie Administrator oder Besitzer einer Telegram-Gruppe sind, können Sie einen Anruf direkt aus dem Menü der Gruppe starten. Auf diese Weise können Sie Live-Streaming mit mehreren Teilnehmern durchführen, sie in Audio und Video aufzeichnen, sie teilen oder sie für Zwecke wie z. B. Bildung verwenden.


In der folgenden Abbildung sehen Sie, wie Sie einen Gruppenanruf mit Telegram Desktop starten können: Gehen Sie zum Chat desselben und im oberen rechten Teil des Bildschirms befindet sich das Symbol eines Bildschirms. Wenn Sie darauf klicken, können Sie entscheiden, ob Sie den Anruf sofort starten oder ihn für eine bestimmte Zeit planen möchten.
