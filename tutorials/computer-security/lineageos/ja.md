私たちのスマートフォンにプリインストールされている従来のアンドロイド・システムは、多くのよく知られた問題を引き起こしている。それは、常にデータを追跡することにつながるグーグル・サービスの集中的な統合、メーカーが押し付ける不要なスポンサー・アプリケーション（ブロートウェア）、数年後にアップデートの追跡を放棄し、まだ機能しているデバイスをプログラムによる陳腐化に追いやることなどである。



LineageOSは、これらの問題に対するエレガントな回答として自らを提示している。オープンソースコミュニティの産物であり、CyanogenMod（2016年末に廃止）の直接の後継であるLineageOSは、ユーザーがスマートフォンを再びコントロールできるようにする無料のAndroidベースのモバイルオペレーティングシステムだ。2016年12月に正式にローンチされたこのプロジェクトは、現在世界中で440万以上のアクティブインストールを誇り、20を超えるブランドの数百の携帯電話モデルをサポートしている。



![lineageos-homepage](assets/fr/01.webp)



*プロジェクトとその目的を紹介するLineageOS公式ウェブサイト*。



## リネージュOSとは？



### 理念と目標



LineageOSは、Android Open Source Project (AOSP)をベースとしたオープンソースのモバイルオペレーティングシステムで、世界中のボランティア貢献者の広大なコミュニティによって開発されています。このプロジェクトは、スマートフォンの寿命を延ばすと同時に、合理的でプライバシーに配慮したAndroid体験を提供することを目指している。



このプロジェクトは、歴史上最も人気のある代替Android ROMの1つであるCyanogenModの灰から立ち上がった。2016年12月にCyanogenMod Inc.が閉鎖されると、コミュニティはその前身を特徴付けた革新の精神とオープンソースの哲学を保持しながら、LineageOSを作成するために動員された。



OEMのAndroidディストリビューションとは異なり、LineageOSはデフォルトでGoogleのアプリケーションをプリインストールせず、ブロートウェアを完全に排除している。ユーザーは、必要不可欠なアプリケーション（電話、メッセージ、カメラ、ブラウザ）のみを含む最小限のシステムからスタートし、後から追加するものを自由に選択できる。



### インパクトとコミュニティ



224カ国で440万台以上がアクティブにインストールされているLineageOSは、世界で最も広く採用されているAndroid代替ソフトのひとつです。ブラジルが200万人以上のユーザーを獲得してトップに立ち、中国と米国がそれに続いており、無料でカスタマイズ可能なAndroidの普遍的な魅力が実証されています。




## 主な特徴



### Interfaceとユーザー・エクスペリエンス



**ピュアAndroid**：LineageOSは、メーカーのオーバーレイや余計なアプリケーションを排除し、AOSPに近い本物のAndroid体験を提供します。Interfaceは、Androidユーザーに馴染み深いままでありながら、ブロートウェアがないため最適な流動性を提供します。



**デフォルトでGoogleフリー**：法的および倫理的な理由から、Googleのサービスはプリインストールされていません。この "Google-free "アプローチは、お客様の個人データの完全なコントロールを保証し、バックグラウンドで実行されるサービスを回避することでパフォーマンスを向上させます。



### カスタマイズと安全性



**高度なカスタマイズ**：ナビゲーションボタンの再設定、カスタマイズ可能なシステムテーマ、さまざまなコンテキスト（仕事、プライベート、ゲーム）に合わせた使用プロファイルなど、純正Androidでは利用できない多くのオプションがLineageOSによってアンロックされます。



**Outil Trust**：デバイスのセキュリティ状態を監視し、潜在的な脅威を警告する統合機能により、システムのセキュリティをリアルタイムで可視化します。



**拡張アップデート**：LineageOSコミュニティは毎月セキュリティパッチを提供することにコミットしており、製造中止となったデバイスも最新のAndroidセキュリティパッチを受け続けることができます。



## 対応機器



LineageOSは20以上のメーカーの数百のデバイスをサポートしている：Samsung、Xiaomi、OnePlus、Motorola、Sony、Google Pixel、Fairphone、その他多数。この幅広い互換性は、Pixelデバイスに限定されたGrapheneOSのような代替品と比較して、このプロジェクトの大きな利点の1つである。

### 人気デバイス



公式統計によると、最も使用されているモデルには、価格帯や年齢層が異なる様々なデバイスが含まれており、LineageOSが古いスマートフォンに新しい命を吹き込むだけでなく、新しいスマートフォンを最適化する能力もあることを示している。



### インストール前の重要なポイント



**アンロック可能なブートローダー**：メーカー／オペレーターがロック解除を許可しているか確認してください。Huaweiのように、最近のモデルでこの可能性を排除しているブランドもあれば、特定の手順を課しているブランドもあります。



**正確なモデル**：お使いのデバイスに正確に対応するROMをダウンロードすることが重要です。似たような商品名の2つのモデルは技術的に異なる場合があり（例えばGalaxy S10 vs S10 5G）、異なるイメージが必要です。



**スケーラブルなサポート**：移植にはボランティア開発者が必要なため、新しいデバイスはすぐにはサポートされないかもしれません。逆に、デバイスのメンテナがプロジェクトから脱退した場合、サポートが停止する可能性もあります。



## インストール



### 必須条件



⚠️ **トラブルを避けるため、始める前にこの説明書を完全にお読みください**！



**純正ファームウェアに戻す（必要な場合）:**。




- Androidフラッシュツール**：Googleの公式ツール[flash.android.com](https://flash.android.com)を使用して、ウェブブラウザ（Chrome/Edgeが必要）からPixelデバイスを簡単に純正Androidに復元できます。
- 代替**：[developers.google.com/android/images](https://developers.google.com/android/images)から手動でファクトリーイメージを作成する。



*必須前提条件テスト：***。




- 純正システムでデバイスを最低1回**起動させる。
- すべての機能をテスト**：SMS、通話、Wi-Fi、モバイルデータ
- 重要**：SMSの送受信、通話（WiFi、4G/5G経由を含む）ができることを確認してください。純正システムで動作しない場合、LineageOSでも動作しません！
- 最近の端末**：IMSをプロビジョニングするために、純正システムでVoLTE/VoWiFiを少なくとも1回使用する必要があるものもあります。



**システム準備




- アクティベーションをブロックする可能性のあるファクトリー・リセット・プロテクションを避けるため、デバイスからすべてのGoogle**アカウントを削除する。
- フルバックアップ** ：携帯電話を完全に消去します。写真、連絡先、アプリケーション、重要なファイルをバックアップします。



**ADBとFastbootツール:** [公式LineageOSガイド](https://wiki.lineageos.org/adb_fastboot_guide#installing-adb-and-fastboot)に従ってAndroid SDK Platform Toolsをインストールしてください。adb version`と`fastboot --version`でインストールを確認してください。



**電話構成




- デベロッパーオプション**を有効にします：設定」>「バージョン情報」>「ビルド番号」を7回タップします。



![android-version](assets/fr/06.webp)



*開発者モード*を有効にするには、[設定]> [電話について]に移動します。





- 開発者向けオプションで**USBデバッグ**を有効にする
- OEM Unlock**を有効にする（ブートローダーのロック解除に不可欠です）



![developer-options](assets/fr/07.webp)



*開発者向けオプション、USBデバッグ、OEMロック解除を有効にする*。



### 詳細なインストール



⚠️ **これらの手順はLineageOS 22.2専用です。各ステップを正確に実行してください。失敗しても先に進んではいけません！



#### ステップ1：ファームウェアのチェック



**ファームウェアが必要です：続行する前に、お使いのデバイスにAndroid 13がインストールされている必要があります（Pixel 4の場合）。ファームウェアとは、ストックシステムに含まれるデバイス固有のイメージを指します。



![pixel4-info](assets/fr/04.webp)



*ダウンロード・リンクとインストール・ガイドを含むPixel 4の公式ページ*。



![downloads-page](assets/fr/05.webp)



*LineageOSビルドのダウンロードページとファイル



**Pixel 4特有のダウンロード:**。




- LineageOS**をビルドする：[download.lineageos.org/devices/flame/builds](https://download.lineageos.org/devices/flame/builds)
- 必要なファイル**：このページから3つの必要なファイルをダウンロードしてください：
  - `lineage-22.2-YYYYMMDD-nightly-flame-signed.zip` (メインROM)
  - dtbo.img` （パーティション・デバイス・ツリー・ブロブ）
  - `boot.img` (リカバリ用LineageOS)



⚠️ **重要**：メーカーのOSバージョンではなく、Androidバージョンを確認してください。カスタムROM（たとえLineageOSであっても）であっても、この要件が満たされていることを保証するものではありません。



💡 **ヒント**：ファームウェアに疑問がある場合は、続行する前に純正システムに戻してください！



#### ステップ2：ブートローダーのアンロック



⚠️ **このステップでは、すべてのデータが削除されます！





- ADB接続をテストする**：デバイスをUSBで接続し、コンピューターのターミナルから`adb devices`コマンドでテストする。



![adb-devices](assets/fr/08.webp)



*ADB devices`*コマンドでADB接続をチェックする





- 携帯電話で接続を許可する



![usb-debugging-auth](assets/fr/09.webp)



*コンピュータのRSAフィンガープリントでUSBデバッグを有効にする*。





- ブートローダー・モードに入る** ：


```
adb -d reboot bootloader
```


または**音量ダウン+電源**を押したままデバイスをオフにする





- fastboot**の接続を確認してください：


```
fastboot devices
```



![fastboot-mode](assets/fr/10.webp)



*ターミナルでFastbootコマンドを実行し、接続を確認する*。



![bootloader-screen](assets/fr/11.webp)



*Pixel 4のfastbootによるシステム情報*の表示





- ブートローダーのロック解除** ：


```
fastboot flashing unlock
```


デバイス上で、ボリュームキーを使ってナビゲートし、**Power**ボタンを押して "Unlock the bootloader "を選択し、操作を確認します。



![unlock-bootloader](assets/fr/12.webp)



*デバイスのブートローダーアンロックの確認*。



⚠️ **ロック解除確認後、自動的に再起動します。





- 自動再起動後**、開発者向けオプションでUSBデバッグを再度有効にしてください。




#### ステップ3：追加のパーティションをフラッシュする



⚠️ **リカバリーが正しく機能するために必要**です。





- ブートローダーを再起動するボリュームダウン+電源
- Flash** (`/path/to/` はファイルをダウンロードしたフォルダに置き換えてください) ：


```
fastboot flash dtbo /chemin/vers/dtbo.img
```



![flash-partitions](assets/fr/13.webp)



*fastboot*によるdtboとboot.imgパーティションのフラッシュ



#### ステップ4: LineageOSリカバリーのインストール





- フラッシュリカバリ** (`/path/to/`はファイルをダウンロードしたフォルダに置き換えてください) ：


```
fastboot flash boot /chemin/vers/boot.img
```




- リカバリー**で再起動して確認する



#### ステップ5：LineageOSのインストール





- リカバリーで再起動するボリュームダウン＋電源 → リカバリーモード



![recovery-mode](assets/fr/14.webp)



*Interface、LineageOSのリカバリーからメインメニュー*を表示する。





- ファクトリーリセット** ：ファクトリーリセット"→"データのフォーマット/ファクトリーリセット "と入力してください。



![factory-reset](assets/fr/15.webp)



*LineageOS*リカバリーのファクトリーリセットプロセス





- メインメニューに戻る
- リネージュOSをサイドロードする** ：
   - デバイス上で"アップデートの適用" → "ADBから適用"
   - PCの場合： `adb -d sideload /path/to/lineageos.zip`。



![apply-update](assets/fr/16.webp)



*リカバリー*で「アップデートの適用」を選択し、「ADBから適用」を選択する。



![sideload-process](assets/fr/17.webp)



*サイドロード*によるLineageOSのインストール中



![sideload-terminal](assets/fr/18.webp)



*インストールが進行中の端末でサイドロード・コマンドを実行する*。



正常です：プロセスが47%で停止したり、"Success "エラーが表示されることがありますが、これは正常です！



#### ステップ6：最初のスタートアップ





- 再起動**："今すぐシステムを再起動する"
- 初回起動**：最大15分



インストール完了!



### 注意点



⚠️ **警告**：LineageOSは保証なしで「現状のまま」提供されます。全ての機能が動作するよう最善を尽くしますが、インストールは自己責任で行ってください！



**重要なチェック項目




- ファームウェアの互換性**：お使いのモデルのダウンロードページで、必要なファームウェアのバージョンを確認してください。
- LineageOSのインストール後、ブートローダーを再ロック**しないこと
- お使いのデバイスの説明**に従ってください。



## コンフィギュレーションとアプリケーション



### 最初のスタートアップ


Interfaceは、Googleを使わず、純正アンドロイドに近い合理化されたものです。シンプルな設定：言語、Wi-Fi、アカウント不要。



### 代替アプリケーション



**安全なアプリケーション・ソース



**F-Droid** ：microG用LineageOSにプリインストールされているか、直接ダウンロード可能なベンチマーク・オープンソース・アプリケーション・ストアです。F-Droidは、トラッカーや悪意のあるコンポーネントがないことを保証し、透過的に検証されコンパイルされたオープンソースソフトウェアのみを提供しています。



**オーロラストア**：GoogleアカウントなしでGoogle Playストアにアクセスするための匿名クライアント。Auroraは、あなたのプライバシーを維持しながら、主流のアプリをダウンロードすることができ、共有匿名アカウントを借用します。



**必須代替アプリケーション





- ナビゲーション**：オーガニックマップ（OpenStreetMapベースのオフライン地図）
- 通信**：Signal（エンドツーエンド暗号化メッセージ）、K-9 Mail（無料メールクライアント）
- メディアNewPipe（広告なし、トラッキングなしのYouTube）、VLC（ユニバーサルメディアプレーヤー）
- 生産性**：Nextcloud（セルフホスティングクラウド）、シンプルカレンダー（CalDAV同期）
- セキュリティ**：Bitwarden (パスワードマネージャー)、Aegis Authenticator (2FAコード)



これらのアプリケーションは、そのほとんどがF-Droid経由で利用可能で、モダンで機能的なユーザー体験を提供しながら、グーグルのサービスを完全に置き換えることができる首尾一貫したエコシステムを形成している。



## 使用とメンテナンス



### 日々の経験



LineageOSは、Androidの体験を一変させ、流動性と応答性を優先させる。合理化されたInterfaceは、特に古いデバイスで威力を発揮し、重いオーバーレイや余計な処理がないおかげで、メーカー製ROMよりも一般的にパフォーマンスが優れている。



基本的な機能（通話、SMS、写真、ブラウズ）はシームレスに動作し、カスタマイズ・ツールは安定性を損なうことなく個人の好みに合わせてシステムを細かく調整することを可能にする。



### OTAアップデートシステム



LineageOSは、特に使いやすいOver-The-Airアップデートシステムを採用しています。新しいバージョンは通知によって自動的に提案され、インストールは数回クリックするだけで、複雑な技術的介入は必要ありません。このプロセスにより、インストール済みのデータやアプリケーションは完全に保持されます。



こうした定期的なアップデートは、特にメーカーが生産を終了したデバイスにとっては大きな財産であり、最新のアンドロイド・セキュリティ・パッチの恩恵を受け続けることができる。



### 推奨されるベストプラクティス



*インストール後のセキュリティ：***。




- ロック解除用の強力な暗証番号またはパスワードを設定する。
- ストレージの暗号化が有効になっているか確認する（通常はデフォルトで有効になっている）
- F-Droid経由でBitwardenのようなパスワードマネージャーをインストールする。



**予防メンテナンス




- セキュリティのための定期的なOTAアップデート
- アプリケーションのインストールを信頼できるソースに制限する（F-Droid、Aurora Store）
- アプリケーションに付与された許可を定期的に見直す
- 時々再起動することで、パフォーマンスを最適化し、メモリを解放します。

## GrapheneOSとLineageOSの違いは？



### 異なるアプローチ



**GrapheneOS**は最大限のセキュリティに特化し、Google Pixels上でのみ動作し、専用のセキュリティチップを利用します。このシステムには、エクスプロイトに対する数多くの高度な緩和策が組み込まれており、アプリケーションのサンドボックス化が大幅に強化されています。



**LineageOS**は、可能な限り多くのデバイスにおいて、セキュリティ、プライバシー、利便性のバランスをとっている。そのアプローチはより現実的で、絶対的なセキュリティよりも拡張された互換性を目指している。



### Googleサービスの管理



**GrapheneOS**：オプションでサンドボックス化されたGoogle Playシステムを提供。Google Playはインストールできるが、特別なシステム特権はなく、厳密なサンドボックスで実行される。このユニークなアプローチにより、厳格なセキュリティ管理を維持しながらGoogleエコシステムを利用することができます。



**LineageOS**：Googleサービス(GApps)、microG(無料代替)、または完全にGoogleフリーのインストールを選択できます。ニーズに合わせて最大限の柔軟性を提供します。



### 技術比較



| **Aspect** | **GrapheneOS** | **LineageOS** |
|------------|----------------|---------------|
| **Compatibilité** | Pixels uniquement | Centaines d'appareils |
| **Sécurité** | Mitigations avancées | Sécurité AOSP standard |
| **Google Play** | Sandboxé optionnel | Installation classique possible |
| **Installation** | Interface web + USB | Procédure manuelle technique |
| **Philosophie** | Sécurité avant tout | Équilibre et liberté de choix |

### 使用上の推奨事項



**GrapheneOS**は、あなたがPixelを所有し、最大限のセキュリティが最優先であり、強化された保護のための制約を受け入れるのであれば、選択してください。



**LineageOS**は、非Pixel端末をお持ちの方、プライバシーと実用性のバランスを重視する方、Googleエコシステムとの妥協レベルを自由に選びたい方におすすめです。



## 結論



LineageOSは、あなたのAndroidスマートフォンをコントロールするための成熟した選択肢を提供します。プライバシーと日常の実用性を兼ね備えた理想的な選択です。
