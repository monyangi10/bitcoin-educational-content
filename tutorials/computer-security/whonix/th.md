**Whonix** เป็นการกระจายตัวของ Linux ที่พัฒนามาจาก **Debian** ออกแบบมาเพื่อให้สภาพแวดล้อมที่รวม **ความปลอดภัย**, **การไม่เปิดเผยตัวตน** และ **ความเป็นส่วนตัว** เรียนรู้ง่ายและเข้ากันได้กับอินเทอร์เฟซต่างๆ (เครื่องเสมือน, Qubes OS, โหมด Live) โดยค่าเริ่มต้นจะรวมการกำหนดเส้นทางการจราจรเครือข่ายผ่าน **Tor**, **ไฟร์วอลล์คู่** (ไฟร์วอลล์หนึ่งบน Gateway และอีกหนึ่งบน Workstation), **การป้องกันเต็มรูปแบบจากการรั่วไหลของ IP/DNS** และเครื่องมือในการปกปิดกิจกรรมของคุณจากผู้สังเกตการณ์เครือข่ายอย่างมีประสิทธิภาพ รวมถึง ISP ของคุณ ไม่ใช่แค่ระบบที่ไม่เปิดเผยตัวตนเท่านั้น แต่ **Whonix** ยังเป็นสภาพแวดล้อมการพัฒนาที่ปลอดภัยอย่างสมบูรณ์อีกด้วย



## ทำไมถึงเลือก Whonix?





- ฟรี**: เช่นเดียวกับการแจกจ่าย Linux ส่วนใหญ่ Whonix เป็นระบบโอเพ่นซอร์สที่ได้รับอนุญาตให้ใช้ฟรีทั้งหมด มันถูกพัฒนาในโอเพ่นซอร์ส โดยมีชุมชนที่กระตือรือร้นและโปร่งใส
- ความเป็นส่วนตัว ความปลอดภัย และการไม่เปิดเผยตัวตน**: วัตถุประสงค์หลักของ Whonix คือการมอบสภาพแวดล้อมที่มีความปลอดภัยสูงสุด ซึ่งข้อมูลทั้งหมดของคุณจะได้รับการปกป้องและการสื่อสารของคุณจะถูกเข้ารหัสผ่านเครือข่าย Tor
- ใช้งานง่าย**: Whonix มาพร้อมกับกราฟิก Interface ที่ตั้งค่าไว้ล่วงหน้าและใช้งานง่าย เหมาะสำหรับผู้ใช้มือใหม่ ไม่จำเป็นต้องเป็นผู้เชี่ยวชาญก็สามารถได้รับประโยชน์จากการป้องกันขั้นสูงได้
- สภาพแวดล้อมที่เหมาะสมสำหรับการพัฒนาที่ปลอดภัย**: Whonix ช่วยให้คุณพัฒนา, ทดสอบ, ตรวจสอบ หรือรันโปรแกรมโดยไม่ต้องเปิดเผยที่อยู่ IP จริงของคุณหรือเปิดเผยพฤติกรรมการท่องเว็บหรือการสื่อสารเครือข่ายของคุณ
- เซสชันแบบใช้แล้วทิ้งและโหมดสด**: Whonix สามารถเปิดใช้งานในโหมดสดหรือผ่านเครื่องที่ใช้แล้วทิ้ง (เช่น ผ่าน **Qubes OS**) ทำให้สามารถดำเนินงานที่สำคัญได้โดยไม่ทิ้งร่องรอยถาวรหลังจากเซสชันสิ้นสุดลง
- การติดตั้งที่ค่อนข้างง่าย**: มีการจัดเตรียมภาพที่พร้อมใช้งานสำหรับการติดตั้งอย่างรวดเร็วในเครื่องเสมือน (VirtualBox, KVM, Qubes) ระบบมีการจัดทำเอกสารและอัปเดตเป็นประจำ



## การติดตั้งและการกำหนดค่า



ก่อนที่จะดำเนินการติดตั้ง Whonix สิ่งสำคัญคือต้องทราบว่าการแจกจ่ายนี้ **ยังไม่พร้อมใช้งานอย่างเป็นทางการ** ในฐานะระบบหลักที่สามารถติดตั้งโดยตรงบนฮาร์ดดิสก์ (ในโหมด bare metal) กล่าวอีกนัยหนึ่งคือคุณ **ยังไม่สามารถติดตั้ง Whonix เป็นระบบปฏิบัติการโฮสต์แบบคลาสสิก** เช่น Ubuntu หรือ Debian มาตรฐาน



อย่างไรก็ตาม มีหลายรุ่นให้เลือกใช้ ทำให้ Whonix สามารถใช้งานได้ทั้งแบบ **volatile** (โหมดสด, เซสชันชั่วคราว) หรือ **persistent** (ผ่านเครื่องเสมือนหรือการรวมใน Qubes OS)



สำหรับการใช้งานระยะยาวและเสถียร **การใช้ virtualization เป็นวิธีเดียวที่แนะนำอย่างเป็นทางการในปัจจุบัน** คุณสามารถรัน Whonix โดยใช้ **VirtualBox** (Whonix-Gateway และ Whonix-Workstation) หรือรวมเข้ากับระบบเช่น **Qubes OS** ในบทแนะนำนี้ เราจะมุ่งเน้นไปที่การติดตั้ง VirtualBox



### ข้อกำหนดเบื้องต้น



ก่อนที่คุณจะติดตั้ง Whonix ในโหมดเสมือน ตรวจสอบให้แน่ใจว่าเครื่องของคุณตรงตามข้อกำหนดทางเทคนิคขั้นต่ำ การจำลองเสมือนต้องการทรัพยากรบางอย่างที่คอมพิวเตอร์ทุกเครื่องอาจไม่มี ดังนั้นจึงจำเป็นอย่างยิ่งที่โปรเซสเซอร์ของคุณจะต้องรองรับเทคโนโลยีการจำลองเสมือน (Intel VT-x หรือ AMD-V) และเปิดใช้งานตัวเลือกนี้ใน BIOS/UEFI



นี่คือสเปคที่แนะนำสำหรับประสบการณ์ที่ราบรื่นและเสถียรกับ Whonix:





- หน่วยความจำเข้าถึงโดยสุ่ม (RAM)**: แนะนำอย่างยิ่งให้มีอย่างน้อย **8 GB** ยิ่งมี RAM มากเท่าไหร่ คุณก็สามารถจัดสรรทรัพยากรให้กับเครื่องเสมือน (Gateway และ Workstation) ได้มากขึ้น ซึ่งจะช่วยปรับปรุงประสิทธิภาพการทำงาน
- พื้นที่ว่างในดิสก์ที่ใช้ได้**: กรุณาให้มีพื้นที่ว่างในดิสก์อย่างน้อย 30 GB** ซึ่งรวมถึงพื้นที่ที่จำเป็นสำหรับเครื่องเสมือนสองเครื่อง, ไฟล์ระบบ และข้อมูลหรือสแน็ปช็อตใดๆ
- โปรเซสเซอร์**: แนะนำให้ใช้โปรเซสเซอร์ที่มีอย่างน้อย **4 คอร์จริง** (8 เธรดลอจิคัล) โดยเฉพาะอย่างยิ่งหากคุณต้องการรันบริการหรือเครื่องมืออื่นๆ พร้อมกัน



### ดาวน์โหลด Whonix



Whonix มีให้เลือกหลายรุ่น ขึ้นอยู่กับประเภทของสภาพแวดล้อมที่คุณต้องการใช้งาน สำหรับผู้ใช้ส่วนใหญ่ (Windows, Linux หรือ MacOs) รุ่น VirtualBox เป็นรุ่นที่ติดตั้งได้ง่ายที่สุด คุณสามารถดาวน์โหลดอิมเมจได้โดยตรงจาก [เว็บไซต์ทางการ](https://www.whonix.org/wiki/VirtualBox)



⚠️ Whonix **ไม่สามารถใช้งานร่วมกับ** MacBooks ที่ใช้โปรเซสเซอร์ Apple Silicon (สถาปัตยกรรม ARM) ได้

ก่อนอื่น ให้เริ่มเครื่อง **Whonix-Gateway-Xfce** เครื่องนี้มีหน้าที่ในการกำหนดเส้นทางการจราจรทั้งหมดผ่านเครือข่าย **Tor** หากไม่ได้เปิดเกตเวย์ การจราจรจะไม่ถูกกำหนดเส้นทางผ่าน Tor และคุณจะสูญเสียการไม่เปิดเผยตัวตน

### อัปเดตระบบ



เปิดเทอร์มินัลแล้วใส่คำสั่งต่อไปนี้เพื่ออัปเดตรายการแพ็กเกจ:



```shell
sudo apt update
```



จากนั้นรันคำสั่งต่อไปนี้เพื่อติดตั้งการอัปเดตที่มีอยู่:



```shell
sudo apt full-upgrade
```



## ค้นพบ Whonix



**Whonix** เป็นระบบที่ออกแบบมาเพื่อมอบสภาพแวดล้อมการคำนวณที่ **ปลอดภัย**, **ไม่ระบุตัวตน** และ **เป็นความลับ** เหมาะสำหรับการท่องอินเทอร์เน็ตโดยไม่ทำให้ตัวตนหรือข้อมูลของคุณถูกเปิดเผย เพื่อให้บรรลุเป้าหมายนี้ มันมาพร้อมกับแอปพลิเคชันที่มีประโยชน์ในชีวิตประจำวันหลายตัวที่ออกแบบมาเพื่อเสริมสร้างความปลอดภัยทางดิจิทัลของคุณตั้งแต่เริ่มต้น


### KeepassXC



**KeePassXC** เป็นโปรแกรมจัดการรหัสผ่านที่รวมอยู่ใน Whonix ช่วยให้คุณสามารถ **สร้าง, เก็บรักษา และจัดการ** รหัสผ่านของคุณได้อย่างปลอดภัย โดยไม่ต้องจำทั้งหมดด้วยตนเอง รหัสผ่านจะถูกเก็บไว้ใน **ฐานข้อมูลที่เข้ารหัส** ซึ่งป้องกันด้วยรหัสผ่านหลัก



### Tor Browser



**Tor Browser** เป็นเว็บเบราว์เซอร์เริ่มต้นของ Whonix มันพึ่งพาเครือข่าย **Tor** ซึ่งจะเปลี่ยนเส้นทางการรับส่งข้อมูลของคุณผ่านรีเลย์หลายแห่งทั่วโลก ทำให้แทบจะเป็นไปไม่ได้ที่จะระบุที่อยู่ IP จริงของคุณ



https://planb.academy/tutorials/computer-security/communication/tor-browser-a847e83c-31ef-4439-9eac-742b255129bb

### Electrum Bitcoin Wallet



**Electrum** เป็น Bitcoin wallet ที่เบาและรวดเร็ว ติดตั้งล่วงหน้าบน Whonix เพื่อให้คุณจัดการ **ธุรกรรมสกุลเงินดิจิทัล** ได้โดยไม่เปิดเผยตัวตน มันไม่ได้ดาวน์โหลด Blockchain ทั้งหมด แต่ใช้เซิร์ฟเวอร์ระยะไกลเพื่อรับข้อมูลที่จำเป็น ทำให้มันเบากว่า Wallet แบบเต็มมาก



https://planb.academy/tutorials/wallet/desktop/electrum-efec9166-46b5-4937-8cee-6bc310975177

Whonix เป็นมากกว่าแค่ระบบปฏิบัติการ: มันคือ **สภาพแวดล้อมที่ปลอดภัย** ที่แท้จริง ออกแบบมาเพื่อปกป้องการไม่เปิดเผยตัวตน ความเป็นส่วนตัว และกิจกรรมที่ละเอียดอ่อนของคุณ ด้วยสถาปัตยกรรมที่ใช้ Tor การแบ่งพาร์ติชันอย่างชาญฉลาดระหว่าง Gateway และ Workstation และเครื่องมือที่ติดตั้งล่วงหน้าเช่น Tor Browser, KeePassXC และ Electrum มันมอบโซลูชันสำเร็จรูปสำหรับใครก็ตามที่ต้องการ **ท่องเว็บอย่างไม่ระบุตัวตน**, **ทำงานอย่างปลอดภัย** หรือ **จัดการข้อมูลลับ**



เพื่อเสริมความปลอดภัยในระบบ Unix ของคุณ ลองดูบทแนะนำของเราเกี่ยวกับการตรวจสอบเครื่องของคุณ: ตรวจสอบช่องโหว่ด้านความปลอดภัยในระบบปฏิบัติการของคุณและตรวจสอบให้แน่ใจว่าข้อมูลของคุณไม่ได้ถูกละเมิด
