![image](assets/cover.webp)


LN VPN เป็นบริการ VPN ที่ปรับแต่งได้ซึ่งรับเฉพาะการชำระเงินด้วย lightning เท่านั้น วันนี้ฉันจะแสดงวิธีการใช้งานและทิ้งร่องรอยให้น้อยลงเมื่อท่องเว็บอินเตอร์เน็ต


มีผู้ให้บริการ VPN คุณภาพมากมาย และเราได้ทำการรีวิวอย่างครอบคลุมในบทความนี้ (hyperlink) อย่างไรก็ตาม LN VPN โดดเด่นกว่าใคร และเราไม่พลาดโอกาสที่จะแนะนำให้คุณรู้จัก


ผู้ให้บริการ VPN ส่วนใหญ่ เช่น ProtonVPN และ Mullvad มีตัวเลือกให้ชำระเงินด้วยบิตคอยน์ แต่ต้องสร้างบัญชีและซื้อแผนการใช้งานในระยะยาวหรือระยะสั้น ซึ่งอาจไม่เหมาะกับงบประมาณของทุกคน


LN VPN ช่วยให้การใช้งาน VPN ตามความต้องการเป็นเวลาสั้นๆ เพียงหนึ่งชั่วโมงเป็นไปได้ ด้วยการใช้การชำระเงินด้วยบิตคอยน์ผ่านเครือข่าย lightning การชำระเงินผ่าน lightning ที่รวดเร็วและไม่ระบุตัวตนเปิดโลกแห่งความเป็นไปได้สำหรับการชำระเงินขนาดเล็ก


หมายเหตุ💡: **คู่มือนี้อธิบายวิธีการใช้ LN VPN จากระบบ Linux Ubuntu 22.04 LTS**


## ข้อกำหนดเบื้องต้น: Wireguard


ในแง่ที่ง่ายที่สุด Wireguard ถูกใช้เพื่อสร้างช่องทางที่ปลอดภัยระหว่างคอมพิวเตอร์ของคุณและเซิร์ฟเวอร์ระยะไกลที่คุณจะใช้ในการท่องอินเทอร์เน็ต ที่อยู่ IP ของเซิร์ฟเวอร์นี้จะปรากฏเป็นของคุณในช่วงระยะเวลาการเช่าที่คุณจะทำสัญญาโดยการทำตามคำแนะนำนี้


คู่มือการติดตั้ง Wireguard อย่างเป็นทางการ: https://www.wireguard.com/install/


```
Wireguard installation
$ sudo apt-get update
$ sudo apt install wireguard
```


## ข้อกำหนดเบื้องต้น: Lightning Bitcoin Wallet


หากคุณยังไม่มี Lightning Bitcoin wallet ไม่ต้องกังวล เราได้สร้างคู่มือที่ง่ายมากสำหรับคุณที่นี่ (ส่วนการสอน LN สามารถช่วยคุณได้)


## ขั้นตอนที่ 1: Contract a Lease


จาก https://lnvpn.com คุณจะต้องเลือกประเทศของ IP ออกของอุโมงค์ VPN และระยะเวลา เมื่อกำหนดพารามิเตอร์เหล่านี้แล้ว ให้คลิกที่ ชำระเงินด้วย lightning


![image](assets/1.webp)


ใบแจ้งหนี้สายฟ้าจะถูกแสดงขึ้น และคุณเพียงแค่สแกนด้วยสายฟ้า wallet ของคุณ


เมื่อชำระเงินใบแจ้งหนี้แล้ว คุณจะต้องรอสักครู่ตั้งแต่ไม่กี่วินาทีจนถึงสองนาทีเพื่อให้การตั้งค่าการกำหนดค่า Wireguard ของคุณถูกสร้างขึ้น หากใช้เวลานานกว่านี้เล็กน้อย ไม่ต้องตกใจ เราได้ทำขั้นตอนนี้หลายสิบครั้งแล้ว และบางครั้งมันก็ใช้เวลานานกว่านี้เล็กน้อย

ข้อความต่อไปนี้ได้รับการแปลเป็นภาษาอังกฤษโดยคงรูปแบบการจัดวาง markdown เดียวกันกับข้อความต้นฉบับ:


หน้าจอถัดไปจะปรากฏขึ้นและคุณเพียงแค่ต้องคลิกที่ "Download as File" เพื่อรับไฟล์การตั้งค่าของคุณ ซึ่งจะมีชื่อที่ดูเหมือน lnvpn-xx-xx.conf โดยที่ "xx" จะสอดคล้องกับวันที่ปัจจุบัน

## ขั้นตอนที่ 2: เปิดใช้งานอุโมงค์


ก่อนอื่น คุณจะต้องเปลี่ยนชื่อไฟล์ config ที่ได้รับจากขั้นตอนก่อนหน้าเพื่อให้สามารถถูกจดจำโดยอัตโนมัติโดย Wireguard


ไปที่โฟลเดอร์ดาวน์โหลดของคุณ ไม่ว่าจะในหน้าต่างเทอร์มินัลหรือด้วยตัวสำรวจไฟล์ และเปลี่ยนชื่อไฟล์ lnvpn-xx-xx.conf เป็น wg0.conf ดังนี้:


```
$ sudo ln -s usrbin/resolvectl usrlocal/bin/resolvconf
$ sudo wg-quick up ~/Downloads/wg0.conf
```


ไปเลย! อุโมงค์ถูกเปิดใช้งานแล้ว!


## ขั้นตอนที่ 3: ตรวจสอบ


ใช้บริการออนไลน์เช่น whatismyip เพื่อตรวจสอบว่าที่อยู่ IP สาธารณะของคุณตอนนี้เป็นที่อยู่จาก VPN ที่คุณเพิ่งเปิดใช้งาน


## ขั้นตอนที่ 4: ปิดใช้งาน


เมื่อสัญญาเช่าของคุณหมดอายุ คุณจะต้องปิดการเชื่อมต่อเพื่อกลับมาใช้งานอินเทอร์เน็ตได้อีกครั้ง จากนั้นคุณสามารถทำซ้ำขั้นตอนที่ 1 ถึง 3 ได้ทุกเมื่อที่คุณต้องการทำสัญญาเช่ากับ LN VPN


ปิดการใช้งานอุโมงค์:


```
$ sudo ip link delete dev wg0
```


นั่นไง! ตอนนี้คุณรู้วิธีใช้ LN VPN ซึ่งเป็นบริการ VPN ที่ไม่เหมือนใครแล้ว!
