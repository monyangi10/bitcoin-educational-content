LN VPN یک سرویس VPN قابل تنظیم است که فقط پرداخت‌های لایتنینگ را می‌پذیرد. امروز به شما نشان خواهم داد که چگونه از آن استفاده کنید و هنگام مرور اینترنت ردپای کمتری بگذارید.


ارائه‌دهندگان بسیاری از خدمات VPN با کیفیت وجود دارند، اما LN VPN برجسته است و نمی‌توانستیم فرصت معرفی آن به شما را از دست بدهیم.


اکثر ارائه‌دهندگان خدمات VPN مانند ProtonVPN و Mullvad گزینه پرداخت با بیت‌کوین را ارائه می‌دهند اما نیاز به ایجاد حساب کاربری و خرید یک طرح برای مدت طولانی‌تر یا کوتاه‌تر دارند که ممکن است با بودجه همه سازگار نباشد.


LN VPN استفاده از VPN را به صورت درخواستی و برای مدت زمان کوتاه تا یک ساعت امکان‌پذیر می‌کند، به لطف اجرای پرداخت‌های Bitcoin از طریق Lightning Network. پرداخت‌های سریع و ناشناس، دنیایی از امکانات را برای پرداخت‌های خرد باز می‌کنند.


توجه💡: **این راهنما نحوه استفاده از LN VPN از یک سیستم Linux Ubuntu 22.04 LTS را توضیح می‌دهد.**


## پیش‌نیازها: Wireguard


به زبان ساده، Wireguard برای ایجاد یک تونل امن بین کامپیوتر شما و سرور راه دوری که از طریق آن به اینترنت مرور خواهید کرد، استفاده می‌شود. این IP Address این سرور است که به عنوان IP شما برای مدت زمان اجاره‌ای که با دنبال کردن این راهنما Contract خواهید بود، ظاهر می‌شود.


راهنمای نصب رسمی Wireguard: https://www.wireguard.com/install/


```
Wireguard installation
$ sudo apt-get update
$ sudo apt install wireguard
```


## پیش‌نیازها: Lightning Bitcoin Wallet


اگر هنوز Lightning Bitcoin Wallet ندارید، نگران نباشید، ما یک راهنمای بسیار ساده برای شما اینجا ایجاد کرده‌ایم. (بخش آموزش LN می‌تواند به شما کمک کند)


## مرحله 1: Contract یک اجاره نامه


از https://lnvpn.com، شما باید کشور خروجی IP تونل VPN و مدت زمان را انتخاب کنید. پس از تنظیم این پارامترها، روی "پرداخت با لایتنینگ" کلیک کنید.


![image](assets/1.webp)


یک Invoice صاعقه به نمایش گذاشته خواهد شد و شما فقط باید آن را با Wallet صاعقه خود اسکن کنید.


پس از پرداخت Invoice، باید چند ثانیه تا حداکثر دو دقیقه صبر کنید تا تنظیمات پیکربندی Wireguard شما تولید شود. اگر کمی بیشتر طول کشید، نگران نباشید، ما این روش را ده‌ها بار انجام داده‌ایم و گاهی اوقات فقط کمی بیشتر طول می‌کشد.

متن زیر به انگلیسی ترجمه شده است در حالی که همان قالب‌بندی مارک‌داون متن اصلی حفظ شده است:


صفحه بعدی ظاهر خواهد شد و شما فقط باید روی "Download as File" کلیک کنید تا فایل پیکربندی خود را دریافت کنید، که نامی شبیه به lnvpn-xx-xx.conf خواهد داشت که در آن "xx" به تاریخ فعلی مربوط می‌شود.

![image](assets/2.webp)


## مرحله 2: تونل را فعال کنید


ابتدا، باید فایل پیکربندی که در مرحله قبل به‌دست آورده‌اید را تغییر نام دهید تا به‌طور خودکار توسط Wireguard شناسایی شود.


به پوشه دانلود خود بروید، چه در یک پنجره ترمینال یا با استفاده از مرورگر فایل، و فایل lnvpn-xx-xx.conf را به wg0.conf به این صورت تغییر نام دهید:


```
$ sudo ln -s usrbin/resolvectl usrlocal/bin/resolvconf
$ sudo wg-quick up ~/Downloads/wg0.conf
```


آنجاست! تونل فعال شد!


## مرحله 3: تأیید کنید


از یک سرویس آنلاین مانند whatismyip استفاده کنید تا تأیید کنید که IP عمومی شما Address اکنون همان IP از VPN است که به تازگی فعال کرده‌اید.


## مرحله ۴: غیرفعال کردن


وقتی اجاره‌نامه شما منقضی می‌شود، باید اتصال را غیرفعال کنید تا دوباره به اینترنت دسترسی پیدا کنید. سپس می‌توانید هر زمان که بخواهید برای ایجاد یک اجاره‌نامه با LN VPN مراحل 1 تا 3 را تکرار کنید.


غیرفعال کردن تونل:


```
$ sudo ip link delete dev wg0
```


این هم از این! اکنون می‌دانید چگونه از LN VPN، یک سرویس VPN منحصر به فرد، استفاده کنید!
