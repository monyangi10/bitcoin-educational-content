In diesem Tutorial lernen wir Aurora Store kennen, eine einfache, quelloffene und sichere Alternative zum Google Play Store.



# Erste Schritte mit Aurora Store



Aurora Store ist ein Produkt von *Aurora Open Source Software*, einem Kollektiv von Entwicklern und Datenschutz-Enthusiasten, das sich auf die Entwicklung freier, quelloffener Tools konzentriert, die Ihr digitales Leben und Ihre Freiheit respektieren.


Der Aurora Store ist einer der beliebtesten alternativen App-Stores zum Google Play Store auf der Android-Plattform, und zwar aus den folgenden Gründen




- Ein stromlinienförmiger Interface**: Der Interface macht Ihnen nicht das Geringste aus, so dass Sie sich leicht zurechtfinden können.
- Zugang zu Millionen von Anwendungen**: Der Aurora Store fungiert als Portal für eine breite Palette von mobilen Anwendungen. Wenn die Anwendung, die Sie suchen, im Google Play Store zu finden ist, finden Sie sie auch im Aurora Store.
- Kein Google Play-Dienst erforderlich**: Dank der MicroG-Technologie können Android-Anwendungen auf Ihrem Telefon ohne Google Play-Dienste ausgeführt werden.



Sie können Aurora Store von der [offiziellen Website] (https://auroraoss.com/aurora-store) herunterladen, indem Sie auf die Schaltfläche "Release" in den Build-Typen klicken. Wählen Sie die Option "Vanilla", wenn Sie eine andere Telefonmarke als Huawei verwenden. Neuere Huawei-Telefonmodelle haben ihre eigene Version von Aurora Store, da sie aufgrund der US-Sanktionen nur eingeschränkt auf Google-Dienste zugreifen können.



![download](assets/fr/01.webp)



Sie können Aurora Store auch von anderen Anwendungsspeichern wie F-Droid oder IzzyOnDroid herunterladen.



https://planb.network/tutorials/computer-security/data/f-droid-2cd1aae5-7028-4c04-8fbe-95aeaf278ef4

Wir empfehlen jedoch, die APK-Datei direkt von der offiziellen Website herunterzuladen, um die Authentizität und Integrität von Aurora Store sicherzustellen.



https://planb.network/tutorials/computer-security/data/integrity-authenticity-21d0420a-be02-4663-94a3-8d487f23becc

Eine APK-Datei (Android Package Kit) ist das Paketformat, das vom Android-Betriebssystem für die Verteilung und Installation von Anwendungen verwendet wird. Es ist das Äquivalent zu **.exe**-Dateien unter Windows oder **.dmg**-Dateien unter macOS.




Sobald die Anwendung heruntergeladen wurde, installieren Sie sie manuell und gewähren Sie der Aurora Store-Anwendung Installationsrechte.



![permissions](assets/fr/03.webp)



Da es keine Google Play-Dienste benötigt, können Sie sich anonym mit dem Aurora Store Interface verbinden und alle im Google Play Store verfügbaren Anwendungen nutzen. Aurora Store ist ein Portal, das die Funktionalität von Google Play Store widerspiegelt. Es besitzt, lizenziert oder vertreibt keine Anwendungen.



![login](assets/fr/06.webp)



Wenn Sie eine anonyme Verbindung herstellen, erstellt Aurora Store eine temporäre E-Mail Address, die sich alle 24 Stunden ändert. Es werden keine persönlichen Informationen gespeichert und bei jeder Verbindung wird eine anonyme Sitzung erstellt.


In den Anwendungseinstellungen können Sie die Häufigkeit der Aktualisierungen und den gewünschten Aktualisierungsmodus (automatisch oder manuell) festlegen.



![update-modes](assets/fr/02.webp)



Über den Aurora Store können Sie auf den Katalog der im Google Play Store verfügbaren Anwendungen zugreifen, ihre Beschreibungen lesen, Nutzerfeedback erhalten und sie auf Ihrem Telefon installieren. Das bedeutet jedoch nicht, dass alle Anwendungen kostenlos sind. Wenn Ihre Anwendung im Play Store kostenpflichtig ist, müssen Sie zuerst die Zahlung vornehmen, entweder über die Play Store-Anwendung oder über die offizielle Play Store-Website.



## Eigenschaften



Aurora Store spiegelt nicht nur den Google Play Store wider, sondern enthält auch Funktionen, die Ihnen alle Informationen über die Anwendungen liefern, die Sie installieren möchten. Mit diesen Funktionen haben Sie mit Aurora Store die Kontrolle über Ihre Daten und Ihr digitales Leben.





- Unabhängigkeit von Google:**



Wenn Sie sich dazu entschließen, die Google Play-Dienste zu deaktivieren, um Ihre Daten zu schützen, oder wenn Sie ein Telefon verwenden, das die Google-Dienste nicht mehr unterstützt, können Sie mit Aurora Store dank MicroG, einem Open-Source-Projekt, die im Google Play Store verfügbaren mobilen Anwendungen installieren und nutzen. Diese Funktion ist besonders nützlich, wenn Sie sich in einem Land befinden, das unter US-Sanktionen steht.



![microG](assets/fr/07.webp)





- Download-Kontrolle:**



Mit Aurora Store können Sie mobile Anwendungen manuell herunterladen, indem Sie den Code der Version angeben, die Sie installieren möchten. Die Download-Kontrolle ist ein Vorteil, wenn Ihre Anwendungs-Updates mehr Daten benötigen, als sie benötigen: Sie haben die Kontrolle über Ihre Daten.



![manual](assets/fr/04.webp)





- Berechtigungen:**


In der Beschreibung der Anwendung, die Sie installieren möchten, finden Sie eine vollständige Liste der Berechtigungen, die Sie dieser Anwendung nach der Installation erteilen werden.



![autorisations](assets/fr/05.webp)





- Analyse des Datenschutzes:**



Aurora Store integriert Exodus, eine Lösung zur Analyse und Erkennung von Trackern, die von Ihren Android-Anwendungen verwendet werden. Ein Tracker ist ein kleines Stück Software, das Daten über Sie oder Ihre Aktivitäten sammelt. Durch die Anzeige von Exodus-Analysedaten können Sie mit Aurora Store wirklich wissen, welche Art von Daten von der Anwendung, die Sie installieren möchten, gesammelt wird.



![privacy](assets/fr/08.webp)



Aurora Store ist mehr als nur ein Anwendungsspeicher, es ist eine Open-Source-Lösung, mit der Sie die Kontrolle über die Verwendung Ihrer Daten übernehmen können. Sie liefert Ihnen nützliche Informationen über Ihre Anwendungen und ermöglicht es Ihnen, Ihr digitales Leben vor missbräuchlicher Erfassung zu schützen.
