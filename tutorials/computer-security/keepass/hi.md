डिजिटल युग में, हमें अपने दैनिक जीवन के विभिन्न पहलुओं को कवर करने वाले कई ऑनलाइन खातों का प्रबंधन करने की आवश्यकता है, जिनमें बैंकिंग, वित्तीय प्लेटफॉर्म, ईमेल, फाइल स्टोरेज, स्वास्थ्य, प्रशासन, सोशल नेटवर्क, वीडियो गेम आदि शामिल हैं।


इन सभी खातों पर अपनी पहचान प्रमाणित करने के लिए, हम एक पहचानकर्ता (आमतौर पर ईमेल पता) और एक पासवर्ड का उपयोग करते हैं। बड़ी संख्या में अद्वितीय पासवर्ड याद रखना असंभव होने के कारण, कोई व्यक्ति एक ही पासवर्ड का बार-बार उपयोग करने या उसे याद रखने में आसान बनाने के लिए उसमें थोड़ा बदलाव करने के बारे में सोच सकता है। हालांकि, ये तरीके आपके खातों की सुरक्षा को गंभीर रूप से खतरे में डालते हैं।


पासवर्ड के लिए पालन करने का पहला सिद्धांत यह है कि उन्हें दोबारा इस्तेमाल न करें। प्रत्येक ऑनलाइन खाते को एक अद्वितीय और पूरी तरह से अलग पासवर्ड से सुरक्षित किया जाना चाहिए। यह महत्वपूर्ण है क्योंकि, यदि कोई हमलावर आपके किसी एक पासवर्ड को हैक करने में सफल हो जाता है, तो आप नहीं चाहेंगे कि उसे आपके सभी खातों तक पहुंच प्राप्त हो जाए। प्रत्येक खाते के लिए एक अद्वितीय पासवर्ड होने से संभावित हमलों को रोका जा सकता है और उनका दायरा सीमित किया जा सकता है। उदाहरण के लिए, यदि आप किसी वीडियो गेम प्लेटफॉर्म और अपने ईमेल के लिए एक ही पासवर्ड का उपयोग करते हैं, और गेमिंग प्लेटफॉर्म से संबंधित किसी फ़िशिंग साइट के माध्यम से वह पासवर्ड हैक हो जाता है, तो हमलावर आसानी से आपके ईमेल तक पहुंच सकता है और आपके अन्य सभी ऑनलाइन खातों पर नियंत्रण कर सकता है।


दूसरा महत्वपूर्ण सिद्धांत पासवर्ड की मजबूती है। पासवर्ड को मजबूत तब माना जाता है जब उसे ब्रूट फोर्स विधि से तोड़ना मुश्किल हो, यानी बार-बार कोशिश करके अनुमान लगाना मुश्किल हो। इसका मतलब है कि आपके पासवर्ड यथासंभव यादृच्छिक, लंबे और विभिन्न प्रकार के अक्षरों (छोटे अक्षर, बड़े अक्षर, संख्याएँ और प्रतीक) से युक्त होने चाहिए।


पासवर्ड सुरक्षा के इन दो सिद्धांतों (अद्वितीयता और मजबूती) को रोजमर्रा की जिंदगी में लागू करना मुश्किल साबित हो सकता है, क्योंकि अपने सभी खातों के लिए एक अद्वितीय, यादृच्छिक और मजबूत पासवर्ड याद रखना लगभग असंभव है। यहीं पर पासवर्ड मैनेजर काम आता है।


पासवर्ड मैनेजर मजबूत पासवर्ड बनाता है और उन्हें सुरक्षित रूप से स्टोर करता है, जिससे आप अपने सभी ऑनलाइन खातों को अलग-अलग याद रखने की आवश्यकता के बिना एक्सेस कर सकते हैं। आपको केवल एक पासवर्ड, मास्टर पासवर्ड याद रखना होता है, जो आपको मैनेजर में सेव किए गए सभी पासवर्ड तक पहुंच प्रदान करता है। पासवर्ड मैनेजर का उपयोग करने से आपकी ऑनलाइन सुरक्षा बढ़ती है क्योंकि यह पासवर्ड के पुन: उपयोग को रोकता है और व्यवस्थित रूप से रैंडम पासवर्ड बनाता है। साथ ही, यह आपकी संवेदनशील जानकारी तक पहुंच को केंद्रीकृत करके आपके खातों के दैनिक उपयोग को सरल बनाता है।

इस ट्यूटोरियल में, हम सीखेंगे कि अपनी ऑनलाइन सुरक्षा को बेहतर बनाने के लिए लोकल पासवर्ड मैनेजर को कैसे सेट अप और उपयोग किया जाए। यहाँ, मैं आपको KeePass से परिचित कराऊंगा। हालाँकि, यदि आप शुरुआती हैं और एक ऐसा ऑनलाइन पासवर्ड मैनेजर चाहते हैं जो कई डिवाइसों पर सिंक हो सके, तो मैं आपको Bitwarden पर हमारे ट्यूटोरियल को फॉलो करने की सलाह देता हूँ।

## KeePass का परिचय


KeePass एक निःशुल्क और ओपन-सोर्स पासवर्ड मैनेजर है, जो स्थानीय प्रबंधन के लिए निःशुल्क और सुरक्षित समाधान चाहने वालों के लिए आदर्श है। यह एक ऐसा सॉफ़्टवेयर है जिसे आपके पीसी पर इंस्टॉल किया जा सकता है और बिना किसी प्लगइन के इंटरनेट से कनेक्ट नहीं होता है। यह Bitwarden से बिल्कुल अलग है, जिसके बारे में हमने पिछले ट्यूटोरियल में चर्चा की थी। KeePass के विपरीत, Bitwarden कई उपकरणों पर सिंक्रोनाइज़ेशन की अनुमति देता है और इसलिए आपके पासवर्ड को ऑनलाइन सर्वर पर स्टोर करना आवश्यक होता है।


KeePass डिफ़ॉल्ट रूप से Bitwarden की तरह ब्राउज़र एक्सटेंशन का समर्थन नहीं करता है; इसलिए, आपको सॉफ़्टवेयर से अपने पासवर्ड मैन्युअल रूप से कॉपी और पेस्ट करने होंगे। हालाँकि यह एक कमी लग सकती है, लेकिन ऑटो-फिल का उपयोग करने के बजाय पासवर्ड कॉपी और पेस्ट करना आपकी ऑनलाइन सुरक्षा के लिए एक अच्छा तरीका है।


KeePass को हल्का और उपयोग में आसान बनाने के साथ-साथ उच्च सुरक्षा मानकों का पालन करने के लिए डिज़ाइन किया गया है। यह सॉफ़्टवेयर आपके क्रेडेंशियल्स की सर्वोत्तम सुरक्षा के लिए आपके डेटाबेस को स्थानीय रूप से एन्क्रिप्ट करता है। KeePass एकमात्र पासवर्ड मैनेजर है जिसे ANSSI (फ्रांसीसी साइबर सुरक्षा प्राधिकरण) द्वारा मान्यता प्राप्त है।


KeePass का एक मुख्य लाभ इसकी लचीलापन है। इसे कई अलग-अलग तरीकों से इस्तेमाल किया जा सकता है, जैसे कि कंप्यूटर पर इंस्टॉल किए बिना USB स्टिक पर। इसके अलावा, इसके प्लगइन वातावरण (https://keepass.info/plugins.html) की बदौलत, KeePass को विशिष्ट आवश्यकताओं के अनुरूप अनुकूलित किया जा सकता है।

## KeePass को कैसे डाउनलोड करें?


KeePass की स्थापना प्रक्रिया आपके द्वारा उपयोग किए जा रहे ऑपरेटिंग सिस्टम के आधार पर भिन्न होती है। Windows या Linux उपयोगकर्ताओं के लिए, स्थापना अपेक्षाकृत सरल है। हालाँकि, यदि आप macOS का उपयोग कर रहे हैं, तो एक अतिरिक्त चरण आवश्यक है क्योंकि KeePass को .NET प्लेटफ़ॉर्म पर विकसित किया गया है, जिसे macOS सीधे तौर पर सपोर्ट नहीं करता है। इसलिए, आपको Apple उपकरणों पर KeePass चलाने के लिए एक संगत वातावरण कॉन्फ़िगर करना होगा।


Debian/Ubuntu उपयोगकर्ताओं के लिए, टर्मिनल खोलें और निम्नलिखित कमांड दर्ज करें:


```bash
sudo apt-get update

sudo apt-get install keepass2

```

For Fedora:

```

sudo dnf install keepass

```

For Arch Linux:

```

sudo pacman -S keepass

```

If you are on a Windows computer, go to the [official KeePass download page](https://keepass.info/download.html), and download the latest version of the installer:
![KEEPASS](assets/notext/02.webp)
Click on the downloaded file to run it, then follow the instructions of the setup wizard to complete the installation (see next section).

For macOS users, the installation is a bit more complex. If you wish to use the original version of KeePass as on Windows, follow the instructions below. Otherwise, you can opt for [KeePassXC](https://keepassxc.org/), an alternative version compatible with macOS, which offers a slightly different interface.

To use KeePass, you will need a runtime environment for .NET applications. I recommend installing Mono for this. Go to the [official Mono page](https://www.mono-project.com/download/stable/#download-mac) in the "*macOS*" section, and click on the link to download the installation package (`.pkg`).
![KEEPASS](assets/notext/03.webp)
Open the downloaded `.pkg` file and follow the instructions to install Mono on your Mac.
![KEEPASS](assets/notext/04.webp)
Next, go to the official KeePass website and download the latest portable version in `.zip` format.
![KEEPASS](assets/notext/05.webp)
After downloading the `.zip` file, double-click to extract it. You will get a folder containing several files, including `KeePass.exe`. Open a terminal, navigate to the KeePass folder (replace `xx` with the version number):

```

cd ~/Downloads/KeePass-2.xx

```

And finally, run KeePass with Mono:

```

मोनो KeePass.exe

```
