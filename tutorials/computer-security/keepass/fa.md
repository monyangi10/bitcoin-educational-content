در عصر دیجیتال، ما نیاز داریم تا تعداد زیادی از حساب‌های آنلاین را که جنبه‌های مختلف زندگی روزمره ما را پوشش می‌دهند، مدیریت کنیم، از جمله بانکداری، پلتفرم‌های مالی، ایمیل‌ها، ذخیره‌سازی فایل، سلامت، مدیریت، شبکه‌های اجتماعی، بازی‌های ویدئویی و غیره.


برای احراز هویت خود در هر یک از این حساب‌ها، از یک شناسه، که اغلب یک ایمیل Address است، به همراه یک رمز عبور استفاده می‌کنیم. با توجه به عدم امکان به خاطر سپردن تعداد زیادی رمز عبور منحصر به فرد، ممکن است وسوسه شوید که از همان رمز عبور استفاده کنید یا یک پایه مشترک را کمی تغییر دهید تا به خاطر سپردن آن آسان‌تر شود. با این حال، این روش‌ها امنیت حساب‌های شما را به شدت به خطر می‌اندازند.


اولین اصل برای گذرواژه‌ها این است که از استفاده مجدد آنها خودداری کنید. هر حساب آنلاین باید با یک گذرواژه منحصر به فرد و کاملاً متمایز محافظت شود. این امر مهم است زیرا اگر یک مهاجم موفق به نفوذ به یکی از گذرواژه‌های شما شود، نمی‌خواهید به همه حساب‌های شما دسترسی پیدا کند. داشتن یک گذرواژه منحصر به فرد برای هر حساب، حملات احتمالی را جدا کرده و دامنه آنها را محدود می‌کند. به عنوان مثال، اگر از همان گذرواژه برای یک پلتفرم بازی و ایمیل خود استفاده کنید و آن گذرواژه از طریق یک سایت فیشینگ مرتبط با پلتفرم بازی به خطر بیفتد، مهاجم می‌تواند به راحتی به ایمیل شما دسترسی پیدا کرده و کنترل همه حساب‌های آنلاین دیگر شما را به دست گیرد.


اصل دوم ضروری، قدرت رمز عبور است. یک رمز عبور زمانی قوی تلقی می‌شود که به سختی بتوان آن را با روش‌های حدس و خطا شکست. این بدان معناست که رمزهای عبور شما باید تا حد امکان تصادفی، طولانی و شامل تنوعی از کاراکترها (حروف کوچک، حروف بزرگ، اعداد و نمادها) باشند.


اعمال این دو اصل امنیت رمز عبور (یگانگی و استحکام) می‌تواند در زندگی روزمره دشوار باشد، زیرا به خاطر سپردن یک رمز عبور منحصر به فرد، تصادفی و قوی برای همه حساب‌های ما تقریباً غیرممکن است. اینجاست که مدیر رمز عبور وارد عمل می‌شود.


یک مدیر رمز عبور، رمزهای عبور قوی تولید و به‌صورت امن ذخیره می‌کند و به شما اجازه می‌دهد به تمام حساب‌های آنلاین خود دسترسی داشته باشید بدون اینکه نیاز به حفظ کردن آنها به‌صورت جداگانه داشته باشید. شما فقط نیاز دارید یک رمز عبور را به خاطر بسپارید، رمز عبور اصلی، که به شما دسترسی به تمام رمزهای عبور ذخیره‌شده در مدیر را می‌دهد. استفاده از یک مدیر رمز عبور امنیت آنلاین شما را افزایش می‌دهد زیرا از استفاده مجدد از رمزهای عبور جلوگیری می‌کند و به‌صورت سیستماتیک رمزهای عبور تصادفی تولید می‌کند. اما همچنین استفاده روزانه شما از حساب‌هایتان را با متمرکز کردن دسترسی به اطلاعات حساس شما ساده می‌کند.

در این آموزش، ما یاد خواهیم گرفت که چگونه یک مدیر رمز عبور محلی را برای افزایش امنیت آنلاین خود راه‌اندازی و استفاده کنیم. در اینجا، من شما را با KeePass آشنا خواهم کرد. با این حال، اگر مبتدی هستید و می‌خواهید یک مدیر رمز عبور آنلاین داشته باشید که قادر به همگام‌سازی در چندین دستگاه باشد، توصیه می‌کنم آموزش ما در مورد Bitwarden را دنبال کنید:

## مقدمه‌ای بر KeePass


KeePass یک مدیر رمز عبور رایگان و متن‌باز است که برای کسانی که به دنبال یک راه‌حل رایگان و امن برای مدیریت محلی هستند، ایده‌آل است. این نرم‌افزار باید روی رایانه شما نصب شود و بدون افزودن پلاگین‌ها، با اینترنت ارتباطی ندارد. این رویکرد به‌طور ریشه‌ای با رویکرد Bitwarden متفاوت است که در یک آموزش قبلی به آن پرداختیم. Bitwarden برخلاف KeePass، امکان همگام‌سازی در چندین دستگاه را فراهم می‌کند و بنابراین نیاز به ذخیره رمزهای عبور شما روی یک سرور آنلاین دارد.


به طور پیش‌فرض، KeePass از استفاده از افزونه‌های مرورگر مانند Bitwarden پشتیبانی نمی‌کند؛ بنابراین، شما نیاز خواهید داشت که به صورت دستی رمزهای عبور خود را از نرم‌افزار کپی و جای‌گذاری کنید. اگرچه این ممکن است به نظر محدودیت بیاید، کپی و جای‌گذاری رمزهای عبور به جای استفاده از پر کردن خودکار، یک روش خوب برای امنیت آنلاین شماست.


KeePass به گونه‌ای طراحی شده است که هم سبک و هم آسان برای استفاده باشد، در حالی که به استانداردهای امنیتی بالا پایبند است. این نرم‌افزار پایگاه داده شما را به صورت محلی رمزگذاری می‌کند تا از اعتبارنامه‌های شما به بهترین شکل محافظت کند. KeePass همچنین تنها مدیر رمز عبور است که توسط ANSSI (مرجع امنیت سایبری فرانسه) تأیید شده است.


یکی از مزایای اصلی KeePass انعطاف‌پذیری آن است. می‌توان از آن به روش‌های مختلفی استفاده کرد، مانند استفاده بر روی یک USB بدون نیاز به نصب بر روی کامپیوتر. علاوه بر این، به لطف [محیط افزونه](https://keepass.info/plugins.html)، KeePass می‌تواند برای برآورده کردن نیازهای خاص‌تر سفارشی‌سازی شود.

## چگونه KeePass را دانلود کنیم؟


فرآیند نصب KeePass بسته به سیستم‌عاملی که استفاده می‌کنید متفاوت است. برای کاربران ویندوز یا لینوکس، نصب نسبتاً ساده است. با این حال، اگر از macOS استفاده می‌کنید، به دلیل توسعه KeePass بر روی پلتفرم .NET که به‌طور مستقیم توسط macOS پشتیبانی نمی‌شود، یک مرحله اضافی لازم است. بنابراین، شما باید یک محیط سازگار را پیکربندی کنید تا KeePass بتواند بر روی دستگاه‌های اپل اجرا شود.


برای کاربران Debian/Ubuntu، ترمینال را باز کرده و دستورات زیر را وارد کنید:


```bash
sudo apt-get update

sudo apt-get install keepass2

```

For Fedora:

```

sudo dnf install keepass

```

For Arch Linux:

```

sudo pacman -S keepass

```

If you are on a Windows computer, go to the [official KeePass download page](https://keepass.info/download.html), and download the latest version of the installer:
![KEEPASS](assets/notext/02.webp)
Click on the downloaded file to run it, then follow the instructions of the setup wizard to complete the installation (see next section).

For macOS users, the installation is a bit more complex. If you wish to use the original version of KeePass as on Windows, follow the instructions below. Otherwise, you can opt for [KeePassXC](https://keepassxc.org/), an alternative version compatible with macOS, which offers a slightly different interface.

To use KeePass, you will need a runtime environment for .NET applications. I recommend installing Mono for this. Go to the [official Mono page](https://www.mono-project.com/download/stable/#download-mac) in the "*macOS*" section, and click on the link to download the installation package (`.pkg`).
![KEEPASS](assets/notext/03.webp)
Open the downloaded `.pkg` file and follow the instructions to install Mono on your Mac.
![KEEPASS](assets/notext/04.webp)
Next, go to the official KeePass website and download the latest portable version in `.zip` format.
![KEEPASS](assets/notext/05.webp)
After downloading the `.zip` file, double-click to extract it. You will get a folder containing several files, including `KeePass.exe`. Open a terminal, navigate to the KeePass folder (replace `xx` with the version number):

```

cd ~/Downloads/KeePass-2.xx

```

And finally, run KeePass with Mono:

```

mono KeePass.exe

```
