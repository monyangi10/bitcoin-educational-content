ตามชื่อที่แนะนำ เบราว์เซอร์คือซอฟต์แวร์ที่ใช้ในการท่องอินเทอร์เน็ต มันทำหน้าที่เป็นประตูเชื่อมระหว่างเครื่องของผู้ใช้กับเว็บ แปลโค้ดของเว็บไซต์ให้เป็นหน้าเว็บที่โต้ตอบได้และอ่านได้ การเลือกเบราว์เซอร์ของคุณมีความสำคัญมาก เพราะมันไม่เพียงแต่มีผลต่อประสบการณ์การท่องเว็บของคุณเท่านั้น แต่ยังมีผลต่อความปลอดภัยและความเป็นส่วนตัวออนไลน์ของคุณด้วย


ระวังอย่าสับสนระหว่างเบราว์เซอร์กับเครื่องมือค้นหา เบราว์เซอร์คือซอฟต์แวร์ที่คุณใช้เพื่อเข้าถึงอินเทอร์เน็ต (เช่น Chrome หรือ Firefox) ในขณะที่เครื่องมือค้นหาคือบริการ เช่น Google หรือ Bing ที่ช่วยให้คุณค้นหาข้อมูลออนไลน์


วันนี้ Google Chrome เป็นเบราว์เซอร์ที่มีผู้ใช้มากที่สุดอย่างท่วมท้น โดยคิดเป็นประมาณ 65% ของตลาดโลกในปี 2024 Chrome ได้รับการชื่นชมในด้านความเร็วและประสิทธิภาพ แต่ก็ไม่จำเป็นต้องเป็นตัวเลือกที่ดีที่สุดสำหรับทุกคน โดยเฉพาะอย่างยิ่งหากความเป็นส่วนตัวเป็นสิ่งสำคัญสำหรับคุณ Chrome เป็นของ Google ซึ่งเป็นบริษัทที่มีชื่อเสียงในการรวบรวมและวิเคราะห์ข้อมูลจำนวนมหาศาลจากผู้ใช้ และในความเป็นจริง เบราว์เซอร์ภายในของพวกเขาเป็นหัวใจสำคัญของกลยุทธ์การเฝ้าระวังของพวกเขา ซอฟต์แวร์นี้เป็นส่วนประกอบสำคัญในปฏิสัมพันธ์ออนไลน์ส่วนใหญ่ของคุณ การควบคุมการรวบรวมข้อมูลบนเบราว์เซอร์ของคุณเป็นประเด็นสำคัญสำหรับ Google

มีครอบครัวหลักหลายกลุ่มของเบราว์เซอร์ แต่ละกลุ่มมีพื้นฐานจากเอนจินการเรนเดอร์เฉพาะ เบราว์เซอร์อย่าง Google Chrome, Microsoft Edge, Brave, Opera หรือ Vivaldi ล้วนมีพื้นฐานจากเบราว์เซอร์ Chromium ซึ่งเป็นเวอร์ชันที่เบาและโอเพ่นซอร์สของ Chrome ที่พัฒนาโดย Google เบราว์เซอร์เหล่านี้ทั้งหมดใช้เอนจินการเรนเดอร์ Blink ซึ่งเป็น fork ของ WebKit ที่มีต้นกำเนิดจาก KHTML การครอบครองตลาดของ Chromium ทำให้เบราว์เซอร์ที่พัฒนาจากมันมีประสิทธิภาพเป็นพิเศษ เนื่องจากนักพัฒนาเว็บมักจะปรับแต่งเว็บไซต์ของตนสำหรับ Blink เป็นหลัก


Safari ซึ่งเป็นเบราว์เซอร์ของ Apple ใช้ WebKit ซึ่งก็มาจาก KHTML เช่นกัน


ในทางกลับกัน เบราว์เซอร์อย่าง Mozilla Firefox, LibreWolf และ Tor Browser ใช้ Gecko ซึ่งเป็นเอนจินการเรนเดอร์ที่แตกต่างออกไป ซึ่งมีต้นกำเนิดมาจากเบราว์เซอร์ Netscape


การเลือกเบราว์เซอร์ที่เหมาะสมขึ้นอยู่กับความต้องการของคุณ แต่ถ้าคุณกังวลเกี่ยวกับความเป็นส่วนตัวของคุณ และดังนั้นความปลอดภัยของคุณ ฉันขอแนะนำให้ใช้ Firefox สำหรับการใช้งานทั่วไป และ Tor Browser สำหรับความเป็นส่วนตัวที่มากยิ่งขึ้น ในบทแนะนำนี้ ฉันจะแสดงวิธีเริ่มต้นใช้งาน Tor Browser อย่างง่ายดาย

## บทนำสู่ Tor Browser


Tor Browser เป็นเบราว์เซอร์ที่ออกแบบมาโดยเฉพาะเพื่อการท่องอินเทอร์เน็ตที่ปลอดภัยและเป็นส่วนตัวมากที่สุดเท่าที่จะเป็นไปได้ เบราว์เซอร์นี้มีพื้นฐานมาจาก Firefox และดังนั้นจึงใช้เอนจินเรนเดอร์ Gecko

Tor Browser ใช้เครือข่าย Tor เพื่อเข้ารหัสและส่งข้อมูลของคุณผ่านเซิร์ฟเวอร์รีเลย์หลายตัวก่อนที่จะส่งไปยังปลายทาง กระบวนการการส่งข้อมูลแบบหลายชั้นนี้ที่เรียกว่า "*onion routing*" ช่วยซ่อนที่อยู่ IP จริงของคุณ ทำให้ยากต่อการระบุตำแหน่งและกิจกรรมออนไลน์ของคุณ อย่างไรก็ตาม การท่องเว็บจะช้ากว่าการใช้เบราว์เซอร์มาตรฐานที่ไม่ใช้เครือข่าย Tor เนื่องจากเป็นการส่งข้อมูลทางอ้อม

ไม่เหมือนกับเบราว์เซอร์อื่น ๆ Tor Browser รวมคุณสมบัติเฉพาะเพื่อป้องกันการติดตามกิจกรรมออนไลน์ของคุณ เช่น การแยกเว็บไซต์ที่เข้าชมแต่ละแห่งและการลบคุกกี้และประวัติการเข้าชมโดยอัตโนมัติเมื่อปิด นอกจากนี้ยังออกแบบมาเพื่อลดความเสี่ยงของการระบุตัวตนโดยทำให้ผู้ใช้ทุกคนดูเหมือนกันมากที่สุดเท่าที่จะเป็นไปได้กับเว็บไซต์ที่เข้าชม


คุณสามารถใช้ Tor Browser เพื่อเข้าถึงเว็บไซต์มาตรฐาน (`.com`, `.org`, ฯลฯ) ได้อย่างดี ในกรณีนี้ การรับส่งข้อมูลของคุณจะถูกทำให้ไม่ระบุตัวตนโดยผ่านโหนด Tor หลายตัวก่อนที่จะถึงโหนดทางออกที่สื่อสารกับเว็บไซต์สุดท้ายบนเครือข่ายปกติ

![TOR BROWSER](assets/notext/03.webp)

คุณยังสามารถใช้ Tor Browser เพื่อเข้าถึงบริการที่ซ่อนอยู่ (ที่อยู่ที่ลงท้ายด้วย `.onion`) ในสถานการณ์นี้ การรับส่งข้อมูลทั้งหมดจะยังคงอยู่ภายในเครือข่าย Tor โดยไม่มีโหนดออก ซึ่งรับประกันความเป็นส่วนตัวอย่างสมบูรณ์สำหรับทั้งผู้ใช้และเซิร์ฟเวอร์ปลายทาง โหมดการทำงานนี้ถูกใช้เพื่อเข้าถึงสิ่งที่บางครั้งเรียกว่า "*dark web*" ซึ่งเป็นส่วนหนึ่งของอินเทอร์เน็ตที่ไม่ได้ถูกจัดทำดัชนีโดยเครื่องมือค้นหาแบบดั้งเดิม

![TOR BROWSER](assets/notext/04.webp)


## ความแตกต่างระหว่างเครือข่าย Tor และเบราว์เซอร์ Tor คืออะไร?


เครือข่าย Tor และเบราว์เซอร์ Tor เป็นสองสิ่งที่แตกต่างกันซึ่งไม่ควรสับสน แต่พวกมันทำงานร่วมกันได้ เครือข่าย Tor เป็นโครงสร้างพื้นฐานระดับโลกของเซิร์ฟเวอร์รีเลย์ ซึ่งดำเนินการโดยผู้ใช้ ซึ่งทำให้การจราจรทางอินเทอร์เน็ตไม่ระบุตัวตนโดยการส่งผ่านหลายโหนดก่อนที่จะนำไปยังปลายทางสุดท้าย นี่คือการกำหนดเส้นทางหัวหอมที่มีชื่อเสียง


ในทางกลับกัน เบราว์เซอร์ Tor เป็นเบราว์เซอร์เฉพาะที่ออกแบบมาเพื่ออำนวยความสะดวกในการเข้าถึงเครือข่ายนี้อย่างง่ายดาย มันรวมการตั้งค่าที่จำเป็นทั้งหมดเพื่อเชื่อมต่อกับเครือข่าย Tor โดยค่าเริ่มต้น และใช้เวอร์ชันที่ปรับปรุงของ Firefox เพื่อมอบประสบการณ์การท่องเว็บที่คุ้นเคยในขณะที่เพิ่มความเป็นส่วนตัวและความปลอดภัยสูงสุด


เครือข่าย Tor ไม่ได้ใช้เฉพาะกับเบราว์เซอร์ Tor เท่านั้น แต่ยังสามารถใช้กับซอฟต์แวร์และแอปพลิเคชันต่างๆ เพื่อรักษาความปลอดภัยในการสื่อสารของพวกเขาได้อีกด้วย ตัวอย่างเช่น คุณสามารถเปิดใช้งานการสื่อสารผ่านเครือข่าย Tor บนโหนด Bitcoin ของคุณเพื่อซ่อนที่อยู่ IP ของคุณจากผู้ใช้รายอื่นและป้องกันการสอดส่องการจราจรที่เกี่ยวข้องกับ Bitcoin ของคุณโดยผู้ให้บริการอินเทอร์เน็ตของคุณ

โดยสรุปแล้ว เครือข่าย Tor คือโครงสร้างพื้นฐานที่ให้ความเป็นส่วนตัวในการท่องอินเทอร์เน็ตของเรา และ Tor Browser คือซอฟต์แวร์ที่ช่วยให้เราสามารถใช้เครือข่ายนี้เป็นส่วนหนึ่งของการท่องเว็บของเรา

![TOR BROWSER](assets/notext/12.webp)

เมื่อเปิดใช้งานครั้งแรก เบราว์เซอร์จะแสดงหน้าที่เชิญชวนให้คุณเชื่อมต่อกับเครือข่าย Tor เพียงคลิกที่ปุ่ม "*Connect*" เพื่อสร้างการเชื่อมต่อ

![TOR BROWSER](assets/notext/13.webp)

หากคุณต้องการให้ซอฟต์แวร์เชื่อมต่อกับเครือข่าย Tor โดยอัตโนมัติในครั้งถัดไปที่คุณใช้งาน ให้เลือกตัวเลือก "*เชื่อมต่อโดยอัตโนมัติเสมอ*"

![TOR BROWSER](assets/notext/14.webp)

ในการค้นหาบนอินเทอร์เน็ต เพียงป้อนคำค้นหาของคุณในแถบค้นหาและกดปุ่ม "*enter*"

จากนั้น คุณจะได้รับผลลัพธ์จากเครื่องมือค้นหาของคุณในลักษณะเดียวกับเบราว์เซอร์อื่น ๆ

## วิธีการกำหนดค่า Tor Browser?


ที่ด้านบนของหน้าจอเบราว์เซอร์ของคุณ คุณจะพบตัวเลือกในการนำเข้ารายการโปรดของคุณ ซึ่งจะช่วยให้คุณสามารถรวมบุ๊กมาร์กจากเบราว์เซอร์เก่าของคุณเข้าสู่ Tor Browser ได้โดยอัตโนมัติ
