## Introduction

Bitcoin est un projet open source, transparent, décentralisé…

Plus vous vous enfoncez dans le terrier du lapin — plus vous vous intéressez au Bitcoin au-delà de l'aspect spéculatif — plus vous devenez bitcoiner.

De facto, un désir naturel émerge en vous et vous incite à découvrir, adopter, et à partager avec vos pairs des projets qui partagent ces mêmes valeurs et qui développent des applications indispensables dans nos activités quotidiennes.

Dans ce tutoriel, allons à la découverte  d'un outil de gestion de la vie privée qui limite les permissions et l'accès à vos données aux applications installées sur votre smartphone : **Shelter**.

**Shelter** est une application mobile, libre et open source (FOSS) qui vous fournit un espace isolé où vous avez la possibilité d'installer ou de cloner des applications.    

Selon le dépôt GitHub de Shelter, les principaux cas d'utilisation sont :  
- l'installation d'applications dans le "Work Profil" pour les isoler ;  
- le gel des applications dans le "Work Profil" pour les empêcher de s'exécuter ou de se réactiver lorsque vous ne les utilisez pas activement ;
- l'installation de deux copies de la même application sur le même téléphone (clonage).

Pour fonctionner, Shelter exploite la fonctionnalité **Work Profil** (Profil professionnel) du système d'exploitation mobile **Android**. Par conséquent, Shelter n'est pas disponible pour les utilisateurs du système IOS d'Apple. 

Mais qu'est-ce que la fonctionnalité **Work Profile** (Profil Professionnel) ?

## La fonctionnalité Work Profil 

En effet, c'est une fonctionnalité qui vous permet de séparer vos données et applications professionnelles de vos données et applications personnelles. Cela vous permet donc d'utiliser le même téléphone pour le travail et pour la vie privée. 
Votre entreprise peut uniquement supprimer le profil professionnel ou certaines applications de ce profil sans toucher à vos données personnelles.

Point important ❗️ Shelter dépend entièrement de la qualité de l’implémentation du **Work Profile** par Android sur votre téléphone. Si votre système a des bugs liés à cette fonctionnalité, le fonctionnement de Shelter en sera affecté.  
 

## Débuter avec Shelter

### Installer Shelter

Rendez-vous sur le magasin d'applications et d'outils libres et open source **F-droid**. 

https://planb.academy/tutorials/computer-security/data/f-droid-2cd1aae5-7028-4c04-8fbe-95aeaf278ef4

D'abord, sur l'interface **Nouveautés** de F-droid, cliquez sur l'icône de recherche, et tapez **Shelter** dans la barre de recherche.  

![screen](assets/fr/03.webp)

![screen](assets/fr/04.webp)

Ensuite, cliquez sur son logo lorsqu'il apparait dans la liste de résultat de recherche. Vous serez redirigé vers une interface où vous trouverez notamment une description détaillée de l'application, ses différentes versions…

Enfin, cliquez sur le bouton **Installer**.

![screen](assets/fr/05.webp)

 Puis, tapez sur **Installer** à nouveau dans le menu contextuel qui apparaît pour autoriser ladite installation.

![screen](assets/fr/06.webp)

En outre, vous avez la possibilité d'installer Shelter depuis son dépôt [Git](https://gitea.angry.im/PeterCxy/Shelter).

![screen](assets/fr/07.webp)

Votre application est installée, appuyez sur le bouton **Ouvrir** pour y accéder.

![screen](assets/fr/09.webp)

Un petit guide vous est présenté lors de la première ouverture de l'application. Prenez le temps de lire les informations, c'est un plus dans la compréhension de l'utilisation de Shelter. Appuyez sur **Suivant** à chaque fois, afin de parcourir tout le guide.

![screen](assets/fr/10.webp)

Shelter vous rappelle que vous devez vous assurer que votre téléphone n'est pas en mode **Ne pas déranger**, avant de lancer le processus de configuration de l'application.

![screen](assets/fr/11.webp)

Appuyez sur le bouton **Accepter et continuer**, attendez la finalisation de la configuration de votre profil professionnel puis tapez sur le bouton **Suivant**.

### Tour rapide des interfaces

Une fois dans l'application, vous avez :

- l'interface **Profil Principal** (espace personnel) qui renferme la liste de toutes vos applications personnelles installées sur votre téléphone ;

![screen](assets/fr/14.webp)

- et l'interface **Shelter** (espace du profil professionnel) qui contient par défaut un certains nombre d'applications comme Android Switch, Assistant Google, etc.

![screen](assets/fr/15.webp)

Dans le coin supérieur droit, vous disposez : 

- de l'icône de **barre de recherche** pour rechercher telle ou telle application selon l'interface sur laquelle vous vous situez (1) ;
- de l'icône de **gel** de toutes les applications de l'interface sur laquelle vous vous situez (2)

![screen](assets/fr/16.webp)

- et de l'icône des **trois points** qui vous permet d'accéder aux paramètres, d'installer un APK dans Shelter… 

![screen](assets/fr/17.webp)                        

![screen](assets/fr/18.webp)

En fonction du lanceur d'applications (Launcher) que vous utilisez sur votre téléphone, le menu d'applications peut être divisé en deux onglets (**Personnelles et Professionnelles**) bien visibles (1) ou carrément vos applications professionnelles sont regroupées dans un dossier nommé dans le cas d'espèce **Android Work** sur l'écran d'accueil (2).

![screen](assets/fr/19.webp)

![screen](assets/fr/20.webp)

Toutefois, quel que soit l'endroit où elles sont visibles sur votre téléphone, les applications du profil professionnel se distinguent par une petite **icône de valise/porte-documents bleue** dans le coin inférieur droit de leur icône.

![screen](assets/fr/21.webp)

Pour utiliser une application depuis **Shelter**, il suffit de cliquer sur l'application et d'appuyez sur **Lancer** lorsque le menu contextuel apparait.

![screen](assets/fr/22.webp)

## Installation d'applications dans le Work Profil

Sur l'interface **Shelter**, cliquez sur l'icône de l'application Google Play Store pendant un petit moment. Puis sur le menu contextuel (écran d'information) qui apparait, tapez sur **Lancer**. 

![screen](assets/fr/23.webp)

Appuyez sur le bouton **Connexion** et ajoutez une adresse e-mail professionnelle pour vous connecter.

![screen](assets/fr/24.webp)

![screen](assets/fr/25.webp)

Accédez ensuite à Play store et téléchargez les applications professionnelles dont vous avez besoin.
Comme vous pouvez le constater, l'application **Signal**, une fois téléchargée, se retrouve dans la liste de vos applications professionnelles sur l'interface **Shelter**.

![screen](assets/fr/26.webp)

![screen](assets/fr/27.webp)

Découvrez comment prendre Signal en main en consultant le tutoriel suivant :

https://planb.academy/tutorials/computer-security/communication/signal-8dfb5572-6962-4f1c-bfa5-3192da4e9a4e

Par ailleurs, vous avez la possibilité d'installer dans Shelter une application téléchargée sous format APK (Android Package Kit) sur votre téléphone.
Le format de fichier **APK**  est le format standard utilisé par Android pour la distribution et l'installation d'applications mobiles.

Pour le faire, tapez sur l'icône des **trois points** puis sur **Installer une APK dans Shelter**. Vous serez redirigé vers le stockage interne de votre téléphone pour choisir le fichier APK à installer en cliquant dessus.

Essayons d'installer le fichier APK de cette application **GPS local**. 

![screen](assets/fr/28.webp)

Donnez l'autorisation à Shelter de pouvoir installer des applications sur votre téléphone en activant le bouton.

![screen](assets/fr/29.webp)

![screen](assets/fr/30.webp)

Appuyez sur le bouton **Installer** pour installer l'application.

![screen](assets/fr/31.webp)

Bingo ! Notre application est maintenant visible dans la liste des applications professionnelles et fonctionne correctement.

![screen](assets/fr/32.webp)

## Gel d'applications dans le Work Profil

Pour geler une application, rendez-vous sur Shelter. Accédez à l'onglet des applications professionnelles. À partir de cet onglet, sélectionnez l'application à geler.

Appuyez sur l'icône de l'application pendant un petit moment jusqu'à l'apparition du menu contextuel. Dans ce menu contextuel, cliquez sur **Geler**.

![screen](assets/fr/34.webp)

Une fois l'application gelée, elle qui était visible dans le dossier **Android Work** (1) devient invisible (2). Vous pouvez le dégeler quand vous le souhaitez dans Shelter.

![screen](assets/fr/35.webp)

![screen](assets/fr/36.webp)

## Clonage d'une application

Dans Shelter, vous pouvez cloner directement une application que vous utilisez déjà dans votre espace personnel. 
Pour cela, rendez-vous dans l'onglet **Profil Principal** puis cliquez sur l'application à cloner pendant un petit moment jusqu'à l'apparition du petit menu contextuel. 

Essayons le clonage de l'application **Telegram**. 
Lorsque le menu contextuel apparait, tapez sur **Cloner vers Shelter (Profil Professionnel)**.

![screen](assets/fr/37.webp)

Acceptez l'installation en cliquant sur **Installer**.

![screen](assets/fr/38.webp)

**Telegram** est bien installé et maintenant visible dans l'onglet **Shelter**.

![screen](assets/fr/39.webp)

Le processus inverse aussi est possible, c'est-à-dire cloner une application disponible dans l'onglet **professionnel** vers l'onglet **principal (personnel)**. 
Pour le faire, depuis l'onglet professionnel, appuyez sur l'application à cloner pour faire apparaitre le menu contextuel. Puis cliquez sur **Cloner vers le Profil Principal**. Essayons de cloner l'application GPS Local. 

![screen](assets/fr/40.webp)

Appuyez sur **Installer** pour accepter l'installation.

![screen](assets/fr/41.webp)

L'application est désormais disponible dans votre menu d'applications personnelles.

![screen](assets/fr/42.webp)

## Désinstallation de Shelter

Diverses raisons peuvent vous amener à désinstaller Shelter. Soit parce que votre nouvel emploi ne nécessite plus l'utilisation, soit parce que vous venez d'acquérir un nouveau téléphone exclusivement dédié à votre travail… 
Quoi qu'il en soit, vous devez supprimer avant tout le **Profil Professionnel**. Dans les **paramètres** de votre téléphone, rendez-vous dans **Mots de passe et Comptes*** puis supprimez le profil professionnel.

![screen](assets/fr/43.webp)

Ensuite, revenez dans le menu de vos applications et désinstallez l'application **Shelter** maintenant.
