## Einführung



Bitcoin ist ein quelloffenes, transparentes, dezentralisiertes Projekt.



Je tiefer man in den Kaninchenbau einsteigt - je mehr man sich für Bitcoin über den spekulativen Aspekt hinaus interessiert - desto mehr wird man zum Bitcoiner.



De facto entsteht in Ihnen der natürliche Wunsch, Projekte zu entdecken, zu übernehmen und mit Gleichgesinnten zu teilen, die dieselben Werte teilen und unverzichtbare Anwendungen für unsere täglichen Aktivitäten entwickeln.



In diesem Tutorial lernen wir ein Tool zur Verwaltung der Privatsphäre kennen, das die Berechtigungen und den Zugriff auf Ihre Daten auf Anwendungen beschränkt, die auf Ihrem Smartphone installiert sind: **Shelter**.



**Shelter** ist eine freie und quelloffene (FOSS) mobile Anwendung, die Ihnen einen isolierten Bereich bietet, in dem Sie Anwendungen installieren oder klonen können.



Laut dem GitHub-Repository von Shelter sind die wichtigsten Anwendungsfälle :




- installation von Anwendungen im "Arbeitsprofil", um sie zu isolieren;
- anwendungen im "Arbeitsprofil" einfrieren, um zu verhindern, dass sie ausgeführt oder reaktiviert werden, wenn Sie sie nicht aktiv nutzen;
- die Installation von zwei Kopien derselben Anwendung auf demselben Telefon (Klonen).



Shelter nutzt die Funktion **Arbeitsprofil** des mobilen Betriebssystems **Android**. Aus diesem Grund ist Shelter für Apple IOS-Nutzer nicht verfügbar.



Aber was ist die Funktion **Arbeitsprofil**?



## Die Funktion Arbeitsprofil



Mit dieser Funktion können Sie Ihre geschäftlichen Daten und Anwendungen von Ihren privaten trennen. Das bedeutet, dass Sie dasselbe Telefon sowohl für die Arbeit als auch für Ihr Privatleben nutzen können.


Ihr Unternehmen kann nur das berufliche Profil oder bestimmte Anwendungen aus diesem Profil löschen, ohne dass Ihre persönlichen Daten davon betroffen sind.



Wichtig ❗️ Shelter hängt vollständig von der Qualität der Android-Implementierung des **Arbeitsprofils** auf Ihrem Telefon ab. Wenn Ihr System Fehler im Zusammenhang mit dieser Funktion aufweist, wird der Betrieb von Shelter beeinträchtigt.




## Erste Schritte mit Shelter



### Unterstand installieren



Besuchen Sie den **F-droid** Store für kostenlose und quelloffene Anwendungen und Tools.



https://planb.academy/tutorials/computer-security/data/f-droid-2cd1aae5-7028-4c04-8fbe-95aeaf278ef4

Klicken Sie zunächst auf der Oberfläche **Neue Funktionen** von F-droid auf das Suchsymbol und geben Sie **Shelter** in die Suchleiste ein.



![screen](assets/fr/03.webp)



![screen](assets/fr/04.webp)



Klicken Sie dann auf das Logo, wenn es in der Liste der Suchergebnisse erscheint. Sie werden zu einer Schnittstelle weitergeleitet, auf der Sie eine detaillierte Beschreibung der Anwendung und ihrer verschiedenen Versionen finden.



Klicken Sie abschließend auf die Schaltfläche **Installieren**.



![screen](assets/fr/05.webp)



Tippen Sie dann im erscheinenden Kontextmenü erneut auf **Installieren**, um die Installation zu autorisieren.



![screen](assets/fr/06.webp)



Sie können Shelter auch aus seinem [Git]-Repository (https://gitea.angry.im/PeterCxy/Shelter) installieren.



![screen](assets/fr/07.webp)



Drücken Sie auf die Schaltfläche **Öffnen**, um auf Ihre Anwendung zuzugreifen.



![screen](assets/fr/09.webp)



Wenn Sie die Anwendung zum ersten Mal öffnen, wird Ihnen ein kurzer Leitfaden angezeigt. Nehmen Sie sich die Zeit, die Informationen zu lesen. Sie sind eine große Hilfe, um zu verstehen, wie man Shelter benutzt. Drücken Sie jedes Mal auf **Weiter**, um die gesamte Anleitung durchzugehen.



![screen](assets/fr/10.webp)



Shelter erinnert Sie daran, dass Sie sicherstellen sollten, dass sich Ihr Telefon nicht im **Nicht stören**-Modus befindet, bevor Sie mit der Konfiguration der Anwendung beginnen.



![screen](assets/fr/11.webp)



Drücken Sie die Schaltfläche **Akzeptieren und fortfahren**, warten Sie, bis die Konfiguration Ihres Profils abgeschlossen ist, und drücken Sie dann die Schaltfläche **Weiter**.



![screen](assets/fr/12.webp)



![screen](assets/fr/13.webp)



### Ein kurzer Rundgang durch die Schnittstellen



Sobald Sie in der Anwendung sind, haben Sie :





- die Schnittstelle **Hauptprofil** (persönlicher Bereich), die eine Liste all Ihrer persönlichen Anwendungen enthält, die auf Ihrem Telefon installiert sind;



![screen](assets/fr/14.webp)





- und die **Shelter**-Schnittstelle (professioneller Profilbereich), die standardmäßig eine Reihe von Anwendungen enthält, wie Android Switch, Google Assistant usw.



![screen](assets/fr/15.webp)



In der rechten oberen Ecke finden Sie :





- das Symbol **Suchleiste**, um eine bestimmte Anwendung zu finden, je nachdem, auf welcher Oberfläche Sie sich befinden (1) ;
- das **Gel**-Symbol für alle Anwendungen auf der Oberfläche, auf der Sie sich gerade befinden (2)



![screen](assets/fr/16.webp)





- und das Symbol mit den **drei Punkten**, mit dem Sie auf die Einstellungen zugreifen und eine APK in Shelter... installieren können.



![screen](assets/fr/17.webp)



![screen](assets/fr/18.webp)



Je nachdem, welchen Launcher Sie auf Ihrem Telefon verwenden, kann das Anwendungsmenü in zwei deutlich sichtbare Registerkarten unterteilt sein (**Persönlich und Professionell**) (1), oder Ihre professionellen Anwendungen sind in einem Ordner namens **Android Work** auf dem Startbildschirm zusammengefasst (2).



![screen](assets/fr/19.webp)



![screen](assets/fr/20.webp)



Anwendungen mit professionellem Profil sind jedoch überall auf Ihrem Telefon durch ein kleines **blaues Koffersymbol** in der rechten unteren Ecke ihres Symbols gekennzeichnet.



![screen](assets/fr/21.webp)



Um eine Anwendung aus **Shelter** heraus zu nutzen, klicken Sie einfach auf die Anwendung und drücken Sie **Starten**, wenn das Kontextmenü erscheint.



![screen](assets/fr/22.webp)



## Installation von Anwendungen im Arbeitsprofil



Klicken Sie auf der Oberfläche **Shelter** kurz auf das Symbol der Anwendung Google Play Store. Tippen Sie dann im Kontextmenü (Informationsbildschirm), das erscheint, auf **Starten**.



![screen](assets/fr/23.webp)



Drücken Sie die Schaltfläche **Verbinden** und fügen Sie eine geschäftliche E-Mail-Adresse hinzu, um eine Verbindung herzustellen.



![screen](assets/fr/24.webp)



![screen](assets/fr/25.webp)



Gehen Sie dann zum Play Store und laden Sie die gewünschten Geschäftsanwendungen herunter.


Wie Sie sehen können, erscheint die **Signal**-Anwendung nach dem Herunterladen in der Liste Ihrer professionellen Anwendungen auf der **Shelter**-Oberfläche.



![screen](assets/fr/26.webp)



![screen](assets/fr/27.webp)



In der folgenden Anleitung erfahren Sie, wie Sie mit Signal arbeiten können:



https://planb.academy/tutorials/computer-security/communication/signal-8dfb5572-6962-4f1c-bfa5-3192da4e9a4e

Sie können auch eine im APK-Format (Android Package Kit) heruntergeladene Anwendung auf Ihrem Telefon installieren.


Das **APK**-Dateiformat ist das von Android verwendete Standardformat für die Verteilung und Installation von mobilen Anwendungen.



Tippen Sie dazu auf das Symbol **drei Punkte** und dann auf **APK in Shelter installieren**. Du wirst zum internen Speicher deines Telefons weitergeleitet, wo du die APK-Datei zur Installation auswählen kannst, indem du darauf klickst.



Versuchen wir, die APK-Datei für diese **GPS local**-Anwendung zu installieren.



![screen](assets/fr/28.webp)



Geben Sie Shelter die Erlaubnis, Anwendungen auf Ihrem Telefon zu installieren, indem Sie die Schaltfläche aktivieren.



![screen](assets/fr/29.webp)



![screen](assets/fr/30.webp)



Drücken Sie die Schaltfläche **Installieren**, um die Anwendung zu installieren.



![screen](assets/fr/31.webp)



Bingo! Unsere Anwendung ist nun in der Liste der professionellen Anwendungen sichtbar und funktioniert einwandfrei.

## Bewerbungsstopp im Arbeitsprofil



Um eine Bewerbung einzufrieren, gehen Sie zu Shelter. Gehen Sie zur Registerkarte Professionelle Anwendungen. Wählen Sie auf dieser Registerkarte die Anwendung aus, die Sie einfrieren möchten.



Drücken Sie einen Moment auf das Anwendungssymbol, bis das Kontextmenü erscheint. Klicken Sie in diesem Kontextmenü auf **Freeze**.



![screen](assets/fr/34.webp)



Sobald sie eingefroren ist, wird die Anwendung, die im Ordner **Android Work** (1) sichtbar war, unsichtbar (2). Sie können sie jederzeit im Shelter wieder freigeben.



![screen](assets/fr/35.webp)



![screen](assets/fr/36.webp)



## Klonen einer Anwendung



In Shelter können Sie eine Anwendung, die Sie bereits verwenden, direkt in Ihren persönlichen Bereich klonen.


Gehen Sie dazu auf die Registerkarte **Hauptprofil** und klicken Sie einen Moment auf die Anwendung, die Sie klonen möchten, bis das kleine Kontextmenü erscheint.



Versuchen wir nun, die Anwendung **Telegram** zu klonen.


Wenn das Pop-up-Menü erscheint, tippen Sie auf **Cloner to Shelter (Professional Profile)**.



![screen](assets/fr/37.webp)



Akzeptieren Sie die Installation, indem Sie auf **Installieren** klicken.



![screen](assets/fr/38.webp)



**Telegram** ist nun installiert und auf der Registerkarte **Shelter** sichtbar.



![screen](assets/fr/39.webp)



Der umgekehrte Prozess ist ebenfalls möglich, d. h. das Klonen einer Anwendung, die auf der Registerkarte **Professionell** verfügbar ist, auf die Registerkarte **Haupt (persönlich)**.


Tippen Sie dazu auf der Registerkarte Profi auf die zu klonende Anwendung, um das Kontextmenü aufzurufen. Klicken Sie dann auf **Klonen zum Hauptprofil**. Versuchen wir nun, die Anwendung GPS Local zu klonen.



![screen](assets/fr/40.webp)



Drücken Sie **Install**, um die Installation zu bestätigen.



![screen](assets/fr/41.webp)



Die Anwendung ist nun in Ihrem persönlichen Anwendungsmenü verfügbar.



![screen](assets/fr/42.webp)



## Deinstallation von Shelter



Es gibt viele Gründe, warum Sie Shelter deinstallieren möchten. Entweder, weil Sie ihn in Ihrem neuen Job nicht mehr benötigen oder weil Sie gerade ein neues Telefon erworben haben, das ausschließlich für Ihre Arbeit bestimmt ist...


In jedem Fall müssen Sie zuerst das **Professional-Profil** löschen. Gehen Sie in den **Einstellungen** Ihres Telefons zu **Kennwörter und Konten*** und löschen Sie das Profi-Profil.



![screen](assets/fr/43.webp)



Gehen Sie dann zurück zu Ihrem Anwendungsmenü und deinstallieren Sie die Anwendung **Shelter** jetzt.



Jetzt wissen Sie, wie Sie mit der Shelter-Anwendung ein **Arbeitsprofil** erstellen können, das von Ihren persönlichen Anwendungen isoliert und getrennt ist. Wenn Sie diese Anleitung nützlich fanden, hinterlassen Sie mir bitte einen grünen Daumen. Sie können es gerne weitergeben.
