오늘날 웹 브라우저 환경에서 Google Chrome은 65% 이상의 시장 점유율로 독주하고 있지만 이러한 헤게모니는 개인정보 보호와 기술 다양성에 대한 중요한 의문을 제기합니다. 대부분의 인기 브라우저와 마찬가지로 Chrome은 검색 데이터를 대량으로 수집하여 Google의 광고 알고리즘에 공급합니다.



![Parts de marché des navigateurs](assets/fr/01.webp)



이러한 현실에 직면하여 최신 사용자 경험과 프라이버시 존중이라는 다른 철학을 가진 새로운 브라우저들이 등장하고 있습니다. 2024년에 출시된 Zen Browser는 이러한 접근 방식의 일환으로, Firefox를 기반으로 하지만 오늘날의 사용자를 위해 재설계된 혁신적인 대안을 제공합니다.



Zen 브라우저는 세로 탭, 세션을 정리할 수 있는 작업 공간, 분할 보기와 같은 생산성 기능을 갖춘 독특한 Interface이 돋보입니다. 하지만 이러한 인체공학적 혁신 외에도 원격 측정 기능 없음, 강화된 추적 방지 보호 기능, 투명한 커뮤니티 개발 등 개인 정보 보호에 대한 Commitment이 이 브라우저를 더욱 흥미롭게 만듭니다.



이 튜토리얼에서는 Zen 브라우저가 생산성, 개인화 및 개인 정보 보호를 결합하여 브라우징 방식을 어떻게 변화시킬 수 있는지 알아보세요.

## Zen 브라우저 소개



**Zen Browser**는 2024년 3월부터 열정적인 자원봉사자 커뮤니티가 개발한 Mozilla Firefox에서 파생된 무료 오픈 소스 웹 브라우저입니다. 주요 기술 회사의 브라우저와 달리 Zen 브라우저는 상업적 기업의 지원을 받지 않으며 커뮤니티의 기부금으로만 자금을 조달합니다.



**중요 참고:** Zen 브라우저는 현재 **베타 버전**입니다. 일상적인 사용에는 안정적이지만, 잦은 업데이트와 가끔 버그가 발생할 수 있습니다.



이 프로젝트의 철학은 "더 차분한 인터넷에 오신 것을 환영합니다"라는 슬로건으로 요약됩니다. 이러한 접근 방식은 개인 데이터보다는 사용자 경험을 중시하는 브라우저를 지향하며 아름다움, 속도, 프라이버시 사이의 완벽한 균형을 추구합니다.



### 생산성을 위해 재설계된 Interface



Zen 브라우저는 몇 가지 혁신으로 브라우징 경험을 혁신합니다:



**세로 탭: ** 기존 브라우저와 달리 Zen은 탭을 가로가 아닌 세로 사이드바에 표시합니다. Arc 브라우저에서 영감을 얻은 이 인체공학적 선택은 디스플레이 공간을 최대화하고 특히 많은 탭이 열려 있을 때 탐색을 개선합니다.



**작업 공간: ** 전용 공간에서 프로젝트 또는 테마별로 탭을 정리하세요. 예를 들어, '작업' 공간에는 업무용 탭을, '보기' 공간에는 독서용 탭을 배치하는 식입니다. 한 공간에서 다른 공간으로 즉시 전환할 수 있습니다.



**분할 보기:** 여러 사이트를 하나의 창에 나란히 표시하여 정보를 비교하거나 여러 소스를 동시에 작업하는 데 이상적입니다.



**한눈에 보기:** 새 탭을 열지 않고도 임시 모달 창에서 링크를 빠르게 미리 볼 수 있어 문맥을 잃지 않고 참고자료를 참조하는 데 적합합니다.



### 기본 개인정보 보호



Zen 브라우저는 기본적으로 강력한 개인 정보 보호 기능을 통합하고 있습니다:





- 향상된 추적 방지 기능: ** 추적기, 타사 쿠키 및 지문 스크립트 자동 차단
- 원격 분석 사용 안 함:** 외부 서버로 데이터 전송 안 함
- DNS over HTTPS:** 모니터링을 방지하기 위해 DNS 요청을 암호화합니다
- Google 종속성 감소:** Zen 브라우저는 Google 서비스에 대한 대부분의 연결을 제거하지만 일부는 남아있을 수 있습니다(보안 브라우징, 업데이트)



### Zen 모드를 통한 고급 사용자 지정



Zen은 브라우저의 모양과 동작을 변화시키기 위해 커뮤니티에서 만든 테마 및 수정 사항 갤러리인 **Zen Mods**를 통해 독특한 사용자 지정 에코시스템을 제공합니다.



**인기 Zen 모드 추천:**





- 슈퍼핀** ⭐: 고정된 탭을 스타일화된 버튼으로 전환하여 보다 전문적인 Interface 룩을 연출할 수 있습니다
- 응집력**: URL 표시줄, 사이드바, 북마크를 통합하는 일관되고 투명한 스타일링
- 더 나은 찾기 표시줄**: 인체공학적 개선을 위해 검색창을 상단으로 이동합니다
- 호버링 시 사이드바 확장**: 마우스오버 시 사이드바 자동 확장, 화면 공간 최대화
- 언로드된 탭 개선**: 비활성 탭에 대한 시각적 표시기를 통해 메모리 관리 최적화
- 정리된 URL 바**: Interface 정제된 Address 바, 불필요한 Elements 제거
- 투명 젠**: 부드러운 애니메이션과 우아한 투명도 효과



**Zen 모드 설치:**




- 공식 Zen 모드 갤러리](https://zen-browser.app/mods)로 이동합니다
- 사용 가능한 테마 갤러리 찾아보기



![Galerie Zen Mods](assets/fr/12.webp)





- 원하는 모드의 "설치"를 클릭합니다



![Installation SuperPins](assets/fr/13.webp)


*예시: 인기 있는 슈퍼핀* 모드 설치하기





- 테마가 즉시 적용됩니다
- 언제든지 비활성화하거나 다른 기능을 사용해 볼 수 있습니다



Zen 모드는 시각적 테마에만 국한되지 않고 Interface의 동작(애니메이션, 요소 레이아웃, 사용자 지정 단축키)을 수정하기도 합니다. 이러한 모듈식 접근 방식을 통해 각 사용자는 자신만의 이상적인 브라우징 환경을 만들 수 있습니다.



![Gestion des Zen Mods](assets/fr/16.webp)


*파라미터에 설치된 젠 모드 관리를 위한 Interface**



**⚠️ 개인화 및 지문 인식에 대한 중요 참고 사항 :**


Zen 브라우저(테마, 확장 프로그램, 모드)를 더 많이 커스터마이징할수록 사용자의 **디지털 발자국**이 고유해져서 추적할 수 있게 됩니다. 이는 근본적인 타협입니다:




- 개인화 극대화** = 최적의 사용자 경험, 그러나 쉽게 식별할 수 있는 고유한 각인
- 기본 구성** = 일반적인 설치 공간은 많지만 개인화된 경험은 적습니다



Zen Browser는 완벽한 익명성보다 사용자 경험을 선택했습니다. 완벽한 익명성을 우선시한다면, 모든 사용자에게 균일한 구성을 적용하는 Tor 브라우저나 Mullvad 브라우저를 추천합니다.



또한 Zen은 Firefox 기반이기 때문에 전체 Firefox 확장 프로그램 생태계와 호환됩니다.



## 장점과 단점



### ✅ 하이라이트





- 개인 정보 보호 설계:** 추적 방지 보호 활성화, 원격 측정 비활성화, 데이터 수집 금지
- 혁신적인 Interface:** 세로 탭, 작업 공간, 분할 보기로 생산성 획기적 향상
- 빠른 업데이트: ** 보안 패치를 위해 72시간 이내에 Firefox와 동기화합니다
- 고급 사용자 지정:** 커뮤니티 테마, 미세 조정, Firefox 확장 프로그램 호환성
- 오픈 소스 및 커뮤니티: ** 투명한 코드, 협업 개발, 빅 테크로부터의 독립성



### ❌ 전류 제한





- 모바일 버전 없음:** PC(Windows, macOS, Linux)에서만 사용 가능
- DRM 비호환성: ** Netflix, Disney+, Spotify 및 기타 스트리밍 서비스는 현재 작동하지 않습니다
- 젊은 프로젝트:** 소규모 팀, 커뮤니티 지원, 가끔 버그 발생
- 학습 곡선:** 가로 탭에 익숙한 사용자에게는 적응이 필요하며, Interface이 다릅니다

### 최적화된 기본 설정



Zen 브라우저는 처음부터 대부분의 브라우저를 능가하는 높은 수준의 개인 정보 보호 설정을 적용합니다:





- 엄격한 추적 방지 보호: ** 기본적으로 '표준' 수준이 활성화되어 차단됩니다:
  - 교차 사이트 추적 쿠키 및 슈퍼쿠키
  - 광고 트래커 스크립트(구글 애널리틱스, 페이스북 픽셀 등)
  - CPU를 사용하여 Miner 암호화폐를 생성하는 크립토마이너
  - 캔버스, WebGL, 오디오 컨텍스트를 통한 핑거프린팅





- 완전한 쿠키 격리: ** 퍼스트 파티 격리는 한 사이트가 다른 사이트의 쿠키를 읽지 못하도록 차단합니다
- 원격 측정이 대부분 비활성화됨:** 대부분의 데이터 수집이 제거되었지만 Mozilla/Google 서비스에 대한 일부 연결이 남아 있을 수 있으며 추가 수동 구성이 필요할 수 있습니다
- 기본적으로 보안 DNS 사용:** 요청 감시를 방지하기 위해 DNS-over-HTTPS 활성화
- HTTPS 전용 사용:** 모든 사이트에서 암호화 연결 강제 사용

**표준(기본값 권장):**




- 보호와 성능 간의 균형, 페이지가 정상적으로 로드됨
- 차단: 소셜 트래커, 사이트 간 쿠키, 비공개 창에서 콘텐츠 추적, 크립토마이닝, 지문 인식기
- 전체 쿠키 보호** 포함: 사이트별로 쿠키를 분리하여 사이트 간 추적을 방지합니다



**엄격(최대 보호):**




- 보호 기능이 강화되었지만 특정 사이트 또는 콘텐츠가 손상될 수 있습니다
- 차단: 소셜 트래커, 교차 사이트 쿠키, **모든** 창에서 콘텐츠 추적, 알려진 브라우저 및 의심되는 브라우저
- 숙련된 사용자에게 권장



**사용자 지정(세분화된 제어):**




- 차단할 트래커와 스크립트를 정확하게 선택하세요
- 별도의 옵션: 쿠키, 콘텐츠 추적, 크립토마이닝, 알려진/의심되는 추적기

- 시작 페이지**: 익명화된 Google 검색 결과, 네덜란드 기준(RGPD)
- Searx**: 분산형 메타검색 엔진, 로그 없음, 오픈 소스
- 브레이브 검색**: Google이 아닌 독립 색인
- 피하기**: 구글, 빙, 야후(대규모 데이터 수집)



**3. 보안 DNS(DNS over HTTPS) 구성:**


설정 > 개인정보 및 보안 > DNS over HTTPS(페이지 하단)



![Configuration DNS sur HTTPS](assets/fr/19.webp)



**기본 보호:**




- Zen은 보안 DNS 사용 시기를 자동으로 결정합니다
- 보안 DNS를 사용할 수 있는 지역에서 사용
- 보안 공급업체에 문제가 있는 경우 기본 DNS로 폴백
- VPN, 자녀 보호 또는 회사 정책에 따라 자동으로 비활성화됩니다



**보호 강화(권장):**




- 보안 DNS 사용 시기를 제어하고 공급업체를 선택합니다
- 필요한 경우 선택한 공급업체와 함께 DNS 시스템으로 폴백 사용
- 기본 제공업체:** Cloudflare(빠른 익명 로그)
- 대안:** 이용 가능 여부에 따라 Quad9, NextDNS로 전환 가능



**최대 보호(고급 사용자):**




- Zen **항상** 보안 DNS만 사용
- DNS 시스템 사용 전 안전 경고
- 경고:** 보안 DNS를 사용할 수 없는 경우 사이트가 로드되지 않을 수 있습니다



**4. HTTPS 모드만 활성화:**


설정 > 개인정보 및 보안 > HTTPS 모드 전용 > **활성화**




- 모든 연결을 HTTPS로 강제 연결
- 사이트가 HTTP만 지원하는 경우 알림



**5. 기본 권한 관리:**


설정 > 개인정보 보호 및 보안 > 권한:




- 위치**: 차단(카드 서비스 제외)
- 카메라/마이크**: 차단(사례별로 승인)
- 알림**: 차단(스팸 방지)
- 자동 재생**: 오디오 및 비디오 차단



### 권장 안전 확장 기능



**필수 확장 기능:**




- 유블럭 오리진**: 가장 효과적인 광고 차단기 및 트래커
  - 추천 목록: 이지리스트, 이지프라이버시, 피터 로우의 광고 및 추적 서버 목록
  - 숙련된 사용자를 위한 고급 모드





- ClearURLs**: URL 추적 매개변수(utm_source, fbclid 등)를 삭제합니다
- 쿠키 자동 삭제**: 탭을 닫으면 쿠키와 브라우징 데이터를 자동으로 삭제합니다
- Decentraleyes**: 구글/클라우드플레어 CDN을 피하기 위해 JS 라이브러리를 로컬에서 서비스합니다



**고급 확장 기능(숙련된 사용자):**




- NoScript**: 세분화된 자바스크립트 제어(많은 사이트를 망가뜨릴 수 있음)
- 프라이버시 오소리** (EFF): 트래커의 행동 탐지
- 임시 컨테이너**: 각 탭을 별도의 컨테이너에 격리



## Zen 브라우저의 DRM 부재에 대한 이해



### DRM이란 무엇인가요?



**DRM(디지털 저작권 관리)**은 디지털 콘텐츠를 암호화하여 복제를 방지하는 보호 기술입니다. 보호된 미디어를 해독하고 읽기 위해서는 전용 브라우저 모듈(예: **구글 와이드바인**)이 필요합니다.



**DRM이 필요한 서비스:**




- 동영상 스트리밍:** 넷플릭스, 디즈니+, HBO 맥스, 아마존 프라임 비디오
- 프리미엄 음악:** Spotify Premium, YouTube Music, Deezer
- 온라인 교육: ** Udemy, Coursera(보호된 동영상)



### Zen 브라우저에 DRM이 없는 이유



**주요 이유:**


1. **비용 및 복잡성:** Google Widevine 라이선스는 무료가 아니며 번거로운 승인 절차가 필요합니다


2. **프로젝트 철학: ** DRM은 오픈 소스 접근 방식 및 Google로부터의 독립성과 호환되지 않는 독점적인 '블랙박스'입니다


3. **제한된 리소스:** 팀은 Interface 혁신과 기밀 유지에 중점을 둡니다



### 실질적인 영향



**❌ 액세스할 수 없는 서비스:**


넷플릭스, 디즈니+, 스포티파이 프리미엄, 유튜브 프리미엄, 유데미/코세라 유료 교육 과정



![Erreur DRM Prime Video](assets/fr/17.webp)


*DRM으로 보호된 콘텐츠 재생 시도 시 일반적인 오류 메시지*



**✅ 기능적 서비스:**


무료 유튜브, 트위치, 비메오, 뉴스 사이트, 소셜 네트워크, 팟캐스트



**우회 솔루션:**




- 스트리밍에만 Firefox/Crome 사용
- 기본 애플리케이션(넷플릭스, 스포티파이)
- DRM이 없는 콘텐츠 선택(YouTube, Twitch, Bandcamp, PeerTube)



**현재 상태: ** Zen 팀은 Widevine 라이선스 취득 절차를 시작했지만 일정은 정해지지 않았습니다. 이 절차는 전적으로 Google의 승인에 달려 있습니다.

## 결론



Zen 브라우저는 웹 브라우저 생태계에 신선한 바람을 불러일으키고 있습니다. 혁신적이고 생산적인 Interface와 개인 정보 보호에 대한 엄격한 존중을 결합하여 거대 기술 기업의 브라우저에 대한 신뢰할 수 있는 대안을 제시합니다.



세로 탭과 작업 공간은 여러 프로젝트를 동시에 처리하는 사용자의 브라우징 경험을 완전히 바꿔놓습니다. '노 구글' 철학과 커뮤니티 개발은 디지털 주권에 관심이 있는 사용자에게 일관된 선택이 될 것입니다.



아직 몇 가지 제한 사항(모바일 없음, DRM 누락)이 있지만 Zen 브라우저는 일상적으로 사용하기에 충분히 성숙했으며, 활발한 커뮤니티 덕분에 빠르게 개선되고 있습니다.



생산성과 프라이버시를 모두 중시하는 비트코인과 기술 사용자라면 Zen 브라우저를 사용해 볼 가치가 있습니다. 이 새롭고 더 평온하고 효율적인 브라우징 방식을 채택할 수도 있습니다.
