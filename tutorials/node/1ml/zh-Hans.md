## 导言



Lightning Network 是建立在 Bitcoin 基础上的快速、低成本支付解决方案，可实现即时、安全的交易。观察该网络对于了解其工作原理、拓扑结构和节点状态至关重要。Lightning explorer（如 1ML）可用于可视化网络的公共数据，包括活动节点、开放通道和可用容量，为用户、开发人员和节点运营商提供有价值的概览。



## 访问 1ML 并了解主页



要访问 1ML，只需打开浏览器并输入 [https://1ml.com](https://1ml.com)。这将带您进入主页，作为 Lightning Network 的全球仪表板。



![capture](assets/fr/03.webp)



该页面在屏幕顶部显示几个重要的统计数据，包括 ：





- 网络上**个活跃节点**的总数，即参与发送和接收闪电付款的计算机。
- 开放通道的**数量**，相当于这些节点之间可以进行资金转账的连接。
- 网络总容量**，以比特币（BTC）为单位，表示所有公共通道的容量总和。



这些数字定期更新，以反映网络的现状。通过这些数据，我们可以了解网络的规模、发展和稳健性。



![capture](assets/fr/04.webp)



页面下方提供了带有排名的列表：





- 连接最多的节点**，它们与其他节点之间的通道最为畅通。
- 节点上**高的容量**，表示哪些节点可以传输最大的数据量。
- 就容量而言最重要的通道。



还可使用筛选器按地理位置或其他标准对这些列表进行细化。



本页是探索 Lightning Network 和了解其一般拓扑结构的理想起点。

## 探索 "闪电 "节点



要探索 1ML 上的节点，可从页面顶部的搜索栏开始。您可以输入**节点 ID**，即节点的唯一标识符，也可以输入**别名**，这是一个更容易记住的名称。



搜索完成后，点击相应节点即可进入其详细页面。



![capture](assets/fr/07.webp)



该页面显示了几条重要信息：





- 节点 ID**：这一唯一标识符是一长串字符，可在整个网络中精确识别节点。



![capture](assets/fr/08.webp)





- 别名**：这是节点所有者选择的公开名称。



![capture](assets/fr/09.webp)





- **通道数**表示节点与其他节点打开的连接数量，而**总容量**表示这些通道中可用比特币的总和。拥有大量通道和高容量的节点通常连接良好，能够在网络上快速转移大量资金。



![capture](assets/fr/10.webp)





- **正常运行时间**，即可用性，衡量节点保持在线活动和可访问的时间，反映其可靠性。另一方面，节点的**年龄**表示该节点在网络上存在的时间，反映其在 Lightning Network 中的稳定性和经验。



![capture](assets/fr/11.webp)



这些数据有助于您了解节点在 Lightning Network 中的重要性和可靠性。例如，拥有大量信道、高容量和高正常运行时间的节点往往是网络中的重要角色。



## 探索雷电通道



### 什么是闪电通道？



闪电通道是两个网络节点之间的直接连接。它使这两个节点能够交换即时、低成本的付款，而无需每次交易都经过 Bitcoin 主链。通道是 Lightning Network 快速和可扩展的基础。



### 阅读有关 1ML 的频道信息



在 1ML 上，每个通道都有自己的页面或说明表，其中包含许多重要数据：



从节点页面，您可以访问其频道列表。点击频道后，1ML 会显示一个专门页面，其中包含几条关键信息。



![capture](assets/fr/12.webp)



![capture](assets/fr/13.webp)



主要可见数据如下





- 伙伴节点**：每个通道连接两个节点。1ML 显示这两个标识符及其各自的别名。



![capture](assets/fr/14.webp)





- 通道容量**：这是该通道锁定的比特币总量。该容量代表可通过该通道支付的最大限额。



![capture](assets/fr/15.webp)





- 通道年龄**：表示通道开放的时间。老通道通常是两个节点之间关系稳定的标志。



![capture](assets/fr/16.webp)



### 航道能见度限制



要知道，1ML 只显示 Lightning Network 的**部分**。资源管理器只显示**公开频道**，即已在网络上公布的频道。出于保密或战略原因而经常使用的私人频道是不可见的。此外，1ML 不显示渠道每一方资金的确切分布情况，也不显示支付情况或特定时刻实际可用的流动性。因此，显示的信息可用于分析网络的**总体结构，但不能用于分析其实际财务活动或详细的流动性状况。



## 按地点探索 Lightning Network



### 按国家和地区划分的节点



1ML 允许您根据**地理细分**探索 Lightning Network。通过主页或专门栏目，可以按国家或地区显示节点。该功能基于节点运营商提供的位置信息。


在导航栏中，您将看到**位置**链接。在页面上，节点按洲、国家和城市分组。



![capture](assets/fr/17.webp)



选择一个国家，1ML 就会显示相关节点的列表，以及该地理区域的节点数量和可见总容量等综合统计数据。



#### 这说明了什么？





- 技术采用**：一个地区有大量节点，表明 Bitcoin 用户、公司或服务正在积极采用 Lightning Network。这表明技术成熟，愿意利用闪电的优势（交易速度快、成本低）。
- 经济生态系统** ：节点的密集存在也预示着 Bitcoin 周围的本地经济结构：接受 "闪电 "服务的商家、开发工具的初创企业、社区活动等。
- 安全性和恢复能力**：多样化的地理分布增强了网络在面临本地中断或限制时的恢复能力。节点越分散，网络就越分散，越难以审查。
- 政策和法规**：一些国家可能会因为有利的监管框架或积极主动的社区而加快采用速度。相反，在监管严格或充满敌意的地区，节点数量会较少。



#### 地理数据的局限性



不过，请记住，"闪电 "节点的地理位置有其局限性和偏差：





- 近似 IP 位置**：1ML 通常使用节点的公共 IP 地址来估计其位置。但是，如果使用 VPN、云服务器（AWS、Google Cloud）或托管在与节点所有者实际居住地不同的国家，这种方法就会失真。
- 虚拟节点**：一些运营商出于可靠性和可用性的考虑，将其节点托管在远程服务器上，这可能给人一种物理位置的错觉。
- 用户流动性**：节点运营商可能会改变位置、移动基础设施或在不同地区开设多个节点，从而使数据读取变得更加复杂。
- 隐蔽节点**：有些节点不公布自己的 IP 地址，或使用各种方法隐藏自己的位置，导致无法进行地理定位。



## 1ML 具体使用案例



### 了解网络拓扑结构



1ML 可让您直观地了解 Lightning Network** 的**总体结构。通过观察节点之间的连接、通道数量和总体容量，可以了解网络是如何组织的，哪些节点发挥着核心作用，以及理论上支付流是如何流通的。



如果我们要了解 Lightning Network 的工作原理，就必须了解这一点，而不仅仅是为了组合使用。



### 确定重要节点



借助 1ML 提供的排名（连接节点最多、容量最大、年龄最大），可以识别出在网络中占据重要位置的节点。这些节点通常是闪电支付的主要网关。



![capture](assets/fr/18.webp)



### 检查节点的公共可见性



对于节点操作员来说，1ML 允许您检查您的节点是否在 Lightning Network 上**公开宣传**。如果一个节点出现在 1ML 上，这就意味着它是可见的，其他节点可以访问它以打开公共频道。


这对诊断可见性或连接性问题非常有用。



### 观察 Lightning Network 随着时间的推移而演变



通过比较不同时期的全球统计数据，1ML 使我们能够观察到 Lightning Network 的演变：节点数量的增减、总容量的变化或地理分布的变化。


这些观察结果为 Lightning Network 的成长、成熟和趋势提供了一个有趣的视角。



## 1ML 最佳实践和限制



### 公共数据≠完整现实



1ML 完全基于 Lightning Network 的**公开**数据。这意味着该工具只显示了现实的一部分。许多渠道可能是私有的，一些节点可能没有公布，渠道中的实际流动性也不可见。因此，必须将 1ML 作为全局分析工具，而不是网络的详尽代表。



### 隐私和闪电



Lightning Network 的设计非常注重**支付隐私**。单笔交易在 1ML 上不可见，确切的频道余额也不公开。这一限制不是资源管理器的过错，而是 Lightning Network 的基本功能，旨在保护用户隐私。



### 不要妄下结论



容量大或通道多的节点并不一定在所有情况下都更 "可靠 "或更 "高效"。同样，整体网络容量的暂时下降也不一定意味着结构性问题。在解释数字时，总是应该事后分析，并结合实际情况。



### 与其他工具的互补性



1ML 是探索 Lightning Network 的绝佳起点，但最好与 Lightning 组合、节点管理界面和其他探索器等其他工具结合使用。这种方法可以提供更全面、更细致的网络视图。



## 1ML 连接选项（高级功能）



1ML 提供**登录/创建账户**选项，可在网站上看到，但这并非查看 Lightning Network 数据的**必要**。本教程目前讨论的所有功能**无需账户**即可使用。



该连接主要针对**Lightning 节点操作员**。特别是，它能使节点与 1ML 账户关联，以便管理某些公共信息，如节点的展示、联系链接和其他元数据。该功能旨在提高节点在资源管理器中的可见度和识别度。



需要注意的是，1ML 不是**托管服务。创建账户并不意味着可以使用节点的资金、渠道或付款。它只是从声明和信息的角度与探索者互动。



因此，在学习或探索 Lightning Network 的过程中，该选项可被视为**可选**，仅供高级用户使用。



## 结论



1ML 是通过公开数据观察和了解 Lightning Network 的重要工具。通过它，您可以探索网络结构、分析节点和渠道，并跟踪 Lightning Network 采用情况随时间的整体演变。1ML 无需开设账户或处理资金，为任何希望深入了解闪电如何运作的人提供了一个便捷的途径。
