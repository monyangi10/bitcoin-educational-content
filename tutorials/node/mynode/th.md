[My Node](https://mynodebtc.com/) เป็นวิธีที่ง่ายที่สุดและทรงพลังที่สุดในการรัน Bitcoin และ Lightning node! เราผสมผสานซอฟต์แวร์โอเพ่นซอร์สที่ดีที่สุดเข้ากับอินเทอร์เฟซ การจัดการ และการสนับสนุนของเรา เพื่อให้คุณสามารถใช้ Bitcoin และ Lightning ได้อย่างง่ายดาย เป็นส่วนตัว และปลอดภัย


## ประเภทของการตั้งค่า Node


มีการตั้งค่า Node หลายแบบ MyNode ยอดเยี่ยมมาก มีแอปมากมายที่มาพร้อมกับมัน และยิ่งกว่านั้นถ้าคุณจ่ายสำหรับเวอร์ชันพรีเมียม มิฉะนั้นจะเป็นเรื่องน่าเบื่อมากในการดาวน์โหลดแอปทั้งหมดด้วยตัวเอง MyNode ทำให้ง่ายมากอย่างที่คุณจะเห็น


ตัวเลือกทางเลือกที่คล้ายกันคือ RaspiBlitz GUI ไม่สวยเท่า และแอปมีการซ้อนทับกันมากกับแอปที่มาพร้อมกับ MyNode แต่ Raspiblitz เป็นซอฟต์แวร์โอเพ่นซอร์สฟรี (FOSS) และ MyNode ไม่ค่อยเป็น อีกความแตกต่างคือ MyNode ทำงานใน Docker container ฉันพบว่า Docker น่ากลัวและยากต่อการแก้ไขปัญหา นี่เป็นปัญหาเฉพาะเมื่อคุณพบข้อผิดพลาดหรือบั๊ก นักพัฒนามีการเสนอความช่วยเหลือสำหรับผู้ใช้พรีเมียมและยังมีกลุ่มแชท Telegram ด้วย


RaspiBlitz เป็นเพียงโปรแกรมหลายตัวที่ติดตั้งบน Linux โดยไม่มี Docker ฮาร์ดไดรฟ์ภายนอกสามารถเชื่อมต่อกับเครื่อง Linux อื่นที่มี Bitcoin Core ได้อย่างง่ายดาย และคุณก็พร้อมใช้งานทันทีหากจำเป็น


อีกทางเลือกหนึ่งคือการติดตั้งเฉพาะ Bitcoin Core และ Electrum Server หลากหลายประเภท (มีหลายแบบ) บนระบบปฏิบัติการ ฉันมีคู่มือสำหรับ Linux (Raspberry Pi), Mac และ Windows


## รายการซื้อของ



- Raspberry Pi 4, หน่วยความจำ 4Gb หรือ 8Gb (4Gb ก็เพียงพอแล้ว)
- อุปกรณ์จ่ายไฟอย่างเป็นทางการของ Raspberry Pi (สำคัญมาก! อย่าใช้ของทั่วไป, จริงจังนะ)
- เคสสำหรับ Pi. เคส FLIRC นั้นยอดเยี่ยมมาก ทั้งเคสทำหน้าที่เป็นฮีทซิงค์และคุณไม่จำเป็นต้องใช้พัดลมซึ่งอาจมีเสียงดัง
- การ์ด microSD ขนาด 16 Gb (คุณต้องการหนึ่งอัน แต่มีหลายอันก็สะดวก)
- เครื่องอ่านการ์ดหน่วยความจำ (คอมพิวเตอร์ส่วนใหญ่จะไม่มีช่องสำหรับการ์ด microSD)
- External SSD 1Tb hard drive.

หมายเหตุ: SSD มีความสำคัญมาก อย่าใช้ฮาร์ดไดรฟ์ภายนอกแบบพกพาแม้ว่าจะราคาถูกกว่าก็ตาม


- สายเคเบิลอีเธอร์เน็ต (สำหรับเชื่อมต่อกับเราเตอร์ที่บ้านของคุณ)
- คุณไม่จำเป็นต้องมีจอภาพ

## ค้นหาที่อยู่ IP ของ Pi


คุณไม่จำเป็นต้องมีจอมอนิเตอร์กับ MyNode อย่างไรก็ตาม คุณจำเป็นต้องมีคอมพิวเตอร์อีกเครื่องในเครือข่ายภายในบ้าน หาก Pi ของคุณไม่ได้เชื่อมต่อผ่านอีเธอร์เน็ต และคุณต้องการพึ่งพา WiFi การค้นหา IP จะต้องใช้ทักษะคอมพิวเตอร์ขั้นสูง ช่วยไม่ได้ ขอโทษที คุณจำเป็นต้องมีการเชื่อมต่ออีเธอร์เน็ต (ปัญหามาจากการที่ต้องการเข้าถึงจอมอนิเตอร์และระบบปฏิบัติการเพื่อเชื่อมต่อกับ WiFi และป้อนรหัสผ่าน)


ตรวจสอบเราเตอร์ของคุณ เพื่อดูรายการ IP ทั้งหมดของอุปกรณ์ที่เชื่อมต่อทั้งหมด


ฉันพิมพ์ 192.168.0.1 ในเบราว์เซอร์ (ตามคำแนะนำที่มาพร้อมกับเราเตอร์ของฉัน) ลงชื่อเข้าใช้ และสามารถเห็นอุปกรณ์ “MyNode” ที่มี IP 192.168.0.18 โปรดทราบว่า IP เหล่านี้ไม่สามารถมองเห็นได้จากอินเทอร์เน็ตสาธารณะ (พวกมันจะผ่านเราเตอร์ก่อน) พวกมันเป็นเพียงตัวระบุสำหรับอุปกรณ์ในเครือข่ายภายในบ้านของคุณเท่านั้น


การค้นหา IP เป็นสิ่งสำคัญ


**หมายเหตุ:** คุณสามารถใช้เทอร์มินัลบนเครื่อง Mac หรือ Linux เพื่อค้นหาที่อยู่ IP ของอุปกรณ์ที่เชื่อมต่อผ่าน Ethernet ทั้งหมดในเครือข่ายภายในบ้านโดยใช้คำสั่ง “arp -a” ผลลัพธ์อาจไม่สวยงามเท่ากับที่เราเตอร์จะแสดง แต่ข้อมูลทั้งหมดที่คุณต้องการอยู่ที่นั่น หากไม่ชัดเจนว่าอันไหนคือ Pi ให้ลองผิดลองถูกดู


## SSH เข้าไปใน Pi


คุณมีตัวเลือกในการเข้าสู่ระบบอุปกรณ์จากระยะไกลโดยใช้คำสั่ง SSH แต่ไม่จำเป็น (จำเป็นถ้าเป็น RaspiBlitz Node) อย่างไรก็ตาม ฉันจะแสดงวิธีให้คุณดู เพราะมันสะดวกมาก


เปิดคอมพิวเตอร์ Mac หรือ Linux (สำหรับ Windows ให้ดาวน์โหลด putty ซึ่งเป็นเครื่องมือ SSH) และพิมพ์:


```bash
ssh admin@192.168.0.18
```


ใช้ที่อยู่ IP ของคุณเอง ชื่อผู้ใช้สำหรับอุปกรณ์ MyNode คือ “admin” โดยค่าเริ่มต้น รหัสผ่านคือ “bolt” โดยค่าเริ่มต้น


หากคุณเคยใช้ Pi ของคุณมาก่อนและสลับการ์ด micro SD คุณจะได้รับข้อผิดพลาดนี้:


![image](assets/17.webp)


สิ่งที่คุณต้องทำคือไปที่ตำแหน่งที่ไฟล์ “known_hosts” อยู่และลบมันออก มันปลอดภัย ข้อความแสดงข้อผิดพลาดจะแสดงเส้นทาง สำหรับฉันมันคือ /Users/MyUserName/.ssh/


อย่าลืม “.” ก่อน ssh ซึ่งบ่งบอกว่าเป็นไดเรกทอรีที่ซ่อนอยู่


จากนั้นลองใช้คำสั่ง ssh อีกครั้ง


คราวนี้คุณจะเห็นผลลัพธ์นี้:

หากคุณต้องการลองใช้โหนดอื่นด้วย เช่น RaspiBlitz คุณจะต้องมี SSD external hard drive เพิ่มเติม และ micro SD card อีกใบเพื่อทำการแฟลช มิฉะนั้น อุปกรณ์จะเหมือนเดิม เพียงแค่คุณไม่สามารถรัน MyNode และ RaspiBlitz พร้อมกันได้อย่างชัดเจน หากคุณต้องการทำเช่นนั้น ก็ถึงเวลาซื้อ Raspberry Pi อีกตัวแล้ว


ตอนนี้ที่คุณมีโหนดทำงานแล้ว ใช้มัน อย่าปล่อยให้มันนั่งอยู่เฉยๆ โดยไม่ทำอะไรให้คุณ ติดตามบทความ (และวิดีโอ) ของฉันเกี่ยวกับวิธีการเชื่อมต่อ Electrum Desktop Wallet กับ Electrum Server และ Bitcoin Core ที่นี่
