[마이 노드](https://mynodebtc.com/)는 Bitcoin와 라이트닝 노드를 가장 쉽고 강력하게 실행할 수 있는 방법입니다! 최고의 오픈 소스 소프트웨어와 Interface, 관리 및 지원을 결합하여 Bitcoin와 Lightning을 쉽고, 비공개적으로, 안전하게 사용할 수 있습니다.


## 노드 설정 유형


다양한 노드 설정이 있습니다. MyNode는 훌륭합니다. 함께 제공되는 앱이 많으며 프리미엄 버전을 구매하면 더 많은 앱을 이용할 수 있습니다. 그렇지 않으면 모든 앱을 직접 다운로드하는 것은 매우 지루합니다. MyNode를 사용하면 보시다시피 매우 쉽습니다.


비슷한 대안으로 RaspiBlitz가 있습니다. GUI는 그다지 좋지 않고 앱이 MyNode와 함께 제공되는 앱과 많이 겹치지만, Raspiblitz는 무료 오픈 소스 소프트웨어(FOSS)이고 MyNode는 그렇지 않습니다. 또 다른 차이점은 MyNode가 Docker 컨테이너에서 실행된다는 점입니다. 저는 Docker가 어렵고 Hard가 문제 해결에 어려움을 느낍니다. 이는 오류나 버그가 발생하는 경우에만 문제가 됩니다. 개발자는 프리미엄 사용자를 위한 도움말을 제공하며 텔레그램 채팅 그룹도 있습니다.


RaspiBlitz는 Docker 없이 Linux에 여러 프로그램을 설치한 것일 뿐입니다. 필요에 따라 외장형 Hard 드라이브를 Bitcoin core가 있는 다른 Linux 머신에 쉽게 연결하여 사용할 수 있습니다.


또 다른 옵션은 운영 체제에 Bitcoin core와 다양한 일렉트럼 서버(여러 가지가 있음)를 설치하는 것입니다. Linux(라즈베리 파이), Mac 및 Windows용 가이드가 있습니다.


## 쇼핑 목록



- 라즈베리 파이 4, 4Gb 메모리 또는 8Gb(4Gb면 충분)
- 공식 라즈베리 파이 파워(매우 중요! 일반적이지 않게, 진지하게)
- 파이를 위한 케이스. FLIRC 케이스는 굉장합니다. 케이스 전체가 방열판으로 되어 있어 시끄러울 수 있는 팬이 필요 없습니다
- 16Gb microSD 카드(하나만 있으면 되지만 몇 개만 있으면 편리함)
- 메모리 카드 리더기(대부분의 컴퓨터에는 microSD 카드용 슬롯이 없습니다).
- 외장 SSD 1Tb Hard 드라이브.

참고: SSD가 중요합니다. 저렴하더라도 휴대용 외장 Hard 드라이브는 사용하지 마세요


- 이더넷 케이블(가정용 라우터 연결용)
- 모니터가 필요하지 않습니다

## Pi의 IP Address 찾기


마이노드에는 모니터가 필요하지 않습니다. 하지만 홈 네트워크에 다른 컴퓨터가 필요합니다. Pi가 이더넷으로 연결되어 있지 않고 WiFi를 사용하려는 경우 IP를 찾으려면 고도의 컴퓨터 기술이 필요합니다. 도와드릴 수 없습니다. 이더넷 연결이 필요합니다. (WiFi에 연결하고 비밀번호를 입력하려면 모니터와 운영 체제에 액세스해야 하기 때문에 문제가 발생합니다).


라우터에서 연결된 모든 디바이스의 모든 IP 목록을 확인하세요.


브라우저에 192.168.0.1(라우터와 함께 제공된 지침)을 입력하고 로그인했더니 IP가 192.168.0.18인 "MyNode" 디바이스를 볼 수 있었습니다. 이 IP 주소는 인터넷에 공개적으로 표시되지 않으며(라우터를 먼저 통과해야 함), 홈 네트워크에 있는 디바이스의 식별자라는 점에 유의하세요.


IP를 찾는 것이 중요합니다.


**참고: Mac 또는 Linux 컴퓨터의 터미널에서 "arp -a" 명령을 사용하여 홈 네트워크에 연결된 모든 이더넷 연결 장치의 IP Address을 찾을 수 있습니다. 출력은 라우터에 표시되는 것만큼 예쁘지는 않지만 필요한 모든 정보가 있습니다. 어느 것이 파이인지 명확하지 않은 경우 시행착오를 수행하세요.


## Pi로 SSH 접속


SSH 명령으로 장치에 원격으로 로그인할 수 있는 옵션이 있지만 필수는 아닙니다(RaspiBlitz 노드인 경우). 어쨌든 매우 편리하기 때문에 방법을 보여드리겠습니다.


Mac 또는 Linux 컴퓨터를 열고(Windows의 경우 SSH 도구인 퍼티를 다운로드합니다) 입력합니다:


```bash
ssh admin@192.168.0.18
```


고유 IP Address를 사용합니다. MyNode 디바이스의 사용자 이름은 기본적으로 "admin"입니다. 비밀번호는 기본적으로 "Bolt"입니다.


이전에 Pi를 사용하다가 마이크로 SD 카드를 바꾼 적이 있다면 이 오류가 발생할 수 있습니다:


![image](assets/17.webp)


"known_hosts" 파일이 있는 위치로 이동하여 삭제하기만 하면 됩니다. 안전합니다. 오류 메시지에 경로가 표시됩니다. 제 경우에는 /사용자/내사용자명/.ssh/였습니다


Ssh 앞에 '."를 붙이는 것을 잊지 마세요. 이는 숨겨진 디렉터리임을 나타냅니다.


그런 다음 ssh 명령을 다시 시도합니다.


이번에는 다음과 같은 출력이 표시됩니다:


![image](assets/18.webp)


삭제한 파일이 삭제되었으며 새 지문을 추가합니다. 예와 <입력>을 입력합니다.


비밀번호를 입력하라는 메시지가 표시됩니다. "Bolt"입니다

필요한 경우 다운로드 중에 기기를 끄는 것이 안전합니다. 설정으로 이동하여 기기 전원을 끄는 버튼을 찾으세요. 코드를 잡아당기면 설치 또는 Hard 드라이브가 손상될 수 있습니다.


초기에 "설정"으로 이동하여 빠른 동기화를 비활성화하세요. 초기 블록 다운로드가 처음부터 다시 시작됩니다.


![image](assets/25.webp)


가이드 작성을 계속 진행하고 싶었기 때문에 앞서 준비한 또 다른 마이노드를 소개합니다. Blockchain가 동기화되고 여러 '앱'이 활성화 및 동기화된 상태의 페이지 모습입니다:


![image](assets/26.webp)


Electrum 서버도 동기화해야 하므로 Bitcoin Blockchain이 동기화되는 즉시 버튼을 클릭하여 해당 프로세스를 시작하면 하루나 이틀 정도 걸립니다. 다른 모든 기능은 활성화하도록 선택하면 몇 분 안에 활성화됩니다. 물건을 클릭하고 탐색할 수 있습니다. 아무것도 깨지지 않습니다. 만약 무언가가 고장난다면(저도 이런 일이 있었지만, 저렴한 부품을 사용했기 때문인 것 같습니다) 다시 플래시하고 다시 다운로드를 시작해야 합니다. MyNode의 문제점은 "다시 플래시"해야 하는 경우 Blockchain 동기화를 처음부터 다시 시작해야 한다는 것입니다. 이 문제를 해결할 수 있는 기술적인 방법이 있지만 쉽지 않습니다.


다른 노드(예: RaspiBlitz)도 사용하려면 추가 SSD 외장형 Hard 드라이브와 플래시용 마이크로 SD 카드가 필요합니다. 그렇지 않으면 동일한 장비이지만 MyNode와 RaspiBlitz를 동시에 실행할 수 없습니다. 그렇게 하고 싶다면 다른 라즈베리 파이를 구입해야 합니다.


이제 노드가 실행 중이니 아무 일도 하지 않고 그냥 두지 말고 사용하세요. 여기에서 일렉트럼 데스크톱 Wallet를 일렉트럼 서버 및 Bitcoin core에 연결하는 방법에 대한 제 글(및 동영상)을 참조하세요.
