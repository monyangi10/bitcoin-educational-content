Alby Hub ni porogaramu nshasha y’inkomoko yuguruye ivuye kuri Alby, ishirahamwe riri inyuma y’urubuga rwa Lightning ruzwi cane. Alby Hub ni Wallet yizigama ifise node yoroshe gukoresha ya Lightning, ishobora gushikwako aho hose kugira ngo uyifatanye n’amaporogarama menshi. Alby Hub ni Interface yoroshe gukoresha mu gucunga amanode y’umuravyo.


Muri iyi nyigisho, turaza kuraba uburyo butandukanye bwo gukoresha Alby Hub ingene woyifatanya na Alby Go, app ya Alby yo kuri telefone ngendanwa canke Extension ya Alby Browser. Ivyo bizokuronsa ubushobozi bwo gukoresha Sats yawe mu rugendo mu gihe wigenga mu burongozi bw’urudodo rwawe.


![ALBY HUB](assets/fr/01.webp)


## Alby Hub ni iki?


Alby Hub iteguwe kuba igikoresho gishasha c’ibendera mu bidukikije vya Alby. Iyi porogarama ituma abayikoresha bashobora gucungera mu buryo bworoshe Wallet yabo bwite yo kwizigama n’uruzitiro rwa Lightning rwinjijwemwo, mu gihe bagumana Ownership y’imfunguruzo zabo (ukwizigama).


Alby Hub ni igikoresho gishobora guhindura cane. Ishobora gushitsa ivyo abatangura n’abateye imbere bakeneye. Abashasha bazoyikoresha kugira ngo bakoreshe mu buryo bworoshe urudodo rw’umuravyo nyarwo ku giti cabo, batabwirizwa guhangana n’ibintu bigoranye biri inyuma. Ku bakoresha bazi utuntu n’utundi, Alby Hub ishobora gukoreshwa nka Interface yuzuye ku bijanye n’uburongozi buteye imbere bw’urudodo rwa Lightning ruriho.


Bivanye n'ivyo ukeneye, Alby Hub iboneka mu mice 4:




- Igicu ca Alby :**


Ivyiza ku bashasha, iyi nzira ya mbere ni uburyo bw’igicu ca Alby. Ishobora kugufasha gukoresha Hub ku serveri irongowe na Alby, ushobora kuyironka biciye ku Alby Hub Interface yawe. Naho Alby ari we acungera server, wewe uraguma ufise ubusegaba ku mahera yawe, kuko imfunguruzo zawe zishirwa mu nzira hakoreshejwe ijambobanga uzi wewe wenyene. Ariko rero, imfunguruzo zawe zitegerezwa kuguma zipfutse muri RAM kugira ngo iyo node ikore, ivyo bikaba bituma zishobora gushikirwa n’ingorane iyo umuntu yinjiye kuri server. Ni ugusenyera ku mugozi umwe gushimishije ku batangura, ariko birahambaye kumenya ingorane zishobora guterwa n’ivyo.


Inyungu nini y’iyi nzira ni uko uronka node ya Lightning ikora 24/7, utabwirizwa gucungera hosting wewe nyene. Ikindi, ama backups y’uruzitiro rwawe rwa Lightning yoroshe kandi yikora, ugereranyije n’amahitamwo yikorera aho utegerezwa gucungera ama backups y’imirongo wewe nyene.


Alby Cloud ni igikorwa cishurwa [Suzuma igiciro cabo](https://albyhub.com/#igiciro) kugira ngo umenye vyinshi. Amahera aca akurwa kuri Wallet yawe biciye ku Lightning Invoice isohowe na Alby. Ivyo bikorwa biciye ku nzira ya NWC itunganya node yawe kugira ngo yishure amafagitire ya Alby ajanye n’ukwiyandikisha kwawe.





- Alby Hub afise urudodo ruriho :**


Niba usanzwe ufise node yakira, nk’akarorero kuri Umbrel canke Start9, Alby Hub ishobora gukoreshwa nk’uburongozi buteye imbere Interface, mu buryo bumwe na ThunderHub canke RTL.




- Alby Hub yo mu karere :**


Birashoboka kandi gushiramwo Alby Hub kuri PC yawe, naho iyo nzira idakora cane, kuko PC yawe itegerezwa kuguma ikora ibihe vyose kugira ngo ushobore gushika kure kuri node ya Lightning. Ariko rero, iyo nzira yoshobora kuba ibereye ivyo ukeneye vyihariye.




- Alby Hub kuri server y'umuntu ku giti ciwe :**


Ku bakoresha bateye imbere, Alby Hub ishobora gukoreshwa kuri server y’umuntu ku giti ciwe n’itegeko ryoroshe. Iyi nzira ntivugwa muri iyi nyigisho, ariko ushobora kubona amabwirizwa yihariye [ku GitHub ya Alby](https://github.com/getAlby/hub?tab=soma-ov-dosiye#docker).


Iyi nyigisho yibanda canecane kuri Interface, izoba ari imwe ataco ushizeko. Turaba kandi ingene twokoresha Alby Hub n’uburyo bwo gukoresha igicu c’amahera, hanyuma dukoresheje uburyo bwo gukoresha node-in-box (Umbrel canke Start9).

## Uburyo bwo kwiyakira


Niba ushaka gukoresha Alby Hub nka Interface ku node yawe ya Lightning iriho, urafise uburyo bwinshi: kuyishira kuri server, mu karere kuri mudasobwa yawe, canke biciye ku node-in-box (Umbrel canke Start9). Gukoresha Alby Hub muri izo ntunganyo ni ubuntu. Tugiye kwibanda ku mahitamwo ya node-in-box, kuko mbona ko amahitamwo ya server, ata n’umwe ashobora gushika ku mubiri, atanga ingorane zisa n’iza verisiyo y’igicu, kandi gushiramwo mu karere kuri PC akenshi ntibibereye.


Kugira ngo ushireho ivyo kuri Umbrel (intambwe za Start9 zirasa), utegerezwa kubanza kugira urudodo rwa LND rwamaze gutunganirizwa.

## Gutanguza Alby Hub


Fyonda kuri buto ya "*Tangira*".

Mu nsiguro y’igicu yishurwa, imfunguruzo zawe zibikwa kuri server ya Alby, zigashirwako iri jambobanga wewe wenyene uzi, hanyuma zigafungurwa zigabikwa muri RAM gusa kugira ngo zishireko umukono ku bikorwa iyo bikenewe.

## Noshobora gute gusubira kuronka ama bitcoins yanje?


Imbere yo kohereza amahera kuri Alby Hub yawe, birahambaye ko utahura ingene woyagarura iyo habaye ingorane, hamwe n’amakuru asabwa kugira ngo uyagarure. Ivyo bigenda birahindagurika bivanye n’ingene amahera azosubizwa n’uburyo bwo kwakira node yawe.


Ku bakoresha igicu barihwa, gusubirana neza amafaranga yawe bisaba Elements zitatu z’ingenzi:



- Invugo yawe yo gukira;
- Ushobora gushika kuri konti yawe ya Alby, kugira ngo ubone ama backups yikora.


Kubura n’imwe muri ayo makuru 2 vyotuma udashobora kugarura ama bitcoins yawe yose.


Ku bakoresha Alby Hub ku gikoresho cabo bwite, uburyo bwo gusubirana buranditswe [hano](https://guides.getalby.com/uburongozi-bw’abakoresha/alby-konti-n’umucukumbuzi-alby-hub/ububiko-n’ugusubirana#alby-hub-yiharuwe-na-).


Niwaba washizeho Alby Hub ku nzira iriho, uzokenera gukurikira uburyo bwo gusubizaho iyo nzira. Nk'akarorero: Umutaka utanga [uburyo] (https://github.com/getumbrel/umutaka/blob/2b266036f62a1594aa60a8a3be30cfb8656e755f/inyandiko/ububiko/README.md) kugira ngo uyishire mu buryo butazwi biciye kuri Tor. Menya ko gusa ama backups yikora ava kuri Alby atuma ushobora gusubirana Hub yawe yose ata n’imwe upfutse.

## Gufungura umurongo w'umuravyo


Ubu ko ufise amahera ya onchain, urashobora gufungura umurongo mushasha wa Lightning. Ni vyiza gufungura imihora myinshi, ufise amahera ahagije kugira ngo ushobore kwama uriha ataco uhagarika. Benshi mu ba LSP (*Abatanga ibikorwa vy’umuravyo*) basaba n’imiburiburi 150.000 Sats kugira ngo bafungure umurongo nawe.

![ALBY HUB](assets/fr/38.webp)
