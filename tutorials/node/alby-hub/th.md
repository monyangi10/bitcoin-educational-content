Alby Hub เป็นซอฟต์แวร์โอเพ่นซอร์สล่าสุดจาก Alby บริษัทที่อยู่เบื้องหลังส่วนขยายเว็บ Lightning ที่ได้รับความนิยม Alby Hub เป็น wallet ที่ผู้ใช้ดูแลเองได้พร้อมโหนด Lightning ที่ใช้งานง่ายที่สุด สามารถเข้าถึงได้จากทุกที่เพื่อรวมเข้ากับแอปพลิเคชันหลายสิบตัว Alby Hub เป็นอินเทอร์เฟซที่ใช้งานง่ายสำหรับการจัดการโหนด Lightning


ในบทแนะนำนี้ เราจะมาดูวิธีการต่างๆ ในการใช้ Alby Hub วิธีเชื่อมต่อกับ Alby Go, แอปมือถือของ Alby หรือส่วนขยายเบราว์เซอร์ของ Alby ซึ่งจะช่วยให้คุณใช้จ่าย sats ของคุณได้ทุกที่ทุกเวลา ในขณะที่ยังคงความเป็นอิสระในการจัดการโหนดของคุณ

## Alby Hub คืออะไร?


Alby Hub ถูกกำหนดให้เป็นเครื่องมือเรือธงใหม่ในระบบนิเวศ Alby ซอฟต์แวร์นี้ช่วยให้ผู้ใช้สามารถจัดการ wallet ที่ดูแลตนเองได้อย่างง่ายดายด้วยโหนด Lightning ที่รวมอยู่ในตัว ในขณะที่ยังคงความเป็นเจ้าของกุญแจของตนเอง (การดูแลตนเอง)


Alby Hub เป็นเครื่องมือที่ปรับตัวได้สูง สามารถตอบสนองความต้องการของทั้งผู้เริ่มต้นและผู้ใช้ขั้นสูง ผู้เริ่มต้นจะใช้มันเพื่อดำเนินการโหนด Lightning จริงได้อย่างง่ายดายด้วยตนเอง โดยไม่ต้องจัดการกับความซับซ้อนที่อยู่เบื้องหลัง สำหรับผู้ใช้ที่มีประสบการณ์มากขึ้น Alby Hub สามารถใช้เป็นอินเทอร์เฟซที่สมบูรณ์สำหรับการจัดการขั้นสูงของโหนด Lightning ที่มีอยู่แล้ว


ขึ้นอยู่กับความต้องการของคุณ Alby Hub มีให้เลือก 4 รูปแบบ:




- Alby Hub Cloud :**


เหมาะสำหรับผู้เริ่มต้น ตัวเลือกแรกนี้คือตัวเลือกคลาวด์ Alby ซึ่งช่วยให้คุณสามารถปรับใช้ Hub บนเซิร์ฟเวอร์ที่จัดการโดย Alby ได้โดยตรง ซึ่งสามารถเข้าถึงได้ผ่านอินเทอร์เฟซ Alby Hub ของคุณ แม้ว่า Alby จะจัดการเซิร์ฟเวอร์ แต่คุณยังคงมีอำนาจอธิปไตยเหนือเงินทุนของคุณ เนื่องจากคีย์ของคุณถูกเข้ารหัสโดยใช้รหัสผ่านที่รู้เฉพาะคุณเท่านั้น อย่างไรก็ตาม คีย์ของคุณจะต้องถูกถอดรหัสใน RAM เพื่อให้โหนดทำงาน ซึ่งในทางทฤษฎีจะทำให้เสี่ยงหากมีผู้เข้าถึงเซิร์ฟเวอร์ทางกายภาพ เป็นการประนีประนอมที่น่าสนใจสำหรับผู้เริ่มต้น แต่สิ่งสำคัญคือต้องตระหนักถึงความเสี่ยง


ข้อได้เปรียบหลักของตัวเลือกนี้คือคุณสามารถเปิดใช้งาน Lightning node ได้ตลอด 24/7 โดยไม่ต้องจัดการโฮสติ้งด้วยตัวเอง นอกจากนี้ การสำรองข้อมูลของ Lightning node ของคุณยังง่ายขึ้นและเป็นอัตโนมัติ เมื่อเทียบกับตัวเลือกที่โฮสต์เองซึ่งคุณต้องจัดการการสำรองข้อมูลช่องทางด้วยตัวเอง


Alby Cloud เป็นบริการที่มีค่าใช้จ่าย [ตรวจสอบราคาได้ที่นี่](https://albyhub.com/#pricing) สำหรับรายละเอียดเพิ่มเติม ค่าธรรมเนียมจะถูกหักโดยอัตโนมัติจาก wallet ของคุณผ่านใบแจ้งหนี้ Lightning ที่ออกโดย Alby ซึ่งดำเนินการผ่านการเชื่อมต่อ NWC ที่ตั้งค่าโหนดของคุณให้ชำระใบแจ้งหนี้ Alby ที่เกี่ยวข้องกับการสมัครสมาชิกของคุณโดยอัตโนมัติ





- Alby Hub กับโหนดที่มีอยู่ :**


หากคุณมีโหนดที่โฮสต์อยู่แล้ว เช่น บน Umbrel หรือ Start9, Alby Hub สามารถใช้เป็นอินเทอร์เฟซการจัดการขั้นสูงได้ ในลักษณะเดียวกับ ThunderHub หรือ RTL.




- Alby Hub ท้องถิ่น :**


นอกจากนี้ยังสามารถติดตั้ง Alby Hub โดยตรงบนพีซีของคุณได้ แม้ว่าตัวเลือกนี้จะไม่ค่อยสะดวกนัก เนื่องจากพีซีของคุณต้องเปิดใช้งานตลอดเวลาเพื่อเข้าถึงโหนด Lightning จากระยะไกล อย่างไรก็ตาม ทางเลือกนี้อาจเหมาะสมกับความต้องการเฉพาะของคุณ




- Alby Hub บนเซิร์ฟเวอร์ส่วนตัว :**


สำหรับผู้ใช้ขั้นสูง Alby Hub สามารถติดตั้งบนเซิร์ฟเวอร์ส่วนตัวได้ด้วยคำสั่งง่ายๆ ตัวเลือกนี้ไม่ได้ครอบคลุมในบทแนะนำนี้ แต่คุณสามารถค้นหาคำแนะนำเฉพาะได้ที่ [บน GitHub ของ Alby](https://github.com/getAlby/hub?tab=readme-ov-file#docker)


บทแนะนำนี้มุ่งเน้นไปที่อินเทอร์เฟซเป็นหลัก ซึ่งจะเหมือนกันไม่ว่าจะเลือกตัวเลือกใดก็ตาม เราจะดูวิธีการปรับใช้ Alby Hub ด้วยตัวเลือกคลาวด์แบบชำระเงิน จากนั้นด้วยตัวเลือก node-in-box (Umbrel หรือ Start9)


![ALBY HUB](assets/fr/02.webp)


สำหรับการติดตั้งในเครื่องคอมพิวเตอร์ของคุณ [ดาวน์โหลดและติดตั้งซอฟต์แวร์ตามระบบปฏิบัติการของคุณ](https://github.com/getAlby/hub/releases) จากนั้นทำตามคำแนะนำเดียวกันบนอินเทอร์เฟซ

## ตัวเลือกการโฮสต์ด้วยตนเอง


หากคุณต้องการใช้ Alby Hub เป็นอินเทอร์เฟซสำหรับโหนด Lightning ที่มีอยู่ของคุณ คุณมีตัวเลือกหลายอย่าง: ติดตั้งบนเซิร์ฟเวอร์, ติดตั้งในเครื่องบนคอมพิวเตอร์ของคุณ, หรือผ่าน node-in-box (Umbrel หรือ Start9) การใช้ Alby Hub ในการกำหนดค่าเหล่านี้ไม่มีค่าใช้จ่าย เราจะมุ่งเน้นไปที่ตัวเลือก node-in-box เนื่องจากฉันพบว่าตัวเลือกเซิร์ฟเวอร์ที่ไม่มีการเข้าถึงทางกายภาพมีความเสี่ยงคล้ายกับเวอร์ชันคลาวด์ และการติดตั้งในเครื่องบนพีซีมักไม่เหมาะสม


ในการตั้งค่านี้บน Umbrel (ขั้นตอนสำหรับ Start9 เหมือนกัน) คุณต้องมีโหนด LND ที่กำหนดค่าไว้แล้ว


เข้าสู่ระบบอินเทอร์เฟซ Umbrel ของคุณและไปที่ร้านค้าแอปพลิเคชัน

## เปิดตัว Alby Hub


คลิกที่ปุ่ม "*เริ่มต้นใช้งาน*"


![ALBY HUB](assets/fr/18.webp)


Alby Hub จะขอให้คุณเลือกพาสเวิร์ด พาสเวิร์ดนี้มีความสำคัญมาก เนื่องจากจะใช้ในการเข้ารหัส wallet ของคุณ ในเวอร์ชันคลาวด์ที่ต้องชำระเงิน คีย์ของคุณจะถูกเก็บไว้บนเซิร์ฟเวอร์ Alby โดยเข้ารหัสด้วยพาสเวิร์ดนี้ที่มีเพียงคุณเท่านั้นที่รู้ จากนั้นจะถูกถอดรหัสและเก็บไว้ใน RAM เท่านั้นเพื่อเซ็นชื่อธุรกรรมเมื่อจำเป็น


ดังนั้นจึงจำเป็นอย่างยิ่งที่จะต้องเลือกใช้รหัสผ่านที่แข็งแกร่ง ใครก็ตามที่มีรหัสผ่านนี้อาจสามารถเข้าถึงโหนดของคุณได้ ตรวจสอบให้แน่ใจว่าคุณได้ทำการสำรองข้อมูลรหัสผ่านนี้ไว้ในรูปแบบทางกายภาพอย่างน้อยหนึ่งครั้งบนกระดาษ หรือดียิ่งกว่านั้นบนแผ่นโลหะเพื่อเพิ่มความปลอดภัย


เมื่อคุณเลือกและบันทึกรหัสผ่านของคุณอย่างระมัดระวังแล้ว ให้คลิกที่ "*Create Password*"


![ALBY HUB](assets/fr/19.webp)


คุณสามารถเข้าถึงโหนด Lightning ของคุณได้แล้ว


![ALBY HUB](assets/fr/20.webp)


การดำเนินการแรกที่ต้องทำคือการบันทึกวลีการกู้คืนของคุณ ซึ่งเป็นที่มาของคีย์ของคุณ ในการทำเช่นนี้ ให้คลิกที่ "Settings" วลีนี้ช่วยให้คุณกู้คืนการเข้าถึง wallet ของคุณได้หากคุณเปิดใช้งานการสำรองข้อมูลอัตโนมัติ


![ALBY HUB](assets/fr/21.webp)


จากนั้นไปที่แท็บ "*Backup*" ป้อนรหัสผ่านของคุณเพื่อเข้าถึง


![ALBY HUB](assets/fr/22.webp)


จากนั้นคุณจะสามารถเข้าถึงวลีการกู้คืน 12 คำของคุณได้ ทำการสำรองข้อมูลทางกายภาพหนึ่งหรือมากกว่าของวลีนี้บนกระดาษหรือโลหะ และเก็บไว้ในที่ปลอดภัย


![ALBY HUB](assets/fr/23.webp)


เมื่อคุณบันทึกวลีแล้ว ให้ทำเครื่องหมายในช่องเพื่อยืนยันว่าคุณได้บันทึกแล้วและคลิก "*Continue*".


![ALBY HUB](assets/fr/24.webp)


## ฉันจะกู้คืนการเข้าถึงบิตคอยน์ของฉันได้อย่างไร?


ก่อนที่จะส่งเงินไปยัง Alby Hub ของคุณ สิ่งสำคัญคือต้องเข้าใจวิธีการกู้คืนเงินในกรณีที่เกิดปัญหา รวมถึงข้อมูลที่จำเป็นสำหรับการกู้คืนนี้ กระบวนการจะแตกต่างกันไปตามลักษณะของเงินที่จะกู้คืนและโหมดการโฮสต์ของโหนดของคุณ


สำหรับผู้ใช้คลาวด์ที่ชำระเงิน การกู้คืนบิตคอยน์ของคุณอย่างสมบูรณ์ต้องการสามองค์ประกอบที่สำคัญ:



- วลีการกู้คืนของคุณ;
- เข้าถึงบัญชี Alby ของคุณเพื่อดึงข้อมูลสำรองอัตโนมัติ


การขาดข้อมูลทั้ง 2 ชิ้นนี้จะทำให้ไม่สามารถกู้คืนบิตคอยน์ของคุณได้อย่างเต็มที่


สำหรับผู้ที่ใช้งาน Alby Hub บนอุปกรณ์ของตนเอง กระบวนการกู้คืนได้ถูกบันทึกไว้ [ที่นี่](https://guides.getalby.com/user-guide/alby-account-and-browser-extension/alby-hub/backups-and-recover#alby-hub-self-hosted-with-an-alby-account)


หากคุณติดตั้ง Alby Hub บนโหนดที่มีอยู่แล้ว คุณจะต้องปฏิบัติตามกระบวนการกู้คืนของระบบปฏิบัติการโหนดนั้น ๆ ตัวอย่างเช่น: Umbrel มี [ตัวเลือก](https://github.com/getumbrel/umbrel/blob/2b266036f62a1594aa60a8a3be30cfb8656e755f/scripts/backup/README.md) ในการเข้ารหัสสถานะล่าสุดของช่องทาง Lightning ของคุณและบันทึกอย่างไดนามิกและไม่ระบุตัวตนผ่าน Tor โปรดทราบว่าเฉพาะการสำรองข้อมูลอัตโนมัติจาก Alby เท่านั้นที่ให้คุณกู้คืน Hub ของคุณได้อย่างสมบูรณ์โดยไม่ต้องปิดช่องทางใด ๆ


## ซื้อช่อง Lightning แรกของคุณ


ตอนนี้คุณสามารถทำตามคำแนะนำที่ Alby Hub ให้ไว้ได้แล้ว คลิกที่ปุ่มเพื่อเปิดช่องทางแรกสำหรับเงินสดที่เข้ามา


![ALBY HUB](assets/fr/25.webp)


เลือก "*Open Channel*". หากคุณไม่ได้ตั้งใจที่จะเป็นโหนดการกำหนดเส้นทางและไม่จำเป็นต้องมีโหนดโดยเฉพาะ ฉันแนะนำให้คุณเลือกช่องทางส่วนตัว


![ALBY HUB](assets/fr/26.webp)


Alby Hub จะ generate ใบแจ้งหนี้ให้คุณชำระ การชำระเงินนี้ครอบคลุมค่าธรรมเนียมการทำธุรกรรมที่จำเป็นในการเปิดช่องของคุณ รวมถึงค่าบริการของ LSP (*ผู้ให้บริการสายฟ้า*) ที่จะเปิดช่องไปยังโหนดของคุณ ทำให้คุณสามารถรับชำระเงินได้ทันที


![ALBY HUB](assets/fr/27.webp)


เมื่อชำระใบแจ้งหนี้และยืนยันการทำธุรกรรมแล้ว ช่องทาง Lightning แรกของคุณจะถูกสร้างขึ้น


![ALBY HUB](assets/fr/28.webp)


ในแท็บ "*Node*" คุณจะเห็นว่าตอนนี้คุณมีเงินสดเข้ามา ทำให้คุณสามารถรับชำระเงินผ่าน Lightning ได้


![ALBY HUB](assets/fr/29.webp)


ในการรับชำระเงิน ให้คลิกที่แท็บ "*Wallet*" แล้วคลิกที่ "*Receive*".


![ALBY HUB](assets/fr/30.webp)


ป้อนจำนวนเงินและเพิ่มคำอธิบายหากจำเป็น จากนั้นคลิกที่ "*Create Invoice*"


![ALBY HUB](assets/fr/31.webp)


ฉันได้รับการชำระเงินครั้งแรกจำนวน 120,000 sats.


![ALBY HUB](assets/fr/32.webp)


โดยการกลับไปที่แท็บ "*Wallet*" คุณสามารถตรวจสอบยอดคงเหลือ wallet ของคุณได้ โปรดทราบว่า Alby Hub จะกัน 354 sats โดยอัตโนมัติเมื่อคุณทำการชำระเงินครั้งแรก สำหรับแต่ละช่องทาง Lightning ที่คุณเปิดหลังจากนั้น Alby Hub จะกันเงินสำรองโดยอัตโนมัติเท่ากับ 1% ของความจุของช่องทางนั้น เงินสำรองนี้เป็นมาตรการรักษาความปลอดภัยที่ช่วยให้โหนดของคุณสามารถกู้คืนเงินของช่องทางได้ในกรณีที่มีการพยายามฉ้อโกงโดยเพื่อนของคุณ นั่นเป็นเหตุผลที่ว่าแม้ว่าฉันจะส่ง 120,000 sats แต่มีเพียง 119,646 sats เท่านั้นที่แสดงในยอดคงเหลือของฉัน


![ALBY HUB](assets/fr/33.webp)


## การฝากบิตคอยน์แบบออนเชน


หากคุณต้องการมีเงินสดออกเพื่อทำการชำระเงิน คุณสามารถเปิดช่องทางด้วยตัวเองได้เช่นกัน ในการทำเช่นนี้ คุณจะต้องมีบิตคอยน์ในเครือข่ายใน wallet ของคุณ


จากแท็บ "*Node*" ให้คลิกที่ "*Deposit*"


![ALBY HUB](assets/fr/34.webp)


ส่งบิตคอยน์ไปยังที่อยู่ที่แสดง ที่อยู่นี้ได้มาจากวลีการกู้คืนที่คุณบันทึกไว้ก่อนหน้านี้


![ALBY HUB](assets/fr/35.webp)


ฉันส่ง 72,000 sats ตอนนี้พวกมันปรากฏใน "*ยอดเงินออม*" ซึ่งรวมถึงเงินทั้งหมดที่ฉันเป็นเจ้าของบนเชน และไม่ใช่บน Lightning


![ALBY HUB](assets/fr/36.webp)


## เปิดช่องทาง Lightning


ตอนนี้ที่คุณมีเงินบนเชนแล้ว คุณสามารถเปิดช่องทาง Lightning ใหม่ได้ ขอแนะนำให้เปิดหลายช่องทาง โดยมีจำนวนเงินเพียงพอเพื่อให้แน่ใจว่าคุณสามารถชำระเงินได้โดยไม่มีข้อจำกัด ผู้ให้บริการ LSP (*Lightning Service Providers*) ส่วนใหญ่ต้องการขั้นต่ำ 150,000 sats เพื่อเปิดช่องทางกับคุณ


ในแท็บ "*Node*" ให้คลิกที่ "*Open Channel*"


![ALBY HUB](assets/fr/37.webp)


เลือกขนาดของช่องสัญญาณของคุณ ฉันแนะนำว่าอย่าเปิดช่องสัญญาณที่เล็กเกินไป โดยคำนึงว่านี่คือโหนด Lightning และเครื่องที่โฮสต์คีย์ของคุณไม่ได้ให้ระดับความปลอดภัยเท่ากับฮาร์ดแวร์ wallet ดังนั้นควรระมัดระวังกับจำนวนเงินที่คุณเลือกบล็อก


![ALBY HUB](assets/fr/38.webp)


ในเมนู "*ตัวเลือกขั้นสูง*" คุณสามารถเลือก LSP ที่จะเปิดช่องของคุณด้วย หรือป้อนโหนด Lightning อื่นด้วยตนเอง


![ALBY HUB](assets/fr/39.webp)


จากนั้นคลิกที่ "*Open Channel*"


![ALBY HUB](assets/fr/40.webp)


โปรดรอในขณะที่ช่องของคุณได้รับการยืนยันบนเชน


![ALBY HUB](assets/fr/41.webp)


ช่องใหม่ของคุณจะปรากฏในแท็บ "*Node*"


![ALBY HUB](assets/fr/42.webp)

## ปรับแต่งที่อยู่ Lightning ของคุณ


Alby เสนอทางเลือกให้คุณใช้ที่อยู่ Lightning ซึ่งช่วยให้คุณรับการชำระเงินบนโหนดของคุณโดยไม่ต้องสร้างใบแจ้งหนี้ generate ด้วยตนเองทุกครั้ง โดยค่าเริ่มต้น Alby จะกำหนดที่อยู่ Lightning ให้คุณ แต่คุณสามารถปรับแต่งได้ เข้าสู่ระบบบัญชีออนไลน์ Alby ของคุณ คลิกที่ชื่อของคุณที่มุมขวาบน จากนั้นเลือก "*Settings*"


![ALBY HUB](assets/fr/63.webp)


ไปที่เมนู "*Lightning Address*"


![ALBY HUB](assets/fr/64.webp)


แก้ไขที่อยู่ของคุณ จากนั้นยืนยันโดยคลิกที่ "*อัปเดตที่อยู่ lightning ของคุณ*"


![ALBY HUB](assets/fr/65.webp)


โปรดทราบว่าเมื่อที่อยู่ของคุณถูกเปลี่ยนแล้ว มันจะไม่เป็นของคุณอีกต่อไป ดังนั้นโปรดตรวจสอบให้แน่ใจว่าคุณจะไม่ส่ง sats ไปยังที่อยู่นั้นอีก


และนั่นคือทั้งหมด ตอนนี้คุณรู้วิธีใช้ Lightning กับโหนดของคุณเองโดยใช้เครื่องมือ Alby Hub แล้ว หากคุณพบว่าบทแนะนำนี้มีประโยชน์ ฉันจะขอบคุณมากหากคุณกดนิ้วโป้งสีเขียวด้านล่าง โปรดอย่าลังเลที่จะแบ่งปันบทความนี้บนเครือข่ายสังคมของคุณ ขอบคุณมาก!
