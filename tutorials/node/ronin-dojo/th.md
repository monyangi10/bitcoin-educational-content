## RoninDojo คืออะไร?


Dojo เป็นเซิร์ฟเวอร์โหนด Bitcoin แบบเต็มที่พัฒนาโดยทีม Samourai Wallet คุณสามารถติดตั้งได้อย่างอิสระบนเครื่องใดก็ได้


RoninDojo เป็นเครื่องมือช่วยติดตั้งและบริหารจัดการสำหรับ Dojo และเครื่องมืออื่นๆ อีกมากมาย RoninDojo นำการใช้งานดั้งเดิมของ Dojo มาปรับปรุงและเพิ่มเครื่องมืออื่นๆ เข้าไป พร้อมทั้งทำให้การติดตั้งและการจัดการง่ายขึ้น


พวกเขายังมีฮาร์ดแวร์แบบ "plug-and-play" ที่ชื่อว่า RoninDojo Tanto ซึ่งมี RoninDojo ติดตั้งล่วงหน้าบนเครื่องที่ประกอบโดยทีมของพวกเขาเอง Tanto เป็นโซลูชันที่ต้องชำระเงิน เหมาะสำหรับผู้ที่ไม่ต้องการลงมือทำเอง


รหัสสำหรับ RoninDojo เป็นโอเพ่นซอร์ส ดังนั้นจึงสามารถติดตั้งโซลูชันนี้บนฮาร์ดแวร์ของคุณเองได้เช่นกัน ตัวเลือกนี้มีความคุ้มค่าแต่ต้องการการจัดการเพิ่มเติมเล็กน้อย ซึ่งเราจะทำในบทความนี้


RoninDojo คือ Dojo ดังนั้นจึงสามารถผสานรวม Whirlpool CLI เข้ากับโหนด Bitcoin ของคุณได้อย่างง่ายดายเพื่อให้ได้ประสบการณ์ CoinJoin ที่ดีที่สุด ด้วย Whirlpool CLI ไม่เพียงแต่คุณสามารถให้บิตคอยน์ของคุณรีมิกซ์ได้ตลอด 24/7 โดยไม่จำเป็นต้องเปิดคอมพิวเตอร์ส่วนตัวของคุณ แต่คุณยังสามารถปรับปรุงความเป็นส่วนตัวของคุณได้อย่างมากอีกด้วย


RoninDojo รวมเครื่องมืออื่นๆ อีกมากมายที่พึ่งพา Dojo ของคุณ เช่น เครื่องคิดเลข Boltzmann ซึ่งกำหนดระดับความเป็นส่วนตัวของธุรกรรม, เซิร์ฟเวอร์ Electrum เพื่อเชื่อมต่อกระเป๋าเงิน Bitcoin ต่างๆ ของคุณกับโหนดของคุณ, หรือเซิร์ฟเวอร์ Mempool เพื่อสังเกตธุรกรรมของคุณอย่างเป็นส่วนตัว

เมื่อเปรียบเทียบกับโซลูชันโหนดอื่น ๆ เช่น Umbrel ซึ่งฉันได้นำเสนอให้คุณในบทความนี้ RoninDojo มุ่งเน้นอย่างลึกซึ้งไปที่โซลูชัน "On Chain" และเครื่องมือที่เพิ่มประสิทธิภาพความเป็นส่วนตัวของผู้ใช้ ดังนั้น RoninDojo จึงไม่อนุญาตให้มีการโต้ตอบกับ Lightning Network

RoninDojo มีเครื่องมือน้อยกว่าเมื่อเทียบกับ Umbrel แต่ฟีเจอร์สำคัญไม่กี่อย่างสำหรับผู้ใช้ Bitcoin ที่มีอยู่บน Ronin นั้นมีความเสถียรอย่างมาก


ดังนั้นหากคุณไม่ต้องการคุณสมบัติทั้งหมดของเซิร์ฟเวอร์ Umbrel และต้องการเพียงโหนดที่เรียบง่ายและเสถียรพร้อมเครื่องมือที่จำเป็นเพียงไม่กี่อย่างเช่น Whirlpool หรือ Mempool แล้ว RoninDojo อาจเป็นทางออกที่ดีสำหรับคุณ


ในความเห็นของฉัน การพัฒนาของ Umbrel มุ่งเน้นไปที่ Lightning Network และเครื่องมืออเนกประสงค์อย่างมาก มันยังคงเป็นโหนด Bitcoin แต่เป้าหมายคือการทำให้มันเป็นมินิเซิร์ฟเวอร์ที่สามารถทำงานหลายอย่างพร้อมกันได้ ในทางตรงกันข้าม การพัฒนาของ RoninDojo มุ่งเน้นไปที่ทีมที่ Samourai Wallet และเน้นไปที่เครื่องมือที่จำเป็นสำหรับผู้ใช้ Bitcoin เพื่อให้สามารถมีความเป็นอิสระอย่างเต็มที่และการจัดการความเป็นส่วนตัวที่เหมาะสมบน Bitcoin


โปรดทราบว่าการตั้งค่าโหนด RoninDojo นั้นซับซ้อนกว่าโหนด Umbrel เล็กน้อย


ตอนนี้ที่เราได้วาดภาพของ RoninDojo แล้ว มาดูกันว่าเราจะตั้งค่าโหนดนี้อย่างไร


## จะเลือกฮาร์ดแวร์อะไรดี?


ในการเลือกเครื่องที่โฮสต์และรัน RoninDojo คุณมีตัวเลือกหลายอย่าง


ตามที่อธิบายไว้ก่อนหน้านี้ ตัวเลือกที่ง่ายที่สุดคือการสั่งซื้อ Tanto ซึ่งเป็นเครื่อง plug-and-play ที่ออกแบบมาโดยเฉพาะสำหรับวัตถุประสงค์นี้ ในการสั่งซื้อของคุณ ไปที่นี่: [link](https://shop.ronindojo.io/product-category/nodes/).


เนื่องจากทีม RoninDojo ผลิตโค้ดแบบโอเพนซอร์ส จึงสามารถนำ RoninDojo ไปใช้บนเครื่องอื่นได้เช่นกัน คุณสามารถค้นหาเวอร์ชันล่าสุดของตัวช่วยติดตั้งได้ที่หน้านี้: [link](https://ronindojo.io/en/downloads.html) และเวอร์ชันล่าสุดของโค้ดได้ที่หน้านี้: [link](https://code.samourai.io/ronindojo/RoninDojo)


ส่วนตัวแล้ว ฉันติดตั้งมันบน Raspberry Pi 4 8GB และทุกอย่างทำงานได้อย่างสมบูรณ์แบบ


อย่างไรก็ตาม โปรดทราบว่าทีม RoninDojo ระบุว่ามักจะมีปัญหากับเคสและอะแดปเตอร์ SSD ดังนั้นจึงไม่แนะนำให้ใช้เคสที่มีสายสำหรับ SSD ของเครื่องคุณ แต่ควรใช้การ์ดขยายพื้นที่จัดเก็บที่ออกแบบมาเฉพาะสำหรับเมนบอร์ดของคุณ เช่น การ์ดขยายพื้นที่จัดเก็บ Raspberry Pi 4


นี่คือตัวอย่างวิธีการตั้งค่าโหนด RoninDojo ของคุณเอง:



- Raspberry Pi 4
- เคสพร้อมพัดลม
- การ์ดขยายพื้นที่จัดเก็บที่เข้ากันได้
- สายไฟ
- การ์ด micro SD อุตสาหกรรมขนาด 16GB หรือมากกว่า
- SSD ขนาด 1TB หรือใหญ่กว่า
- สายเคเบิล Ethernet RJ45, แนะนำหมวดหมู่ 8


## วิธีการติดตั้ง RoninDojo?


### ขั้นตอนที่ 1: เตรียมการ์ด micro SD ที่สามารถบูตได้


เมื่อคุณประกอบเครื่องเสร็จแล้ว คุณสามารถเริ่มการติดตั้ง RoninDojo ได้ ในการทำเช่นนี้ ให้เริ่มต้นด้วยการสร้าง micro SD card ที่สามารถบูตได้โดยการเบิร์นดิสก์อิมเมจที่เหมาะสมลงไป


ใส่การ์ด micro SD ของคุณลงในคอมพิวเตอร์ส่วนบุคคล จากนั้นไปที่เว็บไซต์ทางการของ RoninDojo เพื่อดาวน์โหลดภาพดิสก์ RoninOS: https://ronindojo.io/en/downloads.html.


ดาวน์โหลดดิสก์อิมเมจที่ตรงกับฮาร์ดแวร์ของคุณ ในกรณีของฉัน ฉันได้ดาวน์โหลดอิมเมจ "MANJARO-ARM-RONINOS-RPI4-22.03.IMG.XZ":


![Download RoninOS disk image](assets/2.webp)


เมื่อดาวน์โหลดภาพแล้ว ให้ตรวจสอบความสมบูรณ์โดยใช้ไฟล์ .SHA256 ที่สอดคล้องกัน ฉันจะอธิบายรายละเอียดวิธีการทำในบทความนี้: วิธีตรวจสอบความสมบูรณ์ของซอฟต์แวร์ Bitcoin บน Windows?


คำแนะนำเฉพาะสำหรับการตรวจสอบความสมบูรณ์ของ RoninOS มีให้บริการในหน้านี้: https://wiki.ronindojo.io/en/extras/verify.


ในการเบิร์นภาพนี้ลงในการ์ด micro SD ของคุณ คุณสามารถใช้ซอฟต์แวร์เช่น Balena Etcher ซึ่งคุณสามารถดาวน์โหลดได้ที่นี่: https://www.balena.io/etcher/.


ในการทำเช่นนี้ ให้เลือกภาพใน Etcher และแฟลชลงบนการ์ด micro SD:


![Burn disk image with Etcher](assets/3.webp)


เมื่อการดำเนินการเสร็จสิ้น คุณสามารถใส่การ์ด micro SD ที่สามารถบูตได้ลงใน Raspberry Pi และเปิดเครื่องได้เลย


### ขั้นตอนที่ 2: กำหนดค่า RoninOS.


RoninOS เป็นระบบปฏิบัติการของโหนด RoninDojo ของคุณ มันเป็นเวอร์ชันที่ปรับแต่งของ Manjaro ซึ่งเป็นการแจกจ่าย Linux หลังจากเริ่มเครื่องของคุณและรอสักครู่ คุณสามารถเริ่มการกำหนดค่าได้


ในการเชื่อมต่อจากระยะไกล คุณจะต้องค้นหาที่อยู่ IP ของเครื่อง RoninDojo ของคุณ ในการทำเช่นนี้ คุณสามารถเชื่อมต่อกับแผงการจัดการของกล่องอินเทอร์เน็ตของคุณ หรือคุณยังสามารถดาวน์โหลดซอฟต์แวร์เช่น https://angryip.org/ เพื่อสแกนเครือข่ายภายในของคุณและค้นหาที่อยู่ IP ของเครื่องได้


เมื่อคุณพบ IP แล้ว คุณสามารถควบคุมเครื่องของคุณจากคอมพิวเตอร์เครื่องอื่นที่เชื่อมต่อกับเครือข่ายท้องถิ่นเดียวกันโดยใช้ SSH ได้


จากคอมพิวเตอร์ที่ใช้ macOS หรือ Linux เพียงแค่เปิดเทอร์มินัล จากคอมพิวเตอร์ที่ใช้ Windows คุณสามารถใช้เครื่องมือเฉพาะทางอย่าง Putty หรือใช้ Windows PowerShell โดยตรง


เมื่อเปิดเทอร์มินัลแล้ว ให้พิมพ์คำสั่งต่อไปนี้:

```
ssh root@192.168.?.?
```


เพียงแทนที่เครื่องหมายคำถามสองตัวด้วย IP ของ RoninDojo ที่คุณพบก่อนหน้านี้

### ขั้นตอนที่ 5: ดาวน์โหลด proof-of-work chain และเข้าถึง RoninUI.


เมื่อติดตั้งเสร็จสิ้น โหนดของคุณจะเริ่มดาวน์โหลดเชน Bitcoin proof-of-work นี่เรียกว่าการดาวน์โหลดบล็อกเริ่มต้น (IBD) ซึ่งปกติจะใช้เวลาระหว่าง 2 ถึง 14 วัน ขึ้นอยู่กับการเชื่อมต่ออินเทอร์เน็ตและอุปกรณ์ของคุณ


คุณสามารถติดตามความคืบหน้าของการดาวน์โหลดเชนได้โดยการเข้าถึงอินเทอร์เฟซเว็บ RoninUI


หากต้องการเข้าถึงจากเครือข่ายท้องถิ่น ให้พิมพ์สิ่งต่อไปนี้ในแถบที่อยู่ของเบราว์เซอร์ของคุณ:



- ป้อนที่อยู่ IP ของเครื่องของคุณโดยตรง (192.168.?.?)
- หรือป้อน: ronindojo.local


โปรดจำไว้ว่าให้ปิด VPN ของคุณหากคุณกำลังใช้งานอยู่


### ความเป็นไปได้ในการหักมุม


หากคุณไม่สามารถเชื่อมต่อกับ RoninUI จากเบราว์เซอร์ของคุณได้ ให้ตรวจสอบการทำงานที่ถูกต้องของแอปพลิเคชันจาก Terminal ของคุณที่เชื่อมต่อกับโหนดของคุณผ่าน SSH เชื่อมต่อกับเมนูหลักโดยทำตามขั้นตอนก่อนหน้า:



- พิมพ์: SSH username@192.168.?.? แทนที่ด้วยข้อมูลประจำตัวของคุณ



- กรุณาใส่รหัสผ่านผู้ใช้ของคุณ


เมื่ออยู่ในเมนูหลัก ไปที่: **RoninUI > Restart**


หากแอปพลิเคชันเริ่มต้นใหม่อย่างถูกต้อง แสดงว่าเป็นปัญหากับการเชื่อมต่อจากเบราว์เซอร์ของคุณ ตรวจสอบว่าคุณไม่ได้ใช้ VPN และตรวจสอบให้แน่ใจว่าคุณเชื่อมต่อกับเครือข่ายเดียวกันกับโหนดของคุณ


หากการเริ่มต้นใหม่ทำให้เกิดข้อผิดพลาด ให้ลองอัปเดตระบบปฏิบัติการแล้วติดตั้ง RoninUI ใหม่อีกครั้ง ในการอัปเดต OS: **System > Software Updates > Update Operating System**


เมื่อการอัปเดตและรีสตาร์ทเสร็จสิ้น ให้เชื่อมต่อกับโหนดของคุณอีกครั้งผ่าน SSH และติดตั้ง RoninUI ใหม่: **RoninUI > Re-install**


หลังจากดาวน์โหลด RoninUI อีกครั้ง ให้ลองเชื่อมต่อกับ RoninUI ผ่านเบราว์เซอร์ของคุณ


**เคล็ดลับ:** หากคุณออกจาก RoninCLI โดยไม่ได้ตั้งใจและพบว่าตัวเองอยู่บนเทอร์มินัล Manjaro เพียงแค่ป้อนคำสั่ง "ronin" เพื่อกลับไปยังเมนูหลักของ RoninCLI โดยตรง


### เข้าสู่ระบบเว็บ


คุณยังสามารถเข้าสู่ระบบอินเทอร์เฟซเว็บ RoninUI จากเครือข่ายใดก็ได้โดยใช้ Tor ในการทำเช่นนี้ ให้ดึงที่อยู่ Tor ของ RoninUI ของคุณจาก RoninCLI: **Credentials > Ronin UI**


ดึงที่อยู่ Tor ที่ลงท้ายด้วย .onion แล้วเข้าสู่ระบบ Ronin UI โดยป้อนที่อยู่นี้ในเบราว์เซอร์ Tor ของคุณ ระวังอย่าให้ข้อมูลรับรองต่างๆ ของคุณรั่วไหล เนื่องจากเป็นข้อมูลที่ละเอียดอ่อน


![RoninUI web login interface](assets/20.webp)


เมื่อเข้าสู่ระบบแล้ว คุณจะถูกขอให้ใส่รหัสผ่านผู้ใช้ของคุณ ซึ่งเป็นรหัสผ่านเดียวกันกับที่คุณใช้ในการเข้าสู่ระบบผ่าน SSH


![RoninUI web interface management panel](assets/21.webp)

หลังจากดาวน์โหลดบล็อกเชนทั้งหมดแล้ว ดัชนีจะทำการบีบอัดฐานข้อมูล การดำเนินการนี้ใช้เวลาประมาณ 12 ชั่วโมง คุณยังสามารถติดตามความคืบหน้าได้ภายใต้ "Indexer" บน RoninUI

## วิธีใช้ RoninDojo?


เมื่อดาวน์โหลดและบีบอัดเชนเสร็จสิ้น คุณสามารถเริ่มเพลิดเพลินกับบริการที่เสนอโดยโหนด RoninDojo ใหม่ของคุณได้ มาดูกันว่าจะใช้งานอย่างไร


### การเชื่อมต่อซอฟต์แวร์ wallet กับ electrs.


ยูทิลิตี้แรกของโหนดที่คุณติดตั้งและซิงโครไนซ์ใหม่จะเป็นการกระจายธุรกรรมของคุณไปยังเครือข่าย Bitcoin ดังนั้นคุณอาจต้องการเชื่อมต่อซอฟต์แวร์การจัดการ wallet ที่แตกต่างกันของคุณเข้ากับมัน


คุณสามารถทำสิ่งนี้ได้โดยใช้ Electrum Rust Server (electrs) แอปพลิเคชันนี้มักจะถูกติดตั้งล่วงหน้าบนโหนด RoninDojo ของคุณ หากไม่ คุณสามารถติดตั้งด้วยตนเองจากอินเทอร์เฟซ RoninCLI ได้


เพียงไปที่: **Applications > Manage Applications > Install Electrum Server**


ในการรับที่อยู่ Tor ของ Electrum Server ของคุณ จากเมนู RoninCLI ไปที่:

**ข้อมูลประจำตัว > Electrs**


คุณเพียงแค่ต้องป้อนลิงก์ .onion ในซอฟต์แวร์ wallet ของคุณ ตัวอย่างเช่น ใน Sparrow Wallet ไปที่แท็บ:

**ไฟล์ > การตั้งค่า > เซิร์ฟเวอร์**


ในประเภทเซิร์ฟเวอร์ ให้เลือก `Private Electrum` จากนั้นป้อนที่อยู่ Tor ของ Electrum Server ของคุณในช่องที่สอดคล้องกัน สุดท้าย คลิกที่ "Test Connection" เพื่อทดสอบและบันทึกการเชื่อมต่อของคุณ


![Sparrow Wallet connection interface to an electrs](assets/23.webp)


### เชื่อมต่อซอฟต์แวร์ wallet กับ Samourai Dojo


แทนที่จะใช้ Electrs คุณยังสามารถใช้ Samourai Dojo เพื่อเชื่อมต่อซอฟต์แวร์ที่เข้ากันได้ wallet ของคุณกับโหนด RoninDojo ของคุณได้ ตัวอย่างเช่น Samourai Wallet มีตัวเลือกนี้


ในการทำเช่นนี้ เพียงแค่สแกนรหัส QR การเชื่อมต่อของ Dojo ของคุณ เพื่อเข้าถึงจาก RoninUI ให้คลิกที่แท็บ "Dashboard" แล้วคลิกที่ปุ่ม "Manage" ในกล่องของ Dojo ของคุณ จากนั้นคุณจะสามารถเห็นรหัส QR การเชื่อมต่อสำหรับ Dojo และ BTC-RPC Explorer ของคุณได้ เพื่อแสดงรหัสเหล่านี้ ให้คลิกที่ "Display values"


![Retrieving the connection QR code to Dojo](assets/24.webp)


ในการเชื่อมต่อ Samourai Wallet ของคุณกับ Dojo คุณจะต้องสแกน QR โค้ดนี้ระหว่างการติดตั้งแอปพลิเคชัน:


![Connecting to Dojo from the Samourai Wallet application](assets/25.webp)


### ใช้ Mempool Explorer ของคุณเอง


เครื่องมือที่จำเป็นสำหรับผู้ใช้ Bitcoin, ตัวสำรวจช่วยให้คุณตรวจสอบข้อมูลต่าง ๆ เกี่ยวกับเครือข่าย Bitcoin ได้ ด้วย Mempool, ตัวอย่างเช่น คุณสามารถตรวจสอบค่าธรรมเนียมที่ผู้ใช้อื่นใช้ในเวลาจริงเพื่อปรับค่าของคุณตามนั้น คุณยังสามารถตรวจสอบสถานะการยืนยันของธุรกรรมของคุณหรือดูยอดคงเหลือของที่อยู่ได้อีกด้วย


เครื่องมือสำรวจเหล่านี้สามารถทำให้คุณเสี่ยงต่อความเป็นส่วนตัวและต้องให้ความไว้วางใจกับฐานข้อมูลของบุคคลที่สาม เมื่อคุณใช้เครื่องมือออนไลน์นี้โดยไม่ผ่านโหนดของคุณเอง:



- คุณเสี่ยงที่จะรั่วไหลข้อมูลเกี่ยวกับ wallet ของคุณ



- คุณไว้วางใจผู้จัดการเว็บไซต์สำหรับโซ่ proof-of-work ที่พวกเขาโฮสต์

### การใช้เครื่องคิดเลขของ Boltzmann


เครื่องคำนวณ Boltzmann เป็นเครื่องมือที่ช่วยให้คุณคำนวณเมตริกขั้นสูงต่างๆ บนธุรกรรม Bitcoin ได้อย่างง่ายดาย รวมถึงระดับเอนโทรปี ข้อมูลทั้งหมดนี้จะช่วยให้คุณสามารถวัดระดับความเป็นส่วนตัวของธุรกรรมและตรวจจับข้อผิดพลาดที่อาจเกิดขึ้นได้ เครื่องมือนี้ติดตั้งล่วงหน้าในโหนด RoninDojo ของคุณแล้ว


ในการเข้าถึงจาก RoninCLI ให้เชื่อมต่อผ่าน SSH จากนั้นไปที่เมนู:

**Samourai Toolkit > Boltzmann Calculator**


ก่อนที่จะอธิบายวิธีการใช้งานบน RoninDojo ฉันจะอธิบายว่าเมตริกเหล่านี้แสดงถึงอะไร คำนวณอย่างไร และใช้สำหรับอะไรบ้าง


ตัวชี้วัดเหล่านี้สามารถใช้สำหรับธุรกรรม Bitcoin ใด ๆ ก็ได้ แต่พวกมันน่าสนใจเป็นพิเศษสำหรับการศึกษาคุณภาพของธุรกรรม Coinjoin


1. ตัวบ่งชี้แรกที่คำนวณโดยซอฟต์แวร์นี้คือจำนวนของการจัดเรียงที่เป็นไปได้ ซึ่งถูกระบุบนเครื่องคิดเลขว่า "nb combinations" โดยพิจารณาจากค่าของ UTXOs ตัวบ่งชี้นี้แสดงถึงจำนวนของการจับคู่ที่เป็นไปได้จากอินพุตไปยังเอาต์พุต


**หมายเหตุ:** หากคุณไม่คุ้นเคยกับคำว่า `UTXO` ฉันขอแนะนำให้อ่านบทความสั้น ๆ นี้:


> กลไกของธุรกรรม Bitcoin: UTXO, อินพุต, และเอาต์พุต

กล่าวอีกนัยหนึ่ง ตัวบ่งชี้นี้แสดงถึงจำนวนการตีความที่เป็นไปได้สำหรับธุรกรรมที่กำหนด ตัวอย่างเช่น โครงสร้าง Whirlpool 5x5 Coinjoin จะมีจำนวนการผสมผสานที่เป็นไปได้เท่ากับ 1496:


![Schema of a Coinjoin transaction on kycp.org](assets/32.webp)


เครดิต: KYCP


2. ตัวชี้วัดที่คำนวณได้ตัวที่สองคือเอนโทรปีของธุรกรรม เนื่องจากจำนวนของการจัดเรียงที่เป็นไปได้อาจสูงมากสำหรับธุรกรรมหนึ่ง ๆ จึงสามารถเลือกใช้เอนโทรปีแทนได้ เอนโทรปีแสดงถึงลอการิทึมฐานสองของจำนวนการจัดเรียงที่เป็นไปได้ สูตรของมันคือดังนี้:



- E: เอนโทรปีของธุรกรรม
- C: จำนวนชุดค่าผสมที่เป็นไปได้สำหรับการทำธุรกรรม


**E = log2(C)**


ในทางคณิตศาสตร์ ลอการิทึมฐานสอง (ฐาน 2) เป็นฟังก์ชันผกผันของฟังก์ชันยกกำลังของ 2 กล่าวอีกนัยหนึ่ง ลอการิทึมฐานสองของ x คือกำลังที่จำนวน 2 ต้องถูกยกขึ้นเพื่อให้ได้ค่าของ x


ดังนี้:


**E = log2(C)**


**C = 2^E**


ตัวบ่งชี้นี้แสดงในรูปแบบบิต ตัวอย่างเช่น นี่คือการคำนวณเอนโทรปีของธุรกรรม Whirlpool 5x5 Coinjoin โดยมีจำนวนการจัดเรียงที่เป็นไปได้ที่กล่าวถึงก่อนหน้านี้เท่ากับ 1496:


**C = 1496**


**E = log2(1496)**


**E = 10.5469 bits**


ดังนั้น การทำธุรกรรม Coinjoin นี้มีค่าเอนโทรปี 10.5469 บิต ซึ่งถือว่าดีมาก


ตัวบ่งชี้นี้ยิ่งสูงเท่าใด การตีความธุรกรรมก็ยิ่งหลากหลายมากขึ้นเท่านั้น และดังนั้นธุรกรรมก็ยิ่งเป็นความลับมากขึ้นเท่านั้น


ลองมาดูอีกตัวอย่างหนึ่ง นี่คือตัวอย่างการทำธุรกรรมแบบ "คลาสสิก" ที่มีอินพุตหนึ่งรายการและเอาต์พุตสองรายการ:


![Bitcoin transaction schema on oxt.me](assets/34.webp)


เครดิต: OXT


ธุรกรรมนี้มีการตีความได้เพียงแบบเดียว:


**[(inp 0) > (Outp 0 ; Outp 1)]**


ดังนั้นเอนโทรปีของมันจะเท่ากับ 0:


**C = 1**,

**E = log2(C)**,

**E = 0**


3. ตัวชี้วัดที่สามที่ส่งคืนโดยเครื่องคำนวณ Boltzmann คือประสิทธิภาพของ Tx ที่เรียกว่า "Wallet Efficiency" ตัวชี้วัดนี้ช่วยให้สามารถเปรียบเทียบธุรกรรมที่ป้อนเข้ากับธุรกรรมที่ดีที่สุดที่เป็นไปได้ในโครงสร้างเดียวกันได้อย่างง่ายดาย


เราจะนำเสนอแนวคิดของเอนโทรปีสูงสุด ซึ่งแสดงถึงเอนโทรปีที่สามารถบรรลุได้สูงสุดสำหรับโครงสร้างธุรกรรมที่กำหนด ตัวอย่างเช่น โครงสร้าง Whirlpool 5x5 Coinjoin จะมีเอนโทรปีสูงสุดที่ 10.5469 ตัวบ่งชี้ประสิทธิภาพจะเปรียบเทียบเอนโทรปีสูงสุดนี้กับเอนโทรปีจริงของธุรกรรมที่ป้อน สูตรของมันคือดังนี้:



- ER: เอนโทรปีที่แท้จริงแสดงในบิต
- EM: เอนโทรปีสูงสุดที่มีโครงสร้างเดียวกันแสดงในบิต
- Ef: ประสิทธิภาพที่แสดงในบิต


**Ef = ER - EM**


**Ef = 10.5469 - 10.5469**


**Ef = 0 บิต**


ตัวบ่งชี้นี้ยังแสดงเป็นเปอร์เซ็นต์ ดังนั้นสูตรจะกลายเป็น:



- CR: จำนวนชุดค่าผสมที่เป็นไปได้จริง
- CM: จำนวนการจัดเรียงที่เป็นไปได้สูงสุดที่มีโครงสร้างเดียวกัน
- Ef: ประสิทธิภาพที่แสดงเป็นเปอร์เซ็นต์


**Ef = CR/CM**


**Ef = 1496/1496**


**Ef = 100%**


ประสิทธิภาพ 100% หมายความว่าธุรกรรมนี้มีความเป็นส่วนตัวสูงสุดเท่าที่เป็นไปได้เมื่อเทียบกับโครงสร้างของมัน


4. ตัวชี้วัดที่คำนวณได้ตัวที่สี่คือความหนาแน่นของเอนโทรปี มันช่วยให้เราสามารถเชื่อมโยงเอนโทรปีเข้ากับแต่ละอินพุตหรือเอาต์พุต ตัวชี้วัดนี้สามารถใช้เพื่อเปรียบเทียบประสิทธิภาพระหว่างธุรกรรมที่มีขนาดต่างกันได้


การคำนวณนั้นง่ายมาก: เราแบ่งเอนโทรปีของธุรกรรมด้วยจำนวนอินพุตและเอาต์พุตที่มีอยู่ ตัวอย่างเช่น สำหรับ Whirlpool 5x5 Coinjoin เราจะมี:


ED: ความหนาแน่นของเอนโทรปีที่แสดงในบิต

E: เอนโทรปีของธุรกรรมที่แสดงในบิต

T: จำนวนอินพุตและเอาต์พุตทั้งหมดในธุรกรรม


T = 5 + 5 = 10

ED = E / T

ED = 10.5469 / 10

ED = 1.054 บิต


ข้อมูลชิ้นที่ห้าที่ให้โดยเครื่องคิดเลข Boltzmann คือ ตารางความน่าจะเป็นของการเชื่อมโยงระหว่างอินพุตและเอาต์พุต ตารางนี้จะให้คะแนนความน่าจะเป็น (คะแนน Boltzmann) ว่าอินพุตที่กำหนดสอดคล้องกับเอาต์พุตที่กำหนด


หากเรานำตัวอย่างของเรากับ Whirlpool Coinjoin ตารางความน่าจะเป็นจะเป็นดังนี้:


| %       | Output 0 | Output 1 | Output 2 | Output 3 | Output 4 |
|---------|----------|----------|----------|----------|----------|
| Input 0 | 34%      | 34%      | 34%      | 34%      | 34%      |
| Input 1 | 34%      | 34%      | 34%      | 34%      | 34%      |
| Input 2 | 34%      | 34%      | 34%      | 34%      | 34%      |
| Input 3 | 34%      | 34%      | 34%      | 34%      | 34%      |
| Input 4 | 34%      | 34%      | 34%      | 34%      | 34%      |


ที่นี่เราสามารถเห็นได้ว่าแต่ละอินพุตมีความน่าจะเป็นเท่ากันที่จะเชื่อมโยงกับแต่ละเอาต์พุต


อย่างไรก็ตาม หากเรายกตัวอย่างของธุรกรรมที่มีหนึ่งอินพุตและสองเอาต์พุต มันจะมีลักษณะดังนี้:


| Input | Output 0 | Output 1 |
| ----- | -------- | -------- |
| 0     | 100%     | 100%     |

ในตัวอย่างนี้ เราสามารถเห็นได้ว่าความน่าจะเป็นของผลลัพธ์แต่ละอย่างที่มาจากอินพุต 0 คือ 100%


ยิ่งความน่าจะเป็นนี้ต่ำ ระดับความเป็นส่วนตัวก็ยิ่งสูงขึ้น


6. ข้อมูลชิ้นที่หกที่ถูกคำนวณคือจำนวนของลิงก์ที่เป็นเชิงกำหนด อัตราส่วนของลิงก์ที่เป็นเชิงกำหนดจะถูกจัดเตรียมไว้ด้วย ตัวชี้วัดนี้เน้นจำนวนของลิงก์ระหว่างอินพุตและเอาต์พุตของธุรกรรมที่กำหนดซึ่งมีความน่าจะเป็น 100% หมายความว่าพวกมันไม่สามารถปฏิเสธได้


อัตราส่วนบ่งชี้จำนวนลิงก์เชิงกำหนดในธุรกรรมเมื่อเทียบกับจำนวนลิงก์ทั้งหมด


ตัวอย่างเช่น การทำธุรกรรม Coinjoin Whirlpool ไม่มีลิงก์เชิงกำหนดระหว่างอินพุตและเอาต์พุต ตัวบ่งชี้จะเป็นศูนย์และอัตราส่วนจะเป็น 0%


อย่างไรก็ตาม สำหรับธุรกรรมที่สองที่ศึกษา (1 อินพุตและ 2 เอาต์พุต) ตัวบ่งชี้คือ 2 และอัตราส่วนคือ 100%

### เชื่อมต่อ Bisq.


Bisq เป็นแพลตฟอร์มแลกเปลี่ยนแบบเพียร์ทูเพียร์ที่ให้คุณซื้อและขายบิตคอยน์ ใช้งานร่วมกับซอฟต์แวร์เดสก์ท็อปที่ทำงานบน Tor และช่วยให้คุณแลกเปลี่ยนบิตคอยน์โดยไม่จำเป็นต้องเปิดเผยตัวตนของคุณ

Bisq รักษาความปลอดภัยในการแลกเปลี่ยนแบบเพียร์ทูเพียร์ผ่านระบบลายเซ็นหลายตัว 2/2 คุณสามารถใช้ซอฟต์แวร์นี้ร่วมกับโหนด RoninDojo ของคุณเองเพื่อเพิ่มประสิทธิภาพความเป็นส่วนตัวของการแลกเปลี่ยนของคุณและเชื่อถือข้อมูลจากบล็อกเชนของโหนดของคุณเอง


ในการดาวน์โหลดซอฟต์แวร์ Bisq ไปที่เว็บไซต์ทางการของพวกเขา: https://bisq.network/


ในการเริ่มต้นใช้งานซอฟต์แวร์ ฉันขอแนะนำให้อ่านหน้านี้: https://bisq.network/getting-started/


ในการดึงลิงก์การเชื่อมต่อจาก RoninDojo ของคุณ คุณจะต้องเชื่อมต่อกับ RoninCLI ผ่าน SSH จากนั้นไปที่เมนู:

กรุณาใส่รหัสผ่านของคุณ จากนั้นกดปุ่มเว้นวรรคเพื่อตรวจสอบช่อง:

คัดลอกที่อยู่ภายใต้ "Bitcoin Daemon"


คุณยังสามารถดึงที่อยู่ Bitcoind Tor V3 ของคุณจากอินเทอร์เฟซ RoninUI ได้โดยเพียงแค่คลิกที่ "Manage" ในกล่อง "Bitcoin Core" บน "Dashboard":

### คุณสมบัติอื่น ๆ


โหนด RoninDojo ของคุณยังรวมถึงคุณสมบัติพื้นฐานอื่นๆ คุณมีความสามารถในการสแกนข้อมูลเฉพาะเพื่อให้แน่ใจว่ามีการพิจารณาข้อมูลนั้นแล้ว


ตัวอย่างเช่น บางครั้งอาจเป็นไปได้ว่า wallet ของคุณที่เชื่อมต่อกับ RoninDojo ไม่พบบิตคอยน์ที่เป็นของคุณ ยอดคงเหลือเป็น 0 แม้ว่าคุณจะแน่ใจว่าคุณมีบิตคอยน์ใน wallet นี้ มีสาเหตุที่เป็นไปได้หลายประการที่ต้องพิจารณา รวมถึงข้อผิดพลาดในเส้นทางการอนุมาน และในบรรดาสาเหตุเหล่านั้น ยังเป็นไปได้ว่าโหนดของคุณไม่ได้สังเกตที่อยู่ของคุณ
