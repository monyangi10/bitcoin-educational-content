Hakurikijwe agaciro k’ama UTXO ari muri iyo nzira, iki kigereranyo giharura umubare w’uburyo ivyo bikoresho bishobora gufatanywa n’ivyo bishobora gusohoka. Mu yandi majambo, rigena umubare w’insobanuro zishobora kwemerwa igikorwa gishobora gukuramwo umuntu abona ivy’inyuma ariko aragisuzuma.

### Entropi:

Ikimenyetso ca kabiri giharurwa ni entropie y'ugucuruza, kigaragazwa na `Entropie`.


Mu bijanye n’ubuhinga bwo gukingira amakuru n’amakuru, entropi ni ingero y’ingero y’ukudakeka canke ukudashobora kumenya imbere y’igihe bifitaniye isano n’inkomoko y’amakuru canke inzira y’imburakimazi. Mu yandi majambo, entropie ni uburyo bwo gupima ingene amakuru agoye gutegekanya canke gutekereza ku bintu.


Mu bijanye n’isesengura ry’uruzitiro, entropi ni izina ry’ikigereranyo, rikomoka kuri entropi ya Shannon kandi [yavumbuwe na LaurentMT](https.


Iyo igikorwa c’ugucuruza kigaragaza umubare munini w’imigwi ishoboka, kenshi birabereye cane kwerekeza ku ntungamubiri yaco. Ico kigereranyo kiremesha gupima ubukene bw’ubumenyi bw’abahinga ku bijanye n’imiterere nyayo y’ugucuruza. Mu yandi majambo, uko entropie igenda irakura, ni ko igikorwa co kumenya inyigisho za Bitcoin hagati y’ibintu vyinjizwa n’ibintu bisohoka kirushiriza kugora abasesangura.


Mu bikorwa, entropi ihishura nimba, mu kubona kw’umuntu wo hanze, igikorwa gitanga insobanuro nyinshi zishoboka, zishingiye gusa ku rugero rw’ibintu vyinjizwa n’ibisohoka, ata kwihweza ibindi bigereranyo vyo hanze canke vyo mu mutima n’ubuhinga bwo kumenya ibintu. Entropie nini rero ni co kimwe n’ubuzima bwite bwiza bw’ugucuruza.


Entropie isobanurwa nk’umubare w’ibintu bishobora gukorana. Akira formule ikoreshwa:

```plaintext
E: the entropy of the transaction
C: the number of possible combinations for the transaction

E = log2(C)
```


Mu mibare, logarithme ya kabiri (logarithme ya base-2) ihuye n’igikorwa co guhinduranya 2. Mu yandi majambo, logarithme ya kabiri ya `x` ni yo `2` itegerezwa kuduzwako kugira ngo umuntu aronke `x`. Ico kimenyetso rero kigaragazwa mu bice.


Reka dufate akarorero ko kubara entropie y’ugucuruza kwa CoinJoin gutunganijwe hakurikijwe uburyo bwa Whirlpool 5x5, nk’uko twabibonye mbere, butanga umubare w’imigwi ishoboka ya `1.496`:

```plaintext
C = 1,496
E = log2(1,496)
E = 10.5469 bits
```

Gutyo, iyo nzira ya CoinJoin yerekana entropie y’ibice `10.5469`, ivyo bikaba bibonwa ko bishimishije cane. Uko ako gaciro karushiriza kuba kanini, ni ko n’insobanuro zitandukanye iyo nzira yemera, gutyo bigakomeza urugero rw’ubuzima bwite bwayo.

Ku bijanye n’ugucuruza 8x8 CoinJoin yerekana insobanuro `9.934.563`, entropie yoba:

```plaintext
C = 9,934,563
E = log2(9,934,563)
E = 23.244 bits
```


Reka dufate akandi karorero n'ugucuruza gusanzwe, gufise input imwe n'ibisohoka bibiri: [1b1b0c3f0883a99f1161c64da19471841ed12a1f78e77fab128c69a5f578ccce](https://GW -17.ikibanza/tx/1b1b0c3f0883a99f1161c64da19471841ed12a1f78e77fab128c69a5f578ccce) Mu bijanye n'iyi nzira, insobanuro yonyene ishoboka ni: `(Mu.0) > (Inyuma.0 ; Inyuma.1)`. Ku bw'ivyo, entropie yayo ishingwa kuri `0`:

```plaintext
C = 1
E = log2(1)
E = 0 bits
```


### Umwimbu:

Ikimenyetso ca gatatu gitangwa n'Ibarabara rya Boltzmann citwa `Ubushobozi bwa Wallet`. Ico kigereranyo gisuzuma ubushobozi bw’ugucuruza mu kugereranya n’ugucuruza kwiza gushobora kwiyumvirwa mu ntunganyo isa.


Ivyo bituma tuganira ku ciyumviro c’uburemere burengeye ubundi bwose, buhuye n’uburemere burengeye ubundi bwose umubumbe w’ubudandaji wihariye woshobora gushikako mu vyiyumviro. Ubushobozi bw’ugucuruza buraheza buharurwa mu guhangana n’iyo entropi nkuru n’intropi nyayo y’ugucuruza kwasuzumwe.


Iryo formule rikoreshwa ni iri:

```plaintext
ER: the actual entropy of the transaction expressed in bits
EM: the maximum possible entropy for a given transaction structure expressed in bits
Ef: the efficiency of the transaction in bits

Ef = ER - EM
```


Nk'akarorero, ku mibumbe ya Whirlpool 5x5 y'ubwoko bwa CoinJoin, entropie nini cane ishirwa kuri `10.5469`:

```plaintext
ER = 10.5469
EM = 10.5469
Ef = 10.5469 - 10.5469 = 0 bits
```


Iki kigereranyo naco kigaragazwa nk'ijanisha, formule yaco rero ni:

```plaintext
CR: the actual number of possible combinations
CM: the maximum number of possible combinations with the same structure
Ef: the efficiency expressed as a percentage

Ef = CR / CM
Ef = 1,496 / 1,496
Ef = 100%
```


Ubushobozi bwa `100%` rero bwerekana ko igikorwa co gucuruza gitera imbere cane ubushobozi bwaco bwo gukingira ubuzima bwite bivanye n’imiterere yaco.


### Uburemere bwa Entropi:

Ikimenyetso ca kane ni uburemere bw'uburemere, bugaragara ku gikoresho `Uburemere bw'uburemere`. Itanga iciyumviro ku entropie ijyanye n’inyungu canke isohoka ryose ry’ugucuruza. Ico kigereranyo kigaragaza ko ari ngirakamaro mu gusuzuma no kugereranya ubushobozi bw’ibikorwa vy’ubudandaji vy’ubunini butandukanye. Kugira ngo uyiharure, ushobora gusa kugabanya entropie yose y’ugucuruza n’umubare wose w’ibintu vyinjizwa n’ibisohoka birimwo:

```plaintext
ED: the entropy density expressed in bits
E: the entropy of the transaction expressed in bits
T: the total number of inputs and outputs in the transaction

ED = E / T
```


Reka dufate akarorero k’indege y’umuyagankuba 5x5 CoinJoin:

```plaintext
T = 5 + 5 = 10
E = 10.5469
ED = 10.5469 / 10 = 1.054 bits
```


Reka kandi tubaze uburemere bw'uburemere bw'umuriro ku Whirlpool 8x8 CoinJoin:

```plaintext
T = 8 + 8 = 16
E = 23.244
ED = 23.244 / 16 = 1.453 bits
```


### Igitigiri ca Boltzmann:

Amakuru ya gatanu atangwa n’Ibarabara rya Boltzmann ni imbonerahamwe y’uguhuza ibishoboka hagati y’ibintu vyinjizwa n’ibisohoka. Iyi mbonerahamwe yerekana, biciye ku manota ya Boltzmann, ubushobozi bw’ibintu vy’uko ikintu kinaka gifitaniye isano n’igisohoka.


Ni ingero rero y’ingero y’ibishoboka bishingiye ku bintu bishobora gutuma habaho isano hagati y’ivyo winjiza n’ivyo usohoka mu gucuruza, bishingiye ku rugero rw’umubare w’ibintu vyiza vyabaye muri ico kintu n’umubare wose w’ibintu bishobora kuba, mu rutonde rw’insobanuro.


Dufashe akarorero ka Whirlpool CoinJoin kandi, imbonerahamwe y’ibishoboka bishingiye ku bintu vyoshobora kwerekana amahirwe yo guhuza hagati y’ivyo umuntu yinjiza n’ivyo asohora, igatanga ingero y’ingero y’ukudasobanuka kw’amashirahamwe mu gucuruza:

| Input 3 | 34%      | 34%      | 34%      | 34%      | 34%      |

Aha, turashobora kubona neza ko ikintu cose cinjijwe gifise amahirwe angana yo gufatanywa n’igisohoka cose, ivyo bikaba bituma ubuzima bwite bw’ugucuruza butera imbere.

Guharura igiharuro ca Boltzmann birimwo kugabanya umubare w’insobanuro ikintu kinaka kibamwo n’umubare wose w’insobanuro ziriho. Gutyo, kugira ngo hamenyekane amanota ahuza inyungu No. 0 n’isohoka No. 3 (`insobanuro 512`), hakoreshwa uburyo bukurikira:

Interpretations (IN.0 > OUT.3) = 512
Total Interpretations = 1496
Score = 512 / 1496 = 34%

### Amahuza y'Ivyiyumviro:

Amakuru ya gatandatu atangwa ni umubare w’amahuza y’ivy’ubuhinga, yuzuzwa n’igipimo c’ayo mahuza. Ico kigereranyo kigaragaza ingene amasano ari menshi hagati y’ibintu vyinjizwa n’ibintu bisohoka mu bikorwa vy’ubudandaji vyasuzumwe ataco bimaze, n’ubushobozi bwa `100%`. Ku rundi ruhande, igipimo kiratanga iciyumviro ku buremere bw’ayo mahuza y’ivy’ubuhinga mu rwego rwose rw’amahuza y’ugucuruza.

Nk'akarorero, igikorwa ca Whirlpool c'ubwoko bwa CoinJoin nta mahuza y'ivy'imbere gifise, rero kigaragaza ikimenyetso n'igipimo ca `0%`. Ku rundi ruhande, mu bikorwa vyacu vya kabiri vyoroshe vyasuzumwe (n’injiza imwe n’isohoka bibiri), ikimenyetso gishirwa kuri `2` maze igipimo kigashika kuri `100%`. Gutyo, ikimenyetso c’ubusa kigaragaza ubuzima bwite bwiza cane kubera ukutagira amasano ataziguye kandi atagira aho ahengamiye hagati y’ibintu vyinjizwa n’ibisohoka.
