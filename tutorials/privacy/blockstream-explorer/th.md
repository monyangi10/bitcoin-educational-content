Blockstream Explorer เป็นโครงการที่อำนวยความสะดวกในการสำรวจธุรกรรมและสถานะทั่วโลกของโปรโตคอล Bitcoin รวมถึง [*sidechain*](https://planb.academy/en/resources/glossary/sidechain) Liquid ที่พัฒนาโดยบริษัท Blockstream



ริเริ่มในปี 2014 โดย Blockstream บริษัทที่ก่อตั้งโดย Adam Back, [blockstream.info](https://blockstream.info) explorer มีเป้าหมายเพื่อให้โครงสร้างพื้นฐานที่แข็งแกร่งสำหรับ Bitcoin, รับประกันการทำงานร่วมกันและการติดตามธุรกรรมระหว่างเลเยอร์ (on-chain และ Liquid), ในขณะที่เพิ่มความปลอดภัยและความเป็นส่วนตัวของผู้ใช้



ในบทแนะนำนี้ เราจะมาดูสิ่งที่ทำให้มันแตกต่าง บริการของมัน และวิธีที่มันเสนอการตรวจสอบการดำเนินงานและสถานะของชั้น on-chain และ Liquid ของ Bitcoin อย่างไร้รอยต่อ



## เริ่มต้นใช้งาน Blockstream explorer



### นำทางช่องหลัก



เมื่อคุณไปที่ Blockstream.info explorer บน "**Dashboard**" โปรโตคอล Bitcoin main chain จะถูกเลือกโดยค่าเริ่มต้น จากอินเทอร์เฟซนี้ คุณจะมีภาพรวมของ :





- ขนาดของสายหลัก: บล็อกที่ขุดเมื่อเร็ว ๆ นี้



![blocks](assets/fr/01.webp)



ส่วนนี้ให้ข้อมูลเกี่ยวกับบล็อกที่ถูกขุดล่าสุด, เวลาประทับ, จำนวนธุรกรรมที่รวมอยู่ในแต่ละบล็อก, ขนาดเป็นกิโลไบต์ (kB) และการวัดของแต่ละบล็อกในหน่วยน้ำหนัก (**WU** = *Weight Units*) การวัดครั้งสุดท้ายนี้เป็นที่น่าสนใจ เนื่องจากช่วยให้เราประเมินการเพิ่มประสิทธิภาพของบล็อกได้ โดยที่แต่ละบล็อกของเชนหลักถูกจำกัดไว้ที่ `4,000,000 WU` หรือ `4,000 kWU`





- ธุรกรรมล่าสุด



![transactions](assets/fr/02.webp)



ส่วนการทำธุรกรรมให้ข้อมูลเกี่ยวกับตัวระบุเฉพาะของการทำธุรกรรม มูลค่า bitcoin ที่เกี่ยวข้อง ขนาดในหน่วยไบต์เสมือน (vB) - ซึ่งแสดงถึงผลรวมของข้อมูลทั้งหมด (อินพุตและเอาต์พุต) - และอัตราค่าธรรมเนียมที่เกี่ยวข้อง ตัวอย่างเช่น การทำธุรกรรมที่มีขนาด `153 vB` ที่อัตรา `2 sat/vB` จะมีค่าธรรมเนียม `306 satoshis`



### การสำรวจของไหล



จากเมนู "**Blocks**" คุณสามารถติดตามประวัติของสายหลักทั้งหมดกลับไปยังบล็อกสุดท้ายที่ถูกขุดได้



![blocs](assets/fr/03.webp)



โดยการคลิกที่บล็อกเฉพาะ คุณสามารถรับรายละเอียดเพิ่มเติมเกี่ยวกับข้อมูลและธุรกรรมที่รวมอยู่ในนั้นได้ ตัวอย่างเช่น สำหรับบล็อก 919330: คุณจะเห็นแฮชของบล็อก คุณยังสามารถนำทางไปยังบล็อกก่อนหน้าได้ เนื่องจากแต่ละบล็อกที่ถูกขุด (ยกเว้น Genesis) จะเชื่อมโยงกับบล็อกก่อนหน้า โดยรักษาแฮชของบล็อกก่อนหน้าไว้



![metadata](assets/fr/04.webp)



โดยการคลิกที่ปุ่ม **"Details "** คุณสามารถรับข้อมูลเพิ่มเติมเกี่ยวกับบล็อกนี้ เช่น สถานะของมัน ซึ่งยืนยันว่ามันได้ถูกเพิ่มไปยังเชนหลักที่ถูกเก็บรักษาและเผยแพร่แล้ว คุณยังสามารถดูความยากที่บล็อกนี้ถูกขุด: ความยากนี้แสดงถึงพลังการคำนวณที่จำเป็นในการแก้ปัญหาการเข้ารหัสของ mining และจะถูกปรับทุกๆ 2016 บล็อก (ประมาณ 2 สัปดาห์)



![details](assets/fr/05.webp)



ด้านล่างส่วนรายละเอียดนี้ เราจะพบธุรกรรมทั้งหมดที่รวมอยู่ในบล็อกนี้



ธุรกรรมแรกในบล็อกเรียกว่า **ธุรกรรมคอยน์เบส** มันถูกใช้เพื่อจัดสรรรางวัล mining ของนักขุด (รวมถึงค่าธรรมเนียมทั้งหมดที่เกี่ยวข้องกับธุรกรรมที่รวมอยู่ในบล็อกและการให้บล็อก) บิตคอยน์ที่สร้างขึ้นโดยธุรกรรมนี้สามารถใช้จ่ายได้ก็ต่อเมื่อมีการขุดบล็อกติดต่อกันอีก 100 บล็อก กล่าวอีกนัยหนึ่ง เพื่อที่จะใช้มัน นักขุดจะต้องรอการผลิตบล็อก **919430** นี่เป็นที่รู้จักกันในชื่อ [*"ช่วงเวลาการเติบโต "*](https://planb.academy/fr/resources/glossary/maturity-period).



Coinbase เป็นธุรกรรมพิเศษ: เป็นธุรกรรมเดียวที่ไม่มีอินพุตจริง เนื่องจากไม่ได้ใช้จ่ายบิตคอยน์จากธุรกรรมก่อนหน้าใด ๆ




![coinbase](assets/fr/06.webp)



ธุรกรรมอื่นๆ ทั้งหมดแบ่งออกเป็นสองส่วน: อินพุตและเอาต์พุต



สำหรับการใช้บิตคอยน์เป็นอินพุตในธุรกรรมใหม่ ผู้เริ่มต้นธุรกรรมต้องพิสูจน์การครอบครองโดยการให้ลายเซ็นที่ตรงกับสคริปต์เฉพาะ บิตคอยน์แต่ละชิ้น (UTXO) มีสคริปต์ที่โดยทั่วไปต้องการลายเซ็นเฉพาะที่เฉพาะกุญแจส่วนตัวของผู้ถือครองเท่านั้นที่สามารถให้ได้ สคริปต์เหล่านี้คือ ***scriptSig*** (ใน ASM) เขียนใน Bitcoin Script และสามารถมีหลายประเภท ในตัวอย่างนี้ เราจะเห็นว่า UTXOs ที่ใช้เป็นประเภท P2SH ไปยังเอาต์พุตประเภท P2WPKH (*Pay-to-Witness-Public-Key-Hash*)



คุณสามารถติดตามประวัติของ UTXO เฉพาะโดยใช้ฮิวริสติกส์ เราขอเชิญคุณค้นพบฮิวริสติกส์ Bitcoin ที่แตกต่างกันและวิธีการเสริมสร้างความลับของธุรกรรมของคุณบน Bitcoin :



https://planb.academy/courses/65c138b0-4161-4958-bbe3-c12916bc959c

![trxs](assets/fr/07.webp)



ลองพิจารณาตัวอย่างของค่าใช้จ่ายขาออกของธุรกรรมนี้ โดยการคลิกที่ตัวระบุธุรกรรม เราจะถูกนำไปยังส่วน **ธุรกรรม** ในหน้ารายละเอียดธุรกรรม



![transaction](assets/fr/08.webp)



จากหน้านี้ คุณสามารถตรวจสอบได้ว่าธุรกรรมถูกบรรจุอยู่ในบล็อกใด ขึ้นอยู่กับประเภทของที่อยู่ที่ใช้ ธุรกรรมสามารถปรับปรุงข้อมูลของตนเอง (*virtual bytes*) และจ่ายค่าธรรมเนียมธุรกรรมน้อยลงได้ ตัวอย่างเช่น ธุรกรรมนี้ประหยัดค่าธรรมเนียมได้ 53% โดยใช้รูปแบบที่อยู่ SegWit Bech32 แบบเนทีฟที่เริ่มต้นด้วย `bc1q`



![trx_details](assets/fr/09.webp)



## Liquid layer



Liquid Network เป็น[*ไซด์เชน*](https://planb.academy/en/resources/glossary/sidechain)และโซลูชันโอเพ่นซอร์สระดับ 2 สำหรับโปรโตคอล Bitcoin โดยเฉพาะ มันช่วยให้การทำธุรกรรมบิตคอยน์เร็วขึ้นและเป็นความลับมากขึ้น



บน Blockstream.info explorer คลิกที่ปุ่ม **"Liquid"** เพื่อสลับไปยังเครือข่าย Liquid



![liquid](assets/fr/10.webp)



เมื่อคลิกที่หนึ่งในธุรกรรมที่เราต้องการติดตาม เราจะเห็นว่าจำนวนบิตคอยน์ถูกแทนที่ด้วยคำว่า "**Confidential**" บนเครือข่ายนี้ ธุรกรรมสามารถเป็นความลับได้ ดังนั้นเราจึงไม่สามารถเห็นจำนวนของแต่ละ UTXO ไม่ว่าจะเป็นการเข้าออกของธุรกรรม



![liquid_trx](assets/fr/11.webp)



อย่างไรก็ตาม เราสังเกตว่าหลักการและกลไกที่มีอยู่ในชั้นหลักของโปรโตคอล Bitcoin นั้นเหมือนกัน: สคริปต์ล็อกบิตคอยน์และการติดตาม UTXO



![liquid_details](assets/fr/12.webp)



Liquid Network ยังให้สินทรัพย์ดิจิทัลที่ไม่ใช่เงินฝากซึ่งสามารถใช้โดยองค์กรได้อีกด้วย ในเมนู **"Assets "** คุณจะพบรายการสินทรัพย์ที่ลงทะเบียนแล้ว จำนวนรวม และโดเมนที่เกี่ยวข้องกับสินทรัพย์เหล่านั้น



![assets](assets/fr/13.webp)



สำหรับแต่ละสินทรัพย์ คุณสามารถติดตามประวัติของการออกและการทำธุรกรรมการเผาไหม้ (การลบยอดรวมที่หมุนเวียน)



![assets_trxs](assets/fr/14.webp)




## ตัวเลือกเพิ่มเติม



Blockstream.info explorer ยังรวมถึงการแสดงภาพและการติดตามธุรกรรมบน Testnet, Bitcoin, on-chain และ Liquid Network ด้วยเช่นกัน



![testnet](assets/fr/15.webp)



เมื่อคุณเข้าสู่เครือข่าย Testnet คุณจะไม่ได้ใช้บิทคอยน์จริง แต่คุณจะมีคุณสมบัติทั้งหมดที่อธิบายไว้ข้างต้น



![liquid_testnet](assets/fr/16.webp)



เครือข่ายนี้มีความยาวของโซ่ที่แตกต่างกัน ซึ่งคุณสามารถเชื่อมต่อและทดสอบการทำงานของกลไก Bitcoin และ Liquid ได้





- ส่วน API ถูกจัดเตรียมไว้สำหรับผู้ที่ต้องการรวมฟังก์ชันบางอย่างของ Explorer เข้ากับแอปพลิเคชันของตนเอง ผ่านทาง API นี้คุณสามารถสอบถามข้อมูลเกี่ยวกับสายโซ่หลักของเลเยอร์ต่างๆ (on-chain และ Liquid) ติดตามธุรกรรม และค้นหาค่าธรรมเนียมเฉลี่ยสำหรับธุรกรรมในบล็อกได้ เป็นต้น



![api](assets/fr/17.webp)



คุณพร้อมแล้วที่จะใช้ศักยภาพเต็มที่ของ Blockstream Explorer สำหรับการสอบถามบล็อกเชนบนเลเยอร์ on-chain และ Liquid เราหวังว่าคุณจะพบว่าบทแนะนำนี้ให้ข้อมูลที่เป็นประโยชน์ และขอแนะนำบทแนะนำของเราเกี่ยวกับตัวสำรวจ Bitcoin อีกตัว:
