Ashigaru Terminal เป็นการปรับแต่งของทีม Ashigaru จาก Sparrow Server ซึ่งนำโปรโตคอล Whirlpool coinjoin มาใช้ ซอฟต์แวร์นี้เป็นการสานต่อจากงานที่เริ่มต้นโดย Samourai Wallet โดยเฉพาะในส่วนของ Whirlpool GUI ซึ่งยึดหลักการพื้นฐาน: การดูแลรักษาตนเองและการรักษาความลับ



ซอฟต์แวร์นี้เป็น fork ของเซิร์ฟเวอร์ Sparrow ที่ได้รับการปรับเปลี่ยนและเพิ่มประสิทธิภาพเพื่อการผสานรวมอย่างเต็มรูปแบบกับระบบนิเวศ Whirlpool ซึ่งเป็นโปรโตคอล ZeroLink coinjoin ที่พัฒนาขึ้นโดยทีม Samourai เดิมที



Ashigaru Terminal ทำงานจากอินเทอร์เฟซ TUI แบบมินิมอลและสามารถใช้งานได้บนคอมพิวเตอร์ส่วนบุคคลหรือบนเซิร์ฟเวอร์เฉพาะ มันช่วยให้คุณโต้ตอบโดยตรงกับ Whirlpool เพื่อเริ่มต้น "*Tx0*", จัดการบัญชี "*Deposit*", "*Premix*", "*Postmix*" และ "*Badbank*" และดำเนินการรีมิกซ์อัตโนมัติเพื่อเสริมความลับของส่วนต่างๆ ของคุณ



โดยสรุป Ashigaru Terminal จะมีประโยชน์อย่างยิ่งหากคุณต้องการสร้าง coinjoins โดยใช้ Whirlpool



ในบทแนะนำแรกนี้ ฉันจะพาคุณผ่านการติดตั้งและการใช้งานของ Ashigaru Terminal บทแนะนำที่สองซึ่งมีความก้าวหน้ามากขึ้นจะอุทิศให้กับการสร้าง coinjoins จริง



https://planb.academy/tutorials/privacy/on-chain/ashigaru-whirlpool-e566803d-ab3f-4d98-9136-5462009262ef

## 1. ติดตั้ง Ashigaru Terminal



ในการติดตั้ง Ashigaru Terminal คุณจะต้องใช้ Tor Browser เนื่องจากไบนารีจะถูกแจกจ่ายผ่านเครือข่าย Tor เท่านั้น หากคุณยังไม่ได้ทำ [ติดตั้งบนเครื่องของคุณ](https://www.torproject.org/download/)



### 1.1. ดาวน์โหลด Ashigaru Terminal



จาก Tor Browser ไปที่ [หน้าการเผยแพร่ของ Git repository ของพวกเขา](http://ashicodepbnpvslzsl2bz7l2pwrjvajgumgac423pp3y2deprbnzz7id.onion/Ashigaru/Ashigaru-Terminal/releases/) เพื่อดาวน์โหลดเวอร์ชันล่าสุดของ Ashigaru Terminal สำหรับระบบปฏิบัติการของคุณ



```txt

```

### 1.2. ตรวจสอบ Ashigaru Terminal



ก่อนที่จะรันซอฟต์แวร์บนอุปกรณ์ของคุณ คุณจำเป็นต้องตรวจสอบความถูกต้องและความสมบูรณ์ของมัน นี่เป็นขั้นตอนที่สำคัญ เพราะจะป้องกันไม่ให้คุณติดตั้งเวอร์ชันปลอมที่อาจทำให้บิตคอยน์ของคุณถูกคุกคามหรือทำให้อุปกรณ์ของคุณติดไวรัส



เปิดแท็บเบราว์เซอร์ใหม่และไปที่ [เครื่องมือตรวจสอบ Keybase](https://keybase.io/verify) วางเนื้อหาของไฟล์ `.txt` ที่คุณเพิ่งดาวน์โหลดลงในช่องที่ให้ไว้ จากนั้นคลิกที่ปุ่ม `Verify`



![Image](assets/fr/02.webp)



เพื่อกระจายแหล่งที่มาของการตรวจสอบของคุณ คุณยังสามารถเปรียบเทียบข้อความกับข้อความที่มีอยู่บนเว็บไซต์ clearnet [ashigaru.rs](https://ashigaru.rs/download/) ในส่วน `/download` ได้อีกด้วย



![Image](assets/fr/03.webp)



หากลายเซ็นถูกต้อง Keybase จะแสดงข้อความยืนยันว่าไฟล์ได้รับการลงนามโดยนักพัฒนาของ Ashigaru แล้ว



![Image](assets/fr/04.webp)



คุณยังสามารถคลิกที่โปรไฟล์ `ashigarudev` ที่แสดงโดย Keybase และตรวจสอบว่าลายนิ้วมือคีย์ของพวกเขาตรงกันอย่างแม่นยำ: `A138 06B1 FA2A 676B`



![Image](assets/fr/05.webp)



หากเกิดข้อผิดพลาดในขั้นตอนนี้ แสดงว่าลายเซ็นไม่ถูกต้อง ในกรณีนี้ **อย่าติดตั้งซอฟต์แวร์ที่ดาวน์โหลดมา** ให้เริ่มต้นใหม่ตั้งแต่ต้น หรือขอความช่วยเหลือจากชุมชนก่อนดำเนินการต่อ



Keybase ได้ให้แฮชที่ผ่านการตรวจสอบแล้วของแอปพลิเคชันแก่คุณ ตอนนี้เราจะตรวจสอบว่าแฮชของไฟล์ `.deb`, `.zip` หรือ `.dmg` ที่คุณดาวน์โหลดตรงกับแฮชที่ได้รับการยืนยันบน Keybase หรือไม่ ในการทำเช่นนี้ ให้ไปที่ [HASH FILE ONLINE](https://hash-file.online/)



คลิกที่ปุ่ม `BROWSE...` และเลือกไฟล์ `.deb`, `.zip` หรือ `.dmg` ที่ดาวน์โหลดในขั้นตอนที่ 1.1 จากนั้นเลือกฟังก์ชันแฮช `SHA-256` และคลิกที่ `CALCULATE HASH` เพื่อ generate แฮชสำหรับไฟล์ของคุณ



![Image](assets/fr/06.webp)



ไซต์จะแสดงแฮชของซอฟต์แวร์ จากนั้นเปรียบเทียบกับแฮชที่คุณตรวจสอบบน Keybase.io หากทั้งสองตรงกันอย่างสมบูรณ์ การตรวจสอบความถูกต้องและความสมบูรณ์ถือว่าสำเร็จ คุณสามารถใช้ซอฟต์แวร์ได้แล้ว

### 1.3 เปิดตัว Ashigaru Terminal





- Debian / Ubuntu



ในการติดตั้งซอฟต์แวร์ ให้รันคำสั่ง :



```bash
cd ~/Downloads
sudo apt install ./ashigaru_terminal_v1.0.0_amd64.deb
```



แก้ไขคำสั่งตามเวอร์ชันที่ดาวน์โหลดแล้ว



จากนั้นตรวจสอบการติดตั้ง:



```bash
/opt/ashigaru-terminal/bin/Ashigaru-terminal --version
```



จากนั้นเปิดซอฟต์แวร์:



```bash
/opt/ashigaru-terminal/bin/Ashigaru-terminal
```





- วินโดวส์**



คลิกขวาที่ไฟล์ `.zip` ที่คุณได้ดาวน์โหลดและตรวจสอบแล้ว จากนั้นเลือก `Extract All...` เพื่อแยกเนื้อหาของมันออกมา



เมื่อการแยกไฟล์เสร็จสิ้น ให้ดับเบิลคลิกที่ไฟล์ `Ashigaru-terminal.exe` เพื่อเปิดซอฟต์แวร์

## 2. เริ่มต้นใช้งานกับ Ashigaru Terminal



Ashigaru Terminal เป็นโปรแกรม TUI (*Text-based User Interface*) ซึ่งเป็นอินเทอร์เฟซแบบมินิมอลที่ทำงานโดยตรงในเทอร์มินัล คุณโต้ตอบกับมันโดยใช้เมนูและคีย์ลัด แต่ไม่มีสภาพแวดล้อมกราฟิกแบบคลาสสิกจริงๆ



![Image](assets/fr/09.webp)



ใช้งานง่าย: ใช้ปุ่มลูกศรบนแป้นพิมพ์ของคุณเพื่อเลื่อนผ่านเมนู และกดปุ่ม `Enter` เพื่อยืนยันการกระทำหรือยืนยันการเลือก



## 3. การเชื่อมต่อโหนดของคุณกับ Ashigaru Terminal



โดยค่าเริ่มต้น Ashigaru Terminal จะเชื่อมต่อกับเซิร์ฟเวอร์สาธารณะ Electrum ซึ่งแน่นอนว่าส่งผลให้เกิดความเสี่ยงในแง่ของความลับและอธิปไตย ดังนั้นเราจะกำหนดค่าให้เชื่อมต่อโดยตรงกับ Electrum Server ของคุณเอง



ในการทำเช่นนี้ ให้เปิดเมนู `Preferences > Server`



![Image](assets/fr/10.webp)



คลิกที่ปุ่ม `< Edit >`



![Image](assets/fr/11.webp)



เลือก `Private Electrum Server` จากนั้นคลิก `<Continue>`.



![Image](assets/fr/12.webp)



ป้อน URL และพอร์ตของเซิร์ฟเวอร์ของคุณ คุณสามารถระบุที่อยู่ Tor ใน `.onion` ได้ จากนั้นคลิกที่ `< Test >` เพื่อตรวจสอบการเชื่อมต่อ



![Image](assets/fr/13.webp)



หากการเชื่อมต่อสำเร็จ ข้อความ `Success` จะปรากฏขึ้น พร้อมกับรายละเอียดของเซิร์ฟเวอร์ของคุณ



![Image](assets/fr/14.webp)



หากคุณยังไม่มีโหนด Bitcoin ฉันขอแนะนำให้คุณเข้าร่วมหลักสูตรนี้:



https://planb.academy/courses/3cd9cb94-82e8-417a-9c5a-02afc2589426

*ในกรณีของฉัน สำหรับบทแนะนำนี้ ฉันจะตัดการเชื่อมต่อจากเซิร์ฟเวอร์ Electrs ของฉันเพราะฉันกำลังทำงานบน testnet อย่างไรก็ตาม การดำเนินการยังคงเหมือนกันทุกประการบน mainnet*



## 4. สร้าง wallet บน Ashigaru Terminal



ตอนนี้ที่ซอฟต์แวร์ถูกตั้งค่าอย่างถูกต้องแล้ว เราสามารถเพิ่ม Bitcoin wallet ได้



คุณมีสองตัวเลือก:




- คุณสามารถสร้าง wallet ใหม่ตั้งแต่เริ่มต้นและใช้มันเฉพาะกับ Ashigaru Terminal ในกรณีนี้ คุณจะต้องเปิดซอฟต์แวร์นี้ทุกครั้งที่คุณต้องการโต้ตอบกับ wallet ของคุณ;
- นอกจากนี้ คุณสามารถนำเข้า Ashigaru wallet ที่มีอยู่ของคุณโดยตรงจากโทรศัพท์ของคุณเข้าสู่ Ashigaru Terminal ข้อเสียของวิธีนี้คือมันลดความปลอดภัยของการตั้งค่าของคุณลงเล็กน้อย เนื่องจาก wallet ของคุณจะถูกเปิดเผยต่อสภาพแวดล้อมที่มีความเสี่ยงสองแห่งแทนที่จะเป็นหนึ่ง อย่างไรก็ตาม มันมีข้อดีที่สามารถปล่อยให้ Ashigaru Terminal ทำงานอย่างต่อเนื่องเพื่อผสมเหรียญของคุณ ในขณะที่ยังสามารถใช้จ่ายเหรียญเหล่านั้นจากระยะไกลผ่านแอปมือถือ Ashigaru



ในบทแนะนำนี้ เราจะเลือกใช้วิธีที่สอง อย่างไรก็ตาม หากคุณต้องการสร้าง wallet ใหม่ทั้งหมด ขั้นตอนจะยังคงเหมือนเดิม: ความแตกต่างเพียงอย่างเดียวจะอยู่ในระหว่างการสร้าง ซึ่งคุณจะต้องบันทึกวลีช่วยจำใหม่ของคุณและ passphrase ใหม่ของคุณ



โปรดทราบว่า Ashigaru Terminal ไม่อนุญาตให้คุณใช้จ่ายบิตคอยน์ของคุณโดยตรง คุณสามารถซิงโครไนซ์ wallet เดียวกันบน Ashigaru Terminal และแอป Ashigaru (ซึ่งฉันจะทำในบทแนะนำนี้) หรือบน Sparrow Wallet



หากคุณยังไม่มี wallet ในแอปพลิเคชัน Ashigaru คุณสามารถทำตามบทแนะนำเฉพาะทาง:



https://planb.academy/tutorials/wallet/mobile/ashigaru-9f903b55-2e55-4b06-9627-80f8e178158f

- ธุรกรรม": ช่วยให้คุณสามารถตรวจสอบประวัติของธุรกรรมที่เชื่อมโยงกับบัญชีนี้ได้ ในกรณีของฉัน บางรายการได้ปรากฏขึ้นแล้ว เนื่องจากฉันได้ทำบางรายการด้วยแอปพลิเคชัน Ashigaru บน wallet นี้



![Image](assets/fr/23.webp)





- รับ`: สร้างที่อยู่ใบเสร็จใหม่ที่ว่างเปล่าสำหรับวาง satss ในบัญชีเงินฝาก



![Image](assets/fr/24.webp)





- ที่อยู่`: แสดงรายการที่อยู่ทั้งหมดของคุณ ไม่ว่าจะเป็นของเชนภายในหรือภายนอกบัญชีของคุณ



![Image](assets/fr/25.webp)





- `UTXOs`: แสดงรายการ UTXOs ที่มีอยู่ทั้งหมดของคุณ



![Image](assets/fr/26.webp)





- `Settings`: ให้สิทธิ์การเข้าถึง wallet *descriptor* ของคุณ คุณยังสามารถตรวจสอบ seed ของคุณ ปรับ *Gap Limit* หรือเปลี่ยนวันที่สร้างของ wallet ของคุณได้



![Image](assets/fr/27.webp)
