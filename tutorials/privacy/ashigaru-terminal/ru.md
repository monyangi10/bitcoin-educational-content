Ashigaru Terminal - это адаптация командой Ashigaru сервера Sparrow, реализующая протокол коинджойнта Whirlpool. Это программное обеспечение является продолжением работы, начатой Samourai Wallet, в частности, над графическим интерфейсом Whirlpool, основополагающими принципами которого являются: самосохранение и сохранение конфиденциальности.



Это программное обеспечение представляет собой fork сервера Sparrow, модифицированный и оптимизированный для полной интеграции с экосистемой Whirlpool, протоколом объединения монет ZeroLink, изначально разработанным командами Samourai.



Ashigaru Terminal работает с минималистичным интерфейсом TUI и может быть развернут на персональном компьютере или на выделенном сервере. Он позволяет напрямую взаимодействовать с Whirlpool для инициирования "*Tx0*", управления счетами "*Deposit*", "*Premix*", "*Postmix*" и "*Badbank*", а также выполнять автоматические ремиксы для усиления конфиденциальности ваших частей.



Одним словом, Ashigaru Terminal будет особенно полезен, если вы хотите создавать коинджоины с помощью Whirlpool.



В этом первом уроке я расскажу вам об установке и работе Ashigaru Terminal. Второе, более продвинутое руководство будет посвящено собственно созданию коинджоинов.

## 1. Установите терминал Ашигару



Для установки Ashigaru Terminal вам понадобится Tor Browser, так как двоичные файлы распространяются только через сеть Tor. Если вы еще не сделали этого, [установите его на свою машину](https://www.torproject.org/download/).



### 1.1. скачать Ashigaru Terminal



Из Tor Browser перейдите на [страницу релизов их Git-репозитория](http://ashicodepbnpvslzsl2bz7l2pwrjvajgumgac423pp3y2deprbnzz7id.onion/Ashigaru/Ashigaru-Terminal/releases/), чтобы загрузить последнюю версию Ashigaru Terminal для вашей операционной системы.



```txt
ashicodepbnpvslzsl2bz7l2pwrjvajgumgac423pp3y2deprbnzz7id.onion/Ashigaru/Ashigaru-Terminal/releases/
```



![Image](assets/fr/01.webp)



Загрузите следующие 2 файла для вашей операционной системы:




- Двоичный файл (`ashigaru_terminal_v1.0.0_amd64.deb` для Debian/Ubuntu, `.dmg` для macOS или `.zip` для Windows) ;
- Файл с подписанными хэшами: `ashigaru_terminal_v1.0.0_signed_hashes.txt`.



### 1.2. Проверьте терминал Ашигару



Прежде чем запускать программу на своем устройстве, необходимо проверить ее подлинность и целостность. Это очень важный шаг, поскольку он позволяет избежать установки мошеннической версии, которая может скомпрометировать ваши биткоины или заразить ваш компьютер.



Откройте новую вкладку браузера и перейдите на страницу [Keybase verification tool](https://keybase.io/verify). Вставьте содержимое только что скачанного файла `.txt` в соответствующее поле, затем нажмите кнопку `Проверить`.



![Image](assets/fr/02.webp)



Чтобы разнообразить источники проверки, вы также можете сравнить это сообщение с тем, которое доступно на сайте clearnet [ashigaru.rs](https://ashigaru.rs/download/), в разделе `/download`.



![Image](assets/fr/03.webp)



Если подпись действительна, Keybase выведет сообщение, подтверждающее, что файл был подписан разработчиками Ashigaru.



![Image](assets/fr/04.webp)



Вы также можете щелкнуть по профилю `ashigarudev`, отображаемому Keybase, и проверить, что отпечаток его ключа точно совпадает: `A138 06B1 FA2A 676B`.



![Image](assets/fr/05.webp)



Если на этом этапе появляется ошибка, значит, подпись недействительна. В этом случае **не устанавливайте загруженное программное обеспечение**. Начните с самого начала или обратитесь за помощью к сообществу, прежде чем продолжить.



Keybase предоставил вам подтвержденный хэш приложения. Теперь проверим, совпадает ли хэш загруженного вами файла `.deb`, `.zip` или `.dmg` с хэшем, подтвержденным на Keybase. Для этого перейдите по ссылке [HASH FILE ONLINE](https://hash-file.online/).



Нажмите на кнопку `BROWSE...` и выберите файл `.deb`, `.zip` или `.dmg`, загруженный в шаге 1.1. Затем выберите хэш-функцию `SHA-256` и нажмите на кнопку `CALCULATE HASH`, чтобы получить generate хэш для вашего файла.



![Image](assets/fr/06.webp)



На сайте будет показан хэш программы. Сравните его с хэшем, который вы проверили на Keybase.io. Если они совпадают, значит, проверка подлинности и целостности прошла успешно. После этого вы можете использовать программное обеспечение.



![Image](assets/fr/07.webp)



### 1.3 Запуск терминала Ашигару





- Debian / Ubuntu



Чтобы установить программное обеспечение, выполните команду :



```bash
cd ~/Downloads
sudo apt install ./ashigaru_terminal_v1.0.0_amd64.deb
```



Измените порядок в соответствии с загруженной версией.



Затем проверьте установку:



```bash
/opt/ashigaru-terminal/bin/Ashigaru-terminal --version
```



Затем запустите программу:



```bash
/opt/ashigaru-terminal/bin/Ashigaru-terminal
```





- Windows**



Щелкните правой кнопкой мыши на файле `.zip`, который вы скачали и проверили, затем выберите `Extract All...`, чтобы извлечь его содержимое.



После извлечения дважды щелкните по файлу `Ashigaru-terminal.exe`, чтобы запустить программу.



![Image](assets/fr/08.webp)



## 2. Начало работы с терминалом Ashigaru



Ashigaru Terminal - это программа TUI (*Text-based User Interface*), то есть минималистичный интерфейс, работающий непосредственно в терминале. Вы взаимодействуете с ним с помощью меню и сочетаний клавиш, но без какого-либо классического графического окружения.



![Image](assets/fr/09.webp)



Он прост в использовании: используйте клавиши со стрелками на клавиатуре для перемещения по меню и нажимайте клавишу `Enter` для подтверждения действия или выбора.



## 3. Подключение вашего узла к терминалу Ашигару



По умолчанию Ashigaru Terminal подключается к публичному серверу Electrum. Это, безусловно, представляет риск с точки зрения конфиденциальности и суверенитета. Поэтому мы настроим его на прямое подключение к вашему собственному Electrum Server.



Для этого откройте меню `Параметры > Сервер`.



![Image](assets/fr/10.webp)



Нажмите на кнопку `< Редактировать >`.



![Image](assets/fr/11.webp)



Выберите `Private Electrum Server`, затем нажмите `<Продолжить>`.



![Image](assets/fr/12.webp)



Введите URL и порт вашего сервера. Вы можете указать Tor-адрес в `.onion`. Затем нажмите на `< Test >`, чтобы проверить соединение.



![Image](assets/fr/13.webp)



Если подключение прошло успешно, появится сообщение `Success`, а также сведения о вашем сервере.



![Image](assets/fr/14.webp)



Если у вас еще нет узла Bitcoin, я рекомендую вам пройти этот курс:



https://planb.academy/courses/3cd9cb94-82e8-417a-9c5a-02afc2589426

*В моем случае, для этого руководства, я собираюсь отключиться от моего сервера Electrs, потому что я работаю в testnet. Однако на mainnet работа остается строго идентичной.*



## 4. Создайте портфолио на Ashigaru Terminal



Теперь, когда программное обеспечение правильно настроено, мы можем добавить портфель Bitcoin.



У вас есть два варианта:




- Вы можете создать новый wallet с нуля и использовать его исключительно в Ashigaru Terminal. В этом случае вам придется открывать эту программу каждый раз, когда вы захотите взаимодействовать с wallet;
- Кроме того, вы можете импортировать существующий Ashigaru wallet прямо с телефона в Ashigaru Terminal. Недостатком этого метода является то, что он несколько снижает безопасность вашей установки, так как ваш wallet подвергается двум рискованным средам вместо одной. С другой стороны, это преимущество заключается в том, что вы можете оставить Ashigaru Terminal работать постоянно, чтобы перемешивать ваши монеты, в то время как вы можете тратить их удаленно через мобильное приложение Ashigaru.



В этом уроке мы выберем второй способ. Однако если вы предпочитаете создать совершенно новый портфель, процедура останется той же: единственное отличие будет в процессе создания, когда вам нужно будет сохранить новую мнемоническую фразу и новый passphrase.



Обратите также внимание, что Ashigaru Terminal не позволяет напрямую тратить ваши биткойны. Вы можете синхронизировать один и тот же кошелёк в Ashigaru Terminal и приложении Ashigaru (что я сделаю в этом уроке) или в Sparrow Wallet.



Если у вас еще нет wallet в приложении Ashigaru, вы можете следовать специальному руководству:



https://planb.academy/tutorials/wallet/mobile/ashigaru-9f903b55-2e55-4b06-9627-80f8e178158f

Перейдите в меню `Кошельки`.



![Image](assets/fr/15.webp)



Затем выберите `Создать/восстановить wallet...`. Опция `Открыть Wallet...` позволяет получить доступ к портфолио, уже сохраненному в Ashigaru Terminal, позднее.



![Image](assets/fr/16.webp)



Дайте своему портфолио название.



![Image](assets/fr/17.webp)



Затем выберите тип портфеля `Hot Wallet`.






![Image](assets/fr/18.webp)



Именно на этом этапе процедура отличается в зависимости от вашего первоначального выбора:




- Если вы хотите создать новый портфель с нуля, нажмите на `<Generate New Wallet>`, определите passphrase BIP39, затем аккуратно сохраните мнемоническую фразу и passphrase на физическом носителе;



https://planb.academy/tutorials/wallet/backup/backup-mnemonic-22c0ddfa-fb9f-4e3a-96f9-46e2a7954270



- Если вы хотите использовать тот же wallet, что и в приложении Ashigaru, введите 12 слов вашей мнемонической фразы и ваш passphrase BIP39 прямо в соответствующие поля. Пишите слова в нижнем регистре, целиком, по порядку, через пробел, без цифр и лишних символов.



После завершения этого шага нажмите `< Далее >`.



*Примечание*: Если вы не можете нажать на эту кнопку, значит, ваша мнемоническая фраза недействительна. Внимательно проверьте, что ни одно из слов не является неверным или неправильно написанным.



![Image](assets/fr/19.webp)



Затем вам нужно будет установить пароль. Он будет использоваться для разблокировки вашего терминала Ashigaru wallet и защиты его от несанкционированного физического доступа. Он не участвует в криптографическом выводе ваших ключей: другими словами, даже без этого пароля любой человек с вашей мнемонической фразой и passphrase сможет восстановить ваш wallet и получить доступ к вашим биткоинам.



Выберите длинный, сложный, случайный пароль. Храните копию в надежном месте: в идеале на физическом носителе или в безопасном менеджере паролей.



Нажмите `< OK >`, когда закончите.



![Image](assets/fr/20.webp)



## 5. Использование портфеля



Затем вы можете выбрать, к какому счету обращаться. На данный момент мы не инициировали никаких миксов, поэтому будем обращаться к счету `Deposit`.



![Image](assets/fr/21.webp)



Управление идентично Sparrow, так как Ashigaru Terminal - это fork сервера Sparrow. Вы найдете те же самые меню:



![Image](assets/fr/22.webp)





- транзакции": позволяет просмотреть историю транзакций, связанных с этим аккаунтом. В моем случае некоторые из них уже появились, так как я совершил несколько с помощью приложения Ashigaru на этом же wallet.



![Image](assets/fr/23.webp)





- receive`: генерирует новый, пустой адрес квитанции для размещения сатов на депозитном счете.



![Image](assets/fr/24.webp)





- адреса`: отображает список всех ваших адресов, независимо от того, принадлежат ли они к внутренней или внешней цепочке вашего аккаунта.



![Image](assets/fr/25.webp)





- `UTXOs`: список всех доступных вам UTXO.



![Image](assets/fr/26.webp)





- `Настройки`: предоставляет доступ к *дескриптору* вашего портфеля. Вы также можете ознакомиться со своим seed, настроить *лимит разрыва* или изменить дату создания портфеля.



![Image](assets/fr/27.webp)
