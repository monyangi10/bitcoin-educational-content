ในบทแนะนำนี้ คุณจะค้นพบทุกสิ่งที่คุณจำเป็นต้องรู้เกี่ยวกับการติดฉลาก UTXOs ใน Bitcoin wallet ของคุณและเกี่ยวกับการควบคุมเหรียญ เราเริ่มต้นด้วยส่วนทฤษฎีเพื่อทำความเข้าใจแนวคิดเหล่านี้อย่างเต็มที่ ก่อนที่จะย้ายไปยังส่วนปฏิบัติที่เราสำรวจวิธีการใช้ฉลากในซอฟต์แวร์หลัก Bitcoin wallet อย่างเป็นรูปธรรม


## การติดฉลาก UTXO คืออะไร?

"Labelling" เป็นเทคนิคที่เกี่ยวข้องกับการเชื่อมโยงคำอธิบายหรือป้ายกำกับกับ UTXO เฉพาะภายใน Bitcoin wallet คำอธิบายเหล่านี้จะถูกเก็บไว้ในเครื่องโดยซอฟต์แวร์ wallet และจะไม่ถูกส่งผ่านเครือข่าย Bitcoin ดังนั้น Labelling จึงเป็นเครื่องมือสำหรับการจัดการส่วนบุคคล


ตัวอย่างเช่น หากฉันได้รับ UTXO จากการทำธุรกรรม P2P ผ่าน Bisq กับ Charles ฉันสามารถกำหนดป้ายกำกับว่า `Bisq P2P Purchase Charles`


การติดฉลากช่วยให้สามารถจดจำแหล่งที่มาหรือจุดหมายปลายทางที่ตั้งใจไว้ของ UTXO ซึ่งช่วยให้การจัดการกองทุนง่ายขึ้นและเพิ่มประสิทธิภาพความเป็นส่วนตัวของผู้ใช้ การติดฉลากมีความเกี่ยวข้องมากยิ่งขึ้นเมื่อรวมกับฟังก์ชัน "การควบคุมเหรียญ" การควบคุม Coin เป็นตัวเลือกที่มีอยู่ในกระเป๋าเงิน Bitcoin ที่ดี ซึ่งช่วยให้ผู้ใช้สามารถเลือกด้วยตนเองว่า UTXOs ใดจะถูกใช้เป็นอินพุตเมื่อสร้างธุรกรรม


การใช้ wallet พร้อมการควบคุมเหรียญ ร่วมกับการติดฉลาก UTXO ช่วยให้ผู้ใช้สามารถแยกแยะและเลือก UTXOs สำหรับการทำธุรกรรมของพวกเขาได้อย่างแม่นยำ จึงหลีกเลี่ยงการรวม UTXOs จากแหล่งที่มาต่างๆ การปฏิบัตินี้ช่วยลดความเสี่ยงที่เกี่ยวข้องกับ Common Input Ownership Heuristic (CIOH) ซึ่งแนะนำว่ามีการเป็นเจ้าของร่วมกันของอินพุตของการทำธุรกรรม ซึ่งอาจทำให้ความเป็นส่วนตัวของผู้ใช้ถูกประนีประนอมได้


กลับไปที่ตัวอย่างของ UTXO ที่ไม่มี KYC ของฉันจาก Bisq; ฉันต้องการหลีกเลี่ยงการรวมมันกับ UTXO ที่มาจากแพลตฟอร์มแลกเปลี่ยนที่มีการควบคุมซึ่งรู้จักตัวตนของฉัน โดยการวางป้ายกำกับที่แตกต่างกันบน UTXO ที่ไม่มี KYC และบน UTXO ที่มี KYC ฉันจะสามารถระบุได้ง่ายว่า UTXO ใดที่จะใช้เป็นอินพุตเพื่อตอบสนองการใช้จ่าย โดยใช้ฟังก์ชันการควบคุมเหรียญ


## วิธีการติดฉลาก UTXO ของคุณอย่างถูกต้อง?

ไม่มีวิธีการติดฉลาก UTXOs ที่เป็นสากลซึ่งเหมาะกับทุกคน ขึ้นอยู่กับคุณในการกำหนดระบบการติดฉลากเพื่อให้คุณสามารถหาทางรอบ ๆ wallet ของคุณได้อย่างง่ายดาย

เกณฑ์สำคัญในการติดฉลากคือแหล่งที่มาของ UTXO คุณควรระบุเพียงว่าการมาถึงของเหรียญนี้ใน wallet ของคุณเป็นอย่างไร มันมาจากแพลตฟอร์มแลกเปลี่ยนหรือไม่? การชำระบิลโดยลูกค้าหรือไม่? การแลกเปลี่ยนแบบเพียร์ทูเพียร์หรือไม่? หรือมันเป็นเงินทอนจากการซื้อ? ดังนั้น คุณสามารถระบุได้ว่า:


- `การถอนเงิน Exchange.com`;
- `การชำระเงิน ลูกค้า David`;
- `P2P ซื้อ Charles`;
- `เปลี่ยนจากการซื้อโซฟา`

![labelling](assets/en/1.webp)

ในการปรับปรุงการจัดการ UTXO ของคุณและปฏิบัติตามกลยุทธ์การแยกกองทุนภายใน wallet ของคุณ คุณสามารถเพิ่มตัวบ่งชี้เพิ่มเติมในป้ายกำกับของคุณเพื่อสะท้อนการแยกเหล่านี้ได้ หาก wallet ของคุณมีสองประเภทของ UTXOs ที่คุณไม่ต้องการผสมกัน คุณสามารถรวมเครื่องหมายในป้ายกำกับของคุณเพื่อแยกแยะกลุ่มเหล่านี้อย่างชัดเจน


เครื่องหมายแยกเหล่านี้จะขึ้นอยู่กับเกณฑ์ของคุณเอง เช่น ความแตกต่างระหว่าง KYC UTXO (รู้จักตัวตนของคุณ) และ no-KYC (ไม่ระบุตัวตน) หรือระหว่างกองทุนมืออาชีพและส่วนบุคคล โดยใช้ตัวอย่างป้ายที่กล่าวถึงก่อนหน้านี้ สิ่งนี้สามารถแปลได้ว่า:


- `KYC - Withdrawal Exchange.com`;
- `KYC - ลูกค้าการชำระเงิน David`;
- `ไม่มี KYC - P2P ซื้อ Charles`;
- `ไม่มีการยืนยันตัวตน - เปลี่ยนจากการซื้อโซฟา`

![labelling](assets/en/2.webp)

ในกรณีใด ๆ โปรดจำไว้ว่าการติดฉลากที่ดีคือการติดฉลากที่คุณจะสามารถเข้าใจได้เมื่อคุณต้องการ หาก Bitcoin wallet ของคุณมีวัตถุประสงค์หลักเพื่อการออม อาจเป็นไปได้ว่าฉลากจะมีประโยชน์กับคุณในอีกหลายทศวรรษ ดังนั้นให้แน่ใจว่าฉลากนั้นชัดเจน แม่นยำ และครอบคลุม


นอกจากนี้ยังแนะนำให้คงการติดฉลากของเหรียญตลอดการทำธุรกรรม ตัวอย่างเช่น ในระหว่างการรวม UTXO แบบไม่ต้องใช้ KYC ให้แน่ใจว่าทำเครื่องหมาย UTXO ที่ได้ไม่เพียงแค่เป็น `consolidation` แต่เฉพาะเจาะจงว่าเป็น `no-KYC consolidation` เพื่อรักษาร่องรอยที่ชัดเจนของแหล่งที่มาของเหรียญ


ในที่สุดแล้ว ไม่จำเป็นต้องใส่วันที่บนฉลาก ซอฟต์แวร์ wallet ส่วนใหญ่จะแสดงวันที่ทำธุรกรรมอยู่แล้ว และสามารถดึงข้อมูลนี้ได้เสมอจากบล็อกเอ็กซ์พลอเรอร์โดยใช้ TXID ของมัน


## บทแนะนำ: การติดฉลากบน Specter Desktop


เชื่อมต่อและเปิด wallet ของคุณบน Specter Desktop จากนั้นเลือกแท็บ `Addresses`


![labelling](assets/notext/3.webp)

ที่นี่ คุณจะเห็นรายการที่อยู่ทั้งหมดของคุณ รวมถึงบิตคอยน์ใด ๆ ที่ถูกล็อกไว้ในนั้น โดยค่าเริ่มต้น ที่อยู่จะถูกระบุโดยดัชนีภายใต้คอลัมน์ `Label` หากต้องการเปลี่ยนป้ายกำกับ เพียงคลิกที่มัน ป้อนป้ายกำกับที่ต้องการ แล้วคลิกที่ไอคอนสีน้ำเงินเพื่อยืนยัน

![labelling](assets/notext/4.webp)


ป้ายกำกับของคุณจะปรากฏในรายการที่อยู่ของคุณ


![labelling](assets/notext/5.webp)


คุณยังสามารถกำหนดป้ายกำกับล่วงหน้า เมื่อคุณแชร์ที่อยู่รับของคุณกับผู้ส่งได้อีกด้วย ในการทำเช่นนี้ ให้เข้าถึงแท็บ `Receive` และจดบันทึกป้ายกำกับของคุณในช่องที่กำหนดไว้


![labelling](assets/notext/6.webp)

![labelling](assets/notext/7.webp)


เปิดหน้าต่างใหม่ คลิกที่ช่อง `Description` และพิมพ์ป้ายกำกับของคุณ


![labelling](assets/notext/8.webp)


เมื่อป้อนป้ายกำกับแล้ว คุณสามารถปิดหน้าต่างนี้ได้


![labelling](assets/notext/9.webp)


ป้ายกำกับของคุณถูกบันทึกเรียบร้อยแล้ว คุณสามารถค้นหาได้ภายใต้แท็บ `Description`


![labelling](assets/notext/10.webp)


ในแท็บ `Coins` ซึ่งคุณสามารถควบคุมเหรียญได้ ป้ายกำกับของคุณจะพบในคอลัมน์ `Label`

## บทแนะนำ: การติดฉลากบน Green Wallet

![labelling](assets/notext/12.webp)


พิมพ์ป้ายของคุณ จากนั้นคลิกที่ปุ่ม `บันทึก` สีเขียว


![labelling](assets/notext/13.webp)


คุณจะสามารถค้นหาฉลากของคุณได้ทั้งในรายละเอียดของธุรกรรมของคุณและบนแดชบอร์ดของ wallet ของคุณ


![labelling](assets/notext/14.webp)


## บทแนะนำ: การติดฉลากบน Samourai Wallet


ใน Samourai Wallet มีวิธีการต่างๆ ที่ช่วยให้คุณสามารถกำหนดป้ายกำกับให้กับธุรกรรมได้ สำหรับวิธีแรก เริ่มต้นโดยการเปิด wallet ของคุณและเลือกธุรกรรมที่คุณต้องการเพิ่มป้ายกำกับ จากนั้นกดปุ่ม `Add` ซึ่งอยู่ถัดจากกล่อง `Notes`


![labelling](assets/notext/15.webp)


พิมพ์ป้ายของคุณและยืนยันโดยคลิกที่ปุ่ม `เพิ่ม` สีน้ำเงิน


![labelling](assets/notext/16.webp)


คุณจะพบป้ายกำกับของคุณในรายละเอียดของธุรกรรมของคุณ แต่ยังอยู่บนแดชบอร์ดของ wallet ของคุณด้วย


![labelling](assets/notext/17.webp)

สำหรับวิธีที่สอง ให้คลิกที่จุดเล็ก ๆ สามจุดที่มุมขวาบนของหน้าจอ จากนั้นคลิกที่เมนู `Show Unspent Transaction Outputs`

![labelling](assets/notext/18.webp)


ที่นี่ คุณจะพบรายการที่ครอบคลุมของ UTXOs ทั้งหมดที่มีอยู่ใน wallet ของคุณ รายการที่แสดงนี้เกี่ยวข้องกับบัญชีเงินฝากของฉัน อย่างไรก็ตาม การดำเนินการนี้สามารถทำซ้ำได้สำหรับบัญชี Whirlpool โดยการนำทางจากเมนูเฉพาะ


จากนั้นคลิกที่ UTXO ที่คุณต้องการติดป้าย จากนั้นคลิกปุ่ม `Add`


![labelling](assets/notext/19.webp)


พิมพ์ป้ายกำกับของคุณและยืนยันโดยคลิกที่ปุ่ม `เพิ่ม` สีน้ำเงิน จากนั้นคุณจะพบป้ายกำกับของคุณทั้งในรายละเอียดของธุรกรรมของคุณและบนแดชบอร์ดของ wallet ของคุณ


![labelling](assets/notext/20.webp)


## บทแนะนำ: การติดฉลากบน Sparrow Wallet


ด้วยซอฟต์แวร์ Sparrow Wallet สามารถกำหนดป้ายกำกับได้หลายวิธี วิธีที่ง่ายที่สุดคือการเพิ่มป้ายกำกับล่วงหน้าเมื่อสื่อสารที่อยู่รับไปยังผู้ส่ง ในการทำเช่นนี้ ให้คลิกที่เมนู `Receive` จากนั้นคลิกที่ช่อง `Label` และป้อนป้ายกำกับที่คุณเลือก ป้ายกำกับนี้จะถูกเก็บรักษาและสามารถเข้าถึงได้ตลอดซอฟต์แวร์ทันทีที่ได้รับบิตคอยน์ที่อยู่ดังกล่าว


![labelling](assets/notext/21.webp)


หากคุณลืมติดป้ายที่อยู่ของคุณเมื่อได้รับแล้ว ยังสามารถเพิ่มได้ในภายหลังผ่านเมนู `Transactions` เพียงคลิกที่ธุรกรรมของคุณภายในคอลัมน์ `Label` จากนั้นป้อนป้ายที่ต้องการ


![labelling](assets/notext/22.webp)


คุณยังมีตัวเลือกในการเพิ่มหรือแก้ไขป้ายกำกับของคุณจากเมนู `Addresses`


![labelling](assets/notext/23.webp)


ในที่สุด คุณสามารถดูป้ายกำกับของคุณในเมนู `UTXOs` ได้ Sparrow Wallet จะเพิ่มลักษณะของผลลัพธ์ในวงเล็บหลังป้ายกำกับของคุณโดยอัตโนมัติ ซึ่งช่วยแยกแยะ UTXOs ที่เกิดจากการเปลี่ยนแปลงจากที่ได้รับโดยตรง


![labelling](assets/notext/24.webp)
