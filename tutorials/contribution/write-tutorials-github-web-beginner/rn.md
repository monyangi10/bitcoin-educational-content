## 1 - Rema ishami rishasha


Ugurure umucukumbuzi wawe maze ugende kuri paji yawe ya Fork iri mu bubiko bwa Plan ₿ Network. Iyi ni Fork mwashinze kuri GitHub. URL ya Fork yawe igomba gusa n'iyi: `https://github.com/[izina-ryawe-ry'ukoresha]/Ibirimwo-vy'inyigisho vya Bitcoin`:


![GITHUB](assets/fr/01.webp)


Raba neza ko uri kw'ishami ry'i `dev`, hanyuma ukande kuri buto ya "*Sync Fork*". Niba Fork yawe itari ku gihe, GitHub izogusaba guhindura ishami ryawe. Bandanya n'iyi nkuru nshasha:


![GITHUB](assets/fr/02.webp)


Fyonda ku ishami rya `dev`, hanyuma ushire izina ry'ishami ryawe rikora kugira ngo umutwe waryo ugaragaze neza intumbero yaryo, ukoresheje uturongo kugira ngo utandukanye amajambo. Nk'akarorero, nimba intumbero yacu ari ukwandika inyigisho yo gukoresha Green Wallet, ishami ryoshobora kwitwa: `tuto-Green-Wallet-loic`. Uhejeje kwinjiza izina rikwiye, kanda kuri "*Rema ishami*" kugira ngo wemeze iremwa ry'ishami ryawe rishasha rishingiye kuri `dev`:


![GITHUB](assets/fr/03.webp)


Ubu rero ukwiye kuba uri kw’ishami ryawe rishasha ry’akazi:


![GITHUB](assets/fr/04.webp)


Ivyo bisigura ko amahinduka yose uzogira azobikwa kuri iryo shami ry’umwihariko gusa.


Ku ngingo nshasha yose utegura gusohora, rema ishami rishasha rivuye kuri `dev`.


Ishami muri Git rigereranya verisiyo y'umugambi isa n'iyo, iguha uburenganzira bwo gukora ku bihinduwe ataco bihinduye ku ishami ry'ingenzi, gushika igikorwa cawe giteguye gushirwamwo.


## 2 - Kwongera amadosiye y'inyigisho


Ubu ishami rikora ryaremewe, ni igihe co gushiramwo inyigisho yawe nshasha.


Mu madosiye y'ishami ryawe, uzokenera kurondera ububiko bubereye bwo gushiramwo inyigisho yawe. Uko amadosiye atunganijwe vyerekana ibice bitandukanye vyo ku rubuga rwa Plan ₿ Network. Mu karorero kacu, kuko turiko twongerako inyigisho kuri Green Wallet, genda ku nzira ikurikira: `Bitcoin-ibirimwo-inyigisho\inyigisho\Wallet` ihuye n'igice ca `Wallet` c'urubuga:


![GITHUB](assets/fr/05.webp)


Mu dosiye `Wallet`, rema ububiko bushasha bwihariye cane cane inyigisho yawe. Izina ry’iyi dosiye rikwiye kwerekana neza porogarama ivugwa muri iyo nyigisho, hakoreshejwe uturongo kugira ngo amajambo ahuzwe. Nk'akarorero kanje, dosiye izokwitwa `Green-Wallet`. Fyonda kuri "*Ongera dosiye*" hanyuma kuri "*Rema dosiye nshasha*":


![GITHUB](assets/fr/06.webp)


Injira izina rya dosiye ikurikiwe n'agace `/` kugira ngo wemeze ko yaremwe nk'idosiye.


![GITHUB](assets/fr/07.webp)


Muri iyi dosiye nshasha y'inyigisho yawe, ukeneye kwongerako ibintu vyinshi:




- Rema dosiye `assets` kugira ngo ushiremwo ibigereranyo vyose bikenewe ku nyigisho yawe;
- Muri iyi dosiye `itunga`, rema dosiye ntoyi yitwa hakurikijwe kode y'ururimi rw'umwimerere rw'inyigisho. Nk'akarorero, nimba inyigisho yanditswe mu congereza, iyi dosiye ntoyi ikwiye kwitwa `en`. Shira ibishushanyo vyose vy’inyigisho (ibishushanyo, amashusho, amafoto, n’ibindi) muri iyi dosiye.
- Dosiye `tutorial.yml` itegerezwa kuremwa kugira ngo wandike amakuru y'inyigisho yawe;
- Dosiye y’ikimenyetso itegerezwa kuremwa kugira ngo wandike ibirimwo vy’ukuri mu nyigisho yawe. Iyi dosiye itegerezwa kwitwa hakurikijwe kode y’ururimi yanditswemwo. Nk'akarorero, ku nyigisho yanditswe mu gifaransa, dosiye ikwiye kwitwa `fr.md`.


Mu ncamake, ng'ubu uburongozi bwa dosiye (tuzobandanya tuzirema mu gice gikurikira):


```
bitcoin-educational-content/
└── tutorials/
└── wallet/ (à modifier avec la bonne catégorie)
└── green-wallet/ (à modifier avec le nom du tuto)
├── assets/
│   ├── fr/ (à modifier selon le code de langue approprié)
├── tutorial.yml
└── fr.md (à modifier selon le code de langue approprié)
```


## 3 - Uzuza dosiye ya YAML


Reka dutangure n’idosiye ya YAML. Mu gasandugu ko kurema dosiye nshasha, shiramwo `inyigisho.yml`:


![GITHUB](assets/fr/08.webp)


Uzuza dosiye `tutorial.yml` mu gukopa iyi nkuru ikurikira:


```
id:

project_id:

tags:
-
-
-

category:

level:

professor_id:

# Proofreading metadata

original_language:
proofreading:
- language:
last_contribution_date:
urgency:
contributor_names:
-
reward:
```


Akira ahantu hakenewe:



- id**: UUID (_Ikimenyetso kidasanzwe co kw'isi yose_) kigaragaza inyigisho mu buryo bwihariye. Ushobora kuyigira generate ukoresheje [igikoresho co kuri Internet] Igisabwa conyene ni uko iyo UUID iba random kugira ngo ntihagire amakimbirane n’iyindi UUID iri kuri platform;



- project_id**: UUID y’ishirahamwe canke ishirahamwe riri inyuma y’igikoresho kigaragazwa mu nyigisho [bivuye ku rutonde rw’imigambi](https://github.com/PlanB-Network/Bitcoin-ibirimwo-ivy’inyigisho/igiti/iterambere/ibikoresho/imigambi). Nk'akarorero, nimba uriko urahingura inyigisho yerekeye porogaramu ya Green Wallet, ushobora kubona iyi `project_id` muri dosiye ikurikira: `Bitcoin-ibirimwo-ivy'inyigisho/ibikoresho/imigambi/umurongo/umugambi.yml`. Aya makuru yongerwa kuri dosiye ya YAML y’inyigisho yawe kubera ko Plan ₿ Network ibungabunga urutonde rw’amashirahamwe yose n’imiryango yose ikora kuri Bitcoin canke imigambi ijana. Mu kwongerako `project_id` y'ikigo gihuye n'inyigisho yawe, urema ihuriro hagati y'izo Elements zibiri;



- tags**: amajambo 2 canke 3 y’ingenzi ajanye n’ibirimwo mu nyigisho, yatowe gusa [ku rutonde rw’amajambo ya Plan ₿ Network]



- category**: Urutonde ruto rujanye n’ibirimwo mu nyigisho, hakurikijwe imiterere y’urubuga rwa Plan ₿ Network (nk’akarorero, ku bikoresho: `ibikoresho`, `ibikoresho`, `bikoresho`, `ibikoresho`);



- urugero**: Urugero rw'ingorane z'inyigisho, rwatowe muri:
    - 'umutanguzi`
    - `hagati`
    - `iteye imbere`
    - 'umuhinga'



- professor_id**: `id_y'umwigisha` (UUID) yawe nk'uko yerekanwa kuri [umwirondoro wawe w'umwigisha](https://github.com/Ikigo-C'Umugambi/Bitcoin-ibirimwo-ivy'inyigisho/igiti/dev/abaporofeseri);



- ururimi_rw'umwimerere**: Ururimi rw'umwimerere rw'inyigisho (nk'akarorero, `fr`, `en`, n'ibindi);



- gukosora**: Amakuru yerekeye ingene umuntu akora ivyo gukosora. Uzuza igice ca mbere, nk'uko gukosora inyigisho yawe bwite biharurwa nk'ukwemeza kwa mbere:
    - ururimi**: Ururimi rw'ugukosora (nk'akarorero, `fr`, `en`, n'ibindi).
    - itariki_yanyuma_y'intererano**: Itariki y'umusi.
    - vyihutirwa**: 1
    - amazina_y'abaterankunga**: Indangamuntu yawe ya GitHub.
    - impembo**: 0


Kubindi bisobanuro ku bijanye n'ikarata yawe y'umwigisha, urashobora kuraba inyigisho ihuye:

## 6 - Wongereko ikimenyetso n'igipfukisho


Muri dosiye `assets`, ukeneye kwongerako dosiye yitwa `logo.webp`, izokora nk'igishushanyo gito c'ingingo yawe. Iyi shusho itegerezwa kuba mu buryo bwa `.webp`, kandi itegerezwa kuba ifise ubunini bw'inzira kugira ngo ihure n'umukoresha Interface.


Ufise umwidegemvyo wo guhitamwo ikimenyetso ca porogarama ikoreshwa mu nyigisho, canke irindi shusho ryose ribereye, igihe cose ritagira uburenganzira bwo gukoresha. Ikindi, wongereko ishusho yitwa `cover.webp` aho hantu nyene. Ivyo bizokwerekanwa hejuru y’inyigisho yawe. Raba neza ko iri shusho, nk’ikimenyetso, ryubahiriza uburenganzira bwo gukoresha kandi rihuye n’ivyo inyigisho yawe ivuga.


Kugira ngo wongere amashusho muri dosiye `/assets`, ushobora kuyakurura ukayakura mu madosiye yawe yo mu karere. Raba neza ko uri muri dosiye `/assets` no ku ishami ry'iburyo, hanyuma ukande kuri "*Gukora amahinduka*".


![GITHUB](assets/fr/26.webp)


Ubu rero ushobora kubona amashusho aboneka muri dosiye.


![GITHUB](assets/fr/27.webp)

## 8 - Kwongera amashusho


Ububiko bw'ururimi buri muri dosiye `/assets` (mu karorero kanje: `/assets/en`) bukoreshwa mu kubika ibishushanyo n'ibishushanyo bizojana n'inyigisho yawe. Uko bishoboka kwose, niwirinde gushiramwo amajambo mu mashusho yawe kugira ngo ivyo uvuga bishike ku bantu bo kw’isi yose. Ego ni ko, porogarama yerekanwa izobamwo amajambo, mugabo niwashiramwo ibishushanyo canke ibindi bimenyetso ku mafoto y’ivyo bikoresho, nubikore ata majambo canke, nimba ari ngombwa, ukoreshe Icongereza.


Kugira ngo ushire amazina ku mashusho yawe, ukoreshe gusa imibare ihuye n’urutonde rwayo mu nyigisho, ikaba iteye nk’imibare ibiri (canke imibare itatu nimba inyigisho yawe irimwo amashusho arenga 99). Nk'akarorero, ushire izina ry'ishusho yawe ya mbere `01.webp`, iya kabiri `02.webp`, n'ibindi.


Amashusho yawe ategerezwa kuba ari mu buryo bwa `.webp` gusa. Niba ari ngombwa, urashobora gukoresha [porogarama yanje yo guhindura amashusho](https://github.com/LoicPandul/Igihinduzi c'amashusho).


![GITHUB](assets/fr/32.webp)


Ubu ko wongeyeko amashusho yawe muri dosiye nto, ushobora gukuraho dosiye y'ikinyoma `.gitkeep`. Gufungura iyi dosiye, ukande ku tudodo dutatu turi hejuru iburyo, hanyuma ukande kuri "*Delete file*".


![GITHUB](assets/fr/33.webp)


Bika ivyo wahinduye ukanda kuri "*Gukora ivyo wahinduye...*".


![GITHUB](assets/fr/34.webp)


Kugira ngo winjize igishushanyo kivuye muri dosiye yawe mu nyandiko yawe y'ubuhinga, koresha itegeko rikurikira ry'Ikigereranyo, wiyubare kugaragaza umwandiko ukwiye n'inzira y'ishusho ibereye y'ururimi rwawe:


```

```


Ikimenyetso c’ugutangara kiri mu ntango cerekana ishusho. Ivyanditswe bisubirira, bifasha mu gushikira no gukoresha ibisobanuro, bishirwa hagati y’ibifunga vy’inzira. Ubwa nyuma, inzira ija ku gishushanyo yerekanwa hagati y’ibifunga.


![GITHUB](assets/fr/35.webp)

**-> Impanuro:** Igihe usangira amadosiye mu ruhame, nk'amashusho, birahambaye gukuraho amakuru atari ngombwa. Kugira ngo worohereza iyo nzira, ushobora gukoresha ibikoresho vyihariye nka [Exif Cleaner](https://exifcleaner.com/), bigufasha gusukura amakuru y’inyandiko ukoresheje uburyo bworoshe bwo gukurura no gutera.


## 9 - Gutanga inyigisho


Uhejeje kwandika inyigisho yawe mu rurimi uhisemwo, intambwe ikurikira ni ugutanga **Pull Request**. Umuyobozi azoca yongerako ubuhinduzi bubuze mu nyigisho yawe, akoresheje uburyo bwacu bwo guhindura bwikora bukoresheje ugukosora kw’abantu.


Kugira ngo ukomeze n'Igisabwa co Gukuraho, umaze kubika ivyo wahinduye vyose, fyonda ku buto "*Gutanga*", hanyuma kuri "*Gufungura igisabwa co gukuraho*":


![GITHUB](assets/fr/36.webp)


Igisabwa co gukura ni igisabwa gikorwa co gushiramwo amahinduka ava mw’ishami ryawe mw’ishami ry’ingenzi ry’ububiko bwa Plan ₿ Network, ivyo bikaba bituma umuntu ashobora gusubiramwo no kuganira ku mahinduka imbere y’uko ahurizwa hamwe.


Imbere y’uko ubandanya, nusuzume neza aha hepfo y’igitabu Interface ko ayo mahinduka ari yo wari witeze:


![GITHUB](assets/fr/37.webp)


Raba neza, hejuru ya Interface, ko ishami ryawe rikora ryifatanijwe n'ishami rya `dev` ry'ububiko bwa Plan ₿ Network (ari ryo shami nyamukuru).


Injira umutwe uvuga muri make amahinduka wipfuza gufatanya n'ububiko bw'inkomoko. Yongerako igitekerezo gitoyi kidondora ayo mahinduka (niba ufise inomero y'ikibazo ijanye n'uguhingura inyigisho yawe, wibuke kwandika `Ifunga #{inomero y'ikibazo}` nk'igitekerezo), hanyuma ukande kuri buto ya Green "*Create pull request*" kugira ngo wemeze igisabwa co gufatanya:


![GITHUB](assets/fr/38.webp)


PR yawe izoca iboneka mu gice ca "*Pull Request*" c'ububiko nyamukuru bwa Plan ₿ Network. Ico ubwirizwa gukora ubu ni ukurindira gushika umuyobozi akumenyesheje ko intererano yawe yashizwe hamwe, canke ngo asabe ibindi bihindurwa.


![GITHUB](assets/fr/39.webp)


Amaze gufatanya PR yawe n'ishami ry'ingenzi, turagusavye gukuraho ishami ryawe rikora (mu karorero kanje: `tuto-Green-Wallet`) kugira ngo ugume ufise amateka meza ya Fork yawe. GitHub izoca iguha ubu buryo kuri paji yawe ya PR:


![GITHUB](assets/fr/40.webp)


Niba wipfuza guhindura intererano yawe umaze gutanga PR yawe, intambwe zo gukurikira zivana n’ingene PR yawe imeze ubu:




- Nimba PR yawe ikiri yuguruye kandi itarahurizwa hamwe, nukore ivyo guhindura ku ishami ry’akazi rimwe. Impinduka z’ukwiyemeza zizokwongerwa kuri PR yawe ikiri yuguruye;
- Mu gihe PR yawe yoba yamaze guhurizwa hamwe n’ishami ry’ingenzi, uzokenera gusubira gukora iyo nzira kuva mu ntango mu guhingura ishami rishasha, hanyuma ugatanga PR nshasha. Raba neza ko Fork yawe ihuye n'ububiko bw'inkomoko bwa Plan ₿ Network ku ishami rya `dev` imbere yo gukomeza.


Niba ufise ingorane z’ubuhinga zo gutanga inyigisho yawe, ntukekeranye gusaba imfashanyo kuri [umugwi wacu wihariye wo gutanga intererano](https://t.me/PlanBNetwork_ContentBuilder). Murakoze cane!
