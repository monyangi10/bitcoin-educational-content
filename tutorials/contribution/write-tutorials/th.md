ภารกิจของ Plan ₿ Academy คือการทำให้ทรัพยากรการศึกษาชั้นนำเกี่ยวกับ Bitcoin สามารถเข้าถึงได้ในหลายภาษามากที่สุด เนื้อหาทั้งหมดที่เผยแพร่บนเว็บไซต์เป็นโอเพ่นซอร์สและโฮสต์บน GitHub ทำให้ทุกคนมีโอกาสเข้าร่วมในการเพิ่มคุณค่าให้กับแพลตฟอร์ม การมีส่วนร่วมสามารถทำได้หลายรูปแบบ: การแก้ไขและการพิสูจน์อักษรของข้อความที่มีอยู่ การแปลเป็นภาษาอื่น การอัปเดตข้อมูล หรือแม้กระทั่งการสร้างบทเรียนใหม่ที่ยังไม่มีในเว็บไซต์ของเรา


ในบทแนะนำนี้ เราจะมาดูวิธีการเพิ่มเนื้อหาใหม่ เช่น บทแนะนำหรือหลักสูตร ลงในแพลตฟอร์ม Plan ₿ Academy


## ความแตกต่างระหว่างการสอนแบบแนะนำและหลักสูตรคืออะไร?


ส่วนหลักสองส่วนของแพลตฟอร์มของเราคือหลักสูตรและบทแนะนำ หลักสูตรให้ทรัพยากรการศึกษาเชิงทฤษฎี ในขณะที่บทแนะนำเสนอเนื้อหาปฏิบัติที่แสดงวิธีการใช้เครื่องมือเฉพาะ (เช่น ฮาร์ดแวร์ wallet หรือซอฟต์แวร์) หรือการปฏิบัติพิเศษ (เช่น การรักษาความปลอดภัยของวลีช่วยจำหรือการตรวจสอบความถูกต้องของซอฟต์แวร์)


หลักสูตรมีความยาวและมีโครงสร้างมากกว่าบทเรียนตามธรรมชาติ ควรสำรวจหัวข้อเฉพาะที่เกี่ยวข้องกับ Bitcoin หรือระบบนิเวศของมันอย่างลึกซึ้ง แม่นยำ และละเอียดถี่ถ้วน


[ค้นหาหลักสูตร Plan ₿ Academy](https://planb.academy/courses)

[ค้นหาบทเรียน Plan ₿ Academy](https://planb.academy/tutorials)

## ฉันจะเสนอหลักสูตรใหม่ได้อย่างไร?


หากคุณมีแนวคิดสำหรับหลักสูตร Bitcoin ที่ยังไม่มีบนแพลตฟอร์มและต้องการเขียน คุณสามารถติดต่อเราได้ที่ที่อยู่นี้:


contact@planb.network


ในข้อความของคุณ โปรดแนะนำตัวเองสั้น ๆ และอธิบายแนวคิดของหลักสูตรของคุณ หากคุณมีแนวคิดเกี่ยวกับโครงสร้างของหลักสูตรแล้ว โปรดระบุบทต่าง ๆ ด้วย เราจะติดต่อกลับโดยเร็วที่สุดเพื่ออธิบายวิธีดำเนินการต่อและหากจำเป็นจะหารือเกี่ยวกับโครงสร้างของหลักสูตรของคุณ


## ฉันจะส่งบทแนะนำใหม่ได้อย่างไร?


ในการเพิ่มบทแนะนำใหม่ไปยัง Plan ₿ Academy คุณสามารถทำได้โดยตรงบน GitHub โดยการส่ง Pull Request ในบทแนะนำด้านล่างและบทแนะนำที่มาพร้อมกัน ฉันจะพาคุณไปทีละขั้นตอนผ่านกระบวนการนี้และแจ้งให้คุณทราบถึงแนวทางที่ต้องปฏิบัติตามเพื่อให้แน่ใจว่าบทแนะนำของคุณเข้ากันได้กับแพลตฟอร์ม ในขณะที่ยังคงรักษาประวัติการมีส่วนร่วมที่ชัดเจน


## 1 - เลือกธีมสำหรับบทแนะนำของคุณ


เรากำลังมองหาบทแนะนำเกี่ยวกับเครื่องมือที่เกี่ยวข้องกับ Bitcoin หรือระบบนิเวศของมันเป็นหลัก เนื้อหานี้สามารถแบ่งออกเป็นหกหมวดหมู่หลัก:




- พอร์ตโฟลิโอ ;
- โหนด ;
- Mining ;
- พ่อค้า;
- Exchange ;
- การรักษาความลับ


นอกเหนือจากหัวข้อเหล่านี้ที่เกี่ยวข้องกับ Bitcoin โดยเฉพาะ Plan ₿ Academy ก็กำลังมองหาการมีส่วนร่วมในหัวข้อที่เน้นอธิปไตยของบุคคล เช่น:




- เครื่องมือโอเพ่นซอร์ส ;
- ไอที ;
- การเข้ารหัส ;
- พลังงาน ;
- คณิตศาสตร์;
- เศรษฐกิจ;
- DIY;
- LifeHacking...


ตัวอย่างเช่น ปัจจุบันเรามีบทแนะนำเกี่ยวกับ Tails, Nostr และ GrapheneOS เครื่องมือเหล่านี้ไม่ได้เกี่ยวข้องโดยตรงกับ Bitcoin แต่เป็นระบบที่อาจน่าสนใจสำหรับเราในฐานะส่วนหนึ่งของการเคลื่อนไหวสู่ความเป็นอธิปไตยในโลกดิจิทัล เนื้อหาเหล่านี้สามารถรวมเข้ากับหมวดหมู่ย่อยของส่วน "อื่นๆ" ได้


คุณสามารถออกแบบบทแนะนำตั้งแต่เริ่มต้น หรือใช้บทแนะนำที่เคยเผยแพร่บนเว็บไซต์ของคุณ (โดยที่คุณเป็นเจ้าของลิขสิทธิ์) และแชร์บน Plan ₿ Academy ด้วย โดยเพิ่มลิงก์ไปยังบทความต้นฉบับ


ไม่ว่าคุณจะเลือกแบบใด โปรดจำไว้ว่าทุกเนื้อหาที่เผยแพร่บน Plan ₿ Academy ได้รับอนุญาตภายใต้สัญญาอนุญาตแบบเสรี [CC-BY-SA](https://creativecommons.org/licenses/by-sa/4.0/) สัญญาอนุญาตนี้อนุญาตให้ใครก็ตามสามารถคัดลอกและอาจแก้ไขเนื้อหาของคุณได้ โดยมีเงื่อนไขเพียงอย่างเดียวว่าต้องให้เครดิตแหล่งที่มาเดิมอย่างถูกต้อง


## 2 - ติดต่อทีม Plan ₿ Academy


เมื่อคุณเลือกธีมสำหรับบทแนะนำของคุณแล้ว ขั้นตอนต่อไปคือแจ้งให้เราทราบถึงความตั้งใจของคุณที่จะเพิ่มเนื้อหานี้ลงในแพลตฟอร์ม วิธีที่ง่ายที่สุด หากคุณมีบัญชี Telegram คือ [เข้าร่วมกลุ่มของเรา](https://t.me/PlanBNetwork_ContentBuilder)


![TUTO](assets/fr/39.webp)


แนะนำตัวเองสั้นๆ และระบุเนื้อหาเฉพาะที่คุณต้องการเขียน รวมถึงภาษาที่คุณต้องการเขียน โดยส่งข้อความในช่อง "General" สมาชิกในทีมจะสร้าง issue บน GitHub ที่สอดคล้องกับการมีส่วนร่วมในอนาคตของคุณ


หากคุณไม่มีบัญชี Telegram และไม่ต้องการสร้างบัญชี คุณยังสามารถติดต่อเราทางอีเมลได้ที่ที่อยู่นี้


paolo@planb.network


## 3 - การเลือกเครื่องมือที่เหมาะสมในการมีส่วนร่วม


ในการมีส่วนร่วมกับ Plan ₿ Academy คุณมี 3 ตัวเลือกขึ้นอยู่กับระดับประสบการณ์ของคุณกับ GitHub:




- ผู้ใช้ที่มีประสบการณ์**: ดำเนินการต่อด้วยวิธีการที่คุณคุ้นเคยและเพียงแค่ปรึกษาบทแนะนำสั้น ๆ ด้านล่างซึ่งสรุปโครงสร้างไฟล์ของที่เก็บ Plan ₿ Academy ข้อกำหนดเฉพาะและวิธีการทำงาน:


https://planb.academy/tutorials/contribution/content/write-tutorials-git-expert-0ce1e490-c28f-4c51-b7e0-9a6ac9728410


- ระดับกลาง (GitHub Desktop)**: หากคุณไม่คุ้นเคยกับการใช้ Git ตัวเลือกแรกคือการตั้งค่าสภาพแวดล้อมในเครื่องของคุณเองอย่างง่ายดายเพื่อมีส่วนร่วมใน Plan ₿ Academy วิธีนี้แนะนำสำหรับการมีส่วนร่วมที่สำคัญ เช่น การเขียนบทแนะนำแบบเต็มรูปแบบ ในการทำเช่นนี้ ให้ทำตามบทแนะนำทีละขั้นตอนด้านล่าง:


https://planb.academy/tutorials/contribution/content/write-tutorials-github-desktop-intermediate-4a36a052-1000-4191-890a-9a1dc65f8957


- ผู้เริ่มต้น (อินเทอร์เฟซเว็บ)**: คุณยังสามารถเลือกใช้อินเทอร์เฟซเว็บของ GitHub ได้โดยตรง โดยไม่ต้องตั้งค่าสภาพแวดล้อมในเครื่องอย่างสมบูรณ์ วิธีนี้สามารถพิจารณาใช้สำหรับการมีส่วนร่วมเล็กน้อย อย่างไรก็ตาม สำหรับการมีส่วนร่วมที่สำคัญ เช่น การเพิ่มบทเรียนใหม่ทั้งหมด ตัวเลือกนี้อาจซับซ้อนกว่าการตั้งค่าสภาพแวดล้อมในเครื่อง ฉันจะอธิบายวิธีการทำในบทเรียนนี้:


https://planb.academy/tutorials/contribution/content/write-tutorials-github-web-beginner-e64f8fed-4c0b-4225-9ebb-7fc5f1c01a79

เลือกหนึ่งในสามตัวเลือกนี้ตามระดับความรู้ของคุณ จากนั้นเริ่มเขียนบทแนะนำ Plan ₿ Academy แรกของคุณ!
