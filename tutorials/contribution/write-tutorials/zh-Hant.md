Plan ₿ Network 的使命是以盡可能多的語言提供 Bitcoin 上領先的教育資源。網站發佈的所有內容都是開放原始碼，並託管在 GitHub 上，讓任何人都有機會參與豐富平台的內容。貢獻可以有多種形式：修正和校對現有的文字、翻譯成其他語言、更新資訊，甚至創造我們網站上尚未提供的新教學。


在本教程中，我們將探討如何在 Plan ₿ Network 平台上新增內容，例如教學或課程。


## 教學和課程有什麼不同？


我們平台的兩個主要部分是課程和教學。課程提供理論教育資源，而教學則提供實用內容，說明如何使用特定工具 (例如 Hardware Wallet 或軟體) 或特定作法 (例如保護 Mnemonic 短語或檢查軟體真偽)。


課程的篇幅自然比導修課更長、更有條理。課程應深入、精確且詳細地探討與 Bitcoin 或其生態系統相關的特定主題。


[Discover Plan ₿ Network 課程](https://planb.network/courses)


![TUTO](assets/fr/37.webp)


教學通常較短。這些教學以圖片為輔助，逐步說明如何執行特定的 Bitcoin 相關任務。


[探索 Plan ₿ Network 教程](https://planb.network/tutorials)

## 如何建議新課程？


如果您有尚未出現在平台上的 Bitcoin 課程構思，並且想要撰寫，您可以透過下列 Address 聯絡我們：


contact@planb.network


在您的訊息中，請簡短地自我介紹並描述您的課程構思。如果您對課程結構已有想法，也請附上個別章節。我們會儘快回覆您，說明如何進行，必要時也會討論您的課程結構。


## 如何提交新教學？


若要在 Plan ₿ Network 上新增教學，您可以直接在 GitHub 上提交 Pull Request。在下面的教學和隨附的教學中，我會帶您逐步完成這個過程，並告知您需要遵循的準則，以確保您的教學與平台相容，同時保持清晰的貢獻歷史。


## 1 - 為您的教學選擇主題


我們主要尋找與 Bitcoin 或其生態系統相關的工具教學。這些內容可分為六大類：




- 投資組合 ；
- 節點 ；
- Mining ；
- 商家；
- Exchange ；
- 保密性。


除了這些與 Bitcoin 具體相關的主題之外，Plan ₿ Network 也在尋求強調個人主權的主題的貢獻，例如：




- 開放原始碼工具；
- IT ；
- 密碼學 ；
- 能源 ；
- 數學；
- 經濟；
- DIY；
- LifeHacking...


例如，我們目前有 Tails、Nostr 和 GrapheneOS 的教學。這些工具與 Bitcoin 並無直接關係，但它們是我們可能感興趣的系統，是邁向數位世界主權的一部分。這些內容可以整合到「其他」部分的子類別中。


您可以從頭開始設計教學，或是將之前在您網站上發表的教學（只要您擁有版權）也放到 Plan ₿ Network 上分享，並加入原始文章的連結。


無論您選擇哪一種方式，請記住 Plan ₿ Network 上發佈的所有內容都是在免費 [CC-BY-SA](https://creativecommons.org/licenses/by-sa/4.0/) 授權下使用。此許可授權任何人複製並可能修改您的內容，唯一條件是原始來源必須正式註明。


## 2 - 聯絡 Plan ₿ Network 團隊


一旦您選好教學的主題，下一步就是通知我們您打算將這些內容加入平台。如果您有 Telegram 帳戶，最簡單的方法就是 [加入我們的群組](https://t.me/PlanBNetwork_ContentBuilder)。


![TUTO](assets/fr/39.webp)


在「General」頻道發送訊息，簡短地自我介紹並說明您想要撰寫的特定內容，以及您想要撰寫的語言。接下來就會有團隊成員在 GitHub 上建立與您未來貢獻對應的問題。


如果您沒有 Telegram 帳戶也不想建立，也可以透過下列 Address 以電子郵件與我們聯絡


paolo@planb.network


## 3 - 選擇正確的工具來貢獻


若要貢獻 Plan ₿ Network，您有 3 種選擇，視您對 GitHub 的使用經驗而定：




- 有經驗的使用者**：繼續使用您的常規方法，只需參考下面的簡短教程，其中總結了 Plan ₿ Network 儲存庫的檔案結構、具體要求和工作方法：


https://planb.network/tutorials/contribution/content/write-tutorials-git-expert-0ce1e490-c28f-4c51-b7e0-9a6ac9728410


- 中級 (GitHub 桌面)** ：如果您不熟悉使用 Git，第一個選擇是輕鬆建立自己的本機環境，為 Plan ₿ Network 做出貢獻。對於重要的貢獻，例如撰寫完整的教學，建議使用此方法。要做到這一點，請按照下面的教學一步一步來：


https://planb.network/tutorials/contribution/content/write-tutorials-github-desktop-intermediate-4a36a052-1000-4191-890a-9a1dc65f8957


- 初學者 (web Interface)** ：您也可以選擇直接使用 GitHub 的網頁 Interface，而不需要建立完整的本機環境。小規模的貢獻可以考慮使用此方法。不過，對於重大的貢獻，例如新增完整的新教學，這個方法可能會比建立本機環境更複雜。我將在本教程中解釋如何做到這一點：


https://planb.network/tutorials/contribution/content/write-tutorials-github-web-beginner-e64f8fed-4c0b-4225-9ebb-7fc5f1c01a79

根據您的知識程度從這三個選項中選擇一個，然後開始撰寫您的第一份 Plan ₿ Network 教學！
