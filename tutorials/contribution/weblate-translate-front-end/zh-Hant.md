Plan ₿ Network 的使命是在 Bitcoin 上提供一流的教育資源，並將其翻譯成盡可能多的語言。網站發佈的大部分內容都是開放原始碼，並託管在 GitHub 上，讓任何人都能參與豐富平台的內容。貢獻的形式有多種：修正和校對現有內容、更新資訊，或創作新的教學內容加入平台。


在本教程中，我們將告訴您如何輕鬆地為我們網站上靜態 Elements 的翻譯做出貢獻。平台上的資料主要分為兩大類：


- 前端資料/靜態 Elements（頁面、按鈕等）；
- 教育內容 (教學、課程、資源...)。


為了翻譯教育內容，我們使用 [人工智慧](https://github.com/Asi0Flammeus/LLM-Translator)。然後，為了糾正這些檔案中的最終錯誤，我們邀請校對人員貢獻心力。如果您希望校對某些內容，請參閱下列教學：


https://planb.network/tutorials/contribution/content/contribution-proofreading-review-tutorial-1ee068ca-ddaf-4bec-b44e-b41a9abfdef6

另一方面，如果您有興趣翻譯網站的靜態 Elements（不包括教育內容），您就找對地方了！為了有效翻譯前端內容，我們使用 Weblate 工具，此工具使用起來非常簡單，並有助於翻譯的方法。


如果您希望在 Plan ₿ Network 中加入全新的語言，請務必透過我們的 [Telegram 群組](https://t.me/PlanBNetwork_ContentBuilder) 聯絡 Plan ₿ Network 團隊。如果您沒有電報，您可以寄送電子郵件到 mari@planb.network。請務必撰寫一小段簡報，說明您是誰以及您所使用的語言。

我們的團隊成員會給您具體的指示，並在 Github 上開啟相關「問題」以協調您的工作。


在按照本教學添加新的語言到 Weblate。


https://planb.network/tutorials/contribution/content/weblate-add-new-language-eef2f5c0-1aba-48a3-b8f0-a57feb761d86

當您準備好開始翻譯時，請回到本教學，並閱讀下列要點。


## 在 Weblate 上註冊



- 前往 [Plan ₿ Network 的自助式 Weblate](https://weblate.planb.network/)：

![weblate](assets/01.webp)


- 如果您已經擁有 Weblate 帳戶，請點選 ` 登入 `：

![weblate](assets/02.webp)


- 如果您沒有帳號，請點選「註冊」：

![weblate](assets/03.webp)


- 輸入您的電子郵件 Address，以及使用者名稱和全名 (您可以使用假名)，然後按一下「註冊」：

![weblate](assets/04.webp)


- 在您的電子郵件收件匣中，您應該已收到 Weblate 寄來的確認訊息。按一下連結以確認您的註冊：

![weblate](assets/05.webp)


- 選擇一個強大的密碼，然後按一下`變更我的密碼`：

![weblate](assets/06.webp)


- 現在您可以回到 Plan ₿ Network 面板：

![weblate](assets/07.webp)


## 開始翻譯



- 按一下 `Website Elements` 專案 (不是詞彙表)：

![weblate](assets/08.webp)


- 您會到達一個 Interface，在這裡您可以看到正在進行中的語言：

![weblate](assets/09.webp)


- 選擇您的語言。例如，以法語為例：

![weblate](assets/10.webp)


- 若要開始翻譯，只需按一下「翻譯」按鈕：

![weblate](assets/11.webp)


- 您將被重新導向至工作 Interface：

![weblate](assets/12.webp)


- 然後 Weblate 會自動建議句子、段落或甚至單字翻譯到`語言`方塊中。就您的情況而言，您可能會看到英文主字串，以及另一個您的語言文字方塊：

![weblate](assets/13.webp)


- 您的任務是翻譯指定的字串。您必須在您所選擇的語言對應的方塊中插入您的文字。例如，如果您正在處理法文版本，請將您的翻譯內容寫在`法文`方塊中：

![weblate](assets/14.webp)


- 按一下「自動建議」標籤：

![weblate](assets/15.webp)


- 在這裡，Weblate 會為您展示人工智慧所做的翻譯：

![weblate](assets/16.webp)


- 如果建議的翻譯似乎與您相關，您可以按一下 `Clone to translation` 按鈕：

![weblate](assets/17.webp)


- 該建議現在已放入您的工作方塊中：

![weblate](assets/18.webp)


- 然後您可以手動修改建議：

![weblate](assets/19.webp)


- 一旦翻譯結果令您滿意，請按一下「儲存並繼續」按鈕。當您確定翻譯完成後，請務必取消勾選「需要編輯」方塊：

![weblate](assets/20.webp)


- 就是這樣！您的翻譯已成功儲存。Weblate 會自動將您重定向到下一個要翻譯的項目。如果您回到與您的語言對應的儀表板，您可以看到任何類型的字串都有不同的翻譯狀態。例如，如果您只需要專注於「未翻譯字串」，您可以點選特定的索引標籤：

![weblate](assets/21.webp)


- 如果您需要搜尋特定的字詞，不論是您的語言或原文，請按一下「搜尋」，然後在那裡插入：

![weblate](assets/22.webp)


## 翻譯指引



- 當您發現插入在大括弧「{」內的字詞時，您不需要翻譯它們。例如，在 「您的帳戶已建立，{{userName}}!」中，您將翻譯整個句子，但保留英文中的 "username"。
- 當您在字串中找到 "Plan ₿ Network 「時，請確定不要翻譯 」network"（將 Plan ₿ Network 視為商標）。此外，請務必使用 Bitcoin 的 ₿！
- 如果您發現只有「網路」一詞，您可以將其翻譯。
- 不要翻譯「B-CERT」，因為它是另一個固定的字。
- 如果您發現以空格結束的字串，可以不填。
- 有些字串可能在最後一個字和標點符號之間有空格：除非文法暗示，否則不要在您的目標語言中留下空格。例如，"Contact information :「 應更正為 」Contact information:"。在這種情況下，請使用正確的方式來翻譯。您也可以在原始英文版本中加入註解，告訴管理員這個問題。


## 新功能


- 我們正在努力為任何字串新增「解釋」區段，並附上截圖，以協助您找到特定句子/字詞在網站上出現的位置。目前，如果您對某些字詞有任何疑問，並需要找到它們在網站上的具體位置，您可以在「評論」部分提出問題，或在本教程開頭提到的 Telegram 群組中詢問翻譯協調員。
