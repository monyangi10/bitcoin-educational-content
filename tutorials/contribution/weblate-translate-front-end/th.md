ภารกิจของ Plan ₿ Academy คือการจัดหาทรัพยากรการศึกษาชั้นหนึ่งเกี่ยวกับ Bitcoin และแปลเป็นภาษาต่างๆ ให้ได้มากที่สุด เนื้อหาส่วนใหญ่ที่เผยแพร่บนเว็บไซต์เป็นโอเพ่นซอร์สและโฮสต์บน GitHub ทำให้ทุกคนสามารถมีส่วนร่วมในการเพิ่มคุณค่าให้กับแพลตฟอร์มได้ การมีส่วนร่วมสามารถทำได้หลายรูปแบบ: การแก้ไขและพิสูจน์อักษรเนื้อหาที่มีอยู่, การอัปเดตข้อมูล, หรือการสร้างบทเรียนใหม่เพื่อเพิ่มบนแพลตฟอร์ม


ในบทแนะนำนี้ เราจะแสดงวิธีการมีส่วนร่วมในการแปลองค์ประกอบคงที่บนเว็บไซต์ของเราได้อย่างง่ายดาย ข้อมูลบนแพลตฟอร์มแบ่งออกเป็นสองหมวดหมู่หลัก:


- ข้อมูลส่วนหน้าส่วนประกอบคงที่ (หน้า, ปุ่ม, ฯลฯ);
- เนื้อหาด้านการศึกษา (บทเรียน, หลักสูตร, ทรัพยากร...)


ในการแปลเนื้อหาด้านการศึกษา เราใช้[ปัญญาประดิษฐ์](https://github.com/Asi0Flammeus/LLM-Translator) จากนั้นเพื่อแก้ไขข้อผิดพลาดที่อาจเกิดขึ้นในไฟล์เหล่านี้ เราขอเชิญผู้ตรวจทานให้มีส่วนร่วม หากคุณต้องการตรวจทานเนื้อหา โปรดดูบทแนะนำต่อไปนี้:


https://planb.academy/tutorials/contribution/content/proofreading-review-tutorial-28236c98-23b2-4efd-9563-953f08707017

ในทางกลับกัน หากคุณสนใจในการแปลองค์ประกอบคงที่ของเว็บไซต์ (ไม่รวมเนื้อหาการศึกษา) คุณมาถูกที่แล้ว! เพื่อแปลส่วนหน้าของเว็บไซต์อย่างมีประสิทธิภาพ เราใช้เครื่องมือ Weblate ซึ่งใช้งานง่ายมากและช่วยให้การแปลเป็นเรื่องง่ายขึ้น


หากคุณต้องการเพิ่มภาษาที่ใหม่ทั้งหมดใน Plan ₿ Academy โปรดติดต่อทีม Plan ₿ Academy ผ่าน [กลุ่ม Telegram](https://t.me/PlanBNetwork_ContentBuilder) หากคุณไม่มี Telegram คุณสามารถส่งอีเมลไปที่ mari@planb.network อย่าลืมเขียนการนำเสนอเล็กๆ เกี่ยวกับตัวคุณและภาษาที่คุณพูด

สมาชิกในทีมของเราจะให้คำแนะนำเฉพาะเจาะจงแก่คุณและจะเปิด "issues" ที่เกี่ยวข้องบน Github เพื่อประสานงานของคุณ


ก่อนที่จะทำตามบทแนะนำเฉพาะนี้เพื่อเพิ่มภาษาใหม่ใน Weblate


https://planb.academy/tutorials/contribution/content/weblate-add-new-language-eef2f5c0-1aba-48a3-b8f0-a57feb761d86

เมื่อคุณพร้อมที่จะเริ่มการแปล ให้กลับมาที่บทแนะนำนี้และทำตามจุดต่อไปนี้

## เริ่มการแปล



- คลิกที่โครงการ `Website Elements` (ไม่ใช่พจนานุกรม):

![weblate](assets/08.webp)


- คุณจะเข้าสู่ส่วนติดต่อที่คุณสามารถดูภาษาที่กำลังดำเนินการอยู่:

![weblate](assets/09.webp)


- เลือกภาษาของคุณ ตัวอย่างเช่น เรามาเลือกภาษาฝรั่งเศส:

![weblate](assets/10.webp)


- ในการเริ่มต้นการแปล เพียงคลิกที่ปุ่ม `Translate`:

![weblate](assets/11.webp)


- คุณจะถูกเปลี่ยนเส้นทางไปยังอินเทอร์เฟซการทำงาน:

![weblate](assets/12.webp)


- จากนั้น Weblate จะเสนอประโยค ย่อหน้า หรือแม้แต่คำให้แปลในช่อง `language` โดยอัตโนมัติ ในกรณีของคุณ คุณอาจจะเห็นสตริงหลักภาษาอังกฤษ และกล่องข้อความอีกอันสำหรับภาษาของคุณ:

![weblate](assets/13.webp)


- งานของคุณประกอบด้วยการแปลสตริงที่ระบุ คุณต้องใส่ข้อความของคุณในช่องที่ตรงกับภาษาที่คุณเลือก ตัวอย่างเช่น หากคุณกำลังทำงานในเวอร์ชันภาษาฝรั่งเศส ให้เขียนคำแปลของคุณในช่อง `French`:

![weblate](assets/14.webp)


- คลิกที่แท็บ `Automatic suggestion`:

![weblate](assets/15.webp)


- ที่นี่ Weblate แสดงการแปลที่ทำโดยปัญญาประดิษฐ์:

![weblate](assets/16.webp)


- หากคำแปลที่แนะนำดูเหมาะสมสำหรับคุณ คุณสามารถคลิกที่ปุ่ม `Clone to translation`:

![weblate](assets/17.webp)


- ข้อเสนอแนะถูกวางไว้ในกล่องงานของคุณแล้ว:

![weblate](assets/18.webp)


- คุณสามารถแก้ไขคำแนะนำได้ด้วยตนเอง:

![weblate](assets/20.webp)

![weblate](assets/21.webp)


- หากคุณต้องการค้นหาคำเฉพาะ ไม่ว่าจะเป็นในภาษาของคุณหรือในภาษาต้นฉบับ ให้คลิกที่ "ค้นหา" และใส่คำลงไปที่นั่น:

![weblate](assets/22.webp)


## แนวทางการแปล



- เมื่อบัญชีของคุณถูกสร้างขึ้นแล้ว, {{userName}}!
- เมื่อคุณพบ "Plan ₿ Academy" ในสตริง, ให้แน่ใจว่าอย่าแปลคำว่า "network" (พิจารณา Plan ₿ Academy เป็นเครื่องหมายการค้า) นอกจากนี้, ใช้ ₿ ของ Bitcoin เสมอ!
- หากคุณพบคำว่า "network" เพียงคำเดียว คุณสามารถแปลคำนี้แทนได้
- อย่าแปล "B-CERT" เนื่องจากเป็นคำที่กำหนดไว้แล้ว
- หากคุณพบสตริงที่ลงท้ายด้วยช่องว่าง คุณสามารถปล่อยไว้ได้
- บางสตริงอาจมีช่องว่างระหว่างคำสุดท้ายและเครื่องหมายวรรคตอน: อย่าปล่อยไว้ในภาษาเป้าหมายของคุณเว้นแต่ไวยากรณ์จะบ่งบอกถึงมัน ตัวอย่างเช่น "Contact information :" ควรถูกแก้ไขเป็น "Contact information:" ในกรณีนี้ ให้แปลโดยใช้วิธีที่ถูกต้อง คุณยังสามารถเพิ่มความคิดเห็นเพื่อบอกผู้ดูแลเกี่ยวกับปัญหานี้ในเวอร์ชันภาษาอังกฤษต้นฉบับ
