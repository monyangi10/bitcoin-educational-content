ภารกิจของ Plan ₿ Academy คือการจัดหาทรัพยากรการศึกษาชั้นหนึ่งเกี่ยวกับ Bitcoin และแปลเป็นภาษาต่างๆ ให้มากที่สุด เนื้อหาส่วนใหญ่ที่เผยแพร่บนเว็บไซต์เป็นโอเพ่นซอร์สและโฮสต์บน GitHub ทำให้ทุกคนสามารถมีส่วนร่วมในการเพิ่มคุณค่าให้กับแพลตฟอร์มได้ การมีส่วนร่วมสามารถทำได้หลายรูปแบบ: การแก้ไขและพิสูจน์อักษรเนื้อหาที่มีอยู่, การอัปเดตข้อมูล, หรือการสร้างบทเรียนใหม่เพื่อเพิ่มลงในแพลตฟอร์ม


เว็บไซต์ของเราปัจจุบันมีให้บริการในหลายภาษา และเรากำลังทำงานอย่างต่อเนื่องเพื่อเพิ่มภาษาใหม่ ๆ เพื่อปรับปรุงกระบวนการแปลสำหรับส่วนหน้าของเรา เราใช้เครื่องมือ Weblate ซึ่งช่วยให้เราร่วมมือและจัดการการแปลได้อย่างมีประสิทธิภาพ เป็นเครื่องมือที่ใช้งานง่าย


หากภาษาพื้นเมืองของคุณยังไม่มีในเว็บไซต์ของเราและคุณต้องการเพิ่มภาษา บทแนะนำนี้เหมาะสำหรับคุณ!


ก่อนอื่น โปรดติดต่อทีม Plan ₿ Academy ผ่าน [กลุ่ม Telegram ของเรา](https://t.me/PlanBNetwork_ContentBuilder) หากคุณไม่มี Telegram คุณสามารถส่งอีเมลไปที่ mari@planb.network อย่าลืมเขียนแนะนำตัวสั้น ๆ เกี่ยวกับตัวคุณและภาษาที่คุณพูด


## กำลังตรวจสอบว่ามีภาษาบน Weblate หรือไม่


เพื่อตรวจสอบว่าภาษาของคุณอยู่ในกลุ่มภาษาที่เรากำลังดำเนินการอยู่หรือไม่



- ไปที่ [แพลตฟอร์ม Weblate ของเรา](https://weblate.planb.network/projects/planb-network-website/):

![language](assets/01.webp)


- ในเมนู `องค์ประกอบเว็บไซต์` คุณจะพบรายการของภาษาทั้งหมดที่กำลังดำเนินการ:

![language](assets/02.webp)

หากภาษาของคุณอยู่ในรายการนี้ คุณไม่จำเป็นต้องเพิ่มอีกครั้ง เพื่อมีส่วนร่วมในการพิสูจน์อักษรใน weblate ค้นพบบทแนะนำต่อไปนี้:


https://planb.academy/tutorials/contribution/content/weblate-translate-front-end-8213b931-650f-4efd-8f4e-9a8ae5ce6295



หากไม่มีภาษาของคุณ ให้ทำตามบทแนะนำด้านล่างเพื่อเพิ่มภาษาใหม่


## การเพิ่มภาษาใหม่ใน Plan ₿ Academy



- ขั้นตอนแรกคือการสร้างบัญชีบน Weblate โดยคลิกที่ "Register" ที่มุมขวาบน (หากคุณต้องการความช่วยเหลือ คุณสามารถกลับไปที่บทแนะนำที่กล่าวถึงข้างต้นได้)



- เมื่อสร้างบัญชีของคุณแล้ว ให้ไปที่เมนู `Website elements` และเลือกแท็บ `Languages`:

![language](assets/03.webp)


- คลิกที่ `+` ที่มุมซ้ายบนของหน้าต่าง:

![language](assets/04.webp)


- เปิดรายการแบบเลื่อนลงและเลือกภาษาที่คุณต้องการเพิ่ม หากภาษาที่คุณกำลังมองหาไม่มีในรายการแบบเลื่อนลง คุณสามารถติดต่อที่ [Telegram group](https://t.me/PlanBNetwork_ContentBuilder) เพื่อให้ทีมงานของเราสร้างขึ้นเอง:

![language](assets/05.webp)


- คลิกที่ `เริ่มการแปลใหม่`:

![language](assets/06.webp)


- จากนั้นคุณจะไปถึงหน้าการจัดการการแปลสำหรับภาษาของคุณ:

![language](assets/07.webp)


- ในการเริ่มแปลองค์ประกอบคงที่ของเว็บไซต์ ให้คลิกที่ปุ่ม `Translate`: ![language](assets/08.webp)

หากต้องการคำแนะนำเกี่ยวกับกระบวนการแปล โปรดดูบทแนะนำเฉพาะของเราด้านล่างนี้:


https://planb.academy/tutorials/contribution/content/weblate-translate-front-end-8213b931-650f-4efd-8f4e-9a8ae5ce6295

ขอแสดงความยินดี คุณได้เริ่มกระบวนการแปลองค์ประกอบคงที่บนเว็บไซต์ของ Plan ₿ Academy แล้ว!

พวกเขารวมสตริงทั้งหมดบนเว็บไซต์ ยกเว้นเนื้อหาด้านการศึกษา (หลักสูตร, บทเรียน...) ซึ่งเราจะใช้วิธีการกึ่งอัตโนมัติอื่น (การแปลด้วย AI + การพิสูจน์อักษรโดยผู้ร่วมมือ)


ขอบคุณมากสำหรับการมีส่วนร่วมที่มีคุณค่า!
