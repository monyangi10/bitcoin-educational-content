## Mu ncamake


Niba ushaka gukosora canke gusubiramwo ibirimwo vya Plan ₿ Academy, winjire mu [mugwi wacu wa Telegaramu](https://t.me/Uwukora_Ibirimwo) wihariye intererano. Imenyekanishe muri make, hanyuma ugaragaze ivyo ushaka gusubiramwo be n’ururimi. Kora ivyo ukosora kw’ishami rikora maze ushiremwo PR iyo isuzuma ryawe rirangiye.


Nimba utatahuye igice kiri hejuru, ntuhagarike umutima—iyi nyigisho ni iyawe!


---

## Plan ₿ Academy ni iki?


Plan ₿ Academy ni urubuga rwo kwigirako ivya Bitcoin biciye mu nyigisho nyinshi n’inyigisho z’umuntu wese. Ubu, ibirimwo vyose vyacu ni ivy’inkomoko yuguruye, bishizwe ku bubiko bwa GitHub, kandi vyugururiye intererano zo gusubiramwo no kurema ibirimwo.


Plan ₿ Academy ifise intumbero yo kuba ikibanza ca bose c’urubuga rwa Bitcoin, igahuza no kuremesha imiryango ya Bitcoin kw’isi yose.


## Kubera iki wotanga?


Plan ₿ Academy mu vy’ukuri ni indimi nyinshi, kuko intumbero yacu ari ugutuma ibikoresho vya Bitcoin bishikira umuntu wese kw’isi yose. Twemera ko ururimi rudakwiye kuba intambamyi ku mahera y’igihugu.


Kugira ivyo bishoboke, dukoresha uburyo bwa Human x AI. Biciye kuri [porogarama] yacu(https://github.com/Asi0Flammeus/LLM-Translator), turashobora guhindura ubwacu ibirimwo mu ndimi nyinshi. Ariko rero, turakeneye umuntu avuga neza kugira ngo ayisuzume kugira ngo abone ko ibirimwo bitahurwa kandi ko ata majambo adakwiriye arimwo.


Niba ushaka gufasha kwubaka uyu mugambi w’inyigisho wo guteza imbere inyigisho za Bitcoin mu kibano cawe, urashobora gutangura usubiramwo inyigisho canke inyigisho.


Ikindi, uko dukunda cane *Value4Value* model, tuzohemba intererano zo gusubiramwo. Ingororano izoshingira ku burebure bw’ibirimwo, ku kuntu ururimi rwacu ruhindurwa rwonyene, no ku kuntu ibirimwo bigoye.


## Noba nkeneye kumenya ingene GitHub ikora?


Ntukeneye kuba umuhinga mu vya GitHub kugira ngo usuzume ibirimwo. Naho vyama ari vyiza gutahura ivyo uriko urakora (n’igituma), urashobora gukurikiza amabwirizwa ari aha hepfo kugira ngo ushiremwo intererano.


Ariko rero, nimba utazi Git na GitHub kandi ushaka kumenya vyinshi, ushobora kuraba ingingo yacu y’intango kuri ibi bikoresho:


https://planb.academy/tutorials/contribution/others/create-github-account-a75fc39d-f0d0-44dc-9cd5-cd94aee0c07c

## 1. Rema konti



- Rema konti kuri [GitHub](https://github.com/) kugira ngo utangure gutanga. Niba ushaka ubuyobozi bwo kurema konti yawe ya GitHub, genda ku nyigisho twavuze haruguru.



- Uzokenera kandi konti kuri porogarama y’ubutumwa [Telegram](https://telegram.org/).


## 2. Niwinjire mu mugwi wa Telegaramu



- Injira [mu mugwi w'abahinga mu vy'ubuhinga bwa Plan ₿ Academy](https://t.me/Ubuhinga bwo kwubaka ibirimwo) kuri Telegram.


![REVIEW](assets/fr/01.webp)



- Imenyekanishe muri make kandi ugaragaze ivyo ushaka gutangako n'ururimi mu butumwa buzorungikwa ku muhora "General".
- Umwe mu bagize umugwi azoheza areme ikibazo kuri GitHub ku bijanye n'intererano yawe izoza. Uzosangayo amakuru yose akenewe kugira ngo utangure gusubiramwo.


![REVIEW](assets/fr/02.webp)


## 3. Suzuma kandi uhindure ibirimwo biciye ku Interface ya GitHub



- Ikibazo kirimwo insobanuro (raba \**A\**) n'amahuza ku birimwo vy'umwimerere na dosiye yo gusubiramwo.
- Niba ushaka kwitwararika isubiramwo, usige igitekerezo ku kibazo (raba \**B\**) kugira ngo werekane ko uzoba uriko uragikorako. Ivyo bituma umuntu adasubiramwo utwigoro.
- Fyonda ku nzira iri iruhande ya dosiye kugira ngo usubiremwo (raba \**C\**) kugira ngo ubone ibirimwo.


![REVIEW](assets/fr/03.webp)


*Ushobora ubu kubona GitHub Interface ya dosiye yo gusubiramwo.*


- Igipande c'ibubamfu kigufasha guca mw'ishami ryawe (raba \**A\**) no mu madosiye (raba \**B\**).

*Fyonda ku kimenyetso c'ikaramu kugira ngo utangure gusubiramwo (raba **C**).*


![REVIEW](assets/fr/04.webp)



- Niba ari intererano yawe ya mbere, GitHub izogusaba Fork ububiko bw'inkomoko. Fyonda kuri buto gusa.


![REVIEW](assets/fr/05.webp)



- Uzoca ushika kuri Editor Interface (raba \**A\**), aho ushobora kubona inzira ya dosiye n’ishami rihuye (raba \**B\**) maze ugahindura hagati y’uburyo bwo guhindura n’uburyo bwo guca irya n’ino (raba \**C\**).


![REVIEW](assets/fr/06.webp)


NB. Ni vyiza gusubiramwo mw’idirisha rya kode, n’ibirimwo vy’umwimerere vyerekanywe mw’idirisha ritandukanye. Nk'akarorero, urashobora kuraba umwandiko ku rubuga rwa Plan ₿ Academy canke mu mbonerakure ya GitHub ku rubuga rw'umucukumbuzi. Koresha `ctrl + F` kugira uronke amakosa kandi ukosore mu mbonerahamwe ya kode uko ugenda.


## 4. Gukora amahinduka



- Ushobora gutangura gusubiramwo no gukosora ibirimwo uko bikenewe.
- Iyo ushaka guhagarika ikiganiro cawe, urashobora guhindura, naho nyene isubiramwo ritarangiye. Ni nk’ukuzigama iterambere ryawe kuri GitHub.
- Fyonda kuri buto ya "Guhindura...".


![REVIEW](assets/fr/07.webp)



- Yongerako ubutumwa budondora ivyo wahinduye (raba \**A\**).
- Yongerako ibindi bisobanuro mu ndondoro nimba bikenewe (raba \**B\**).
- Fyonda kuri buto ya "Gutanga amahinduka" (raba \**C\**).


![REVIEW](assets/fr/08.webp)



- Niba atari intererano yawe ya mbere, GitHub izogusaba gukora ishami rishasha. Ntushobora kwigera wiyemeza ku ishami ry'ingenzi rya `dev`. Niba waramaze gukora amasezerano ku ishami risanzwe rikora, hitamwo iryo shami. Ahandiho, hitamwo "Rema ishami rishasha" kugira ngo ureme irindi rishasha.


![REVIEW](assets/fr/13.webp)



- Uzoca ubona urupapuro ruvuga mu ncamake ivyo wahinduye vyose. Niwabicamwo, urashobora kubona ivyo vyahinduwe: ibintu vyasubiwemwo biboneka mu rangi itukura, mu gihe ibintu vyongereweko biboneka mu rangi ya Green.


![REVIEW](assets/fr/09.webp)



- Nimba utaraheza gusubiramwo kandi wipfuza gukomeza igikorwa cawe mu nyuma, urashobora gusohoka gusa.
- Igihe cose umaze gukora amahinduka yawe, abikwa kuri GitHub Fork yawe.
- Kugira ngo usubire gukora, genda kuri Fork yawe y'ububiko bw'ibirimwo bwa Plan ₿ Academy biciye ku rubuga rwawe rwa GitHub.


![REVIEW](assets/fr/14.webp)



- Hitamwo ishami ririho iterambere ryawe (raba \**A\**), urondere dosiye yawe y’ibirimwo (raba \**B\**), hanyuma ukande ku kimenyetso c’ikaramu kugira ngo ukomeze gusubiramwo.


![REVIEW](assets/fr/15.webp)



## 5. Gukora PR n’ishami ryiyemeje



- Niba vyose bisa neza kandi wararangije guhindura, kanda kuri buto ya Green "Create Pull Request".


![REVIEW](assets/fr/10.webp)



- Uzoca ushika kuri page ya PR. Igisabwa co gukura ni igisabwa co gutanga amahinduka yakozwe ku ishami ry’ububiko bwa kure kugira ngo bisubirwemwo kandi bishoboke ko bihurizwa hamwe mw’ishami ry’ingenzi.
- Wongereko umutwe n’insobanuro ngufi kuri PR yawe. Umutwe w’ikiganiro ukwiye guhura n’ikibazo uriko urakorako. Nk’akarorero, nimba umutwe w’ikinyamakuru ari `[UBUHINDURO] Ivyigwa CYP201 - FR`, koresha uwo mutwe nyene ku PR yawe (raba \**A\**). Mu ndondoro, vuga ikibazo gifitaniye isano ukoresheje umubare waco, ubanzirijwe na "Ifunga". Ivyo bizoca bifunga ikibazo iyo PR imaze guhurizwa hamwe (raba \**B\**).
- Niba vyose bisa neza, kanda kuri buto ya Green "Rema Igisabwa co Gukurura" (raba \**C\**).


![REVIEW](assets/fr/11.webp)


## 6. Amategeko yo gukurikiza igihe uhindura ibirimwo


Kugira ngo umenye vyinshi ku bijanye n'ingingo ngenderwako zo gukoresha igihe ukora ugukosora kuri Plan ₿ Academy mu bijanye n'ururimi canke amategeko y'imiterere, turagusavye urabe inyigisho "intererano-ugukosora-ingingo ngenderwako" muri iki gice nyene!


## 7. Rindira isubiramwo



- Murakoze cane, PR yanyu yarashitse! Ushobora gukurikirana ingene itera imbere mu gice ca "Ibisabwa vyo gukuraho" ku [bubiko bwa GitHub bwa Plan ₿ Academy](https://github.com/PlanB-Irezo/Bitcoin-ibirimwo-inyigisho/ibikuraho):


![REVIEW](assets/fr/12.webp)



- Murakoze cane ku ntererano yanyu! Inyuma y’ukwemererwa, umunyamuryango w’umugwi wa Plan ₿ Academy azoguhamagara kugira ngo akurungikire impembo yawe. \**Ivyishyurwa bikorwa gusa biciye ku Lightning Network\**.
- Niba ushaka gusubira gutangura, subira ku ntambwe ya 2 wohereze ubutumwa bushasha mu mugwi wa Telegram kugira ngo ukomeze gutanga umusanzu ku bindi birimwo. Raba neza ko uhuza Fork yawe imbere yo kurema ishami rishasha rikora. Kugira ngo ubikore, genda kuri paji y'intango ya Fork yawe, uhitemwo ishami ry'ingenzi ryitwa `dev` (raba \**A\**), hanyuma ukande kuri buto ya "Sync Fork" (raba \**B\**). Niba hariho ivyahinduwe kuri Fork yawe, buto ya "Vugurura ishami" izoboneka. Gusa ukande kuri yo kugira ngo ukore ivy’uguhindura.


![REVIEW](assets/fr/16.webp)



- Niba ufise ivyo uvuga, ibibazo, canke ivyo uvuga ku bijanye n’uwo mugambi, ntutinye kohereza ubutumwa [ku mugwi w’abatanga intererano](https://t.me/PlanBNetwork_ContentBuilder).
