ภารกิจของ PlanB คือการจัดหาทรัพยากรการศึกษาชั้นนำเกี่ยวกับ Bitcoin ในหลายภาษาให้มากที่สุด เนื้อหาทั้งหมดที่เผยแพร่บนเว็บไซต์เป็นโอเพ่นซอร์สและโฮสต์บน GitHub ซึ่งช่วยให้ทุกคนสามารถมีส่วนร่วมในการพัฒนาแพลตฟอร์มได้


นอกเหนือจากบทเรียนและการฝึกอบรมแล้ว Plan ₿ Academy ยังมีห้องสมุดขนาดใหญ่ที่มีเนื้อหาการศึกษาที่หลากหลายเกี่ยวกับ Bitcoin ซึ่งทุกคนสามารถเข้าถึงได้ [ในส่วน "BET" (_Bitcoin Educational Toolkit_)](https://planb.academy/resources/bet) ฐานข้อมูลนี้รวมถึงโปสเตอร์การศึกษา, มีม, โปสเตอร์โฆษณาชวนเชื่อที่มีอารมณ์ขัน, แผนภาพทางเทคนิค, โลโก้ และเครื่องมืออื่น ๆ สำหรับผู้ใช้ เป้าหมายของโครงการนี้คือการสนับสนุนบุคคลและชุมชนที่สอน Bitcoin ทั่วโลกโดยการจัดหาทรัพยากรภาพที่จำเป็นให้กับพวกเขา


คุณต้องการมีส่วนร่วมในการเพิ่มพูนฐานข้อมูลนี้ แต่ไม่รู้ว่าจะทำอย่างไรใช่ไหม? บทแนะนำนี้เหมาะสำหรับคุณ!


*เป็นสิ่งสำคัญที่เนื้อหาทั้งหมดที่รวมอยู่ในไซต์ต้องปราศจากสิทธิ์หรือเคารพใบอนุญาตของไฟล์ต้นฉบับ นอกจากนี้ ภาพทั้งหมดที่เผยแพร่บน Plan ₿ Academy จะมีให้ภายใต้ใบอนุญาต [CC-BY-SA](https://creativecommons.org/licenses/by-sa/4.0/).*

- กรอกไฟล์นี้ด้วยข้อมูลที่เกี่ยวข้องกับเนื้อหาของคุณโดยใช้เทมเพลตนี้:


```yaml
project:
type:
links:
download:
view:
- en:
tags:
-
-
contributors:
-
```


นี่คือรายละเอียดที่ต้องกรอกในแต่ละช่อง:



- `project`**: ระบุรหัสประจำตัวขององค์กรของคุณบน Plan ₿ Academy หากคุณยังไม่มีรหัส "project" สำหรับบริษัทของคุณ คุณสามารถสร้างได้โดยทำตามบทแนะนำนี้


https://planb.academy/tutorials/contribution/resource/add-builder-b5834c46-6dcc-4064-8d68-1ef529991d3d

หากคุณไม่มี คุณสามารถใช้ชื่อของคุณ นามแฝงของคุณ หรือชื่อบริษัทของคุณได้โดยไม่ต้องสร้างโปรไฟล์โครงการ



- `type`**: เลือกลักษณะของเนื้อหาของคุณจากสองตัวเลือกต่อไปนี้:
 - `Educational Content` สำหรับเนื้อหาทางการศึกษา
 - `เนื้อหาภาพ` สำหรับเนื้อหาประเภทอื่นๆ ที่หลากหลาย



- `links`**: ให้ลิงก์ไปยังเนื้อหาของคุณ คุณมีสองตัวเลือก:
 - หากคุณเลือกที่จะโฮสต์เนื้อหาของคุณโดยตรงบน GitHub ของ PlanB คุณจะต้องเพิ่มลิงก์ไปยังไฟล์นี้ในระหว่างขั้นตอนต่อไปนี้
 - หากเนื้อหาของคุณถูกโฮสต์ไว้ที่อื่น เช่น บนเว็บไซต์ส่วนตัวของคุณ โปรดระบุลิงก์ที่เกี่ยวข้องที่นี่:
     - `download`: ลิงก์สำหรับดาวน์โหลดเนื้อหาของคุณ
     - `view`: ลิงก์เพื่อดูเนื้อหาของคุณ (อาจเป็นลิงก์เดียวกับลิงก์ดาวน์โหลด) หากเนื้อหาของคุณมีหลายภาษา ให้เพิ่มลิงก์สำหรับแต่ละภาษา



- `แท็ก`**: เพิ่มสองแท็กที่เกี่ยวข้องกับเนื้อหาของคุณ ตัวอย่าง:
 - บิทคอยน์
 - เทคโนโลยี
 - เศรษฐกิจ
 - การศึกษา
 - มีม...



- `contributors`**: ระบุรหัสผู้ร่วมให้ข้อมูลของคุณหากคุณมี

- กรุณาใส่ข้อมูลในไฟล์ YAML โดยใช้เทมเพลตนี้:


```yaml
name:
description: |

```



- สำหรับคีย์ `name` คุณสามารถเพิ่มชื่อของเนื้อหาของคุณ;
- สำหรับคีย์ `description` คุณเพียงแค่ต้องเพิ่มย่อหน้าสั้น ๆ ที่อธิบายเนื้อหาของคุณ คำอธิบายต้องเป็นภาษาเดียวกับชื่อไฟล์ คุณไม่จำเป็นต้องแปลคำอธิบายนี้เป็นทุกภาษาที่รองรับบนเว็บไซต์ เนื่องจากทีม PlanB จะทำการแปลด้วยโมเดลของพวกเขา
