---
term: Cashu
definition: Uburyo bwa ecash chaumien bufunguye bukora kuri Bitcoin na Lightning ku bijanye n'izina n'ingatane.
---

Itegeko ry'amafaranga y'ubuhinga bwa none ry'i Chaumian, risa n'iry'ubuhinga bwa eCash bwa David Chaum, ariko rikoreshwa kuri Bitcoin na Lightning Network. Mu buryo bwihariye, Cashu yahumekewe n'ubundi buryo bw'amahera y'ubuhinga bwa none bwashikirijwe mu 1996 na David Wagner bwitwa "*amahera y'ubuhinga bwa none ata RSA*". Cashu ishobora gukoreshwa ku bikoresho vy’ububiko kugira ngo server ntishobore kumenya ba nyir’ayo mahera canke amakuru y’ivyo akora, gutyo bigatuma habaho ugutera imbere mu bijanye n’ubuzima bwite. Abakoresha barashobora generate Cashu tokens muri Exchange ku bitcoins, zishirwako umukono na server atazi uwuzikoresha. Ivyo bimenyetso birashobora rero guhererekanwa hagati y’ababikoresha ubwo nyene, mu bwiherero, kandi ata n’amahera.
