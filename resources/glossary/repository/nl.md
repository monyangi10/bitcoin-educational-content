---
term: REPOSITORIE
---

Een centrale datastructuur gebruikt in Git waar versiebeheerinformatie van een project wordt opgeslagen. Een repository bevat de complete geschiedenis van alle wijzigingen, branches en tags. Elk repository is een onafhankelijke verzameling van bestanden en mappen, vergezeld van een geschiedenis van commits, wat samenwerking en het volgen van veranderingen in de tijd mogelijk maakt. De Bitcoin core repository wordt bijvoorbeeld hier op GitHub gehost: https://github.com/Bitcoin/Bitcoin.


> ► *In het Engels wordt het een "repository" genoemd. Het is gebruikelijk om de afkorting "repo" te gebruiken om naar een Git repository te verwijzen.*
