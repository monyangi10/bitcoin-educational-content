---
term: REPOSYTORIUM
---

Centralna struktura danych używana w Git, w której przechowywane są informacje o wersjach projektu. Repozytorium zawiera pełną historię wszystkich zmian, gałęzi i tagów. Każde repozytorium jest niezależnym zbiorem plików i folderów, któremu towarzyszy historia commitów, umożliwiająca współpracę i śledzenie zmian w czasie. Na przykład, repozytorium Bitcoin Core jest hostowane na GitHub tutaj: https://github.com/Bitcoin/Bitcoin.


> w języku angielskim jest ono określane jako "repozytorium". Powszechne jest używanie skrótu "repo" w odniesieniu do repozytorium Git
