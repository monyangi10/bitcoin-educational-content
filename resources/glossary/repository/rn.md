---
term: UBUBIKO
---

Igishushanyo c'amakuru nyamukuru gikoreshwa muri Git aho amakuru y'uguhindura umugambi abikwa. Ububiko burimwo amateka yose y'amahinduka yose, amashami n'ibimenyetso. Buri bubiko ni ihuriro ryigenga ry’amadosiye n’amadosiye, riherekejwe n’amateka y’ibikorwa, bikaba bishoboza gukorana no gukurikirana amahinduka mu gihe. Nk'akarorero, ububiko bwa Bitcoin core burashirwa kuri GitHub hano: https://github.com/Bitcoin/Bitcoin.


> ► *Mu congereza, ryitwa "ububiko". Ni ibisanzwe gukoresha ijambo ry'inyongera "repo" kugira ngo yerekeze ku bubiko bwa Git.*
