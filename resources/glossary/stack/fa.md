---
term: پشته
definition: ساختار داده LIFO که در اسکریپت بیت‌کوین برای ذخیره و دستکاری عناصر موقت در حین اجرا استفاده می‌شود.
---

در زمینه زبان اسکریپتی که برای تعیین شرایط هزینه به UTXOهای Bitcoin استفاده می‌شود، پشته یک ساختار داده LIFO (*آخرین ورودی، اولین خروجی*) است که برای ذخیره موقت Elements در طول اجرای اسکریپت استفاده می‌شود. هر عملیات در اسکریپت این پشته‌ها را دستکاری می‌کند، جایی که Elements می‌تواند اضافه (*push*) یا حذف (*pop*) شود. اسکریپت‌ها از پشته‌ها برای ارزیابی عبارات، ذخیره متغیرهای موقت و مدیریت شرایط استفاده می‌کنند.


هنگام اجرای یک اسکریپت Bitcoin، می‌توان از ۲ پشته استفاده کرد: پشته اصلی و پشته جایگزین (alt). پشته اصلی برای اکثر عملیات اسکریپت استفاده می‌شود. این پشته جایی است که عملیات اسکریپت داده‌ها را اضافه، حذف یا دستکاری می‌کند. از سوی دیگر، پشته جایگزین برای ذخیره موقت داده‌ها در طول اجرای اسکریپت استفاده می‌شود. اپکدهای خاصی مانند `OP_TOALTSTACK` و `OP_FROMALTSTACK` به شما اجازه می‌دهند تا Elements را از پشته اصلی به پشته جایگزین و بالعکس منتقل کنید.


به عنوان مثال، زمانی که یک تراکنش تأیید می‌شود، امضاها و کلیدهای عمومی به پشته اصلی منتقل شده و توسط اپکدهای متوالی پردازش می‌شوند تا تأیید شود که امضاها با کلیدها و داده‌های تراکنش مطابقت دارند.
