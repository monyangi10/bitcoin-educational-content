---
term: 스택
definition: 비트코인 스크립트 실행 중에 임시 요소를 저장하고 조작하는 데 사용되는 LIFO(후입선출) 데이터 구조.
---

스택은 스크립트 실행 중에 임시 Elements을 저장하는 데 사용되는 LIFO(*Last In, First Out*) 데이터 구조로, Bitcoin UTXO에 지출 조건을 붙이는 데 사용되는 스크립팅 언어의 맥락에서 볼 때, 스택은 스크립트 실행 중에 임시 Elements을 저장하는 데 사용됩니다. 스크립트의 각 작업은 이러한 스택을 조작하며, 여기서 Elements을 추가(*push*)하거나 제거(*pop*)할 수 있습니다. 스크립트는 스택을 사용하여 표현식을 평가하고, 임시 변수를 저장하고, 조건을 관리합니다.


Bitcoin 스크립트를 실행할 때 메인 스택과 대체(대체) 스택의 두 가지 스택을 사용할 수 있습니다. 메인 스택은 대부분의 스크립트 작업에 사용됩니다. 이 스택에서 스크립트 작업이 데이터를 추가, 제거 또는 조작합니다. 반면 대체 스택은 스크립트 실행 중에 데이터를 임시로 저장하는 데 사용됩니다. OP_TOALTSTACK` 및 `OP_FROMALTSTACK`과 같은 특정 옵코드를 사용하면 Elements를 메인 스택에서 대체 스택으로 또는 그 반대로 전송할 수 있습니다.


예를 들어, 트랜잭션의 유효성이 검사되면 서명과 공개 키가 메인 스택에 푸시되고 연속적인 연산으로 처리되어 서명이 트랜잭션 키와 데이터와 일치하는지 확인합니다.
