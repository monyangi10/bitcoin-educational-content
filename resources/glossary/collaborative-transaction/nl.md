---
term: Collaborative transaction
definition: Bitcoin-transactie waarbij verschillende entiteiten als input betrokken zijn, zoals coinjoins.
---

Een Bitcoin transactie waarbij meerdere verschillende entiteiten betrokken zijn als input voor de transactie. Het is ook een transactiemodel dat kan worden gebruikt bij ketenanalyse. Het typische voorbeeld van een collaboratieve transactie is de CoinJoin, waarbij verschillende gebruikers gelijkwaardige bedragen als inputs samenvoegen om het volledige bedrag (min transactiekosten) als outputs van hetzelfde bedrag terug te krijgen, om tracering te voorkomen door de geschiedenis van de munten te doorbreken.
