---
term: HWI
definition: رابط استاندارد شده برای تعامل بین نرم‌افزارهای کیف پول و کیف پول‌های سخت‌افزاری.
---

مخفف "*Hardware Wallet Interface*". این یک Interface استاندارد است که تسهیل‌کننده‌ی یکپارچه‌سازی و تعامل بین نرم‌افزار مدیریت Bitcoin Wallet و کیف‌پول‌های سخت‌افزاری است. به‌طور خاص، HWI هم یک کتابخانه پایتون و هم یک ابزار خط فرمان است. این ابزار با استفاده از PSBTs (تراکنش‌های Bitcoin نیمه‌امضا شده) و احتمالاً Descriptors (توصیف‌گرهای اسکریپت خروجی)، ارتباط بین این اجزا را آسان‌تر می‌کند. این Interface ابتدا برای Bitcoin Core توسعه داده شد و سپس به یک استاندارد تبدیل شد که توسط اکثر نرم‌افزارهای Wallet استفاده می‌شود.
