---
term: Witness Transaction
---

In the RGB protocol, Witness Transaction refers to the Bitcoin transaction that closes Single-Use Seal around a message incorporating a Multi Protocol Commitment (MPC). This operation consists of either spending an existing UTXO, or creating a new one, in order to lock the contractual commitment written into the protocol. The Witness Transaction is therefore a On-Chain proof that the state of the RGB contract has been fixed at a specific point in time.
