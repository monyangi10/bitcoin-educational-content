---
term: MAPPER
---

Dalam konteks ilmu komputer, pemetaan mengacu pada proses mengasosiasikan Elements dari satu kumpulan data dengan Elements dari kumpulan data lainnya dengan cara yang sistematis. Asosiasi ini memungkinkan data di set pertama untuk diganti dengan data di set kedua, atau untuk bertransisi dari satu set ke set lainnya. Teknik ini sering digunakan dalam operasi transformasi data.
