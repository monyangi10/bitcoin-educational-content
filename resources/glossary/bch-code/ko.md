---
term: BCH 코드
---

데이터 시퀀스에서 오류를 감지하고 수정하는 데 사용되는 오류 수정 코드 클래스입니다. 즉, BCH 오류 수정 코드를 사용하면 전송된 데이터의 임의 오류를 식별하고 수정하여 정보가 목적지에 온전하게 도착하도록 할 수 있습니다. "BCH"라는 약어는 이 코드를 개발한 사람들의 이니셜에서 유래했습니다: 보스, 레이 차우두리, 호퀜헴입니다.


이 코드는 컴퓨팅, 특히 SSD, DVD 및 QR코드에서 널리 사용됩니다. 예를 들어, 이러한 오류 수정 코드 덕분에 QR코드가 부분적으로 가려져 있어도 그 안에 인코딩된 정보를 검색할 수 있습니다.


Bitcoin에서는 Bech32 및 Bech32m Address 형식(SegWit 이후 도입)의 체크섬에 BCH 코드가 사용됩니다. 이는 이전에 레거시 주소에 사용되었던 단순한 Hash 함수에 비해 체크섬 크기와 오류 감지 기능 간의 개선된 절충점을 나타냅니다. 그러나 Bitcoin 컨텍스트 내에서 BCH 코드는 오류 정정이 아닌 오류 감지에만 사용됩니다. 따라서 Bitcoin Wallet 소프트웨어는 수신 주소의 오류를 식별하고 사용자에게 경고하지만 잘못된 Address을 자동으로 수정하지는 않습니다. 오류 수정 기능을 통합하면 필연적으로 알고리즘의 오류 감지 기능이 저하될 수 있기 때문에 이러한 선택은 의도적인 것입니다.
