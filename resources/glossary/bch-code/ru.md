---
term: КОД BCH
---

Класс кодов коррекции ошибок, используемых для обнаружения и исправления ошибок в последовательности данных. Другими словами, коды коррекции ошибок BCH используются для поиска и исправления случайных ошибок в передаваемой информации, чтобы гарантировать, что она дойдет до места назначения в целости и сохранности. Аббревиатура "BCH" означает инициалы имен изобретателей этих кодов: Bose, Ray-Chaudhuri и Hocquenghem.


Этот инструмент используется во многих областях вычислительной техники, включая твердотельные накопители, DVD-диски и QR-коды. Например, благодаря этим кодам с коррекцией ошибок, даже если QR-код частично закрыт, все равно можно извлечь информацию, которая в нем закодирована.


Как часть Bitcoin, коды BCH используются для контрольной суммы в форматах Bech32 и Bech32m (после SegWit) Address. Они представляют собой лучший компромисс между размером контрольной суммы и возможностью обнаружения ошибок, чем простые функции Hash, ранее использовавшиеся для адресов Legacy. В контексте Bitcoin коды BCH используются только для обнаружения ошибок, а не для их исправления. Таким образом, программное обеспечение портфеля Bitcoin будет определять и сообщать пользователю о любых ошибках в принимаемом Address, но не будет автоматически менять неправильный Address. Такой выбор обусловлен тем, что интеграция исправления ошибок неизбежно влияет на возможности алгоритма обнаружения ошибок.
