---
term: OP_CHECKHASHVERIFY (CHV)

---

Opcode nshasha yashikirijwe mu 2012 muri BIP17 na Luke Dashjr yotanga ibikorwa nk'ivya `OP_EVAL` canke P2SH. Yari igamije gucapura iherezo rya `scriptSig`, kugereranya igisubizo n'isonga ry'ikirundo, no guhindura igikorwa kitagira akamaro iyo izo mpapuro zibiri zidahuye. Iyi opcode ntiyigeze ishirwa mu ngiro.
