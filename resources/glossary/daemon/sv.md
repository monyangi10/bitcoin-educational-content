---
term: Daemon
definition: Datorprogram som körs i bakgrunden, till exempel bitcoind för Bitcoin Core.
---

Typ av datorprogram som arbetar i bakgrunden, oberoende av användarkontroll. Dessa program utför uppgifter som hantering av nätverkstjänster, systemövervakning och svarar på förfrågningar utan att kräva direkt ingripande. I samband med Bitcoin Core kallas daemon för "bitcoind". Den representerar den version av Bitcoin Core som saknar användaren Interface och som enbart arbetar via kommandoraden.
