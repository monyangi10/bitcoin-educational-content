---
term: Wallet.dat

definition: File terenkripsi di Bitcoin Core yang menyimpan kunci pribadi dan informasi dompet pengguna.
---
File di Bitcoin Core yang menyimpan informasi tentang dompet pengguna, seperti kunci pribadi dan riwayat transaksi. File `wallet.dat` dienkripsi untuk memastikan keamanan dana. Sejak versi 0.16.0, file ini telah dipindahkan ke folder `/wallets`.
