---
term: Bosminer
definition: Software de minería de código abierto en Rust de Braiins, diseñado para ASICs con soporte de StratumV2.
---

Software Mining de código abierto desarrollado en Rust por Braiins (antes Slush Pool) como alternativa más moderna a CGMiner. BOSminer está diseñado específicamente para ASICs. Integrado en el ecosistema BraiinsOS, permite una gestión avanzada de ASIC, con funciones como overclocking, supervisión detallada del rendimiento y compatibilidad con StratumV2.
