---
term: Electrum server

definition: Pengindeks pada node lengkap yang memungkinkan dompet Electrum berfungsi tanpa blockchain lengkap.
---
Pengindeks yang diimplementasikan pada sebuah node Bitcoin yang memungkinkan perangkat lunak dompet Electrum (atau dompet lain yang mendukungnya) untuk beroperasi tanpa mengunduh seluruh _blockchain_ Bitcoin. Server Electrum menyimpan indeks lengkap alamat dan transaksi Bitcoin, sehingga memungkinkan kueri yang cepat dari perangkat lunak lain. Secara sederhana, Electrum Server bertindak sebagai jembatan antara perangkat lunak dompet dan node Bitcoin yang menyimpan _blockchain_.
