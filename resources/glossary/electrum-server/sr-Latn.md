---
term: ELECTRUM SERVER
---

Indekser implementiran na punom Bitcoin čvoru koji omogućava Electrum Wallet softveru (ili bilo kojem drugom Wallet koji ga podržava) da radi bez preuzimanja celog Bitcoin Blockchain. Electrum Server održava kompletan indeks Bitcoin adresa i transakcija, omogućavajući brze upite iz drugih softvera. Da pojednostavimo, Electrum Server deluje kao neka vrsta mosta između Wallet softvera i Bitcoin čvora koji drži Blockchain.
