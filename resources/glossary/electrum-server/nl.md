---
term: Electrum server
definition: Indexer op een full node waarmee Electrum-wallets kunnen functioneren zonder een volledige blockchain.
---

Een indexer die geïmplementeerd is op een volledig Bitcoin knooppunt, waardoor Electrum Wallet software (of elke andere Wallet die het ondersteunt) kan werken zonder de volledige Bitcoin Blockchain te downloaden. Een Electrum Server onderhoudt een complete index van Bitcoin adressen en transacties, zodat andere software deze snel kan opvragen. Om het te vereenvoudigen, fungeert een Electrum Server als een soort brug tussen Wallet software en een Bitcoin knooppunt dat de Blockchain bevat.
