---
term: SERWER ELECTRUM
---

Indeksator zaimplementowany na pełnym węźle Bitcoin, który pozwala oprogramowaniu Electrum Wallet (lub dowolnemu innemu Wallet, który go obsługuje) działać bez pobierania całego Bitcoin Blockchain. Serwer Electrum utrzymuje kompletny indeks adresów i transakcji Bitcoin, umożliwiając szybkie zapytania z innego oprogramowania. Upraszczając, Electrum Server działa jako rodzaj pomostu między oprogramowaniem Wallet a węzłem Bitcoin, który przechowuje Blockchain.
