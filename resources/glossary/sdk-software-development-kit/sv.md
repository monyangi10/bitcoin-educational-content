---
term: SDK (software development kit)
definition: Uppsättning programvaruverktyg för att utveckla applikationer på en specifik plattform.
---

En uppsättning programvaruverktyg som tillhandahåller de resurser som krävs för att utvecklare ska kunna skapa applikationer på en specifik plattform. Ett SDK innehåller bibliotek, utvecklingsguider, kodexempel och kompileringsprocesser. SDK:er underlättar och påskyndar utvecklingen genom att erbjuda återanvändbara moduler. På Bitcoin finns BDK (*Bitcoin Dev Kit*) och LDK (*Lightning Dev Kit*).


> ► På engelska kallas SDK:er ibland också "devkit" För mer information, se definitionerna av **BDK** och **LDK**.
