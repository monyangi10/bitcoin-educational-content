---
term: SDK (software development kit)
definition: Set softwaretools om applicaties te ontwikkelen op een specifiek platform.
---

Een set softwaretools die ontwikkelaars de nodige hulpmiddelen bieden om toepassingen op een specifiek platform te maken. Een SDK bevat bibliotheken, ontwikkelingsgidsen, codevoorbeelden en compilatieprocessen. SDK's vergemakkelijken en versnellen de ontwikkeling door herbruikbare modules aan te bieden. Op Bitcoin is er de BDK (*Bitcoin Dev Kit*) en de LDK (*Lightning Dev Kit*).


> in het Engels worden SDK's soms ook "devkit" genoemd Zie voor meer informatie de definities van **BDK** en **LDK**.
