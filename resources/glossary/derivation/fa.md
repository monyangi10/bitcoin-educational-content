---
term: اشتقاق
definition: فرآیند تولید کلیدهای فرزند از یک جفت کلید والد در یک کیف پول HD.
---

به فرآیند تولید جفت کلیدهای فرزند از یک جفت کلید والد (کلید خصوصی و کلید عمومی) و یک chain code درون یک Wallet قطعی و سلسله‌مراتبی (HD) اشاره دارد. این فرآیند امکان تقسیم‌بندی شاخه‌ها و سازماندهی یک Wallet به سطوح مختلف با تعداد زیادی جفت کلید فرزند را فراهم می‌کند که همگی با دانستن تنها اطلاعات بازیابی پایه (عبارت Mnemonic و هر passphrase احتمالی) قابل بازیابی هستند. برای استخراج یک کلید فرزند، از تابع `HMAC-SHA512` با chain code والد و الحاق کلید والد و یک شاخص ۳۲ بیتی استفاده می‌شود. دو نوع استخراج وجود دارد:


- اشتقاق عادی، که از کلید عمومی والد به عنوان مبنای تابع `HMAC-SHA512` استفاده می‌کند؛
- مشتق‌گیری سخت‌شده، که از کلید خصوصی والد به عنوان مبنای تابع `HMAC-SHA512` استفاده می‌کند؛


نتیجه HMAC-SHA512 به دو بخش تقسیم می‌شود: ۲۵۶ بیت اول به کلید فرزند (خصوصی یا عمومی پس از پردازش از طریق ECDSA) تبدیل می‌شود و ۲۵۶ بیت باقی‌مانده به chain code فرزند تبدیل می‌شود.
