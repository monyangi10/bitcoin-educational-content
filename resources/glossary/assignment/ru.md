---
term: Assignment
---

В логике протокола RGB Assignment является эквивалентом выхода транзакции, которая изменяет, обновляет или создает определенные свойства в состоянии Contract. Assignment состоит из двух Elements:




- Seal Definition (ссылка на конкретный UTXO);
- Owned State (данные, описывающие состояние, связанное с этим новым держателем).


Таким образом, Assignment указывает на то, что часть состояния (например, актив) теперь распределена между конкретными держателями, идентифицированными через Single-Use Seal, связанный с UTXO.
