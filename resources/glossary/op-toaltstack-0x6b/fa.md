---
term: OP_TOALTSTACK (0X6B)
definition: Opcode که بالای پشته اصلی را به پشته جایگزین منتقل می‌کند.
---

بالای پشته اصلی (*main stack*) را می‌گیرد و آن را به پشته جایگزین (*alt stack*) منتقل می‌کند. این opcode برای ذخیره موقت داده‌ها برای استفاده بعدی در اسکریپت استفاده می‌شود. آیتم منتقل شده به این ترتیب از پشته اصلی حذف می‌شود. سپس `OP_FROMALTSTACK` برای بازگرداندن آن به بالای پشته اصلی استفاده خواهد شد.
