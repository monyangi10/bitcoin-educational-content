---
term: Bcoin
definition: پیاده‌سازی جایگزین پروتکل Bitcoin که با JavaScript برای Node.js توسعه یافته و برای توسعه‌دهندگان و شرکت‌ها طراحی شده است.
---

اجرای جایگزین پروتکل Bitcoin که با استفاده از JavaScript برای Node.js توسعه یافته است. برخلاف Bitcoin Core، که اجرای اصلی نودهای Bitcoin است (حدود 99% از شبکه)، Bcoin بیشتر به نیازهای شرکت‌ها و توسعه‌دهندگانی که به دنبال ادغام ویژگی‌های خاص مرتبط با Bitcoin در سیستم‌های خود هستند، توجه دارد. Bcoin شامل مجموعه‌ای جامع از کتابخانه‌ها برای ادغام یک Full node، نود SPV، HD Wallet، ایندکسر و ویژگی‌های دیگر است.
