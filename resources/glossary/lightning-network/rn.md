---
term: Lightning network
definition: Protocol yisumbuye yemerera ihererekanya rya Bitcoin rinyaruka kandi ridahenze biciye muri canal z'uburihi.
---

Itegeko ry’i Layer ryubatswe hejuru y’itegeko rya Bitcoin, rigamije gutuma habaho ugucuruza vyihuta kandi bihenda. Bituma habaho uburyo bwo kwishura hagati y’abaje mu nama, aho amafaranga ashobora gukorwa hafi mu kanya nk’ako gukubita kandi ataco atanga, ataco umuntu akeneye kwandika kuri Blockchain. Ivyo bibanza birashobora kuguma vyuguruye hafi ibihe bidahera, kandi bisaba gusa gukoresha amafaranga kuri Blockchain iyo bifunguwe canke bifunze. Lightning Network ifise intumbero yo gutuma Bitcoin ishobora gukoreshwa neza no gutuma ishoboka ku bijanye n’amahera y’agaciro gatoyi.
