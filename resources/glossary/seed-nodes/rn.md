---
term: Seed nodes
definition: Urutonde rudahinduka rw'ama node ya Bitcoin y'icese rwashizwe mu buhinga bwa code kugira ngo habeho uburyo bwo kwinjira bwa mbere.
---

Urutonde rw'ibice vya Bitcoin vya bose, vyinjijwe mu buryo butaziguye muri kode y'inkomoko ya Bitcoin core (`Bitcoin/src/imbuto z'uruhererekane.h`). Iryo lisiti rikora nk’aho ama node mashasha ya Bitcoin yifatanya n’urubuga, ariko rikoreshwa gusa iyo imbuto za DNS zitatanga inyishu mu masegonda 60 y’ivyo zisavye. Muri ivyo, iyo node nshasha ya Bitcoin izokwifatanya n’izo node za seed kugira ngo ishinge uruja n’uruza rwa mbere n’urubuga no gusaba amaderesi IP y’izindi node. Intumbero nyamukuru ni ukuronka amakuru akenewe kugira ngo ukore Initial Block Download (IBD) no gukorana n’uruzitiro rufise ibikorwa vyinshi vyirundanijwe. Urutonde rw’ibihimba vya seed rurimwo ibihimba hafi 1000, vyamenyekanye hakurikijwe ingingo ngenderwako yashizweho na BIP155. Gutyo, amanode ya seed agereranya uburyo bwa gatatu bwo gufatanya n’urubuga ku manode ya Bitcoin, inyuma y’ugukoresha bishoboka kwa dosiye ya `peers.dat`, yaremwe n’iyo node ubwayo, no gusaba imbuto za DNS.


> ► *Iciyumviro, ama node ya seed ntashobora gutera urujijo n'"imbuto za DNS," ari zo buryo bwa kabiri bwo gushinga amahuzu.*
