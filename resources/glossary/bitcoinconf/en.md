---
term: bitcoin.conf
definition: Configuration file allowing for the customization of Bitcoin Core node parameters.
---

Configuration file used to customize the operation of a Bitcoin node running the Bitcoin Core client. Located in the Bitcoin Core data directory, this text file allows node operators to define various parameters and options that influence the node’s behavior.
Parameters configurable in `bitcoin.conf` include: the mempool size, network connection restrictions, minimum transaction relay fees, and various security and performance options.
