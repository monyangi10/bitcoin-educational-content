---
term: RPC (fjärranropsprotokoll)
definition: Protokoll som gör det möjligt att köra kommandon på distans på en Bitcoin-nod.
---

Ett datorprotokoll som gör det möjligt för ett program att utföra en procedur på en annan fjärrdator som om den utfördes lokalt. Specifikt i Bitcoin-sammanhang används det för att göra det möjligt för applikationer att interagera med bitcoind. Det kan användas för att utföra kommandon på en Bitcoin-nod, t.ex. skicka transaktioner, hantera plånböcker eller få tillgång till information på Blockchain. Säkerheten för denna interaktion säkerställs genom autentisering via en `.cookie`-fil eller referenser, så att endast auktoriserade klienter kan utföra RPC:er på noden.
