---
term: COMPLETE KNOT
---

A computer running a Bitcoin protocol client, which downloads, verifies and stores the entire Blockchain, i.e. the complete transaction history from the Genesis block onwards. A complete node autonomously verifies all transactions and blocks according to Bitcoin consensus rules. This type of node therefore provides the highest level of verification for its user, and minimizes the need to rely on a third party. The full node requires more storage resources, computing power, RAM and bandwidth than a lightweight node (SPV).
