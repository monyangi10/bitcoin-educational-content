---
term: کد زنجیره
definition: مقدار نمک (salt) رمزنگاری ۲۵۶ بیتی که در اشتقاق HD کلیدهای کیف پول استفاده می‌شود.
---

در زمینه اشتقاق سلسله مراتبی تعیین‌پذیر (HD) کیف‌پول‌های Bitcoin، chain code یک مقدار نمک رمزنگاری ۲۵۶ بیتی است که برای generate کلیدهای فرزند از یک کلید والد، طبق استاندارد BIP32 استفاده می‌شود. chain code در ترکیب با کلید والد و شاخص فرزند به‌طور تعیین‌پذیر generate یک جفت کلید جدید (کلید خصوصی و کلید عمومی) بدون افشای کلید والد یا سایر کلیدهای فرزند مشتق‌شده استفاده می‌شود.


بنابراین، برای هر جفت کلید یک chain code منحصر به فرد وجود دارد. chain code یا از طریق هش کردن seed مربوط به Wallet و گرفتن نیمه راست بیت‌ها به دست می‌آید. در این حالت، به عنوان یک chain code اصلی شناخته می‌شود که با کلید خصوصی اصلی مرتبط است. به طور متناوب، می‌توان آن را با هش کردن یک کلید والد با chain code والد و یک شاخص به دست آورد، سپس بیت‌های راست را نگه داشت. این سپس به عنوان یک chain code فرزند شناخته می‌شود.


بدون دانستن chain code مرتبط با هر جفت والدین، استخراج کلیدها غیرممکن است. این فرآیند داده‌های شبه‌تصادفی را به فرآیند استخراج وارد می‌کند تا اطمینان حاصل شود که تولید کلیدهای رمزنگاری برای مهاجمان غیرقابل پیش‌بینی باقی می‌ماند، در حالی که برای دارنده Wallet تعیین‌پذیر است.
