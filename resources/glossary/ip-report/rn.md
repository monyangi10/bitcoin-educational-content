---
term: RAPORO ya IP
---

Buto iri kuri ASICs kugira ngo ubone ningoga IP Address y’imashini ku rubuga rw’aho hantu. Kugira ngo ukoreshe iyo nzira, kanda kuri buto ya *IP Report* iri kuri ASIC mu masegonda makeyi. Hanyuma, hakoreshejwe porogarama zidasanzwe ziri kuri orodinateri ifatanye n’urwo rubuga nyene, IP Address ya Miner izogaragara ku rubuga. Iyi buto ituma vyoroha kuronka imashini iri ku rubuga rw’aho uba iyo ufise imashini nyinshi.
