---
term: Ip report
definition: دکمه‌ای روی ASICها که امکان یافتن سریع آدرس IP آن‌ها را در شبکه محلی فراهم می‌کند.
---

دکمه‌ای که بر روی ASICها وجود دارد تا به سرعت IP دستگاه Address را در شبکه محلی پیدا کند. برای استفاده از این ویژگی، دکمه *IP Report* را بر روی ASIC برای چند ثانیه فشار دهید. سپس با استفاده از نرم‌افزار تخصصی بر روی یک کامپیوتر که به همان شبکه متصل است، IP دستگاه Miner Address بر روی صفحه نمایش داده خواهد شد. این دکمه پیدا کردن یک دستگاه در شبکه محلی را زمانی که چندین دستگاه دارید آسان می‌کند.
