---
term: IP-RAPPORT
---

Knop aanwezig op ASIC's om snel het IP van de machine te vinden Address op het lokale netwerk. Om deze functie te gebruiken, drukt u enkele seconden op de knop *IP Report* op de ASIC. Vervolgens wordt met behulp van gespecialiseerde software op een computer die op hetzelfde netwerk is aangesloten, het IP Address van de machine op het scherm weergegeven. Vervolgens wordt met behulp van speciale software op een computer die op hetzelfde netwerk is aangesloten, de IP Address van de Miner op het scherm weergegeven. Deze knop maakt het gemakkelijk om een machine op het lokale netwerk te lokaliseren als je meerdere machines hebt.
