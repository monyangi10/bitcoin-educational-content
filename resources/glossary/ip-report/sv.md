---
term: Ip report
definition: Knapp på ASICs som gör det möjligt att snabbt hitta deras IP-adress på det lokala nätverket.
---

Knapp som finns på ASIC för att snabbt hitta maskinens IP Address i det lokala nätverket. För att använda den här funktionen, tryck på *IP Report*-knappen på ASIC i några sekunder. Med hjälp av en specialiserad programvara på en dator som är ansluten till samma nätverk visas sedan Miner:s IP Address på skärmen. Den här knappen gör det enkelt att lokalisera en maskin i det lokala nätverket när du har flera maskiner.
