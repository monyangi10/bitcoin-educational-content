---
term: Ip report
definition: ASIC-laitteissa oleva painike, jonka avulla niiden IP-osoite löytyy nopeasti paikallisverkosta.
---

ASIC-ohjaimissa oleva painike koneen IP Address:n nopeaa löytämistä varten lähiverkossa. Voit käyttää tätä toimintoa painamalla ASIC:n *IP Report*-painiketta muutaman sekunnin ajan. Sen jälkeen Miner:n IP Address näkyy näytöllä samaan verkkoon liitetyn tietokoneen erikoisohjelmiston avulla. Tämän painikkeen avulla on helppo paikallistaa kone lähiverkossa, kun käytössä on useita koneita.
