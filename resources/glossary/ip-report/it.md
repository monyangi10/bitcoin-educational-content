---
term: RAPPORTO IP
---

Pulsante presente sugli ASIC per trovare rapidamente l'IP Address della macchina sulla rete locale. Per utilizzare questa funzione, premere il pulsante *IP Report* sul ASIC per alcuni secondi. Quindi, utilizzando un software specializzato su un computer collegato alla stessa rete, l'IP Address del Miner verrà visualizzato sullo schermo. Questo pulsante semplifica la localizzazione di una macchina sulla rete locale quando si dispone di più macchine.
