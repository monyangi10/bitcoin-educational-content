---
term: RELATÓRIO DE PI
---

Botão presente nos ASICs para encontrar rapidamente o IP da máquina Address na rede local. Para utilizar esta funcionalidade, prima o botão *IP Report* no ASIC durante alguns segundos. Depois, utilizando um software especializado num computador ligado à mesma rede, o IP Address do Miner será apresentado no ecrã. Este botão facilita a localização de uma máquina na rede local quando se tem várias máquinas.
