---
term: Ip report
definition: Tombol pada ASIC yang memungkinkan alamat IP-nya ditemukan dengan cepat di jaringan lokal.
---

Tombol yang ada pada ASIC untuk menemukan IP Address mesin dengan cepat pada jaringan lokal. Untuk menggunakan fitur ini, tekan tombol *IP Report* pada ASIC selama beberapa detik. Kemudian, dengan menggunakan perangkat lunak khusus pada komputer yang terhubung ke jaringan yang sama, Alamat IP penambang akan ditampilkan di layar. Tombol ini memudahkan untuk menemukan mesin pada jaringan lokal bila Anda memiliki beberapa mesin.
