---
term: LAPORAN IP
---

Tombol yang ada pada ASIC untuk menemukan IP Address mesin dengan cepat pada jaringan lokal. Untuk menggunakan fitur ini, tekan tombol *Laporan IP* pada ASIC selama beberapa detik. Kemudian, dengan menggunakan perangkat lunak khusus pada komputer yang terhubung ke jaringan yang sama, IP Address Miner akan ditampilkan di layar. Tombol ini memudahkan untuk menemukan mesin pada jaringan lokal bila Anda memiliki beberapa mesin.
