---
term: CONDENSADO
---

Em criptografia, significa o resultado (ou saída) produzido pela aplicação de uma função criptográfica Hash a um conjunto de dados. O condensado é uma sequência de caracteres de tamanho fixo, geralmente representada como uma série de números e letras em notação hexadecimal (base 16). Este resultado tem a particularidade de ser quase único e específico para os dados de entrada, de modo que uma alteração mínima na entrada produzirá um condensado completamente diferente. As funções criptográficas Hash são concebidas para serem unidireccionais e resistentes a colisões, tornando muito difícil recuperar os dados originais a partir do condensado, ou encontrar duas entradas distintas que produzam o mesmo condensado.
