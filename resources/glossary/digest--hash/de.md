---
term: CONDENSAT
---

In der Kryptografie das Ergebnis (oder die Ausgabe), das durch Anwendung einer kryptografischen Hash-Funktion auf einen Datensatz entsteht. Das Kondensat ist eine Zeichenfolge fester Größe, die im Allgemeinen als eine Reihe von Zahlen und Buchstaben in hexadezimaler Notation (Basis 16) dargestellt wird. Dieses Ergebnis hat die Besonderheit, dass es nahezu einzigartig und spezifisch für die Eingabedaten ist, so dass eine minimale Änderung der Eingabe ein völlig anderes Kondensat ergibt. Kryptografische Hash-Funktionen sind so konzipiert, dass sie unidirektional und kollisionssicher sind, so dass es sehr schwierig ist, die ursprünglichen Daten aus dem Kondensat wiederherzustellen oder zwei unterschiedliche Eingaben zu finden, die das gleiche Kondensat ergeben.
