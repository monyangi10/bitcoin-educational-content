---
term: BIP0084
definition: Derivation standard for native SegWit addresses (bc1q...) using index 84' in the derivation path.
---

Defines the standard for deriving SegWit V0 addresses (`bc1q...`) within a deterministic and hierarchical wallet. It specifies the use of index `84'` at the `purpose` depth of the HD wallet for P2WPHK script types.
