---
term: BIP0078
definition: Payjoin protocol improving privacy by involving the recipient in the transaction inputs.
---

Introduces a protocol for using Payjoin on Bitcoin, a transaction structure that improves payment privacy by involving the recipient as an input contributor alongside the payer. 
This BIP is inspired by BIP79, which had already proposed a similar concept. 
However, these two BIPs are not the origin of the Payjoin idea.
The Stowaway implementation by Samourai Wallet already existed beforehand, and this transaction structure was first mentioned by LaurentMT in 2015 under the name “steganographic transaction.”
