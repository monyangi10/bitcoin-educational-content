---
term: BIP0071
definition: Definition of the MIME type for BIP70 payment messages. Never widely adopted.
---

Defines a MIME (*Multipurpose Internet Mail Extensions*) media type, in accordance with the RFC 2046 standard, for bitcoin payment request messages in BIP70. 
MIME is an Internet standard that extends the format of electronic messages, enabling the structured transfer of various types of data. 
By defining a specific MIME type for payment messages, BIP71 ensures that wallet software sending payment protocol messages via email or HTTP follows Internet standards for proper message encapsulation.

However, because this enhancement was bundled with BIP70, it was never widely adopted.
