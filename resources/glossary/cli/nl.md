---
term: CLI
definition: Command-line interface waarmee interactie met software mogelijk is via tekstcommando's in een terminal.
---

Acroniem voor "Command Line Interface," een methode voor interactie met software die gebaseerd is op het invoeren van tekstcommando's in een terminal of console. De CLI onderscheidt zich van de GUI (Graphical User Interface), die aanwijsinteracties (met een muis) en interactieve visuele Elements biedt.
