---
term: Cgminer
definition: Open-source těžařský software pro CPU a GPU, převzatý do projektu BOSminer od Braiins.
---

Open-source software Mining původně vyvinutý pro CPU a GPU. CGminer umožňuje uživatelům spravovat a optimalizovat operace Mining pomocí funkcí, jako je vícevláknové zpracování, správa více fondů a sledování výkonu v reálném čase. Tento software již dnes není skutečně udržován, ale převzal jej open-source projekt BOSminer společnosti Braiins.
