---
term: Cgminer
definition: Software de minería de código abierto para CPU y GPU, retomado en el proyecto BOSminer de Braiins.
---

Software Mining de código abierto desarrollado originalmente para la CPU y la GPU. CGminer permite a los usuarios gestionar y optimizar sus operaciones de Mining, con funciones como el multihilo, la gestión de múltiples pools y la supervisión del rendimiento en tiempo real. En la actualidad, el software ya no se mantiene realmente, sino que ha pasado a manos del proyecto de código abierto BOSminer de Braiins.
