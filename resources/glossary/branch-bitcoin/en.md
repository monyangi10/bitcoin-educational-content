---
term: BRANCH (Bitcoin)
---

In Bitcoin, a branch is a sequence of valid blocks in which each new block is linked to the previous one. Blockchain represents the branch with the most accumulated work, but there may be competing branches resulting from branching. Each branch shares at least one common block with the main branch, namely the Genesis block. The part of the blocks specific to the branch is sometimes called the "segment". When two competing branches are present, honest nodes synchronize on the one with the most accumulated work. A branch is sometimes called "strong" if it has more accumulated work than another, and "weak" otherwise.
