---
term: Bitcoin network
definition: Infrastructure of interconnected nodes running the Bitcoin protocol to broadcast and verify transactions and blocks.
---

Refers to the global infrastructure of the Bitcoin system. The network consists of all the nodes (computers) running software  that implements the Bitcoin protocol and connects to each other. Each node communicates peer-to-peer with other nodes in order to: download and verify the blockchain, validate and broadcast new blocks, and validate and broadcast new transactions.
