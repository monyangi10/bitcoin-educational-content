---
term: Addr.dat

definition: Un vecchio file in Bitcoin Core che memorizzava informazioni su peer di rete. Sostituito da peers.dat a partire dalla versione 0.7.0.
---
Nome del vecchio file utilizzato in Bitcoin Core per memorizzare le informazioni sui peer (cioè i nodi) della rete con cui il nodo dell'utente ha interagito o potrebbe interagire. Dalla versione 0.7.0 questo file è stato sostituito dal file peers.dat.
