---
term: Addr.dat

definition: File lama di Bitcoin Core yang menyimpan informasi tentang peer jaringan. Digantikan oleh peers.dat sejak versi 0.7.0.
---
Nama file lama yang digunakan di Bitcoin Core untuk menyimpan informasi tentang peers (_node_) di jaringan yang berinteraksi dengan _node_ pengguna atau yang berpotensi berinteraksi dengan _node_ pengguna. File ini telah digantikan oleh file peers.dat sejak versi 0.7.0.
