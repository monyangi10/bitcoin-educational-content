---
term: Addr.dat
definition: An old file in Bitcoin Core that stored information about network peers. Replaced by peers.dat since version 0.7.0.
---

Name of the old file used in Bitcoin Core to store information about peers (i.e., nodes) in the network that the user's node has interacted with or could potentially interact with. This file was replaced by the peers.dat file starting with version 0.7.0.
