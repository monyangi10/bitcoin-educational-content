---
term: کلید اصلی
definition: کلید خصوصی مشتق شده از seed که به عنوان نقطه شروع برای تمام کلیدهای یک کیف پول HD عمل می‌کند.
---

در زمینه کیف‌پول‌های HD (Deterministic Hierarchical)، کلید خصوصی اصلی یک کلید خصوصی منحصر به فرد است که از Wallet's seed مشتق شده است. برای به‌دست‌آوردن کلید اصلی، تابع `HMAC-SHA512` بر روی seed اعمال می‌شود و از کلمات "*Bitcoin seed*" به‌صورت دقیقاً به‌عنوان کلید استفاده می‌شود. نتیجه این عملیات یک خروجی 512 بیتی تولید می‌کند که 256 بیت اول آن کلید اصلی را تشکیل می‌دهد و 256 بیت باقی‌مانده chain code اصلی را تشکیل می‌دهند. کلید اصلی و chain code اصلی به‌عنوان نقطه شروع برای مشتق‌سازی تمام کلیدهای خصوصی و عمومی فرزند در ساختار درختی HD Wallet عمل می‌کنند. بنابراین، کلید خصوصی اصلی در عمق 0 مشتق‌سازی قرار دارد.
