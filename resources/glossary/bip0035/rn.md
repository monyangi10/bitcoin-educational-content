---
term: BIP0035
definition: Ishirwaho ryemera ama node gusangira amakuru y'amapool yayo (itunshwa ririndiriye) n'abandi bakoresha mu muhora.
---

Iciyumviro cemerera node ya Bitcoin gusangira amakuru yerekeye Mempool yayo, ikidengeri c’ibikorwa birindiriye kwemezwa.

Ivyo bifasha abandi bari muri iyo nama kuronka amakuru y’igihe nyaco ku bikorwa bitaremejwe mu kohereza ubusabe bwihariye ku nzira. Imbere ya BIP35, ama node yashobora gutanga amakuru gusa ku bikorwa vyari vyaramaze kwemezwa.

Ivyo bifasha ama wallet ya SPV mu kubaha uburenganzira bwo kuronka amakuru y’ibikorwa bitaremejwe, bifasha abacukuzi kwirinda kubura ibikorwa vy’amahera menshi inyuma yo gusubira gutangura, kandi bikorohereza isesengura rya Mempool n’ibikorwa vyo hanze.
