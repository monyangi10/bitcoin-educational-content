---
term: Bitcoin Core
definition: Referencyjne oprogramowanie open-source implementujące protokół Bitcoin, służące jako pełny węzeł i portfel.
---

Bitcoin Core jest referencyjnym oprogramowaniem open-source dla systemu Bitcoin i jest główną implementacją protokołu Bitcoin do tej pory. Jest rozwijany i utrzymywany przez dużą grupę wolontariuszy. Początkowo nazwany "Bitcoin Qt", jest trzecim klientem w historii Bitcoin po Bitcoin autorstwa Satoshi Nakamoto i bitcoind. Został opracowany na podstawie oryginalnego kodu Satoshi i wprowadził graficznego użytkownika Interface dla użytkownika. Co więcej, nawet dzisiaj graficzny Interface Bitcoin Core nazywa się `Bitcoin-qt`. Został on spakowany z `bitcoind` od wersji 0.5. Oprogramowanie Bitcoin Core służy kilku celom. Przede wszystkim działa jako klient Full node. Bitcoin Core zawiera również Wallet dla użytkowników, którzy chcą przechowywać, zarządzać i przeprowadzać transakcje bezpośrednio z Bitcoin Core.
