---
term: Bitcoin 核心
---

Bitcoin Core 是 Bitcoin 系統的參考開放原始碼軟體，也是迄今為止 Bitcoin 通訊協定的主要實作。它由一大群志願貢獻者開發和維護。最初命名為「Bitcoin Qt」，是 Bitcoin 歷史上繼 Satoshi Nakamoto 的 Bitcoin 和 bitcoind 之後的第三個用戶端。它由 Satoshi 的原始程式碼開發，並為使用者引入了圖形化使用者 Interface。此外，即使在今天，Bitcoin Core 的圖形化 Interface 仍被稱為「Bitcoin-qt」。自版本 0.5 起，它就被打包在 `bitcoind` 中。Bitcoin Core 軟體有幾個用途。首先，它充當 Full node 用戶端。Bitcoin Core 也包含一個 Wallet，供希望直接使用 Bitcoin Core 儲存、管理和進行交易的使用者使用。
