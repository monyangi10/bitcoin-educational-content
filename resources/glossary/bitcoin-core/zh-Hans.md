---
term: Bitcoin Core

definition: 实现比特币协议的开源参考软件，充当全节点和钱包。
---
Bitcoin Core 是比特币系统的参考开源软件，也是迄今为止比特币协议的主要实施方式。它由一大批志愿贡献者开发和维护。它最初被命名为 “Bitcoin Qt”，是比特币历史上继中本聪的比特币和 Bitcoind 之后的第三个客户端。它是根据中本聪的原始代码开发的，并为用户引入了图形化用户界面。此外，时至今日，Bitcoin Core 的图形界面仍被称为 “bitcoin-qt”。从 0.5 版开始，它便作为 `bitcoind` 的一部分一同发布。Bitcoin Core 软件有几个用途。首先，它是一个完整的节点客户端。Bitcoin Core 还包括一个钱包，供想要直接使用 Bitcoin Core 存储、管理和进行交易的用户使用。
