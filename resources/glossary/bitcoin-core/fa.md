---
term: Bitcoin Core
definition: نرم‌افزار مرجع متن‌باز که پروتکل Bitcoin را پیاده‌سازی می‌کند و به عنوان یک نود کامل و کیف پول عمل می‌کند.
---

Bitcoin Core نرم‌افزار مرجع متن‌باز برای سیستم Bitcoin است و تا به امروز اصلی‌ترین پیاده‌سازی پروتکل Bitcoin محسوب می‌شود. این نرم‌افزار توسط گروه بزرگی از مشارکت‌کنندگان داوطلب توسعه و نگهداری می‌شود. در ابتدا با نام "Bitcoin Qt" شناخته می‌شد و سومین کلاینت در تاریخچه Bitcoin پس از Bitcoin توسط Satoshi Nakamoto و bitcoind است. این نرم‌افزار از کد اصلی Satoshi توسعه یافته و یک رابط کاربری گرافیکی Interface برای کاربر معرفی کرده است. علاوه بر این، حتی امروز نیز رابط گرافیکی Interface از Bitcoin Core به نام `Bitcoin-qt` شناخته می‌شود. از نسخه 0.5 به بعد، با `bitcoind` بسته‌بندی شده است. نرم‌افزار Bitcoin Core چندین هدف را دنبال می‌کند. اول و مهم‌تر از همه، به عنوان یک کلاینت Full node عمل می‌کند. Bitcoin Core همچنین شامل یک Wallet برای کاربرانی است که می‌خواهند به طور مستقیم با Bitcoin Core تراکنش‌ها را ذخیره، مدیریت و انجام دهند.
