---
term: Bitcoin core
---

Bitcoin core is de open-source referentiesoftware voor het Bitcoin systeem en is de belangrijkste implementatie van het Bitcoin protocol tot nu toe. Het wordt ontwikkeld en onderhouden door een grote groep vrijwilligers. Aanvankelijk heette het "Bitcoin Qt", en het is de derde client in de geschiedenis van Bitcoin, na Bitcoin van Satoshi Nakamoto, en bitcoind. Het werd ontwikkeld op basis van Satoshi's originele software. Het werd ontwikkeld uit de oorspronkelijke code van Satoshi en introduceerde een grafische gebruiker Interface voor de gebruiker. Bovendien heet de grafische Interface van Bitcoin core vandaag de dag nog steeds `Bitcoin-qt`. Het is verpakt met `bitcoind` sinds versie 0.5. Bitcoin core software dient verschillende doelen. Eerst en vooral fungeert het als Full node client. Bitcoin core bevat ook een Wallet voor gebruikers die willen opslaan, beheren en transacties direct met Bitcoin core willen uitvoeren.
