---
term: Wallet 가져오기 형식(WIF)
---

다른 지갑 간에 더 쉽게 가져오거나 내보낼 수 있도록 Bitcoin 개인키를 인코딩하는 방법입니다. WIF는 버전에 대한 정보, 해당 공개 키의 압축, 입력 시 오류 감지를 위한 체크섬을 포함하는 `Base58Check` 인코딩을 기반으로 합니다. WIF 개인키는 압축되지 않은 키의 경우 `5`, 압축된 키의 경우 `K`와 `L` 문자로 시작하며, 원본 개인키를 재구성하는 데 필요한 모든 문자를 포함합니다. WIF 형식은 서로 다른 Wallet 소프트웨어 간에 개인 키를 전송할 수 있는 표준화된 수단을 제공합니다.
