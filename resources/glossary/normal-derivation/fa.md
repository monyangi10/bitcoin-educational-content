---
term: مشتق‌گیری عادی
definition: فرآیند تولید کلیدهای فرزند با استفاده از کلید عمومی والد در کیف‌پول‌های HD.
---

فرآیند تولید کلیدهای فرزند در کیف‌پول‌های HD. مشتق‌گیری معمولی از کلید عمومی والد به عنوان ورودی برای تابع `HMAC-SHA512` استفاده می‌کند، که امکان تولید کلیدهای عمومی فرزند از کلید عمومی والد و chain code والد را فراهم می‌کند. این فرآیند شامل الحاق کلید عمومی والد و یک شاخص کمتر از $2^{31}$ است، که به دنبال آن `HMAC-SHA512` با chain code والد اعمال می‌شود. نتیجه به دو بخش تقسیم می‌شود: 256 بیت اول به کلید خصوصی والد اضافه می‌شود تا کلید خصوصی فرزند به دست آید، در حالی که 256 بیت باقی‌مانده chain code فرزند را تشکیل می‌دهد. این روش تضمین می‌کند که کلید عمومی توسعه‌یافته می‌تواند برای استخراج کلیدهای عمومی فرزند استفاده شود. در مشتق‌گیری استاندارد، مشتق‌گیری معمولی در تمام سطوح مشتق‌گیری از عمق حساب استفاده می‌شود. در نشانه‌گذاری مسیرهای مشتق‌گیری، یک مشتق‌گیری معمولی زمانی شناسایی می‌شود که فقط شاخص بدون هیچ آپاستروف `'` وجود دارد.
