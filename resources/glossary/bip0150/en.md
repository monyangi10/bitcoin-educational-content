---
term: BIP0150
definition: Peer-to-peer authentication mechanism to strengthen security and restrict access to certain node services.
---

BIP150 proposes a peer authentication mechanism for the Bitcoin network to improve security and verify node ownership. 
This mechanism allows node operators to restrict access to certain services or grant priority data flows only to specific peers, using mutual authentication to prevent man-in-the-middle (MITM) attacks. 
This BIP will remain in draft status, but it will serve as a lesson for BIP324 (*P2P transport V2*), which is now optionally implemented in Bitcoin Core.
