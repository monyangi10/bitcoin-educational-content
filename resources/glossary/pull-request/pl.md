---
term: ŻĄDANIE ŚCIĄGNIĘCIA
---

W kontekście GitHub i innych platform hostujących kod, Pull Request to wniosek złożony przez współtwórcę o zintegrowanie zmian z gałęzi Fork do gałęzi głównego repozytorium. Wywołuje to przegląd kodu i dyskusję przed potencjalnym scaleniem zmian. Proces ten jest szeroko stosowany w rozwoju implementacji węzłów Bitcoin, zwłaszcza Bitcoin Core.


> termin "Pull Request" jest często skracany jako "PR"
