---
term: GUKURA IKIBAZO
---

Mu bijanye na GitHub n’izindi nzira zo kwakira kode, Pull Request ni igisabwa gikorwa n’uwutanga umusanzu kugira ngo ashiremwo amahinduka yiwe avuye mw’ishami rya Fork yabo mu ishami ry’ububiko nyamukuru. Bituma habaho isubiramwo ry'amategeko n'ibiganiro imbere y'uko amahinduka ashobora guhurizwa hamwe. Iyi nzira ikoreshwa cane mu gutegura ibikorwa vya Bitcoin, cane cane Bitcoin core.


> ► *Ijambo "Isaba ry'ugukura" rikunda gufatwa mu buryo bugufi ngo "PR".*
