---
term: VERZOEK
---

Binnen de context van GitHub en andere code hosting platformen, is een Pull Request een verzoek van een bijdrager om hun wijzigingen van een branch van hun Fork te integreren in een branch van het hoofdrepository. Het triggert een code review en discussie voordat de wijzigingen mogelijk samengevoegd kunnen worden. Dit proces wordt veel gebruikt bij de ontwikkeling van implementaties van Bitcoin knooppunten, met name Bitcoin core.


> ► *De term "Pull Request" wordt vaak afgekort als "PR".*
