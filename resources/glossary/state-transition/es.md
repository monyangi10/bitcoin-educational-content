---
term: State Transition
---

En el protocolo RGB, State Transition es la operación utilizada para cambiar el estado de un Contract a una nueva configuración. Puede modificar tanto los datos de Global State como los de los estados propios. Cada transición se comprueba estrictamente según las reglas definidas en Schema de Contract, que garantiza que las modificaciones cumplen las restricciones establecidas en Genesis. Esta operación se ancla en Blockchain Bitcoin a través de un Commitment.
