---
term: State Transition
---

No protocolo RGB, State Transition é a operação utilizada para alterar o estado de um Contract para uma nova configuração. Pode modificar os dados do Global State e dos Owned States. Cada transição é rigorosamente verificada de acordo com as regras definidas no Schema do Contract, o que garante que as modificações estão em conformidade com as restrições estabelecidas no Genesis. Esta operação é ancorada no Blockchain Bitcoin através de um Commitment.
