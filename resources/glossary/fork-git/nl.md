---
term: Fork (GIT)
---

Binnen de context van Git vertegenwoordigt het het aanmaken van een kopie van een bestaande repository op een nieuwe account, waardoor de gebruiker het project kan aanpassen, testen of ontwikkelen onafhankelijk van de originele repository. Vorken maken open-source samenwerking en bijdragen aan projecten mogelijk zonder de bronrepository te beïnvloeden.
