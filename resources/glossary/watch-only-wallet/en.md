---
term: Watch-only wallet
definition: Wallet allowing to view transactions and balance without possessing the private keys to spend funds.
---

A Watch-only wallet is a type of software that allows a user to view transactions associated with a specific Bitcoin key or set of keys, without owning the corresponding private keys. It provides visibility on the balance and history of transactions, without allowing the user to spend the funds in the wallet. For example, the Sentinel application is a Watch-only wallet.
