---
term: Watch-only wallet
---

Een Watch-only wallet is een type software waarmee een gebruiker transacties kan bekijken die geassocieerd zijn met een specifieke Bitcoin sleutel of set sleutels, zonder de bijbehorende privésleutels te bezitten. Het geeft inzicht in het saldo en de geschiedenis van transacties, zonder dat de gebruiker het geld in Wallet kan uitgeven. De Sentinel applicatie is bijvoorbeeld een Watch-only wallet.
