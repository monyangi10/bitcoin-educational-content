---
term: Extra transaction proof
definition: Dados suplementares no protocolo RGB para validar um compromisso do tipo Tapret.
---

No protocolo RGB, o ETP é a parte do Anchor que integra os dados adicionais necessários para validar um Commitment do tipo Tapret (no contexto do Taproot). Inclui, entre outras coisas, a chave pública interna associada ao script Taproot e as informações específicas necessárias para o *Script Path Spend*. Este componente assegura assim uma verificação exacta dos compromissos criptográficos.
