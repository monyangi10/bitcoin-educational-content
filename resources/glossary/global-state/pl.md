---
term: Global State
---

W protokole RGB, Global State wyznacza zestaw publicznych właściwości Contract, ustanowionych podczas Genesis. Działa jako rejestr publiczny, grupując stany Elements, które są dostępne dla wszystkich i mogą być aktualizowane przez autoryzowane przejścia zgodnie z zasadami określonymi przez Contract Schema. W przeciwieństwie do Owned States, które są powiązane z konkretnymi podmiotami, Global State nie należy do żadnej konkretnej strony.
