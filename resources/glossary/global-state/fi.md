---
term: Global State
---

RGB-protokollassa Global State nimeää Genesis:n aikana määritetyn Contract:n julkisten ominaisuuksien joukon. Se toimii julkisena rekisterinä, joka ryhmittelee Elements:n tilat, jotka ovat kaikkien käytettävissä ja joita voidaan päivittää valtuutetuilla siirtymillä Contract:n Schema:n määrittelemien sääntöjen mukaisesti. Toisin kuin omistetut tilat, jotka liittyvät tiettyihin entiteetteihin, Global State ei kuulu millekään tietylle osapuolelle.
