---
term: RAMO (GIT)
---

No contexto do Git, representa uma separação do fluxo de trabalho principal, permitindo o desenvolvimento paralelo sem afetar o ramo principal (normalmente chamado `master` ou `main`). As ramificações permitem que modificações, testes e experiências sejam realizadas num ambiente isolado, antes de serem integradas no projeto principal através de uma fusão.
