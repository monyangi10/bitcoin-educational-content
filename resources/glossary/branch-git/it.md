---
term: BRANCO (GIT)
---

Nel contesto di Git, rappresenta una separazione dal flusso di lavoro principale, consentendo uno sviluppo parallelo senza influenzare il ramo principale (solitamente chiamato `master` o `main`). I rami consentono di eseguire modifiche, test ed esperimenti in un ambiente isolato, prima di essere integrati nel progetto principale tramite un merge.
