---
term: BRANCHE (GIT)
---

Im Zusammenhang mit Git steht er für eine Trennung vom Hauptarbeitsablauf, die eine parallele Entwicklung ohne Beeinträchtigung des Hauptzweigs (in der Regel `master` oder `main` genannt) ermöglicht. Zweige ermöglichen die Durchführung von Änderungen, Tests und Experimenten in einer isolierten Umgebung, bevor sie über eine Zusammenführung in das Hauptprojekt integriert werden.
