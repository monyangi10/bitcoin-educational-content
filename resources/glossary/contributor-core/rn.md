---
term: Umuterankunga (Core)
definition: Umuntu afasha mu kurema Bitcoin Core mu gutanga code no gusuzuma izatanzwe.
---

Umuntu atanga umusanzu mu bijanye na Bitcoin core (ugushirwa mu ngiro kw’ibihimba vyinshi ku rubuga rwa Bitcoin) ni umuntu agira uruhara n’umwete mu gutegura porogarama mu kwandika kode, mu gusubiramwo no mu kugerageza amahinduka yashikirijwe n’abandi. Mu buryo butandukanye n’ababungabunga, abatanga intererano ntibafise ububasha bwo gufatanya amahinduka muri kode nyamukuru; uruhara rwabo ahubwo ni ugutanga ibisabwa vyo gukura (PR) no kugira uruhara mu biganiro no kwemeza ivyo vyiyumviro. Umuntu wese afise icipfuzo co gutanga umusanzu ataco akeneye gutorwa canke kwemererwa mu buryo butegekanijwe, ivyo bikaba bitandukanye n’abacungera umugambi bajejwe inshingano zikomeye z’uburongozi n’ugufata ingingo mu mugambi.
