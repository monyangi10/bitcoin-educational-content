---
term: BIP0381
---

Introduces functions for descriptors, notably `pk(KEY)` (Pay-to-PubKey), `pkh(KEY)` (Pay-to-PubKey-Hash), and `sh(SCRIPT)` (Pay-to-Script-Hash). These functions standardize how Legacy script types are described within descriptors.
BIP381 was implemented along with all other descriptor‑related BIPs (except BIP386) in Bitcoin Core v0.17.
