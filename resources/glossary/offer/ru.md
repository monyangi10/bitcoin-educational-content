---
term: ОФЕРТА
---

В BOLT12 *предложения* - это статичные QR-коды для совершения нескольких платежей на Lightning Network. В отличие от обычных счетов, *предложения* можно использовать повторно. Их можно использовать для нескольких запросов generate и Invoice. Когда пользователь сканирует QR-код, содержащий *предложение*, он отправляет сообщение с запросом нового Invoice на соответствующий узел Lightning. В ответ узел выдает Invoice, который может использовать плательщик. Таким образом, *предложения* обеспечивают уникальный идентификатор для получения нескольких платежей от разных пользователей в Lightning.
