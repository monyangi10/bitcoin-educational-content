---
term: I2P
definition: Anonymt kommunikationsnätverk som krypterar data och döljer identiteter på internet.
---

Anonymt kommunikationsnätverk utformat för att säkerställa konfidentialiteten i utbyten på Internet. Precis som TOR krypterar I2P nätverksdata och använder ett routingsystem som kallas "mix network". Denna metod garanterar kommunikationens anonymitet genom att skicka den via mellanliggande servrar som omdirigerar informationen utan att känna till avsändarens eller mottagarens identitet. Precis som TOR kan I2P användas på en Bitcoin-nod för att säkra dess nätverkskommunikation.


> akronymen "I2P" står för "Invisible Internet Project"
