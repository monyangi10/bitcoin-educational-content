---
term: Hot Wallet (ПРОГРАММНОЕ ОБЕСПЕЧЕНИЕ)
---

Hot Wallet - это программное устройство, предназначенное для защиты и управления закрытыми ключами Bitcoin Wallet. Hot Wallet - это когда фраза восстановления Bitcoin Wallet хранится на вычислительном устройстве с помощью программного обеспечения, которое не предназначено только для использования Bitcoin и подключено непосредственно к Интернету. Например, приложение Samurai Wallet на вашем смартфоне будет считаться Hot Wallet.
