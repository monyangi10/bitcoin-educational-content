---
term: Blocks index
definition: Structure LevelDB dans Bitcoin Core cataloguant les métadonnées et emplacements des blocs.
---

Structure de données LevelDB dans Bitcoin Core qui catalogue des métadonnées sur tous les blocs. Chaque entrée dans cet index renseigne des détails tels que l'identifiant du bloc, sa hauteur dans la blockchain, son emplacement sur le disque, et d'autres métadonnées. Cet index permet de localiser rapidement un bloc sur le disque.
