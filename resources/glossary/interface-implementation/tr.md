---
term: Interface Implementation
---

RGB protokolünde Interface Implementation, bir Interface ile bir Schema arasındaki yazışmayı oluşturan bildirimler kümesini ifade eder. Kullanıcı veya cüzdan gibi uygulamalar tarafından doğru yorumlanmalarını sağlamak için ikili Contract verilerinin anlamsal çevirisini sağlar.
