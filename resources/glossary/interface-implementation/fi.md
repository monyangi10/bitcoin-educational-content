---
term: Interface Implementation
---

RGB-protokollassa Interface Implementation tarkoittaa ilmoitusten joukkoa, jolla vahvistetaan Interface:n ja Schema:n välinen vastaavuus. Se mahdollistaa binääristen Contract-tietojen semanttisen kääntämisen, jotta käyttäjä tai sovellukset, kuten lompakot, voivat tulkita ne oikein.
