---
term: Interface Implementation
---

В протоколе RGB под Interface Implementation понимается набор деклараций, устанавливающих соответствие между Interface и Schema. Он позволяет осуществлять семантический перевод двоичных данных Contract, чтобы обеспечить их правильную интерпретацию пользователем или приложениями, такими как кошельки.
