---
term: Client-side Validation
---

Process by which each party (customer) verifies a set of privately exchanged data, according to the rules of a protocol. In the case of the RGB protocol, this exchanged data is grouped together in what are known as consignments. Unlike the Bitcoin protocol, which requires all transactions to be published in On-Chain, RGB allows only commitments (anchored in Bitcoin) to be stored in public, while the essential contract information (transitions, attestations, proofs) remains in off-chain, shared only between the users concerned.
