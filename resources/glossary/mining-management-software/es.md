---
term: SOFTWARE DE GESTIÓN Mining
---

Software dedicado a la gestión y optimización de las operaciones de Mining. Este tipo de software permite supervisar en tiempo real el rendimiento de ASIC, el consumo de energía, las temperaturas y las tasas de Hash. También ofrece funciones para gestionar granjas de Mining a gran escala, con ajustes automatizados (como overclocking), detección de fallos y gestión remota de dispositivos.
