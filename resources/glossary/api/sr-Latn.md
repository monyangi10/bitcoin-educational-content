---
term: API
---

Akronim za "*Application Programming Interface*". U opštem kontekstu računarstva, API je skup pravila i specifikacija koje softver može pratiti kako bi međusobno komunicirao. Omogućavaju programerima pristup funkcionalnostima ili podacima iz aplikacije, operativnog sistema ili druge usluge za sopstveni softver.


> ► *Na engleski, može se prevesti kao "application programming Interface" ili direktno kao "programming Interface".*
