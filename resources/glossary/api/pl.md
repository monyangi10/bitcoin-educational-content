---
term: API
---

Skrót od "*Application Programming Interface*". W ogólnym kontekście informatyki API to zestaw reguł i specyfikacji, których oprogramowanie może przestrzegać, aby komunikować się ze sobą. Umożliwiają one programistom dostęp do funkcji lub danych z aplikacji, systemu operacyjnego lub innej usługi dla własnego oprogramowania.


> w języku angielskim można to przetłumaczyć jako "programowanie aplikacji Interface" lub bezpośrednio jako "programowanie Interface"
