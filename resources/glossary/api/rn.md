---
term: API
---

Inyuguti y'inyongera y'ikirundi "*Iporogarama y'ibikorwa Interface*". Mu bijanye n’ubuhinga bwa none muri rusangi, API ni amategeko n’ibisobanuro porogarama zishobora gukurikiza kugira ngo zivugane n’izindi. Bituma ababikora bashobora kuronka ibintu canke amakuru yo muri porogarama, kuri sisitemu ikoresha canke ku yindi mirimo yo gukoresha muri porogarama zabo bwite.


> ► *Mu gifaransa, ivyo bihindurwa ngo “Interface y’iporogarama y’ibikorwa” canke gusa “Interface y’iporogarama.”*
