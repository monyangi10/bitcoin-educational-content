---
term: API
---

Acroniem voor "*Application Programming Interface*". In de algemene context van computergebruik is een API een verzameling regels en specificaties die software kan volgen om met elkaar te communiceren. Ze geven ontwikkelaars toegang tot functionaliteiten of gegevens van een applicatie, een besturingssysteem of een andere service voor hun eigen software.


> ► *In het Engels kan het vertaald worden als "application programming Interface" of direct als "programming Interface".*
