---
term: 比特币实现

definition: 应用比特币协议规则的软件，Bitcoin Core 是其参考实现。
---
指应用并遵循比特币协议所定义规则的软件。通常所说的 “比特币” 就是电子货币系统。它是一个规定规则的协议。它由构成网络的节点具体体现。这个系统没有具体的代码。它只是一套广泛的默许规则，由用户通过他们的节点达成共识。然而，真正包含计算机代码且能够被开发、维护和修改的，是比特币的节点软件。它们是比特币协议的独立实施，可以连接到网络的其他部分。

在比特币的实现中，Bitcoin Core 是迄今为止最普遍的方案，因为它代表了约 99% 的节点网络。不过，还有一些小众的实施方案，如 BTCsuite、Bcoin 和 Bitcoin Knots。尽管软件客户端多种多样，但 Bitcoin Core 在网络中的主导地位使其对比特币协议的应用几乎具有排他性的影响力。因此，可以说 Bitcoin Core 实际上代表了比特币协议本身。
