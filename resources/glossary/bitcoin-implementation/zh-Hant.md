---
term: gw-0 實作
---

指應用和遵循 Bitcoin 協定所定義規則的軟體。一般稱為「Bitcoin」的是電子貨幣系統。它是一個指定規則的協定。它具體地由組成網路的節點來表示。這個系統沒有具體的代碼。它只是一套由使用者透過節點達成共識而強加的廣泛默認規則。然而，那些有電腦程式碼，因此可以開發、維護和修改的，反而是 Bitcoin 節點軟體。這些是 Bitcoin 通訊協定的獨立實作，可以連接到網路的其他部分。


在 Bitcoin 實作中，Bitcoin Core 是迄今為止最普遍的，因為它代表了約 99% 的節點網路。然而，也有少數的實作，例如 BTCsuite、Bcoin 和 Bitcoin Knots。儘管軟體用戶端種類繁多，但 Bitcoin Core 在網路中的主導地位使其對 Bitcoin 協定的應用幾乎具有排他性的影響力。因此，可以說 Bitcoin Core 有效地代表了 Bitcoin 協定本身。
