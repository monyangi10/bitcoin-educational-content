---
term: Koordinator Coinjoin
definition: Server yang mengoordinasikan coinjoin tanpa dapat menghubungkan input dengan output pengguna.
---

Sebuah server pusat yang menghubungkan para pengguna yang ingin melakukan _coinjoin_ dan mengkoordinasikan pembangunan transaksi kolaboratif, tanpa dapat mengaitkan input dengan output spesifik dari setiap pengguna. Hal ini dimungkinkan dengan penggunaan _blind signature_ Chaum dan jaringan Tor. Tidak seperti pengacak Bitcoin tradisional, koordinator _Coinjoin_ tidak pernah memiliki bitcoin peserta dan tidak dapat mengasosiasikan input dan output.
