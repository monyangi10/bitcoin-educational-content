---
term: Coinjoin-koordinator
definition: Server som samordnar coinjoins utan att kunna koppla användarnas ingångar till utgångar.
---

En central server som kopplar samman användare som vill göra coinjoins och samordnar konstruktionen av samarbetstransaktioner, utan att kunna associera inmatningar med varje användares specifika utmatningar. Detta möjliggörs genom användning av Chaums blinda signaturer och Tor-nätverket. Till skillnad från en traditionell Bitcoin-shuffler tar CoinJoin:s koordinator aldrig deltagarnas bitcoins i besittning och kan inte associera input och output.
