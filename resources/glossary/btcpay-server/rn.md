---
term: BTCPay Server

definition: Igikoresho co kuriha gifunguye gituma umuntu yakira amahera ya bitcoin ata muvunyi abaye hagati.
---

BTCPay Server ni ubuhinga bwo kwishura bufunguye bufasha abacuruzi n’abakoresha kwemera kwishura Bitcoin ata kwizigira uwundi muntu kugira ngo bakoreshe amafaranga.

Yatangujwe mu mwaka w’2017, BTCPay Server itanga umuti wo kwishura Bitcoin ku mbuga z’ubudandaji, ifise ibintu biteye imbere nk’ugushigikira ama wallets y’ibikoresho, ibikoresho vyo gutanga amafaranga n’ivyo guharura amafaranga, hamwe n’uguhuza na Lightning Network.

Iterambere ryayo ryatangujwe na Nicolas Dorier, mu kwishura ku bikorwa vya Bitpay, nk'uko abivuga, vyazimije abakoresha mu guteza imbere SegWit2x nk'"ukuri" Bitcoin. Ukwo kurwanya kwashizwe mu rwandiko rwa Nicolas Dorier ubu ruzwi cane mu kwezi kwa munani 2017:


> "_Ivyo ni ibinyoma, ukwizigira kwanje muri wewe kwacitse, nzoguhindura uwutagira akamaro_".
