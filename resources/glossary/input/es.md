---
term: ENTRADA
---

En el contexto de Bitcoin, una entrada dentro de una transacción se refiere a los UTXOs (*Unspent Transaction Outputs*) utilizados como fondos originales para satisfacer las salidas. Cada entrada contiene referencias a UTXOs anteriores, que serán consumidos por la transacción. Estas entradas se utilizan para alimentar nuevos UTXO que se crearán como salidas de la transacción, y que luego podrán gastarse en transacciones futuras.


El papel de la transacción Bitcoin es, por tanto, consumir UTXOs como entradas, y crear nuevos UTXOs como salidas. La diferencia entre ambos corresponde a las tasas de transacción que puede recuperar el Miner que valida el bloque.


Desde un punto de vista más amplio, en informática, el término "entrada" se refiere generalmente a los datos suministrados a una función, algoritmo o sistema como operandos o información necesaria para realizar una operación o cálculo. En este sentido, el término se utiliza de forma más genérica para describir cualquier cosa suministrada a un proceso con el fin de obtener un resultado o salida. Por ejemplo, cuando se pasan datos a través de una función criptográfica Hash, esta información se denomina "entrada".
