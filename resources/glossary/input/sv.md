---
term: Input
definition: UTXO som används som källa för medel i en Bitcoin-transaktion, som refererar till en tidigare utgång.
---

Inom ramen för Bitcoin avser en input i en transaktion UTXO:er (*Unspent Transaction Outputs*) som används som ursprungliga medel för att uppfylla outputs. Varje input innehåller referenser till tidigare UTXO:er, som sedan kommer att förbrukas av transaktionen. Dessa inmatningar används för att mata nya UTXO:er som kommer att skapas som utmatningar av transaktionen och som sedan kan spenderas i framtida transaktioner.


Bitcoin-transaktionens roll är därför att konsumera UTXO:er som input och att skapa nya UTXO:er som output. Skillnaden mellan de två motsvarar de transaktionsavgifter som kan återvinnas av Miner som validerar blocket.


I ett bredare perspektiv, inom datavetenskap, avser termen "input" i allmänhet data som levereras till en funktion, algoritm eller ett system som operander eller information som krävs för att utföra en operation eller beräkning. I denna mening används termen mer generellt för att beskriva allt som tillförs en process för att erhålla ett resultat eller en output. När t.ex. data skickas genom en kryptografisk Hash-funktion kallas denna information för "input".
