---
term: ENTRADA
---

No contexto do Bitcoin, uma entrada numa transação refere-se aos UTXOs (*Unspent Transaction Outputs*) utilizados como fundos originais para satisfazer as saídas. Cada entrada contém referências a UTXOs anteriores, que serão depois consumidos pela transação. Estas entradas são utilizadas para alimentar novos UTXOs que serão criados como outputs da transação e que podem depois ser gastos em transacções futuras.


O papel da transação Bitcoin é, portanto, consumir UTXOs como inputs e criar novos UTXOs como outputs. A diferença entre os dois corresponde às taxas de transação que podem ser recuperadas pelo Miner que valida o bloco.


De um ponto de vista mais amplo, em informática, o termo "input" refere-se geralmente aos dados fornecidos a uma função, algoritmo ou sistema como operandos ou informações necessárias para efetuar uma operação ou cálculo. Neste sentido, o termo é utilizado de forma mais genérica para descrever tudo o que é fornecido a um processo com o objetivo de obter um resultado ou uma saída. Por exemplo, quando os dados são passados através de uma função criptográfica Hash, esta informação é referida como "entrada".
