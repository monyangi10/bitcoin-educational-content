---
term: Färgade mynt
definition: Metod för att representera icke-naturliga tillgångar på Bitcoin via metadata kopplad till transaktioner.
---

En metod som föreslogs 2012 av Yoni Assia, Vitalik Buterin och Meni Rosenfeld och som gör det möjligt att representera och hantera icke-inhemska tillgångar på Bitcoin Blockchain. Tanken var att koppla metadata till specifika transaktioner för att "färga" bitcoins för att indikera deras koppling till fysiska tillgångar. Den första funktionella implementeringen, Open Assets-protokollet, utvecklades av Flavien Charlon 2013. Detta protokoll möjliggjorde märkning av bitcoins med hjälp av `OP_RETURN`.
