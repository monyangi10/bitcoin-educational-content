---
term: ONION_PRIVATE_KEY
definition: فایل قدیمی Bitcoin Core که کلید خصوصی سرویس مخفی Tor V2 را ذخیره می‌کند.
---

فایلی که قبلاً در Bitcoin Core برای ذخیره یک کلید خصوصی مرتبط با یک سرویس مخفی Tor V2 برای گزینه `-listenonion` استفاده می‌شد. این فایل از نسخه 0.21.0 به بعد دیگر استفاده نمی‌شود و به جای آن از Tor V3 استفاده می‌شود.
