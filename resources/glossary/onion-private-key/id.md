---
term: ONION_PRIVATE_KEY

definition: File Bitcoin Core lama yang menyimpan kunci privat untuk layanan tersembunyi Tor V2.
---
File yang sebelumnya digunakan di Bitcoin Core untuk menyimpan kunci privat yang terkait dengan layanan tersembunyi Tor V2 untuk opsi `-listenonion`. File ini tidak lagi digunakan sejak versi 0.21.0 yang mendukung Tor V3.
