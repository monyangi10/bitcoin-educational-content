---
term: Anchors.dat

definition: Idosiye ya Bitcoin Core yatanze ayo mibotoro y'i IP y'ikinini ubwiganze yari ihuye n'ubwiyunge bwikinini mbere y'imvamo, kugira ngo isobera ihurira nubwiganze bukeye.
---

Dosiye ikoreshwa mu mukiriya wa Bitcoin Core kugira ngo ubike amaderesi IP y’ibice vy’inyuma umukiriya yari ahuye navyo imbere y’uko afunga. Anchors.dat iremwa igihe cose node ihagaze kandi igafutwa iyo isubiye gutangura. Ivyo bihimba bifise aderesi za IP ziri muri iyi dosiye bikoreshwa mu gufasha gushinga amahuzu ningoga iyo iyo node isubiye gutangura.
