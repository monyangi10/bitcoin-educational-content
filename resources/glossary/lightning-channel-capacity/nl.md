---
term: Lightning kanaalcapaciteit
definition: Maximale hoeveelheid bitcoins die via een Lightning-kanaal kan worden verzonden, gedefinieerd bij de aanmaak ervan.
---

Het aantal bitcoins dat is vergrendeld in een Address met meerdere handtekeningen dat een betalingskanaal vertegenwoordigt op de Lightning Network. De capaciteit van een kanaal is dus de maximale hoeveelheid Sats (satoshis) die kan worden verzonden via dit specifieke kanaal. Het wordt gedefinieerd op het moment van de creatie van het kanaal door de som van de fondsen die een partij toewijst aan het kanaal. De "inkomende capaciteit" verwijst naar de maximale hoeveelheid bitcoins die een node via een kanaal kan ontvangen. De "uitgaande capaciteit" staat voor de maximale hoeveelheid bitcoins die een knoop via een specifiek kanaal kan versturen.
