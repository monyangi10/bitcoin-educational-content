---
term: RGB contract
definition: Set of digital rights between actors via the RGB protocol, defined by a Schema.
---

Refers to a set of rights executed digitally between several actors via the RGB protocol. It has an active state and a business logic, defined by a Schema, which specifies which operations are authorized (transfers, extensions, etc.). A contract's state and validity rules are expressed in Schema. At any given time, the contract evolves only in accordance with what is permitted by this Schema and by the validation scripts (run, for example, in AluVM).
