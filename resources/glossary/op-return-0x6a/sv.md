---
term: OP_RETURN (0X6A)
definition: Opcode som gör en utgång ospenderbar, används ofta för att registrera godtycklig data.
---

Anger ett ogiltigt skript, vilket gör att utdata som innehåller det bevisligen inte kan spenderas. Nätverksnoder kan därmed ta bort denna utdata från sin UTXO-uppsättning. `OP_RETURN` används ofta för att registrera godtyckliga data i Blockchain.
