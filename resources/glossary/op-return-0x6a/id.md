---
term: OP_RETURN (0X6A)

definition: Opcode yang membuat output tidak dapat dibelanjakan, sering digunakan untuk mencatat data sembarang.
---
Mengindikasikan skrip yang tidak valid, membuat output yang berisi skrip tersebut tidak dapat digunakan. Dengan demikian, node jaringan dapat menghapus output ini dari set UTXO mereka. `OP_RETURN` sering digunakan untuk merekam data arbitrer dalam _blockchain_.
