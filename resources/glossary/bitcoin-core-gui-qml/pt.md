---
term: Bitcoin NÚCLEO GUI-QML
---

Projeto em curso para criar uma aplicação de portfólio e de nós Bitcoin baseada no Bitcoin Core, mas com um Interface gráfico intuitivo desenvolvido com QML (*Qt Modeling Language*). O objetivo do Bitcoin Core GUI-QML é modernizar a experiência do utilizador do Interface do Bitcoin Core para proporcionar uma experiência mais simples, mais suave e mais dinâmica.


O Interface tradicional do Bitcoin Core é desenvolvido principalmente em C++ com Qt Widgets, que é um sistema que produz interfaces de utilizador muito clássicas. Este novo projeto baseia-se agora em QML, que é uma linguagem declarativa utilizada para desenhar GUIs com animações, transições e um Interface mais responsivo e estilizado. Isto aproximará o Core das interfaces de aplicações modernas com interações ricas. O objetivo é tornar o software disponível tanto para PC como para dispositivos móveis.
