---
term: Bitcoin कोर GUI-QML
---

Bitcoin Core पर आधारित Bitcoin पोर्टफोलियो और नोड एप्लिकेशन बनाने के लिए चल रही परियोजना, लेकिन QML (*Qt मॉडलिंग लैंग्वेज*) का उपयोग करके विकसित एक सहज ग्राफ़िकल Interface के साथ। Bitcoin Core GUI-QML का उद्देश्य Bitcoin Core के Interface उपयोगकर्ता अनुभव को आधुनिक बनाना है ताकि एक सरल, सहज और अधिक गतिशील अनुभव प्रदान किया जा सके।


Bitcoin Core का पारंपरिक Interface मुख्य रूप से C++ में Qt Widgets के साथ विकसित किया गया है, जो एक ऐसा सिस्टम है जो बहुत ही क्लासिक यूजर इंटरफेस बनाता है। यह नया प्रोजेक्ट अब QML पर निर्भर करता है, जो एक घोषणात्मक भाषा है जिसका उपयोग एनिमेशन, संक्रमण और अधिक प्रतिक्रियाशील, स्टाइलिज्ड Interface के साथ GUI डिज़ाइन करने के लिए किया जाता है। यह Core को समृद्ध इंटरैक्शन के साथ आधुनिक एप्लिकेशन इंटरफ़ेस के करीब लाएगा। इसका उद्देश्य सॉफ़्टवेयर को PC और मोबाइल डिवाइस दोनों पर उपलब्ध कराना है।
