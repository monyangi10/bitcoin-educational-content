---
term: Bitcoin CORE GUI-QML
---

Progetto in corso per la creazione di un'applicazione per portafogli e nodi Bitcoin basata su Bitcoin Core, ma con una grafica Interface intuitiva sviluppata utilizzando QML (*Qt Modeling Language*). L'obiettivo di Bitcoin Core GUI-QML è quello di modernizzare l'esperienza utente di Interface di Bitcoin Core per offrire un'esperienza più semplice, fluida e dinamica.


Il Interface tradizionale di Bitcoin Core è sviluppato principalmente in C++ con Qt Widgets, un sistema che produce interfacce utente molto classiche. Questo nuovo progetto si affida ora a QML, un linguaggio dichiarativo utilizzato per progettare interfacce grafiche con animazioni, transizioni e un Interface più reattivo e stilizzato. In questo modo Core si avvicinerà alle moderne interfacce applicative con interazioni ricche. L'obiettivo è quello di rendere il software disponibile sia su PC che su dispositivi mobili.
