---
term: Bitcoin CORE GUI-QML
---

Pågående prosjekt for å lage en Bitcoin-portefølje- og nodeapplikasjon basert på Bitcoin Core, men med en intuitiv grafisk Interface utviklet ved hjelp av QML (*Qt Modeling Language*). Målet med Bitcoin Core GUI-QML er å modernisere Interface-brukeropplevelsen av Bitcoin Core for å levere en enklere, smidigere og mer dynamisk opplevelse.


Bitcoin Core's tradisjonelle Interface er hovedsakelig utviklet i C++ med Qt Widgets, som er et system som produserer svært klassiske brukergrensesnitt. Dette nye prosjektet baserer seg nå på QML, som er et deklarativt språk som brukes til å designe GUI-er med animasjoner, overganger og et mer responsivt, stilisert Interface. Dette vil bringe Core nærmere moderne applikasjonsgrensesnitt med rike interaksjoner. Målet er å gjøre programvaren tilgjengelig på både PC og mobile enheter.
