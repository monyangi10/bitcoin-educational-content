---
term: Bitcoin CORE GUI-QML
---

Probíhající projekt na vytvoření aplikace pro portfolio a uzly Bitcoin založené na jádru Bitcoin, ale s intuitivním grafickým rozhraním Interface vyvinutým pomocí jazyka QML (*Qt Modeling Language*). Cílem projektu Bitcoin Core GUI-QML je modernizovat uživatelské prostředí Interface v Bitcoin Core a poskytnout jednodušší, plynulejší a dynamičtější prostředí.


Tradiční Bitcoin Core Interface je vyvíjen především v jazyce C++ s Qt Widgets, což je systém, který vytváří velmi klasická uživatelská rozhraní. Tento nový projekt se nyní spoléhá na QML, což je deklarativní jazyk používaný k návrhu grafických uživatelských rozhraní s animacemi, přechody a citlivějším, stylizovanějším Interface. Tím se jádro přiblíží moderním aplikačním rozhraním s bohatými interakcemi. Cílem je zpřístupnit tento software na počítačích i mobilních zařízeních.
