---
term: Bitcoin Core GUI-QML
definition: Umushinga w'inyuma h'ishusho bigezweho kuri Bitcoin Core hakoreshejwe QML kugira ngo imikoreshereze yoroha.
---

Bitcoin core GUI-QML ni umugambi ubandanya wo kurema Bitcoin Wallet n’ikoreshwa ry’ibihimba bishingiye kuri Bitcoin core, ariko rifise igishushanyo gisanzwe Interface cakozwe hakoreshejwe QML (*Ururimi rwo Gushushanya Qt*).

Intumbero ya Bitcoin core GUI-QML ni uguhindura umukoresha Interface wa Bitcoin core kugira ngo ashobore gutanga ubumenyi bworoshe, buteye igomwe kandi bufise ubushobozi.


Bitcoin core Interface ya kera ikoreshwa cane cane muri C++ n’ibikoresho vya Qt, ubuhinga bukora ama interface y’abakoresha. Uyu mugambi mushasha, ariko, wishingikiriza kuri QML, ururimi rwo gutangaza rukoreshwa mu guhingura GUIs zifise amashusho, amahinduka, na Interface ishobora kwishura kandi iteye neza. Ivyo bizotuma Core yegerana n’ibikoresho vy’ubuhinga bwa none bifise imigenderanire myinshi kandi bigamije kuboneka ku bikoresho vyo ku biro no ku bikoresho vy’amaboko.
