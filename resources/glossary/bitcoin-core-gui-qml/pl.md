---
term: Bitcoin CORE GUI-QML
---

Trwający projekt mający na celu stworzenie portfela Bitcoin i aplikacji węzłowej opartej na Bitcoin Core, ale z intuicyjnym graficznym Interface opracowanym przy użyciu QML (*Qt Modeling Language*). Celem Bitcoin Core GUI-QML jest modernizacja interfejsu użytkownika Interface w Bitcoin Core, aby zapewnić prostsze, płynniejsze i bardziej dynamiczne doświadczenie.


Tradycyjny Bitcoin Core Interface jest rozwijany głównie w C++ z Qt Widgets, który jest systemem tworzącym bardzo klasyczne interfejsy użytkownika. Ten nowy projekt opiera się teraz na QML, który jest deklaratywnym językiem używanym do projektowania GUI z animacjami, przejściami i bardziej responsywnym, stylizowanym Interface. Przybliży to Core do nowoczesnych interfejsów aplikacji z bogatymi interakcjami. Celem jest udostępnienie oprogramowania zarówno na komputerach PC, jak i urządzeniach mobilnych.
