---
term: Bitcoin KERN GUI-QML
---

Laufendes Projekt zur Erstellung einer Bitcoin-Portfolio- und Knotenanwendung auf der Grundlage von Bitcoin Core, aber mit einem intuitiven grafischen Interface, das mit QML (*Qt Modeling Language*) entwickelt wurde. Das Ziel von Bitcoin Core GUI-QML ist es, die Interface-Benutzererfahrung von Bitcoin Core zu modernisieren, um eine einfachere, reibungslosere und dynamischere Erfahrung zu bieten.


Das traditionelle Interface von Bitcoin Core wird hauptsächlich in C++ mit Qt Widgets entwickelt, einem System, das sehr klassische Benutzeroberflächen erzeugt. Dieses neue Projekt stützt sich nun auf QML, eine deklarative Sprache, die für die Gestaltung von Benutzeroberflächen mit Animationen, Übergängen und einem reaktionsfähigeren, stilisierten Interface verwendet wird. Dies wird Core näher an moderne Anwendungsschnittstellen mit reichhaltigen Interaktionen heranführen. Ziel ist es, die Software sowohl auf PC als auch auf mobilen Geräten verfügbar zu machen.
