---
term: Bitcoin CORE GUI-QML
---

Текущий проект по созданию портфеля Bitcoin и приложения для узлов на основе Bitcoin Core, но с интуитивно понятной графикой Interface, разработанной с использованием QML (*Qt Modeling Language*). Цель Bitcoin Core GUI-QML - модернизировать пользовательский интерфейс Interface в Bitcoin Core, чтобы сделать его более простым, плавным и динамичным.


Традиционный Bitcoin Core в основном разрабатывается на C++ с использованием Qt Widgets, который представляет собой систему, создающую очень классические пользовательские интерфейсы. Новый проект теперь опирается на QML, который является декларативным языком, используемым для разработки графических интерфейсов с анимацией, переходами и более отзывчивым, стилизованным Interface. Это приблизит Core к современным интерфейсам приложений с богатым взаимодействием. Цель - сделать программу доступной как на ПК, так и на мобильных устройствах.
