---
term: Bitcoin Core GUI-QML
definition: Modern grafisch interfaceproject voor Bitcoin Core dat QML gebruikt voor een vlottere gebruikerservaring.
---

Lopend project om een Bitcoin portfolio- en knooppuntentoepassing te maken, gebaseerd op Bitcoin core, maar met een intuïtieve grafische Interface ontwikkeld met behulp van QML (*Qt Modeling Language*). Het doel van Bitcoin core GUI-QML is om de Interface gebruikerservaring van Bitcoin core te moderniseren om een eenvoudigere, soepelere en meer dynamische ervaring te leveren.


Bitcoin core's traditionele Interface is voornamelijk ontwikkeld in C++ met Qt Widgets, een systeem dat zeer klassieke gebruikersinterfaces produceert. Dit nieuwe project vertrouwt nu op QML, een declaratieve taal die gebruikt wordt om GUI's met animaties, overgangen en een meer responsieve, gestileerde Interface te ontwerpen. Dit brengt Core dichter bij moderne applicatie-interfaces met rijke interacties. Het doel is om de software beschikbaar te maken op zowel PC als mobiele apparaten.
