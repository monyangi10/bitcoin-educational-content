正在进行的项目旨在创建一个基于 Bitcoin Core 的 Bitcoin 组合和节点应用程序，但使用 QML（*Qt 建模语言*）开发直观的图形化 Interface。Bitcoin Core GUI-QML 的目标是使 Bitcoin Core 的 Interface 用户体验现代化，提供更简单、更流畅、更动态的体验。


Bitcoin Core 的传统 Interface 主要是用 C++ 和 Qt Widgets 开发的，这是一个能生成非常经典的用户界面的系统。这个新项目现在依赖 QML，QML 是一种声明式语言，用于设计带有动画和过渡效果的图形用户界面，以及反应更灵敏、更风格化的 Interface。这将使 Core 更接近具有丰富交互功能的现代应用界面。我们的目标是使该软件既能在个人电脑上使用，也能在移动设备上使用。
