---
term: Bitcoin Core GUI-QML
definition: Modern graphical interface project for Bitcoin Core using QML for a smoother user experience.
---

Bitcoin Core GUI-QML is an ongoing project to create a Bitcoin wallet and node application based on Bitcoin Core, but with an intuitive graphical interface developed using QML (*Qt Modeling Language*). 
The aim of Bitcoin Core GUI-QML is to modernize the user interface of Bitcoin Core to deliver a simpler, smoother, and more dynamic experience.

The traditional Bitcoin Core interface is mainly developed in C++ with Qt Widgets, a system that produces rather classic user interfaces. This new project, however, relies on QML, a declarative language used to design GUIs with animations, transitions, and a more responsive and stylized interface. This will bring Core closer to modern application interfaces with rich interactions and is aimed to be available on both desktop and mobile devices.
