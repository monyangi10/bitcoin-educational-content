---
term: Bitcoin CORE GUI-QML
---

Ongoing project to create a Bitcoin portfolio and node application based on Bitcoin Core, but with an intuitive graphical Interface developed using QML (*Qt Modeling Language*). The aim of Bitcoin Core GUI-QML is to modernize the Interface user experience of Bitcoin Core to deliver a simpler, smoother and more dynamic experience.

Bitcoin Core's traditional Interface is mainly developed in C++ with Qt Widgets, which is a system that produces very classic user interfaces. This new project now relies on QML, which is a declarative language used to design GUIs with animations, transitions and a more responsive, stylized Interface. This will bring Core closer to modern application interfaces with rich interactions. The aim is to make the software available on both PC and mobile devices.
