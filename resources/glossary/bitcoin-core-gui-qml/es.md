---
term: Bitcoin CORE GUI-QML
---

Proyecto en curso para crear una aplicación de carteras y nodos de Bitcoin basada en Bitcoin Core, pero con una interfaz gráfica intuitiva de Interface desarrollada con QML (*Qt Modeling Language*). El objetivo de Bitcoin Core GUI-QML es modernizar la experiencia de usuario de Interface de Bitcoin Core para ofrecer una experiencia más sencilla, fluida y dinámica.


El Interface tradicional de Bitcoin Core está desarrollado principalmente en C++ con Qt Widgets, que es un sistema que produce interfaces de usuario muy clásicas. Este nuevo proyecto se basa ahora en QML, que es un lenguaje declarativo utilizado para diseñar interfaces gráficas de usuario con animaciones, transiciones y un Interface más sensible y estilizado. De este modo, Core se acercará más a las interfaces de aplicaciones modernas con interacciones enriquecidas. El objetivo es que el software esté disponible tanto en PC como en dispositivos móviles.
