---
term: Bitcoin Core GUI-QML
definition: Projekt för ett modernt grafiskt gränssnitt för Bitcoin Core som använder QML för en smidigare användarupplevelse.
---

Pågående projekt för att skapa en Bitcoin-portfölj och nodapplikation baserad på Bitcoin Core, men med en intuitiv grafisk Interface som utvecklats med QML (*Qt Modeling Language*). Syftet med Bitcoin Core GUI-QML är att modernisera Interface-användarupplevelsen av Bitcoin Core för att leverera en enklare, smidigare och mer dynamisk upplevelse.


Bitcoin Core's traditionella Interface är huvudsakligen utvecklad i C++ med Qt Widgets, vilket är ett system som producerar mycket klassiska användargränssnitt. Det nya projektet förlitar sig nu på QML, som är ett deklarativt språk som används för att designa GUI:er med animationer, övergångar och en mer responsiv, stiliserad Interface. Detta kommer att föra Core närmare moderna applikationsgränssnitt med rika interaktioner. Målet är att göra programvaran tillgänglig på både PC och mobila enheter.
