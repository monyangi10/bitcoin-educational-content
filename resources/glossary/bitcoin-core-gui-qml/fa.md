---
term: Bitcoin Core GUI-QML
definition: پروژه رابط گرافیکی مدرن برای Bitcoin Core با استفاده از QML برای تجربه کاربری روان‌تر.
---

پروژه‌ای در حال انجام برای ایجاد یک پورتفولیو Bitcoin و برنامه نود بر اساس Bitcoin Core، اما با یک رابط گرافیکی شهودی Interface که با استفاده از QML (*زبان مدل‌سازی Qt*) توسعه یافته است. هدف از Bitcoin Core GUI-QML مدرن‌سازی تجربه کاربری Interface از Bitcoin Core برای ارائه تجربه‌ای ساده‌تر، روان‌تر و پویاتر است.


هسته Bitcoin سنتی Interface عمدتاً با C++ و Qt Widgets توسعه یافته است، که سیستمی است که رابط‌های کاربری بسیار کلاسیک تولید می‌کند. این پروژه جدید اکنون به QML متکی است، که یک زبان اعلانی است که برای طراحی رابط‌های کاربری گرافیکی با انیمیشن‌ها، انتقال‌ها و یک Interface پاسخگوتر و استایل‌دارتر استفاده می‌شود. این امر هسته را به رابط‌های برنامه‌های مدرن با تعاملات غنی نزدیک‌تر خواهد کرد. هدف این است که نرم‌افزار را هم بر روی رایانه‌های شخصی و هم دستگاه‌های موبایل در دسترس قرار دهد.
