---
term: Assume utxo

definition: 一个Bitcoin Core参数，允许使用假定有效的UTXO集快照快速同步新节点，然后在后台验证历史。
---
Bitcoin Core 多数派客户端中的配置参数，允许刚刚初始化（但尚未完成 IBD）的节点推迟在给定快照之前的交易和 UTXO 集验证。该概念基于使用由 Core 提供并假定准确的 UTXO 集（给定时间所有现有 UTXO 的列表），这使得节点能够非常快速地在累积工作量最多的链上进行同步。由于节点跳过了漫长的 IBD 步骤，它能够非常迅速地为用户提供功能。

Assume UTXO 将同步（IBD）分为两部分：首先，节点执行 Header First Sync（仅验证区块头），并将 Core 提供的 UTXO 集视为有效；然后，一旦其功能正常，节点将在后台验证完整的区块历史，更新一个由其自身验证的新 UTXO 集。如果后者与 Core 提供的 UTXO 集不符，它将提供错误消息。

因此，Assume UTXO 能够通过推迟交易和 UTXO 集的验证过程，利用 Core 中提供的更新快照，从而加速新比特币节点的准备工作。
