---
term: Assume utxo
definition: 一個Bitcoin Core參數，允許使用假定有效的UTXO集快照快速同步新節點，然後在後台驗證歷史。
---
Bitcoin Core 多數派用戶端中的配置參數，允許剛剛初始化（但尚未完成 IBD）的節點推遲在給定快照之前的交易和 UTXO 集驗證。該概念基於使用由 Core 提供並假定準確的 UTXO 集（給定時間所有現有 UTXO 的列表），這使得節點能夠非常快速地在累積工作量最多的鏈上進行同步。由於節點跳過了漫長的 IBD 步驟，它能夠非常迅速地為用戶提供功能。

Assume UTXO 將同步（IBD）分為兩部分：首先，節點執行 Header First Sync（僅驗證區塊頭），並將 Core 提供的 UTXO 集視為有效；然後，一旦其功能正常，節點將在背景驗證完整的區塊歷史，更新一個由其自身驗證的新 UTXO 集。如果後者與 Core 提供的 UTXO 集不符，它將提供錯誤訊息。

因此，Assume UTXO 能夠透過推遲交易和 UTXO 集的驗證過程，利用 Core 中提供的更新快照，從而加速新比特幣節點的準備工作。
