---
term: PHOENIXD
---

Gespecialiseerde implementatie van een Lightning (bliksem) node, ontworpen om betalingen van een server te versturen en te ontvangen. Phoenixd gebruikt dezelfde software als de Phoenix mobiele Wallet/node, maar in tegenstelling tot de laatste, die uitsluitend bedoeld is voor smartphones, kan phoenixd op een server draaien als een daemon. In plaats van een grafische gebruiker Interface (GUI), is het uitgerust met een HTTP API Interface en ondersteunt het automatisch kanaal- en liquiditeitsbeheer. Phoenixd is een project ontwikkeld en onderhouden door het Franse bedrijf Acinq.
