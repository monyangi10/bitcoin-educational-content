---
term: PHOENIXD

---

Ishirwa mu ngiro ry’umwihariko ry’uruzitiro rw’umuravyo (umuravyo), rwagenewe kohereza no kwakira amahera avuye kuri server. Phoenixd ikoresha porogarama imwe na Phoenix mobile wallet/node, ariko itandukanye n’iya nyuma, igenewe gusa telefone ngendanwa, phoenixd ishobora gukora kuri server nk’iya daemon. Aho kugira urubuga rwo gukoresha (GUI), rufise urubuga rwa HTTP API kandi rufasha gukoresha umurongo n’ugucungera amahera. Phoenixd ni umugambi wateguwe kandi ukabungabungwa n’ishirahamwe ry’abafaransa ryitwa Acinq.
