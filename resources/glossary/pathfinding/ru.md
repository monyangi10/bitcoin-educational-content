---
term: PATHFINDING
---

Процесс, используемый узлом для определения оптимального пути для маршрутизации платежа через сеть канала Lightning. Поиск пути осуществляется узлом-плательщиком, который должен выбрать наиболее подходящие промежуточные узлы для достижения получателя. Этот выбор основывается на ряде критериев, таких как комиссия, пропускная способность канала и временные блокировки.


Алгоритмы поиска пути строят граф топологии сети на основе данных сплетен и оценивают различные возможные маршруты между узлами плательщика и получателя. Эти маршруты ранжируются от лучшего к худшему в соответствии с различными критериями. Затем узел тестирует эти маршруты, пока не добьется успеха в осуществлении платежа по одному из них.
