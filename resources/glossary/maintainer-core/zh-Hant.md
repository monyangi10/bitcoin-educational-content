---
term: 維護員（核心）
---

在 Bitcoin 核心專案（Bitcoin 網路上的主要執行節點）的背景下，維護者是負責管理專案的個人。他們對專案的執行負責。他們負責管理 Bitcoin Core GitHub 資源庫，並設定新版本的發行時間表。他們主要負責合併貢獻者提出的拉取請求 (PR)。


換句話說，當程式碼修改提案通過各階段的驗證後，就是這些維護人員承擔了將新程式碼與 Bitcoin Core 軟體合併的重大責任。在進行合併之前，維護者會檢查程式碼是否符合專案的基本原則，是否已達到所需的最低標準，同時他們也會判斷貢獻者對這項修改的普遍共識。在撰寫本定義時，Bitcoin Core 只剩下五位維護者：Hennadii Stepanov、Michael Ford、Ava Chow、Gloria Zhao 和 Ryan Ofsky。
