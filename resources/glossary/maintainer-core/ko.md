---
term: 유지 관리자 (핵심)
---

Bitcoin 네트워크에서 노드를 주로 구현하는 Bitcoin core 프로젝트의 경우, 유지 관리자는 프로젝트 관리를 맡은 개인입니다. 이들은 프로젝트 구현에 대한 책임을 집니다. 이들은 Bitcoin core GitHub 리포지토리를 조정하고 새 버전 출시 일정을 설정하는 일을 담당합니다. 이들은 주로 기여자가 제안한 풀 리퀘스트(PR)의 병합을 담당합니다.


즉, 코드 수정 제안이 다양한 검증 단계를 통과한 경우, 새로운 코드를 Bitcoin core 소프트웨어와 병합하는 중요한 책임을 맡은 것은 바로 이러한 유지 관리자입니다. 이 병합을 진행하기 전에 관리자는 코드가 프로젝트의 기본 원칙을 준수하는지, 포함해야 하는 최소 기준에 도달했는지 확인하고, 이 수정에 대한 기여자들의 일반적인 합의도 판단합니다. 이 정의를 작성하는 현재, Bitcoin core에 남아 있는 메인테이너는 Hennadii Stepanov, Michael Ford, Ava Chow, Gloria Zhao, Ryan Ofsky 등 5명뿐입니다.
