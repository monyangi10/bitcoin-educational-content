term: Payload
definition: Amakuru ngenderwako arungikwa mu gice kinini c'amakuru.
---

Mu bijanye n’ubuhinga rusangi, umuzigo w’inyungu ni amakuru y’ingenzi atwarwa mu mpapuro nini z’amakuru. Nk’akarorero, muri SegWit V0 hejuru ya Bitcoin Address, umuzigo w’inyungu uhuye na Hash y’urufunguzo rwa bose, ata metadata atandukanye (HRP, itandukaniro, verisiyo ya SegWit n’umubare w’igenzura). Nk'akarorero, kuri Address `bc1qc2eukw7reasfmrafevp5dhv8635yuqays50gj`, dufise :




- `bc`: igice gisomwa n'umuntu (HRP) ;
- `1`: igitandukanya ;
- `q`: Ubuhinga bwa SegWit. Iyi ni verisiyo 0;
- `c2eukw7reasfcmrafevp5dhv8635yuqa`: umuzigo w'inyungu, muri iki gihe, Hash y'urufunguzo rwa bose ;
- `ys50gj`: umubare wo gusuzuma.
