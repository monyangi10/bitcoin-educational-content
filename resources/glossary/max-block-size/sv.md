---
term: MAX_BLOC_SIZE
definition: Konstant som definierar den maximala storleken ett block kan ha på Bitcoin.
---

En konstant som anger den maximala storleken som ett block kan ha på Bitcoin. Historiskt sett var denna gräns satt till 1 MB, en åtgärd som infördes av Satoshi Nakamoto 2010 för att förhindra skräppost och för att upprätthålla en viss nivå av nätverksdecentralisering.
