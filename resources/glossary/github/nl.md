---
term: GITHUB
---

Een platform voor het beheren en hosten van broncode dat samenwerking tussen ontwikkelaars vergemakkelijkt. Zoals de naam al doet vermoeden, is GitHub gebouwd op het versiebeheersysteem Git. Dit platform maakt het mogelijk om wijzigingen in de code bij te houden, versies te beheren en samenwerking aan te moedigen via tools als pull requests en issues. GitHub is een essentieel hulpmiddel geworden voor ontwikkelaars, vooral binnen de Bitcoin gemeenschap waar de meeste projecten, waaronder Bitcoin core, de belangrijkste implementatie van het protocol, worden gehost. In 2018 nam Microsoft GitHub over voor 7,5 miljard dollar.
