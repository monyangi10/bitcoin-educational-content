---
term: GITHUB
---

Jukwaa la kudhibiti na kupangisha msimbo wa chanzo ambao hurahisisha ushirikiano kati ya wasanidi programu. Kama jina lake linavyoonyesha, GitHub imejengwa kwenye mfumo wa udhibiti wa toleo la Git. Mfumo huu huruhusu kufuatilia mabadiliko ya misimbo, kudhibiti matoleo, na kuhimiza ushirikiano kupitia zana kama vile maombi ya kuvuta na masuala. GitHub imekuwa zana muhimu kwa wasanidi programu, haswa ndani ya jumuiya ya Bitcoin ambapo miradi mingi, ikiwa ni pamoja na Bitcoin Core, utekelezaji mkuu wa itifaki, inapangishwa. Mnamo 2018, Microsoft ilinunua GitHub kwa dola bilioni 7.5.
