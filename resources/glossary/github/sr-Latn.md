---
term: GITHUB
---

Platforma za upravljanje i hostovanje izvornog koda koja olakšava saradnju među programerima. Kao što ime sugeriše, GitHub je izgrađen na Git sistemu za kontrolu verzija. Ova platforma omogućava praćenje promena koda, upravljanje verzijama i podstiče saradnju putem alata kao što su pull requests i issues. GitHub je postao ključni alat za programere, posebno unutar Bitcoin zajednice gde je većina projekata, uključujući Bitcoin Core, glavnu implementaciju protokola, hostovana. Godine 2018, Microsoft je kupio GitHub za 7,5 milijardi dolara.
