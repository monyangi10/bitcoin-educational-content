---
term: GITHUB
---

Urubuga rwo gucunga no kwakira kode y’inkomoko yoroshe gukorana hagati y’abahinguzi. Nk’uko izina ryayo rivyerekana, GitHub yubatswe ku buryo bwo kugenzura verisiyo ya Git. Iyi nzira iremesha gukurikirana amahinduka ya kode, gucunga verisiyo, no guhimiriza ubufatanye biciye mu bikoresho nk’ibisabwa n’ibibazo. GitHub yabaye igikoresho gihambaye ku bategura, cane cane mu muryango wa Bitcoin aho imigambi myinshi, harimwo na Bitcoin core, ishyirwa mu bikorwa nyamukuru ry’amasezerano, icumbikiwe. Mu mwaka w’2018, Microsoft yaguze GitHub ku madolari miliyari 7 n’igice.
