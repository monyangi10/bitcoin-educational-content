---
term: GITHUB
---

Platforma do zarządzania i hostowania kodu źródłowego, która ułatwia współpracę między programistami. Jak sama nazwa wskazuje, GitHub opiera się na systemie kontroli wersji Git. Platforma ta umożliwia śledzenie zmian w kodzie, zarządzanie wersjami i zachęcanie do współpracy za pomocą narzędzi takich jak pull requests i issues. GitHub stał się niezbędnym narzędziem dla programistów, zwłaszcza w społeczności Bitcoin, gdzie hostowana jest większość projektów, w tym Bitcoin Core, główna implementacja protokołu. W 2018 roku Microsoft przejął GitHub za 7,5 miliarda dolarów.
