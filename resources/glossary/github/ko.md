---
term: 깃허브
---

개발자 간의 협업을 용이하게 하는 소스 코드를 관리하고 호스팅하는 플랫폼입니다. 이름에서 알 수 있듯이 GitHub는 Git 버전 관리 시스템을 기반으로 합니다. 이 플랫폼을 사용하면 코드 변경 사항을 추적하고, 버전을 관리하고, 풀 리퀘스트 및 이슈와 같은 도구를 통해 협업을 장려할 수 있습니다. 특히 프로토콜의 주요 구현인 Bitcoin core을 포함한 대부분의 프로젝트가 호스팅되는 Bitcoin 커뮤니티 내에서 GitHub는 개발자에게 필수적인 도구가 되었습니다. 2018년에 Microsoft는 75억 달러에 GitHub를 인수했습니다.
