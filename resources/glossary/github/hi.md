---
term: GitHub
---

स्रोत कोड को प्रबंधित करने और होस्ट करने के लिए एक प्लेटफ़ॉर्म जो डेवलपर्स के बीच सहयोग की सुविधा देता है। जैसा कि इसके नाम से पता चलता है, GitHub को Git संस्करण नियंत्रण प्रणाली पर बनाया गया है। यह प्लेटफ़ॉर्म कोड परिवर्तनों को ट्रैक करने, संस्करणों को प्रबंधित करने और पुल अनुरोधों और मुद्दों जैसे उपकरणों के माध्यम से सहयोग को प्रोत्साहित करने की अनुमति देता है। GitHub डेवलपर्स के लिए एक आवश्यक उपकरण बन गया है, विशेष रूप से Bitcoin समुदाय के भीतर जहां प्रोटोकॉल के मुख्य कार्यान्वयन Bitcoin कोर सहित अधिकांश परियोजनाएं होस्ट की जाती हैं। 2018 में, Microsoft ने 7.5 बिलियन डॉलर में GitHub का अधिग्रहण किया।
