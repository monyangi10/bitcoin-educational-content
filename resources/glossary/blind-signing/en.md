---
term: Blind signing
definition: Action of signing a transaction without being able to verify the details, potentially dangerous with hardware wallets.
---

The process whereby a user signs a Bitcoin transaction without being able to check the precise details of what they are signing. With hardware wallets, signing blind is dangerous, as the user must rely on the connected computer to display information (recipients, fees, amounts, etc.). 
This creates a security risk, as the computer is connected to the internet and has a much larger attack surface than the dedicated signing device. If the computer is compromised, it can manipulate the displayed details and display a malicious transaction to be signed without the user being aware of it. To avoid this risk, it is recommended to use hardware wallets with an integrated screen and to verify transaction details directly on the device before signing.
