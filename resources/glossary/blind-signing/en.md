---
term: SIGN BLIND
---

The process whereby a user signs a Bitcoin transaction without being able to check the precise details of what he is signing. With hardware wallets, signing blind is dangerous, as the user must rely on the connected computer to display information (recipients, fees, amounts, etc.). This exposes them to vulnerabilities, as the computer is connected to the Internet and has a much larger attack surface than the signing device. If the computer is compromised, it can manipulate the details displayed and send a malicious transaction to be signed without the user being aware of it. To avoid this risk, we recommend using hardware wallets with an integrated screen, and checking the information on it before signing transactions.
