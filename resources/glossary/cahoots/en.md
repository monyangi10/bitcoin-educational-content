---
term: CAHOOTS
---

In the context of the Samourai Wallet and other wallet software that implements it, a Cahoot refers to any type of transaction carried out collaboratively between multiple users. 
Participating in a Cahoot means jointly creating a transaction by exchanging partially signed transaction data. These exchanges can be done manually, using QR codes, or automatically through the Soroban communication network.

Types of Cahoot transactions include:
* Stowaway transactions (Payjoin);
* Stonewall x2 transactions;
* Joinbot transactions.
