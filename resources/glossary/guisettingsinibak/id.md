---
term: Guisettings.ini.bak

definition: Berkas cadangan pengaturan antarmuka grafis Bitcoin Core.
---
File di Bitcoin Core yang digunakan untuk menyimpan cadangan pengaturan antarmuka pengguna grafis (GUI). Cadangan ini dibuat ketika menggunakan opsi `resetguisettings`, yang akan mengatur ulang pengaturan GUI ke nilai bawaan. File ini memungkinkan pengguna untuk mengembalikan konfigurasi sebelumnya jika diperlukan.
