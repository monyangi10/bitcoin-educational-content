---
term: Owned State
---

Mu bijanye n'amasezerano ya RGB, Owned State yerekeza ku gice ca Contract State gipfutse muri Assignment kandi gifitaniye isano n'umuntu kanaka biciye ku Single-Use Seal yerekana UTXO. Ico gice kigereranya, nk’akarorero, umutungo w’ubuhinga bwa none canke uburenganzira bumwe bumwe bw’amasezerano buhabwa umuntu ku giti ciwe. Mu gutandukanya Leta ifitaniye isano n’uwuyifise wese, Owned State iremeza uburongozi bw’umutungo bubereye kandi butekanye mu masezerano y’ubwenge ya RGB.
