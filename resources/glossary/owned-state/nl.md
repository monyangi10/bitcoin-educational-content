---
term: Owned State
---

In de context van het RGB protocol verwijst Owned State naar het deel van Contract State dat is ingekapseld in een Assignment en expliciet geassocieerd is met een bepaalde houder via een Single-Use Seal die naar een UTXO wijst. Deze component vertegenwoordigt bijvoorbeeld een digitaal bezit of een specifiek contractueel recht dat aan een individu is toegewezen. Door de toestand gekoppeld aan elke houder te isoleren, garandeert Owned State nauwkeurig en veilig eigendomsbeheer binnen RGB slimme contracten.
