---
term: OP_2OVER (0X70)
definition: Opcode-ی که عناصر سوم و چهارم استک را به بالای آن کپی می‌کند.
---

دو Elements که در موقعیت‌های چهارم و سوم از بالای پشته قرار دارند را کپی کرده و سپس آن‌ها را در بالای پشته قرار می‌دهد. برای مثال، اگر پشته به این صورت باشد:


```text
A
B
C
D
```


`OP_2OVER` تولید خواهد کرد:


```text
C
D
A
B
C
D
```
