---
term: SCRIPT
---

Een op stack gebaseerde programmeertaal die wordt gebruikt om bestedingsvoorwaarden op te stellen en zo indirect bitcoins te beveiligen. Een script is in wezen een lijst instructies, samengesteld uit logische operatoren en commando's om stacks te manipuleren. Het wordt gematerialiseerd door het gebruik van opcodes die specifieke instructies geven die worden uitgevoerd door nodes om de geldigheid van een transactie te verifiëren. Script is een niet-Turing complete taal. Het kan worden gecategoriseerd als een intermediate-level taal (bijna low-level) geïnspireerd door Forth.
