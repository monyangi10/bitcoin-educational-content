---
term: Script
definition: Stackbaserat programmeringsspråk som används för att ställa villkor för att spendera bitcoins.
---

Ett stackbaserat programmeringsspråk som används för att fastställa utgiftsvillkor och därmed indirekt säkra bitcoins. Script är i huvudsak en lista med instruktioner som består av logiska operatorer och kommandon för att manipulera stackar. Det materialiseras genom användning av opkoder som ger specifika instruktioner som utförs av noder för att verifiera giltigheten av en transaktion. Script är ett icke-Turingkomplett språk. Det kan kategoriseras som ett mellannivåspråk (nästan lågnivå) inspirerat av Forth.
