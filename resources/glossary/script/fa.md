---
term: Script
definition: زبان برنامه‌نویسی مبتنی بر پشته (stack) که برای تعیین شرایط خرج کردن بیت‌کوین‌ها استفاده می‌شود.
---

یک زبان برنامه‌نویسی مبتنی بر پشته که برای تعیین شرایط خرج کردن استفاده می‌شود و به این ترتیب، به طور غیرمستقیم بیت‌کوین‌ها را امن می‌کند. Script اساساً یک لیست از دستورات است که از عملگرهای منطقی و فرمان‌هایی برای دستکاری پشته‌ها تشکیل شده است. این زبان از طریق استفاده از اپ‌کدها که دستورالعمل‌های خاصی را ارائه می‌دهند و توسط نودها برای تأیید اعتبار یک تراکنش اجرا می‌شوند، به کار گرفته می‌شود. Script یک زبان غیر تورینگ کامل است. می‌توان آن را به عنوان یک زبان سطح میانی (تقریباً سطح پایین) که از Forth الهام گرفته شده است، دسته‌بندی کرد.
