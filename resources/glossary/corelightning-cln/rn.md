---
term: Core-lightning (CLN)
definition: Ishirwa mu ngiro rikuru rya Lightning Network mu rurimi rwa C na Rust, ryakozwe na Blockstream.
---

Ishirwa mu ngiro rikomeye ry’umurongo wa Lightning Network wanditswe mu ndimi C na Rust. Core-Lightning yateguwe na Blockstream, yateguwe kugira ngo ibe yoroshe kandi ikore neza cane. Iratandukanye n'ubwubatsi bwayo bufise ibice, bituma abahinguzi bashobora kwongerako bitagoranye ibiranga. Iyi nzira yahinduwe izina mu 2022. Izina ryayo ry'intango ryari "C-Lightning".
