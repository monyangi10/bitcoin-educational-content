---
term: ISSUE
---

Binnen de context van Github en andere platforms voor het hosten van code, is een issue een rapport dat een bug signaleert, een verbetering voorstelt of een nieuwe functie voorstelt. Het dient als discussiepunt voor bijdragers en maakt het mogelijk om taken bij te houden die moeten worden voltooid of problemen die moeten worden opgelost in het project. Als Bitcoin gebruiker kun je de open-source software die je gebruikt helpen door mogelijke bugs te rapporteren via issues op de repository van het project.
