---
term: Block height
definition: Sequence number of a block relative to the Genesis block, used to identify a block in the blockchain.
---

The block height is the sequence number of a block relative to the first block, known as the Genesis Block, which is indexed at height 0.  It represents the number of blocks preceding the block in question. 
For example, if a block has a height of 650,000, it means that 650,000 blocks were mined before it. Block height is used to identify a specific block within the blockchain. 
Contrary to popular belief, block height does not determine which chain is valid, as nodes synchronize on the chain that has accumulated the most work, not necessarily the longest chain.
