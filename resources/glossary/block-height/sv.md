---
term: Blockhöjd
definition: Sekvensnummer för ett block i förhållande till Genesis-blocket, används för att identifiera ett block i blockkedjan.
---

Avser sekvensnumret för ett visst block i förhållande till det första blocket, känt som "Genesis-blocket", som är indexerat på höjd noll. Denna indikator talar om för oss hur många block som föregår blocket i fråga. Om ett block till exempel har höjden 650 000 betyder det att det finns 650 000 block som föregår det. Blockhöjd används för att identifiera ett specifikt block inom Blockchain. I motsats till vad många tror avgör inte blockhöjden vilken kedja som är giltig, eftersom noderna synkroniserar på den kedja som har ackumulerat mest arbete, inte nödvändigtvis den längsta kedjan.
