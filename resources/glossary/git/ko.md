---
term: GIT
---

모든 유형의 소프트웨어 프로젝트를 효율적으로 관리하도록 설계된 분산 버전 관리 시스템입니다. 개발자는 시간이 지남에 따라 프로젝트의 소스 코드 변경 사항을 추적하고, 이전 상태로 되돌리고, 브랜치를 관리하고, 변경 사항을 병합할 수 있습니다. Git은 파일 충돌의 위험 없이 여러 사람이 같은 프로젝트에서 동시에 작업할 수 있도록 하여 개발자 간의 협업을 촉진합니다. 각 개발자는 로컬에서 작업한 다음 변경 내용을 중앙 리포지토리와 동기화할 수 있습니다. 2005년 라이너스 토발즈가 만든 Git은 소프트웨어 업계에서 버전 관리의 사실상 표준이 되었습니다. Bitcoin core을 포함한 Bitcoin 노드 구현의 개발은 Git으로 관리됩니다.
