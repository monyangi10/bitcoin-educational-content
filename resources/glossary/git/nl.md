---
term: GIT
---

Gedistribueerd versiebeheersysteem dat is ontworpen om elk type softwareproject efficiënt te beheren. Het stelt ontwikkelaars in staat om wijzigingen in de broncode van een project in de loop van de tijd bij te houden, terug te gaan naar vorige versies, branches te beheren en wijzigingen samen te voegen. Git vergemakkelijkt de samenwerking tussen ontwikkelaars door meerdere mensen tegelijkertijd aan hetzelfde project te laten werken, zonder het risico op bestandsconflicten. Elke ontwikkelaar werkt lokaal en kan dan zijn wijzigingen synchroniseren met de centrale repository. Git werd in 2005 gecreëerd door Linus Torvalds en is de de facto standaard geworden voor versiebeheer in de software-industrie. De ontwikkeling van Bitcoin node implementaties, inclusief Bitcoin core, wordt beheerd met Git.
