---
term: GIT
---

分散式版本控制系統，設計用來有效管理任何類型的軟體專案。它能讓開發人員追蹤專案源代碼隨時間所做的變更、還原到之前的狀態、管理分支以及合併變更。Git 可讓多人同時處理同一專案，而不會有檔案衝突的風險，從而促進開發人員之間的協作。每個開發人員都在本地工作，然後可以將他們的變更與中央儲存庫同步。Git 於 2005 年由 Linus Torvalds 創建，目前已成為軟體產業版本控制的實際標準。包括 Bitcoin Core 在內的 Bitcoin 節點實作的開發都是使用 Git 管理的。
