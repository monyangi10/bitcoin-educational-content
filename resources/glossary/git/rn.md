---
term: Git
definition: Uburyo bwo kugenzura inyandiko bukwiragijwe mu micungire y'imishinga y'ubuhinga bwa mudasobwa dufashanya.
---

Uburyo bwo kugenzura verisiyo bukwiragijwe bwagenewe gucunga neza ubwoko bwose bw'umugambi wa porogaramu. Bishoboza abahinguzi gukurikirana amahinduka yakozwe kuri kode y’inkomoko y’umugambi mu gihe c’igihe, gusubira mu bihugu vya kera, gucunga amashami, no gufatanya amahinduka. Git yorohereza ugukorana hagati y’abahinguzi mu kwemerera abantu benshi gukora ku mugambi umwe icarimwe, ata ngorane z’amatati y’amadosiye. Buri muhinguzi akora aho hantu kandi ashobora guhuza amahinduka yiwe n’ububiko bwa mbere. Git yashinzwe mu 2005 na Linus Torvalds, yacitse ikigereranyo c'ukuri co kugenzura verisiyo mu bijanye n'amaporogarama. Iterambere ry'ibikorwa vya Bitcoin, harimwo na Bitcoin core, rirongorwa na Git.
