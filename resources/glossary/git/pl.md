---
term: GIT
---

Rozproszony system kontroli wersji zaprojektowany do efektywnego zarządzania dowolnym typem projektu oprogramowania. Umożliwia on deweloperom śledzenie zmian wprowadzanych do kodu źródłowego projektu w czasie, przywracanie poprzednich stanów, zarządzanie gałęziami i scalanie zmian. Git ułatwia współpracę między programistami, umożliwiając wielu osobom jednoczesną pracę nad tym samym projektem, bez ryzyka konfliktów plików. Każdy deweloper pracuje lokalnie, a następnie może synchronizować swoje zmiany z centralnym repozytorium. Stworzony w 2005 roku przez Linusa Torvaldsa, Git stał się de facto standardem kontroli wersji w branży oprogramowania. Rozwój implementacji węzła Bitcoin, w tym Bitcoin Core, jest zarządzany za pomocą Git.
