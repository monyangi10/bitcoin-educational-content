---
term: Git
definition: Distribuerat versionskontrollsystem för att hantera samarbetsprojekt inom programvara.
---

Distribuerat versionskontrollsystem som är utformat för att effektivt hantera alla typer av programvaruprojekt. Det gör det möjligt för utvecklare att spåra ändringar som gjorts i ett projekts källkod över tid, återgå till tidigare tillstånd, hantera grenar och slå samman ändringar. Git underlättar samarbetet mellan utvecklare genom att flera personer kan arbeta med samma projekt samtidigt, utan risk för filkonflikter. Varje utvecklare arbetar lokalt och kan sedan synkronisera sina ändringar med det centrala arkivet. Git skapades 2005 av Linus Torvalds och har blivit de facto-standarden för versionshantering inom programvaruindustrin. Utvecklingen av Bitcoin-nodimplementationer, inklusive Bitcoin Core, hanteras med Git.
