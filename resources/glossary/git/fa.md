---
term: Git
definition: سیستم کنترل نسخه توزیع شده برای مدیریت پروژه‌های نرم‌افزاری مشارکتی.
---

سیستم کنترل نسخه توزیع‌شده‌ای که برای مدیریت کارآمد هر نوع پروژه نرم‌افزاری طراحی شده است. این سیستم به توسعه‌دهندگان امکان می‌دهد تغییرات ایجاد شده در کد منبع یک پروژه را در طول زمان پیگیری کنند، به حالت‌های قبلی بازگردند، شاخه‌ها را مدیریت کنند و تغییرات را ادغام کنند. Git همکاری بین توسعه‌دهندگان را با اجازه دادن به چندین نفر برای کار بر روی یک پروژه به‌طور همزمان، بدون خطر تداخل فایل‌ها، تسهیل می‌کند. هر توسعه‌دهنده به‌صورت محلی کار می‌کند و سپس می‌تواند تغییرات خود را با مخزن مرکزی همگام‌سازی کند. Git که در سال ۲۰۰۵ توسط لینوس توروالدز ایجاد شد، به استاندارد د فاکتو برای کنترل نسخه در صنعت نرم‌افزار تبدیل شده است. توسعه پیاده‌سازی‌های نود Bitcoin، از جمله Bitcoin Core، با Git مدیریت می‌شود.
