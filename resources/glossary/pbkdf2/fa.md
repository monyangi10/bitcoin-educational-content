---
term: PBKDF2
definition: تابعی برای استخراج کلیدهای رمزنگاری از یک رمز عبور از طریق تکرار (iterations).
---

`PBKDF2` مخفف *Password-Based Key Derivation Function 2* است. این یک روش برای ایجاد کلیدهای رمزنگاری از یک رمز عبور با استفاده از یک تابع مشتق است. این روش به عنوان ورودی یک رمز عبور، یک نمک رمزنگاری و به صورت تکراری یک تابع از پیش تعیین شده (اغلب یک تابع Hash مانند `SHA256` یا یک `HMAC`) را به این داده‌ها اعمال می‌کند. این فرآیند بارها تکرار می‌شود تا یک کلید رمزنگاری generate شود.


در زمینه Bitcoin، از `PBKDF2` همراه با تابع `HMAC-SHA512` برای ایجاد seed از یک Wallet (seed) قطعی و سلسله‌مراتبی از یک عبارت بازیابی ۱۲ یا ۲۴ کلمه‌ای استفاده می‌شود. نمک رمزنگاری استفاده شده در این مورد BIP39 passphrase است و تعداد تکرارها `2048` می‌باشد.
