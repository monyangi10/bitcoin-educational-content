---
term: OP_3DUP (0X6F)
definition: Opcode-ی که سه عنصر بالای استک را تکثیر می‌کند.
---

نسخه‌های تکراری از سه Elements بالای پشته را گرفته و سپس آن‌ها را در بالای پشته قرار می‌دهد. به عنوان مثال، اگر پشته به صورت زیر باشد:


```text
A
B
C
D
```


`OP_3DUP` تولید خواهد کرد:


```text
A
B
C
A
B
C
D
```
