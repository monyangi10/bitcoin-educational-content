---
term: bitcoin-cli

definition: Antarmuka baris perintah untuk berinteraksi dengan node Bitcoin Core dan daemon bitcoind-nya.
---
`Bitcoin-cli`, singkatan dari "*Bitcoin Command Line Interface*", adalah sebuah antarmuka baris perintah yang dirancang untuk berinteraksi dengan proses Bitcoin Core yang sedang berjalan, terutama daemon, `bitcoind`. Ini adalah program mandiri yang menyediakan sarana bagi pengguna untuk berkomunikasi dan menjalankan perintah untuk mengontrol dan meminta status `bitcoind`. Selain kemampuan manajemen jaringan, seperti pemantauan transaksi dan blok, `bitcoin-cli` juga menawarkan fitur dompet, yang memungkinkan pengguna untuk melakukan transaksi Bitcoin dengan mengirim dan menerima dana.
