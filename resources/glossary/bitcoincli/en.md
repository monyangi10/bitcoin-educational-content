---
term: bitcoin-cli
definition: Command-line interface for interacting with a Bitcoin Core node and its bitcoind daemon.
---

`Bitcoin-cli`, short for “Bitcoin Command Line Interface”, is a command-line interface for interacting with a running Bitcoin Core instance, particularly its daemon, `bitcoind`. It is a standalone program that allows users to send commands to control and query the state of the `bitcoind` instance. In addition to network management capabilities, such as monitoring transactions and blocks, `bitcoin-cli` also offers wallet functionalities, allowing users to perform Bitcoin transactions by sending and receiving funds.
