---
term: Backdoor
definition: A secret mechanism intentionally introduced into a system allowing unauthorized access, bypassing normal security procedures.
---

A backdoor is a secret mechanism that provides privileged access to a computer system, software, function, algorithm or data, without going through the usual authentication or security procedures. Unlike a security flaw, backdoors are intentionally introduced into the source code by malicious developers. They can be used to spy on, manipulate or steal sensitive information.
