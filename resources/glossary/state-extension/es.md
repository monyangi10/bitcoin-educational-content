---
term: State Extension
---

En el contexto del protocolo RGB, una State Extension es una operación contractual utilizada para iniciar una actualización del estado de una Contract mediante el rescate de Valencias previamente declaradas. Este procedimiento utiliza un derecho público registrado en la Contract para desencadenar una modificación, que sólo se hace efectiva una vez que ha sido cerrada por una State Transition. Esta última incluye la actualización definitiva de la Contract State.
