---
term: State Extension
---

Im Kontext des RGB-Protokolls ist ein State Extension eine vertragliche Operation, die dazu dient, eine Aktualisierung des Zustands eines Contract durch die Einlösung zuvor gemeldeter Valenzen einzuleiten. Bei diesem Verfahren wird ein in der Contract registriertes öffentliches Recht verwendet, um eine Änderung auszulösen, die erst dann wirksam wird, wenn sie durch eine State Transition abgeschlossen wurde. Letzteres beinhaltet die endgültige Aktualisierung der Contract State.
