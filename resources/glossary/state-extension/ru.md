---
term: State Extension
---

В контексте протокола RGB, State Extension - это договорная операция, используемая для инициирования обновления состояния Contract путем погашения ранее объявленных валентностей. Эта процедура использует публичное право, зарегистрированное в Contract, для запуска модификации, которая вступает в силу только после ее закрытия State Transition. Последняя включает в себя окончательное обновление Contract State.
