---
term: State Extension
---

No contexto do protocolo RGB, um State Extension é uma operação contratual utilizada para iniciar uma atualização do estado de um Contract através do resgate de valências previamente declaradas. Este procedimento utiliza um direito público registado no Contract para desencadear uma modificação, que só se torna efectiva depois de ter sido encerrada por um State Transition. Este último inclui a atualização definitiva do Contract State.
