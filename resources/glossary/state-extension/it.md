---
term: State Extension
---

Nel contesto del protocollo RGB, un State Extension è un'operazione contrattuale utilizzata per avviare un aggiornamento dello stato di un Contract attraverso il riscatto di valenze precedentemente dichiarate. Questa procedura utilizza un diritto pubblico registrato nel Contract per attivare una modifica, che diventa effettiva solo dopo essere stata chiusa da un State Transition. Quest'ultimo include l'aggiornamento definitivo del Contract. Quest'ultimo include l'aggiornamento definitivo del Contract State.
