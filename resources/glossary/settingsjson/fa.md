---
term: Settings.json
definition: فایل Bitcoin Core که تنظیمات رابط کاربری گرافیکی را ذخیره می‌کند.
---

فایلی که در Bitcoin Core برای ذخیره تنظیمات رابط کاربری گرافیکی Interface (GUI) استفاده می‌شود. این فایل اطلاعات مربوط به پیکربندی‌های کاربر، مانند کیف‌پول‌های باز را نگه می‌دارد. هنگامی که Bitcoin Core مجدداً راه‌اندازی می‌شود، این فایل امکان باز شدن خودکار کیف‌پول‌هایی که قبل از بسته شدن برنامه فعال بودند را فراهم می‌کند. اگر یک Wallet از طریق GUI بسته شود، از این فایل نیز حذف می‌شود و بنابراین در شروع بعدی به‌طور خودکار باز نخواهد شد.
