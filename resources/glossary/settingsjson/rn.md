Dosiye ikoreshwa muri Bitcoin core kugira ngo ubike amategeko y'umukoresha w'igishushanyo Interface (GUI). Igumya amakuru yerekeye imiterere y'abakoresha, nk'akarorero amasakoshi yuguruye. Iyo Bitcoin core isubiye gutangura, iyo dosiye irashoboza gusubira gufungura ubwavyo ama wallets yari akora imbere y’uko porogarama ifunga. Iyo Wallet ipfungiwe biciye kuri GUI, nayo irakurwa muri iyi dosiye, kandi rero, ntizosubira gufungurwa ubwo nyene ku ntangiriro ikurikira.
