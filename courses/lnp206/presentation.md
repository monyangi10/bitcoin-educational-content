# Presentation

## A Hands-on course for Lightning Developers

The Breeze Nodeless SDK Hands-on course is an intensive 4-hour course designed for developers who want to build Lightning Network applications without the complexity of running their own Lightning node.

This practical course combines theoretical understanding with hands-on coding experience. You'll learn about Breeze's ecosystem, explore the Nodeless SDK's capabilities through live demonstrations, and build a complete Lightning-enabled application called "Bitcoin Wars."

### What Makes This Course Unique

Unlike traditional courses, this course emphasizes learning by doing. After a brief theoretical introduction, you'll spend the majority of the time actually writing code, implementing Lightning payment flows, and solving real-world integration challenges.

The "Bitcoin Wars" project serves as your practical playground—a gamified application that demonstrates Lightning payments in action. You'll complete the missing pieces of this application, experiment with the SDK, and receive immediate feedback from the instructor.

### Who Should Attend

This course is ideal for:

- Developers looking to integrate Lightning payments into applications
- Startup founders exploring Lightning Network capabilities
- Mobile/web developers wanting to add Bitcoin functionality
- Anyone interested in building on Bitcoin's Layer 2

### What You'll Take Away

By the end of this course, you will:

- Know how to implement core Nodeless SDK methods
- Have built a working Lightning-enabled application
- Understand payment flows, error handling, and best practices
- Know where to find resources and how to contribute to the community
- Have hands-on experience that you can immediately apply to your own projects

### Course Format

The Course is divided into two main parts:

**Part I - Theory**

- Introduction to Breez and its ecosystem
- Practical overview of the Nodeless SDK through code examples
- Walkthrough of SDK documentation

**Part II - Hands-on Coding**

- Environment setup with Git
- Exploration of the Bitcoin Wars application
- Individual coding challenges to complete the application
- Solution review and discussion
- Community resources and next steps

All participants will receive access to the complete source code, documentation references, and community channels for continued learning after the course.

# Course Requirements

To get the most out of this hands-on course, we recommand to the participants to have:

- Basic programming knowledge (JavaScript, Python, Rust, or similar)
- Familiarity with Git and version control
- Understanding of basic Lightning Network concepts
- A laptop with development environment (IDE/text editor)
- Git installed and configured
