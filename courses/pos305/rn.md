Icizere ntikiri cyoroshye, cyane cyane iyo kivuga amafaranga. Iri somo ryo gutangura rirakuyobora muri BTCPay Server, igikoresho gikomeye kiguha uburenganzira bwo kwakira ubwishyu bwa Bitcoin utishingikirije ku bandi bantu. Uziga ibanze byo kuba umukoresha wa BTCPay Server

## Incamake y'amashure


<chapterId>785ed2bc-94ae-4962-a26a-edf5742a3c72</chapterId>


Murakaze mu nyigisho ya POS 305 kuri Server ya BTCPay!


Intumbero y’iri huriro ni ukukwigisha ingene woshiramwo, gutunganya no gukoresha BTCPay Server mu bucuruzi bwawe canke mu muryango wawe. BTCPay Server ni umuti w’inkomoko yuguruye ushobora kugufasha gukora amahera ya Bitcoin wigenga, ata nkomanzi, kandi ataco utwara. Iyi nyigisho igenewe cane cane abakoresha bateye imbere bipfuza kumenya neza BTCPay Server yikoresha kugira ngo bashobore kwinjira mu bikorwa vyabo vya misi yose.


**Igice ca 1: Intangamarara ya Serveri ya BTCPay**

Tuzotangura n’ugushikiriza rusangi BTCPay Server, harimwo n’ibarabara ryo kwinjira, gucunga konti y’abakoresha, no kurema iduka rishasha. Iyi ntangamarara izogufasha gutahura BTCPay Server Interface no gutahura ibintu vy’ishimikiro bikenewe kugira ngo utangure gukoresha iki gikoresho.


**Igice ca 2: Intangamarara yo gucungera imfunguruzo za Bitcoin**

Umutekano w’amahera yawe ya Bitcoin ni ikintu gihambaye cane. Muri iki gice, tuzokwihweza ivy’uruvyaro rw’imfunguruzo z’ubuhinga bwa none, gukoresha ama wallets y’ibikoresho kugira ngo ukingire izo mfunguruzo, n’ingene wokorana n’imfunguruzo zawe biciye kuri BTCPay Server. Uzomenya kandi ingene wotunganya Server ya BTCPay Lightning Wallet kugira ngo ukoreshe neza amafaranga yawe.


**Igice ca 3: Serveri ya BTCPay Interface**

Iki gice kizokuyobora biciye ku mukoresha wa Server ya BTCPay Interface. Uzomenya ingene wogendagenda ku rubuga rw’ibikoresho, gutunganya amasetingi y’ububiko na server, gucunga amahera, no kwungukira ku bikoresho vy’inyongera vyinjijwe. Intumbero ni ukuguha ibikoresho bikenewe kugira ngo ushobore guhindura uko ushiramwo ibintu bivanye n’ivyo ukeneye vyihariye.


**Igice ca 4: Gutunganya Serveri ya BTCPay**

Ubwa nyuma, tuzokwibanda ku gushiramwo BTCPay Server mu bidukikije bitandukanye. Waba ukoresha LunaNode, Voltage, canke node ya Umbrel, uzokwiga intambwe z’ingenzi zo gukoresha no gutunganya Server yawe ya BTCPay, uzirikana ivy’ibidukikije vyose.


Ni mwiteguye kumenya neza BTCPay Server no gukura ubucuruzi bwanyu? Reka tugende!


## Ivyiza vy'umwanditsi Bitcoin na BTCPay Server


<chapterId>e1fe6294-3c82-5203-9537-779f9087c35a</chapterId>


Reka dutangure dutahuye ico BTCPay Server ari co n’inkomoko yayo. Turaha agaciro uguseruka n’ingingo mfatirwako zimwe zimwe kugira ngo umuntu agire icizigiro mu kibanza ca Bitcoin.

Umugambi wo mu kirere waraciye izo nzira. Umuhinga mu bijanye n’ugutegura BTCPay Server, Nicolas Dorier, ivyo yarabifashe we ubwiwe maze asezerana ko azobigira ivya kera. Ehe turi, haciye imyaka myinshi, kandi turiko turakora tuja muri aka kazoza, k’inkomoko yuguruye yuzuye, buri musi.


> Ivyo ni ibinyoma, ukwizigira kwanje kuri wewe kwacitse, nzoguhindura uwutagira akamaro.
> Nikola Doriyeri

Inyuma y’amajambo Nicolas yavuze, igihe co gutangura kwubaka carashitse. Ibikorwa vyinshi vyagiye mu co ubu twita BTCPay Server. Abantu benshi baripfuje gutanga intererano muri ico gikorwa. Abamenyekana cane ni r0ckstardev, MrKukks, Pavlenex, n’umucuruzi wa mbere yakoresheje iyo porogarama, astupidmoose.


Open source bisobanura iki, kandi ni igiki kija muri uwo mugambi?


FOSS ni ijambo risobanura Porogaramu y’Ubuntu n’Ifunguye. Iryo rya mbere ryerekeza ku majambo yemerera umuntu wese gukopa, guhindura, mbere no gukwiragiza amaverisiyo (mbere no kugira ngo aronke inyungu) y’iyo porogarama. Ivyo vya nyuma vyerekeye gusangira ku mugaragaro kode y’inkomoko, guhimiriza abantu bose gutanga umusanzu no kuyinoza.

Ivyo bikwegera abakoresha bazi utuntu n’utundi bafise igishika co gutanga umusanzu kuri porogarama bamaze gukoresha kandi bagakuramwo agaciro, amaherezo bikaba vyerekana ko baroraniwe mu kwemera kuruta porogarama z’abanyagihugu. Bihuye n’inyifato runtu ya Bitcoin ivuga ko “amakuru yipfuza kuba ataco amaze.” Bihuza abantu bafise urukundo rwinshi bagakora umuryango kandi biraryoshe cane. Cokimwe na Bitcoin, FOSS ntaco ishobora gukora.


### Imbere y'uko dutangura .


Iryo shure rigizwe n’ibice vyinshi. Benshi bazokwitwararikwa n’umwigisha wawe wo mw’ishure, ibidukikije vy’Iyerekanwa ushobora gushikako, server yakira abantu, kandi bishoboka ko n’izina ry’urubuga. Niwaheza iri shure wigenga, urashobora kumenya ko ibidukikije vyerekanwa ko ari DEMO bitazogushikira.

NB. Niwakurikiza iri shure mw’ishure, amazina y’abakozi yoshobora gutandukanywa bivanye n’ingene ishure ryawe ritunganijwe. Ibihinduka mu mazina ya Server bishobora kuba bitandukanye kubera ivyo.


### Ubuhinga bw'amashure


Igice cose gifise intumbero n’isuzuma ry’ubumenyi. Muri iki cigwa, tuzovuga kuri kimwe cose muri ivyo maze dutange incamake y’ibintu nyamukuru ku mpera y’igice c’icigwa cose (ni ukuvuga igice). Ivyerekanwa birashirwa ahabona kugira ngo bihe inyishu ziboneka no gukomeza ivyiyumviro nyamukuru mu bijanye n’ivyo umuntu abona. Intumbero zishirwaho mu ntango y’igice c’icigwa cose. Izo ntumbero zirarenga urutonde rw’ibintu vyo gusuzuma. Ivyo biguha ubuyobozi bwo kumenya ubuhinga bushasha. Ivyiyumviro vy’ubumenyi biragenda birushiriza kugorana, uko ugutegura Server yawe ya BTCPay biheza.


### None abanyeshure baronka iki n’ivyo vyigwa?


Mu gihe c'Ishure rya BTCPay Server, umunyeshure arashobora gutahura ingingo ngenderwako z'ishimikiro, z'ubuhinga n'izitari iz'ubuhinga, za Bitcoin. Amahugurwa menshi yo gukoresha Bitcoin biciye kuri BTCPay Server azotuma abanyeshure bashobora gukoresha ibikorwa remezo vyabo vya Bitcoin.


### Amaderesi y'urubuga ahambaye canke amahirwe yo guhamagara


Ishirahamwe ryitwa BTCPay Server Foundation, ryatumye Alekos na Bas bandika iryo shure, riri i Tokyo mu Buyapani. Umushinge wa BTCPay Server ushobora gushikirwa biciye ku rubuga rwashizwe ku rutonde.

### Gukorana na Serveri ya BTCPay


Intumbero y’iki gice c’amashure ni ukuronka ugutahura rusangi ku bijanye n’ubuhinga bwa BTCPay Server. Mu kibanza rusangi, ni vyiza gukurikira iyerekanwa ry’umwigisha no kuraba igitabu c’inyigisho ca BTCPay Server kugira ngo ukurikize hamwe n’umwigisha. Uzomenya ingene wokora Wallet biciye mu buryo bwinshi. Ingero ni nk’imiterere ya Hot Wallet n’ibikoresho vy’amasakoshi bihujwe biciye ku BTCPay Server Vault. Izo ntumbero ziboneka mu bidukikije vy’Iyerekanwa, zigaragazwa kandi zihabwa uburenganzira bwo kuzironka n’umwigisha wawe w’amashure.


Niwakurikiza iyi nyigisho wewe nyene, urashobora kuronka urutonde rw’abashitsi b’abandi ku ntumbero z’Iyerekanwa kuri https://directory.btcpayserver.org/filter/hosts. Turahanura cane kudakoresha izo nzira z’uwundi muntu nk’ibidukikije vyo gukoreramwo; ariko rero, zikora intumbero ibereye yo gutanguza ikoreshwa rya Bitcoin na BTCPay Server.


Nk’umumenyerezwa w’inyenyeri za BTCPay Server, ushobora kuba ufise ubumenyi bwa kera bwo gushinga urudodo rwa Bitcoin. Iyi nyigisho igenewe canecane porogarama za BTCPay Server.


Benshi mu mahitamwo ari muri BTCPay Server bari mu buryo bumwe canke ubundi mu yindi porogarama ijana na Bitcoin Wallet.


### Igikoresho co kwinjira muri Serveri ya BTCPay


Uko wakiriwe mu bidukikije vy’Iyerekanwa, usabwa ‘Kwinjira’ canke ‘Gukora konti yawe.’ Abarongozi ba server barashobora guhagarika ubuhinga bwo gukora konti nshasha kubera imvo z’umutekano. Ibimenyetso vya Server ya BTCPay n’amabara y’ubuto birashobora guhindurwa kuko Server ya BTCPay ni Porogaramu y’Isoko Yuguruye. Uwundi muntu ashobora gushiramwo ikimenyetso c’umweru kuri porogarama maze agahindura ukuntu isa.


![image](assets/en/001.webp)


### Rema idirisha rya konti


Gukora konti kuri Server ya BTCPay bisaba imirongo ya Email Address ibereye; akarorero@email.com yoba urudodo rubereye kuri Imeyili.


Ijambobanga rikeneye kuba rifise nibura inyuguti 8, harimwo inyuguti, imibare n’inyuguti. Uhejeje gushinga ijambobanga rimwe, uzobwirizwa kugenzura ko ijambobanga ari rimwe n’iryo wanditswe mu kibanza ca mbere c’ijambobanga.


Igihe ivyicaro vyose bibiri vya Email na Password vyuzuye neza, fyonda ku buto ‘Create Account’. Ivyo bizobika Email n’ijambobanga kuri Server ya BTCPay y’umwigisha.

### Konti yashizweho n'umuyobozi wa serveri ya BTCPay


Umuyobozi w'Intangamarara ya Serveri ya BTCPay arashobora kandi gukora amakonti ya Serveri ya BTCPay. Umuyobozi w’instance ya Server ya BTCPay arashobora gufyonda kuri ‘Server Settings’ (1), gufyonda ku ‘Bakoresha’ (2), hanyuma gufyonda kuri buto “+ Yongera Umukoresha” (3) iri hejuru iburyo bw’Users. Mu Ntumbero (4.3), uzomenya vyinshi ku bijanye n’ubugenzuzi bw’umuyobozi w’Ikonti.


![image](assets/en/003.webp)


Nk’umuyobozi, wokenera Email y’ukoresha Address maze ugashinga ijambobanga ry’ibanga. Ni vyiza ko Umuyobozi amenyesha uwukoresha guhindura iri jambobanga imbere yo gukoresha konti kubera imvo z’umutekano. Iyo Umuyobozi atashizeho Ijambobanga kandi SMTP ikaba yaratunganijwe kuri server, uwukoresha azoronka imeyili iriko uruja n’uruza rw’ubutumire kugira ngo akore konti yiwe kandi ashireho ijambobanga we nyene.


### Akarorero


Igihe ukurikiye inyigisho n’umwigisha, kurikiza urubuga rwatanzwe n’umwigisha maze ushireho konti yawe ku bidukikije vy’Iyerekanwa. Raba neza ko imeyili yawe Address n’ijambobanga ryawe vyabitswe neza; uzokenera ivyo bimenyetso vyo kwinjira ku ntumbero zisigaye z'iyerekanwa muri iri shure.


Umwigisha wawe ashobora kuba yarakoranije imbere y’igihe imeyili Address maze akarungika uruja n’uruza rw’ubutumire kugira ngo ukore iki gikorwa. Niba wategetswe, reba Email yawe.


Igihe ufata amashure ata mwigisha, fungura konti yawe ukoresheje ibidukikije vy’iyerekanwa rya BTCPay Server; genda kuri


https://Mainnet.iyerekanwa.btcpayserver.org/kwinjira.


Iyi konti ikwiye gukoreshwa gusa mu bijanye n’iyerekanwa/ugutoza abantu ntiyigere ikoreshwa mu bijanye n’ubudandaji.

### Ni igiki kijanye n’uguhingura imfunguruzo za Bitcoin?


Amasakoshi ya Bitcoin, iyo aremwe, akora ico bita "seed". Mu ntumbero ya nyuma, wararemye "seed", Urutonde rw'amajambo yavutse imbere ruzwi kandi nk'amajambo ya Mnemonic. seed ikoreshwa mu gukuraho imfunguruzo za Bitcoin kandi ikoreshwa mu kohereza canke kwakira Bitcoin. Amajambo ya seed ntashobora kwigera asangizwa n’abandi canke abo mu runganwe rwawe batizigira.


Igihugu ca seed gikorwa hakurikijwe ingingo ngenderwako y'inganda izwi kw'izina rya "Hierarchical Deterministic" (HD).

### Aderesi


Serveri ya BTCPay yubatswe kuri generate Address nshasha. Ivyo bigabanya ingorane yo gusubira gukoresha urufunguzo rwa bose canke Address. Gukoresha urufunguzo rwa bose rumwe bituma gukurikirana amateka yawe yose y’ukwishyura vyoroha cane. Kwiyumvira imfunguruzo nk’ibipapuro vy’ugukoresha rimwe gusa vyotuma urushiriza kugira ubuzima bwite. Turakoresha kandi Amaderesi ya Bitcoin, ntimuvyivange n’imfunguruzo za bose.


Address iva ku rufunguzo rwa bose biciye ku “buryo bwo gukora hashing.” Ariko rero, amasakoshi menshi n’amafaranga, azokwerekana Aderesi aho kwerekana izo mfunguruzo za bose. Aderesi, muri rusangi, ni ngufi kurusha imfunguruzo za bose kandi akenshi zitangura na `1`, `3`, canke `bc1`, mu gihe imfunguruzo za bose zitangura na `02`, `03`, canke `04`.



- Amaderesi atangura na `1.....` aracari amaderesi asanzwe cane. Nk'uko vyavuzwe mu kigabane, "Guhingura iduka rishasha," izo ni amaderesi y'iragi. Ubwo bwoko bwa Address bugenewe ibikorwa vya P2PKH. P2Pkh ikoresha ubuhinga bwa Base58, ivyo bikaba bituma Address ishobora gutahura inyuguti nini. Igishushanyo caco gishingiye ku rufunguzo rwa bose rufise umubare w’inyongera nk’ikimenyetso.



- Amaderesi atangura na `bc1...` ariko aragenda buhoro buhoro aja mu maderesi asanzwe cane. Ivyo bizwi kw’izina rya (akavukire) Aderesi za SegWit. Ivyo bitanga urutonde rwiza rw’amahera kuruta ayandi Maderesi yavuzwe. Amaderesi y'imvukira SegWit akoresha ubuhinga bwa Bech32 kandi yemera gusa inyuguti ntoya.



- Aderesi zitangura na `3...` zirakoreshwa cane cane n'abahindura amaderesi y'ibiziga. Izo aderesi zivugwa mu kigabane "Gukora iduka rishasha," aderesi za SegWit zizingiwe canke zishizwe mu nzira. Ariko rero, zirashobora gukora nk'"Multisig Address". Iyo ikoreshejwe nka SegWit Address, hariho ukuzigama ku mahera y'ugucuruza, kandi, bike kuruta SegWit y'imvukira. P2SH Amaderesi akoresha ubuhinga bwa Base58. Ivyo bituma iba Sensitive, nk'iragi rya Address.



- Amaderesi atangura na `2...` ni amaderesi ya Testnet. Bagenewe kwakira amabarabara y’ibara ry’agahama (tBTC). Ntukwiye kwigera uvanga ivyo maze ukarungika Bitcoin kuri izo aderesi. Kubera intumbero z'iterambere, ushobora generate na Testnet Wallet. Hariho amazi menshi kuri internet kugira ngo ubone Testnet Bitcoin. Ntukigere ugura Testnet Bitcoin. Testnet Bitcoin ni yo icukurwa. Ivyo bishobora kuba ari vyo bituma umuhinguzi akoresha Regtest aho gukoresha Regtest. Iyi ni ikibanza co gukiniramwo abahinguzi, habuze ibice bimwebimwe vy’urubuga. Ariko rero, Bitcoin ni ngirakamaro cane ku bijanye n’iterambere.


### Imfunguruzo za bose


Imfunguruzo za bose ntizikoreshwa cane mu bikorwa muri iki gihe. Igihe kirenze, abakoresha Bitcoin baramaze kuzisubirira n’Aderesi aho kuzisubirira. Biracariho kandi biracamenyererwa rimwe na rimwe. Imfunguruzo za bose ni, muri rusangi, imirongo miremire cane kuruta aderesi. Nka kurya kw’amaderesi, atangura n’ikimenyetso kinaka.



- Ubwa mbere, `02...` na `03...` ni ibimenyetso vy'urufunguzo rwa bose bimenyerewe cane vyashizwe mu buryo bwa SEC. Ivyo birashobora gutunganirizwa maze bikahindurwa amaderesi yo kwakira, bikakoreshwa mu guhingura amaderesi ya multi-sig canke mu kugenzura umukono. Ibikorwa vya Bitcoin vya kera vyakoresha imfunguruzo za bose nk’igice c’ibikorwa vya P2PK.



- Ariko rero, amasakoshi ya HD akoresha uburyo butandukanye. `xpub...`, `ypub...` canke `zpub...` vyitwa imfunguruzo za bose zagutse, canke xpubs. Izo mfunguruzo zikoreshwa mu gukura imfunguruzo nyinshi za bose nk'igice ca HD Wallet. Nk’uko xpub yawe ifise amakuru y’amateka yawe yose, bisobanura amafaranga yawe ya kera n’ayo muri kazoza, ntukigere ubisangira n’abantu utizigira.


### Incamake y'ubuhinga


Muri iki gice, mwarize ibi bikurikira:



- Ubutandukane hagati y’amaderesi n’ubwoko bw’amakuru y’urufunguzo rwa bose n’ivyiza vyo gukoresha aderesi ku rufunguzo rwa bose.

### Kubika imfunguruzo za Bitcoin


Amaze gukora ijambo seed, urutonde rw'amajambo 12 - 24 yashizwe muri iki gitabu rusaba ko umuntu akora ububiko bwiza n'umutekano, kuko ayo majambo ari yo nzira yonyene yo gusubira gukoresha Wallet. Ubuhinga bwa HD wallets n’ingene butanga amaderesi mu buryo butegekanijwe hakoreshejwe seed imwe bisigura ko amaderesi yose waremye azoshigikirwa hakoreshejwe uru rutonde rumwe rw’amajambo Mnemonic, aserukira seed yawe canke ijambo ryo gusubirana.


Gumana ijambo ryawe ryo gukira ryiza. Iyo umuntu ashitseko, cane cane afise umugambi mubi, arashobora kwimurira amahera yawe. Gukomeza seed mu mutekano no mu mutekano, mu gihe kandi bibuka ko ari ivy’ugusangira hagati yabo. Hari uburyo bwinshi bwo kubika imfunguruzo z’ibanga za Bitcoin, imwe yose ikaba ifise ivyiza n’ibibi vyayo, mu bijanye n’umutekano, ubuzima bwite, uburyo bwo kubika, n’ububiko bw’umubiri. Kubera akamaro k’imfunguruzo z’ibanga, abakoresha Bitcoin bakunda kubika no kuzigama ata nkomanzi izo mfunguruzo mu “kwizigama” aho gukoresha ibikorwa vyo “kwizigama” nk’amabanki. Bivanye n'uwubikoresha, bategerezwa gukoresha umuti wo kubika Cold canke Hot Wallet.


### Hot na Cold ububiko bw'imfunguruzo za Bitcoin


Kenshi, amasakoshi y’ama Bitcoin akoreshwa mu ma Hot Wallet canke Cold Wallet. Ivyinshi mu bintu bishobora gutuma umuntu agira ico akora biri mu kuntu bimeze neza, mu buryo bworoshe gukoresha be no mu bijanye n’umutekano. Buri bumwe muri ubwo buryo burashobora kandi kubonwa mu muti w’umucungezi. Ariko rero, ivyo guhinduranya aha ahanini bishingiye ku mutekano no ku buzima bwite kandi birarenga urugero rw’iri shure.


### Hot Wallet


Hot wallets ni uburyo bwiza bwo gukorana na Bitcoin biciye ku mbuga ngurukanabumenyi, ku rubuga canke ku biro. Wallet yama ifatanye na internet, bikaba bituma abayikoresha bashobora kohereza canke kwakira Bitcoin. Ariko rero, ivyo na vyo nyene ni intege nke zayo; Wallet, nk’uko yama iri kuri internet, ubu irashobora guterwa cane n’abasuma canke porogarama mbi ziri ku gikoresho cawe. Mu Serveri ya BTCPay, amasakoshi ya Hot abika imfunguruzo z’ibanga ku nkuru. Umuntu wese yinjira mu bubiko bwawe bwa BTCPay Server yoshobora kwiba amahera muri iyi Address nimba ari umunyaruyeri. Igihe BTCPay Server ikora mu bidukikije vyakira abashitsi, ukwiye kwama uzirikana ivyo mu nkuru yawe y’umutekano kandi vyiza ni uko udakoresha Hot Wallet muri ivyo. Iyo BTCPay Server ishizwe ku bikoresho vy’ubuhinga bwa none kandi bicunzwe nawe, urutonde rw’ingorane ruragabanuka cane, ariko ntirwigera ruzimangana burundu.


### Cold Wallet


Abantu ku giti cabo bimurira Bitcoin yabo mu Cold Wallet kuko ishobora gutandukanya imfunguruzo z’ibanga kuri internet, gutyo ikabakingira ibintu bishobora kubatera ubwoba kuri internet. Gukuraho uruja n’uruza rwa internet mu nzira y’ubuhinga bigabanya ingorane z’ubuhinga butari bwiza, ubuhinga bw’ubutasi, n’uguhindura SIM. Ububiko bwa Cold bwemerwa ko buruta ububiko bwa Hot ku bijanye n’umutekano n’ukwigenga, igihe cose hafashwe ingingo zihagije kugira ngo umuntu ntatakaze imfunguruzo z’ibanga za Bitcoin. Ububiko bwa Cold burabereye cane ku bipimo vyinshi vya Bitcoin, bitagenewe gukoreshwa kenshi kubera ukuntu Wallet igoranye.


Hari uburyo butandukanye bwo kubika imfunguruzo za Bitcoin mu bubiko bwa Cold, kuva ku masakoshi y’impapuro gushika ku masakoshi y’ubwonko, ku masakoshi y’ibikoresho, canke, kuva mu ntango, dosiye ya Wallet. Ivyuma vyinshi bikoresha BIP 39 gushika generate ijambo seed. Ariko rero, muri porogarama ya Bitcoin core, nta n’umwe yemeranya ku bijanye n’ingene yokoreshwa. Porogaramu ya Bitcoin core izoguma generate dosiye ya Wallet.dat, ukeneye kubika ahantu hatagira umurongo.

### Uko worungika


Igikorwa kirungika ca serveri ya BTCPay cohereza amafaranga kuva kuri serveri yawe ya BTCPay On-Chain Wallet. BTCPay Server iremeza uburyo bwinshi bwo gusinya amafaranga yawe kugira ngo ukoreshe amahera. Igikorwa kishobora gusinywa n’;



- Hardware Wallet
- Amasakoshi ashigikira PSBT
- HD urufunguzo rw’ibanga canke imbuto zo gukira.
- Hot Wallet


#### Hardware Wallet


BTCPay Server ifise ubufasha bwa Hardware Wallet bushizwemwo, bugufasha gukoresha Hardware Wallet yawe na BTCPay Vault ata makuru asohoka ku maporogarama canke ku ma server y’abandi. Ugushiramwo Hardware Wallet muri BTCPay Server bigufasha kwinjiza Hardware Wallet yawe no gukoresha amahera yinjira n’ukwemeza kworoshe ku gikoresho cawe. Imfunguruzo zawe z’ibanga ntizigera ziva muri iyo mashini, kandi amahera yose araremezwa kuri Full node yawe, kugira ngo ntihagire amakuru asohoka.


#### Gusinyana na Wallet ishigikira PSBT


PSBT (Ibikorwa vya Bitcoin Vyashizweko Umukono Mu Gice) ni uburyo bwo guhinduranya amafaranga y’Ibikorwa vya Bitcoin bikeneye gusinywa bimwe bishitse. PSBT ishigikiwe na BTCPay Server kandi ishobora gusinywa n’ibikoresho n’ibikoresho bihuye.


Ukwubaka igikorwa co gucuruza Bitcoin gishizweko umukono ku buryo bushitse bica muri izi ntambwe zikurikira:



- PSBT yubakwa n’ibintu vyinjira n’ibisohoka, ariko nta mikono
- PSBT yoherejwe hanze ishobora kwinjizwa hanze na Wallet ishigikira iyi nzira .
- Amakuru yerekeye amafaranga ashobora gusuzumwa no gushirwako umukono hakoreshejwe urupapuro rwa Wallet.
- Dosiye ya PSBT yashizweko umukono irungikwa hanze iva kuri Wallet ikaza ikajanwa na BTCPay Server
- Serveri ya BTCPay itanga amafaranga y'iherezo ya Bitcoin
- Ugenzura igisubizo maze ukagitangaza ku rubuga

#### Gusubira gupima

### Huza urudodo rw'imbere (umuyobozi wa serveri)


Umuyobozi wa Serveri ashobora gukoresha Node y'Umuco yo mu mutima ya Serveri ya BTCPay. Utitaye ku gushirwa mu ngiro kwa Lightning, gufatanya n’uruzitiro rw’imbere rwa Lightning ni co kimwe.


Genda mu bubiko bw'ibintu vya kera, hanyuma ukande kuri "Lightning" Wallet mu rutonde rw'ibubamfu. BTCPay Server itanga uburyo bubiri bwo gutegura: gukoresha urudodo rw’imbere (Umuyobozi wa Server gusa ku buryo busanzwe) canke urudodo rw’imbere (uguhuza hanze). Abarongozi ba server bashobora gukanda kuri "Koresha urudodo rw'imbere". Nta yindi ntunganyo ikenewe. Fyonda kuri buto ya "bika" maze ubone itangazo rivuga ngo, "BTC Lightning node yahinduwe". Ubu iyo duka yararonse ubushobozi bwa Lightning Network.


### Huza urudodo rw'inyuma (umukoresha wa server/nyiri ububiko)


Ku mburabuzi, ba nyen'amaduka ntibemerewe gukoresha umuyobozi wa server Lightning Node. Iryo huriro rikeneye gukorwa ku nzira yo hanze, yaba nzira ifiswe na nyen’iduka imbere yo gushinga Server ya BTCPay, plugin ya LNBank iyo ibonetse n’umuyobozi wa server, canke umuti w’umucungezi nka Alby.

### Rusangi


Mu gice c’Ivyagezwe rusangi, ba nyen’amaduka barashinga ikimenyetso cabo n’ivyo kwishura. Igihe iduka ryatangura gushirwaho, izina ry’iduka ryarahawe; ivyo bizokwerekanwa mu mitunganyirize rusangi iri munsi y'Izina ry'Iduka. Aha, nyen’iduka arashobora kandi gushinga urubuga rwiwe kugira ngo rujane n’ikimenyetso n’Indangamuntu y’Iduka kugira ngo Umuyobozi amenye mu rutonde rw’amakuru.


#### Ikimenyetso


Nk’uko BTCPay Server ari FOSS, nyen’iduka ashobora gukora ivy’ibara ry’ibintu kugira ngo bihure n’iduka ryiwe. Gushinga ibara ry’ikimenyetso, ubike ibimenyetso vy’ikimenyetso cawe, wongereko CSS ku mapaji y’abantu bose/abaguzi (Ifagitire, Ibisabwa vyo kwishura, Gukuraho amahera)


#### Guhembwa


Mu mirongo y’ukwishura, ba nyen’amaduka barashobora gushinga amafaranga y’iduka ryabo (yaba Bitcoin canke amafaranga yose y’amahera).


#### Kwemerera umuntu wese gukora amafagitire


Iyi nzira igenewe abahinguzi canke abahinguzi bari hejuru ya BTCPay Server. Iyo iyi nzira ikoreshejwe ku iduka ryawe, iremesha isi yo hanze gukora amafagitire ku nkuru yawe ya BTCPay Server.


#### Yongerako amafaranga y'inyongera (amafaranga y'urubuga) ku mafagitire


Igikoresho kiri muri BTCPay co kurinda abacuruzi ibitero vya Dust canke abaguzi kugira ngo ntibazokwigere bagira amahera menshi mu nyuma igihe umucuruzi akeneye kwimurira amafaranga menshi ya Bitcoin icarimwe. Nk’akarorero, umukiriya yararemye Invoice ku madolari 20 aca ayishura igice, arishura 1$ incuro 20 gushika Invoice yishuwe yose. Ubu uwo mudandaza arafise igikorwa kinini cane co gucuruza, ivyo bikaba bituma igiciro ca Mining kirushirizaho iyo uwo mudandaza afashe ingingo yo kwimurira ayo mahera mu nyuma. Ku buryo busanzwe, BTCPay ikoresha igiciro c’uruja n’uruza c’inyongera ku giciro cose ca Invoice kugira ngo yishure iyo nsiguro ku mudandaza igihe Invoice yishurwa mu bikorwa vyinshi. BTCPay itanga uburyo bwinshi bwo guhindura iyo nzira y’uburinzi. Ushobora gusaba amafaranga y'urubuga:



- Iyo gusa umukiriya yishuye Invoice incuro zirenga imwe (Mu karorero kari hejuru, iyo umukiriya yaremye Invoice ku 20\$ maze ariha 1\$, Invoice yose ikwiye gutangwa ubu ni 19\$ + amafaranga y’urubuga. Amafaranga y’urubuga akoreshwa inyuma y’ukwishurwa kwa mbere)
- Ku kwishura kwose (harimwo n’ukwishura kwa mbere, mu karorero kacu, umubare wose uzoba 20\$ + amafaranga y’urubuga ubwo nyene, mbere no ku kwishura kwa mbere)
- Ntukigere wongerako amafaranga y'urubuga (uzibira amafaranga y'urubuga yose)


Naho nyene irinda amafaranga y’ubudandaji ya Dust, irashobora kandi gutuma ubudandaji bugira ico bukoze iyo butamenyeshejwe neza. Abakiriya boshobora kuba bafise ibindi bibazo maze bakiyumvira ko uriko urabahemba amahera menshi.


#### Invoice irahera iyo amahera yose atarishwe inyuma?


Igihe ca Invoice gishirwa ku minota 15 ku buryo busanzwe. Ico gihe gikora nk’uburyo bwo kwikingira uguhinduka, kuko gifunga umubare wa Bitcoin hakurikijwe ibiciro vya Bitcoin-to-fiat Exchange. Iyo umukiriya atarishe Invoice mu kiringo categekanijwe, Invoice ifatwa ko yarangije igihe. Invoice ifatwa ko "yishuwe" igihe nyene igikorwa kiboneka kuri Blockchain (ata vyemezo bitagira), kandi "yuzuye" iyo ishitse ku mubare w'ivyemezo umucuruzi yasobanuye (kenshi 1-6). Igihe gishobora guhindurwa n’iminota.


#### Nimuzirikane Invoice yishuwe naho amahera yishuwe yoba ari X% make kuruta ayo mwiteze?


Iyo umukiriya akoresheje Exchange Wallet kugira ngo yishure ataco akora Invoice, Exchange ifata amahera makeyi. Ivyo bisigura ko iyo Invoice itabonwa ko yuzuye. Invoice igaragara ko "yishuwe igice". Ushobora gushinga igipimo c’ijanisha hano nimba umucuruzi ashaka kwemera amafagitire atarishwe.

### Urubuga


Muri Server ya BTCPay, biroroshe cane gukora "Igitabo" gishasha. Mu mategeko y'Iduka rya Serveri ya BTCPay - Urupapuro rw'Ibikoresho, nyen'iduka arashobora gukora urubuga rushasha mu gufyonda kuri "+ Rema urubuga". Webhooks zituma BTCPay Server yohereza ibintu vya HTTP bijanye n’iduka ryawe ku bandi ba serveri canke ubufatanye bw’ubudandaji.


![image](assets/en/061.webp)


Ubu uri mu mbonerahamwe yo kurema Webhook. Raba neza ko uzi URL yawe ya Payload maze uyishire muri Server yawe ya BTCPay. Mu gihe washizeko URL y'umuzigo, munsi yayo yerekana ibanga ry'urubuga. Kopa ibanga rya webhook maze urihe ku mpera. Igihe vyose bimaze gutegurwa, ushobora guhindura muri BTCPay Server ku "Gusubira gutanga ubwavyo." BTCPay Server izogerageza gusubira gutanga amakuru yose yananiwe inyuma y’amasegonda 10, umunota 1, no gushika incuro 6 inyuma y’iminota 10. Ushobora guhinduranya hagati y'ibintu vyose canke ugaragaza ibintu ku vyo ukeneye. Raba neza ko ushoboza urubuga maze ukande kuri buto ya "Ongera urubuga" kugira ngo ubike.


![image](assets/en/062.webp)


Webhooks ntizigenewe guhura na API ya Bitpay. Hariho IPN zibiri zitandukanye (mu majambo ya BitPay: "Imenyekanisha ry'Ukwishura Ubwo nyene") muri Server ya BTCPay.



- Urubuga
- Imenyekanisha


Koresha gusa URL y'Imenyekanisha igihe urema invoice biciye ku Bitpay API.


### Abakora ivyo kwishura


Abakora ivy’ugutanga amahera bakorana n’iciyumviro c’amahera muri BTCPay Server. Igikoresho co gutanga amahera co gukoranya amafaranga menshi no kuyarungika icarimwe. Kubera ko hariho abakora ivy’ugutanga amahera, nyen’iduka arashobora gukoresha ubuhinga bwo kwishura mu buryo bw’imigwi. Server ya BTCPay itanga uburyo bubiri bwo kwishura mu buryo bwikora: On-Chain na off-chain (LN).


Nyir’iduka arashobora gukanda no gutunganya izo nzira zompi zo kwishura zitandukanye. Uwufise iduka yoshobora gushaka gusa gukoresha umurongo wa On-Chain rimwe mu masaha X, mu gihe umurongo wa off-chain woshobora gukoresha mu minota mikeyi. Ku On-Chain, ushobora kandi gushinga intumbero y'ibara ry'ibara ry'ibara ry'ibara ry'ibara ry'ibara. Ku mburabuzi, ibi bishirwa kuri 1 (canke igice gikurikira kiboneka). Urabona ko gushinga umurongo w’ugutanga amahera off-chain ufise gusa igihe c’igihe kandi nta ntumbero y’uguhagarika. Lightning Network vyishyurwa ubwo nyene.


![image](assets/en/063.webp)

![image](assets/en/064.webp)


Abafise amaduka bashobora gusa guhindura processeur ya On-Chain iyo bafise Hot Wallet ifatanye n’iduka ryabo.


![image](assets/en/065.webp)


Uhejeje gushinga umurongo w’ugutanga amahera, urashobora kuwukuraho canke ukawuhindura ningoga mu gusubira ku rubuga rw’umurongo w’amahera mu mitunganyirize y’Iduka rya Server ya BTCPay.


**Inyishu**


Igikoresho co kwishura On-Chain - Igikoresho co kwishura On-Chain gishobora gukora gusa ku iduka ryatunganijwe rifise Hot Wallet ifatanye. Iyo ata Hot Wallet ihuye, Server ya BTCPay ntifata imfunguruzo za Wallet kandi ntizoshobora gukora ivyo kwishura ubwayo.

#### Invoice ibisobanuro

Inyemezabuguzi za Server ya BTCPay zishobora kwoherezwa hanze mu buryo bwa CSV canke JSON. Kugira ngo umenye vyinshi ku bijanye n’ugushora Invoice hanze n’ugucungera amafaranga.

#### Gushira mu bubiko amafagitire


Nk’inyishu y’uko ata Address yosubira gukoresha ikintu ca BTCPay Server, birasanzwe kubona amafagitire menshi yarangije igihe kuri paji ya Invoice y’iduka ryawe. Kugira ngo ubihishe mu vyo ubona, bihitemwo muri list maze ubishireko ikimenyetso c’uko biri mu bubiko. Inyemezabuguzi zashizweko ikimenyetso c’uko zishizwe mu bubiko ntizifutwa. Ukwishura kuri Invoice yashizwe mu bubiko bizoguma bimenyekana na Server yawe ya BTCPay (ikibanza ca paidLate). Ushobora kubona inyemezabuguzi z’ububiko bw’iduka igihe cose uhisemwo inyemezabuguzi z’ububiko mu bubiko bw’akayunguruzo k’ubushakashatsi.

- Igihe gishingiye ku gutanga amafaranga y’abakozi bikorera ku giti cabo (aho umuntu akoresha yemera ko uwo mukozi yikorera ku giti ciwe ashobora gukura amahera muri Wallet yiwe uko igihe kigenda kiramenyekana)

### Ivyo kwishura


Ivyo kwishura biboshe ku [Gukuraho Ivyishyurwa](https://docs.btcpayserver.org/Gukuraho Ivyishyurwa/). Iyi nzira ishobora kugufasha gukora amafaranga yo kwishura muri BTCPay yawe. Iki kintu kigufasha gukora ivyo kwishura (gusubiza amahera, kwishura umushahara, canke gukura amahera).


#### Akarorero ka 1: Gusubiza amahera


Reka dutangure n’akarorero ko gusubizwa amahera. Umukiriya yaguze ikintu mw'iduka ryawe, ariko ikibabaje, bategerezwa kukigarura. Bashaka ko basubizwa amahera. Muri BTCPay, urashobora gukora [Gusubizwa](https://docs.btcpayserver.org/Gusubizwa/) maze ugaha umukiriya uruja n’uruza rwo gusaba amahera yiwe. Umukiriya amaze gutanga Address yiwe kandi akayisaba, izokwerekanwa mu gice c’Ivyishurwa.


Igitigiri ca mbere gifise ni Ukurindira Kwemererwa. Abakozi bo mu maduka barashobora kumenya nimba hari benshi barindiriye, hanyuma umaze guhitamwo, ukoresha buto Ibikorwa.


Amahitamwo kuri buto y'igikorwa



- Kwemeza amafaranga yatowe
- Kwemeza no kohereza amafaranga yatowe
- Gukuraho amafranga yatowe


Intambwe ikurikira ni ukwemeza & kohereza amafaranga yatoranijwe, nk’uko dushaka gusubiza umukiriya. Suzuma Address y’Umukiriya, yerekana amahera be n’uko twipfuza ko ayo mahera akurwa mu vyo asubizwa canke atarivyo. Uhejeje gukora amasheki, gusinya ku giciro ni intambwe yonyene isigaye.


Ubu umukiriya araronka amakuru mashasha kuri paji y’Ibisabwa. Ashobora gukurikirana ivy'ugucuruza nk'uko yahawe uruja n'uruza rwa Block explorer n'ugucuruza kwiwe. Igihe igikorwa cemejwe, ikibanza caco kirahinduka kikaja 'Kwararangiye'.


#### Akarorero ka 2: Umushahara


None rero reka twinjire mu gutanga umushahara, kuko ivyo biva imbere mu iduka atari nk’uko Umukiriya asaba. Iciyumviro kiri mu vyo ni co kimwe; ikoresha amahera y’ugukwega. Ariko aho kurema ugusubizwa amahera, tuzokora [Ugukuraho Ukwishura](https://docs.btcpayserver.org/Ugukuraho/).


Genda ku rubuga rw'Ivyishyurwa vy'Ivyishyurwa muri server yawe ya BTCPay. Hejuru iburyo, kanda kuri Buto Rema Gukura Ukwishura.


Ubu turi mu kurema Payout, uyihe izina n’amahera wipfuza mu mafaranga watoye. Uzuza Insobanuro, kugira ngo umukozi amenye ico bivuga. Igice gikurikira ni nk’ico gusubiza amahera. Uwo mukozi yuzuza urupapuro rwa Destination Address n’amahera ashaka gusaba muri iyo Payout. Ashobora gufata ingingo yo kubigira 2 claims zitandukanye, ku ma adresse atandukanye, canke mbere igice c’ikirego ku miravyo.


Niba hariho amafaranga menshi arindiriye, ushobora kuyashira hamwe kugira ngo asinywe maze yoherezwe. Iyo umaze gusinya, amahera yishurwa aja ku rubuga In Progress maze yerekana Transaction. Iyo yemejwe n’urubuga, amahera yishurwa aja ku rubuga rwitwa Completed. Igipande cuzuye ni co gusa c’intumbero z’amateka. Ifise amafaranga yishurwa yatunganijwe n'amafaranga ari ayayo .

### Ibishirwamwo mburabuzi na porogaramu


Server ya BTCPay izana n’ibikoresho bishobora guhindura Server ya BTCPay ngo ibe irembo ry’ukwishyura ry’ubudandaji. Iyo wongeyeko Point Of sale, urubuga rwa Crowdfund, n’ubuto bwo kwishura bworoshe, BTCPay Server iba umuti woroshe gukoresha.


### Ikibanza co kugurisha


Imwe mu nzira zisanzwe za BTCPay Server ni ahantu h’ugurisha (PoS). Hakoreshejwe plugin ya PoS, nyen’iduka arashobora gukora Webshop ataco akoresheje Server ya BTCPay; nyen’iduka ntakeneye inyishu z’ubudandaji bwo kuri interineti z’uwundi muntu kugira ngo akoreshe Webshop. PoS ishingiye ku rubuga irashoboza abakoresha bafise amaduka y’amatafari n’amabuye kwemera bitagoranye Bitcoin, ata mahera canke uwundi muntu, ataco baciye muri Wallet yabo. PoS ishobora kugaragazwa bitagoranye kuri tablettes canke ku bindi bikoresho bifasha gukoresha urubuga. Abakoresha barashobora gukora mu buryo bworoshe inzira ngufi y’ibarabara ry’imbere kugira ngo bashobore kwihuta gushika kuri porogarama y’urubuga.


#### Uko wokora ikibanza gishasha co kugurishako


BTCPay Server ishoboza ba nyen’amaduka gukora ningoga Point of Sale mu mice myinshi. BTCPay Server iratahura ko atari iduka ryose ari ubudandaji bwo kuri interineti, kandi atari iduka ryose ari ibarabara canke iresitora, kandi rizana n’imiterere myinshi y’imirongo ngenderwako y’i PoS yawe.


Iyo nyen'Iduka akanze kuri "Ikibanza co kugurisha" mu murongo wiwe w'ibubamfu, BTCPay Server ubu izosaba izina; iri zina rizoboneka mu murongo w'ibubamfu. Fyonda kuri Create kugira ngo ureme PoS.
