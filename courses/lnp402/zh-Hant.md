# 提升您的 LN 開發技能

歡迎使用 SDK 開啟您的 LN 之旅。

在本課程中，您將學習 Rust 書本的基本知識，接著使用 SDK 進行一些 LN 程式設計，最後再進行一些實際練習。我們來自不同背景的老師會引導您學習實用技能，並解釋現今 LN 工程師經常面對的各種挑戰。

本課程是在 2023 年 10 月 LN Tuscany 活動期間由 Fulgur'Ventures 舉辦的 LIVE 研討會中拍攝。
