## 線上瀏覽

<chapterId>3a935da9-fa6e-57eb-bf85-7b3ec35e6ee2</chapterId>

瀏覽網際網路時，必須避免某些常見錯誤，以維護您的線上安全。以下是一些避免這些錯誤的提示：

### 謹慎使用軟體下載：

建議從出版商的官方網站而非一般網站下載軟體。

範例：使用 www.signal.org/download 取代 www.logicieltelechargement.fr/signal。

此外，建議您優先使用開放原始碼軟體，因為這些軟體通常較為安全，而且不含惡意軟體。開放原始碼」軟體是指其程式碼為所有人所知且可存取的軟體。這可以驗證，除其他事項外，是否有隱藏的存取權限來竊取您的個人資料。

> 獎勵：開放源碼軟體通常都是免費的！這所大學 100% 開放原始碼，因此您也可以在我們的 GitHub 上查看我們的程式碼。

### Cookie 管理：錯誤與最佳作法

Cookie 是網站為了在您的裝置上儲存資訊而建立的檔案。雖然有些網站需要這些 Cookie 才能正常運作，但它們也可能被第三方網站利用，尤其是用於廣告追蹤目的。根據 GDPR 等法規，您可以（並建議）拒絕第三方追蹤 Cookie，但同時接受網站正常運作所需的 Cookie。每次造訪網站後，最好手動或透過擴充套件或特定程式刪除相關 cookies。有些瀏覽器甚至提供選擇性刪除 cookie 的功能。儘管有這些預防措施，但重要的是要了解不同網站所收集的資訊仍會互相連結，因此在便利性與安全性之間找到平衡點是很重要的。

> 注意：同時限制安裝在瀏覽器上的擴充套件數量，以避免潛在的安全性和效能問題。
### 網頁瀏覽器：選擇、安全性

瀏覽器有兩大系列：基於 Chrome 的瀏覽器和基於 Firefox 的瀏覽器。

雖然這兩個系列都提供類似的安全層級，但建議避免使用 Google Chrome 瀏覽器，因為它有追蹤程式。Chrome 瀏覽器的輕量替代品，如 Chromium 或 Brave，可能是首選。Brave 的內建廣告封鎖程式尤其值得推薦。可能需要使用多種瀏覽器才能存取某些網站。

![](assets/notext/6.webp)

### 隱私瀏覽、TOR 及其他可讓您更安全匿名瀏覽的替代方案

隱私瀏覽雖然不會向您的網路服務供應商隱藏瀏覽內容，但可以讓您不在電腦上留下本機痕跡。Cookie 會在每次會話結束時自動刪除，讓您可以接受所有 Cookie 而不被追蹤。隱私瀏覽在購買線上服務時非常有用，因為網站會追蹤我們的搜尋習慣，並據此調整價格。但是，需要注意的是，私人瀏覽建議用於臨時和特定的會話，而不是一般的網路瀏覽。

另一個更先進的選擇是 TOR (The Onion Router) 網路，它透過遮蔽使用者的 IP Address 來提供匿名性，並允許存取 Darknet。TOR 瀏覽器是專為使用 TOR 網路而設計的瀏覽器。它允許您訪問傳統網站和 .onion 網站，這些網站通常由個人操作，可能具有非法性質。

TOR 是合法的，記者、自由活動人士和其他希望逃避專制國家審查的人都使用 TOR。然而，重要的是要了解 TOR 並不能確保瀏覽的網站或電腦本身的安全。此外，使用 TOR 可能會導致網際網路連線速度變慢，因為資料在抵達目的地之前會先經過三台其他人的電腦。還必須注意的是，TOR 並非保證 100% 匿名性的萬無一失的解決方案，不應該用於非法活動。

## VPN 和網際網路連線

<chapterId>5aac83f4-a685-54b0-9759-d71bea7eeed2</chapterId>

### VPN

保護您的網際網路連線是線上安全的重要一環，而使用虛擬私人網路 (VPN) 是加強此安全性的有效方法，不論是對企業或個人使用者而言皆然。

VPN 是一種工具，可加密透過網際網路傳輸的資料，使連線更加安全。在專業環境中，VPN 讓員工可以安全地遠端存取公司內部網路。交換的資料經過加密，讓第三方更難截取。除了確保存取內部網路的安全外，使用 VPN 還可讓使用者透過公司內部網路路由其網際網路連線，讓人以為他們的連線是來自公司。這對於存取受地理限制的線上服務尤其有用。

![](assets/en/8.webp)

### VPN 種類

VPN 有兩種主要類型：企業 VPN 和消費者 VPN，例如 NordVPN。企業 VPN 往往較昂貴且複雜，而消費者 VPN 一般較容易使用且容易上手。例如，NordVPN 允許使用者透過位於另一個國家的伺服器連線至網際網路，可以繞過地理限制。

然而，使用消費者 VPN 並不能保證完全匿名。許多 VPN 供應商都會保留使用者的相關資訊，這可能會損害使用者的匿名性。雖然 VPN 對於改善線上安全性很有用，但它們並非通用的解決方案。它們對於某些特定用途是有效的，例如存取受地理位置限制的服務或改善旅行時的安全性，但它們無法保證完全安全。選擇 VPN 時，必須優先考量可靠性和技術性，而非知名度。收集個人資訊最少的 VPN 供應商通常是最安全的。iVPN 和 Mullvad 等服務不收集個人資訊，甚至允許以 Bitcoin 付款以增加隱私。

![](assets/notext/9.webp)

最後，VPN 也可以用來阻擋線上廣告，提供更愉快、更安全的瀏覽體驗。然而，重要的是要自行研究，找到最適合您特定需求的 VPN。建議使用 VPN 以加強安全性，即使在家瀏覽網際網路時也是如此。這有助於確保線上交換資料的安全性更高。最後，請務必檢查網址和 Address 欄中的小掛鎖，以確認您正在您打算造訪的網站。

### HTTPS 與公共 Wi-Fi 網路的

就線上安全性而言，必須瞭解 4G 通常比公共 Wi-Fi 更安全。但是，使用 4G 會很快耗盡您的行動數據計劃。HTTPS 協定已成為網站資料加密的標準。它可以確保使用者與網站之間交換的資料是安全的。因此，確認您造訪的網站是否使用 HTTPS 通訊協定至關重要。

在歐盟，資料保護受《一般資料保護條例》（GDPR）規範。因此，使用歐洲的 Wi-Fi 存取點供應商（例如 SNCF）會比較安全，因為他們不會轉售使用者的連線資料。然而，僅僅是網站顯示掛鎖並不能保證其真實性。使用證書系統驗證網站的公開金鑰來確認其真實性是很重要的。雖然資料加密可防止第三方截取交換的資料，但惡意個人仍有可能冒充網站，以純文字傳輸資料。

為了避免線上詐騙，確認您所瀏覽網站的身份是非常重要的，尤其是要檢查副檔名和網域名稱。此外，要警惕那些在 URL 中使用相似字母來欺騙使用者的騙子。

## 電腦使用

<chapterId>16745632-b56b-5423-9873-ddf70fdf1efd</chapterId>

在現今的數位世界中，我們電腦的安全性是大家最關心的議題。今天，我們將 Address 三個重點：


- 選擇電腦

### 選擇電腦和作業系統

關於電腦的選擇，新舊電腦在安全性方面沒有顯著差異。但是，作業系統之間存在安全性差異：Windows、Linux 和 Mac。

![](assets/notext/11.webp)

關於 Windows，建議不要每天使用管理員帳戶，而是建立兩個獨立的帳戶：一個管理員帳戶和一個日常使用的帳戶。由於 Windows 使用者眾多，而且很容易從使用者切換到管理員，因此通常較容易受到惡意軟體的威脅。另一方面，威脅在 Linux 和 Mac 上則較不常見。

作業系統的選擇應基於您的需求和喜好。Linux 系統近年來有了長足的進步，變得越來越人性化。Ubuntu 對於初學者來說是一個有趣的選擇，它的圖形化 Interface 非常容易上手。您可以在保留 Windows 的同時，將電腦分割以試用 Linux，但這可能會很複雜。通常最好是使用專用電腦、虛擬機器或 USB 密鑰來測試 Linux 或 Ubuntu。

![](assets/notext/12.webp)

### 軟體更新

關於更新，規則很簡單： **定期更新作業系統和應用程式是必要的。

在 Windows 10 上，更新幾乎是連續不斷的，因此不要阻擋或延遲更新是非常重要的。每年都會發現約 15,000 個漏洞，突顯出保持軟體更新以防範病毒的重要性。一般而言，軟體支援會在發佈後 3 到 5 年間結束，因此必須升級至較高版本，才能繼續從安全性中獲益。

該規則幾乎適用於所有軟體。事實上，更新的目的不是讓您的機器過時或變慢，而是保護它免受新的威脅。有些更新甚至被視為主要更新，如果沒有這些更新，您的電腦就會面臨嚴重的攻擊風險。

舉一個具體的錯誤例子：無法更新的破解軟體代表著雙重潛在威脅。病毒會在從可疑網站非法下載的過程中出現，而且使用不安全，無法抵禦新形式的攻擊。

### 防毒軟體


- 您需要防毒軟體嗎？需要
- 您必須付款嗎？視情況而定！

防毒軟體的選擇和執行非常重要。Windows Defender 是 Windows 內建的防毒軟體，是安全且有效的解決方案。對於免費的解決方案而言，它是非常好的，比網路上許多免費的解決方案好得多。事實上，從網際網路上下載的防毒軟體可能是惡意或過期的，因此必須謹慎使用。

對於希望投資於付費防毒軟體的使用者，建議選擇可智慧型分析未知與新興威脅的防毒軟體，例如 Kaspersky。防毒軟體更新對於防禦新的威脅至關重要。

![](assets/notext/13.webp)

> 注意：Linux 和 Mac 因其使用者權限分離系統，通常不需要防毒軟體。
最後，這裡有一些保障電腦和資料安全的好方法。選擇有效且方便使用的防毒軟體非常重要。在您的電腦上採取良好的作業方式也很重要，例如不要插入不明或可疑的 USB 鑰匙。這些 USB 隨身碟可能含有惡意程式，插入後會自動啟動。一旦插入 USB 密鑰，檢查 USB 密鑰將毫無用處。有些公司曾因不小心將 USB 隨身碟遺留在停車場等容易接觸的地方，而成為駭客入侵的受害者。

像對待自己的家一樣對待您的電腦：保持警覺、定期更新、刪除不必要的檔案，並使用強大的密碼以確保安全性。加密筆記型電腦和智慧型手機上的資料對防止盜竊或資料遺失至關重要。Windows 的 BitLocker、Linux 的 LUKS 以及 Mac 的內建選項都是資料加密的解決方案。建議您毫不猶豫地啟動資料加密，並將密碼寫在紙上，放在安全的地方。

總而言之，選擇適合您的作業系統，並定期更新以及安裝的應用程式是非常重要的。此外，還必須使用有效且方便使用的防毒軟體，並採取良好的作業方式，以確保電腦和資料的安全。

### 駭客如何攻擊？

為了好好保護自己，了解駭客如何嘗試滲透您的電腦是非常重要的。事實上，病毒通常不會神奇地出現，而是我們的行為所造成的後果，甚至是無意中造成的！

### 網路釣魚，警惕詐騙電子郵件：

請注意！電子郵件是第一個攻擊媒介，以下是一些提示：


- 對旨在擷取您的憑證和密碼等敏感資訊的網路釣魚嘗試保持警覺。避免點擊可疑的連結，以及在未確認寄件者合法性的情況下分享您的個人資訊。
- 謹慎處理電子郵件附件和圖片：

電子郵件的附件和影像可能含有惡意軟體。請勿下載或開啟來歷不明或可疑寄件者的附件，並確保您的防毒軟體是最新的。

這裡的黃金法則是仔細檢查寄件者的全名以及電子郵件的來源。如有疑問，請刪除！

### 贖金軟體和網路攻擊類型：

贖金軟體是一種惡意軟體，會加密使用者資料，並要求贖金才能解密。這種類型的攻擊越來越常見，對公司或個人來說可能非常麻煩。為了保護自己，必須建立最敏感檔案的備份！這並不能阻止贖金軟體，但可以讓您直接忽略它。

![](assets/notext/14.webp)

定期將重要資料備份到外部儲存裝置或安全的線上儲存服務。這樣，在發生網路攻擊或硬體故障時，您可以在不遺失重要資訊的情況下復原資料。

簡單的解決方案：


- 購買外接式 Hard 硬碟機，將資料複製到硬碟機上。這樣做兩次，並將其中一個硬碟機存放在另一個地方，有助於防止潛在火災）。
- 使用 ProtonMail Drive、Sync 或甚至 Google Drive 建立「雲端」備份。只需將您的敏感資料上傳至此線上主機即可。然而，請注意您的資料可能已上傳至網際網路，並由可信賴的第三方持有。

### 您是否應該付錢給駭客？

不，在發生贖金軟體或其他類型的攻擊時，一般不建議向駭客支付贖金。支付贖金並不能保證復原您的資料，而且會鼓勵網路罪犯繼續進行惡意活動。相反，請優先進行預防並定期備份您的資料，以保護您自己。

如果您在電腦上偵測到病毒，請中斷電腦與網際網路的連線，執行完整的防毒掃描，並刪除受感染的檔案。然後更新您的軟體和作業系統，並變更密碼以防止進一步入侵。

## 密碼管理員

<chapterId>0b3c69b2-522c-56c8-9fb8-1562bd55930f</chapterId>

### 什麼是密碼管理器？

密碼管理器是一種工具，可讓您儲存、generate 和管理不同線上帳戶的密碼。與其記住多個密碼，您只需要一個主密碼就可以訪問所有其他密碼。

有了密碼管理器，您再也不用擔心忘記密碼或把密碼寫在某個地方。您只需要記住一個主密碼。此外，這些工具大多會為您提供 generate 強密碼，以加強您帳戶的安全性。

### 一些受歡迎的管理員之間的差異：


- LastPass：最受歡迎的管理器之一。它是第三方服務，這表示您的密碼會儲存在他們的伺服器上。它提供免費版本和付費版本，Interface 使用方便。
- Dashlane：它也是一項第三方服務，擁有直覺的 Interface 和其他功能，例如追蹤信用卡資訊和安全備註。

![](assets/notext/17.webp)

### 自行託管以獲得更多控制權：


- Bitwarden：它是一個開放原始碼的工具，這表示您可以檢閱其程式碼來驗證其安全性。雖然 Bitwarden 提供託管服務，但它也允許使用者自行託管，這表示您可以控制密碼的儲存位置，可能提供更多安全性和控制。
- KeePass：這是一個開放原始碼解決方案，主要用於自我託管。您的資料預設儲存於本機，但如果您願意，也可以使用不同的方法同步密碼資料庫。KeePass 因其安全性和靈活性而廣受認可，不過對於初學者來說，它可能略顯不便。

![](assets/notext/18.webp)

(註：選擇第三方服務或自行託管服務，取決於您對技術的適應程度，以及您對控制權與便利性的優先順序。一般而言，第三方服務對大多數人而言較為方便，而自行託管則需要較多技術知識，但在安全性方面可提供更多的控制與安心)。

### 什麼是好的密碼：

好的密碼一般是：


- 長：至少 12 個字元。
- 複雜：大小楷字母、數字和符號的混合。
- 唯一性：請勿在不同的帳戶中重複使用相同的密碼。
- 非基於個人資訊：避免出生日期、姓名等。

為了確保您帳戶的安全性，建立強大且安全的密碼至關重要。密碼的長度不足以確保其安全性。字元必須完全隨機，才能抵抗暴力攻擊。事件的獨立性對於避免最可能的組合也很重要。常見的密碼如 "password "很容易被破解。

要建立強大的密碼，建議使用大量隨機字元，不要使用可預測的字或模式。此外，包含數字和特殊字符也很重要。但是，需要注意的是，某些網站可能會限制使用某些特殊字符。非隨機產生的密碼很容易被猜中。密碼的變更或增加都是不安全的。網站無法保證使用者所選密碼的安全性。

隨機產生的密碼提供較高層級的安全性，儘管它們可能較難記。密碼管理器可以 generate 更安全的隨機密碼。使用密碼管理器，您不需要記住所有密碼。重要的是要逐步用管理器生成的密碼取代舊密碼，因為舊密碼更強更長。請確保您密碼管理器的主密碼也是強大且安全的。

## 雙因素驗證

<chapterId>9391e02e-e61b-5a86-93e0-91a07f217d35</chapterId>

### 為何實施 2FA

雙因素驗證 (2FA) 是額外的 Layer 安全機制，用來確保嘗試存取線上帳戶的人是他們所聲稱的人。2FA 不需要輸入使用者名稱和密碼，而是需要第二種驗證方式。

這第二步可以是：


- 透過 SMS 傳送的臨時代碼。
- 由 Google Authenticator 或 Authy 等應用程式產生的驗證碼。
- 您插入電腦的實體安全鑰匙。

![](assets/notext/19.webp)

有了 2FA，即使駭客取得您的密碼，如果沒有第二個驗證因素，他們也無法存取您的帳戶。因此，2FA 對於保護您的線上帳戶免受未經授權的存取是非常重要的。

### 選擇哪個選項？

強大驗證的不同選項提供不同程度的安全性。


- SMS 不被視為最佳選擇，因為它只能提供擁有電話號碼的證明。
- 2FA (雙因素認證) 更為安全，因為它使用多種類型的證據，例如知識、持有和識別。一次性密碼 (HOTP 和 TOTP) 比 SMS 更安全，因為它們需要加密計算，並儲存在本機而非記憶體中。
- 硬體令牌 (例如 USB 鑰匙或智慧卡) 可為每個網站產生唯一的私人密碼匙，並在允許連線前驗證 URL，提供最佳的安全性。

為了以強大的驗證達到最佳安全性，建議使用安全的電子郵件 Address、安全的密碼管理器，並使用 YubiKeys 進行 2FA。此外，建議您購買兩個 YubiKeys 以預防遺失或被盜，例如在家中和身上各保留一份備份。

至於 SIM 2FA 的潛在威脅，這裡有一個常見的例子：當攻擊者將使用者的電話號碼連結到攻擊者所控制的 SIM 卡，從而竊取使用者的電話號碼時，就會發生 SIM 卡交換攻擊。攻擊者有幾種方式可以成功完成攻擊；但是，這種威脅通常只會對高檔案和有興趣的人造成重大威脅。

生物辨識技術可作為替代品，但其安全性比不上知識與擁有的結合。生物辨識資料應保留在驗證裝置上，不得在線上洩露。重要的是要考慮與不同驗證方法相關的威脅模型，並據此調整作法。

最後，我們不妨簡單介紹一下 HOTP 和 TOTP OTP：HOTP 是基於 HMAC（Hash-based Message Authentication Code，Hash 訊息驗證碼）演算法的一次性密碼，而 TOTP 則是基於時間的 OTP。這類演算法的主要特點是密碼只能使用一次，每個產生的值都是獨一無二的，並且在使用者的裝置 (用戶端) 和驗證服務 (伺服器) 之間存在共用金鑰。這兩種系統的差異在於因子的移動方式：TOTP 是以時間為基礎，而 HOTP 系統則是以計數器為基礎。

### 獲得實際經驗：

實務經驗的重要性不容小覷。在擁有網路安全團隊的公司尋求實習機會或初級職位，是應用理論知識的絕佳方式。此外，參與道德駭客競賽或網路安全模擬，也能在真實情境中精進您的技能。

專業網路的力量是無價的。加入專業協會、駭客空間或線上論壇可提供一個與其他專家交流 Exchange 想法的平台。同樣地，參加網路安全會議和研討會不僅能讓您學習，還能幫助您與業界專業人士建立聯繫。

威脅的不斷演變需要定期監控新聞和專業論壇。在這個信任至高無上的行業中，以道德和誠信行事在您職業生涯的每個階段都至關重要。

### 高效的密碼管理與認證強化：學術方法

在 Découvre Bitcoin 在學院內提供的「安全 101」訓練單元中，我們討論了密碼管理員的重要性。必須考慮三個層面：在網站上建立、更新和實施密碼。

一般不建議使用瀏覽器擴充套件來自動填寫密碼。這些工具會讓使用者更容易受到網路釣魚攻擊。Renaud 是公認的網路安全專家，他比較喜歡使用 KeePass 進行手動管理，也就是手動複製和貼上密碼。擴充套件往往會增加攻擊面，可能會導致瀏覽器效能變慢，因此會帶來很大的風險。因此，建議在瀏覽器上使用最少的擴充套件。

密碼管理器通常鼓勵使用額外的驗證因素，例如雙因素驗證。為了達到最佳的安全性，建議您在行動裝置上保留 OTP (一次性密碼)。AndoTP 提供開放原始碼解決方案，可在手機上產生和儲存 OTP 代碼。雖然 Google Authenticator 允許匯出驗證碼種子，但對於 Google 帳戶上的備份，信任度仍然有限。因此，建議使用 OTI 和 AndoTP 應用程式來自主管理 OTP。

![](assets/notext/21.webp)

數位繼承和數位哀悼的問題，提出了在一個人死後，有一個傳輸密碼的程序的重要性。密碼管理器可將所有數位秘密安全地儲存在一個地方，從而促進這種轉移。密碼管理器也可以識別所有開啟的帳戶，並管理其關閉或轉移。建議將主密碼寫在紙上，但應保存在隱蔽且安全的地方。如果 Hard 磁碟機已加密且電腦已上鎖，即使發生盜竊案，也無法取得密碼。

### 邁向後密碼時代：探索可信的替代方案

密碼雖然無處不在，但卻有許多缺點，包括在驗證過程中可能發生風險傳輸。微軟和 Apple 等領導廠商提供了創新的替代方案，例如生物識別技術和硬體令牌，顯示出拋棄密碼的漸進趨勢。

例如，Passkeys 提供加密隨機鑰匙，並結合本地因素 (生物特徵或 PIN)，這些因素由供應商託管，但仍在他們的能力範圍之外。雖然這需要更新網站，但這種方法不需要密碼，因此提供了高層級的安全性，卻沒有傳統密碼的限制或數位保險箱的管理問題。

Passkiz 是另一種可行且安全的密碼管理選擇。然而，一個主要的問題仍然存在：在供應商失敗時的可用性。因此，互聯網巨頭最好能提出系統來保證這種可用性。

直接驗證到相關服務是一個有趣的選擇，可以不再依賴第三方。然而，網際網路巨擘所提供的單一登入 (SSO) 也會在可用性和審查風險方面造成問題。為了防止資料洩漏，將驗證過程中收集的資訊數量降至最低是非常重要的。

### 電腦安全：安全作業的必要條件與人為疏失的相關風險

簡單的做法和使用預設密碼 (例如「admin」) 都可能危害電腦安全。複雜的攻擊不一定會危害電腦安全。例如，YouTube 頻道的管理員密碼是以公司的私人原始碼寫成的。安全漏洞通常是人為疏忽的結果。

還應該注意的是，網際網路是高度集中的，主要由美國人控制。DNS 伺服器可能會受到審查，並經常採用欺騙性的 DNS 來封鎖某些網站的存取。DNS 是一種老舊且安全性不足的通訊協定，可能會導致安全問題。新的通訊協定（如 DNSsec）已經出現，但仍未被廣泛使用。要繞過審查和廣告封鎖，可以選擇其他 DNS 供應商'。

侵擾性廣告的替代方案包括 Google DNS、OpenDNS 及其他獨立服務。標準 DNS 通訊協定會讓網際網路服務供應商看到 DNS 查詢。DOH (DNS over HTTPS) 和 DOT (DNS over TLS) 加密 DNS 連線，提供更高的隱私性和安全性。這些通訊協定因其強化的安全性而在企業中廣泛使用，並由 Windows、Android 和 iPhone 原生支援。若要使用 DOH 和 DOT，必須輸入 TLS 主機名稱，而非 IP Address。您可在線上取得免費的 DOH 和 DOT 供應商。DOH 和 DOT 可避免「中間人」攻擊，從而改善隱私和安全性。其他主要注意事項

作為 Découvre Bitcoin 學院「安全 101」訓練模組的一部分，我們也討論了 Lightning 認證。此系統可為每個服務產生不同的識別碼，而無需提供電子郵件 Address 或個人資訊。由使用者控制分散式身分是可能的，但分散式身分專案缺乏標準化與規範化。為了避免惡意攻擊，建議使用 Nuget 和 Chocolaté 等套件管理器，這些套件管理器允許下載微軟商店以外的開放原始碼軟體。總括而言，DNS 對於線上安全至關重要，但必須對 DNS 伺服器可能遭受的攻擊保持警覺。
